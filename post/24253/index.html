关于 NFT 以疯狂的价格出售的消息越来越少。

甚至一些名人似乎也无法将他们的 NFT 项目推向高潮。

那么炒作结束了吗？泡沫会破灭吗？

就目前而言，NFT看起来是一条闪闪发光的致富之路，尽管实际的买卖情况并不完全清晰。至今最著名的例子是，艺术家Beeple的一件作品被佳士得(Christie’s)以6900万美元的天价拍卖。佳士得称这是“大型拍卖行推出的第一件纯数字作品。”

为什么买家要花那么多钱买一件可以完全复制并被任何人免费欣赏的作品呢?中标者得到了保存在区块链上的文件，该文件赋予了中标者对该艺术品名义上的“所有权”。但这个词并不意味着任何形式的独家访问。许多NFT交易甚至不包括版权或复制权利。在这种情况下，“所有权”只是为买家提供了所谓的“炫耀权利”，理论上，这些权利可以转移到另一个买家手中。换句话说，买家获得的是几行计算机代码的专有权，而不是有形资产。

但是， 在现实中，代币化的数字艺术只是许多可能的NFT之一。简单地说，NFT是通用的真实性数字证书。任何能从区块链的所有权转移、可验证和互操作中获益的事物，将来都可能被NFT所代表。

NFT的来龙去脉

NFT 的早期版本可以追溯到 2012 年，当时创造了“彩色硬币”，这种小面额的比特币代表了任意数量资产的所有权。五年后出现了CryptoKitties，这是一款在线游戏，允许玩家在区块链上收养和交易虚拟猫。不久之后，对NFT（意味着独特且不可替代）的需求激增，以涵盖任何可以想象的创作——甚至是纽约时报的专栏，“售价”为560,000 美元。（当然，《纽约时报》保留版权。）

NFT的现有问题

耗电量大： 我们不妨从最大和报道最多的问题开始。NFT 目前主要在以太坊上铸造和交易。该区块链使用工作量证明共识机制，比特币和其他流行的区块链也是如此。

众所周知，保持这些网络安全所需的工作量证明和挖矿需要大量电力，而在气候变化成为一个巨大问题的时代，这是不可接受的。质押证明共识机制是新一代区块链技术，它允许在没有相关能源需求的情况下实现安全的去中心化网络。

大多数新链现在都在使用 PoS，甚至以太坊也在努力在未来几年内摆脱 PoW。

Gas 费用： 区块链上的交易不是免费的，它们需要付费。这是防止拒绝服务攻击的一种简单而有效的安全方法。然而，当必须以区块链代币支付费用时，这可能会引起问题，该代币的估值随加密货币市场的变化而变化。

对于较旧的区块链，例如具有高法定价值代币的以太坊，这意味着以 ETH 支付的 gas 费用会非常昂贵。这个问题有解决方案。像双代币经济这样的想法为稳定的廉价gas费和高价值代币提供了一种方式。

密钥管理和用户知识： 区块链具有一些独特的特征。其中最重要的一项是密钥管理。在设置存储代币和资产的钱包时，用户必须设置一个或多个称为密钥的唯一密码。如果这些密码丢失，任何人都无法再次访问钱包。

密钥恢复非常有限，甚至不存在，这意味着区块链资产永久丢失是一个非常现实的问题。这主要可以通过为新用户提供良好的入门体验来纠正，以便他们制作良好的密码并正确存储它们。

NFT寻求新的发展

金融NFT

金融NFT包括从保险和债券，到代币化的现实世界资产——它们将成为迄今为止NFT技术的最大单一用途。

最近一个主要的例子出现在Uniswap v3中，它开始以NFT的形式发行流动性供应代币(对做市商提供的代币的索赔)——这是来自单一协议的价值13亿美元的NFT资产。

除了Uniswap外，已经开始以各种方式用于金融目的。这包括:

  • 由 Nexus Mutual 承保的 Yearn.finance 保险合同在 Rarible 上作为 NFT 出售

  • Solv 协议将投资合约编入 NFT，以提高项目代币的透明度和转移。

  • UFiT DAO 将不同的可替代和不可替代代币组合成“篮子”NFT。

  • 现有的 NFT 与 Charged Particles 的计息 ERC20 代币相结合。

NFT将在DeFi中占据很大比例的价值，这个快速增长的行业已经拥有近600亿美元的价值。仅保守的 10% 的 DeFi 头寸代表 NFT 就相当于 60 亿美元的金融 NFT 资产——而 DeFi 的蛋糕还没有完成增长。

举个例子，现实社会中一家销售豪华房地产的公司可能会使用 NFT 来验证财产的部分所有权。存储在以太坊区块链上，相关的 NFT 取代了对纸质文件的需求。

NFT+DeFi

目前，DeFi使得从可替代代币获得回报和以它们为抵押贷款变得异常容易。由于允许用户这样做，像Compound和Yearn这样的去信任的协议变得非常受欢迎。

但当涉及到NFT时，根本没有多少基础设施可以借用或赚取回报。对于一个价值数十亿美元的新市场来说，这是一个巨大的需求。同样值得一提的是，金融NFT的持有者不仅仅是那些不关心机会成本的艺术收藏家——持有金融 NFT 的人需要考虑每一分钱。

随着潜在的金融 NFT 热潮即将来临，像Drops这样的项目，它可能会解决金融 NFT 持有者在寻找充分利用其金融 NFT 资产的方法时面临的许多挑战。

Drops 是一个多功能平台，使这些金融 NFT 持有者能够以他们的 NFT 资产去信任地借入可替代资产，包括ETH和稳定币。这些贷款可以根据借款人的意愿使用，用于抵押、流动性挖矿，或者只是为了避免在所有者需要快速资金来源时出售 NFT 资产。Drops 计划在 2021 年底之前将流动性挖矿保险库内置到他们的平台中，使 NFT 持有者能够通过质押他们的 NFT 无缝地获得可量化的回报。

结语

我们所有人对 NFT 构想的发展方向也不甚清晰。

这些项目的范围很广，从融入当前游戏世界到处理现实世界的问题，例如活动票务，再到数字身份和社区建设，再到金融领域。
