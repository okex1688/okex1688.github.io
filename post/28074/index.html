2021年是NFT市场爆发增长的一年，不管是第一季度销售额突破20亿美元，还是近期NBA球星库里花费18万美元买下了NFT头像，都证明了NFT目前的火热程度。与此同时，用户对NFT也产生了更深层的理解，没有切入痛点的产品和不能创造真正价值的项目将逐渐被市场所淘汰。同样也是对项目方的一个挑战，激励从业者们创造出更优秀的产品。

DNFT 协议是一个面向 Web3.0 的去中心化跨 NFT 网络。 它允许用户跨区块链铸造、交易和管理他们的 NFT 资产，例如加密艺术、游戏和元宇宙资产以及数据模型。通过将Data-model作为NFT(DAAN)的概念来呈现，DNFT开创了NFT应用于人工智能行业的场景。又将如何实现AI+NFT的落地呢？

AI虽然让人类社会更加地智慧，但一些看似基本的问题其实并没有解决。很多艺术家，除了固定的圈层，他们的创意和作品很难被更广泛的市场和受众接触到。随着成熟的人工智能技术的逐步落地，我们可以开始积极地利用它来帮助肯定，补偿艺术家的辛勤工作，区块链和智能合约与AI的结合，让这一切变得更可能。

NFT的落地已经有一段时间了，市场从最初的疯狂逐渐渐入理性，逐渐沉淀出更多高品质的协议和平台，艺术家也会更放心的把他们的作品上传到平台上进行公平竞投。任何一个市场脱离了理性的发展都会变得投机从而昙花一现。NFT市场也充斥着估价的“过分估高”及“过分看低“的双重异象。似乎传统的艺术品市场和NFT之间并没有”价值共识“。我们一直在思考一个问题，哪一个市场的”价值标准“更准确或者说更理性。

NFT 的稀缺性和独特性是其大规模增长的关键原因。 然而，NFT 的优点导致了一个难以解决的问题，即缺乏流动性。 NFT 跨链资产网络将是这个问题的解决方案。它为生态系统的整合带来了更多的可能性。为解决NFT的跨链流通问题，DNFT将提供怎样的解决方案来实现流动性的赋能呢？
