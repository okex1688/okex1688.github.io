UNI V3 版本在 5月5日发布，对于 V3 的上线，市场还是有不少期待。

UNI  V3 的核心内容有：粒度控制的聚合流动性，多级费率、范围订单、历史预言机和软件许可保护

总的来说，V3 重点是改善资本效率，一切服务资本效率，把同质化的 LP 变成了非同质化的 NFT LP，成了不可替代的流动性。

那么问题来了，在 DeFi 生态中 DEX 中的LP可以参与各种挖矿，现在 LP 变成了 NFT LP，那么接下来怎么挖矿呢？

今天介绍一下基于 UNI V3 开发的项目 Visor，提供了多种 NFT LP 挖矿玩法，或许能玩出新高度呢，一切交给市场检验。

V3 的 LP NFT化，官方解释

在 V3 中的 LP 头寸包含更多自定义输入，例如集中流动性，范围订单和弹性费用，因此所有变量都必须包含在单个不可替代令牌（NFT）中。

对此 Uniswap 解释说，他们期望策略最终将被标记化。

随着时间的流逝，我们期望令牌化越来越复杂的策略，使LP可以参与，同时保持被动的用户体验。这可能包括多个位置，自动重新平衡以集中于市场价格，费用再投资，贷款等。

为了向 V3 LP提供这些功能，需要将特定的NFT外设合同包装在LP NFT周围。

任何人都可以创建一个外围合同，将单个流动性头寸（包括已收取的费用）包装在ERC-721不可替代的代币中。

针对 V3 的 NFT LP，Visor 给出了他的解决方案。

什么是 Visor

Visor.Finance 是开发用于流动性挖掘的NFT智能保管库，因为下一代流动性挖掘将是将NFT Smart Vault引入去中心化金融。

简单说，V3 搞出了 NFT LP，Visor 的目标是把 NFT LP 玩出花来。当然基于 NFT LP 做二次开发，搞新玩法的项目还有很多，后面陆续介绍。

Visor 如何玩 NFT LP

Visor 对于 NFT LP 给出了三个玩法，分别如下：

LP 激励

2020年5月，Ampleforth是在奖励计划中利用这些LP代币的首批项目之一。Ampleforth将一部分代币分配给在Uniswap上提供流动性然后将各自的LP代币存放在他们称为“间歇泉”的奖励合约中的参与者。

从那时到现在，该模型已被采用和重新设计了数百次，并在DeFi中建立了某种标准，即流动性寻求者如何激励流动性提供者。

LP 时间锁

在Uniswap上提供大量的流动性，可以对 NFT LP 进行时间锁定，以便参与者知道除非指定了这些流动性，否则这些流动性就无法消除。

同时，也可以设定 NFT 时长的权重，简单说参与流动性挖矿不退出的时间越长，奖励越大。这点在 Ampleforth 中已经用了快1年多了。

LP 抵押借贷

MakerDAO通过了一项治理提案，该提案允许某些LP代币被接受作为抵押品，以铸造其稳定币Dai。这将允许借款人将LP代币作为协议中的抵押品进行存入，以获取贷款。

这里是简单的基础功能，玩家可以根据需要，通过这些基础功能组合出 NFT LP 的挖矿玩法。

可以预见的是，未来挖矿玩法在 V3 发布之后，会组合出更多玩法，更多激励真正长期看好项目并长期提供流动性的玩家。作为用户，也需要迎接接下来新的玩法。
