行业内外的许多朋友和我聊天时，总会遇到种种类似的问题。对于投资者朋友来说，他们希望把握NFT资产的宏观情况与最新动向，经常会问到持有哪些NFT资产？这些资产估价多少？对于刚接触到NFT的新手来说，会想要了解NFT是什么？如何可以低门槛地购买到NFT？NFT流行趋势是什么？以及，最受欢迎的NFT是什么？等等。

可以看出，以上的问题都涉及到信息的获取和聚合。如今，NFT 迎来爆发式增长。在人人都在关注这个赛道的同时，总会感觉 NFT 世界的信息较为分散。比如，单单所接触到的，就已有成百家国内外在各大不同链上的 NFT 交易所。刚进入 NFT 领域的人很难立刻对此有一个宏观的认识。这个时候，了解赛道的综合数据就显得至关重要。

众所周知，在数字资产诞生以来，CoinMarketCap，CoinGecko已经成为了查找 FT 相关的数据、知识及必不可少的信息入口。而这种形式的平台通常被称为“数据聚合器”（Data Aggregator），“聚合”能够帮助用户直接捕获了绝大多数内容价值。比如 Google 聚合了各个网站的内容，比如亚马逊上面，聚合了成千上万个商家与用户。

因此，对于任何想要深入了解 NFT 的爱好者，都可以透过数据框架和分析，形成对 NFT 行业的清晰的思考框架。

数据定量分析框架

横向数据 - 时间维度

在数据的横向维度上，分为历史数据，近期数据和当前数据等。比如，7D/30D/90D内的 NFT 历史市场趋势，以及最新的NFT市场表现。通过观测时间轴，了解到对应的数据涨跌幅至关重要。此外，一些累计数据，比如每个项目和总累计交易额，总持有者等数据也值得参考。

纵向数据 - 同类比较

纵向维度上，NFT类型/项目间的比较也很重要。在项目类型上，投资者也需要了解不同NFT领域的市场占比，比如交易卡、艺术品、收藏品和虚拟世界等各个NFT类型目前的市场占有情况。特定的NFT项目的交易量占比也值得关注，甚至有些头部项目的交易量总和会超过半数。通过这些数据，用户可以轻易判断项目当前是处于第一梯队，还是第二梯队。周成交额也值得关注，比如是否在10万美元以上，或是在1万美元以上。

颗粒数据 - 细分指标

除了宏观的市场数据，我们也需要细分维度的数据来更加具体的了解NFT项目。

例如项目详情页访问量，呈现在搜索结果的频率，交易者人数，交易额，NFT持有人数，社群活跃度等。对于某个NFT资产，用户也可以通过历史交易数据了解到历史成交价格、购买者或出售者等等。

有了这些数据的支持，投资者更加轻松的持有、管理以及投资NFT资产。比如，最近很火的Bored Ape Yacht Club，每个猴子的发行价是0.08ETH，以及他们后续的价格波动和变化。

我们还可以根据市场成交记录来判断具体 NFT 的潜力。在NFT 市场上，我们可以找到过往的交易数据，尽管历史数据未必可以预示未来，但是在出售和购买时，可以作为定价的参考标准。

此外，项目详情页访问量，呈现在搜索结果的频率，交易者人数，交易额，NFT持有人数，社群活跃度等指标也可以作为影响因子，帮助自己更好地做出投资决策。

代币数据 - 投资机会

在 NFT 代币方面，需要了解它的市值、交易量、上所情况，项目生态，社群建设等方面，哪些项目具有较大潜力，项目的回报率有多少，平均涨幅有多大，以及在不同时间维度上，哪几个项目涨幅最大等等。

研究定性分析逻辑

除了分析 NFT 数据，还需要在哪些方面来判断 NFT 的投资价值呢？有以下三个方面可以参考。

实用性

首先是实用性，实用性价值取决于NFT的使用方式。具有高实用价值的两个主要类别是游戏资产和门票。实用性的另一个层面是在不同的应用中使用NFT的能力。想象一下，如果你能在不同的游戏中使用同一个资产，那么这类 NFT 价值相对会更高。

所有权历史

其次是所有权历史，NFT 价值取决于NFT的发行者和以前的所有者的身份。拥有较高所有权历史价值的NFT通常是由著名艺术家或具有强大品牌的公司创造或发行。你可以通过找到巨鲸持有者的地址，并列出他们拥有的其他NFTs，作为一个指标参考。

流动性

第三流动性，高流动性意味着NFT的价值更高。流动性是衡量 NFT 是否有更高价值的因素之一。符合ERC-721或者ERC-1155标准的NFT可以很容易地在二级市场上与任何持有ETH的人进行交易，这在一定程度上增加了潜在买家的数量。投资者或许更倾向于投资交易量大的NFT类别，因为流动性降低了持有NFT的风险。在相关平台关闭后NFT失去实用价值的极端情况下，只要有人愿意买卖，高流动性的NFT仍有价值。另一方面，不基于以太坊的NFT标准受到缺乏流动性的影响，在这些平台上创建的NFT的价值往往被打折扣。

发现价值，捕获价值

下面以 NFTGO 上的数据为例，进一步分析一下数据的维度的几个要点。

首先，需要关注的是全网 NFT 市场数据，其中包括实时全网 NFT 交易与资产数据，包括 NFT 市值、总交易额、总交易量以及 NFT 及其持有者的数量，用于判断 NFT 整体市场趋势和热度。

其次，可以看一下过去7天/30天/90天内的市场数据走势，以及买方和卖方市场比例等，透过这些宏观数据判断市场供需情况。

正如上文所提到，NFT流动性，某种程度上反映NFT的换手率，是一个非常重要的指标。 这个概念在股票市场中也比较常见，通常会采用换手率来衡量一支股票的流动性强弱。换手率 越高，意味着该支股票的交易越活跃，整体质量高，更具投资价值。

流动性的概念与此类似，可以根据这个指标衡量NFT以一个合理价格顺利变现的能力。尤其是对于那些不打算长期持有NFT，并想要通过NFT买卖获得收益的朋友，流动性指标确实是必须要关注的数据之一。

当前 NFT 市场分布与热门项目是很重要的数据维度之一，从而直观了解当前市场时下最火热的 NFT 细分领域和项目。不管是对于刚了解NFT的新手，还是投资者来说，这些数据都能帮助快速建立对NFT世界的基础认知。同时，通过查看比例变化也能第一时间快速发现热门项目，感知到NFT市场趋势变化。

NFT 排行榜

上述的数据大多是宏观的一些维度，微观的层面上的颗粒数据也同样需要关注。哪些是高价值项目与资产？NFT巨鲸都是谁？NFT 与 NFT Token 投资价值高低？

最典型的是NFT项目的排行榜，包括 NFT 的价值排行，增值最快的项目排行，还有资产价值排行。

在排序维度和方式上，除了市值/增长/交易人数/交易额等维度之外，之前提到过的流动性指标。通过排序，基本上可以快速发现目前市场流通性最高的NFT项目，捕捉到潜在的投资机会。

此外，值得一提的是巨鲸排行榜。除了可以快速知道巨鲸的NFT持有数和价值，更能窥探巨鲸具体持有的NFT及购买动向。身边有些朋友确实会选择去追随巨鲸的步伐，了解市场重要玩家行踪，这也是探索市场趋势的关键一环。

虽然这不能构成科学的投资依据，但通过了解巨鲸持有了哪些NFT，购入或出售了哪些NFT，大家或许能发现一些投资机会点。

NFT 元数据/自然语言搜索

元数据也叫Metadata，每个NFT通行证都会有一个自己的Metadata URl，里面定义了NFT一些更详细的信息，比如NFT的名称、描述、图片的URL以及特殊属性。以 Bored Ape Yacht Club 为例，每只猿猴都有不同的颜色、形状、名称等。通过元数据进行搜索，可以帮用户更快锁定目标NFT范围。

举个例子，通过在搜索框输入“Bored”，查看到全网的 Bored Ape 资产。对于海量搜索结果，可以在左边一栏进一步细分筛选维度，从类别/项目/区块链三个维度进一步细化搜索结果，同时也可以按照当前价格/创建时间/交易时间进行排序。

点进去单个的NFT资产，可以知道它的基本属性，了解到所有历史交易信息，包括交易次数、交易时间、历史价格、历史持有者等。在做出NFT购买决策时，可以借助该NFT的历史信息，特别是历史持有者。值得一提的是，一个NFT是否曾被巨鲸或是名人持有过，或许也是NFT估值的一个关键决策依据。

说过数据维度后，再说说数据深度，我们需要了解项目的基本信息，以及当前的市场表现，如市值、交易额、发行NFT数量、持有者以及流通性等。

市场数据是所有建模和分析工具的核心，不仅用于理解市场，也用于进行各种产品之间的相关性分析，如NFT+Layer2，NFT+DeFi 等。

最后，理解是什么赋予了NFT及其代币价值是至关重要的。不管在哪个领域，以不同的角度和思维来评估风险，以及辨别新的 NFT 项目中蕴藏的各种机会，这是一种能力，同时也需要更好的工具。
