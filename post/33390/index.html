之前MadNFTs社区为大家带来过一篇干活满满的关于NFT工具Discord的文章，此次我们为大家带来另外一个在 NFT 空间中经常使用的工具，甚至已经成为很多人与 NFT 交互方式中不可或缺的一部分。那就是“SuperSea”。

首先，什么是SuperSea？

您可能想查看它的网站，您可以在这里找到：https://nonfungible.tools/supersea 。SuperSea 是一个承诺提供“卓越的 OpenSea 体验”的 Chrome 插件。从我通过icy.tools（我常常查看的另一个工具）获得的试用开始， 我使用 SuperSea 已经有一段时间了。我在试用期结束时发现它非常有用，以至于不久之后我就购买了终身会员资格，并且没有回头。

Supersea有什么作用呢？

1）添加了许多 OpenSea UI 增强功能

2）在揭示时抓取稀有NFT的元数据

3）NFT 上架通知/狙击 

4）添加一个 QuickBuy 按钮，允许您跳过 OpenSea 的许多窗口。

让我们更详细地看一下这些中的每一个。

首先是 OpenSea UI 增强功能。 你有多少次购买了 NFT，却只是坐在那里 30 分钟想知道它是什么样子，而 OpenSea 需要花时间来揭示它。通过Supersea，你将不再需要时间等待！SuperSea 在您的收藏中的每个 NFT 旁边添加了大量选项来帮助解决这个问题。我在下面包含了一个屏幕截图，以便您可以看到这些是什么：

其中大部分都是不言自明的，但为了完整起见，我们将一一介绍。 

1) Queue OpenSea Refresh - 这允许您从您的收藏页面对 OpenSea 服务器上的元数据刷新进行排队—超级方便。

2) 从源中替换图像 - 这会查询位于nft后面的元数据，并自动将图像拉到您的 UI 中，而无需等待 OpenSea 在其服务器上更新它。当您想在其他人之前看到您的nft时非常有用。

3) Mass-Queue OpenSea Refresh / Replace Image for Collection - 这和听起来一样 - 这些按钮的作用与前面提到的相同，但将对给定集合中的所有项目应用该操作。如果您一次mint了 10 个 NFT，这将非常有用。

4.2) 查看原始数据 - 如果图像加载需要一段时间，并且您想快速查看 NFT 具有哪些特征，或者实际上，如果您只想自己找到图像 URL 并将其粘贴到您的浏览器。知识就是力量，等等。

5) Etherscan - 查看合约 - 最后一个是不言自明的。单击此按钮将带您直接进入 Etherscan 上的nft合约。正如我们之前所看到的，您可以从那里查看合约功能、检查持有人、检查待处理的交易等

希望正如您开始看到的那样，所有这些对于简化您的 OpenSea 体验非常有用，并且只会让使用该平台更加愉快。最棒的是它也是无缝集成的，并且用户界面是一流的。

谈到集成，Supersea下一个特性——稀有度和底价——实现得非常好，你会认为它是 OpenSea 的默认 UI 的一部分。看看下面的截图。每个nft都会更新以包含稀有度数据、持有地点和收藏的底价。

每个nft都有一种颜色，具体取决于 SuperSea 对其稀有度的排名，如下所示：

未排名 - 浅灰色 (-) ；

常见 - 深灰色（底部 50%） ；

罕见 - 绿色（前 50%） ；

稀有 - 蓝色（前 10%） ；

史诗 - 紫色（前 1%） ；

传奇 - 橙色（前 0.1%）。

这使得一目了然地了解nft在收藏中的潜在价值变得超级容易，尽管很明显，在做出购买决定之前，您总是应该用自己的研究、知识和判断来补充这一点！但这些稀有物品的酷之处在于它们随处可见。就像，即使您正在查看集合的新列表或销售活动时，它仍然存在。这确实可以帮助您了解 NFT 的各个等级的售价。

底价检查器也非常有用。您可以手动检查您的 NFT 的底价哦……是的，SuperSea 还具有一项功能，可以在您的 NFT 中拖动并根据底价估算您的投资组合的价值。

所有这些排名功能听起来真的很有用。但这一切从何而来？谁做的这些东西？那就是SuperSea 通过查看 NFT 合约的所有铸造nft的元数据并查看其特征等来创建自己的排名。

那么你如何获得排名？现在这已经成为我最喜欢的功能之一。无需手动请求 - 您不必等待有人为您开始排名。他们为此有一个 Discord 机器人。是自助服务。SuperSea 的Discord 可以在这里找到：https ://discord.gg/jE8qH3PV。

你在那里，你会想要点击“排名请求”频道。到达那里后，您将需要使用几个关键命令： 

- /request 

- /queue-unrevealed

让我们依次来看。如果您输入 /request 然后按 Tab，它会自动要求您输入“OpenSea URL”。这非常简单 - 您只需剪切并粘贴您想要排名的集合的 OpenSea URL，按 Tab 键，按 Enter 键，然后就可以了！

完成此操作后，机器人将自动开始从集合中提取元数据，并开始对其进行排名，以更新工具上显示的排名。使用上面示例中的命令，您将在 Discord 中看到这一点…

您也可以使用一些参数来自定义请求，如下面的屏幕截图所示。- 您可以设置开始和结束nft ID - 如果已经排名，您可以通过“refetch-all”请求新的排名 - 有时“慢模式”在某些收藏中效果更好。

值得注意的是 - 对于超过 20,000 个令牌的集合 - 您可能希望使用 start-ID 和 end-ID 函数对它们进行分批排名。

那么 /queue-unrevealed 呢？这是一项新功能，可让您告诉机器人定期检查未披露的合约，直到它们披露为止。如果您知道何时会发生这种情况，则可以节省您在揭示时请求排名的时间。你使用这个很像 /reveal 命令，将 OpenSea 链接粘贴到命令旁边。还有与我们之前看到的相同的修饰符，但除了“天”和“小时”之外，您还可以指定集合何时显示。为机器人省去一些麻烦！

所以我们已经通过了自助服务排名。老实说，在我看来，这是整个工具最酷的功能之一。排名后，您可以刷新 OpenSea 上的相关收藏页面，您的排名将直面您，准备狙击。

狙击？怎么做？这是它变得非常有趣的地方。SuperSea 有一个“上架通知器”，它将爬取 OpenSea 上的新列表 - 每 2 秒检查一次 - 以查找特定价格或稀有度的列表。想在绝大多数用户中获得优势吗？就是这样。列表通知选项的屏幕截图如下所示。要到达这里，您只需转到相关集合的“Activity”，然后单击 OpenSea 帮助添加的“Listing Notifiers”按钮（如下所示！）。

让我们依次浏览不同的选项。“最低/最高价格”是不言自明的。在此处设置您可能希望为列表设置的最低和最高价格。“最低稀有度”让您设置最低稀有度（罕见、稀有、传奇等）。

“特征”屏幕将有助于列出集合中的所有特征，以及具有该特征的 NFT 数量。仅此一项就可以让您了解可能很少见的东西，并且您可以选择要狙击的特定特征。 下面是示例@AnonymiceNFT
