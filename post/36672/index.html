写在前面

都知道OpenSea一家独大，但是OpenSea是如何成为最受欢迎的NFT市场的？作者在文中叙述了OpenSea在18年的加密寒冬中如何一步步做出正确的选择：他们预判了有潜力的赛道；用自己的方式建立了供需关系；从用户体验感最大化的方向去打磨产品…通过这些OpenSea一跃成为了NFT交易市场独角兽，而在这之后，OpenSea又将何去何从？

让我们把时间回溯到2018 年的冬天，此时加密市场已经接近崩盘，甚至像BTC和 ETH 这样的蓝筹股也从历史高点下跌了 90%，那些曾经有着宏大愿景的项目一个又一个被关闭。所以在感恩节晚宴上，你的叔叔可能会讽刺地问：“那么……你的比特币投资怎么样？”，场面一度极其尴尬。

但是在其他加密项目纷纷放弃之际，有一个项目仍然坚持着它的信念和魄力继续奋战。这就是于2018 年 2 月上线的OpenSea，继续努力追求成为“加密行业的 eBay”的愿景。这种坚持最终得到了回报，OpenSea成为了最受欢迎的NFT市场，最近的估值为133亿美元。本文试图分析 OpenSea 取得巨大成功的原因以及它未来的发展方向。

首先，什么是 NFT？

NFT（非同质化代币）是一种在区块链上存储资产所有权的方式。它可以是图片、音乐等数字资产 ，甚至是土地所有权、文凭 等现实世界的资产。由于区块链是完全去中心化和抗审查的（至少以太坊是），你不需要通过第三方来维护这种所有权的记录。

OpenSea 是一个与底层区块链交互的市场，用于创建和交易代表数字资产（如图片、音乐，甚至是以太坊上的域名）的 NFT，市场是促进产品销售和购买的途径。想想亚马逊、Uber和Airbnb。大多数市场都没有产品的库存，他们只是允许其他人销售（也有例外，比如亚马逊也销售他们自己的产品）。

从上面可以明显看出，OpenSea 已经一骑绝尘，或者我应该说一骑绝水（明白了吗？明白了吗？）[译者：原文可能想借由水滴取代沙尘比喻竞争优势巨大]。在过去 30 天里，它促成了价值约 45 亿美元的交易（即 150 万ETH），远远超过所有竞争对手。综上所述，eBay 在上个季度每月进行的交易价值约为 66 亿美元。OpenSea和eBay交易完全不同的商品，OpenSea 在加密领域运营，却仍能与传统市场保持同步，对此我表示赞赏。

他们做对了什么？

一个项目的成功不能仅仅归因于几个因素，它需要出色的产品、优质的团队、适合的契机以及运气 。这些似乎OpenSea 都拥有，但我们仍然可以尝试去评估他们到底做了什么正确的事情而达到现在的位置。

快速建立供需关系

正如 Lenny Rachitsky（Airbnb 的前产品经理）在他的Newsletter中所讨论的那样，推出交易市场的最大难题之一，可以用“鸡和蛋的问题”来描述。当平台上没有用户时，你如何说服供应商排列他们的商品，如果没有商品，又如何获得买家的关注？

大多数传统市场通过先建立供应渠道 而获得成功。OpenSea 也是如此。正如The Generalist报道的那样，他们竭尽全力建立初始供应，理查德·陈（加密投资公司 1confirmation 合伙人）在下面的引文中完美地总结了这一点：

“Devin 和 Alex（OpenSea 的联合创始人）在 Discords 中出色地发现了新的 NFT 项目，并击败了 Rare Bits （OpenSea当时强力的竞争对手），成功让这些项目在 OpenSea而非 Rare Bits上架并交易 。我们在 2018 年 4 月投资时，OpenSea 的交易量约为Rare Bits 4 倍。” 

- 理查德·陈

这些努力促成了OpenSea早期与多个项目/艺术家的合作，包括支持Axies（可以说是基于以太坊的最受欢迎的游戏），使用其原生代币 MANA购买Decentraland（类似于 Minecraft 的基于以太坊的虚拟游戏），与美国职业棒球大联盟合作销售 MLB 数字收藏品，与死侍官方合作销售数字收藏品，与德国足球俱乐部拜仁慕尼黑合作列出球员卡 NFT，并支持ENS 域名的交易。这份详尽的清单展示了团队与尽可能多的项目建立稳定供应合作的热情，以及在游戏、艺术、体育、娱乐业、域名等广泛类别中迎合 NFT 的远见。

OpenSea的全速前进得到了平台以用户为中心的功能的助力，这些功能使得供应商能够轻松上手使用OpenSea。NFT 的上市是无需许可的，无需任何批准或复杂的审批流程。 此外，OpenSea 创造了一种将在以太坊上铸造新 NFT 的成本从卖方转移到买方 的方法。这两个功能都显著降低了新晋艺术家/项目的进入门槛 ，极大地提升了平台上 NFT 库存。

一旦建立了稳定的供应，传统市场就会开始专注于建立需求，对于 OpenSea，建立需求的过程要简单一些。OpenSea将自己定义为NFT平台并获得了早期合作伙伴，源源不断的新项目开始在OpenSea上架NFT，同时也为OpenSea带来了客户。因为OpenSea支持NFT 在每次销售中获得版税，使项目本身也受到激励，因此会鼓励其客户使用 OpenSea。此外，从 2021 年夏天开始的 NFT 狂热也推动了 OpenSea 的需求增长。一旦加密世界接受了这种区块链的非 DeFi 用例，并在以太坊上创造了新的交易纪录，猜猜谁会在那为此提供便利，并赚得盆满钵满？

用户体验感最大化

正如 Sarah Tavel 在她的博客文章中所讨论的，市场不应该仅仅专注于增加供应，还需要实现产品功能来最大限度地提高用户的体验感。就像Uber通过信用卡无感支付一样，Airbnb 通过建立优质房东计划，奖励满足要求的房东一个特殊的展示徽章，从而使他们更容易获得客户的青睐。OpenSea 在这方面也表现出色。

他们的一大成就是实现了强大的过滤和排序功能 ，以便轻松发现 NFT。可能这看起来微不足道，但事实并非如此。每个 NFT 集合都有不同的特征和稀有度。OpenSea 允许“自定义”过滤和排序，根据正在查看的集合进行更改，以便用户可以根据该特定集合细化他们的搜索。

不同集合的不同过滤器

接下来，他们采取了具体措施来解决困扰 NFT 爱好者和一般以太坊用户的最大问题——即高得令人望而却步的Gas费。Gas是以太坊网络上每笔交易收取的手续费，支付给运行网络的个人，也就是矿工。但由于对以太坊的需求量高得离谱，而当前网络的容量还相当低，因此每笔 NFT 交易的费用达到几十美元到几百美元不等（取决于网络拥堵情况）。虽然 OpenSea 无法控制以太坊网络底层的Gas费，但他们实施了多项功能，至少在一定程度上降低了这些成本。

他们使拍卖能够在链下进行，以便在最终销售期间仅使用一次Gas费，允许在链下降低拍卖价格，这样价格变化就不需要每次都支付Gas费，引入“懒人一键铸造”，以便艺术家可以免费铸造 NFT，Gas费由买家承担。OpenSea与 Polygon 集成，Polygon是以太坊的一条侧链，其交互成本更低。

在产品的财务方面Opensea也保持着关注，OpenSea 还允许使用除 ETH 之外的加密货币购买 NFT，尤其是Dai和USDC等稳定币。这特别有用，因为用ETH 购买会在许多国家引发税收事件，而使用稳定币是避免这种情况的简单方法。

做出正确的选择

好的产品只是成功的关键的元素之一，但不是唯一的。每个项目在其生命周期中都面临着无数的选择，而他们做出的选择决定了他们是在大海中乘风破浪还是倾覆沉没。OpenSea 在通往顶峰的过程中做出了许多正确的选择。

首先是他们对自己使命拥有信念 。据The Generalist报道，他们坚定的信仰是一种叫做 ERC-721 的东西。这是一个由以太坊核心开发人员构建的标准，用于规定在以太坊上创建和转移 NFT。该标准的创建意味着该领域的重要性足以引起以太坊开发人员的关注，并且所有未来的 NFT 项目都将遵守该标准。因此，他们在 ERC-721 标准之上创建了一个市场，然后可以支持未来所有的 NFT。通过这样做，OpenSea 通过聚合 NFT 供应为买家提供了更加便捷的体验 。在此之前，每个项目都必须创建自己的市场，例如，CryptoPunks 和Axies有自己的NFT交易市场（现在仍然如此）。

对 NFT 的这种信念也是让他们在 2018 年的加密货币寒冬中幸存下来的原因。当加密货币市场低迷且没有多少项目能够生存下来时，OpenSea 经营着一支只有 7 名员工的精干团队，靠着不断收取交易费以维持公司运营，并继续在行业深耕。

但在最近，由于代码中的一些错误，导致许多高价值 NFT 以较低的价格出售，此时他们面临了一些更艰难的选择。他们决定赔偿受影响用户的损失。同样，在某些情况下，他们还冻结了被盗的 NFT 。虽然他们因此受到了一些人的批评，因为这违背了审查抵制的基本去中心化加密精神，但这些步骤肯定有助于建立与用户之间的信任。

OpenSea 的下一步是什么

在征服世界后你会做什么？你要粉碎每一个敢于推翻你的竞争对手，并继续你的统治地位。要做到这一点，要么是在你的领域中以很大的优势成为第一，要么是扩展到其他领域或赛道，亦或两者皆是。

OpenSea 已经是NFT 市场中的主导力量。然而，他们面临的市场新晋黑马 LooksRare 真实有力的威胁。这个最近推出的 NFT 市场相较于OpenSea来说有很多优势：费用低于 OpenSea，并且拥有原生代币，这大大增加了它对加密原住民的吸引力，除此之外他们还与代币持有者分享其所有手续费。从前面的表格中可以看出，LooksRare 已经超越了所有其他平台，并且正在接近 OpenSea 的交易量和交易金额。

那么 OpenSea 是否会推出自己的代币呢？这没有那么简单。作为一家美国公司，OpenSea 与美国监管机构相处融洽，不能冒险推出代币，那样只会招致监管机构的愤怒，从而阻碍他们的运营。因此，OpenSea 的前进道路是最大限度地提高用户的体验感，并可能扩展到其他赛道。以下是一些潜在的选择：

  • 扩展移动端。 OpenSea 已经有一个移动应用程序，但它非常有限，甚至不允许交易 NFT。该应用程序需要抽象出创建钱包、存储助记词（可以为高级用户执行此操作）和复杂的Gas费估算的过程。基本上可以让加密新手无缝接入使用。

  • 探索 NFT 的碎片化 ，其中 NFT 的原始所有者可以将所有权拆分为多个较小的代币，然后其他人可以购买这些代币来共享 NFT 的所有权。现在大多数人都买不起BYAC 和 CryptoPunks这样的蓝筹 NFT，这样的拥有方式将成为新的需求。原始所有者可以从增加的流动性以及通过出售碎片化的NFT获得的潜在“策展人”费用中受益。双赢！

  • 考虑与 DeFi 协议合作 （或创建自己的协议）以允许使用 NFT 作为抵押品来借钱，当然是要从蓝筹 NFT开始。这种跨领域合作可以解锁全新的用户群和对 OpenSea 的需求。

  • Vitalik 最近谈到了不可转让的 NFT，这些 NFT 具有潜在用途，例如颁发大学学位、土地所有权等。虽然还有许多考虑因素尚未解决（例如，如果由于安全问题所有者想要转移到他们自己的其他钱包），OpenSea 可以引导社区的讨论，甚至可能为其编写 ERC 标准！他们甚至可以探索潜在的创建 B2B 解决方案，为可能想要使用它的企业提供不可转让的 NFT 解决方案。

应该注意的是，上面提到的一些选项，例如 NFT 的碎片化和作为抵押品的效用，可能会使 OpenSea 受到监管机构的审查。所以他们必须小心行事。事实上，监管是 OpenSea 需要持续小心的一大潜在障碍， 因为所有加密货币的监管普遍存在不确定性。事实上，作为加密社区的主导者，他们甚至可以考虑游说对加密友好的法律，以帮助生态系统向前发展。毕竟，权力越大，责任越大！

译者笔记：

之前一直不太理解，为什么OpenSea会一家独大。不管是从OpenSea的IPO路线，还是其卡顿的体验，抑或是它中心化的运营思维，这一切都不太“Web3”，也是一直为大家所诟病的方面。从作者的讲述中，知道了OpenSea在寒冬中的坚持，在行业混沌时拨开云雾的壮举，在产品上寻求用户极致体验感的努力，也就渐渐明白OpenSea为何会成为行业独角兽。但是我想说的是不进则退，如果OpenSea面对“LooksRare”们的强力攻击，不拿出用户为主的态度，其行业主导的地位终将被蚕食。不过反过来说，作为一个“Web3人”，我挺乐意见到这种局面的，各相竞争、百花齐放，或许是更好的时代。
