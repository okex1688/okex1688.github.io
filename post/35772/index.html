区块链技术的流行预示着从中心化机构到去中心化机构的根本转变，使用区块链技术的公司往往以各种方式脱颖而出。比特币和其他加密货币作为一种更加透明和安全的数字货币的原始概念，Web 3——一种去中心化的网络体验，NFT——一种艺术业务的转变，以及Metaverse——一种将在未来几年内让人们大吃一惊的转变，区块链正在每个行业形成趋势。

因此，我们如何押注 .com、.in、.eu 和 .gov 等已建立的域名？非 .com 域名似乎在区块链领域中更受欢迎。例如，.xyz 域是区块链组织最常用的域名之一。

2015年，谷歌的母公司Alphabet Inc.选择www.abc.xyz 作为其网址后，xyz域名的受欢迎程度急剧上升。这主要是因为美国广播公司使用的是www.abc.com，而宝马的车队管理部门使用的是www.alphabet.com（实际上现在仍然如此）。结果，.xyz 域名受到了很多人的关注。

如何注册 .xyz 域名？

注册 .xyz 类似于注册普通的 .com、.in 域名。注册 .xyz 域名的分步指南：

第 1 步：选择您要注册的域名。示例：coincodecap.xyz、blockwish.xyz 等

第 2 步： https ://xyz.xyz/ 。访问这个网站。 XYZ Registry 是 .xyz 域名背后的公司。该项目在 GitHub 上开放供贡献或反馈。

XYZ 是一家为全球个人和企业提供域名和相关服务的技术组织。作为行业领导者，XYZ 提供各种商品，从其旗舰域名.xyz 到特定类别的域名。XYZ 成立于 2011 年，旨在为希望数字化存在的互联网消费者提供创新和选择。此外，它通过连接 X、Y 和 Z 代，为所有年龄段的人提供了使用新域名建立他们的项目、品牌和企业的机会。

.xyz X Web 3.0

如果你最近访问过一家加密公司的网站，你无疑会看到一个以“.xyz”结尾的 URL，而不是更昂贵的 .com。.xyz 已成为许多 Web3 企业的首选 URL 结尾，从 Block（以前称为 Square）到Mirror、Proof、Propel 和 Login 等区块链初创公司。这意味着什么，为什么它在 web3 世界中变得如此流行？

.xyz 是第一个没有内在含义的真正通用域名后缀，旨在为人们提供域名选择。例如：

  • .com 用于商业用途，

  • .in 代表印度

  • .net 用于网络和

  • .org 用于组织

不完全属于这些类别或想要脱颖而出的用户应该选择.xyz 作为他们的 TLD。现在你知道为什么大多数 Web3 公司都使用它了。Web3 不是对 Web2 的升级，而是一种全新的 Web 体验。虽然 .xyz 域目前由互联网监管机构 ICANN管理，但目前有几方正在努力开发该系统的去中心化替代方案以支持 Web3。XYZ 与 Web3 公司的结盟可能会基于去中心化网络中的身份和所有权开辟大量新的货币化机会。

以太坊名称服务 (ENS)是 Web3 的域名服务 (DNS)。ENS 允许用户为他们的所有加密地址和一个可搜索的数据库生成一个全球身份，以使加密钱包和交易更容易访问，即使它们存储在几个不同的平台上。用户现在可以使用本地 .eth 域或 a.xyz 域在 ENS 中构建配置文件，以在他们的社交媒体上共享。

Coincodecap.eth是一个ENS 域名，代表与 Coincodecap 连接的以太坊地址。eth.xyz 功能已自动提供给每个 .eth 名称。只需将“.xyz”附加到任何 .eth 名称的末尾，即可在任何 Web 浏览器中查看 ENS 配置文件。例如，在任何浏览器中，Coincodecap.eth 的 ENS 配置文件都可以在 Coincodecap.eth.xyz 找到。档案材料是使用 ENS 的可公开访问数据自动生成的，并最终由相应的 ENS 用户管理。

结论

.xyz 域名在区块链初创公司中很受欢迎，因为它代表了去中心化和下一波 Web3 应用程序。此外，它还赋予他们不属于技术行业中占主导地位的陈词滥调 TLD 的权力。
