在过去的一年里，我们采访了 50 多家 Web 3 元宇宙公司。

今天总结一下我的收获。

“Web 3 元宇宙”指的是数字虚拟世界，通常通过代币或 NFT 包含一些用户所有权元素。

我们遇到的每个项目都属于以下三类之一：

  • 基础设施

  • 体验层（世界内）

  • 增值层（世界外）

一个常见的比喻是：元宇宙=游戏。

但是一个更适合的比喻是Layer1

与 L1 一样，元宇宙是开发人员创建体验 + 服务的平台。

就像 2017 年的 L1 一样，我们看到创建元宇宙的人比在元宇宙之上创建东西的人更多。

虽然开放世界 MMORPG（大型多人在线角色扮演游戏）的潜在市场规模很大，但我认为最大的虚拟世界将不仅仅包含游戏。

与“胖协议”类似，获胜的元宇宙基质层将比任何单个应用程序更有价值。

我们采访过的和投资的许多公司都在构建本质上增强的、用户拥有的MMORPG。

@AxieInfinity @illuviumio @staratlas @decimated_game 是一些例子——即熟悉的游戏体验 + 通过 NFT 获得游戏内物品的所有权。

“游戏”听起来微不足道，但被不冷落和忽视！

2021 年：暴雪公司88亿美元收入中61%来自游戏内购。 最近被微软以680亿美元价格收购。

2021年，AXS 产生了13亿美元的收入，如今估值为140亿美元。

P2E 游戏提供了巨大的机会，并且已经催生了基础设施公司（例如$LOOKS $GF $GUILD），工具（例如@CrucibleNetwork）和一个价值 5.5 亿美元的公会市场（例如$YGG $MC)。1100家游戏公会拥有 100 万玩家，公会可以说是当今 DAO 的最大用例。

虽然 P2E 游戏令人兴奋并且可以演变成更多游戏，但价值 1 万亿美元的登顶机会在于创建一个虚拟世界，其中游戏只是众多用例之一。

想象一个广义的虚拟世界——无需许可的创作+商业，并且由不可更改的数字权利定义。

类似于 Facebook 如何在一个数字平台上复制我们的社交体验，获胜的元宇宙将把多个用例捆绑到一个开放的虚拟世界中。

我们已经看到各种公司提供元宇宙活动和店面作为服务，尽管很多都为时过早。

一些项目已经为实现这一愿景构建了一个通用的基础，但我们距离完全成熟可能还有 5 到 10 年的时间。这些项目包括@Decentraland @theSandboxGame @SomniumSpace @cryptovoxels @NFtworlds  等等。

成功的最大机会是为 Meta 试图创造的东西创造一个开放的替代方案。

即任何 开发人员都可以在其上构建任何用户都可以访问的社交、娱乐、工作、教育和商业体验的平台。

但是我们仍处于早期阶段。

例如，今天，虽然您可以在@decentraland探索和玩游戏，但给人的体验仍然一言难尽

我们看到的大多数元宇宙购物仍然是 NFT买卖，而且大多数只是重定向链接到OpenSea。

我们采访过的一些项目正在做元宇宙世界内的 PoS 和店面。

尽管元宇宙雄心勃勃，但它们也面临着冷启动问题：当通过构建自己的元界可以获得更多价值时，你如何激励开发者在新的元宇宙之上进行构建？ （看看各种$LOOT 分叉项目）。如何让用户加入一个空白的虚拟世界？

我们看到的一种解决方案是降低创造世界内体验的障碍。例如@nftworldsNFT 使用 Minecraft（而不是 Unity）来创建他们的世界。 这利用了一个超过 1.3 亿 MAU 的很多人都已经熟悉的行为套件。

由于大多数虚拟土地所有者仍然是被动的投机者，元宇宙世界租赁模式也将促进发展。

用户可以出租土地供开发商店建造，以换取费用。

这是目前可以使用@nftworldsNFT在游戏中完成的，但可以通过新的 DeFi模式进行推广。

我们看到的第三种常见解决方案是专业化。

不同于开启一个通用的元宇宙，一些团队选择在一个小众社区中进行扩展和提升。

例如@Cryptovoxels 最初是一个以艺术为中心的元宇宙，但它没有理由不能扩展到其他领域。

在提升方面——我们遇到的大多数元宇宙通过代币销售或土地销售筹集资金。

我最初的预感是，在成熟时，土地应该比世界上的交易货币更值钱。

今天，这并不令人惊讶。

总而言之，我相信随着时间的推移，数字体验趋向于沉浸式，因此元宇宙是不可避免的。

Facebook（Meta） 愿意花费 100亿美元来实现这一点的事实不仅增加了它的可能性，而且也增加了对用户拥有的替代方案的需求。

元宇宙项目的现状让我想起了 2017 年的 Web 3。

许多想法与当今的技术限制脱节，但也有许多轶事证据表明产品市场契合度指日可待。

最后，Web 3 元宇宙的想法可能是 1 年、5 年或 10 年之后才能获得一些大的成就……

但我们致力于加倍努力并帮助共同解决这个问题。
