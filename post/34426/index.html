NFT 交易平台推荐一：OpenSea

OpenSea是现今最大的NFT交易平台。OpenSea拥有丰富的NFT种类，上架商品分为九大类：艺术、音乐、域名、虚拟世界、交易卡、收藏品、体育和功能性资。

当中最受欢迎的类别「收藏品」 包括有：CryptoPunks、BAYC、Pudgy Penguins等不同NFT项目。

以Bored Ape Yacht Club为例，靠着球星Stephen Curry 、饶舌歌手Eminem 、名嘴Jimmy Fallon 、DJ  Steve Aoki等知名人士相继入手，早前在OpenSea的地板价超越了CryptoPunks，及后再宣布在OpenSea交易平台上的总交易量正式突破10亿美元。

2022年1月，这龙头NFT交易平台完成3亿美元融资，估值增长至133亿美元，在半年内身价大涨8倍。

  • NFT 交易平台名称：OpenSea

  • NFT 交易平台佣金：2.5%

  • 背后投资者：Binance、Coinbase、Paradigm、Coatue

资料来源：HYPEBEAST、DappRadar、CoinGecko、OpenSea

OpenSea：7人团队打造独角兽  

  • 过往历史

OpenSea 由 Devin Finzer（Pinterest 前员工） 和 Alex Atallah（苹果前员工） 于 2017 年 12 月创立，是一家 Y-Combinator 支持孵化的公司。

不过有意思的是，创始人 Devin 最开始向 Y-Combinator 申请孵化的项目方向并非 OpenSea，而是在看到 CryptoKitties 火爆后，意识到 NFT 市场开始形成后迅速地将项目方向转向 OpenSea。

根据官方信息显示，OpenSea 当前共有 21 名员工，而创始人 Devin 此前曾表示，在 2020 年的很大一部分时间里，OpenSea 是一个 7 人团队 ，不过随着 NFT 市场规模的逐渐扩大以及公司的进一步融资，团队也在不断扩张。

  • 发展现状

目前，OpenSea 已经发展成为最大的去中心化数字商品交易市场，涵盖最广泛的市场类别，包括艺术类、音乐类、域名、游戏资产以及卡牌等，根据 OpenSea 官网数据显示，当前 OpenSea 上共约有 828,100 种收藏品类型，NFT 数量超过 3 千万。

用户除了可以在 OpenSea 上进行 NFT 的自由买卖外，还可以通过 OpenSea 的「收藏品管理器」，自由创建自己的 NFT 作品，而且无需支付任何 Gas 手续费，即使是卖出这些 NFT 也无需支付 Gas 手续费。

对于 NFT 项目方以及相关数据服务方来说，OpenSea 也像是一个 NFT 开放平台。需要 NFT 相关数据的服务方可以通过 OpenSea 提供的 API 获取数据信息。对于 NFT 项目方，他们可以直接将项目的 NFT 交易市场链接到 OpenSea 上，轻松打造自己的交易市场，这样项目开发人员就可以专注于项目本身，而无需花费精力从零开始打造自己的交易市场。

目前 OpenSea 的主要收入模式还是依靠交易手续费，仅收取 2.5% 的交易手续费，尽管这费率结构比正常加密货币交易所的手续费费率高出近十倍，但是在 NFT 交易市场领域，该费率结构基本处于行业最低水平，其他 NFT 交易市场的手续费费率基本在 10%~20% 之间。

而传统艺术品拍卖行的手续费更高，据了解，苏富比、佳士得拍卖艺术收藏品收取买家佣金约为 12-25%，收取卖家佣金约为 2%-9%。

  • OpenSea 数据基本面

2021 年，随着链游、元宇宙、Aavtars 等赛道的火爆，OpenSea 成为了 NFT 交易的首选市场，OpenSea 也就此开挂，在各方面数据指标屡创新高。

不过值得一提的是，并不是所有 NFT 交易市场都是跟 OpenSea 一样为综合开放的交易市场，例如 SuperRare 和 Foundation 这些 NFT 交易市场，是属于精心策展、偏向专业化的 NFT 交易平台，所以对平台的 NFT 有所要求，平台的艺术家也都是经过严格筛选的，并非所有用户都可自由发行 NFT。而近期火爆的 NFT 基本都是属于社区项目以及一些游戏卡牌道具等，OpenSea 便更理所应当地成为了它们交易的首选场所。

NFT 交易平台推荐二：Nifty Gateway

Nifty Gateway是孖生兄弟Duncan Cock Foster和Griffin Cock Foster创立的NFT交易平台，2019年被加密货币平台Gemini收购。

成功引入Pak、Beeple、FEWOCiOUS等知名艺术家。

Nifty Gateway这个NFT交易平台对创作者要求较高，创作者需要提交申请，附带简短视频介绍，并列明项目中长期目标。NFT首次和二次销售均会收取5%销售费用。

这个NFT交易平台的拍卖玩法多样，每场拍卖都是被精心策划，甚至有不同主题；就连拍卖制度一般常见的都有五种：公开竞标、无声竞标、抽签、随机礼包、开放版数。

Nifty Gateway团队人脉网络强大，合作发行NFT作品的名人或单位包括知名DJ Steve Aoki、杂志PlayBoy、名媛Paris Hilton、拍卖行苏富比等等，每每都能掀起话题。

交易方面，Nifty Gateway支援加密货币之外，还提供法币买NFT交易服务，用家可以信用卡或金融卡直接购买NFT币，而且将出售的利益直接兑现至银行帐户。不过，目前Nifty Gateway只容许美国用户提取法定货币。

  • NFT 交易平台名称：Nifty Gateway（按这进入）

  • NFT 交易平台佣金：5%

  • 背后投资者：Boost VC

资料来源：Benzinga、chou00.xyz

其中佳士得拍卖的首个NFT作品为Beeple的作品：《Everydays – The First 5000 Days》引起艺术圈、互联网、传统拍卖和区块链领域人群的大规模关注。

NFT 交易平台推荐三：SuperRare

SuperRare是一个专门为专业、知名艺术家打造的NFT平台，也是最早成立的NFT平台之一。

用户想在SuperRare出售作品，必须经过严格审核，审核失败率也相当高。另外，SuperRare对作品版权有较完善的保护机制，是专业艺术工作者，或是非常着重版权问题的创作者 ，SuperRare是不二之选的NFT交易平台

  • NFT 交易平台名称：SuperRare（按这进入） 

  • NFT 交易平台佣金：15% 

  • 背后投资者：confirmation、Velvet Sea Ventures、Samsung NEXT

NFT 交易平台推荐四：Foundation

Foundation是采用社群主导和自动限时拍卖机制的NFT交易平台，生成的作品也会被放到OpenSea发行。

社群主导的策展模式意思是，通过最初邀请到平台上的50位艺术家，每当他们成功售出一件原创作品，便会获得两个邀请名额，可以邀请新人入驻平台。

如此类推。如果艺术家恶意买卖邀请码，将被剥夺永久入驻资格。入驻方式还有 —— 所有经过Twitter认证的社群成员在加入后，可获得5票用在支持5位可入驻的艺术家，排名前50的便可加入这交易平台创建NFT。

交易方面，Foundation的艺术家用户将NFT作品设定底价，首个出价后，作品会自动开始24小时限时拍卖。截至2022年1月10日，Foundation这NFT交易平台的交易量金额达50,004.90 ETH

  • NFT 交易平台名称：Foundation

  • NFT 交易平台佣金：15%

资料来源：Foundation

NFT 交易平台推荐五：MakersPlace

MakersPlace这NFT交易平台极度看重加密艺术品的质量，因此会为艺术家或创作者每件NFT作品制作区块链指纹和数码签署，以证明作品的身份和来源。

这也成为了艺术品的一种独特象征，即使作品被复制，也不会有原始的签名版本。

MakersPlace亦引进了社交功能，创作者可通过浏览量、喜爱程度等功能进行分析。同时，这NFT 交易平台提供数位钱包给创作者存放作品。交易方面，除了以太坊，目前接受信用卡和Paypal支付。 

  • NFT 交易平台名称：MakersPlace

  • NFT 交易平台佣金：15% 

  • 背后投资者：Bessemer Venture Partners、Pantera Capital、Uncork Capital

资料来源：CoinDesk

NFT 交易平台推荐六：Async Art

Async Art本身就是一场艺术实验，主打艺术、音乐和全新的蓝纸三大类作品。

以这NFT交易平台上的图像作品为例—— 不再单纯是静态图片，而是由master图层和多个layer组成，拥有者甚至可以随意改变不同图层的显示方式和设定。换句话说，作品会随着时间不断演变。

这种独特体验受到各界欢迎，如2020年3月由Async Art 联合13名加密艺术家共同推出的加密艺术品《First Supper》拍出高达103枚ETH的价钱。

相比其他平台，佣金较低也是跑出的原因之一。Async Art 对一次销售收取10%费用，二次销售则只是1%。 

  • NFT 交易平台名称：Async Art

  • NFT 交易平台佣金：10% 

  • 背后投资者：Lemniscap、Galaxy Interactive  

资料来源：Async Art

NFT 交易平台推荐七： Zora

Zora是一个邀请制的数码艺术平台，专门提供限量NFT币化产品及作品。

这个NFT 交易平台有个既定规则，每位新加入的艺术家有3个邀请名额，可以邀请其他好友或艺术家。

现在，Zora已开放铸造功能，任何人都可以铸造NFT作品。2021年8月，Zora推出CryptoPunks拍卖行Punk House，是目前唯一的公开且无需许可的CryptoPunks的链上拍卖场所，而且推出全新的拍卖模式—— 组团竞价。

  • NFT 交易平台名称：Zora

  • NFT 交易平台佣金：无

  • 背后投资者：CoinFund、Venrock、1kx  

资料来源：Coindesk

NFT 交易平台推荐八： Rarible

Rarible由Alexander Salnikov和Alexei Falin于2020年创办，主要团队成员都在莫斯科。

Rarible是第一家透过发行专属平台币—— RARI币，让用户能参与发展决策的NFT交易平台。

另外，Rarible上架的作品可以在OpenSea市集中看到 ，这让Rarible卖家接触到更多用户，从而提升整体交易量。

截至2022年1月10日，Rarible这NFT交易平台的交易量金额达2.74亿美元 。

  • NFT 交易平台名称：Rarible

  • NFT 交易平台佣金：2.5% 

  • 背后投资者：CoinFund、Venrock、1kx  

资料来源：The College Investor 、Rarible

KnownOrigin ： KnownOrigin是个老牌加密艺术平台，由于其无限制的模式使得大量的艺术家申 请，现在是一个著名的数字艺术家集散地，类似北京 798，但目前已暂 停申请 。根据平台的规定，创作者在申请成功后，可每24小时上传1件作品。当作品被首次销售后，KnownOrigin会收取15%的服务费，创作者获得85%的收入；而在二次销售中，平台将收取2.5%的服务费，创作者可获得12.5%的收入，卖家则获得85%的收入。当然，如果创作者和其他艺术家进行合作，其可在NFT铸造时预先设定每笔销售的百分比，例如10%、25%、50%等。

Async Art ： Async Art 是个建立在以太坊上的可编程加密艺术平台，定位新颖**，适合二次艺术创作者，特别是可编程作品的展示。** Async Art里面的作品主要是由“Master”和“Layer”两部分组成，Master是作品的主体形式，一个Master可由多层Layers构成，Async Art允许艺术作品根据“图层变化”而进行变化。创作者在申请成为Async艺术家后，无需任何编程知识，只需在作品上传过程中，将作品切为图层即可。在作品出售中，创作者可设置“立即购买”或者“开放式拍卖”，拍卖结束时间由自己决定。Async Art在作品销售中的收益采取了“抽水”的模式，其在首次销售中将收取10%的服务费，剩余90%归艺术家所有，二次销售中则收取1%的服务费。当然，创作者还可获得10%的版税。不过，对于定制艺术品，Async Art将收取20-30%的服务费，创作者则获得70%-80%的收入。

NFT 交易平台比较（Solana）

Solana有「以太币杀手」之称，靠着交易快速和交易费低两大特点备受追捧，成为最受欢迎的区块链之一。

Bitcoin.com数据显示，Solana单是2021年12月的NFT销售额就高达27亿美元，在众多区块链排名第三

以下就列出Magic Eden和Alpha Art两个支持Solana的NFT平台作介绍和比较：

NFT 交易平台推荐九： Magic Eden

Magic Eden 是为Solana区块链打造的新NFT平台，可购买、销售和铸造NFT。

据加密数据网站DappRadar资料显示，Magic Eden由2021年9月成立至今，已录得2,035万美元交易量

除了NFT迷不会陌生，专攻亚洲市场的Monkey Kingdom，截至2022年1月12日，Magic Eden上最受欢迎的NFT项目头三位包括有：GGSG: Galactic Geckos、Shadowy Super Coder DAO和Degenerate Ape Academy。

2021年12月初，Magic Eden官方报告中指出，平台每月用户量是OpenSea的一半，但每日交易量却高出2至3倍。另外，0%上市费和2%的交易费，降低了NFT买家和创作者的门槛。  

  • NFT 交易平台名称：Magic Eden

  • NFT 交易平台佣金：无 

  • 官方支持钱包：Phantom、Slope、Sollet

资料来源：DappRadar、CoinGecko、AMBCrypto

Alpha Art Market算是第二大的，仅次于SOLANART，对于作品的审核非常的高，基本上可以杜绝假的项目上线；

  • NFT 交易平台名称：Alpha art

  • NFT 交易平台佣金：2% 

  • 官方支持钱包：Phantom、Slope、Solflare

如何选择NFT 交易平台？

根据市场活跃度选交易平台

从近1 个月的交易额 Top 10 NFT 交易市场来看，OpenSea 以$33 亿交易额稳居第一，次之是1 月中旬强势崛起的后起之秀LooksRare，前面2 个NFT交易市场的交易额，佔了整体交易额近80% 的份额，把其他NFT 交易市场远远甩在后面。

作为玩家寻找最适合的NFT 市场，除了参考以上建议之外，还需要注意以下几点：

  • NFT 市场提供的信息

好的NFT 市场需要给玩家提供完好的项目信息。玩家需要能够看到可供购买的NFT 数量、每个NFT 的价格，以及更深入的信息，例如买卖次数、持有者、历史交易信息等。有了这些信息，能够帮助你做出正确的选择。

  • 支持的钱包是否足够便利

作为一个好的NFT 交易市场需要更简单让不同设备的玩家参与进来。它需要支持多个NFT 钱包以及支付选项，这会方便玩家可以灵活选择不同的支付方式购买NFT。

  • NFT 市场的安全性

一个好的NFT 交易市场需要有高度安全的环境来保障创作者以及玩家的安全。比如对一些隐私信息进行高级加密，同时要求身份验证或是为交易提供保险来防止欺诈。

  • 更多高级的搜索功能

在购买NFT 时，快速地获得可靠和准确的数据非常重要。另外，平台应该提供更多可视化数据，让玩家能够清楚了解市场上发生的事情，以及辅助判断哪些NFT 更值得投资。

一个好的NFT 市场应该提供不同的搜索功能来满足玩家的需求，无论玩家是在搜索具体的名字、类型，还是创作者，都应该提供与此相关的正确信息。

  • NFT 市场的社区评价

在你决定要购买NFT 时，可以从别人的评论中更好地了解这些NFT 市场的体验。当你决定要从特定的某个NFT 市场购买NFT 时，你自己能够避开一些坑，以及预见自己会有怎样的体验。
