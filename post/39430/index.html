Web3 真正的样子是什么？

一段时间以来，区块链领域一直在兜售 Web3 田园诗的承诺，以及它将如何彻底改革现有的互联网，围绕太多登录凭据、安全性、可访问性和内容审核的问题都困扰着当今的互联网。

新的去中心化技术几乎可以解决所有这些问题，但有一个问题，这些网络通常仍然建立在遗留基础设施之上，因此面临许多相同的缺陷。

幸运的是，不必如此，并且已经有一些平台可以提供未来 Web3 体验的真实愿景。

Web2 现在的样子

Web2，我们已经进入了 20 年的大部分时间的互联网时代，已经很好地服务于普通大众，并为我们提供了一些改变生活的服务。流媒体娱乐、移动互联网和零工经济都是这个电信时代给我们带来的革命，话虽如此，它并非没有缺点。

Web2 服务在很大程度上已由少数大型集中式公司拥有和运营，这导致了一些令人讨厌的做法，尤其是在数据收集方面，科技行业的一些知名人士也参与其中。

用户失去了对自己信息的控制权，他们对此并不满意，话虽如此，这些服务已经变得如此无所不包，以至于许多人别无选择，只能与他们互动。

当前 Web2 环境的另一个问题是它仍然在旧的服务器基础架构上运行，如果没有正确设置或监控，这些集中式网络通常容易被攻击者访问。

通常只需要利用数据库或设计不当的密码来危及主要平台的安全性，在过去十年中，当前系统的缺陷导致网络犯罪和多次数据泄露的流行率大幅上升。

互联网上也存在关于内容审核的持续争论，有些人将内容审核视为保护人们免受欺凌、仇恨言论和错误信息的一种手段。其他人则深切关注这对言论自由意味着什么，没有一种哲学是普遍认同的，这引发了对该主题的持续、激烈的讨论。

Web3 的可能性

我们如何到达这里是可以理解的，但它需要改变，幸运的是，正在发生的转变正在迎来 Web3 时代。这种新范式建立在区块链技术之上，由比特币和以太坊等加密货币推广。

区块链本质上是去中心化的、不可篡改的账本 —— 将可编程的自动执行智能合约添加到等式中，这些新颖的账本有可能形成新互联网的骨干。

该技术不仅允许创建新形式的货币，还允许创建去中心化应用程序 (dApp) 以及称为非同质化代币或 NFT 的数字资产，这些服务和商品以及许多其他协议是定义 Web3 并将构建即将到来的元宇宙的构建块。

区块链的性质构成了如何在 Web3 基础设施中解决 Web2 的许多问题的基础，例如，它们允许创建所谓的去中心化自治组织或 DAO。这些是可以监督平台或项目开发的治理系统，但由社区的每个成员拥有和运行。

不需要大型科技公司，所有用户都对未来的决策有公平的发言权，以及参与其中的补偿。

其次，去中心化的本质意味着网络和网络上的任何服务都不会因为孤立的停电而停机。只要网络的一部分处于活动状态，它就应该正常运行，直到可以使更多节点重新联机。

这是可能的，因为这种系统的开源性质意味着成千上万的节点应该在世界各地运行，实际上消除了网络完全关闭的可能性。

最后，Web3 服务的去中心化治理还可以允许由社区而不是大型科技公司掌控的内容审核。每个用户都可以对给定平台上允许的内容以及大多数人所说的内容进行投票，这不会取悦少数人，而是将决定权交给少数高管的一步。

是什么阻碍了去中心化的未来

这在理论上都很棒，但仍然存在阻碍 Web3 真正愿景的主要问题。目前，大约 50% 的基于云的节点仅托管在 Amazon Web Services (AWS) 上。

这意味着，如果它们的底层主机离线，代表相当多网络的这些节点可能会面临严重的中断。事实上，这并不是史无前例的，而且已经多次发生，给各种各样的服务带来了问题。

此外，这种集中化开辟了允许审查节点的可能性，因为亚马逊可以选择随时关闭它们。

这导致 Web3 一目了然，但有一个巨大的、集中的故障点，破坏了它的真正潜力。幸运的是，这并非必须如此，完全由用户拥有的互联网不仅是可能的，而且它也已经在创建中。

正在部署真正去中心化的开源网络，不依赖任何集中式云架构，在任何情况下，它们都不能被公司或政府操纵或关闭。最重要的是，它们旨在提供与现有 Web2 基础架构上构建的服务一样快或更快的服务。

这就是将 Web3 的实际愿景带入主流的原因，现在许多区块链提供的主要是一个可靠的概念证明，但缺乏一些可以让它们兑现承诺的关键功能。

通过放弃对遗留系统的依赖并选择真正的去中心化，明天的元宇宙可以真正由最常使用它的人拥有。
