Web3.0因占据风口受到市场关注。web3.0是什么？与分布式存储又有什么关系？分布式存储怎样赋能Web3.0时代？IPFS领域重磅参与者，焜耀科技（原力区）品牌商务总监柏礼、石榴矿池技术负责人李白、开源矿池业务合伙人伯言、RRMine上海办事处负责人兰琪、西部世界总裁助理Leon罗祎果、KeyPool品牌负责人Kimmy Wen，于18日下午的圆桌论坛《分布式存储如何赋能web3.0时代？》中针对以上问题进行了精彩讨论。本场圆桌由巴比特8问主持人小别主持。

以下是圆桌论坛内容整理：

主持人：数据在存储上存在什么难题需要被解决？

柏礼 ：1、存储需求井喷爆发，真实存储容量难以满足
Web 2.0 存储需求的爆发不是直接来自个人或者企业级螺旋式的上涨趋势。随着互联网的快速发展以及5G时代、物联网时代的到来，短视频、数字化时代，让消费端的存储需求大增，这就连带上云诉求激增，进而带来企业级存储业务的增长。碎片化的存储需求积累起来，让企业面临着巨大的存储需求。

2、多元技术混合问题涌现，存储升级能够兼容
计算、云端、AI等等多元技术的快速发展，存储作为载体页面领着不同类型的数据对于容量、存储时间等要求的差异性，也让市场对更多项存储技术、存储介质多元共生。
假如处理不当会面临着延迟问题、并发访问、安全隐患等问题，尤其突出的是成本问题，存储如何满足数据的灵活性、应用感知灵敏度等会成为未来首要的市场关注点。
总的来说，存储不是基于存储本身的问题，而是在众多技术升级中一个互相匹配的一环，需要相互匹配，相互兼容的发展。未来，存储问题的解决不仅要面对存储本身，还要跳出存储本身，解决技术互相融合的问题。

伯言 ：这个问题跟开源矿池的定位非常匹配。第一是成本问题，之前从数据生成到存储相对比较断层，大家一直没考虑清楚数据的产生、存储、应用的成本。Filecoin作为分布式存储的龙头项目，大家其实致力于去降低数据存储的成本，因为所有的生产力推动都是经济效益的推动。
然后是数据隐私问题，数据的所有权权益一直都没有比较明确的划分，数据都在网上作二次传输。很多朋友都会遇到这样的问题：手机里面做任何操作都会被第三方APP获取，然后被推荐电商、游戏甚至金融服务，这就是在数据产生过程当中没有得到有效的保护，所有数据产生的额外收益归大数据供应商所有。
目前，全球数据还是指数级增长。数据产生后，数据的分类、存储、应用是非常庞大的应用资源。目前Filecoin全网的存储空间有超过4 EB，但是存储数据对目前的商用还不够大。到目前为止分布式存储尚未解决的问题主要还是带宽传输，这跟国家号召的5G新基建包括带宽的降费提速的政策是息息相关的。无论是从技术还是经济效益层面，我觉得还有很大的提升空间。

Kimmy Wen ：目前大部分数据还是存储于中心化服务器上。主要有几个弊端，一是集中存储容易造成单点风险，容易被攻击；带宽比较贵，导致资源冗余。去中心化存储（分布式存储）可以很大程度上解决这个问题。当然去中心化存储还是一个进程，用户的使用体验还待完善，但目前是非常好的开端。

主持人：Filecoin存储工具有什么特点？

李白 ：Filecoin相比以前的区块链系统主要实现点对点文件存储、传输和激励层，可跟最早的迅雷BT作类比，但这种协议缺乏相应的激励。Filecoin在点对点网络中加入激励层，用户可以通过存储、传输数据获得激励。Filecoin的slogan是让人类数据永存，小节点可以存一份有价值的数据不断获得奖励，而传统云服务厂商无法对访问量很少但却有非常重要价值的数据持久存储，这就是Filecoin存在的意义。
Filecoi系统在实际使用中还存在一些问题，主要是底层算法还无法实现实时把封装好的数据返回然后提供给用户。未来两三年的时间可能会进行版本的升级，算法会有更大的改进。

兰琪 ：IPFS技术应运而生，我们既可以享受存储的便利，也可以享受它带来的安全和隐私服务。很多互联网公司巨头有权限查阅我们存在中心化服务器上的数据，数据越来越多，存储故障压力越来越大，数据安全性难以保障，IPFS可以解决以上的困境和难题。

罗祎果 ：Filecoin作为IPFS协议的激励层被创造出来，设定了一个更加标准化的要求来规范存储服务商的行为，为应用需求方提供一个更加有保障的系统，其实IPFS是14年就出现到现在已经非常成熟的技术了，在filecoin出现之前，IPFS的存储服务商市场鱼龙混杂。
举个例子，今天你存储了数据进去，明天服务商就离线，你的数据找不回来，这是当年比较混乱的一个局面，存储的数据是图形、文本、视频等个人数据，它是属于个人一个阶段性的记忆来看，是非常有价值的。人类历史所有文明的记录都是这样一点一滴积累下来，我们创始人倪一叶先生讲过IPFS可以做到永久存储，实际上不管是错误也好，正确也好，人类历史上所有文明全部被记录下来将是一件非常伟大的事情。
更安全也好、数据所有权归个人所有也好，我觉得特点这个事，简单的对比上一代的存储技术不足以突出它的价值，它能够从技术角度让人类文明做到永续存在这个才是它最大的价值。

主持人：从技术的角度来分享一下分布式存储如何构建Web3.0。

罗祎果 ：Web3.0的底层是如何构建的，我觉得应该先从数据现存的问题来引入。
现在的数据是被存储在指定的中心化服务器内，黑客只要对目标服务器进行攻击，就可以获得大量用户数据，比如平台账号密码、甚至信用卡信息。举个例子，如果你曾经注册过谷歌账号，不妨现在登陆上去看一下，谷歌将会告诉你你的哪些密码已经被泄露。
分布式存储它采用可扩展的系统结构，利用多台存储服务器分担存储负荷，利用哈希地址定位存储的信息，这么做不但提高了系统的可靠性、可用性和存取效率，还易于扩展。分布式它是一种独特的系统架构类型，由一组通过网络进行通信、为了完成共同任务而协调工作的计算机节点组成。每个节点都参与数据的管理和存储，一个节点坏了，剩下的节点仍然能够继续工作，维护整个网络的稳定与安全。
分布式存储技术、区块链是与 Web 3.0 的方向契合度最高，特别是智能合约跟区块链账本这两块，在这里面，代码即一切，代码即规定，一切规则被写进协议，一旦开启，便无法篡改。代码开源公开透明，所有人都可以随时随地检索、检查。区块链节点分布式的协同共识，把交互所需要的信任需求降到最低。无需信任，只需验证代码。
分布式存储系统中，用户数据存储在去中心化的数据库上，不需要再去租用特定的服务器。因此，每次数据的访问都需要经过存储方你的同意才可以被授权，而且由于分布式技术把数据分开存储的记录方式，不存在单点攻破后的数据泄露问题，能够更好地保护用户隐私。
分布式存储有三个特性，首先是高扩展，基于标准硬件跟分布式架构，理论上节点数量无限，同时还有EiB级体量的扩展；然后是容易运维，块、对象、文件等多种类型存储可以统一管理；最后是上线快，标准硬件以集采方式供货，不用单独采购。
Web 3.0 可以把数据和数据带来的收益回归个人，私钥掌握在用户自己手里，也就是你把自己数据的使用权都掌握在自己手中，你可以选择保护它不然它被利用，也可以选择将其出售获得利益。
最后布个道，自由，并不是想做什么就做什么，而是不想做什么的时候就可以不做。早期的Web给世界带来了无限可能性的同时却没有把数据自由交给我们。基于分布式存储构建的Web 3.0将会大大提升用户的自由度。

李白 ：我们逐渐靠近Web3.0。想象一下，画一幅画并以NFT形式发布在以太坊平台上，把画（一个大文件）放在去中心化的存储网络上，全世界所有人都可以通过网络访问到这幅画，也可以通过智能合约进行交易，获得版权。这幅画在去中心化的网络Filecoin上永久存在下去。如果发生数据丢失，质押的损失远远超于继续存储数据维持联网的代价，所以目前来看数据丢失概率非常小。去中心化的存储、交易以及检索，这些属性合在一起会产生很有意思的应用。
应用的出现需要一定的时间周期，比如最初开始15年就有以太坊，等大家想到要用它做FIL是19年。一个应用的出现要有基础设施，基础设施有了以后要经历几年的思考和发现，才有真正的Web3.0的应用，这一天不会太远。

主持人：从应用层角度分享一下Filecoin如何构建Web3.0。

Kimmy Wen ：首先我们需要了解Web3.0，它到底是什么。Web3.0这个概念是由Web2.0这概念衍生而来的，而Web2.0这个概念是O’Reilly在20年前提出的，简单来说就是让用户通过Web端生成以及分享你的内容，现在逐渐演变成了平台经济。Web2.0存在的明显弊端是，平台经济产生的所有数据都存于平台上，所有产生的利润最后大多数都是进了平台方的口袋。
Web3.0的好处就是更开放透明的网络机制。Filecoin最大的强项就是引入Fil的激励机制，引申出的存储市场让每个人都有数据交换及提取数据价值的权利，Filecoin构建了Web3.0底层基础设施建设。现在Filecoin有很多应用，包括基于Filecoin的视频应用、播放器、游戏等等，还有现在热度非常高的NFT，顺便我也提下我们的应用Flame，创作者可以在这个平台上发布内容，协同创作，发起众筹… 而他们将拿到主要利润。所以我们说Filecoin本身是web3.0的底层基础设施，它将继续为Web 3.0的应用赋能。

伯言 ：Filecoin建立在存储空间的基础上，是给Web3.0做基础设施建设。Filecoin的存储是立身之本，如果所有的用户都采用以币挖币的形式，而不去提升Filecoin的存储空间，那Filecoin未来的市场价值就会大大下降。
现在Filecoin停留在数据填充获得算力的阶段。下一步可能进入有效数据存储的挖矿阶段，有效数据的存储应用会给日常生活中商业数据的上链带来新的尝试。相信很多人在短期内高估了Filecoin，但是长期内是低估了这个项目。我们应该从更长远的角度来看，无论做投资也好，还是Web3.0基础建设也好，有非常长的建设时期。经历过初级的建设时期之后，市场的红利期会远超过大家的想象。分布式存储或者Web3.0都是未来的趋势，希望大家顺应趋势，不要做逆势的事情。

主持人：当下分布式存储究竟遇到了什么样的难题，应该怎么解决？

柏礼 ：1、存储需求井喷爆发，真实存储容量难以满足
Web 2.0 存储需求的爆发不是直接来自个人或者企业级螺旋式的上涨趋势。随着互联网的快速发展以及5G时代、物联网时代的到来，短视频、数字化时代，让消费端的存储需求大增，这就连带上云诉求激增，进而带来企业级存储业务的增长。碎片化的存储需求积累起来，让企业面临着巨大的存储需求。

2、多元技术混合问题涌现，存储升级能够兼容
计算、云端、AI等等多元技术的快速发展，存储作为载体页面领着不同类型的数据对于容量、存储时间等要求的差异性，也让市场对更多项存储技术、存储介质多元共生。
假如处理不当会面临着延迟问题、并发访问、安全隐患等问题，尤其突出的是成本问题，存储如何满足数据的灵活性、应用感知灵敏度等会成为未来首要的市场关注点。
总的来说，存储不是基于存储本身的问题，而是在众多技术升级中一个互相匹配的一环，需要相互匹配，相互兼容的发展。未来，存储问题的解决不仅要面对存储本身，还要跳出存储本身，解决技术互相融合的问题。

兰琪 ：分布式存储需要很多企业和用户参与进来，贡献自己的服务器和有效数据。随着行业发展火热，我担心越来越多的从业者缺乏认真严谨负责的态度、缺乏专业知识，会从不够合规的角度去表述，或多或少会对这个行业的长远发展和健康产生一些不利的影响。这也是我认为目前面临的比较严重的困境。最早一批从业者需要向外介绍分布式真实情况、应用价值等等。希望我们都能够把这个行业做得越来越健康，越来越合规。

**主持人：分布式存储有没有能够给大家带来兴奋的最新进展？
**
柏礼 ：我们目前关注比较多的是Venus，这块昨天我们的CTO李昕进行了很详细的阐述，Venus除了同步很多网络以及模块化的工作之外，目前还做了很多标准化的工作，主要目的是为了降低了挖矿的门槛。Venus通过把整个软件分成了很多的组件。另外我们的Venus分布式矿池服务马上上线，可以更好的为整个网络的中小矿工来提供服务，促进分布式存储行业的标准化。

李白 ：最近有两大趋势都在研究和开发中。一是把分布式存储和NFT结合，NFT产出的文件可以存在分布式网络中，这是圈内非常迫切的需求，但目前还没有成品。第二是在分布式网络中把以太坊和智能合约迁移过来，Filecoin现在不支持智能合约，如果支持的话，流动性挖矿之类的都可以迁移过来，这有待进一步开发。
一般取得突破性进展都要经历熊市，一旦进入牛市无论是市场关注还是程序员本身都会比较浮躁，如果遇到熊市可能会爆发出一些好项目。

伯言 ：目前Filecoin有吸引很大一部分比特币、以太坊矿工，甚至有传统存储企业入场，这对Filecoin后面发展的蓄力有非常大的帮助。大家觉得短期成本非常高，但是实际上Filecoin后面的发展可以参考公司市值，比如说阿里云的千亿美金市值。对比传统领域，整个项目的市值还是非常低的。

兰琪 ：随着Filecoin的火爆出圈，更多主流资本主流机构都在入局。RRMine的有效算力超过了460P，单节点有效算力突破100P。所以我们RRMine也会持续加大对于分布式存储基建的投入，为更多的广大用户带来更良好的产品体验。目前有推出IPFS满存算力、独立矿机产品和待加工产品，在市面上具备一定的性价比。相信分布式存储的算力比拼也会过渡到真实数据的比拼上。RRMine也会携手传统互联网企业，与对数据隐私安全有需求的企业共同联手，助力分布式企业共同发展。

罗祎果 ：基于分布式存储的应用数量还是挺多的。不过最近热度最高、应用最广泛的就属NFT领域了，他的数据层、逻辑层可以是以太坊、可以是波场，但是应用层还放在中心化服务器上面，那就不可称之为NFT，一但中心化服务器ban了包括他的图像、文本，NFT就只剩下token。从应用这一点来看，完全可以用新的技术把所有老的商业再完整的做一遍。

**
Kimmy Wen** ：Filecoin整体还是越来越合规和商业落地。协议实验室最近推出了几个很大的措施，一个是公证人制度，不同地区对数据的监管不一样，在不同地区安排公证人是最初期对数据监管的措施，也会让真实世界的数据逐步迁移到去中心化网络。然后还有存储竞赛，也是让真实数据一步一步存储到Filecoin上。整体发展方向不单局限于挖矿，一定是会往服务层进行发展，这也是协议实验室的初衷。

主持人：对于FIL未来价值走势有什么判断？不作为投资建议，只是一种参考。

Kimmy Wen ：现在挖矿成本中只有不到10%出自于硬件，现在SAFT半年期，Fil6都释放完毕，这些因素都会对未来的币价上涨有一定支持。但当然现在币价也高，还是存在风险，大家自己来衡量。

罗祎果 ：它肯定会有增长的趋势，长期我们非常看好，短期振荡没有问题，长期还是很有价值。
