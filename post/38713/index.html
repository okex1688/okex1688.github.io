随着 Crypto 和元宇宙逐渐出圈，Web3 成为热门话题，并被冠以「互联网下一个时代」的宏大愿景。与此同时，Web3 社交赛道开始兴起，对中心化的 Web2 社交平台造成冲击。

在 Web2 背景下，各大社交巨头垄断了用户产生的大量数据，并因此获益；而基于区块链技术构建的 Web3 去中心化社交产品，将由用户完全掌握个人数据并授权第三方进行使用，同时用户也将通过自己的行为数据获得经济效益，永远不会面临「封号」的尴尬处境。

可以说，Web3 产品天然站在了 Meta（Facebbok）、推特等互联网社交巨头的“对立面”。但纵览历史进程，从 Web1 到 Web2，再到如今的 Web3，Web3 是技术发展推动下的必然趋势。时代的洪流滚滚向前，巨头们终究不能也无法扼杀 Web3 社交赛道的蓬勃发展。

本期，我们将盘点目前市场上多个兴起的 Web3 社交产品。需要说明的是，目前 Web3 社交赛道并没有明确的划分标准，诸如 ENS 域名、Mask Network 插件以及钱包等基础工具，都可以在广义上被划分到这一赛道。本文并没有将这些产品纳入统计范围之内，而是盘点了具有代表性的项目。

（1）Project Galaxy：链上成就系统

作为币安 Launchpool 首发的核心项目，Project Galaxy 的关注度在目前 Web3 社交产品中相对较高。Project Galaxy 核心目标是打造 Web3 开放的数据凭证网络，帮助用户积累“链上简历”，同时帮助开发人员利用数据接口，构建更好的产品及社区。

具体来说，其数据凭证网络包括两个部分：Galaxy ID 以及 Galaxy Credentials。所谓数据凭证，是指那些能够作为凭证依据提取出来的非量化行为数据，这些凭证是对 Web3 参与者在链上足迹的记录。例如， 为某个项目提供早期流动性、参与某个 AMA 活动等等。

这些数据凭证与用户的 Galaxy ID 绑定，参与者可以通过这个 ID，可以让用户展示自己成就的历史；同时，Galaxy ID 也是元宇宙中的身份认证器，为开发者提供了大量有关 Web3 参与者的信用数据。Project Galaxy 想要将 Galaxy ID 打造成普世的用户名，成为每个参与者在多链多钱包的 Web3 世界里面的身份标识。

Project Galaxy 的核心产品是 OAT Campaign。简单说，OAT 面向的用户是活动组织者和社区经理，提供的是一个 NFT 的制作分发平台，大大降低了社区发行激励类 NFT 的门槛。在这个平台上，任何社区都可以自行、便利地分发纪念类的 NFT。

但 Project Galaxy 在用户体验上也存在一些问题。Galaxy 的 OAT 是用户主动获取，项目方通常会设定 OAT 领取时间，过期则取消资格，一旦项目方通知不及时或者用户没有关注到信息发布，错过领取时间，则其所做出的奖励可能无法被认定。另外，用户通常需要通过 google form 等外部工具提交地址（Project Galaxy 也不支持域名识别绑定），同样可能错过收集时间，导致效果 OAT 的奖励效果并不明显。

（2）Mojor：面向Web3用户的社区平台

Mojor 是一个面向 Web3 的去中心化社区应用程序，旨在让任何人通过铸造 NFT 来创建和拥有一个真正属于自己的社区。MOJOR 的使命是帮助人们获取他们的在线社交图谱，并创建一个基于 Web3 的一体化社区平台。

简单来说，Mojor 是在借鉴 Discord 社区信息架构的基础上，增加了面向 web3 应用场景的 BOT（机器人）。用户可以在 Mojor 中进行聊天交流，同时通过一系列 BOT 完成交易、DAO 治理等 Web3 需求。

Mojor 用户通过钱包地址即可登录社区，无需注册，有效保护用户隐私。用户还可以在 Mojor 社区中直接搜索用户地址进行对话交流，打破了 Web2 与 Web3 无法互通的局面。在 Mojor 中，假如某用户看中了 BAYC# 6666 号 NFT，但该藏品并未上架出售，用户可以在 BAYC 社区中搜索该地址，找到后即可与之聊天，向对方进行开价。

对于 Web3 项目组织者来说，Mojor 的出现更是赋予其对社区实现控制所有权。用户创建社区服务器过程也是铸造该服务器 NFT 的过程，拥有服务器 NFT 就拥有了该服务器绝对的所有权和服务器最高权限，是一个真正属于创建者的社区，并且也可以通过转让服务器 NFT 的方式将服务器所有权转让给其他人。

此外，在 MOJOR 社区内，成员可以独立构建自己的去中心化社交图谱，包括社区数据、关系、POAP、个人行为等。此外，它还允许用户通过整合来自其他平台的成员信息，包括 DeFi 和 GameFi 数据、社交媒体信息等，从而形成用户在 Web3 世界的通用身份 ID。用户的 DID 也因社区沟通和交流，而变得更具价值，丰富社交图谱。

举个例子，一个项目方想给用户发放白名单， 你可以点击他的个人详情页，看他的链上数据，比如地址创建时间、链上活跃度以及交易量、拥有多少 NFT 以及稀有度，从而对用户价值有了一个清晰地了解，从而提升项目初始阶段用户质量，避免羊毛党抢占社区份额。并且，项目方还可以使用白名单机器人（WL Bot），可以设置相关参数自动进行抽奖、收集地址，解放社区管理员和 mod 工作量，提升社区治理效率。

Mojor 未来面临的困境是，如何在项目冷启动阶段，吸引更多的 NFT 项目方以及其他 Web3 项目入驻，从而形成正向循环，采取一定的激励措施或许会是一个不错的选择。

（3）Lens Protocol：去中心化社交图谱

Lens Protocol 致力于打造一个去中心化的、可组合的社交图谱，由著名的 DeFi 协议 Aave 团队开发。该协议主要用来帮助开发人员构建 Web3 原生社交媒体，任何基于 Lens 构建的应用程序都可以扩展社交图谱，并使生态系统中的所有应用程序受益。Lens Protocol 的创始人  Stani Kulechov 表示，Lens 是「开放的、可组合的 Web3 社交媒体协议，允许任何人创建非托管的社交媒体资料，构建新的社交媒体应用程序」。

从产品构建上，目前的 Web2.0 社交媒体，注册账户、认证身份、发布内容、关注他人、评论、收藏内容都是基本功能；在 Lens 协议产生的社交产品中，这些功能将融入 NFT 技术。

在使用 Lens 协议的社交媒体上，钱包将是用户身份的载体，创建一个社交账户相当于在区块链链上创建了一个配置文件，这个具有用户个人资料信息的配置文件将以 NFT 承载。

具体而言，Lens 把用户的主页铸造为“Profile NFT”，包含用户生成的所有帖子、镜像、评论和其他内容的历史记录，该 NFT 的所有权让用户可以控制自己的内容。只有拥有“Profile NFT”的用户才可以发布内容、评论和转发，其他用户可以关注一个主页，并获得“Follow NFT” 。此外，用户还能收集创作者发布的内容，获得相应的 “Collect NFT”。

当然，Lens Protocol 也存在一些问题。比如，其产品构建虽然采取了模块化的开发方式，但依然是照搬传统 Web2 社交产品结构，并没有足够的新意推动用户采用。此外，目前社区开发者较少，Lens Protocol 可组合 NFT 协议成熟度究竟达到怎样的程度，数据上链后 polygon 能否满足需求，这些落地实践问题都考验着项目组织者。

（4）DeFine：基于NFT搭建社交网络

DeFine 是基于 NFT 的去中心化社交网络，该平台支持创作者与其粉丝群体之间通过 NFT、社交代币等数字资产进行各种社交互动与交流。 其主要功能有四个：数字信用系统、元宇宙档案、NFT 展厅、个人代币发行（社交通行证）。

数字信用系统：基于 NFT 的社会材料将为未来基于 NFT 的 DeFi 功能铺平道路。例如，用户可以抵押他们的荣誉徽章（Poap），就好像他们在用“信用”贷款一样；如果用户未能按时还款，其荣誉徽章将被销毁并贴上非荣誉标签。

元宇宙数据：基于 NFT 的社交数据将在不同的元宇宙中使用，其中用户的活动反映在他们的个人数据中。例如，用户可以通过在不同的元宇宙中取得成就来获得荣誉勋章。

NFT 展厅：通过将资产附加到区块链和元宇宙以扩展 NFT 在艺术之外的应用，NFT 与可量化的实际权益绑定，以解决 NFT 背后资产的主观性。

社交/粉丝通行证铸造：创造者可以推出自己的社交交通通行证，并定制通行证的使用，以提高粉丝参与度，探索与自己社区互动的新方式。创作者可以使用其通行证奖励任何活动的粉丝，包括参与 NFT 销售、为社区做出贡献、为新项目和计划提供财务支持等。通过解锁参与特殊 NFT发布、私人社区、商品或内容的优先级，社会交通证将作为货币在创造者社区中流通。

DeFine 的问题在于，项目为用户建立基于 NFT 的社交档案，却并不存在留存用户的强激励行为。另外，Poap 与 DeFi 结合落地的难度较大，在未来相当长一段时间内都很难实现。

（5）BBS Network：去中心化版 Reddit

BBS Network 是一个类似于 Reddit 或贴吧的去中心化社交网络，用户可以在其中创建论坛来发布、评论和分享想法，同时从他们的内容中产生收入。任何用户都可以在该平台创建 BBS，每个 BBS 都可以在自己独特的域上运行，同时仍然链接到所有其他 BBS ，其中的创作者、策展人和利益相关者以自动和透明的方式获得项目的原生代币奖励。

在 BBS 上发布的每个帖子实际上都是 NFT，用户可以购买、出售甚至租赁，同时收集他们拥有的任何帖子产生的广告收入。在 BBS 上，帖子的影响力由实际市场力量决定，其创造的价值会在帖子的创建者、当前所有者和板块管理员之间共享，这会激励每个人尽力使 BBS 尽可能吸引观众和利益相关者。

每个 BBS 所有者都可以设置自己社区的审核规则和内容政策，用户可以自由选择访问和参与哪些板块以满足他们的需求。BBS 可以使用插件模块进行定制和增强，促进用户的选择和多样性。

BBS Network 可能面临的问题是：NFT 的价值验证需要靠浏览量、评论等数据，然而一个已经完全曝光的帖子又如何对广告主产生对应的价值呢？帖子被更多人看到意味着其在潜在购买者心中的意向价位越高，但也意味着更少的人能够被广告触达，这可能会使投资者与广告主之间错配。另外，如何能在不引起用户反感的情况下让广告触达已浏览 NFT 的用户？

（6）Torum：去中心化Twitter

Torum 是基于 BSC 的一个专为 Web3 而设的社交平台，旨在为用户建立一个完整加密生态系统，支持 DeFi、NFT 以及社交功能。

社交是 Torum 的核心与基础，其社交界面类似于 twitter，用户可以进行发帖分享、浏览订阅及热门帖子、发表评论等活动。Torum 将加密用户的活动整合到社交板块中。在“companies”板块中可以看到入驻公司的业务动态，在“clans”板块可以加入各类不同主题的群组，“airdrop”板块提供空投信息及参与机会，“news”板块以聚合器的方式推送新闻，待推出的“torumgram”板块允许用户在torum上使用telegram，“lounge”板块则提供语音聊天服务。

Torum 最明显的问题是，其愿景太过宏大，功能众多，但各个板块之间要想产生协同效应较难，容易头重脚轻，最后可能导致众多功能逐渐沦为鸡肋。目前的情况是，出了社交之外的其他板块数据并不理想。此外，这些功能的开发、运营维护，也会加重了项目自身负担。

（7）Mem Protocol：去中心化版「知乎」

Mem Protocol 旨在打造 Web 3.0 版本的“知乎”，让用户通过分享知识赚取收益。通过智能合约，Mem 用户可以在区块链上验证自己的身份，并建立自己的可验证的声誉体系。该协议还构建了一个界面来浏览链上钱包的链上数据，并设计一个开放协议来记录钱包和代币生成的交互和关系。

Mem 将网络参与者的信息——包括社交媒体历史、链上活动和专业经验——存储在加密的保险库中。该数据由用户拥有和控制，当用户允许时，这些数据可以作为“社会信用评分”或“社交图谱”。

通过使用声誉体系和智能合约，Mem用户可以使用他们的身份在区块链上进行那些在现实世界的动作，例如提出问题、为赏金筹集资金以及建立自己的可验证的、特定于场景的声誉体系。

Mem 希望使用社交代币和智能合约来促进建立基于该协议的应用程序和社区，例如产品评论、社交媒体和求职等广泛应用。

Mem Protocol 的问题是，其竞争对手遍布整个 Web2 网络，包括知识星球、得到、分答等等，并不具备特殊优势。链上数据凭证，在知识付费中能否成为核心竞争力目前还是一个未知数。

（8）FaceDAO：「人肉挖矿」的Web3社交平台

随着 Web 3 的到来，用户已越来越关心获得自己数据所有权的重要性，并希望出现一个为所有用户提供平等利益的社交媒体平台。

在机制设计上，FaceDAO 将用户的注意力货币化，社区是最终的受益者，每个生态的参与者和建设者都可以获得代币奖励。通过一个公平的代币奖励系统，内容创建者和消费者都能在每次点击中获得代币。在 FaceDAO，用户只要自然且正常的使用产品，自然会获得生态对他做出的奖励，每天访问、点赞、评论、分享、发帖……所有积极的行为都将自动奖励代币，堪比「人肉挖矿」。

此外，FaceDAO 利用人脸识别技术为用户在以太坊上创建唯一的数字身份。FaceDAO 允许用户自行掌握自己的数据，并提供了一种更安全、更私密的方式来共享信息。采用了人的面容作为其平台的登陆门槛，每个用户个体的面容是平台内一切数据的载体。在 FaceDAO 中，每一张面容都是一份私钥。

FaceDAO 的问题在于，该平台虽然通过代币奖励用户，但数额较少，并且在没有广告客户的背景下，这种长期烧钱的模式并不会持久。

（9）总结：Web3社交成为下一风口

围绕 Web3，社交平台似乎开启了一个新的时代。浪潮愈演愈烈，传统巨头开始纷纷下场。

Meta、TWitter、Google 等巨头都发现了这群想要取代自己的公司，并且积极拥抱 Web3。毕竟当前的互联网巨头们不仅有强烈的危机意识，同时也有强烈的创新意识。比如像谷歌、苹果之类的巨头都投入了巨大的资金在对未来的技术进行研究；推特宣布推出新功能，将拥有 NFT 的用户头像显示为六边形；Meta、Youtube、Reddit、Google 都宣布尝试推出 NFT 产品，加入 Web3 产品的研发中。

目前加密市场原生的 Web3 产品，虽然有着一定的先发优势，但仍然有很大的进步空间。特别是当巨头们蜂拥而至时，留给前者的生存空间势必越来越小。狭路相逢勇者胜，这一场好戏才刚刚拉开序幕。
