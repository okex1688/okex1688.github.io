原文标题：《Dani’s vision》

分析

随着令人期待的 SOLID（Dani 和 Andre 的最新项目）的推出，Dani Token 生态系统很可能会在短期内大幅增长。在我们进入市场去买卖受追捧的 rebase 庞式币之前，让我们先回过头来，试着了解一下当前 Dani/Andre 生态系统。

Dani 从事加密技术已经有一段时间了，由于他积极的个性和参与了大量的 DeFi 协议，他最近变得越来越受欢迎。Dani 的追随者称自己为「Frog Nation（青蛙国度）」，他们是我见过的最忠实的 Token 持有者。

Andre 主要以他创建的 Yearn Finance 而闻名，这是 DeFi 最著名的收益聚合器之一。他还参与了 Keep3r 网络的开发工作，该网络似乎仍在开发中。

Abracadabra

Abracadabra 是 2021 年较成功的 DeFi 项目之一，累计 TVL 为 60 亿美元，并推出了 SPELL 治理 Token 。对 SPELL 来说，也是可观的一年。从远低于 1 美分上涨到近 5 美分，为投资者提供了相当大的回报 (从低点上涨 3400% 左右)。

Abracadabra 为 DeFi 用户提供借贷，耕作和质押一站式服务。Abracadabra 最初始于以太坊，其最近又开发了跨链项目，被用于 Avalanche 和 Terra 生态系统中。

Dani 从一开始就参与了 Abracadabra 项目，这是在他推文最活跃的项目之一。无论是关于 Token 销毁公告，还是 Abracadabra 的原生稳定币 MIM(magic internet money) 的宣传。看起来 Dani 更喜欢 Abracadabra 而不是 Popsicle Finance。

Abracadabra 的声誉之所以提高，部分原因是 Tetranode 的进入，这条互联网巨鲸最近以笔记的形式分享了一些 alpha。

看到关于 Curve Wars 的内容了吗?Dani 从不糊弄，这就是为什么 Abracadabra 最近一直有热度的原因。即使在每周都有故事来来去去的加密推特圈（Crypto Twitter）环境中，Dani 和 Frog Nation 也没有让 Abracadabra 从话题中消失。这是值得肯定的。

在 veSPELL 和减少释放的计划下，很难不看好 Abracadabra 的长期前景。我们已经通过 Degenbox 策略在 Abracadabra 上看到了跨链兼容性。添加一些激励措施来吸引用户转向 Fantom 并没有什么不好，而且还可能在 Fantom DeFi 生态系统中创造一些获利机会。

Wonderland

Wonderland 和它的 rebase Token TIME，是在 2021 年最具影响力的 DeFi 项目之一。作为 Avalanche 上的第一个 OHM 分叉，Wonderland 一度颠覆了 OHM 的市值。

TIME 的成功可以感谢 TikTok，许多年轻的投资者因 TikTok 抓住了这个项目和它高得离谱的 APY。尽管发展之路颠簸，且 TIME 目前与其价格高点相聚甚远，但 Wonderland 仍能设法保持其 PoL 和热度。让我们称之为「Dani 效应」。

尽管绝大多数的 OHM 分叉都没有提供什么价值，且有的很快就消亡了，但由于 Avalanche 的低成本和 Dani 对项目的营销，Wonderland 实际上有能吸引用户入圈 能够将新用户纳入加密货币。

尽管绝大多数 OHM 分叉几乎没有提供任何价值，且有的就即刻消失了，但由于 Avalanche 的低费用和 Dani 对该项目的营销，所以 Wonderland 实际上有吸引新用户入加密圈的能力。

尽管 Wonderland 最近有些挣扎，但的确没人会与支票过不去，而 Wonderland 最终可能会在某个时候成为 Dani 的个人支票簿。我支持这些说法吗？并不见得。

我只是相信，如果没有其他选择，Dani 可以拿走一大笔资金，投资到他想投资的地方。如果这恰好将 Wonderland 国库视为一个以 DAO 形式运作的风险基金，那就顺其所愿吧。这肯定会是一种投资未来金融的有趣方式，而且对 Dani 来说，拥有协议的股份是最符合利益的，这些协议可以证明自己在未来是成功的。

这一切都只是猜测。

Olympus 和 Wonderland 经历了一些价格打击，许多人开始怀疑，这种新的协议模式是否适合运营，是否能将价值回报给股东。也许我们不得不再次提高 APY 才能让人们感到安全感。我认为有太多的过度反应，人们正忘记 9 位数资金所产生的可能性。无论如何，为更大的波动做好准备。

Popsicle

曾几何时，Popsicle Finance 曾是所有人的焦点，而 Abracadabra 抢了这个风头。从本质上讲，Popsicle 是一个收益聚合器，旨在成为 DeFi 的做市商。Dani 表示，到 2022 年第四季度，他希望 Popsicle 能够带来一个完全去中心化的基于订单簿的期权交易平台。对于一个与 Dopex 等先行者竞争的平台来说，这是一个相当崇高的目标。

Popsicle 的 TVL 高于 5000 万美元，并希望提高资本效率并为用户提供尽可能高的收益。作为一个区块链不可知论的收益聚合器，Popsicle 为那些希望在探索以太坊区块链之外的可能性的同时获得更高 APY 的人提供了许多机会。

Popsicle 与 Dani 实现高效金融基础设施的首要主题一致，在 Popsicle 的帮助下，将跨链生态系统中曾未链接的孤岛连接起来。Dani 的目标是实现一个开放的 DeFi，在这个 DeFi 中，用户可以与任何他们所需要的链进行交互，甚至在他们不知道正在使用的哪一条链的情况下。Popsicle 实现这一目标的主要产品之一是 Sorbetto。虽然 Popsicle 最近没有受到太多关注，但 Dani 最近的推文表明他仍然对这个项目很感兴趣。

Sushi

2021 年底最令人意想不到的公告之一是 Dani 向 Sushiswap 的提议，以及随后的 SUSHI 动荡。在 Sushiswap 的核心团队发生了一些争吵之后 (我在这里就不赘言了，因为有太多的戏剧性和废话)，Dani 以一个完美时机的提议，扭转了局面。他的计划包括全面专注于 Sushi 成为多链 DEX 的能力，以及投票托管 Token 经济学为用户提供价值。

这会成功吗？可能不会。

老实说，我没跟上有关 Sushiswap 事件的任何进展，我的时间表因 Curve Wars 变得一团糟。然而，我认为这对 Dani 来说更像是一个宠物项目，他最近与 Andre 的关系可能会占用他更多的时间。

也许他确实成功了，看到 Sushiswap 在 Fantom 上获得一些关注会很有趣。我认为这对需要 DeFi 协议来大幅改进 UI 的区块链来说大有裨益。但那只是我的个人意见。

但是，如果 SUSHI 要实施投票托管 Token 经济学，事情就会变得一团糟。世界将会翻个底朝天。Dani 会让 cz 的净资产缩水。这就是为什么拥有一点点 SUSHI 很重要，以防 Dani 能实现他的梦想。听天由命吧。

Yearn

对于 Andre 的项目历史，他最出名的是创建了 Yearn Finance。即使在上面交易就像稳定币那样，但 Yearn 是一只野兽，拥有超过 50 亿美元的 TVL，并且是所有加密货币中最开放和最高效的 DAO 之一，Yearn 并没有让人失望。从一个公平的发布方式开始，Yearn 发布了他们的原生 Token YFI，这是加密领域一次最受尊重的 Token 发行之一。

最近，Yearn 宣布对其 Token 经济学和 Token 回购机制进行修改。从本质上讲，他们想成为 YFI 的黑洞，因为用赚取的收入回购他们的 Token 只会使他们长期受益，这就是 PoL。

Yearn 将永远在 DeFi 中占有一席之地。其作为历史前三的地位无可争议。这要感谢 Andre 和其敬业的团队。我相信 Andre 产出的任何项目或信息都基于这个原因。

Keep3r 网络

老实说，并没有太多的信息是关于 Keep3r 的。用他们自己的话来说，Keep3r 将自己描述为「用于需要外部 DevOps 的项目以及外部团队寻找 keeper 工作的去中心化网络」。

也有关于 veKP3R 的传言，此外还有 Andre 创建了一个新的 DeFi 原语，允许 LP 出售 theta 以赚取其头寸的额外收益。关于这方面的更多信息，可以在 Twitter 上关注 @JackNiewold, 他比我更了解 DeFi。

无论 Keep3r 的结果如何，它可能在 Curve Wars 或甚至 Andre 的新 Token 具有某种功能。我没有什么想法。不过，这个概念相当有趣，因为它已经在计划扩展到更多的链，包括 Fantom。

一个非常复杂的谜题

我将 Dani 和 Andre 的新 Token 称为 SOLID，因为它似乎是根据 Andre 的 Medium 帖子中的信息命名的。我知道它也可以称为 ve(3,3) 但我会在这里碰碰运气并参考它更可能的名称。毕竟，命名一个 Token 为 ve(3,3) 是相当蹩脚的，可能不会被大众所接受。

在我看来，SOLID 试图扩展我们在 DeFi 中看到的问题解决方案。回到 2020 年的 DeFi 之夏，我们看到流动性挖矿很快就被淘汰了。新的协议将会出现，LP 将获得巨大的激励，一旦奖励结束，协议将随着 LP 的离开而消失。重复不断。

OlympusDAO 能够通过他们对协议拥有的流动性（PoL）的新方法来解决这个问题。Olympus 不需要担心唯利是图的投机者，其能够鼓励投资者入股，最终以债券的形式进行投资，总额超过 7 亿美元。债券允许 Olympus 在特定的时间范围内控制他们的流动性，债券会被设定为一个特定的时间段，这意味着储户的资金基本上被锁定并交于 Olympus。

尽管有这种创新，OHM 的价格还是从高点迅速下跌。对于 TIME 和其他 OHM 分叉也可以这样说，但 TIME 对于 Dani 尤为重要。

我们现在的处境是，Olympus 和 Wonderland 都拥有巨大的资产，但几乎与它们无关。我说他们对资金库无能为力，因为现实地想想，他们能做什么？显然，他们的 Token 具有某种形式的内在价值。如果 OHM/TIME 的估值低于资金库资产价值，那么 Token 被低估。但这很无聊。

我们没有人来加密圈寻找被低估的资产。我们来这里是为了高 APR。我是否看到了 Olympus 或 Wonderland 解决此问题的路径？目前没有。rebase 庞氏经济有点超前，我认为这两种协议都处于一个棘手的情况。他们已经走到了死胡同。

继续，回想一下 Curve Wars。Curve 和 Convex 抢尽风头，展示了 Ve 式 Token 经济学的力量以及像 Tetranode 这样的巨鲸影响力。许多人很快就抓住了这个新机会，像 YFI 和 RBN 这样的 Token 宣布采用类似 Curve 的 Token 经济学。尽管加密推特圈转向 FOAN 和 SOLID 等其他趋势，但 Curve Wars 仍然非常重要，在接下来的几个月里，更多的协议很可能会采用 Ve 式 Token 经济学。 Token 被锁仓，供应减少，价格上涨。就这么容易。

两个 DeFi 巨头

Dani 和 Andre 都有很多承诺和项目，并都与 Curve Wars 有某种联系。在这种情况下，由于 Dani 与 Abracadabra 的关系，他的参与程度略高。在目睹了 DeFi 中的创新和猜测之后，我可以假设这两个人走到了一起，并寻求一种方法来结合 DeFi 中两个最大的 meta。

ve(3,3) Token 经济学

是的，这是一个真事。是的，它可能会引起很大的争议，或者说会涨得很厉害。人们喜欢庞氏骗局，所以这几乎是必然的。老实说，我不确定这将如何长期发展，但是，如果目标是短期的价格升值，从最近的两个 meta 中搭便车比想出一个新的 meta 要好。让我们来看看这是如何运作的。

我要发布的所有信息都来自 Andre 的 Medium 帖子。如果在你阅读本文时，其 Token 发布了或有信息更新，对不起，那这篇文章已经过时了。在加密推特圈，事情发展得很快。

SOLID 计划通过 Ve 经济学将通缩供应的好处、锁币的二级市场（以前无法出售）和对质押者的激励（rebase 奖励）结合起来。

从根本上看 SOLID，该 Token 似乎旨在为治理 Token 提供市场。如果你能够在二级市场上出售锁定的 Token ，则可以将其视为出售选票。由于投票托管经济学的锁仓机制，由于奖励会增加，锁仓时间越长则更有意义。如果一伙人选择以最长时间内锁仓其 Token ，那他们可以获得奖励并抛售其 Token 。但之前似乎没有这样的积极性事件。

正如我之前提到的，我相信 SOLID 试图解决 DeFi 中的许多问题，其中一个更大的问题是在 Curve Wars 中缺乏购买v作为投票代理的能力。这与 [Redacted] 的做法类似，但有一些细微差别。[Redacted] 通过债券购买 CVX、CRV 和 OHM，而 SOLID 尚未提及相关信息。Curve Wars 还没有迁移到 Fantom，但谁知道未来会发生什么。

SOLID 旨在成为一个公平的启动 Token ，因为它将通过空投分发到 Fantom 上的 TVL 前 20 的协议，这是一个大胆的举措，但可能会奏效。这上面到处都是 Andre 的名字，就像 Yearn Finance 对他们的原生 YFI 所做的那样。向社区空投的策略也与 Dani 的反 VC 理念相关，并有助于激励他们向 Fantom 过渡。这是一个多赢的局面。

SOLID 有一些有趣的 Token 经济学，因为供应被锁定的越多，就会有更少的排放量，此外锁仓的 APR 更高，这就是 (3,3) 特点。

通过为协议提供流动性，这可以作为一个 AMM，它不受流动性挖掘控制，而是由社区控制。这是对 PoL 的一个更有趣的看法，本质上是颠覆了这个想法。看看这些协议如何发挥作用，以及它们如何利用它们所处的位置，将是一件有趣的事情。

总的来说，这是一种非常新颖的方法。我认为如果一切顺利的话，这将很容易进入到 DeFi 的上层梯队，并有可能改变 meta。我不确定这一切何时发布，但我会持续关注。听天由命。

DeFi 的终极头目

我对这种情况的看法

从局外人的角度来看，感觉 Dani 这一次做的太多了。不管他是否能看到自己的梦想成真，像他这样施展才华是没有意义的。当然，在 Fantom 生态系统中现在有很多机会，而且 rebase 托管投票 Token 听起来很有吸引力。但有人能实现如此规模的愿景是毫无意义的。

如果 Dani 看到 frog nation 改进的 Sushi，并给 Fantom 带来数十亿的 TVL，并通过 Abracadabra 获得跨链的未来，我会感到震惊。这并不是说我不尊重 Dani 的职业道德，因为很明显他是个研磨者。与我们所见过的策略相比，这是一种非常不同的策略。Dani 正准备完成有史以来最成功的横向收购。他几乎什么都要插手。

在写这篇文章之前，我想知道 Popsicle Finance 是否已经在 Dani 的生活中消失了。我一直在努力为 Wonderland 想出一个路线图，以及 Dani 将如何想出更多为 TIME 持有者提供价值的方法。我试图设想一个跨链的世界，在这个世界里，Sushiswap 是大家都可以使用的 DEX。虽然我仍然持怀疑态度，但我的脑海里开始浮现出这些想法。

如果你还没有阅读它，去看看 @j0hnwang 在 Twitter 上的这个话题。他在描述 DeFi 的一些霸主方面做得非常好。Dani 正在尝试组装 DeFi 的乐高死星（乐高帝国的终极行星轰击武器），尽可能多地抓起他能抓到的积木块。这对他来说会有一个很好的结果吗？我很好奇。

DeFi 天启三骑士

Dani 的很多币表现已经远远超过了大部分市场，特别是当你考虑到许多旧的 DeFi Token 表现不佳的时候。只要看看 SPELL 和 ICE 的图表，从低点买入其中任何一个，都会让你获得任何 TradFi 基金经理都会垂涎的回报。它们现在的价值是否稍微被高估了？也许有一点。但如果我们在加密推特圈上看到了什么，即使是已被高估的，它也总是会涨得更高。事实上，我们甚至看到没有价值的币也在上涨。

这一直是一个非常有趣的兔子洞，让人深陷其中。我不得不做大量的阅读，但更多的是去思考。加密货币的发展速度如此之快，以至于几乎不可能预测下周会发生什么，更不用说明年了。

无论你站在哪一边，你都必须尊重 Dani 和 Andre 的勤劳工作。在 DeFi 中，很少有人拥有这样的职业道德，能够忍受他们所受到的审查和强烈反对的人就更少了。无论 SOLID 发生什么，加密推特圈都会有收获。

人们也可以赞扬 Dani 和 Andre 为鼓励跨链的未来而尽其所能。不是所有加密推特圈的人都能负担得起每天在以太坊上交易，更不用说那些希望体验 DeFi 的普通投资者。希望 Fantom 能在这里取得一些成功，并实现其目标，让大众进入一个可负担得起的快速区块链。

不管这个项目是否成功，Frog Nation 和他的 Twitter 追随者为 Dani 建立起了壕沟。我很难相信他不会继续保持高成功率。Dani 重塑了摇滚明星的形象，超越了在加密推特圈内流行的意义。他已经集结了一支军队，现已经招募了一位最受尊敬的建造者来帮助他创造终极庞氏 Token 。

愿我们都能成功。
