5 月 20 日至 6 月 3 日，首届 ETH Shanghai 黑客松大赛作为以太坊上海峰会 ( https://ethshanghai.org/ )(ETH Shanghai Summit)的平行活动，以“下一代社交”为主题，全程在线上举行，遍邀全球 Web3 爱好者、开发者共同关注互联网和社会责任之间的密切联系，就当下热切探讨的公链、链上数据分析及去中心化社交等议题点燃想象力，来共同定义 Web3 社交的未来。

大赛筹集了总计超过 14 万美元的奖金，聚集了最前沿的 Web3 协议开发者们为黑客松提供丰富的资源和 SDE 工具。通过黑客松大赛，主办方希望共同建设一个更加开放的社交空间，并创建一个共管、共有的去中心化治理结构。

黑客松期间共收到超过 900 名选手报名和 200 余个项目提交，经过评审的打分后，24 个使用了赞助商提供的开发工具的优秀项目脱颖而出，涵盖多种Web3应用场景，如：婚恋、链游、邮件、社交等等。

1，Metamail （金奖）

官网：https://metamail.ink/

MetaMail提供 Web3 邮件服务，兼容普通邮件，可以自由且免费地跟任何其他邮箱（如 Gmail 和 QQ 邮箱）收发邮件。

MetaMail 用户可以把钱包地址 / ENS 作为邮箱地址，通过数字签名和端到端加密来加强邮件安全性。

访问metamail.ink，连接钱包，就可以得到自己的邮箱，后缀为@mmail.ink。整个过程只需要一个签名来验证账户的所有权，完全免费，没有gas费。用户不会泄露私钥或其他信息。

在 MetaMail 中，编辑、发送和接收邮件与其他邮件服务几乎相同。在发送之前，用户需要对电子邮件的内容进行签名以确保其完整性。签名将存储在电子邮件 EML 的标头中。电子邮件服务提供商和收件人可以随时验证签名。

2，ETHChat

官网：https://eth-chat-git-v2-georgefane.vercel.app/

ETHChat 在 XMTP 和 CyberConnect 中提供了过滤器服务，用户可通过此过滤器筛选出来自自己关注的人的私信。用户界面将由关注、被关注、互关三个部分组成，并可通过 ETHChat 发送关注请求。

3，Web3NFT.social

官网：https://dev.web3nft.social/

Web3NFT.social是面向 Web3 新手用户的 Web3 NFT 开放平台。

目前有两个主要特点：

  • 用户可以关注/取消关注的每个钱包地址的链接树主页，并与钱包地址聊天

  • 聊天系统具有全功能过滤逻辑，支持 Cyber Connect 数据和钱包地址的链上数据

后续，在 Web3NFT.Social 平台上会陆续推出 NFT 的 SaaS 服务，如提供自动化 NFT 生成服务，在主页上可以设置基于 LIT（https://litprotocol.com/）的链上权限验证以实现在平台上的各类服务，以便服务更多 Web2 用户来使用 Web3。

在其发布的路线图中，平台还将添加 $NTB 代币供用户购买和作为平台币使用（每项活动都需要使用 $NTB 支付）。同时添加更多功能，使 W3NS 更像一个开发人员开放平台，人们不需要重新发明基础设备，只需要专注于构建应用程序功能。

4，Superfluid Ceramic Addressbook

官网：https://superfluid-ceramic-addressbook.vercel.app/

目前，支持 Arweave钱包登陆。

10，Popula

官网：https://testnet.popula.io/

Popula 是致力于 Web3 社区经济的基础设施，它使创作者经济的参与者能够通过创作和社会参与建立他们的互联网声誉和货币化。

具有如下特点：

  • 一个用户友好的 Web3 社区平台，用于放大创作并增强社会参与度，其中内容、数据的所有权和治理权真正属于用户。

12， Fraction Interface

官网：https://frac-tion.vercel.app/

Fraction允许用户在特定资产上赚取收益，能够将资产（通常是稳定币）包装到 ERC-1155 并以自定义代币的超流形式赚取收益。Fraction还提供铸造灵魂绑定NFT的能力，将用户绑定到他们的资产并为他们赢得更多的信息流兴趣。

13，DAOU（铜奖）

官网：https://www.daou.io/

DAOU 使用 Soulbound Token 和 Social Oracle 将您在 DAO 中的活动数据：Discord 活动和参与、论坛问答、Github 代码贡献、DAO 角色和连接转换为链上可验证数据。Social Oracle是指激励同一个DAO中的成员对DAO成员产生的数据进行签名和验证，从而使这些数据更加可信，并根据这些数据给每个成员打分，这个分数就是DAO中的成员Credit。Social Oracle 验证的数据将通过 DAO 发布到 Soulbound Token，然后可以对外使用。例如：信用贷款、Soulbound 代币持有者的空投、防止 DDoS等。

github地址：https://github.com/DimensionDev/Maskbook/pull/6388

Def Network 收集链上钱包行为，例如合约交互和代币交易，将它们整合到易于理解的统一配置文件中，然后使该配置文件可用于可能需要它的其他系统——例如生成白名单、构建 SocialFi 应用程序或营销工具等。Def Network还为 Mask 创建了一个插件，以使用 def/POAP/Galaxy 显示社交资料。

15，Scam Sniffer（最佳 Web2-Web3 桥梁项目）

Scam Sniffer是一个Mask插件，可以帮助人们对抗加密诈骗者。

Scam Sniffer汇总了来自 Coingecko 和 Opensea 的热门项目，一旦找到推文中提到的项目，将检查链接和推特帐户信息，诈骗链接将被聚合并报告给 MetaMask 网络钓鱼检测，每个 MetaMask 用户都可以共享保护！

16，Zksafebox

github地址：https://github.com/busyapedao/zksafebox-contract

ZKSafebox 为资产、私钥和密码提供双重安全保障。ZKSafebox是基于零知识证明的资产保险箱，取出资产的时候需要密码，密码不触网。若密码被盗，保险箱依然保持安全；若私钥被盗，保险箱也依然保持安全；只有密码和私钥都被盗的情况下保险箱才会陷入风险。如果用户忘记密码，可通过社交恢复。

19， Arche Network（最佳社交游戏项目）

github地址：https://github.com/Archenetwork?tab=overview&from=2022-05-01&to=2022-05-31

Arche正在构建社区驱动的 Web3 游戏生态系统，以增强和团结游戏玩家、公会和创作者。一旦用户连接了其所有的钱包地址，Arche 将读取并分析附加的钱包地址的链上关于 web3 游戏的数据，然后为用户生成一张 gamefi DID 信息卡。Arche 将根据 Arche 从用户钱包地址获取的数据提供各种 NFT 奖牌，用户可以收集符合条件的 NFT 奖牌并显示在可定制的 DID 卡中。

根据用户钱包地址的交易数据，Arche 可以计算出用户玩的每个 web3 游戏的 ROI 率。此外，为用户和游戏提供标签系统以建立用户档案。在玩家卡中，可以看到用户（除了他们将卡设为私有）PFP NFT、徽章、游戏卡、个人标签、愿望清单、历史评论等。

Arche还将与游戏和公会合作，举办季节性的web3游戏比赛，为游戏和公会寻找高水平的玩家。

22，Kwil Database Cli（银奖项目+最佳 Arweave 生态项目）

github地址：https://github.com/xendarboh/kwildb-cli

Kwil Database Cli是一个交互式终端，命令行客户端和数据库导入工具。在概念上仿照 mysql 和 psql 终端客户端，以获得传统舒适的开发者体验。它使用户能够交互式地输入查询，向 Kwil 数据库发出查询（可选择同步到区块链），并看到查询结果。SQL 输入可以作为一个数据库导入工具从文件中读取。一个配置向导允许 Kwil Data Moat 配置轻松创建并快速恢复。
