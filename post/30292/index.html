IGNITION 是一场专注于 Web3、DeFi、游戏和数字艺术/收藏 NFT 的全球黑客马拉松，Solana Ventures 今天很高兴宣布本次大赛的结果。IGNITION 是 Solana 的第四届黑客松，也是今年举办的第三届以及最后一场比赛。从 8 月 31 日到 10 月 15 日，黑客松汇集了来自世界各地的加密货币先驱和开发者，他们在 Solana 区块链开发项目，为后来被称为“Solana Summer（Solana 之夏）”的项目添砖加瓦。

IGNITION 面向全球的团队，最终将近 6,000 名参与者向评委提交了 568 个项目。微软、Jump Crypto、渣打银行、Forte 和 Metaplex 作为本次全球黑客松的赛道赞助商，携手 Solana Ventures 共同为参赛者提供高达 500 万美元的全球奖金池和种子轮资金。

“Solana 生态在过去一年经历了惊人的增长，而 IGNITION 黑客马拉松无疑为区块链行业的优秀团队提供了一个绝佳的展示舞台。”Solana Labs 首席执行官 Anatoly Yakovenko 表示 ，“我们非常高兴看到 Solana 出现的这些超级优秀的项目，并期待它们的发展。”

来看那一下获奖项目都有哪些：

总冠军

Katana，一个专注于跨风险范围开发投资产品的资产管理协议。Katana 获得了 75,000 美元的总冠军奖励以及参加第一届 Solana 年度会议 Breakpoint 门票，他们将在大会上展示项目。

DeFi 赛道

DeFi 赛道的第一名为 Atrix，一个支持在 Serum 无缝创建流动性池和挖矿的 AMM，奖金为 65,000 美元。

本赛道的其他获奖项目包括：

第二名：Superposition，去中心化固定收益协议。

第三名：Aver，去中心化和最小化信任的博彩市场。

第四名：01 Protocol，在 Solana 区块链提供永久期权的去中心化衍生品协议。

第五名：Fluidity，允许用户通过 Fluid 资产使用他们的加密货币赚取收益的系统。

Web3 赛道

Web3 赛道的第一名为 Solana 协议的桌面客户端 Project Citadel，它获得了 45,000 美元的奖金。

本赛道的其他获奖项目包括：

第二名：Bulldozer，让（基于 Anchor 开发架构）Solana 程序开发更轻松简单的 low code 平台。

第三名：Alon，基于 Solana 编译器套件 WebAssembly 端口的 IDE。

第四名：Chaincrunch，链上分析平台。

第五名：Dialect，支持应用-用户以及用户-用户通讯的链上消息传递协议和移动应用程序。

游戏赛道

Gaming 赛道的第一名为 Rushdown Revol，它是一个收藏品和元宇宙框架，奖金为 45,000 美元。

本赛道的其他获奖项目包括：

第二名：Portals，基于 Solana 的网页元宇宙平台。

第三名：Eizper Chain，一款融合冒险与竞技场战斗的多人动作角色扮演游戏。

第四名：StepN，一款 move-2-earn 的手机游戏，让数百万非加密货币的跑步人群加入加密货币。

第五名：Titan Analytics，为元宇宙构建的商业分析平台。

数字艺术和收藏品赛道

数字艺术和收藏品赛道的第一名为 Neftify，一个针对 play-to-earn 的 NFT 借贷平台，它获得了 45,000 美元的奖金。

本赛道的其他获奖项目包括：

第二名：SolanaFloor，NFT 数据分析平台。

第三名：Cofre，点对点 NFT 托管兑换平台。

第四名：Sonanalysis NFT Flex，NFT 展示和估值产品。

第五名：Banksea：基于流动池抵押的 NFT 的贷款产品。

社区奖

社区奖获得者是 DarleyGo，一款结合 NFT 的区块链赛马游戏，其奖金为 10,000 美元。
