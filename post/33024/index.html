占据NFT市场90%以上份额的OpenSea自传出有IPO计划之后，用户对这个起源于加密领域的NFT交易平台颇有不满，他们认为OpenSea平台利润来源于用户每笔NFT交易手续费的贡献，但该平台却未以DAO的形式发放Token给予用户回报，不符合Web3产品为用户赋能的理念。

此外，多次有用户在OpenSea账户出现丢失NFT资产的情况，该平台还出现过员工内幕交易的丑闻，种种不佳表现令NFT玩家渴望能有OpenSea的竞品出现，通过竞争的方式倒逼NFT交易市场成熟。

已经有一些NFT交易平台开始发力。具有NFT产品交易功能的平台LooksRare在1月10日宣布，空投其通证LOOKS给OpenSea上的交易用户，「苦OpenSea久矣」的用户立即跟进。

相比以同样方式还击「OpenSea不空投」的OpenDAO， LooksRare已经开发出相对完整的产品，且已投入使用，不少用户在上面完成过多笔交易。这无疑是竞品向OpenSea发出挑战的一步。

LooksRare宣布空投后的24小时内，数据网站Dune Analytics显示，1月12日LooksRare交易量超过2.75亿美元，领先于OpenSea的1.6亿美元。获得空投的目的带来了大量用户参与NFT交易的数据。

尽管当下LooksRare的交易量无法反映真实需求，但对用户来说，一个站在OpenSea对面的平台出现了。而事实上，在LooksRare之前，NFT交易平台赛道上一直有竞品存在，本期DeFi蜂窝将会盘点NFT交易市场的代表产品及运营情况。

NFT交易平台LooksRare

简介

LooksRare（LOOKS）是建立在以太坊上的NFT交易平台。2021年8月，LooksRare开始筹备，并声称要做一个以社区为中心的NFT交易平台，祭出口号「By NFT people, for NFT people」。

「成就于NFT用户，回报于NFT用户」的口号恰好击中了OpenSea未发代币而产生的市场情绪，LooksRare不但通过发行Token的方式吸引流量，还计划把平台的收入奖励给参与交易的用户、NFT收藏者和创作者。

1月10日，LooksRare 揭开面纱，宣布其平台通证LOOKS的空投计划，凡是在2021年6月16日至2021年12月6日期间，在OpenSea交易总金额在3 ETH以上的用户，就可领取LOOKS空投，领取数量的多少取决于用户在OpenSea的交易资金量。

LOOKS是LooksRare平台的原生Token，供应总量为10亿，主要用来投票决策交易奖励分配、质押收益、平台发展方向等提案方面的治理。

 LooksRare官网

平台特点

作为一个NFT交易平台，LooksRare已然具备基础交易功能，并表示自己不是一个简简单单的「分叉产品」或「仿品」，它从头构建了新产品，保障产品功能后续更新迭代的灵活性，希望能通过努力为NFT市场设定新的行业标准。

LooksRare推出了一些与OpenSea不同的功能：

  • 以Token回报社区——LooksRare会对平台的交易用户和NFT创作者进行奖励、赋权和回馈。

  • 即时版税——OpenSea每两周会向创作者分配一次版税收入，LooksRare则实行版税的实时支付，创作者的版税不需要等待数周才能收到，只要人们交易NFT，创作者就可以即时领取版税，此外，版税会以ETH的形式分发给创作者，而非任何ETH的封装资产，这样就为创作者省去兑换ETH的过程及GAS费。

  • 交易奖励——当用户在LooksRare平台上购买或出售指定系列里的NFT，用户可以赚取交易奖励，奖励以LOOKS发放。

  • 共享平台收益——在LooksRare平台上，用户每次交易NFT需支付2% 手续费给平台，对比OpenSea的2.5%少了一些；且这些手续费会完全分发给 LOOKS的质押者，用户只需要质押LOOKS就可以获得平台每日交易手续费的分红。

目前，LooksRare支持索引以太坊上的所有NFT产品，用户可以在该平台上购买NFT获取LOOKS奖励，也可通过质押LOOKS赚取平台每日手续费分成WETH及LOOKS奖励。

LooksRare无论是从产品功能还是商业模式上，都较OpenSea做出了实际的社区化改善，但最终能否在NFT交易市场胜出，还需要时间验证。毕竟，OpenSea经过时间的沉淀，有更丰富的NFT产品种类，用户形成了交易习惯。

作为一个新平台，LooksRare也想了一些办法丰富NFT商品，它将用户「自行上架NFT」作为领取空投的门槛之一，以此增加平台的商品种类，但由于用户上传的产品种类有限，无法一蹴而就地达到OpenSea的商品数量和质量水平，这也是目前限制LooksRare发展的一大问题。

运营数据

Dapprader数据显示，LooksRare近24小时交易用户为2859人，交易金额为5.8亿美元，已超越OpenSea当日的1.56亿美元交易额。

加密艺术策展平台SuperRare

简介

SuperRare（RARE）是一家老牌NFT综合交易平台，创建于2018年，同样是建立在以太坊上。相较OpenSea的「人人都可创建NFT」，SuperRare对NFT作品质量要求更高，它为创作者设置了一些门槛，比如，艺术家首先要向平台提出申请，SuperRare每周进行严厉审查，审查通过后才能上架。

如果说OpenSea走的是亲民路线，SuperRare则选择了高端精品路线，主要为在区块链上铸造的加密艺术品服务，在这里，用户可以观摩和购买世界顶尖艺术家的作品。

SuperRare官网

从服务对象来说，OpenSea服务的是NFT创作者和购买用户，SuperRare则由艺术家、收藏家和策展人组成，并希望能为真实、高质量的艺术策展建立一个流通场所。

SuperRare认为，收藏本质上是一种社交，收藏家和艺术家可以围绕他们共同的爱好轻松互动，该平台希望自己能提供一个供加密艺术爱好者交流和收藏作品的平台。

RARE是SuperRare的平台通证，供应总量为10亿，主要用来为社区财政部的资金分配、网络和协议的改进提案进行投票治理。

平台特点

SuperRare产品也在不断升级迭代，在最初的1.0版本中，核心团队亲自挑选艺术家，并将他们的作品铸造为NFT，提供交易市场后，将艺术家的作品出售给收藏家。

2021年8月18日，SuperRare2.0诞生，并推出SuperRare Spaces和SuperRareDAO等新功能。

  • SuperRareSpaces——称为社区画廊，它将允许独立策展人、艺术家及推广者在该板块推出他们自己的画廊页面，策展人、收藏家和社区成员通过这个画廊举办特定主题的展览并发布作品，同时帮助艺术家进行更多的实际推广和销售策略。

    画廊需经过社区成员审核，SuperRare的平台通证RARE持有者将进行投票，选出画廊的经营者，当选的经营者将挑选艺术家及其作品，按照他们认为合适的方式进行营销，经营者可以通过此画廊展览的NFT产品销售获得佣金。

  • SuperRareDAO——它是一个由RARE持有者和 SuperRare 治理委员会管理的去中心化自治组织，负责监督SuperRare Spaces，以及管理展览推出时间、社区财政和平台的未来发展规划。

    SuperRare治理委员会是一个由被提名的社区成员组成的小组，根据RARE持有者的投票来执行决策。

运营数据

Dappradar数据显示，SuperRare的24小时交易用户数为29人，NFT交易额为69万美元。

多链NFT交易平台Rarible

简介

Rarible（RARI）是一个运行在以太坊上的NFT交易和铸造平台，也是一个多链社区驱动型的开源平台，允许用户零门槛创作和展示自己的作品，并拥有NFT的所有权。

Rarible NFT交易平台专注于跨链兼容性，目前已集成到Flow和 Tezos公链上。RARI是该平台的原生通证，供应总量为2500万，主要用来奖励Rarible的交易用户和创作者，还可以用来投票决定平台的交易费用、产品升级及改进提案等。

Rarible官网

平台特点

Rarible在产品形态上与OpenSea有众多重合属性，它们都属于综合性的NFT交易平台。自OpenSea上线NFT创作功能之后，它与OpenSea之间的产品越来越相似。

但在2021年下半年，Rarible团队在产品上做出了多个改进和创新。

首先 Rarible 开发了APP版本，成为了首个有移动端的NFT交易应用，用户使用手机就可以随时随地管理或交易NFT资产，也可直接将一些作品铸造为NFT，比如，用手机拍照或上传手机上的图片来制作NFT，这对于摄影师来说无疑是好消息。

其次，Rarible还为NFT交易用户和创作者开发了交流工具Rarible Messenger，用户可以直接在Rarible上与收藏家、艺术家和创作者联系，这就为NFT交易平台扩展出了社交功能。

此外，Rarible还推出了开源协议Rarible Protocol，这里汇集了NFT市场上的发行和交易数据，为NFT应用的开发或创业人员提供了开发组件及模板，开发者可以使用这些工具任意构建产品，无需从零开始。

Rarible还形成了自己的社区DAO，为创作者提供多种获得资金和曝光的机会，并计划奖励这些创作者。

Rarible为开发者和创作者提供了更多的选择，不排除会成为NFT交易赛道上的一匹黑马。

运营数据

根据Dappradar数据显示， Rarible平台的24小时交易用户数为199人，NFT交易量为29万美元。

Fantom上的NFT平台 Artion

简介

Artion是构建在Fantom链上的NFT交易平台，于2021年9月推出。

作为一个新项目，该产品受到了广泛关注，很大原因来自它的发起人Andre Cronje，他也是DeFi协议Yearn（YFI）和Curve（CRV）创始人，同时也是Fantom网络的技术架构师。去年9月，Cronje在推特上多次公开提到Artion这个NFT交易产品，名人效应为该平台带来了曝光。

Artion推出有两个背景，一是当时以太坊链上GAS费居高不下，每笔交易换算为法币达100美元左右，而NFT交易往往需要支付巨额GAS费。二是当时OpenSea爆出员工内幕交易的丑闻，老鼠仓行为引发市场强烈不满。

这时，Andre Cronje在推特上发问：为什么没有产品与OpenSea较量？几天之后，他再次发声表示，已经有团队在构建新产品。没多久，Fantom基金会就推出了NFT交易平台Artion。

Artion官网

产品特点

Artion测试网在页面上与OpenSea相当类似，但和OpenSea不同的是，由于它搭建在Fantom上，用户创建及交易NFT的速度更快，且价格低廉，GAS费几乎可以忽略不计。此外，相比OpenSea收取2.5%的手续费，Artion不收取任何交易手续费。

使用Artion平台的成本低且易于操作，这一下就消除了创作者和收藏家进入NFT市场的主要障碍，对于用户来说无疑是利好。更重要的是，当时Fantom基金会提出「NFT桥接器」概念，未来，Artion将包括一个Ethereum-Fantom桥，允许用户将以太坊上的NFT无缝传输到Fantom链上，用户交易及转让NFT无需再承受巨额GAS费，这一计划得到了不少行业人士的支持。

可惜的是，自2021年9月Artion推出测试网后，就再无新动作，「Ethereum-Fantom桥计划」还能否实施成了一个未知数。目前，Artion网站依旧停留在测试阶段，官方也没有表示会有新动作。
