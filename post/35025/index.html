关于如何监管 NFT，各国都在积极地研究探讨，但并未得出定论，存在争议也较多。不过随着市场发展、人们对 NFT 认知的进一步深入、监管政策的进一步完善，NFT 的炒作将逐渐降温。当人们对 NFT 价值的期望逐渐回归正常，持有的风险逐渐可控，相信 NFT 的金融性会有所降低，其功能性也逐渐会体现在连接现实与元宇宙方面。

北京时间 2 月 12 日，由国际奥委会官方授权的冰墩墩数字盲盒于 Flow 链上交易市场nWayPlay 上线发售，总共 500 个，单价 99 美元。据中国基金报 2 月 14 日消息，冰墩墩 NFT 数字藏品两日内已经暴涨近千倍。涨幅最大的是北欧联合北京 2022(Nordic combined Beijing 2022) 吉祥物冰墩墩的别针系列，目前共有 13 个 NFT 在nWayPlay 在售，最低报价已达 2000 美元，最高报价 88888 美元。与最初的发行价 99 美元相比，暴涨了近 1000 倍。

据财联社 2 月 16 日消息，纽约证券交易所（NYSE）已向美国专利商标局（USPTO）提交申请，为各种数字商品提供在线市场，包括 NFT、加密货币、数字媒体和艺术品。韩国央行也在近期宣布，其正在央行数字货币（CBDC）第二阶段测试中研究如何使用 CBDC 支付 NFT 等多种数字资产的可能性。NFT 的持续火热和多国官方机构的布局让曾经一度对 NFT 持怀疑态度的人们开始逐渐转变看法。

NFT 是什么

NFT 的火热是从 2021 年开始，2021 年也被很多媒体称为 NFT 元年，据柯林斯统计，NFT 一词在 2021 年的使用率增长了 110000%，击败了包括“元宇宙”“混合办公”“疫苗”等在内的一系列词汇候选者，当选为《柯林斯词典》2021 年度词汇。在NFT“元年”，NFT 大放异彩，2021 年 3 月 11 日，数字艺术家 Beeple 的作品《Everydays ：The First5000 Days》在佳士得拍卖行从 100 美元起拍，最终以 6934 万美元售出，刷新了数字艺术拍卖品的最高成交价纪录，此次拍卖彻底引爆了 NFT 市场；3 月22 日，Twitter 前 CEO Jack Dorsey 在 2006 年的第一条推特消息发行的 NFT 副本以将近 300 万美元的价格出售；5 月，一组包含 9 个朋克头像的 NFT作品《CryptoPunks》在佳士得以 1696.25 万美元成交……

NFT 的英文全称是 Non-Fungible Tokens，中文通常翻译为“不可同质化代币 / 不可替代代币”。通俗来讲，NFT 就是用于表示数字资产（包括 jpg和视频剪辑形式）的唯一加密货币令牌，并且可以流通买卖。因为 NFT“不可替代”的特性，这意味着它所代表的产品是独一无二的——或许会有很多复制品，但拥有 NFT 的仅有一件。

NFT 的概念是由加密猫Crypto Kitties 的创始人兼 CTO Dieter Shirley 在 2017 年正式提出的。当时，随着比特币在短短的 2 个月内从 5000 美金上涨到 20000 万美金，加密猫游戏迅速走红。在加密猫推出后，很多用户问到 Dieter 加密猫和 ERC20（市面上绝大部分加密代币的通证标准）如何交互，但是加密猫和 ERC20 是无法实现兼容的，因为加密猫是基于另一个通证标准 ERC721，是非同质化通证， ERC20 是同质化通证。为了更好的解释加密猫与 ERC20 的区别，Dieter 想出了一个固定而简单的回答，随后 Dieter 就提出了 NFT 的概念，即非同质化通证的概念。

NFT（非同质化通证）最鲜明的两个特点就是唯一性和不可拆分性。每一枚非同质化通证各不相同，都是独一无二，无法相互替代的通证。此外，NFT不可拆分，不能像加密代币那样拆分成许多份。NFT 将稀缺性引入到加密代币领域——不同于加密代币每一枚价值相等，每一个 NFT 的价值都是不同的，相互之间不可替代。从此，数字资产不再是加密代币一家独大，而是由同质化的加密代币与非同质化的 NFT 代表的加密资产组合而成。

也正是由于 NFT 的不可拆分且唯一的两大特性，NFT 通常会链接到某些特定数字物品，比如游戏皮肤、数字收藏品等代表的无形资产所有权。NFT 的核心价值同质化的商品是可以等价交换的，其价值来自于市场供求关系。

为何是 NFT 连接元宇宙和现实

2021年伴随NFT迅速崛起的还有元宇宙概念，元宇宙概念的迅速火爆全球，也给 NFT 带来了新的活力。有声音表示，NFT 是元宇宙的基础，没有它就没有元宇宙，也有相关人士表示 NFT 是打开元宇宙大门的钥匙。那么 NFT 和元宇宙存在什么样的关系呢？

要了解 NFT 和元宇宙的关系首先要对元宇宙有一个清晰的认知，清华大学新媒体研究中心执行主任沈阳表示：“元宇宙是整合多种新技术而产生的新型虚实相融的互联网应用和社会形态，它基于扩展现实技术提供沉浸式体验，以及数字孪生技术生成现实世界的镜像，通过区块链技术搭建经济体系，将虚拟世界与现实世界在经济系统、社交系统、身份系统上密切融合，并且允许每个用户进行内容生产和编辑。”简单来说，元宇宙就是对现实世界的虚拟化、数字化，在这一过程中根据需要对内容生产、经济系统、用户体验以及实体世界内容等进行大量升级改造而形成的虚拟世界。

无论在虚拟世界还是现实世界，一个完整的经济系统是使其运转的核心要素，而 NFT 的唯一性和不可替代性使其成为元宇宙中数字身份、数字资产和经济系统中不二选择。用户可以通过现实世界的货币购买带有 NFT 标记的虚拟世界产品，从而保障其在虚拟世界中的产权——由于虚拟世界的可复制性使得一般手段难以保障数字产品的产权。

元宇宙的建设离不开现实中广大用户主动生产内容，NFT 的可交易性提的高了用户生产内容的积极性，进而促进了元宇宙的发展。

NFT 的唯一性、不可拆分性和可交易性等特性，实现了资产在现实世界和虚拟世界的流通，为元宇宙经济系统的搭建起到了不可替代的作用。

海外热炒上天，国内谨慎发展

随着元宇宙的不断发展，NFT 的作用还会逐渐显现，但就目前来看，无论是元宇宙还是 NFT 都处于发展初期，过度火热的状态不得不让人感受到其中似乎充斥着炒作和投机。

国外 NFT 艺术品屡屡被拍出天价，也带动了国内艺术家、投资机构和互联网平台的积极性，纷纷开始探索布局。蚂蚁集团是国内第一家进军 NFT 的互联网平台，2021 年 6 月，蚂蚁集团在支付宝上推出小程序“蚂蚁链粉丝粒”。一经发布，就遭到市场追捧。随后，腾讯、字节跳动、小红书、百度等多家互联网平台陆续进军 NFT，一个新的行业生态正在悄然形成。

网 易 旗 下 游 戏《永 劫 无 间》IP 授 权 发 行 了《NARAKAHERO》系列 NFT 盲盒，上线仅 15 分钟便被抢购一空；腾讯旗下的 PCG 推出了号称“国内首个 NFT 交易平台”——“幻核”App ；小红书推出了 NFT 数字收藏品；百度旗下的百信银行也推出了 NFT 数字藏品；而蚂蚁集团，虽然对外宣传时有些轻描淡写，但是实则对 NFT 业务抱以厚望。

但互联网平台对于 NFT 的市场监管依然抱有几分戒备，在发布 NFT 之后，首先做的事就是更名。2021 年 10 月，支付宝小程序“蚂蚁粉丝粒”及腾讯旗下 NFT 发行平台“幻核”App 内页中，NFT 字样全部消失，改为“数字藏品”。在更名的同时，各平台内部对于 NFT 字样进行删减后，关于数字藏品的定义和介绍也全部进行了更新，特别强调与虚拟货币的不同，只保留了区块链技术对其稀缺性的赋予。包括后来进场的京东、B 站、网易、小红书在内的互联网平台，都强调自身是数字藏品，很少再见到NFT 字样，在介绍相关产品时也纷纷强调其与虚拟货币的不同。

除了 NFT 字样上不再显示，各平台对于二级交易也进行了严格管控。我国对于虚拟货币的监管一直比较注重，2021 年多部门对虚拟货币挖矿与交易进行了清退和禁止，而 NFT 也具有明显的代币属性，各平台对于 NFT 的谨慎也就理所当然了。2021 年 6 月支付宝上线 NFT 时，其炒作热度就引起了开发团队的担忧，随后闲鱼迅速下架 NFT相关内容。

由于 NFT 炒作的关键就在交易上，所以不开放交易成为合规的前提，而国内的联盟链也与公链不一样，并不可以自由交互。所以不放开二级交易，成为国内互联网平台试水数字藏品的底线。

从数字藏品上线起，蚂蚁集团就多次强调拒绝虚拟货币炒作，不支持用户进行交易。蚂蚁集团旗下的数字藏品仅开放转赠功能，且需要持有 180 天以上，购买者必须通过实名认证，绑定身份信息。

包括腾讯、京东在内的所有互联网平台都选择类似的策略：只开放购买或领取功能，且必须完成实名认证，并持有一定时间后才陆续开放转赠功能。各平台施行如此严格的措施，就是为了杜绝相关炒作的风险。

警惕狂热背后的风险

国内互联网平台对于 NFT 的风险控制侧面肯定了国内对于新事物发展的监管，从原来的先疯狂上马再接受监管，到现在的把风险放在首位，是互联网平台对自身行为负责、对消费者负责的表现。毫无疑问，NFT 现阶段存在泡沫，而热炒上天的 NFT泡沫破裂之时，最受伤的无疑还是“散户”。目前在 NFT 的生产、交易、监管等阶段仍有不少问题尚待解决。

首先，NFT 产品确权问题仍存漏洞。虽然区块链技术可以对已经上链的数字内容溯源，保证其真实性和所有权归属。但由于在被铸成 NFT 之前的资产确权环节存在漏洞，并不能排除其他人再次复制、裁剪、二次编辑的可能。《Everydays: TheFirst 5000 Days》天价成交后，此内容又被作者分割成 105 个小块，分别注册 NFT，再一次进行拍卖。这说明 NFT 并没有真正实现数字内容的资产化，用户只有使用权，而并非完整的所有权。这一重大漏洞将导致 NFT 对于区块链游戏和元宇宙的经济体系构建成为纸上谈兵。

其次，在 NFT 产品生产、交易过程中会造成一定量的能源消耗。在比特币迅速升值的时候， “比特币挖矿”曾经产生大量的电力消耗，再后来也一度被多国禁止、叫停。NFT 虽然是数字资产，存在于虚拟中，但其制作和交易都会消耗现实中的大量能源。奥地利建筑师 Chris Precht，制售了过百件 NFT 艺术品，他表示 100 件 NFT 艺术品耗费的电量，足以供一名普通欧盟居民使用 20 年。

最后，防范 NFT 产品交易成为新的洗钱渠道。由于 NFT 使用的是区块链技术，其去中心化的属性会给洗钱机构带来便利。美国财政部 2 月初发布的《通过艺术品交易开展洗钱和恐怖融资的研究》报告中，将 NFT 归类于新兴在线艺术品市场，并指出该市场可能会带来新的风险。报告将 NFT 定义为底层区块链上的数字单位或代币，代表了图像、视频、音频文件和其他媒介形式的所有权，或物理资产、数字财产的所有权。报告指出，NFT 可用于进行自我洗钱。犯罪分子可以用非法资金购买 NFT 并与自己进行交易，以在区块链上创建销售记录；然后，NFT 将被出售给一名不知情的个人，罪犯将从后者与犯罪无关的干净资金中获利。

针对这些问题，如何监管 NFT，各国都在积极地研究探讨，但并未得出定论，存在争议也较多。不过随着市场发展、人们对 NFT 认知的进一步深入、监管政策的进一步完善，NFT 的炒作将逐渐降温。当人们对 NFT 价值的期望逐渐回归正常，持有的风险逐渐可控，相信 NFT 的金融属性会有所降低，其功能性也逐渐会体现在连接现实与元宇宙方面。
