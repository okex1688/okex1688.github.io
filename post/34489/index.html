目前都有哪些Web3声誉身份系统？

原标题：《Web3 Reputation Market Map》

在 Web3 时代，去中心化身份越来越被广泛采用。一个人的部分在线活动在链上是公开的，可通过加密钱包搜索到，用户在链上创造、贡献、赚取和拥有的东西，都反映了他们的喜好，也逐渐积累成该用户的财产和声誉。

因而去中心化的声誉和身份系统作为解锁加密市场下一个全新用例的重要类别，这篇文章将试图对目前存在的各种数字声誉项目进行分类，并最终确定它们是如何在定义我们数字身份方面发挥关键作用的。

社会声誉系统

这是当前存在的主要类别，可以上传用户生成的内容，用户在该特定生态系统中可以创建身份和彼此交流。随着时间的流逝，社会声誉系统与用户会一起构建出一个相关联的社会图谱，这里有两个符合要求的经典例子：

Twitter

用户可以上传包含文本、图片、视频等的推文，其他用户可以互相关注，以及点赞、转发和评论，以创建感知价值的社交图谱。

Github

GitHub 是一个通过 Git 进行版本控制的软件源代码托管服务平台，开发人员可以在公开的存储库中编写代码。而作为拥有许多 Github 明星的存储库中的主要贡献者，他们将在开发人员圈子中创造一定的地位和声誉。如主要的核心比特币/以太坊开发人员，在加密圈据可以获得很多声誉。

不过，我敢肯定还有许多其他的社会声誉媒介，但它们在 Web3 中将不占有主导地位。

贡献声誉系统

与社会声誉不同，贡献声誉系统将更为客观，这取决于用户完成的实际工作，而不是其他人对该用户工作的看法。同时，贡献声誉系统往往还与具有去中心化、透明等特性的 DAO 相结合，展示出了很强的竞争性。

Layer3

Layer3 是一种协调、追踪和奖励贡献者的机制，提供了赏金市场，补充 DAO 工具包中缺失的部分。用户可以通过发布推特内容或重新设计前端等贡献，赚取治理 Token。

通过这种方式，任何人都可以为 DAO 做出贡献，这些贡献在链下和链上都能被跟踪。最终，随着时间的推移，最好的用户拥有更多可以追踪的数据点。

Gitcoin

Gitcoin 是一个开源软件开发激励平台，旨在通过区块链支持各类开源项目，促进开源运动的发展。在 Gitcoin 上，一个项目可以通过创建赏金或任务供其他人完成，而用户则可以赚取 Token 来完成上述任务。每个用户对任务的贡献将在链上可被追踪，并帮助其建立因完成具有明确目标任务的声誉。

虽然这可能与 Github 有相似之处，但不同之处在于这个特定的声誉向量是高度定义的并且范围很窄，即：0xabc 为项目 X 完成了 Y 数量 Token 的写作赏金。

Coordinape

Coordinape 是 Yearn 为 DAO 推出的去中心化薪资管理系统，允许去中心化团队和 DAO 自动为贡献者发放奖励或赠款，而无需自上而下的管理或雇用 HR。

Coordinape 是一个强大的工具，可以在 DAO 的本地环境中追踪贡献声誉。DAO 中的许多贡献者可以评估彼此的工作，以判断谁贡献最多和创造了最大的价值，而在评估其他成员的工作时，往往评价的是工作质量和工作量而不是社会影响力。

身份提供商

BrightID

当人们想到身份认证时，他们通常指的是在链上桥接链下身份的提供商。但我个人认为这不是你如何在去中心化系统中扩展信任的未来，尽管我同时承认它们有自己的用例可以满足某些需求。

在我看来，使用身份提供者的最大原因是提供硬女巫保护，代价是在隐私权衡中引入一些用户摩擦。这个领域的主要参与者可能是 BrightID，它要求用户通过一些新颖的机制证明其是人类。它不像 KYC 那样严格，但还是有些麻烦。

BrightID 是 Web3 时代去中心化社交身份认证系统，主要基于以太坊开发，另外结合了链下会议视频等非隐私信息渠道的认证方式。通过 BrightID，个人无需提交身份资料等敏感信息，而是通过熟人之间信任机制来进行链上账户的认证。

凭证

由于加密货币中可用的开放和公共数据集，最新出现的声誉系统类别可能是凭证类应用。许多凭证提供者和应用程序会检查用户是否在某个时间点执行了某些行为，并发布应用内徽章或链上 NFT 来推荐此凭证。

RabbitHole

RabbitHole 为用户创建任务，用户在完成任务后可获得免费 Token，同时用户的 Web3 操作和行为还将获得「认证」。这里很酷的是，用户可以看到 RabbitHole 和 BrightID 之间的可组合性。我确实认为我们会看到声誉和身份的可组合性将成为数字声誉的一个重要议题。

Project Galaxy

Project Galaxy 是一个 NFT 基础设施服务商和 Web3 认证数据网络，并旨在将他们的 GalaxyID 产品打造成一个身份提供者。相较于 RabbitHole，Project Galaxy 通常与项目合作并举行为用户创建凭据的竞赛，不过更多的则是围绕「迷你挑战」和「迷你忠诚度」活动，这些活动将引导用户可以收集 NFT。

Sismo

Sismo 通过使用零知识证明向用户的其他帐户（如 ENS 以及 twitter 或 github）颁发徽章（NFT）来建立其以太坊个人资料，并且用户可以在保密的情况下将其声誉汇总到公开资料中。这种方法很新颖，因为它展示了零知识证明如何在下一个加密领域发挥作用。

POAP

POAP 也称为出勤证明协议，NFT 会发给那些参加虚拟或实体活动的人，甚至可以证明他们确实参加了。然后用户可以在其钱包中利用这些 NFT 凭证来创建显示自己参与活动历史的徽章集合。

数据解决方案

这可能是我们将在未来几年开始看到更多起飞的更新颖和令人兴奋的领域之一。自我主权数据的概念并以分散的方式存储它是我们仍处于早期阶段的东西。有两个著名的竞争者在这个领域深入研究，我很想提请注意：

Ceramic

Ceramic 是用于构建具有可组合数据的 Web3 应用程序的分散数据网络。该网络由三个核心部分组成；可扩展的、分散的数据基础设施；用于存储、修改和检索数据的开放 api；社区创建的可重用数据模型市场。（作者是 Ceramic 开发商 3Box Lab 的投资者。）

这整个想法有点像 Amazon S3，但数据库可在无许可数据网络上进行验证，并且数据库操作使用私钥/公钥对用户进行身份验证。与 IPFS 不同，用户可以更新数据，但对数据的更改、更新、存储方式也有更多的颗粒度。

Spruce

Spruce 是一个数字身份认证系统，致力于保护用户的隐私，并且正在重新构想企业与政府间可信赖的交互方式。并且，Spruce 还正在创建开源软件产品，以帮助其客户从不受信任的数据转变为可以私下共享的可验证信息。

积分

ARCx

ARCx 通过预定义算法持续评估用户的行为，用户的分数是动态的、数据丰富的和可验证的。与我们过去看到的其他数字声誉向量不同，ARCx 可以从链上和链下来源中提取数据，并对其进行量化。考虑到在链上使用 ARCx 的潜力，他们将可以为以最适合协议的方式来创建有意义的差异化用户体验。

总结

如您所见，加密货币中新兴的声誉和身份证明非常令人兴奋，并且在接下来的几个月和几年内会有许多有趣的应用进入市场。更巧妙的是，这些声誉向量中的许多都可以相互组合，同时可以作为 API 进行混合，而这在 Web2 中还没有真正实现。
