Layer 2 能给 NFT 带来什么？这个问题似乎成为了最受加密社区关注的一个问题，预计在四季度上线的ZKBox，或许能给我们一个满意的答案。

不管你承不承认，2021年，加密行业里的最大风口就是 NFT。

从 Elon Musk 的自创舞曲，到姚明的限量版葡萄酒，再到支付巨头 Visa 大手笔购入 CryptoPunk 7610，一时间 NFT 成为了各路人马热议和关注的新叙事，“万物皆可 NFT”更是成为一句经典。

目前，绝大多数 NFT 智能合约仍基于以太坊构建，但随着垂直市场愈发火爆，相关交易活动也逐渐增多，以太坊网络诸多限制也成为制约 NFT 进一步发展的桎梏，比如区块大小限制、高 Gas 费和交易费、缓慢的确认速度、单点失败问题等等，除交易受限外，还有不少其他因素也在影响 NFT 发展，比如铸造 NFT 成本过高，NFT 公链之间互通困难等。

但是，以太坊依然是当前生态最丰富、机制最完善的底层区块链，可以说，NFT 项目短时间内仍无法离开以太坊——在这种情况下，Layer 2 的优势就开始发挥作用，毫不夸张地说，通过扩容和提速，Layer 2 能够有效解决 NFT 发展中遇到的各种问题。

Layer 2 为 NFT 赛道赋予更大发展空间

如果说以太坊是 NFT 最底层的基础设施，那么在这个“地基”上搭建的 Layer 2 平台将成为 NFT 的最佳交易场地，不仅可以让各类 NFT 项目根据自身特色来铸造、创作，还能提升 NFT 流动性，为各个项目引流，吸引用户、投机者和收藏家进入市场。

在基于以太坊 Layer 2 的 NFT 平台“第一梯队”中，最值得关注的无疑是 ZKBox 和 Immunetable X，后者是第一个基于 Layer 2 的 NFT 协议，允许任何人构建自己的去中心化市场、区块链游戏和其他 dApp。而 ZKbox 则是由 L2 Labs 打造的一款基于以太坊 Layer 2 的 NFT 平台，可以在 Layer 2 上实现 NFT 的发行、转账、买卖功能，帮助用户免除高昂的手续费，并且快速实时的进行交易。

ZKbox 平台使用零知识证明等技术，通过大量计算来保证 Layer1 和 Layer2 状态的一致性，可以支持以太坊主链的 NFT 充值到 Layer2，以及 Layer2 的 NFT 提现至以太坊主链，大大节省用户的使用和交易成本。

Immutable X 和 ZKBox 对比图

通过对⽐不难发现，Immutable X 和 ZKBox 都是基于L2的 NFT 协议。Immutable X ⽬前主要着重于协议 API设计，⽬前只有官⽅能够发⾏ NFT 并进⾏交易的功能，用户想要自主发行 NFT 需要联系官⽅，提交资料后方可发⾏，这对于个⼈⽤户不太友好。

更严重的是，Immutable X 牺牲了数据可⻅性，降低了 L2 与 L1 交互安全性，虽然官⽅声称后续将会组建所谓的“数据可⻅性委员会”，但这种⽅案同样需要依赖于委员会自身诚实性，跟 Validium 其实差异不⼤。

ZKBox 则注重⾃身协议以及应⽤的构建，有 Web、App、API 等多种⽅式构建，⽤户便利性较⾼，使⽤⻔槛低。未来，ZKBox 平台将推出各类插件和移动端 App，ZKBox 作为基础工具方便其他平台的用户一键发布 NFT，如 Twitter 、Reddit、电报等插件。这种便利的插件类似微信小程序，一旦被开发得当，将很将被市场熟知，席卷币圈。

  • 用户友好性方面，ZKBox 优势更为突出

ZKBox 会将应用程序接口向所有项目开源，并支持平台一键购买 NFT ，首版产品将支持 NFT 商场（用户可在商城中通过币种、时间、价格、类别自由筛选和搜索喜爱的 NFT 产品）、NFT 钱包（ZKBox 会打通 ZKSwap 入口，并提供 ZKSwap L2 钱包内容；支持从 L1 账户充值 NFT 或接收其他L2 账户的 NFT 充值、转账、提现）、以及 NFT（支持单个和批量 NFT 创建）三大功能，后续迭代版本还将拓展拍卖、盲盒发售等功能。

  • 在包容性方面，ZKBox 更具优势

为了将更大的玩家群带入生态系统，ZKBox 支持现有的 L1 NFT 存入 L2，使 L1 的原生 NFT 可以在 L2 上直接交易，这将帮助玩家节省 Gas 成本并提高交易速度。此外，通过将所有 NFT 保存在同一个 L1 智能合约中，ZKBox 有效消除了 NFT 创建者在 L1 中部署智能合约的需要。

你会发现，ZKBox 通过分发和创建高质量的 NFT 来创建一个功能强大的 Layer 2「NFT 世界」 ，其基础层功能完全可以满足艺术家、名人、品牌、网红和知识产权在其平台上创建、交易和转移 NFT 的全部能力。虽然 ZKBox 和 Immunetable X 都是基于 Layer 2 的 NFT 协议，也都基于 ZK-Rollups 技术方案实现与以太坊 Layer 1 互通互联，但从具体实现过程和相关数据可以看出，ZKBox 明显更胜一筹。

值得一提的是，ZKBox 之所以有如此亮眼的优势，其实还有另一个因素——

　02 　

ZKBox 背后的支持者大有来头

ZKBox 是由 L2 Labs Foundation 打造的一款基于以太坊 Layer2 的通用 NFT 协议。L2 Labs 专注于Layer 2 创新项目，旗下品牌还包括 ZKSwap 和其他支付协议及侧链。

现阶段，L2 Labs Foundation 正在不断构建和完善 Layer 2生态系统，利用「ZK 生态」中的 ZKSwap、L2 Cash等其他项目优势，加速整个生态发展。 ZKSwap是一个 Layer 2 去中心化交易所，其历史最高锁仓量达到 16 亿美元，这些生态项目都能对 ZKBox 提供有力支撑——毋庸置疑，一个完整的 Layer NFT 世界即将孕育而生。

ZKBox 的研发团队具有丰富的 DEX、智能合约和 NFT 社交网络平台开发经验，主要包括区块链行业的早期布道者、行业资深开发者等。团队希望通过发行和创造高质量的 NFT，提高平台品质，线上联动线下，扩大 NFT 的生态版图，打造一个通用友好型的 Layer2 NFT Metaverse。目前团队正在紧锣密鼓地开发 ZKBox，预计将在今年四季度上线。  

Layer 2 将为 NFT 平台带来更多价值

虽然以太坊仍是大多数 NFT 项目的首选公链，但越来越多优质 NFT 项目也开始探索其他公链，比如 NBA Top Shot 便是基于 Flow 公链，Rarible 也是基于自身的公链搭建的，这便会造成一个问题，不在以太坊上开发的项目用户交易活动会被限制在自身的公链上进行，就像一个个生态孤岛，难以捕获到以太坊上巨量的存量用户。

如果现有的 NFT 公链能连接到以太坊上，并结合 Layer 2解决方案来减少传输成本，既可以达到增加双方的相互价值，又可以达到“用户互用”的目的，实现双赢。 举个例子，如果有其他公链上的 NFT 项目接入到 ZKBox，那么就有可能实现 NFT 自身功能与以太坊的完美对接，这可能也是未来 Layer 2 NFT 平台的发展趋势之一，不仅可以提高NFT项目的可扩展性，而且可以将以太坊系统里的大量用户带到优秀的 NFT 项目中，甚至有可能让更多优秀的NFT项目快速出圈。

你会发现，通过 ZKBox，未来可以将“孤岛”上的优秀产品源源不断输送到以太坊上去，那么将为 NFT 市场带来源源不断的增量用户。

总结

NFT 市值目前大约在 300 亿美元左右，相比于整个加密市场规模而言依然相对较小，因此发展潜力巨大。随着 ZKBox 这些基于 Layer 2 的 NFT 平台逐渐成熟，势必会为这个正处在“发育阶段”的新兴领域带来更多用户和更大发展。

从这个角度来看，Layer 2 的意义非比寻常，作为以太坊生态的关键组成部分，每一次市场爆发都需要Layer2 的助力——2019 年和 2020 年是 DeFi，而这一次，无疑是 NFT。
