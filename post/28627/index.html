Gala是一个游戏和NFT平台，旨在建立一个致力于区块链游戏的生态系统。GALA是平台的功能型代币，可用于奖励网络节点运行者，参与网络治理，通过玩各种平台游戏获得GALA代币，在Gala商店购买游戏内物品和NFT。

Gala项目介绍

Gala Games 希望将游戏行业带入一个与现在完全不同的方向。Gala Games 指出，要制作用户真正想玩的区块链游戏。讲究乐趣第一，玩家可以真正拥有他们在我们的游戏中赢得的东西。如果您在 Gala 游戏平台上赢得或赢得了一把魔剑，那就是您的了。您的物品和任何继承的状态都是区块链上的可验证资产，您可以与其他人交易或在游戏中玩。

Gala Games Network 内的一切都取决于玩家拥有的节点生态系统。Gala 网络受到像您一样的用户的支持，他们从他们的家用计算机上操作 Gala 节点。目前， 超过 8,000 个玩家运行的节点为我们的生态系统提供动力。Gala网络由用户支持，可在家里的电脑上操作Gala节点。用户通过购买Gala Node的运营许可，根据各人对平台网络的贡献获得奖励。其奖励包括：GALA代币、限量版NFT以及帮助Gala游戏生态系统发展的机会。

中心化的 Gala Games 生态系统渴望敞开大门，让玩家重新获得应有的控制权。通过权力下放，玩家不仅拥有他们的游戏内物品，而且他们对 Gala Games 路线图的发展有发言权。玩家和节点所有者将通过分布式投票机制参与进来，以帮助确定应将哪些游戏添加到平台或应由去中心化的 Gala Games 生态系统资助哪些游戏。

总的来说，随着互联网变得更加主流并且实体媒体逐渐被淘汰，游戏玩家和消费者一直在失去对他们内容的控制。许多游戏已经开发出模型，玩家可以使用需要现实世界金钱的游戏货币购买皮肤和其他物品。这为玩家增加了一种新的体验，因为他们能够自定义他们的头像并获得稀有物品。这些玩家通常会在这些游戏物品上花费数百美元，而且很容易买到，也很容易被带走。由于游戏管理员的禁令，玩家可能无法访问他们支付的所有内容。花在获得这些物品上的所有时间和金钱都完全消失了。游戏公司保留了所有的钱，而玩家现在一无所有。玩家可以购买物品，但他们并没有像他们应该的那样拥有自己的物品。玩家购买使用物品的特权，但永远无法真正拥有它。这就是 Gala Games 正在改变的东西。

资产和游戏账号的严密管控扼杀了很多创造性思维。Gala Games 希望把大门敞开，让玩家重新获得他们应得的控制权。这发生在 Gala Games 的《城市之星》中。玩家可以拥有游戏内物品，例如 CraneBot。该物品可以帮助加快游戏中的建造过程，并且玩家可以随时交易、出售或将物品赠送给其他用户。这就是不可替代代币或 NFT 的力量，它们之所以能够存在，是因为区块链。
