L2 淘金热进行时 一文速览 L2 的价值捕获策略

虽然以太坊 L2 提供了诸多好处，比如 Gas 费用的价格低、几乎即时的交易确认以及继承了以太坊 L1 的安全性，但一直没有一个很好的催化剂来推动 L2 的采用，直到 Arbitrum 上首个主要的收益农场 ArbiNYAN  的推出，推动着 Arbitrum 这个 L2 网络的 TVL (锁仓总价值) 在不到 5 天的时间里从 2.38 亿美元上升到了超过 25 亿美元 (见下图) ！

上图：Arbitrum L2 网络中的 TVL 增长情况。图源: L2 Beat

尽管仍处于早期阶段，但这已经充分证明了人们对以太坊 L2 的巨大需求 。当然，这也提出了一个问题：投资者如何才能获得这种趋势的敞口？让我们来探索其中几种方法。

本文任何内容均不构成投资或财务建议。在做出任何决策之前，请先自行研究。

机会 1: 基础设施

投资者可以不必直接押注于单个应用或 L2 网络的成功，相反，投资者可以在投资组合中配置 L2 的基础设施层，选择那些为 L2 提供动力的工具来获得敞口。

这种策略可能适合于这类投资者，即他们想要避免在 L2 生态系统中选择谁会成为赢家，但仍希望获得那些将从 L2 增长中受益的资产的敞口。此外，由于这些基础设施拥有的网络效应，加上其他竞争性服务面临的巨大进入障碍，许多基础设施协议处于垄断地位，其面临的竞争有限 。

让我们来看看已经集成到 L2 中的一些基础设施。

  • Chainlink (LINK)

DeFi 协议需要安全的、信任最小化的预言机喂价 ，以实现合约定价和及时清算等活动。例如，几乎所有部署在 Optimistic L2 网络上的主要 DeFi 协议，包括 Synthetix、Aave、Curve、SushiSwap、Dopex 和其他协议，都已经将 Chainlink (LINK) 预言机集成到它们的喂价中，以实现及时清算等活动。

除了总体上提供一个广泛的 DeFi 敞口，Chainlink 还可能在未来更深入地集成到 Arbitrum。

来源：https://twitter.com/ChainLinkGod/status/1397981952846622720?s=20

  • The Graph (GRT)

协议不仅需要访问_链下数据_，也需要访问_链上数据_。

去中心化数据索引协议 The Graph 是基础设施堆栈中的另一个关键组件，它提供了链上数据的索引 ，允许应用程序以一种去中心化和无须信任的方式来查询链上数据。该协议支持多种不同的网络，包括 Arbitrum 和 Optimism 等以太坊 L2 网络，并被一些已在 L2 部署的 Dapps 应用使用，比如 Uniswap、Synthetix 和 Futureswap。

类似于 Chainlink，The Graph 可以提供广泛的 L2 网络和优秀 Dapps 的敞口，而无需押注于特定的 L2 网络或应用。

  • Hop Protocol & Connext (追溯性耕作)

虽然与 L1 相比，L2s 的快速交易确认及其 Gas 费用的降低显著改善了用户体验，但 L2s 仍然面临一系列独特的挑战 ：其中值得注意的是，从 Optimistic Rollups 中取款至主网需要等待 7 天的时间；在不同的 L2s 之间迁移流动性也很困难。

Hop Protocol 和 Connext 这两个项目旨在解决这一问题。这两个项目允许用户在兼容于 EVM (以太坊虚拟机) 的不同网络之间无缝地转移资产，并允许用户“快速”取款 ，让用户能够避免取款等待期。

虽然这两个项目尚未发行代币，但这些系统值得试验，因为它们有可能 为早期用户提供追溯性空投！

机会2: Dapps 应用

投资者获得 L2 敞口的第二种方式是通过投资热门的应用 。任何区块链或 L2s 只会对构建在其上的应用程序提供尽可能好的体验——如果某个网络上没有什么值得使用的应用，那就没有人会使用该网络。

正因为如此，在一个或多个 L2s 上运行的应用可以为投资者提供不同程度的风险敞口 。重要的是，L2s 代表了新兴市场，既有的 Dapps 协议可以在 L2s 上增长其用户基数，从而增加其使用量和收入。

主要有两类应用可以为投资者提供 L2 敞口：从 L1 迁移至 L2 的“早期采用者” 和**“L2原生应用”** ，后者是指仅在 L2 部署的应用，或者在 L2 部署之后在功能上有了实质性增加的应用。

比较而言，L2 的“早期采用者” 可能会被证明“更安全”，因为这些应用并不完全依赖于底层的 L2 的成功，而**“L2原生应用”** 可能代表了对某个特定 L2 网络更加集中的押注。

虽然下方列出的应用不太全面 (还有很多的机会未被包含进来)，但我们还是来看看下面这些例子吧。

  • 早期采用者 #1: Uniswap (UNI)

Uniswap 是以太坊 L1 上规模最大的 DEX (去中心化交易所)，同时也是 Optimism L2 网络的最早采用者这一。Uniswap 于 2021 年 7 月部署到了 Optimism 网络上，撰文时该协议在 Optimism 网络上有着超过 3,200 万美元的流动性 ，并在最近几天实现了超过 1,000 万美元的日交易量 。见下图：

上图：截至撰文时 Uniswap 在 Optimism 网络上的 TVL 和日交易量情况。来源：https://info.uniswap.org/#/optimism/

除了 Optimism，Uniswap 最近也在 Arbitrum 网络上部署。该部署使 Uniswap 获得了更大的使用量，在过去 4 天里该协议在 Arbitrum 网络上共计锁定了 3,700 万美元的价值，每天产生了超 2,000 万美元的交易量 。

  • 早期采用者 #2: SushiSwap (SUSHI)

SushiSwap 是另一个重要的去中心化交易所，也是 L2 的早期采用者。SushiSwap 已经在 Polygon L2 网络占据了相当大的市场份额，同时也部署在其他区块链上，如 Fantom、Avalanche 和 Harmony。此外，SushiSwap 也是最早在 Arbitrum 上部署的协议之一。

自 9 月 10 日以来，SushiSwap 协议在 Arbitrum 上的部署取得了巨大成功，撰文时其流动性超过 2700 万美元，每日交易量超过 2400 万美元 。此外，Sushiswap 的表现有时甚至超过了它的主要竞争对手 Uniswap，某些日子的交易量超过了 1.25 亿美元。见下图：

上图：截至撰文时，Sushiswap 在 Arbitrum 网络上的 TVL 和日交易量情况。来源：https://analytics-arbitrum.sushi.com/

虽然 SushiSwap 团队表示，考虑到他们的多链战略，他们可能会推迟部署到 Optimism 网络上 ，但似乎他们很有可能最终会部署到该网络。

  • 早期采用者 #3: Curve (CRV)

Curve 是另一个迅速拥抱 L2 的去中心化交易所。Curve 已经几乎垄断了 Polygon 和 Fantom 网络上的同类资产兑换，最近该协议也在 Arbitrum 网络上线 。目前，Curve 在 Arbitrum 上提供了 2 个同类资产的流动性池以及 1 个 V2 流动性池 (见下图)，撰文时这些池中共计持有超过 1.31 亿美元的流动性，并实现了超过 590 万美元的日交易量。同时 Curve 也是一个热门的收益耕作场所，以最大程度降低流动性提供者的无常损失风险。

上图：Curve 在 Arbitrum 网络上提供的 3 个流动性池。来源：https://arbitrum.curve.fi/

与 SushiSwap 一样，虽然目前还不清楚 Curve 协议是否计划在 Optimism 网络上部署，但考虑到该协议的多链倾向，该协议似乎也有可能最终会在 Optimism 上部署。

  • L2 原生应用 #1: Synthetix (SNX)

由于 Synthetix 合成资产的交易有着计算密集型的性质，因此受到 L1 高昂的 Gas 费用的影响，导致 Synthetix 协议在以太坊 L1 的增长受限。为此，Synthetix 协议开始致力于在 L2 上解锁其全部能力 。

Synthetix 这个合成资产协议是 Optimism 的早期支持者和采用者，并且_其他项目_也基于 Synthetix 在 Optimism 上的部署而构建出来，比如 Optimism 上的 Kwenta 交易所 允许用户铸造和交易 Synthetix 的合成资产 (Synths)；期权协议 Lyra 也已经在 Optimism 上线，目前该协议提供 2 个激励性池子，锁仓价值超过 1,200 万美元。

  • L2 原生应用 #2: Dopex (DPX)

Dopex 是另一个可能受益于 L2 带来的交易费降低和更快的交易确认。

Dopex 是一个去中心化期权协议，由 Tetranode 和 DeFiGod 等 DeFi 知名人士支持。目前，该协议的农场锁定了超过 7500 万美元的价值。虽然该期权协议目前仍处于测试网阶段，但它正在计划在 Arbitrum 上部署其主网。

这意味着，该项目不仅可以为投资于新兴的 DeFi 衍生品行业提供一种投资方式，还可以增加对整个 Arbitrum 生态系统的敞口。

  • L2 原生应用#3: dYdX (DYDX)

去中心化永续合约交易所 dYdX 是另一个通过 L2 释放其功能的衍生品协议。

dYdX 已经基于 StarkWare 的可扩展性引擎 StarkNet 搭建了其自身的 L2 Rollup ，并于最近推出了流动性挖矿计划，其中包括对一些用户进行了价值 50,000 美元的 DYDX 代币追溯性空投 。因此，该协议的吸引力开始增加，其日交易量在 6 周内增长了 10 倍以上，目前一直保持在 6 亿美元以上。

L2 资产的追溯性耕作

也许投资者从某个特定 L2 网络的成功中获益的最直接的方式是投资于该 L2 网络的原生代币 。

虽然 L2 原生代币的设计空间正在浮现，并且我们不清楚是否所有的 L2 网络都会发行自己的原生代币，但在提供对特定网络的敞口方面，L2 原生代币能够提供与 L1 原生代币类似的角色。虽然目前还不清楚是否所有的 L2s 都将发行代币，但用户有机会将自己置于获得追溯性 L2 原生代币奖励的位置 。

  • Loopring (LRC)

Loopring 是以太坊扩容生态系统中最突出的特定于应用的 Rollups 之一，它提供了一整套基于 ZK-Rollup 搭建的产品，包括一个 AMM (自动化做市商) 和订单簿交易所。

投资者可以获得对该协议的敞口，该协议在过去一年以 LRC 代币 的方式产生了超过 270 万美元的收入，LRC 代币用于治理该系统，并作为最后的抵押品。投资者可以直接购买该代币，或者通过向该交易所提供流动性 来获取该代币奖励。此外，在指定的交易对 (见下图)上的交易量排在前 25 名的交易者 也可获得 LRC 代币奖励 (2021年9月9日开始，持续28天；结算周期为7天，共4个周期)。更多详情见：

https://medium.com/loopring-protocol/loopring-l2-liquidity-mining-round-14-1e28fdb3f6a9

  • Immutable X (IMX)

Immutable X 是一个建立在 StarkWare 之上的 ZK-Rollup。

该协议针对新兴的区块链游戏领域的 NFT 交易和使用进行了优化，目前正在通过「边玩边赚」(Play-to-Earn) 的模式以 IMX 代币 奖励的方式来激励用户使用该平台。IMX 代币在该网络的运行中扮演着几个关键角色：用户需要使用 IMX 来支付 20% 的交易费，且用户可以使用该代币来对治理提案进行投票。此外，IMX 持有者可以将该代币进行质押 (stake)，从而获得用户向该网络支付的交易费收益。

  • Optimism & Arbitrum (追溯性耕作)

在谈论 L2 时，如果不提及 Optimism 和 Arbitrum，显然是不合适的。

这两个最大的 Optimistic Rollups 网络都都筹集了数百万美元的资金，目前都没有发行本地代币。此外，尽管它们上线的时间很短，但这两个 L2 网络却产生了数百万美元的交易费，而这些费用并没有以任何方式流向其用户。虽然目前还不清楚它们是否会发行本地代币，但如果它们为早期采用者启动追溯性空投，那么成为这个两个 L2 网络的活跃用户可能是值得的。

总结

L2 的季节终于来了。

现在，有着不同风险承受力和不同敞口需求的投资者可以通过好几种方式来从 L2 的迅速发展中获益。无论是通过基础设施、Dapps 应用，还是本地资产耕作，投资者们很多不同的机会。

撰文：Ben Giove，Bankless 撰稿人
