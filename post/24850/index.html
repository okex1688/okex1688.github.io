背景

从2017年的加密猫，到2021年的NBA Top Shot，NFT的变化可谓是加密世界的另一个成长曲线，而令人意外的是NFT在今年的出圈，似乎一下子从“无人问津”变成了“万人空巷”，从艺术品，到音乐，到版权，甚至到一些奢侈品，NFT正以前所未有的姿态出现在大众视野里。

而在上周，CNN和福克斯广播开始了在NFT领域的尝试。以“CNN的财富《改变我们的时刻》”为主题的NFT记载着历史上值得关注的时刻被记录在Flow区块链上。而福克斯广播公司与其子公司BentoBox娱乐公司合作，启动了一支价值1亿美元的基金。

NFT的势头正在席卷各行各业，而目前又以以太坊上的NFT最为耀眼，毕竟超过80%的NFT作品都出现在了以太坊上，这也是ERC721的魅力，毕竟最早的NFT就是从该标准中诞生的，必然会受到后来者的关注。

尽管区块链行业已经有了像Flow这样专注于NFT赛道的公链，但依然会有其他公链进入这个赛道，毕竟在“水大鱼大”的日子里，谁也不能确定自己一定能成为最后那个“鲤鱼跃龙门”的玩家。

当然，作为当前炙手可热的波卡生态，NFT同样席卷了不少项目，而本文我们将重点为大家阐述NFT在波卡生态的建设与发展，供诸位了解与学习。

NFT究竟是什么？

虽然NFT已然火爆出圈，为人所知，然而刨根究底的讨论其究竟是什么？能够完整的回答出来的人并不是特别多。所以在探究波卡能够给NFT带来的价值之前，我们还是需要不遗余力的用一些篇幅来为大家说明究竟什么是NFT？为什么说NFT具有非凡的价值？

NFT是什么？

众所周知，通证是构建于现有区块链之上的智能合约，NFT也是一种通证，是Non-Fungible Token的缩写，即非同质化通证。与Fungible Token（同质化通证）不同的是，NFT具有不可篡改、不可分割、不可替代且独一无二的特点。

我们常见的比特币、以太坊、波卡等等都是Fungible Token，同种通证之间可以相互替代并且进行拆分。

我们在这里举一个最通俗易懂的例子。如果一个名叫Alice的人手里有5个DOT，这5个DOT之间没有任何的区别。Alice可以将0.5个DOT送给Bob，也可以在Bob向她买1个DOT的时候，给他任意的一枚DOT，这些无差别且能进行分割的DOT就是Fungible Token。

而突然有一天Alice突发奇想，给这5个DOT编号，并且为这5个DOT赋予了一个头像，那么Bob购买的DOT就是独一无二的了。当他购买1枚DOT时，他需要做出选择。同时他不能购买0.5个被编号和赋予图像的DOT，图像不能被切割，并且其中任意两个DOT都不相同，并不能互换。就好像世界上不会存在完全一致的指纹，也不会存在完全一致的NFT。

也就是说，NFT实际上是一种可溯源且确权的、不可拆分、不可篡改且具有价值的可编程通证。这一类的通证具有独特的数据和属性，就好像集邮爱好者收藏的邮票，每一枚都具有不同的魅力与价值，且不能拆分（撕毁无效）。

NFT的起源

NFT的出现并不是一个巧合，早在2013年，就有艺术家通过比特币的交易脚本以ASCII字符画的形式进行尝试。以寻求区块链技术所带来的不可篡改、去中心化、兼具安全性并且保护隐私的优势，来改变当时艺术领域所面临的掮客坐拥金山银山，艺术家却穷困潦倒的困境，并解决盗版所带来的一系列麻烦。

然而由于当时技术的局限性，这样的应用并没有得到发展。一直到2017年，以太坊推出通证标准ERC721，使得智能合约下的每份基于ERC721的标准都拥有一个uint256类型的独立编号，让它们可以具有独特的属性，并对应唯一的物品。

很快认为通证标准ERC721能够为大多数数字艺术和收藏品提供动力的Larva Labs开发了一组名叫CryptoPunks的像素头像项目，总量一万个，并由拥有以太坊钱包的人自行领取。至今为止，已经有9446个CryptoPunks在二级市场上流通，CryptoPunks＃7804更是拍出了757万美元的高价。

而在CryptoPunks发行短短6个月以后，Dapper Labs开发的小游戏Cryptokitties迅速走红，掀起了加密猫的热潮，引爆了整个NFT的市场。

自此之后，越来越多的NFT进入了我们的视野，无论是被大家津津乐道的数位艺术家Beeple的NFT作品，《每一天：前5000天》在全球知名拍卖行佳士得拍出了4.5亿人民币的天价，还是NBA TOP SHOT让球星卡从小众游戏变成了全民狂欢。

以及无论是币安推动的NFT for Good一站式开放数字艺术公益捐赠平台，还是旨在建造全球最大NFT身份验证库并为慈善组织提供基金的Ethernity Chain。

NFT不再仅仅是一个概念，而是真正的成为了我们生活中的一部分。

NFT有怎样的意义

那么NFT究竟有什么样的意义？

NFT的意义其实拆开来讲，我们可以另外写一篇长文，这里我们就简单讲讲我们对于NFT的一个主要的理解。

需要明确一下，许多朋友也许以为NFT只是相对于整个区块链领域里其中一个小版块，跟DeFi比，体量差得很远。

但是在我们看来，NFT其实是一个很大的方向，大到什么程度呢？NFT是可以与FT分庭抗礼的，而DeFi也只是FT领域里的其中一部分。NFT+FT就好像是太极阴阳一样，只有两者结合才是完整的数字世界里所有价值的表达，而这就是NFT最大的意义所在。所以，对于NFT，大家可以尽情畅享，它未来的价值无可限量。

NFT的发展与分类

“不积跬步无以至千里”，NFT的发展并不是一蹴而就的。从2017年至今经历漫长的标准的进化和类别的分化，才有了如今一片欣欣向荣的景象。为了让大家对NFT有一个更加全面的理解，我们仅就NFT标准的进化和分类进行一个简单的说明。

NFT的标准进化

正如上文中提到的，NFT的出现源于通证标准ERC721的推出。ERC是Ethereum Request for Comments的缩写，即以太坊开发者提交的标准提案，后面的数字代表着编号，在经过公开审议后，将成为以太坊改进提案并且作为以太坊的通用标准之一。

而NFT起源于2017年年底的EIP-721（Ethereum Implementation Proposal 721）。该标准也是以太坊上对NFT的第一个标准，正是由2017年最火爆的NFT项目CryptoKitties的CTO Dieter Shirley 所创建和发布的，没错，他发明了ERC721，并基于它创建了最火爆的项目之一，可谓是才华横溢。随后CryptoKitties的火爆也让ERC721打响了知名度，Decentraland等越来越多的项目也基于ERC721开始尝试NFT的探索了。

当然，ERC721并不完美，它仅限于构建NFT，具有一定的局限性，对于构建复杂的基于NFT的游戏来说，ERC721显然不够好用，于是越来越多的倾向于NFT的标准也在不断被提出。2019年由Enjin发布的ERC1155便是另一个知名的NFT标准。

ERC1155好像打通了任督二脉的绝顶高手，使得NFT和FT能够在这种标准中并存，并且相互转化，同时还带来了半同质化通证的新概念。ERC1155支持所有的元数据语言本地化，使所有的通证都通用，而不仅仅像ERC721那样只支持一种语言，并且一个智能合约可以容纳无数类型的通证，不会造成数据的浪费，还可以支持在单个交易中批量传输多个代币ID，并且在遗留元数据、ID替换等多个方面都有了改善。

ERC1155就好比ERC721的进化版，更加的强大且具有效率。

然而并不是只有以太坊的通证标准才可以构建NFT，波卡也推出了用于构建NFT的RMRK规范，而RMRK规范的诞生开启了NFT的新世界的大门。

除了最基本的发送、铸造、销毁、上架、购买等功能，RMRK规范带来了全新的交互模式。即EQUIP / UNEQUIP（将子NFT装备到父插槽中）、SET（在NFT上设置自定义值）、RESADD（将新资源添加到NFT作为集合的发行人）、REACCEPT（接受将新资源添加到现有NFT）。并使嵌套NFT、多资源NFT、有条件渲染NFT和NFT DAO成为了可能。

RMRK规范让用户手中的NFT不再仅仅是一个通证，而是成为构建元宇宙的基础。用户可以打造拥有其他NFT的NFT，也让特定的NFT在不同的条件下展现出不同的状态和视觉效果。甚至可以让同一个NFT被Audible、Kindle、Opensea识别到音视频等不同的状态，还可以根据NFT分解同质化通证，由社区确定某一个NFT的状态与流转。

而我们也相信，在未来还会有更为先进的NFT标准，并且为NFT注入源源不断的活力。

NFT生态的发展

与此同时，整个NFT生态系统也从最开始单一的游戏NFT、艺术NFT，发展成为了公链、DApp、资产等多个方向并行发展的生态系统。

NFT的起源来自于以太坊，毋庸置疑目前最活跃最古老的NFT公链还是以太坊，超过70%的项目在以太坊上发展，然而其高昂的Gas费对高频交易并不友好，用户体验感差，对存留和拓展新的用户是不利的。于是我们有了Flow、Wax、Efinity等多个NFT公链。

其中Efinity是由Enjin公司打造的基于波卡的可拓展NFT跨链网络，即保障了吞吐量的提高和交易速度的提升，也保证了基于不同的公链的NFT之间的交互与交流。为NFT的发展提供了一个良好的平台。

而与NFT相关的DApp不再局限于游戏NFT，在艺术NFT、游戏NFT、NFT平台，甚至NFT与DeFi的结合上，NFT生态中的DApp都有了长足的发展。并逐渐领跑整个加密领域。

基于波卡Layer2的解决方案Plasm推出了首款NFT游戏DApp PolkaPet，在4分钟内售罄800张，并搭建了Plasm生态中的NFT社区，大大降低了NFT的交易成本，实现了NFT的跨链并增强了其易用性。Plasm可以将钱包与Twitter绑定，大大的降低了使用者的门槛。

而NFT最早发展起来的资产方向逐渐的走向了成熟。英国香水公司Rook Perfumes创建了Perfume DAO，参与者可以通过购买NFT参与创造香水；保时捷数字子公司Forward 31推出NFT初创公司Fanzone，发布德国足球队球员卡，以扩大对足球联赛的支持；TechCrunch创始人、加密货币投资者Michael Arrington将自己在乌克兰基辅的公寓以NFT的名义出售。

无论从短期还是中长期来看，NFT生态仍在发展之中，开发者和创造者都将为我们创建源源不断的超乎想象的NFT新用例。

波卡NFT的发展情况

目前波卡生态的NFT才刚刚开始，各个项目方还属于跑马圈地的状态，分别在NFT的不同领域探索，有对NFT标准创新的，有想要打造NFT市场的，也有构建NFT公链的，也有基于NFT打造一个虚拟世界的，以获得官方Grant（津贴）为准，我们可以简单了解一下现在波卡生态中的NFT项目都在做怎样的探索。

Bit.Country

Bit.Country是一个去中心化的永恒元宇宙。用户可以创建自己的小元世界（Bit Country）、街区（Blocks）、土地（Subdivison）和物品（Items）作为加密收藏品和非同质化代币 （NFTs）。在这个世界里，用户会拥有自己的交易市场，NFTs，社区代币，粉丝经济和3D虚拟世界。在去中心化Web3.0时代里，让任何人在区块链网络上创建自己的元宇宙，社区和经济作为自己的小元世界（Bit Country）。

Unique Network

Unique Network 是基于 Polkadot 和 Kusama 设计的 NFT 公链，通过开源的组件库，使任何人都可以自定义 NFT 铸币方案，并且规避了以太坊上高 gas 费等问题。

RareLink Network

RareLink 协议为动态 NFT 创建基础设施，支持跨链传输并为新的去中心化金融 (DeFi) 应用程序提供支持。动态NFT是使用预言机与外部数据和系统进行通信并做出反应的永久智能合约。预言机允许 NFT 使用外部数据/系统作为铸造/销毁 NFT、点对点交易和检查状态的机制，可以响应现实世界的事件。

NFTMart

NFTMart是一个全可扩展的、去中心化、基于独立公链的跨链交易平台，也是首个独立主链的专业去中心化NFT交易市场。

DNFT Protocol

DNFT Protocol是基于Substrate的跨链去中心化NFT资产网络，致力于打造新一代NFT协议。旨在打造一个去中心化的NFT和游戏中心，为波卡和Substrate生态的跨链应用提供支持。

CryptoGraph

CryptoGraph是由偶像和艺术家创造的独一无二的数字收藏品，并且永远支持善举。Cryptograph平台的核心是希望激励创作者，并以一种模式支持慈善事业，使创作者、收藏家和我们的平台的利益永久保持一致。

Starry Network

一个功能性的 NFT 协议。现在 Starry 协议包括 NFT 的批量铸造/转移、可组合令牌（Graph Token）、可分割 NFT 和添加了 NFT 贡品的 Moloch DAO。是波卡上第一个支持批量铸造的项目，第一个支持可组合代币的项目，第一个支持分片代币的项目，第一个完全支持 Moloch DAO 的项目。

DEGO Finance

DEGO是去中心化世界的乐高积木，通过NFT+DeFi的多样化组合，构建理想的去中心化金融生态。DEGO采用模块化组合设计理念，将产品的各种要素组合起来，形成具有特定功能的通用模块，并通过与其他模块相结合，创建一个具有多种功能和性能的新系统，像乐高积木一样达到1+1＞2的效果，并产生无限的可能。

Gamepower

Gamepower是第一个建立在Kusama和波卡上的完全去中心化的游戏发布平台。发现新游戏，且可以自由交易用户的数字游戏，并获得具有真实世界价值的稀有收藏品。

当然，除了以上获得了官方Grant支持的项目。也有一些NFT项目并没有获得官方的Grant，其中有的是还没有通过官方的审核，也有的一些项目是没必要获得Grant的加持，Efinity就是后者中值得关注的一个项目。

NFT未来的发展趋势

链上原生资产发展（元宇宙方向）

元宇宙，英文叫做Metaverse，最早来源于作家尼尔·斯蒂芬森（Neal Stephenson）的科幻小说《雪崩》，描述了一个人们以虚拟形象在三维空间中与各种软件进行交互的世界。

根据维基百科的解释，元宇宙是指一个集体的虚拟共享空间，由虚拟增强的物理现实和物理持久的虚拟空间的聚合创造，包括所有虚拟世界、增强现实和互联网的总和，由持久的、共享的3D虚拟空间连接到一个可感知的虚拟宇宙。如果觉得复杂，不妨可以简单理解为虚拟空间组成的宇宙。 

元宇宙Metaverse一词，由“Meta”和“Universe”组成，分别是“超越”和“宇宙”的意思，我们可以理解为，现实中的宇宙就只有这一个，但是在虚拟空间中，我们可以创造出无数个不同的虚拟宇宙，这是超越现实宇宙的Metaverse。也因此，元宇宙被认为是互联网的未来。

但是，根据这样的描述来看，元宇宙并不仅限于区块链，现有的互联网技术也可以去构建元宇宙的部分。实际上，在互联网方面，已经有资本先一步布局，开始投资元宇宙领域了。就在今年3月份一家沙盒游戏平台Roblox便第一个将 " 元宇宙 " 的旗帜插到纽交所上，上市首日是指便突破了400亿美元，瞬间引爆了资本圈，而Roblox背后便有全球游戏巨头腾讯的身影。随后，众多游戏厂商，游戏相关产业等也纷纷宣布进军元宇宙这片蓝海，英伟达、知名游戏公司Epic Games、Facebook、微软、字节跳动等巨头也不甘示弱，很快元宇宙成为资本热捧的香饽饽了。

但是，今年5月份的一场腾讯与DD373游戏交易平台的一场官司，却引发了我们的思考。这个官司究竟谁对谁错，我们不去细究，主要是在于腾讯在其辩护中的一句话，“游戏账号的所有权不属于玩家，而是属于游戏厂商。”引发了广泛的讨论。原来，我们辛辛苦苦在自己游戏账号上花费了大量的时间和金钱，到头来，这个账号所有权还不属于我自己？那么，如果是腾讯这样的中心化巨头按照同样的逻辑来构建的虚拟世界，里面的财产岂不是还不属于我自己了？

所以，去中心化的元宇宙才是未来最具发展潜力的方向，因为大家都会更希望我们就像在电影《头号玩家》里一样，在虚拟空间所花费的时间是有意义和价值的，我们所获得的虚拟资产也应该是有价值且归属于我们的。

自然地，能够承载去中心化虚拟资产的技术，莫过于链上原生资产的加密货币和NFT了。尽管现在元宇宙的发展还很早期，但是一些结合元宇宙属性比如有3D的虚拟空间展示，有原生的NFT资产，有通证经济的设计等项目，正在逐渐探索新的可能。随着与元宇宙相关的一些比如AR/VR、智能设备、5G等技术逐渐成熟，以及更多创作者和资本在这个领域的沉淀，去中心化性质的元宇宙一定会引领新一轮的浪潮，届时NFT与加密货币必将再次披上华服站上历史的舞台。

链下实体结合的发展

除了链上的NFT以外，NFT的非同质化这个属性，也吸引了众多传统企业的关注，纷纷尝试利用自身的影响力结合NFT获得新的增长点。

NBA Top Shot就属于老的商业模式上链，可以享受区块链带来的便利。NBA Top Shot是一款基于NFT的应用，由NBA官方与Crypto Kitties背后的团队Dapper Labs所推出的一款基于区块链的数字收藏游戏，用户可以购买、出售和收集官方授权的数字收藏品，其中包含各个球员的标志性时刻。

而这款应用能够快速获得几十甚至上百万的用户，并且在2021年的2、3月份能够有每月上亿美元的成交额，其实是由于它有一个基础。NBA之前就有实体的球星卡，本身就拥有庞大的群众基础，此次的NBA Top Shot相当于将有图案的实体卡变成了有视频的数字收藏品，比起只有图片，球星的标志性时刻显然更有可收藏性，而且NFT比实体卡有诸多优势。实体卡成交必须线下成交，光是这点就限制了其流通性，反观NFT可以做到全球任何人随时随地都可以交易，交易层面几乎没有门槛。

其次，实体卡容易造假，如何辨别真伪是一项费时费力的技术活，甚至由此诞生了Beckett Media这样经营近40年的专业市场收藏平台，可见这样的需求和市场是不容小觑的，但是NFT可以在链上很容易的得到验证，就不存在这样的防伪问题。

而刚经过上链后，老的商业模式降低了交易门槛，防伪成本，又释放了流动性，一举多得，这才是NBA Top Shot火爆的真相。由此可以看出，老的商业模式上链，可以获得NFT的优势，从而焕发第二春，是属于已经证实可行的一套解决方案，值得现有的传统企业借鉴和探索。比如，可以进一步延伸把一些权益或者证明作为NFT来售卖，我们甚至可以见到把VIP资格作为NFT售卖，或者租用的情况，又或者把好比是巴菲特的晚餐资格这类权益作为NFT，我们可以把某个明星的某一天拿出来卖，把粉丝经济玩起来等等。

当然，这仅仅只是链下实体企业结合NFT的很小的一块应用方向。毕竟，这还只是旧城改造，还没有涉及到传统企业利用NFT探索出新的商业模式这个领域，此处由于篇幅限制，我们不做过多的展开，NFT是一个很大的领域，我们会不断剖析分解，可以期待我们后续关于NFT的解读。

两者互相影响

如果说链上NFT资产可以看做是去中心化的表达，链下实体结合的NFT可以看做是中心化的延伸，那么我们需要正视一个观点是，一定要包容地看待技术和商业的发展，不能非此即彼，中心化也有中心化的好处，去中心化也有去中心化的优势，所以，两种NFT的发展方向，更有可能是各自有各自的生态，而在某些方面又能互相结合，共建共享。不过，现在还在NFT发展的早期，舞台才刚刚搭好，NFT会带来怎样惊艳的表演，让我们拭目以待吧。

后记

尽管NFT在经历过去半年的炒作后，开始出现了小幅度的降温，但这丝毫不影响NFT在其他行业的发展，我们看到越来越多的传统企业开始涉足NFT市场，不管是传统零售企业，还是互联网科技企业，纷纷向NFT市场伸出了“兴趣”之手。

而现阶段又以以太坊上的NFT最为知名，但NFT的跨链问题同样困扰着行业的发展，这其中除了像波卡这样的跨链项目，还有其他公链也在探索NFT的跨链问题，只是本文不做展开讨论，但这个方向同样值得我们关注。

至于波卡生态中谁能成为NFT赛道的黑马或者王者，其实现在还很早期，毕竟波卡也才刚刚启幕，这需要我们花费更多的时间去观察、去实践，或许波卡生态中也会诞生更独特的NFT项目或者产品，这才是波卡生态给NFT可能带来的改观。
