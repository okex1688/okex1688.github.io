当奥运遇到区块链，Flow 成“幕后功臣”

近期北京 2022 冬奥会在社会上引发公众关注，冬奥会吉祥物“冰墩墩”更是被抢购一空，同样火爆的不仅仅是吉祥物，Flow 区块链上的生态团队 nwayplay 联合国际奥委会发布的官方授权 Play To Earn 手游《Olympic Games Jam:Beijing 2022》同样也吸引了不少粉丝，在短短几天内安卓市场的下载量已经在一万次以上。

不仅如此，基于 Flow 区块链上的“冰墩墩数字盲盒”更是“一墩难求”，而为其提供基础区块链服务的公链 Flow 也因此获得不少用户和投资者的讨论。

Flow——既熟悉，又陌生的公链

说到 Flow 公链，实际上大部分加密货币投资者的第一感觉是既熟悉又陌生。熟悉是因为 Flow 区块链的开发团队 Dapper Labs 曾经是火爆以太坊的 NFT 加密猫的创作团队，同时 Dapper Labs 的 CTO 也是以太坊 ERC721 代币标准的主要制定者，而 ERC721 目前是大部分 NFT 使用的标准，可见 Dapper Labs 在 NFT 行业内的影响力。

加密猫在 17 年在以太坊上火爆的同时也引发了以太坊链上的拥堵，也正因为此，Dapper Labs 团队开发了 Flow 公链，并在 Flow 公链上主要发展 NFT、区块链游戏、动画、音乐等生态产品。不仅如此，Flow 代币在加密市场的表现也非常耀眼，从 CoinList 公募上线到目前为止，涨幅接近百倍，引发了市场的强烈关注。另外 Dapper Labs 官方推出的 NBA 授权热门 IP 产品 NBA Topshot 也使得 Dapper Labs 再次引发热议，币圈投资者即使身处币圈，没有参与投资 Flow 生态，也会耳濡目染，对 Flow 有所了解。

而加密投资者对 Flow 感觉陌生主要是因为 Flow 目前主打与传统 IP 方发行 NFT 以及构建相应生态的公链，大部分 Flow 生态的用户主要以 NBA 粉丝用户为主，以及加密艺术市场、NFT 交易市场、部分链游以及元宇宙相关的项目产品。

也就是 Flow 生态中大部分用户主要以传统 IP 方的用户为主，比如像 NBA Topshot，相应的加密行业中对 Flow 生态 NFT 炒作者相对较少。投资加密货币的朋友更加关注与代币的可投资性，而很少关注于 Flow 生态中合作的 NFT 相应的玩法。同时又由于 NFT 的市场投资回报率并不像 ERC20 这类同质化代币这样简单明了，因此在 Flow 生态中 NFT 的投机属性有所降低，对加密投机者来说吸引能力也有所降低。

当然即使这样，这种链上生态投机属性的降低也丝毫不减市场对 Flow 的追捧。主要是因为 Flow 生态面对的是传统 Web2.0 的 IP 生态方和大部分普通用户群体，比如像 NBA Topshot 用户中大部分为 NBA 粉丝，只有少部分为加密货币 NFT 的投资者。同时对于体育类的球迷而言，他们更加愿意长期持有相关 NFT 产品，就好比传统印刷的 NBA 宣传收藏品，而非投机炒作，这与大部分币圈 NFT 的投机者理念是不同的。

Flow 的 IP 生态

传统行业的 IP 生态对于 Flow 来说是发展的重点，其中最成功的就是 NBA Topshot。2020 年 Flow 的开发团队 Dapper Labs 获得 NBA 官方授权并基于 Flow 公链的 NBA Topshot 成功引发市场关注，截止目前 NBA Topshot 已经拥有接近 200 万注册用户，交易流水接近 10 亿美金，NBA Topshot 的成功使得 Flow 成为拥有众多体育 IP 的赛事方都看到了布局区块链 Web3.0 以及 NFT 生态的必要性。

NBA Topshot 的成功让很多传统手持天量用户的 IP 品牌方看到了加密数字资产的魅力，而 Flow 也自然成为这些品牌商的首选，目前 Flow 已经和多个国际品牌商达成了合作关系，其中包括：文学音乐、体育动画、游戏以及传统出版物等诸多领域，我们下面对部分具有一定影响力的合作 IP 进行简单介绍。

  • 文学音乐

在音乐领域，Flow 和知名的音乐发行公司华纳音乐和环球音乐都有相应的合作，并计划在未来推出音乐相关的 NFT 产品，目前相关开发仍在进行中。

在儿童文学领域，Flow 和风靡美国的儿童文学 Dr.suess 合作，推出 NFT 收藏 APP Seussibles，目前该款 APP 已经上线安卓和苹果应用商店。

  • 体育

体育类一直是 Flow 主打的方向，比如 20 年发行的 NBA Topshot，作为一个典型的成功案例，NBA Topshot 的成功也吸引了大量体育赛事团队考虑在 Flow 上发行相关的 IP 作品。目前除了值得关注的冬奥会冰墩墩数字盲盒产品之外，美式橄榄球 NFL、ICC 板球世界杯、UFC 终极格斗冠军赛、La Liga 西甲等也纷纷计划或已经在 Flow 上合作推出相应的 NFT 作品。

当然除了 Flow 官方自己合作的体育赛事项目之外，实际上对于其他与体育相关的项目方而言，Flow 也是一个不错的选择。比如像传奇体育项目 Sportsicon 这样以体育明星为主打的粉丝俱乐部 Lion Club 就计划将发行的 NFT 迁移到 Flow 区块链上，利用 Flow 区块链在体育 IP 领域的巨大优势，从而吸引更多粉丝群体。

  • 游戏

虽然 Flow 在体育方面拥有非常成熟的 IP 以及市场，但是这也并不影响 Flow 在游戏领域的发展，和体育等传统 IP 进行合作的模式类似，Flow 在游戏领域主要还是与主流的游戏厂商进行深度合作，并计划开发相应的爆款产品。

比如 Flow 和 ubisoft（育碧）进行深度合作，并计划在 Flow 链上开发相应的 3A 大作，ubisoft 在游戏领域具有多年的开发经验，曾经也出品了多部经典的游戏作品，比如《刺客信条》、《孤岛惊魂》等家喻户晓的游戏作品，Flow 与 ubisoft 的合作也会进一步提升 Flow 生态游戏的多样化发展。

除了像 ubisoft 这样的大型游戏公司之外，Flow 也接纳各种风格的区块链游戏团队进行入驻，比如像加密版精灵宝可梦——ChainMonsters，在上线不到两个月内就收获了 35000+ 的活跃用户量，10000 Alpha Access 上线后被抢光。

另外像知名的加密游戏开发商和投资机构 Animoca Brands（投资 Sandbox 等知名游戏）也计划在 Flow 公链中推出收藏品游戏平台 MotoGP Ignition。通过 MotoGP Ignition，粉丝们可以在区块链上拥有和交易他们最喜爱的 MotoGP 摩托车和人物收藏品，这也体现出 Flow 生态中游戏版块的活跃程度。

DeFi 与基础设施

其他基础设施，包括 DeFi、流动性挖矿，同质化代币跨链生态等。

虽然 Flow 主打 NFT 以及相关的传统 IP 生态群体，但是对于 Flow 而言，让更多加密货币投资者认识并了解它也是项目的目标之一。Dapper Labs 也将在 2022 年加大在 Flow 区块链中 DeFi 版块与区块链基础设施等方面的投入力度，以便于目前的主流区块链生态实现对接。

比如近期 Flow 官方和 Circle 进行合作，并将 USDC 引入 Flow 生态之中。虽然 Flow 也有相应的合规出入金工具，但是面对全球用户而言，美元稳定币仍然在一定程度上能够给予用户更大的选择（尤其是对资本有严格管控的国家）。这在方面可以提高 Flow 生态用户支付 NFT 以及其他数字藏品的渠道，另外主流稳定币的引入也会加快 Flow 生态中 DeFi 的发展。

除了 USDC 之外，目前像 ChainLink 等预言机也开始支持 Flow 公链。我们都知道，预言机属于 DeFi 发展阶段非常重要的组件，目前市场上预言机大部分份额由 ChainLink 占据，对于 Flow 而言，ChainLink 的支持也会进一步推动 Flow 中 DeFi 和其他金融属性项目的发掘。

在跨链、Swap 和钱包方面，Flow 生态 Blocto 钱包也推出了自己的跨链桥和 Swap 功能，其他的基础跨链设施已经在计划之中。Blcoto 钱包是 Flow 生态中最受欢迎的钱包，主要为用户提供了私钥托管形式和非托管形式的两种钱包模式。托管模式主要为了考虑到大量普通圈外用户的体验，而非托管模式则更加适合币圈内的加密货币投资者。除此之外，Flow 团队以及着手于与其他更多的钱包开发方进行合作，并计划让更多加密货币钱包支持 Flow 区块链，其中也包括了建立类似于 Metamask 这样的浏览器插件钱包计划。

基础设施方面，Flow 整体仍然有较大的发展空间，因此 Dapper Labs 已经开始建立一个类似于加密黑客松这样的 FLIP Buildation 开发者项目激励活动。开发者可以通过 FLIP Buildation 为 Flow 生态进行基础设施的开发完善，包括钱包、API 接口以及同质化代币项目的开发等等，并获得相应的代币奖励，从而实现 Flow 和开发者共赢的局面。

Flow 的公链架构

Flow 公链的设计和大部分区块链是有所不同的，用官方的话来说，这是一种流水线的设计方式，流水线的形式可以大大提高链上任务的处理能力。

一般我们在以太坊等公链上看到的区块链节点基本上都是包括从收集交易信息、执行合约、进行验证以及生产区块等多功能于一身，这对于区块链节点的性能和稳定性有非常高的要求，同时这样的区块链也面临着诸多性能方面的瓶颈。也正因为此，Flow 官方发布的加密猫在以太坊上直接造成拥堵，这就好比一个车间里一个工人同时做多个岗位的工作，效率相当低下。

Flow 则是通过将节点进行功能区分，主要分为收集节点、执行节点、验证节点和共识节点，不同节点根据其资源的需求和硬件配置来执行不同的任务，这种流水线的方式可以有效提高节点的效率，同时防止节点作恶。虽然这种设计整体上增加了区块链系统架构的复杂性，但是相对于以太坊即将升级的 ETH2.0，以及目前的一些 layer2 等扩容方案，Flow 的架构目前还是相对比较稳定的。

另外，Flow 专门为其公链设计了 Cadence 开发语言，Cadence 是一种面向资源型的编程语言，与以太坊上使用的 Solidity（面向对象）是有所不同的。从合约语言上 Flow 已经对智能合约做了相应的优化处理，因此对于 Flow 区块链来说，自然会容纳更多交易，从而避免类似拥堵以太坊这样的事件发生，也正因为此，Flow 可以被看做更加适合 NFT 以及元宇宙、区块链游戏等拥有大量 IP 的合作方入驻和生态的建立。

虽然 Flow 新型的智能合约语言提高了 Flow 区块链整体的运行效率，但是同样，市场中相应的 Cadence 开发人才相对较少也是一个比较突出的问题，而相应的审计也成为很多开发者关注的事情。不过 Flow 的 Cadence 开发语言学习成本较低，因此这并不是一件很难的事情。

开发者在完成项目开发之后，目前需要向 Flow 官方或者认证第三方提交以完成项目合约的审计工作，而在未来一段时间里，Flow 官方也将加大开发者对学习 Cadence 开发语言和进行相应项目开发的奖励活动，并且逐步转向完全无许可部署。

总结

对于 Flow 而言，拥有大量顶级 IP 合作伙伴是一笔宝贵的财富，同时相应的 Flow 生态上的合作项目普遍水平相对较高，受众也主要以传统用户为主，这与目前大部分公链所走的路是不一样的。很显然，Flow 主打的是吸引 Web2.0 的 IP 对粉丝转向 Web3.0 的过渡，这部分用户对于 NFT 的炒作和投机性质相较于以太坊等链会小很多，也就是当 NFT 熊市到来的时候，其他公链会受到很大的影响。而对于 Flow 来说，所受影响有限。

另外 Flow 目前主要发展 NFT 和元宇宙、链游等板块，而对于 DeFi、跨链、去中心化等方面仍然有一定的发展空间，Flow 也需要趁目前体育年（比如近期即将举办的超级碗比赛）的到来加大基础设施等方面的开发和生态项目的激励，这样才能在未来不仅仅吸引 Web2.0 顶级 IP，同时也能和其他公链有更紧密的联系，最终促进 Flow 生态的建设和发展。
