长久以来，美妆行业为客户提供以亲近肌肤和治疗为主的感官体验，这个行业正在被改造成为一个虚拟世界，而在这个虚拟世界中用户无法感受到、闻到或者真正触摸到自己面前的东西，这一切听起来似乎有些疯狂，但是，这些情况确实正在发生。 

品牌必须迅速适应并学习如何在数字领域建立自己的地位 。 

Charlotte Tilbury、Lottie London、圣罗兰、雅诗兰黛、Gucci和Nars等品牌已经开始将自己沉浸到元宇宙中。

据报道，今年2月，欧莱雅提交了17项与元宇宙相关的商标申请，分别与NFT、虚拟香水、 化妆品和虚拟人造型有关。Rhianna不久前也为自己Savage X Fenty和Fenty Beauty品牌申请了Fenty商标。

这些品牌以包容性而受到认可，他们正在为其他一些具有包容性的品牌加入虚拟世界发起挑战。 

雅诗兰黛的全球消费者营销高级副总裁乔恩·罗曼 (Jon Roman) 表示：“就一个人如何通过虚拟形象之类的东西来表达自己而言，元宇宙本质上是无限的，而正是这一点将释放出更多超现实的美，以及具有包容性和抽象自我表达的新领域。”

简而言之，在元宇宙中，人们可以成为任何自己想成为的人，并且可以随心所欲地生活，没有在现实生活中可能遇到的任何约束或限制 。

美是更身临其境的体验和教育

化妆师Alex Box对此表示赞同并接受了关于虚拟的可能性。“美是建立在商品化之上的，目前我们需要的是关于这一点的更身临其境的体验和教育，而不是想着取代这一观点 。随着技术和洞察力的发展，体验的复杂性也会随之提高。” 

元宇宙已经吸引了无数精通数字技术的人群，尤其是女性游戏玩家，女性受众比想象的要多得多。 

事实上，WeAre Social的一份报告发现，在16到44岁的女性中，有80%在接触互联网及电子游戏，而全球世界指数的另一项调查发现，2021年，53%的女性粉丝下载了免费电子游戏。

这也解释了为什么Charlotte Tilbury经过深思熟虑决定赞助女孩游戏玩家节。4月，该品牌推出了其第一家虚拟商店，名为悄悄话派对之虚拟美妆仙境 （Pillow Talk Party Virtual Beauty Wonderland）。

在这个虚拟商店里，顾客能够看到Charlotte本人的3D立体化身，而这使品牌立刻就刻入了游戏玩家的记忆中。

要进入CharlotteTilbury的虚拟世界，用户可以直接通过网页浏览器或手机浏览器访问这个魔法门户 。 

用户只需扫描一个二维码，Charlotte就会神奇地出现在面前，旁边则是漂浮和旋转的3D美妆产品。

简单几步，用户就可以与全球专业艺术家一起开启大师班的大门，享受与朋友一起购物的机会。

试想一下通过数字时尚应用Drest进入虚拟空间会是一种怎样的体验，没错，Gucci Beauty就是第一个推出该应用美容模式的品牌，该应用允许用户试用Gucci 29款虚拟美妆产品。

不久之后，Nars和Drest也分别推出了为期9天的美妆活动，用户可以使用品牌的30种产品在虚拟模特身上进行定制 。除此之外，消费者还可以将试用外观和产品保存在个性化的情绪板上。 

当谈到特定的“元宇宙”时，可供选择的数字游乐场可不仅仅只有一个。 Decentraland似乎是当下最受欢迎的虚拟空间，它真正关注人们的自由和自我表达。同时它也是消费者可以访问NFT和收集可穿戴POAP（出勤证明协议）的地方。 

NFT最初被用于艺术界，它是可以将信息记录到到区块链这个无法被更改的系统并溯源的数字藏品。 

这有点像一件原创艺术品，它们永远存在并且不能被复制，而POAP类似于一张票、一个程序或一件T恤，被用于验证用户是否出席了某项活动 。

 Lottie London的全球营销总监Nora Zukauskaite解释道：“去年，当我们初次涉足这个数字空间，与Ciate London合作推出我们的第一个NFT时，反响非常热烈，发行的NFT在24小时内就全部售罄。” 

之后，在Decentraland中的拉斯维加斯市区与洛杉矶的名星专用美甲师Chaun Legend举办了一场活动，展示了Ciate发布的新系列并宣布他们将在六年后重新进入美甲领域这个理想的场所。

 “在现实生活中，我们会被限制在某个地点或某个时区，并且会在我们所处的位置上停滞不前，但在这里我们可以以一种新的方式向广大观众展示我们的品牌和产品 ，”Nora继续说道。“它释放了巨大的潜力，我们即将举办首届Decentraland舞会。舞会在一个独特的场所，我们还将有1000个独特的POAP，其中包括Lottie London畅销的发光彩妆、舞会皇冠和胸花。” 

雅诗兰黛最近也入驻了Decentraland，成为其3月与元宇宙时装周合作的独家美妆品牌。 

雅诗兰黛邀请用户走进标志性的高级夜间修复“小棕瓶”，为他们解锁独家POAP和一个NFT可穿戴设备 ，为他们的虚拟人赋予发光的光环，只要用户愿意，光环就会持续发光。 

对于狂热的美妆爱好者来说，这是一种令人狂热的产品，品牌进入虚拟世界意味着他们可以接触到一些尚不熟悉雅诗兰黛的新客户群体。“这不是针对特定年龄的群体，而是为了吸引新的消费者社区并与之互动，这些用户已经沉浸在元宇宙中并有浓厚的兴趣与美容和时尚品牌建立虚拟联系，”Roman解释道。 

虽然目前主要的元宇宙人口的确都是Z世代和千禧一代，但任何可以上网的人都可以参与其中。 

WGSN预测，到2024年，品牌将瞄准一个新的美容角色——Beautyversals。作为认为数字产品与实体产品同样重要的受众，他们也更喜欢与虚拟大使互动，而不是那些现实生活中的专家。 Dermalogica已经开始在内部试水，他们并不专注于品牌针对消费者的活动，而是创建了第一位内部数字大使 Natalia 。 

Natalia就新的治疗方法、产品和方案对治疗师进行培训。Dermalogica打开了一个全新的维度，Natalia的头像可以实时显示人们在现实生活中永无法体验的皮肤老化。

元宇宙就是新一代的未来

Wunderman Thompson Intelligence的全球总监Emma Chiu认为，现实世界和元宇宙这之间的临界点将成为新的现实 。 

“数字接触点绝不是面对面活动的替代品，但元宇宙正在提供另一种比我们所经历的更复杂的体验。深入元宇宙是有意义的，但同时我们也需要一些转向现实世界的活动。我们所认知的现实将会改变，虽然这可能是针对老一代人的调整，但对于现在出生的人来说，元宇宙就是他们的未来。”

希望品牌有机会为其产品和消费者添加一层“深度服务” ，并为消费者提供一种尝试、享受和使用美妆产品的新方式；也期待更多品牌开始向虚拟商店敞开大门，带领用户一起感受NFT的魅力。
