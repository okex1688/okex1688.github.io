以下是关于NFT游戏的对话记录，对话者包括Sky Mavis/Axie Infinity联合创始人兼首席运营官Aleksander Larsen、Yield Guild Games联合创始人兼首席执行官Gabby Dizon和A16z的合伙人Jonathan Lai，此对话发生在 GDC 2022 期间 A16z 主办的活动中。

Jon：谢谢大家的加入，我是Jon，a16z的 合伙人。今天我们讨论的话题内容离我们很近，并且是GDC 每个人都关心的话题，就是 NFT、Web3 以及它们对游戏的影响。所以，我很荣幸有两位 Web3 游戏行业的杰出人物加入。

Axie Infinity 背后的Sky Mavis 公司的联合创始人兼首席运营官 Aleks Larsen 和 Yield Guild Games 的联合创始人 Gabby Dizon。

让我们开始吧。Aleks 和 Gabby，在过去的一年里，我们听到了很多关于 NFT 和 Web3 游戏将如何彻底改变人们的游戏方式，以及改变行业发展方式的消息，让我们把它分解成细节。首先，从玩家的角度来看，我们今天可以用 NFT 或其他 Web3 基础设施做哪些以前不可能的事情？

Gabby：作为来自发展中国家的玩家，菲律宾一直是游戏的启动国家之一，因为有很多喜欢玩游戏的人并且在社交方面非常活跃，但人们并不总是能够因为他们在游戏中花费的时间和精力而获得现实世界的奖励。现在有了游戏 NFT，人们可以拥有自己的资产，进而参与到游戏经济中。因此，对于来自我国和其他能够参与游戏经济（例如 Axie Infinity）的发展市场的人来说，这是巨大的改变。它使人们能够拥有数字资产，这些资产以NFT的形式拥有自己的价值，例如 Axies 或代币。它真的改变了人们看待游戏的方式。我认为几乎有 2% 的菲律宾人会玩 Axie Infinity，所以它对整个人群来说是一个巨大的游戏规则改变。

Aleks：我认为我们在 Axie 中看到的一件事是人们更高的参与度和保留率，而且人们也更加关心这部分资产。当我们研究 NFT 实际上是什么时，它是您数字身份的一部分。所以，我真的相信我们制作 Axie 和 Sky Mavis 的部分原因是我们希望人们拥有自己的数据，当您开始在线旅程时，尤其是年轻人，其中很多都是在游戏中。每当你玩游戏时，我坚信你应该拥有属于你的东西。

这对很多人来说都是有意义的，不仅仅只是关于货币价值。这是关于人们对这些游戏资产的情感联系。过去，如果游戏关闭，你的数据就会消失。现在，至少你有一些切实的证据证明，“嘿，我确实在那个游戏中做到了这一点。” 那是有意义的。所以，对我来说，这真的很重要，这在以前是不可能的。因此，虽然金钱是吸引人们的原因，但我相信他们会因为其他东西而留下来，并使它们变得独一无二。

Jon：非常有趣的是，您提到的第一件事就是所有权，这些是数字资产的概念，没有人可以从你身上拿走。如果游戏关闭，如果 Axie Infinity 明天关闭，您的 Axie 仍会在您的钱包中。你可以交易它们，它们有一些内在价值。我们从 Web2 开发者那里得到的反驳是，数字所有权可能只是Web2游戏开发者对现有游戏的一个政策改变。例如，在魔兽世界中，他们可以只允许人们用黄金交易账户；在暗黑破坏神中，有一段时间有一个真钱拍卖行，人们可以在这里买卖他们在游戏中发现的武器和盔甲。如果你能再深入一层，除了拥有和交易你的数字物品的能力之外，Web3和NFT的特别之处是什么？

Aleks：这实际上与我所说的身份有关。玩家花费数年时间在《魔兽世界》等游戏中寻找游戏资产。他们获得的物品对他们有价值，是因为获得它们所花费的时间很多，而不是销售价值。如果他们随后退出游戏，但想稍后再回来向某人证明他们做了所有这些很酷的事情，则不确定他们是否在那里。游戏开发者可以很容易地删除这些游戏资产。

Gabby：是的，NFT 是数字财产。您可以围绕这些资产完全制作自己的游戏，拥有 NFT 将拥有数字所有权的概念转移到数字世界。

Jon：从开发者的角度来看，如果你现在正在开发游戏，应该如何思考 NFT 和探索 Web3？我们看到大量游戏开发者从 Web2 世界转移到 Web3。是什么在激励他们？有什么令人兴奋的？

Aleks：我们与很多游戏开发者交流过，他们都很好奇，“这项创新是如何发生的？什么是双币模型？什么是 SLP？” 说实话，其中很多只是关于实验，试图弄清楚这项新技术到底能带来什么。对于进入这个领域的游戏开发者来说，他们必须非常非常仔细地查看代币。我向许多创业者强调，尤其是那些试图通过出售代币来筹集资金的创业者，这些代币是多么重要。因为这是玩家立即拥有的流动性，或者投资者立即拥有的流动性，是你作为一个游戏工作室或开发者必须长期维护的东西。

与其出售 NFT 赚取少量现金，不如实际奖励早期的社区成员，并通过给他们 NFT 将他们与您的社区联系起来。当你卖东西给玩家的时候，他们会觉得你欠他们什么；但当你送玩家一些东西的时候，他们会觉得是他们欠你一些东西作为回报，而这种感激之情异常强烈。这就是你可以开始建立社区核心成员的方式。

Gabby：我对 Web2 和 Web3 之间的区别的看法是，你真正从资产作为基础层开始——NFT 是你的 IP 、知识产权的基础层。与其考虑游戏，然后考虑可以放什么资产，不如考虑基础资产本身是什么？让我们以 Yuga Labs 为例。你有BAYC，然后从那里空投获得MAYC。所以，你实际上是从这些资产中一个一个地创建你的世界。当你拥有基础资产时，你会想，为我的社区创造三个月的游戏体验、六个月的游戏体验或两年的游戏体验。这意味着，所有的这些都使用同样的基础资产层来奖励你的社区，不仅有不同的体验，而且还有不同的资产，他们可以通过持有你的基础资产来获得。

Jon：我发现 Yuga Labs 以及他们在ApeCoin上所做的事情真的很吸引人，因为我认为这是自下而上构建游戏的一个很好的例子。他们基本上给了社区一组原语，即 NFT 本身，并告诉他们，“嘿，我们想围绕这个建立一个宇宙。” 然后他们把商业权利，给了NFT持有人。您如何将其与传统的游戏开发模式相协调？当你有一个中心化的游戏工作室，像 Sky Mavis 这样的组织，确保游戏玩起来有趣，没有作弊者等等，你是否觉得这些方法是不一致的？它们是如何调和的？

Aleks：从某种意义上说，Sky Mavis 方法实际上是混合的。当我在看 Yuga Labs 时，他们在做什么？他们正在赠送在各种事情上使用 BAYC 的商业权利。这很棒——你可以释放创造力。作为游戏开发者，我们正在尝试逐步去中心化。在 Axie，用户可以获得可观的货币价值，但如果他们想更深入地参与生态系统，他们需要与我们达成协议。在我们的案例中，我们正在努力使激励措施与持有基础代币（如 AXS）的人保持一致。所产生的收入要么返还给玩家，要么进入我们的国库，然后使代币持有者受益。

现在，对于 ApeCoin，我对此有更强烈的看法，因为我不看好货币代币。我相信代币需要有一种清晰的方式让人们捕获某种价值，无论是资金流还是管理平台，仅将它们用于贸易并不是我认为会使价格上涨的事情，这实际上是很多事情的意义所在。我宁愿专注于，特别是如果你要发行代币，人们为什么要持有它？让人们想要长期持有您的资产，而不仅仅是将其作为货币。

Gabby：制作游戏真的很难，但是 NFT 和 Web3 社区赋予了很多东西。我最喜欢的例子之一是 RTFKT 和 CloneX，它们刚开始是一堆 NFT。但随后他们又推出了另一个 NFT，Space Pods，它就像一个人们可以装饰的物理 3D 空间。很多人实际上是作为创作者进入的，并且正在装饰自己的 pod，甚至学习 Blender 来这样做。因此，让您的社区感觉，“我是这个经济体的真正所有者，我可以与开发人员共同创造。” 当然，你可以在那里设置一些界限。有一种只有开发者才能创造的核心游戏体验。

Jon：我们今天在 Web3 中面临的挑战之一是没有那么多玩家。举一个个人例子，两天前我通关了艾尔登法环，这是一款畅销的3A主机游戏。他们最近宣布上个月有 1200 万人玩艾尔登法环。不过，据我所知，目前还没有一款区块链游戏累计有 1200 万玩家，更不用说每月活跃玩家了。您认为要在区块链游戏中获得艾尔登法环这种级别的成功需要做什么？

Aleks：目前在 Axie，我们每天有大约 160 万活跃玩家，这还是在没有任何应用商店上架的游戏。你不能在iOS上下载游戏，在 Android 上获取它的唯一方法是直接下载 APK。话虽如此，当我们可以进一步降低进入壁垒时会发生什么？作为一名开发者，这并不是为了赚很多钱，而是要给予一些回报，因为游戏玩家永远被认为是理所当然的，实际上是被剥削的。他们被压榨到了极致，尤其是被手机游戏工作室压榨到了极致。

Aleks：作为一名移动手机游戏开发者，我们沿着这条路走下去，发现实际上最糟糕的并不是游戏开发者，是中间商。是您必须花钱买广告的方式。现在，链游的可能之处在于，你可以通过投放代币直接向消费者推销，这也是 Axie Infinity 能够在如此短的时间内获得如此多用户的原因。到去年年底，我们从 36,000 人增加到现在的 250 万人。通过进一步减少障碍，让人们可以免费玩游戏，这就是达到这些数字所需要的。

Gabby：我们在 Axie 社区的基础上创建了 YGG。社区发现，基本上，通过这种双重登录系统，你实际上可以将 NFT 借给其他人，而无需放弃钱包访问权限。如果我让你访问用户名和密码，而不让你访问我的区块链钱包，你就可以玩游戏、体验游戏并赚取价值，而无需信任拥有你资产的人。因此，我们从 2020 年开始就这样做了，现在这在菲律宾和世界其他地区的 Axie Infinity 中已经真正爆发了。我们是作为玩家社区建立的，或者作为 DAO（去中心化自治组织）运行。现在正在购买不同类型游戏的资产，我们现在已经购买了超过 40 款游戏的资产。无论是卡牌游戏，或策略，或 RPG，我们现在拥有让玩家进入并享受这些游戏的资产，并且无需购买 NFT 即可开始游戏。当然，如果他们非常喜欢它，那么他们可能会在之后购买一些东西。

可以看到 YGG 已经取得了巨大的增长，其中很多是在东南亚、印度、拉丁美洲等发展中地区，人们在这些地区玩这些游戏并赚取额外收入。但我认为我们将看到类似于拥有银行账户的魔兽世界公会的演变。公会本身可能会购买一座城堡或一个王国，它会生产建筑物。人们会进来并收获资源。因此，在数字资产之上拥有这种公会层，为您提供了前所未有的新游戏原语。

Jon： Gabby，关于 DAO 的话题，也许只是一个有争议的问题……今天的 DAO 基本上已经成为 Web3 游戏发行商了吗？如果您正在开发一款新的 Web3 游戏，您希望 YGG 查看代币设计，查看游戏设计文档并在其上签字。这样一来，您的社区、您的学者和奖学金管理人员都会知道这是一个合法的项目，他们应该投入时间和金钱。你认为从这个角度来看，DAO 真的变得太强大了吗？因为他们实际上已经成为未来 Web3 游戏和一般 dApp 的守门人，甚至在游戏之外。在您构建公司并发展壮大时，您如何看待这一点？

Gabby：这是一个非常好的问题。我认为在 Web3 中赢得和创造价值的方式与以前大不相同。以前，如果你有一些专有的东西，你想把它关起来并收费，这就是为什么 Facebook 和谷歌在他们的算法之上发展得如此之大。在 Web3 中，你的成长方式是通过复合网络效应，这意味着如果 YGG 加入并帮助游戏发布、帮助代币设计和购买资产，我们不会把关，也不会让其他公会进来。我们实际上是在设置设计，以便数百个其他公会可以进来，享受游戏，并拥有繁荣的经济。我们实际上通过把关减少了网络效应，从而降低了我们所做工作的价值。这确实是 Web2 和 Web3 在创造价值方面的核心区别。

Jon：还有最后一个问题是，观众中有很多 Web2 开发人员对加密货币感到好奇，并且想了解更多关于 Web3 的信息。您对他们有什么一般性建议？他们应该如何开始？

Aleks：说实话，NFT 是可以用作构建模块的基本要素。如果你做对了，你能获得不少经验。但是你需要最基础开始，在你准备好之前，不要卷入所有围绕融资的炒作。

Gabby：了解 Web2 和 Web3 中社区价值的差异，对于创建游戏至关重要。社区在游戏中一直很有价值，对吧？但在 Web3 中，它们实际上可以成就或破坏您的游戏。所以，加入一个 DAO、加入社区、加入一个公会，并对一些提案进行投票，或者从 DAO 的角度体验游戏，这真的让您了解玩家社区拥有的数字资产网络效应有多么强大。

Jon：谢谢大家加入我们。
