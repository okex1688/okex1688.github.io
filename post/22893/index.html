随着 NFT 走入主流，越来越多的艺术家、项目方都开始加入到 NFT 领域之中。自 2021 年开年以来，最大的 NFT 交易平台 OpenSea 的月度交易量呈爆发式增长，其中，Meebits、Bored Ape Yacht Club、Ghxsts 等新项目的火爆是推动交易量大幅增长的主要原因之一。

不过，需要注意的是，NFT 项目层出不穷，但并非每一个项目都能迎来成功，绝大多数项目的结局都是一级市场发售结束后便无人问津，缺乏长期生命力。因此，在我们购买 NFT 时便需要 DYOR，做好自己的研究。

本文为 Bankless 作者 William M. Peaster 撰写的关于如何对 NFT 进行 DYOR 的文章，旨在帮助每一个人找到更优质的 NFT 项目、避免踩坑。

律动 BlockBeats 将全文翻译如下。

「Do your own research（做好你自己的研究）」通常也被人们叫做「DYOR」。

这个四个字母组成的缩略词常常被人们用在 DeFi 领域，加密经济就好比是一片荒蛮的西部地区，而最终你只能依靠你自己去研究那些加密领域中的项目，了解清楚他们的来龙去脉之后，再把你的资金投入其中。

最近，我刷到了交易员 Alex Krüge 发布的一条推文，其中就写到了他列出来的一份关于在涉及 DeFi 代币时如何 DYOR 的操作流程，具体如下：

近期，我一直在研究 DeFi 领域以及 NFT 生态系统，而这些步骤可以帮助我深入研究 DeFi 项目，帮了大忙。而且，我的兴趣也被激发了出来，因为我认为我们也可以将 Alex Krüge 的建议用于研究那些 NFT 项目。于是为了方便大家研究 NFT 项目，我将上述建议清单做了些许改动。

而且我需要补充的一点是，这些研究步骤可以帮助大家了解新诞生的 NFT 项目，这是更有成效且这也是必要的，此外，对于那些诞生时间较长的项目，大家也可以用这个模型来对其进行研究。

具体操作步骤

（1）查看项目的推特和网站。

对于加密领域的项目而言，推特是兵家必争之地。无关新老，几乎所有优质的项目都会用心经营它们推特，而且通过用心打理推特账户，它们就会更容易对接到一些更优质的资源。另外，你还需要去看一下它们的官网，在大多数情况下，你与 DApp 的交互都是在官网上进行的，所以你需要仔细查看。

（2）分析项目团队。

一些 NFT 项目会公开自己的团队成员名单，但是也有一些会保持匿名，甚至还有些项目方会公开一部分、匿名一部分。对于团队的分析没有什么参考标准，主要取决于你是否熟悉这些人、他们信誉如何、之前做过那些项目等等。

（3）确定 NFT 的上链程度。

你需要了解这个项目的 NFT 的存储方式。是完全上链存储，还是通过 IPFS、Arweave 等进行去中心化存储，还是完全在链下存储其元数据。这关系到你的 NFT 的安全问题，若一个项目采用的是完全链下存储，你想收藏就只能去信任项目方不会做恶，而这是十分危险的。

（4）翻看项目 Discord 和 Telegram 的聊天记录。

一般而言，NFT 项目都会建立一个 Discord 或 Telegram 群组，而当你想了解这个项目时，加入到这些群组当中是一个绝妙的方法，在群组中你便可以感受得到整个项目社区的脉搏、情绪与活力。而且，这些社区也是之后进行二级市场交易磋商的好地方。

（5）关注其他 NFT 持有者的言论。

虽然你要知道的是，一个人自己的观点可能是片面的，你不必过于依赖某个人的看法，也不必过分神话某一个人，不过为了更全面地了解项目情况，与这些 NFT 持有者进行简单的沟通或者关注他们的看法是十分有必要的。你可以通过推特的标签或者 Reddit 进行搜寻。

（6）分析链上活动。

你可以使用 OpenSea 的排名页和 NonFungible 的市场历史中心等工具来追踪 NFT 项目的交易活动，你可以通过这些数据了解这些项目在近期和历史上的表现。

此外，如果你正在研究的 NFT 项目有自己的原生代币，你可以通过以下步骤更好地了解以 NFT 为中心的资产的基本情况。

（1）在 CoinGecko 中搜索代币。

CoinGecko 是一个很棒的加密货币市场数据仪表板，可以很容易地找到关于代币的各种基本指标。此外，它还十分关注新项目的代币，所以你想了解的代币基本都可以在这个网站找到。

（2）研究市值、完全流通市值和交易量。

市值指的是一个项目正在流通的代币价值，完全流通市值是指项目所有代币完全流通后的总价值，交易量是指在任何一天、一周、一个月等时间内通过这些资产交易的资金量。这些是你了解任何加密资产时都需要关注的一些核心指标。

（3）研究代币经济和预售等信息。

NFT 项目的原生代币能在 NFT 生态系统中做些什么事情？质押、治理或是其他的吗？有多少投资者已经投资购买了代币、他们的成本是多少？这些问题可以帮助你更好地衡量你是否应该将你的资金投入到这个项目中。

总结

你需要做好自己的研究，在这片充满未知宝藏与危险的水域中找到最正确的道路。好消息是，上述信息基本上都可以在网络中免费搜索到，而每一个人都可以通过深入探索并分享自己的知识成为一名一流的 NFT 研究者。
