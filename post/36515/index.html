近些天，一直在寻找公链上的社交应用协议，可以很明显发现的是，web3的社交已经脱胎于web2的形态，以公链上的数据为核心要素去设计。

本文中，笔者整理了名为Relation社交图谱协议的相关资料以及其应用Relation ONE的设计方式。

整体看来，Relation协议主要以社交图谱为切入点，其认为社交图谱是社交平台中最关键的数据，社交图谱是一种关联数据，也是一种索引数据，这种数据，只要将其所有权保留在用户手中，聚集的数据就能成为有价值的资产。当一些需求方使用数据时，可以将数据变现，当数据的价值通过NFT集成时，也可能表现出其他市场价值。

以下为项目资料整理。

当用户登录Facebook、Twitter等社交平台时，在平台上的活动会形成了一个社交图谱，成为社交巨头拥有的盈利数据资产。在 Web 3 时代，这些数据的所有权和效用正在向用户转移。

尽管去中心化社交的未来充满希望，但当前的 Web3 DApps很难像 Web2 应用程序一样的大规模增长。一方面，绝大多数 DApp 没有足够的成本和精力专门开发一套社交网络智能合约，这使得这些应用程序的用户停留在 Discord 和 Telegram，而不是停留在其网页中。另一方面，由于去中心化和用户匿名的特点，DeFi、NFT、GameFi等当前项目的社交属性并不重要。这些 DApp 没有 Web2 应用积累的丰富生态资源，也没有大量中心化的用户数据或应用商店可以依赖。它还导致加密用户之间缺乏足够的互动作为增长策略。

Relation Labs 正在引入解决这些问题的产品。包含Relation Graph、Relation Link Tool Suite和Relation One。

Relation Graph

Relation Graph 是一个用 Rust 编写并部署在 WASM 上的图数据库，是 Relation 的底层核心引擎。作为Relation Social Graph构建的基础，提供了图存储、图查询、图计算、图分析、数据库基础技术能力。它将扩展用于图形可视化和图形神经网络的技术模块。

具体来说，Relation Graph 支持将 Web3 多链生态系统中的多维数据以非结构化的方式组织起来，方便数据聚合查询、关联分析和深度挖掘，有效提升 Web3 应用程序使用数据的能力。特别是支持类SQL的SPARQL查询语言，方便数据工程师进行低门槛、高效的数据分析，可以根据业务需求编写复杂复杂的查询代码，从Relation Graph中挖掘数据的价值。Relation Graph 在 N 度关系查询中的效率提升是关系数据库的十倍以上。

Relation Link Tool Suite

Relation Link Tool Suite允许开发人员无需太多成本和时间即可创建新的社交图谱，提供了一种增长策略。

工具套件为每条链上的 DApp提供开源的Relation Graph 访问 API、Relation Dashboard API、模块化社交功能组件和多链智能合约钱包解决方案。特别是可以访问关系图的开发者或 DApp 将能够通过内置的好友关系协议创建和管理他们的链上联系人，从而增加社交属性。同时，开发者还可以在授权数据中进行源代码查询，获取数据关系结果。

开发者还可以查询Relation全网公共数据，包括数据控制者、Gas消耗、DataSpace创建数量、日志安全监控等信息。此外，Tool Suite 允许在 Data Space 上存储和管理多个私钥，只需一个用户身份权限进行签名，简化了为用户导入大量私钥的繁琐操作，并实现了具有多种不同身份验证和身份的多链钱包解决方案。通过统一的认证方式进行授权。

Relation One

Relation 通过其一站式社交应用 Relation One 打破中心化的社交图谱困境，为用户构建个性化的可视化 Web3 社交图谱，并将数据的所有权和隐私权交还给用户。用户还能够从自主数据贡献中获得收入。

通过 Relation One 添加 Metamask、Internet Identity 或 Polkadot 等地址后，用户可以通过 Relation ID、用户名和链上地址搜索添加好友。用户将从系统中获得好友推荐。用户可以体验个人Profile管理、链上身份管理、P2P聊天、群聊、DAO、NFT头像展示、好友邀请、私域游戏排名等社交功能。

Relation One 会为每个用户生成一个数据空间来存储他们唯一的私钥，以保护用户数据隐私。同时，Relation 目前正在为每位用户补贴 1T Gas Fee，以创建个人数据空间，实现零成本交互。此外，Relation One 已经支持 MetaMask、Internet Identity、Polkadot 等钱包应用，可以快速登录，在不同区块链之间共享社交数据。未来它还将与 Solona、Near、Cosmos、Polygon、Ceramic 和 Arweave 兼容，以便任何链上的加密用户都可以使用他们的社交网络。

此外，Relation One 还会同步所有用户在不同应用程序上的社交网络，让他们在各个感兴趣的领域、其他平台等上结识志同道合的人。即使相关平台关闭，他们也不会拥有担心丢失他们创建的内容或他们的朋友。对于 DApp，Relation One 模块可以更直观地了解用户的兴趣，并允许他们更准确地接触相关群体。

为改变过去社交巨头单方面剥削数据价值的局面，Relation One 未来计划引入经济模型和数据变现方案，以激励用户通过开放许可向应用程序或相关利益相关者提供数据价值的意愿。
