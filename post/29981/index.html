NFT

“不可替代/非同质化”一词在经济学中通常用于表示拥有独特的对象，并描述无法被其他对象替代的事物，因为它们具有一组独特的属性。

作为记账单位的“代币”是由智能合约的计算机算法控制的分布式区块链中的账户，其中记录了代币持有者账户上的余额值，从而可以将它们从一个钱包转移到另一个。

不可替代/非同质化代币 (NFT)是指代表数字文件、图像、音频、视频、视频游戏收藏品和其他创意产品的加密代币。与要求所有代币相同的加密货币不同，每个 NFT 都是独一无二的或限量的。

为简化起见，在每个区块链中，所有代币都是平等且可互换的（例如，所有比特币都是相同的，一个比特币可以换成另一个，并且什么都不会改变）。其中，由于 NFT 的出现，每个代币都无法被伪造、分裂或替换。这项技术非常适合保护独特事物的权利，即艺术品、电脑游戏的虚拟对象，甚至房地产。

标准

NFT最初是作为特定的代币标准创建的，以支持在计算机游戏中使用区块链，其中包括用于广受好评的 CryptoKitties 游戏的以太坊 ERC-721 标准和更新的 ERC-1155 标准。

ERC-721 是第一个在以太坊区块链上表示不可替代数字资产的标准。ERC-1155 标准提供了“半可替代性”，还提供了一组扩展的 ERC-721 功能（因此，ERC-721 可以使用 ERC-1155 构建）。

与用唯一标识符代表一项资产的 ERC-721 不同，ERC-1155 代币的唯一标识符代表一整类可替代资产，用户可以将任意数量的资产转让给其他人。

基于ERC-998标准的组件是模板，根据这些模板，NTF既可以是不可替代的资产，也可以是可替代的资产。

虽然大多数 NFT 活动传统上发生在以太坊区块链上，但其他 NFT 解决方案开始出现在不同的区块链上。例如，由 Mythical Games 团队创建的 DGoods 项目专注于为多个区块链提供多功能标准，从 EOS 开始。Cosmos 项目正在开发一个 NFT 模块，该模块可用作 Cosmos SDK 的一部分。在由 Cryptokitties 团队创建的 Flow 区块链中，Cadence 编程语言将每个 NFT 呈现为用户存储在其帐户中的对象。它具有类型系统提供的强大所有权规则。最重要的是，像比特币现金、NEO、Tron、WAX 和 Tezos 这样的区块链也有决定其用途的代币标准。

优点与缺点

NFT 的主要好处与近年来分布式账本技术 (DLT) 提供的一切密切相关。区块链在技术上确保了 NFT 的真实性，并保证直接向内容创建者付款。

然而，尽管 NFT 解决方案具有明显的优势，但在其出现的同时，也很难不关注行业中形成的几个问题。

苏富比的分析师强调了以下与NFT和加密艺术的快速扩散相关的技术缺陷：

  • 难以确定 NFT 的价值；

  • 难以评估加密艺术的独特性，也难以了解存在多少副本；

  • 很难找到 NFT 的原始所有者。

关于 NFT 销售，版权保护问题越来越频繁地出现。上传内容以打开非策划的 NFT 平台时，你会不停地对这一事实感到好奇——它可以代表您出售任何媒体文件（无论是几秒钟前拍摄的自拍照还是在互联网上找到的图片）。许多骗子利用了这一点。一个典型的例子是最近发生的让-米歇尔·巴斯奎亚特 (Jean-Michel Basquia t)的 NFT 绘画事件，该绘画于 2021 年 4 月拍卖，两天后在平台发现卖家不拥有它后才将其移除。

所有这些问题仍然存在。研究人员和工程师尚且没有找到解决方案。

历史

NFT 的原型是Colored coins——2012年在比特币网络上创建的实验资产。佩佩青蛙角色的插图也是首批不可替代的比特币令牌之一。其中很多都在 eBay 上出售，而且稀有的Rare Pepes 系列后来在纽约拍卖。

所有 NFT 图像最初都托管在Counterparty 上，这是一个支持资产创建和交易的点对点金融平台，拥有去中心化交易所和加密代币。然而，在 21世纪10年代末，随着类似平台的出现，Counterparty 失去了托管 NFT 的垄断地位。

基于以太坊的 NFT 先驱是CryptoPunks项目。该系列包含 10,000 张像素化的朋克图片，每张图片都具有一组独特的特征。使用流行的钱包（例如 MetaMask）购买的每个朋克，都显着降低了与 NFT 的交互障碍。奇怪的是，CryptoPunks 不符合 ERC721 标准，因为它尚未发明，代表 ERC721 和 ERC20 的混合体。今天，鉴于其有限的供应和在加密社区中的成熟品牌形象，CryptoPunks 可能被视为主流数字古董。

值得一说的是，CryptoPunks 首要地位的问题仍然开放，如在2015年，在这个项目出现在以太坊的两年前，有人创造了Etheria游戏，用户在里面买六角地块的NFT虚拟土地。

2017 年底，允许用户繁殖数字猫的在线游戏CryptoKitties获得了显着认可，为大规模采用开辟了道路。该游戏运行在五个以太坊智能合约上，用户可以通过他们的以太坊地址轻松地与其进行交互。该游戏非常直观，非常简单，非常受欢迎，在发布后不久就占了整个以太坊网络流量的 25% 左右。该项目很快筹集了 1250 万美元的投资，由于炒作，一些小猫的售价已超过 10 万美元。

CryptoKitties 成为首次在区块链基础上部署大规模非金融应用程序的一个例子。在那一刻，加密社区意识到了 NFT 的潜力，其供应量显着增加。在CryptoKitties引起的兴奋结束之后，现有项目的受众增加了，在接下来的几年（2018 年和 2019 年）出现了 100 多个 NFT 项目。NFT 的格局开始以前所未有的热情发展，以至于甚至Cryptotitties项目也出现了。

2021 年：第 1 部分

2021 年，NFT 生态系统继续以前所未有的速度增长。根据NonFungible.com的 NFT 行业统计报告， 2020 年初NFT市场超过 2.5 亿美元，比之前的 2019 年增长近 300%。除此之外，在几个月内，加密货币投资者纷纷涌入9000 万美元用于 NFT 和数字收藏品，导致数字市场的整体市值增长了1785%，从 2021 年初的 2300 万美元增加到 3 月底的 4.32 亿美元。

在所有 NFT 细分市场中，虚拟世界和艺术品占据了大部分。尽管如此，电脑游戏的交易数量仍然领先（2020 年近 50% 的 NFT 交易与它们相关），收藏品也不甘落后。

最常用的 NFT 市场是一个通用的OpenSea平台，允许用户铸造和交易 NFT，包括 ERC721 和 ERC1155 标准，从网页能够查到它们的数据并检查统计数据。创建于 2017 年，今天，它提供了 300 多种类型的资产和超过 10,000,000 个铸造的代币。几乎所有的加密集合都可供用户使用，包括CryptoPunks和Hashmasks，以及来自许多流行区块链游戏的项目，例如SoRare、CryptoKitties、Axie Infinity、Gods Unchained等等。2021 年 6 月，OpenSea 的销售额创下了约 1.5 亿美元的历史新高。

另一个流行的购买、销售和创建 NFT 的市场是Mintbase，它擅长涵盖音乐、摄影和新闻等利基 NFT 类别。

Rarible和Zora平台在加密艺术的粉丝和艺术家本身中赢得了巨大的成功，因为它们没有被策划。要将 art-NFT 上传到它们上面，用户只需要注册并支付交易费用即可铸造代币。与他们不同的是，在SuperRare、Nifty Gateway、MakersPlace或Known Origin等策划服务上，要开始销售数字艺术品，内容创作者需要提交申请表，该申请表具有相当严格的选择标准和漫长的等待专家决定的时间，这通常会吓跑 NFT 的创建者。

除此之外，许多数字艺术作品已经被放置在虚拟世界的在线收藏中。

Cryptovoxels是第一个基于区块链的虚拟世界，拥有多个艺术画廊，用户可以在其中以 NFT 的形式购买艺术品。随着虚拟现实空间变得越来越流行，在其中展示数字艺术对加密社区来说变得更加普遍和令人兴奋。

专家认为，2021 年已成为艺术 NFT 的繁荣年。仅在最初的几个月，销售额惊人。例如，2 月份，加拿大歌手格莱姆斯在 Nifty Gateway 平台上以约 600 万美元的价格售出了 10 幅数字画作。当月晚些时候，带有 Nyan Cat meme 的 NFT 售价略低于 600,000 美元。

“Everydays: The First 5000 Days”是一幅JPEG格式的数字图像，分辨率为21069 × 21069像素，是美国艺术家Mike Winkelmann (Beeple)创作的《每日》系列图像的拼贴图。

最引人注目的事件是 Beeple 的作品“每一天：前 5000 天”的拍卖，这是在著名的佳士得拍卖的第一件 NFT 艺术品。它以 6930 万美元的价格售出，使 Winkelmann 创造的这个 NFT成为整个加密艺术史上最昂贵的NFT 。

在 Beeple 作品创纪录的拍卖之后，拍卖行宣布在 5 月初以 1690 万美元的价格拍卖了 9 件 CryptoPunk 。这是 NFT 首次与安迪·沃霍尔 (Andy Warhol) 和让-米歇尔·巴斯奎特 (Jean-Michel Basquiat) 的作品一起提供。

所有这些事件和许多其他事件都引起了广泛的受众和潜在的新内容创作者的共鸣，他们的数量在 2021 年的头几个月呈指数增长。

2021 年：第 2 部分

尽管人们对非功能性金融领域的兴趣很淡，但他们的市场仍在增长。销量一直保持在高水平，并继续打破记录。

2021 年的前六个月，NFT 的销售额达到 25 亿美元（根据DappRadar 的数据）。同时，这个数字被大大低估了，因为统计来源只跟踪链上销售，即在区块链上，没有考虑 NFT 的大量链下销售，例如拍卖。NonFungible.com的数字是 13 亿美元，不包括大约 80 亿美元的 DeFi NFT。

DappRadar是一个追踪多个区块链上的NFT交易的公司，这些区块链包括以太坊、Flow、Wax和BSC

主要交易所的 NFT 每周交易量从年初增长到 2 月份的峰值，5 月份下降 82% 至 3530 万美元。然而，自那以后，NFT 活动有所增长，6 月最后一周的交易量达到 7800 万美元，比上个月增加了 120%。

这种反弹可归因于玩家们对NFT生态系统的重燃兴趣。到目前为止，NFT 解决方案的创建者已经学会了如何与社区互动并在他们的 NFT 平台上投资激励措施，以适当地激励受众并培养本土传道者。

现在，与NFTs相关的新闻继续在加密货币和艺术社区以大致相当的兴奋程度引起狂热的复兴。

就最新的标志性拍卖而言，苏富比拍卖行以 1290 万美元的价格售出了 Banksy 的《Love is in the Air》，这标志着首次组织拍卖可以用加密货币支付的实物艺术品。

苏富比成立于 1744 年，以约 1230 万美元的加密货币出售了这颗 101.38 克拉的稀有钻石。交易于 7 月 9 日在香港进行。拍卖接受传统货币、比特币和以太坊的投注。不过拍卖行未指定有关买方和销售加密货币的信息。

这些事件对于全球艺术界采用加密货币至关重要。

6 月下旬，媒体巨头Twitter通过 Rarible 平台分发了 140 个 NFT，其中包含各种 Twitter 徽标图像和虚拟 Tamagotchi 宠物，支持并延续了这一趋势。

值得一提的是，The Bored Ape Yacht Club，一套 10,000 个独特的数字猿 NFT，最近成为收藏家的热门话题。它们的总销售额跃升至 6100 万美元，而 7 月份 OpenSea 上的平均售价超过 3500 美元，比 4 月份的发布价格上涨了 1500%。

作为 NFT 技术的另一个潜在用例，音乐也为将艺术家的创造力货币化提供了机会。 2021 年春天，贾斯汀·布劳 (3LAU) 成为世界上第一位将其音乐专辑代币化并以 1160 万美元出售的艺术家。与此同时，美国乐队Kings of Leon 发行了他们的第八张录音室专辑，也以NFT的形式，以200 万美元价格出售。Steve Aoki、Linkin Park 的 Mike Shinoda 和 Deadmau5 等艺术家以他们自己的例子来支持围绕 NFT 技术优势的狂热。

除了视觉艺术和音乐，视频 NFT 也越来越受欢迎。例如，最近，CNN 以 NFT 的形式出售历史新闻。其中包括独家资料、总统选举结果、太空旅行等重大事件。此外，以安东尼霍普金斯为主角的新长片《零接触》将仅以NFT的形式在专业市场 NFT Vuele 上分发。只有在购买相应的代币后才能访问它。

加密游戏的受欢迎程度正在稳步上升。在此类游戏中，加密货币和 NFT 通常被用作玩家交易和交换的游戏内物品。这些游戏中最著名的可能是Gods Unchained，这是一种基于以太坊的纸牌游戏，其中每张可收藏的纸牌都由智能合约确认。

另一个流行的加密游戏是Axie Infinity。加密社区记得其中一名玩家以 150 万美元购买土地。具体来说，这片土地由九个连续的创世块组成——游戏中最有价值的土地。价格如此之高，因为只有220个这样的包裹。随着越来越多的玩家加入游戏，这种有限数量的碎片的成本可能会更高。

与 NFT 相关的活动远不止于此。他们创作的变化永远不会停止。

6 月初，英国计算机科学家、万维网的发明者蒂姆·伯纳斯-李 (Tim Berners-Lee)拍卖了一些源代码作为NFT。一位匿名的收藏家以 540 万美元的价格买下了它。

与此同时，生物学先驱 George Church 和他创立的公司 Nebula Genomics宣布他们打算将 Church 的基因组作为 NFT 出售。这个想法在科学界引发了争议，因为剑桥大学的遗传学家Church（丘奇）因其有争议的提议而在学术界广为人知，其中包括创建基于 DNA 的约会应用程序以及复活长毛象。

提到复活就顺便提几句：今年夏天，一个基于以太坊的虚拟去中心化墓地项目Gone to Mars启动。1089 个 NFT 胶囊已经准备出售，等待未来的主人。密封到 2050 年，它们将与持有者的信息一起传递给下一代。

自 2020 年以来，将 NFT 技术的优点与去中心化金融的功能相结合的优势出现了。例如，在Aavegotchi游戏中，每个角色都是存储在 Aave 借贷平台上的用户的抵押品，在NIFTEX上可以借入和借出 NFT，以保持部分所有权，在NFTfi 上，不可替代的代币在获得时可以用作抵押品贷款。这些只是几个例子。

由于 Berners-Lee 的原始发明，数字内容的富有想象力的表示形式以及不可替代的代币用例的极限是无限的。NFT 存在于时尚、艺术、摄影、金融等领域。

今天，NFTs 生态系统继续发展和繁荣。上面列出的许多项目已经成功地使用了区块链技术很长时间，已经拥有了大量的创意人力资本、收藏家和忠实的观众。

与此同时，越来越多的内容创作者正在发现将他们的创造力货币化的可能性。区块链技术和 NFT 正逐渐被相关领域的影响者和主要参与者所认可。群众一直对这个领域表现出兴趣。据NonFungible 称，每周有 10 到 2 万人有兴趣购买 NFT。基于美丽和原创想法的项目不断出现，为在科学和工业的边界使用不可替代的代币技术开辟了新的可能性。

在 2021 年初的繁荣之后，有一件事仍然很清楚，曾经看似小众的 NFT 生态系统从小型加密社区涌现，成为金融、艺术、游戏和许多其他领先创意领域的平等参与者行业。
