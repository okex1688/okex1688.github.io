DeFi热点

1.Uniswap周交易量达到100亿美元历史新高

金色财经报道，去中心化交易所Uniswap在周二创造了新的每周交易量记录，累计交易额超过100亿美元。

2. Uniswap V3 版核心和外围合约已部署到以太坊主要测试网

据官方消息，Uniswap 表示Uniswap V3 版本的核心和偏外围（periphery）合约现已部署到主要的以太坊测试网，包括 Ropsten、Rinkeby、Kovan 和 Goerli。Uniswap 智能合约代码由两个 github 项目组成。一个是 core，一个是 periphery。Uniswap 表示，这些合约地址非最终地址，当对外围存储库进行最终更新时，将会被更改。

3.Uniswap创始人：V1发布后被批评不会起作用，因为被动的LP将被主动的做市商超越

Uniswap创始人Hayden Adams对此表示，在发布Uniswap V1后几个月里，收到的主要批评是，它永远不会起作用，因为被动的LP将被主动的做市商超越。

4.DeFi基准利率跌至8.33%

金色财经报道，据同伴客数据显示，04月21日DeFi去中心化金融基准利率为8.33%，较前一日下跌7.17%。同期美国国债抵押回购率（Repo Rate）为0.02%，二者利率差为8.31%。 DeFi基准利率代表了DeFi融资难易程度，利率越高说明融资成本越高，利率越低说明融资成本越低。其与Repo Rate的利率差则便于DeFi与传统市场作进行同类比较。

5.KyberSwap 团队开发的一站式 DeFi 钱包 Krystal 已上线移动端应用

4月21日消息，由 KyberSwap 团队开发的一站式 DeFi 钱包工具 Krystal 已正式上线移动版应用，用户可下载 iOS 和安卓版，未来几周还将推出网页版。 目前 Krystal 已集成 Uniswap 和 Kyber Network 以支持交易，后续将增加更多 DEX 协议，包括 SushiSwap、聚合器等。另外 Krystal 还集成了货币市场协议 Aave 和 Compound，为用户提供赚取利息的功能。

6.DeFiBox宣布与BitKeep钱包达成战略合作

据官方消息，近日，一站式DeFi资产管理及收益追踪平台DeFiBox宣布与去中心化多链数字钱包BitKeep达成战略合作，用户可直接通过BitKeep钱包查看多链DeFi项目投资情况并追踪挖矿收益。双方将在DeFi生态及数据等方面展开更深层次合作。 BitKeep是一款去中心化多链数字钱包，致力为于全球用户提供安全简单的一站式数字资产管理服务。同时支持46+主流公链、22000+币种资产、3500+DApps应用，目前用户数已突破347万。 DeFiBox是一站式DeFi资产管理和信息聚合平台, 其独具特色的资产看板模块可协助用户实时追踪多链DeFi项目收益，优化投资组合，目前已支持以太坊、Heco、BSC和OKEx Chain公链专区。

7.Chocoswap即将推出“香草庄园”并向社区馈赠2300枚CHOCO

据官方消息，全生态跨链聚合交易平台Chocoswap于4月21日推出权益性代币CHOCO，并将推出集NFT、盲盒等多种衍生玩法的“香草庄园”。CHOCO将是“香草庄园”的重要入场券之一。同时，当Chocoswap聚合交易及跨链聚合交易功能上线后，根据Choco 持仓数量进行手续费折扣分级，持仓越多，折扣越大。 CHOCO没有任何预售、没有任何投资机构，全部通过社区回馈和流动性挖矿发行，初始定价10美元，将面向Chocoswap社区馈赠2000枚，并于近日启动流动性挖矿。同时，Chocoswap和Codebank携手，面向社区送出300枚CHOCO。

NFT

1.Clubhouse Media计划推出代币化数字艺术品和NFT内容之家

Clubhouse Media Group新闻稿称，该公司打算建立一个新的内容之家，聚集顶尖的艺术家、加密理财专家、行业影响者、加密投资者和NFT爱好者。Clubhouse Media计划对创作者制作的内容进行代币化，在平台上销售，同时对数字艺术品进行代币化，以备出售。

2.孙宇晨：NFT并不是泡沫，是新的艺术流派，是概念性的革命

据最新消息，波场TRON创始人兼BitTorrent CEO孙宇晨正在火星财经进行视频直播，解析NFT行业发展机遇，他在直播中表示：“NFT并不是泡沫，是新的艺术流派，NFT看的人越多，越有价值，这是艺术品革命，概念性的革命。我创办的JUST NFT基金将只专注于顶级艺术家与艺术品，原则上JUST NFT基金收藏的艺术品单价不低于100万美金，中位数价格在1000万美金左右。 NFT领域艺术品交易市场良莠不齐，频被诟病，JUST NFT基金设置入选艺术品门槛，一方面是基于优化市场秩序的考量，另一方面，我始终坚信，艺术领域的价值来自于头部效应，只有金字塔顶尖的艺术家才具有超越时间的价值。我们不完全拒绝其他类别的NFT，但我们始终以艺术品本身的价值作为考量标准，我们欢迎更多优质的作品入驻，百花齐放才是春。”

3.奥运季军具滋哲发起NFT项目，将于4月23日在opensea.io进行拍卖

4月21日消息，在2012年伦敦奥运会上获得铜牌的韩国足球运动员具滋哲(Koo Ja-Cheol)，通过数字创意公司“ Pinseul Collective”发起了NFT项目，该NFT的创建是为了纪念2012年他夺得的伦敦奥运会铜牌。具滋哲亲自签署了数字签名，该NFT总共发行了4个版本，将从23日晚上7点到26日晚上7点在NFT交易平台“ opensea.io”进行拍卖和销售。

4.NFT游戏《无限舰队》获得720万美元投资

基于比特币的NFT游戏《无限舰队》（Infinite Fleet）迄今为止已经吸引了超过720万美元的总投资。此前1月份消息，Infinite Fleet发行商推出STO，旨在为区块链在线游戏筹集800万美元资金。

5.去中心化流动性协议Tokemak筹集400万美元

去中心化流动性协议Tokemak获得400万美元的投资，由DeFi投资基金Framework Ventures领投，Electric Capital、Coinbase Ventures、North Island Ventures、Delphi Digital和ConsenSys等参投。这笔资金将为Tokemak于2021年第二季度末的推出做准备，其将在以太坊主网上部署其流动性网络。
