5月27日，金沙江创投合伙人朱啸虎参与的Web3.0游戏StepN，宣布将清退中国大陆用户。

据公告内容显示，为积极主动响应相关监管政策，StepN将清查中国大陆帐户，若发现中国大陆地区用户，StepN将依据使用条款对其账户于2022年7月15日（UTC+8）24:00停止提供GPS及IP位置服务。如果用户预期长期会在该地区的GPS或IP位置登陆及使用帐户，官方鼓励用户自行决策处理应用内的资产。在此期间，更多细节将通过官方社交媒体公告、邮件、应用内提示等方式通知使用者。

网传StepN国内团队在紫金港办公室的团队被杭州西湖分局问话，也有传言称，该公司将离开中国大陆。

公告发布之后，该款游戏相关代币Green Satoshi 代币 (GST) 和StepN 代币 (GMT) 立即下跌了两位数 。GST从大约2.08美元跌至1.53美元，跌幅超过 26%。GMT从1.24美元跌至0.83美元，跌幅超30%。

据“中投网”统计，著名投资人朱啸虎买的NFT鞋花了大概2700美金，当初预计3个月才能回本。如今这项清退政策，无疑让回本期限延长，也加剧了该游戏的不确定性。

StepN的经济模型设计“未必是庞氏”

StepN是一款基于Solana ( SOL ) 和 BNB Chain ( BNB ) 的流行Move to Earn“运动即赚钱”游戏，由现在居住在澳大利亚的两名中国移民创建。

其团队参考了比特币的创建者中本聪，也使用匿名，希望成为智能手机用户领先的加密健身游戏。这个游戏，在去年的Solana Ignition黑客马拉松中获得第四名。

锐思此前介绍过的Axie Infinity游戏，是加密世界Play to Earn“游戏即赚钱”的领导者。这个项目激发了新的Web3.0应用程序开发热情，无数新的Web3.0应用程序渴望将赚钱模型应用于各自的生态系统。 

StepN在“游戏即赚钱”的基础上加入了社交模式，编写了一系列程序让其NFT运动鞋的所有者可以边走路边赚钱 。

该游戏是双重代币模式，使用游戏代币GST和治理代币GMT。StepN的玩家必须首先花费SOL代币或者BNB代币购买虚拟的NFT运动鞋，配备NFT运动鞋后，可在户外步行或跑步以赚取GST代币。赚取的代币，可用于升级和铸造新的NFT运动鞋。当不再需要鞋子时，运动鞋可以转售或出租给他人。

在健康和健身领域，这种激励行为并不是一个新概念。成立于2018年的项目SweatCoi，是首批向用户支付加密货币的应用程序之一。 还有其他加密健身应用程序，例如Dotmoovs、Calo、Step还有前几年国内流行的跑步产品“趣步”，每个应用程序都有各自的应用内奖励代币。因此StepN 并不是这种模式的先驱，但它确实是当下最火的Web3.0应用，重振了“运动即赚钱”经济模型。

4月的时候，StepN运动鞋每天可以为用户赚取高达20-30美元的收入，而高级运动鞋可以让用户以GST的当前价格赚取数百美元。 当时在二级交易市场上，最便宜的StepN运动鞋售价约为2千美元，最贵的则能达到近3万美元。

该公司在其官方网站上表示，“StepN旨在推动数百万人走向更健康的生活方式，应对气候变化并将公众连接到Web 3.0。”

在红杉资本、币安和Alameda Research等加密风险投资领域的知名人士的支持下，该公司第一季度的利润为2000万美元 。截至5月底，StepN的累计用户量达到了62万。

朱啸虎也在朋友圈称他购买NFT鞋子的这几万块花得挺值，并称赞了StepN的经济模型设计，觉得“未必是庞氏”，有机会跑通，“值得体验学习一下”。但很快，他又在这条状态下面补充了一条评论，“这样的模式在国内都有监管风险”。

Web3.0没有国界，但政策有

“都要清退了，鞋子的价格从20枚BNB跌到了4枚BNB。”参与StepN的用户小孙说。

小孙是2017年-2018年参与过“趣步”的老玩家。他说，当时的趣步“没什么投入，下载个app就能玩，如果不去买币，就算不赚钱，也不至于赔钱。后来趣步被立案了，变成了卖手表设备的，这个项目也就不了了之。”热爱参与跑步加密项目的他也是StepN的早期用户，今年2月前入的场。并表示，“早期参与的都回本了。”

“但是，大概是四月底的时候，StepN官方看到SOL链涨的太快了，打压了一次价格。那一个星期官方连续更改了四次游戏规则，导致市场很多没了信心。 ”

4月底之前，Mint（铸造）鞋盒只用GST代币，小孙解释道，“鞋盒是生鞋子的盲盒，开出来就是鞋子。”官方为了拉涨GMT代币的价格，把规则改成了必须使用GST+GMT代币才能Mint（铸造）。结果是，GMT代币价格没拉起来，还连带着GST代币一路阴跌。

看到其两种原生代币的价格都在一路下跌，官方又更改了规则。“从两个鞋子 + GST可以生一个鞋子，改成两个鞋子 + GST + GMT才能生一个鞋子出来，然后又改成两个鞋子 + GMT，现在又改回来了还是两个鞋子 + GST+ GMT。”小孙认为，项目方这种朝令夕改的方式，增加了用户对StepN的不信任。

宣布清退大陆用户之后，StepN的市场信心更是骤降。 “前几天买一双鞋子五万块钱，现在地板价一万不到。那么对于五万买鞋子的人，现在一天才跑几百块，回本周期肯定变长了。”小孙表示，他刚入场的时候几天就能回本，而现在入场的话，就算代币稳住现在的价格，回本大概需要一个月时间。

按照StepN模拟器的计算，SOL代币购买的鞋子回本周期最短的大约在37天。小孙认为，由于官方设定7月15日才清退，现在入场的国内用户回本也许还有望，“但之前成本高的人要赔钱了 ”。

7月15日清退以后，大陆用户还能否通过其它方式参与StepN呢？小孙说，“目前来看是不能跑了，无解。只能去国外跑，因为就算搭梯子能改IP，最重要的是改不了GPS 。”因此，他的朋友圈有不少在海外的参与者开启了“海外代跑业务”。甚至有人表示，已经在跟最大的华人超市洽谈，希望与外卖团队合作。

StepN团队的Jerry指出，中国大陆用户占其平台整体用户群的5%，暗示公司退出该市场不会对其财务成功产生重大影响。

其首席营销官Shiti Manghani在周五的直播中说，“从一开始，我们就一直遵守中国的当地法规，这是我们长期战略的一部分。我们希望成为一个可持续的长期项目。”

Manghani补充，“作为清退的一部分，我们需要实施GPS技术以防作弊，”她说，“中国在GPS监管方面的规章制度不同，所以我们不是唯一一家。耐克刚刚删除了他们的Nike Run Club应用程序。我们必须遵守当地的法规，这将是我们对任何国家的立场。
