揭秘OpenSea的成功秘诀 它是如何一步步成为最受欢迎的NFT市场的

2018 年的冬天，那时候加密市场已经崩溃，甚至像比特币和以太坊这样的蓝筹股也从历史高点下跌了 90%。 一个接一个地，那些曾经大张旗鼓地启动的项目开始走向倒闭。 到了感恩节，你叔叔伯伯挖苦地问：“额……你的比特币投资怎么样了？”，这场晚餐真的很尴尬。

在其他项目都躺平的时候，有一个有信念和进取心项目还在继续建设。 OpenSea 于 2018 年 2 月推出，努力追求其成为“加密商品领域的eBay”的愿景。 凭借作为最受欢迎的 NFT 市场，并且最近估值 133 亿美元，OpenSea的坚持得到了回报。 这篇文章试图分析促使 OpenSea 作为产品取得巨大成功的原因以及它未来将走向何方。

首先，什么是 NFT？

嗯，NFT（不可替代代币）是一种在区块链上存储资产所有权的方式。它可以是图片、音乐等数字资产，甚至是土地所有权、文凭等现实世界的资产。由于区块链是完全去中心化且抗审查的（至少以太坊是），因此您不必信任第三方来维护该所有权的记录。

OpenSea 是一个与底层区块链交互的市场，用于创建和交易代表数字资产（如图片、音乐，甚至是以太坊上的域名）的 NFT。市场是促进产品销售和购买的途径。想想亚马逊、优步和Airbnb。大多数市场的一个关键特性是他们并不真正拥有产品的库存，他们只是允许其他人在这个市场上销售（有一些例外，比如亚马逊也销售他们自己的库存产品）。

OpenSea 真的那么受欢迎吗？

让我们看看数字。下表根据过去 30 天的用户数、交易数量和ETH交易额比较了不同的 NFT 市场。这些统计数据来自 DappRadar，它查询以太坊区块链以跟踪以太坊应用程序的各种指标。

从上面可以明显看出，OpenSea 已经一骑绝尘。在过去 30 天里，它促成了价值约 45 亿美元的交易（即 150 万ETH），远远超过所有竞争对手。综上所述，eBay 在上个季度每月进行的交易价值约为 66 亿美元。尽管二者处理完全不同的产品（OpenSea处理区块链数字资产，eBay处理实物商品）。尽管在加密领域运营，OpenSea 仍能够跟上传统市场的脚步，对此表示赞赏。

OpenSea做对了什么才获得这样的成功？

一个项目的成功不能仅仅归因于几个因素。它需要出色的产品、出色的团队、出色的时机和大量的运气。虽然 OpenSea 似乎拥有所有这些，但我们至少可以尝试评估他们到底做了什么正确的事情才能达到现在的位置。

快速建立供需关系

正如 Lenny Rachitsky（Airbnb 的前产品经理）在他的时事通讯中所讨论的，推出市场的最大障碍之一就是所谓的“鸡与蛋的问题”。当平台上没有用户购买产品时，您如何说服供应商在您的市场上上架他们的产品。除非已经有产品库存可以出货，否则您如何说服买家到您的市场购买？

大多数传统市场都是通过首先专注于建立供应而获得成功的。 OpenSea 也是如此。正如 The Generalist 报道的那样，他们竭尽全力建立初始供应，这在 Richard Chen（加密投资公司 1confirmation 的普通合伙人）的以下引文中得到了完美的总结：

“Devin 和 Alex [OpenSea 的联合创始人] 在 Discords 中出色地发现了新的 NFT 项目，并击败了 Rare Bits [它当时的紧密竞争对手]，让这些项目在 OpenSea 上上市并吸引了大部分交易量在 OpenSea 而不是 Rare Bits 上。我们在 2018 年 4 月投资时，OpenSea是Rare Bits 交易量的约 4 倍。”- Richard Chen

所有这些努力都促成了OpenSea早期与多个项目/艺术家的合作。这包括支持 Axies（可以说是基于以太坊的最受欢迎的游戏）；使用其原生代币 MANA 购买 Decentraland（类似于 Minecraft 的基于以太坊的虚拟游戏）；与美国职业棒球大联盟合作销售 MLB 数字收藏品；提供官方死侍数字收藏品；与德国足球俱乐部拜仁慕尼黑合作上线球员卡 NFT；并支持所有花哨的 .eth ENS域名的交易。这份非详尽的清单展示了团队与尽可能多的项目合作以建立稳定供应流的热情，以及在游戏、艺术、体育、娱乐业、域名等广泛类别中迎合 NFT 的远见.

这种海纳百川得到了以用户为中心的功能的补充，这些功能使供应商能够轻松上手。 NFT 的上市是无需许可的，无需任何批准或复杂的进入门槛。此外，OpenSea 创造了一种将在以太坊上铸造新 NFT 的成本从卖方转移到买方的方法。这两个功能都大大降低了新晋艺术家/项目的参与门槛，这极大地促进了平台上广泛的 NFT 库存。

一旦建立了稳定的供应，传统市场就会开始专注于建立需求。对于 OpenSea，执行此操作要简单一些。在忙于获得早期的合作伙伴关系并将自己建立为上架 NFT 的平台之后，所有较新的项目都开始在那里上架他们的 NFT 并带来他们的客户。项目本身也受到激励，以鼓励其客户使用 OpenSea，因为该项目将在原始 NFT 的每次二次销售中获得特许权使用费。此外，从 2021 年夏天开始的 NFT 狂热推动了 OpenSea 的需求增长。一旦加密世界接受了这种区块链的非 DeFi 用例，促使以太坊上的交易创纪录，猜猜谁在那里，准备好为所有这些交易提供便利并赚到大海那么大的财富了吗？

最大化用户幸福感

正如 Sarah Tavel 在她的博客文章中所讨论的那样，市场不应该只专注于增加供应，还需要实施产品功能以最大限度地提高用户的*幸福感*。 就像优步通过自动充值信用卡创建无摩擦支付一样，Airbnb 通过建立超级房东计划，满足要求的房东获得一个特殊的展示徽章，从而使他们对新预订更具吸引力。 OpenSea 在这方面也表现出色。

他们的一大成就是实现了强大的过滤和排序功能，以便轻松发现 NFT。 这可能看起来微不足道，但事实并非如此。 每个 NFT 收藏品都有不同的特征。 OpenSea 允许根据正在查看的收藏品进行“自定义”过滤和排序，以便用户可以根据该特定分类来细化他们的搜索。

针对不同收藏品的不同过滤规则

接下来，他们采取了具体措施来解决困扰 NFT 爱好者和一般以太坊用户的最大问题，即高得令人望而却步的 gas 成本。gas成本是以太坊网络上每笔交易收取的费用，该费用支付给运行网络的个人，也称为矿工。但是由于对以太坊的需求非常高，而当前网络的容量相当低，因此每笔 NFT 交易的这些费用可能从几十美元到几百美元不等（取决于网络拥堵情况）。虽然 OpenSea 无法控制底层以太坊网络的 gas 成本，但他们实施了多项功能，至少在一定程度上降低了这些成本。他们允许拍卖在链下进行，以便在整个销售期间只使用一次 gas，允许拍卖价格在链下降低，这样价格变化就不需要每次都需要 gas，引入了“惰性铸造”，以便艺术家可以免费铸造NFT，gas 由买家在首次购买 NFT时支付，同时OpenSea集成了Polygon，Polygon 是另一个与以太坊交互但费用较低的区块链网络。

继续关注产品的金融方面，OpenSea 还允许使用除 Eth 之外的加密货币购买 NFT，尤其是像 Dai 和 USDC 这样的稳定币。这特别有用，因为用以太币购买会在许多国家引发税收事件，而使用稳定币是避免这种情况的简单方法。

做出正确的选择

好的产品只是成功的驱动力之一；这是一个关键的因素，但不是唯一的。每个项目在其生命周期中都面临着无数的选择，而他们做出的选择决定了他们是在大海中乘风破浪还是倾覆沉没。 OpenSea 在通往顶峰的过程中做出了许多正确的选择。

首先是他们对自己使命的信念。据 The Generalist 报道，使他们坚持信念的原因是一种叫做 ERC-721 的东西。这是一个由核心以太坊开发人员并行构建的标准，用于规定在以太坊上创建和转移 NFT。该标准的创建意味着该领域的重要性足以引起以太坊开发人员的关注，并且所有未来的 NFT 项目都将遵守该标准。因此，他们在 ERC-721 标准之上创建了一个市场，然后可以支持所有未来的 NFT。通过这样做，OpenSea 通过聚合 NFT 供应为买家提供了更加便捷的体验。在此之前，每个项目都必须创建自己的市场，例如，CryptoPunks 和 Axie 有自己的市场（现在仍然如此）。

对 NFT 的这种信念也是让他们在 2018 年的加密货币寒冬中幸存下来的原因。当加密货币市场极度低迷，没有多少项目能够生存下来时，OpenSea 经营着一支只有 7 名员工的精干团队，不断收取交易费以维持公司的生存，并继续建造。

最近，由于代码中的错误，OpenSea上许多高价值 NFT 以较低的价格出售时，他们面临了一些更艰难的选择。他们决定赔偿受影响用户的损失。同样，在某些情况下，他们还会冻结被盗的 NFT。虽然他们受到了一些人的批评，因为这违背了抗审查的基本加密精神，但这些手段肯定有助于建立与用户的信任。

OpenSea 的下一步是什么

征服世界之海后做什么？ 你会努力粉碎每一个敢于推翻你的竞争对手，并继续你的统治地位。 您可以通过在你的类别中大幅排名第一或扩展到其他用例或两者兼而有之来做到这一点。

OpenSea 已经是其 NFT 市场类别中的主导力量。 然而，他们面临着来自新来的 LooksRare 的非常真实的威胁。 这个最近推出的 NFT 市场对OpenSea来说有很多东西需要关注——他们费用低于 OpenSea，拥有一个原生代币，这大大增加了它对加密原生代的吸引力，并与代币持有者分享其所有费用。 从前面的表格中可以看出，LooksRare 已经超越了所有其他平台，并且正在接近 OpenSea 的交易数量和 Eth数量。

那么 OpenSea会推出自己的代币吗？这不是那么简单。作为一家美国公司，OpenSea更喜欢与美国监管机构相处融洽，因此不能冒险推出代币，这只会招致监管机构的愤怒，从而阻碍他们的运营。因此，OpenSea 的前进道路是继续最大限度地提高用户的幸福感，并可能扩展到其他用例。以下是一些潜在的选择：

  • 开发移动端。 OpenSea 已经有一个移动APP，但它的功能非常有限，甚至不允许交易 NFT。该APP需要抽象出创建钱包、存储种子（可以为高级用户执行此操作）和复杂的gas估算的过程。基本上，他们需要让体验足够无缝，可以吸引加密新手。

  • 探索 NFT 的碎片化，NFT 的原始所有者可以将所有权拆分为多个较小的代币，然后其他人可以购买这些代币。这允许 NFT 的共享所有权。接触像 BYAC 和 CryptoPunk 这样的蓝筹 NFT 的机会，这些现在大多数人都买不起，这将释放新的需求。原始所有者可以从增加的流动性和他们可以通过出售碎片化部分获得的潜在“策展人”费用中受益。双赢！

  • 考虑与 DeFi 协议合作（或创建自己的协议！）以允许使用 NFT 作为抵押品来借钱，至少一开始是蓝筹 NFT。这可以解锁全新的用户群和对 OpenSea 的需求。

  • Vitalik 最近谈到了不可转让的 NFT，这些 NFT 具有潜在用途，例如颁发大学学位、土地所有权等。虽然还有许多考虑因素尚未解决（例如，如果所有者因为安全问题而想要转移到他们自己的第二个钱包），OpenSea 可以引领社区的讨论，甚至可能为其编写 ERC 标准！他们甚至可以探索潜在的创建 B2B 解决方案，为可能想要使用它的企业提供不可转让的 NFT 实施。

应该注意的是，上面提到的一些选项，例如 NFT 的碎片化和作为抵押品的效用，可能会使 OpenSea 受到监管机构的审查。 所以他们必须小心行事。 事实上，监管是 OpenSea 需要持续监控的一大潜在障碍，因为所有加密货币的监管普遍存在不确定性。 事实上，作为加密社区的主导者，他们甚至可以考虑游说对加密友好的法律，以帮助生态系统向前发展。 毕竟，权力越大，责任越大！
