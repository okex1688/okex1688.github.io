撰文：杨树

SevenX Ventures 成立于 2020 年，是一家亚洲新锐加密投资基金，专注于发掘区块链领域内早期创业者和项目标的，且以一级市场投资和持续为创业者服务为主，它的三位创始合伙人和团队也都是在这个行业有 5 年以上经验的老兵。

‎在投资风格方面，SevenX Ventures 主张「浸入式投资 」——除了资金支持外，还在融资规划、金融产品设计、经济体系、营销等方面为所投项目全方位赋能，一站式‎提供资本、见解、运营等专业知识，以帮助区块链创业项目成长。

此外，除了传统的 DeFi、NFT、GameFi、元宇宙、Web3 等赛道，‎SevenX Ventures 持续跟进以太坊生态，还特别关注 Arweave 生态、NEAR 生态、波卡（Polkadot）生态的投资机会，在多个公链生态的创业项目中多有布局。

据不完全统计，SevenX 在加密领域的公开投资组合共计约 80 个，涵盖区块链网络、DeFi、NFT、GameFi 与元宇宙、Web3 等诸多领域 。此外 SevenX 除了传统的投资项目，还结合「FOF」属性，投资布局了一众业内的头部基金伙伴。

风投基金

Lemniscap 是一家专注于早期的新兴加密资产和区块链初创公司的投资基金，代表项目包括Algorand、Avalanche、FTX、The Graph 等。

gumi Cryptos Capital 是一家日本游戏上市公司出资成立的加密风险投资机构，团队主要base在日本和美国，代表项目包括 OpenSea、YGG、Celsius Networks、1inch Network 和 Qredo 等。

Sparkle Ventures 是一家总部位于法国的，由 Animoca 和 Sandbox 的创始人共同发起的一个新基金，以支持 Web3 并参与元宇宙的建设。

Meta Web Venture Capital 是由 Near 原亚太区负责人发起成立的一起新锐基金，主要在 Near生态布局投资，代表项目包括 Aurora、Octupos、Orderly 等。

BitKraft Ventures 是全球领先的，专注于游戏、Web3 和沉浸式技术的投资机构，在游戏行业有10年以上的投资经验，代表项目包括 Alethea、YGG、Immutable、Horizon、Ember Sword 等。

32bit Ventures 是由 Civitas 的首席产品官成立的一家专注于区块链游戏和生态系统的投资机构，拥有10年以上的游戏行业从业经历。

Arcanum Capital 是由 Ploygon 的创始人主要出资，专注于印度市场的区块链风投基金。

Bonfire Union Ventures 是一家专注Web3的风投基金，是由全球头部的 Web3 协议 Mask 出资成立的，在内容、社交领域的投资十分广泛，代表项目包括RSS3、Meson、Scroll 等。

C2 Ventures 是一家专注于早期 Web3 项目投资的风投基金，由 Ciara Sun 创立，旨在为下一代 Web3 建设者提供资金和运营方面的支持。代表项目包括 Project Galaxy、Kemet Trading、Truflation、Orderly 和 Gala Music 等。

Geekcartel 是一个DAO Ventures，由一群资深的白帽、技术极客和社区KOL联合成立，帮助项目在安全审计、资金安全、社区建设上赋能，代表项目包括 GO+、Project Galaxy、Star Sharks 等。

区块链网络

Aurora 是建立在 NEAR 上、针对以太坊的 Bridge + EVM 解决方案，可帮助以太坊用户和 Dapp 轻松迁移至 NEAR 区块链。

Oasis Network 是主打隐私保护的 Layer1 公链，通过 Proof- consensus 机制智能合约为用户提供高安全性、隐私性和可扩展性。此外通过 Tokenized Data 模型，Oasis Network 帮助用户管理和利用数据的潜在利益，并通过将他们的数据质押到应用程序中来获得有吸引力的奖励。

Octopus Network 是基于 NEAR 构建的应用链网络，允许开发者和 NEAR 实轻松集成，从而发布自己的区块链，并如同 Cosmos、Polkadot 等应用链协议一样，支持可定制的加密经济规则和治理规则。

Unique Network 是基于 Polkadot 和 Kusama 设计的 NFT 公链，通过开源的组件库，使任何人都可以自定义 NFT 铸币方案，规避了以太坊上高 Gas 费等问题，是 Polkadot 生态重要的NFT 基础设施。

NEAR 是一个 Layer1 公链，其特色包括了优化的分片设计、Nightshade共识与「大区块」设计。SevenX 认为在正统性上 NEAR 是最能够承接以太坊的公链，其对以太坊 2.0 体系的理解非常深刻，并已经实际进行了开发与落地：POS+分片的体系设计。此外 NEAR 账户体系的设计和对大规模并发型应用的支持，使得 NEAR 可能成为承接 Web2 -Web3 应用转型的最具有潜力的基础设施。

Arweave 是一个Layer 1 永久性存储公链，其特色包括了优化的分片设计、Nightshade 共识与「大区块」设计。

DeFi

SevenX 认为 DeFi 行业是由一个类似房屋的「三层结构」组成的，分为「地基-柱子-房顶」三层 ，图左侧则代表对 DeFi 这个「房屋」影响最大的元素，类似「水泥沙子和石头」。

其中最底层的地基是「现金」和「资产」，即一个金融体系里最基本的两个元素。现金即稳定币，资产即各式区块链原生或外界映射入区块链的资产，在地基之上，生长着五根柱子，分别对应：交易、借贷、资产管理、支付和衍生品。

在 DeFi 建筑的侧面则是扩容、跨链、隐私、预言机，这些「水泥」和「沙子」都是影响大楼构建的基础设施，这些基础设施的变化也将影响或者改变 DeFi 行业的变化。

交易 & 流动性

虽然交易赛道目前以 AMM 机制为主流，但 SevenX 认为当公链性能进一步得到提升，或 L2 解决方案落地后，交易方式可能会回归「订单簿」（orderbook）。

DODO 是一个采用主动做市算法（PMM）的去中心化交易所。

Ref Finance 是 NEAR 生态的 AMM 和稳定币兑换市场，旨在为 NEAR 生态系统提供 DeFi 一站式服务，将在 2022 年下半年致力于集中流动性功能，并推出 Zap Swap 功能，允许任何人在一个操作中访问任何 LP 代币。

CowSwap 作为建立在 Gonsis 链上的去中心化交易所，是从 Gonsis 团队内部孵化的主打聚合流动性和抗 MEV 的 DEX。

Zenlink 是基于 Polkadot 的底层跨链 DEX 协议，致力于成为 Polkadot 的 DEX 组合中心。通过接入基于 Substrate 打造的开放、通用的跨链 DEX 协议 Zenlink DEX Protocol，使得所有平行链能够一键构建 DEX 并实现流动性共享。

WOO Network 是一个零 Taker 费率聚合 Swap，可将中心化的交易所和去中心化的交易订单聚合，并提供给机构用户和 DEX 协议。

Hypersea DEX 是一个智能流动性管理协议，旨在提高流动性提供者（LP）的利益并减少交易者的风险敞口。

Shadow Labs 是由全球Cefi头部对冲基金 Hashbot 和 DODO、SevenX 联合成立的专注于 DeFi 流动性管理和产品设计的 DeFi 基金，目前服务客户主要包括 DODO、DYDX、 iZUMi、GMX、YIN。

收益 & 管理

SevenX 认为在未来 DeFi 协议将真正扮演协议的角色，即用户不直接和基础协议进行交互，而是通过聚合器和 Dashboard 来和底层协议发生交互，聚合器将成为流量的聚合入口，再将用户流量分发到下方的各类协议中。

Zerion 是全球最头部的 DeFi 和 NFT 资产组合面板+跨链交易聚合器，并开发了自己的非托管钱包，平均日活5万用户。

Furucombo 是一个多链 DeFi 聚合器，用户可以像基金经理一样创建 DeFi 组合策略池并从中受益，也可以自主选择优秀的 DeFi 组合型策略并从中获得收益，旨在建立一个跨链基础设施，使得 Furucombo 成为不同链上 DeFi 的门户。

Phuture 提供一种「开放式设计」理念，用来创建 DeFi 投资策略，该协议将包括独特、可扩展、聚合的指数架构，使该协议能够在多个价格指数之间进行实时再平衡，支持用户在 Web3 环境中建立，创造和投资指数。

Swivel Finance 是去中心化固定利率贷款和利率衍生品协议，主打利率衍生品。

YIN Finance 是一个面向 Uniswap V3 市场的主动聚合流动性管理协议，用户可以通过订阅协议中的 CHI（YIN Finance 的主动流动性管理策略）实现更优的流动性管理，从而实现更高收益。

借贷 & 衍生品

借贷方向上，虽然目前超额抵押的 Compound 和 Aave 处于绝对领先位置，但 SevenX 考虑未来随着 DID 和链下身份的打通，信用贷款将会初见端倪并快速发展。

Opium 是一个通用而全面的链上衍生品协议，允许用户创造、清算和交易任何去中心化衍生品。

Burrow 是NEAR 生态系统的借贷协议，专注于生息资产，特别是那些利息固定的资产，例如 Staking 衍生品（stNEAR、stETH、stSOL）。

Shield Official 是链上衍生品交易的加密原生解决方案，旨在成为 Web3 结构性产品的一站式商店，产品包括创新衍生品 Shield Perpetual Options，可实现高资本效率、低 LP 交易对手风险，并且没有无常损失；以及加密结构性产品 Shield Vaults，可帮助用户提高投资组合回报。

Strips Finance 是点对池模式的去中心化利率衍生品，用户能与资金池以利率为标的交易衍生品。

synfutures 是开放式合成衍生品交易所，专注于搭建链上合成衍生品市场，涵盖开放式由 LP 添加任意资产、任意到期日交易对；合成自动做市商模型，单币种做市而支持包括跨链、链下以及指数等合成资产交易；3. 自动清算人模式，并在链上实现部分平仓这三大功能。

钱包

钱包将作为基础设施，串联起几乎所有的区块链应用。在之前几年的市场中，钱包作为盈利模式并不清晰的项目，并没有得到资本热捧。随着公链生态的成熟，越来越多的真正有使用价值的 DApp 将不断涌现，钱包将成为具有盈利模型的基础设施。有了流量的聚合，钱包可以向下的聚合器、交易、借贷、支付、衍生品等所有支柱赛道延伸，甚至深入到现金和资产地基。

Blocto 是一个跨链智能钱包，旨在成为用户进入数字货币、DApp 及 NFT 的入口，已支持 BNB Chain、以太坊、Flow、Solana 和 Tron 等网络，内嵌了跨链 DEX Bloctoswap，并且给其他 DApp 提供钱包 SDK 解决方案。

Sender 是 NEAR 生态的原生头部钱包，目前已经接入了 NEAR 上 20 多个 头部协议。

BitKeep 集「钱包」「BitKeep Swap」「NFT Market」「DApp」「Discover」五大功能为一体，主打去中心化多链数字钱包，未来将从封闭型组织转向 DAO 治理模式，建立支持多链生态的区块链基础设施。

基础设施

Vega Protocol 是一种以去中心化方式交易衍生品的 Layer2 解决方案，允许用户以匿名方式启动衍生品市场，包括期货、交易和期权，同时遵守相关规定并去除金融中间商环节。

Acala 是波卡生态的 DeFi 基础设施，旨在联合波卡 Polkadot 生态创建开放式金融框架，已推出了跨链多资产抵押的稳定币 Honzon 协议、释放 Staking 资产流动性 Homa 协议以及去中心化交易所等核心 Layer1 模块。

Bifrost 是波卡生态 DeFi 基础协议，致力成为质押资产提供流动性的基础设施，目前推出面向 Staking 和波卡平行链卡槽 PLO 的衍生品 vToken，可实现 Staking、PLO 等质押权益的转让通道，实现质押资产风险对冲。

Orderly Network NEAR 生态里原生的头部金融基础设施，由 NEAR 团队和 WOO 联合孵化，主打 NEAR 上的订单簿（orderbook）交易。

everFinance 旨在为所有人提供没有时空界限的去中心化金融服务，其目标是提升用户体验，降低开发门槛，为所有人提供去中心化可信的金融应用，已经推出的 everPay 是一个区块链易用性的解决方案和应用协议，在保障可信的前提下，为用户提供接近互联网的支付结算体验。

Fusotao 建立在 Octopus 上面的金融基础设施应用链，不仅本身有一个订单簿（orderbook）交易协议，还可直接一键基于他们发行自己的 DEX。

稳定币

TiTi Protocol 新一代稳定币发行协议，创新地开发了垄断 AMM 以及 Use to Earn 的激励模式，在协议收入来源以及流动性管理与激励上向市场提供了更高效的设计。

发行

Dao Maker 作为加密孵化机构，可为加密项目提供区块链 SaaS 解决方案、通证经济学设计等服务。

Dora Factory 是基于波卡的 DAO 即服务基础设施，它基于 Substrate 的开放、可编程的链上治理协议平台，可为新一代去中心化组织和开发者提供二次方投票、曲线拍卖、Bounty 激励、跨链资产管理等可插拔的治理功能，同时开发者可以向这个 DAO 即服务平台提交新的治理模块，并获得持续的激励。

保险

InsurAce.io 是一个去中心化 DeFi 保险协议，可提供「一揽子」式保险服务，同时兼具投资功能，能够为用户提供有保障有收益的保险服务。

Amulet Protocol 是 Solana 生态的去中心化保险协议，通过协议控制的承保价值（PCUV）集中收益支持索赔池、金库和其他储备层，以减少质押资本的损失。

跨链

Swing 是一个跨链流动性协议，加密开发人员和交易者可使用 Swing 在以太坊、Solana 等区块链之间进行跨链交易和流动性迁移。

O3Swap 是一个跨链聚合交易协议，可通过与以太坊、BNB、Neo 等主流公链和 Layer2 跨链交互，进行多链 DEX 的流动性聚合，从而实现不同链上主流资产的自由兑换。

NFT

如果 FT 对应着区块链世界的「钱」，NFT 则对应着区块链世界的「物」，换句话说，任何在 区块链世界拥有特定属性的对象都可以用 NFT 来表达。

所以 SevenX 的理念是整个 NFT 赛道的结构也遵循底层⸺中间层⸺上层流量的逻辑。在最底层是支持NFT 的 L1 公链，在此之上是 NFT 协议标准，各个项目方可能会推出自己的 NFT 协议标准以适合不同的应用场景，再之上是 NFT 的跨链协议和扩容方案、NFT 的 MetaDate 存储方案，在此之上是一些非常有趣的智能合约集合，用于在 NFT 之间进行组合和交互。

在这些协议层基础设施之上，则是 NFT 重要的几个应用场景，包括游戏、社交、收藏品（虚拟地产）、 IP 和 NFT as PASS。在整个 NFT 体系的最上层，则是流量来源。这些流量未来将会以有组织的方式围绕着不同的资产和项目进行分布。

在 NFT 体系的左侧和右侧，分别是对于整个体系最为重要的两个群体：收藏者群体（买家）和创作者群体（卖家），他们分别为 NFT 带来了购买力和资产。来自右侧的创作者使得更多的资产可以以 NFT 的形式被铸造出来，进入 NFT 的整体使用、流通和金融体系，并被来自左侧的收藏者购买。

Pixelynx 是一个音乐元宇宙发行平台。创作者可以根据已有模板（主题，美术要素，游戏与经济机制等）快速生成不同的元宇宙，在其中进行游戏或者演出，同时方便粉丝获取体验与资产。

Smart Token Labs 是一个代币标准化协议，借助该协议 NFT 持有者可以创建继承原始特质，同时又容纳全新自定义特质的「合成 NFT」。

Whale Shark 由加密资产市场最大的 NFT 个人买家之一 WhaleShark 创立，WHALE 代币间接由 WhaleShark 的 NFT 资产支持，这些资产存在称为「Vault」的地址中，主要由 WhaleShark 掌管。

Accursed Share 是一个动态 NFT 技术方案提供商以及 NFT 发行平台，用户铸造的 NFT 可以根据外部数据的变化而产生变化。

MintGate 是支持接入多个链（以太坊，xDAI，Polygon，Near 等）的 NFT 铸造平台，主要解决现有 NFT 铸造与分发平台的低可用性与高使用门槛，创作者可以先将 NFT 的内容放到MintGate 上，粉丝选择支付 Gas 费，将该 NFT 铸造在任何一条支持的链上。

myNFT 基于 Flow 公链，旨在连接东西方市场，为用户提供优质的发行、 交易、社交体验的，同时它可聚合管理与展示了用户的 NFT，且聚合了用户基于此的社交关系。

Taker Protocol 是一个面向新型资产（如 NFT 等）的租赁和借贷协议，旨在为用户提供报价机制、为 NFT 等新型资产持有人提供抵押借贷等服务。

AFKDAO 作为游戏 NFT 流动性协议，通过 ERC721 资产来创建可委托的 NFT 资产来构建 NFT 的 DeFi 基础设施，包括无需信任的资产管理以及无抵押 NFT 借贷等，由其专有 ERC4610 协议的委托功能提供支持。

GameFi & 元宇宙

SevenX 认为 Web3 游戏赛道从下至上由基础设施层、产业链上游、中游和下游组成，并有其他服务贯穿游戏产业上中下游。

游戏

FearNFT 是一个唯一带 IP（Whack It）的恐怖系列游戏 + NFT 收藏 + Gamification。

Civitas 是一个创新型城市建设元宇宙游戏，用户可以参与到一个城市从零开始建设的整个发展历程，并选择自己喜爱的活动，包括：资源采集、贸易、经商、建筑等。

StableBattle 是一款适用于手机和 PC 的 P2E 休闲游戏。

Cyberverse 是一款赛博朋克的射击类游戏和星际大战的元宇宙开发世界。

电子竞技

Ignite Tournaments 是全球首个面向移动手机端的 Web3 游戏电竞锦标赛平台，旨在激励全球数十亿非职业游戏玩家、电竞爱好者组织自己的电竞锦标赛，并从中获得收益。

游戏公会

YGG 是成立时间最久、规模最大的链游公会。

YGG SEA 是 YGG 旗下第一个区域子 DAO，专注于马来西亚、印度尼西亚、越南和泰国的玩家。

发行

OP Games 是一个 Web3 游戏发行平台，通过将 NFT 和 DAO 融为一体，让用户深度参与到自己喜爱的游戏，把游戏归属权更好的交给开发者和社区，共建游戏生态。

Colizeum 给传统的竞技类游戏提供了一整套游戏上链、游戏分发和游戏竞猜的解决方案。

元宇宙

Bit.Country 是建立在 Substrate 上去中心化的虚拟世界和 NFT 项目，旨在为元宇宙游戏带来丰富的知识和联系。

Metametaverse 是 MetaMask 创始架构师 Joel Dietz 推出的元宇宙互操作性项目，它会有自己的 Layer1 区块链，拥有一种新编程语言「metametalang」，允许用户建立自己的元宇宙。新编程语言可用于在 「立方体」（cubes）内运行游戏和模拟，以及映射平行现实。

Ethereum Towers 旨在建立一个以社区为中心的垂直元宇宙世界，由 4388 个居民拥有的公寓和公共区域组成。

MetaEstate 是一家提供全方位加密元宇宙服务的公司，旨在帮助更多用户零门槛迈入元宇宙的世界，为土地所有者、艺术家等提供更加沉浸、实时和多元的泛娱乐体验，，并为用户提供包括买卖/租赁元宇宙地产、搭建及沉浸式体验、营销策划及活动执行、广告、可穿戴周边设计在内的各项服务。

Matrix World 是一个 3D 可编程的多链元宇宙，可以实现 3D 低代码的 AI Generate Contents（AIGC）与 User Generate Contents（UGC）创造，以及能为元宇宙世界提供信标，通过代码融合多元宇宙空间。

基础设施

Rangers Protocol 是由 MixMarvel 孵化的 NFT 公链基础设施，为区块链游戏和资产提供更高性能更优化体验的跨链服务。

预测市场

Azuro 是一个利用智能合约构建的去中心化投注协议，包括经典投注和预测市场，通过流动性提供、前端管理、预言机提供数据、去中心化（DAO）治理构建链上博彩机制。

Web3

SevenX 认为 Web3 其实是一种对于网络权利的唤醒 ，是一种对于新的秩序的向往，所以当下最重要的是唤醒，唤醒更多人拾起这份权利，建立这份共识，因此 Web3 整体结构可分为四层：

  1. 最底层是用户看不到的智能合约和协议层；

  2. 第二层是用户可以使用但是看不到的存储层；

  3. 第三层是被封装后用户可以使用但是看不到的数据层；

  4. 第四层是可看到可使用的应用层。

社交 & 内容

Mask Network 是一个帮助用户从 Web2 无缝过渡到 Web3 的桥梁，允许用户在传统社交巨头的平台上无缝发送加密信息、加密货币，甚至是去中心化应用（比如 DeFi 、NFT 和 DAO）。

ShowMe 作为 Web3 社交中间件，主打以订阅为核心的 NFT 社交，创作者可通过灵活的订阅方式来吸引并沉淀粉丝。

Yup 是一款将社交影响力可视化的去中心化社交协议，能植入在各个现有的社交平台中（Twitter、Youtube、Facebook、Instagram 等），对用户的跨平台点赞、 评分、分享和其他社交动作进行管理，并通过建立的社交偏好、创建和分享的内容和信用度来赚取收益。

t2.world 是一个去中心化的出版平台，提供「Read-and-Earn」机制来鼓励深度阅读。

ChainFeeds 是一个开源 Web3 内容聚合平台，由链闻原研究总监潘致雄创立，目前已上线 Alpha 测试版本，旨在从信息源开源计划 RSS Aggregator for Web3 开始，为行业用户提供标准且高质量的内容获取工具。

Akord 是一个基于 Arweave 的存储、社交以及隐私保护协议。

身份 & 隐私

Clique 是一个专注于隐私技术、旨在将 Web2 连接到 Web3 的身份和信誉系统。

Litentry 旨在提供 Web3 去中心化身份管理服务，它通过将不同链上网络身份的信息整合，包括 DeFi、链上治理等所产生的数据，创造一个链上的身份信用体系，帮助解决目前大部分区块链中空投低效、缺少信用借贷、PoS 权力集中等问题。

Zecrey 是基于 zk-Rollup 的 Layer 2 通用隐私跨链协议，可实现不同公链（以太坊、Near、Solana、BNB Chain 等）的数字资产的隐私跨链交易。

Integritee Network 是基于 Substrate 开发的 TEE 隐私协议。

DAS 是基于 Nervos 的多链账户系统，可提供以 .bit 为后缀的全局唯一的身份体系，从而作为数字货币收款账户、域名以及作为访问一般互联网服务的账户使用。

存储

Crust Network 是基于 IPFS 的去中心化存储协议。

AR.IO Network 是 Arweave 向外扩容的重要基础设施，能更好的降低用户访问 Arweave 网络数据的门槛。

Ardrive 是一种功能齐全的文件永久存储应用，具有隐私、抗审查的特性，可提供更加安全的永久性数据存储方案，是 Arweave 生态面向广大 Web2 世界的用户的门户和流量入口。

Koii Network 是 AR 上的共识中间件、状态中间件，旨在打造一个快速产生共识的框架，AR 上的 DApp 们可以在这个框架上产生自己的共识。Koii 允许任意语言的任意代码以任务的形式发布，由节点来完成并领取奖励，进而让任意代码都可以在 AR 上面运行。

基础设施

Cassava Network 是由传音孵化的非洲区块链基础设施网络，通过有利于内容创作者和消费者的共识激励，来连接内容创造和消费。

ParaState 允许开发者用流行的编程语言编写与以太坊兼容的智能合约，并在波卡 Polkadot 上运行。

Spheron Protocol 可以让任何项目通过非常简单的操作将 Web2 前端和应用部署在Arweave 上。

Meson Network 是 Web3 带宽协议，通过挖矿的形式聚合闲置服务器，调度带宽资源并将其服务于文件以及流媒体加速市场，包含传统网站、视频、 直播及区块链存储方案。

Go+ Security 是 Web3 数据驱动的安全网络协议，面向各类区块链用户提供开放、无权限、用户驱动的安全服务。

Deeper Network 专注于去中心化网络协议，结合了一些网络硬件设备、安全技术、共享经济和区块链技术为全球用户提供分布式网络服务。

Decentology 是一个智能合约模块化的开发平台与市场，主打智能合约组件的交易市场和给开发者提供模块化的开发平台。

数据

Web3 数据赛道的结构可以被划分为四个层次，分别是最底层的数据来源，第二层的数据获取，第三层的数据查询与索引，还有最顶层的数据分析与应用。

Footprint Analytics 是一个社区驱动的数据分析平台，用户可以在上面查看不同链上各类协议的数据，能够很好的填补专业化用户与纯小白用户之间的空白。

DeepDAO 是针对 DAO 的综合数据平台，基于多个维度对 DAO 组织进行分析、探索和排名。

SubQurey 是波卡生态的数据索引服务，面向 Polkadot 和 Substrate 项目提供处理和查询项目相关的数据的服务，目前正在向 Avalanche 和 Cosmos 生态扩张。

Kwil DB 是一个基于 Arweave 的去中心化 SQL 数据库。

DeBank 通过数据的聚合和解析，为各种协议、钱包等前段入口产品提供资产分布和资产交易记录和行为的数据 API 服务，同时还有一个面向 C 端的仪表板和钱包，旨在成为 DeFi 行业的数据基石。

CyberConnect 旨在连接 Web3 上的每个人，为 Web3 构建社交图谱基础设施，构建一个可公开访问、去中心化的社交图谱协议，将数据所有权返还给 Web3 用户。

RSS3 一个开放的信息联合协议，旨在支持 Web3 中高效和分散的信息分发。

小结

Web3 不会一蹴而就。而在 Web2 到 Web3 范式转变的投资浪潮中，无论是触手庞大、四面出击的老牌基金，还是像 SevenX 这样「坚持早期投资，寻找非共识标的」 的新锐团队，都需要凭借各自对 Web3 的认知画像，去发现项目，并在趋势到来之前投资布局，时间会给出答案。
