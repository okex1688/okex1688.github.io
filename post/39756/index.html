作者：张烽

2022 年 6 月 29 日，NFT 租赁市场 Double Protocol 推出的可租赁 NFT 标准 EIP-4907 已通过以太坊开发团队最终审核，成为以太坊上第 30 个状态为 “Final”（最终版）的 ERC 标准。

据悉，该标准通过双角色的设置，实现了 NFT 所有权与使用权的分离，并首创了到期自动收回使用权功能。该技术协议用原有 ERC721 既有标准延伸，重新定义 NFT 使用者与拥有者。拥有者可以通过合约出租 NFT 给使用者进行协议操作，但使用者无法转移 NFT 所有权，且有到期自动归还功能。

ERC4907 这一标准的应用将极大地降低游戏、元宇宙场景、会员卡等 Utility NFT 租赁的开发和集成成本，让 NFT 资产更具流动性。 据悉，目前确认应用 ERC4907 标准的项目已达 12 家。

ERC4907 协议能否成为发展新动能，一举解决困扰 NFT 市场的诸多问题，我们试作一简要分析。

一、ERC4907 为什么会出现？

一段时间以来，境内外 NFT 市场虽然还是在持续受到关注，但其中存在的问题却也不得不让人担忧。

一是市场不断降温。 不管是境外的 NFT 产品还是国内数字藏品，普遍而言已经不象之前的销售这么顺畅了。据统计，全球范围内上个月 NFT 市场销量已经比去年 9 月缩水 90%，5 月以来 NFT 市场总体销售额从 46 亿美元下降到 15.9 亿美元，很多 NFT 项目销售出现断崖式下跌，一些 NFT 产品价格大幅下降。

二是 NFT 市场乱象也开始出现。 包括欺诈、剽窃、系统漏洞与钓鱼攻击、平台内幕交易甚至跑路、资产被盗各种问题层出不穷，可谓是野蛮生长。

三是 NFT 市场法律与监管问题持续受到关注。 包括产品知识产权侵权问题，特别是一直被诟病的 NFT 产品金融化操作问题，平台监管问题等，无论在境内外都不断出现。

概而言之，NFT 市场问题反映的焦点和瓶颈是流动性稀缺。而造成 NFT 流动性稀缺的原因，是其技术协议初级，应用场景单一，难以表达复杂经济关系，不可避免导致 NFT 市场金融化投机盛行。

正如 2022 年 6 月 8 日，我就有关 NFT 相关问题接受第一财经 " 财经夜行线 " 采访所认为，目前 NFT 市场基础设施不够完善，应用场景相对缺乏，技术协议比较初级，规范治理体系尚待健全，司法保障还需完善等问题。而其中技术协议比较初级可以说是造成这些问题的关键。

在 ERC4907 之前，ERC721 只能单一地进行特定化，而 ERC1155，ERC798 也只是在特定化上进行了有限的拓展，而且都只能一次性买断的方式来实现对 NFT 产品的使用，可以说无法更充分地实现数字产品的利用，无法更充分实现数字产品的流转，无法更充分地实现与线下权益的结合，无法更明晰地区分金融属性与商品属性的风险。

二、ERC4907 可能激发哪些应用场景？

商品所有权和使用权的分离、资产所有权和经营权的分离，本质上说是经济运行和资源配置的必然要求。商品或资产本身是兼有价值和使用价值属性，资产本身需要通过经营使其发挥最大价值，但是每个人每个机构能力偏好、资源秉赋和整合水平不同，客观上最合理地实现资源配置是应有之义。

将所有权和使用权分离的 ERC4907，简单地说就是把 NFT 的权利人分为所有权人和使用权人两种类型。所有权人可以根据情况需要为使用权人设定使用权限，但只有所有权人才可以转让 NFT。

什么东西可以出租？几乎大多数的有形或无形财产都可以出租，实现所有权和使用权分离。

出租权是著作权财产权类型之一。我国《著作权法》第十条规定，著作权包括下列人身权和财产权：其中列举的第七项就是出租权，即有偿许可他人临时使用视听作品、计算机软件的原件或者复制件的权利，计算机软件不是出租的主要标的的除外。因此，可以基于这一标准，将自己基于 NFT 持有的数字版权作品进行出租，租赁人成为 user 角色，承租人可以对作品进行使用，但不能进行转让所有权的操作。

元宇宙中的房产土地的利用装修可能就更方便了。元宇宙中的房东可以为装修工设定 user 权限，对房产进行相应的装修管理，装修完了甚至可以直接出租给个人或机构使用，大大丰富了应用场景。理论上说，也可以使用 NFT 对线下的房地产进行出租管理。

游戏道具使用中，可能会让游戏高手和投资人实现一定程度的完美结合。有的人喜欢下棋，有的人喜欢看人下棋。有了 ERC4907，可以让喜欢下棋和看人下棋的人各得其所。有的人在游戏中只是要体验一把某种特殊的装备，但是其可能价格高昂，他可能就只需要通过租赁使用几天，而对于装备所有权人来说，也可以更好地促进其价值发挥。

艺术品投资和创作利用，也可能会在一定程度上实现完美结合。目前艺术品 NFT 产品功能还是比较简单，但未来结合元宇宙场景，很可能可以将现实场景中艺术品出租可以照搬到数字空间，从装饰到欣赏、从利用再到改编，可以为艺术品的使用打开更广阔的空间。

数字内容和数字产品所有权和使用权分离，将会使 NFT 能发挥空间更为广泛。NFT 产品更容易与线下权益结合，更容易实现不同时空的结合，实现不同生态的结合，也更容易阻隔金融风险。由于 NFT 产品的使用场景增加了，实现价值途径增加了，流动性也会相应增加，其金融炒作风险反而可能降低。

三、ERC4907 可能需要注意什么问题？

设定出租权利，理论上同样也是一种交易，同样可能会产生相应的问题，这里列出了一些可能存在的隐忧，另外随着市场的进一步实践可能会有其他问题，需要我们保持持续关注。

（一）要更加紧密结合应用场景

NFT 价值本身还属于比较初级，租赁权的设定和定价如果没有现实传统的经济关系为作参考，可能产生随意性比较大、流于形式而形成新的金融操作，甚至形成更为复杂的金融关系。

在传统经济关系中，租赁等所有权与使用权分离的情况已经相当成熟。但尽管如此，仅以房产租赁为例，其具体租赁需求、出租价格、协议履行环节与周期等也是非常复杂的。而对于数字产品的出租，无论是基于纯线上权益，还是与线下权益结合，其中必然会出现很多需要进一步摸索的环节。因此这种操作，要更加结合应用场景，否则很可能成为一种技术游戏，伤害的可能不仅仅是 user，甚至可能 owner 影响更大。

（二）注意防止把租赁业务金融化异化

可以预见，后期很可能会基于 ERC4907 协议，出现专门经营出租或租赁的机构，基于 NFT 的出租或租赁权进行规模化操作。我们回顾一下传统长租公寓业务模式中出现的近乎疯狂的金融化操作，对于 NFT 规模化租赁提个醒还是必要的。另外，市场上很可能会出现一些变相的融资项目，或者以现实金融产品作为底层的 NFT 产品，会打着 ERC4907 的名义进行发行和交易，这很可能会蕴藏积累相当大的风险。

（三）用户要注意自己买的 NFT 产品是什么权利

未来，NFT 产品的数字内容更加丰富，其权益类型和内容甚至期限也会更加复杂。作为用户可能要注意分辨自己将要入手的 NFT 是属于什么内容，尤其是纯粹从投资角度去参与的，头脑要保持清醒。目前看来，其协议内容似乎还不能将租赁权进行转让即转租，只有 NFT 的所有人即 owner 才能将 NFT 产品出售，因此在使用和持有 NFT 产品时，一定要谨慎。

既然 NFT 技术是商品和服务数字化的技术工具，那么其技术协议不但应当反映传统现实生活中丰富多彩的各种权益，而且应当反映更为复杂的很多传统世界实现不了但却可以在数字世界实现的权益。我们看到事情似乎正在朝着这个方向发展。可以预见，ERC4907 可能是技术协议更加丰富的开始，相信未来会有更多贴近应用场景反映现实经济关系的技术协议出现。
