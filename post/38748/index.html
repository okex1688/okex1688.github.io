要想在未来取得成功，你现在必须要学习 web3 了。

首先，这 7 个改变世界的 Web 3 概念可以让您快速上手：

1. WEB3 - 在以太坊等区块链计算机网络上迅速出现的新版互联网

Web3 改变了互联网，为人类创造了一个更好的地方。

在 1990 年代初期，web1 带来了 HTML 和 URL。它允许用户在页面之间导航，但内容是静态的。

Web1 只是网络的只读版本。

在 2000 年代初期，web2 带来了交互的能力。Google、YouTube、Facebook 和 Twitter 这些平台使我们我们可以观看、搜索和分享内容。

但是，当我们创建内容时，这些内容都“属于”这些平台。

Web2 是网络的读写版本。

但主导 web2 的科技巨头通过跟踪和货币化用户数据发了大财。

到 2021 年，Alphabet（谷歌）、亚马逊、苹果、Facebook 和微软的年收入总和约为 1.2 万亿美元。

在 2020 年代，web3 正在为用户带来数字所有权。

现在您可以拥有数字资产、控制您的内容和数据等等。

Web3 是网络的读 + 写 + 自己的版本。

2. 数字所有权 - 您可以创建或购买数字资产并完全控制它

您拥有数字产权，可以执行交易或移动您的资产。任何价值都属于您。

由于不可替代的代币（NFT），数字所有权成为可能。

所有权、稀缺性和独特性很难在网上建立。NFT 的出现允许在线资产具有可验证的稀缺性和无法操纵的所有权。

他们将赋予实物资产价值的元素带入了数字领域。

通过数字所有权，您：

• 拥有您的在线身份和社交媒体内容

• 可以将其从一个在线平台转移到另一个

• 拥有您的在线个人、医疗数据和浏览历史记录

• 可以参与并受益于互联网经济。

以下是对代币为互联网和您的数字生活带来的所有权可能性的更深入研究：https://future.com/tokens-are-a-new-digital-primitive/

3. 无信任- 一个无需相关人员相互信任即可执行交易的系统

系统创建了所需的信任。

无需信任的系统消除了参与者相互了解或信任以使系统正常工作的需要。

无需中介机构（例如银行或交易所）来确保结果。计算机代码负责执行任务。

区块链技术使用智能合约来促进用户之间的无信任交互。

它允许来自世界各地的陌生人在没有中间人的情况下开展业务。

智能合约为他们执行交易。用户信任代码并消除中间人的存在。

去信任系统改变了商业、金融甚至投票选举系统。

4. 无许可 - 一个允许任何人参与而无需中央机构批准的系统

像比特币、以太坊和 Solana 这样的公共区块链是无需许可的。

这是一个开放的网络，任何人都可以参与用于验证交易的共识过程。

无许可系统的主要特征是：

• 完全透明

• 开源

• 匿名

• 缺乏中央权威机构

Web3 被构建为无需许可的。

每个人都可以有平等的参与机会，没有人被排除在外。

这是一种范式转变，让数十亿人能够：

• 参与经济

• 开发新的应用程序

• 扩大创意机会

• 建设一个更美好的世界。

来自@DefiantNews 的“无许可”意味着什么以及为什么它对 DeFi 很重要：https://www.youtube.com/watch?v=7cHzZ-oB-34‌

5. 零知识（ZK）技术——一种证明对特定信息集的知识而不实际揭示该信息是什么的方法

这是一个没有任何敏感数据交换的身份验证系统。

零知识技术是保护和维护您在 web3 中的在线安全和隐私的强大方法。

假设您需要达到一定年龄才能注册服务，使用 ZK，您可以在不分享您的私人信息的情况下提供您的年龄证明。

ZK 是另一个将改变我们生活方式的 web3 组件。

它将允许任何人在他们的手机上进行完全私密的交易，然后在无需信任的区块链上进行结算。

该视频展示了 ZK 如何实现隐私保护用例：https://www.youtube.com/watch?v=7VmI8fRYHMQ‌

6. 去中心化——将监督和决策从中心化的组织（个人、公司或一群人）转移到去中心化的网络

去中心化是区块链技术和 web3 运动各个方面的基础。

区块链在政治上是去中心化的（没有人控制它们）

架构去中心化（无基础设施中心点）

但在逻辑上是中心化的（一个普遍同意的状态，系统就像一台计算机）。 @VitalikButerin

https://medium.com/@VitalikButerin/the-meaning-of-decentralization-a0c92b76a274‌

去中心化网络通过提供公平的竞争环境赋予企业家、创造者和开发者权力。

它们是构建互联网的更好方式。

更多关于为什么去中心化很重要的内容：来自@cdixon

https://onezero.medium.com/why-decentralization-matters-5e3f79f7638e‌

7. 可组合性——将系统的组件重新配置成更大的结构以及一个系统的输出成为另一个系统的输入的能力

在 web3 中，可组合性带来更多选择和更好的用户体验。

任何人都可以采用现有的想法，使其更易于使用或适应新的用例。

在软件中，可组合性是一种指数力量。

开发人员可以引导他们自己的项目，而无需从头开始构建。

非加密业务可以接入开放的生态系统。

NFT 可以用作抵押品。

可组合性改变世界！更多来自@ljxie

总结

以上介绍了7 个改变世界的 web3 概念，即：

1. Web3

2. 数字所有权

3. 无信任

4. 无需许可

5.零知识技术

6.去中心化

7. 可组合性
