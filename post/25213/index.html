上学那会看到同学在玩一个叫《模拟人生》的游戏，就好奇地凑过去问他这游戏的目的是干啥，怎么通关？

他回答说：“这游戏没有目的，也不存在通关，就是扮演一个虚拟的你，游戏世界里的你想干啥都行”。

当时习惯了RPG这种关卡设计，与CS、星际这种目的性很强的游戏的我，表示不能理解。然而直觉上，我觉得这是个很有新意的游戏，虽然对我并没有什么吸引力。

后来有了《魔兽世界》，沉迷其中的我似乎过着两种生活，一种是线下的现实中的我，一种是线上艾泽拉斯世界的我 ，我时不时会想起几年前看到的那款游戏《模拟人生》。

有一次教一个9岁的小朋友计算机，看到他在玩一个叫做Roblox的游戏，而且似乎在他们这个年龄层非常的火爆，我一看这不就是一个线上的像素版或者说乐高版的《模拟人生》？

再后来《头号玩家》火爆上映。出了影院，我的脑子里再次浮现出那个年少时期看到的游戏 ——《模拟人生》。

你爱或者不爱，信或者不信，元宇宙就在那里，一步步的朝你走来……

Web2.0时代的元宇宙

元宇宙目前有个相对流行的定义：元宇宙是一个平行于现实世界，又独立于现实世界的虚拟空间，是映射现实世界的在线虚拟世界，是越来越真实的数字虚拟世界。

还有个相对流行的公式：Metaverse （元宇宙）= Create （创造）+Play （娱乐）+Display （展示）+Social （社交）+Trade （交易），如果按照这个定义和公式，你会发现《魔兽世界》或者《Roblox》几乎已经可以近似定义为一个元宇宙。

然而这只是Web2.0时代的元宇宙，这些元宇宙是彻底割裂的。

《魔兽世界》里的金币没办法去《Roblox》里面用，《魔兽世界》里的装备也没法带到《Roblox》里，《魔兽世界》的账号更是没法用来登录《Roblox》。因为究其本质，这些游戏或者说《元宇宙》的数据，都存在于创造他们的不同公司的服务器里，既不公开可见，也无法互通。

这不是我们想要的元宇宙，真正的元宇宙Web3.0才是起点。

Web3.0时代的元宇宙

当前Web3.0时代的元宇宙体验并不怎么好，甚至远远不如2.0时代，如果你既玩过《Decentraland》，又玩过《魔兽世界》的话，你肯定知道我在说什么。

然而这不重要，Web3.0时代的元宇宙才刚刚开始，还处在从0到1的时期，10年之后回看，一切将完全不同。因为Web3.0时代的元宇宙，有着2.0时代完全不具备的几个基础设施和技术条件，比如区块链、NFT、AR/VR、AI等。

让我假想一个Web3.0时代的元宇宙场景：

你回到家，带上头盔（眼镜），用私钥签名，进入一个叫做《地球》的元宇宙程序。你在客厅对着镜子看了看自己，满意的臭美了5秒钟，程序里的你比真实的你帅气不少，但是仔细一看，又能看出许多你的气质和神韵。

今天有演唱会，那个当年在《堡垒之夜》里和1000万粉丝在游戏里一起狂欢唱歌的大明星Travis Scott，现在在《元宇宙》也要搞演唱会了，你可不想错过。

于是你兴致冲冲的跑进后院，把之前一直在种的几棵菜（流动性挖矿）给收了，然后撕开一个卷轴，打开通向Uniswap大卖场的传送门，进去后在里面把刚收的菜卖掉，换成了手里印着ETH Logo的几颗金币 。

接着你扭头来到了隔壁OpenSea的售票处，里面那个好看的卖票小姐姐对着你甜甜一笑，你不由的心神一荡，脸一红，赶紧拿刚才赚到的金币换成了晚上Travis Scott的NFT门票。

出了门却突然反应过来，刚才的小姐姐大概率是AI，或者说NPC，自己居然被一个AI给“撩”了。这是你听到“叮”的一声，原来是系统在提示你，由于近期你在Uni提供了不少的流动性，完成了“种菜达人”的成就，系统提示你你的个人信誉点提升了 。

你抬头看了看天上飘着的“AAVE——您最忠实的宇宙银行”广告，琢磨着是不是可以去那边拿信用点贷点款，《地球》的玩家越来越多，最近好的地段地皮越来越值钱，据说当时Uniswap可是花了大价钱才买下来现在市中心的那一片地，盖了这个大卖场。

贷点款买块地脚不错的地，将来不管是升级自己的房子，还是卖给其他需要入驻的APP，应该都不会是亏钱的买卖。

离上面的场景我们有了什么，还缺什么？

1、我们有了区块链，有了可组合型，有了NFT，但还缺DID，缺TPS，缺链上的微博微信，缺万链归一……

只用过币安火币，没有玩过Uniswap和Defi的玩家，很难体验到Web3.0的那种“爽”感。

没错，助记词和私钥看起来很反人性。

然而你拥有一个私钥，所有应用对你开放，还可以互相组合的那种体验，你在适应之后根本回不去2.0时代那种每个应用需要申请一套独立的用户名密码的“反人类”。

你可能都不记得你拥有多少套账户，从最早的邮箱，到QQ，到微信，论坛，微博……对了，大概率对我们这些人来说，还有几十个交易平台的用户名密码……

DeFi给我们展示了一个Web3.0时代没有前端，没有炫酷画面的简陋版元宇宙雏形。这个元宇宙里所有数据公开记录在链，个人数据个人控制，私钥就是你穿梭这个元宇宙的唯一凭证。

这个元宇宙里目前有了可组合型，也有着NFT这种赋予物品唯一属性的技术或者说协议。然而我们还没有成熟的DID - （去中心化身份）系统，区块链的世界，谁说不需要身份证？你的账号，你的所有行为，构成了在元宇宙里那个“真实而又虚拟”的你。

我们没有TPS，哪怕是目前号称最速的Solana，在人数激增的时候也会卡到不行。和可以支撑淘宝双11上亿用户同时购买的2.0时代的基础设施相比，当前的区块链，弱的像个婴儿，ETH的rollup与分片，也还至少还有几年的路要走。

我们没有基于链上的各类通讯或是社交APP，Status做了那么多年也没几个用户，链上的邮箱，论坛，微信，微博，目前几乎还是空白一片，新项目永远都是DEX，借贷，NFT……

我也很难想象多链并存世界的元宇宙，因为每一个链都需要自己的账户系统，私钥，助记词……如果说每条链上都要有一个或者多个《元宇宙》程序，每个元宇宙都有自己的Uniswap，Opensea，AAVE，游戏……那这个web2.0时代的差别似乎又没有那么大了。

在我眼里，终极的元宇宙，应该基于一条唯一的区块链。然而要等到这样一个万链归一的时代，不知道还需要多久。

2、VR与AI——我们已经起步，需要的只是时间

也许，VR和AI进化的速度，要超过前面区块链上一堆美好的却有待实现的技术。毕竟VR和AI源于传统行业，已经发展了许多年，在Web2.0时代的元宇宙，他们扮演着同样重要的角色。

这里有两个关键词，Quest2，AYAYI。

强烈建议有条件的朋友入手一套Quest2，或者去体验店里体验一下也是很好的，它会让你对游戏，演唱会，纪录片等形式有一个全新的体验。

虽然Quest 2离最终的VR/AR形态依旧差距很大，但体验完后，你也许会乐观的觉得，《头号玩家》也没有想象的那么遥不可及。

至于Ayayi么……刚才场景里被一个AI小姐姐迷倒脸红心跳的场景，你觉得离我们很远么？

至于元宇宙的经济系统，等到实现的那一天，我相信比特币或是以太坊在人们心中的地位，和当下相比，不可同日而语。

一个元宇宙的悲观论调

最近看到朋友一篇文章，觉得很有意思，是关于元宇宙的“悲观论调”。

然而这个悲观，并非指元宇宙，而是人类自身。 众所周知，科技的发展其实依然陷入“停滞期”，绝大多数“低垂的果实”，早已被摘完。的确，我们有了互联网，有了苹果，有了特斯拉，有了facebook，进入了信息时代，生活更加方便，物质更加丰富……

然而1927年的美国航母，航速33节，90多年后的戴高乐号航母，航速却只有25节。1970年，第一台波音747从纽约飞伦敦用了八小时，50年后，依旧还是8小时。

人类50年前登上了月亮，当时的科幻小说普遍认为2021年，我们应该会有在天上飞着的汽车，穿梭于月球或者火星的星际飞船。然而现实是50年后的今天，我们不仅没再去过月亮，反而大家能做的是宅在家逛淘宝，刷抖音。

那么在一个没有星际旅行，没有登月，没有深空探索的地球，在一个贫富差距日益加大，躺平和宅文化日益流行的社会，元宇宙会是什么？

它本质上就是一个人类内卷的产物，因为无法向外探索和拓展边界，那么必然相内以某种方式卷曲， 去一个虚拟世界里获得现实世界因无力而无法得到的满足感 。

所以，元宇宙会不会是一个可悲的结局？也许看过《未来简史》和《三体》的朋友，会对这个观点有着更多的感悟。
