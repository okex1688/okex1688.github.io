越来越难分清到底谁是NFT的真正标签，艺术品、游戏、区块链、元宇宙、币圈，还是炒作？

今年被称为NFT 的“元年”，但是NFT到底是什么？

12岁小朋友靠卖一系列的像素画，一个暑假赚了大约250万人民币；Twitter联合创始人将一条仅有5个单词的推特以NTF形式拍卖，成交价超过290万美元，真“一字万金”；加密艺术家Beeple的数字作品“First 5000 Days”在佳士得单一拍品网上以6900万美元价格成交；B圈孙宇晨斥资1050万美元买下一个头像……

碰上NFT，一张图、一段视频、一幅画、一首歌，甚至一个表情包，身价就能呈指数型爆炸式增长。

包括今年元宇宙概念的持续热炒，也让市场看到了NFT的商业机会。在一些行业人士看来，元宇宙为NFT提供了更加多元的应用场景，NFT将成为实现虚拟物品数字资产化和流通交易的重要工具。

币圈新的掘金点又一次点燃了大众的文艺热情。其实，最早创作的数码艺术作品要追溯到表情符号。1982年9月19日，卡内基梅隆大学的计算机科学教授——科斯特·法尔曼，发明了人类进入信息时代以来使用最广泛的两个符号：笑脸和哭脸符号。现在聊天软件里著名的小黄脸表情包就是来自于法尔曼教授的创意。

现在这两个文字表情包，正放在全世界最大的收藏品拍卖公司——美国遗产拍卖行的主页上进行拍卖。现在的价格已经超过35万美元。因为这套文字表情每套只有六个字符，计算下来，每个字符的价格接近6万美元。

要创作开创性的作品不容易，而NFT蹿红，正在将大量普通数码艺术品的身价不断抬高。

从最开始无意间的尝试，到如今数字艺术品开始炒作、出现泡沫，我们通过梳理了几幅NFT数字艺术品，追溯NFT的发展史，还原这场逐渐变形的狂欢。

CryptoPunks：无心插柳的尝试

CryptoPunks算是最早的NFT项目 。这是一整套像素风格的图标，一共有1万个。最终生成的朋克有5种类型：外星人、僵尸、猿猴、男性、女性，有87种不同的属性_（发型、发色、眼镜、五官、肤色等）_。受70年代伦敦朋克运动的影响，大多人物发型张扬。

作者声称没有任何两个头像是一样的，每个都只能出售给一个人。

今年5月，其中9个作品首次在线下拍卖行拍卖，最终以超过1600万美元的总价售出，引起轰动。

9月28号的佳士得拍卖中，下图的头像又以3385万港元（约合人民币2876万）成交，远超预估价680万港元。

除了拍卖行，在全球最大的NFT交易平台opensea，最近7天的成交榜上，CryptoPunks也长期排名第一。

这个项目一开始并不成功。

2017年，Larva Labs的创始人John Watkinson和Matt Hall设计了一个自动生成头像的程序，本意是作为游戏或者社交软件的形象备用，但并没有太大反响 。为了丰富头像的多样性，他们运行调整了上百次原程序。后来他们尝试性地将图像上传到币圈，作品的独特性开始吸引收藏家的关注，后来逐渐成名。  

从opensea最新成交的几笔订单来看，现在每个头像售价100ETH左右，折合人民币28.7万元。

作为第一个吃到螃蟹的项目，天价背后更多的是“第一”这个特点的收藏价值。

前不久，和此相关的国内新闻“朋克头像+孙宇晨”上了热搜。

2021年8月31日，孙宇晨花1050万美金买了一个Tpunk的NFT作品，并且设置成自己的微信和推特头像。

孙宇晨就是那位花了3000万拍下巴菲特的午餐，最后又放了巴菲特鸽子的那位。这个价值1000万的头像，尺寸是24*24像素。可以说这是一个只包含了576个像素的数码图像，其中每个像素的价格高达一万七千多美元。

Tpunk项目其实是孙宇晨自家的公司——波场TRON生态中的NFT项目，类似CryptoPunks，总数也是1万，风格也偏朋克。此次高调购买是否是对CryptoPunks像素头像护城河的一次冲击，为自家生意宣传，各界猜测颇多。

NFT分为两类， CryptoPunks属于“不可繁殖NFT”类别的第一个项目，是无心插柳。另一类“可繁殖NFT”，第一个项目CryptoKitties也恰巧同年开始。

CryptoKitties销售各种猫的NFT，每只猫的花色、大小、品种都不同，而且性别不一样的猫可以繁殖下一代，继续出售。这种玩法后期在游戏中应用广泛，流行的元宇宙游戏axie infinity最突出，游戏中以宠物为核心，每只宠物都是一个NFT，品类、大小、技能等都不一样。这种可繁殖的NFT每次繁殖，最初所有者都会收到版权费提成。也因此衍生出一批在游戏中挣钱的专业玩家。

虽然数码艺术品的开端并不是来自专业艺术家，但这些项目的发展开辟了一种新的模式，为数码时代的复制粘贴找到了一种可能解法。 

the first 5000 days：数字艺术家入局

谈到NFT艺术品，都绕不开《the first 5000 days》，它是目前全球成交价最高的顶峰之作。

作者名叫Beeple_（毕普尔）_。根据作者自述，从2007年5月1日开始，每天原创并上传一幅数码艺术作品，并取名为《everydays》。这些作品有的是他的随笔，有的是精细度很高的艺术作品。截止到2021年1月7号，13年半的时间里，坚持了整整5000天，最后把这5000张数码作品拼贴到一起，取名为《the first 5000 days》，完成了这幅数码大作。

2021年3月11日，这幅画以6934.6万美元_（约合人民币 4.5 亿元）_的价格通过佳士得拍卖行拍出，进入目前在世艺术家的作品交易额排行榜的第三位。这件作品也是佳士得拍卖行首件以NFT形式拍卖的纯数码艺术品。

和上面不同，这幅作品的特点在于艺术性。 毕普尔在组合时以时间顺序为基础，将主题和颜色相近的作品放在一起，整幅作品左半边偏亮，右半边偏暗，带有强烈的个人艺术风格。

这位来自南卡罗莱纳州的平面设计师兼动态艺术家，作品以创新前卫见长，擅长将流行文化、科技、时事融入创作，在业内颇有名气。毕普尔Instagram粉丝超180万。

艺术家的专业价值是天价的基础，这种组合形式的大作，更是它的加成元素。 并且，这位艺术家在去年的一次NFT成交中就打响了名号。

去年 10 月，他在NFT交易平台 Nifty Gateway 上以 660 万美元售出一件10秒的视频作品——《Crossroad》（《十字路口》），刷新当时交易记录。在这幅作品里，一个巨人趴在地上，周围人来人往，却没有人停下来，哪怕多看一眼。

这幅作品比较抽象，也吸引不少媒体解读背后内涵。

毕普尔只是最具代表性的人物，早从2017年开始，NFT的交易方式逐渐被越来越多的艺术家接受，因为其核心价值——版权保护，正是当下所缺。 数码时代复制一件作品的代价非常低，有时正版发布后的几秒种，就能看到盗版。而NFT能非常方便地帮助作者完成完全不可更改的版权声明，也就是确权。这就为日后解决版权纠纷，提供了可靠的证据。

此次，毕普尔在作品《the first 5000 days》天价成交之后，做出了观众万万意想不到的举动。他居然立即趁热打铁，把这幅画裁成了105个小块，然后每一块又注册了一个NFT作品，再分别拍卖。结果每一个小块都卖出了十几万美元的好价钱。

虽然画作分割就属于完全不同的作品，是完全不同的NFT，操作上没有任何技术和法律问题，一般从业者从责任考虑都不会重复销售作品。不知道此次《the first 5000 days》最开始6934万美元的买家作何感想。

除了绘画技术占据NFT艺术的优势之外，如果作品对象本身就有特点，那么也一定会有人追捧，就如明星的流量变现。

NBA Top Shot：流量变现

这款NBA授权的NFT球星卡，包括一个几秒钟的短视频——记录球星在比赛场上的精彩瞬间，还有对应的球星封面的文字描述。新秀、当红巨星、退役老将都涉及。球星越著名，镜头越精彩，价格越贵。

据巴比特资讯，这种卡脱手暴涨而“卖飞”是常事，如一球迷用几百美元入手Series 1的詹姆斯卡，1000美元出手，起初窃喜大赚，过几天发现涨到3000美元。今年2月23日，竟然以20.8万美元的价格售出，创下NBA Top Shot单卡最高售价。不少球迷甚至还总结出“不能卖”的真经。

流量在本质上代表着大众认可度，也是市场最准确的指南针，利用得当，就是最好的营销。 作为一个大的流量池，NBA的全球忠粉无数，其中大量中青年男性已经有一定的经济实力，有能力消费球星周边。

这属于消费群体明确且潜力巨大的市场，NFT新玩法加持后，扩张迅速也确实在情理之中。具有关注度的名人尝试自建NFT，也是一样的原理。

姚明拥有位于美国纳帕产区的酒庄「YAO FAMILY」，今年发布其葡萄酒顶级品牌「The Chop」，也采用了拍卖限量版NFT的方式。

该NFT收藏品在Opensea上交易，大多定价在0.2ETF-0.3ETF之间_（约合人民币5000元-8000元）_。拍得者会获得这瓶实体酒和保真证书，特点是，这份证书会有姚明的亲笔签名。姚明的这轮销售，虽然定价稍高，但有明星效应和NFT热点的优势，最终平台销售一空。

除了名人之外，推特属于名平台，今年早些时候的推特NFT就是平台流量的变现。

2021年3月23日，推特联合创始人、首席执行官杰克·多西_（Jack Dorsey）_拍卖了他在推特上发表的第一篇推文，最终成交价超过291万美元。这篇2006年3月的推文内容是：我刚设置了我的推特。

多西宣布拍卖所得将捐给慈善机构。

正是平台流量基数大，相关纪念结点才具有收藏价值。

天价成交背后，越来越多人开始担心NFT艺术品，最终是否会成为稍纵即逝的泡沫？NFT艺术品代表着的“唯一”是否可靠？

《白痴》：艺术品的炒作

英国著名街头艺术家班克斯为了讽刺拍卖行炒作艺术品的行为，曾专门创作过一幅名叫《白痴》的版画。在他的这幅画里，以为拍卖师正在口若悬河，煽动台下的收藏家出价，而正在兜售的作品只是一个空空的相框，上面写着：我真不敢相信 你们这群白痴买下了这个东西。

更加讽刺的是，这幅画被一家 区块链公司花费9.5万美元_（约人民币61万）_买了下来，然后在一次直播中当场烧毁，随后它的电子版被制作成了NFT艺术品，放在了NFT的交易网站opensea上面拍卖，最后它的成交额超过240万人民币，翻了两番。比较讽刺。

那么，价格高昂的原理是什么？

我们可以把上传的图片，文本，word文档或者任意一段数码信息，通过一种叫做哈希算法的数学处理，生成一段长度固定的字符串，这个字符串的最大特点就是不可逆。

也就是没有任何人可以通过这段字符串可以反向推出原始文件是什么。并且原始信息只要有一丁点的不同，哪怕只是一个字节的不同，生成的字符串也会完全不一样。最后附上拥有者的身份信息，每个作品+所有者，就会产生一个不可替代的令牌，即NFT 。

因此，毕普尔的《the first 5000 days》和分割后的105份分属完全不同的作品，严格上讲，不算二次销售。钻了技术的空子。

NFT技术能够保证买家是这幅作品的唯一所有者呢？其实不会。

因为只要有哪怕一个字节的不同，最终生成的数字指纹都会完全不一样，这种“唯一”是有漏洞的。比如，想要复制一幅作品，在它隐蔽的地方稍做改动，甚至只是给它增加一个边框，都会形成一个新的NFT作品，称不上版权问题。真正能够保证卖家权益的还要靠与作者的交易约定，双方的诚信。

元宇宙：让万物真的可以NFT

当元宇宙的概念逐渐走红时，NFT不再局限于艺术品，已经成为通往元宇宙世界的关键密钥。

元宇宙，即Metaverse，这也是今年大火的概念。入局的公司名单包括，Facebook、微软、腾讯、阿里巴巴、字节跳动、B站、网易、耐克、戴森、LV、三星、LG……Facebook甚至all in元宇宙改名为Meta。

大家引用较多的概念是，元宇宙是一个可以映射现实世界、又独立于现实世界的虚拟空间。说白了，其实就是沉浸式体验，在虚拟世界里开展另一段“现实”人生。

伴随着元宇宙的热潮，外界都在期待其与NFT的联动。

“正像是物理社会中资产凭证的重要性一样，元宇宙生态也具有大量数字化资产，需要资产凭证来促进元宇宙经济循环。”电子技术标准化研究院区块链研究室主任李鸣认为，数字化资产凭证是元宇宙生态的关键要素，而NFT可以成为元宇宙中数字化资产凭证的一种表现形式，并且将随着元宇宙的发展逐步演进。

彼时，或许大到实体资产、艺术品、商业创意、知识产权，小到个人的微博、朋友圈、视频、游戏、玩具、宠物、照片……似乎现实世界的一切都可通过NFT连接到区块链世界。

也就是说，元宇宙会成为NFT的最佳容器。

在红利时期，普通玩家们纷纷入局。只要是唯一性的作品，都可以上传。作品价值的衡量在于购买者。有人上传一个简单的色块。

类似推特创始人杰克的方式，也有人把自己的推特截图做成NFT，来纪念特殊的时刻。如下图就是纪念自己的推特订阅数888和粉丝数88.8K，都由数字8组成，连价格也定位0.8ETH。

也有人上传几个简单的文字。

只是，这是一场关于未来的布局。

首先需要面对的是，元宇宙也为NFT带来了许多炒作的空间。

据凤凰网科技报道，原价39元的2022年杭州亚运会数字火炬，在拍卖平台上的标价高达314.9万。初始售价为“10个支付宝积分+9块9”的敦煌飞天NFT，最高价格也被炒到了150万元。

狂欢之下，NFT正在偏离原本的价值。对于普通人而言，也很难分清到底谁是NFT的真正标签，艺术品、游戏、区块链、元宇宙、币圈，还是炒作？
