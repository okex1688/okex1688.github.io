介绍

许多团队开始看到 web3 的潜力并开始进入该领域。 想法正在形成。 团队正在聚集在一起。 各种融资轮次正在关闭。

毫无疑问，我们将看到伟大的事情浮出水面，但这个行业仍然是新的，一些团队可能缺少 web 3 和代币等领域的专业知识。

如果您的团队来自传统游戏背景，我将在下面分享有关启动 web 3 游戏的一些关键挑战的高级概述。

有一个令人信服的“为什么是 NFT”

为什么需要区块链？ 它解决了什么“问题”？ 这是一个有趣的问题，应该在非常基本的层面上理解和回答。 这也是 NFT 的主要反驳论点之一。

我认为同样重要的一个较少问的问题是区块链技术可以实现哪些新系统或模型，从而使游戏变得更好？ 归根结底，区块链是一种工具，那么游戏应该如何最好地使用它呢？

可以回答这个问题的一些想法包括：

  1. 向那些对游戏应该如何运作没有“根深蒂固”观点的领域深入了解的人学习并获得反馈。

  2. 分析其他 NFT 游戏的功能——谁玩？ 他们为什么玩？ 他们如何进行分销？ 社区管理？

**掌握做开放经济设计的能力
**

今天的大多数游戏在设计时都考虑到了封闭经济。 他们玩、赚钱和留住玩家的方式是基于玩家不能交易的基本“法则”。 一旦你引入可交易的 NFT 或代币，这条法则就会被打破，现有的游戏设计模型也是如此。

搞清楚开放经济游戏设计需要对过去进行分析（例如 2000 年代的 MMO 在现实世界中进行非法交易），以及对未来的创新。

开放经济游戏设计是一种罕见的技能组合，对于许多现有的游戏设计师来说可能具有挑战性，因为它要求他们抛弃强烈持有的假设。

一些具有挑战性的问题包括：

  1. 有人可以在游戏中购买任何物品吗？ 这是否会破坏他们或其他人的游戏体验，就像在暗黑破坏神 3 RMAH 中一样？

  2. 您希望您的玩家如何将资产的财务价值纳入他们的游戏玩法？ 这会让游戏更有趣还是更有压力？

  3. 您的系统会激励哪些行为？ 你如何阻止它被机器人或“黄金农场”超越？

  4. 如果您的新玩家增长为零，您将如何使游戏经济可持续（价格和收入）？

在游戏之外思考并取胜

我们习惯了游戏就是产品。 打造一款出色的手机游戏意味着专注于应用内体验。 然而，NFT 和区块链的加入实际上迫使开发者专注于游戏之外的领域。 通常不是很重要的事情突然之间可能会成为使游戏成功的关键差异化因素。

“产品”不再只是游戏，还包括您的网站、您的社区、您的代币、您的市场、您的“工具”等等。 这为创新提供了重要机会，不容忽视。

你如何才能创造一种令人惊叹的游戏外体验来补充甚至超越你的游戏内体验？

#4 了解代币设计和操作

代币是一种非常有吸引力的工具，可用于 NFT 游戏。代币有很多好处，主要是它们是一种协调机制，可以使激励与所有参与者保持一致。

但是，对代币的理解和实现也很复杂。

首先是代币本身的设计 。会有固定供应吗？供应会波动吗？代币是如何从经济中引入或移除的？它们的用途是什么，用户为什么想要它们？

其次，还有代币的金融和投资方面 。目前，大多数代币交互都不是来自实际用户，而是来自这个“金融”生态系统中的其他参与者。游戏需要能够管理他们的代币库，了解如何与交易所合作，做市商做什么，DEX 和 CEX 之间的区别，什么是质押和流动性挖矿，等等。

第三，代币在操作上难以使用 。假设您的游戏成功并且您拥有价值 10 亿美元的代币。你如何安全地存储这些？贵公司中的哪些人可以访问它们？您如何使它们能够以编程方式部署在您的游戏中，或授权您的团队将它们用作奖励？不太容易。

最后，实施代币背后有很多法律复杂性 。许多事情可能会限制代币的设计，你被“允许”做什么，甚至只是你可以说什么。

作为合作伙伴和利益相关者建立社区

免费游戏和 NFT 游戏之间的一个很大区别在于，NFT 游戏中的社区通常拥有资产——无论是 NFT 还是代币。这意味着他们实际上是游戏的“股东”或部分所有者。通常在比赛结束之前。由于社区成员如此投入，不仅在经济上而且在情感上，社区建设的重要性在 NFT 游戏中变得更加重要。

除了社区管理的重要性之外，NFT 游戏在奖励和赋能社区方面也有巨大的机会。通常在免费游戏中，社区实际做事的动力有限。内容创作者、主播、开发者和其他社区成员通常无法从游戏中谋生，而是依靠广告等东西来赚钱。 NFT 和 web3 为游戏提供了颠覆这种模式的机会。

游戏如何与所有社区成员创造双赢的局面，从而激励社区中的每个人为游戏的成功做出贡献？

选择正确的区块链解决方案

就他们选择使用的技术而言，开发人员很难做出选择。 并非所有区块链都是相等的，理解区块链（以太坊、比特币）、侧链（Polygon、Ronin）和第 2 层扩展解决方案（Immutable X、Arbitrum）之间的区别很重要。

选择“错误”的成本可能会非常高，或者可能会扼杀你的游戏。

从本质上讲，区块链技术的选择是安全性、去中心化和可扩展性（区块链的三难困境）之间的权衡。 团队必须问自己，其中哪一个对他们最重要，因为区块链的性质意味着不可能同时拥有这三个。

其他要问的问题可能是：

  • 大多数人在它之上构建什么？ （网络效应）

  • 哪个对您的受众有最好的可用性？

  • 哪些开发者体验最好？

  • 100年后哪个会继续存在？

  • 其中哪些可能会影响您的融资或招聘？

了解 Web3 的“阴暗面”：模因文化、影响力者和不良行为者

最后，对于过渡到 web3 领域的人来说，最难理解的可能是加密的“黑暗面”。这些是大多数人甚至不会想到的事情，但在这个领域中很常见。它们在大多数情况下可能不会影响游戏，但当它们影响时会有所作为。

  • 您如何掌握 web3 的模因文化并利用它来发挥自己的优势？

  • 您如何区分“真正”的影响者与那些众所周知的“倾销”他们的关注者或只是通过发帖获得报酬的影响者？

  • 您如何与代币交易所和做市商合作，并区分那些有“不正当”行为（即贿赂或价格操纵）的人和其他人？

  • 您如何区分真正的顾问和长期支持者（钻石手）的加密 VC 与一旦解锁就会出售您的代币的那些？

  • 你怎么知道加密社区可能如何对 NFT 销售的价格、供应或时间决定做出反应？最近发生的事件如何改变这种反应？

构建 NFT 游戏所需的技能

NFT 游戏的新世界非常令人兴奋，开发和投资这些“新”能力的团队将是最有可能成功的团队。

web3 游戏中的下一代“工作”可能围绕以下内容展开：

  • 激励与经济设计

  • NFT 和代币集成

  • 社区和生态系统发展

  • F2P 到 NFT 游戏发布过渡

  • 用户驱动的内容和 IP 许可

  • 游戏+网络相交的产品开发

  • 区块链工程与安全

结论

综上所述，从传统的免费游戏过渡到 web3 有很多挑战。 每个主题都有自己的挑战和复杂性，没有正确或错误的答案。 NFT 游戏的部分乐趣在于为每个游戏找到自己的解决方案和方法。
