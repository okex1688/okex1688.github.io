**有人天生为王，有人自带光环，但无人能够完美一生。 **

Web3无疑是2022年少数几个还能维持光鲜亮丽的“词”。

Cryptopolitan数据显示，2022年第一季度，Web3.0初创企业已获得超1.73亿美元的投资，而2020年全年的融资数据仅为2000万美元。特别是2022年4月，Web3安全公司CertiK完成8800万美元的B3轮融资，Insight Partners、Tiger Global、Advent International，高盛、红杉资本、Lightspeed等国际顶级投资机构纷纷抛出橄榄枝。据悉，该公司在过去九个月内完成四轮融资，总融资额2.3亿美元，估值20亿美元，CertiK将Web3的光鲜亮丽表现得淋漓尽致。

正所谓树秀于林风必摧之，Web3的火热也引起了不少质疑声，例如艾隆·马斯克就与推特创始人杰克·多西（Jack Dorsey）就在推特上多次嘲讽Web3，当然还有一部分投机者打着Web3的幌子坑蒙拐骗割韭菜。那么Web3未来会朝着理想前进还是快速结束光鲜亮丽的一生就此打住？不妨从李鬼维基百科身上去看Web3光鲜亮丽的一生。

维基百科的Web3精神

将维基百科与Web3放在一起，主要原因在于维基百科具有Web3精神，即开放协作和数据主权。以太坊联合创始人 Gavin Wood（加文·伍德）在 2014 年提出了 Web 3.0的概念，认为基于区块链技术构建的 Web3 平台和应用程序不会由巨头所有，而是由每个用户拥有，他们将通过帮助开发和维护这些服务来获得所有权。对此，Gavin，运营了一个支持去中心化技术项目的 Web3 基金会，以及专注于为 Web3 构建区块链基础设施的公司 Parity Technologies（开发区块链项目 Polkadot）。

基于此我们再来看维基百科。维基百科自诞生20多年来，至今已收纳5500万篇文章，拥有189种语言版本，目前有近30万编辑(或称“维基人”)自愿贡献他们的时间来编写、编辑、锁定、讨论和打磨这个庞大的百科全书的每一个角落。他们称之为“项目”，致力于他们所说的五大支柱：维基百科是百科全书；维基百科是从中立的角度写的；维基百科是所有人都可以使用、编辑和传播的免费内容；维基百科的编辑应该以尊重和礼貌对待彼此；维基百科没有死板的规则，从这个角度来看维基百科亦满足Web3的开放协作精神。（WiKi是多人协助的写作系统）

网站的背后是维基媒体基金会(Wikimedia Foundation)，这是一家总部位于旧金山的非营利组织，负责服务、筹款、法律问题和其他推动项目发展的举措。维基媒体基金会最重要的任务是：更好地管理志愿者编辑，他们中约有4万人在英文维基百科上最为活跃，其中超过500人处于活跃的“管理”状态，这使他们能够行使权力，比如屏蔽和编辑受保护的页面。除此之外维基百科还有由全球无数志愿学者、玩家、学生等等有知识的人共同建筑的，该计划的参与者叫做维基百科人参与用户管理，与志愿协助管理工作的“管理员管理平台上的各种内容，从这个角度来看，维基百科不由巨头所有，而是由每个用户拥有，具备一定的数据主权。

Web3精神不是万能的

维基百科的Web3精神的确让维基百科得到了超乎想象的发展，成立仅3年后就超过了当时全球最大的百科全书——《大英百科全书》8 万词条的规模，并且在短短10年就成为了全球最受欢迎的50个网站之一，并且挤入世界品牌500强之列，目前维基百科有600万多个条目，它每天的访问量多达上亿次，从其成绩来看的确体现了Web3的无限潜力。

但是正如中国一句古话——灯下黑说言，即便是有Web3精神加持的维基百科近些年来其暴露的弊端也预发明显严重了。其一是“古希腊式的贵族假民主”，例如维基百科虽然号称管理员都是由用户自己选举，可以随时罢免，但是常常因为各种理由或同情难以真正罢免管理员，仅有少数因长期不在维基百科活跃的管理员真正不再担任管理员。再比如虽然维基百科号称任何人都可以在其平台上自由创建和编辑内容，但是用户创建的条目，如果被管理员认为包含商业信息则可能会被删除，即便它并不包含商业信息，以及知名度很高。这一点会让用户创建条目时的难度加大，一些维基人本身就在某企业工作或某学校上学，却常常苦于在自己公司/学校网站以外难以找到不含所谓宣传性的信息的内容，而编辑不了相关条目，总的来说其受益者仍逐渐被少数“贵族”掌握。

其二，维基百科没有实现去中心化，其管理权限最终还是落在了少数几个人手中，其弊端则是权力的滥用，例如2021年9月，维基媒体基金会以“存在安全风险”和“渗透维基系统”为由，对中国大陆维基百科编辑用户以及管理员用户组采取措施，永久封禁7名编辑，并撤销了另外12名编辑的管理员的权限。

正如马克思在《资本论》那句脍炙人口的话一样：如果有10%的利润，资本就会保证到处被使用；有20%的利润，资本就能活跃起来；有50%的利润，资本就会铤而走险；为了100%的利润，资本就敢践踏一切人间法律；有300%以上的利润，资本就敢犯任何罪行，甚至去冒绞首的危险，Web3固然在效率与权益上都更友好，但如果权力不关在笼子里那一定是一场新的灾难。
