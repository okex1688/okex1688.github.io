1. Kusama 什么时候开始拍卖？

在一轮拍卖和众贷功能测试后，Rococo 网络又完成一次 bug 修复和升级，目前已经重新启动成功。Parity 正在监控它的稳定性，然后再向社区开放平行链的注册。Gavin 正在更改一些不兼容的问题，这些更改将改变平行链执行 XCM 的方式，一旦完成，大家就可以随意打开 channel 并随意玩耍。在本次更新的新版本中，极大地提高了最大跨链消息的大小，所以这次大家会有足够的自由去测试。

目前 Rococo 上有 550 个验证人，但不是所有的验证人都在参与平行链的共识，波卡联合创始人 Robert 正在解决这个问题，将所有验证人都用到平行链共识中，这意味这次重启过后，Parity 和平行链团队会最大程度的增加每条平行链的验证人（目前设置的是 5 条，相信未来几天会慢慢增加到 10个），并进行稳定性测试。

预计最快需要测试一周到两周的时间，如果没什么问题，应该就可以上线 Kusama 网络。根据 Gavin 之前提到的平行链上线流程，Kusama 网络首先会通过治理投票（投票正常时间是 7 天投票 + 8 天执行期，如果技术委员会加速投票的话，需要 3 个小时的投票期，并直接执行）上线一条系统平行链 Statemint，之后才会开启需要通过拍卖的平行链。所以，一个大概的，并不保证的预计是最少一周，最多四周上线 Statemint 后，（可能也要通过治理启动 Kusama 网络拍卖和众贷的功能，投票时间同上）才会开启 Kusama 的竞拍。但是，这是 No promise 的预计, Expect chaos。

2. 上线 Statemint 有什么用？

根据波卡官方公布的信息，Statemint 是 Parity 计划开发并提议的一个通用资产平行链，会作为首批公共利益平行链之一。该平行链提供在 Polkadot 和 Kusama 网络中部署资产的功能。从 Gavin 在北京 meetup 上公布的视频来看，Statemint 的功能非常简单，即专门用来进行资产的发行。对于不能第一时间竞拍到平行链的项目来说，可能大家也会基于 Statemint 发行代币进行流通。包括 USDT 也会使用 Statemint 在波卡上发行 USDT 稳定币，Tether 的 CTO Paolo Ardoino 前些天对外公布，会在 Statemint 启动后在其上发行 USDT 。

3. 如果我想用 KSM 或者 DOT 支持某个团队成为平行链，有最低限额和最大限额吗？

理论上没有最低 KSM/DOT 的支持数量，但是会有一个上限，这依赖于这个团队需要的最大众贷金额，以及愿意分配多少奖励出去，所以一定有一个最大限额，但团队不用，这个限额也不同。大家可以在拍卖正式开始时，根据团队选择。

4. Polkadot 会不会上线 Coinbase？

目前没有看到波卡官方公布过相关信息。虽然大家都希望上线，我们期待一下吧！

5. 我的 DOT 为什么不能从钱包转出来？

为了减少尘埃账户，每个 DOT 账户的可用余额需要至少保留一个 DOT，所以如果你要转账全部的 DOT，没有剩下一个 DOT 的时候，或者转账小于 1 个 DOT 到一个空账户的时候，都会出现失败的情况。如果想把全部的 DOT 转出来，并且计划放弃账户，可以在转账的时候把 keep-alive 这个选项关掉。

6. 为什么我的 DOT 总是质押不成功？

质押 DOT 存在一个最小质押金额，并且这个金额在变化，目前显示是 257 个 DOT，也就是至少需要 257 个 DOT，才可以质押成功！你可以在这里查看实时的最低质押数：https://polkaview.network/dot
