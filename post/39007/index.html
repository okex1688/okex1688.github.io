在这篇文章中，我们将深入研究Clone X，以了解他们的全部情况，并查看链上数据，看看炒作是否真实。

介绍

在上一期的_More_ T__hen JPEGs 中，我们介绍了行业王者Bored Ape Yacht Club (BAYC)。 BAYC 的成功掀起了一波动物卡通 PFP 收藏的热潮。之后的大部分系列都在效仿，但只有RTFKT（Clone X 背后的工作室）开辟了自己的道路。

虽然 BAYC 以合作的形式成功吸引了阿迪达斯的注意，但当 RTFKT 被这家运动服装巨头收购时，他们获得了耐克的最大支持。在第二期_More_ T__hen _JPEGs_中，我们研究了 Clone X 以了解它们的全部情况，并利用 Nansen 的 NFT 分析来揭示对该系列更深层次的见解。

背景故事

Clone  X 的历史

Clone X并不是RTFKT（发音与Artifact相同）的第一次尝试。在发布Clone X之前，RTFKT已经将自己打造成了一个领先的Metaverse品牌。他们从创造虚拟运动鞋开始，慢慢发展到可以在Decentraland使用的时尚单品，甚至举办活动允许用户通过使用他们的NFT获得实体运动鞋。

如果没有自己的数字头像，一个Metaverse的品牌将是不完整的。 Clone X 的推出为 RTFKT 品牌开启了新的维度。拥有一系列 3D 头像为整个生态系统奠定了一个超越服装的基础。这个系列是与当今世界上最具影响力的日本艺术家之一Takashi Murakami（村上隆）合作完成的。

该系列于 2021 年 11 月 27 日为现有的 RTFKT NFT 持有者预售了1万个 NFT，并在 2 天后公开上线剩余的1万个。最初以荷兰式拍卖形式发售，起拍价为一个mintvial 3 ETH。每个mint vial允许持有者兑换一个随机生成的Clone。然而，对网站的攻击迫使该团队将发售的剩余vial改为标准化的2 ETH mint价格。

收购

在阿迪达斯进入metaverse和NFTs之后，围绕在每个人心中的问题是“耐克将有什么动作？”在Clone X推出大约2周后，耐克正式宣布收购RTFKT。收购整个RTFKT团队和产品线是对他们所建立的品牌质量的证明。

发展

自收购以来，RTFKT一直在继续发展，并与耐克一起建立了他们的生态系统。Clone X持有者已收到多个空投，所有这些空投都有不同的用途和用例。第一个是用作虚拟家庭的 Space Pods，随后很快推出了 PodX，它是 Space Pods 的扩展，带有 Nike 品牌特色。

MNLTH 是其中最神秘的一个。 NFT持有者有各种虚拟任务要做，社区必须团结起来才能找到谜语的答案。这又是 RTFKT在规划自己的道路，为他们的社区创造互动体验，而不是简单地在预定时间透露他们的计划。MNLTH持有者可以选择燃烧他们的MNLTH NFT 以获得3个独立的NFT —— Skin Vial Evo X、Dunk Crypto Kicks & MNLTH 2。

具有“Murakami drips”特征的Clone持有者也有权在Takashi Murakami最新的NFT系列Murakami.Flowers Seed中获得白名单，Murakami.Flowers Seed NFT可以变成Murakami.Flowers NFT。

深入 研究 链上 数据

要深入了解 Clone X 宇宙及其成功，让我们深入研究 Nansen上的各种NFT分析仪表板，以评估该系列的实力。

概 览

以下是该系列的总体概览：

  • 二级销售交易量约20.5万ETH （刷量交易过滤后），不过滤刷量交易二级销售交易量21万 ETH

  • 最高单日成交量 12,898 ETH（耐克发布收购公告的那天）

  • 过去30天日均交易量约为 456 ETH（刷量交易过滤后）

  • ~9.4K 持有者

凭借二级销售的5%版税，RTFKT 仅从 Clone X 就获得了大约 1.05万枚 ETH 的版税。

持有 时间 分布

大约有9.3万个唯一地址持有该系列中的1个NFT。地址数量在披露当天有3.5K个，但第二天迅速跃升至 6.3K，此后一直上升。超过 83% 的 NFT 已经被持有至少60天，约 74% 至少90天。地板价在4月达到 20 ETH 的峰值，此后跌至约 7.7 ETH。

持有者持有 NFT 的时间反映了他们对项目的看法，持有的时间越长，他们就越乐观。 74%的NFT持有至少 90 天，这使 Clone X 与其他蓝筹股项目一致，如 BAYC (74%) 和 Doodles (75%)。需要注意的是，与 BAYC 和 Doodles 相比，Clone X 是一个更新的系列，考虑到这一点，会使得资历分布更加令人印象深刻。

钻石 玩家 ？

除了检查拥有可观利润钱包的比例外，让我们看一下仅持有1个 Clone X NFT 的钱包比例和钻石玩家的地址数量，以评估当前地板价的可持续性。

Clone X 有相当大比例的钱包只持有1个Clone币。 Clone X此比例为71% ，低于 Nansen Blue Chip-10 指数中其他社交 NFT系列如 BAYC (81%)、MAYC (79%)、Doodles (79%)、Azuki (74%)，但与 CryptoPunks（72%）相似。虽然 NFT 的分布不像其他蓝筹股项目那样广泛，但 71% 仍然是一个健康的数字。加上持有时间分布，我们可以推断，即使 NFT 的分布不那么广泛，但持有者也会长期参与其中。

大约1.06万个Clone由从未出售过Clone的钱包所持有。虽然这个数字一直在下降，但我们看到它在当前的水平开始稳定，并逐渐呈现上升趋势。

为了了解持有者的心理，另一个需要查看的指标是上架活动。

该系列只有7.8%的Clone X NFT上架，0.24% NFT的价格在地板价的15%范围内，大多数 NFT的价格在16 - 31 ETH范围之间。这表明大多数持有者并没有考虑出售，而一小部分持有者在期待高于当前地板价。

Clone的上架数量一直在逐渐下降，直到最近几天上架数量飙升至1084，是过去两个月来的最高水平。

在这样的价格下买入是否明智？

价格已经从高位回落，但现在买入的价格是否就是最低价呢？

我们分析了聪明投资者的趋势和购买活动以获得一些洞见。

看一下聪明投资者的NFT持有趋势，有两件事很突出。

  • Smart NFT Sweepers 和 Smart NFT 交易者的数量随着时间的推移而减少，Smart NFT交易者持有的NFT数量显著下降（从41个NFT减少到5个）

  • 聪明 NFT 持有者和聪明NFT 早期采用者的数量和他们持有的NFT数量都在增加。

从整体上看，聪明投资者持有的 Clone X NFT 数量有所下降，主要是由于 NFT 交易员退出他们的头寸所致。交易员通常的时间跨度较短，并且在动荡的市场条件下更具防御性，这解释了 5 月中旬风险敞口的下降。然而，倾向于长期时间跨度的聪明投资者的地址一直在增加。这表明他们对该系列的长期前景持乐观态度。

深入研究自 4 月 29 日以来的交易数据，我们发现大部分NFT的购买都来自普通买家。在过去 2 周，来自聪明投资者的地址仅完成了 2 次购买，并且购买价格接近当时的平均售卖价格。

Clone X 持有者还拥有哪些其他系列？

不出所料，Clone X持有者拥有的前5个系列都与RTFKT或 Murakami 有关，其中 PodX 是最常见的。除了 RTFKT 生态系统，我们看到持有者也对 Bored Ape 宇宙感兴趣。 Bored Ape Yacht Club 及其相关系列构成了 NFT 的下一部分。

有趣的是，我们看到阿迪达斯的NFT系列排在第8位。虽然 Travis Scott 可能会选择“Checks over stripes”（意思是选择耐克而不是阿迪达斯），但该规则似乎在Metaverse中并不普遍。

Clone 的未来

除了发布与NFT 相关的实物商品之外，RTFKT 还利用AR工具来帮助连接这两个世界。这使得他们的虚拟商品可以在现实世界中呈现，并在整个社交媒体上共享。Clone X 是他们正在创造的宇宙的基础，并作为该世界居民的头像。通过开源开发和创作者挑战（creator challenge）带来的想法，他们允许任何人参与构建这个宇宙。能有像耐克这样的巨头资源，只会增强他们的交付能力，使他们能够以只有耐克这样规模的品牌才能做到的规模来执行。

结论

Clone是 RTFKT 正在构建的世界的一个组成部分。该团队的不同之处在于他们能够建立一个将数字与实体相结合的品牌。现在判断 RTFKT 还为 Clone X 持有者准备了什么还为时过早，但如果过去的表现能预示未来的成功，我们可能会看到这个团队不断壮大并继续给人留下深刻印象。该NFT系列有相当多的持有者，他们对这个NFT的未来持乐观态度，具有长期视野的聪明投资者持有者似乎也同意这种看法。
