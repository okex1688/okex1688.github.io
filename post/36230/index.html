继今年1月在国际消费电子展（CES）首次宣布将NFT功能集成到新型电视计划后，三星电子新近与国际知名的NFT市场合作，迈出了智能电视售卖NFT的实质步伐。也许就在今年，人们可望一边看电视节目、一边交易数字NFT艺术品，还可以方便地参与三星的元宇宙派对了。

合作开发“智能电视 NFT 平台”

国际电子巨头三星电子与知名的NFT 市场Nifty Gateway 合作，将共同开发“首个智能电视 NFT 平台”，可用于购买和交易数字艺术品和收藏品。

在3月31日的新闻发布中，科技公司Nifty Gateway宣布，已经将其NFT 市场整合到三星的 2022年高端电视产品中。Nifty Gateway是Griffin兄弟俩于 2018 年创立的基于以太坊的交易市场， 2019年被加密资产交易所 Gemini 收购。

三星与Nifty Gateway 合作开发基于智能电视机的 NFT 观看、购买和交易体验。Nifty Gateway NFT 平台将集成到三星 2022年新的高端电视系列中，包括其 QLED 和 Neo QLED 套装，而三星 The Frame 和 Micro LED 套装的所有者可以下载 Nifty Gateway 应用程序以获得访问权限。

双方合作的解决方案，将利用Nifty Gateway 的托管钱包系统，可以保护用户的数字收藏品，而不必存放在外部钱包中。同时，三星用户们还可以搜寻经过验证的NFT，使用包括借记卡、信用卡或加密货币在内的各种支付方式，购买数字收藏品。

“通过我们与全球电子行业领导者之一的三星开展合作，我们将支持数百万主流用户接触到 NFT，”Griffin Cock Foster对采访的媒体介绍，“去年，NFT迎来了专门针对艺术行业的突破性时刻。创作者和收藏家们已经看到了这项技术的力量。”“今年，我们希望通过首次提供无缝购买和收集 NFT 的方式，见证主流消费者的持续增长。”

据悉，三星与Nifty Gateway合作后，用户将能够在 Nifty Gateway 平台上查看各类数字艺术品，包括来自Beeple、Daniel Arsham、XCOPY、Pak等知名艺术家的6000多件数字艺术作品。去年秋天以来，Nifty Gateway开始超越自己的策展，为客户提供来自其他市场的更广泛的 NFT 收藏品。

首发交易NFT的智能电视

今年1月上旬的2022年拉斯维加斯国际消费电子展 (CES)，韩国电子巨头三星在首次宣布将NFT功能集成到该公司新型电视中的计划。三星参展的产品，包括了一系列配备App的电视，可支持用户们浏览、购买和销售不可替代代币 (NFT)数字资产。

当时，三星官方解释：“这个App具有一个直观的集成平台，用于发现、购买和交易数字艺术品。”三星也把该App称为“NFT 管理”功能，成为实质性的NFT平台。

该App将作为新的 Micro LED、Neo QLED 和 The Frame 型号的标准配置，成为新的“智能中心”（Smart Hub）的一部分，“将内容管理和发现功能放在用户观看偏好的前面和中心位置。”

据悉，该平台将“从多个市场引入NFT，用户可以预览数字艺术品并查看其他相关细节”，包括“谁创作了作品，以及相关的区块链元数据”等细节，“已经拥有 NFT 的人，也可以使用该平台来展示他们的收藏。”

包括 The Frame在内的多款三星电视型号，此前曾作为可兼作数码相框的设备进行销售，用户由此可以在不看电视或流媒体视频时展示传统或数字艺术品。

为了支持电视观众深入研究更广泛的 NFT 世界，包括利用自己的电视机进行交易，三星很可能还会利用已在区块链钱包中使用的一些技术，这些技术已成为三星旗舰版智能手机的标准配置。

开设 “ 限时 ”元宇宙 商店

1月7日，三星在最知名的元宇宙项目 Decentraland (MANA) 上，开设了一家“限时”商店。

三星将该商店命名为三星 837X，对应三星位于纽约的美国旗舰实体店的地址-曼哈顿华盛顿街 837 号，称其为“来自于三星位于纽约市旗舰店的突破性虚拟世界体验”。

虚拟商店虽然仅在Decentraland的有限时间内运营，但会与实体旗舰店的活动形成互动。比如赢得独家 NFT的“神秘任务”，在实体店和虚拟店举行“现场混合现实派对”等。

另外，三星还将与Veritree一起在虚拟商店运营一块可持续森林，计划在2022年第一季度结束时“种植” 220万棵树。Veritree 是一个基于Cardano (ADA)公链的气候控制平台。

为了吸引Dencentraland玩家们的注意力，只要参观三星虚拟商店，就有机会获得 837X NFT 徽章，然后徽章持有者参加抽奖活动，为他们的 Decentraland 头像赢得从史诗级到神话级的三种限量版可穿戴收藏品。

推出 新型 数字钱包

2月份，三星推出了具有新钱包功能的最新旗舰智能手机，支持用户存储数字文件以及加密资产。

这款名为 Galaxy S22 Ultra 的设备，能够以数字形式存储 ID 文件，包括数字信用卡和借记卡，学生证、驾照和身份证，用于房屋和汽车钥匙的数字钥匙，以及登机牌、冠状病毒疫苗接种记录等。预计在今年年内首先在韩国国内市场推出。

三星称这款钱包是“一种无缝、方便和安全的体验，让日常生活变得更轻松”，它“将数字支付、身份证、钥匙和资产管理结合到一个工具中，以简化日常生活与工作之需。”

近年来，包括三星在内的多家韩国公司一直在研究涉及区块链钱包的解决方案。这些钱包支持用户存储多种数字文件，包括基于区块链的数字化驾驶证。

越来越多的韩国企业使用分布式身份识别（DID）技术，开始采用数字化文档的形式，这些文档几乎普遍存储在钱包中，与用于存储加密货币和不可替代代币 (NFT) 的加密钱包没有差异。
