Loot 是与众不同的，这是以前从未出现过的，它是 NFT 领域的范式转变。

那 Loot 到底是什么？

Loot 是一种仅包含文本的链上 NFT，是的，它只是 text 文字！

这是以文本形式而存在的冒险项目列表，当你铸造一个时，你会得到一组物品，嗯，是以文本的形式。

就这么简单。

那它的发行者 @dhof 是如何分发它的呢？

他在 3 天前在 Twitter 上公布了这个项目：

  1. 它是免费领取的；

  2. 铸造者只需支付 gas 费用即可铸造 Loot；

  3. 起初没有网站；

  4. 除了那条推文之外，没有其他任何信息；

  5. 你必须通过 @etherscan 上的合约直接铸造 Loot；

而目前一个 Loot NFT 的地板价是 4 ETH !!! （译者注：当前已升至 7ETH）

你可能会困惑，为什么一个只有白色文本和黑色背景的 NFT 会值这么多钱。

下面我们来解释原因……

通常，NFT 项目采用的自上而下的方法。

这意味着创作者创造了一些类似于猿的头像图片、游戏物品或艺术 NFT。

然后，创作者扩展生态系统，为 NFT 带来更多价值，这会为收藏者的收藏品增加价值 / 价格。

创作者为收藏者创造了 hodl 的激励——增加了 NFT 的稀缺性。这可能是为持有者空投另一个 NFT，等等。

基本上，创作者卖的是像房子一样的「成品」。 然后，创作者会让社区变得更好或给你额外的家具。

而 @dhof 的 Loot 采取了不同的做法，这是一种自下而上的方法，相比创作者在发行 NFT 后为其带来价值，@dhof 做的是让社区来决定 NFT 的价值。

这是构建 NFT 项目的一种新的、更去中心化的方式。

通过这种方法，NFT 的价值不再与单个实体或个人挂钩。

听起来是否有点熟悉？

这就是以太坊的工作方式！你可以直接跳入 Github 并编写代码，你可以通过撰写新 EIP 提案来为以太坊贡献新想法。

相比卖一栋房子，@dhof 卖的是砖头，他让你决定用这些砖块建造什么。

你可以用它来盖房子，你可以建一座桥，你可以建一个体育场，这是你的自由。

Dom 创建了一个画布，然后收藏家来画它。

loot 项目的未来取决于收藏家，如果人们购买 Loot 而没有人去建造任何东西，那么，Loot 大概会是一文不值的。

但如果人们购买 Loot， 并在它上面建造了一些东西，那么它的上限就是天空。

好吧……那么人们在用 Loot 建造什么呢？

  • 稀有度发现工具

  • 专属社区

  • 艺术

  • PFP

  • 元宇宙

  • 游戏？

  • 小说？

这个社区甚至已经构建了一个 SDK 来围绕 Loot 构建东西。

不相信我？

你可以听听 Dylan Field、stani.jpg、jstn.eth 以及 John Palmer 等人的意见。

我们可能会进入 NFT 领域的新时代。

没有领导，没有路线图，没有核心开发者， 没有核心艺术家，一切都是去中心化的。

当然，这并不是财务建议，这 100% 是一个实验项目。
