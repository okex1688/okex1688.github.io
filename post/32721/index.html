2021年是Layer2生态迅速发展的一年，链捕手盘点了Arbitrum、Starkware、Optimism、zkSync、Metis、Boba Network、Aztec等七个主要项目在2021年的主要进展与发展状况。

1、 Arbitrum

Arbitrum最初是在普林斯顿大学的一个学术扩容项目，在2020年由研究人员、工程师和以太坊爱好者组成的世界级团队Offchain Labs 构建，于2020年10月推出测试网。Arbitrum解决了区块链的三难问题：可扩展性、去中心化、安全性。 Arbitrum rollup有能力在不影响主网的去中心化和安全性的情况下扩展以太坊的规模，也是最兼容EVM的方案。

目前，Aave、Curve、MakerDAO、Chainlink、MCDEX、Uniswap V3、WePiggy、Sushi swap、SynFutures等主流项目都部署到Arbitrum。根据L2BEAT数据显示，截至1月6日，以太坊Layer2上总锁仓量已达到60亿美元，其中Arbitrum为锁仓量最高的为扩容方案，为27.4亿美元，占比46.8%。

目前，以Arbitrum为主要建设重心的项目包括GMX、Dopex、Tracer、Premia、Umami Finance、Swapr、Cap等项目，锁仓量均在2000万美元以上。

主要进展：

2021年5月28日，Arbitrum 向开发者开放主网测试版Arbitrum One。

2021年8月底，Arbitrum开发商Offchain Labs宣布以12亿美元估值完成1.2亿美元B轮融资，Lightspeed Venture Partners领投，Polychain Capital、Ribbit Capital、Redpoint Ventures、Pantera Capital、Alameda Research和Mark Cuban等参投。

2021年9月1日，Offchain Labs宣布正式上线 Arbitrum 主网，已删除白名单机制并向用户开放所有项目，同时公布了资产桥接教程。同时，Arbitrum还在Immunefi平台启动100万美元漏洞赏金计划，主要用于悬赏智能合约层面的漏洞发现。

2021年10月，Arbitrum 免许可代币桥接功能对所有项目开放。该功能的默认桥接合约类型为标准 ERC-20，支持基本ERC20 功能以及在 L1 和 L2 上的可传输性；此外，也支持自定义ERC-20，允许项目方或个人对 ERC20 代币部署治理、快照等功能。

2021年10月，Arbitrum 宣布即将推出 Arbitrum One 的下一个版本 Arbitrum Nitro，新版本与 EVM 更兼容且比当前的技术快一个数量级。团队预计 Arbitrum Nitro 运行后第 2 层执行速度将提高 20 至 50 倍，成本会下降。

2、 StarkWare

StarkWare成立于2018年，是基于STARK、 ZK-rollups 技术的L2扩容解决方案，旨在为区块链应用程序提供安全、去信任和可扩展的操作，目前StarkWare总TVL达到12.1亿美元，dYdX、Sorare、DeversiFi、Immutable X、Celer Network等较为重要的协议均在StarkWare上部署。暂未发行代币。

主要进展：

2021年3月，Starkware完成由Paradigm领投7500万美元融资，红杉、Founders Fund、Pantera、Three Arrows和Alameda Research跟投。

2021年11月，StarkWare在 C 轮融资中筹集了 5000 万美元，估值 20 亿美元，该轮融资由红杉资本领投，其他投资方包括 Paradigm、三箭资本、Alameda Research 、IOSG Ventures、Founders Fund等。

2021年11月底，StarkWare上线了基于 ZK Rollup 概念StarkNet Alpha主网，支持与其他 StarkNet 合约以及通过 L1<>L2 消息与 L1 合约的可组合性。

2021年12月，StarkWare提出Layer3解决方案。

3、 Optimism

Optimism是采用的 Optimistic Rollup 方案更容易支持通用型智能合约，即当前的一些当红 DeFi 项目可以更快地实现 Layer 2 无缝迁移。通过在链下运行计算，Optimism 能够将以太坊 gas 费用降低 10,000% 或更多，并将吞吐量提高 200 倍。

目前，Optimism生态的主要项目包括Synthetix 、Lyra、Uniswap 、Perpetual Protocol 等。

主要进展：

2021年1月，上线基于以太坊主网的测试网，得到 Uniswap、Compound 以及合成资产 Synthetix 的深度参与 。

2021年2 月 ，Optimism完成由a16z领投 2500 万美元的 A 轮融资。

 2021年8月，Optimism 支持美元稳定币 USDC，用户可在 Optimism Gateway 上选择 USDC 进行存款。

2021年9月，Optimis将主网的的吞吐量限制提高了一倍。在开始限制更高的gas价格之前，每天最多可以处理20万笔交易。

2021年11月，Optimism完成EVM（以太坊虚拟机）等效性升级，将OVM2.0更改合并至其主要分支中。

2021年12月，Optimism取消白名单，完全开放部署应用程序权限。

4、 zkSync

zkSync是一个由Matter Labs开发的L2的扩展解决方案，使用ZK-Rollup技术处理以太坊区块链上的交易，同时仍然保持其安全性。zkSync上的gas费用相当于以太坊gas的1%，比使用Optimistic Rollup更便宜，每秒2000+交易(tps)，而以太坊上是14tps，此外还可以随时将资产移回L1。

2021年大事件：

2021年5月，zkSync推出zkSync 1.x测试网，支持NFT和兑换功能。

2021年6月，zkSync 2.0上线zkEVM测试网Alpha版，用户可使用区块浏览器查看zkSync 2.0上的活动。

2021年8月，Matter Labs宣布zkSync 2.0版本将延期上线。

2021年11月，zkSync开发团队Matter Labs完成由a16z领投的5000万美元融资，Placeholder、Dragonfly、1kx、Blockchain.com、Crypto.com、Consensys、ByBit、OKEx、Alchemy、Covalent等参投。

2021年11-12月，zkSync开发团队Matter Labs相继与OKEx、火币、Banxa等达成合作，为该项目提供法币出入口。

5、 Metis

Meits是一个Layer 2 + DAO基础协议，采用Optimistic Rollup技术路线，于今年4月上线Alpha测试网，并于今年11月上线主网，目前TVL近2.7亿美元。

Metis生态主要项目为去中心化交易平台Netswap，目前总TVL超过8500万美元。

主要进展：

2021年3月，Metis完成A轮融资，投资机构包括 OK Block Dream Fund、ConsensusLab、YBB Foundation、Gate.io等。

2021年11月，Metis宣布联合合作伙伴推出 1 亿美元生态基金，分配给包括 DeFi、游戏和 DAO 等垂直领域内的项目。

2021年12月，Metis相继与Poly Network、cBridge等跨链桥合作，支持用户将其它网络资产转入Metis网络。

6、 Boba Network （ BOBA ）

Boba 是下一代以太坊 Layer 2 Optimistic Rollup扩展解决方案，可降低Gas费用、提高交易吞吐量并扩展智能合约的能力，由 Enya 团队作为 OMG 基金会核心团队 Enya 构建，由 OMGX Network 改名而来。Boba提供由社区驱动的流动性池支持的快速退出，将 Optimistic Rollup退出期从 7 天缩短到仅几分钟。

目前，Boba的TVL约为4.2亿美元，主要生态项目为去中心化交易所OolongSwap。

主要进展：

2021年8月，去中心化基础设施解决方案的开发商Enya推出Boba Network的主网测试版。

2021年9月，Boba主网上线，OMG基金会为该网络推出新治理代币BOBA。

2021年10月，Boba Network主网上线 OVM 2.0 版本，开发者无需修改即可在 Boba Network 上部署以太坊 Layer1 上的合约。

2021年12月，跨链网络cBridge支持Boba Network。

7、 Aztec

Aztec是目前以太坊生态中尤其关注隐私问题的Layer2解决方案，旨在为以太坊带来可扩展性和隐私性，致力于通过零知识证明实现负担得起的私人加密支付。目前，Aztec的TVL为866万美元。

主要进展：

2021年3月， Aztec 正式在主网上启动 Aztec 2.0 Rollup，已启用针对 ETH 的隐私 Rollup 服务和区块浏览器（Block Explorer）。

2021年12月，Aztec面向以太坊DeFi的私人桥接Aztec Connect已经上线测试网，可节约高达100倍的gas以及完全私人交易默认。

2021年12月，Aztec Network 完成由Paradigm领投的1700万美元A轮融资。
