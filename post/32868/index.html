区块链项目Chainlink将智能合约与现实世界数据连接起来，垄断了预言机市场。对智能合约和预言机的需求是相互关联的，因此无论哪个智能合约区块链成功，都可能导致Chainlink的增长。

就其本身而言，智能合约是相当基础的。与网络或计算机应用程序不同，它们无法访问来自现实世界的数据，并且仅限于区块链上存在的数据。这对于简单的用例可能已经足够，例如智能合约托管账户，但创建更复杂的用例就变得不可能了。

区块链预言机解决了这个问题，让智能合约能够有效地从外界检索数据以用于合约。 预言机通过向众多不同来源询问信息来获取特定数据，例如 Coinmarketcap 和 Coingecko 的加密货币价格。然后，他们取这些数据的平均值并将其提供给智能合约，以确保结果准确。所有不同类型的数据，包括价格信息、体育比赛比分、随机数，甚至天气信息都可以在区块链应用程序中获取和使用，这为智能合约打开了一个全新的世界。

Chainlink 是最早的也是最大的区块链预言机项目。它由 Sergey Nazarov 和 Steve Ellis 于 2017 年创建，作为连接链上和链下数据的一种方式。从那时起，它已经发展到有效地垄断了智能合约生态系统，现在被用来确保超过 760 亿美元的链上价值，并集成在1000多个项目中。几乎每个智能合约平台都能够与 Chainlink 集成，包括 Ethereum、Avalanche、Solana、Cardano、Polygon、Polkadot 和 Cosmos。他们还拥有向其预言机提供数据的优质公司，例如美联社和 AccuWeather，与亚马逊网络服务等巨头合作。在加密货币开发界，Chainlink可能成为预言机的代名词。

为了访问 Chainlink 的数据，智能合约需要支付 LINK，然后将其适当地分发给数据提供商。这自然意味着，随着对智能合约和数据需求的增加，对 LINK 代币的需求也会增加。

Chainlink 当前最大的问题之一是缺乏持有 LINK 代币的动力。一旦购买 LINK 以用于智能合约并将其发送给数据提供商，他们就没有动力持有它，然后再出售它来赚取利润。因此，LINK 的价格目前是相当投机的。幸运的是，Chainlink 计划在 2022 年的某个时候为供应商引入一种质押模型。这将使供应商能够在他们的 LINK 上赚取收益，从而有动力持有他们的 LINK，最终减少供应并增加需求，从而导致价格上涨。不幸的是，公众可能无法使用质押，但如果他们选择创建某种委托模型，这种情况可能会改变。

尽管 Chainlink 没有看到与其他代币一样令人兴奋的价格走势，但它有必要的使用案例，不会很快消失。凭借其对区块链预言机的垄断，任何智能合约平台的成功都意味着 LINK 的成功，而质押将进一步增加对代币的需求。对 Chainlink 的质押实际上是对区块链未来的押注，这比押注一个特定的智能合约平台更加安全。
