以太坊代币是去中心化金融生态系统的支柱，有朝一日可能成为全球金融体系的支柱。让我们看看到底什么是以太坊代币，以及人们如何在零编程经验的情况下创建自己的代币。

以太坊于 2015 年发布时，其最具革命性的功能之一是能够在其区块链上创建和发布智能合约。

智能合约是用代码编写的，并且具有自动执行的属性，这意味着如果编程正确，则始终会出现正确的结果。 因此，即使参与合约的一方试图恶意行事，他们也无法做到，合约将按预期执行。智能合约有无限的可能性，它们使商业、金融和其他行业更高效、更可靠以及更安全。

智能合约最常见的用途之一是创建加密货币。 例如，某人可以创建一个智能合约，并在该智能合约中编写一行代码，成为 100 个 XYZ 代币的所有者。尽管这个人只是简单地创建了这枚代币，它没有任何价值，但他们现在可以将它用于许多不同的用途，例如与朋友交易，作为收藏品持有，甚至创建公司，像发行公司股票一样分发这枚代币。事实上，这与 tZERO 公司的目标相似，他们试图将公司股票代币化，以便它们可以在区块链上进行交易。

许多最大的加密货币要么开始于以太坊，要么仍然是以太坊上的代币。例如，在其主网上线之前，Tron 只是以太坊区块链上的一个代币，其他代币包括 Chainlink、Shiba Inu、 FTX 代币以及所有稳定币。

现在我们对代币及其历史有了一些了解，让我们学习如何创建自己的代币，而无需任何编码经验。

第一步是下载并设置 MetaMask 钱包。 MetaMask 是一个基于浏览器的以太坊钱包，允许我们与区块链进行交互。在完成设置过程时，请务必记下您的私钥，因为这是在您丢失计算机时访问钱包的唯一方法。设置完成后，您的 MetaMask 应如下所示：

接下来，将网络更改为 Ropsten 测试网。 由于以太坊区块链上的所有内容都具有经济影响，因此存在测试网络，开发者可以在其中部署免费版本的以太坊智能合约。请单击 MetaMask 顶部的“以太坊主网”（Etherum Mainnet）按钮，单击“显示/隐藏测试网络”，然后选择 Ropsten 测试网。

现在，我们需要获取一些免费的以太坊来创建代币。请访问 https://faucet.dimensions.network/，输入您的以太坊地址，然后等待 5-10 分钟，让以太坊出现在您的钱包中。

现在，我们准备好编码了**！首先，前往 https://remix.ethereum.org，这是一个用于创建和发布智能合约的在线开发环境。** 然后，单击合同文件夹正上方的“创建新文件”图标，并将其命名为“TokenName.sol”。要创建代币，请复制并粘贴以下代码：

//SPDX-License-Identifier: UNLICENSED

pragma solidity >=0.4.22<0.9.0;

import “@openzeppelin/contracts/token/ERC20/ERC20.sol”;

contract TestCoin is ERC20 {

constructor() ERC20(“TestCoin”, “TEST”) {

_mint(msg.sender, 100000000000000000000000000);

}

}

您可以将名称从“TestCoin”和“TEST”更改为您喜欢的任何名称，也可以修改初始供应量，即“mint”功能旁边的数字。

最后，点击键盘上的 “control + s ”保存并编译代码，然后单击屏幕一侧的图标打开部署程序。

在这里，将环境从 JavaScript VM (London) 更改为 Injected Web3，并在出现提示时连接您的 MetaMask。最后一步是从“合同”下拉菜单中选择您刚刚创建的合同，然后单击部署。

现在，如果您访问 https://ropsten.etherscan.io 并搜索您的地址，将能够在您的帐户中看到您的新代币，并且您可以随心所欲地使用它们。祝贺您成功创建了属于自己的代币!
