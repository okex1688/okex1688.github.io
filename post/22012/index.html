5月11日，全球领先的数字资产服务商火币宣布，将于新加坡时间5月20日正式开启第七期Huobi Global优选上币通道（Huobi Prime）。

火币Prime旨在挖掘区块链潜力应用，聚集行业资源，推动优质区块链项目快速落地和发展，在全球范围内遴选头部优质项目，通过打折交易的方式，让利给HT持有者。此前，火币Prime于2019年3月20日启动，先后优选出六期项目在主板上进行交易。

根据官方规则，本期Prime活动分为两轮：第一轮为抽签，第二轮为等比例瓜分。抽签轮中，用户在活动页面中使用HT下单参与抽签，下单完成并获得有效订单号后，即视为成功参与抽签。抽签结束后，页面会显示用户所中签数，系统将自动扣除中签数所对应的HT，并分配对应代币与返回剩余HT。第二轮则是等比例分配，规则按买一价格上的挂单数量进行等比例分配。

值得注意的是，所有参与Prime的用户均需要满足一定的持仓要求。5月14日00:00:00至5日19日23:59:59，要求用户每日最低持仓达到300 HT，并于5月20日看到最终签数资格。

本期优选上币规则具体如下：

1. 交易方式： 用户可在Huobi Global的HT交易区，以交易的方式进行买卖；

2. 参与资格： 2021年5月14日00:00:00-2021年5日19日23:59:59每日最低持仓达到300 HT；用户可每日在持仓日历中查看HT持仓数额，在2021年5月20日看到最终签数资格；

3. 本期Prime用户成交限额： 对应HT兑换比例将会在抢购当日公布；

4. 交易规则：

（1） 第一轮为抽签。

用户在活动页面中使用HT下单参与抽签，下单完成并获得有效订单号后，即视为成功参与抽签。

抽签结束后，页面会显示用户所中签数，系统将自动扣除中签数所对应的HT，并分配对应代币与返回剩余HT。

第一轮新增星级用户概念，即2019年9月12日00:00:00-2021年5日19日23:59:59每日最低持仓达到300HT，即获得档数对应的额外签数。星级用户最多可获得10签。

注意：每签中签概率独立，中签后按比例成交。下单的最小单位为1签对应的HT数量,若用户下单数量不足1签则不允许下单；如实际下单的HT数量少于可下单签数的HT数量，则按最少签的下单数进行抽签。

（2） 第二轮为等比例分配，规则是按买一价格上的挂单数量进行等比例分配。用户可下单额度等于最大成交限额（前端显示的最大成交限额）减去第一轮已成交额，并于第二轮结束后按比例成交。

两轮活动皆设置阶梯限价期，每轮设置一个限价区间（最高可交易价格），用户可在限定的价格范畴内进行买卖交易。

5. 活动具体时间与每轮分配比例如下 ：

限价期：2021年5月20日20:00——2021年5月20日20:29:29；限价期分为两轮，每轮10分钟，中间10分钟作为清盘结算时间，共计30分钟。

第一轮：20:00:00—20:09:59；

清盘结算：20:10:00—20:19:59；

第二轮：20:20:00—20:29:59；

6.项目信息将另行公布 。

第二轮为等比例匹配成交详情

用户可根据前端显示的最大成交限额全额下单。

第二轮可下单额度 = 最大成交限额（前端显示的最大成交限额）– 第一轮已成交额

例：若一名持仓HT用户最大成交限额为4800HT，第一轮中分配4签并中了1签，按比例瓜分后成交50HT，则第二轮可下单额度为4750HT。

有交易额度的用户可在交易时间挂出限价买单，在交易时间结束时，针对买一价的所有买单，系统将进行等比例匹配成交。

其他注意事项：

1. 第二轮结束后所有部分成交的订单系统自动撤销。

2. 自由交易阶段开盘保护：为避免市场过于火热而产生非理性操作，平台将于开放自由交易后5分钟内限制使用市价交易功能，且限价交易的挂单价格不得高于第二轮价格的10倍。5分钟后恢复正常。

3. 数字资产交易是一种高风险的交易方式，请用户谨慎参与，理性决策，注意风险。Huobi Global不对交易行为承担担保、赔偿等责任。

4. 如有作弊行为，Huobi Global将取消其在该市场的交易份额。
