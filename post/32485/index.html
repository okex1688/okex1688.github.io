原文标题：《This is Why as A Crypto Native, I Also Don’t Like Web 3.0/DAO》

Jack （Twitter 创始人 Jack Dorsey）是对的，我们不能再胡说八道了，要告诉人们现实。我知道 Jack 偏心比特币，但他对 VC 的看法是正确的。

关于WEB 3.0，我觉得有些东西是不对的。 我是一个加密领域原住民，我一直在使用各类 DeFi 协议和我的代币，我卸载了社交媒体 app，以便在我的手机上为各种加密货币相关应用程序腾出空间。就在几天前，我参与的一个协议就被黑掉了，我自然亏了不少。是的，我就是一个Degen（degenerate的缩写，通常用于自嘲是“赌徒”）。

即便如此，我也并不买这些 WEB 3.0 的帐。

你可能已经听腻了 WEB 3.0 这个词了。最近，Twitter 的前 CEO Jack Dorsey 在推特上发布了上述推文，引起了争议。而我可以说很难不同意他的看法。

我也恶心透了 WEB 3.0。

WEB 3.0 是圈外人编造出来的

Web 3.0 这个词不知从何而来，甚至连我这样的一些加密原住民都感到奇怪。因为我们中的许多人从智能合约和区块链刚诞生的时候就一直在参与所谓的 Web3 或 DAO，认为这没什么大不了。这只是我们一直在做的事情。我们不讨论它，因为它和行话一样，存在于所有人的概念中。我们只是重复做一些事情，交易，投资，使用 DeFi ，说 gm 之类的。

我在2021年中期开始听到关于 web3.0 的杂音，大约是在 NFT 的市场炒作即将达到顶峰的时候。巧合的是，在这个时候，我开始注意到有一群新的、浮夸的人开始陆续入场。其中许多人我以前从未见过。

Crypto是一个紧密结合的社区。当你看到一个 Twitter 账户有 “没有被你关注的人关注”。那你就知道是个外来者。那些蜂拥而至且大喊 “网络 3.0 ，去中心化” 的局外人多是 VC （风险投资人/机构），不久，紧接着是 TradTech（传统/非加密货币技术人员）。简而言之，那些来自硅谷的人。

这就像商量好的一样。

我经常在Twitter上闲逛，我大部分清醒的时间都花在这上面。因此，当我看到一波 VC 账户向新的东西进军时，我就立刻注意到了这一点。在过去，他们的简介都是 “科技 “或 “电动车 “或 “创造者经济"之类的关键词。现在他们把自己重塑为 “网络3.0 “和 DAO 的发言人。

说实话，风险投资公司根据风向调整自己的方向并不奇怪，因为他们一直都秉持着“哪里有金矿就会去哪里”的原则。

更不用说他们过去的金矿已经被榨干了。科技股和 SPAC ，风险投资公司的旧爱，今年都受到了重创。他们可能早已获利，现在他们需要一个新的游乐场。

我只是不喜欢他们称加密领域是 “我的东西”。我们的角落里的小世界是我们每天一点一点建立起来的（而且好像它还没有现有的 VC /鲸鱼问题）。VC 们会毁掉一切，就像他们在传统科技领域用高估的 SPAC 做的事情一样。

这些人不过是一群骗子和淘金客。

Web 3.0具有2017年ICO的气息

如果一个东西在市场上有闪亮的、好得不能再好的包装，你就知道它是充满了谎言。加密领域在 2017 年经历了这种荒诞剧，例如当时大量的 ICO。

因此，当 Web 3.0 的人带着夹生的叙事和一篮子花言巧语来的时候，简直就是想重演 2017 年的戏码。

Web 3.0 被推销得好像它是互联网的未来。散发着盲目的热情，特别是针对加密领域的希望。他们被描述成了是关于一个更好的世界的梦想。

Web 3.0 借用了加密货币术语 gm 和 wagmi ，并在四处大喊大叫，以便让每个人都相信它就是一个加密原生概念，Web 3.0将通过推动社会发展，从而改变他们的生活（这主要意味着购买他们的代币），让人们产生一种虚假的归属感。

不要误会我的意思，加密原住民们也相信加密货币是未来。但在这一点上，大多数人都知道，比特币作为货币的替代品是值得商榷的，而Dogecoin只是用来搞笑的。

同时，围绕着 Web 3.0 的都是加密独角兽崛起和币价冲上月球的氛围，而这种感觉让人不寒而栗。

因为加密原住民都知道，真正的加密领域是无情的。鲸鱼，直接或间接地，表达它们想吃掉你的想法（收割散户）。黑客和 Rug Pull 很多，所谓的治理并没有什么卵用，你依然可以在短时间内失去损失超过 70 % 的本金。

大量的 Rug Pull 、暗箱操作，以及深陷熊市早就磨炼了长期参与者的心智。在他们看来诚实的罪犯更受赞赏。是的，如果一个人/项目直接明说要收割我们这些韭菜，而不是伪装着要拯救世界一样，这样其实更好。

如果新的参与者被所谓的“web3”蒙蔽了双眼，看不到所有这些负面的东西，他们只被灌输了希望和梦想，那是很危险的。围绕 web3.0 的营销也是对人们的不安全感和对归属感的渴望的掠夺，让他们极具市场操控性。

这都是那些风险投资公司的弱点。他们根本不知道加密领域如何运作。他们认为他们可以带着自己的那一套堂而皇之的闯入，并被当作国王一般簇拥。他们认为我们无法看穿他们的谎言，当然，事情的发展恰恰相反。这也是一个糟糕的时机，因为在过去的两年里，加密领域已经非常成熟，并且已经开始变得反 VC 。

Web 3.0。定义不明确，没有产品

“先建设，后市场 “从来不是那些资本分配者喜欢的策略。因此，在传统金融市场上才会出现没有产品，却估值 10 亿美元的公司。

显然，由风险投资驱动的 Web 3.0 领域也正朝着同样的方向发展。

首先，什么是Web 3.0？它有一个不明确的定义。它是如此模糊，没有人能够真正解释它是什么。在某些情况下，任何与区块链有关的东西都是 Web 3.0 ，尽管它基本上像老派公司一样在老派基础设施之上运作，比如 NFT 平台 OpenSea 和一个中心化交易所CoinBase。

“Web 3.0 —— 就是 Web 2.0 上面撒上了加密货币和区块链”。

在对 DAO 进行研究时，我经常发现那些 DAO 除了一些空气币和 discord 频道外，什么都没有。他们的代币几乎都是我从来没有听说过的币，当我在 CoinGecko 上查看时，排名却可以排到第 700 位。

它们是空气币中战斗机。

目前的Web 3.0运动充满了噪音。人们叫嚣着gm和wagmi，在实际上什么都没有的情况下建立一种虚假的社区感。我没有看到任何产品、任何服务。我看到的是糟糕的代币经济学。这篇文章甚至承认他们将不断需要新人（他们称之为 “客户”）来维持代币价格。这不就是一个赤裸裸的庞氏骗局么？？文章链接：https://every.mirror.xyz/y_WLA-Tk3VF5uPqHi-glDLVVfHxLUbjXakRI7SMISas

该作者就是一个风险投资人。

Web3热潮令人担忧

我经常思考学习加密货币的最佳途径是什么，我得出了一个结论，即 DeFi 途径是最好的。你先在中心化交易所进行交易，了解市场氛围，理解价格行为，然后再转到 DeFi ，甚至可以探索的更深。简单地增加你的投资。随着时间的推移学习。

这个想法可能很激进，但我依旧建议你在 “毕业 “之前，多走走弯路，多尝试失败的滋味。只有在这之后，你才有资格投资于风险更大的非流动性资产，如 NFT 或超低市值 DAO 代币。

我想说的是，相对于 DeFi 来说，整个 Web 3.0/DAO 的事情还处于萌芽阶段。它的大部分都是未经测试和探索的，因此直接入场这些领域，是新韭菜们的大忌。

Web 3.0 是在试图重塑加密货币的形象。我必须承认，加密货币与投机，或黑幕交易脱不了干系。是的，它并不完美，但这就是我喜欢它的原因。把事实讲出来并不丢人。

一些局外人觉得有必要给加密领域赋予一个新的名字和一个有吸引力的叙事。例如《纽约时报》就在其最新的文章《硅谷新的致富快车》中称其为：加密货币初创公司。

我不同意这种品牌重塑。这样的结果只不过是 TradFi 和 TradTech（基本上是 web 2.0）伪装的Web 3.0。它在本质上不是加密领域。还是那句话，他们只是外来的淘金客。简直令人厌恶，更不用说一旦他们搞砸了，他们就会迅速的离开这里，而我们不得不收拾他们的烂摊子时，他们又会给我们贴上新的标签。

Crypto 这个词已经足够了。

最后想写的话

在VC驱动的 Web 3.0 中，你的 DAO 代币将不会比你的 SPAC 股票走势好。事实上，由于它们的空气币性质，在投机和波动性方面会差很多。DAO 将是一群虚构的组织，没有实际的协议和产品，只为叙事服务。你可以投票，但你永远都不知道这是否重要。

我将对你说实话。作为一个老 Degen，我人生的感觉基本上就是在试图躲开鲸鱼和VC。如果这种现实被封装在一些 “安全的” Web 3.0 包装中提供给无知的加密领域新手的话，那就已经不能只用龌龊来形容了。
