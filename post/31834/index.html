12月14日谷歌趋势数据显示，过去3个月，全球“web3”一词搜索量大幅上涨。实际上，与之相对的是，过去一段时间以来，有关web3.0的讨论逐渐升温。

什么是Web3.0

Web 1.0指的是万维网发展的第一阶段，时间大约从1991年到2004年。在Web 1.0中，内容创作者很少，绝大多数用户只是内容的消费者。个人网页很常见，主要由托管在ISP经营的网络服务器上或是托管在免费的网络托管服务（如：GeoCities）上的静态网页组成。

随着Web 2.0的出现，普通的网络用户可以通过低成本的网络托管服务或通过专属服务器（诸如Blogger或LiveJournal），拥有社交网络个人资料（诸如Myspace和Facebook等网站）和个人博客。通常，内容是动态生成的，允许读者以一种与以往不同的方式发表评论。

Web 2.0是一种新的互联网方式，通过网络应用（Web Applications）促进网络上人与人间的信息交换和协同合作，其模式更加以用户为中心。典型的Web 2.0站点有：网络社区、网络应用程序、社交网站、博客、Wiki等等。

Web 3.0是针对Web 2.0提出的，较有名的首次提及是在2006年初Jeffrey Zeldman的博客中一篇批评Web 2.0的文章中。实际上，自2006年底以来，Web 3.0一词正受到越来越多的关注，也是越来越多争论的焦点，这个现象正持续到现在。关于如何定义Web 3.0，及其所代表的含义的争论非常激烈，观点也琳琅满目。

2006年5月，英国计算机科学家，万维网发明者蒂姆·伯纳斯-李曾说，人们不停地询问Web 3.0是什么。我认为当SVG在Web 2.0的基础上大面积使用——所有东西都起波纹、被折叠并且看起来没有菱角——以及一整张语义网涵盖着大量的数据，你就可以访问这难以置信的数据资源。

2006年11月的Technet峰会上，Yahoo雅虎创办人兼首席执行官杨致远指出，目前对Web 2.0的归档和讨论很多。借助网络级别所能达到的性能，网络的力量已经到达了一个临界点。我们同时也看到最近4年出现了更高级的设备以及更高级的与网络交互的方法，不仅呈现在游戏机和移动设备这样的硬件，同时也呈现在软件层面。你不一定得是计算机科学家才能创作出一个程序。这种现象在Web 2.0里初现端倪，而3.0将更加深化，是一个真正的公共载体……专业、半专业和消费者的界线越来越模糊，创造出一种商业和应用程序的网络效应。

Netflix奈飞创始人Reed Hastings阐述了定义Web术语的简单公式。Web 1.0是拨号上网，50K平均带宽，Web 2.0是1M平均带宽那Web 3.0就该是10M带宽，全影像的网络，这才感觉像Web 3.0。

2010年11月16的网络高峰会（Web 2.0 Summit）中“互联网女皇”Mary Meeker指出Web 3.0由“社交网络、移动设备和搜索（Social Networking, Mobile and Search）”所组成。

Web3.0的争议

实际上，Web 3.0目前还很难被定义，在传统互联网人看来，它是更人性化的互联网，融入了更新的技术，大数据、人工智能等，许多事计算机都帮你想到了。但这会导致信息垄断、用户毫无隐私等问题。

因此，在区块链从业者看来，Web 3.0应该是去中心化的网络。例如，Web3.0的目标是在无中介的读写网络中提供更好的以用户为中心的体验。技术使得个人在缺省情况下能控制数据隐私和所有权。Web 3.0大致有一下4个特性标签：统一身份认证系统、数据确权与授权、隐私保护与抗审查、去中心化运行。因此，在区块链从业者眼中，Web 3.0其实是基于区块链思维而进化出的全新网络形态。

而近一年来，区块链行业关于web3.0的讨论也逐渐升温。当然，其中充满了争议，有人认为web3.0是互联网的未来，也有人认为web3.0还很遥远。

美国创业孵化器Y Combinator总裁Sam Altman称，2020年代进行的投资的平均风险回报将比2010年代的投资差很多。但web3可能仍然有2010年的回报，不过大多数风险投资公司会错过它。

12月2日，埃隆马斯克回复美国创业孵化器Y Combinator总裁Sam Altman称，“Web 3听起来像是胡说八道（bullshit）。”12月4日消息，Elon Musk 在社交媒体上再次发表嘲讽 Web3 类内容，本次发表的图片暗指 Web3 拥护者喜欢不分场合的向别人推销 Web3、去中心化组织（DAO）等加密领域概念。随后，以太坊核心开发者eric.eth在sassal.eth转推马斯克推文下表示，目前市场的大多数人都已经不喜欢Elon Musk，Web 3和加密DeFi不需要Elon。

波卡Polkadot创始人Gavin Wood将这种在全球范围内进行人际互动的方式称为“后斯诺登时代网络”（Post-Snowden Web）。展望未来10年或20年，Gavin Wood表示，未来大规模多用户应用程序框架可以为用户提供“实际”保证。当被要求预测十年后的Web 3，Gavin Wood回答说：“这将是一个非常有趣的社会实验，我们将看到世界对隐私、自我主权和透明度的关注，知道系统或服务规则如何运作，非常真实。”

a16z合伙人Chris Dixon在推特上发文表示，不能因为当前新兴技术存在缺陷就反对新兴技术发展，任何重要技术运动都需要经历很多年时间过程，善意的批评可以改善技术，需要将“暂时出现的缺点”和“彻底限制”区分开，尤其是Web 3和加密，现在的确存在交易费用过高、能源消耗过大的问题，这是该新兴技术的暂时缺陷，但出于不同发展阶段，我们可以思考更多解决方案，但不能把“暂时缺陷”视为彻底否认新兴技术的理由。

值得一提的是，12月9日美国国会的加密听证会上，a16z 政策主管 Tomicah Tilleman 表示：这是国会议员首次通过委员会全体听证会这个平台来强调 Web3 是互联网的未来。这是关于去中心化技术的全国性讨论中一个历史性的转折点，委员会成员承认 Web3 平台有潜力解决许多他们关心的问题，包括汇款和金融普惠。

总体而言，web3.0正在受到越来越多的关注，同时也注定了其在未来一段时间将充满争议。
