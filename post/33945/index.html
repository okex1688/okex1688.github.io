在2021 年，感觉世界已经充满了艺术 NFT。

创作者意识到他们现在可以直接与最狂热的粉丝建立联系，而不是依赖那些从他们的辛勤工作中获取大量价值的中介。

在当前的演变中，这就是 NFT 所代表的东西：创造者和粉丝之间的直接联系，从而在创造的价值和获得的价值之间建立更牢固的关系。

但我们只看到 NFT 在艺术界爆发。世界各地都有创作者：作家、音乐家、摄像师、设计师、艺术家等。

他们都将从数字所有权和 NFT 中受益。

我们已经看到其中一些平台起飞，例如为作家打造的Mirror协议，但其中大部分潜力仍未实现。

今天，Lucas 和 Cooper 向我们展示了这些未开发的创作者垂直领域和音乐 NFT 世界的幕后故事。

音乐NFT的看涨案例

新兴的NFT中出现了一个新的领域。一种为主流采用带来最大潜力的产品。

音乐 NFT 就是可收藏的音频文件。

他们让粉丝有机会收集他们喜爱的歌曲，而现在由于 web3的到来，这些歌曲拥有了数字稀缺性。

收集音乐并不新鲜，但在线收集音乐却是前所未有。

就像我们收集黑胶唱片、刻录 CD 或制作播放列表一样，音乐NFT 提供了一种新的媒介，通过接触潜在的创意作品及其创作者来策划你的品味。在传统世界中，音乐收藏品代表着价值50亿美元的市场。在数字世界中，天空则是它的极限。

在本文中，我们将探讨音乐 NFT 的上升趋势，并仔细研究这种面向下一代独立艺术家的新模式。

一种新的货币化模式

音乐 NFT 是艺术家的一种新的货币化模式。

从本质上讲，音乐NFT 只是数字收藏品。它们不具有固有的所有权，仅作为可以由收藏家买卖的资产存在。

在一个主要由流媒体和巡回演出主导的世界中，音乐 NFT 通过建立一个渴望收集稀缺数字作品的超级粉丝社区来为艺术家提供杠杆。

像 Royal 这样的平台正在探索具有版税的音乐 NFT，但在本文中，我们将严格关注音乐 NFT 作为收藏品。

收藏品提供了最直接和最简单的货币化机制，因为在艺术家拥有 100% 的权利时没有监管障碍。

对于那些不熟悉的人，像 Spotify 这样的传统流媒体平台每百万streams支付约 3,700 美元。

相比之下，音乐NFT 的售价高达数千美元——这有力地证明了在 web3 中建立社区比在 Spotify 上玩数字游戏更有优势。

虽然音乐NFT 的销售在很大程度上取决于加密货币的宏观状态，但在任何市场条件下，一个专门的超级粉丝收藏基础的好处都适用。这是利用艺术家的 100 名忠实粉丝的直接盈利模式。

您无需成为流行歌星即可谋生 - 音乐 NFT 只需少数忠实粉丝即可让您最大化您的收入潜力。它们是 Creator DAO 的基石，也是迈向音乐微观经济的第一步。

我在这篇文章中深入探讨了音乐 NFT，并为那些不了解的人总结了当前格局的广泛布局。

放大来看，目前有几个平台引领着 Music NFT 的潮流。

这些平台提供了加密货币如何修复音乐行业的第一个真实实例。

Catalog

Catalog 是一个 1/1 音乐NFT 市场。

与 SuperRare 或 Foundation 类似，所有 NFT 都以 1/1 作品的形式铸造 - 这意味着该歌曲永远不会被铸造其他副本（理论上）。Catalog 基于 Zora 协议构建，是第一个专门为收集音频而设计的市场。

迄今为止，Catalog 的销售额已超过 200 万美元，平均售价为 3,707 美元。

以下是 Catalog 上最畅销的艺术家，以及根据对比每百万streams 3,700 美元（或每stream 0.0037 美元）估算的 Spotify。

Spotify 的收益计算为每stream 0.0037 美元，ETH 为 2,800 美元。

平均而言，顶级 web3 艺术家通过销售音乐 NFT 获得的收入是他们从一年的 Spotify流媒体中获得的估计收入的 7.5 倍。

这清楚地表明了这一点：独立艺术家通过音乐 NFT 可以获得比流媒体更多的收入。同样重要的是，这完全是附加的——销售 Music NFT 并不会从你的流媒体收入中扣除。

在广泛的范围内，目前平均目录销售在 Spotify 上转化为大约一百万个stream。在没有基础平台费用的情况下，该销售额的 100% 返还给创作者。

但这些销售才刚刚开始。Catalog开始经历曲棍球棒的增长，月销售额直接在六位数范围内，平均销售价格呈上升趋势。与 SuperRare 的 1/1 艺术市场的八位数相比，Catalog 和 1/1 音乐NFT 的上行空间很大。

来源：analytics.catalog.works

Sound

Sound 是一个提供专辑而非 1/1 的音乐 NFT 市场。

每个专辑都有一个聆听派对，收藏者在其中聆听完整曲目，然后继续收集 NFT，并能够在曲目上发表评论。

收藏家出现在“观众”选项卡中 - 让他们成为焦点，并有权在早期被推荐。当 Sound NFT 在二级市场上出售时，该评论栏也有同样的权利。

此前已经连续 30 次售罄，所有 Sound NFT 均以 0.1 ETH 的基准价格定价。这意味着每个 Sound NFT 代表 75,000 个 Spotify stream，每个售罄的标准 25个专辑的总stream价值超过 180 万。

值得一提的是，Daniel Allan 为他的 EP Overstimulated 推出了 一次100个专辑空投，一分钟内全部售罄并带来 10 ETH 的收入。作为参考，到目前为止，新发行的 EP 仅从 Spotify 产生了 444 美元的估计收益。

较低的价格点为二级市场活动留下了更多空间，让我们第一次看到为什么空投已成为吸引新收藏家的一个谢林点。

除了一级销售，二级销量一直在上升，并为每位艺术家带来越来越多的收入。Sound.xyz 的平均下载量仅在版税方面就产生了大约 420,000 stream的收入。

来源：https://dune.xyz/0xnative/Sound-Music-NFTs

关键要点

一位收集音乐 NFT 的粉丝的价值几乎是 Spotify 上的每月听众的 100 倍。一般：

1 个Catalog NFT = 1,000,000+ 个stream

1 Sound NFT = 2,500,000+ stream

这显示了 Music NFT 作为独立艺术家的货币化模型的影响力。它让收藏家能够在今天评估他们的作品，而不是指望一首热门歌曲和等待流媒体费用的到来。如果它是好音乐，收藏家会以他们认为值得购买的价格购买。

音乐 NFT 的数量在增加，铸造 NFT 的唯一艺术家总数在增加，唯一收藏家的数量也在增加。但与 NFT 市场的其他部分相比，它仍然微不足道。

音乐 NFT 即将迎来重大突破。

从上个月的交易量来看，Catalog 和 Sound 的一级和二级销售额加起来不到 100 万美元，仅占 OpenSea 的 NFT 交易量的 0.015%。

您决定该变量是否值得布局。

双重生活

对于发布 Music NFT 的艺术家来说，你需要兼顾双方。

虽然 web3 艺术家看到了财务上的成功，但在 web2 中不断扩大的足迹也同样重要。

Web3 让您可以专注于粉丝的质量。Web2 让您可以专注于粉丝的数量。这种将新的 web2 粉丝转变为新的 web3 收藏家的关系是当今独立艺术家最大的机会。

随着时间的推移，预计会有更多的艺术家探索加入web3 意味着什么，并只能作为音乐 NFT 发布，而且仅作为音乐 NFT 发布的曲目，从未分发给 Spotify 或 Apple Music 等 DSP。

展望未来

想象一下，能够在您最喜欢的艺术家的职业生涯早期对其进行押注。有了音乐 NFT，这个未来并不遥远。

虽然今天我们正在探索收集音乐的方法，但这个循环不可避免地会导致一个成熟的 DAO 经济，在这个经济中，艺术家将所有销售额的一部分转入社区主导的金库。你应该意识到，社区金库通常意味着以后会有项目代币。

以 Daniel Allan 和他的 Overstimulated EP 等早期例子为案例，我们都应该期待在不久的将来看到更多的 艺术家DAO。

在那之前，让这些音乐 NFT 蓄力，而你可以开始收集。
