虽然加密货币市场开年受制于比特币价格的持续下跌略显萧瑟，但 NFT 市场的热度未见丝毫减退。今年 1 月 NFT 市场成交量突破 50 亿美元再创历史新高，作为市场份额占比超 90% 的门户 OpenSea，在今年年初完成的 C 轮融资中估值也已飙升至 133 亿美元。不过促成 OpenSea飞速「膨胀」的巨大流量，同样成为了其必须直面的严峻考验。

近期 OpenSea 又一次因安全问题被推上风口浪尖，因其挂单 Bug 和钓鱼攻击使用户资金蒙受了损失，加之去年 9 月 OpenSea 爆出其前高管利用职务之便进行内幕交易的丑闻，引发了「Web3 先行者」们对中心化 NFT 市场弊端的重新审视，而那些去中心化 NFT 交易市场及其他显露潜力的交易市场，也迎来了一个难得的瓜分 OpenSea 碗里蛋糕的窗口期。

本文将对各条链上现有的 NFT 交易市场以及各大交易所和传统巨头进军 NFT 领域推出的交易市场做出整理，盘点 38 个 NFT 交易市场现状，并整理了各平台已公开的交易费用和版税信息。

各公链上的 NFT 市场

NFT 的这波浪潮激发市场中出现更多的 NFT 市场，一些主流 NFT 平台可能只支持自己的平台币或者 ETH 来进行 NFT 交易，各公链为了发展自己生态的 NFT，争相推出了 NFT 交易市场。 对于这些 NFT 市场来说，它们未来发展的潜力基于所在的链未来发展的潜力。（注：先后顺序根据各公链市值及 NFT 交易市场首字母排列）

Cardano

Artano

Artano 是一个由社区驱动的 NFT 市场，它成立了一个艺术家和策展人委员会，希望最终由用户来为该平台策划应用程序以及指导 Artano 的艺术方向。Artano 有两种版税，分别是通用版税和 Artano 版税：用户在 Artano 之外销售他们的 NFT，并且他们销售的市场支持 CIP 标准，则将收取通用版税；艺术家在 Artano 铸造 NFT 后可以选择百分比的版税，但如果 Artano 版税设置高于通用版税，可能会导致经销商前往其他平台出售 NFT。

首次销售费时，Artano 收取 5%服务费用，艺术家获得 95% 的利润，二次销售时，Artano 收取 2.5% ，卖家获得 97.5% 的利润减去艺术家设定的版税。艺术家参与 Artano 上的策展，策展人将收到双方协议的佣金。

CNFT.IO

CNFT.IO 是 Cardano NFT 市场，于 2021 年 7 月由一个来自世界各地去中心化的团队推出。网站上可显示 NFT 交易量排名、24 小时以及 7 天交易量，CNFT 按交易量、地板价和其他资产对 NFT 进行排名。使用体验中比较亮眼的部分包括 CNFT 可通过邮箱登录，整体浏览较为流畅，可通过连接Cardano 钱包购买 NFT。CNFT 的市场费用为 2.5%（最低为 1 ADA ），CNFT 的版税为 2.5%（最低 1 ADA ）。

Tokun

Tokhun 是一个基于 Cardano 区块链的 NFT 市场，于 2021 年 3 月在 Mary Hardfork 之后成立。Tokhun 最初设计为 NFT 铸造平台，之后演变为智能合约驱动的 NFT 市场，数据结构基于CIP-25 标准并且是高度可定制的。Tokhun 平台的销售佣金为销售费用的 2.5% （至少 2.00 ₳DA），铸造费用采用了阶梯定价模型，铸造第一个 NFT 的价格为 2 ₳DA，第 2 至 20 个铸造价格为 1.5 ₳DA。用户可先通过谷歌账号登录 Tokhun，然后可自己创建一个钱包。

Solana

Magic Eden

Magic Eden 是 Solana 上 NFT 市场，包括 NFT Launchpad、NFT 拍卖等，官网页面还了统计数据，包括各 NFT 项目的地板价、交易量、平均价格等，还展示 Magic Eden 平台的 24 小时交易量和总交易量。Magic Eden 的 NFT 上架费用为 0，对所有交易收取 2% 的费用，版税由嵌入其元数据的创作者设定，可以在项目详细信息页面上查看每个创建者设置的版税百分比。根据 Dune Analytics 的数据，Magic Eden 目前是 Solana 上交易量最高的 NFT 市场。

Solanat

Solanat 是 Solana 上 NFT 市场，每笔交易收取 3% 的市场费用，版税由创作者设定。Solanat 于 2 月 17 日上线了 V2 版本，其中对钱包管理、数据分析、增强特定收藏的属性预览等功能进行了升级。

Terra

Luart

Luart 是基于 Terra 构建的一体化 NFT 平台，包含 NFT 市场、NFT Launchpad、NFT 盲盒、质押 LUART 代币获得 NFT 等功能。在其 NFT 二级交易市场中，每笔交易向卖方收取 6% 的费用，其中 3.5% 支付给项目方，1.5% 给 LUA Power，1% 给 Luarj。

Polkadot

RMRK

RMRK 是 Polkadot 的金丝雀网络 Kusama 上的一组标准，由五个「NFT lego」原语组成。用户可根据乐高创建任意复杂度的 NFT 系统。Singular 是基于 RMRK 1.0 的 Kusama 原生 NFT 市场，目前还未推出版税。

Fontom

图源：Fontom 基金会公示生态图

PaintSwap

PaintSwap 是 Fantom Opera 网络的去中心化平台，包含流动性挖矿、质押和 NFT 市场。其二级市场于去年 9 月发布，发布一个月后 FTM 交易量超过 100 万，目前已推出第 2 版，主要变化是将主要货币从 BRUSH 切换为 FTM，每笔销售会收取 2.5% 的佣金，其中 50% 用于购买和销毁代币 BRUSH，另外 50% 将发送给 DAO 基金，继续支持围绕 BRUSH 和市场的开发。PaintSwap 支持两种类型的版税，分别为 ERC2981 和 PaintSwap 的精选页面收藏版税，两者的上限都是 7.5%。

NFTKEY

NFTKEY 是一个去中心化的非托管 NFT 市场， 2021 年 3 月 26 日启动，支持以太坊、币安智能链、Fantom、Avalanche 和 Harmony 上NFT 上架、拍卖、交易。NFTKEY 收取 2% 的交易费用，默认的创作者版税为 2%，但创作者可以在 0% 到 8% 的范围内设定。

Artion

Fantom 基金会推出了 NFT 市场平台 Artion ，其 V2 版本已于 2 月 9 日上线，界面与 OpenSea 十分相似， Fantom 基金会表示，Artion 之后将由社区接管。

Artion 称其平台的优势在于：

交易速度快，通常可在 1-2 秒内得到确认；

交易费用低，通常创建和交易 NFT 只需几美分；

零平台费用，通过拍卖或直接报价交易 NFT，Artion 不收取任何费用。

Opera House

Opera House 是为社区打造的 Fantom Opera NFT 市场，目前处于测试阶段，由 Nova Network 提供技术支持。交易时 Opera House 收取 1 %的费用，版税由创作者自己设定，上限为 15%，目前支持 ERC-721，未来将支持 ERC-1155 和 ERC-2981。

ZooPet

Zoo 是一个建立在 Fontom 上的去中心化应用程序生态，旗下包括交易所聚合器 ZooTrade、去中心化交易所ZooDEX、交易和资产管理解决方案 ZooTracker 以及 NFT 市场 Zoo Pet。 ZooPet 支持 NFT 铸造和交易，分为原创艺术品、摄影、收藏品等类别，支持给艺术家消费以及查看每个 NFT 的交易历史等。

Dfinity

Entropot

Entropot 是 Dfinity 生态的 NFT 市场，包含 NFT Launchpad 和 NFT 交易市场，二级市场的交易费用为 1%，创作者可收取高达 10%的版税，目前支持的钱包包括 Stoic Wallet 和 Plug Wallet。

Tezos

Objkt

原本 HEN 是 Tezos 上第一个 NFT 交易市场，于 2021 年 3 月上线，由巴西人 Rafael Lima 创立，最初想帮助南美范围内没有西方媒体资源支持的艺术家做自己的 NFT 展销平台，但最终受到全球范围内独立艺术家的欢迎和入驻。但其因为智能合约存在漏洞，在一个黑客事件后，第三方拍卖工具 Objkt 加紧建设，完善了相关功能，声称要做 Tezos 上 NFT 的汇总平台，Tezos上所有的艺术品都可以在上面交易，11 月 11 日时 HEN 主站下线。 Tezos 上的艺术品更像是把传统艺术展中的艺术作品拿到链上销售，NFT 更加具有文化形式而不是价值炒作形式，另外，Tezos 上的 NFT 都更加弱中心化，交易费用为 2.5% ，版税由创建作品的艺术家自己设置。

Flow

BloctoBay

BloctoBay 是多合一跨链智能合约钱包 Blocto 内置的非托管 NFT 市场，Blocto 还包含对开发人员友好的 SDK以及内置的 DEX BloctoSwap 以及 Blocto Launchpad。

Eternal

Eternal 是基于 Flow 网络的视频和游戏 NFT 市场，Eternal 与创作者和电子竞技玩家合作，将他们的视频从他们的直播和游戏变成粉丝可以收集或交易的时刻，Eternal 于去年 9 月完成了 450 万美元融资，Mark Cuban 参投。Eternal 的 NFT 二级市场中，5%手续费归 Eternal 用于支付网络和运营费用， 10% 归创作者所有。

NBA Top Shot Marketplace

NBA Top Shot 是美国国家篮球协会和女子国家篮球协会进军 NFT 世界的尝试，在其市场上，可以从世界一流的篮球联赛中购买具有收藏价值的时刻（视讯短片和比赛集锦）和艺术作品。

该市场是 Dapper Labs 基于 Flow 区块链构建的交易市场。

Oasis

Oasis 为社区用户建立一个进入存储在 Oasis 网络中的元宇宙身份 MBuddy，创建了 999 朵AI Rose，MBuddy 成员可以申领AI Rose，该 NFT 还可以在专属的 NFT 市场 Metamirror 交易，目前 AI Rose 的地板价为 3800 ROSE （约），交易总量为 105 万 ROSE。

Arbitrum

Treasure DAO

Treasure DAO 是 Arbitrum 上的去中心化 NFT 生态系统，专为元宇宙项目构建，主要有GameFi 游戏、NFT、代币 MAGIC 这三大部分组成。最初用户可以通过质押 Loot NFT获得 MAGIC。现在可用 MAGIC 购买其 NFT 市场的 NFT，其中版税为 5%。至少 2.5% 会被重新分配给 Treasure DAO，其余部分归创建者所有。

多链 NFT 市场

SuperFarm

SuperFarm 是一个跨链 NFT 综合平台，目前已经发布 NFT 发布平台（NFT Launchpad）、NFT 挖矿（NFT Farming），按照其路线图还将 推出 NFT 生成工具、NFT 市场平台以及 NFT 交易平台。SuperFarm 采用多链策略，将支持以太坊、币安智能链、Polygon 等，今年 4 月份将在 Fantom 上发布独家 NFT 以及启动 NFT 挖矿。SuperFarm 的 NFT Launchpad 由 Immutable X 提供支持，可实现快速、零 Gas 的 NFT 交易。

Venly

Venly（原 Arkane Network）是一个多链 NFT 平台，支持 Polygon、币安智能链、Avalanche，目前推出了 BETA 版。在销售和拍卖时，Venly 市场对每笔成功的销售收取 2% 的费用， NFT 创建者收取另外 10% 的费用。Venly 在对 NFT 分类时，分为游戏中的物品、收藏品和数字艺术品。

对 OpenSea 发起吸血鬼攻击的 NFT 市场

Infinity：Infinity 是较早对 OpenSea 发起吸血鬼攻击的协议，早在去年 10 月 Infinity 就表示向OpenSea 用户发放空投，该协议由社区驱动，每次销售收取 1.5% 的交易费并将其纳入社区控制的资金库。

LooksRare：1 月 10 日 LooksRare 对 OpenSea 用户发放 LOOKS 代币，LooksRare 相较于 OpenSea 2.5% 的交易费来说，LooksRare 只收取 2%的交易费用，并且交易费用 100% 由 LOOKS 质押者赚取，并且以交易激励机制来鼓励用户进行 NFT 交易。

X2Y2 ：X2Y2 也对 OpenSea 用户发放空投，收取 2%的交易费用，费用也用于为代币质押者提供质押奖励，此外，X2Y2 鼓励用户挂单，用户上架 NFT 的奖励是代币质押奖励的两倍还多。

交易所推出 NFT 市场

当各条链上正在发展的去中心化 NFT 市场还在为如何吸引用户而努力时，在用户群体上已占据规模优势的交易所纷纷开始进军 NFT 这个领域。中心化交易所的托管服务、品牌与分销、合作和知识产权等都存在较大优势，NFT 市场可能虽然不会是其主营业务，但也将是交易所全面发展不可或缺的一部分。

币安去年 6 月推出 NFT 市场，是首批推出 NFT 市场的加密货币交易所之一，用户的交易所账户可自动与 NFT 账户相连，使用户完全无门槛进入，竞标 NFT 资产的操作与其他同类平台一样简单，买家可以在该平台上使用 ETH、BNB 和 BUSD，具体取决于创建者可接受哪个数字资产。

加密货币交易平台 Gemini 收购的 NFT 市场 Nifty Gateway 促成过数字艺术家 Beeple 和歌手 Grimes 等的 NFT 销售。

FTX US 于去年 10 月份推出了 FTX NFT 市场，刚开始布局在Solana，12月份又支持以太坊。

Bybit 于今年 1 月份推出了 Bybit NFT Marketplace，Bybit 用户无需将他们的个人钱包地址连接到平台，通过 Bybit 现货账户在以太坊 ERC-721 标准上即可买卖或交易 NFT。

交易所巨头 Coinbase 也表示计划推出 NFT 市场，Coinbase 的用户基数庞大并且有合规优势，虽然尚未正式推出，但其 NFT 市场的白名单上有超过 370 万名用户排队。

加密货币交易平台 Kraken 也计划推出 NFT 交易市场，用户将可以使用 NFT 作为贷款的抵押品，该市场将提供托管服务，交易所正在研究如何确定所存 NFT 的清算价值。

韩国加密交易所 Bithumb 首席执行官 Backyoung Heo 在接受采访时表示，Bithumb 今年通过子公司推出 NFT 平台，可能会类似于 OpenSea。

传统巨头 NFT 市场

如果说交易所的战略加入只是圈内人对 NFT 板块轮动的重视的话，那传统巨头的进入就是 NFT 破圈之旅的证据。

Meta（前 Facebook）计划推出 NFT 市场， 并允许用户在其个人资料中创建和显示 NFT，同时还在酝酿铸造代币的原型功能。社交巨头 Meta 参与 NFT 市场确实会增加 NFT 对普通人的曝光率。

美国国家橄榄球联盟（NFL）和运营 NBA Top Shot 的公司 Dapper Labs 合作推出了基于 Flow 区块链的 NFT 市场 NFL All Day。

社交巨头 LINE 计划 在 2022 年第一季度上线由旗下区块链暨 NFT 子公司 LINE NEXT 构建的 LINE NFT 市场「DOSI」，进入元宇宙。

纽约证券交易所申请虚拟商品和 NFT 相关商标，计划建立和 OpenSea 相似功能的 NFT 市场。

小结

无论是加密世界原生的协议、中心化交易所还是传统世界的企业都在对标 OpenSea 开发 NFT 市场，可见目前盘踞在 NFT 板块龙头的 OpenSea 很难做到永远领先。OpenSea 问题迭出的现状也在给正在开发中和投入使用初期的 NFT 平台敲起了安全警钟，把保护用户资产放在第一位的 NFT 市场才会持续吸引越来越来用户进入。而在 OpenSea 的基础上建立了创新机制的后起之秀，如 LooksRare 和 X2Y2，其成功与否还需要经过时间的检验。就当前 NFT 繁荣发展的趋势来看，各 NFT 市场将继续与 OpenSea 同步成长，最终沉淀出几个最有实力生存下去的 NFT 市场。
