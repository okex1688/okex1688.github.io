注意力经济

什么是最稀缺的资源？当我们想到稀缺资源时，我们想到的是有形商品，如石油或钻石。它们是已经被过度使用的商品，不再足以满足我们的需求。然而，在今天的信息时代，我们自己的注意力已经成为一种稀缺资源。在过去，我们是客户，现在，我们的注意力已经成为产品。在所谓的 “注意力经济 “中，我们的注意力不断受到来自社交媒体、电视或新闻来源的信息轰炸。

注意力经济的核心是将用户的注意力商品化。注意力经济伴随着社交媒体平台的崛起而兴起。这些平台对人们来说是免费的；用户只是通过向平台提供他们的个人资料来 “付费”，然后平台可以根据广告将到达特定目标受众的承诺，向公司出售广告。用户还 通过 关注内容来 “付费”，因为创作者获得的浏览量和参与度越高，他们就能从平台上 或 通过做品牌交易赚取更多的钱。

从本质上讲，注意力经济是由一个恶性循环促成的，我们是注意力经济的产物，但同时也是被不知不觉地操纵来加强它的顾客。

由注意力经济衍生的创作者经济

社交媒体平台涌现了许多有才能的创作者， 创作者渴望受到关注。这意味着他们需要持续创造出对广泛人群有吸引力的内容。此外，创作者可以熟悉不同平台的算法，创造他们认为算法会推荐给更多人的内容，扩大他们的受众。

平台也受到激励，希望有很多创作者能带来更多用户的关注。正因为如此，平台为创作者提供工具，如过滤器、智能分析和自动广告。这些工具，加上平台的影响力，可以让创作者有机会进行流量变现。这种现象， 即 许多人所说的 “创作者经济”。

这种崛起的独立经济打破了消费者与内容互动的传统方式。随着创作者经济的发展，思想领袖和有远见的人认为Web 3.0是互联网的下一个阶段，预计将取代当前版本的互联网，是未来增长的支柱。

虽然目前社交媒体是创作者直接与追随者联系的最有效的传播方式，但在未来，它可能不是正确的解决方案。虽然任何人都可以在社交媒体上发布信息，但用户放弃了一些控制权，因为他们要遵守这些平台的准则。此外，像TikTok和Instagram这样的平台试图通过他们的寻租优势来 “倒卖创作者合作关系”，压榨用户和创作者的红利，因为他们已经直接与广告商做生意了。 即 社交媒体巨头正在 “定位自己，从创作者的收入中抽成”

友好型创作者平台

Substack、Ko-fi、Patreon和Twitch，它们允许创作者直接出售他们的内容，这可以被认为是 “创作者友好 “平台。在《福布斯》进行的一项研究中，63%的人表示他们曾经给过内容创作者小费。这些平台仍然从创作者的收入中抽取一定比例，但明显低于其他社交媒体平台。例如，YouTube会从YouTuber的广告收入中抽取约32%，而Patreon会从创作者的每月Patreon收入中抽取5%、8%或12%（基于该创作者的会员计划而定）。他们也没有像其他社交媒体平台那样严格的审查规则，这可以让创作者有 更多 自由。

然而，这种吸引力已不再明显。现在，这些平台被视为累赘，他们对创作者的作品也有一定程度的所有权和控制权，因为他们有权力删除帖子或 将创作者的内容隐藏起来，不让粉丝看到。他们还拥有内容控制权，创作者会调整他们的内容以适应平台的广告规则和算法。

传统社交媒体平台和创作者平台的痛点：

  • 数据隐私：

Web 2.0: Facebook、Google、Twitter 中心化企业掌握了大量个人数据，一旦遭受黑客攻击或企业倒闭，用户的数据会造成泄漏，企业可对内部数据库进行篡改。

  • 数据主权：虽然用户以个人信息（数据）换取良好的用户体验,大部分中心化企业出售数据以换取广告收入。

  • 守门人的审查：

Web 2.0:许多社交媒体平台根据创作者的政治信仰对其进行审查，任意对一些内容进行删除，内容创作所有权掌握在中心化平台手里。

如何用区块链技术重塑创作者经济？

区块链是一个去中心化的数字交易账本。它使用加密技术，使所有发生的交易都可以追踪和透明。

区块链是去中心化和完全透明的，这一事实最初使它通过加密货币对金融部门非常有吸引力。然而，人们已经开始将区块链用于其他方面，它对内容创作者和媒体行业变得非常有利。

NFTs给予创作者所有权

非同质化代币（NFTs）是存在于区块链上的独特数字代币。独特 “意味着它们不能被复制或分割 - 它们是独一无二的。现在，围绕NFTs的大部分炒作是针对数字艺术。然而，NFTs可以是任何数字产品：歌曲、电影、视频，甚至推特。因为它们存在于区块链上，所以是可以追踪的。这对创作者很有吸引力，因为与现实生活不同，创作者可以知道他们的作品一旦售出后在哪里。他们还可以对NFT进行编程，如果它被转售，就把销售额的一个百分比给自己和所有以前的所有者，激励所有者保留其价值。

FTs（同质化代币简称 例如BTC）

同质化代币（FTs）是可以相互替换的。它们不是唯一的。同质化代币彼此具有相同的价值。可替换代币的一个例子是比特币：一个比特币与另一个比特币的价值总是相等的。FTs也存在于区块链上，使其可以追踪，创作者可以铸造属于自己的社交代币出售给粉丝。

DAO

去中心化自治组织（DAO）可以被认为是存在于区块链上的社区的一个共享的、完全数字化的银行账户。它们是社区像合作社或公司一样行事的一种方式，并为自己建立一个可持续的模式。一个社区可以发行FT（同质化代币），并使用该代币来决定成员获得什么样的特权，而且成员可以投票对DAO进行修改。

这些技术如何影响 创造者 经济？

创作者可以将他们的内容作为NFT上传到去中心化社交媒体平台。当NFT被转售时，收入可以归于创造者。创作者也可以创建和发行自己的FT（社交代币），他们的粉丝可以直接购买。这些代币持有者可以组织一个DAO，并将该DAO作为连接彼此和内容创造者的一种方式，允许采用一种社区驱动的方法，使粉丝融入进创作者的社区，一荣俱荣，而不仅仅只是当一个追随者。DAO还可以获得所有NFT销售额的一部分，高额的NFT销售甚至可以增加创作者的社交代币价值，这将使代币持有人受益。

这种基于代币的去中心化系统与创作者友好型平台的区别在于，通过购买创作者的社交代币，粉丝本质上是在投资该创作者。社交代币可以增加其价值，这使得它们对代币持有者有价值。这就解决了目前传统社交平台寻租的问题，即粉丝无法选择基于他们喜爱的创作者的成功而获得货币收益，所有收益归于平台所有。

已经在探索区块链技术作为创作者的工具的公司

这些技术所述的创作者经济中的潜在作用已经开始实现了，像Rally和Roll这样的公司正允许创作者创建他们自己的社交代币。也有许多NFT市场，如OpenSea，创作者用于铸造NFTs销售给市场用户。同时涌现出了一些创建DAO的工具和基础设施，方便创作者建立完完全全属于自己的粉丝经济。

总的来说，创作者经济与NFT和DAO相结合，为新的集体所有权模式奠定了基础。这意味着将注意力转移到创作者和观众的创造力上，让他们直接受益，而不是依赖中介机构。

创作者经济赋予了创作者更多的代理权。他们可以依靠其忠实支持者的收入。同时也改变了内容创作者的盈利模式，现在没有任何单一中心化平台可以垄断其创作者的巨额利润，这会激励创造者们释放出真正的潜力，促进整个创作者经济的竞争和创新。
