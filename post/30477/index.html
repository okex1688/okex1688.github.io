目前在 OpenSea 上有 400 万个NFT，在其他平台（如 Rarible、Foundation、SuperRare、NBA Top Shot 等）上还有着数百万个NFT。知道 NFT 世界是巨大且不断扩大的，可以肯定地说并非所有这些资产都会升值。那么应该如何选择 NFT 项目？我们发现 NFT 的实用性（定义如下）是对 NFT 及其未来潜在用途的一个很好的介绍。

实用型 NFT：NFT 不仅是艺术品，更重要的是作为分配数字资产的用途。实用型 NFT 可以是一件与购买的 NFT 相匹配的实物艺术品；它也可以是对活动的特殊访问、独家会员资格或是未来在元宇宙中使用的道具（游戏道具等）。

下面，笔者将介绍 10个或是大家耳熟能详的、或是刚刚问世不久的实用型 NFT 项目。

VeeFriends（veefriends.com）  

VeeFriends 是知名投资人 Gary Vaynerchuk 所创立的 NFT 系列的名称。 Gary 手绘了 268 个简笔画作为 10255 个 VeeFriend NFT 的基础。这些草图带有独家福利,如礼品盒、导师会议和会议门票等。利用 NFT 技术及其智能合约功能围绕他的创意和商业热情建立一个社区。通过拥有一个 VeeFriend NFT 来成为 VeeFriends 社区的一员并可以访问 VeeCon。

VeeCon 是世界上第一个 NFT 票务会议（ ticketed conference）组织。VeeCon 是 VeeFriends 代币持有者作为一个社区聚集在一起以建立社交、分享观点和共同学习的地方。

Crypto Baristas（cryptobaristas.com）

Crypto Baristas 是世界上第一家 NFT 资助的咖啡馆。

Crypto Barista 的第 1 季将推出 60 个喜欢咖啡因的角色，这些 NFT 的所有权将助力于概念化和开设世界上第一家 NFT 资助的咖啡馆。Crypto Barista 的所有者在未来的所有咖啡馆空间和网站上都可以终生享受咖啡的福利。所有者还是“咖啡师银行”的决策者，咖啡银行的资金是从项目收入中留出的 15%，用于未来在咖啡领域的使用。 Barista Bank 的一些潜在用途可以支持咖啡领域的慈善组织、推进 Crypto Barista 项目或一些更新奇的玩法。

Crypto Baristas 的目标是建立一个由欣赏艺术、咖啡因、创业和创新的志同道合的人组成的社区。该项目的目标是首先在纽约开设一个以艺术和创新为前沿的实体空间。除此之外，加密咖啡师的未来将取决于项目的持有者和追随者。Crypto Barista 项目旨在解决许多 NFT 项目中有时缺乏的三个部分：治理、社区和所有权。该项目将通过季节性发布而发展，每个季节都专注于咖啡领域的新努力，同时为所有持有人提供津贴和治理的权力。

Moon Boyz（moonboyz.net）

Moon Boyz 是生活在以太坊区块链上的 11,111 个独特角色的集合。采用独特的 3D 设计，每个 NFT 都拥有不断增长的社区和很棒的实用程序的正式会员资格。

这个一个设定很有趣的项目，用其官网的介绍来说：“2021 年，11111 名退化成 jpeg 的百万富翁，将他们的意识加密到元宇宙中，试图通过这种办法来逃离我们正在崩溃的社会。他们正在共同努力实现一个共同的使命：征服元宇宙的不同部分并建立一个新的月球秩序。” Moon Boyz 意图用图片 NFT 对应每一个持有者，以此来建立一个带有门槛的社区，让 11111 个人进行一个基于 NFT 的脑洞游戏。

MekaVerse（themekaverse.com）

MekaVerse 是 8,888 个生成的以太坊 NFT 数字艺术作品 Mekas（一种类似高达的机器人）的集合，包含受日本 Mecha 宇宙启发的数百种元素。每件艺术品都是原创的，有自己的颜色与风格。这样做的目的是使每个 Meka 都独一无二，以便促使玩家优先考虑品质而不是数量。

自Mekaverse于2021年8月份开通推特账户，在未开售之前其粉丝就一度逼近原NFT社区神话—Bored Ape Yacht Club。同时 Mekas 实体化具有巨大的潜力。Mekaverse 团队对 3D 打印充满了热情，并希望用先进材料将理想变为现实。

Nouns（nouns.wtf）

Nouns 作为一个 NFT 项目，由 DAO 组织管理，其试图引导身份、社区、治理和资金，社区可以用它来创造长期价值和项目扩展。Nouns 将伪随机生成图像，并结合拍卖、社区治理和一种新颖的渲染技术，每个 NFT 被称为一个 Noun（名词），且每天都会铸造一个 Noun，持续时间不限。 Noun 项目的独特之处在于形成自己的 DAO 资金库，从 NFT 销售中收集 100% 的资金。所有 Noun 都被拍卖，销售收入进入项目金库。NounsDAO 已经在 DeFi 治理协议基础上进行了修订，使 Noun 的持币者创建关于金库基金和协议升级的提案，并对其进行投票表决。

Noun 的渲染是一个让人眼前一亮的技术。其为每个特征生成的整数，可以调出存储在另一个名为 NounsDeor 的智能合约中的图像。图像是以名为 RLE 的无损数据压缩格式存储为字节串，最终可通过合约渲染呈现为 SVG。当 UI 调用 tokenURI 时，智能合约将从这些压缩的 RLE 数据派生的 SVG 予以返回。tokenURI 返回渲染一个高保真图像所需外全部数据，无需从链下调用任何对象。这一手段之前未在任何区块链艺术项目中出现过，很有趣。

Creature World（creature.world）

来自纽约市的 21 岁视频艺术家 Danny Cole  致力于通过他的“生物世界”项目来实现想象力。

该项目本身在 NFT 领域来说是一个很神秘的存在。 其网站作为一个互动空间，生物在其中引导访问者完成虚拟旅程。 目前尚不清楚个人所购买的角色最终是否会进入虚拟世界或项目的总体意图，但是这个很新奇的展开方式，一个蕴含着“秘密”的 NFT 项目，让旁观者十分好奇。

Adam Bomb Squad（abs.thehundreds.com）

如果你是街头时尚的粉丝，那么 Adam Bomb Squad 项目会是一个很适合的选择。 Adam Bomb Squad 项目由 The Millions（2003 年推出的著名街头服饰品牌）创建。该项目希望将身份、社区和所有权带入 NFT 空间。 “Adam Bomb”是数百个世界著名的标志/角色，Adam Bomb 项目旨在记录这个角色的 18 年历史和使用。该项目的梦想是超越基础设施，希望利用 NFT 解决在推广品牌方面的所有权问题。

Claylings（claylings.io）

The Claylings 是一个备受追捧的 NFT 项目，目前在整体趋势交易量中排名第 5。Claylings 项目希望将粘土动画带入区块链，打算创造出 4,040 个角色。 该项目的目标之一是制作一部 Claymation 短片，该短片将通过塑造的 NFT 粘土角色为出演者，如果你是这些“出演者” NFT 的持有者，当短片上映的时候一定会让你感到很酷。

Autograph.io（autograph.io）

Autograph.io 更像是一个市场，而不是一个专注于运动员、娱乐、文化和独特数字体验的项目。该项目由 Tom Brady 创立，并得到了大量的投资，同时有着包括来自 Naomi Osaka、Wayne Gretzky、Tiger Woods 等人的具有代表性的顾问委员会。

运动员推动了 Autograph.io 当前关注 NFT 的重点，包括专业人士的真实签名和私人 Discord 频道入口。由于所有相关人员的支持和支持，观看这个项目的展开将会很有趣。

Decentraland（decentraland.org）

Decentraland 是基于以太坊区块链上建立的一个虚拟世界，为用户构建了一个居于链上的虚拟世界。如今已经是人尽皆知的 NFT 项目之一，在元宇宙赛道持续占据着重要的位置。如果要说到区块链上对应的虚拟世界，绝大多数人想到的都会是 Decentraland。

用户可以在Decentraland的主体世界里参观其他玩家拥有的建筑、参与位于各建筑内的活动与游戏、触发一些特殊剧情（捡到收藏品等）、和其他偶遇的玩家通过语音或文字对话，操纵自己的Avatar在这个虚拟世界里尽情畅游。而且，用户还可以发挥创造力，通过Decentraland提供的制作器（Builder）创建属于自己的建筑，把它置于自己的世界里或对外销售。

确定什么是值得关注的 NFT 项目或值得投资的因素，并不总是那么容易。 在笔者选择这 10 个 NFT 项目时，收集的不仅仅是具有艺术品性质的项目，更多的侧重点在于 NFT 的实用价值。在决定投资 NFT 时，请注意应找到一个可以使你从中持续受益的实用项目。 例如，Crypto Baristas 是一个以咖啡为主题的 NFT 项目，该项目为纽约的一家咖啡馆提供资金，同时向持有人提供含“咖啡因福利”（例如打折咖啡、免费饮品等）。 VeeFriends 是另一个实用 NFT，它更多的是一种社交资格，一个可以聆听、参与投资圈达人会议的资格。

那些一鸣惊人的项目，往往在初期没那么容易看到它的潜力，因此除了通过一定的标准来构建模型进行评价之外，可以多关注 NFT 的实用价值，从而辅助大家寻找值得关注、投资的 NFT 项目。
