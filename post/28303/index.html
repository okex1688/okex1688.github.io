NFTs 正以猛烈的势头复苏。

在 2021 年上半年创造了超 25 亿美元的交易量之后，由于人们对作为绝大多数 NFT 交易活动聚集地的以太坊 所带来的环境影响的批评越来越多，以及市场对 NFTs 存在泡沫性的声音，这些基于区块链的数字所有权契约 (也即 NFTs) 的销量一度出现急剧下降。

但时间快进到过去几周，NFTs 真的又卷土重来了 ！大量新的 NFT 项目和趋势占据了新闻头条。本文将探讨当下 NFT 领域的最热门趋势，这些趋势正在引发有关加密货币最新态势的讨论，同时推动着价值数百万美元的资本移动。

俱乐部的会员资格：Bored Apes

NFT 爱好者们可能会将 CryptoPunks (加密朋克) 视为标志性的 NFT 品牌，但 NFT 领域的风向变化无常，就在上月早些时候，一个新的 NFT 王冠竞争者出现了：Bored Ape Yacht Club  (BAYC ，无聊猿猴游艇俱乐部) 。当前，NFT 领域正在为 BAYC 疯狂，它是一个描绘猿猴各种面部表情的 NFT 系列，总共只有 10,000 个 Bored Apes (无聊猿猴)，见下图。

上图：Bored Ape Yacht Club (BAYC) 中的几只无聊猿猴 NFTs 头像

根据 CryptoSlam 的数据，Bored Apes 的售价也很可观：上周，一只 Bored Ape NFT 以 600 ETH (当时价值约 255 万美元) 的价格被售出 (见下图)；截至撰文时，该 NFT 收藏品系列的交易总额超过了 4 亿美元。此外，当前有 100 多只 Bored Ape NFTs 正在 Sotheby’s (苏富比拍卖行) 进行拍卖，预计拍卖总额将高达 1800 万美元！

这些 Bored Ape NFTs 的持有者将他们的猿猴作为自己的 Twitter 头像 ，因此这是一种身份的象征 ，但这些持有者也获得了许多额外的特权 ：加入 Discord 聊天组与其他 Bored Apes 持有者 (包括 NBA 球星 Steph Curry) 进行聊天；仅限 Bored Apes 持有者参与的商品 (比如街头服装品牌 The Hundreds 与 BAYC 联名的服装)；以及 BAYC 额外向 Bored Apes 持有者免费提供 的 NFTs ，比如 Mutant Apes Yacht Club (MAYC，突变猿猴游艇俱乐部) 和 Bored Ape Kennel Club  (BAKC，无聊猿猴狗舍俱乐部)，见下图。

上图：Mutant Apes Yacht Club (MAYC) 系列中的几只突变猿猴 NFTs 头像

上图：Bored Ape Kennel Club (BAKC) 系列中的几只狗狗 NFTs 头像。

这些 NFTs 都是围绕「社区 」而蓬勃发展的。Bored Apes 和其他的 NFT 收藏品系列 (包括 Pudgy Penguins (矮胖企鹅)、Weird Whales (怪异鲸鱼)、Gutter Cat Gang 等) 感觉就像**会员俱乐部** ，这类项目正在推动第二波 NFTs 的购买热潮。

简而言之，对一些人来说，NFTs 代表了历史和文化中的一个独特时刻。Visa 最近购买了一个 CryptoPunk 头像，并认为 NFTs 是“历史性的商业产品”，如果 Visa 是正确的，那么这些数字图片重新点燃人们的兴趣就不足为奇了。

元老级NFTs：EtherRocks、CryptoPunks

如果你对 NFT 持怀疑态度，认为这些猿猴和企鹅的图片卖出数千美元有些不可思议，那么一些元老级的 NFTs 可能会更加让你瞠目。

EtherRocks 是一组由 100 个以「石头」为主题的 NFT 收藏品系列，每块「石头」除了颜色不同之外没有任何区别。然而，这些卡通石头的最低价在 8 月初超 10 万美元，截至 8 月底，EtherRocks 的交易量达到数百万美元。截至撰文时，这些「石头」的_最低价_已经达到了 800 ETH (约合 300 万美元)。

EtherRocks 的成名有两个原因。首先，这些 NFTs 的灵感来自于 20 世纪 70 年代流行的经典 Pet Rock (宠物石) 玩具；其次，作为早在 2017 年创建的首批 NFTs 之一，EtherRocks 有着更久的历史，且总共是有 100 个，这使得们已经成为了 NFT 收藏家们的热门资产。

上图：EtherRocks 系列中的几块 NFT 石头

与此同时，CryptoPunks  (加密朋克) 目前仍然是 NFT 所有权的「黄金标准 」，原因类似：它们存在的时长 。CryptoPunks 自 2017 年以来一直存在，并一直保持其价值。按照 NFT 的标准来说，CryptoPunks 是“元老级的”。

就像 Bored Apes 一样，将 CryptoPunk 作为 Twitter 头像也是一种身份的象征 ，以至于一些 CryptoPunks 的所有者正在将自己的 CryptoPunks 出租给其他人以赚取收益 。

大企业正在纷纷买入：Visa、Budweiser

在今年上半年的 NFT 热潮中，许多公司已经发现 NFT 的潜力并开始推出自己的 NFTs 来进行产品推广，比如美国快餐连锁店 Taco Bell 在 3 月推出以玉米卷为主题的 NFTs，卫生纸品牌 Charmin 也在 3 月份推出了以卫生纸为主题的 NFTs (见下图) 等等。

上图：Charmin于今年3月份推出的以卫生纸为主题的 NFTs。图源：Rarible

但在最近的一波 NFT 热潮中，各大品牌的做法略有不同，而且相当有趣：它们正在购买现有的 NFTs，而不是创建自己的 NFTs 。

上周，Visa 宣布以约 50 ETH (约合 16.5 万美元) 的价格购买了一个 CryptoPunk NFT (见下图)，目的是将其加入该公司的收藏品行列。8 月 23 日，Visa 在推特上表示：“今天，随着我们进入 NFT 商业的新时代，Visa 欢迎 CryptoPunk #7610 加入我们的收藏品。”

上图：8月23日，Visa 在推特上宣布购买了编号为 #7610 的 CryptoPunk NFT 头像。

Visa 的加密货币主管 Cuy Sheffield 非常看好 NFT，并相信它们会持续下去。他在一份声明中说：“NFT 作为数字优先的体育纪念品正在获得发展动力。”他补充说，Visa 预计 NFT“未来几年将出现大量新的用例”。

而 Visa 并不是最近唯一购买 NFTs 的大企业。

啤酒生产商 Budweiser (百威) 上周也购买了 NFT。这家啤酒巨头不仅以 30 ETH 的价格购买了一个名为 beer.eth 的 ENS 域名，还在 Opensea 上以 8 ETH  (当时约合 2.6 万美元) 的价格买入了一张描绘了「火箭啤酒」的 NFT 图片 ，并将之设置为该公司 Twitter 账户的头像 (见下图)。该 NFT 图片是 Rocket Factory NFT 系列中的其中一个，由艺术家 Tom Sachs 设计。

上图：Budweiser 的官方 Twitter 账号

上图：Budweiser 的以太坊账户中持有的 NFTs 资产，截图来源：https://opensea.io/0x01b9ae223e71b215209a0b6873ef63d7925862cc

碎片化的 NFTs

碎片化的 NFTs已经出现几个月了，但它们在最近今天开始声名鹊起。

虽然加密货币亿万富翁们正在兴高采烈地抢购 Bored Apes 和 CryptoPunks，但这些价格高达六、七位数的 NFTs 远远超出了普通加密货币用户能够支付得起的价格范围。但与传统艺术品不同的是，单个 NFT 是可以进行碎片化的，也即将其拆分成多个 (更便宜的) 部分 ，以供不太富裕的人购买 。

许多加密货币用户正在以 DAO (去中心化自治组织) 的方式聚集联合起来购买 NFTs ，从而获得对这一新资产类别的敞口。他们甚至将一张以 DogeCoin (狗狗币) 头像原型的原始 NFT 图片以碎片化的方式进行拍卖，将这整张 NFT 图片的价值推高到了 3.02 亿美元。

上图：2021年6月，DogeCoin (狗狗币) 原型柴犬的主人 Atsuko Sato 将这张狗狗币的原型图片铸造成了一个 NFT，并以创纪录的 1696 ETH (当时约合 550 万美元) 的价格卖给了实验性艺术 DAO 组织 PleasrDAO，之后 PleasrDAO 将此 NFT 碎片化成数十亿个部分。

但将 NFT 所有权进行碎片化也带来了另一个代价。据美国 SEC (证券交易委员会) 委员 Hester Pierce 称，碎片化的 NFTs 非常类似于未经注册的证券 。

“NFT 的碎片化让更多人能够从投资蛋糕中分一杯羹。但是，向投资者提供 _NFT 部分所有权_的公司可能没有意识到这样做带来的监管影响，”DeFi 交易平台 Swarm Markets 的联合创始人 Philipp Pieper 说道。

游戏化的 NFTs：Axie Infinity、Supdrive

如果不提到 NFTs 在游戏行业的应用 ，那本文显然是不完整的。NFTs 在游戏中的应用正在加密领域吸引大量的关注。

基于以太坊的区块链游戏 Axis Infinity 迅速成为了最突出和成功的加密游戏，自 6 月以来，其累计的 NFT 交易总额超过 16 亿美元，超过任何其他 NFT 收藏品系列或项目。这是一款怪物战斗的游戏，其中的生物 (称为 Axies) 本身就是 NFTs，且玩家通过玩游戏可以获得加密货币奖励 ，这些代币奖励在某些国家足以维持生计。

即便参与 Axie Infinity 游戏的最低成本 (需要至少购买三只 Axies 宠物) 是价值几百美元的 ETH，但这也没有阻止这款游戏的发展：Axie Infinity 现在有超过 100 万的日活跃用户 。没有任何一款加密游戏能够与 Axie Infinity 近期的发展势头相媲美。

此外，由 NFT 驱动的幻想足球游戏《Sorare 》拥有狂热的粉丝基础，而即将推出的 3D 元宇宙游戏《The Sandbox 》也拥有许多潜力和杰出的合作伙伴/投资者。

上图：基于 NFT 的区块链游戏《The Sandbox》

在 NFT 游戏领域的其他方面，视频平台 Vine 的联合创始人 Dom Hofmann 正在领导一个称为 Supdrive 的 NFT 视频游戏项目。Supdrive 将作为一个链上幻想游戏控制台 (On-chain fantasy game console)，使得每个可用的 NFT 都将是一个可玩的视频游戏。

生成艺术：Art Blocks

无论你是否相信，区块链不仅可以**托管代表某件艺术品所有权的 NFT 资产** ，实际上还可以**创造艺术品本身** 。

这正是当前蓬勃发展的生成艺术 NFT 市场 的基础：存储在区块链上的脚本或算法在铸造过程中产生原创的、独一无二的 NFT 艺术作品。Art Blocks 是一个生成随机艺术品的平台，是该领域最大的玩家。

这个基于以太坊的平台 (即 Art Blocks) 包括了各种艺术家的大量个人作品，跨越了不同的艺术风格和方法，而且最近几乎所有这些 NFTs 作品的售价都在激增 。仅在 8 月份，Art Blocks 的交易额就达到了 5.83 亿美元，因为收藏者们在新价格下跌时掀起了抢购狂潮，花高价在二级市场上购买这些 NFTs 生产艺术品。

8 月底，艺术家 Dmitri Cherniak 的生成艺术 Ringers NFT 系列中的一幅作品在 Art Blocks 上以 1800 ETH (当时价值约 566 万美元) 的价格卖给了 Starry Night Capital。见下图：

同一周早些时候，艺术家 Tyler Hobbs 的生成艺术 Fidenza NFT 系列中的一幅作品 (见下图右) 也以 1000 ETH (当时价值约合 330 万美元) 的价格被售出。见下图：

与此同时，由 CryptoPunks 的创作团队 Larva labs 发起的一个名为 Autoglyphs 的早期小型生成艺术 NFT 系列中，最近也有多个 NFT 的销售价格超过了 100 万美元。

上图：Larva labs 推出的生成艺术 Autoglyphs NFT 系列中的几幅艺术品

共同构建世界：Loot

这个趋势目前正在由一个称为 Loot 的 NFT 项目引领，并且已经产生了巨大的影响，可能被证明会对未来产生很大的影响。

Loot 是 Dom Hofmann 推出的一个由 NFT 驱动的项目，每个 Loot NFT 的核心是一个由虚构的幻想世界_武器和装备_组成的列表 (包括武器、胸甲、头甲、腰甲、脚甲、手甲、项链和戒指)。就是这样。Loot 没有涉及什么艺术作品或者方向，也没有特定公司或创造力来支配它们的使用。但 Loot 是故意被设计成这样的。

Hofmann 在 8 月下旬公开免费发布了 Loot 项目，之后社区已经在围绕 Loot 开发广泛的工具、衍生项目、公会和其他举措 ，以充实更多内容。它最终会成为一款成熟的幻想游戏吗？或者一种去中心化的知识产权？

现在下结论还为时过早。但加密社区正在讨论这些可能性，加上 Loot NFTs 的需求非常大 (其中一个 Loot 的售价高达 954,000 美元，见下图)，而且总量只有 8,000 个的 Loot 系列的交易量已经达到 1.59 亿美元。

上图为 Loot Bag #748，在 OpenSea 上以约 954,000 美元的价格被售出

至于接下来 Loot 会发生什么，完全取决于社区的集体想象和创造力了。
