前言：1947年，由Thomas T.Goldsmith Jr.和Estle RayMann申请专利，用八个真空管模拟导弹互相攻击的游戏机，这是有记录以来的第一个游戏设备。而最早有记录的电子游戏诞生于1952年，是一款井字棋游戏（Tic-Tac-Toe），《Noughts＆Crosses》，运行于真空管计算机上。从1947年到2021年，视频游戏经历了从第一代家用游戏机和Pong克隆衍生——专注策略和解谜游戏开发的大型计算机游戏时期——以《太空大战》（SpaceWar）游戏为代表的第一代网络游戏——以《拳王》为代表的街机电子游戏盛行的黄金时代——基于ROM的墨盒上进行游戏数据存储的第二代游戏机——早期家庭游戏——大型网络对战游戏——基于区块链技术及治理机制的次世代游戏的发展历程。

在区块链技术日趋成熟的今日，从第一款区块链的NFT游戏《加密猫》诞生时起，全球的开发者就关注到了区块链游戏这个空缺的市场并纷纷涌入其中。在如雨后春笋般涌出的区块链游戏中，称得上真正意义的游戏项目屈指可数。大多数区块链游戏项目印证了一句老话“画皮难画骨”，粗制滥造的游戏层出不穷，更别提什么匠人精神。从游戏逻辑到风格界面、从玩法设置到游戏生态，抄袭之风盛行。

在此之时，一个号称史上最懂区块链游戏本质的团队凭借NEXTYPE声名鹊起。有人不禁好奇，这样一个团队到底有何过人之处？

原因无他，NEXTYPE的游戏研运团队深耕传统游戏领域多年，连续推出多款全球现象级游戏产品，具备扎实的游戏开发功底和丰富的运营经验，曾在全球创造过破16亿美金总流水的业绩。NEXTYPE的游戏研运团队的经验并非一蹴而就，不少传统游戏玩家体验畅玩过的ACT动作游戏、AVG冒险游戏、FPS第一人称视点射击游戏、MMORPG大型多人在线角色扮演游戏、SLG策略游戏、TCG卡牌游戏、SIM模拟经营等类型的游戏，NEXTYPE团队在过去的15年里均有优秀作品面世。

据了解，NEXTYPE并非只是一个游戏项目，更是一个可跨链的游戏、NFT、DeFi 的综合应用生态，为用户提供高品质的基于 Dapp 的游戏+DeFi、游戏+NFT 资产合成、NFT+DeFi 的多形态产品。NEXTYPE团队也被支持者誉为“江湖游戏人”，这与NEXTYPE团队高调的行事作风息息相关。NEXTYPE团队认为，真正意义上的游戏，应该包括符合玩家特征的玩法系统，有成长黏性的数值体系和令人着迷的经济系统，并且在美术品质上要与时代接轨。

同时，NEXTYPE将按照标准化的敏捷开发方式，进行产品矩阵的布局，不断的推出高品质的各类型游戏，告诉区块链世界，未来的区块链游戏应该是一个什么样子。“没有人定义区块链游戏一定就是Dapp，它对实现真正的游戏价值有相对的局限性，而NEXTYPE也将在传统游戏产品内加入区块链技术，特别是NFT和DeFi的应用，那么将带给传统游戏巨大的变革，同时也为极大地拓展了传统游戏开发者的思维边界。”而这个大言不惭的团队，其实一直怀揣着赤子之心。NEXTYPE想要打造的是一个次世代的区块链游戏产品矩阵，将传统游戏的开发思路引入到区块链游戏世界，在各种玩法类型的游戏陆续上线之后，最终将组合成一个由NEXTYPE所畅想的Metaverse元宇宙。

小编认为，如果有着王者的实力，别人眼中的张扬也许只是一种诚实；如果实力平平，张扬就成了一种无知。究竟NEXTYPE团队是属于哪种呢？还看小编为大家深度探析NEXTYPE的野心才能做出判断。

除了上面所展示的对于提升区块链游戏品质的执着之外，小编还了解NEXTYPE还可以为不同的区块链项目发起者提供基于GameFi（游戏+DeFi）的加密令牌的发行通道，也可以为网络游戏提供 NFT资产升级的底层技术支持，并通过 Dapp 的方式用 NFT 建立起区块链世界与传统游戏世界的价值桥梁。让设想成为现实的是，NEXTYPE的核心创新技术，包括通过开发并部署NCCP 协议（NEXTYPE Cross-Chain Protocol），而此协议支持公链其所属的NFT资产可以通过该协议实现跨链存储、交换、流转，并为N-DEX提供有效支持；还有负责自有NFT价值交换协议的NVEP（NEXTYPE Value Exchange Protocol），实现了NEXTYPE生态内NFT-NFT，NFT-FT的多形态SWAP价值交换，全面提升各类型NFT价值。

简而言之，NEXTYPE为各类型区块链项目提供了融合DEFI模型的游戏化加密令牌发行通道及NFT支持，同时实现了NEXTYPE生态内多个游戏间NFT通证的兑换流通，多形态的价值交换成为了提升各类NFT价值的关键原因。

目前，NEXTYPE即将在HECO上推出一款融合了DeFi+NFT的虚拟挖矿游戏《矿场大亨》（也是NEXTYPE的首发游戏）。这是一款通过游戏化的虚拟挖矿可以获得平台通证的游戏，同时也是通证的首发应用场景，而它最大的特色是NX-L型矿证在高级矿池中的存在，实现了GAME+DeFi与BTC的联动与创新，并且通过创新的合作矿池为区块链项目方匹配潜在高价值区块链用户。以小编初步体验下来看，《矿场大亨》在游戏的美术、交互、动效、成长线等方面已经几乎近似传统休闲游戏的体验，但同时又在游戏内可以通过智能合约的方式进行质押挖矿，这种感受几乎可以对区块链玩家进行无死角打击，让人欲罢不能！作为对玩家资产的安全性保障，《矿场大亨》的智能合约已经通过了CertiK的合约代码审计，玩家们可以放心畅玩！

另外小编还了解到NEXTYPE正在开发一款专门铸造和交易NFT的游戏《NFT大师》，玩家可以在游戏内轻松定制自己可交易的NFT，让用户完整的感受完成NFT铸造、投放、获取收益的过程。游戏在推出的同时将携带众多知名IP甚至是全球顶级IP共同登场，为用户定制带有复合多场景权益的NFT，打造一个IP+NFT+GAME的无敌阵营。到时候会是怎样一番景象，让我们拭目以待！

就NEXTYPE推出的游戏产品本身的游戏逻辑与玩法、玩家体验、DeFi+NFT技术落地、产品美学的艺术设计等多维度因素而言，NEXTYPE相对于其他半路出家的区块链游戏团队来说，专业实力与对游戏的见解能力处于不同次元。近期NEXTYPE也高调官宣正式加入了区块链游戏联盟BGA（Blockchain Game Alliance），即将于本月25日参加由BGA组织的产品分享会，届时可以了解NEXTYPE更多消息！

小编认为，也许对于行事如此张扬的团队来说，其游戏的丰富经验与完整战略思路的提出，只是NEXTYPE团队的先发优势。究竟随着时间的流逝，NEXTYPE综合应用游戏平台到底能否持续研发出更多元化的产品？能否持续推动DeFi+NFT应用市场的发展，从而征服万亿级游戏应用生态市场？各位玩家不妨持续关注，与小编一起见证NEXTYPE的未来与发展。
