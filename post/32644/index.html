DeFi

1.Umbrella Network与PlayMining合作扩展跨链功能

1月5日消息，DeFi预言机Umbrella Network宣布与PlayMining达成合作伙伴关系，以继续扩展跨链功能。PlayMining是一家全球性的GameFi和NFT娱乐公司。 通过Umbrella Network的去中心化预言机，PlayMining DEAPCoin资产的定价数据将通过智能合约提供给dApp。 DEAPCoin是一种支持PlayMining生态系统的加密货币，该生态系统包括加密游戏和一个NFT市场等，DEAPCoin-USD定价数据现在可以在Umbrella Network上获得。

2.Livepeer完成2000万美元B轮股权融资，TigerGlobal等参投

1月5日消息，去中心化流媒体Livepeer宣布完成2000万美元B轮股权融资，本轮融资由TigerGlobal、Digital Currency Group、North zone和Warburg Serres等机构及包括英国对冲基金经理Alan Howard在内的天使投资人参投。 据悉，Livepeer是一个基于以太坊的去中心流媒体，旨以分散的方式和较低的成本对视频进行转码。转码是重新格式化原始视频文件以确保最佳观看体验的过程，任何人都可以加入Livepeer并进行转码，用户将以以太坊或Stablecoin赚取费用。（TheBlock） 金色财经此前报道，2021年7月底，Livepeer完成2000万美元B轮融资，DigitalCurrencyGroup领投。

3.神鱼：Cobo 2022年将大力推动DeFi合规基金和DaaS链上业务

据官方消息，神鱼在Cobo内部发出全员信，在全员信中神鱼表示2021年是无穷的行业的机会与更大力度的政策壁垒相互交织的一年，同时并表示2022年Cobo将增强自身合规意识、积极拥抱监管、正确的处理好合规与发展的关系。目前正在新加坡和香港申请相关的合规牌照，同时也在不断探索欧洲及美国地区的合规化途径。 并在全员信中表达了对于Cobo的未来战略规划，将继续坚持区块链基础设施中间键的战略目标，大力发展DaaS、DeFi Fund和WaaS等多个优势，深化自身技术安全及用户安全意识培养。并将70%-80%的战略重心放在已有和即将深耕的领域，分出20%-30%的精力会去做一些探索性的业务。

4.DeFi平台WonderFi以超2亿美元收购加密交易市场Bitbuy

1月5日消息，DeFi平台WonderFi Technologies Inc.宣布已与Bitbuy母公司First Ledger Corp.达成协议，将以2.06亿美元的现金和股票收购Bitbuy交易平台，此次收购预计于今年第一季度完成。据了解，Bitbuy于去年11月成为加拿大第一个获批的加密交易市场，用户数超37.5万，通过该平台进行的交易超44亿美元。去年12月，WonderFi首次宣布对Bitbuy母公司进行战略投资。

NFT热点

1. NFT项目“变异猿”MAYC交易总额突破6亿美元，10天增长超1亿美元

金色财经报道，1月5日，据最新数据显示，NFT项目“变异猿”MUTANT APE YACHT CLUB交易总额已突破6亿美元，创下历史新高，本文撰写时为645,386,008美元，当前交易总量为22,315笔。Mutant Ape被设计成“变异猿”BoredApe Yacht Club的低级别NFT，通过Mutant Ape为现有猿猴NFT持有者提供了一种全新的NFT，即变异（mutant）版猿猴，并同时吸引新用户进入BAYC生态，成为低级别会员，该NFT项目交易总额在12月26日达到5亿美元，这意味着在短短10天内增长超过1亿美元。

2.NFT DAO将于1月11日向OpenSea用户发放NAO Token空投

1月5日消息，去中心化自治组织NFT DAO将于1月11日向OpenSea用户发放NAO Token空投。NAO总量为100万亿，其中50%将空投给OpenSea用户、20%用于质押奖励、20%用于Naffiti NFT市场开发、10%用于LP奖励。快照已于2021年12月31日20时在区块高度13916168处进行，其中OpenSea上的NFT交易数量（ERC）占40%权重，OpenSea上NFT拥有的数量占40%权重，累计消耗的Gas费用占20%权重。用户必须在2022年6月31日7:59之前领取空投Token，任何剩余的Token将被发送到DAO金库用于社区计划。 需要注意的是，由于合约未披露审计情况或存在风险，用户谨慎操作注意资产安全。

3.The Sandbox 宣布与舒淇等多个香港合作伙伴在元宇宙中创建 Mega City

1月5日消息，土地沙盒游戏TheSandbox于今日宣布与多个来自电影、音乐、娱乐、表演、金融、房地产和游戏行业等新的合作伙伴达成合作，共同创建一个大型文化中心MegaCity。新的参与者通过在开放元宇宙中获得LANDNFT来加入TheSandbox的虚拟世界，并在MegaCity中创建体验。为了庆祝新的合作伙伴的加入，TheSandbox将于2022年1月13日推出新一轮LAND销售，供玩家购买位于新合作伙伴附近的LAND。

4. DreamVerse 即将上线欧易NFT市场，400枚土地NFT盲盒上架

据官方消息，DreamVerse 土地NFT盲盒将于1月5日20:00（HKT）上线欧易NFT市场，开启限时抢购。本轮DreamVerse 土地NFT盲盒发售数量400个，每个定价0.25ETH，每个用户最多可Mint 5个NFT。 Dreamverse是一个将众多明星及影视、动漫、游戏等顶级IP世界和现实世界高逼真、精细化还原的虚拟世界。拥有非常多IP，获得日本漫画家手冢治虫多个作品的主要角色3D形象NFT及元宇宙授权，且已和麦迪达成合作，授权麦迪在元宇宙建立麦迪庄园的权力。

5.CES 2022展会将首次以单独类别展出NFT及区块链技术

1 月 5 日消息，据报道，2022 年国际消费类电子产品展览会（CES）今日开幕，其中 NFT、加密数字货币以及其他区块链技术将首次被划分为一个单独类别进行展出，预计展会将吸引 7.5 万名观众和包括亚马逊、Meta、谷歌等传统科技巨头在内的超过 2200 家展商参展。 据悉，本次参展的 NFT 相关企业包括 NFT 销售与交易平台 Autograph 以及一些数字货币公司。此外，主营互联网硬件业务的 Netgear 将在 CES 上展示其用于 NFT 艺术的数字画布，同时消费电子巨头三星将推出世界上第一个基于电视屏幕的 NFT 浏览器和市场聚合器。 CES 由美国电子消费品制造商协会主办，旨在促进尖端电子技术和现代生活的紧密结合。该展始于 1967 年，现已成为全球各大电子产品企业发布产品信息和展示高科技水平及倡导未来生活方式的窗口。

6.澳洲电信巨头TPG接班人与22岁NFT投资者Ishan Haque联合推出5000万美元元宇宙基金

1月5日消息，澳洲电信服务提供商TPG接班人Jack Teoh与个人投资者Ishan Haque合作推出5000万美元元宇宙基金Geminio Capital，旨在投资元宇宙、GameFi游戏等。Axie Infinity的种子投资者Sherwin Lee，ASX 200成分公司如Aristocrat和WiseTech Global等上市公司董事Arlene Tansey担任基金顾问。 

7.Cosmos Theta升级拟于Q1进行，将启用NFT模块、跨链账户等功能

1月5日消息，Messari分析师Mason Nystrom发推表示，Cosmos Theta升级（定于2022年Q1进行）将会带来巨大变更。具体而言，Cosmos Theta升级将潜在启用： 1. 元交易； 2. NFT模块； 3. 跨链账户； 4. Liquid质押； 5. Hub上的CosmWASM实例； 6. Gravity Bridge升级。

GameFi热点

1.NFT游戏Starbots完成240万美元融资，AU21 Capital等参投

1月5日消息， Solana 上的机器人对战 NFT 游戏 Starbots 宣布完成 240 万美元私募轮融资，本轮融资由 Impossible Finance、Defi Alliance、Solar Eco Fund、Parsiq、OKEx Blockdream Ventures、TomoChain、Good Game Guilds、Real Deal Guild、Gamee Studio、LuaVentures、Kyber Network、Solscan、Kyros Ventures、BigCoin Capital、TK Ventures、Exnetwork Capital、Hub Capital、AU21 Capital、Basics Capital、Formless Capital、Quadhorn Guild 参投。 Starbots 是一个基于 Solana 的机器人对战 NFT 游戏，玩家可在游戏内根据不同战略任意组装自己的机器人、赢得战斗、完成任务和征服新土地来获得 GEAR Token。

2.Islandswap上线OEC首周TVL突破1200万美金

据官方消息，2021年12月28日在OEC主网上线的islandswap，上线仅1周时间，TVL突破1200万美金，最高超过1400万美金。 Islandswap是一个搭建在OEC上的岛屿养成类游戏，游戏分为池子、IFO、创建主基地、Market、兑换、NFT盲盒、幸运抽奖6大板块，是由玩家、DAO 和游戏开发者共同设计的，集DeFi+GameFi+NFT市场于一体的的综合平台。
