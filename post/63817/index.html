当 AI、Web3 与二次元文化相遇，会碰撞出怎样的火花？对创作者而言是否意味着新机遇？

11 月 22 日晚，LBank 最新 Launchpad 项目、「Web3 版 Midjourney」 ACGN Protocol 社区大使 Ichika 做客星球超话社区，详解 ACGN Protocol 如何颠覆 ACG 创作者经济。

据 Ichika 介绍，ACGN 是一个功能全面的平台，不仅提供图像和文本内容生成的 AI 工具，还提供 AI 模型训练工具，能够一站式满足创作者的需求。ACGN 将人工智能技术与 Web3 技术相结合，为用户提供在去中心化网络上创建和交易 NFT 内容的机会。

最近 ACGN 与 LBank 合作进行测试代币 $ACGNxp 空投活动，产品上线后 $ACGNxp 将以 1: 1 的比例兑换成 AIGN 生态代币 $AIMEME。此外，ACGN 未来将推出创作者激励计划。

以下为社区对话实录，由Odaily星球日报整理：

Odaily星球日报：请向大家简单介绍一下 ACGN Protocol。

Ichika： ACGN Protocol 是一个集成尖端人工智能技术的开创性平台，为创作者和爱好者带来更多可能。ACGN Protocol 集成先进的人工智能技术，是一个开创性的 Web3 ACGN（动画、漫画、游戏、小说）创作者生态系统。ACGN Protocol 基于开源创作者协议，为用户提供领先的人工智能工具，使他们能够创建源自原始知识产权（IP）的衍生内容。

ACGN Protocol 为用户提供基于人工智能的先进创作工具，允许无缝上传衍生作品。我们的平台通过提供用于内容创作的人工智能辅助工具、训练人工智能模型以及探索各种多样化素材等功能，促进了用户的创作过程。此外，用户可以在平台上进行 NFT 交易，使用视觉小说创作工具轻松地加入插图、音乐、配音和字幕，从而创作出吸引人的视觉小说等内容。

Odaily星球日报：在过去的两年里，人工智能的概念一直很热门。ACGN Protocol 选择在人工智能领域开发产品，肯定具备其独特的优势。您能简单介绍一下 ACGN 的具体优势吗？为什么 ACGN 能脱颖而出?

Ichika： ACGN Protocol 在人工智能领域脱颖而出，具备以下几个独特的优势：

  • 综合平台： ACGN 不仅提供图像和文本内容生成的 AI 工具，还提供 AI 模型训练工具，打造了一个全面的创作者生态系统。这个功能全面的平台能够一站式满足用户的创意需求，涵盖了从创作到训练的所有内容。

  • Web3 集成： ACGN 将人工智能技术与 Web3 技术相结合，为用户提供在去中心化网络上创建和交易内容的机会。这种集成允许创作者在去中心化环境中制作、分发和交易 NFT 内容，从而获得更广泛的曝光和收入机会。

  • 社区驱动： ACGN 注重社区建设，鼓励创作者和用户之间的互动和合作。这种社区驱动的方式促进了思想交流和创新，使 ACGN 的内容更加多样化和丰富。

  • 用户友好： ACGN 的工具和平台设计简单、对用户友好，即使没有专业设计或编程背景的用户也可以轻松上手。这种用户友好性吸引了更多的创作者和普通用户加入 ACGN 生态系统。

综上所述，ACGN Protocol 不仅仅是一个应用 AI 技术的平台，它还是一个综合的生态系统，集创造、交易、社交互动和技术创新于一体。这些多方面的优势使 ACGN 在竞争激烈的人工智能领域脱颖而出。

Odaily星球日报：听完您对项目优势的介绍，我感受到了 ACGN 的独特之处。请问 ACGN 有计划进行用户可以直接参与的活动吗?

Ichika： ACGN Protocol 与 LBank 交易所已经举办 $ACGNxp 空投活动，为用户提供参与 ACGN 生态系统的独特方式。通过此次空投活动，用户有机会获得 ACGN 的测试代币 $ACGNxp。一旦产品上线，这些测试代币将以 1: 1 的比例兑换成 AIGN 生态系统代币 $AIMEME。我们为此次空投活动提供总共 500 万枚测试代币，活动今天刚刚结束了。

本周 ACGN Protocol 的第一代产品 AI Image Generator V1 已经上线，全网Web3用户都可以参与使用。项目还在 X（原 Twitter）、Galxe、QuestN 等多个平台举办 GA 活动，有兴趣的朋友们欢迎参加。

Odaily星球日报：作为一个创作者工具，确保创作者的隐私至关重要。ACGN Protocol 如何确保平台内用户数据和交易的安全？

Ichika： ACGN Protocol 非常重视用户数据和交易的安全，实施了一系列措施来保护创作者的创作隐私和用户的个人信息安全。

  • 加密技术： 该平台采用先进的加密技术，确保平台上所有的数据传输都经过加密，防止数据在传输过程中被拦截或篡改。

  • 去中心化存储： ACGN 采用去中心化存储技术，将用户数据分散存储在多个节点上，避免了单台服务器集中存储方式带来的风险。这种去中心化存储方式提高数据安全性，防止单点故障的发生。

  • 智能合约安全性： ACGN 依赖智能合约进行 NFT 交易和创作者奖励。该平台经过严格的安全审计，以确保智能合约的安全性和稳定性，降低与合约漏洞相关的风险。

  • 匿名交易： ACGN 允许用户进行匿名交易，保护用户身份信息。用户可以参与 NFT 交易和其他活动，而无需透露个人身份。

通过这些措施，ACGN Protocol 致力于确保平台内用户数据和交易的安全，为创作者提供一个安全可信的创作环境。

Odaily星球日报：您能和我们聊聊全球二次元热潮吗？ACGN Protocol 有针对创作者的特别计划吗?

Ichika： 必须有，ACGN Protocol 非常重视二次元文化的全球影响力。为此，我们制定了一系列专项计划，旨在支持和激励创作者，促进二次元文化的创作和发展：

  • 创作者激励计划： ACGN 计划在产品上线后的一个季度内建立创作者激励计划。通过在专门网页上的社区投票，成员将决定每个时期的获奖者。该计划鼓励创作者在平台上传高质量的 2D 作品，包括动画、漫画、游戏、小说等。每种类型的作品都有专门的版块。优秀作品将获得丰厚的奖励和推广机会，希望激励创作者创作出更多精彩的内容。

  • 创作资源支持： 为了促进 ACGN 生态系统蓬勃发展，该平台将为创作者提供各种创作资源支持，包括图像、音乐和动画库。这有助于创作者更方便地创作 2D 作品。值得注意的是，我们的人工智能创作工具可以帮助创作者轻松将自己的想法变成现实。

  • 社区和项目合作： 我们积极与 2D 社区以及 Web3 社区合作，组织各种创意大赛、展览和活动。这些活动旨在为创作者提供展示作品、扩大影响力的机会，并为用户提供更多精彩的内容。

  • 创作者交流与合作： ACGN 不仅仅是一个创作工具，也是一个充满活力的创作者社区，鼓励创作者之间的交流和合作。该平台为创作者提供专属的交流空间，让创作者互相学习和合作，共同推动二次元文化的繁荣发展。

Odaily星球日报：您能详细说明一下平台内 $AIMEME 的各种用例吗?

Ichika： $AIMEME 代币在 ACGN 生态系统中有各种用例，旨在促进平台开发和用户参与。以下是 $AIMEME 代币在 ACGN 生态系统中的具体应用：

  • 创作者激励和奖励 ： $AIMEME 代币将用于奖励和激励 ACGN 平台上的创作者。创作者通过发表优秀作品、参加创意大赛并吸引观众，可以获得 $AIMEME 代币作为奖励，鼓励其创作出更多精彩的内容。

  • 平台服务支付： 用户可以使用 $AIMEME 代币支付 ACGN 平台内的各种服务费用，包括创作者工具使用费、交易手续费等。这种方式为用户提供了更灵活的支付选项，从而促进代币的流通。

  • 社区治理： 持有 $AIMEME 代币的用户可以参与 ACGN 社区的治理决策，包括提出建议、投票支持或反对提案等。这种方法允许社区成员积极地为平台的发展做出贡献，并共同决定未来的方向。

  • 项目和社区合作： $AIMEME 代币可用于在平台内建立合作伙伴关系。代币持有者可以与 ACGN 平台合作，推动生态系统的发展并分享生态价值。

总而言之，$AIMEME 代币在 ACGN 生态系统中具有多种功能，不仅为创作者提供激励，还为用户提供更好的体验和参与机会。这些用例将有助于建立一个充满活力、多样化的 ACGN 生态系统，促进社区的繁荣和发展。

社区成员：二次元文化在东亚的发展比较好，是否也意味着 ACGN 主要也是面对东亚用户，是否有欧美发展的计划？

Ichika： ACGN 文化在东亚地区的发展确实较为显著，但 ACGN Protocol 是有计划在全球范围内推广和发展的。虽然二次元文化在东亚拥有广泛的影响力，但 ACGN Protocol 有可能吸引并服务全球范围内对这一文化感兴趣的用户。在 ACGN Protocol 的发展计划中，可能会考虑面向欧美等地区的用户扩展，包括定制化的推广活动、多链发展、多地区项目合作，以及线上、线下与当地创作者和社区的合作等。

社区成员：$AIMEME总量是多少？有没有通缩机制？

Ichika： $AIMEME总量 10 亿枚；有明确的通缩机制，除了产品使用等用途之外，会定期对收入进行销毁。

社区成员：ACGN 如何产生利润/收入来支持项目？它的收入模式是什么？如何让投资者与项目双赢？

Ichika： 产品未来会通过设置不同的权益、设置不同方案，为不同需求的创作者提供最合适的套餐。通过平台收入、手续费、社区激励、高级功能订阅等途径获取收益。
