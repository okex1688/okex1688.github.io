以太坊融资平台 Gitcoin 的第十一轮捐助活动正在如火如荼地进行中，本期 NFT 板块又涌现出了一批高质量的创新项目。

随着 NFT 行情遇冷、市场普遍回调，人们开始逐渐把关注的焦点从众多常见的头像、Loot 系 NFT 项目转移向其他更新鲜、更优质的项目身上。律动 BlockBeats 对参与本期捐助活动的 NFT 项目进行了筛选，将用户关注度较高、产品模式较为新颖的项目进行了梳理。

StartFi

StartFi 是一个为 NFT 构建的内容创作者服务平台。简化 NFT 铸造和购买流程，帮助 NFT 作者筹集资金。提供资金的社区成员可以获得 NFT 奖励和 STFI 平台代币收入。StartFi 所有平台的费用都将使用 STFI 代币交易。

Audio Software for EVM

Audio Software for EVM 旨在打造一款音乐 NFT 播放应用，通过这款应用，用户可以播放自己在以太坊上持有的音乐 NFT 并将它们保存在本地。

Zero Utility Token

Zero Utility Token 旨在通过创新型产品打通 NFT 与 DeFi 的边界，是一个完全由社区开发的代币，目前仅由捐赠和产品补贴组成。所有收入都将被放入项目钱包，由多签保护，且开发者钱包密钥也已被销毁。社区开发的 Zero Utility Tools 也有着极高的实用性，其中 MultiSender 能够同时向多个地址发送 ERC-1155 和 ERC-721 NFT，该工具目前可以在以太坊和 Polygon 网络上使用；NFTForge Immolator 可以为用户铸造具有动态属性的 NFT，如用户可以铸造具有定时销毁功能的 NFT，或者在 NFT 钱包的持有人预设的代币余额低于阈值的情况下进行销毁。Zero Utility Token 则可以用来以更低的 Gas 费向多个用户快速创建、分发和管理 NFT。

Dataverse

Dataverse 是一款 NFT 收藏插件，为 OpenSea、Foundation、SuperRare 等各大主流 NFT 平台提供一键收藏功能和汇总功能。方便用户能够集中查看不同平台喜欢的 NFT 作品情况。该工具测试版目前已上架谷歌浏览器扩充插件商店。

EthBlockArt

EthBlockArt 是一个基于以太坊链上的在线生成艺术平台，与 ArtBlocks 类似，用户可以在 EthBlockArt 选择购买不同系列的艺术品，不过不同的是 EthBlockArt 允许用户自己对生成艺术品的部分参数进行调整，而不是纯随机生成。

Cross-Asset Swap

Cross-Asset-Swap 是一款类似于「NFT 版 1inch」的协议，允许用户在任何 ERC-20、ERC-721、ERC-1155 代币之间进行交易。具体流程为先将 ERC-20 代币和 NFT 打包，通过 NFTX、NFT20 等 NFT 指数基金将 NFT 转换为 ERC-20 代币，再通过 Uniswap 等交易平台交易为想要得到的 NFT 指数代币，再将这些指数代币通过 NFTX、NFT20 等转换为 NFT。不过目前还不支持 P2P 交易，用户需等待其他用户匹配报价。

MintFund

Mint Fund 是一个社区项目，创建的目的是支持那些想加入加密世界的艺术家们，Mint Fund 会为这些艺术家提供铸造 NFT 所需的 Gas 费，并未他们提供指导、辅助和建议。该基金对所有艺术家开放，但目前优先考虑有色人种和 LGBTQIA+的艺术家，以及北美和欧盟以外国籍的艺术家。

每一次的热潮都不可能永久地持续，总会有遇冷的那一天，NFT 亦如是。所以当 OpenSea 单日交易量从超 3 亿美金一路走低至 5000 万美金，各种曾经热门的 NFT 项目交易量大幅下跌时，我们需要从疯狂回归理性，寻找更多优质、创新的项目并帮助他们发展。
