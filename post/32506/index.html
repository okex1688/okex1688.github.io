GameFi热点

1. 多链NFT游戏Blockchain Monster Hunt完成首次BCMC销毁

12月31日消息，多链NFT游戏Blockchain Monster Hunt宣布，随着Testnet V.3的推出，已经完成2777778枚BCMC（价值3972222美元）的首次销毁。

2.Twitch联合创始人的游戏NFT平台Fractal已上线

金色财经报道，Twitch联合创始人Justin Kan基于Solana的游戏NFT平台Fractal于周四正式推出，推出了一系列play-to-earn游戏。Fractal的第一批游戏合作伙伴包括十多个Web3游戏，例如The Sandbox以及play-to-earn游戏Genopets和SynCity。该平台既是用户直接从公司购买NFT以在游戏中使用的市场，也是点对点交易的二级市场。

3.CeloLaunch实验室已对MetaCelo游戏完成评估

据官方推特消息，CeloLaunch实验室已经完成MetaCelo元宇宙NFT游戏的项目评估。MetaCelo是一款建立在Celo 网络上的“Play-To-Earn”元宇宙NFT游戏。同时是一款开放世界的角色扮演区块链游戏，玩家将成为怪物训练师并体验自己的冒险。 CeloLaunch实验室致力于识别、投资和授权可行的区块链企业家、初创公司和社区权力，为行业项目提供融资，帮助项目在Celo和其生态系统中做大。

4.NFT赛马游戏Zed Run发布2022年路线图

12月31日消息，NFT赛马游戏Zed Run发布2022年路线图，要点如下： 1. 改进等级制度，时间为第一季度。 2. ZED RUN代币。代币分发将分为三个主要类别：游戏内活动；增长激励；现有用户。 ZED RUN代币将分四个阶段启动： 第一阶段：初始空投（第一季度），将空投给符合条件的马厩主人和ZED RUN NFT持有者。分配将基于以下因素：拥有的赛马数量；拥有的赛马类型；ZED RUN NFT所有权的时间。 第二阶段：赛马比赛（第一季度）。 第三阶段：赛马退役（第二季度）。 第四阶段：租赁（第二季度）。 3. 终极锦标赛和赛马体验（从第一季度开始）。

NFT热点

1. MetaBox将联合Pokemon推出限量版NFT

据官方消息，MetaBox已成立MetaDAO基金用于DAO管理和慈善事业，并将联合Pokemon口袋怪兽推出限量版NFT。

2 . Readoo将于2022年1月推出Genesis NFT

12月31日消息，Web3小说平台Readoo将于2022年1月推出Genesis NFT。据悉，Readoo是基于Solana上的去中心化Web3平台，作者和读者可以通过NFT版税获得写作和阅读的奖励。

3. 宫崎骏的十部画作首次以NFT结合形式发售

12月31日消息，宫崎骏的十部经典画作，将授权机甲NFT项目MetaRim作为NFT背景使用。这也是宫崎骏第一次将其画作与NFT相结合。 据悉，MetaRim将于1月5日开启白名单预售。宫崎骏十款NFT画作将随机分发到参与预售活动的地址中。

4. 价值220万美元的被盗BAYC被NFT市场OpenSea“冻结”

金色财经报道，NFT收藏家Todd Kramer今天宣布，他收藏的价值220万美元的Bored Ape Yacht Club NFT被“黑客入侵”，NFT市场OpenSea已经为他“冻结”了这些资产。该收藏家在纽约经营Ross+Kramer艺术画廊。Kramer在推特上说，他点击了一个链接，该链接看起来是一个真的NFT dapp。但结果证明这是一次网络钓鱼攻击，导致他的16个NFT被盗，包括1 个 Clonex、7 个 Mutant Ape Yacht Club 和 8 个 Bored Ape Yacht Club NFT 。目前价值约 615 ETH（228 万美元）被盗，现在无法在 OpenSea 上交易。

5. 周杰伦和昆凌获赠PHANTACi NFT

金色财经报道，周杰伦和昆凌分别通过Instagram宣布已获赠PHANTACi NFT。此前消息，NFT平台Ezek宣布将于2022年1月1日与PHANTACi合作发布Phanta Bear系列NFT，该系列NFT共将发行10000枚。据悉，潮流品牌PHANTACi成立于2006年，由流行音乐巨星周杰伦与好友Ric共同打造。
