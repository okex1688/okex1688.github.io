2020 年，加密金融全面复苏，最重要的议题就是 DeFi。到了 2021 年，热点轮转到非同质化代币（NFT）。
NFT 并非新事物。然而，直到基于以太坊的游戏《加密猫》正式推出， 关于 NFT 的想法才得以实现。

从那以后，NFT 狂潮席卷而来，一些独特的收藏品估价急速上升。来自全球的艺术家、艺人、名人和其他知名人士的发声，也推动 NFT 获得更高的知名度，就连苏富比和佳士得等传统机构也开始入局这一领域。

有着 15 年从业经验的金融分析师、宏观经济专家 Luis Aureliano 表示，NFT 使得区块链与普罗大众更为贴近。“虽然加密货币功能越来越多，但由于吸引力有限，其采用势头缓慢。NFT 成功地吸引了人们的兴趣，一定程度上要归功于众多名人代言，引起了主流社会的关注。"

在 Luis 看来 NFT 背后的平台是其创新的堡垒，而不断上涨的原生代币价格则反映了平台的受关注度。

“当一些 NFT 平台首次出现时，它们的产品被认为是毫无用处的。现在，随着流行文化、艺术和收藏进入他们的生态系统，提高区块链的参与度，这些组织笑到了最后。相应的代币估值也呼应了这一现实，并预示了 NFT 的未来的增长潜力。

下面这十个 NFT 项目及其代币值得重点关注。因为它们的进步和管理工作，为区块链的可用性和覆盖面开辟了新纪元。

LeCube

LeCube 是链游老兵 MixMarvel 即将正式上线的新品。该团队的曾在 2018 年出品《云斗龙》等多款火爆链游，一度超过《加密猫》。

LeCube 主打 NFT 社区创造以及 NFT+DeFi 的新玩法，旨在改变目前 NFT 定价不成熟以及共识快速流失等困境。

首先，玩家可以向智能合约中质押任意加密货币来获得 Cube。在「创作中心」中，玩家用 Cube 可以拼成 NFT作品。该 NFT 作品可以被放到「市场」进行出售。也可以将其放到「版权中心」，供其他玩家付费引用。先放入版权中心的作品可以被优先引用。整个 NFT 的搭建，都是在链上进行，每个元素到作品都是原生的区块链资产，整个创作过程都是在链上。

目前 LeCube 处于测试阶段，今年 Q2 将正式上线。从 3 月开始，MixMarvel 的原生代币 MIX 开始飞速上涨，涨幅一度超过 4 倍，也显示了市场对其新品的期待与认可。

（LeCube 测试网页面）

Decentraland

如果你是一个喜欢游戏，并且对区块链技术感兴趣的人，那么这个平台应该会让你感到兴奋。

Decentraland 是一个去中心化的在线虚拟现实游戏平台，2018年项目发起，但 2020 年才最终上线。该平台基于以太坊构建，主要目标是支持虚拟财产的创建、开发、销售和托管。

在 Decentraland 上，用户可以购买虚拟财产发展在线和虚拟业务。平台上的虚拟土地可以用来建造不同的结构，这些结构可以被代币化或出售以获得利润。

LAND，是一种 ERC-721代币，是平台上代表土地的不可替代资产。整个游戏中的 LAND 有 9 万个，大小为 33*33 英尺。不过，LAND 的高度是没有限制的。因此理论上说，所有者可以继续建造更高的建筑物。

就像在真实的城市中一样，LAND 的集合形成了一个区域，一组具有相似主题的建筑。MANA是 Decentraland 的原生代币，采用 ERC-20 标准，它赋予了用户对其区域相关事项的投票权。除此之外，它还可以用来购买 LAND 和平台上的其他虚拟服务和资产，目前价格是 0.9858美元。

LAND 在 Decentrland 上用途非常广，我们已经看到 Rarible、SuperRare、MakerDAO等区块链公司将他们的 LAND 地块用于虚拟办公室和画廊，也有些用户将其用于虚拟会议和现场活动。

（Decentrland官网）

Enjin

这是另一个面向游戏玩家的区块链技术平台。Enjin 本质上是一个供用户存储和管理游戏中使用的虚拟资产的场所。因此，无论是代表游戏中物品（如配件或武器））的NFT，还是游戏中的货币，都可以在 Enjin 上存储、管理和销售。

Enjin 基于以太坊构建，原生加密货币是 ENJ。用户可以创建数字资产，并将其整合到他们需要的应用和游戏中，并且可以定制资产到需要的平台上工作。这些资产以 ENJ 进行估价，目前 ENJ 价格为 2.39 美元。能够为多种游戏环境铸造数字资产，是 Enjin 的价值所在。目前，该平台已经与微软等主要巨头合作，他们认识到该平台可以为其生态系统带来的巨大潜力和价值。

（Enjin官网）

Ethernity

Ethernity Chain 是目前最热门的 NFT 平台之一，其代币 ERN 在 3 月获得大幅上涨。

在 2021 年 3 月 8 日推出后，ERN 首发价格为 0.275 美元；但在 Polkastarter 上进行IDO 后，ERN 价格很快上涨到 3.5 美元；接下来五天，交易价格上涨到 33 美元，并在 3 月 27 日达到 74 美元的历史最高价；虽然此后价格跌至 38.7 美元，但 ERN 在不到一个月的时间里的价格轨迹表明其潜力巨大。

Ethernity Chain 是一个通过 NFT 出售运动卡、收藏品和数字艺术品的数字艺术市场。其目标是将部分收益用于帮助慈善机构，促进社会公益。它还帮助艺术家和其他创作者拍卖他们的作品，并将部分收益捐给艺术家所选择的慈善机构。

Ethernity 还在继续发展，最近它宣布著名数字艺术品的创造者Jason Hauser合作，向授权革新方向迈出了一大步。

（Ethernity网站收藏品截图）

Theta

Theta 区块链是一个基于区块链技术的去中心化视频平台，旨在通过激励用户在网络上共享其带宽来解决流媒体行业的问题，用户可以获得 THETA 代币作为回报。该项目成立于 2018年，已经获得三星电子和索尼公司的风投基金的支持，这些都表明它得到了主要行业参与者的支持。作为唯一的去中心化视频流媒体和传输的端到端平台，Theta 拥有巨大的潜力。

THETA 目前价格是 11.79 美元，随着 Theta 区块链的不断发展，还有更多的提升空间。有趣的是，THETA 曾一度跻身市值排名前十的加密资产，也表明了其发展潜力巨大。

（THETA官网）

Superfarm

另一个值得关注的 NFT 平台是 SuperFarm。它的创建者将其描述为一个跨链 DeFi 协议，它可以将任意代币转换为 NFT 农场，而无需任何编码，这将解决许多加密项目为其基础资产和代币创建应用场景的难题。该平台包括一系列工具，项目可以使用这些工具为代币增加实用性并激励行为。

SuperFarm 代币目前的交易价格为 2.77 美元，是一个月前的两倍多，因为越来越多的人开始认识到它的实用性和价值。使用 SuperFarm 进行收益耕作以及 NFT 挖矿的便利性，使其成为一个了不起的平台。它的用户体验非常出色，其 NFT 基础设施也很全面，该平台拥有：

  • NFT 市场：允许用户交易 NFT 资产的市场。
  • NFT 和 ERC-20 代币生成：生成 NFT 和 ERC-20 代币以启动定制化挖矿。
  • NFT 挖矿：抵押 SUPER 代币并获得 SuperFarm 合作伙伴挖矿池中的独家NFT奖励。
  • NFT 多元宇宙游戏：结合 NFT 和 SuperFarm 的可互操作的元宇宙视频游戏。

（SuperFarm官网）

KIWIE

KIWIE 是一个数字艺术市场，通过 NFT 出售街头艺术品。对于绘制的每一幅街头艺术品，还将创建与之匹配的数字艺术品，并添加包含实物坐标的 NFT。凡是在这个平台上购买数字艺术品的人，也就拥有了这幅街头艺术品。一旦街头艺术品被重新喷漆或销毁，数字艺术品就会永久转化为幻影版本（ghost version）。

这个平台上的所有艺术品都是以 Fat Monster（胖怪物）为原型创作，平台的目标是在 195 个国家拥有艺术品。

（Fat Monster）

Unicly

另一个值得关注的 NFT 平台是 Unicly。虽然它还没有发布，但该平台在 NFT 市场的潜力已经预示着其将在未来取得成功。

Unicly 的卖点在于它能够将 NFT 拆分。NFT 通常被认为是代表唯一数字资产的唯一代币，这也一直是它们的主要卖点。但这也带来不小的挑战，尤其是在出售高价 NFT 时。有了Unicly，用户将有可能拥有 NFT 的一小部分，从而获得更多的流动性。该平台将推出 uTokens，它将赋予这些 NFT 的持有者投票权，决定是出售还是持有。

（Unicly 官网）

Terra Virtua

乍一看，Terra Virtua 可能看起来很像 Decentraland。但是，Terra Virtua 提供的体验与后者并不相同。Terra Virtua是一个增强虚拟现实平台，允许销售、收集和创造NFT。

Terra Virtua 将允许用户购买或创建 NFT 环境，如赛车场和虚拟休息室，并在其中填充建造、赚取或购买的可收集的NFT。

Terra Virtua 提供来自诸如《环太平洋》和《迷失太空》等娱乐特许经营权的收藏品。Terra Virtua 是为收藏家建立的，以展示和探索他们的收藏品，所有这些都是在区块链去中心化系统保障的安全环境中进行的。

Hoard Platform

Hoard Platform 已宣布推出 NFT 市场，用户可以在其中贷款、租赁和交易 NFT。贷款人可以在使用 NFT 作为抵押品的同时，发放有溢价的稳定币贷款。

Hoard 还支持 HRD 抵押，用户可以成为 Uniswap 等去中心化交易所的流动性提供者，获得 UNI v2 代币、手续费以及其他奖励。

Hoard 旨在将其 NFT 市场和软件开发联系起来。这种独特的价值主张将通过提供基础设施，将游戏中的项目整合到以太坊区块链上来实现。
Hoard SDK是 Hoard 提供的软件开发套件，它是一个易于使用的开源工具，可以将任何游戏与其区块链整合。该工具将确保游戏开发人员在尝试成为区块链编码领域的专家时，可以继续与 NFT 进行交易而不会陷入困境。

游戏内的资产（如虚拟房地产或收藏品）现在可以轻松地进行代币化，玩家可以更轻松地交易物品。 

（Hoard官网）

写在最后

有加密专家和分析师认为，2021 年是属于 NFT 的。到目前为止，一些动向也表明，这种预测并不是不着边际的。

无论是已经售出的数百万件 NFT 艺术品（最大的是 Beeple 的 6900 万美元的艺术品），还是多个 NFT 平台的发展，都证实了 NFT 已成气候。

目前 NFT 还处于早期阶段，还有很多亟待完善的不足之处。希望整个行业能共同携手，砥砺前行。
