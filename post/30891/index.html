原标题 | YGG 投资版图再扩张！与 8 款边玩边赚游戏达成合作！

Yield Guild Games（YGG）是一个去中心化的游戏公会，向玩家出借游戏 NFT 资产，玩家可以在前期没有投入的情况下开始玩区块链游戏并获得收入。从 2021 年 10 月起，YGG 宣布与 8 款 NFT 游戏达成了合作并进行投资，包括 Influence、CyBall、Thetan Arena、KOGs SLAM！、MOBOX、Aavegotchi、DeHorizon 和 Genopets。YGG 成立于 2020 年底，为了满足玩家的需求，YGG 目前的投资组合中有超过 18 款游戏，包括 Axie Infinity、Splinterlands 和 Zed Run 等等。

在今年早些时候，YGG 完成了 2240 万美元的融资。其中第一轮由 Delphi Digital 领投，投资金额为 990 万美元；中间轮由 BITKRAFT 领投，随后追加的一轮投资由 a16z 领投。加上首次向游戏玩家出售代币获得的 1250 万美元，YGG 正在使用这些资金购买优质的 NFT 游戏资产，旨在为社区提供在区块链游戏经济中边玩边赚的机会。关于 YGG 加密货币资产的全面概述，可以在最新的资产和财务报告中找到。

YGG 合作伙伴关系主管 Sarutobi Sasuke 介绍了 YGG 评估 NFT 游戏潜力的方法，他表示：“社区参与度是 YGG 购买游戏的关键因素。我们团队会审查游戏玩法和游戏经济模型，以确保游戏是宏大且公平的，对玩家投入的时间和努力要给予充分的奖励。当然游戏公会、团队和举行联赛也是很重要的组成部分，因为可以激励了公会以及学员之间的集体努力。

以下是 YGG 最近合作的游戏介绍，包括购买的游戏资产的细节以及玩家可以享受到的好处。

Influence

由 Marina Ortega 设计的 Influence 概念图

Influence 是一款沉浸式太空战略大型多人在线游戏（MMO），提供了引人入胜的游戏玩法，用户可以掌握 NFT 游戏资产的真正所有权。玩家通过多种途径进行竞争，包括采矿、建造、交易、研究和战斗，并通过第三视角控制船只和建造的设施在沉浸式 3D 宇宙中进行互动。玩家可以通过游戏获得他们即将推出的代币 SWAY，Influence 允许玩家交易游戏中的资源、组件、船只和小行星。

YGG 从 OpenSea 购买的小行星

YGG 已经购买了影响力排名第 46 位的小行星——KJ-44947，这是一颗 C 型碳质小行星。除此以外，YGG 还购买了大约 30 颗较小的小行星供 YGG 社区使用。

CyBall

CyBall 展览模式 3v3 比赛中的 UI 概念界面

CyBall 是一款以足球为主题的、基于 NFT 的游戏，玩家可以收集、交易、指导和战斗 CyBlocs ，CyBlocs 是虚拟机械人角色，是区块链上的可收集和可交易的 NFTs 。玩家训练他们的 CyBlocs ，在各种游戏模式中竞争，获得不同程度的奖励。现有的 CyBlocs 也可以指导初级的 CyBlocs ，创造下一代独特的 NFTs 。

在与 CyBall 的合作中，YGG 获得了部分创世 CyBloc NFT 卡包，游戏一开始 YGG 就能为至少 1000 名 CyBall 学员提供。

Thetan Arena

这是一款 MOBA 手游，玩家可以召集朋友组成团队，与他人战斗，并通过游戏技能和团队合作获得代币奖励。这是一款免费的游戏，玩家可以在游戏中赚取名为 Thetan Coin（THC）的代币。与此同时，游戏还有一个边玩边赚模式，可以利用高级英雄赚取更多代币。

Thetan Arena 部分英雄一览

YGG 参加了 Thetan Arena 的代币销售，并购买了 250 个普通英雄，100 个 史诗英雄和 50 个传奇英雄。

KOGs SLAM!

KOGs 图片，来源：FROX 网站

由 RFOX 开发的 KOGs SLAM！是一款区块链手游，其规则类似于风靡 1990 年代的 POGs 游戏。然而，KOGs SLAM！采用了一种进化的数字形式，它使用 NFT 收藏品作为游戏碎片，不仅能玩游戏，还可以获得奖励。KOGs 是可收集的、可玩的、可交易的、可竞技的 NFT，搭建在 WAX 区块链平台上。KOGs SLAM！向东南亚地区的数百万用户介绍了一种从玩到赚的模式，玩家可以从日常任务和锦标赛中赢得 KOGs 和游戏代币 KOINS，并且可以将游戏收益换成现实生活中的收入。

来自 RFOX VALT 网站的概念图

YGG 还获得了 RFOX VALT 游戏区的两个 Marquee Brand SHOPs 的其中之一，这是一个互动和完全沉浸式的购物体验，结合了游戏元素，形成了自己独立的元宇宙虚拟现实。YGG 会员将能够访问 YGG 的 SHOP ，与社区互动，收集 NFT ，玩游戏，以及进行更多其他体验，所有的这些都在 VR 中进行。

MOBOX

MOBOX MOMO 神秘盒子

MOBOX 作为 Binance 智能链上的一个领先项目，是一个社区驱动的游戏平台，它将产量耕作和游戏中的 NFT 结合起来，创造了一个免费游戏和边玩边赚的生态系统。MOBOX 已经在其平台下发布了四款游戏，即《 MOMO：区块争霸》、《MOMO：代币大师》、《MOMO：农民》和《交易行动》。MBOX 是 MOBOX DAO 的原生治理代币，用来获得额外的角色、彩票券和奖励。

YGG 已经获得了 5 个 YGG 品牌的传奇款 MOMO NFT 以及 100 个 KEYS ，这些 KEYS 可以在所有的 MOBOX 游戏中使用，用来耕种 MBOX 代币。

Aavegotchi

X 射线视角的 Aavegotchi 堡垒地图

Aavegotchi 带来了虚拟宠物游戏宠物蛋的怀旧感，只是这次的电子宠物变成了居住在以太坊区块链上并可以运用 Polygon 网络的像素化幽灵。在ERC-721 标准的支持下，这些 NFT 的价值由其稀有程度决定，稀有程度通过多个因素计算，包括基本特征、配备的可穿戴设备和抵押品的数量，比如 Aave 协议的有息代币。玩家可以通过连接他们的钱包访问他们的土地，并将其出租给 Aavegotchis ，从而分享耕种产量的分成，并解锁更多的耕种潜力，以及社交和探索游戏的全部广度。

YGG 及其 subDAO Yield Guild SEA（东南亚）和 Yield Guild ASIA 已经购买了独家的 YGG 主题土地（REALM土地）以及 Aavegotchi 头像。并且还将购买可穿戴设备，为今年12月 Gotchiverse 上线时的游戏做准备。

DeHorizon

DeHorizon 英雄

YGG 的学员们还可以加入 DeHorizon 元宇宙史诗般的奇幻世界，玩家们可以在这里玩得开心，赚得开心。DeHorizon 游戏建立在 Ethereum-EVM 兼容链上，可以让玩家们探索史诗般战斗的开放世界，铸造 NFT 怪兽，开采资源，踏上英雄的征程，驯服野生生物，以及还有更多其他互动体验。DeHorizon 游戏包括团体对团体的冒险游戏 DeVerse、每周战斗游戏 DeTournament、骑龙幻想体育游戏 DeQuidditch ，以及更多其他游戏。

YGG 购买了 DeHorizon 游戏中的资产，包括独家武器、装备、龙族赛车队、 DeMeta Pass ID 和 40 个名为 JuJus 的怪兽，这些怪兽也将进行特别定制来代表 YGG 。

Genopets

Genopets 水晶

YGG 还通过与 Genopets 合作，继续扩大其在 Solana 生态系统中的游戏资产收藏。Genopets 是一款在结合边动边赚游戏机制和激励体育活动方面处于前沿的游戏。Genopets 是生成 NFT，可以进化、升级和定制，具有 8 个可变的仿生特征和四肢，属于 5 种元素类型之一。游戏通过不同期的解谜小游戏将宠物蛋的培育与 Pokémon 的训练和战斗相结合，将用户在现实生活中的运动转换为 XP （经验值），以便在游戏中取得进展。

YGG 已经购买了定制品牌 Genopets Habitats 和 Genopets 治理代币 GENE

关于 YGG

Yield Guild Games（https://yieldguild.io/）是一个去中心化的游戏公会，汇集投资者的资金购买能够产生收益的 NFTs，通过运用玩家的在游戏中所花的时间和精力，来优化社区拥有的资产，让其发挥最大的效用，收获最高的回报。YGG 融合 NFT 和去中心化金融（DeFi），通过发展虚拟世界和区块链游戏的内容和经济，来完成为社区成员创造价值的使命。

2021 年 5 月发布的一部名为《边玩边赚：NFT游戏在菲律宾》的纪录短片，揭示了一个农村社区在新冠疫情期间通过游戏赚取收入的故事。这部纪录片由 Yield Guild Games、 Delphi Digital 和 Emfarsis Consulting 合作制作，展示了 NFT 和加密货币在发展中国家是如何为缺乏就业机会和政府补助金的年轻人和老年人提供变革性的经济机会。
