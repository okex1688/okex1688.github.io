互联网的下一个乌托邦会是Web3吗？

Web3，正在成为互联网的新爆点。

2021年底，在美国众议院举行的加密货币听证会上，一句“我们如何确保 Web3 革命发生在美国？”蹿红。不久后，特斯拉 CEO 埃隆·马斯克在个人社交媒体平台发文表示，“有人看过 Web3 吗? 我没有找到。”

有人认为，Web3 代表了互联网的下一个阶段，也许还能代表人类社会的下一个阶段；也有人认为，被炒起来的Web3只是一场醒不来的梦。

Web3的本质是什么，谁可能会选择Web3？已经有了一些实践和应用的Web3，真的会成为互联网的下一个“乌托邦”吗？

一个关于“信任”的问题

尽管Web3还没有一个明确的定义和实现时间，但回溯Web1、Web2时代的历程不难发现，互联网的问题，最终是一个关于“信任”的问题。

1994年，Yahoo的成立带来了网络黄页，查找第一次有了目录和分类；四年后，Google则推出了大受欢迎的搜索服务功能。习惯了参与和互动的网民们，偶尔会怀念这些由静态网页组成的互联网早期形态。

1996年末，YAHOO公司就已经奠定了它在门户网站的领导地位

各类网站用户阅读内容和广告，互联网公司依靠点击量盈利。搜索引擎与门户网站，构成了单向性提供和理解内容、由开放协议主导的Web1时代，内容提供商和搜索引擎通过HTTP/HTML等协议，连接浏览器，进而接触消费者。

2005年，Twitter 和 Facebook的出现开启了Web2时代，只能阅读的网络演变为新的社交空间，你可以在Twitter上分享日常，也可以在APP上评价或售出商品。互联网由被动地接收信息迭代为主动创造信息。

互联网变得更有吸引力，却也涌现出更多信任问题。

用户在开源项目中，选择信赖技术、信任机制与代码，来执行任务；而在封闭源代码的商业模式中，用户选择信任公司运营者，能够保证他们的信息不被泄漏、数据网络处于安全状态。

但现实往往与想象相悖。

2017年5月，勒索蠕虫WannaCry病毒感染了150多个国家的电脑终端，它的核心攻击代码源于美国国家安全局NSA的承包商泄露出来的攻击工具，后来有研究者指出是某国的黑客利用这个工具写成了这个极具破坏性的勒索蠕虫，受害者达200余万。

2020年8月，B2B营销公司OneMoreLead被发现在一个不安全的数据库中存储了用户的私人数据，该公司将该数据库完全开放。据称，至少6300万人的姓名、电子邮件地址和工作场所信息直接暴露给拥有网络浏览器的人。

像《Polkadot Lightpaper》中所说，“当下的互联网世界却是这样的，用户向科技巨头们免费提供个人数据后，除了向神祈祷个人隐私不被泄露、窃取、滥用之外，别无选择——这些中心化世界的巨头们就像恶魔一样一口一口将我们吞噬。”

个体化意识和隐私安全被不断强调后，对于Web3.0的种种想象应运而生。

既然信任协议不能保证用户安全，Web3.0给出的答案是，“无需信任”。

以太坊的联合创始人、“Web3.0”概念的提出者Gavin Wood认为，Web3.0的定义是“少一点信任，多一点真实”。在他眼中，信任意味着任何人和组织在得到授权后，能够以任意方式行使互联网权力。如此一来，信任可能会变得盲目，变得不再是真正的信任。

Gavin Wood，英国计算机科学家，以太坊的联合创始人

Web3的设想改变了过往的信任机制。以前是用户信任企业和巨头，但在Web3的世界里，用户使用“去中心化”的平台，这是一种几乎无需信任承诺的架构，在这个平台中，大型科技企业将被从传播系统中移除，让互联网中大部分是免费、开源并且独特真实的交易信息。

互联网的数据和应用将不再由Facebook、亚马逊、苹果这样的科技巨头作为中介垄断，而是让每个用户拥有自己的数据，并能在不同网站上使用。

用户参与的一切互联网交易、数据传输，都将消除“中间人”，用户也不再需要因为使用网络泄露个人数据。Web3支持者认为，新的世界需要比特币一类真正“去中心化”的产品存在，没有中介，互联网为集体所有，基本上不受政府管控。

概念提出者Gavin认为，Web3 的目的不是取代科技巨头，而是在尝试一种社会运动——让权力的所属更加自由。“这是我能看到的保护自由世界的唯一方式，这是我们过去 70 年来享受的生活，也是我们让它在未来 70 年继续运行的唯一方法。”

商机

新事物的出现往往蕴藏商机，而资本常常流向尊重它的地方。

Twitter正在研究如何将Web3概念融入社交网络。“Web3意味着所有创造出的价值都能被更多的人分享，而不仅仅是企业主、投资者和员工。”Twitter高级项目经理Esther Crawford说，“很长一段时间Web3都非常理论化，但现在有一股迅猛增长的势头。”

Crawford表示，Twitter希望有一天能通过与加密货币关联的账户（而非Twitter账户）登录社交网络并发布消息——不是用加密版Twitter取代Twitter，而是在标准版Twitter的基础上引入Web3功能。

Web3的概念首先应用在了DAO（Decentralized Autonomous Organizationd，去中心化自治组织）的实践中。

2021年底，红杉资本官方将简介变更为“从想法到落地。我们帮助有胆识的人缔造传奇DAO，从创意到代币空投。”

红杉资本简介变更

加密投资巨头A16z则在今年一月发布了塑造web3未来的“10条原则清单”，其中包括释放DAO的潜力，和拥抱监管稳定币的金融包容性。

事实上，DAO的想法在比特币被设计出不久后就被提出：如果比特币可以破除金融中间人，那么也许公司或其他组织也能在没有分级管理的情况下运作。简单来说，DAO意味着互联网自治组织，没有集权领导，通过建立在智能合同中的链上规则来运作。

A16z负责人Ben Schecter认为，互联网工作的未来，就在于DAO和加密网络。新的未来工作将由围绕加密协议形成的网络实现，价值捕获将越来越多地从组织转移到个人身份。

Web3时代下的DAO应用将意味着对以往朝九晚五生活的颠覆。

举例来说，传统的赚钱方式是「工作赚钱」，但收入的未来是「X-to-earn」——玩游戏赚钱，学习赚钱，创造赚钱，工作赚钱。 而这一切都将依托于DAO，它可以在企业系统的背景下协调这些活动。

Ben Schecter提到，作为开放经济体的 DAO 将推动 X-to-earn 的趋势，人们的工作将更灵活、更流畅、更有趣。 加密经济的开放性将允许人们参与多个 DAO 和加密网络，混合、匹配不同的收入和回报，DAO 将扩大对多种类型参与者开放的机会类型和数量，包括代币持有者、赏金猎人和核心贡献者。

例如，代币持有者可以获得主要 DeFi （去中心化金融）协议的赠款、各种代币的被动收益收入；赏金猎人将通过完成激励性的链上行动来赚取；网络参与者可以通过玩Axie Infinity 或其他即将出现的P2E游戏来赚钱……支持者认为，DAO 最终将取代传统模型。

DAO也已和NFT发展成为了一种共生关系，一些DAO被建立的目的就是为了让用户拥有有价值的 NFT。

一些品牌和明星也抓住了商机，推出了NFT-Web3项目，尝试用数字代币联结现实世界的服务和产品。

2021年12月18日，周杰伦的潮流品牌PHANTACi与NFT发行平台Ezek合作，宣布将在元旦推出名为Phanta Bear的NFT项目，1月1日当天，Phanta Bear NFT一售而空。

据了解，目前，包括耐克、塔可贝尔和可口可乐在内的多个公司已经推出了公益筹款相关的NFT，或正在对区块链技术进行投资。

2021年，以6935万美元售出的NFT作品《Everydays: The First 5000 Days》

DAO和NFT自治的特点，显然与Web3畅想中的去中心化不谋而合。

实际上，Web3的概念也的确因为区块链技术改变过。2016年，区块链技术风靡互联网之前，Web3的概念更强调“语义网”与“万物互联”，互联网内容的个性化筛选与交互是Web3概念中的主角。

区块链技术的到来显然让Web3有了新的可能——让用户从科技巨头手中夺取互联网数据的主权。

但距离Web3的实现仍然有不短的路。

在技术规模化上，加密货币现在的市场规模没有想象中高，比特币显然不足以撼动和占领货币市场。量子运算技术则是区块链最大的威胁──采取RSA加密算法的区块链，对现在的科技水平仍然是暂时不可攻克的难题。

除此以外，Web3的反对者也认为，真想要走到互联网的“无政府”去中心化状态，恐怕并不可行。

马斯克在采访中曾对Web3表示怀疑：“现在看起来更像是营销的流行语而不是现实。”

Twitter前CEO转载相关漫画

软件工程师Geoffrey Huntley则表达了更多质疑：“目前，大多数区块链，尤其是在NFT领域使用的区块链是无权限的。”

他认为，Web3的实现，意味着用户存储NFT的数字钱包可能被发送任何内容——包括暴力色情或其他数字形式的骚扰。

这时，你很难确定Web3的到来究竟让用户拥有了更多权力，还是丧失了更多权力。反对者认为一个不被治理的社区终究是空谈。

Twitter前CEO Jack Dorsey认为，用户并不拥有Web3，Web3的实际拥有者是项目背后的风投机构（VC）及其有限合伙人（LP），“Web3永远不能逃离他们的激励，最终成了一个带有不同标签的中心化实体。”

维护和反对Web3的声音同样激烈。但更多入局还在加速，1月11日，NFT游戏开发商 Zynga宣布将被游戏巨头 Take-Two Interactive 以 127 亿美元的价格收购，副总裁Wolf认为，“基于 Web3.0 的 Token 系统将使用户能够改变玩家之间的互动方式，通过 NFT 创造一种整合的体验，让用户成为主人。”

不论如何，从DeFi的兴起到NFT的热潮，再从元宇宙到Web3，旧世界已经松动，更多人在涌入新概念下的技术市场。

正如布莱恩·阿瑟所说，“新技术在现有技术的基础上发展，现有技术又来源于先进的技术，技术的’组合’和’递归’特征，将彻底改变我们对技术本质的认识。”

Web3是否真的是一场“把主权归还给人的革命”，还需要等待更多技术的大规模落地，但可以确定的是，和以往一样，新技术和产业还会在这个阶段不断涌现，迎接互联网的每个新浪潮。
