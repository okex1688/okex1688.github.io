有哪些最实用的 DeFi 工具可以帮助您了解和跟踪更广泛的 DeFi 生态系统呢？

在本文中，我们将为每个 DeFi 投资者展示最实用的工具。

1. VFat Tools

https://vfat.tools/

vFat Tools集产量挖矿分析、挖矿计算器、仪表板和发现服务于一体。

它是 yieldfarming.info 的一个分支，它已提升功能以适用于多个区块链及更多其它项目。用户可以使用 vFat 工具来计算收益或通过不断更新的列表发现新的矿池。

优点

  • 自由

  • 干净的界面

  • 提供产量和矿池信息

  • 支持许多项目和矿池

缺点

  • 没有用于可视化分析的图形和图表

2. DEFI PULSE

https://defipulse.com/

DEFI PULSE是一个您可以通过锁定的总价值找到 DeFi 协议项目的最新分析和排名的一个平台。

DeFi Pulse 监控以太坊区块链上每个协议的底层智能合约，他们通过提取这些智能合约持有的 ETH 和 ERC-20 代币的总余额，每小时刷新一次图表。

优点：

  • DeFi 协议项目锁定的总价值概览

  • 能够查看 TVL 的折线图和趋势

  • 查看你最喜欢的 DeFi 协议的排名

缺点：

  • 不跟踪协议 TVL 的跨链部署

  • 协议 TVL 更新缓慢

3. DeFiLlama

https://defillama.com

DeFiLlama —您可以在其中跟踪协议统计数据，例如 TVL、1h 变化、7d 变化等。

他们列出了所有链上的 DeFi 项目，目前支持 Ethereum、Binance Smart Chain、Avalanche、Solana、Polygon 和 Terra。

您还可以按类别搜查 DeFi 协议项目，例如 DEX、资产、贷款、收益、保险等。

优点：

  • 跨多个区块链跟踪更多协议项目

  • 对类似比较的协议项目进行分类

缺点：

  • 没有

4. Dune Analytics

https://duneanalytics.com**/**

Dune Analytics是一种用于区块链研究的工具，用户可以在其中查询、提取和可视化来自区块链的大量数据。您可以创建自定义查询来获取数据，将这些查询的结果可视化，然后使用仪表板用您的数据讲述故事。

优点：

  • 非常强大的功能集，可从区块链数据创建自定义查询和仪表板

  • 可以使用熟悉的语言 SQL 进行查询

  • 能够分叉查询或使用模板快速入门

  • 内置可视化服务

缺点：

  • 没有

5. Revert Finance

https://revert.finance/

Revert Finance是 DeFi 中流动性提供者的分析工具，旨在为投资者提供关键和准确的流动性池信息。它目前支持在 Uniswap V2 和 V3 以及 Sushiswap 中跟踪 LP 位置。

优点：

  • 显示损益、GAS 成本、流动资产等的有用分析功能

  • 美观的用户界面

缺点：

  • 极其有限的流动池支持限制了它的实用性

6. APY Vision

https://v2.apy.vision/

APY Vision是一种一体化仪表板和分析服务，用于管理您的 DeFi 流动资金情况并实时跟踪您的非永久性损失。它跟踪您的流动资金的进入和退出以产生洞察力。该服务是由一个小团队软件开发人员和流动性提供商构建的，他们看到了向 AMM 提供流动性的机会，并在最近推出了V2版本。

优点：

  • 非永久性损失与持有的复杂分析

缺点：

  • 付费模式锁定了许多有用的功能

7. Messari

https://messari.io/

Messari是一个加密研究平台，通过专业级工具、研究和数据为加密经济带来透明度。

如果您订阅 Pro 或 Enterprise 计划，Messari 会为其数据工具以及研究、情报和见解提供监视列表、筛选器和图表。他们的基本计划——非常适合临时用户——提供基本的筛选、图表功能和指标。

优点：

  • 免费版可用于基本功能

  • 用于加密研究的宝贵信息宝库

缺点：

  • 为了充分利用平台（例如 API），建议使用 Pro 或 Enterprise 计划

8. Token Terminal

https://www.tokenterminal.com/

Token Terminal是一个平台，可聚合领先区块链和去中心化应用程序上的金融数据，类似于彭博终端在去中心化世界中的运作方式。

许多投资者使用 Token Terminal 来跟踪协议项目指标，如市值、交易量、TVL、收入等，它来自第三方数据源，如 The Graph、Alchemy、Coingecko、DeFi Pulse 等。

优点：

  • 有许多可用于顶级协议的可跟踪财务指标

  • 通过每 4 小时更新一次的数据追踪供方和所有者的收入

  • API 可用（仅限专业版）

缺点：

  • API服务成本相对昂贵

9. DeFi Saver

https://app.defisaver.com/

DeFi Saver是一个一站式仪表板和 DeFi 管理应用程序，用于创建、管理和跟踪您的 DeFi 资金。

使用 DeFi Saver，您可以在 Aave、dYdX 和 Compound 等借贷协议之间交换贷款头寸、管理您的 DeFi 投资组合、再融资贷款、管理您的流动性宝库、质押 ETH，甚至可以使用Recipe Creator建立自定义交易。

优点：

  • 与支持的协议交互的一站式平台

  • 许多有用的功能，例如Recipe Creator和loan shifter

  • 模拟模式可用

缺点：

  • 有限的协议支持

其它值得推荐的工具：

1. Cancel Ethereum Transactions

  https://cancel-ethereum-transactions.web.app/

Cancel Ethereum Transactions是一个简单的 web 应用程序，允许您取消或加速已广播到 Ethereum 网络的交易。

2. Fees.wtf

https://fees.wtf/

Fees.wtf是一个网站，显示您的钱包花费了多少以太坊GAS费用。您还可以使用该工具下载交易数据。

3. Sandwiched.wtf

https://sandwiched.wtf/

Sandwiched.wtf是一种工具，可让您查看 MEV 交易者及其机器人通过将您的交易夹在 Uniswap v2 上获得了多少利润。这是Alchemist的社区项目。

4. Degenscore

https://atomic.blue/degenscore

Degenscore允许您查看您的钱包承受了多少次rugpulls (指一些项目通过包装自己来骗取用户质押和投资,随后立刻卷款逃跑)、黑客攻击和模仿。它衡量了你在 DeFi 中的退化程度。
