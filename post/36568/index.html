写在前面

而同样于今年3月14日，发生了一件对于NFT玩家来说震撼度绝不亚于“微软收购暴雪”的事件：无聊猿（Bored Ape Yacht Club）的母公司“Yuga Labs”收购了由 Larva Labs 创建的大名鼎鼎的 CryptoPunks和 Meebits系列NFT，同时Yuga Labs还从 Larva Labs 购买了 400 个 CryptoPunks 和 1,700 个 Meebits，但其知识产权和资产购买涉及的金额均未公开。

如果你看不懂这段话，那就不太妙了。

NFT是什么？

NFT ，即非同质化代币 ，其概念可对比BTC、ETH这些同质化代币以进行区分。同质化代币是每个都一样，我的一个BTC和你的一个BTC都是相同的价值，也不能区分。而每一个NFT的价值却都是不相同的，其价值是通过它们的外在样式或者是哈希值 来体现。

NFT的表现形式可以是一张图片、一段音乐、一段视频，也可以是一个游戏角色、一个游戏道具等等，其最小单位就是1个。

NFT存在于链上，它在你的钱包里，就算项目方没有提供交易市场，你也可以通过钱包或者NFT交易所进行交易和转移。

那NFT还有哪些类型？

NFT有多种类型，并且一个NFT也不一定只是一种类型，就像一个人的职业，他可以同时是程序员、作家、摄影师，一个NFT也可以属于多种类型。

PFP

Profile Picture：个人资料图片**。** 刚才说到的Punks和BAYC都属于这个范畴，可以拿来做社交平台的头像。而这也是被调侃得最多的地方，比如我们时常听见有人调侃：“XXX花了XX万美金买了个头像”。其实持有NFT是人们想通过使用一个合集的NFT来寻找有相同共识的人，特别是在这个NFT价值不菲的时候，其作用就可等同于一件奢侈品。对了，这个圈子有个约定俗成的规则——就是只能使用自己拥有的NFT做头像。

Metaverse

刚才说到的SandBox便是属于这个范畴，它不仅仅是一张图片，它也是元宇宙的基建 。SandBox构建了底层框架，往上的应用层全靠购买地块的用户搭建。

那要怎么判断NFT的价值？

团队

和其他Crypto项目一样，团队是至关重要的一环。团队是否实名、是哪个国家的团队、NFT绘制者是谁、团队是否有优质项目的运营经验、是否在认真做事、人脉资源是否广阔等等。

项目信息

  • 白皮书： 官网以及白皮书的完整度，是否有抄袭痕迹。项目NFT总数，白名单数量&获得方式，mint价格，mint时间。

  • 路线图： 查看项目方有无路线图，路线图是否有新意，是否有落地实行的可能性，观察路线图的履约能力。

社区热度/共识

这里提到一个NFT的特点，NFT的个体价值差异大，流动性较差，价值由社区的热度和共识决定，地板价是NFT价值的体现。但一旦热度消失，共识破灭，NFT没人买盘就卖不掉，瞬间归零，毫无价值。所以查看社区热度、共识是评断NFT价值非常重要的一环，目前主要观察Twitter和Discord的热度。

Twitter： 粉丝数量、账号创建时间、推文数量、转评赞数量以及质量，关注列表有多少NFT圈的，有多少NFT KOL关注了该项目方。

Discord： 总人数&在线人数、运营能力（看团队是否熟悉web3的运营&玩法）等。

IP

IP是凝聚社区共识的重要一环。

  • 阿狸、冷兔是在web2中广为人知的卡通角色。

  • Adidas：家喻户晓的品牌。

  • Mfers：meme社区文化。

审美

不同于公链、预言机这些硬核的技术项目，NFT大多是以图片的形式表达，同时NFT还寄托着投资者的情感羁绊，所以审美也是NFT重要的一环。

  • 可爱画风：在满篇都是潮酷炫的猴狗猫的NFT市场，可爱的、简易的、人形的Doodle脱颖而出。

制作成本

NFT的制作成本从一定程度反映了团队的投入、认真程度，越精良的制作当然也越受投资者的青睐，往往也有更好的共识。

同一合集的NFT却为何价值不同？

NFT引入了**「稀有性」** 的维度，人总是追求个性，追求与他人不同的独特性。例如BAYC10,000个猿猴中，吹气球的有120个（1.2%），皇冠的元素只有2个（0.02%），物以稀为贵，一般来说数量越少的价值越高。当然，也会有审美上的偏差，喜欢就值得付出更多的溢价。

怎么找到可以投资的NFT？

NFT和其他项目的投资方式差异较大，没有机构的投融资，地板价*数量 的市值计算方式可信度也不太高。NFT的参与者基本上分为项目方、大户投资者和散户投资者，所以投资的方式也和常规项目有些差异。

刚才已经讲过怎么判断NFT的价值，就不再赘述，这里仅仅分享如何“发现”项目。说到发现项目，首先我们要了解一下NFT的发行方式。

NFT的发行方式

从NFT的发行渠道直接购买NFT通常等同于传统Crypto项目的一级投资，而NFT的发行方式主要有：

  • 白名单： 现阶段的NFT项目大多采用两轮发售模式，第一轮采用白名单的形式进行mint。获取白名单的方式通常采用以下方式：

  1. Twitter抽奖：由项目方或者合作KOL进行抽取，一般要求转赞评以及@好友。

  2. 完成社区任务：比较优质的项目通常有较多的投资者参与，为了凝聚社区共识，提高白名单门槛，项目方通常会要求投资者完成相应任务获取白名单。例如邀请好友进Discord，或者要求Discord达到一定的等级，亦或干脆让投资者绘制项目IP同人作品等。

  3. 持有其他项目NFT：这是NFT相互联动、赋能的一种方式。

  • 荷兰拍： 蓝筹项目Azuki就使用了荷兰拍的方式进行售卖，先以一个较高的价格进行公开发售，价格随着时间的推移而逐渐降低直至全部售罄。虽然如此，Azuki在初始价格Ξ1的时候就已全部售罄，2022年4月12日地板价达到22.9ETH。

  • Free+gas： 也有很多冷启动的项目，项目采用免费的方式进行售卖，投资者仅仅需要支付gas费用便可以领取NFT。（某些项目是前一部分名额免费）

发现NFT的方法

  • 数据整理网站

通过数据整理网站的日历功能，查看即发行的NFT。

  • 通过Twitter信息

由项目方或者合作KOL进行抽取，一般要求转赞评以及@好友。可以关注NFT KOL、互粉好友以及搜索Tag#whitelist去发现新项目。

  • 通过gas异常

‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍通过浏览器插件发现gas异常。

进入工具网站

https://compass.art/collections/minting 查询正在mint的项目。（项目燃烧的gas多，说明热度高）

通过详情页找到项目官网进行mint。

  • 聪明地址收集方法

    查看热点项目爆发前期的购入的地址，查看以往胜率，去伪存真，日常查看他们minted的项目。

Gas异常以及聪明地址的方式最适合新蓝筹的前两个仿盘 ，虽然没有过多的时间去分析，但新蓝筹往往是有热点效应的，前两个仿盘fomo情绪严重。

建立投资项目库

大部分的项目无法通过一级的方式获得，可以建立投资项目库，跟踪准蓝筹的项目进度，进而投资。

  1. 关注NFT燃烧排行榜 （链接）；NFT交易额涨幅排行榜  (链接) ；单位时间mint数量  (链接) ；Twitter、社群 出现该项目频率。通过以上数据将有一定热度的项目收集起来进一步分析。

  2. 通过NFT价值判断模型给项目打分。

  3. 关注项目方Twitter跟踪进展（尤其需要熟悉项目路线图）。

  4. 投资不错的标的。
