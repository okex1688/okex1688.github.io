大数据时代，云技术VS去中心化

在数据产业化领域，本地存储的方案已经逐渐被淘汰，云储存服务正在成为各行各业布局大数据的基础，据相关机构统计，2020年全球中心化云存储数据量规模达2376EB，预计未来5年全球中心化云存储数据量规模将维持28%的年均复合增长率；另据国际权威机构Statista的预测，到2035年，全球大数据存储量将达到2142ZB，2026年全球云存储市场规模将达到1697亿美元。

然而云数据服务大多采取中心化管理模式，必然会有海量的隐私数据存储在服务供应商的服务器中，黑客攻击、机房故障、违规操作，甚至是主动泄露等安全风险无法得到有效解决；同时云数据中心的建造和数据迁移成本并不低，导致云服务的价格较高；此外，大多数云数据的应用以企业为主，个人仅能作为数据提供者，且无法掌控数据的应用场景，隐私问题亟待解决。

在这种情况下，去中心化的云存储，就成为了业界探索的一个重要方向，Cere Network则是更近一步，升级打造了去中心化云数据平台。

开放的数据生态，巧妙的双赢模式

Cere Network 是搭建在 Polkadot 链上的去中心化数据云平台，旨在对数据的集成和协作进行优化。它是一个完全开放和可互操作的数据中心，其开放式数据注册（Open Data Registry）架构和中心化数据查看器(Decentralized Data Viewer)可以让用户自行定义数据共享程度，并明确知晓数据的流向和应用场景。Cere Network也会帮助企业部署合规开发工具，让企业在充分运用数据的同时，保护用户的权利和隐私。Cere Network巧妙的解决了传统区块链隐私的问题，实现了企业与用户的双赢局面。

Cere Network用定制的 Substrate 组件建立了自己的第一层区块链网络，其私有网络和Substrate 上企业资产的新智能合约标准得到了一个Web3 基金会的资助，并且Cere Network 正在参与 Substrate 建设者计划，在完成DDC 授予后，Cere Network能够直接被 Polkadot 生态内的任何其他项目和开发人员使用，全面提升网络的生态深度与广度，为实现消费级场景落地奠定坚实的基础。

面向消费场景，布局 NFT领域

目前的去中心化存储仍在起步和探索阶段，但其中也不乏落地应用的项目与场景，例如典型的IPFS分布式存储生态，Microsoft微软、万维网、谷歌浏览器、以太坊、Netflix、阿波罗计划档案馆等众多机构或网站均已加入其中，但Cere Network要做的，不是简单的数据存储，而是将数据协作，乃至商业智能化进度提升到一个更高的水平。

Cere Network的数据服务不仅能够让企业自己获得数据权限，通过开放数据市场 (ODM)功能，外部的供应商和合作伙伴也可实时访问目标公司的数据，公开的数据将经过筛选和匿名化操作，避免敏感数据泄露，极大的提升了商业协作间的效率和安全系数，这对于高频次、高数额、多元化的消费级场景应用来说是必不可少的。凭借这一优势，Cere Network已经与全球范围内，涉及旅游、媒体、银行和零售行业的2000多家企业建立合作关系，将区块链应用带到消费企业界。

此外，Cere Network也在积极布局NFT领域，将NFT于Cere Network的去中心化数据云相结合，为创作者、机构和品牌的内容NFTS化提供完美的解决方案。

相较于传统的云数据服务，Cere Network拥有去中心化模式带来的高效、安全、低成本等众多优势，打破了大数据时代下企业对数据应用的垄断，让C端用户也能作为受益者享受数据带来的红利，为数据市场合理合规的发展提供了重要的实例参考。针对消费场景的战略布局，让Cere Network跳出了传统数据服务的框架，Cere注定不会只是一个简单的数据服务平台，而是一个有深度、有广度的数据生态系统。
