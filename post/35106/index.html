本篇整理了将近 40 个实用的 NFT 工具，包括交易聚合器、热度追踪工具、专用稀有度查询工具、铸造日历、NFT 新闻聚合器以及即将启动的一个 NFT 抢先铸造工具等。

高效且实用的交易工具往往有助于先人一步发现市场机会。

对于 NFT 掘金者而言，监测热门 NFT 项目、及时获取铸造预告、实时了解铸造趋势以及便捷查看稀有度数据的工具缺一不可。本篇汇总中，Foresight News 整理了将近 40 个实用的 NFT 工具，包括交易聚合器、热度追踪工具、专用稀有度查询工具、铸造日历、NFT 新闻聚合器以及即将启动的一个 NFT 抢先铸造工具等。

NFT 数据分析和热度追踪工具

Nansen

Nansen 的 NFT 数据分析板块仅开放给会员，包括市场概括、Mint 排行榜以及 NFT 聪明钱交易动向、NFT 系列合约可视化分析、NFT 指数概览和稀有度排名等数据，还添加了清洗交易过滤选项，并针对一小部分 NFT 系列提供估值服务。

其中，Mint 排行榜标注了 FOMO 等级、铸造成交额、铸造地址数以及聪明地址铸造数等，并可根据这些分类进行排序。

Dune Analytics 仪表板

过去 4 小时到 1 年不等的时段内可按成交额、持有者等排序的 NFT 系列排行榜。

涵盖持有者数量、成交额、唯一持有者比例、地板价变化等的 NFT 项目仪表板。

交互式 NFT 地板价跟踪器。

NFTGo

NFTGo 是一个数据分析聚合平台，提供 NFT 市场数据的可视化、多维度排行榜、交易聚合（可跳转至 OpenSea、Rarible、LooksRare、Element、Mintverse 上交易）、NFT 发行（NFT Drops）日历服务。除成交情况、持有者等基础指标外，NFTGo 还支持查看过去一个时段新项目的铸造趋势，并标明了铸造人数、巨鲸铸造情况以及 FOMO 等级。

在巨鲸追踪页面中，NFTGo 还汇总了 1101 个巨鲸地址和相关交易、铸造活动，这些地址持有的 NFT 价值高达 39.4 亿美元，占总市值的超 23%。

NFTSCAN

NFTSCAN 是一个 NFT 浏览器，提供热门项目排行榜和铸造趋势排行榜。

CryptoSlam

CryptoSlam 提供 NFT 仪表板，提供 NFT 系列成交额和粉丝代币市值排名榜和具体基础指标。另外，CryptoSlam 还会针对以太坊、Solana、Flow 等多个区块链上的 NFT 项目铸造提供预告服务。

TheNFTscore

TheNFTscore 在以太坊、Solana 等多个区块链上提供稀有度查询、发行预告和热门 NFT 系列排行榜。

icy.tools

目前，icy.tools 免费的数据仅包括过去 1 小时和 1 小时以上按成交额计算的 NFT 系列排行榜以及铸造趋势排行榜，1 小时以下数据需要付费。icy.tools 付费服务还包括创建 NFT 系列和钱包的关注列表，以及获取实时更新。

DeGenData

提供热门项目的进一步细化数据。

NiftyRiver

NiftyRiver 提供基本的 NFT 系列热度、铸造分析服务以及稀有度查询。值得一提的是，NiftyRiver 还对骗局项目进行了标注，如，截止撰稿时，以太坊上过去 1 小时按铸造数量来看，前三个 NFT 系列均为骗局，前 10 个中有 6 个为骗局。巨鲸监测开放给会员。

Flips

Flips 允许查看地板价、持有者和上架数据的可视化趋势图表，还提供 OpenSea 上的实时销售数据。

WGMI

WGMI 能够跟踪多个热门 NFT 项目的地板价、成交量变化。

Coniun

Coniun 提供 NFT 市场基本分析数据以及 NFT 项目预告，同时包含推特粉丝、Discord 成员数量等社交数据供用户参考。

Context

Context 是一个 NFT 玩家追踪工具，允许用户关注任何钱包地址，以及探索精选艺术家和收藏家的最新动向，比如实时跟踪 CryptoPunk 持有者的交易活动，包括铸造、购买、出售、上架、出价以及发送等活动。

Context 支持 Foundation、SuperRare、Zora、OpenSea、Rarible、CryptoPunks、ArtBlocks 七个交易市场平台。

Moby

Moby 主要提供实时 NFT 铸造、销售和巨鲸活动数据。

NFTNerds

NFTNerds 是一个二级市场 NFT 交易热度追踪工具，需要连接 MetaMask 钱包使用。NFTNerds 包含 Firehose 和 Trending 两个板块，其中，Firehose 能够实时显示 OpenSea 上发生的每一笔交易（包括上架信息，可直接购买），另外还同时按成交额和笔数提供过去 5 分钟最活跃的收藏品系列；Trending 板块允许用户根据交易笔数、成交额、平均价格、7 天地板价和成交额变化图表来对收藏品系列进行排名，统计时段短则 1 分钟，多则 7 天。

MintyScore

MintyScore 主要对即将要发行的、已经发行的 NFT 收藏品系列以及领域内有影响力的人物进行排名，从而帮助用户挖掘热门项目。排名参考因素推特粉丝量、Discord 成员数量以及用户评分等。

compass

compass 提供 NFT 系列热度和铸造趋势统计数据，还允许用户关联钱包后直接查看持有 NFT 的系列数据，以及将感兴趣的 NFT 系列添加至关注列表。监控地址等部分高级功能需成为会员才能解锁。

NFT 稀有度专用查询工具  

入门

rarity.tools

需自行搜索 NFT 收藏品系列再按稀有度排名或者直接输入 ID 查看，显示排名、分数。

Rarity Sniffer

Rarity Sniffer 也是一个稀有度查询工具，更新较快，还提供正在或即将开放铸造的 NFT 项目日历。

RarityRanks

提供网站和插件，由 Doge Pound Studios NFT 项目背后的团队创建，浏览 OpenSea 和 Immutable X 上收藏品时同时显示稀有度排名和分数。

进阶

SuperSea

提供网站和插件，可方便查看 OpenSea 上所有 NFT 的地板价，以及提供购买需求匹配通知，还可跳过 OpenSea 缓存直接从源中获取图像，不过稀有度排名和绕过 OpenSea 确认步骤直接购买功能仅开放给订阅者和会员。

高阶

Trait Sniper

Trait Sniper 提供网站和插件。插件允许用户浏览 OpenSea 上 NFT 时显示每个 NFT 的稀有度排名、地板价，并标注了普通、Top 10 等标签，跳转到 Trait Sniper 网站后能看到各个特点的地板价。

Trait Sniper 网站还显示最新上架和实时销售数据。更快速地显示排名和特征、在 OpenSea 上的购物车功能以及同时竞价多个 NFT 等功能仅供订阅者使用。当前，LooksRare 上已内置 Trait Sniper 稀有度数据。

日历预告工具

除此前提到的 Gem、NFTGo、Rarity Sniffer、TheNFTscore、MintyScore、Coniun 同时提供 NFT 铸造日历外，其他 NFT 铸造预告平台还包括 ForesightNews Calendar、NFT Calendar、NFT Drops Calendar、UpcomingNFTlist等，不过这些项目中一部分会为项目方提供推广服务，因此用户还须自行判别风险。

NFT 抢先铸造工具

Yobot（即将启动）

Yobot 是由 Nascent 团队孵化的平台，目前处于试验阶段，允许用户通过支付服务费使用 flashbots 将其交易放到以太坊内存池中，以抢先铸造热门 NFT。

NFT 交易聚合器

Gem

Gem 聚合了 NFTX、NFT20、Rarible、OpenSea、Larva Labs 的 CryptoPunks 市场中的 NFT，之后也会聚合 LooksRare，此外，还支持用户在交易 NFT 时使用持有的多种 ERC-20 代币进行支付，也可以组合不同代币支付。

此外，Gem 其他优势包括 Web3 购物车功能、节省 Gas 费、集成 Dune Analytics 数据和稀有度数据等。

Genie

Genie 主要支持 NFT 批量购买和出售功能，可让用户在单次交易中从多个市场购买多个 NFT（最多 60 个），并通过聚合交易降低交易成本。聚合的市场平台包括 OpenSea、NFTX、NFT20、Rarible以及 Larva Labs 的 CryptoPunks。

此外，用户还可使用 Gem 和 Genie 扫货监测工具 GemBot、Genie Sweeps Bot 来监测扫货交易。

点对点交易工具

NFT Trader

NFT Trader 创建了自定义资产交换智能合约，允许用户交换 ERC-20、ERC-721 和 ERC-1155 代币和以太坊代币的任意组合。

sudoswap

sudoswap 由开发者 0xmons创建，使用开源的 0x 协议来创建和结算 ERC-20、ERC-721 和 ERC-1155 代币的任意组合之间的交换。

Swap.kiwi  

Swap.kiwi 由 Ape Punk 的持有者 niftynaut 在 2021 年 6 月创建，仅支持 ERC-721 和 ETH 代币。  

NFT 投资管理工具  

Zapper  

Zapper NFT 资产管理页面涵盖地板价和持有该 NFT 的数量。  

NFTBank  

NFTBank 是一个NFT 投资管理平台，还发布了移动应用，允许用户跟踪其 NFT 投资组合和获取 NFT 对应特征下的地板价，还提供价格估算服务。  

Evaluate Market  

Evaluate Market 是一个 NFT 探索和资产组合管理平台，并且为多数 NFT 系列添加了稀有度排名功能。
