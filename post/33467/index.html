2021 年绝对是 “NFT元年”，从快速兴起到席卷全球，国内外众多机构、品牌都趋之如骛。日前，“NFT”的搜索量已正式超过“crypto”。从无人问津到路人皆知，甚至当选《柯林斯词典》2021年度词汇，据统计，在2021年一年的时间里，NFT 一词的使用率增长了 110000%。

在 2021年，NFT 也经历了一件件出圈事件，也经历了一次次变革，国内外更有互联网大厂加速跑步入局。今天，笔者按照时间梳理，为大家盘点一下 2021 年国内外的十大NFT大事记。

Everydays: The First 5000 Days

2021 年 3 月，艺术家 Beeple 的加密艺术作品《Everydays: The First 5000 Days》登陆顶级拍卖会佳士得，并以 6900 万美元的天价成交，在当时这件作品也成为了在世艺术家的作品中排名第三高的艺术品。

这件事在艺术圈等多个领域均引起了极大的轰动。在当时，主流媒体对 NFT 的报道还停留在 NBA Top Shot 的财富效应以及 NFT 是否是一个新型骗局，Beeple 和其作品的出圈让人们开始认识到 NFT 作为一种新型媒介对数字艺术来说有多么重要。

这件由 5000 幅 Beeple 每日创作的作品拼接而成的巨作也让无数 NFT 的关注者高呼「这是一场伟大的 NFT 革命」、「NFT 的时代已经来临」。

Bored Ape Yacht Club

5 月初，Bored Ape Yacht Club（BAYC）诞生。BAYC 绝对可以说是 2021 年表现最优秀的 NFT 系列，无论是价格涨幅还是知名度、社交资本等层面，BAYC 带领着一种 NFT 头像收藏品上升到了此前无法想象的高度。如今，Stephen Curry、Post Malone 等来自各界的知名人士纷纷买入 BAYC，这也让 BAYC 持续获得更高的热度，地板价也攀升至 51.9 ETH，约合 21 万美元。

其实，如果把这些头像单纯视为「收藏品」，保留版权是完全可以理解的，然而在 Web3 时代中，这些头像已经成为了藏家们的「数字身份」，而无法掌握在自己手里的「身份」与 Web3 的宗旨相违背，更不用说 Larva Labs 还直接将一些「数字身份」授权给影视公司当作电影角色。而 BAYC 爆火的一大原因便是藏家使用自己的猿猴形象制作了众多周边产品，这种自发地、大面积的宣发也为 BAYC 积累了更大的知名度。

截止发稿时，Bored Ape Yacht Club 二级市场总交易额达 8.98 亿美元，最高成交额约为 243.76 万美元。

支付宝

6月23日零点，支付宝在“蚂蚁链粉丝粒”小程序上限量发售了NFT皮肤。其中，与敦煌美术研究院联合推出的付款码NFT皮肤分为两款，“祥瑞鹿王”及“幸运飞天”；与热门国产动漫《刺客伍六七》联名推出的付款码NFT皮肤也分为“伍六七”和“伍六七玄武国”两款。

每款皮肤限量8000个，需支付10个支付宝积分和9.9元抢购，支付宝用户可以在支付宝首页搜索“蚂蚁链粉丝粒”进入官方小程序抢购。

作为全球知名的金融服务机构，支付宝目前拥有超过10亿的用户量。支付宝上线NFT的举动，是对NFT的一次非常好的推广，不仅让更多人知道NFT，也为加密资产世界NFT进军传统世界进行了一次很好的铺垫。

网易

6月26日下午13点33分，网易联合Nervina Labs 在淘宝发行他们的首个NFT作品。该作品通过网易文创旗下的《三三工作室》发行其首个IP向NFT作品——小羊驼三三纪念金币。该NFT全球限量333枚，售价133元。用户可通过注册基于 Nervos （CKB）的 NFT 平台「秘宝」，接收 NFT。

网易文创是一家以内容IP起家的品牌，先后孵化了「网易王三三」、《戏精宿舍》漫画、「小羊驼三三 Family」等 IP ，全平台粉丝超 2600 万。而「小羊驼三三 Family」也已经正式入驻 Nervos NFT 平台金色传说，后续还将上线更多 NFT 作品。

IDA（Integrated Digital Art）

7月30日，首批IDA（(Integrated Digital Art，一体化数字艺术品）上线当天单笔最大认购金额超一千万。首批IDA选择齐白石《贝叶工虫图》、潘天寿《荷花图》、傅抱石《待细把江山图画》等名家画作，均为荣宝斋珍藏精品，由荣宝斋采用其独有的复制工艺1:1制作，每幅画全球限量复制1万张。不仅完整的展现了原作的精美笔触、深浅层次，同时还出具防伪证书，为作品真实性提供了凭证，使其兼顾了高品质与收藏性 。

IDA解决方案指的是通过将艺术品实物与数字所有权凭证分离的方式，藏家可将艺术品委托专业机构保管，而通过所持有的数字艺术品所有权凭证实现权属证明，很好的解决了艺术品流动与保管、鉴真的问题。

目前，国内数字藏品市场还没有专门的实物NFT。而IDA是在NFT数字艺术品基础上的改良升级，通过采用“实物+数字化”的创新模式，既拥有实物又拥有基于区块链的数字艺术品所有权凭证，具有专业收藏、礼赠佳品、自由交易、安心托管等产品特点。

在第三阶段销售期间，2021年IDA首批10幅作品在全球最大的数字艺术品交易平台OpenSea上线，最终以近10万美元的总体价格成交，率先打通了科技助力中国文化全球化的创新型商业模式。

第四批次IDA海外公开销售渠道将于2022年3月18日正式启动，销售预定开放时间为2022年1月18日至2022年3月8日 ，IDA第四批次将发售4万幅数字艺术品，按每幅零售价8折发售。

腾讯“幻核”APP

8月1日，腾讯旗下PCG事业群推出国内首个NFT交易平台“幻核”App，首期限量发售300枚“有声《十三邀》数字艺术收藏品NFT”。

腾讯研究院曾发文指出，在联盟链上发行NFT，腾讯有深度思考，联盟链的支付方式应该是法币，以及联盟链上合作的交易平台也应是合规的。

目前，幻核除了《十三邀》数字艺术收藏品外，截至目前共发行10款共29067份数字收藏品，共计1556348元。

Loot (for Adventurers)

时间来到 9 月，一个绝对会被记录在 NFT 历史上的项目诞生了，它就是 Loot (for Adventurers)。Loot (for Adventurers) 的创始人是大名鼎鼎的 Dom Hofmann，他曾于 2012 年推出了短视频 App Vine 并于 2016 年被推特以 3000 万美元的价格收购，而在被收购的同年，抖音才刚刚诞生。

Loot 的爆火是因为它打破了现有的模式。发行模式采用的是通过合约免费铸造而非官网发售，且二级市场版税比例设定为 0；在社区构建方面，其他的 NFT 都是先有一个产品然后吸引人们加入社区，而 Loot 则是先吸引人们加入社区然后再由社区成员共同创作一个产品。

不过，由于 Loot 缺少捕获后续价值的方式，所以目前二级市场的表现没有达到人们的预期，不过 Loot 带来的范式转变让我们对于 NFT 与社区产生了全新的认知。截止发稿时，Loot 二级市场交易额约为 2.68 亿美元，最高成交额约为 170.34 万美元。

字节跳动

10月1日字节跳动旗下的TikTok宣布推出首个NFT系列——TikTok Top Moments，相关NTF由社区定义创作者设计，并由他们所创作的热门视频启发。

10月3日TikTok在《纽约时报》买下整版广告宣传介绍将引入 NFT 以奖励内容创作。TikTok高管接受《纽约时报》采访时表示，内容创作者社区是TikTok的核心，希望通过 NFT 为内容创作赋能，让TikTok始终能够以最前沿的方式去了解和鼓励平台内容创作。

与国内的阿里、腾讯的数字收藏品平台基于联盟链不同的是，TikTok Top Moments将分为唯一版和限量版NFT，唯一版将在以太坊上发行，限量版则由NFTL2协议ImmutableX支援，属于完全碳中和的NFT。足见其布局NFT的野心。

Merge

12 月，加密艺术家 Pak 的作品 Merge 吸引了人们的眼球。单看NFT本身，黑色背景加上一个小小的圆形，无数人质疑「这也算艺术？」，其实人们对于 Pak 一直有这样的「偏见」。

Pak此次发售的「小圆球」会随着每一次交易与藏家本来拥有的「小圆球」融合，融合得越多，「小圆球」就会变得越大，而在之后，这些「小圆球」会按照规则兑换成 Pak 发行的 ASH token。而这样的融合机制既能让 NFT 与藏家产生互动，又能起到通缩的作用。并且 Merge 还打破了原有的「先买入，再碎片化」的流程，变成了「先买碎片，再融合」的玩法。此外，Merge 与 ASH 的链接也让艺术品拥有了金融属性，让 token 拥有了艺术属性。

从小正方体组成的 CUBE 到最初是一张白纸的 LostPoets，人们总是认为这些「自己也能做得出来」。的确，只从单个 NFT 的外观来说，几乎没有制作门槛，但 Pak 其实并不能算是「视觉艺术家」，相比于视觉，Pak 作品的机制设计才更加吸引人，可以说是那些 NFT 加上整套玩法机制共同构成了「一件艺术品」，Pak 是一位「数字观念艺术家」。

Merge 发售的总交易额高达 9200 万美元，一举超过 Beeple 在佳士得拍卖取得的成绩，而 Pak 作品的总交易额也超过 Beeple，在所有加密艺术家中排名第一。

京东

12月17日，京东旗下NFT发行平台“灵稀”已上线京东APP小程序，首批以京东形象代表5款吉祥物“Joy”为代表的数字藏品，价格为9.9元人民币一枚，每一版数量为2000份，上线即售罄。随后发行了4个“四大才女牙刻图”以及5个“山海神奇动物”。

在购买规则方面，“灵稀”与鲸探（蚂蚁粉丝粒前称）、幻核、IDA一致，购买前需实名注册，一经购买，不得退换。购买后记录将会记录在京东旗下区块链网络之中，无法篡改。

根据灵稀平台的用户协议，购买用户可以使用数字藏品进行研究、展览、欣赏、收藏。但严禁利用数字藏品进行炒作、场外交易或以任何非法方式使用。“灵稀”方面称将随时根据法律法规及业务变更对服务进行调整的权利。

2021 年是 NFT 飞速发展的一年，Crypto 的发展速度本就比传统行业更快，再加上 NFT 拥有着比 Crypto 其他赛道更容易出圈的文化属性，让此前从未关注 NFT 甚至从未关注 Crypto 的人们蜂拥而至，这也使得 NFT 能在短短一年中经历数次变革。

至于 NFT 未来会变成什么样子，谁也无法下定论，但可以预见的是，现有的优质项目会持续不断地沿着自己开辟的道路前进，并吸引更多人的加入，同时也会有更多的新项目会去尝试探索 NFT 的边界，探索在今天我们还没有办法想象的事情。
