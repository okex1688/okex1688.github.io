本文主要分析了目前关于DAO生态的项目，它们遵循了Web3的去中心化和通证化性质，介绍了目前正在运行的平台、让每个DAO的参与者都能够有更清晰的认知，具体如下：

DAO生态全景图  

目前，有100+个DAO管理着超过100亿美元的资产。

DAO操作系统  

  • DAO的第一章是用于创建它们的操作系统。

  • 这些项目为社区提供了不同的模板、框架和工具，以汇集资源并开始他们的第一个DAO。

  • 它们通常提供智能合约和接口，以促进分散社区的链上操作。

  • DAO操作系统使任何人都可以在有限的技术技能下轻松启动DAO。

DaoStack

DAOstack 是一个用于集体智慧和人类关联新形式的操作系统。用于去中心化区块链治理，特别是可扩展和有弹性的治理协议，可以有效支持大量人群决策的处理。

Daohaus

DAOHaus 是一个用于上线和运行去中心化自治组织（DAO）的无代码平台 。平台上所有的 DAO 均使用 Moloch 的开源代码作为编程框架，用户可在平台上加入已经创建好的 DAO 组织，也可以自己创建新的 DAO 组织。平台可支持创建的 DAO 组织类型有公会（Guilds）、俱乐部（Clubs）、风险投资（Venture）、奖助（Grants）和产品（Products），其中公会适用于劳工和自由职业者合作，俱乐部适用于朋友间的协作，风险投资为链上风投基金，奖助为社区资助分配，产品则帮助完成项目。

Aragon

Aragon是以太坊区块链上的一个可以让任何人创建和管理任意组织 的dapp。 aragon network是一个由通证控制的数字管理组织**，专注于为经济增长创造最好的条件** 。本质上，aragon network是个可以让任意组织、企业家和投资人高效安全协作，并且没有技术漏洞和恶意参与方的生态系统。aragon network的链上组织将会使用aragon core进行搭建，它是由一个solidity语言的去中心化自治组织（dao）和网页端去中心化应用（dapp）组成。一开始，aragon core会专注于资本化的公司类组织，但它的模块化功能也足够适应其他种类的组织。

Colony

Colony 是一个社区驱动的 Avalanche 生态系统加速器 ，由 Colony 的 CLY 通证提供支持。简单来说，Colony：为 Avalanche 建成的项目提供早期资金为现有的 Avalanche Defi 协议提供流动性购买并质押 Avalanche (AVAX) 和即将推出的子网通证购买选定的 Avalanche 项目以形成指数。

Orca

Orca 是 Solana 上用户友好的 DEX。 在 Orca 上交换资产时，用户将享受快速的掉期，最低的交易费用以及便捷的功能，例如我们的“公平价格指示器”，这使您很容易知道您是否获得交易的市场价格。用户也喜欢我们的收藏品，这是您可以通过加入 Orca 生态系统获得的特殊通证。

CW

Seascape 是一个基于 Substrate 的游戏生态系统平台，专为 DeFi 和 NFT 经济设计。 着眼于可用性和质量，目标是创造一个环境，使用户、开发人员和利益相关者能够以激励方式参与、获取并分享经验。通过允许用户进行合作并从他们的游戏资产中获利，Seascape 创建了一个生态系统，用户可以在其中获得与开发者一样多的利润。

Grants DAO  

DAO的第一个实际用例是赠款。

社区捐赠资金，并使用DAO投票决定如何以治理提案的形式将资金分配给各种贡献者。

赠款DAO的治理最初是通过不可转让的股票进行的，这意味着参与在很大程度上是由社会资本而不是财务回报驱动的。

赠款DAO表明，利基社区在资本分配方面比正式机构更灵活。

Metacartel ventures

去中心化自治组织 MetaCartel 发布 MetaCartel Ventures ，简称 MCV，以盈利为目的且专注以太坊应用生态投资的 DAO 去中心化自治组织。

MetaCartel 组织将会包含三类的角色，分别是：魔法师，他们将积极参与管理行动，比如寻找新的投资机会、尽职调查或管理资产，任何非合格投资者只能以魔法师角色参与该 DAO；哥布林必须是合格投资者，他们将不会像魔法师一样参与管理，但是他们也可以像魔法师一样积极参与，只是他们可以选择藏在背后；召唤师需要管理整个项目的财务、合规等很多具体事务。

Mint fund

Mint 播客，独家介绍了当今的创建者如何使用社交型通证、DAO 和 NFT 等 Web3 原生组件，构建未来社区。

Compound Grants

Compound Grants 为使 Compound 变得更好的项目、想法和团队提供捐赠。此赠款计划，对于申请人而言尽可能快且容易。具体可以申请快速补助、社区赠款。

快速补助，在申请后的两周内即可提供，金额少于 $125k，最快的选择。社区赠款，金额超过 $125k，流程背后的想法很简单：在公共论坛上发布，以征询社区成员的想法和意见。

协议 DAO  

协议DAO将权力从核心团队转移到社区手中，为项目向市场发行可替代通证提供了一种新的方式。

虽然DAO的第一章以不可转让的股票为特色，但协议DAO是第一个发行具有二级市场价值的可转让ERC20通证的。

这些令牌通常用于管理协议，这意味着令牌持有者拥有提出，投票和实施对网络基础机制的更改的唯一权限。

项目通常会就如何分配通证进行投票，从而为流动性挖掘，收益农业，公平发布以及介于两者之间的一切打开了大门。

协议DAO为任何网络提供了一个框架，以发行（希望）由其社区拥有和运营的令牌。

Maker

MakerDAO 是以太坊上的智能合约系统，提供第一个去中心化稳定币 Dai (以太美元) 和衍生金融产品。Dai 由链上资产足额抵押担保发行，和美元保持 1:1 锚定。个人和企业可以通过兑换 Dai 或者抵押借 Dai，无中心化风险地获得对冲资产和流动性。Maker 平台在抵押贷款、保证金交易、国际转账、供应链金融等方面都已经有落地应用。

Compound

Compound 是首个从 Coinbase 获得风险投资的项目，也是 Coinbase 为促进区块链创新所做努力的一部分体现。

Compound 为放置在交易所和钱包的加密资产提供应有的利息。每种资产的利率会依据该资产的借款需求进行动态调整。Compound 希望吸引对冲基金、经验丰富的投资者和借贷者。实际上，Compound 能获得 Coinbase 的支持一点都不奇怪，因为该公司的技术与 Coinbase 正彻底改变传统金融的公司的理念保持一致。

Uniswap

U niswap是基于以太坊区块链构建的去中心化交协议 ，被称为去中心化的 Shapeshift 或者 Bancor，可提供ERC-20通证的非托管交易。它的第一个本(Uniswapv1)自2018年11月开始运行(在DevCon上发布)，而第二个版本(Uniswapv2)于2020年5月发布。

Aave

AAVE 是一个芬兰语的单词，意为「幽灵」，代表 Aave 致力于为去中心化金融创建透明、开放的基础架构而努力 。它曾经的主要产品是 ETHLend ，一个去中心化的以加密货币为抵押的借贷市场。

yEarn

yEarn.finance 前身是 iearn.finance，自动化的 DeFi 收益率聚合器，在推出治理通证 YFI 之前，它的管理资产总量约为 800 万美元，综合收益率约 10.5%。它的工作原理主要是自动储蓄稳定币，并通过 AAVE、Compound 和 dYdX 等多种协议来实现收益。

yEarn.finance 是重新推出的产品，带来一套全新的收益率工具，如 ytrade、yliquidate、yleverage、ypool 和智能合约信用委托贷款。

Sushi

SushiSwap 流动性封装项目，克隆自 Uniswap，最大的不同是其发行了 SUSHI 通证，并且交易手续费中 0.25% 直接分给活跃的流动性提供者。团队希望用 SUSHI 通证经济模型，优化 Uniswap。但 Uniswap 创始人 Hayden Adams 表示，Sushi 只是任何有能力的开发人员通过一天的努力创造出来的东西，试图利用炒作和 Uniswap 创造的价值来获利。

Synthetix

Synthetix 从前叫做 Havven，是一个去中心化的支付网络 ，被设计用于日常的数字货币消费。Synthetix 网络使用双币系统来降低价格波动。网络交易的手续费被用作抵押品在区块链上发行一种新的有资产背书的稳定通证，Nomin 通证。

Curve

Curve 旨在使用高级粘合曲线，创造更深层次的区块链上流动性。目前处于测试阶段。 它是以太坊上交易流动的资金池，旨在达成极高效的稳定币交易，并为流动资金提供者提供低风险的补充费用收入，且没有机会成本。

Curve 允许用户以专为稳定币设计的低延误、低费用算法，在 DAI 与 USDC 之间进行交易。在后台，将流动资金池提供给 Compound 协议，为流动资金提供者产生收入。

Index  coop

Index Cooperative 是 Set Protocol 发起的去中心化指数社区，也称为 Index Cooperative 或者 Index。在社区成员的共同贡献下，Index 有机会成为全球最大的加密货币资产管理平台。

Index Cooperative 是首个由社区发起的去中心化组织，专注于创建及管理加密货币指数。 为了实现这一愿景，他们希望 Index Coop 可以与全球的金融人才，加密专家和团队一起来创建、管理优质的加密货币指数，治理系统参数，同时将协议嵌入更多的分发渠道。

PieDAO

PieDAO 是技术团队 Dexlab 的新项目。此前，Dexlab 通过持续交付知名的 DeFi 产品、基础架构，已建立了一个由用户、参与贡献者组成的全球社区。

PieDAO 是一个社区管理的平台，提供流动性、基础架构和管理，允许任何人使用数字和合成资产创建自己的 PIE（类似于 ETF 的分配策略） 。每种策略均在全球范围内可用，没有最低投资额。

Tornado

Tornado.cash 可帮助你收回隐私，以隐藏发送地址的方式将 Ether 发送到任何地址。 通过零知识证明实现。你可以使用此应用将 ETH 存入非托管智能合约，然后轻松生成证明你已经执行了存款，但未透露原始地址。而后，应用会将此证明发送给运营商，运营商会将其提交给智能合约，然后智能合约将 ETH 发送给所需的收件人，并向运营商支付少量费用。

Gitcoin

开源软件开发激励平台 Gitcoin，其愿景是通过区块链支持各类开源项目，促进开源运动的发展。由 ConsenSys 支持。

如果你是某开源项目的发起者，或者你正在管理某 Github 开源项目的代码仓库，你可以在 Gitcoin 上发起 Issues 悬赏，这个 Issues 可能是修复某个 bug、开发某个功能或者解决某个安全问题，感兴趣的开发者就可以领取 Issues 悬赏，并在 Github 上进行代码协作。如果发起人接受了你提交的 Pull Request，悬赏绑定的 ETH 便会自动打到开发者的钱包地址里。

简单来讲，Gitcoin 把 Github 上的开源协作机制通证化，现在他们则可以在兴趣与志愿之外，获得一点经济上的回报和奖励。

Lido

Lido 是以太坊上流动性质押的解决方案。

KeeperDAO

KeeperDAO，分布式系统开发企业 Talo、数字资产理财平台 Amber Group 联合推出，聚焦 DeFi 的链上流动性，旨在通过经济激励策略鼓励通证持有者参与流动性资金池，以协调保证金交易、贷款和交易所等应用领域中的清算和再平衡。也就是说，KeeperDAO 允许户将资金集中到以太坊智能合约中，并通过链上套利和清算的机会来获得共同利益，这既让所有参与者以博弈方式获得被动收入，又确保去中心化金融应用的流动性和有序性。为了最大化对闲置资金的利用，资金池中所有资产也将持续给 Compound、dYdX 等提供借贷。此外，KeeperDAO 参与者不需要自己搭建基础设施和机器人，设计付费优先策略等等，这大大降低了准入门槛。 

KeeperDAO 协议，可以从经济上激励「管理者」策略的集中参与，管理涉及保证金交易、借贷、清算等。这使参与者能够以博弈论最优的方式获得被动收入，同时确保去中心化金融应用保持流动性和有序性。

Badger

Badger 是比特币现金的安全身份认证，作为持有 BCH 等数字资产的网页版钱包，并作为去中心化应用的桥梁。Badger DAO 致力于构建产品和基础架构，以将比特币引入 DeFi。

RaribleDAO

Rarible 是 NFT 发行与交易平台，创建和出售由区块链保护的数字藏品。

Olympus

OlympusDAO 协议，开源且可扩展的点对点电子货币系统。

Rari capital

Rari Capital，最聪明的自动交易顾问，可确保获得最高的收益，而不仅仅是贷款。轻松存入 DAI，其将重新平衡以最大化你的回报。他们所做的不只是借用基本协议。他们进行做市，提供保险等等。

Cream Finance

Cream Finance 是基于智能合约的开放式包容性金融系统，将推动新兴财务系统的边界，使其尽可能安全地包含尽可能多的用户和资产。其将首先在 Binance Smart Chain（BSC）上通过算法货币市场协议启动。

Audius

Audius 是一家基于区块链的多中心化音乐共享平台，由 Roneil Rumburg、Ranidu Lankage 创建，旨在去除音乐流服务的中间环节和额外费用，让艺术家直接拿到自己作品应有的收入份额。具体说来，该平台的用户可以花钱来购买 Audius 通证，也可以通过收看广告来赚取通证。当他们从多中心化音乐库中听歌时，区块链钱包就会自动用通证付钱，价格不到一美分。其中，大约有 85% 的收益，是艺术家自己所得。剩下部分由歌曲提供方以及软件开发人员所得。而与之对比，当下行业中其它一些流媒体音乐服务商，只能为艺术家提供 70% 的收入份额。

Universe

Universe 是由社区驱动的 NFT 平台，基于以太坊构建，旨在为艺术家和歌迷创建可持续生态 。用户可在 Universe 以社区化的方式铸造并通证化 NFT、拍卖 NFT、在 NFT 市场交易等。

Inverse finance

Inverse Finance 稳定币收益协议，旨在存入 Dai，赚取 ETH。由埃及区块链架构师 Nour Haridy 发布。存入的 Dai 将投资于 Yearn 等收益优化协议，收益将持续转换为 ETH 并分配给持有者，Dai 可随时提现。

Haridy 表示 YFI 团队将 Inverse Finance 金库列入白名单，这也意味着 Inverse Finance 的储户将成为 Yearn v2 第一批用户。

Aavegotchi INSTA

Aavegotchi 是针对加密数字藏品的 DeFi 抵押，收藏、竞价并结合 Aavegotchis，以获得终极版的游戏化 DeFi 体验！

OceanDAO

Ocean是用以交易基于以太坊通证的 0x Relayer 和流动资金池。

投资DAO  

随着协议DAO将新的通证带入世界，团体联合起来投资它们似乎是合乎逻辑的。

经过长期的非营利性DAO，投资俱乐部为会员打开了开关，专注于创造回报。

虽然这些DAO比Grant DAO有更多的法律限制，但它们表明，任何个人群体都可以聚集在一起，以较低的进入门槛投入更多的资本。

投资DAO允许成员在早期阶段汇集资本并投资于项目。

MetaCartel

以太坊社区创建了一个新的非银行投资工具范式。

Flamingo

Flamingo 是集成资产跨链、自动做市商 DEX、合成稳定币及自动做市永续合约交易 AMM Perpetual 等的全栈 DeFi 协议 。其作为 Neo 在 DeFi 领域的基础设施，不仅是补全 Neo DeFi 版图的重要组件，还将对 DeFi 目前面临的几个亟待解决的问题提供新颖的解决方案。

Flamingo 借助 Poly Network 将成为 DeFi 市场中首个支持跨链事务一致性的金融协议，释放被局限在单链DeFi市场规模和处理效率的潜力。基于超额抵押范式的借贷以及割裂的流动性资源使得 DeFi 资金使用效率和流动性的潜力尚未充分发挥，而 Flamingo 其将成为 DeFi 领域首个打通 AMM 流动性池和合成抵押资产池的金融协议，在 DeFi 的资金使用效率、资产规模上限和流动性实现全面优化。

Komorebi

Komorebi 相信加密世界将成为全球经济、文化、数字互动的支柱，希望通过投资该行业的杰出女性、非二元创始人，使这个未来成为现实。

Komorebi 是富有诗意的日语表达方式，意思是阳光透过树木的光影。其组织的目标是照亮和提升一群在加密领域被低估的、有远见的创业者。

其核心运营团队由 she256 非营利组织的成员组成，致力于创建区块链女性组织，增加区块链领域的多样性和打破进入门槛，专注于通过教育和社区建设增加行业的多样性。

BitDAO

BitDAO 旨在推动开放式金融发展的去中心化自治组织 。其创立的初衷是分配大量资金和人才资源以推动 DeFi 增长，尤其是在资金、研发和流动性方面。BitDAO 还会通过捐赠支持区块链技术的发展，并通过通证交换支持现有和新兴项目。

项目拥有持续性的定期投资，未来将成为 DAO 控制的最大资金池之一，这些资产可用于为合作伙伴提供流动性和推动新的合作，如去中心化交易平台 DEX、借贷和综合性项目。

Duck DAO

DuckDAO 社区不同于金融界以前见过的任何事物。其由正在寻找长期机会的投资者组成，积极活跃于 Twitter、4chan、Reddit 等博客和社交媒体上捕获投资机会。

服务 DAO  

随着如此多的通证进入世界，项目需要人才。服务 DAO 是人才分配器，使用链上凭据将资源从一个 DAO 汇集和分配到另一个 DAO。

服务DAO为个人创建分散的工作组，为开放的互联网工作 - 本质上是作为加密原生人才机构。

从法律到创意，从治理到营销，从开发到资金管理，Service DAO 为 Web3 创建渠道。 工作通常以ERC20通证作为奖励 - 提供对为网络创造的价值的所有权。

服务DAO探索工作的未来，以及加密原生世界中的就业情况。

DX DAO

DXdao 由 Gnosis 团队于 2019 年 3 月发起**，基于 DAO Stack 创建，旨在以一种去中心化的组织模式开发 DeFi 产品** 。该项目部署完成后 Gnosis 会撤出，以保证不在 DXdao 中保留任何类型的控制权或预挖资产，确保一切都是公平公正的。

PartyDAO

PartyDAO 是 NFT 集体竞标的产品 ，组建派对，邀请朋友，一起出价并玩得开心。

Metafactory

Fire eyes

Fire Eyes DAO 诞生于 Raid Guild 治理部门，是由 Meme 驱动的开发组织，集合了专注于通证经济学与治理的加密原著民，正参与构建去中心化治理 。他们信奉 Meme 模因驱动的开发，对 DeFi 和 DAO 生态具备深入了解，能够清有效地为所有权经济奠定基础。

DeepDAO

DeepDAO 专注于去中心化世界的分析 ，关注 DAOStack、Moloch、Aragon、Colony 等所有 DAO 平台，跟踪投票、会员资格、提案和财务数据。DeepDAO 旨在为去中心化的治理系统提供全面的信息分析内容，已涵盖 100 多个 DAO 的财务和治理数据，目前正在不断扩大 DAO 的覆盖范围以及量化和衡量 DAO 中治理流程动态指标。

NeptuneDAO

NeptuneDAO 是 The LAO 新启动的第二个 DAO，也是其建立全数字化硅谷的延续，旨在为区块链项目提供流动性 。最初将开放 70 名成员席位，每个成员可贡献 60 个以太坊获得 100,000 枚 Neptune 通证，上限为 900,000 枚 Neptune 通证或 540 个以太坊。

NeptuneDAO 将充分利用 The LAO 成员的专业知识和背景，确保 DeFi 协议获得所需的支持。其任何成员都可以提出流动性和投资策略，再由其余成员投票表决，战略部分将由前 ConsenSys 首席战略官 Sam Cassatt 及 Aligned Capital 执行。另外，OpenLaw 还将为 dapp 开发等提供策略支持。

Llama

Llama.community 可让你为加密社区资金设定预算，跟踪资金流入和流出，管理进展。它涉及社区对 Snapshot 提案进行投票；多重签名钱包的签名者执行社区投票的内容；以及在电子表格上更新预算以反映所花费的金额。Llama.community 旨在简化工作流程。

LexDAO

LexDAO 为促进法律合作的去中心化自治组织。其实施了一种信誉计划，正计划转移至 Aragon DAO 代理程序，以便在更多地址之间分配接纳功能。

Dorg

Doingud

DoinGud 是一个 NFT 市场和社交平台 ，致力于让艺术家、创作者、收藏家、策展人及其他社会组织相互支持，找到归属、维持、自治，从而全力专注于创造好的未来，并赋予其经济价值。

YGG

Yield Guild Games 是 NFT 去中心化自治组织，主要帮助投资者汇聚资产并投注 NFT 获利，最终与支持者共享收益 ，目前 YGG 主要专注于游戏 NFT 领域。就像房地产一样，游戏的虚拟经济活动将在虚拟土地上进行。由于虚拟土地稀缺，因此区块链游戏中对虚拟土地的需求将不断增加。

HoneyDAO

Myco

Myco 是一个团体共同创造和分配价值的平台，从 Outpost 演进而来。其核心使命是帮助人们增强彼此的同理心和相互理解的能力，并使自由表达最大化，同时能够各自消费的内容。  

Uniwhales

UniWhales 旨在 Uniswap 上分析市场动向，追踪 Uniswap 上的巨鲸，呈现实时数据。

YamDAO   

Yam Finance 是一个试验性协议 ，由 Brock Elmore、Trent Elmore、Clinton Bembry、Dan Elitzer、Will Price 设计和推出，融合了可编程货币和治理方面的创新，具有弹性供应、完全由链上治理驱动、分配机制公平、不分配给团队等特点。最初设定的目标是每枚 YAM 为 1 美元。Yam Finance 在现有的弹性供应模型的基础上增加了一个关键的补充机制，即每次供应扩展的部分都用于购买 yCRV （高收益美元稳定币）并将其添加到 Yam 储存库中由 Yam 社区治理。YAM 总供应量为 500 万枚（未来可能会通过重置发生变化），重置 / 变基周期（Rebase Period）为每 12 小时一次，即北京时间 4 点和 16 点。提交治理提议时 YAM 的阈值为总供应量的 1％，提案投票时间为 48 小时。 

AladdinDAO

AladdinDAO 旨在利用 DeFi 行业中最优秀的人才，作为去中心化资产管理者，利用竞争的机制，让普通用户也可以识别出最令人兴奋、收益最丰厚、最值得信赖的 DeFi 产品。 其创始成员由重量级加密投资机构和 DeFi 高手组成，其中包括 Polychain、Digital Currency Group、1kx、Multicoin、CMS、Nascent、Alameda、DeFi Alliance、Robert Leshner、Kain Warwick、Hart Lambur、Alex Pack、Ashwin Ramachandran 和 Sharlyn Wu，将致力于通过 AladdinDAO 为混乱的加密世界带来秩序，推动全行业做大做强。

社交DAO  

成为DAO的成员意味着什么？在一个由投机主导的行业中，Social DAO专注于社会资本而不是金融资本。社交DAO是群聊的自然演变，朋友成为同事。

社交媒体将每个人都变成了一家媒体公司，而社交 DAO 则将每一次群聊都变成了数字业务。

他们挑战成为社区一部分意味着什么，并提供了成为数字原生部落一部分的方法。

社交DAO表明，加密不仅仅是快速赚钱，互联网是结识志趣相投的人的最佳场所。

Radicle

Radicle 堆栈建立在 IPFS 上，无需集中式服务器即可运行 。配置主机非常简单， 项目包含 git 库，直接从终端访问。Radicle 具备可编程属性，支持你调整自己的代码协作服务，用 Radicle 语言编写的 P2P 程序，可据你的需要进行更改，或者创建新的。

其治理通证 RAD 支持大量集成以太坊的功能支付、Radicle 网络公共所有权、集体治理和长期可持续性。RAD 通证持有用户与基于以太坊的协议交互时无需付费，也就是说，通证持有人可避免付费或享受一定付费折扣，并参与网络治理。

总结  

理论上，任何组织都可以是DAO。任何人都可以开始一个DAO，它没有自上而下的等级制度，具备包容和开放的等特性。

DAO的规则都是用智能合约和开源编码的 。该组织所做的是完全透明的。

DAO最终的目标是能够在分布式环境中高效地工作，这样组织中的每个人都能发出自己的声音，而不会影响组织的整体性能。 虽然DAO仍处于早期阶段，但在未来将得到不断的完善。
