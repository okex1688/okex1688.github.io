近日，Binance Labs 第 4 季全球孵化计划已正式启动，通过该计划，其将继续支持更多有潜力的区块链项目。在过去 3 个赛季的孵化计划中，已有众多优质的加密平台和协议成功入选并获得了 Binance Labs 的支持，如 Polygon、Perpetual Protocol、Injective Protocol 和 Dune Analytics 等。

而此次第 4 季，则有超过 500 个项目提出了申请，也反映了加密行业日益增长和多样化的趋势。最终，Binance Labs 选择了 14 个最具潜力及最适合该计划的项目，分别分布在了 Web3 基础设施/工具、GameFi / Metaverse、DeFi 等赛道。接下来，律动 BlockBeats 将对这些项目一一做出介绍。

Web3 基础设施 / 工具

FilSwan

FilSwan 提供了一个完整的去中心化存储和计算解决方案，使 Web3 开发人员能够轻松地跨多个链集成去中心化存储、边缘计算和支付。

在 Filswan 项目中，一个任务可以包含一个或者多个离线交易。其中，任务类型则包括公开任务和客户端授予的私有任务，同时公开任务又分为自动竞价和手动竞价两种形式，这是一种公开竞价的交易集合。

NuLink

NuLink 通过 API 构建了一个去中心化应用的隐私保护解决方案，使开发人员、初创公司都能够轻松构建自己的应用，并轻松实施最佳的安全和隐私。

目前，NuLink 正在与 NuCypher 合作，获得了 Polkadot Web3 基金会的资助，其核心库和 API 的第 1 版将于 2022 年第二季度发布。

Starton

Starton 是一种用于构建和扩展 Web3 应用的 API，这个 API 平台提供了区块链技术的简单性和可扩展性，可将任何应用连接到区块链上，并允许开发人员更快、更安全地在其产品中开发 Web3 功能。

Grindery

Grindery 是一个中间件 Web3 自动化工具，如用 DAO 支付工资单，供个人和组织连接 dApp 和协议以交换数据、支付和服务等。

此外，Grindery 构建的软件通过简化支付来帮助 DAO 提高生产力和透明度，并已在 Google Chrome 商店中发布了第一款产品作为 Chrome 扩展程序。2021 年 12 月，在 The DAO Global Hackathon 上，Grindery 还获得了金融与运营类的第二名。

Qwestive

Qwestive 是一个 Web3 CRM（社区关系管理）平台，通过代币持有者洞察和 Web3 原生社区管理工具帮助项目保留和扩大用户群。目前，Qwestive 正在向 100 个社区推出封闭 Beta 版计划。

Reveel

Reveel 是一个供 Web3 创作者跟踪其投资组合、自动化收益分享和报告链上收入的平台。

GameFi / Metaverse

Gamic Guild

Gamic Guild 是目前非洲最大的 GameFi 游戏公会，拥有约 13000 名活跃玩家和手机用户的社区，为用户提供多样化的 NFT 游戏，旨在通过玩游戏来改变众多非洲青年的生活。

最好的 NFT 游戏往往需要资金，这阻碍了玩家访问这些最有价值的游戏。Gamic Guild 希望将他们的娱乐时间转化为金钱，并通过为玩家提供游戏和赚钱所需的必要 NFT 来消除这些障碍。

CoralApp

CoralApp 是一个基于区块链的健身和社交平台，用户可在由区块链技术支持的 Metaverse 中亲自体验健康和健身。

CoralApp 将在多条链上进行构建，包括 BNB 链、Polygon、Avalanche (EVM)、Solana 和 Zilliqa 平台，测试版产品则于 2022 年第二季度末推出，正式项目启动计划于 2022 年第四季度初发布。

AlterVerse

AlterVerse 是 BNB 链上的游戏元宇宙。而游戏中，Sky City 则是 AlterVerse 的中心枢纽，玩家可通过购买土地、建造、购物、社交、收集资源、制作 NFT、购买、出售和交易，以及其他通过多种方式来获利。

从 Sky City，玩家还可前往众多的 OutPost 服务器之一。OutPost 位于巨型空间站上，这是一款第一人称射击风格的游戏，同时也包括任务、采矿和制作等。并且，玩家能拥有并定制您自己的船员室，通过驾驶强大的 XR75 战斗机战斗，以及社交和参加各种比赛。

Playbux

Playbux 旨在将现有的交易所、贷款平台和 NFT 集成到该平台中，以便任何人都可以在 Playbux 元宇宙中与其他人或 DeFi 平台进行交互，以及玩游戏、参加相应活动，和元宇宙中的朋友一起出去玩。

ChapterX

ChapterX 是一个身临其境的 3D 平台，只需使用加密钱包，每个用户都可以拥有唯一的 3D 头像和去中心化标识符，从而使用他们的元身份进行相互交互。同时，ChapterX 重新构想了 NFT 项目与用户互动的新方式，特别是 GameFi 和 PFP 项目，通过平台的基础设施工具包来创建自己的元经济。

此外，ChapterX 旨在将 Meta Identity 带入 Socialverse，使项目能够在三维景观中设计、建造并与社区互动。

DeFi

CODA Ventures

CODA Ventures 是 BNB 链上的收益生成协议，为用户提供了对一流可持续产量产生策略的被动访问。用户获得更高的收益通常需要持续监控、复杂的步骤或对衍生品的高度理解，CODA Ventures 则旨在消除这些高障碍并为用户提供轻松的访问。

Ambrosia

Ambrosia 是法国一个 DeFi 储蓄账户，允许任何用户将他们的法定储蓄投资到稳定币并以简单的方式赚取收益。其中年利率为 6%，每日分配，可随时取款。

OpenLeverage

OpenLeverage 是部署在以太坊和 BNB 链上一种无需许可的借贷和保证金交易协议，使交易者能够在 DEX 上做多/做空任何交易对。

OpenLeverage 的创始团队来自金融证券交易和 Crypto 相关背景，在衍生品交易和风险管理以及构建基于区块链的金融系统方面拥有多年经验，曾与高盛、汇丰银行等机构有着相应合作。
