上线仅三个月的 Uniswap V3 已经孕育出欣欣向荣的应用生态，在流动性管理、交易工具、流动性挖矿等多个领域繁衍出 43 个项目，其中大部分集中在流动性管理类。

Uniswap V3 版本 的上线可谓是一石激起千层浪，除了新版本本身带给用户的使用改变，这个以太坊上最大的 DeFi 项目在这次迭代中也催生出了许多新的项目以及给其他项目带来了新的机会。如果说一个开源的 DeFi 应用它的代码可能是很容易被复制的，但是应用的生态却是很难复制的，伴随 Uniswap V3 建立的生态圈无疑是在其经济堡垒形成的护城河。距离 Uniswap V3 版本上线已近三月，Uniswap V3 也已经部署到以太坊扩容网络 Arbitrum 和 Optimism ，是时候盘点一下哪些应用在该生态建设中注入力量了。

https://twitter.com/haydenzadams/status/1379180378578124808

Uniswap 创始人Hayden Adams 此前发布推特询问有什么项目在 Uniswap V3 建造，或者将在 Uniswap V3 上建造什么项目，根据各项目方的回复，据链闻统计，截止 2021 年 7 月 19 日，Uniswap V3 在流动性管理、交易工具、流动性挖矿等多个领域繁衍出的各个项目共计 43 个，其中最多的是流动性管理类的应用有 21 个，可见 Uniswap V3 自定义流动性范围的特定给了各个项目更多的发挥空间。

Uniswap V3 的更新包括聚合流动性的粒度控制 、范围订单 、多级费率 、高级预言机 等，新版本赋予了用户更多的自主权，用户可通过自定义流动性范围来实现集中流动性，提高资本效率。然而这也意味着对流动性提供者的要求更高，也就促使了新的依托于 Uniswap V3 的专业做市机构及主动做市策略出现。例如 Lixir、Charm Alpha Vault、Visor、Method Finance 等。

流动性管理

Visor Finance （https://www.visor.finance/）

Visor Finance 是一款主动流动性管理工具。提供自托管挖矿、奖励费用累积和时间锁定功能，包括 Visor Vault、Hypervisor、Supervisor 三个组件。Visor 机枪池可以锁定 LP NFT，防止项目方撤资跑路。Hypervisor 可根据预定交易范围，获得流动性奖励。Supervisor 可更新 Hypervisor 预设参数，管理资产，执行策略。

Visor Finance 宣布成立新组织 Gamma，将资助 Uniswap V3 上主动的流动性提供者相关策略的研究和实施。Visor 希望通过 Gamma，获得合适的数据科学和金融建模资源，以便实施最具吸引力的主动流动性策略，为 Visor 的金库所有者提供服务。Visor 表示将为 Gamma 提供最初的 50 万美元资金。

Method Finance （https://app.methodfi.co/）

与 Visor 接近，以 NFT 金库的形式让用户自托管 Uniswap LP NFT。

DeepGo （ht__tps://deepgo.io/）

DeepGo 是一个数字资产管理平台，开发了 CDO 模式与 Uniswap V3 结合使用，CDO 全称为 Collateralized DEX Offering，帮助用户通过质押标的风险分级，实现持续融资。做市商在 Uniswap V3 在提供初始流动性之后，将 LP Token 作为质押物锁定在 CDO 协议中，从而持续获得流动性买盘。

Lixir （https://app.lixir.finance/）

Lixir 是 Uniswap V3 的一个做市策略提供商，能够让做市资金在保持集中，且随着市场价格移动，确保市场价格附近有较多的流动性，同时可帮助减少无常损失。

https://twitter.com/lixirfinance/status/1384768152731635712

Charm Alpha Vault （https://alpha.charm.fi/）

Charm Alpha Vault 是由链上期权协议 Charm Finance 推出新协议 Alpha Vaults，可用来自动管理 Uniswap V3 的流动性，使其集中流动性获得更高的收益，并定期进行再平衡以减少无常损失。Alpha Vaults 的份额是 ERC-20 的形式，所以是可组合和可替换的。Alpha Vaults 将采用被动的再平衡策略，可以提高整体的效率，据他们的回测数据表明，被动的再平衡策略可以超过主动的再平衡策略。Charm Finance 表示 Alpha Vaults 即将在主网上线。

Liquid Network （https://liquid.net/）

Liquid Network 是基于侧链的交易者和交易所结算网络，可实现更快，更机密的比特币交易以及数字资产的发行。Liquid 表示目前正在建立代币化的头寸管理器，用于围绕可定制的价格点提供 V3 流动性，以及根据价格波动重新平衡头寸的策略。

YIN Finance  （https://yin.finance/）

YIN Finance 是一个主动流动性管理协议，用户可以通过订阅协议中的主动流动性管理策略 CHI，来实现有效的高收益流动性管理。在 Uniswap V3 中，用户需要手动收取费用进行再投资，而 YIN 会为用户自动进行再投资。并且 YIN 采用多方收益挖矿机制，在 Yin 中管理流动性，不仅可以赚取费用和 Yin，还支持接入 UniswapV3 挖矿。YIN Finance 会在不同的公链、DEX 中尝试为用户提供主动性流动管理服务，现在已在以太坊进行部署，计划将会部署于 Polygon 以及 Solana。

Gelato Network （https://www.gelato.network/）

Gelato Network 是一个以太坊智能合约自动执行工具，推出 Uniswap V3 自动再平衡实施方案 G-UNI，用来降低 Uniswap V3 的无常损失风险。G-UNI 的工作原理是通过其策略，始终以提供高于或低于当前交易价格 5% 至 10% 的流动性为目标。由 Gelato 机器人监测平均价格，每 30 分钟将决定是否重新平衡，只有当平均交易价格超出当前头寸的界限时，才会发生重新平衡，这时头寸会被撤回，并根据新的交易价格进行适当调整后重新存入。

Gelato Network 推出的第一款产品为 Sorbet Finance，该产品是基于 Uniswap 的订单薄型 DEX，支持用户设定目标价格并在 Uniswap 市场价达到目标价时执行购买或出售订单。

小结  

Uniswap 从落地起就是 DeFi 浪潮中的创新派，V3 版本升级同样还是给大家带来了很多新功能，就如同一场刚开始的实验，而这场实验成功与否的检验标准除了数据之外还能以用生态的繁荣程度来衡量。

Uniswap V3 不仅给了用户更多的选择和机会，也给了更多项目机会，在流动性管理 方面，用户可以选择配合各种流动性的工具一起使用创造更多的玩法，也可以自定义流动性打造自己的专属流动性池。

V3 的创新不止于此，还包括 LP 代币不再是标准的 ERC 20 代币而是 NFT ，让 DeFi 和 NFT 有了结合点，使得 LP 可以设定不同并且独特的作市条款。与之前相比，NFT 的采用让 V3 可以签署和实现更复杂的金融活动，也增加了去中心化金融的灵活性。Uniswap V3 也向其他 DeFi 产品提供预言机服务 ，并且比 V2 提供的服务更广。

Uniswap 设置了多级费用 的选项，与刚发布时每次需要手动选择费用相比，Uniswap 最近更新的功能为用户自动选择收费等级 ，该应用程序现在会自动默认为大多数 LP 策略建议合理的费用等级，但 LP 仍可以通过单击「编辑」按钮来手动选择费用等级，以显示额外的费用等级及其各自的流动性分配百分比。
