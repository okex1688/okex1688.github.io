随着NFT的世界得到越来越多的关注，人们对与Web 3.0工具、平台和项目进行交互的兴趣也越来越大。现在普遍的共识是，通过将消费者从现实世界环境带入Web3中，以加快这种新的虚拟世界的采用，这就是为什么POAP可能成为加密货币最早(和最广泛采用)的入口之一。

什么是POAP?

POAP是“出勤证明协议”。是用来证明一个用户在在线或离线活动的存在。因为一个人不可能存在于两个不同的身体中，所以用NFT代币的性质来证明它们的唯一性。

出勤证明协议(Proof of Attendance Protocol)是一个在以太坊区块链上开发的去中心化应用程序(dApp)，使用POA网络作为核心基础设施。该协议使用权威证明(PoA)共识机制来确保交易由已验证的帐户(也称为验证者)验证。

本质上，POAP是一个NFT。

POAP主要能够解决两个难题：

  • 复杂性

对于大多数消费者来说，NFT的世界有很多让他们困惑的地方，而对于创造者来说，NFT也是相当复杂且困难的——就目前的情况而言。NFT要求我们必须有计算机科学的学位才能解决这些问题，或者必须至少对智能合约开发有一定的了解。

  • 成本

不仅是技术的复杂性让许多人打消了对NFT的念头，而且另一个进入障碍是创造真正有价值的NFT的成本。

当然，制作一些随机图像并将其生成到区块链的成本并不高，但是智能合约的后端开发和成功项目的必要部分的创建需要有适当的预算。POAP通过减少创建NFT的交易成本解决了这个问题。它在一定程度上消除了一个进入障碍。

正确使用POAP，可以让我们在点击几下鼠标后上传一张图片，并创建一个NFT作为徽章，这是我们参加活动时的收藏品。这些收藏品将在现实世界中发挥作用，而在此之前我们将不得不越过重重障碍去创造它们，而POAP将改变游戏规则。

作为创建者平台的POAP

因此，如上所述，POAP可能对消费者有价值，但是对创造者一方呢?从市场营销的角度来看，这些出勤证明协议有巨大的价值。

一旦有人将他们的钱包与项目连接起来，项目方就可以直接连接到用户的世界，用户同样的也可以直接连接到项目方的世界。一旦钱包连接上，项目方就可以在用户的钱包里空投有未来价值的物品。你可以在任何时候通过钱包有效地与项目方交流——当然，这需要以一种聪明的方式来完成，而不是仅仅空投图像。

POAP是怎么工作的呢？在参与者扫描每个事件的二维码后，该应用程序就会启动验证用户是否真实存在的操作。该软件在扫描QR码时，根据用户的位置数据来检测用户是否在活动现场。下面有两个例子。

让我们用一个真实世界的品牌例子来说明；我去耐克专卖店，为买一双新款耐克运动鞋做准备。我可以在活动中扫描二维码，获得发布活动的POAP，然后当品牌发布扩展系列(卫衣、运动服、帽子等)时，他们可以空投给我一个NFT，这可以给我某种形式的实用价值——比如说可以是抢先购买其他系列的产品等等。对于品牌和项目来说，关键在于使用POAP作为特洛伊木马。空投的任何用途都需要得到真正的重视，而不仅仅是一种进行伪装的推销。

再举个例子，如果耐克能够通过POAP在我的钱包中空投给我一个NFT，那么他们便能够让我更早地接触到其他现实世界中的东西(这能够提高他们的销量)，甚至能够以数字形式在元宇宙中拥有这些东西。在这种情况下，他们设法卖给我另一件东西，我就会更喜欢他们，因为他们让我觉得自己是早期采用者，而且我现在将成为该品牌更积极的倡导者，因为我可以在任何地方将我的虚拟形象换成NFT，所以反过来我也是他们的数字广告!

在我看来，使用这种整合思维的品牌将会在Web 3.0中大获成功。

实际的POAP平台通过消除创建钱包的需要使实现这件事情变得更加容易——项目方可以在活动中提供一个二维码让人们去扫描。这些二维码将他们带到POAP生态系统，在那里他们可以存储这些数字徽章，直到他们决定将自己的数字钱包连接到POAP本身。如果它们能按计划工作，这就会让更多人进入加密货币领域，并尝试各种用途。

通过将实用程序链接在一起，我们还可以提供基于该POAP的代币门控访问。现在许多“开箱即用”的控制系统都在识别POAP，所以这可以增加消费者访问我们有价值的内容、平台、信息和社区。二级市场也认可POAP，所以也可以在那里列出POAP。

变化的浪潮

POAP是一个聪明的发展，虽然它不会让我们成为百万富翁，但是NFT的发布却有这个可能，它激发了社区的某些激情，并为某些人解锁了“下一个层的东西”，这能让他们步入Web 3.0的世界。

可用性——就像POAP——进一步验证了我们将看到许多大变革浪潮将来自 Web 2.0 和 Web 3.0 的混合体以最简单的形式使用区块链，从而让人们体验到他们正在进入的是什么。
