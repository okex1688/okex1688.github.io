本文是日本经济产业省（経済産業省）在今年5月19日发布的一个关于Web3（Web3.0）的政策报告。报告主要背景是，在目前经济局势比较复杂形势下，提出了一些经济和工业政策的新方向，以及日本官方对Web3的一些思考，考虑到（日本）地理和资源限制，需要在网络空间开发一个不受地理或资源限制的全新增长领域，因此会开始关注Web 3.0（Web3）；因为它不受地理或资源限制的约束，可能成为网络空间的一个新的增长领域。我们看这个报告，是作为一个国际动向来的关注，要本着客观辩证的原则去看这些现象。

（以及，由于笔者日语翻译水平有限，因此翻译的信达雅方面，还请广大读者多多担待）

主题

这个报告来源于日本的经济产业部，主要标题是：提出了在经济局势比较复杂形势下，一些经济和工业政策的新方向。

文件定位

文件的定位

  • 根据去年工业结构委员会大会的讨论，小组委员会在过去一年，一直在研讨新的经济和工业政策。

  • 最近动向/思考包括：

①国际局势对国际经济秩序产生了新影响，需要厘清日本究竟应该在新的国际局势扮演怎样的角色？

②新冠病毒的造成的供应链问题，需要严防所有商品和资源供应中断的风险，加强经济建设。

③（考虑到日本的）地理和资源限制，需要在网络空间开发一个不受地理或资源限制的全新增长领域。目前正在关注Web 3.0（Web3），因为它不受地理或资源限制的约束，可能成为网络空间的一个新的增长领域，这些变化正在迎来一个新的的时期，将有可能改变现有的经济秩序。

  • 为进一步发展和深化新的经济和产业政策，本文件发布的目的是根据最新的数据，提供一些新的观点。

主要框架

主要框架：

1.基本观点

2.国际经济秩序重组中的外交政策

3.建立一个以增长为导向的资源自主型经济

4.Web 3.0的潜力和政策

5.要讨论的问题

金融危机后的世界局势：

  • 战后，通过大国主导的贸易自由化，全球化和经济得到充分发展。

  • 但是，差距也在凸显，数字创新造成的财富分配不均，以及中心化思想蔓延，造成的分裂加深，现有的国际经济体系的优势并没有完全发挥出来。加上地缘政治的动荡，我们认为国际经济秩序正处于历史的十字路口。

日本官方对在Web3.0(Web3)领域的观点

什么是Web 3.0（Web3）

  • 随着数字技术和科技的发展，出现了将互联网社会分为三个阶段的思路：

Web 1.0：互联网的早期阶段，除了信件和电话等常规通信手段外，电子邮件可以作为一种通信手段。然而，这是一种单向沟通的互联网（只读）；

Web 2.0: 社交网络服务（Twitter、Facebook等）诞生，使双向（可读、可写）交流成为可能；另一方面，在Web2.0时代，个人数据集中在巨型科技平台的手中；

Web 3.0（Web3）：一个人人都可以拥有并管理自己数据的新世界，具备可验证、数据唯一性、真实性的特点，通过区块链防止篡改，具备鲁棒性（很结实，不容易被破坏），并且个人用户可以在去中心化的情况下自由连接、互动和交易。

  • Web 3.0有望成为新经济活动的前沿阵地，但由于它是去中心化的，提出了监管机构将如何应对风险的问题：例如，会对金融系统产生哪些影响？如何应对洗钱和欺诈性质的问题？

文件提出了，通过Web 3.0，赋能经济和社会的思路：

  • Web3代表了超越国界的经济活动，在网络上迅速发展壮大，定位服务范围在全球市场的初创公司，会相继在Web3领域相继诞生，并不断发展壮大。这会改变现有平台商的垄断结构，通过Web3, 所有人都能为价值创造做出贡献，获得回报 。

  • DAO（去中心化自治组织），其中包括各个利益相关者，这种利益相关者，可以有动力让组织模式和项目取得成功。这样的模式，会让价值创造成为一个整体，因此，大家会更加积极，这种模式有可能使整个国家变得更富足。

  • 日本作为文化创意产业优势较为明显的国家， NFT可以使创作者获益（之前利润集中在平台商），同时，丰富的文化资本（包括漫画和游戏，可以取得突破性进展）(1) 转向可以在全球范围内盈利的 “价值创造型经济” (2) 工业活动可以带来系列优化度和弹性

  • 更便宜、更快、更稳定的服务：基于区块链自动化程序的点对点交易，可以使省略不必要的中间商。

  • 通过供应链的透明度同时实现效率和复原力 的平衡：促进跨公司和行业内数据联通，对供应链的实时了解，既可以提高效率，又能赋能环保等领域。

  • 确保数据真实性：通过降低对中央服务器的依赖，确保数据的真实性，实现韧性经济——例如，防止服务器突然中断和数据造假。元宇宙的发展，可能让人们在全国任何地方舒适地生活、工作和学习，人口向本地的分散和跨境工作和学习的扩大也将增加，实现循环经济，作为一个不受本国资源约束的新增长领域 。

  • NFT机制允许生产者在转售市场上赚取利润，该机制正在推广，创造了一个具有循环性的新市场，减少了对资源的外部依赖。⇒注意：这种情况将取决于技术创新、商业模式、监管和产业促进方面的未来趋势。

各主要国家地区关于Web 3.0方面的布局/趋势

一方面，美国、欧洲官方正在开始在Web3领域发力，希望自己的国家在Web 3.0中更具竞争力，甚至成为全球中心；部分国家，如新加坡和阿联酋，正在率先通过优惠税收和其他支持措施来吸引公司。

另一方面，考虑到金融系统的风险和其他因素，各国正在寻求监管和创新之间的平衡。

美国

2022年3月，拜登签署了一项总统令，以确保数字资产的审慎发展，这为相关机构提供了一个美国强竞争力的发展框架，探讨了如何发行中央银行数字货币，以及数字资产对消费者、投资者和企业产生的影响，以及区块链技术和能源转型之间的关系，还考虑到了和金融稳定相关的风险等。

美国各个州的制度都有不同：例如，怀俄明州已通过一项法案，正式允许DAO（分散的自治组织）成为法律实体。

税务方面：美国个人加密资产的收入被视为资本收益，如果持有一年以上，最高可征收20%的税（如果持有不到一年，则正常累进税）

英国

2022年4月，英国财政部宣布了一个倡议框架，以促进加密资产部门的发展（例如，开始构建针对稳定币的法规，为英国创造一个可以让稳定币作为创新支付发展的环境，使它作为一种新的支付方式而成长。以及，英国皇家铸币厂使用区块链技术创建了一个NFT）目前应该是欧盟加密资产技术社区的成员， 英国财政部长也提到，想要让英国成为全球加密资产技术的中心，保证英国的创新地位。

新加坡

不少与加密资产相关的企业已经进入新加坡，以寻求有利的商业环境（新加坡税务政策相对友好）不过，新加坡金管局在对于加密行业风险管理方面也有一些举措，例如，新加坡出台了，不建议普通公众（散户）进行加密资产交易的指导方针。

阿联酋（UAE）

阿联酋目前设立了一个免征所得税和公司税并允许成立100%外资公司的特区，特别为加密资产相关企业设立孵化中心，这个成立于2021年5月的孵化中心，正在积极吸引外资公司， 目前有超过400家与加密资产相关的企业在阿联酋运营。

围绕Web 3.0（Web3) 的政策讨论

国内系统，如税收、会计、法律制度、知识产权、消费者保护和标准，需要跟上最新的业务发展。部分如知识产权、消费者保护、标准等方面的法律体系没有跟上新的商业现状， 部分企业会去到其他国家地区寻求更好的商业环境，需要创造一个能吸引全球人才的先进环境。

平衡监管和创新，每天都有新兴技术和服务产生，需要以灵活的方式做出应对，注意监管和创新之间的平衡。

税务：以公司为主体发行的Token，会被被认为是未实现的利润，需要缴纳公司税。对于资金紧张的初创企业，Token可以用于筹资和决策过程。

个人加密资产的收入，须缴纳最高55%的一般所得税（股票的资本利得税约为20%）
