加密领域充满了内部斗争。

从一开始，竞争就是这个行业的特征，并非理想主义者设想的“携手合作”。市值排名最靠前的加密资产拥有着“区块链超能力”：流动性带来流动性，资本产生资本，网络效应产生网络效应。获胜的公链能够抓住以上所有元素，并轻松保持住第一的位置。

本文，让我们回望区块链的基础原则，理解何谓“区块链帝国模式”，以及投资者如何利用这一点来驾驭极其复杂的 L1 估值过程。最终，你会意识到，以太坊帝国即将到来。

L1 的获胜关键

加密领域有足够的空间来“做大蛋糕”，因为我们尚处于 Crypto 漫长增长阶段的前期。在行业爆发之初，空间大，容纳的公链多，看起来生态系统可以承载的 L1 似乎不受限制，但这只是假象。实际上，如果 L1 没有竞争至第一名的位置，就会受到其他 L1 的支配。

流动性带来流动性。资本产生资本。网络效应产生网络效应。获胜的公链将抓住所有流动性+资本+网络效应。

货币之战是加密行业的日常。“货币溢价”是加密货币中最稀缺的资源。这是每个区块链都想要，却并非都能拥有的。比特币占大部分，以太坊占有一些，其它的公链只分到一点汤。所谓僧多粥少，就是货币溢价（第四部分详述）的现状。

为“钞能力”而战

同样的竞争也发生在国家层面。当你的货币是全球储备货币时，你就拥有了世界上最强大的资产：印制任何所需货币的能力。 当全世界使用你的钱时，你就是头号人物。任何国家都可以印钞，但如果没有全球需求的支持，可能会很快陷入高通胀。

当你拥有全球第一货币时，全球对你资产的需求会大大减轻货币发行带来的负面影响。由于“石油-美元挂钩体系”，任何新的美元供应都会立即被全球贸易吸收。但浪费性支出和腐败也让人们对美元的信心减弱。基于去中心化区块链网络的 Token 正赶上好时代——跃升至全球储备货币的地位。

链的价值正循环和国家一样

拥有世界储备货币的国家也拥有世界上最强大的军事力量。军事力量通过控制全球贸易来确保货币价值（全球经济在使用它的钱）。

这种权力处于正反馈循环中：一旦成为世界第一国家，由于对世界储备货币的控制，维持军队的成本就会变得更低；主要储备货币的价值进一步补贴了军事成本，这反过来又巩固了货币的价值。

说回到 Crypto 世界：

国家的军队 = 区块链安全；

比特币的大军 = 矿工，PoW 在比特币经济周围竖起了一道电力墙，任何能量弱的人都无法穿透比特币的 PoW 力场；

以太坊的军队 = 持币者，PoS 围绕以太坊经济竖起了一道资本墙，任何资本少于要求的人都无法穿透以太坊的保护墙。

每条链都有安全费用支出。区块链的可持续性是通过优化为生产安全而发行的货币数量来实现的。BTC 和 ETH的价格是影响这些生态系统安全成本的重要因素。如果一条链的资产价值增加了 10 倍，那么安全预算也会相应增加 10 倍。10 倍后的价格也意味着，系统可以在相同的发行量下获得 10 倍的安全级别。

货币溢价与安全问题挂钩

随着 BTC 价格的上涨，其算力供应也会增加。随着 ETH 价格的上涨，人们投资以太坊的兴趣也会增加。

比特币的发行规则是很难再编程的，所以比特币价值的抬升会增加比特币的安全费用支出（至少在它的区块奖励补贴用完之前），这可能会导致安全费用方面的超支。

相比之下，以太坊在发行方面更加灵活。随着 ETH 价格的上涨，ETH 的区块奖励发放减少：

区块 0 - 区块 4369999：5 ETH；

区块 437 万 - 区块 728 万：3 ETH（2017 年，通过 EIP-649 调整）；

区块 728 万至今：2 ETH（2018 年，通过 EIP-1234 调整）。

2021 年，EIP-1559 开始通过烧毁回收多余的 ETH。2022 年年底前，以太坊升级为 PoS 后，发行将进一步减少 90%。

以太坊的安全理念是发行最少数量的 ETH 以达到所需的安全性。

再来类比国家：我们如何以最少的成本优化军队以实现最安全？答案是减少坦克数量，增加无人机数量。

ETH 发行量的减少使得 ETH 更加稀缺，增加了其在二级市场上的价值。二级市场上更高的价格增加了以太坊的安全性，创造了一个正反馈循环，以更少的发行实现更高的安全性。

这就是货币溢价。

L2 呢？也能建立起帝国模式吗？

以太坊的模块化设计结构使其能够无限扩展。以太坊不是试图在单条（主）链之上托管去中心化经济，而是成为其它链的结算层。这就像美国拥有最强大的军事力量，保障和促进各国之间的全球贸易，只要各国采用美元。以太坊现在已经拥有最高的安全性，ETH 在 L1 上的交易，可以保障和便于 L2 之间的交易。

美元的强势并非来自美国经济国内生产的部分，而是来自于各国为了参与全球贸易而对美元的外部需求。美国没有控制德国、法国、阿根廷等国的经济，但它仍然抓住了这些经济体的优势。为了与其它国家进行贸易，这些国家必须将自己的 GDP 转化为美元需求来进出口。

同样，以太坊也不控制 L2 区块链的经济。每个 L2 对自己的经济拥有完全的主权。但是当涉及到将 GDP 从一个 Rollup 输出到另一个 Rollup 时，L2 必须消耗 ETH 才能进行 L1 交易。将数以万计的交易整合到 L1 交易中，是 L2 上的经济活动与以太坊生态系统的其它部分实现互操作的方式。

以太坊模块化设计的美妙之处在于，你可以在其上添加（基本上）无限个 L2，使以太坊作为 L1 成为最基本的可扩展区块链设计，使其从一个国家发展为帝国。

在以太坊上开发新 L2 的成本接近于零。这相当于一个帝国，只要它有必要，就可以轻易展现在新领土或外国的经济影响力。每增加一个 L2 都会增加以太坊的净 GDP，以太坊可以根据市场需求自由增加 L2。 

所有通往以太坊的道路都在增加 ETH 价值。

以太坊的 L1 和 L2 就是美国的联邦与州

模块化以太坊的 L1/L2 结构模拟了美国的联邦/州结构。中心简单，边缘复杂。

美国联邦政府是 L1。它决定了所有状态（L2）都遵守的“法律”。这些法律是为了促进各州之间的互操作性。它为有效的州际贸易提供了信任。

理想状态下，联邦政府只需提供一个所需的最低限度的规则和条例。所有其它的法律和规则，都可以留给各个州（L2）自己决定。

同样的模式也存在于以太坊。

以太坊 L1 上的 EVM 作为法则协调了 L2 之间的经济资源。这个共同的标准帮助 L2 分享彼此增长的经济好处。

共享相同的 L1 协议让每个独立的链从对立变为统一战线。由于底层 L1 提供的互操作性，以太坊上某一 L2 Rollup 的成功对其它以太坊 L2 Rollup 具有积极的影响，并将成为“联合优势”。

L2/州权利模型的绝妙之处在于，每个 L2/州都可以决定什么对自己最有效。

现有，以太坊“联邦”的“州”包括：

Optimistic Rollup：如 Optimism 或 Arbitrum；

Zk-Rollup L2 解决方案：如 zkSync 或 Starknet；

像 Immutable X 的以太坊 L2 扩展性解决方案；

像 Coinbase 或富国银行这样的中心化账本；

像 Hyperledger 那样的联盟链……

在以太坊上构建 L2 是没有限制的，只要遵守 EVM 互操作性标准。“中心的简单”最大化了“边缘的表达”。让所有人都能建造他们想要的东西，激发创造力。

面对 L2，我们是有选择的

L1/L2 结构（联邦/州结构）是一种个人赋权机制。 

如果州政府向你收取的税款比他们提供的服务还多，解决方案可能会是：换个州生活。各州必须相互竞争，让选民满意，并保留他们的居住权。在这方面做得更好的州，将获得更多的经济资源和更多的人口。

在 L2 中也会出现同样的情况。

你常用的 L2 收费变高了吗？它是否在基础设施上投入了足够的资金，它是跟上了创新的步伐，还是落后了？如果 L2 不符合你的要求，你可以换一个。L2 将为用户和锁仓量而相互竞争，这种竞争对用户是有利的。

以太坊帝国

以太坊是全球协调模式的现代实例。随着时间的推移，以太坊基本结构就如美国——通过确保全球贸易以美元计价，来维持其地位。

以太坊帝国将以两种方式巩固自身：

新的 L2 被创建并添加到生态系统中，比如以太坊原生的 Rollup（Arbitrum、Optimism、zkSync）离开了拥挤的中央帝国，在主城堡外建立了新的子领地；

其他安全性低的 L1 加入成为以太坊的新 L2。

这些以太坊的新分支可以自我管理并产生自己的经济。但在每隔几个区块之后，它们就会把总体经济活动捆绑起来，与 L1 进行交易。作为对处理交易征收 ETH 税的交换，以太坊 L1 的安全权力被授予给 L2。

山寨 L1 们可以相比以太坊发行更多的币，这样可以支付更低的安全费用。但如果想要停止货币膨胀，可以随时选择接受以太坊的保护。

用数据说话，对比不同 L1

以太坊目前每天向其网络提供 4500 万美元的安全保障费用。

BTC 目前的通胀率仅为 1.7%，但每天却能提供 4000 万美元的安全保障费用。所以从安全效率的角度，比特币是以太坊的 2.7 倍。

Solana 的通胀率为 7%，而每天的安全支出只有 1100 万美元。以太坊的安全效率是 Solana 的 7.17 倍。

AVAX 的通胀率为 5.5%，而每日安全费用达到 570 万美元。以太坊的安全效率比 Avalanche 高 10.4 倍。

这还是在以太坊切换到 PoS 之前（Solana 和 Avalanche 已经是 PoS 了）。未来，ETH 的发行量下降 90%，同时加强了其安全机制。

掉入“货币溢价”陷阱的竞争链们

Avalanche 和 Solana 都以低费用 L1 来标榜自己，但这迫使它们发行更多代币来支付安全费用。Solana 和 Avalanche 的吞吐量也确实比以太坊高得多。它们产生更多的总区块空间来承载更多数据。如果你有更多的区块空间，你必须增加安全支出来保护额外的空间。

国土面积越大，就需要更多的军队把守。扩张一个庞大的王国就像扩大领土范围，而不是让人们离开王国去建造零星散散的小附属领地。

为了在性能上优于以太坊，山寨 L1 提高了 L1 吞吐量，降低 gas 费。对于这些 L1 的长期货币溢价而言，其后果可能是灾难性的。这种设计选择创造了高发行量，同时也限制了费用收取。这扼杀了货币溢价，算是扩容的一个陷阱。

相比之下，以太坊在其 L2 上的可扩展性产生了较高的 L1 费用，同时最小化了发行 ETH 的需求。

发行更少，收费更高。这就是产生货币溢价的方法。

由于发行已经最小化，来自 EIP-1559 的燃烧费用捕获了以太坊生态系统的经济能量，并将其注入 ETH 的价值（通过使其更稀缺）。ETH 价值上升，需要发行更少的 ETH 来支付安全性。这种发行减少增加了 ETH 的稀缺性，增加了其价值，并进一步减少了发行的需要。这是货币溢价的正反馈循环。

其他试图在 L1 上扩展的链具有负反馈循环。规模化的 L1 需要高发行量，无法获得有意义的费用收入。这迫使供应通货膨胀，减少其稀缺性，并给货币带来下行压力。它不能用费用补贴发行，因为这与这些竞争链的目的背道而驰。随着越来越多的发行量，货币的贬值（由于通货膨胀）引发了进一步的发行需求。

争夺 L1 第一名的战场总是倾向于谁能发行最少的货币以实现最大的安全性。 如果你制造安全感，你就可以拥有一切（用户和资金）。如果安全对你来说是昂贵的，那么你的选择就有限。

竞争链，早点投靠以太坊吧~

4 月 13 日，知情人士称，Arbitrum、Optimism、zkSync 和 Starknet 中或有一方将下个月发行代币。

Optimism 和 Arbitrum 作为已在运行的 L2，从经济上看，不需要发币。它们通过出售区块产生收入，然后每次进行 L1 交易时，它们都要向以太坊经济支付少量的“税”。

Avalanche、Solana、Terra 以及基本上所有“安全效率不处于主导地位”的区块链都有一个漏洞。它们放任通货膨胀越久，问题就越严重。未来，通过成为以太坊 L2，它们的代币发行漏洞可能会降至 0，并立即带来收益，甚至可以增加吞吐量。

竞争链社区的人将我描述为一个“ETH 最大主义者”。事实上，正是去中心化的最大化主义使这个区块链帝国模式得以存在。以太坊对 L1 去中心化和有限区块空间的承诺，鼓励了充满活力且丰富的 L2 生态系统。

最终，你会发现，条条大路通罗马，条条公链通以太坊。
