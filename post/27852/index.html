国内禁止炒币之后，看不懂的NFT又被炒火了。NFT到底是什么？为何突然成为“网红”？它究竟是炒作还是风口？

什么是NFT？

NFT全称叫“非同质化代币”，是一种数字资产，但它跟我们常说的比特币之类的数字货币又不太一样。

它具有区块链管理所有权的独特数字项目，包括：收藏品、游戏项目、数字艺术、活动门票、域名，甚至实物资产的所有权记录。具体形容就是，NFT像独立小店制作，所有定制品都是唯一的。

为什么NFT突然火了？

主要还是因为最近一系列的“天价交易”事件将NFT推进大众视野。

Twitter 创始人通过 NFT 拍卖了自己在 2006 年发布的第一条推文（就5个单词 ），最终成交价 290 万美元，折合人民币 1887 万 。

球星詹姆斯的一段扣篮视频 就拍出了20多万美元的价格，价值一百多万人民币 。

8月22日，孙宇晨在推特表示，花了50万美元 买了一块“JPG石头” 。

这些虚拟石头，是EtherRock于2017年12月推出的 NFT 项目，其共计有 100 个 NFT，每个 NFT 是一张石头图片。

一条推文卖数百万美元、一块石头卖百万元……你觉得花百万美元购买的jpg图片，是炒作还是区块链下一个风口？

另外，NFT火热的行情吸引了众多国际大牌参与其中。截至目前，NFT市场的市场规模已超230亿美元。

而在国内，阿里巴巴、腾讯、网易等大厂也都有所动作。

  • 支付宝与敦煌美术研究所联名推出的NFT付款码皮肤。

  • 网易旗下游戏《永劫无间》IP授权发行的《NARAKAHERO》系列NFT盲盒。

  • 腾讯发布国内首个NFT交易软件“幻核APP”。

NFT的创建门槛又很低！

在交易所OpenSea上，用户可以在该平台直接免费创建NFT，不需要支付任何费用，创作者可以销售获得收入。

这种制作NFT的低门槛，无疑给一些投机者、骗子又带来一次钻空子的机会！

别人的一块虚拟石头都能卖数百万元，既然这么有钱途，那我为什么不也整几块破石头去卖呢？这不是，很多仿盘就出现了：仿盘石头、纸、剪刀……

就是这种炒作热情成就了OpenSea，让其成为世界第一也是最大的NFT市场。同时，也让各种仿盘和诈骗项目横行 。

最近观察一些微信群，以NFT名义建群的数量也在增多。还有很多打着各种各样的“空投”去引人入局！

现在的币圈，真不是一个普通的投资者HOLD住的，又是元宇宙、又是NFT，有几个是真正懂呢？真是替那些进群的人们担心。

NFT 走红“出圈”的同时，也引来大量为暴富而来的投机者，使其充满了投机和泡沫 。

当前大部分国家并未对包括NFT在内的虚拟资产给予明确法律地位，未来监管 仍然存在诸多不确定性 ，一旦出现全球性监管收紧，NFT无疑将遭受重创。

大家，都擦亮眼睛吧！
