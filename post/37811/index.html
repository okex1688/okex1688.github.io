NFT 当前最主要的价值在于，授权艺术家、创作者和收藏家行使他们分发、转售和收藏的权利。

NFT 艺术品《第一顿晚餐》以 103.4 ETH 的价格成功拍卖

NFT 是数字资产吗？是的。因为资产的定义是“被认为具有价值的东西”，所以如果人们愿意购买 NFT，它就是一种数字资产。就像艺术收藏家决定购买齐白石的画作——或者花费 12w 美金，购买自学成才的艺术家 Maurizio Cattelan 用胶带粘在墙上的香蕉——可见购买某物的意愿并不需要基于任何客观现实。

当一位艺术收藏家买了一根用胶带粘在墙上的腐烂香蕉时，他们知道这是一根用胶带粘在墙上的香蕉。那么，我们购买 NFT 是不是也相当于购买被区块链 “粘” 在链上的图片？

NFT 的价值

了解 NFT 重要的是它们如何变得有价值。与 BTC 或 ETH 等 Token 不同，NFT 的价值通常并不是那么偏理性的，而是偏感性或者有权益映射的；如：数字图片文件、权益证明、Pass、甚至某社区入场门票等。因此，NFT 的所有者必须解决区块链上的所有权记录与他们应该拥有的东西（不在区块链上）之间的微妙关系。

考虑一下：你会为自己购买一个 NFT，一个只包含唯一数据串的区块链记录，而不涉及数字或现实世界的资产吗？那你一定没兴趣。如果告诉你它是独一无二的，且周杰伦曾经拥有它，或者其他许多人正在争先恐后的准备高价购买它，我们还会这么想吗？

但是在 NFT 代表数字艺术作品的情况下，NFT 通常包含指向 Internet 上公共媒体文件的链接——该文件可以被任何人访问和复制。那么就会有许多人质疑：购买 NFT 的意义是什么？

至少对于实体艺术来说，伪造是很难的。但在数据的世界中，制造一个复制品是十分容易的；因此 NFT 的价值得以体现——你是其真品的唯一所有人。只有你才有权利决定其归属。

听起来很模糊？但我们反过来想，即使你持有《蒙娜丽莎》，你也无法阻止别人挂一副蒙娜丽莎的仿品在自己家中；NFT 也同样，虽然所有人都可以访问和复制，但只有你才持有其唯一真品。

但当前 NFT 相关的法律还不够完善，等版权、二次创作之类的条款完善后，NFT 和艺术品的结合会有更大的价值体现。

虽然数字艺术品是最开始、也是当前 NFT 最为主要的应用，但事实是，加密行业只是触及了 NFT 技术能力的皮毛。

在下一阶段，NFT 最有可能在哪些领域大展身手呢？

医疗记录和身份证明

医疗记录和身份证明是重要的信息，即使在数字时代这类信息也难以替换。这是 NFT 技术可以提供广泛的实际用例的一个领域，社会中的大多数人都可以从中受益。

与其让大型医疗机构记录、管理和跟踪个人的医疗记录，分配给每个人的唯一 NFT 可以存储所有医疗信息，同时保持机密性和隐私性。

这也将允许人们在寻求治疗时控制与健康提供者共享哪些信息，同时将不太相关的个人数据保密。

有可能有一天，这一过程将从出生时开始，由医疗保健提供者或政府机构向新生儿签发 NFT 出生证明。这也可以是建立可以以 NFT 形式跟踪的数字身份的开始。

借助数字形式的身份识别，个人将能够在需要识别信息时限制共享哪些信息，例如一个应用程序可以验证一个人是否超过 18 岁，而不会泄露通常包含在 ID 中的其他信息，如家庭住址等。

房地产和资产代币化

房地产和其他现实世界资产的代币化可能是 NFT 技术最广泛和范式转换的应用之一，因为它对众多行业产生了影响。

除了作为跟踪和验证项目所有权的简单用例之外，资产的标记化最终将允许它们在去中心化金融 (DeFi) 中的许多不同应用程序中使用。

最终，个人将能够使用代表其房屋契约的 NFT 作为 DeFi 协议的抵押品，这将允许他们以与再融资类似的方式借入资金用于其他地方。

较大物品的代币化，例如酒店或豪华游艇，也允许分散所有权，让那些通常买不起此类物品的人有机会接触它。

可以在区块链上标记和跟踪的内容没有限制，并且很有可能所有形式的资产，包括股票、债券、艺术品、稀有收藏品和奢侈品，有朝一日都将以 NFT 形式表示。

知识产权和专利

NFT 是跟踪知识产权 (IP) 和专利的理想选择，它们提供的一种证明任何内容所有权的方法，可以将当前商标和版权系统提升到一个新的水平。

区块链技术的数据保存功能允许跟踪 IP 的整个历史并加上时间戳，提供了一种提供不可否认所有权的方法。同样，专利或发明的数据可以以 NFT 形式记录在区块链上，提供一种保护和证明所有权的方法。

票务

已经在世界各地的娱乐场所探索和实施 NFT 的一个用例是门票。创建无限数量的独特 NFT 的能力允许演唱会和体育场馆等场所以 NFT 形式发行门票，以便轻松验证或转让。

智能手机在社会中的普及使数字票务成为可能，而 NFT 技术的整合将有助于使这一过程更加高效和易于追踪。

独家会员资格

NFT 技术的最终应用是作为一种通用实用代币，它执行特定功能，例如验证专属俱乐部的会员资格或提供对特定服务的访问权限，即我们通常所称的 Pass。

许多 NFT 项目已经在使用这个功能，他们在专门的网站或 Discord 组，只有在验证该特定集合中 NFT 的所有权后才能访问该功能。

结语

区块链的出现让我们看到了更多价值产生和传递的可能性。我们见证了从无到有的数字资产化过程，同时也在探索资产数字化从零到无限的价值前景。其背后是技术进步催生的思想演变，这让我们不再依赖于当前稳定且低效的体系。从“NFT+”的思想定位出发，我们正在仰望更广阔的星空。

这种资产和价值的转变不会止步于信息互联向价值互联的演进，而是在区块链的支持下，推动人类从中心化的信息体系向去中心化的价值体系整体上链，以及全新的链上治理体系。

或许在互联网诞生数十年之后，我们就能突破链上链下的壁垒，实现大范围资产、信息的上链。链上的价值将占人类生活的大部分。世界将拥抱区块链，人类将进入一个新的生活方式。
