NFT估值需要考虑哪些因素？

不可替代代币（NFT）市场正在飙升，越来越多的人正在创造、购买、出售和交换NFT。这个新市场存在许多挑战，最大的挑战之一是NFT的估值。

NFT的估值是一个很大的未知数。没有关于 NFT 应该如何估值的案例，其最接近的类比，即对艺术品的估值，也是相当模糊的。NFT需要一种新的估值方法，以及一种更好的管理与NFT相关的数字资产的手段，以保持该价值。

公平的市场估值

公平市场价值是指买卖双方在没有任何强制购买或出售的情况，合理了解相关事实的情况下，将NFT转手的价格。这一价值可能包括与NFT相关的数字图像，基于NFT的智能合约，以及基于卖方和买方之间的合同义务，对图像的现实世界主体的展示和复制的权利。

公平市场价值在很大程度上依赖于 “最相关市场 “的概念——即该物品最常向公众出售的市场。 虽然在某些情况下，拍卖可能被认为是 “最相关市场”，但每个NFT的具体性质以及它所在的市场（例如OpenSea），将被用来确定 “最相关市场”。

这就好比艺术品的估价方式。去年刚出版的印刷品不太可能进入拍卖市场。对于独一无二且尚未拍卖的NFT也是如此。因此，在这种情况下，“最相关市场 “是NFT的要价。

NFT 估值需要考虑的因素

链的安全性

底层区块链保持安全对买家来说很重要。以太坊是目前最安全的智能合约平台，随着时间的推移，可以促进NFT的价值。

链上或链下元数据

由于以太坊区块链的存储限制，链上定义为将元数据直接合并到代表代币的智能合约中，而链下则意味着单独托管元数据。

链上元数据使NFT更有价值，部分原因是元数据被合并到代币中，使NFT可以永远存在；而且，部分原因是链上代币必须满足某些以太坊标准，为它们提供流动性溢价并使交易更容易。 

在确定 NFT 是链上还是链下时，关键问题是 NFT 的托管位置，即使用数字资产管理系统（DAMS）。

NFT创建时间

由于 NFT 如此新颖，因此 2020 年之前创建的 NFT 可能被认为是具有更大价值的“数字工艺品”。

创造者和社区

名人艺术家的更广泛的市场化和认可度将影响他们的 NFT 价值，体育行业的某些NFT创作者已经意识到这一点，并与知名艺术家合作，制作出独一无二的NFT。这些 NFT 对购买者具有独特的吸引力，因为它们是独一无二的，而且与可购买的实物类型不同。

稀缺性和真实性

SuperRare等某些 NFT 平台只支持独特的、单一版本的数字艺术品。一些市场还通过稀缺性对其NFT产品进行细分，例如，NBA的Top Shot将其NFT分为 “普通”、“稀有”、“传奇 “和 “终极 “等级。那些被列入较高等级的产品拥有比 “普通 “等级的同类产品更高的价值。

真实性与稀缺性是相辅相成的——例如，佛罗伦萨的乌菲兹美术馆为他们拥有的一幅波提切利作品创建了NFT，这比从游客的iPhone图片中创建的NFT更有价值。然而，如果没有访问DAMS的话，核实互联网上卖家的真实性和管理与NFT相关的知识产权仍然很困难。

铸造数量

创建者总共铸造了多少 NFT，会影响 NFT 的价值。一个艺术家发行的无限量的NFT项目，通常不如从一个承诺铸造25个NFT的艺术家那里购买NFT更有吸引力。

丰富性

丰富性与 NFT 的附加功能有关。音频组件可以尽可能地增加价值：以知名艺术家为特色，或为观众创造一个令人上瘾的循环。将数字资产与现实世界资产或体验相结合的 NFT 也具有更大的价值。最近，金州勇士队拍卖了他们以前的冠军戒指NFT，并为买家提供了切实的好处，这可能是 VIP 篮球比赛体验或带有定制名称雕刻的戒指的仿物理版本。其中一个 NFT 售价超过 80万美元。 

在体验方面，新泽西魔鬼队NFT的购买者还获得了在新泽西魔鬼队校友套房观看2021-22赛季比赛的机会，与魔鬼队其中一个冠军年份的传奇人物一起观看。该NFT的售价为3000美元。然而，该网站上的一些纯数字 NFT 的售价仅为 100.28 美元。

销毁作品

一些创作者用来提高NFT价值的方法是销毁原始作品，要么是在NFT转让给买方时达成协议，要么是在销售发生之前。也许最著名的销毁事件是4月下旬计划拍卖的Basquiat NFT。拍卖赞助商DAYstrom宣称，此次拍卖将包括所有相关的永久知识产权和版权，并且出价最高的人可以根据自己的意愿选择销毁实物图纸。拍卖最终没有发生，因为 Basquiat 的庄园将 NFT 从 OpenSea 撤出，并澄清庄园将保留这幅画的许可和权利。

销毁可能会赋予NFT作为独家数字资产的地位。然而，一旦NFT在互联网上被铸造，销毁原件并不妨碍任何人查看、下载、分享和复制该图像，除非知识产权有可执行的限制。

Origin Protocol最近将 “Charlie Bit My Finger “视频作为NFT进行拍卖，该视频曾是YouTube上浏览量最大的视频。Origin表示，它计划将原始视频从YouTube上删除，以便将内容记录在区块链上。YouTube尚未删除该片段，但该视频现在的标题中加入了 “等待NFT决定”。

合同限制

与其销毁原始作品或让买方选择销毁它，创作者始终可以对物理对象进行 NFT 并保持对象完好无损。例如，艺术史学家本·刘易斯（ Ben Lewis）创作了莱昂纳多·达·芬奇（Leonardo da Vinci）的《救世主》（Salvator Mundi）的NFT，稍作修改，但没有破坏达芬奇的原作。这个NFT仍然在OpenSea上挂着，还没有被卖掉。同样，如上所述，金州勇士队成为第一个推出NFT系列的职业球队，其中包括以 NFT 形式数字复制的勇士队的六枚 NBA 总冠军戒指。最初的销售发生在5月初，并未涉及销毁原始冠军戒指。

创作者可以添加要约，以合同方式限制原件的展示。如果创作者公开展示或使用该作品，则买方可能会对创作者提出违约和版权侵权索赔。

总结

随着 NFT 市场的蓬勃发展，预计估值将变得更加混乱。新市场存在许多挑战，其中最大的挑战之一是 NFT 的估值。由于存在这种不确定性，规划者和客户需要努力了解 NFT 如何创造价值。
