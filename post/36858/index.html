原文标题：《The Four Pillars of the Merge To Proof of Stake: How Ethereum Will Evolve》

以太坊转向权益证明 (PoS) 的合并将是其迄今为止最重要的升级，重要性仅次于创世区块。它将向世界展示一个主流的去中心化的系统如何彻底消除碳足迹。合并将会平息以太坊关于工作量证明（POW）共识机制的争议，并将向社会提供一个更有力的信任基础：以太坊，一个效用持续增长的通缩型货币，ultrasound money。因此，以太坊迁移到 PoS 的重要性不容小觑。

自 2015 年成立以来，ConsenSys 在以太坊的持续发展中发挥了基础性作用。从「Eth2」规划的第一天起，ConsenSys 协议工程团队就一直在不懈地努力实现合并：

  ConsenSys 协议团队的关键贡献  

我们相信像以太坊这样的去中心化网络可以帮助社会取得更多成就， 为将 PoS 带到以太坊，我们付出了大量的人力、产品和资源。

随着合并的临近，我们希望通过说明以太坊将如何升级来重振生态系统的信心。这是合并的四大关键点。 

关键点一：多元开放

PoS 将使网络参与民主化，并通过新的激励机制、验证者和创新，实现更公平地分配奖励。

新的激励措施旨在鼓励单节点验证者更多地参与，同时要求更大的参与者承担责任。以反相关机制为例（ anti-correlation mechanism**）** ， 相对于 Coinbase 或 Lido 这样运营数千个节点的大型质押者，该机制对单节点验证者惩罚力度较小。反相关机制贯彻「权利越大，责任重大」的原则， 鼓励大型参与者与客户端多样性等网络目标保持一致。

PoS 不仅降低了对验证节点的资源要求 ，而且旨在奖励所有参与者，而不仅仅是那些具有最大算力的参与者。虽然仍然会有拥有较多节点和巨鲸质押的网络参与者，但每个节点将有平等的机会获得奖励。

同样重要的是要记住，虽然 PoS 将通过引入新的参与者来帮助以太坊更加多样化，但验证者仍然有责任通过选择多样化的客户端来帮助促进客户端多样性。尽管 ConsenSys 提供免费和开源的以太坊客户端软件，Besu 和 Teku，我们仍然希望用户在我们的客户端之外均匀分布。

像质押池 这样的创新降低了参与门槛，并允许多方汇集他们的 ETH 以满足 32 ETH 的最低质押门槛。ConsenSys 是 Rocket Pool DAO 中支持完全去中心化质押池的 14 个组织之一。

升级到 PoS 将使以太坊更加开放和多样化。客户端、验证者和 Layer 2  的多样性也将确保一个健康且可扩展 的网络，使参与民主化。  

关键点二：能源效率

发展网络达成共识的方法可确保以太坊能够在不损害地球的情况下可持续地支持下一代 Web3 创造者和开发者 。

PoS 将减少每笔交易中 99.95% 的能源消耗，总量大致相当于荷兰的能源消耗，单笔转账能耗可低至流式传输一小时的 Netflix 的能耗（234 瓦）或煮一壶咖啡的能耗（40 瓦）。以太坊将会使世界变得更好，而不是伤害它。 

PoS 的升级也会使以太坊更加模块化 。这种模块化将计算执行与共识层分开，而这将使网络运行的耗能更少。

此外，来自 L2 rollups 和分片的效率提升将使以太坊消耗的能量减少几个数量级。

关键点三：无缝过渡

在以太坊上工作的开发人员无需担忧过渡问题，因为合并前后架构的通用 API 以及对现有组件（如以太坊执行客户端 Besu 和 Geth）的重用将确保无缝过渡 ，并保证为开发人员提供不间断式体验 。这意味着现有应用程序需要重做的概率很低甚至没有 ，但是我们仍建议对其进行全面测试。

PoW 和 PoS 接口的相似性意味着不需要更改 API。Engine API 从一开始就与现有的基础设施和开发人员工具兼容。利用现有组件还可以确保无需重写智能合约或 dapp 或迁移基础设施。对于依赖随机操作码的 dapp，可能需要更新一些基础设施。如果您是以太坊开发人员，并且希望提前开始测试您的 dapps 和基础设施，我们鼓励您使用最近推出的Kiln 测试网来测试。

增加的客户端选择和灵活性 将允许开发人员混合使用以满足他们的需求，例如在执行层使用轻量级客户端和在共识层使用重量级质押客户端。  

迁移到 PoS 还意味着更可预测的区块时间和交易费用 ，这将减少设计高流量解决方案时的不确定性，该解决方案的区块时间为 12 秒，基准费用由 EIP-1559 确定。

关键点四： Ultra Sound Money

合并将带来更新的加密经济模型，减少 ETH 的发行 ，并为最终用户和 DeFi 行业提供更强大的安全保障 。合并还将限制 ETH 的供应，使其更稀缺，因此更有价值。

网络经济本质上是动态的，但是值得注意的是，一些潜在的结果将与 PoS 的迁移伴随而来。一是为支付给验证者而创建的 Token 数量减少（减少 90%），二是销毁作为交易费用收集的 Token，而不是将它们传递给矿工。

攻击链的成本也将显着增加（10-20 倍）。民主化的参与确保了攻击几乎不可能。在当前 35 万验证节点（1,000 万 ETH 参与质押）的基础上，攻击以太坊的成本是 340 亿美元的 51%。PoS 与 EIP-1559 的通缩影响，以及合并后质押 ETH 的预期增加，将继续提高这种攻击成本。 

更重要的是，攻击将变得更容易恢复，部分原因是罚没机制将破坏用于攻击的质押 ETH，需要进一步投资来维持攻击。少数用户激活的软分叉也可以由社区协调以销毁攻击者的 Token。

合并即将来临

我们希望这四大关键点已经阐明了以太坊将如何升级实现 PoS 迁移，并希望重振您的信心和兴趣。

本文的 TL；DR 如下：

1. 开放和多样化 ：所有类型的参与者都可以更轻松地访问以太坊，从而进一步实现去中心化。

2. 节能 ：网络运行需要的算力减少了几个数量级，将以太坊每笔交易的碳足迹降低了 99.95%。

3. 无缝过渡 ：当前在网络上工作的构建者不会中断，继续构建，继续交付，并全力以赴进行测试。

4. Ultra Sound Money ：更新的加密经济模型使以太坊更加安全、稀缺，并提高了交易的最终确定性。

ConsenSys 坚信，像以太坊这样的去中心化网络将人们更紧密的联系起来，消除中心化权力及其固有的缺点，并将使社会更加繁荣。我们将继续贡献我们的人员、产品和资源来完成合并。
