小红书的数字艺术平台 R-SPACE 正式上线了。这是在国内继阿里、腾讯之后又一家涉足 NFT 领域的行业巨头。

与幻核、蚂蚁链粉丝粒略有不同的是，R-SPACE 并非是一个独立的应用，用户可以直接在小红书应用内部购买数字藏品。此外，你还可以在艺术家的主页中找到艺术家自己的 R-SPACE 小空间，里面陈列着艺术家创作和发售的作品，而用户自己的收藏也会在用户主页中的 R-SPACE 中展示出来。

如此一来，小红书中的数字收藏品便不再是一座「孤岛」。当我们展示自己在幻核或粉丝粒中购买的数字收藏品时，只能先截图再发到个人社交媒体中，需要多一步流程。而小红书是一个开放的平台，R-SPACE 除了会将你的收藏呈现给你的粉丝，也会在更大的公域流量中展示出来，如此一来，你便可以向更多人展示自己的收藏品味，结交更多的朋友。在小红书，NFT 的社交功能得到了充分的展示。

从某种角度看，现在的 NFT，似乎比比特币还要出圈，圈内圈外的文体娱乐全面开花。

远的不说，昨天腾讯向自己员工空投了超过 7 万枚「QQ 企鹅头像」数字藏品，基本上腾讯的员工在朋友圈都换上了新头像；地球对面那块时代广场的大屏幕最近几天也被 NFT 刷屏，美国纽约举办了声势浩大的 NFT.NYC 活动，甚至邀请了获得格莱美大奖的说唱歌手 Lil Baby 为他们进行了专场演出。

「The World Has Woken Up to NFTs（世界因 NFT 重燃激情）」的标语出现在纽约时代广场，图片源自网络

再到今天，MAU 1.6 亿的超级平台小红书也推出了自己的数字藏品，将这个概念再次向上亿人推广。但当我们说 NFT 的时候，大家的第一反应是什么？

一个国内的知名 IP 方曾向我们询问过关于 NFT 的事情，他们对 NFT 的第一印象，是「这个东西现在似乎很好卖钱」。

翻看一些国内科技媒体与自媒体对「NFT」的描述，我们不难发现一件 NFT 卖出了怎样的天价是他们最为关心的内容，媒体大肆渲染 Beeple 在电脑上画的画卖出了「4.51 亿人民币」，一个马赛克头像（CryptoPunk）卖出了「上亿美元」……

一个个天文数字伴随着「NFT」这个字样吸引着人们的眼球，再加上 NFT 的概念是从区块链行业演化而得的，便很容易将 NFT 与「泡沫」、「炒作」划上等号。国内的 IP 方在被介绍 NFT 到底是个什么东西时，往往第一个标签就是，「NFT 能赚到钱」。

的确，有很多 NFT 项目的价格被炒作到很高后迅速跌落，看起来似乎 NFT 还是与「炒」有关。

但我们不能这样将 NFT 一棒子打死。

你们有没有想过，如果 NFT 等于「泡沫」的话，为什么阿里、腾讯、小红书等行业巨头宁愿顶着监管的压力与风险也要推出自己的 NFT 平台呢？为什么 LV、Gucci、Burberry 等全球知名品牌也要发行自己的 NFT 呢？为什么余文乐、吴建豪、陈奂仁、阿朵、Higher Brothers 成员 Psy.P 等一众明星也纷纷买入甚至发售自己的 NFT 呢？

是为了赚钱吗？NFT 领域作为最新的热点存在着大量商机与诱人的财富效应，但是对于行业巨头来说，对一个全新赛道的布局主要原因绝对不会是收入，尤其是在目前的大环境中，中国 NFT 的前景并不明朗。这些巨头和明星愿意冒着巨大的风险参与 NFT 的原因是他们看到了 NFT 绝对不只是炒作。

在小红书正式上线自己的数字艺术平台这天，我们决定和大家好好聊一聊 NFT 究竟有什么价值。

NFT 是什么？

让我们先来了解一下到底什么是 NFT。

首先，NFT 不是虚拟货币，虽然名字里都有一个「Token」，但是与其他同样被称为「Token」的虚拟货币有很大不同。NFT 最重要的特征是独特性，也就是说它并不能分割，因此也就不适合作为一般等价物或是计价单位。中国银行原副行长王永利也曾在文章中发表自己的观点称，NFT 翻译成「非同质化权证」更为恰当。

其次，NFT 不是 jpg 图片。虽然现在我们常见的 NFT 都是一张张图片，也会有一些 mp4 等其他格式的艺术作品，但是 NFT 并不是这些图片或视频本身。

那 NFT 到底是什么，我们该如何理解 NFT 呢？

打个比方，我们可以把 NFT 当作一个玻璃杯，装着茶水就变成绿色，装着可乐就变成黑色。NFT 是一个详细记载着各种不可篡改的信息的「超链接」，我们常见的图片只不过是点击超链接后跳转出来的而已。我们看到的 NFT 是什么取决于这个「超链接」链接到了哪里。

有了这样的基本理解，让我们再来看看 NFT 有哪些实用价值，看一看 NFT 真正的价值到底是什么。

NFT 与数字艺术

首先从 NFT 最广泛的应用场景说起，这也是目前国内行业巨头都在做的事情，那就是在数字艺术领域的应用。

在 NFT 技术被广泛采用之前，很少有数字艺术家能以自己的数字艺术作品销售为生，他们很多人只能将其作为兴趣爱好，然后找一个设计师的工作来养家糊口。而很多数字艺术家的作品在网络中广泛传播甚至被盗用时，艺术家本人并不能从中获益甚至连追责都做不到，因为很多时候艺术家本人也并不知道自己的作品被盗用。

NFT 改变了这一局面。NFT 相当于一件艺术品证书，它能够证明这一张图片是艺术家的「原作」或「真迹」，其他的图片只不过是其他人拷贝保存的「副本」，区块链会记录着这件作品的创作者是谁，创作于什么时间，原本可以在互联网被随意复制传播的作品也变得可以分出真品与赝品，数字艺术作品也开始拥有了「被收藏」的权力。

很多人的疑惑在于，即便有 NFT 这项技术，我们依旧可以直接从网络将图片保存下来并发布到各个社交媒体之中，和使用 NFT 之前没什么不同，那这件 NFT 数字艺术品又有什么价值？

我们可以这样理解：在使用 NFT 之前，盗用数字艺术品是在损害艺术家的权益；在使用 NFT 之后，数字艺术品即使被盗用也是在为艺术家及其作品增加价值。NFT 可以让人们知道哪一件作品才是原作，人们盗用艺术品传播、制作其他产品其实都是在为原作做宣传，所创造的价值也可以精准流向原作，而不再像 NFT 诞生前那样无处可去。

此外，NFT 能够为艺术家带来源源不断的版税收入，这已经对艺术行业产生了巨大的影响。传统艺术领域中，艺术家能拿到的收入只有作品第一次售卖的价格，假如作品在一开始只卖出了 100 美元，即使未来拍卖成交价高达数亿美元艺术家也拿不到一分一毫，这显然是不合理的，艺术价值被发现后，艺术的创作者竟然拿不到任何收益。

究其原因，一是因为传统艺术品市场流动性较差，而艺术品的流通过程并不透明，经历几次拍卖或私下交易，艺术品流转到了哪位藏家手中往往会不得而知，也就更不用说给艺术家分版税的事情了。

而通过区块链技术，NFT 艺术品的每一次流通都被详细记录，包括买卖双方及成交价等，且通过智能合约，艺术家可以自动从每一笔交易中分得固定比例的版税，这将会很好地改善艺术家的生存环境。

NFT 与虚拟潮玩

除此之外，NFT 的另一个容易理解的应用便是虚拟潮玩。在日常生活中，我们会买手办、盲盒以及各种小玩具，而当我们将它们买回家后，除了拍几张照片发发朋友圈便只会将其作为小摆件，一些稀有款可能还会有二级市场流通的可能，而一些普通款基本上就只能摆在自己家里。

而只是「摆在自己家里」这一点就会难倒很多潮玩爱好者，一些稀有的手办不仅价格昂贵，更是会占据较大的面积，很多潮玩藏家常说：「这些手办很贵，比它更贵的是北京的房价。」而 NFT 潮玩则可以解决这一问题，虚拟潮玩不会占用空间，而且通过与 AR、VR 技术的结合可以让藏家得到更好的收藏体验。

除此之外，区块链技术也可以保证盲盒的随机性，也能保证潮玩数量的固定，不会因为发行方增发而造成价值稀释。NFT 潮玩也会拥有更高的流动性，在日常交易中，时间成本、运输时的风险都是我们需要考虑的，而 NFT 潮玩既能实时到账，又无需担心交易过程中对藏品本身带来的损害。

NFT 与游戏

NFT 的第三个常见应用场景是游戏。当游戏内的资产变成了 NFT 对玩家来说有什么好处呢？

作为在电子屏幕前成长起来的一代，我们从小到大玩过各式各样的游戏，也难免会遇到一些游戏无法继续运营而关停服务器。一旦服务器关停，你在那款游戏中获得的道具、装备、角色皮肤等等都会随之消失，而你曾经在这款游戏上付出的时间、精力也统统付诸东流。

而当这些游戏资产变成 NFT 存储在你自己的钱包中，即使游戏服务器关停，这些资产也并不会消失，当其他人制作了一款可以集成这些 NFT 的游戏时，这些游戏资产又能重新派上用场。

又或者是这样的一种情况。当你因为某些原因导致游戏账号被封禁，账号内的所有游戏资产也将被冻结在这个游戏账号中，这款游戏可以轻松罚没你的资产。而对于区块链游戏来说，当你的游戏资产变成 NFT 便可以解决这个问题。

Axie Infinity 是当前最著名的区块链游戏之一，玩家可以通过玩游戏赚钱，而在早期超高的收益甚至能帮助东南亚的贫困家庭获得比普通人高出数倍的「工资」。这也让一些人开始同时操作很多账号或者是使用脚本来「刷钱」，引起了官方和其他玩家的不满，于是 Axie Infinity 官方也对这些账号进行了处罚。

处罚的方式很有趣，按照官方的说法，Axie Infinity 只是不再给他们提供游戏服务，而并没有没收他们的游戏资产，他们拥有的小精灵依旧可以在 Decentraland 等其他平台使用。

NFT 让玩家真正地把游戏资产攥在了自己手里。

NFT 与社交

讲完上述 NFT 最为基础的用法，让我们稍微深入一些，来聊一聊 NFT 与社交。

八月底，NBA 巨星史蒂芬·库里花了 18 万美金买下一只 Bored Ape NFT。通常人们会关心的是这十八万美金库里只需要打几秒钟的比赛就能赚到，会关心库里把自己的社交媒体头像更换成了这只猴子，而很少有人会关心为什么库里会买这只猴子，为什么那么多人都想拥有一只 Bored Ape。

是为了投资吗？若只是为了投资，库里也不会在 Bored Ape 系列价格上涨到最高点的时候才买入。是为了收藏吗？若只是为了收藏，库里也大可不必把它换成自己的头像。

其实，比库里买猴子、换头像更有趣的事情，是库里加入了 Bored Ape 社区。

和大家一样更换社交媒体头像，和大家一样加入 Discord 社区，在 Bored Ape 登陆佳士得的那天和大家一样给猴子换上了整洁的黑色西装。在这些时刻，库里似乎并不是个高高在上的 NBA 巨星，他只是一位 Bored Ape 社区成员。

这也是头像类 NFT 在沦为炒作投机工具之前爆火的原因，这些成系列的图片能够很自然地让人们组成一个个社区，在社区中每个人都是平等的，都会互相帮助、共同发展，如果你的偶像和你恰好买了同一个系列的 NFT，你们之间的关系便会不再那么陌生，甚至可能会成为朋友。这便是 NFT 作为「社交资本」的价值。

戴维·迈尔斯在《社会心理学》中提出，统一着装、统一装饰等可以唤醒人们的群体意识，而随之而来的也有每一个人的去个性化，从而导致自我观察的弱化，进一步促使这些群体做出很多出格的举动，而非更加健康地发展。而这些 NFT 也会在唤醒人们群体意识的同时保留每一个人的个性，进而推动整个群体朝着正确的方向前进。

而就在昨天，腾讯为庆祝公司成立 23 周年为其员工发放 23 周年纪念 NFT 数字藏品，很多在腾讯工作的小伙伴也都换上了企鹅头像并在朋友圈分享着。

而这也算是联盟链上的第一个头像类 NFT 项目，通过换头像、在朋友圈炫耀甚至在「腾讯文档版 OpenSea」进行员工之间的挂单、交易，让 NFT 的社交资本属性为人熟知。

NFT 与身份

书接上文，我们继续探讨关于身份的问题。

现如今，我们在现实生活中的一切身份、经历都依赖于中心化机构或平台的证明。

举个例子，你的大学考试成绩记录在学校的档案中，你的成绩单由学校出具，而成绩则是由你的老师手动填写的，而且填写后仍然可以进行修改；而你的工作情况、工作经历、工作水平在领英是需要自己及同事填写的，你的工作能力强不强取决于同事在领英为你输入了怎样的评价。

而如果将你的各种证书、经历铸造为 NFT 并将其存储在区块链上，那么这些不可篡改的 NFT 就将成为最可靠的证明，不需要依赖中心化机构或平台，不会被人篡改，更不用依赖同事的评语。都说互联网没有记忆，但区块链会将人们过往的一举一动都记录在案。

NFT 与元宇宙

最后，我们来聊一聊目前最热门的「元宇宙」。相信大家最近在不少文章中都看到了对元宇宙的如下介绍：

元宇宙一词来源于《雪崩》，书中这样描绘元宇宙：「只要带上耳机和目镜，找到一个终端，就可以通过连接进入由计算机模拟的另一个三维现实。每个人都可以在这个与真实世界平行的虚拟空间中拥有自己的分身。」

每一篇讲元宇宙的文章都会引用这一段话，大概的意思就是我们可以把元宇宙简单理解为「虚拟世界」，而既然元宇宙也是一个「世界」，那就要遵守一些「世界」的法则。就比如，你的东西要真的属于你才对。

这是什么意思呢？让我们来举个例子。现实生活中，假设你常购买的一家球鞋品牌倒闭了，也不会有人会去你家把你曾经购买的该品牌的球鞋拿走，就算有人来拿你也一定不会同意。但是在虚拟世界中，我们却无数次选择妥协，我们在那个世界的资产也应该牢牢把握在自己的手里。

而实现这一愿景的唯一途径便是区块链，有了区块链，才有真正的元宇宙。中心化的元宇宙是很恐怖的，试想一下，在那个宇宙中有一个真实存在的「统治者」，他可以随意更改你拥有的一切，甚至可以毫不费力地将你从这个世界中「抹除」。但是当我们讲元宇宙中的资产上链做成 NFT，既不可以被篡改也不可能被删除，这样的「虚拟世界」才不仅仅是一个「模拟游戏」，才能被称作是「元宇宙」。

当然，你可以只关注那些价格被炒上天的 NFT 艺术品与收藏品，也正如余文乐此前说的，面对这些 1/1 的 NFT，当你知道世界上仅此一件时，你很难按耐住购买的冲动。

但是，不可否认的是，NFT 将会成为未来世界的基础设施，具有改变世界的潜力。在炒作背后，NFT 有着更多实用价值，我们应当将更多的关注投入到技术本身和更加落地的应用场景，而不要一直盯着一个个成交额的天文数字。我们要拥抱这个可以让我们的生活变得更好的新技术，同时拒绝参与到炒作及其他危害行业发展的行为中来。
