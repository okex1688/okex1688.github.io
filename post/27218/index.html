在本文中，我们将讨论 NFT 是什么，数字艺术品分为哪些种类，数字艺术品存储在哪里以及选购 NFT 需要注意哪些事项。

太长不看：NFT 艺术品的原理

简而言之，NFT 艺术品就是创建独一无二的代币来绑定艺术品，要么将艺术品链接至某个代币，要么将艺术品连同其 NFT 存储在区块链上。

NFT 艺术品主要分为两类：可繁殖艺术品和不可繁殖艺术品。不可繁殖艺术品由人类创作。可繁殖艺术品则利用代币独有的 ID 作为 “内核” 决定 NFT 的外观和特征。

NFT 背后的艺术品有三种不同的存储方式：

  1. 与 NFT 一起存储在同一条区块链上

  2. 存储在另一条链上或采用去中心化存储方式

  3. 存储在私有或中心化服务器上

上述三种存储机制会给你的 NFT 带来不同的风险。如果你的 NFT 及其背后的艺术品没有存储在同一条链上，二者之间的链接有可能断开，导致你的 NFT 变得一文不值。

最后，虽然一些区块链目前消耗大量能源，但是人们正在采取行动有计划地进行节能减排。

NFT 是什么？意味着什么？

NFT 是带有唯一标识符（ID）的代币。如果是普通的密码学代币，同一种代币之间无法相互区别，因此是 “同质化的”（每个代币都是相等的，就像现金那样）。NFT（非同质化代币） 是可以识别且独一无二的，也就是说每个 NFT 本质上是不一样的。

- 每个 NFT 都有其专属 ID，因此是非同质化的（彼此之间互不等同） -

你可以将 NFT 理解成具有特殊性质的代币，可以让你创建独一无二的数字资产。

NFT 这一概念为独一无二且可追踪的产品赋予了数字稀缺性。在本文语境下，稀缺性指供给有限（且可控）。无论创建了 1 个还是 100 个 NFT，每个 NFT 都有其专属 ID。

由于 NFT 是独一无二且具有稀缺性的数字产品，你可以将任何东西捆绑到 NFT 上。NFT 有很多重要应用场景。我们来看几个潜在用例：

  • 供应链管理（例如，追踪疫苗从生产到通过分销网络进入诊所或医院的过程）

  • 独一无二的可兑换票券（例如，音乐会门票、彩票、访问令牌等）

  • 奖章和不可交易资产（例如，出勤证明、军训证书、某平台的身份证明等）

NFT 艺术品类型

NFT 艺术品主要分为两类：可繁殖艺术品和不可繁殖艺术品。

不可繁殖艺术品就是我们通常理解的艺术品，即，由人类艺术家创作并上传至区块链与 NFT 绑定的画作。我们看到的大多数 NFT 艺术品都是不可繁殖的。主要例子有：

  • Beeple | 最近以高达 6900 万美元的价格出售一幅 NFT 画作，刷新了 NFT 拍卖记录。

  • OpenSea 等交易平台上的大多数 NFT 都是不可繁殖艺术品，可以去上面看一看！

可繁殖艺术品是通过某种算法创建的，利用 NFT 的独有 ID 作为 “内核” 生成艺术品。每件可繁殖艺术品都是独一无二的，而且只与一个 NFT 绑定。当然了，可繁殖艺术品也需要艺术家提前创作各种 “碎片” 喂给算法，并告诉算法如何将这些碎片拼凑成一副完整的作品。可繁殖艺术品领域已经有了很多成功案例：

  • Crypto Kitties | NFT 可繁殖猫 | DNA 存储在链上，图像存储在链下的中心化私有服务器上

  • Axie | NFT 可繁殖宠物精灵类游戏 | DNA 存储在链上，图像存储在链下但是支持第三方访问

  • AutoGlyphs | NFT 可繁殖艺术品 | 在链上存储和生成

  • ChainFaces | NFT 可繁殖艺术品 | 在链上存储和生成

  • HashMasks | NFT 可繁殖艺术品 | 种子（DNA）存储在链上，图像在链上生成并以去中心化方式存储

一些 NFT 还可以让艺术家收取版税。也就是说，这些 NFT 每出售一次都可以带给其创作者收入，而非只有第一次出售可以。

NFT 背后的艺术品存储在哪儿？

在创建了独一无二的 NFT 之后，我们该如何将艺术品与 NFT 绑定？

- NFT 代币有其专属 ID，可以链接至数字艺术品乃至现实世界的资产 -

有多种方式可以做到这点。虽然有点麻烦，但我们还是有必要了解如何防范诈骗、NFT 代币无法长期持有以及项目设计不佳/草率等情况。

虽然代码（智能合约）有可能被黑，但这是任何在区块链上构建的去中心化应用都无法避免的问题。无论我们采用何种方式存储艺术品，都存在这种风险。

将艺术品存储在同一条链上

关于如何将艺术品绑定到 NFT 上，我们要讨论的第一种方式是将艺术品与 NFT 存储在同一条链上。这是最安全的方式，可以有效保证 NFT 长久存在。

这意味着，即使项目方破产、网站关停，你的艺术品依然安全地存储在区块链上。

- 最安全的 NFT 存储机制是将艺术品与 NFT 一起存储在同一条区块链上 -

示例：

  • AutoGlyphs | NFT 可繁殖艺术品 | 在链上存储和生成

  • ChainFaces | NFT 可繁殖艺术品 | 在链上存储和生成

将艺术品存储在另一条链上或采用去中心化的存储方式

你也可以将艺术品存储在与 NFT 不同的链上或采用去中心化的存储机制（例如，IPFS 和 Arweave 等）。

- 将你的 NFT 代币和艺术品存储在不同的链上或采用去中心化的存储方式 -

当你的艺术品存储在另一条链上或采用去中心化的存储方式时，有些事项将影响你的 NFT 的风险：

  • 如何处理链接？

有很多项目提供艺术品和 NFT 之间链接的托管服务。但是，这也意味着，一旦项目方停止运营，你的 NFT 与艺术品之间的链接就会断开，导致你的 NFT 变得一文不值。

  • 艺术品存储在哪里？

并非所有存储方式都会产生相同的效果。假如你所有的 NFT 艺术品存储在 IPFS 上，就一定要有人 “pin（长期保存）” 这个艺术品。如果 NFT 创建者不再 “pin” 这个艺术品，这个艺术品就会消失，即使链接依然有效也无济于事。

示例：

  • HashMasks | NFT 可繁殖艺术品 | 内核存储在链上，图像在链上生成并采用去中心化的存储方式

尽管如此，上述存储方式在安全性上依然远胜于下面这种……

采用中心化存储方式存储艺术品

中心化存储就是我们最常用的存储方式，例如，存储在 AWS（亚马逊云服务）或私有服务器上。这种存储方式意味着，一旦 NFT 创建者停止托管 NFT 艺术品，买家手中的 NFT 就会变得一文不值。

- 你的 NFT 存储在链上，但是对应的艺术品既不存储在链上，也不采用去中心化的存储方式，而是由发行该 NFT 的公司或团队存储 -

在这种情况下，你的 NFT 面临很高的风险：

  • 如果发行方不再存储你的 NFT 艺术品，你的 NFT 就会变得一文不值。

  • 如果存储机制出现问题，你的 NFT 就会变得一文不值。

示例：

  • Crypto Kitties | NFT 可繁殖猫 | 种子（DNA）存储在链上，图像在链下生成并存储在中心化的私有服务器上

  • Axie | NFT 可繁殖宠物精灵类游戏 | 种子（DNA）存储在链上，图像存储在链下但是支持第三方访问

需要注意哪些事项？

购买 NFT 时，你需要注意以下几点：

交易平台是否正规？

市面上有很多 NFT 交易平台。OpenSea 等大型平台上有很多 NFT 出售，也有很多用户。如果你不确定某个交易平台是否可信，可以带关键词搜索相关信息，如 “交易平台名称 + 诈骗”、“交易平台名称 + 偷走了我的钱”。

是否是原创艺术品？

有时，狡猾的骗子可能会剽窃他人的艺术品并为其创建 NFT，然后声称该 NFT 艺术品是他们原创的。这种情况甄别起来比较困难，但也不是不可能。你需要进行以下操作：

  • 右击 NFT 的图片并复制。

  • 将该图片粘贴到谷歌搜索栏。

  • 与该图片相关的第一个链接（忽略广告）很有可能是原创的链接。

  • 如果这个链接就是你看到的 NFT 的链接，那太好了！这说明你看到的 NFT 是原创的。如果不是，那八成是盗版的。

还有可能导致 NFT 被复制的情况是整个项目在另一条链上发布。在这种情况下，原版 NFT 一般存储在以太坊上。

NFT 艺术品属于优质投资吗？抑或是一场豪赌”？

永远不要在任何密码学货币上投入你所愿承受的亏损范围之外的钱。从长期来看，大多数 NFT 并不保值，在购买 NFT 之前，你必须清楚这一点。

最后的思考：如何看待区块链的能耗问题？

太长不看：很多区块链都在部署节能减排的解决方案。

以太坊（绝大多数 NFT 都在以太坊上创建）即将迎来重大更新，大幅降低以太坊的能耗。

虽然以太坊的能耗问题很严重，但是经常被过分夸大，还被用作一种 “FUD（Fear, Uncertainty and Doubt，害怕、不确定和怀疑）”话术来吓跑想要进入区块链行业的新人。特此声明，区块链项目各有千秋，并非所有区块链都像比特币那样达到堪比一个小国的能耗量。

个人观点：

尽管存在能耗问题，但是区块链不会消亡。此外，区块链技术拥有巨大的潜力，能够以前所未有的方式将（数据所有权、透明度、主权等）权利交到广大消费者手中。

很多大公司都在抹黑去中心化区块链，试图劝退普通用户，为自己争取时间找到区块链的替代方案，继续控制用户的数据，享受所有权利。Facebook 在密码学货币方面的尝试就是个很好的例子（存在 很 多 问 题）。

如果你还是顾及能耗问题不想使用区块链，不妨再等一两年看看。区块链行业正在部署节能减排方案 。
