本文是对 NFT 的总体概述，和波卡中的各种 NFT 方式。

同质化

NFT 的意思是非同质化通证。同质化意味着一组东西的内部元素可互换。理论上，一张 20 美元的钞票在商店里总是值 20 美元，而且和其他任何一张 20 美元的钞票价值相同。但是，它不能和 1 美元或 100 美元的钞票互换。

一张 “喷火龙” 宝可梦卡牌和一张 “杰尼龟” 卡不可互换，但是 “喷火龙” 卡牌之间却可以互换。

同质化是一个范围 —— 对某些人来说同质化的东西，对其他人来说可能是非同质化的。实际上，宝可梦卡作为非同质化资产的典型例子，其实比美元钞票更加可互换，因为每张钞票都有一个对政府机构可能很重要的唯一序列号，而这些卡牌却没有序列号。

图源: Investopedia

此外，如果玩家只关心角色的外观，那么游戏中像 “紫色魔法剑” 这样的数字物品可以与另一把视觉上相同的剑互换。但是，如果另一把剑具有不同的功能，而且这个功能会影响玩家即将开始的冒险的结果，那么外观相同的剑之间绝对是不可互换的。

请记住，对 NFT 的最简单解释是，NFT 是任意的、特定于项目且不可互换的几行数据，可以通过加密方式证明其 “属于” 某人。这些数据可以是任何东西——音乐会门票、出席证、简单的单词、头像、元宇宙中的地块、音频片段、房屋契约和抵押贷款等等。

NFT 标准

通用区块链不是为了原生地理解 NFT 的概念而构建的。它只对自己的原生代币进行原生感知和优化，但构建在这种链之上的实现本质上是一种 “魔改”。

例如，以太坊是一个通用区块链，它没有内置 “通证”（同质化或非同质化）的概念。以太坊中的通证本质上是一些信息表格，各种用户界面以某种方式解释和读取这些表格。阅读此类信息的方式被称为 “标准”。

最普遍的同质化通证标准是 ERC20，你应该也有所耳闻，而最广泛的 NFT 标准是 ERC721，紧随其后的是 ERC1155。必须定义这些标准的缺点在于，它们始终是关于如何阅读电子表格的说明，这些电子表格假装以某种方式提供信息，根据定义，这种方式无法优化。出于这个原因，即使在网络拥堵极低的好日子，与任何 EVM 链上的 NFT 交互也得花个几美元，而 2021 年在以太坊上的每次交互（交易、造币、销售）平均要花 100 美元。

不过这可以防止当前以太坊上收集 NFT 热潮的情况 —— 个人资料图片、生成的 “看一眼然后收起来的” 艺术品、ENS 地址和出席证徽章证明等。

a typical NFT on Ethereum

为了比较，我们可以将它们称为 NFT 1.0：静态 NFT，几乎完全是基于图像的不同稀有度的收藏品。

NFTs 2.0: 在 Polkadot 和 Kusama 上的 NFT

这就是 Polkadot 技术大放异彩的地方，也是 NFT 2.0 发挥作用的地方。

Polkadot 允许异构的特定应用的分片存在，构建者能够针对复杂的 NFT 用例进行本地优化，而无需权衡取舍，因为在其他系统中交互是非常低效和昂贵的。

以下是 Polkadot 生态系统中存在和/或正在开发的 NFT 解决方案.

Unique network

Unique network 是一个专注在 NFT 的区块链，提供例如赞助交易、将可替代代币与不可替代的代币捆绑，以及将 NFT 拆分为可替代代币以获得部分所有权等创新。

迄今为止，Unique Network 已经启动了两个 NFT 项目：从 Hackusama 中出来的 Substrapunks，以及在 Polkadot Decoded 展示过的 Chelobricks。他们目前正在运行一个连接到 Kusama 的 betanet，并且这些 NFT 已经可以在其上进行交易：请参阅 Unqnft.io。

用户可以将 KSM 发送到他们的 Unique Network 托管账户，在那里进行交易，然后将任何赚取的或剩余的 KSM 发回。

Unique Network 旨在使他们的 marketplace 技术开源并且对白名单用户友好。理论上，使用 Unique 的技术为你的项目建立一个新的市场应该是简单的。Unique network 旨在成为 Polkadot 上的平行链，Quartz 是他们在 Kusama 上的网络。

RMRK

RMRK 是一种“黑客”，是直接在 Kusama 中继链之上的强制标准。由于 Kusama 是轻量级的，因此它可以处理与它连接的各种平行链，但这并不意味它有其他复杂的链逻辑，如原生 NFT 或智能合约来启用它们。然而，由于市场需求和 Kusama 的“混乱”性质，RMRK 团队决定从比特币中采取“彩色硬币”的方式，将 NFT 作为涂鸦 ( graffiti ) 在 Kusama 链上实现。

RMRK 标准是一套关于如何解释 Kusama 上的特殊涂鸦 ( graffiti ) 的规则和规范，称为“ remarks 备注”，可通过任何 Substrate 链中的内核系统模块访问。

RMRK 团队刚刚推出了该协议的 2.0 版本，即一组“NFT 乐高积木”，将这些原语放在一起时，构建者可以在没有智能合约的情况下组成任意复杂度的 NFT 系统。这些 “乐高积木”是：

  • NFT 可以拥有其他 NFT，NFT 可以装备其他 NFT 进行视觉变化

  • NFT 可以有多个资源（基于上下文和资源优先级的不同输出）

  • NFT 可以具有用于价格发现和社交机制的链上表情（反应）

  • NFT 有条件渲染（例如，如果蒙娜丽莎有 50 个亲吻? emoji，则显示她脸红）

  • NFT 可以通过可替代的代币由社区管理（NFT 的分割）

即将推出的 3.0 版本（2022 年第一季度）将是所有 RMRK 2.0 逻辑的模块和智能合约 (EVM) 版本，并集成到合作伙伴链中，以便在数十条链中廉价且轻松地传送不可替代的东西。

RMRK 团队正在与 Unique network 密切合作。RMRK 的逻辑和技术将以 runtime 升级（FRAME 模块）的形式部署在 Unique Network 上。

已经有两个基于 RMRK 的 NFT 市场存在了，目前已经启动了数百个项目：

  • Singular：官方市场 https://singular.rmrk.app/

  • Kodadot：第三方市场 https://kodadot.xyz/

此外，可以在 Kanaria 平台 (https://kanaria.rmrk.app/) 上访问和测试具有可组合、嵌套、多资源 NFT 的 RMRK 2.0 功能。

了解更多 RMRK 信息，请查看视频解释（https://url.rmrk.app/rmrkcc）和文档说明（https://docs.rmrk.app/）

Efinity

由以太坊 ERC1155 标准的作者和 Enjin 钱包和 Unity 插件的制造商 Enjin 带头，Efinity 是一个 NFT 桥接链，将于 2022 年登陆 Kusama 和 Polkadot。

他们的计划是建立一个 paratoken，这将成为 Polkadot 生态系统中跨不同平行链的 token 迁移的标准，也可以进出以太坊和其他 EVM 系统。

Moonbeam

Moonbeam 和 Moonriver 是具有以太坊 RPC 端点的完整的 EVM 部署。

这意味着适用其他 EVM 链的整个工具包（Hardhat、Remix、Truffle、Metamask 等堆栈）可供 Moonriver / Moonbeam 用户和开发者使用，从而在吸引现有用户群方面显著领先。

有几十个知名团队在 Moonriver / Moonbeam 上发布他们的产品（或重新发布）。但是，重要的是要注意 Moonbeam 是一个 EVM 链，因此在以下方面将受到与任何其他 EVM 链相同的限制 NFT 的定制和功能丰富性。

然而，一个显著的优势是 Moonriver / Moonbeam 仍然是一个 Substrate 链，这意味着将自定义模块集成到 runtime 仍然是可能的，这使得在链 runtime 级别的 NFT 特定优化成为保持 EVM 兼容性的好方法，同时优化 NFT 的存储和交互。

Uniques

Uniques 是部署在 Statemint 公益平行链上的 FRAME 模块。它是最基本的 NFT 的一种实现 —— 引用一些元数据的数据记录。此元数据引用在冻结之前是可变的，因此 NFT 及其类（派生自它们的实体）是可变的，除非发行者明确将其设为不可变。

Uniques 故意采用非常简单的方法，以使 Statemint 链成为可替代和不可替代的简单平衡链。

桥接

桥接 Substrate 链和 EVM 链需要付出很多努力，但这是 NFT 行业非常需要的功能。合并收藏品和客户群具有重大意义，因此多个项目都在努力实现它。

除了前面提到的 RMRK（XCMP 原生的 Substrate-to-Substrate 无缝远程传输）和 Efinity（Paratoken）之外，以下工作也正在进行中：

  • MyNFT：EVM 到 EVM 的桥接工作。

  • RMRK<->EVM 简化桥：在 RMRK 黑客松期间开发的桥，用于将 RMRK NFT 移植到 EVM 链上的简化 IOU，即将在 2022 年 11 月部署在 Moonriver。

  • RMRK<->EVM 全桥：RMRK 2.0 的 EVM 版本在 2021 年 12 月准备就绪，这意味着将 RMRK 2.0 NFT 从 RMRK（Kusama）完全迁移到 Moonriver（和其他 EVM）成为可能。
