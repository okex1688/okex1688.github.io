在花了两年时间开发Zora之后，我开始意识到我们的方法有多么与众不同。这种差异来自一种全新的心理模型，即加密协议的可能性（并且已经发生）。

加密协议的本质为创建宝贵的公共基础设施开辟了全新的模式。这种观点是如此鲜明，以至于我认为它需要新的语言。

在这篇文章中，我概述了超结构（Hyperstructures）的概念：它们是什么，它们是如何工作的，以及为什么它们将成为未来几十年互联网的基础。

保罗·索莱里 (PaoloSoleri) 的“超结构”， 图右是同比例的帝国大厦

定义

超结构：可以免费且永久运行的加密协议，它们无需维护，不会中断，也不需要中介。

区块链导致了一种称为“超结构”的新型基础设施的出现。

超结构采用在区块链上运行的协议的形式。如果某些东西具有以下这些特征，那么它就可以被认为是超结构：

  1. 不可阻挡：协议不能被任何人阻止，只要底层区块链存在，那它就会运行；

  2. 免费：在协议范围内收取0%的费用，并完全以gas成本运行；

  3. 有价值的：产生所有者可访问和退出的价值；

  4. 扩展性：协议中的参与者有内置的激励措施；

  5. 无需许可：普遍可访问且抗审查，构建者和用户不能被去平台化（即阻止访问）；

  6. 正和：它为参与者利用相同的基础设施创造了一个双赢的环境；

  7. 可信的中立：协议与用户无关；

值得注意的是，仅仅因为一个应用程序是基于协议的，并不立即意味着它是一个超结构。例如，Opensea使用的Wyvern协议（以及其他协议）如果没有在私有数据库中维护链外订单，就无法运行。这意味着，如果Opensea或在其上构建的任何其他平台宕机，市场也会随之无法访问。Coinbase这样的平台也是如此，它促进了加密资产的交换，但如果它宕机，其交易市场也就无法访问。一个更微妙的例子：如果你看到一个完全链上的协议，管理员可以完全升级和改变协议的规则，那么这就不是超结构的，它也属于平台。这并不是说这些例子中的任何一个本质上都是坏的，它们只是不属于超结构。

超结构完全是链上的，并且是为任何参与者创造一个正和生态系统的公共产品。我将在下面详细介绍超结构的每个主要特征。

不可阻挡

传统的基础设施，如电网和社交媒体平台，需要可信的中介机构来维护和运营。如果没有可信的中介机构，基础设施将降级或完全停止运营。这些运营商是私人所有或国家控制的组织，它们需要利润、劳动力或补贴才能长期发挥作用。

由于在区块链上运行的性质，超结构可以永远运行而无法阻止它们。它们可以在没有维护者或操作员的情况下持续运行，并且只要底层区块链正在运行，它们就可以运行——至少可以运行十年。超结构是不可阻挡的，你可以有一次性的创建成本，然后一旦部署，它将完全按照设计运行，并且不会降级。维持超结构不需要更多的劳动力或资本。如果建立在超结构之上的核心团队或平台消失，它仍将完全按照设计运行，并在数十年内全面运行。不可阻挡性是由于区块链而成为可能的全新能力，并且我相信这种能力会改变基础设施的经济性质。

如果我们要缩小一个抽象层次，我们可以将区块链称为超结构的可信&无需信任的（是的，你没看错）运营商。

免费

既然我们已经将“不可阻挡性”确立为一个全新的功能，我将提出我最具推测性和争议性的主张：超结构可以同时永远免费使用，并且拥有极高的价值。这是可能的，因为它们不可阻挡。维护和保持协议永久运行是免费的。部署后，它将完全按照设计的方式工作，不会降级。例如，如果 Uniswap 团队和网站今天消失了，该协议依旧将永久运行下去。这是以前不可能做到的事情。

值得一提的是，在超结构中，仍需要人支付gas费成本才能在当时运行它。

有价值的

超结构可在免费使用的同时，并对于拥有和治理也极具价值。这是我们对 NFT 观察到的一种熟悉的价值模型：媒体可以普遍免费消费，但对于个人或团体的拥有和控制是有价值的。

但在这种情况下，所有权意味着什么？存在可在协议中打开的费用开关。这就产生了一种称为“费用威胁”的动态。这意味着超结构的所有者有权在任何时候通过投票在协议的基础级别开启该费用。这是费用的威胁，因为从长远来看，打开它会破坏价值。开启费用是一种破坏价值的行为，因为它会立即导致一个激励性的分叉，因为现在有一个明确的理由让新进入者自己这样做。

这种销毁权是一种所有权，就像 NFT 的所有者拥有销毁权一样。一个理性的参与者不会这样做，但如果他们愿意，他们可以。在我看来，这种破坏权创造了一种自然的市场力量来赋予真正的价值，因为人们会想要保护这种价值和普遍提供的效用。

除了开启和收取协议费用的权利之外，其他所有权可以包括部署新版本的权利以及对 DAO金库的治理以用于生态系统的资助和发展。

利润和价值之间存在细微差别，进而利润提取和价值创造也存在细微差别。虽然完全有可能在短期内获取利润，但这很可能与长期成功和价值创造直接冲突。我甚至可以说，“营利”是一种扭曲的运作模式，这导致了一种局部优化，它错过了来自协议之上的多样化平台生态系统的更广泛机会。代币所有权和不可阻挡性媒介的性质，意味着我们不再需要提取利润来实现价值创造。我们可以为公众创造一个新的潜在价值体系，这意味着在超结构中创造的价值，远远超过可以在短期内提取的任何利润，并且可以因其为整个社会提供的价值而得到认可。

高昂的费用

超结构具有内置于激励机制中的高昂费用，任何人都可以利用这些激励机制在协议之上以编码方式增加价值。这些费用是透明的，可供整个生态系统使用。为了在超结构之上有效地赚取这些费用，你需要提供协议定义的所需价值。这些激励措施的存在为超结构的社区扩展提供了一种机制，良好的机制设计可能对长寿至关重要。

Uniswap LP费用就是一个很好的例子。LP费用激励参与者提供关键资源流动性。这些费用支付给向池子提供流动性的任何人，但不支付给Uniswap。LP扩展了Uniswap的效用，他们在提供该效用方面没有垄断权，LP只捕获他们所创造的价值。

另一个例子是Zora Finder的费用，它激励参与者提供关键资源分配。这笔费用将支付给找到 NFT 的中标者或最终买家的任何人。Finder 扩展了 Zora 的效用，并且他们不垄断正在展示的 NFT 市场，Finder只捕获他们正在创造的价值。

在超结构中，任何参与者获得的价值都是其产生的价值的同义词。反之亦然，如果任何第三方或实体有能力在不创造价值的情况下从系统中自由提取价值，那么系统就不是超结构。

无需许可

无需许可是部署在区块链上的加密协议所独有的保证。无需许可意味着超结构：

1、普遍可访问，每个人都可以毫无偏见地完全使用它；

2、无法更改其核心操作功能；

无需许可，意味着每个人都可以自信地在超结构上构建自己的平台、应用程序和经济模型，而不会有去平台化的风险。不需要 API 密钥，也不用担心一个平台会突然一时兴起而下架你的整个项目。

同样，这在个人层面提供了审查阻力，因为没有仲裁者或中央实体可以决定限制对基础设施效用的访问。

可信中立

由于无需许可且不可阻挡，超结构是可信中立的。

它们与用户无关，也就是说，它们不歧视或反对任何特定的人，它们会公平地对待每一个人。任何人都可以普遍访问超结构，可以利用、构建并集成到他们自己的系统中，而不存在任何去平台化或排斥的风险。超结构公平地对待每一位参与者，以至于在每个人的能力和需求如此不同的世界中，公平地对待人们是可能的（Buterin，2020）。

正和

作为自由、广阔、不可阻挡、无需许可和可信中立的结果，超结构创造了一个积极的正和环境。

通过永远免费，人们没有动力去一对一地复制相同的功能——因为这样做不会带来额外的价值捕获或风险最小化。在社会层面，一个超结构只需要建造一次。这并不意味着没有创新的动力，人们仍然有动力创造一个具有卓越和差异化功能的替代品。但简单地用相同的工具进行分叉是不会成功的。

通过费用的扩张，超结构允许参与者直接从他们为系统创造的价值中获利，而无需与系统本身竞争。

由于不可阻挡，依赖系统运行的参与者不存在运营风险。他们可以在系统上运行，而无需自己重建和运行它。

由于无需许可，上述所有条件都将保持不变。这再次消除了任何人自己重建系统的需要，只需在系统之上创建并获取你为系统提供的独特价值。

通过可信的中立，没有任何团体或社区被排除在外，这再次意味着如果系统满足你的需求，那就没有理由再去重建。

这意味着你可以拥有一个由潜在竞争参与者组成的丰富生态系统，通过共享流动性和集成的网络效应，在同一基础设施上组织和利用，为每个人带来净利益。

构建一个超结构

随着时间的推移，我会写更多关于构建超结构的细节——我们还在 Zora 学习这一点——但这里有一些值得一提的快速想法：

  1. 超结构是为数百万个接口提供动力，而不仅仅是一个：在协议级别创建机制或任何东西时，确保它尽可能通用。

  2. 将费用作为扩展生态系统的一种方式，而不是从中提取价值：虽然短期内可能获取到价值，但这可能是一种局部优化，它会错过来自协议之上的多样化平台生态系统的更广泛机会。

  3. 采用协议优先的构建方法：专注于开发人员的采用，尽可能多地创建集成——这是一种重要的网络效应，并将超结构巩固为默认设置。

  4. 建立流动性：链上流动性有利于其他进入者加入生态系统，并有利于其他所有人。这是一个关键的网络效应，也降低了分叉的能力/激励。

  5. 尽可能要有所有权，而仅在必要时进行治理：在绝对必要的情况下创建所有权和治理。如果两者中的任何一个都过多，你可能会发现超结构的激励机制会发生扭曲，或者有受到攻击的风险。

  6. 构建周期很长：以这种方式部署协议，更类似于创建硬件而不是软件。设计的周期很长，迁移的摩擦很大，部署的难度很高。

社会基础设施

伊甸园项目：世界上最大的温室拥有超过一百万种植物

鉴于它们（超结构）不可改变和永久的性质，我们有机会创建可以比我们自己的生命更持久的文明基础设施——所有这些都是互联网原生的。

我们从未有过创建基于软件的基础设施的工具，这些基础设施可以像设计的那样代代相传，而不会退化，也不会陷入公地悲剧来维持自身。我们正处在一个世代交替的时刻，在这个时刻，我们有幸建造了第一座超结构。

我认为，对于每一种金融工具，我们都可能看到一种独特的超结构出现：交易所、市场、借贷池、期权等等。对于目前由 web2 和社交媒体平台（如域名、注册管理机构、身份、管理、标签、声誉、表情符号等）运行和运营的非金融公用事业也是如此。整个互联网正在以 Web3 的形式重建，以加密和区块链为核心——我们有机会在私人控制之外重组过去40年来互联网最有价值的功能，并创造以前根本不可能的全新功能。

2010 年的 Facebook 社交图谱

这些基础设施的规模、重要性和雄心壮志最终将以博物馆、电网、运河、水坝和古代道路的方式令人惊叹。超结构的规模将是我们从未见过的，甚至可能比当今的社交媒体规模更大。它们还将根据它们在很长一段时间内对公众的可用性和实用性进行评判。

在设计超结构时，长远的眼光是至关重要的，而在Web3这个追求短期利益的世界中，这通常是缺乏的。

至关重要的是，由于它们不可改变和永恒的性质，我们必须乐观地以最纯洁、最自由和最美丽的形式创造它们。这些超结构很可能被我们的子孙使用，因此我们有责任在我们的头脑中建立它们，我们应该为它们感到骄傲。

未来的孩子，作者Holly Herndon 和 Mat Dryhurst

现在可以创建永远免费运行的超结构，所以我们应该使用这个逻辑作为我们的起点，而不是我们应该避免或忽略的东西。新范式带来了新的价值体系，超结构让我们有机会创造出永远免费的公共产品，同时奖励建设者和参与者创造和贡献这些为整个社会服务多年的宝贵系统—— 这听起来像是值得建立和发现的东西。

我们正在进入未知领域，让我们拥抱它，然后发现什么是可能的。
