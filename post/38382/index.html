Fabric Ventures 很高兴与 Coinbase、EQT 和其他投资者一起投资 Moralis 的 B 轮融资。

在过去十年，提供工具来改进开发者的体验一直是 web2 领域的重要组成部分。通过简化和整合开发者任务以加快生产、质量和部署的需求，创造了一个年收入达到上百亿的市场，并且随着软件继续“吞噬世界”，收入还会保持继续增长。 

开发者体验问题，如果有的话，在 web3 中只会更严重。 web3 的编码语言大多是专门用于这一行业的，并且对期望进入该行业的软件开发人员需要进行一定程度的再培训。此外，web3 中现有开发者的绝对数量仍然很少，这意味着高度熟练的原生用户很少。截至 2021 年 12 月，每月活跃的开发者总数不到 2 万——其中 59% 的开发者当年才进入该领域。

然而越来越多的Web2 开发者进入该领域，这些开发者期望找到让他们工作变得更轻松、更快捷的工具。因此最能够满足这些需求并允许轻松创建 dapp 的供应商将创造出巨大的价值。

Moralis 最适合作为 dapp 开发的必需商店。它为开发者提供了全方位的产品，从教程到易于开发的 SDK，到快速可靠的 Node API（与 Moralis 服务完全集成），再到便于收集各种区块链数据的 Web API （包括 NFT）。打个比方，你可以认为它是 web3 的 Firebase。

尽管公司才成立一年，Moralis 就已展现出强大的吸引力。凭借创始人和 youtube播主Ivan Lijeqvist 的个性魅力和网络活跃度，该公司已经在业内吸引无数眼球，并且每个月都有数以万计的新开发者加入其平台。

对于任何想在 web3 上进行新构建的新团队，提供全方位产品的Moralis 成为他们的首选。他们不仅通过可靠和快速的节点供应来处理后端服务，而且还提供了一个 SDK，极大地减少了开发时间，并允许使用他们的 API 调用相关数据。我们认为，能够提供所有的这些服务对于未来的货币化和价值增值至关重要，并且开发者（尤其是来自 web2 的新开发人员）不会希望与太多独立的产品集成，而宁愿让 Moralis 处理他们所有的后端需要。

对于 Web 和游戏开发者而言，Moralis 为以下语言和框架提供了 SDK：JavaScript、React、Node.js、Unity 和 C#。 Moralis 以缩短87%的平均上市时间而自豪，这是因为 Moralis 通过异步 API 的调用而不必考虑链上特定的实现细节和样板代码。查看代码，可能会注意到它看起来与 Web 2 应用程序类似，API 调用是对 Moralis 服务器进行的，它表示应用程序的前端和区块链之间的连接。

此外，Moralis 在整个开发者社区中获得了出色的评价，我们从现有的投资组合公司中得到了直接参考。 我们非常看重他们API 的质量、跨链支持和所提供的功能套件。生产出色的产品是吸引开发者和产生粘性的最佳方式，我们相信 Moralis 最有可能做到这一点。

我们很高兴通过 Fabric X 计划加深了与 Moralis 的关系。公司各个层面的创始人，从构思到最初的产品市场契合等，一直是 Fabric DNA 不可或缺的一部分，一直以来我们都支持最优质的技术人才在 web3 中发挥作用——这里有如此的多工程师对他们构建的产品感到兴奋，但他们需要时间来真正探索加密领域，不会因为离开之前的工作而有直接的经济压力。注意到在 Web 3 生态系统中开发者和企业家的瓶颈，并意识到他们需要有一个良好的环境来开始他们的 Web 3 之旅，为此我们创建了 Fabric X 计划，并从法国的第一期 Le Crypto Fellowship 开始。我们对 Le Crypto Fellowship 的初步结果感到高兴，并对这么多团队自然而然地将 Moralis 作为他们的首选工具感到震惊。随着我们在不同地区推出更多孵化器，我们期待进一步加强这种关系。
