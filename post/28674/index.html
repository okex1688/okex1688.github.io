2021年9月3日， NFT项目“Loot: for adventures”用了5天时间，就吸引了4600万美元的销售额和超过1.8亿美元的总市值。Loot 于 8 月 27 日由 Dom Hofmann 推出，Dom Hofmann 是视频共享应用程序 Vine 的联合创始人，他也是一个热衷于突破 NFT 应用界限的人**。Loot只是一个有几行字的TXT文本文件。但就是这么简单的NFT项目，引发许多人的争相购买，让人觉得着实不可思议。**

Loot NFT

来源：Loot Project官网

Loot的8行文字写的是8种随机生成的冒险者装备——武器（Weapons）、胸甲（Chest Armor）、头甲（Head Armor）、腰甲（Waist Armor）、脚甲（Foot Armor）、手甲（Hand Armor）、项链（Necklaces）、戒指（Rings）。值得注意的是，这些装备是稀缺性的，且会随机分配。Loot总量为8000个，222个预留给了开发者，现在所有的Loot都已被领取。

每个 Loot NFT 都没有购买的成本，只需要以太坊网络交易的 gas 费。 总共有 8,000 个 NFT在几个小时内全部被认领。许多人现在拥有了一份随机的装备清单，从“巨人的圣胫”到“恐怖的怒吼”坟墓技能魔杖和“倒影之剑”。

而后，Dom Hofmann推出了Synthetic Loot “合成装备”功能，一个智能合约允许任何拥有以太坊钱包的人参与未来与装备相关的工作，而无需拥有一个实际的 NFT。该智能合约确保最终从 Loot 社区中产生的任何项目都不会严格限于原始的 8,000 NFT。

Hofmann在博客上写到：在 Loot 社区上的创作者可以选择将 Synthetic Loot 作为一种允许更广泛的冒险家参与生态系统的方式。同时平台仍然能够区分原始Loot 和 Synthetic Loot。以太坊创始人Vitalik Buterin 在推特上表达了其看法：他赞扬了 Loot 的开源且有效地接受外人在其网络上建设生态。

Vitalik Buterin的推特

我们能看到 Loot NFT 上写着：镭射眼、雪茄、大胡子（仅为举例）等等装备元素，但并没有把画面直接呈现出来。这种“看到了却又没完全看到”的感受能够让人们自发地组成社区，其中最简单的需求便是 Loot NFT 的藏家希望寻找到艺术家将 NFT 的形象画出来。

倘若再进一步，社区中的艺术家、故事写手、开发者等完全可以自发组成一个小团体去创作、发行自己的项目。Loot 对他们的帮助在于建立起了一个社区，社区中的成员们通过 Loot NFT 上面所写的“装备”或者“命题”进行创作，互相认可的人们有机会合作一起推出自己的项目。

总而言之，Loot 是推动者及社区启动的催化剂。

文字都能够NFT化且要价不菲，那么究竟还有哪些新奇NFT呢？实际上，此前也有人炒作虚拟土地NFT；圣马力诺共和国甚至是推出NFT疫苗护照，让接种疫苗的人们能够凭借此来证明接种信息。苏富比拍卖行更是将万维网的源代码NFT化且拍卖出540万美元的高价。

在Decentraland中“炒房”

Decentraland是一个3D的虚拟世界，一个基于以太坊的去中心化的VR平台。用户能够完全控制自己创建的内容环境和应用，其范围可以从任何静态3D场景到具有更多交互功能的应用或游戏。用户可以在这里构建虚拟世界，畅玩游戏，探索NFT艺术博物馆，参加现场音乐会，也可以进行其他社交活动。

Decentraland官网

Decentraland于2015年由Esteban Ordano创立。Ordano是比特币支付供应商BitPay的软件工程师。他因创建了领先的比特币基础设施库Bitcore而倍受币圈的赞赏。2017年8月，Decentraland推出测试版，首次代币发行（ICO）筹集了2000万美元且于2017年12月举行了第一次虚拟土地拍卖。2018年3月Decentraland开设了Decentraland土地市场让地主能够再次买卖土地。2018年6月，开发人员开始在土地上生成作品，该月的LAND销量是大热NFT CryptoKitties销量的两倍。2020年12月，Decentraland项目正式发行。

在Decentraland共享的虚拟世界中，用户可以购买被称为“parcels”的虚拟土地。 土地本身以及在其之上构建的虚拟财产是一种称为LAND的NFT。土地所有者可以控制在其土地上所发布的内容。此内容可以是从场景和图像到游戏和应用程序的任何内容，用户可以以类似于现实世界中的方式探索，旅行和访问场所，该平台也是用户进行交互的点对点网络。

创始人Esteban Ordano在接受采访时表示：“借助Decentraland，我们正在创建一个生态系统，将在加密货币上实现许多解决方案与即将被大量采用的VR系统结合在一起，其结果是一个带有原始经济网络的社会3D世界。 ”他也透露了社区最终的目标是让Decentraland拥有自己的经济系统，人们可以在这里租赁土地给其他人，在产品或服务的广告牌上放置广告，并在游戏内创建聚会场所。

Decentraland 于 2017 年推出测试版时，用户可以以20 美元的价格购买虚拟土地。而如今一个虚拟土地的地板价已经上升到了1.36以太币（5310美元；34000人民币）。

圣马力诺共和国推出NFT疫苗护照

2021年7月2日，圣马力诺共和国（San Marino）宣布发布基于区块链的国家级数字疫苗护照，并采用NFT 技术保障证书的独一无二性。此疫苗护照将成为该国公民出入境的有效凭证，且符合欧盟对于护照隐私保护的要求。

圣马力诺国家的NFT疫苗护照由圣马力诺共和国创新研究院、DNV（一家国际认证公司）、唯链（专注于开发区块链技术的企业）联合开发。该NFT疫苗护照具有高灵活性、高隐私性的特点。该疫苗护照将包含疫苗类型、疫苗计量与批号、接种时间地点等医疗数据，以及包括姓名、出生日期、身份证等在内的个人隐私信息。

圣马力诺共和国发布的NFT疫苗护照样例

对于开创性的NFT疫苗护照，负责该项目三方的高层都发表了各种的看法以及对于未来的展望。

圣马力诺创新机构代表 Lorenzo Spadoni 表示 ：“NFT疫苗护照无疑是重要的工具，可以帮助圣马力诺与欧盟现行标准保持一致，以此做到互通互信，同时也提供了一种 基于区块链技术实现国际‘通用’的新思路 。”

DNV 全球高级副总裁 Renato Grottola则 表示 ：“该护照极其便携且易于使用，不需要下载特定的应用，仅需扫码，任何国家和组织经过持证者同意都能验证证书。唯链提供的区块链技术，特别是 NFT 技术 ，极大提升了各方对信息真实性的信心，大幅降低了伪造的风险。 ”

唯链 CEO 陆扬表示 ：“此前唯链提出 eNFT 概念 ，即企业铸造的商业化 NFT ，强调真实商业场景与 NFT 价值的正向关联。本次唯链与 DNV 合作，帮助圣马力诺共和国发行使用区块链技术的数字疫苗护照就是该理念目前的最佳案例。区块链技术是基于数据隐私实现协同与互信的理想技术手段。基于本次的成功经验，唯链将持续探索多方参与、对数据安全与隐私保护高要求的数字化项目 ，为包括抗疫、公共卫生安全等在内的各个领域提供可靠的区块链技术支持。”

价值540万美元的源代码

2021年6月30日，互联网之父Tim Berners-Lee的万维网源代码NFT在苏富比拍卖行（世界知名拍卖行，以拍卖艺术品及文物而著称）完成拍卖，最终成交价超过540万美元。拍卖所得将归Tim Berners-Lee和其妻子的基金会所有。

该NFT名为《Source Code for the WWW》，铸造于2021年6月15日，起拍价为1000美元。该NFT含有源代码的时间戳原始文件（写于1990年10月3日-1991年8月24日之间），其中有Tim Berners-Lee编写的三种计算机语言和网络协议以及指导早期网络用户如何使用应用程序的原始文档，共计大约9555行代码。除此之外，该NFT还包含了30分25秒的代码动画、Tim Berners-Lee写的一封反映代码和创建过程的信，以及原始文件中完整代码的数字海报。

拍卖的NFT源代码

来源：苏富比拍卖行官网

Tim Berners-Lee是万维网的发明者。他于1989年3月12日首次提出"信息管理系统"，并在同年开发出了世界上第一款Web浏览器。1989年12月，Tim Berners-Lee为这一发明正式命名为“World Wide Web”，也就是此后我们所熟悉的网络前缀www，彻底改变了信息的共享传输方式和创建方式。

然而源代码NFT的拍卖引来了批评人士的质疑，认为这样的拍卖不符合互联网精神。Tim Berners-Lee接受采访时表示：“我并未出售源代码，我出售的只是一张用Python编译的图片，程序是我自己写的，上面有我的签名。如果他们觉得我出售NFT不合适，那我出售书呢？我出售过书，这涉及到金钱，但免费开放的网络，仍然是免费开放的。”

以销售的数量来看，艺术品与收藏品占据了NFT的主流。从转卖率来看，收藏品被转卖最多次。稀缺性使得NFT收藏品在二手市场有着高价格。

此前，Dapper Labs与NBA合作，推出了NBA Top Shot，让球迷可以购买虚拟球星卡以及NBA的精彩片段NFT。Dapper Labs的合作和营销主管Caty Tedman在与Cointelegraph的谈话中表示：“这款产品是为了让球迷们对球场上发生的行为拥有一定的所有权。”

NFT收藏品通过挖掘粉丝的热情从而赋予了NFT收藏品高昂的价格。

此外，NFT在游戏行业中也在蓬勃发展，因为它们允许对游戏中的物品进行NFT化，并可以轻松地与对等交易和市场进行转移或交换。这与禁止出售或转让稀有武器和皮肤等游戏内物品的传统游戏完全不同。由于玩家对自己的数字资产拥有真正的所有权，因此NFT也使游戏体验更加切实和有意义。他们也正在创造一种新的经济，因为玩家现在有潜力通过建造和开发自己的游戏资产来赚钱。

当玩家在游戏中拥有更多的皮肤或装备时，他们的忠诚度就会提高；而当他们的数字资产可以在游戏或平台之间转移，或在公开市场上交易时，他们会将更多的血汗钱投入来交易；当人们能够创造自己的装备或者皮肤再拿去交易，那么会有更多的人热衷于创建游戏NFT。

NFT，带来新的机遇

NFT 为数字业务领导者提供了新的机会。 NFT能够创建新的业务模式、扩展现有产品和服务的价值，例如NBA拓展了收入来源（推出NBA Top Shot）。它们还提供了一种投资新数字资产类别的机会。

音乐团体 Kings of Leon发布了 NFT，包含新专辑发行的数字文件的访问。其中一些 NFT 包含了购买者在未来的Kings of Leon音乐会前排座位的优先权利。NFT创造了一种不同的商业环境、客户（或是粉丝）体验和参与模式，并加强了与用户的品牌共鸣。

NFT 提供了一个探索商业数字化的机会。任何类型的资产都可以数字化。开展业务的条款和条件由NFT制定者设定，而不是由中央中介机构决定，这对于企业来说无疑是个福音。NFT的发行，买卖等都不需要经过政府或是中间人的决定，只需要发布在网络上就可以赚钱。可以说NFT为商业数字化带来了新的机遇。

NFT的风险

随着NFT概念的普及，各类NFT资产类型持续不断的涌现。 许多扎根细分领域的项目已初具规模。但如果NFT项目方突发倒闭或跑路等撤场情况，这必将导致NFT数据元因无法继续被托管、维护而折损甚至消失，没有后续交易支撑，NFT很可能变得一文不值。此外，每个人都能创建NFT也会引来有心人士随意创建NFT，使得欺诈风险大大的提高。

安全方面的隐患。虽然区块链主打安全性，不可篡改其数据。然而不可否认的是区块链技术在各方面还处于早期阶段，各类安全事件频繁发生。2016年的以太坊“The DAO”事件中，由于代码漏洞导致了超过6000万美元的ETH被黑客窃取。随着NFT资产价值的持续上升以及项目体量的扩大，会逐渐成为黑客的下手目标。

当前全球监管环境中NFT未被明确约束。NFT到底是商品还是货币，还有待各国从法律层面进行探讨与定义。 而在此之前，NFT市场势必只能处于无监管状态。由于其匿名性，利用NFT进行洗钱犯罪、非法集资，违禁品交易等犯罪行为极易发生。此外NFT的投机炒作也会迎来政府的监管。
