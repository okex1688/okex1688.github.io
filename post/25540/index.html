毋庸置疑，现在DeFi已经拥有了初步成熟的交易与借贷市场，并且远离了当初只有概念与框架的阶段。

可尽管如此，去中心化交易的资金利用率系统等还有着极大的设计提升空间。

交易与借贷是金融世界的根本组成部分，不过就传统金融系统来说，个人投资者单独的操盘、促成部分其实是非常有限的。

大部分投资者当然更倾向于通过对冲基金、互惠基金等金融管理机构来进行投资。

在TOKEN可以开始在链上进行交易之后，许多基金就开始涌入了市场，为客户提供资产管理服务。虽然如此，但大部分加密资产管理服务还是在链下进行的。

现在，DeFi的各个生态系统正在蒸蒸日上地发展着，用户在平台与服务的选择上有了更多的金融产品和协议可供挑选。

在此基础之上，资产管理项目成为了大家非常关注的一部分，所以今天就带大家来领略一下这个性能强大的DeFi分支板块吧！

为什么选择资产聚合器？

资产聚合器的特征主要有以下这几个点：

1.无监管。 自己的私有财产不需要被转移至平台的监管之下，可以储存在自己的财产里来进行操作。

2.具有可组建性。 许多顶级的资产组合管理平台都与许多不同的DeFi协议连接，因此用户可以做到将DeFi协议接口集合地去组合并使用。

3.协议自动化。 越来越多的资产管理工具现在都是完全基于协议达成自动化的。

程序算法保障了用户，使其无需再频繁地根据市场的大量变化而调整自己的投资组合、流动池，质押mining等，因为这个程序就可以帮你找到最优的解决方案。

4.用户身份匿名化。 运用加密钱包来连接至其协议，说明用户可以保持身份匿名去交易的状态。

5.无地区使用限制。 不管是什么地区，或是地区的什么交易税务政策，利用DeFi这些都不是用户需要考虑的问题，因为DeFi是无国界的。

因此对于入门投资者，或是时间紧缺的投资者来说，运用一些靠谱的资产自动组合管理协议是比较好的选择。

DeFi聚合器集合

也是因为上述这些能使用户极大化跨越DeFi知识门槛，并且使自动收益最大化的特征，资产聚合器现在才能被大家所接纳并且利用，从而引爆市场。

Set Protocol

先来看看Set Protocol（Tokensets），这个平台是DeFi早期发展不可或缺的一部分，其概念即使在现在，也是被广泛运用着的。

“Sets”指的是会保持恒定的一对资产量。 资金若是远离了最初的定量，协议会根据资金量来自动平衡资金。

例如BTCETH50这个Set，不管币价如何波动，BTC与ETH也会一直稳定与其原定的平衡量—50/50。

Set Protocol尽管当时也收获了一些流量，但在研发推出DeFi Pulse指数set之前，还没有推出任何其他比较引人瞩目的金融产品set。

截止到现在为止，DeFi Pulse Index这个set的市值约为1.5亿美金。

在2020下半年COMP引爆市场之后，Set Protocol的表现开始呈现低迷状态。不过单就协议来讲，这个平台对于DeFi新投资用户来说依旧是个极其有利的入口。

狂热的开始

回看历史，Melon和Set Protocol都可以说是DeFi领域最初的金融管理产品。

Melon是由Avantgarde Finance的CEO—Mona ElIsa建立的，创建主旨是为了帮助专业的基金管理者们。

到2020年初，得益于DeFi借贷领域中令人眼花缭乱的利率，Yearn.Finance、Staked的智能投顾RAY等聚合平台看到了机会，也纷纷开始崭露头角。

现在这些大家耳熟能详的收益优化器平台，其实一开始的表现都不温不火，只有在COMP与Yield farming爆发之后才取得了初步的成功。

2020年Compound治理TOKEN币价直线上升

Yearn随后又进一步推出了金库功能，投资者们可以将自己的投资资金存入一个能自动制定理财投资策略的金库中。

与Set Protocol的概念比较相似，但策略会更加复杂，并且针对的是特定的资产。

这是一个特别像专业资金管理的研发项目，不过它更加公开透明化，并且存在于链上。Yearn就是DeFi领域资产管理的初代产物。

Harvest、Rari Capital和APY.Finance都有着相似的资产聚合策略和收益优化方式。Harvest相对来说有着更多中策略方式，不过跟Yearn相比也没有多大区别。

Rari和APY.Finance都是具有极大潜力的收益优化聚合器平台。不同于 投资策略上的选择性，它们推出了风险评价这一指标，可以为投资者提供更多额外信息。

其余聚合器项目介绍

Zapper

Zapper就是聚合器里的龙头协议之一。 Zapper提供的资产管理仪表盘，可以快速地让用户调整与管理自己钱包里散落在多个DeFi协议中的资产，例如Uniswap、Curve等。

同时允许用户查看自己包括钱包余额、投资、收益性mining、存款、债务和流动资金池各个不同DeFi赛道的聚合资产。

且各个赛道资产将以百分比显示，因此DeFi投资理财将通过Zapper协议变得更加简单快捷。

Zapper也可以监控并管理其在多条链（目前支持以太坊、BSC、Polygon）上所有的DeFi资产与负债状况，ROI预测指数可以用于追踪自己各个赛道的资产变动情况。

Zerion

Zerion是一个通用界面平台， 允许用户投资、借贷、兑换和进行与DeFi相关的所有工作，同时保留其资金的全部托管权。

例如，用户可在应用内通过Uniswap交易资产，再通过Maker借入，最后通过Compound v2借出，这一系列的活动都可以通过这一个协议去实现聚合理财。

DeFi Saver

DeFi Saver， 一个非常有利于管理借贷资产的平台，是一个一站式资产管理解决方案， 适用于以太坊上的一系列去中心化DeFi应用程序。

其平台搭建时便考虑到了闪电贷的兼容性，并且有一套平台自动化的金融工具去平衡质押比。

未来需求

随着DeFi的不断成熟，产品层出不穷，我们需要更多聚合工具来帮助大家管理如此广泛遍布的资产。

同时，通过便利实用的收益聚合优化工具让新用户快速上手DeFi，不仅仅代表了方便，也是降低了入门门槛的标准，并帮助DeFi发展的关键。
