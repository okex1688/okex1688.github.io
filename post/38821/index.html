对于互联网世界，在元宇宙之前，最热衷的演进方向其实是第三代互联网——Web 3.0。

Web 1.0 ：最早期的互联网门户网站和 PC 浏览器时代，信息是单向展示和传递的，是一个由网络到用户的过程，用户只能看，不能进行互操作；

Web 2.0 ：动态网站兴起，用户可以通过互联网产生更多的交互和连接，是由用户到用户的过程，尤其打通了互联网社交属性，让现实生活可以映射到互联网，带来了优步、Facebook、Twitter 等巨头。移动互联网也属于 Web 2.0 范畴；

Web 3.0 ：一个终极的、开放的、无需信任、无需权限的理想网络架构，集所有前沿信息技术于一身，消除巨头垄断，创造自由市场，一切以普通用户为核心，自主掌控数据，建设一个民主的互联网。尚未形成较为具体的存在形态和技术架构。

Web2.0 的 S 型曲线

Web 2.0 成就了中心化互联网平台和巨头。以最能代表 Web 2.0 用户规模和估值的电商、媒体和社交平台为例，它们凝聚着巨大的流量和商业模式，但也存在备受诟病的垄断问题和不公平性，引发了今天全世界范围频繁且猛烈的监管干预，巨额的罚款、严厉的批评、增长的停滞，彻底暴露了当今互联网发展的瓶颈和不足。

A16 z 合伙人 Chris Dixon 认为，中心化平台遵循可预测的生命周期——起初他们竭尽所能招募用户和第三方来进行补充，如创作者、开发者和企业，这样做是为了加强他们的网络效应；随着平台呈现 S 曲线发展，它们对用户和第三方的影响力稳步增长；当他们到达 S 曲线的顶部时，他们与网络参与者的关系从正和变为零和，若要继续增长，则需要从用户那里提取数据并与（从前的）合作伙伴竞争。

这段见解一针见血指出了当前 Web 2.0 下互联网平台和巨头面临的发展窘境。互联网巨头们为了巩固或者侵略商业利益，放弃了对新技术的追求，不再投入，更是互建城墙，自我阻碍时代发展的步伐，形成商业气息非常浓厚的传统竞争格局。而人类的进步一定是技术革命的进步，放弃了技术革命，一味追求商业利润，终究是社会的一种退步。

2021 年 10 月 16 日，全球最大的综合性数字发行平台 Steam 宣布清除所有涉及 NFT 的游戏，更是在新手入门页面中更新指出——不应在 Steam 上发布的应用程序包括“基于区块链技术构建的可发行或允许交换加密货币/NFT 的应用程序”。作为一个引领人类游戏文明的超级平台竟然可以随意封杀一种崭新的游戏模式，只因为这种带有数字资产的游戏可能会对 Steam 原有的商业模式带来冲击，影响到自身的利益。当然也有观点认为之所以会下架所有区块链游戏是因为 Steam 认为这些游戏内物品具有价值，而他们不允许其平台上具有现实价值的物品，或者在预防炒作。不论怎样，即便是加密行业从业者都不能轻易给 NFT 游戏定性，因为它存在很多革命性意义，值得被不断探索，而中心化巨头并没有对此类创新技术和模式进行过积极探索，便妄下结论，随意宣判新事物的死刑，也违背了很多用户的意愿。

于是，开放和互通开始成为新的思想启蒙。即便对某个巨头自身来讲，拥有着非常多的垄断资源，涉及全方面的互联网服务，它自身的体系也存在一种互通的建设需求，希望可以通过一个更上层的超级平台对自有的商业体系形成一种凝聚和升级，以创造更高的效率和增量。

最终 Web 3.0 被提上日程。为了解决中心化发展的弊端和瓶颈，Web 3 开始倡导去中心化理念，但尚未对该理念进行明确化的技术定义和实践，只是从最顶层认为应该将互联网的所有权和控制权重新交给所有的用户，让用户可以自主掌控数据、资产、服务，这是互联网对第三代网络的基本认知。正如资深加密人 Ki Chong Tran 撰文写道：“在 Web 3 下，网络是去中心化的，因此没有任何实体可以控制它，建立在其上的去中心化应用（简称 dapps）也是开放的。去中心化 Web 的开放性也意味着没有任何一方可以控制数据或者限制访问。任何人都能在不经过中心化公司允许的情况下构建和连接不同的 dapps。”

Web3.0 在塑造一种公共的力量

Web 2.0 中用户已经被迫习惯了将内容、数据、账户都存储在了中心化的应用和平台中，用户不会轻易地转移平台，否则要重新建立一套繁琐的操作，这便从根本上造成了互联网应用服务间的割裂，且用户权限也是互相孤立的。例如，应用程序之间的互访需要经过繁琐的申请、审批，甚至大型应用之间还互相禁止。此外，Web 2 时代诞生的巨头牢牢将商业利润攥在自己手中，而不是与所有平台用户分成。例如 Roblox 社区的开发者只能分得系统收入的 27% 份额，其余大部分都归平台所有，于是 Roblox 公司仅在 2021 年第二季度就赚了约 5 亿美元，讽刺的是在 120 万开发者群体中只有 1000 名能够获得 3 万美元以上的收入。Web 2 时代造就的公司跟传统商业模式一样，不断从生态系统中榨取越来越多的价值，而不会全部或者大部分反馈给平台的用户。Web 2.0 今天已经将面貌彻底呈现完整，虽然是互联网“万物互联”精神的产物，但终究还是发展成了一个个大型孤岛，所谓的万物互联也仅限于大型孤岛内部的“条条大路通罗马”。

Web 3.0 希望为数据、内容、身份账户甚至包括愈发明朗的资产创造一个公共的开放空间或者平台，不论存在多少独立发展的应用服务程序，它们将不会再拥有用户的一切资源（用户也不会再被迫去选择它们来留存资源）。在提供服务时，应用层都需要接入到这样一种专属于用户的公共空间里，并在征得用户同意的基础上，获取相关生产资料和数据，为用户提供互联网服务。但最终也不能留存用户数据和资产，做到公开透明。应用程序之间在没有了数据和用户壁垒后，会自然地相互开放，互通有无，这便是 Web 3 的一种终极构想或者博弈设计。当消除了中心化商业平台，生态系统所产生的价值也都会以去中心化的形式分散给所有用户，让真正的建设者们享受到生态系统发展的红利。

与此同时，以太坊世界计算机和其他更加超前的底层公链也在引领区块链世界的 Web 3.0 进程，与互联网世界稍有不同，加密行业对 Web 3.0 的建设更为具体一些。其实基于 Web 3.0 对去中心化的核心诉求，以太坊等公有区块链上的基于智能合约的应用都可以称得上 Web 3 产物。但鉴于大多都是未获得商业成功和市场规模验证的尝试，尚不足以代表 Web 3。直到去中心化金融 DeFi 的兴起和壮大，让真正可以实现去中心化运转的产品照进现实，在区块链之上完全按照金融和数学理论建立起了一个数千亿美元量级的新型虚拟经济体，并将权力还归于所有用户，也极大保障了所有用户的隐私、数据、资产等权益。例如，用户可以在 Uniswap 协议上享受到完全去中心化的资产交易服务，自主掌控账户，自主托管资产，不存在平台挪用用户资产的问题，当你想交易的时候，随时将自己的钱包连接到 Uniswap 协议中，完成即刻的交易体验；用户还可以将自己的波动性资产交换为一种去中心化的稳定币，它不会被任何中心机构以各种缘由没收、冻结和转移，是真正属于用户自己的有价值支撑的虚拟资产，代表了人类对自主权利的最高掌控。

但目前也仅限于金融方向（包括这两年最火热的 NFT）的成功实施，而 Web 3 的范畴和定义其实是整个互联网，只有在区块链上建立起超越金融的、更加广泛的去中心化互联网服务，才能真正找到区块链所定义的 Web 3 技术范式。因此，加密行业从未停止这个追求，试图让区块链在 Web 3 时代拥有核心的位置。今天，当元宇宙席卷两大世界时，人们惊讶地发现元宇宙正是对 Web 3 的继承和发扬，正如全球最大的会计师事务所德勤在《The Spatial Web and Web 3.0》报告中指出，在 Web 3.0 时代，与 3 D 空间中的现实世界交互的虚拟空间的形式将会扩展。只不过相较于 Web 3 的形而上，元宇宙更具象，也多了更多实践。

Web3.0 工具箱

区块链投资机构 Outlier Ventures 创始人 & CEO Jamie Burke 在《Depth is what leads to Metaverse Operating system of？》一文中将 Web 3 的详细技术构建归纳为 Web 3 工具箱。站在区块链和去中心化的核心视角，Web 3 工具箱自下而上依次为：点对点网络、交易层、可编程层、去中心化应用层、代理、钱包等六大层。

Web3 工具箱

点对点网络：即 P2 P 网络，是当今区块链架构中的网络底层，具有去中心化属性，全分布 P2 P 节点可以自由加入、退出，并且没有中心节点。

交易层：区块链网络的所有活动都是以交易的形式发送、共识和最终确认的，交易是一切事件表达的最基础原语。

可编程层：以智能合约为核心，让通过交易表达事件具有可编程性，图灵完备的智能合约编程语言几乎可以将任何服务变为链上程序。

去中心化应用层：可以进一步细分为去中心化金融（DeFi）、NFT、去中心化治理（DAO、staking）、去中心化云服务（存储、计算、数据库、查询 & API）、去中心化身份（DID）。

代理层：为钱包和应用之间架起桥梁，方便钱包对各种去中心化应用的集成。而对于用户来说，面对如此庞杂的 Web 3 工具箱，仅需要知道和使用钱包层即可。

通过 Web 3 工具箱可以进一步构造出一种开放元宇宙操作系统，架构自下而上分别为：硬件基础设施（计算和存储）、软件后端（计算和存储）、世界原语和规则、虚拟资产、终端硬件、终端软件。我们尝试去理解一下这种完全基于去中心化技术构建的开放元宇宙操作系统的核心构成。这里的硬件基础设施和软件后端都特制去中心化技术，如区块链核心系统、分布式存储等。核心要素聚集在中间的世界原语和规则、虚拟资产两层，它们负责桥接数字世界和物理世界，主要通过 NFT 来映射物理世界。在世界原语和规则这一层，有对用户身份、所有权和声誉的集成，有对可编程性智能合约的集成，定义了构建一切去中心化活动的基础原语，在其之上便可以开展虚拟资产层的宏伟建设。虚拟资产层大致分为三类：物理与虚拟的映射（如空间建模、虚拟化身）、经济（如数字货币、金融基础设施、交易市场）、内容（如媒体、数据资产）。

干起来吧

不论是 Web 3 还是元宇宙，最重要的是要明确建立一种什么样的开放体系，可以调动起全球的开发者力量，像建设曾经的 Linux 系统和今天的以太坊世界计算机一样，也像今日中国在建设的鸿蒙 OS 技术体系和相关应用生态的步伐一样。

Linux 诞生于互联网时代的初期，改变了 Web 2 的开发范式，提供了一个全世界范围协作的清晰案例。如果说 Microsoft 代表了中心化阵营，而 Linux 则代表了那个时代的去中心化阵营，最初的 Linux 是由一群利用网络协作的志愿者程序员、极客、黑客、天才、爱好者开发的。这群自由人士充满了理想和激情，不需要通过中心组织来维系联系，最直接的语言便是 Git 和代码。

Eric S. Raymond 在《大教堂和集市》（The Cathedral And The Bazaar）一书中提到早期操作系统发展面临的不合理性，跟今天互联网似乎一脉相承——在那时，以传统的企业和金融机构为代表的“大教堂”控制着操作系统的话语权，喜欢按照自己的商业意图开发复杂和封闭的 OS，而且将产品发布周期掌控在自己手中；全世界范围内掌握顶级信息技术能力的极客群体不满于压迫，开始联合起来，形成开源社区，摸索出了一套公平高效的去中心化代码协作体系，通过协作将创新扩展到与用户复杂性相匹配的程度，正如作者所描写到：“代码质量的保证不是通过严格的标准或独裁，而是通过每周发布并在几天内从数百名用户那里获得反馈的原生简单策略，对于开发者引入的突变，创造出一种达尔文式的物竞天择现象。令几乎所有人感到惊讶的是，这种策略非常有效。”这种新型的协作也被称之为“集市”模型。这个体系一运行便是几十年的光阴，最终将 Linux 送上王者地位，让全世界的人享受到了没有商业垄断的开源之光，这是全人类共同的财富。今天的 Web 3 也是在以类似的“集市”模式向根深蒂固的 Web 2“大教堂”发起冲击，借助更为广阔的去中心化运动彻底完成网络民权自由。

开源运动与 Web3.0，来源：《Web3 会让开源的未来更好吗？》作者：Joey DeBruin

同样的传奇也在区块链世界精彩上演。比特币便是这种对抗“大教堂”世界并以去中心化协作形式诞生的革命性技术，不过它在原有的协作体系之上引入了两层意义更为深远的创新——加密协作和加密资产激励。

比特币带来的区块链协作范式更加注重通过开源密码学保障开发群体在网络上的自由和权益，在密码学保障下，去中心化协作更加去中心化，不会被任何中心组织通过垄断网络资源或者实施线上审查线下打击所中止，极客们首次在网络空间中享受到了一种极致的自由，开始淋漓尽致地进行天才创造。加密资产的到来真正让去中心化协作摆脱了现实世界经济来源的束缚，而在这之前，任何开源协作的开发者都需要在现实世界中赚钱维持生计。有人因此只是兼职进行开源运动，无法全身心投入，开源组织也被迫引入中心化世界的薪酬体系和收入来源以支付给核心开发者工资，因此再去中心化的协作也不得不在经济上被中心化世界扼制。

加密资产诞生于去中心化的世界，并且在去中心化世界得到价值认同，所有开发者都可以将自己的劳动和创造转化为远超现实世界薪资水准的报酬，可以在现实世界有需求时转换为法币以维持和改善生活。而且随着区块链 PoS（权益证明）经济模型的普及，不仅是开发者，所有网络参与用户都可以通过持有加密货币来运行特定网络的验证者节点，去中心化协作从代码开发扩展到网络实体的构建和运转上。所有协作者都被激励通过锁定资产来保护网络，并以网络原生资产的形式获得奖励。良好的行为会获得奖励，而恶意的行为也会遭到去中心化惩罚。网络因此可以安全地蓬勃发展，不断扩大自己的规模，而无需中心化控制。

只有在更为具体的技术框架和技术标准确定下，而不是只有一种口号或者精神，才能集所有力量于一点，建立起初代的底层平台。全世界也将以此平台和技术架构为核心纽带开展更上层的全生态级别的繁荣生长，相关规律可以参考以太坊推动去中心化金融体系的繁荣。最终才会逐渐把元宇宙的真正面貌和定义呈现在世人面前。

Web 3 和元宇宙都在追求去中心化的开放性特征，去中心化潮流也是大势所趋。即便是在 Web 2.0 背景下，在一些中心化的巨头平台上也开始出现局部性的去中心化变革进程，例如内容的去中心化。微信公众号、今日头条、抖音短视频等自媒体平台拥有今天的繁荣便是去中心化 UGC 的结果，全民参与去中心化内容创作与发布，中心化平台只起到监管、审查和引导正确价值观的作用。其实在早期的互联网进程中，也存在类似思想，如 Wikipedia 和 GitHub。所以在互联网发展的任何阶段，人们从未停止对去中心化、分布式协作的探索与实践，而这一伟大的进程将在 Web 3 和元宇宙中彻底爆发。不论是古典互联网元宇宙还是加密世界元宇宙，各自都有在去中心化方面的建树和独特优势，符合 Web 3 开放性的大趋势。
