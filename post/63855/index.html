原文作者：Letícia Melo

原文编译：Luccy，BlockBeats

编者按：上周末，一款名为「Fren Pet」的宠物养成类链游得到了 Coinbase 上的 Fren Pet 宠物总交易量已超 18 ETH。据数据显示，Fren Pet 的游戏代币在一个周末内一度涨幅超 300% ，其代币涨幅在当日的 Base 生态中排名第一。

先是 Friend.Tech，后是「FT 版链游」Fren Pet，基于 Base 链上的活跃度每周都在增长，Dapp Radar 内容编辑 Letícia Melo 梳理了 Base 链上值得关注的 8 个 dApp。BlockBeats 提醒，以下信息根据原文编译整理，交易存在风险，请及时关注项目方所发布的信息更新。

自 2023 年 8 月推出以来，Coinbase 的 Layer-2 区块链已成为拥有着广阔前景的 dApp 所在地。在下面的文章中了解 Base 网络上一些最有趣的热门项目，从游戏到 Base 独有的 NFT 和 DeFi dApp。

值得了解的热门基础 dApp

想在这个快节奏的领域寻找下一个机会？立即了解自 2023 年网络向公众开放以来，Base 上最受 Web3 社区关注的项目。

Friend.Tech

Friend.Tech 是一款引起轰动的社交网络和 Web3，它创新了在线互动的方式。在这个 SocialFi 平台上，用户可以购买和出售独特的密钥，从而授予访问私人聊天和独家内容的权限。通过动态定价模型，每个 key 的价值随着需求的增加而增加，从而增加了引人注目的、类似游戏的元素。

尽管媒体反应不一，Friend.Tech 在 Web3 爱好者中占据了一席之地。其「 3, 3 」策略（10% 的交易税中一半有利于内容创作者，另一半有利于平台）让社区保持活跃。即将进行空投的传言也让人们更加兴奋。

Base Name Service

Base Name Service 提供到 Web3 的无缝过渡，在 Coinbase 的 L2 区块链上提供用户友好的「.base」域名。

该服务使用户能够用简单的 web3 用户名替换复杂的钱包地址，从而简化了 Base 上现在的在线活动。Base Name Service 专注于构建无钱包和无 Gas 体验，旨在成为下一个 10 亿用户进入 Web3 世界的门户。

Fren Pet

Fren Pet 是本周 Base 上增加的活动的主角，它以 Web3 的形式重现了电子宠物蛋的魅力。在这个游戏 dApp 上，用户创造像素艺术宠物，通过喂养和清洁来照顾它们。

FrenPet 的独特之处在于，宠物可以通过使用 FP 代币购买必需品来赚取 ETH。虽然目前宠物铸造已停止，但 OpenSea 上的二级市场收购仍然是可能的。该游戏包括与其他宠物「战斗」以获取成长和积分，以及迷你游戏和即将推出的功能，例如宠物之家。

若要参与其中，用户可将 Fren Pet 添加到移动主屏幕，设置一个钱包，并将 Base ETH 兑换为 FP，深入研究这种怀旧与区块链创新的迷人融合。

Base Swap

Base Swap 成为 Base 区块链上领先的原生 DEX。在其短暂的生命周期中，该平台迅速获得了关注，并成为 TVL 方面的主要 DeFi 项目之一。

它以极具吸引力的收益率脱颖而出，为 cbETH、ETH 和 USDC 等加密资产提供超过 150% 的收益率。DeFi dApp 为用户提供了参与手动或自动集中流动性 (CL) 策略的选项，以最大限度地提高费用收入，同时最大限度地减少无常损失。

BaseSwap 最近推出了 FrenPet 农场，这进一步增加了它的吸引力。任何人都可以使用他们的 FP 代币并通过新功能赚取 BSX 和 xBSX 代币。

Cambria Duel Arena

另一款值得关注的游戏是《Cambria Duel Arena》。它以惊心动魄的游戏模式引入了高风险的决斗。玩家以 Base ETH、代币、NFT 等为赌注，然后在一场激烈的生死战中决出胜负，且赢者通吃。

在周末的内测中，早期测试者可以为即将推出的完整游戏《Degen Wars》赢得精华点数和影响力。

这个竞技场只是 Cambria 游戏引擎功能的一瞥，承诺在未来的更新中带来更多兴奋和参与。

Base Paint

Base Paint 邀请艺术家和爱好者加入像素艺术协作平台，在这里创造力与回报相遇。艺术家们创造出独特的画笔，为不断发展的画布做出贡献。

一旦画布的时间结束，它就会被铸造，所有艺术家都会分享收益。这种数字艺术方法培育了一个社区，其中每个像素都是迈向集体杰作的一步。

New Throne

New Throne 将 DeFi 和 GameFi 融合在 Base 上的 PvP 体验中。玩家使用 WETH 代币建立军队并参与战略战斗以从敌人那里夺取奖励。

该游戏提供了各种部队、防御和升级，允许玩家定制策略以获得最大的投资回报率。凭借 THRO 代币和诱人的 NFT 奖励，NewThrone 邀请玩家提升半神地位并征服新领域。

Aerodrome Finance

Aerodrome Finance 是下一代 AMM，旨在充当 Base 区块链上的主要流动性中心。它将强大的流动性激励引擎与投票锁定治理模型相结合，旨在增强用户体验。Aerodrome 借鉴了 Velodrome V2（其 Optimism 版本）建立的功能，旨在将这些功能合并并适应自己的框架。

在其生态系统中，Aerodrome 使用具有双重目的的 NFT。它们允许持有者对代币排放的分配进行投票，并使他们能够获得协议产生的激励和费用。

为什么要在 Base 上构建 dApp？

Base 将以太坊的安全功能与 Optimism OP Stack 的功能相结合。因此，许多 dApp 开发人员将其视为创建项目的绝佳选择。

Base 上的 dApp 在这个超级链的广阔生态系统（具有共享功能的第 2 层链的联合体）中相互流畅集成。

此外，Base 隶属于 Coinbase 旗下，每当该链通过 Coinbase 官方应用程序可用时，开发人员就有可能接触到数百万加密货币投资者。
