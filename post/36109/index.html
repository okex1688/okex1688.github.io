在web3，相信让人们牢牢控制自己的数字身份是 Web3 通过网络赋予用户权力的承诺的核心。Polygon一直在幕后努力实现这一承诺，今天介绍的是Polygon ID，这是互联网下一次迭代的自我主权、去中心化和私人身份。

这个新身份平台的独特之处在于它首次由零知识 (ZK) 密码学提供支持，这是一种隐私和区块链扩展技术。Polygon 已将 ZK 作为其战略愿景的核心，并已向相关项目投入 10 亿美元。Polygon ID 是这个快速增长的产品组合中的最新产品。

Polygon ID 具有以下属性：

  • 用于分散和自主权模型的基于区块链的 ID

  • 零知识原生协议，保障终极用户隐私

  • 可扩展的私有链上验证，以促进去中心化应用程序和去中心化金融

  • 对现有标准和生态系统开发开放

Polygon ID 利用Iden3协议和Circom ZK 工具包。展望未来，这两个项目都将由 Polygon 赞助，同时保持社区倡议的原始精神，为更广泛的开发人员生态系统提供开源协议和工具。

“Polygon ID 默认是私有的，提供链上验证和无许可证明。Polygon 的联合创始人 Mihailo Bjelic 说，现在数字身份领域没有任何东西可以满足所有这些条件。“这也是零知识证明如何帮助我们创造更美好世界的绝佳展示。”

愿景和指导原则

默认私有身份是面向未来的，也是满足世界数字身份需求的唯一解决方案：更强的隐私保证意味着更多可能的用例，这反过来又会导致更多用户加入。

去中心化和自主权原则：让用户控制他们的身份和个人数据是实现社会协调和从第三方手中夺回权力的一种方式。

以人为本的网络（dWeb-of-Trust）的计算信任/声誉：身份属性可以表示为声明（或证明），可以组合起来创建复合证明。

无需通过第三方即可从一个身份向另一个身份公开且无需许可地发布声明：身份不是被动的。机构正在向个人用户提供，而不仅仅是组织，从而启用新的用例。

区块链原生身份和链上验证

可表达声明标准提供了优于不可替代令牌 (NFT) 和可验证凭证 (VC) 的优势。NFT 不是私有的，而且铸造成本很高。虽然 VC 通过选择性披露和 ZK 插件提供一定程度的隐私，但它们的局限性在于应用程序所需的可表达性和可组合性。验证链上的 VC 非常昂贵。

Polygon ID 利用内部专业知识来降低使用 Circom 2.0 编译称为 zkSNARKs 电路的零知识密码结构所带来的复杂性。

开发人员和合作伙伴的入职是通过 ID 客户端工具包完成的，该工具包将包括原生应用程序、SDK 和白标解决方案。

链上私有验证是使用 zkProof 请求语言完成的，这是一种独特的协议，应用程序可以指定用户需要证明的请求私有属性。

Relayer 是一种独特的模型，它通过允许任何身份发出声明来促进 Web3 应用程序。它有助于降低和管理用户索赔的成本，改善隐私并通过用户身份的赞助模型消除复杂性。

关键独特价值主张和优势

对于最终用户：方便所有人的隐私

隐私是一项基本人权，Polygon ID 授权用户收回它。因为默认情况下它是私有的，所以访问控制是基于证明可验证信息而不是与验证者共享它。

  • 仍然可以在用户设备上运行的最高级隐私

  • 使用匿名用户访问应用程序的权利

  • 符合 Web3 隐私精神

对于 Web3 协议：高级和私有链上验证

Polygon ID 允许构建新形式的声誉。一些例子包括 DeFi 中金融原语和社会支付的去中心化信用评分；DAO 的去中心化人物评分、投票权/授权和领域专业知识声誉，以实现新的决策和治理模型；Web3 游戏的玩家声誉档案；用于社交应用的私密且抗审查的 P2P 通信和交互。

  • 可以直接在链上以保护隐私的方式对身份声誉进行加密验证，以触发去信任的执行/操作

  • 无需依赖中间人来执行与用户的交互

  • 能够通过与通用智能合约或 NFT 交互来组成验证，但具有隐私性

对于组织和企业：信任管理的开放生态系统

Polygon ID 是一个完整的平台，可用于构建各种身份和信任服务。该团队正在为信任市场和信任管理创建一个开放且企业就绪的生态系统，以构建具有激励层的新证明和访问服务。

  • d访问即服务

  • 可以部署现有解决方案并创建新解决方案的环境

  • KYC、KYB、认证

  • 具有多种选择以利用新兴加密系统的分销渠道

产品路线图

Polygon ID 目前正在开发中。该解决方案将构建为一组工具、平台服务和示例，供开发人员学习、测试并与他们的应用程序或 dApps 集成，利用 Polygon ID 独特的链上功能。

产品组合（专注于简单性和包容性）如下：

  • Polygon ID Wallet 应用程序 - 一组开源用户和开发人员工具包，用于集成或白标自定义应用程序。

  • Polygon ID 平台——一个开放的公共平台，开发人员和组织可以使用它来定义和管理其应用程序的信任生命周期和其他要求，例如 ZKP、信任源等。

  • Polygon ID Connect - 公共服务平台，用于集成跨用户钱包和应用程序的访问。

预计交货计划：

  • 2022 年第一季度 - Polygon DAO 的 PoC。

  • 2022 年第 2 季度 - 公开发布 ID 钱包应用程序和 SDK，以支持基本功能，如声明发行、私人身份验证、zkProof 生成和验证。

  • 2022 年第三季度 - 完整发布 ID 平台和 SDK，供开发人员实施自定义用例，包括 zkQuery 语言、P2P 通信协议和中继服务。
