2021 年，Chris Dixon 写了一篇关于创作者如何使用 NFT 将他们的“1000 名真正的粉丝”货币化的文章。从那时到现在，NFT 的用例已成倍增加。

举个例子：NFT 会员资格的兴起，要求粉丝持有特定的 NFT 才能加入私人社区并享受其福利。NFT 会员资格为创作者提供了一种加强其品牌的方式，建立有价值的社区，并让粉丝在社区发展中分享收益。

通过在 Twitch 和 Facebook 致力于创作者增长工作，以及撰写关于创作者经济的文章，我相信 NFT 会员资格将成为与广告和订阅相媲美的创作者盈利渠道。根据现实世界的例子，我分析了为什么创作者可能会考虑 NFT 会员资格，它们在实践中是如何使用的，以及创作者经济和 web3 创业公司如何利用它们日益增长的人气。

NFT 会员资格的优势

虽然所有的 NFT 都是存储在区块链上的数字收藏品，但不同的类型可以满足不同的目的。例如，人们经常购买个人资料图片 NFT（PFPs），作为头像使用。

然而，有了 NFT 会员资格，NFT 就是一张专属团体的入场券。想想看：Soho House 或乡村俱乐部，但会员资格首先是在线的，进入取决于你是否是代币持有者，而不是有合适的关系。

越来越多的创作者通过为他们的粉丝建立 NFT 社区来涉足 web3。对于有商业头脑的创作者来说，NFT 会员资格可以在几个方面取代并补充传统的订阅和基于广告的收入模式：

创作者可以通过细分粉丝的付费意愿，从 NFT 中获得更多收益。 在 YouTube 等传统平台上，创作者使用广告锁定不会为内容付费的休闲粉丝，并使用订阅来定位愿意支付少量费用的粉丝。结果，他们从真正的粉丝那里得到了更多的钱。NFTs 通过针对各个级别的粉丝，让创作者获得更多收益。创作者既可以以他们最大的粉丝为目标，推出限量版 NFT 系列，也可以推出后续系列，以瞄准更休闲的粉丝。

创作者可以通过降低中间商费用从 NFTs 中获得更多收益。 传统平台会从创作者订阅和广告中收入抽取的很大一部分，并且可以在未经创作者许可的情况下随时更改政策。使用 NFT，创作者可以保留他们从初始铸币中获得的大部分收入（请参阅此 NFT 铸币指南），然后还可以从版税中获得额外收入。

创作者可以有意义地承认早期的粉丝群体。 在传统平台上，创作者的早期支持者不会因为创作者变得更受欢迎而获得任何回报。当然，有人可以说他们在 YouTuber 成名之前就发现了他们——但这很难证明，而且早期粉丝的身份也不可能带来实际的好处。

另一方面，NFTs 为粉丝提供了一种明确展示他们作为创作者早期粉丝的方式。创作者可以奖励这些早期粉丝，例如参加活动和特别发行。此外，创作者的早期收藏品（从音乐家到运动员）往往会根据创作者的成功而升值。通过这种方式，NFT 直接激励粉丝帮助创作者取得成功。 

NFT 会员资格的缺点

对于创作者来说，NFT会员资格也有潜在的困难之处需要注意。

创作者无法直接控制 NFT 铸造后的价格。 尽管创作者可以设定初始 NFT 铸币价格和他们在二级版税中的份额，但他们对未来 NFT 价格的控制力较弱。例如，如果整个 NFT 市场崩溃，创作者最终从 NFT 中获得的收入可能会少于传统渠道。

另外，投机者可以购买创作者的 NFT 并抬高价格，而不给真正的粉丝参与的机会。为了防止后一个问题，创作者可以在投机者了解该项目之前，尽早向粉丝推广 NFT “允许列表”位置。他们还可以制定规则，禁止任何一个人一次购买过多的 NFT。

如果 NFT 的价值不明确，创作者可能会疏远粉丝。 创作者需要明确表达其 NFT 会员资格的价值，以免让粉丝失望。粉丝要求创作者信守承诺，并期望他们为 NFT 会员提供长期效用。而在没有明确计划如何为会员提供价值的情况下启动 NFT 会员资格会使创作者的声誉面临风险。

管理 NFT 会员社区是一项艰巨的工作。 社区管理是创作者现有内容制作计划之上的一项重要承诺。为了帮助减轻他们的工作量，创作者可以聘请一个团队来保持社区的参与，并鼓励粉丝在提升会员价值方面发挥作用。但是，他们不太可能将参与职责完全外包出去——毕竟，创作者是他们品牌的核心。

创作者如何在实践中使用 NFT 会员资格

获得 NFT 会员资格的最低要求是什么？成功的社区通常包含三个组成部分：

1.创作者已经有一个参与的粉丝群，可以邀请他们购买其 NFTs。

2.粉丝购买 NFT 以加入社区并帮助增加会员价值。

3.创作者及其社区以现实世界的利益形式为会员提供效用。

让我们看看三个能够很好地说明这一模式的 NFT 会员社区的例子。

VeeFriends 是 Gary Vaynerchuk 创建的 NFT 系列，他是预订平台 Resy 和数字机构 VaynerMedia 的联合创始人，同时也是该机构的首席执行官。

  • 创作者： 凭借他的企业家信誉， Gary 建立了一个庞大的、多平台的受众群体。

  • 社区： Gary 通过教育使他的听众加入 VeeFriends 社区，了解为什么 NFT 很重要以及如何首次购买。他为他的 NFT 收藏中的所有角色绘制并创造了背景故事，并将代币与个人联系起来。

  • 实用性： 所有 VeeFriends 持有者均可免费获得参加 VeeCon 会议的三年通行证。首届 VeeCon 于 2022 年 5 月举行，邀请了 Snoop Dogg 和 Beeple 等重量级发言人。VeeFriends 持有者在组织活动中发挥了关键作用。

Moonbirds ，来自 Kevin Rose 的私人社区 Proof Collective 的第二个 NFT 系列。 

  • 创作者： Kevin 主持了流行的加密播客 Modern Finance。

  • 社区： Proof 的第一个NFT 系列有 1000 个通行证。许多收听 Kevin 播客的顶级 NFT 收藏家购买了这些通行证，这为社区注入了人才，并为 Moonbirds 创造了潜在的需求。

  • 实用性： Moonbirds 持有者被激励长期持有他们的 NFT 以解锁独家奖励，例如商品和现实生活中的活动。

Developer DAO‌，是由开发者 Nader Dabit 在离开 AWS 专注于 web3 后推出的 DAO。

  • 创作者： Nader Dabit 在他的 YouTube 频道上建立了观众群，该频道专注于开发者教育。然后，他启动了 Developer DAO，以加速 web3 建设者的教育和影响。

  • 社区： 开发者和其他人才购买了 Developer DAO NFT，放大了会员的吸引力。

  • 实用性： 访问社区本身已成为 NFT 的主要优势。成员们互相帮助举办活动、建设项目和寻找工作。 

以下是说明 NFT 会员项目如何提供长期效用的其他示例：

  • 访问创作者和社区——ZenAcademy： 持有者可以进入 NFT 爱好者的私人社区，并与一位顶级NFT收藏家和教育家 Zeneca 度过工作时间。

  • 知识产权（IP）——Bored Ape Yacht Club： 持有者可以将他们的 BAYC 用于其他目的并卖出。例如一些成员已经建立了衍生项目，如工作室 Jenkins the Valet。

  • 现实生活中的事件——CPG Club： 持有者可以进入 Web3 运营商的社区，定期计划现实生活中的聚会，以及其他福利。

  • 现实生活中的会员和治理——LinksDAO： 持有者可以获得高尔夫商品和私人高尔夫俱乐部会员资格。他们也有权对 DAO 的发展方向进行投票。

  • 教育——CuriousAddys： 持有者可以获得在线 NFT 课程、教育游戏和进入学习社区。

  • 加速器登记——Meta Angels： 持有者可以优先接触 Angel Labs（一个为 NFT 项目创始人提供的 10 周加速器项目）推出的项目。

  • Merch 和 NFT 空投——World of Women： 持有者有机会获得由女性艺术家空投的额外 NFTs 和艺术版画，以及其他福利。

公司如何帮助创作者管理 NFT 会员资格

创作者和社区正在以新的方式尝试使用 NFT 会员资格。随着这一趋势的升温，创作者从 web2 跳转到 web3 的整个用户旅程中都将需要帮助。为了支持创作者，初创企业可以：

创建 NFT 会员资格

  • 定义会员详细信息（例如，目标、受众、数量、价格、福利）。

  • 招募一个团队或选择无代码解决方案来制作艺术品、智能合约和网站。 

向粉丝介绍会员资格

  • 建立社区并告诉粉丝为什么 NFT 会员资格很有价值。

  • 使用允许列表为粉丝和投机者预留 NFT。

与社区合作，为 NFT 会员创造价值

  • 了解哪些持有者对社区贡献最大。

  • 与持有者合作，为 NFT 会员提供持续的效用。

管理业务

  • 管理并找到为财库提供资金的方法，例如通过合作伙伴关系。

  • 管理税收和法律合同。

目前，许多公司已经在解决上述挑战。我很高兴看到更多的创作者和社区使用 NFT 作为创造价值的工具，并分享上升的收益。
