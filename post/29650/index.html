NFT 是泡沫吗？自从以太坊区块链上这一突破性应用的第一个样本吸引了加密货币主管和非加密货币粉丝的想象力以来，这一直是一个亟待解决的问题。

在这篇研究文章中，我们将尽可能明确地解开这个疑问。还有什么比定义本身更好的起点。 

2021年—NFT 元年

NFT（Non-Fungible Tokens 的缩写）并不是什么新鲜事物。以每个表示都是独一无二的方式标记资产的想法已经存在了很长一段时间。NFT 起源于 2013 年的彩色硬币，它是比特币硬币的彩色代表。然而，这些代币存在技术缺陷，只是一个实验。然而，未来革命的基础已经奠定。

2017 年 6 月，Matt Hall 和 John Watkinson 在以太坊区块链上创建了 10,000 个独特的角色。这些字符的数量永远不会改变，并且没有两个项目是相同的。这两位创作者让任何可以访问 Etherum 网络的人免费声称他们的创作，CryptoPunks 正式诞生。 

几个月后，CryptoKitties，一个包含交易和培育数字的游戏被曝光，为 NFT 在区块链领域之外被认可铺平了道路。 

快进四年，你会发现一个市场每天吸引 108,000 个独特的活跃钱包，同时在一个月内产生 52 亿美元。

到 2021 年，区块链行业显示出持续发展的明显迹象。名为 DeFi（去中心化金融的简称）的运动正盛开。围绕该行业的环境不再只是交易加密货币和将区块链技术嵌入到实际商业模式中。摆脱中间人的真正感觉确实是开创性的，并成功吸引了数千人的注意力。 

2020 年 NFT 交易总量为 2.5085 亿美元，几乎是 2019 年水平的三倍。与锁定在 DeFi dapp 中的数十亿美元相比，NFT 市场微不足道。然而，在 2021 年第一季度，一系列事件让 NFT 成为一种现象。  

2 月，NBA Top Shot 在 28 天内筹集了超过 2.24 亿美元。今年 3 月，Twitter 的首席执行官杰克·多尔西拍卖了历史上第一条推文，拍卖了 1,630 E 或 292 万美元。几天后，英国著名拍卖行佳士得为 Beeple 的 NFT 艺术品“前 5000 天”筹集了 6900万美元。感叹号是两个 CryptoPunk 销售额，每笔销售额为 4,200 Ξ或 757 万美元。 

NFT 狂热开始了。当天有数十个项目正在（并且仍在）启动。NFT 的受众越来越多样化，包括区块链开发者、艺术收藏家、好莱坞名人和 14 岁的孩子。每个人都在加入 NFT 的潮流。  

2021 年，NFT 市场的交易量已超过 10B （100亿）美元。在五到六个月的时间里，启动了无数 NFT 项目，推动该领域达到创纪录的 8 月。在那个月，NFT 产生了超过 $5.2B 的交易，吸引了超过 314 万的交易者。交易量较 7 月份增长 311%，较 3 月份增长 680%。  

现在，对 NFT 的需求越来越被供应所平衡。尽管如此，与任何有弹性的商品或服务一样，需求将变得不可持续。最肯定的是，总有一天 NFT 的供应将超过需求，泡沫破裂。 

然而，并非所有 NFT 项目都在泡沫中。其中一些具有支持其价值的某些特征，而另一些则与巨大的行业密切相关，而另一些则将成为您进入元宇宙的门户。 

超越 JPEG 的真正价值

目前，仅在以太坊上就有大约 4,200 个 NFT 项目。不幸的是，并非所有这些都包含为 NFT 收藏品提供超越艺术本身的真正价值所需的一组属性。 

到目前为止，毫无疑问，该领域的每个人都听说过 CryptoPunks、Bored Ape Yacht Club (BAYC) 或Art Blocks 等系列。这些项目属于 NFT 空间的顶层奥林匹斯山。但他们究竟是如何达到这种状态的呢？

接下来，我们将回顾六个重要元素或构建块，它们赋予 NFT 收藏品与简单的 JPEG 完全不同的意义。

用额外的效用奖励持有人

NFT 领域的效用每天都变得越来越重要。个人已经开始意识到，花数千甚至数百万美元购买一张动物图片，仅仅作为社交网络中的头像可能并不值得。用户现在期望通过拥有某个集合来获得未来的好处。 

围绕 NFT 实用程序最有趣的概念之一是资产抵押。NFT，尤其是那些具有重要价值的NFT，可以在借入基础资产时用作抵押品或支付承诺。一旦偿还了贷款加利息，用户就可以收回 NFT。这方面的一个例子是 NFTfi，这是一个允许用户通过锁定他们的 NFT 作为抵押品来借入加密货币的平台。 

截至目前，有超过 1362 万美元的贷款以 NFT 为抵押。如下图所示，Art Blocks、BAYC、CyrptoPunks 和 CryptoKitties 等有价值的收藏品是 NFTfi 中用作抵押品的主要工具之一。使用 NFT 作为抵押品的机会创造了一个额外的流动性层，可能会整体提升 NFT 空间。 

在另一个层面上，公认的 NFT 项目有一个共同的行为，用相关集合中的附加部分来奖励他们的投资者。也许 Larva Labs (LL) 开始了这一趋势。2021 年 5 月，来自 LL 的 Matt 和 John 宣布了他们的下一个 NFT 项目，该项目包含生活在以太坊区块链上的 20,000 个 3D 角色。一半的 Meebits 以 1:1 的比例奖励给 CryptoPunk 所有者。 

通过这种方式，CryptoPunk 持有者能够免费（加上 Gas 费）索取在撰写本文时其底价（集合中最便宜的 NFT）为 4.36 Ξ或 13,300 美元的NFT 。这就像圣诞节收到劳力士一样。该模型被广泛接受并开始被视为标准。在 Meebits 模型之后，几个 NFT 集合遵循 LL 步骤。例如，BAYC 持有者从无聊的猿犬舍俱乐部 (BAKC) 以狗的形式收到了他们的猿类宠物伴侣。Ghxsts 是一个规模较小但很重要的收藏，每月奖励其所有者，代表 12 星座中的每一个。名单可以继续。

然而，用额外的 NFT 奖励所有者并不是提供额外效用的唯一方法。目前，我们正在看到像 CyberKongz 这样的项目，这些项目每天都会通过收藏的原生代币产生奖励。在 Kongz 的情况下，$BANANA。原始 Kongz 持有者将能够在未来 10 年内每天索取 10 个 BANANA 代币。以目前的 BANANA 价格，Genesis Kong 的所有者每天可获得 390 美元或每年 140,400 美元。SupDucks ( $VOLT )等其他项目也将这种代币奖励系统嵌入到各自的模型中。

总而言之，任何 NFT 的效用元素都变得必不可少。虽然奖励额外的 NFT 或产生代币是最常用的模型，但有像 Punks Comic 这样的项目结合了两全其美，以非常有形的方式提供实用性。

从系列到品牌

NFT 中要考虑的第二个要素是一个系列所创造的品牌知名度。NFT 正在成为允许其所有者成为专属集团一部分的品牌。每个系列都与众不同，其中一些正在变得时尚。一个明确的方式来说明这一点，就是 Larva Labs 为自己创建的名称。一旦 Meebits 准备好铸造，尽管铸造价格很容易高于平均水平，但该系列在数小时内就售罄，并且在那一刻没有进一步的效用。似乎与公认名称相关联的 NFT 寻找追求者的难度较小（就像梅西 NFT 一样）。 

另一个例子可以在Bored Ape Yacht Club系列中找到。由于项目的成功，BAYC 的底价从 6 月底的 3 ETH上涨到撰写本文时的 38 ETH或约 115,000 美元。在这些级别上，可以公平地假设，对于大多数收藏家来说，收藏品已经遥不可及。拥有一只无聊猿已成为社会地位的标志。它会产生与穿着豪华服装或驾驶高档汽车相同的情绪。 

为了增加另一个层次，一些系列发布了仅适用于系列所有者的服装。这是品牌在任何方面的字面表示。此外，当知名名人对某些项目发表意见时，品牌知名度会被放大。 

建立根深蒂固的社区

NFT 正在建立强大而特殊的社区。在 NFT 这样新生的空间中，找到一个真正感兴趣的人会让人耳目一新。NFT 爱好者在 Twitter 空间和 Discord 频道中进行讨论、分享经验教训和对潜在风险的警告，甚至更好地分享有关如何改进自己项目的想法。 

Twitter 中的头像运动已成名，与品牌知名度元素一样，名人在社区中也有积极的涌入。Mike Tyson 使用一只蓝猫作为他的 Twitter 个人资料图片这一事实无疑对 Cool Cats 项目产生了积极影响。在其他项目中也会发生同样的情况，比如 VeeFriends，在那里与 Gary Vee 聊天是很常见的。此外，与斯蒂芬库里或 DJ 史蒂夫青木等体育巨星拥有相同的共同信念在几年前是不可想象的。这正是 NFT 社区的全部意义所在——消除这些障碍并建立项目社区。

社区无疑已成为如此重要的元素，以至于像 Loot 这样的项目旨在让其成员建立联系。Loot 是一个模拟龙与地下城 RPG 纸牌游戏的 NFT 合集。继 9 月份首次亮相后，该项目在一周内产生了 1.94 亿美元的交易额。Loot 是一个开创性的项目，因为它是为社区创建的，并将由社区开发。所有未来的改进和发展将由他们的成员负责。项目上限完全由这个小组决定。没有团队成员参与或永远不会参与。 

社区元素已成为共同信仰的基本价值。社区内的联系和根基越牢固，项目成功的机会就越大。

团队元素

谈论团队，这是评估任何 NFT 项目时的另一个相关要素。Loot 的情况非常罕见。规范是团队创建一个项目，进行任何所需的增强或开发，培育项目并有效地传达他们路线图中的里程碑。 

这是成功项目的共同要素，一些团队被广泛披露。Larva Labs、VeeFriends 或 World of Women 就是这种情况。Yuga Labs 或 0N1 Force 等其他团队将在他们的头像角色下保持匿名。

最后，最重要的方面是与团队成员的参与程度。实现目标的团队会变得值得信赖。很容易发现渴望为社区带来价值的团队。但也许最重要的是，他们如何应对所面临的挑战？ 

稀有性和稀缺性问题

每个集合都拥有一组属性和特征，允许算法随机生成每个片段，以便它可以与同一集合中的其他片段不同。分配给每个特征的稀有性使 NFT 收藏品与众不同。这种独特或稀有的元素也推动了需求的增长。确实，某些作品可能比其他作品更具吸引力，即使它们不是那么独特；然而，稀有物品比普通物品更受欢迎，从而提高了前者的价格。

另一个需要考虑的因素是供应，10,000 个头像集合似乎是当前 NFT 市场的标准。这些收藏通常有一定的上限，这意味着供应量将永远保持不变。限制收藏的版本大小增加了稀缺元素，在此过程中欣赏价值。然而，这并不是每个系列的规则。 

例如，像 Avastars 这样的育种项目，包含 25,200 个没有单一克隆的第 1 代片段，将允许它们的持有者将这些特征组合成两个或多个片段，以铸造一个全新的 NFT。尽管供应量会增加，但新铸造的 NFT 将被标记为 Replicant 或 Gen-2，从而明确区分这两种类型。这是一个需要考虑的非常重要的方面。虽然某些系列会扩大其供应量，但原始 NFT 作品应该以某种方式与其他作品区分开来。 

扩展到其他行业

到目前为止，我们已经解决了大多数 NFT 头像项目可以实现的元素或因素。 

他们创建了拥有共同利益的真正社区，并在某些情况下为其所有者提供了有形的奖励。 

虽然这些虚拟形象项目确实是最新 NFT 狂热背后的主要驱动力，但还有其他类别正在帮助 NFT 在不同行业引起轰动。 

游戏升级

以区块链驱动的游戏为例。据 Statista称，它们可能会彻底改变传统游戏行业，该行业的年收入为 1140 亿美元。

首先是游戏赚取的叙述，其次是区块链游戏中 NFT 启用的所有权。玩家将第一次真正拥有并完全控制他们的游戏资产。 

正如DappRadar 的 8 月行业报告中所详述的那样，基于区块链的游戏正在推动行业内的使用。与游戏 dapp（去中心化应用程序）交互的 Unique Active Wallets 为 747,000，比 7 月的数字增长 64%。将这个特定的游戏使用指标与 DeFi 或 NFT 收藏品本身等其他同行类别进行比较，可以清楚地表明游戏是如何支持与区块链 dapp 相关的独特钱包数量的最新激增的。

事实上，仔细观察 8 月份创纪录的 NFT 交易量，18% 来自游戏内收藏品，即可以在区块链游戏中使用的 NFT。 

总而言之，游戏空间有望成为一个更大的市场。Axie Infinity 是一款由越南工作室 Sky Mavis 开发的游戏赚取游戏，每天已经吸引了全球超过 150 万独立玩家。Illuvium 正准备在今年年底推出以太坊第 2 层解决方案 Immutable X，而将在 Solana 上运行的游戏 Star Atlas 也正在大肆宣传。

当然，还有 Metaverse，一个旨在成为个人在虚拟现实中创造、交易和社交的地方。Metaverse 中的重要游戏项目包括 Decentraland、The Sandbox、Blankos Block Party、Ember Sword、Somnium Space 等。

艺术数字化

艺术是 NFT 最适合的应用之一。对局很理想。一方面，NFT 旨在证明真实性，这是艺术作品中最重要的方面之一。另一方面，交易 NFT 的市场是艺术家获得认可的完美场景。利用这个数字空间，同时拥有一种可靠的方式来管理版税。 

NFT 中的艺术以不同的方式存在。有 1/1 的艺术作品，或独一无二的作品。想想蒙娜丽莎或 Beeple 的艺术作品。它们仅以单一且不重复的方式存在。 

此外，还有生成艺术，一种新的艺术创作方式。在生成艺术中，艺术家（或程序员）指示程序运行一个代码，该代码为预定义的模式分配独特的特征。这种类型艺术 NFT 变得如此令人垂涎，以至于这些作品的售价高达 568 万美元。生成艺术项目的一些例子是 Art Blocks 和 Larva Labs 的 Autoglyphs。

体育和 NFT 之间的幻想对决

体育产业很大，尤其是在美国。根据Verified Market Research 的数据，2019 年全球体育交易卡行业价值 $13.82B。世界上最重要的体育协会之一意识到等待被抓住的巨大机会。 

2019 年 7 月，NBA 与 CryptoKitties 背后的区块链公司 Dapper Labbs 成立了合资企业。这标志着 NBA Top Shot 的开始，这是一个在 FLOW 区块链上运行的 NFT 项目。直到今年 2 月，该项目才真正显现出来。2021 年 2 月至 3 月期间，Top Shot 在 256 万笔交易中产生了超过 4.3236 亿美元的收入。 

如果篮球有其官方的 NFT 平台，足球（足球）就不会落后。进入Sorare，一个梦幻足球平台，允许玩家根据以NFT为代表的球员卡交易和管理梦幻球队。在这款游戏中，与传统的奇幻平台不同，奇幻经理可以完全控制他们的玩家。Sorare 的卡牌供应有限，为游戏增加了另一个策略层。虽然 Sorare 还没有达到 Top Shot 的水平，但一些 Sorare 独家球员卡，如罗纳尔多或姆巴佩的，价值数万美元。

此外，体育收藏项目也吸引了大量投资兴趣。Dapper Labs 宣布了一轮 2.5 亿美元的融资，而总部位于法国的 NFT 项目 Sorare 则筹集了 6.8 亿美元，由软银领投。 

总体而言，体育产业在 NFT 领域得到了很好的体现。Ethernity Chain 是一个专门从事 NFT 的区块链，提供了广泛的体育传奇收藏，其中包括贝利、穆罕默德·阿里、莱昂内尔·梅西和丹·马里诺。

时尚品牌加入 NFT 热潮

化身是元宇宙的真正居民。皮肤和游戏中的化妆品在玩家中非常重要。一个明显的例子是像 Fornite 这样的免费游戏，它允许游戏玩家自定义他们的游戏头像。DMarket，一个交易游戏内可穿戴设备的平台，估计皮肤市场每年价值40B 美元。 

来源：Burberry – Blankos 独家鲨鱼雕像

由于市场如此之大，玩家和 NFT 爱好者渴望达到更高的独家性，元宇宙可以成为时尚行业真正的价值生成器。而这些时尚巨头的存在已经被感受到了。8 月，Burberry 和 Blankos Block Party 合作设计了一个包含角色和配饰的协作 NFT 套件。2,250 个 NFT 的集合在不到一分钟的时间内售罄。 

虽然 Burberry 是第一个直接与 metaverse 项目合作的时尚品牌，但它的一些同行已经在进行中。6 月，时尚巨头 Gucci 以 20,000 美元的价格拍卖了其首部官方 NFT，这是一部以 2021 年秋冬系列为灵感的短片。最近，杜嘉班纳 (Dolce & Gabbana) 紧随其后，与专门从事奢侈品和文化产品的策划市场 UNDX 合作，推出了九件套 NFT 系列。 

走向主流

区块链技术，特别是 NFTs，造成的影响达到了新的高度。影响如此之大，以至于所有行业的大品牌都表示有兴趣成为该领域的一部分。8 月，可口可乐拍卖了一款可在 Decentraland 佩戴的品牌 NFT 皮肤。慢慢地，庞大的组织正在这个领域留下他们的足迹。

这不仅仅是品牌推出系列或拍卖稀有作品。NFT 成功地引起了重要投资公司甚至金融机构的注意。例如，总部位于新加坡的对冲基金三箭资本在 8 月底宣布创建 Starry Night。Starry Night 的使命是“组装世界上最好的 NFT 集合”。 

8 月 23 日，支付处理巨头 VISA 通过 Twitter 宣布以 150,000 美元的价格购买 CryptoPunk #7610。该公告引起了重要媒体的关注。最后，上周，苏富比以 2400 万美元的价格拍卖了 BAYC 收藏的 101 件作品。 

总而言之，主流受众将帮助 NFT 空间巩固其价值。虽然 NFT 仍然感觉像是一个新生的空间，但认为它会达到 DeFi 等其他区块链类别中的数字并不牵强。 

NFT的未来

目前，区块链行业还处于非常早期的阶段。目前强劲的需求足以平衡不断增长的供应，许多 NFT 项目可能无法长期保持价值。然而，表现最佳的一组 NFT dapp 将成为该行业未来的关键部分。 

某些 NFT 项目正在用以前无法想象的公用事业奖励其所有者，而其他项目则展示了建立强大社区的力量。总而言之，这些系列正在成为连锁店之外的知名品牌。

影响已经变得如此之大，以至于体育或时尚行业等重要的垂直行业都希望成为这一不断发展的格局的一部分。也许更重要的是，主要机构已经看到了这个概念的真正投资价值。 

NFT 授予的真正所有权将解锁众多应用程序，门票将成为 NFT，对盗版造成潜在的致命打击。保护艺术家的音乐文件将作为 NFT 分发。产生收益的 NFT 可能会变得与许多现有金融产品一样重要。看起来我们正走向一个不可阻挡的虚拟现实，类似于 Ernest Cline 在他的著名小说Ready Player One中所描绘的——一个现实生活资产将被标记化或 NFT 化的世界。哪个泡沫首先破灭仍然是最大的问题。
