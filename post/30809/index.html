NFT 艺术家 Pplpleasr 本周推出其第一个基于 Solana 的项目。

“The Collectoooooor”允许 NFT 拥有者换取更稀有的 NFT。该项目为慈善事业带来约 127000 美元的收益。

数字艺术家 Pplpleasr（又称Emily Yang）是今年 NFT 市场繁荣中的佼佼者之一，曾设计《财富（Fortune ）》杂志封面并将其作为NFT出售。她之前的工作是在基于以太坊的热门 NFT 平台上开展，但现在已经扩展到其竞争对手 Solana 网络，并发布一个创新项目。

本周在里斯本举行的 Solana Breakpoint 会议期间，Pplpleasr 与 Cloud Eater Studios、基于 Solana 的 Metaplex 协议合作创建“The Collectoooooor ”，该项目共计 500 件作品，包括 3D 鸭子、建筑物、UFO 等。

单独来看，任何一件作品都可能看起来简单或平凡。但是，如果一个钱包持有者收集了该系列中所有 13 种独特的物品，他们最终可以选择销毁它们，以换取总共 15 个限量版动画 3D 城市景观中的一个（三种设计，每种设计有五个版本）。

Pplpleasr 告诉 Decrypt，“The Collectoooooor”被设计成“像元宇宙《动物之森（Animal Crossing）》一样，玩家根据配方收集不同的资产来‘制作物品’作为升级。”

任天堂（Nintendo）的《动物之森》游戏系列包含卡通生物的低调生活模拟和在线社交游戏。Switch 上的最新版本于 2020 年 3 月在新冠大流行开始时发布，成为数千万玩家逃离隔离现实的愉悦之旅。

在视觉上，The Collectoooooor 的城市场景捕捉了一点《动物之森》的氛围，带有动画 3D 设置的等距俯视图。Pplpleasr表示，该项目的设计灵感也来自于她之前的作品《鲸鱼之梦（A Whale’s Dream）》，并从里斯本和 Solana 生态系统中汲取视觉线索。

在 Solana 上首次推出一个项目对于 Pplpleasr 来说是新的体验，但她认为这是一个漫长的过程。“对于其他区块链以及 NFT 如何在其他链上运行，其市场/受众、它与以太坊的区别，我都很好奇 。”此外，Solana Labs 首席运营官  Raj Gokal是一位长期支持者，她于 2021 年初首次邀请 Pplpleasr 合作。

除了转向 Solana 发布项目之外，Pplpleasr 还采用一种不同的方式销售“The Collectoooooor” NFT。 Pplpleasr  采用 Fair Launch Protocol（公平启动协议）的方式，其设计目的是避免大量买家同时涌入购买数量有限的 NFT，同时也让社区以去中心化的方式决定价格。

这一过程首先让潜在买家以各自认为合理的价格对单个 NFT 进行出价，然后最终将出价中值设定为“公平启动”价格。在这个案例里，截至周三发布时，其“公平启动”价格是 1 SOL，或约 240 美元。（这些作品目前在二级市场上的售价约为发行价的五倍。）

任何投入1 SOL 或更高金额的竞拍者都会自动收到一张抽奖券，然后从中选出 500名获胜者完成交易。未赢得抽奖的竞拍者将获得自动退款，而出价超过 1 SOL 的竞拍者将获得剩余部分的退款。出价较低的人也可以选择以更高的价格参加抽奖活动。

然而，不管结果如何，每个竞拍者还支付了 0.1 SOL 的费用，将用于慈善事业。截至发稿时，共有 5568 名竞拍者，总费用为 556.8 SOL，约合 127000 美元。Pplpleasr 已经要求 Collectoooooor 社区 Discord 群组中的粉丝们帮助找出捐赠资金的最佳方式。

此前，由于出售《财富》 NFT ，她向专注于维护言论自由的慈善事业捐赠价值超过 65 万美元的 ETH。那次拍卖总共产生价值 130 多万美元的 ETH，她和《财富》向四个选定的组织共同捐赠了一半的收益。

Solana 的 NFT 生态系统在今年夏末开始火起来，其平台提供了一个费用低、交易时间快的以太坊替代方案。在过去的三个月里，Degenerate Ape Academy、Solana Monkey Business 等热门头像收藏品帮助推动了 Solana 网络上价值数亿美元的交易量。

Pplpleasr 是 NFT 领域最知名的艺术家之一，今年早些时候，她为 DeFi 协议创作的作品在年初 NFT 市场繁荣时期大受欢迎。她为去中心化交易所 Uniswap制作的视频广告在 3 月份以 NFT 的形式出售，以价值 52.5 万美元的 ETH 成交。该 NFT 是由粉丝们组成的 PleasrDAO 购买，目的是收集她的作品，以及其他 NFT 和独特的文化艺术品。

除了《财富》封面和 NFT 之外，Pplpleasr 最近还与音乐家 Steve Aoki 合作推出 plsa0k1，这是一个合作金库，用于收集女性艺术家的 NFT 艺术品。Steve Aoki 在本周还宣布与漫画家、《再生侠（Spawn）》的创作者Todd McFarlane合作，推出基于 Solana 的 NFT 市场 OddKey。
