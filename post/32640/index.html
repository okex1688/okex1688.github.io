数据所有权是 Web 3.0 发展的核心要素

从Web1.0到Web2.0，互联网完成了从简单的静态桌面网页推到了互动体验、用户生成的内容和市场经济的升级，如今元宇宙正在扣响Web3.0时代的大门。在加密货币投资巨头灰度（Grayscale）发布的元宇宙 Web 3.0报告中，元宇宙被定义为⼀个互联、体验式的 3D 虚拟世界，位于任何地⽅的⼈都可以在其中进⾏实时社交，并形成⼀个跨越数字和物理世界的恒久且由⽤户⾃⼰拥有的互联⽹经济体系。在这一体系中，数据的归属无疑是核心价值所在。

GlanCaGiorgi在演讲中提出，数据所有权是web3.0的核心，这一点与元宇宙生态的内容十分契合，基于这样的背景，我们能够以数字化身份的形式在虚拟世界中进行交流、商业、工业制造等活动。

海外区块链专家学者Frederck Bravey在主题演讲中指出，web3.0的有着5大关键特征，去中心化兼部分中心化、去中心化自治组织DAO、内容展示、个人的数字身份以及数字资产所有权。

北京中联信数区块链科技院院长李派克也在以《从宇宙重生到元宇宙资本论》为主题的演讲中强调，数字资产所有权应当属于个人。

Web 3.0是一个开放式的加密元宇宙，由全球用户民主地拥有和控制，不同于传统的互联网世界，Web 3.0中的数据价值将回归个人所有。在这个数字宇宙中，每个人都将成为一个数据中心节点，产生数据的同时也在传递着数据，收获价值的同时也在创造价值，数字资产的实质便是数据，因此数据所有权必然是Web 3.0发展的核心要素。

现阶段GameFi是元宇宙的最佳载体

在经过2场精彩的圆桌会议后，论坛演讲的内容主题也开始从技术理论转向应用场景的探讨。

CLIPS Finance的代表Yigu Lin在《2022年Web0发展趋势》主题演讲中指出，Metafi，SocialFi，Gamefi概念将会在未来几年兴起。而这些概念的爆发无非都是基于以下几个模型：Play To Earn边玩边赚/Interact to earn，MetaFi Funding，去中心化自治组织Dao/Guid公会，以及去中心化数字身份。他最后还补充道，不管是Metafi还是SocialFi，还是Gamefi，未来都会将会在元宇宙世界里有共同的交互行为，这是一个共存的世界。

在今年，Axie Infinity、Crypto Kitties、 NBA Top Shot、The Sandbox等一些优秀的区块链游戏已经吸引到了部分主流市场的玩家，区块链游戏、GameFi等概念正在不断破圈，GameFi的成功，不仅仅是因为独特的Play To Earn模式。

元宇宙主要的核心在于对虚拟资产和虚拟身份的承载，它需要创造一个公平、开源、可靠的经济系统，在这个过程中游戏成为了元宇宙+NFT设计的重要载体，对于想要探索元宇宙的人来说，GameFi是绝妙的入口。本次论坛活动的主办方NirvanaMeta同样选择了GameFi作为元宇宙布局的切入点。

NirvanaMeta是一个元宇宙链游平台，在 NirvanaMeta 中所进行游戏、充值、氪金等任何行为所带来的价值，均会回归到玩家，并以MNU作为载体在生态中自由流通。NirvanaMeta依托于专为链游平台打造的高性能公链MetaChain，每秒3000+TPS的高效处理能力能够轻松应对高频词高并发的链游应用需求，确保生态体系平稳安定的运行，平台发行的第一款同名3D新魔幻链游NirvanaMeta（宇宙重生）一经上线便获得了广大玩家的喜爱。

在NirvanaMeta（宇宙重生）游戏中，玩家可以通过战斗、强化、做任务、打副本等方式获得游戏装备、经验、NFT 卡片、通证与金币奖励，部分获得的道具可以实时兑换成可交易的资产。在区块链和分布式技术特性的保护下，NirvanaMeta平台游戏的数据具有较高的安全系数，NFT资产的独一性为其价值提供了保障，通过交易平台，玩家可以自由交易资产或变现，在GameFi中感受元宇宙独特的生态魅力。

除了GameFi等实际应用场景，元宇宙和Web 3.0也在悄然引领着传统组织架构的革新升级，在论坛会议的下半场，VPM亚太地区负责人Alice带来主题演讲《DAO去中心化时代下的组织变革》。

DAO的全称是decentralized autonomous organization，意为去中心化自治组织，是基于去中心化的理念，利用区块链技术运行的组织模式。DAO使用智能合约来建立规则并自动执行决策，通过社区集体治理的方式共同推动组织的发展。Alice指出，VPM公会正在尝试DAO化治理，公会会员分布于世界各地，未来将持续向DAO方向发展。

另一方面，元宇宙和Web 3.0的浪潮也为虚拟现实技术带来了新的活力和发展思路，在主题为《裸眼3D开启未来数字生活》的演讲中，维超智能CEO谭杰夫表示，元宇宙是个跨越许多公司甚至整个科技行业的愿景，可以看作是移动互联网的继任者，而裸眼3D是元宇宙的钥匙，也是未来数字生活的入口。

结语

从DeFi热潮到NFT的兴起，从GameFi的出圈到元宇宙概念的爆发，技术、应用、意识等领域的不断发展积累，正在推动着产业升级革新的到来，元宇宙和Web 3.0何时才能完全落地，我们不得而知，但数字时代的理念无疑已经融入了我们的生活。
