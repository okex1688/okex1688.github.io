这个周末是属于 NFT 的。

当你打开 Dune Analytics 的 OpenSea 数据统计页面便会发现，7 月 31 日和 8 月 1 日两天，这家全球最大的 NFT 交易平台的单日交易额高达 3946 万美元和 5374 万美元，连续两天刷新历史最高纪录，8 月 1 日当天的交易额甚至几乎达到了这个周末前历史最高纪录的三倍。

前不久随着无数头像类 NFT 的诞生，人们逐渐从一开始的兴致勃勃变成后来的审美疲劳，虽然整体交易量还不错，但整个市场的情绪逐渐变得低沉。

那又是因为什么，让 NFT 在这个周末迎来了大爆发呢？

2700 ETH，104 枚 CryptoPunk 

故事要从 7 月 31 日早上说起，人们发现一个匿名地址买入了上百枚 CryptoPunk。

0x650d 开头的地址在 7 月 30 日以近 50ETH 的总价买入两枚 CryptoPunk，当时并未引起关注，而他随后的操作则将所有人的目光全部吸引住了：他在 7 月 31 日上午一小时内连续买入 102 枚 CryptoPunk，算上此前买入的 2 枚，共计花费高达 2700 ETH。该地址买入的 CryptoPunk 价格在 23 ETH 和 29 ETH 之间，平均单价为 25.8 ETH。

一时间，从推特到微信群，海内外的社交媒体都在议论着这只「匿名巨鲸」究竟是何方神圣。

一个推测让 NFT 市场彻底陷入了疯狂。Inside 分析师 Michael Tant 将自己的推测发布在了推特上，他认为，购买这些 CryptoPunk 的匿名地址背后，是三箭资本。

一石激起千层浪，人们开始讨论如果真的是三箭资本入局，将 CryptoPunk 等 NFT 纳入自己的投资组合，那接下来会不会有一大波资本加入，而随着 CryptoPunk 的价格越抬越高，下一个可能会被列入投资组合的 NFT 收藏品会是什么呢？ 

NFT 收藏品的牛市 

在成百上千个 NFT 头像项目中，最先受到影响的是 CryptoPunk 和 Bored Ape Yacht Club。

随着疑似三箭资本的匿名地址大量购入地板价的 CryptoPunk，知名投资人 Gary Vaynerchuk 等社会各界知名人士也纷纷买入了 CryptoPunk，Gary Vaynerchuk 更是直言「对这项资产有着难以置信的信心」。随着人们大量买入，CryptoPunk 的地板价也上升至 33ETH，据 OpenSea 数据显示，近七日 CryptoPunk 总交易额达 4.78 万 ETH，平均成交价高达 46.325ETH。

而另一个受益匪浅的项目便是 Bored Ape Yacht Club。 

在这个周末前，Bored Ape Yacht Club 社群声音越来越小。更早之前，Bored Ape Yacht Club 向每一只猴子的持有者空投了一只狗，并宣布在 7 月会给持有狗的用户以及同时持有猴子和狗的用户发放特殊奖励。

空投来的狗同样具有精心设计的外观以及埋藏的各种彩蛋元素，这使得它们虽然是空投奖励却也能卖出不低的价格，大概数万元人民币左右。正因如此，7 月即将到来的奖励也让持有者们期待不已。然而最终他们等来的却只是一个可以兑换实体篮球的 NFT 空投，而所谓的兑换也仍需要持有者支付 100 美元。

这让很多 Bored Ape Yacht Club NFT 的持有者感到失望，他们开始抛售自己收藏的猴子和狗，猴子的地板价从 6ETH 以上的价格下跌到了接近 4ETH，而狗的地板价也从接近 2ETH 极速下跌到最低 0.5ETH。不够坚定的持有者们认为这个项目在失败的路上渐行渐远，一些坚定的持有者们也纷纷感叹着猴子的第二个熊市到来了。 

在加密世界，代币价格上涨往往会带动整个板块代币价格的提升，这一点放到 NFT 领域同样适用。CryptoPunk 是 NFT 收藏品的 OG，Bored Ape Yacht Club 是 NFT 收藏品的后起之秀，更是近期最热门的项目，二者又同属于「NFT 头像」这一新兴的细分领域，于是从 7 月 31 日开始，始于 CryptoPunk 的狂热终于席卷到了 Bored Ape Yacht Club。 

据 OpenSea 数据统计，截至发稿时，猴子的地板价已经飙升至 8.4ETH，近 7 日平均成交价高达 9.7ETH；狗的地板价也上升至 1.57ETH，近 7 日平均成交价达 1.86ETH。更有趣的是，这并非完全由现有的 Bored Ape Yacht Club 藏家购买所致，总量 1 万枚的 Bored Ape Yacht Club NFT 的持有地址数也在 8 月 1 日突破了 5000，这说明越来越多的旁观者开始加入了收藏的队列中。

而除了这些 NFT 头像项目，Art Blocks、Autoglyphs 等艺术类收藏品项目也应声上涨，Art Blocks 的 Curated、Playground 类别近七日总交易额涨幅数倍，CryptoPunks 制作团队 Larva Labs 推出的 Autoglyphs 近七日总交易额涨幅更是高达十倍有余。 

NFT 收藏品为何爆发 

回顾近日的 NFT 热潮，先是大批头像 NFT 项目被原本的 NFT 玩家抢得不亦乐乎，甚至打起了 Gas 战；再是 Axie Infinity 交易额的一骑绝尘以及 AXS 代币价格的一飞冲天，以及 YGG 代币的公开发行，平时不关注 NFT 的人也开始选择投资 AXS 等 NFT 项目的 ERC-20 代币；再到这个周末，NFT 市场被这一百多枚 CryptoPunk 彻底引爆，越来越多人开始买入自己的第一个 NFT。

究竟是什么原因能让原本担心 NFT 低迷的流动性以及认为 NFT 价值难以判断的投资者们开始对 NFT 领域的布局了呢？

一方面是受传言的影响。三箭资本的影响力无需多说，当人们看到一个有理有据的推测说三箭资本通过购买 CryptoPunk 来布局 NFT 领域后，自然会选择跟从。人们会认为被三箭资本看中的资产一定是有价值的，也一定是有潜力的，而随着三箭资本的布局，未来也一定会有更多资本加入其中，购买的对象也就不仅仅局限于 CryptoPunk，还有可能购买 Bored Ape Yacht Club 等 NFT 头像项目，甚至是 Art Blocks 这样的 NFT 艺术收藏品项目。这也就形成了一股 FOMO 情绪，人们都想赶在下一个资本大批量买入之前以相比之下较低的价格将 NFT 收入囊中。

另一方面是这些成套、成系列的 NFT 收藏品并不像此前人们熟知的 NFT 艺术品那样具有低流动性、价值难以评估等特点，它们虽然是「非同质化」的，但却带有很多「同质化」的属性，ERC-20 代币的投资者们可以用自己熟悉的方式参与到 NFT 的投资中来，比如如果看涨一个 NFT 收藏品系列但又不想关注属性的稀有度以及美观程度，那么便可以采用目前 NFT 收藏品投资者最常见的方式：买入地板价的 NFT，如此一来，只需要等地板价逐渐升高到心理价位后，再将 NFT 以那时的地板价卖出即可。

除此之外，NFTX、NFT20 等指数基金也为 NFT 收藏品的流动性提供了保障，人们可以随时将 NFT 换成对应的 ERC-20 代币卖出，也可以随时买入 ERC-20 代币再兑换成 NFT，解决了人们担心在急需流动资金时无法兑现的后顾之忧。

反观 NFT 艺术品市场，据 cryptoart.io 数据统计，NFT 艺术市场仍旧十分低迷，并未受到这次热潮的影响，看来人们从投资带有「同质化」属性的 NFT 过渡到投资极具独特性的 NFT 的阶段还需要一段时间。

尽管后续推特名为 0x650d 的网友站出来表明这个买入大量 CryptoPunk 的钱包地址是他的，而非三箭资本，但这个推特账号本身也是匿名的，而且在 2021 年 7 月才刚刚创建，所以已经有一些网友认为这个账号背后仍旧是某个大资本在掌控。

况且，NFT 收藏品这一赛道已经被点燃，这架火箭已经发射升空，人们开始认识到 NFT 也不仅仅意味着艺术、完全独特、低流动性，NFT 收藏品市场也同样可以拥有不错的流动性，投资者也可以没有艺术审美基础，可以用投资 ERC-20 代币的方式投资 NFT，而这也会让投资者们慢慢习惯于 NFT 的投资。而这也同样会让资本看到 NFT 投资的可能性，会让他们开始选择将 NFT 放入自己的投资组合当中，也许下一次就真的会有大资本来扫货了。
