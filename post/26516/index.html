NFT 到底是什么？

最近数字资产市场整体属于熊市状态，其中还是呈现出一些亮点，比如元宇宙、GameFi（游戏化金融）和 NFT 板块。实际上元宇宙和游戏化金融都跟 NFT 关系密切，所以整个行业越来越需要深入理解 NFT。现在普遍一谈 NFT 就想到艺术品，我们认为是非常狭隘的。NFT到底是什么？我们结合自己的研究，给大家做一些理论上的探讨。

2017年的时候，我跟元道老师把 Token 这个英文单词翻译成通证，在行业内获得了认可和接受。回想2017年，当时我们对于价值互联网、区块链、未来的开放金融有很多的憧憬。比如当时我们觉得未来的价值互联网会有一个协议层专门用来表示价值；用通证、数字货币表示价值，我们想到通证经济能够调动千万用户的热情，能够创造新的增长模式，我们判断金融、互联网应用和游戏都将在区块链上重构。今天看起来，这些判断都是对的。

但我必须承认，有一点我们没看到，就是 NFT。当时我们所憧憬的价值互联网主要是基于匀质通证的。我们写那篇文章（《通证是下一代互联网数字经济的关键——元道区块链对话之一》）的时候，正值加密猫大红大紫之时。加密猫告诉我们，通证除了匀质的，还有非匀质的。我们觉得很好，NFT 是对 FT 的一个补充，一个点缀，但是绝不是主流。然而，四年之后，我对于这个问题有了完全不同的看法。

我认为，在未来的价值互联网上，大部分的通证都是非匀质的，匀质通证只是一小部分。换句话说，未来的价值互联网主要是建立在非匀质通证——NFT上的。

回顾网络通信的发展历程

在展开论述之前，首先先讲一段往事。关于网络通信的发展历程，其实人类第一个电子通信网络是电话网，不是互联网。

电话网络主要是电路交换网络（circuit switched network）。什么是电路交换网络呢？举个例子，张三要打电话给李四，他把请求发给接线员，接线员拿出一个两头都是插头的线，把张三李四两人的电话线连上，两个人就可以对话。语音信号就可以像水一样，从张三流到李四那里，非常简单、清晰、低时延、高质量。但是有一个问题，一旦这两个人的信道建立之后，这个线路在这个时间只能由张三和李四之间独占，不允许其他人插入进来。这种独占线路情况，给我们中文贡献了一个词汇，就是所谓的“占线”。

因为有占线的问题，如果一直采用过去的电路交换网络，用户数量一多，就到处占线，整个网络不可能做大，我们也绝对不可能像今天这样一边听音乐，一边看新闻，一边玩游戏，因为只要有一个应用占线，其他的应用就不能用了。用一个专业的表达方式，这种网络没有可扩展性。

怎么才让网络通信有可扩展性？1962年，麻省理工的研究组提出了一种新的网络交换协议，即今天互联网所适用的分组交换网络（Packet-switched Network）。分组交换网络最重要的一个进步，就是把过去像流水一样的数据切成一块一块的，放在一个又一个数据包里。

一个数据包长什么样呢？它首先有一个固定格式，一定要包含几个特定的字段，包含一系列自描述信息，比如表示自己是遵守什么协议的，这个包有多大，发送方和接收方的地址，自己的序号，等等。然后，这个包里要有一个内容字段，真正传送的数据放在这个字段里。数据包往往对这个内容字段不做特别的规定，允许你在里面放任何东西。

做一个形象的比拟，数据包这个方案，相当于把水装在很多瓶子里，每个瓶子上面都有包装，写清楚这是由谁发给谁，水的质量、成分，这是第几瓶水等等，这叫自描述信息。数据从流变成包，整个网络通信形式就发生了根本的变化，核心点是把过去简单的基于流的通讯形式，变成基于可自描述的数据包，通过缓冲、存储、复制、路由、转发等等一系列手段，构建高度灵活可扩展的网络。

NFT 是价值互联网的价值数据包

了解了这段往事，我们再仔细看看 NFT？现在的 NFT 基本上都是基于 ERC721 标准的。我们拿起放大镜仔细看一下 ERC721。它规定 NFT 必须有两个必备字段，一个是这个 NFT 的发行合约，另外一个是 TokenID，然后下面有一个巨大的“内容”字段，这个字段是一张白纸，允许我们里面做任何事情。“一张白纸好画最美的图案”，你可以在上面做任何想做的事情，艺术家们最先跑出来说，让我在上面画画吧，于是我们看到现在大量的 NFT 都是艺术品。游戏开发商跑出来说，让我们在上面画土地证和游戏装备吧，于是现在开始有 NFT 虚拟土地和游戏装备了。但是，大家应该意识到，这个内容字段里可以放任何东西，甚至是套娃，再套另一个瓶子。

大家不觉得 ERC721 NFT 很像分组交换网络里的数据包吗？

于是我们看到了有意思的历史的押韵。在通信协议当中，我们从电路交换走向分组交换，关键是把数据变成一个个可以自描述的数据包。而现在价值互联网的发展仍然处于以 ERC20 匀质数据为主的时代。这个技术有很多优势：流动性好、计算性好，协议简单，但是只能在智能合约之间进行熟人社交，它只能完成几个最简单、最基本的动作。如果这种形式继续持续下去，无论在应用层面，还是在网络通信层面、价值互联网的互通层面，这个网络都长不大。

我们认为 NFT 就是价值互联网当中的价值数据包，它是一瓶瓶的水，打上标签之后，用户决定里面放置的内容。在这种情况下，它可以具有巨大的可扩展的优势，可以轻松跨链。智能合约并不需要知道所要处理的NFT由哪个智能合约发行。这是一个非常重要的思想上的变迁。

NFT 发展的重要方向

于是我们猜想，能不能把价值互联网进行一次升级，以 NFT（非匀质通证）来构造价值互联网。在升级的过程中，可能需要对NFT的协议作基本的要求，除了要有合约与Token的ID，还要有区块链（以太坊、波卡、BSC等）的链标识，有了链标识之后，我们就可以把  进行跨链转移。NFT内放置的内容要与业务、应用逻辑挂钩。由此我们将得到重要的能力：跨链 NFT 的相互组合和计算。

这个能力在如今区块链领域刚有萌芽的苗头，相信未来会成为NFT发展的重要方向。这样做有几个突出的优势：

第一，资产的跨链贸易 ，在以NFT做载体，资产跨链贸易可以比今天容易很多。

第二，NFT可以用来做游戏装备 。NFT可以嵌入大量的业务逻辑，装备升级的业务逻辑可以直接写在NFT的内容层。比如一个青铜装备，加上一些钻石和金币，就可以升级为黄金装备等等，这样的逻辑可以直接在 NFT 层面上实现。

第三，更好、更直接、更容易想到的应用，就是用NFT来做可计算票证 ，比如说商业票据、存单、债券、标准合约、提货单、质押凭证、金融衍生品（期权、期货）等等，都可以表达为 NFT。

第四，监管追溯 。由于 NFT 自描述能力强，监管可以基于这些自描述信息进行一系列的数据分析和监管，对其应用进行一系列的规范和要求、约束和限制，可以给未来的开放金融应用带来无穷无尽的想象空间。

如果未来的价值互联网是基于NFT的，它会变成整个价值互联网的一次重大升级。

Solv 利用 NFT 表达可计算票证

Solv 协议做的就是上述第三点，票证。我们利用NFT的内容部分来创造一种标准的规范，表达可计算票证，也就是我们内部称为 vNFT 的新的 NFT 类型。 可计算票证的本质是表格，所以我们构造了一套标准来实现表格计算。举例来说，可以用一个表格来代表提货单信息，把这个表格放在 vNFT 的内容字段里，这个 vNFT 就成了提货单。这一点很容易想到。

但 Solv 协议往前走了一步。大家回顾一下，网络通讯里的数据包是可以缓冲合并的，某种意义上说，数据包是可计算的，也就是可以拆分和合并。我们认为这个能力对于代表价值的 NFT 来说也是十分重要的。于是，vNFT 协议中可以实现这两个表格的合并计算，相当于把两张票据整合了，一张票据是可以提一百本书的，另外一张票据是可以提两百本书的，合起来就是可以提三百本书的一张票据。同一规格的债券，一千块钱的债券和两千块钱的债券可以合成一张三千块钱的债券。这样基本的计算逻辑，已经在我们的协议当中实现了，不仅可以实现合并，还可以实现拆分，主要的线性计算都可以表达。

理想很丰满，但万事开头难，一开始都需要成功应用案例来证明。我们首先选择用 vNFT 来表达Vesting Token，就是大家比较熟知的投资者锁仓份额（不光是投资者，包含所有目前锁仓、未来释放的Token）。比如我们可以把一百个比特币锁在一张票据里，然后设定好释放时间和释放模式（按照每个区块线性释放，还是周期性性释放，事先把释放的规则写好），规定好移动规则，票据可能可以移动，也可以限定不能移动，可以限定它只能移动一次等。我们甚至还可以实现指向某一个特定票据转移Token，就是拿着一张票据，不管这个票据转到谁手里，将来三个月之后有一笔支付直接会传送到这个票据里，实现可移动的帐户。NFT使数字资产更加通用，因此我们选择了这样的方向，目前为止这个思路受到全球专业投资机构的认可。

总结一下，我们认为 NFT 是比 FT 更优秀的数字价值载体，也是价值互联网必然趋势。而优势、超越性主要来自于NFT的自描述能力，而且这种自描述能力是可以被定制化的。在这种观点之下，我们最熟悉的ERC721，只不过是最一般、最普通的NFT，大家在ERC721里面放图画、收藏品、艺术品等。如果我们开脑洞在里面放入其他的东西，实际上它的应用空无限广阔，这会使得我们这个行业出现一次大规模的创新浪潮，包括元宇宙、艺术品、金融票证、游戏等领域。基于 NFT 的价值互联网仍需要大量基础设施创新。我们也希望，Solv 作为一个中国团队在这个伟大进程中能够扮演一个重要的角色。谢谢！
