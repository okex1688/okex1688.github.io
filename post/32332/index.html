1996 年 1 月，比尔·盖茨发表了一篇题为《内容为王》(Content is King) 的文章，后来成为了早期互联网经典文章之一。在该文章中，他描述了互联网的特征，正是这些特征奠定了创作者经济 (creator economy) 的基础。他写道：“互联网令人兴奋的事情之一是，任何人只要有一台个人电脑和一个调制解调器，就可以发布他们创作的任何内容。”

盖茨的文章因其对互联网发展方向的先见之明而被人铭记，但不太为人所知的是，他还发出了一个警告：“要想让互联网蓬勃发展，必须让内容提供者从自己的作品中获得报酬。 ”“长期前景是好的，但我预计短期内会有很多失望。”

盖茨的分析是走在时代之前的。诚然，互联网让几乎所有人都有可能_在网上发布内容_，但同样不可否认的是，在《内容为王》发表 25 年后的今天，作为一名内容创作者要想赚取可观收入已被证明是**难** 以实现 的 。

创作者们的亲身经历说明了这一点：Spotify 上 90% 的流媒体版权费流入了排名前 1.4% 的音乐人手中；排名前 1% 的主播获得了 Twitch 一半以上的收入；1% 的播客获得了大部分的播客广告收入。

“在我看来，我们并没有处于扩张时期，”一位音乐家在谈到 Spotify 上的流媒体收入时对《纽约时报》说道。“从音乐家个人的角度来看，我们的劳动报酬呈下降趋势。”这并不是必然的，也不是初出茅庐的艺人所独有的问题——这影响着 99% 的创作者，包括那些拥有数百万粉丝的名人。当他们甚至挣扎着在网上维持基本的生活时，有些东西人为地阻碍了他们。

互联网本应迎来媒体的黄金时代——一个无限丰富的世界，在这里任何人都可以创作他们想要的东西，每个人都可以找到他们感兴趣的东西。但是，尽管盖茨关于_通过在线内容赚钱_的预测被证明是正确的，但大部分的钱都绕过了创造内容的创作者，而是落在了聚合这些内容的平台 的口袋里。

这将是一个 Web2 互联网 如何_打破_传统媒体商业模式，以及 Web3 的出现如何标志着这种商业模式的_颠覆_，使天平向有利于创作者的方向倾斜的故事。由于没有内置在 Web2 互联网上的原生盈利方法，主流的商业模式是不透明的、基于广告的、依赖于封闭式花园的网络，这给了平台巨大的优势；而即将到了的是，Web3 新的商业模式和技术有望开启一种经济机会，为艺术家和创作者带来一个真正的创作黄金时代 。

注意力经济 & 互联网的原罪

在这个 Web2 互联网如何打破传统媒体商业模式的故事，其核心是一个简单的事实，即互联网不是为了促进资金流动而建立的 。支付没有被嵌入到互联网的基础设施中 ——因为这被认为风险太大。马克•安德森 (a16z联合创始人) 称这是“互联网的原罪”。

支付基础设施的缺乏正是为何_互联网在很大程度上是通过广告盈利_的原因。与其要求用户拿出一张信用卡，然后在网站上输入他们的信息，互联网平台可以无障碍地、间接地通过另一种资产来盈利 ：用户的注意力 。这促使权力从控制内容创作和发行的传统媒体守门人——出版商、唱片公司和电影制片厂等——向那些大规模吸引消费者注意力的互联网平台 转移。

科技博客 Stratechery 的创始人 Ben Thompson 写了大量关于平台 (他称之为“聚合器”(aggregators)) 是如何通过聚合需求赢得了消费者的注意力，并获得了巨大的收入和影响力。比如，YouTube 拥有超过 20 亿的月活跃用户，Facebook 拥有近 30 亿用户，Spotify 拥有 3.65 亿用户。这些庞大的观众数量带来了_巨大的广告收入_。事实上，仅谷歌和 Facebook 就占据了 2020 年数字广告收入的_一半以上_。

广告的商业模式深刻地塑造了平台设计产品的方式。浏览量 (观看次数) 被汇集到已经很受欢迎的内容和创作者那里，创造了一个成功的幂律。关于用户偏好和行为的数据是平台最有价值的资产 ，因此他们封闭自己的生态系统，将用户锁定在自己的网络中，以积累最大的_专有数据库_。

基于广告的收入模式也对内容创作者 产生了巨大的 (负面) 影响。创作者被迫寻求尽可能广泛的受众，并创作吸引广告商的内容。这种商业模式 ——或者缺乏这种模式——对创作者的谋生方式和他们所创作的内容有着深远的影响 (鼓励病毒式传播的、吸引眼球的和励志的内容，同时抑制利基的、有深度的内容)。Web2 互联网最大的影响可能是由于没有可行的商业模式而_无法诞生的创作者和从未被创造出来的创作_。

从注意力经济到所有权经济

以平台为中心、以广告为动力的注意力经济或许已经赢得了 Web2 时代，但它的胜利_并非必然或终局_。创作者对平台的耐心正在迅速增长的_合理性危机_中逐渐消失——即创作者们开始质疑平台对他们的作品、他们与粉丝的关系以及他们获取回报的方式施加如此巨大的控制的权利 。

与此同时，新一代技术正在出现，有望改变创作者经济中的力量平衡 。如果说早期互联网/Web1时代更青睐出版商，Web2 时代更青睐平台，那么下一代创新——统称为 Web3 ——将权力和所有权的天平重新向创作者和用户倾斜 。

这种情况将主要通过四种方式发生：

  1. 引入数字稀缺性 ，恢复创作者的定价权。

  2. 让支持创作者成为一种投资行为 ，而不仅仅是利他行为。

  3. 引入新的可编程经济模型 ，将财富分散到整个创作者领域。

  4. 最重要的是，通过为创作者创造途径，让创作者不仅拥有自己创作的内容，还拥有平台本身 。

综合起来，这四个转变正汇聚在一起，创造 一个新时代 ，在这个时代，新的激励措施会奖励新的行为，让互联网有机会以集体的方式按下“重置”按钮，走向更公平的价值分配。

让我们依次讨论一下。

  • NFTs & 数字稀缺性的引入

稀缺性的名声不好，但这不仅仅是消费者缺乏选择的问题：这更是关于生产者权力的问题，也即在文本的情况中，创作者从他们的创作者获得可观收入的能力 。在我们当前这个充满着大量以平台为媒介的内容世界中，稀缺性并不存在，内容可以很容易地在互联网上复制。虽然有时候可以通过会员制或数字购买 (比如出售电子书、专辑或内容订阅) 让创作者接近稀缺性，但底层内容依然可以被无休止地复制。这种稀缺性的缺乏导致了创作者的内容被非法复制和分发的问题。

NFTs (非同质化代币) 作为一种技术令人兴奋的一个原因是，它们让创作者能够重新获得对自己内容的_控制权_，并重新引入了有助于实现盈利的_稀缺性_。在将他们的作品代币化为 NFT 的过程中，创作者得以创建一条有关作品所有权和来源的_可验证链上记录_。这个过程最终将铸造出一种独一无二的资产 (NFT)，该资产可以追溯到该艺术家。对创作者的作品充满热情的粉丝将愿意为这种权威的作品支付更多的钱，让创作者能够更好地捕捉粉丝的付费意愿。其最终影响不容小觑：内容创作者不再需要拥有数百万粉丝来维持生计，而是可以依靠少数充满热情的人的资助来生存 。

比如，蓬勃发展的音乐 NFT 市场 展示了这种效应。在流媒体平台上，一首流媒体音乐每播放一次都贡献相同的收入 (在 Spotify 上每首歌被付费账号播放一次产生 0.004 美元的版权费)，不管粉丝对歌手的热爱程度有多深。相比之下，在 Catalog 或 Sound 等平台上，超级粉丝以数千美元的价格购买 NFT 音乐 ，而创作者在以前需要播放数千万次才能获得这些收入。Brett Shear 是一位 NFT 音乐收藏家，他拥有 Catalog 平台上的 45 首歌曲。他告诉《时代》杂志：“就像你买了一件你想放在公寓里的艺术品一样，我也想听这些音乐并享受它——拥有它是一种不同的感觉。”

购买 NFT 类似于收集现实世界的商品，让粉丝感觉更接近艺术家，并拥有一些稀有的东西。数字稀缺性和独特性 ——这在 Web2 中是缺失的——在区块链中得以实现 ，为创作者带来了一种新的商业模式，减少平台的经济控制 。

令人兴奋的是，通过 NFT 引入稀缺性并不意味着对底层媒体 (比如一首歌曲) 的访问受到限制，就像付费墙或付费数字下载一样。NFT 底层的实际媒体仍然是公共产品，任何人都可以免费使用。那些认为这削弱了 NFT 的稀缺性 (“右键单击并保存”) 的人根本没有抓住要领。

  • 资助+：支持创作者成为一种投资，而不仅仅是一种利他主义行为

在之前的《100 True Fans》文章中，我描述了创作者可以利用粉丝的_利己主义_来以更高的价格盈利。通过交付实质性的价值和成果， 创作者可以更有效地通过更少的粉丝赚钱和谋生 ：

这代表着_传统的资助模式_ (即用户付费**使创作者获益** ) 向_价值模式_的转变，在价值模式中，用户愿意为**使自己获益** 的东西支付更多的钱。

Web3 将这个想法带到了新的层次，因为所有的代币都是一种投资，不仅能为创作者提供资金 (收益)，而且如果代币的价值增值，持有者也会受益 。Jesse Walden 将“资助+ ” (patronage+) 定义为有盈利可能性的资助 ，这是一种通过代币化所有权引入的现象。在 Web2 中，由于没有 NFT 或社交代币等链上所有权记录，这种投资元素是不可能的 (想象一下视图转售一条从 TikTok 下载的视频，这是无法实现的)。

有什么实际的“资助+”例子吗？今年早些时候，博客平台 The Generalist 的 Mario Gabriele 众筹了 20 ETH (编者注：购买者可获得相应份额的 GENERALIST 代币)，以目的是让一群分析师对 Coinbase 进行深入研究，并委托创作伴随该论文的艺术品。众筹者在简报和艺术作中获得了一定比例的股份，所有这些都被铸成了 NFT。相关的 NFT 销售总额为 28.6 ETH，使得众筹者在短短几周内获得了 43% 的回报。

除了资助和投资之外，另一个好处是成为志同道合的个人团体 (社区) 的成员 。Crypto 领域许多成功的众筹和 NFT 交易都是由用户想要成为某个社区成员的愿望驱动的，而社区是需要代币所有权才可进入的。这与我在《100 True Fans》一文中所写的一个现象相呼应：“人们愿意为专属的、差异化的内容支付高价，并进入一个由志同道合的人组成的网络。”

对于粉丝来说，盈利的可能性加大了他们支持创作者的动机。有趣的是，这还在创作者轨道中引入了一个 Web2 中是从未存在过的全新部分：投机者 。重要的是，所有这些用户——通过成为与创作者的成功相关的资产的所有者——都有动机去帮助扩大创作者的工作。

  • 新的可编程经济模型

创作者经济的一个事实是，创作通常是一种协作行为 。YouTube 的创作者们互相在对方的视频中做主角；音乐家们从彼此的作品中汲取灵感；TikTok 的视频往往是由多位创作者的作品组成的：一位创作者的原声音乐，另一位创作者的编舞。

不幸的是，Web2 系统并不是用来奖励或追踪这种协作性的。在这个赢家通吃的算法平台世界里，价值往往只流向病毒式传播的创作者，而忽略了参与作品创作的其他人。这导致了创作者们的罢工和不满，他们觉得自己的贡献没有得到认可和获益。

而在 Web3 中，代币化的承诺意味着可以建立版税，让某个协作性作品的_整个贡献链 (即每个贡献者)_ 都能从中获利。早期的例子包括 Mirror 和 Foundation 的_收益分流功能_，该功能会自动将收益路由到对某个项目有贡献的各个以太坊地址。

可以想象，任何数字作品都可以利用来自通用媒体库的元素，且收入分成和归属自动计入。Nir Kabessa 写了关于“模因经济”的文章，在模因经济中，在互联网上被重新混合和传播的想法可以成为价值创造的基础：

那些知名的模因 (表情包) GIF 会链接到其 NFT 地址 ，这样当某人想要在其文章中分享该原始的 NFT 时，他可以提取该链上地址。对于模因来说，这是非常强大的，因为这允许它们在每个平台上都能维持其属性和背景信息。因此，有关模因 NFT 的任何行为都可以在几乎所有平台上访问、读取和可用，对特定 NFT 的任何竞价、交换和交易都会被添加到其元数据中。

除了模因 (memes)，如果每一件创意作品都被链接到其来源的链上记录，那么就有可能在互联网上追踪该作品，创作者也可以从对其作品的后续使用来赚钱。

  • DAOs & 社区所有权

我们认为，创作者领域不平等的一个根本原因是，平台通过拥有对内容的生产和传播方式的所有权，对创作者和他们的作品施加了过大的控制。挑战这种控制的最直接方式是改变谁拥有生产资料 。

DAOs (去中心化自治组织) 和 其他集体所有权机制 创造了一种打破平台对创作者环境的中心化控制的途径，使创作者能够在没有外部中介规定参与条款的情况下进行协作 。在 DAO 中，治理系统是由成员决定的，没有外部股东要求榨取利润的压力。相反，在创作者 DAO 中，所有者也即参与者：创作内容、分发内容、消费和重视内容的人。

一个逐渐实现去中心化的创作者平台 的早期例子是 SuperRare ：它是一个 NFT 市场，它将代币分发给该平台的艺术家和收藏家，他们负责治理策展、管理 DAO 金库和未来的产品方向。还有一些组织则以启动社区和代币为先：ElektraDAO 是一个由 42 名音乐家、视觉艺术家、开发者和战略家组成的社区，他们开发了一款以音乐为核心的交互式

DAOs 带来的承诺是通过利益相关者至上来协调激励机制，并消除了攫取价值的需求。其结果是：一个民主化的、去中介化的内容格局，在这里，创作者们可以拥有他们的作品、其分发方式以及估值方式的控制权 。

除了 DAOs 之外，Web3 固有的互操作性使得平台锁定可能是一个比在 Web2 中危害小得多的问题。Web3 的原子单位是帐户 (地址)，用户可以用他们的密钥对控制自己的账户，并且可以在任何应用程序或协议中使用该账户。由于所有智能合约都是透明的，可以公开检查，从而使得不透明和任意的幕后交易更难实现。虽然 Web3 还处于起步阶段，但它正朝着更加开放和基于标准的理念发展，这有利于创作者和用户。

创作者的权力和所有权

自从互联网存在以来，思想家和哲学家们就一直在描绘互联网将使什么成为可能的乌托邦愿景——尤其是在媒体领域。这种乌托邦式的愿景并没有实现。至少现在还没有。

在过去，我把所有权称为初始系统条件，其他一切都是从它开始的。所有权决定激励机制。它决定了机会。它决定了财富是如何被创造的——以及为谁创造。在过去的十年里，我们生活在一个所有权集中于少数几个中心化的技术平台的时代，这些平台拥有数据、终端用户关系以及分发内容和从中盈利的手段。在此期间，虽然用户生成内容 的创作出现了爆炸式增长，这也导致了绝大多数创作者对少数新“看门人”的依赖和普遍的倦怠，以及创作者经济的不可持续性。

幸运的是，即将到来的新发展代表着权力均衡向创作者的转变。随着 Web3 ——数字稀缺性、作为投资行为的资助的成倍增加、可编程的商业模式和社区所有权 ——带来的关键新能力，我们正处于互联网新的创造性文艺复兴的风口浪尖。我相信 Web3 有潜力为每一个在互联网上贡献和创作的人开启不可思议的机会：一个我们一直期待的真正的内容黄金时代 。
