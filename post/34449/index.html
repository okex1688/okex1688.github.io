Web3 是加密的逻辑结果。在数字货币和区块链的去中心化的世界中，网络的去中心化几乎是不可避免的。

虽然 Web3 的概念确实在 2021 年席卷了大部分技术领域，但它还没有发展成一个成熟的领域，目前是由平台、产品和服务的具体例子来定义，这些例子对我们的生活产生了重大影响。

许多行业人士表示，Web3 将在 2022 年经历可观的增长，越来越多的初创公司、组织和大公司创造出有助于普及的、有价值的产品。

然而，Web3 不是加密行业的一个独特、独立的领域，它更像是整个行业的一个范式，一种概念化加密发展的方式，因为它集成了各种不同的链。这包括NFT、DeFi和元宇宙，它们将共同确保 Web3 迎来一个非常强劲的一年。

Web3来了？是的！但它是什么？

加密情绪分析平台Trade The Chain 的研究分析师 Nick Mancini 说：“Web3似乎是2022 年的流行词，但我们认为这只是描述该行业的另一种方式，市场目前正在迈向一个互操作性的时代，在这个时代，NFT、数字资产都和谐共存。”

Mancini 并不是唯一一个认为 Web3 是加密和区块链应该用于什么概念的人，因为它是加密的一个离散子领域。 Livepeer ( LPT ) 的产品开发人员和Web3 Index的创始人Adam Soffer 也有类似的立场，他觉得Web3 更多的是一种与治理和所有权相关的东西，而不是一组定义的技术。他说：“有一致的激励措施，以便服务、工具、平台或协议的用户从他们所做的贡献中获得回报和收益。用户对如何开发服务、工具、平台或协议也有发言权，这就是自治。”

对于 Soffer 来说，今年涉及 Web3 的一个趋势将与社区证明它的确切含义有关。在他看来，目前围绕这个概念有点混乱，太多人认为它指的是完全去中心化的权力。

Soffer 说：“虽然可以去中心化，但一个项目不一定需要完全去中心化才能成为 Web3，任何项目都不可能完全去中心化。” 

对于NFT Storage的联合负责人以及Protocol Labs的业务开发负责人 Jonathan Victor 来说，Web3 以及 NFT、DeFi 和 元宇宙是同一事物的不同方面。他说：“我们正在谈论技术堆栈的不同层次。NFT、DAO、元宇宙——所有这些趋势都意味着一个可信的中立基础层。”

数十亿美元用来支持每一个有价值的想法

假设 Web3 本质上与整个加密圈密不可分，今年它将见证可观的增长。

去中心化社交市场Calaxy的首席运营官兼联合创始人Ceesay说：“去年对于 Web3 及其衍生产品来说是重要的一年，我预计这一趋势将持续到 2022 年。随着行业达到难以预测的高度，社区机会将呈指数级增长，扩大这些子行业所覆盖的人口统计数据。”

Ceesay 补充说：“作为 Web3 的一个子集，NFT 和元宇宙将比 Web3 的其他元素更能吸引加密圈的受众。这主要是因为它们可以与内容创作者、品牌等提供的社交和身体体验交织在一起。DeFi 和 DAO 的发展将为核心加密用户提供必要的讨论空间，以团结并共同克服个人在传统金融世界中经常面临的挑战。”

其他人则认为，在人们对 Web3 的含义有更深入的了解之前，我们不会看到任何特定的加密领域在 Web3 上大放异彩。

Nick Mancini 说：“追求 Web3 项目的组织仍将继续作为一个整体的风险投资，我们将看到更多的企业风险投资部门进入这一领域。尚不清楚任何一个部门或组织完全知道该如何处理元宇宙和Web3领域，因此您将继续看到数十亿资金涌入。”

尽管如此，Mancini 确实建议至少有几个行业将特别倾向于 Web3 理念和组织方法：“游戏和贸易，这两个行业在过去几十年中一直处于增长迹象。”

鉴于 Web3 通常是关于对其社区的平台或应用程序的去中心化权力和治理，其他行业参与者认为Web3 将为具有创新和创造力文化的社区来驱动品牌和公司的一大动力。

Adam Soffer 说：“与客户保持紧密联系的品牌将看到 Web3 项目的巨大成果。”

对Solo Ceesay 而言，内容创作者将引领 Web3 的发展和成长，包括作家、艺术家、音乐家、电影制作人和其他创意人员。他说：“内容创作者将继续成为 Web3 的先驱，考虑到这些社区中一直在讨论，例如所有权和货币化的想法是影响者目前面临的一些最大问题。Web3 技术提供了一种解决方案，允许在粉丝和内容创建者之间更直接、更有效地转移价值——这与中心化各方在 Web2 中对这种价值转移的束缚相去甚远。”

据BitTorrent (BTT)首席执行官Steve Liu称，在内容创建的连续性中，共享内容的社交媒体将成为 Web3 的另一个成熟领域，他说：“Web 3.0 的下一个热门竞争领域将包括利润丰厚的社交媒体领域，它正越来越多地被少数集中式平台主导。”

Web3 基础设施

尽管许多行业都将倾向于尚未定义的 Web3，但其今年增长的关键方面之一应该是开发能够实现其理想的去中心化基础设施。这意味着为去中心化 Web3 应用程序提供基本构建块的协议。

Adam Soffer 说，他创立的 Web3 Index 对提供 Web3 基础设施层的网络协议特别感兴趣，因为这些是新范式的“基础”。“我们在 Web2 应用程序上构建一切时是中心化的。在某些时候，我们需要摆脱这种束缚。”他说。

他指出Arweave ( AR )、Arkash、Helium ( HNT )、Graph ( GRT )、Pocket和Sia (SC)等协议处于 Web3 基础架构的前沿。换句话说，如果 Web3 今年要发展，这些协议将是值得关注的。

正如他所说，这些协议正在帮助促进可互操作的基础架构，从而实现 Web3 中的各种新创新，甚至还可支持 Web2 项目。
