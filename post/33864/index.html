区块链和加密世界由复杂的知识体系构成，我将这一体系分为五个知识阶层，层层深入地揭开加密冰山的面纱。

入门阶层：大多数初学者都熟悉的主题

1. 区块链

一个存在于网络中的分布式、去中心化公开账簿。

2. 账簿

一个用于存储网络区块的不可变数据库。

3. 比特币

中本聪创造的第一代数字货币。

4. 以太坊

一个旨在通过使用智能合约和 dApp 超越数字货币的通用平台。

5. 公链

采用替代方法来解决可扩展性三重困境的一层区块链。

6. Meme Coin

源自互联网 meme 或具有其他一些幽默特征的加密 Token，比如 Dogecoin、Shiba Inu 等。

第二阶层：大多数 crypto 原住民都熟悉的主题

1. 智能合约

在满足指定条件时运行的程序，用于自动执行协议。

2. dApps

提供与普通应用程序类似的功能，但是在点对点网络上运行的应用程序。

3. Gas

在以太坊等公链上，gas 用于描述在网络上执行特定操作所需的计算能力的度量单位。

4. Gas Limit

用户愿意在特定交易上花费的最大 Gas 量。

5. CEX

作为买卖加密 Token 的可信中介交易所，属于托管机构。

6. DEX

去中心化链上交易所，允许用户直接在链上进行点对点交易，是非托管的。

7. 共识机制

一种容错机制，用于促使分布式节点之间就网络的单一状态达成一致。共识机制确保所有节点彼此同步并就交易达成一致，然后将这些内容添加到区块链中。

8.PoW

Proof of work（工作量证明）是共识机制的一种类型，它需要称为「矿工」的网络成员花费精力解决任意数学难题以验证交易并将其添加到区块链中。

9.PoS

Proof of stake（权益证明）是另一种共识机制，它要求用户质押 Token 才能成为网络中的验证者，而验证者负责的内容则与工作量证明机制中矿工的工作相同。

10. 区块

区块链中的数据结构，用于永久记录和存储链上的交易数据。

11. 矿工

负责添加新交易或验证其他矿工所生产区块的计算机或计算机组。

第三阶层：经验丰富的加密投资者才熟悉的主题

1. DeFi

Decentralized finance（去中心化金融）通过使用区块链上的智能合约技术，来提供不依赖于经纪人、交易所或银行等中介的金融工具。

2. NFT

Non-Fungible-Token（非同质化 Token）是存储在区块链上的、不可互换的数据单元，可用于出售或交易。NFT 数据单元的类型可以涉及照片、视频和音频等数字文件形式。

3. 元宇宙

Metaverse 是一个复杂的、大规模且可互操作的实时渲染 3D 虚拟世界网络，能允许无限数量的用户同步和持续地在其中进行体验。

4. 可扩展性三难困境

以太坊创始人 Vitalik Buterin 提出的区块链可扩展性三难困境指出，在去中心化、安全性和可扩展性这三个属性之间进行权衡是不可避免的，你可以照顾到一个或两个属性，但不能同时兼顾三个。

5. 区块难度

区块难度是工作量证明挖矿中最重要的内容之一，它由网络哈希率得出，用于确定矿工能够验证区块的速度。

6. 比特币减半

比特币每 4 年就会经历一次「减半」事件，通过将矿工的奖励减半以减少进入流通供应的比特币数量，旨在收紧供应量，直到 2100 万枚 BTC 被全部开采完。

7. Layer 2

Layer 2 是指构建在现有区块链系统之上的辅助框架或协议，这些协议的主要目标是解决底层网络面临的交易速度和扩展性问题。

8. Yield Farming

Yield Farming 就好比通过在银行存款赚取年化利率。从本质上讲，Yield Farming 是流动性提供者将其资产锁定在流动性池中并以交易费用或治理 Token 排放的形式获得激励的过程。

9. AMM

Automated Market Maker（自动做市商）是支持 DEX 的底层协议，它允许资产通过使用流动性池进行无许可交易，而不是传统的买卖方市场。

10. 流动性池

流动性池是锁定在智能合约中的众筹 Token 池，用于支持去中心化交易所的交易。比如，用户在使用 Uniswap 等流动性池协议进行交易时，不需要买卖双方匹配订单即可完成交易。

11. 加密借贷

加密借贷类似于传统借贷的概念，加密 Token 贷方能够将其闲置资产借给借款人，借款人反过来支付贷方资产的利息。

12. 合成资产

合成资产本质上是 Token 化的衍生品。在传统金融中，衍生品是交易者不拥有但想要买卖的股票或债券的代表，而加密合成资产能允许投资者在区块链上交易任何加密资产。

13. 加密指数

加密指数是接触一篮子不同加密资产的方式，每个指数都由各种不同的加密资产组成。这些指数使投资者无需单独购买即可获得多种 Token 的敞口。

14. 预言机

区块链预言机是将区块链连接到外部系统的实体，从而使智能合约能够根据现实世界的输入和输出执行特定代码。

15. 治理

在没有中央权威的情况下，去中心化网络所依赖的治理结构。区块链治理通常采用各种机制来决定项目发展方向、持续进行的更新，以确保网络高效地运行。

16. 稳定币

价格与加密 Token、法定货币或交易所交易商品挂钩的加密 Token，其中算法稳定币旨在实现价格稳定和资产的流通供应。换句话说，算法稳定币使用的算法能保证在价格上涨时发行更多 Token，并在价格下跌时从市场上买入，一次保证价格的稳定。

17. 闪电贷款

闪贷是一种新型低抵押贷款，它使用户能够即时获得加密贷款，同时无需提供抵押品。

18. 套利

加密套利是一种交易策略，投资者利用数字资产在多个市场或交易所的轻微价格差异，在一个交易所购买资产并立即在价格更高的交易所出售。

19. 创作者 Token

创作者 Token 允许创作者、社交影响者、运动员、音乐家等推出他们自己的加密 Token，以将自己的社群货币化并扩大自己的受众群体。

20. 协议拥有流动性

Protocol Owned Liquidity 是 Olympus DAO 开创的一种提供流动性的新方法，该协议不依赖于向流动性提供者提供激励措施，而是向买家出售折扣的 Token，以换来充当抵押品的 Token，构成协议国库的一部分。

21. 流动性质押

流动性质押允许 Token 持有者质押他们的资产并获得可用于各类 dApp 的流动性衍生品。使用流动性质押，ETH 2.0 质押者能够同时质押他们的以太坊，同时保持流动性。

22. DAO

去中心化自治组织（Decentralized Autonomous Organization）是由计算机程序编码的规则所代表的组织，该程序是透明的，由组织成员控制，不受中央权力的影响。

23. 互操作性

区块链互操作性是指不同区块链网络之间交换和利用数据以在网络各自的区块链之间移动独特类型的数字资产的能力。

24. IPFS

星际文件系统（InterPlanetary File System）是一种点对点网络协议，用于在分布式文件系统中存储和共享数据。IPFS 使用内容寻址来单独标识在连接所有计算设备的全局命名空间中的每个文件。

25. ENS

以太坊域名服务（Ethereum Name Service）是建立在以太坊上的域名查找服务，允许加密用户将他们的机器可读地址转换成人为可读的地址，你可以将它看作是公共以太坊地址的昵称生成器。

第四阶层：正式深入兔子洞

1. EVM

Ethereum Virtual Machine（以太坊虚拟机）是开发人员可以用来在以太坊上创建 dApp 的软件平台，它是所有以太坊账户、智能合约以及以太坊的整个状态历史所在的地方。

2. 内存池

在区块链术语中，内存池（Mempool）是尚未添加到区块且仍未确认的交易的等待区域。

3. 领跑者

领先者（Frontrunners）是以太坊上的一种机器人，它不断扫描内存池以寻找有利可图的交易。领跑者与其他机器人一起参与优先 gas 拍卖（PGA），以获得他们在一个区块中的交易的优先权。

4. 拜占庭容错力

在区块链中，由拜占庭将军问题衍生出的拜占庭容错力是指网络在验证节点出现故障和运行不一致时的容错能力。

5. 女巫攻击

女巫攻击（Sybil Attack）是对计算机的一种攻击，攻击者通过创建大量假名身份并使用它们获得不成比例的网络影响力来破坏服务器的信誉系统。

6. SHA-265

SHA-256 代表 256 位安全散列算法，用于提升加密安全性。密码散列算法能产生不可逆且唯一的散列，可能的散列数越大，两个数值创建出相同散列的机会就越小。

7. 完整节点

完整节点是下载并检查区块链中的每笔交易是否有效的节点，这需要大量资源和数百 GB 的磁盘空间，但它们是最安全的节点，因为它们不会轻易被无效交易区块所欺骗。

8. 轻客户端

如果计算机没有足够的资源来运行完整节点，则可以运行轻客户端。轻客户端不下载或验证任何交易，它们只验证块区块头，并假设该块仅包含有效的交易记录。

9. 梅特卡夫定律

梅特卡夫定律指出，网络的价值与网络中节点数的平方成正比。假设一个网络有 10 个节点，则其价值为：10x10=100。

10. 林迪效应

林迪效应是一种理论化的现象，通过这种现象，一些不易折损的事物（如技术或年龄）的未来预期寿命与其当前年龄成正比。

11. 侧链

侧链是一个单独的区块链，它使用双向挂钩连接到其母链。双向挂钩可以在母链和侧链之间以预定速率实现资产互换。

12. Plasma

Plasma 是以太坊 Layer 2 扩展性解决方案，它是一个允许创建使用以太坊主网作为信任和仲裁层的「子」区块链的框架。

13. Optimistic Rollups

Optimistic Rollups 使用智能合约将交易数据从以太坊主网中继到 Layer 2 网络，其中排序器可以将多个交易记录捆绑成一个批次，然后通过单个交易将该批次提交回主网链上。

14. 零知识证明

在密码学中，通过零知识证明（Zero-knowledge proof），一方可以向另一方证明给定的陈述是真实的，而证明者除了陈述确实是真实的事实之外不会再传达任何其他信息。

15. ZK Rollups

Zero-knowledge Rollups 是另一种以太坊扩容方案，可将交易记录从以太坊主网中取出。ZK Rollups 生成证明交易有效性的加密证明，每批交易记录都包含自己的有效性证明，并提交给主网。

16.Keepers

Keepers 是一类协议外部所有的账户，并受激励在 DeFi 协议中执行特定的操作。Keepers 以固定费用或激励措施的固定百分比的形式获得奖励，例如收取清算抵押债务的费用等。

17. 信标链

信标链是以太坊的升级版本，最终将取代当前使用的传统以太坊区块链。其目的是引入 PoS 新共识机制并协调扩展出的分片及质押者网络。

18. 合并

合并是指当运行在 PoW 上的以太坊主网与作为 PoS 链的 Beacon 链集成时将会发生的事。

第五阶层：最深奥的加密知识

1.MEV

最大可提取价值（Maximal Extractable Value）是指矿工可以通过他们在他们生产的区块中任意包含、排除或重新排序交易以赚取利润的能力。

2. 分片

分片是指将整个以太坊网络分成多个部分，故称为「分片」。每个分片都将包含自己的独立状态，从而大大减少了网络拥塞现象，并增加了以太坊的交易吞吐量。

3. 数据可用性证明

数据可用性证明（Data Availability Proofs）是一项新技术，它允许用户通过仅下载该区块的一小部分来以非常高的概率检查该块的所有数据是否已发布。

4.Merkle 树

Merkle 树是区块链技术的基本组成部分，它是一种数学数据结构，由不同数据块的哈希值组成，作为一个区块内所有交易的汇总。
