「SOL Community > ETH Community.」

Orangie 发了这样一条推文。Orangie 是 Twitter 上的一个拥有超过 160k 粉丝的 NFT KOL，虽然他现在仍然用 MAYC 作为头像，但很明显他已经把目光转向了 Solana NFT。随着 Okay Bears 的爆火，越来越多的 NFT 玩家注意到了 Solana 这条公链。

Solana NFT 在 OpenSea 上崭露头角

NFT 市场里，Solana 的支持者历来以「交易速度快」、「几乎没有手续费」、「低门槛」等优点来宣传布道。但与此同时，对 Solana NFT 的批评也有很多：「到处都是 rug pulls」、「经常宕机」、「不像以太坊那样去中心化」等等，甚至嘲笑它是那些支付不起 Gas 的 NFT 玩家退而求其次选择的「穷人区块链」。

所以真实的 Solana NFT 市场是什么样的呢？本文将从近几个月以来 NFT 市场的变化着手，通过与 Ethereum NFT 市场的对比来探讨 Solana NFT 市场的潜力与前景。

NFT 市场：「悬崖已现」

NFT 的热度从 2020 年开始一路飙升，从 Cryptopunk、BAYC 出现，到 Doodles、Azuki、Moonbirds 等 PFP NFT 项目层出不穷，同时有内马尔、周杰伦、埃米纳姆等众多明星入圈让 NFT 逐步进入大众视野并以极快的速度为人们所熟知。

NFT 项目方们也尝试着从多个方面进行创新。从最开始的创造社交场景的头像 NFT 项目，不同的画风迎合不同的投资者的喜好；到后来的 Pass NFT，提供分析工具权益或是 Alpha 社区权益；发行代币、打造 IP、持续创造收益的「铲子」等各种新玩法不断被设计出来。

NFT 市场的总市值与交易额变化（来源：NFTGO）

在长达近一年的「NFT Summer」中，NFT 的总市值最高曾一度超过了 386 亿美元，新的社交叙事、强大的造富效应吸引了各路资本纷纷入场，NFT 市场空前火热，投资者们都想从这「点图成金」的故事里分一杯羹。

OpenSea 上基于 ETH 的 NFT 交易量持续下滑（来源：Dune）

随着时间来到今年的四五月份，整个加密市场在美联储加息预期下已尽显疲态，五月伊始 Yuga Labs 旗下的「Otherdeed for Otherside」系列的发售更是榨干了 NFT 市场里的最后一丝流动性，Luna/UST 暴雷事件使得投资者们恐慌抛售手中的各种投资资产，NFT 也不例外，Dune 上的链上数据显示，OpenSea 上基于 ETH 的 NFT 交易量持续下滑，最近几天的日交易量甚至不及峰值的十分之一。

「熊市真的要来了」。

进入熊市的过程里，资产缩水只是其中一方面，NFT 的投资者更怕的是流动性骤减。

诚然，与 FT(Fungible Token) 对比，NFT(Non-Fungible Token) 的流动性是最为紧要的话题，「Token 至少还能一键市场卖出回本一点，NFT 卖不出去真的就归零了」。流动性骤减反映到目前的 NFT 交易市场上：几乎所有的 NFT 交易量都下降了，不知名的 NFT 项目一天只有寥寥几单成交量甚至是零成交，迫切套现的投资者争相压低价格继续破坏价格 K 线的走势，早期建立的共识纷纷崩塌，项目方对此无可奈何。

在这样的环境剧变中，乐观的投资者们押注于 NFT 市场的未来，对潜在的增量资金翘首以盼，但危险之处在于，尽管 NFT 市场被寄予厚望，但它的用户和资金增速却已显著放缓。

相比于投资者的热情，NFT 市场寒冰一般的增长数据，像是横在所有 NFT 项目面前的一道悬崖。

Solana NFT 市场：「猛踩油门」

NFT 市场的增量资金不足，让大部分 NFT 项目难以走出存量资金搏杀的「红海」。 头部的蓝筹项目凭借先发优势和头部优势在吃尽市场红利后尚且可以喘息，而那些最近在 ETH 链上发行的新项目面临着诞生即死亡的窘境，之前就存在于市场上的非头部的 NFT 项目也将走向崩溃。

即使如此，面对 NFT 市场的悬崖，Solana 链上的 NFT 项目非但没有减速，反而踩起了油门。

Magic Eden 上基于 Solana 的 NFT 交易在四五月份显著增长（来源：Dune）

Solana NFT 总地板价在过去几个月持续上升（来源：SolanaFloor）

从 Dune 分析数据可以看到，作为 Solana 链上的第一大 NFT 交易平台 Magic Eden 上的 NFT 交易次数在四五月份显著增长。SolanaFloor 上的数据也显示在过去几个月里，基于 Solana 的 NFT 的总地板价是持续上升的。随着 Solana NFT 的持续升温，Magic Eden 的日交易量甚至曾一度超过了 OpenSea。

毫无疑问的，这些数据在 NFT 寒冬里是少有的积极信号，至少可以在某种程度上证明，NFT 市场在 Solana 链上尚且有一定的热度，而且这种热度还在持续上升中。在这其中，一个名为 Okay Bears 的 NFT 项目表现十分亮眼。

Okay Bears 在发售后地板价一路飞涨（来源：Sol Sniper）

4 月 27 日，Okay Bears 在 Solana NFT 市场 Magic Eden 上以 1.5 SOL 的价格公开发售，此后价格一路飞涨，地板价最高一度超过了 260 SOL。在过去的不到一个月的时间里，Okay Bears 创造了十分惊人的交易数据：累积交易次数超过 2 万次，累计交易量超过 140 万 SOL，在 Solana NFT 中总市值位列第二。

Okay Bears 在极短的时间内产生的巨大的财富效应让它迅速爆火，它为 Solana NFT 市场的升温添了一把柴火，证明了在这里一样也存在着「造富机器」。

Solana vs Ethereum ：「公链之争」

作为最早的公链之一，以太坊（Ethereum）曾被冠以「公链之王」的美誉，生态繁荣，各类 DAPP 百花齐放。其他的公链中不乏挑战者，众多的「以太坊杀手」轮番上阵，Solana 就是其中之一。

交易速度和交易手续费

Solana 从最开始就致力于成为「速度最快的高性能公链」。

它采用了历史证明机制（PoH）、Tower BFT 算法、Turbine 传播协议、海平面引擎（Sealevel）等技术搭建了完全不同的系统架构，这让它比其他区块链有着更高的速度和更低的成本。

比特币每秒可以处理大约 7 笔交易（7 TPS），以太坊每秒可以处理 30 笔交易，而 Solana 目前每秒可以处理 65,000 笔交易，它的 TPS 可以与 Visa 和 Mastercard 相媲美，使其在速度和全球可扩展性方面迅速成为行业领导者。虽然 Eth 2.0 未来可能会大幅提高性能，但其上线计划也反复被推迟。

各个链的平均交易手续费（来源：A16Z）

除了交易速度，Solana 的交易手续费也比 Ethereum 低得多。以太坊上的一笔交易通常要花费 10 美元以上的 Gas Fee，而在 Solana 上做一笔交易仅需要 0.00001 美元。

5 月 1 日有超过 70,000 ETH 被烧毁（来源：glassnode）

以 Yuga Labs Otherside 项目的发售为例，来自 Glassnode 和 Data Always 的数据显示，在 5 月 1 日 Otherside 发售期间，当天 Ethereum 上花费了约 2.27 亿美元的手续费，有超过 70,000 ETH 被烧毁，用户平均手续费为 197 美元。

Yuga Labs 表示将退还用户因交易失败而花费的 Gas Fee

在 Otherside 发售后，Yuga Labs 在 Twitter 上发布了一条推文，称它将退还用户因交易失败而花费的 Gas Fee。「很明显，ApeCoin 需要迁移到自己的链上才能适当扩展。」由此可见，面对 Ethereum 上的高额手续费，即使是 Yuga Labs 这种龙头项目也是难以忍受的，NFT 玩家们更是不堪其扰。

去中心化程度

区块链的「三元悖论」

以太坊创始人 Vitalik Buterin 曾提出区块链的发展存在「三元悖论」（Trilemma, 又称不可能三角），它指的是：很难保证去中心化、安全性、可扩展性三者同时兼顾。 与 Ethereum 相比，Solana 在保证安全性和高性能的情况下，牺牲了一部分的去中心化。

从验证者节点的开发角度来看，Solana 目前每秒最多可以处理 65,000 个事务，高吞吐量意味着运行验证器的硬件要求极高。高门槛使得 Solana 基金会成为了唯一在 Solana 区块链上开发核心节点的实体，这意味着它可以控制网络，从而降低了网络的去中心化程度。Solana 网络曾多次宕机，自上而下的解决方案引来了很多质疑的声音：它看起来是一个「拥有更高级数据库的中心化银行」而不是所谓的去中心化区块链。

Messari 公布的公链初始代币分配

在不同的区块链上，将代币委托给验证者是保护网络安全的内在驱动力。因此在比较 Solana 和 Ethereum 网络的去中心化程度时，对比二者的代币分配也十分重要。

与 Ethereum 对比，Solana 内部人员持有的代币数量比例很高。据Messari 的数据，Solana 大约 48% 的代币由内部人士拥有，包括 Solana Labs 团队成员和风险投资机构。此外，另外三分之一的供应已分配给生态系统开发，在 Solana 的公开发售中只分配了相对较少的代币。而 Ethereum 通过公开发售分发了大约 80% 的代币，内部人士只持有总供应量的 15%。

Solana NFT 的前景：「弯道超车」

诺贝尔奖获得者 Herbert Alexander Simon 曾经指出：「在如今这种信息高速发展的时代中，注意力的价值将会超过信息。」

诚然，对于每天层出不穷的新 NFT 项目让人眼花缭乱，谁吸引到了资本的注意力，谁就赢得了先机。

DeGods 创始人 Frank 感谢 Okay Bears 让更多人注意到了 Solana NFT

Okay Bears 的爆火出圈，让更多人了解到了 Solana NFT 一样有很强的造富效应。Solana NFT 市值排名第一的项目 DeGods 的创始人 Frank 甚至发推表示感谢：「DeGod 的持有者要对 Okay Bears 表示感激，它们为 Solana 带来了更多的注意力。」

Ethereum 上的 NFT 利用先发优势，吸引了市场上绝大部分注意力，但面对 Ethereum「赢家通吃」的马太效应，Solana NFT 仍有机会实现「弯道超车」。

Magic Eden 助力 Solana NFT 发展

Magic Eden 牵头「SOLANA SUMMER」

作为 Solana 上第一大 NFT 交易所，Magic Eden 与 Solana 上的 NFT 交流颇多。前有「DeFi SUMMER」、「NFT SUMMER」，Magic Eden 借着 Okay Bears 爆火的东风，甚至想要发起「SOLANA SUMMER」。

三大 ETH NFT 交易所交易量的对比（来源：Dune）

在《ETH 还是 Solana 的 NFT：鸡生蛋的问题》一文中，作者 Mr.Fox 认为：即使 LooksRare、X2Y2 等 NFT 交易平台采用低手续费、代币奖励等「吸血鬼攻击」来与 OpenSea 竞争，但它们依然无法胜出的主要原因是同一个 NFT 在 OpenSea 上的流动性更高。事实确实如此，从 Dune 上的数据可以看到，OpenSea 对于 ETH NFT 的交易量占有率始终维持在 50% 以上。

但对于 Solana NFT，OpenSea 仍旧能够胜出吗？事实并非如此。

Okay Bears 在 OpenSea 和 Magic Eden 的交易量对比（来源：Dune）

从 4 月份开始，OpenSea 开始支持基于 Solana 发行的 NFT 进行交易，Okay Bears 爆火时曾一度冲上了 OpenSea 交易量的榜首。但即便如此，从 Dune 的数据可以看到，绝大部分 Okay Bears 都是在 Magic Eden 进行交易的。

Magic Eden 在 Solana NFT 市场交易量的占比（来源：Dune）

不仅仅是 Okay Bears，Magic Eden 在整个 Solana NFT 市场交易量的占比超过了 95%，也就是说，对于所有的基于 Solana 的 NFT，Magic Eden 上的在流动性方面占有绝对的统治。

由此可见，在「鸡生蛋的问题」里，「ETH 网络效应」为 Ethereum 带去了很多 NFT 用户，而 Solana NFT 暂时落后只是因为比 Ethereum 少了一些先发优势。而在流动性方面，Solana 原生 NFT 交易市场 Magic Eden 只专注于 Solana NFT 的发展，更像是一个潜在的「OpenSea 杀手」，更远一步，它似乎也在挑战整个 Ethereum NFT。

「下沉市场」可能才是主流市场

拼多多让「下沉市场」这一概念闯入大众视野。回顾 2018-2019，拼多多先是从阿里和京东的称霸的电商行业中撕出了一条裂缝，又用一次次的高速增长将这个裂缝变成一片领地。时至今日，拼多多已经成为了电商市场中不可忽视的重量级玩家。

拼多多究竟做对了什么？显然，原因绝对不止有一个，但下沉市场却在这中间起到了绝对重要的作用。

而 Solana NFT 发售价格低、交易手续费低，看起来也像是在开发属于 Web 3.0 的「下沉市场」。

越来越多的 NFT KOL 投资于 Solana NFT（图源：Inspect）

很多 Ethereum NFT 的拥趸者们认为那些跑去 Solana 上投资 NFT 玩家是因为在 Ethereum 上付不起高额的手续费，甚至讥讽 Solana 是「穷人区块链」。节约了更多的手续费去用于投资自己喜欢的 NFT 反而成了缺点，这着实让人费解。而事实上，很多知名的 NFT KOL 和巨鲸也在投资 Solana NFT 了。

诚然，NFT 有一定的社交属性和奢侈品属性，人们在社交媒体上只要通过六边形头像就能猜测屏幕对面的人身价几何。NFT 正在尝试创建一个新的阶级天梯，但对那些站在 Web 3.0 世界门口向里探望的人，动辄几百甚至几千美金的入场门槛并不利于 NFT 出圈，如果押注于未来 NFT 将是每个 Web3.0 成员的标配，那低价的 NFT 也会是市场里的必需品。NFT 不应该只是富人或资本的游戏，它们背后的艺术价值也需要被大众所触及。

Solana NFT 所代表的大众的、平民的「下沉市场」可能正是未来 Web3.0 的主流市场。

总结

公链的竞争是「千军万马过独木桥」，NFT 的发展本质上取决于它所基于的公链的发展。对比于 Ethereum，Solana 目前并不像许多人所希望的那样去中心化，但随着时间的推移相信这个问题会有所改善。对 Solana NFT 来说，平价的理念已深入人心，但仍需更多像 Okay Bears 一样的「爆款单品」来证明自己同样有着惊人的造富能力。

共识的凝聚尚且需要时间，让子弹飞一会。
