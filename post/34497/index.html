比特币开发资金

据估计，每年约有2000万美元用于资助免费和开源比特币软件的开发。一般来说，自由和开放源码软件（free and open source software）募资有一系列复杂的问题，这是一个“公地悲剧”问题。

有人问发行债券是否有意义，但债券是期望从未来收入中获得投资回报的投资工具。因此，大家一致认为，资助比特币开发应该是一种捐赠/慈善模式。

另一种选择可能是自己赚钱并利用利润可持续地资助赠款的捐赠基金？或者利用加拿大的SR&ED税收激励计划的企业——本质上是使用政府拨款来资助发展。

总会有一些关于谁资助开发者的争论，因为担心会施加压力来引导开发朝着有利于资助者的方向发展，而牺牲生态系统中的其他人。

为了减少中心化威胁，该领域的许多盈利公司最好聘请FOSS开发人员，并签订关于他们选择工作的“无附加条件”合约。尽管从公司的角度来看，当收益是间接且无法量化时，总是很难证明雇用FOSS开发人员的成本是合理的。

一个值得效仿的项目可能是FreeBSD，它具有“最慢且不具破坏性”的类似精神。它有一个专门为3个维护者提供资金的基金会。

  • 发布部署BIP和其他功能的赏金是否有意义？

  • 不幸的是，赏金往往会吸引低质量的编码人员

  • 真正的瓶颈是高质的代码审查，而不是代码编写

Bitcoin ACKs是一个项目，可以让你在Bitcoin Core拉取请求上承诺赏金，尽管我不确定赏金是如何支付的；我认为这更像是一种信号机制。

在撰写本文时，似乎大多数FOSS比特币开发者都是由慈善组织或交易所（这往往是业内最赚钱的业务）资助的。也许我们可以尝试让一些大型比特币矿工效仿？

此外，人们普遍认为瓶颈不是金钱，而是可用的开发人员人才。从人才的角度来看，自从软件继续吞噬世界以来，比特币现在正在与几乎所有行业竞争。

我们看到的一件令人担忧的事情是越来越多的人推动协议僵化（ossification）。关于坚持我们所知道的协议与冒险添加可以增加价值但也可能引入漏洞的新功能的优点存在巨大的争论。这里的一个折衷方案是更多地关注资助第二层技术的发展，因为它往往不需要改变共识。

开发文化也存在一些问题。新手可能会害怕向开发列表提出建议，因为它们的对抗性质往往会导致想法被分崩离析。作为一个单独的开发者也可能是主要的；也许应该鼓励更多的团费开发。另一方面，比特币人往往会对组成团对的人做出下意识的反应……

开发人员倦怠还有一个长寿问题。我们不希望有才华的开发人员花费数年时间来支持他们的工作，如果他们的提议不被接受就放弃贡献。

如果你有兴趣帮助资助比特币开发，请查看我在此处列出的资源。https://www.lopp.net/bitcoin-information/charity.html#developers

建立加密优先的家族办公室

开设家族办公室的主要目的是将生活中某部分的管理外包，以扩大你的生活方式。

单一家族办公室的运营成本很高；在每年30万美元到100万美元的范围内。另一方面，多个家族办公室可以共享资源。家族办公室关注什么？

  • 财富管理

  • 法律咨询

  • 遗产规划

  • 安全

  • 礼宾服务

管理资本压力很大，可能导致分析变形；高净值人士可以通过将财务外包给专业人士来高枕无忧……如果他们值得信赖的话。

  • 设立加密家族办公室的提示：

  • 对你的办公室要实现的目标有远见

  • 雇用与你有共同愿景和价值观的员工

  • 银行设置可能很痛苦

  • 与Silvergate和Signature银行相比，人们对First Republic Bank的体验更好

  • 一些私人银行服务的最低资产管理规模高达1亿美元，当然这意味着是传统资产

  • 不幸的是，还没有适用于所有这些的千篇一律的模板

请注意，加密客户比传统富裕客户受到更多审查。然而，银行家为加密客户提供服务的胃口正在增加——他们会去任何有钱的地方。

有消息称，Digital Currency Group目前正在为加密货币人士建立一家名为HQ的财富管理公司，但其开业的时间表尚不清楚。

迪拜——比特币城堡？

就加密资产的监管状况而言，迪拜是一只奇怪的野兽。一方面，没有资本利得税，也没有要求申报你拥有的资产。另一方面，接受任何加密货币进行支付都是违法的。

消息是赌博可能很快就会合法化；我们听说有整层空荡荡赌场的酒店在等待开业。

如果你的加密业务拥有适当的金融许可证，那么你应该是安全的。然而，这些规定并不总是统一执行的。

迪拜与瑞士的Crypto Valley建立了合作关系。作为加密货币企业，你无法在迪拜开设银行账户，但你可以开设在迪拜使用的瑞士账户。

迪拜的互联网监控水平很严格。它可能会导致VPN和安全消息传递应用程序出现问题。阿联酋没有任何言论自由意识——批评政府是一个非常糟糕的主意。你应该假设你的所有数据都受到监控，但从实际的角度来看，这种监控主要用于针对当地持不同政见者而不是大声喧哗的外国人。

如果你在全球范围内购买主要城堡地点，我怀疑你可以找到具有类似税收但也有更多自由的地方。

萨尔瓦多

当地人报告说，低级官僚热衷于提高比特币的采用率，但政府高层通常会一时兴起阻止请求。

Bukele总统有一个来自委内瑞拉的顾问团队，目前还不清楚他们是谁/他们做什么/他们为什么在那里。

如果你能够设法通过保护总统的所有层级的官僚机构和情报力量，那么他仍然很可能会在没有任何通知的情况下改变主意。

萨尔瓦多较贫穷的公民往往对比特币更加兴奋，而较富裕的人则更加怀疑，因为他们记得被以前的腐败政府所摧毁的财富。

大多数萨尔瓦多人不知道托管钱包和非托管钱包之间的区别。事实上，很少有萨尔瓦多人对比特币了解太多，因为它是如何在没有任何教育计划的情况下推出的。现在，其他私人组织正在自行制定计划。

Chivo的每日活跃用户徘徊在500,000左右——约占人口的10%。这些用户中有许多是套利交易者。

有传言说，萨尔瓦多的代表团正在访问其他拉丁美洲国家，就如何复制他们的努力提供指导。采用比特币的主要竞争者是厄瓜多尔和巴拿马，因为它们以类似于萨尔瓦多的美元运作，因此受美联储货币政策的摆布。以及，以出售护照而闻名的国家也同样适合采用比特币。

政府参与钱包/交易所

众所周知，萨尔瓦多一头扎进了他们的比特币部署。他们最初是在Athena的（比特币ATM）钱包上推出的，但在遇到大量问题后，他们转而使用Alphapoint，后者从头开始重写了整个 Chivo系统。

显然，萨尔瓦多政府正试图将他们的技术项目当作快速发展的初创公司来运行，但他们并不是真正的技术人员，也不了解软件开发生命周期。这就是为什么他们不断设定不切实际的目标。在相关的说明中，闪电网络不是Chivo的默认选项的原因也是因为一些官僚的决定，而不是出于任何技术考虑。

虽然Chivo用户对延迟付款有很多抱怨，但这些往往不是由于技术问题，而是由于AML审查过程中的长期积压。

如前所述，政府几乎没有为实际的比特币教育投入任何资源。相反，他们选择根据需要提供技术支持，并希望用户通过激励来进行自我教育。个人受到30美元空投的激励，而商家则受到能够以0手续费接受付款的激励。

另一方面，百慕大采取了稍微不同的方法。他们无意使比特币成为法定货币，而是通过教育公民了解其利益来争取自愿采用加密货币和稳定币。百慕大是一个低税收管辖区，百慕大没有针对收入、资本收益或其他数字资产税的具体税种。政府本身就是早期采用者，自2019年以来已接受USDC的税收和服务。

在未来几年比较和对比这些国家的采用情况将会很有趣。

免费私人城市

什么是免费的私人城市？这是一个融入一个国家的城市，其理解是该城市将遵守国家宪法，但仅此而已。这样的城市将：

  • 要求市民与城市签订合同

  • 有自己的税收计划

  • 运行自己的公共服务

  • 控制自己的边界

目标是改变公民与国家之间的关系，使其成为服务提供者而不是主人之一。这样的城市甚至可以让市民成为股东。

城市将提供哪些服务？可能只是基本的安全性——预计大多数其他服务将被私有化。

该概念以新加坡和摩纳哥为蓝本。自由私人城市的一些例子是洪都拉斯的Prospera和希腊的Mount Athos。经济薄弱的国家可能更愿意托管免费的私人城市。

你可以在freeprivatecities.com了解有关此概念的更多信息。

跨链自托管钱包

与往常一样，安全性和便利性之间存在权衡。我们经常在多资产钱包中看到的是，它们通过在所有资产中使用相同的密钥来创建单点故障。

你还可能遇到不同网络支持相同格式地址的用户体验的噩梦，例如LTC和BTC P2SH地址。由于不同的网络使用不同的派生路径来生成密钥，因此用户可能很难弄清楚出了什么问题，然后花费大量精力来收回资金。

另一个棘手的问题是签名在不同的网络上意味着不同的东西。与智能合约交互时，了解交易的上下文很重要。同样，当使用硬件密钥管理器时，用户可能会产生错误的安全感，而该管理器在不了解底层智能合约的情况下盲目签署数据。如今，不乏利用这一弱点发生的骗局。

用户也很难确切地知道他们授权任何给定的智能合约会执行什么。显然，DeBank仪表板（https://debank.com/）是查看你已批准合约的有用工具。

去中心化自治组织

随着越来越多的群体协作实验通过DAO机制进行，DAO继续获得采用。

DAO最被低估的方面之一可能是DAO到DAO的协作。与典型的B2B合作伙伴关系相比，这种情况的摩擦要小得多，因为它往往会放弃所有的法律合同谈判。

此外，DAO可以通过参与代币交换来相互激励。

DAO不需要通过纯粹的民主“代币投票”系统运作——他们可以选举委员会来委派某些权力，以便比让所有决策都经过全球治理过程更加灵活。事实上，我们看到在DAO中实施了广泛的治理机制，从完全民主到相当专制。

虽然这种情况很少见，但我们已经看到一些企业与DAO建立合作伙伴关系，并通过将风险纳入决策而放弃法律合约。

如果你想要一个好的DAO仪表板，请查看deepdao.io。

由于DAO不是法人实体，因此它们不纳税。他们也没有人力资源部门。另一方面，在为DAO工作时，你不会获得任何传统的好处。你也无法获得工作签证以居住在不同的国家，因为没有国家会承认DAO为雇主。

人们普遍认为，从长远来看，DAO面临的最大挑战是懈怠。一般人不想花时间做研究来决定 DAO的未来。此外，如果在DAO中担任权力职位的人变得懈怠，这可能会带来巨大的挑战，以促进平稳过渡并让他们离开。因此，围绕DAO的可持续性和长寿性存在许多悬而未决的问题。

GameFi——游戏代币化

当游戏遇上金融，当心游戏的庞然大物！

许多当前流行的游戏都瞄准了新兴市场——它们是“为赚钱而玩”或简称为P2E。

一个独特观点是，儿童不能在加密货币交易所开设账户，但他们可以参与游戏经济——因此这可能成为青少年进入加密货币的主要入口。

我们看到一些传统游戏玩家的强烈反对，他们认为激励措施正在破坏游戏文化，或者游戏公司将利润置于可玩性之上。GameFi似乎确实遇到了“本末倒置”的问题，并且经常更关注代币经济学而不是游戏的实际质量。你不能只是将代币贴在游戏上——它需要成为体验/挑战/奖励的一个组成部分。

代币化在颠覆电子竞技行业方面处于有利地位，因为有如此多的中间商从玩家那里获得收益。

不幸的是，许多游戏代币即使不是不可能也很难交易。从监管的角度来看，游戏公司不能允许“cash out游戏”，因为它被认为是一种赌博形式。游戏也很难以一种不被视为赌博的方式实施代币经济学。你必须100%确定玩家不会因为偶然而获得任何奖励。想一想，游戏到处都使用随机性来确保你的体验不会让人感到无聊。因此，为竞争性人类与人类游戏或“为赚钱而玩”平台实施奖励可能更安全。

还有价值流的经济问题——这些游戏是不是一群有钱的西方孩子，他们最终付钱给第三世界国家的人们来玩P2E游戏？这不会不可避免地导致我们在魔兽世界时代看到的一些同样可怕的情况，那里有满是“玩家”的血汗工厂，他们种植黄金并升级角色？

为什么不像在Zebedee看到的那样将比特币集成到游戏中呢？这是一个控制问题——游戏公司更愿意在他们正在构建的生态系统的代币经济学上拥有最大的灵活性。简而言之，GameFi不仅仅是将游戏货币化。

元世界与Web3

Web 2.0：使用用户名和密码登陆

Web 3.0：使用钱包登录

新冠疫情引发了虚拟现实热潮，因为许多发现自己因封锁而受到身体限制的人使用虚拟现实逃到新世界。我本人，自2018年以来一直是VR用户，当我们在2020年的周末开始每周举行比特币VR聚会时，我很高兴。我甚至最终在几个虚拟现实会议上发言！

一种方式是将Metaverse视为提供全新交互级别的下一代网站。对于传统网站，你只需要与运营该网站的公司进行交互。通过虚拟世界，你还可以与“网站”的其他用户进行交互。尽管与当前采用元宇宙的状态有些脱节，因为这些世界往往是围墙花园。因此，互操作性和标准将成为主流采用的关键，因为品牌不希望必须为十几个不同的平台构建相同的用户体验。

Web 2.0：用户被网站通过广告/数据挖掘/监控而货币化。

Web 3.0：用户获取价值并因其惠顾而获得奖励。

在Metaverse端和web3端都存在大量的可扩展性挑战。构建可以同时处理数千个用户的虚拟世界非常困难，因为处理所有交互的复杂性呈指数级增长。类似地，web3扩展可能需要二层协议，因为将每个用户交互永久存储在全局分布式账本上是没有意义的。

我们将在哪里看到这项技术的早期采用？可能在MMORPG中——他们已经在构建虚拟世界，只需要弄清楚如何构建和部署具有流畅用户体验和逻辑代币经济学的加密资产系统。

结论

随着围绕加密货币的生态系统不断发展，本次非会议与会者的多样性也随之而来。试图跟上这个领域的所有发展是压倒性的，所以我非常感谢能够在几天内接受如此多的速成课程。
