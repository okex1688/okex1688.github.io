你可能听说过马克·扎克伯格希望 Facebook 成为元宇宙公司，今年早些时候开发虚拟引擎的公司 Epic Games 宣布完成 10 亿轮融资，以支持元宇宙的长期发展。元宇宙绝对是科技界最热门的流行语。在本文中，我们将简要讨论什么是元宇宙，谁来建造它，最重要的是为什么它对建筑师很重要，以及设计师如何在即将到来的数字经济中发挥重要作用？

什么是元宇宙？

“元宇宙”一词源自 1992 年的科幻小说《雪崩》，由尼尔·斯蒂芬森 (Neal Stephenson) 作为互联网的继承者创造，构成了斯蒂芬森对数字世界在不久的将来可能会如何发展的看法。

你可能会从《头号玩家》和《黑客帝国》等电影中了解元宇宙中的生活，我们可以想象的任何东西都可以存在，我们可以一直连接到元宇宙以延长我们现实生活别样的体验。

谁将构建元宇宙？

与《头号玩家》里中心化的元宇宙大陆不同，科技界认为元宇宙应该是一个开放且可互操作的生态系统，而不是由任何一家公司主导。就像马克扎克伯格说的那样，“在未来，询问一家公司是否正在构建一个元宇宙听起来就像询问一家公司他们的互联网进展如何一样荒谬。 ”

同时，目前最“元宇宙概念股”的Roblox和Epic Games也认为，元宇宙一定要有去中心化的基因，避免被少数势力垄断。Roblox 联合创始人 Neil Rimer 提出：“围绕元宇宙的能量将来自用户，而不是公司 。” Epic CEO Tim Sweeney 也强调：“元宇宙并非来自任何行业巨头，而是数百万人共同创造的结晶。每个人都通过内容创建、编程和游戏设计为宇宙打造自己的风格。贡献还可以通过其他方式为元宇宙增加价值。 ”

多年来，互联网主要是一个带有超链接的 2D 目录，现在它终于开始变得有趣，具有 3D 体验的潜力，并将改变我们的工作、购物、生活方式。

为什么它对建筑师很重要？

全球正经历着从实体经济向数字经济的重大转变，而全球大流行以来，这两者的融合速度急剧加快。

我们已经看到各种实体零售店关门大吉，与此同时，Balenciaga等时尚品牌选择在电子游戏中推出其最新系列。商业房地产业主正在苦苦挣扎，因为租户要么放弃办公室，要么缩小规模，而像 Zoom 这样的视频会议公司在大流行中取得了巨大成功并蓬勃发展。COVID-19 已经改变了我们的工作文化，加速了电子商务的兴起，并改变了企业的运营方式。一些率先构建元宇宙的行业先驱包括游戏、好莱坞和加密货币，除了加入新游戏之外别无选择。

艺术中的 NFT：数字资产的兴起

数字物品的市场已经超过 100 亿美元，为了使数字物品具有真实、持久的价值，它们必须独立于可能随时决定移除或禁用该物品的实体而存在。这就是为什么区块链技术的最新应用NFT(Non-fungible tokens) 出现的原因。“不可替代”意味着它是独一无二的，不能被其他东西取代。因此，NFT 的价值被理解为证明数字项目的创建是“独一无二的”。数字创作可以是任何东西，包括数字建筑。多伦多艺术家 Krista Kim 以超过 50 万美元的价格出售了第一个由 NFT 技术制作的数字家庭。在许多北美城市，这比真正的房子要贵很多。

加密世界的另一个趋势是投资虚拟房地产。作为2021 年初NFT市场狂潮的一部分，虚拟世界中基于区块链的房地产价格飙升。基于区块链的在线世界Decentraland中的一块虚拟土地最近以超过 90 万美元的价格售出。在Decentraland、the Sandbox、Somnimum Space等虚拟世界中，人们可以展示他们的NFT艺术收藏品、与朋友四处走走、参观建筑物和参加活动。这对很多人来说可能是个骗局，这个平台可能还为时过早，但对于设计师来说，利用他们在物理世界中的设计技能并将其扩展到虚拟世界，这绝对是一个有趣且可能有利可图的领域 。

商业模式变革：从顾问到内容创作者

建筑师总是抱怨很难找到完全支持他们创意并按时支付账单的好客户。大多数设计将永远留在纸上，它们可能会因各种原因被拒绝，而大多数“纸架构”由于不被客户认可而毫无价值。当项目出错时，你很可能被起诉。

元宇宙需要海量的内容来娱乐用户。我们需要诸如虚拟游乐园、虚拟电影院、虚拟音乐会、虚拟赌场、虚拟学校、虚拟会议之类的体验，任何你能说出的东西。对于建筑师来说，元宇宙是一个充满可能性的新天地，是一个没有物理世界约束的乌托邦。建筑师可以创建由NFT支持的独特设计，对于喜欢收集独一无二的资产的人是很好的体验。建筑师还可以构建城市、建筑物、家具、雕塑、点云、纹理等数字资产，并将它们多次出售给虚拟世界、游戏和电影。

不管你做什么，试着像内容创造者一样思考，利用你的努力实现指数增长，在数字丛林中找到你在不同平台上推销自己的方法，用高质量的内容接触潜在用户，让他们参与进来并发展你的用户群。

如何将设计前沿延伸至数字世界？

您对新机会感到兴奋，但设计师如何加入游戏？

1.向游戏行业学习

游戏被认为是元宇宙最好的载体，尤其是现在越来越流行的沙盒（the sandbox）游戏，已经有了元宇宙的原始形态。如GTA系列、Minecraft、Fortnite等，都提供了高度自由的游戏体验，不少游戏还提供了自制Mod扩展内容制作的专业工具，以丰富游戏内容并延长游戏寿命。

像SimCity这样的游戏是经典的城市建设游戏，许多城市设计师都觉得它很有启发性。游戏是构建一个想象世界的过程，与现实生活中的设计过程非常相似。环境艺术家将艺术家创作的 2D 艺术转化为3D。使用工具是建筑师或多或少熟悉的 3D 建模软件：Blender、3Ds Max、Maya、ZBrush、Substance Painter等，以及 Unity 和 Unreal 等游戏引擎。

2.学习一些编程

像（软件）开发人员一样思考：

我们已经看到了衍生式设计、参数化设计、算法设计等术语。我不想争论这些术语之间的差异，总的来说，这是一种解决问题的新方法。设计师不是仅仅依靠设计师的直觉和经验，而是将设计过程抽象化，将大挑战分解为小挑战，自己定义那些影响设计输出的参数或条件，迭代设计工作流程和选项，最后分析原理。

像Grasshopper这样的工具已经在建筑领域得到了很好的采用，在您的设计工作流程中应用计算工具来放大计算能力是一种很好的方法。

如果您想学习如何编码，Processing可能是一种在视觉艺术背景下学习编程的好语言。Python也是一种相对容易掌握的语言，在数据分析/可视化、人工智能/机器学习、Web 开发和其他一般用途中具有广泛的应用。

程序内容生成（PCG）：

在计算中，程序内容生成 (PCG) 是一种通过算法而非手动创建数据的方法，通常是通过将人工生成的资产和算法与计算机生成的随机性和处理能力相结合。在计算机图形学中，它通常用于创建纹理和 3D 模型。程序生成通常用于视频游戏和电影中，以允许开发人员制作更大量的内容，以及具有无限内容的游戏的可能性。

在程序化方面，Houdini是一个很好的工具。它专为从事电影、电视、视频游戏和虚拟现实的 3D 动画和 VFX 工作的艺术家而设计。Houdini 将这些世界整合到一个强大的平台中。Houdini 与其他 3D 动画软件的真正区别在于它的程序性。Houdini 有一个开放的环境，并通过各种 API 提供脚本，但 Python 是大多数软件包的首选语言。

CityEngine是由 Esri R&D Center Zurich 开发的商用三维建模软件应用程序，专门用于生成 3D 城市环境。使用程序建模方法，它支持创建详细的大型 3D 城市模型。

探讨：建筑师能为元宇宙带来什么价值？

数字经济中的建筑师危机

建筑师在现实世界中已经面临危机。我们曾经是建筑大师，了解建筑的所有知识和技术。然而，在现代工业社会，结构、机械、电力、预算、施工管理等许多专业已经脱离了建筑学科，剩下的大部分是造型、可视化、协作等，我们缺乏核心技术。

“建筑是一个需要大量时间的职业。最少的架构工作至少需要四五年或六年的时间，对于正在发生的革命来说，这个速度真的太慢了。” 获得普利兹克奖的荷兰建筑师雷姆·库哈斯 (Rem Koolhas) 指出了该行业的不足之处，即它如何适应当今不断变化的社会和技术气候。技术在进步，技术“局外人”的目标是重构或彻底改变 AEC 行业。从工业界到学术界，我们需要改变以适应。

元宇宙是一个新的游乐场，建筑师可以与其他利益相关者一起参与其中。如果我们与游戏设计师或开发者竞争，我们能为游戏带来什么额外价值？

建筑师是连接物理世界和虚拟世界的桥梁

几个世纪以来，建筑师一直引领着物理世界的建设。作为项目经理，与所有学科合作将项目交付给最终用户，我们对项目有全面的了解，而其他顾问更有可能只熟悉整个画面的一部分。

随着AEC行业实践中创建数字孪生的趋势，建筑师可以充当物理世界和虚拟世界之间的桥梁。根据 Autodesk 的说法，“数字孪生是实体资产或资产集的动态、最新复制品，在建筑、工程和建筑领域，无论是建筑物、校园、城市还是铁路，都将设计、施工和真实结合在一起。” 建筑师可以利用虚拟来创建真实世界的场景。数字孪生使用来自连接传感器的数据来讲述建筑物在整个生命周期中的故事。从测试到在现实世界中使用，借助物联网数据，我们可以衡量建筑物健康和性能的具体指标，例如温度和湿度。数字孪生还可以增强物理体验。

元宇宙中将有许多有趣的应用程序，它们托管数字孪生或扫描数据，只需创造性地利用资产并策划两个领域的体验和互动即可。

虚拟世界中的社会责任

建筑师是很好的倾听者和沟通者。我们从大量倾听开始，以了解客户想要什么以及社区想要什么。除了为客户服务外，建筑师倾向于首先将社会影响和责任放在为整个社区服务。

建筑是文化的表现和表达。在概念设计之前，我们试图了解一个特定的文化环境，包括语言、地方记忆和生活方式。虽然元宇宙不是特定于地点的，但用户确实来自不同的文化背景，我们可以设计充满多样性和文化包容性的“本土”元宇宙。

建筑的价值

这是一个很深的话题，我只想再次引用 Rem Koolhas 的话：“建筑用一只脚站在 3000 年前的世界，另一只脚站在 21 世纪。这种几乎像芭蕾一样的伸展使我们的职业出奇地有趣。你可以说我们是最后一个有记忆的职业，或者是最后一个可以追溯到 3000 年前的职业，今天仍然展示了那些漫长道路的相关性。”

历史就像一面镜子，每次看它都会有新的反思。“建筑的价值”因历史而异。从罗马时代的“稳定、实用、美”，到19世纪拉斯金的“建筑七灯（牺牲、真理、力量、美、生命、服从、记忆）”。从勒柯布西耶 (Le Corbusier) 的“迈向建筑”(Toward an Architecture) 将 20 世纪现代建筑摒弃装饰并拥抱纯粹形式作为工业时代的新精神，再到克里斯托弗·亚历山大 (Christopher Alexander) 对软件工程产生重大影响的“模式语言”……我们是否需要在这个数字时代定义建筑的新价值？这对我亲爱的读者来说是一个悬而未决的问题。我们越能定义和实现架构的价值，就越能更好地构建一个伟大的非物质元宇宙。

元宇宙正在不断发展。亲爱的建筑师们，你准备好了吗？
