本文主要以参与 Kusama 平行链竞拍的例子，说一下具体的操作过程，波卡主网竞拍的操作也会类似，只是把 KSM 换成 DOT

和用户有关的概念

Kusama：波卡的先行网，基本复刻波卡主网所有功能，但周期比波卡主网短（比如出块速度更快、抵押解锁时间更短等），它的通证名称是 KSM

平行链插槽：所有波卡项目接入波卡生态的接口，数量有限，接下去一年的总量估计在40个插槽左右

去中心化钱包 或 托管钱包

去中心化钱包指的是用户自行保存私钥，支持波卡的去中心化钱包主要有 imtoken、麦子钱包、polkawallet等。本文的操作主要在去中心钱包中进行。

托管钱包指的有交易所、托管服务商代为保存资产，主流的交易所、云端钱包都属于托管钱包，他们也会推出代理竞拍服务，帮助不熟悉去中心化钱包的用户参与平行链竞拍。

拍卖 Auction

主要由平行链项目方操作，用户不需要直接操作，但可以关注竞拍结果，因为结果会影响到自己参与的 Crowdloan 的奖励情况。

众贷 Crowdloan

指的是用户将 KSM 借给项目方参与竞拍，并获得项目方的通证作为奖励。

用户需要选择一个项目参与，并输入锁定 KSM 通证的数量，这些通证仍在用户钱包中，仅在链上锁定，没有安全风险，但没有抵押挖矿的年化收益。

用户会收到项目方通证作为奖励，具体的比例和锁定时间等由项目方自行确定。

参与风险

主要考虑因素包括

参与锁仓的时间和收益，参与者需要考虑获得的奖励能否覆盖 KSM 本身抵押挖矿的收益。

项目有可能没有成功拍到插槽，赎回的 KSM 可能会对市场形成抛压。

如何选择项目

项目基本面：可以访问 Polkaproject.com 中项目相关信息进行研究。

一般项目基本面研究维度包括：GitHub活跃度, 是否有成熟可用产品, 技术文档和教程是否完善, 技术社区是否活跃, 项目社区是否活跃。

奖励信息：可以访问 Polkaproject.com 的页面中查看整理好的奖励信息，并自行评估收益。

租期 Period：用户需要关注的是租期的最大值，Kusama上租期的单位是6周，租期数值越大，你的KSM锁定时间也越长

硬顶 Cap：可以投入的KSM最大数量，如果希望参与硬顶较低项目的众贷，则需要尽早参与

Karura

Karura 是 Acala 的先行网。

Acala 是波卡生态中重要的 DeFi 基础设施，为波卡生态里 DeFi 项目提供以太坊兼容支持。Acala由支持跨链多资产抵押的稳定币 Honzon 协议和释放质押资产流动性 Homa 协议组成，旨在为全球加密资产带来稳定性，提高流动性。

目前 Acala 得到了 Web3 基金会官方资金支持，并获得 Polychain、Hashkey、KR1 等在内的数家全球知名机构的投资。

Shiden

Shiden 是 Plasm 的先行网。

Plasm 是基于 Substrate 技术的可扩展 DApp 平台，主要是为波卡提高可拓展性。

Plasm团队是最早接入 Rococo 测试网的平行链项目之一。

Plasm 投资机构包括 Binance Labs、HashKey Capital、LongHash Ventures、Digital Finance Group 和 PAKA Ventures 跟投。

Moonriver

Moonriver 是 Moonbeam 的先行网。

Moonbeam Network 构建跨链连接的 DApp，提供与以太坊兼容的智能合约平台，可轻松构建本地可互操作的应用。其由波卡 Polkadot 网络提供支持，由 PureStake 进行开发。借助 Moonbeam 智能合约及与以太坊开发工具集的兼容性，可基于桥的集成，快速构建与远程链上的用户与资产一起使用的去中心化应用。

Moonbeam 投资机构包括 CoinFund、IOSG Ventures、Binance Labs、ParaFi、Coinbase Ventures、分布式资本（Fenbushi Capital）、D1 Ventures、Hypersphere Ventures、HashKey、Arrington XRP、A195、KR1、Mechanism Capital、Divergence Ventures、Signum Capital 等

SherpaX

SherpaX 是 ChainX 的先行网。

ChainX 是波卡生态中的资产跨链项目，将主流的 BTC 等外部资产跨链进波卡生态中，并孵化后续的衍生金融服务。根据官网的介绍，ChainX 的目标是成为波卡生态的资产跨链网关。

根据 ChainX 首席运营官 Kristen 透露，ChainX 计划拿出议会基金和全网算力（可挖出 ChainX 原生代币）的 15-20 % 来激励支持者。

Sakura

Sakura 是 Clover 的先行网。

Clover是一条波卡平行链，提供简单易用的区块链基础设施，并为基于Substrate的应用程序创建一站式兼容EVM框架。

Clover 投资方包括 Polychain、Hypersphere、Bithumb Global 和 Divergence Ventures。

Darwinia

Darwinia Crab 是 Darwinia 的先行网。

Darwinia 达尔文网络是基于 Substrate 开发的跨链桥接网络，专注于建设未来资产互联网络，包括非标资产拍卖市场、稳定币跨链、资产交易兑换等领域。

Darwinia 的投资人包括 SNZ，Hashkey Hub, Waterdrip Capital，SevenX，Bitrise Capital，Consensus Lab等。

Khala

Khala 是 Phala 的先行网。

Phala Network 是波卡上的隐私基础设施。根据官网介绍，Phala 通过TEE-POW 的经济机制来提供 CPU 隐私算力给波卡上的平行链、DEFI、DAPP 和其他隐私数据服务，其核心产品是机密智能合约、波卡生态的暗钱包、Libra桥和 Web3 Analytics。目前 Phala 测试网已经开启TEE挖矿竞赛，目前全球有超过 500 个 TEE 节点在提供隐私计算服务。

Phala 的投资机构包括 Candaq Group、IOSG Ventures、SNZ、水滴资本、币世界、瑞新资本、无极链、株式会社 Incuba Alpha、Exoplanet Capital 和 Blue Mountain Labs。

Crust

Crust Shadow 是 Crust 的先行网。

Crust 提供了 Web3 生态系统的去中心化存储网络，支持包括 IPFS 在内的多种存储层协议，并对应用层提供接口。Crust 的技术栈还能够支持去中心化计算层。Crust 旨在构建一个重视数据隐私和所有权的分布式云生态系统。

根据 Crust 白皮书显示，Crust 会拿出生态发展的约 40% 用于波卡插槽竞拍以及相关的的支出，并且在开拍前制定一个完整的平行链竞拍方案。

Equilibrium

Genshiro 是 Equilibrium 的先行网。

Equilibrium是一个一站式一体化的，可互操作的去中心化金融（DeFi）聚合器，Equilibrium的用户可以仅仅只依靠该平台抵押加密货币并赚取收益，或执行借出和筹集资金提供流动性

Equilibrium 的投资机构包括 KR1、Signum Capital、Hypersphere、Block Dream Fund、Genesis Block Ventures、CMS Holdings、AU21 Capital、Digital Finance Group、PNYX Ventures、FBG Capital、LD Capital、Crasolum Ventures、Inception Capital。

MathChain

MathChain 是基于波卡 Substrate 开发的 Layer 2 平行链。
依托于 MathWallet 百万用户基础，MathChain 将专注于为所有人服务的区块链智能钱包、普惠应用，通过平行链机制成为开源、高效、分布式的 Layer 2 区块链网络。MathChain 是 Substrate Builders Program、Web3 Bootcamp 成员。

MATH 的投资人包括分布式资本、Alameda Research、Binance Labs、FundamentalLabs、Multicoin Capital、NGC Ventures 等著名区块链投资机构。

Bifrost

Bifrost 是一个为 Staking 提供流动性的 Polkadot 平行链，用户可以随时将 PoS 币种通过 Bifrost 转接桥兑换成 Bifrost vToken 从而获得 Staking 收益和流动性。

Bifrost 的投资人包括 NGC、SNZ、DFG、PAKA、CMS Holding、Altonomy、LongHash Ventures等，同时也是 Substrate Builders Program、Web3 Bootcamp 成员
