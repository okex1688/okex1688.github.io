加密从业者的思考，正从单纯的「区块链+传统」一步步地向「区块链原生化」演变。

过去的 12 年，区块链行业已经从「加密极客」发展到大众耳熟能详的地步，截至目前市值规模已经达到千亿美元，发展势头十分迅猛。期间，在区块链的巨大魅力感召下，很多传统行业（诸如公共政策、互联网科技、金融等）的优秀从业者加入了加密世界，为区块链行业的发展做出了巨大贡献。

两类加密从业者

根据从业者是否追求加密行业本源，可以将加密从业者分为两类：

一类是「区块链原生」从业者，他们有一定程度的区块链信仰，追求「加密行业本源」 ，例如诞生于加密社区的艺术作品、在社区中发展壮大的 MEME 文化等。试图不断改善区块链技术，推动行业来改变世界。

另一类是更关注区块链和现实世界的关联，而不是加密领域原生世界，试图将传统与区块链结合起来进行投机 / 创业的从业者，我称呼他们为「区块链的传统行业参与者」。 这些参与者一般会利用自身的传统行业技能，在加密行业找到结合点，进行一些项目投机 / 创业。譬如，大规模并发的交易系统专家，在作为股票交易系统的同时，可以作为加密货币的交易系统。又如，将传统金融市场的各类期权期货交易与区块链相结合，建立比特币等加密货币的期权期货产品。再如，以传统金融市场的量化策略，撬动加密货币的量化交易。

过去我一直认为，区块链的传统行业参与者没有区块链信仰，他们眼中的加密行业属于一个「中性事件」，加密货币和原油、贵金属、玉米、大豆等大宗商品没有本质区别，只是投机、创业的一个标的物。

但最近的一些创业者让我认识到，越来越多传统行业精英，真正地意识到了区块链的力量，他们正越来越「区块链原生化」。

不断向「区块链原生化」的加密从业者

Liquefy 的三个 90 后创业者就让我颇有感触。

这三个年轻人创建 Liquefy 时完成的第一个项目是房地产代币化的项目，试图利用区块链提升传统资产的交易效率。在实践过程中，他们逐渐认识到通过区块链改善实物资产交易，不应该从传统世界找着力点，那样区块链的应用只会流于表面，重复「区块链+能源 = 能源」、「区块链+物联网 = 物联网」的无奈命运；而应该以区块链本身作为着力点，在数字领域、开放金融领域等基础设施有了足够积累的情况下，利用数字证券、DeFi 来更好地撬动传统资产。

带着数字证券的愿景，创始人开始了第一个 DeFi 项目——合成资产平台 Linear Finance。该平台通过与 BSC 合作，解决了当前合成资产协议由于以太坊性能问题带来的各种问题，加上 Gas 费相对合理、交易确认速度较快，平台用户能更加及时、准确的价格构建合成资产，降低交易滑点。该项目在创业之初，就得到了海内外众多加密投资机构的认可与参与。

在 Linear 的基础上，创始人进一步「区块链原生化」，创立了第二个项目 Convergence，通过 AMM 给资产通证建立无国界的流动性、给现实资产建立开放金融世界的流动性。该项目还支持碎片化的高端 NFT 交易，使得长尾市场的用户可以接触到高端加密艺术品。

Linear 解决了合成资产问题、Convergence 解决了流动性问题，Liquefy 的三个创始人不肯止步于此，继续在「区块链原生化」的道路上奔驰——创立了第三个项目「Lithium」，以此连接 Linear 与 Convergence，通过权威预测和分布式预测的博弈论机制来使价格尽可能的频繁生成，解决私人市场中信息不对称的问题，更好地为非流动性资产定价。

Liquefy 的创业过程，是一个不断开拓区块链行业，层层递进地解决实际问题的过程。从这个过程中，我们可以发现，加密从业者的思考，正从单纯的「区块链+传统」一步步地向「区块链原生化」演变。越来越多传统行业精英，正在接受 / 形成区块链信仰，意识到「区块链改变世界」的主体是区块链，应以区块链为「手术刀」来改造世界，而不是将传统世界来套用区块链。

相信随着开放金融相关基础设施的不断完善，实物资产的通证化、各类 AMM 交易等区块链原生化行为是大势所趋，随之而来的人才流入则如江河入海。他们将切身感受到加密世界的魅力，为之倾倒，为之永远 Degen 而热泪盈眶！
