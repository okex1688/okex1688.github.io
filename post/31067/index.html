上半年游戏平台Roblox在纽交所上市，上市当天股价上涨54%，市值突破400亿美元，它在招股书中提到元宇宙，并总结了八大要素，成功将元宇宙概念推到台前。

下半年Facebook改名Meta，扎克伯格宣布公司将在五年内转型为元宇宙公司，为“元宇宙”概念再添一把火。微软、字节跳动、网易等也纷纷加码。

这两个大事件对市场都有轰动性影响，同时代表了元宇宙的两个主要切入场景，游戏和办公。

一、“游戏派”的头号玩家

电影《头号玩家》《失控玩家》所构建的虚拟游戏世界，已经先入为主地为我们打造了一个元宇宙的图景，但要最终实现还是征途漫漫。

“元宇宙第一股”游戏平台Roblox的CEO在采访中说：

我们17年前的商业计划中，就预测了可以让人们聚集在一起的这个新事物。在过去的17年里，我们一直在这一领域进行创新。

2004年Roblox初建时给自己的定位是“平台”，目前是游戏平台，用户可以自己开发游戏，也可以参与其他人的游戏 ，或许以后Roblox会依靠游戏这一锚定领域，延伸出更多的身份，比如社交网络、媒体公司。

2007年，Roblox开始初步构建商业模式，引入虚拟代币Robux。 Robux可用来购买虚拟形象、作品中的特殊技能，玩家能直接通过法币购买Robux，也可以通过创建游戏等多种方式获得。

为激励创作者，2013年Roblox推出的创作者交易计划。玩家在游戏中消费的Robux，游戏创作者能拿到30%，并且可以直接兑换成现金。

比如Alex Balfanz在17岁时发布了一款名为《越狱》的游戏，现在他靠售卖游戏里的皮肤、道具，每年能从这个游戏里赚取超百万美元。这不是个例，去年有超过 1250 名游戏开发者赚到了价值至少1万美元的Robux。

为了让玩家有更强的现场感，Roblox还支持VR设备Oculus Rift和HTC Vive ，玩家可以进入到游戏世界。

Roblox归结元宇宙有8个要素：身份、朋友、沉浸感、低延迟、多元化、随时随地、经济系统和文明。

目前Roblox已经完成虚拟身份、朋友社交、VR沉浸感、场景多元化、Robux经济系统的初步构建，随着可穿戴设备和网络速度的升级，低延迟、随时随地也将不再成为阻碍时，元宇宙的文明会开始建立起来。

Roblox确实是元宇宙游戏中的头号玩家，但在市场上却不是一家独大。

国内的代码乾坤 诞生于2017年，同样是供玩家自行参与和开发游戏的平台，推出的重量级游戏**《重启世界》** 攻克了物理引擎多人多端网络同步的难题。

代码乾坤去年完成了3亿美元的B轮融资，今年获得字节跳动投资，字节跳动游戏业务负责人严授，目前担任代码乾坤的董事。

纯VR的社交游戏类产品《Rec room》，现已支持HTC Vive，Oculus Rift，PSVR和WMR，今年6月23日总用户数超过了300万人。说唱歌手Travis Scott在《堡垒之夜》游戏中举办虚拟演唱会，吸引全球1200万名玩家同时在线观看。

二、“办公派”的社交网络

Meta（原Facebook）和微软将破局的重点放在了办公场景。

Meta今年八月份正式推出了Horizon Workrooms ，Oculus Quest 2用户可以直接下载，进入VR环境举行或参与会议。目前Horizon Workrooms正处于公开测试阶段。

首先，头像定制功能、高质量、低延迟的空间音频，能够很大程度还原真实的工作场景，提高会议效率和同事间互动频率。

虚拟会议室里，同事们在办公桌前围坐一圈，主讲人在白板上展示本次的会议内容，可以随意在白板上写写画画，大家彼此之间有眼神互动、话语沟通，和线下办公别无二致。

其次，信息存储和调用更加便捷。 在VR会议期间可以一键访问电脑，将文件甚至是整个屏幕共享给同事。虚拟空间里有一张无限大的白板，白板上的任何笔记都可以保留下载下来，供后期回看。

还有办公场景定制、日历提醒、会议记录等功能。

扎克伯格在采访中表示：虽然今天的虚拟人物形象还不够逼真，但从很多方面讲，已经感觉比Zoom视频电话更真实了，因为有一种共同的空间感。

微软官方宣布将推出Mesh for Microsoft Teams，最快2022年上半年，用户就可以在 Mesh for Teams预览版体验到沉浸式开会，以及更身临其境的远程协作。

由6个麦克风和16个摄像头组成的全息镜头，采集用户的体态、面部信息，通过混合现实技术为每个用户提供一个虚拟形象。

微软项目经理凯利解释说：“这样做的目的是，你不会被锁定在与你的头像的2D交互中。”

拥有60万员工的服务公司埃森哲，目前正在使用微软搭建的沉浸式空间。新冠疫情爆发之前，埃森哲建立了一个虚拟校园，来自任何地方的员工都可以聚在一起参加聚会。疫情爆发后，埃森哲依靠这套系统培训新员工，帮助员工理解企业文化并建立同事间的联系。

三、元宇宙第一场景

“游戏派”和“办公派”从不同角度推动元宇宙世界的构建。但谁会率先爆发，带来无限的商机？未来谁会成为主宰元宇宙的第一场景？

扎克伯格在接受The Vergecast采访的时候表示：

“有很多人认为元宇宙主要是关于游戏的。娱乐显然是重要组成部分，但我认为它不仅仅事关游戏。”

从迫切性来说，办公派>游戏派。

受新冠疫情的影响，其他许多国家接下来很长一段时间内，都将处于大面积居家办公的模式。虽然现在已经有各种线上办公软件，但是根据其使用的效率和效果来说，远远不如线下。

“办公派”要做到的一点，就是构虚拟交互空间，有办公场景的真实感，全面提升线上办公效率，达到和线下的对等（甚至更好）。

当然，居家办公期间对线上娱乐的要求当然也会更高，从2020上半年游戏和视频公司的盈利情况可见一斑。但现存的游戏已经能暂时满足娱乐的需求，所以人们对新科技给娱乐带来的改变，只有适度期待，没有迫切渴望。

比起锦上添花的“游戏派”，雪中送炭的“办公派”将是攻克未来元宇宙世界的主力。

而疫情控制较好的我国也并非只有娱乐需求。公司每年要花费高额的差旅费进行客户沟通、团队协作、企业培训等，在经济并不景气的现在社会，降低运营成本同样重要。

从使用场景适用度来说，办公派>游戏派。

办公场景相对固定，不是公司就是家里。而玩游戏的场景除了回家，也有可能是工作间歇、外出途中，碎片化的时间和场景，对VR游戏提出了更高的要求。

首先，需要有便携的游戏入口。目前手游的发展如火如荼，也是因为迎合了这一趋势，只要从兜里掏出手机，就可以随时随地进入游戏。

而目前VR游戏需要的设备复杂很多，以目前的Oculus Quest 2来说，重量503克，体积191.5毫米 x 102毫米 x 142.5毫米，除了头盔外还有两个手柄，电池续航时间2-3个小时。

其次，需要不那么沉浸的体验。这简直和VR游戏的初衷背道而驰。毕竟谁也不想在等车开一把游戏的时候，真真切切看到一个怪兽过来，自己因为受到惊吓到处闪躲，所以目前的VR游戏仍有很大的时间空间局限。

从想象力来说，游戏派>办公派。

“办公派”的元宇宙，第一步要复刻线下办公场景，第二步是增加新功能方便办公。只要目前公司制度下的层级体系不改变，想象空间就变得十分有限。而游戏作为一个没有现实对标的纯粹虚拟领域，本身就是天马行空、可以任意扩展。

这也就是为什么办公软件开发来开发去，功能基本差不多，而一代又一代的游戏，不断刺激玩家的兴奋点，实现口碑利益的双丰收

另一点，工作很难变成游戏，但游戏可以是新的工作。

前一阶段火爆的区块链游戏Axie Infinity，无数投入资金的玩家没有时间操作，催生了庞大的代练团队，代练最多的菲律宾一跃成为这个游戏最大的市场；Roblox这类游戏平台上的游戏制作人，可以通过售卖道具获得收益；精通游戏玩法、商业模式的玩家，现在已成为了VR游戏公司的香饽饽……

四、仰望星空+谨慎期待

10月底11月初，元宇宙概念股集体走强，多家公司股价涨停，元宇宙概念“带头大哥”中青宝因股价涨幅明显、较同期创业板综指偏离度较大，已经收到深交所关注函。

市场上，与元宇宙相关的课程299、399、699都不罕见，有人甚至通过卖课赚得盆满钵满。各种游戏的代码抄来抄去、卖来卖去……成功利用人们追逐潮流的心理。

可以肯定的说，目前关键技术都还没有突破、传统现实社会对元宇宙的认知远远不够，所以元宇宙不会出现在“不远的将来”，而是很远很远的将来。

一定要在仰望星空的同时，谨慎期待元宇宙。切勿因为跟风造成财产损失。
