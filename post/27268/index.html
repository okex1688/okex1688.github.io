NFT市场的火爆仍然在继续。

这个夏天，NFT彻底出圈，不仅各大拍卖行拍出各种各样的作品，各种运动员、时尚歌手、艺术家、品牌都在争相推出自己的NFT。国外品牌如路易威登、Burberry、奥迪、保时捷、可口可乐，以及国内的腾讯、支付宝等都在今年夏天发布了NFT藏品。

“乱花渐欲迷人眼”，每天大量的NFT面市，哪些值得购买，这些数据或许能说明一些问题。

1，这些项目成交量较高

首屈一指的当然还是这些天最为火热的NFT游戏，Axie infinity。

热门NFT最近7天交易量 数据来源：Dappradar

另据TokenTerminal数据显示，NFT+区块链游戏项目Axie Infinity单日收入为1227万美元，近7天总收入1.023亿美元，近30天总收入3.085亿美元。

除交易量之外，Axie Infinity参与者的活跃度也很高。

紧随其后的是CryptoPunks。由于巨鲸的加持，CryptoPunks的交易不断创下新纪录。

8月7日，一个CryptoPunk NFT以1500个ETH近440万美元的价格售出，成为迄今为止最昂贵的Punks之一。这个CryptoPunk是10,000个此类收藏品中的第2338号，它是整个系列中仅有的88个僵尸之一。

所以，选对项目的稀缺性同样很重要。显然，唯一的 NFT 是非常稀缺的，因此它们比多数量的作品更有价值。

2，创作交易市场

NFT来势汹汹，各类各具特色的NFT创作交易平台也逐渐完善。以下是对目前相对主流的创作交易平台做出的盘点。用户可根据个人喜好自行选用。

3，盛世买古董

今年三月，佳士得拍卖会上 6900 万美元成交的NFT 艺术作品《Everydays: The First 5000 Days》无疑创下了一个记录。在NFT收藏品领域，还有另一个项目因为高价值闻名遐迩，那就是CryptoPunks。

历史上交易价格最高的NFT 数据来源：dappradar

除了稀缺性，CryptoPunks作为NFT市场元老级的项目，不仅激发了 ERC721 型 NFT 标准，而且还引领了潮流，现在的它显然成为了加密文化乃至“元宇宙”的代表。

与CryptoPunks相似的还有CryptoKitties、以及近期兴起的EtherRock。

EtherRock于2017 年底便已经推出，是一组以石头为主题的 NFT 收藏品，限量 100 个。这个被忽视的早期 NFT 项目最近被重新发现，随着以太坊用户铸造了剩余的石头，这些 NFTs 从上周开始获得很大关注。甚至连 Coinbase 投资者、社交媒体影响者和 NFT 收藏家 Gary Vaynerchuk 也在推特上发布了这一消息，同时讨论了其作为早期 NFT 项目具有的增值效益。

数据来源：EtherRock

4，NFT租赁

如果你很喜欢 CryptoPunk 的图像，但是又负担不起它昂贵的价格，而且不想背负盗窃的骂名，用户可以试试 CryptoPunk.Rent 的租用 NFT 服务。

CryptoPunk.Rent 可以让用户以少得多的价格获得一个 CryptoPunk 的短期使用权（最多99天），在租赁期内，你可以尽情使用这个 CryptoPunk。

除了 CryptoPunk，其他的 NFT 也可以通过这个概念租赁出去。

可租赁的CryptoPunks 数据来源：CryptoPunk.Rent

reNFT 平台是首个提出租赁 NFT 概念的交易平台，在 reNFT 网站上你可以很简单的租借或者出租一个 NFT，选择好租用的时间，支付一定数额的押金和租金， 便可以实现虚拟资产的转移。

不过，需要注意的是，这种租赁行为有一定的风险。

5，千金难买我乐意

如果你有一定的预售，那就买自己喜欢的。毕竟最近有太多的NFT面世，总有一款是自己喜欢的。
