Mask Network 凭借一款插件先于 Twitter 在 NFT 赛道进行前瞻性布局，除 NFT 展示和 NFT 头像外，可能还会在之后让 CryptoPunks 成为项目代言人或 CEO。

社交巨头 Twitter 在近期官宣支持比特币打赏内容创作者的同时，还表示正在探索 NFT 展示功能，希望能在炙手可热的 NFT 市场分一杯羹。

当市场对 Twitter NFT 展示功能充满期待时，殊不知，Mask Network 早已凭借一款插件领跑 NFT 和 SocialFi 赛道，并抢滩布局了包括在 Twitter 上收发红包、发布加密推文、展示并购买 NFT 等在内的多款功能。

而 Mask Network 在做的远不止于此，除了年初上线在 Twitter 中识别展示 OpenSea 等平台 NFT 功能外，Mask 最近还推出 NFT Gallery （NFT 墙）、 NFT Avatar （NFT 头像）、NFT 红包功能，同时还针对部分用户推出了 NFT 头像功能，甚至之后可能支持用 NFT 进行打怪，体验类似于 QQ 宠物一样酷炫的小精灵功能。

「当下的 NFT 与被写进历史课本的『文艺复兴』具有相似的革命性，Mask Network 追逐的正是让更多人能够轻松地参与其中。」在向我们阐述其对 NFT 市场的看法以及对 Mask Network 的目标时，Mask Network 创始人兼首席执行官 Suji Yan 如是说。而关于未来，Suji 还提出了一个大胆的设想。

Mask Network：NFT 产品领跑者

Web3.0、元宇宙以及 SocialFi 是 Mask Network 一直追求的关键方向，也因此，Mask 希望通过一种简单易用的方式让传统社交媒体上的海量用户轻松参与进来并将现实生活连接到一起，从而触及越来越多的人。而在当前 Twitter 就是 Mask 的主战场。

自 2017 年创立以来，Mask Network 主攻在 Twitter 上通过插件模式增加去中心化应用相关功能，目前，Mask Network 插件已经在 Twitter 上实现了一个相对闭环的全套流程，包括发布加密推文、支持多链、发红包、Gitcoin 捐赠、集成 CoinGecko 和 CoinMarketCap 等信息平台、接入 Uniswap、SushiSwap 和 QuickSwap 等去中心化应用等。

而在 NFT 方面，除了此前支持的通过 OpenSea 和 Rarible 购买 NFT 或参与 NFT 竞拍，NFT 市场平台内作品 URLs 的识别和展示，以及购买 Valuebles 上的推文 NFT 之外，Mask Network 在近期明显加快了布局步伐，首先上线了 Twitter NFT 展示（NFT Gallery）功能，用户安装新版本的 Mask 后，在 Twitter bio 中添加以太坊钱包地址或 ENS 域名，便可以在 Twitter Profile 页面上展示个人钱包中的 NFT，同时可以浏览其他在 bio 中填写以太坊地址或 ENS 域名用户的 NFT 收藏。

通过 Mask 插件直接在 Twitter 上查看数字艺术家 Pak 的 NFT 收藏品

当前 Twitter 上不少大 V 或普通 NFT 持有者会将他们的头像换为持有的 NFT，如企鹅、宠物猫、CryptoPunks 等，并将这个头像当作是他们的个人虚拟形象。考虑到此，Mask Network 还推出了另一个令人期待的 NFT 头像功能，用户可以选择其拥有的一个 NFT 用作 Twitter 头像，当前头像功能仅面向部分 KOL 推出，不过，若某一 KOL 设置 NFT 头像，所有安装了 Mask 插件的用户都能看得到。该功能将在之后不断升级和为所有用户开放。

这个 NFT 头像不是简单的 JPG 格式，而是通过直接识别用户钱包里的 NFT 来替换的，并通过相关徽章证明当前 NFT 头像的真实性和价格等信息，而且点击 NFT 头像还可直接链接到 OpenSea NFT 页面。

CryptoPunks 开发公司 Larva Labs 创始人 Matt Hall 的 NFT 头像

在社交平台上集成有关 NFT 的一系列服务以及文件分享离不开存储基础设施。为此，Mask Network 支持用户直接在 Twitter 和 Facebook 上使用去中心化文件存储服务，目前支持 Arweave，之后将支持 IPFS、Sia 以及 Swarm。此外，Mask Network 在积极与其他存储、存储加速网络 Meson 以及链上内容协议 RSS3 等项目合作或进行投资来增加对其底层基础的建设以及拓展应用场景。

值得一提的是，今年 8 月份，由 Twitter 创始人 Jack Dorsey 资助的去中心化社交媒体 Bluesky 开始由 Zcash 早期开发者 Jay Graber 负责运营，而 Mask Network 等 11 个项目成为 Bluesky 的首批社区推荐项目。

Mask 动作频频，创始人如何畅想未来？

Mask Network 最近还以 159 ETH 的价格购入 CryptoPunk #6128，并将这个 CryptoPunk 和另外两个 Loot 资产存入 MaskDAO，团队还购买了其他比如 MoonCat 等在内的 NFT。根据 Suji 设想，「未来 Mask Network 会完全去中心化，这个 Punk 或者之后购买的其他小人可能还会在几年后成为的虚拟代言人，或者代替我去当 Mask 的 CEO，可以每隔几年换一届。」

Suji 表示，Mask Network 团队成员在很早的时候就买了 Loot。Suji 表示，Loot 带有装备，之后 Punk 还可以穿上皮肤装备拿着「石头」、领着「宠物猫」或者「企鹅」去打怪。

有了前一部分可以验证真伪的 NFT 头像后，Suji 称，Mask Network 生态社区还准备参考 QQ 宠物推出小精灵功能，到时候用户的 Punk 小人就可以带着 Loot 装备在自己的 Twitter 页面「闲逛」，也可以去其他 Twitter 用户主页「做客」、「偷菜」，带有小怪兽 NFT 的两个 Twitter 头像之间还可以「互动」。

Suji 在谈及 Mask 的未来规划和畅想时非常兴奋，尽管目前看来这听起来略微有点不切实际，但这种虚拟形象所带来的机遇和革命性作用是非常值得期待的。

Suji 对 NFT 市场的预期

任何一个领域不管是市场情绪、认知以及经济上都或多或少存在周期性。NFT 从今年三、四月份正式出圈至八月份迎来爆炸性增长仅用时不到半年时间，不过近期热度出现了下滑趋势，接下来我们将了解一下 Suji 对 NFT 的看法和未来预期。

在 Suji 看来，「目前 NFT 市场还处于早期阶段，虽然不确定最终是不是泡沫，但目前来看远远未达到顶峰状态。价值判断不能仅依据市场情绪来判断，短期热度的下滑完全不影响之后的发展趋势，NFT 未来还可能有 10 倍、100 倍乃至上千倍的空间。」

Suji 还将 NFT 与文艺复兴作了对比，两者都是革命性的，即，有一小部分人希望改变当前形成的这种意识形态并想要推动社会进化，不过 NFT 的革命性被市场冲淡或者隐藏掉了。此外，Suji 还分享了其在 8 月份参加关于发展区块链和数字经济的《新加坡数字经济论坛》时的经历，赵长鹏、Vitalik Buterin、Three Arrows Capital CEO Su Zhu、吴忌寒以及部分新加坡政客也参与了会议。会议期间有人在一边买着 CryptoPunk，还有人穿着印有彩虹猫的 T 恤，Suji 认为这充分说明在这个领域有很多人是非常看好元宇宙的。

小结

当前 Mask Network 已形成了涵盖 DeFi、NFT、存储、工具、交易以及隐私等领域的一体化生态体系，生态项目多达 50 多个。伴随着 Mask Network 插件功能的新增和迭代，加之未来插件功能拓展到更多的社交平台，以及其始终秉持去中心化和开源的理念，将切实地影响着越来越多的互联网用户，推动 Web2.0 向 Web3.0 的演变革新。

而 NFT 展示和头像功能的实现，以及赋予 NFT 人格角色甚至将其打造成 CEO 专属 IP 的畅想，在带动 NFT 整体价值的进一步提升的同时，还能推动实现真正的元宇宙，即一个与现实世界平行的虚拟世界。

未来，我们将势必会进入到一个全新的 Web3.0 世界，而 Mask Network 先于 Twitter 在 NFT 赛道进行前瞻性布局，将在这当中充当可以绕过复杂基础设施而直接参与 NFT 与 DeFi 等的中间件和一站式入口，能够在不改变普通用户习惯的情况下，依托于 Twitter 庞大的用户数量优势产生潜移默化的量变。
