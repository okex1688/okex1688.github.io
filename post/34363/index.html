如今，春晚不仅是一场文化盛宴，也成为映射我国数字技术发展的一个重要窗口。春晚舞美的“进阶史”背后，是数字技术的发展与进步。

随着元宇宙概念的大火，与之相关的虚拟人、VR/AR/XR等技术也被各大春晚所应用。央视总台春晚有机融合XR、虚拟、全息等技术，呈现了全景式舞美的空间感；北京台春晚将虚拟人“苏小妹”融入古风歌舞；安徽台春晚上，虚拟人“小安”成为主持人……2022军营网络春晚更是首次落地“元宇宙概念”，搭建了多维空间和互动平台，提供了特定场景下跨域交互的沉浸式体验。

春晚已经成为展示新科技的最大流量入口。文化传媒是元宇宙的重要应用场景。在这些“元宇宙”春晚背后，有哪些科技公司的加持？这些公司又是如何布局元宇宙的？

  • 捷成股份：通过元宇宙布局，加强“内容版权”主业优势

在2022央视春晚中，捷成股份（300182.SZ）参与了部分节目的制作及央视春晚高清/4K重播版的制作任务。捷成股份在2021年业绩预告中提到，“仍将加强在 4/8K 超高清技术、三维声技术、虚拟技术等相关高新视频业务领域的布局，积极探索元宇宙时代下相关音视频技术的市场需求及发展方向，不断提升市场竞争力。”

为应对监管环境的变化和现金流压力，目前，捷成股份确立了“以内容版权运营”为核心的发展战略，将业务聚焦在新媒体版权运营及发行方面。

2016年，捷成股份参股子公司世优科技，目前持股比例为24.38%。世优科技是一家专注于VR/AR研发与应用的科技公司，其虚拟数字人、虚拟演播室等相关业务被运用于广电和新媒体行业，目前正在探索虚拟 IP 运营业务，并已经打造了诸如花样滑冰冠军陈露的虚拟分身、妙江天的虚拟 IP 等，未来将在直播、短视频、虚拟演唱会等领域进行布局。

图片来源：世优科技公众号

此外，捷成股份还计划将的新媒体版权运用于世优科技元宇宙分身秀平台，打造元宇宙平台的电影复播。

可以看出，捷成股份主要通过虚拟人赛道切入元宇宙领域，通过虚拟IP运营，加强其在内容版权方面的优势地位。第一，通过虚拟现实技术，可以实现元宇宙场景下的虚拟人IP矩阵打造、虚拟内容版权和生态运营，以及以虚拟 IP 衍生为主的流量变现与增值业务；第二，通过元宇宙分身秀平台，能够将公司的IP和影视剧内容进一步开发利用。

虽在业务布局上具有前瞻性，但捷成股份并未因此获得可观的业务收入。据其相关公告介绍，截至目前，世优科技的收入规模对公司收入及利润不会产生重大影响，对公司后续年度的业绩和经营成果的影响尚存在不确定性。

  • 数码视讯：从元宇宙底层技术切入数字藏品

2022年元宇宙军营网络春晚是首次将“元宇宙概念”落地的春晚，通过 “元宇宙”虚拟舞台，让所有参与者能够在春晚的平行空间里，跨维度、跨时空徜徉其中，由数码视讯（300079.SZ）子公司快视听与中国军网联合推出。

图片来源：洞壹元典公众号

作为一家视频技术服务平台公司，数码视讯业务涵盖传媒科技行业、信息服务行业、公共安全行业、金融科技行业、特种需求定制及其它行业。多元化的业务布局也为其在元宇宙赛道的多维度发展奠定了一定的基础。

数码视讯的元宇宙布局涉及底层技术、数字藏品及交互应用场景等。

数码视讯的安全系统框架、低延时传输、超高清编码、5G视频应用、区块链应用及智能合约等技术可为元宇宙提供底层技术支持；它还创建了数字藏品流转平台“洞壹元典”，洞壹元典平台发展初衷是希望以数字藏品为资产框架，打造有藏品属性的数字内容，构建多元化数字藏品社群，并以内容链接进入元宇宙；而在构建元宇宙交互应用场景方面，数码视讯推出了“元宇宙会议”、“元宇宙直播”，“元宇宙展览”等场景。

  • 蓝色光标：以虚拟人为切入口，“激进式”布局元宇宙

2022年2月1日，蓝色光标（300058.SZ）旗下首个数字虚拟人苏小妹在北京台春晚亮相，苏小妹是蓝色光标虚拟IP业务的首发形象，以传说中苏东坡的妹妹为创意原型，截至2月7日，“真的是苏小妹”抖音账号粉丝量已达39.5万。

图片来源：蓝色光标公众号

蓝色光标在元宇宙上的布局堪称“激进式”布局。从组织架构调整，到业务布局，再到投资合作，蓝色光标似乎在进行一场“全方位”的元宇宙布局。

蓝色光标曾表示，元宇宙相关业务方向是公司未来会重点关注并全力投入的赛道，公司对这一领域的投资额度不设上限，公司管理层和投资部目前正在对相关标的进行全面的调研和谈判，公司可支持超10亿的资金投入布局元宇宙相关赛道，全面构建公司的自有IP、技术和内容核心能力。

然而，蓝色光标作为一家营销科技公司，元宇宙的加持能否为其主营业务带来质的蜕变，似乎还难以确定。元宇宙相关技术和概念仍处于发展早期，将营销活动移植到元宇宙中，可能会面临技术、消费者习惯、收益等多方面的问题。

  • 风语筑：将元宇宙融入文化新基建

2021年11月，数字化体验服务商风语筑（603466.SH）正式宣布布局元宇宙赛道，并计划将浙江深白数字科技有限公司更名为浙江风语宙数字科技有限公司（简称风语宙），经营范围新增虚拟现实设备制造、动漫游戏开发、人工智能应用软件开发、区块链技术相关软件和服务等。此外，风语宙还计划通过自主研发和外部合作的方式，探索元宇宙游戏领域。

在2022年安徽春晚中，风语宙与安徽广电共同打造的虚拟人“小安”成为主持人。除了虚拟人，风语筑还在数字藏品及元宇宙场景等方面进行了探索。

图片来源：风语筑公众号

数字藏品方面，风语筑投资NFT发行技术公司星图比特，并联合闪萌推出国内首个NFT表情包数字藏品；元宇宙场景打造方面，与百度达成元宇宙生态合作关系，作为希壤元宇宙虚拟3D开发商之一，承接3D虚拟建筑、数字空间的开发打造及内容运营，并于希壤内上线风语筑场馆，此外，还与万达影城、星轶影城达成战略合作，将开拓元宇宙泛娱乐时代虚拟场景。

风语筑以展示展览为主业，元宇宙概念中的人机交互、VR/AR等技术能够帮助其展陈内容融入更多的创意、设计、数字艺术和沉浸式体验元素，从而增强各类数字化体验空间的科技感和体验感。

结语

总结以上代表性科技公司的元宇宙布局路径可以看出，它们均成立了相关子公司，来专注元宇宙赛道的发展，并且将虚拟人作为切入元宇宙赛道的重要突破口。同时，在虚拟人的推广和应用上，这些公司也面临政策上的利好。《广播电视和网络视听“十四五”科技发展规划》中指出，要面向新闻、综艺、体育、财经、气象等电视节目研究虚拟形象合成技术。

此外，可以发现，这些科技公司希望通过与元宇宙结合，来助推主营业务的发展。由于元宇宙相关技术、应用场景等仍处在发展初期，这些科技公司在实际推进过程中仍存在较大的不确定性，在元宇宙相关布局上也未成体系，仍处于探索阶段。
