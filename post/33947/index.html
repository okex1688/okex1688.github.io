NFT (非同质化代币) 正在席卷世界。Steve Aoki 和 Jay-Z 等名人在 Twitter 上自豪地展示了他们的 NFT 头像。阿迪达斯和耐克等主流品牌已经进入这个领域：阿迪达斯通过推出自己的 NFT 收藏品并筹集了 2300 万美元，其 NFT 买家将获得阿迪达斯的“数字和实物”产品和体验；耐克收购了 NFT 领域的虚拟设计工作室 RTFKT，以进军元宇宙。一些 NFT 也正通过苏富比和佳士得拍卖行以数百万美元的价格进行拍卖。

随着越来越多的主流采用 NFT，全球越来越难以忽视这一上升的浪潮。在本文中，我们将深入挖掘炒作之外的内容，并涵盖你想了解的关于 NFT 的一切。

NFT 是什么？

NFT 是一种存储在区块链上的有着不可互换的元数据 的加密代币，每个 NFT 都具有唯一性。 相比之下，其他可替代代币 (比如 ERC20 代币) 是可以互换的，BTC 和 ETH 等也是可互换代币的例子。每枚 BTC 都是可以互换的，用户对于他们购买或者接收的是哪枚 BTC 并不在意；而每个 NFT 都是具有唯一可识别性 ，在大多数情况下，用户会在意他们购买或接收的是哪个 NFT。

区块链是分布式和去中心化的数字账本。它们是由按时间顺序连接在一起的数据块组成。目前NFT 领域使用最多的区块链是以太坊 。与此同时，我们也开始看到诸如 Flow 和 Ronin 等这样的小众区块链而崛起，这些区块是基于 NFT 和游戏而构建的。

NFT 的元数据可以在其智能合约中找到，智能合约存储在区块链的区块中。关于 NFT 的一个常见误解是，一些人认为我们看到的 NFT 数字头像或表示 NFT 的 JPEG 图片存储在区块链中——虽然对于少数项目(比如 Anonymice、Chain runners) 是这样，但大多数 NFT 项目是将这些图像存储在外部存储平台 (比如 IPFS、Arweave 等) 上 ，同时存储区块链上的 NFT 元数据包含了连接到这些图片的链接 。原因在于，由于图像的文件大小，将图像存储在区块链上的成本非常高，这使得 NFT 项目完全存储在链上是不经济的。

NFT 解决了什么问题？

互联网通过将一切数字化来解决现实世界的问题。当前比以往任何时候都更容易拥有无限的规模和触及全球的互联网用户，分销的成本也大大降低了，比如新闻通讯 vs 报纸。

然而，这种数字化也缺乏物质世界的好处。所有权和稀缺性几乎不存在，与实体商品相比，对数字资产进行认证要难得多。就像“右击鼠标并保存”NFT 是一个模因一样，这反映了当前的互联网状态。数字资产没有所有权，也几乎没有增值。

就其核心而言，NFT 提供了当前数字经济中缺失的所有权证明 。它创造了一种能力，将数字世界的好处 (全球受众、规模和分销) 与实体世界的好处 (所有权和真实性) 结合起来。

在 NFT 兴起之前，数字资产很难积累和捕获价值。这使得其对于用户来说非常便宜，但是对于创作者们来说却很糟糕。如今，在互联网上追踪和确定真正的所有权是很困难的，而 NFT 解决了这一问题。创作者们会因为他们的工作获得更准确的报酬，并且能够从他们创造的价值中提取更大的比例。用户也受益于能够获得数字资产的所有权，并能够轻松地交易和估值。

NFT 有何用例？

1. 艺术

NFT 最流行的用例是作为可收藏的艺术品 NFT。NFT 为艺术家们提供了一种轻松地将自己的艺术品出售给全球受众，同时确保了所有权。艺术 NFT 大体上可以分为两类：非生成艺术 & 生成艺术。

1) 非生成艺术

当我们想到艺术时，我们通常会想到非生成艺术 。它们是人类在没有自主计算机系统的帮助下创作出来的。数字艺术家能够在 NFT 市场 (比如 OpeaSea) 上架他们的艺术品，这样世界上任何地方的任何人都可以查看和购买他们的作品。

在今天的传统市场上，艺术家们只能从他们的作品的初始销售中赚钱；而 NFT 让艺术家可以选择将版税写入他们 NFT 艺术作品 的智能合约中，这使得他们能够从他们的艺术品价值中获得更加公平的份额。NFT 使得艺术品的真伪和所有权的验证变得容易，这是这项技术带来的巨大好处。这一领域的知名艺术家包括：Beeple、pplpleasr、ferocious 和 Pak。

照片 NFT 也是一个正在兴起的 NFT 用例。一些摄影师们正在将他们的锁屏以 NFT 的形式进行出售，并获得与出售 NFT 艺术品的艺术家类似的收益。Justin avisano 是这个领域最著名的人物之一。当他的“Twin Flames” NFT 系列中的一个 NFT 在克里斯蒂拍卖会上被拍卖时，他创造了历史。这类 NFT 的主要市场包括了 Opensea、SuperRare、Foundation、Rarible 和 Nifty Gateway。

2) 生成艺术

除了由艺术家单独绘制和创造的艺术，艺术品 NFTs 的兴起在很大程度上归功于生成艺术 。生成艺术是通过使用一个自主的计算机系统 (通常是一个算法计算机程序) 来创造的艺术。像 CryptoPunks 和 Bored Ape Yacht Club (BAYC) 这样的个人资料图片 (PFP) 系列就是在这些程序的帮助下创建的。创作者们把每个单独的特征都塑造出了，然后程序就会将这些特征随机组合起来，通过组合这些特征创造出独特的角色。

一些生产艺术 NFT 系列是完全由计算机程序完成的，除了需要创建算法之外，无需任何人力投入。这种类型的 NFT 系列中最受欢迎的就是生成艺术平台 Art Blocks 的一些系列，例如 Snowfro 创作的“Chromie Squiggle”和 Tyler Hobbs 创作的“Fidenza”。OpenSea 是生成艺术 NFT 的领先市场，LooksRare 也是一个受欢迎的新推出的 NFT 市场。

游戏

虽然艺术 NFT 是已经获得了主流关注的用例，但有一个 NFT 系列在销量方面超过了所有的艺术 NFT 收藏品：Axie Infinity ，这是一款 NFT 游戏，目前在交易量方面夺得桂冠 (不包括 LooksRare 上的清洗交易)。这种 NFT 技术与游戏的整合创造了一种全新的游戏模式：边玩边赚/GameFi 。

上图：2022年2月3日交易量靠前的 NFT 系列，数据来源：DappRadar

Axie Infinity 是2021年的一颗新星，并开启了 P2E (边玩边赚) 游戏的崛起。该游戏在 2021 年创造了超过 13 亿美元的收益，并拥有超过 250 万的日活跃玩家。与传统游戏不同的是，传统游戏的发行商通过向玩家出售资产来获取收益，而 Axie Infinity 则通过从市场交易和 Axie 数字宠物的繁殖费中抽取佣金来获取收益的。玩家通过繁殖这些 Axie 宠物来创造出新的 Axie。该游戏团队的动机是创造出最好的游戏以扩大用户基础，而不是向用户出售资产。

上图：Axie Infinity 的累计收益增长情况 (蓝色部分表示其 NFT 市场产生的费用，绿色部分表示 Axie 宠物繁殖时产生的费用)。数据截至2022年2月3日。来源: DappRadar

在游戏中整合 NFT 可以为玩家提供与艺术 NFT 类似的优势，让玩家能够真正拥有对自己的游戏内资产的所有权 。游戏内的道具甚至角色本身都可以是 NFT。在 Axie Infinity 中，为了玩游戏，你需要拥有被称为 Axie 的 NFT 数字宠物，而游戏内部货币是加密货币 (SLP 代币)。在传统游戏中，流入游戏中的钱通常会滞留在游戏中，因为玩家无法从游戏中获取价值。而 NFT 游戏给予玩家对资产的控制权和所有权，他们可以决定对 出售和变现这些资产 。

上图：传统游戏 (左) vs 区块链游戏 (右) 中的价值流动。图源：Nansen

音乐

音乐 NFT 是 NFT 的一个新兴用例。就像艺术 NFT 一样，音乐 NFT 是艺术家从他们创作的作品中获得更大份额价值的关键 。音乐艺术家们通常只能从他们为音乐行业创造的收入中获得约 12% 的收入。最近，音乐艺术家兼制作人 Daniel Allan 将自己制作的一首音乐铸造成了一个 NFT，并在 OpenSea 上面以 1.1 ETH 的价格出售了该音乐 NFT，这个价格相当于在 Spotify 上 120 万次流媒体播放量。

这一趋势也赋予了那些真正粉丝以力量，使他们能够支持和拥有他们最喜欢的音乐艺术家创作的作品的副本 ，而且是直接从艺术家那里获得。这也为艺术家们创造了新的方式，让他们能够以现有方法难以达到的水平吸引粉丝。通过拥有艺术家作品的原创副本，粉丝们能够对艺术家的未来成功进行押注。艺术家能够得到更公平的回报，而作为一个粉丝，你也可以在艺术家的早期帮助他们 。Soun.xyz 和 Catalog 这两个项目正是为音乐艺术家们提供了一个直接将自己的作品以 NFT 的方式出售给粉丝的平台。

体育收藏品

体育 NFT 是一个不断增长和蓬勃发展的 NFT 用例。与棒球交易卡类似，体育 NFT 允许粉丝创建他们最喜欢的球队和球员的收藏品。目前最受欢迎的两个体育 NFT 项目是 NBA Top Shot 和 Sorare 。这两个项目都与相关的组织有官方正式的合作关系并拥有官方许可证。

NBA Top Shot 本质上是创建 NBA 比赛中精彩时刻时刻的 GIF，并随机将其捆绑到数字卡片包中，每个时刻都被铸造成为一个 NFT。这些卡包有着不同的稀缺性，取决于该精彩时刻中的球员、重要性和受欢迎度。某个时刻越稀有，那么其 NFT 供应就越少。普通卡包 9 美元起，传奇卡包 999 美元起。用户可以在 NBA Top Shot 的官方市场上交易该 NFT 系列。在 2020 年总决赛中，Lebron James 的一记传奇扣篮的 NBA Top Shot 以 23 万美元的价格成交，这是迄今为止该系列最高的成交价。

截图源自 NBA Top Shot 市场

Sorare 是梦幻足球和 NFT 的结合。每张球员卡牌都是 NFT，且奖励是以 ETH 的形式支付。用户可以在 Sorare 市场交易这些 NFT 卡牌。Sorare 的好处与 NFT 游戏类似，即玩家拥有对游戏内部资产的所有权，以及从游戏中兑现和提取价值的自由。

通行证

由于通过 NFT 进行身份验证的简单性及其验证所有权的能力，NFT 可以充当访问通行证 。Flyfish Club 就是这样的一个例子，Flyfish Club (FFC) 号称是世界上首个经通过持有其 NFT 才能加入的私人餐饮俱乐部；此外，阿迪达斯的 NFT 收藏品 为持有者提供独家获取该品牌的实体和虚拟商品。虽然一些 NFT 收藏品的唯一目的是作为会员通行证，但这个用例是许多 NFT 收藏品的首要主题，包括 BAYC (无聊猿游艇俱乐部) 也对 Bored Apes (无聊猿 NFT) 持有者提供特殊的福利。所有的 NFT 形式都可以作为各种类型的通行证。

上图：Gary Vaynerchuk 的 Flyfish Club NFT

推动艺术 NFT 兴起的一个关键因素是支持 NFT 项目的社区 。NFT 的持有者可以独家访问某些 Discord 聊天频道，并且在社区中有一种归属感。大型项目 (比如 BAYC) 还会为持有其 NFT 的人举办线下聚会和专属派对。持有某些 NFT 可以让你有资格在早期获得同一个团队的某个新的 NFT 项目的收藏品，或者该团队合作伙伴的其他新的 NFT 项目。与阿迪达斯 NFT 类似，一些 NFT 收藏品 (比如 BAYC 和 Creature World) 也发布了自己的实物商品，其 NFT 持有者可以获得对这些实物商品的早期/独家获取资格。音乐艺术家和游戏也可以向其 NFT 持有者提供类似的福利。

DAO (去中心化自治组织) 和游戏公会 是另一个 NFT 可以作为访问通行证的领域。持有相关的 NFT 将使你成为某个 DAO 或游戏公会的成员，且该 NFT 还可以作为投票代币。Yield Guild Games (YGG) 就是一个以这种方式使用 NFT 的游戏公会组织，其 NFT 持有者可以访问其网站上的独家功能，并注册该公会中的各个不同的项目。

如何创建 NFT？

创建 NFT 的方法有很多，你可以手动去创建，也可以借助 NFT 市场 (比如 OpenSea 和 Rarible) 创建的系统 。创建 NFT 时除了需要掌握一些技术细节之外，还有一件重要的事情需要考虑，即你想要在哪条区块链 上创建你的 NFT：以太坊、Solana、Avalanche、Polygon等等。不同的区块链有不同的权衡和社区，这可能会对 NFT 项目的成功产生很大的影响。

如何购买 NFT？

就交易量而言，最大的 NFT 市场是 OpenSea ，这里可能是你 NFT 交易之旅的起点。虽然今天最受欢迎的 NFT 收藏品都建立在以太坊上，但在其他区块链上也有蓬勃发展的 NFT 项目，比如Solana、Terra 和 Polygon。其他受欢迎的 NFT 市场包括以太坊/Polygon 上的 LooksRare 和Rarible ，Solana 上的 Magic Eden 和 Solanart ，以及 Terra 上的 Random Earth 。

为了与这些 NFT 市场交互，你需要设置一个与 NFT 项目所在的区块链兼容的加密钱包 。最常用的钱包是以太坊的 MetaMask，Solana 的 Phantom 和 Terra 的 Terra Station。一旦你设置了一个钱包，将 ETH/SOL/LUNA 从你选择的中心化交易所转移到该钱包中，你就可以开始了购买了！

如何选择 NFT 进行投资？

当考虑购买 NFT 作为投资时，重要的是要全面地评估 NFT 项目，而不是仅仅依赖于外观。与挑选要投资的 Crypto 项目类似，创始团队的质量、其执行和交付能力以及项目路线图等都是需要考虑的重要因素。也许最重要也是最难衡量的因素是 NFT 项目已经成功构建的社区实力 。NFT 的价值主要来自其社区，而拥有强大社区的项目将能够承受熊市 。我们甚至看到过这样的案例：社区通过交付创始团队失败了的项目而团结起来，重振项目，比如最近的一个例子 Pudgy Penguins (矮胖企鹅)。

真正衡量一个社区实力的唯一方法是融入进去，社区成员的质量比数量更重要。加入 NFT 项目的 Discord 频道，在里面转转，感受一下成员们的气氛。如果谈话主要围绕底价展开，而环境又很糟糕，那么这可能不是你想押注的项目。

随着每天发布的 NFT 项目的数量在增加，几乎不可能跟上所有 NFT 项目的进度。一种策略是跟踪精明的投资者，关注那些激起他们兴趣的项目，NaNsen 平台的 NFT 仪表盘在这方面提供了帮助：你可以在“NFT Gold Mode”中查看特定项目的交易数据，在“Mint Master”上查看 NFT 的铸造数据，并在“NFT Smart Money”上获得过去 24 小时内聪明的钱的动向。

对 NFT 的常见批评

  • “为什么要花几千美元去购买一张我只要右击鼠标就可以保存的 JPEG 图片？”

是的，你确实可以通过“右击鼠标并保存”的方式来获得一件艺术品的副本，但这并不等同于你拥有了实际的 NFT。就像仿冒奢侈品的价值永远无法接近正品一样，即使材料和制造质量完全相同，一个“右击鼠标保存”的 NFT 版本也永远不会有相同的价值。仿冒品，或者在这种情况下，“右击鼠标保存”的版本，只会增加原版的价值 。这创造了一个反馈循环：原始作品被复制 > 获得病毒式传播和吸引力 > 价值上升 > 该作品被复制得更多。

一张特定的图片在世界范围内被看到的次数越多，它就越能被广泛传播，它就越具有标志性，原版的价值也就越高。让《蒙娜丽莎》声名鹊起的是这幅画被盗时的病毒式传播，而不是这幅画出自列奥纳多·达·芬奇之手。登上世界各地报纸的头版，使其成为国际知名和家喻户晓的名字。

你可能会争辩说，实物赝品之所以不值和正品一样的价钱，是因为它们不是 100% 同一件商品。这对于实体商品是正确的，对于 NFT 也是一样的。这个概念更难理解，因为 NFT 是完全数字化的，而且从表面上看，复制的 JPEG 文件与原件是 100% 相同的。然而，就像实物一样，真正重要的是隐藏在其中的内容 。

一个“右击鼠标保存”的图片版本将不会在区块链上有相同的底层数据 ，它不会赋予你拥有真正的 NFT 所给予的同样的利益和权利 。它不会给予你参与事件或访问地点的权利，不允许你访问由持有该 NFT 的个人组成的社区，你无法在 NFT 市场上进行转售，或者获取游戏内的体验等等。

NFT 的实现并不意味着用户将不得不支付更多，它只是意味着用户有一个选择。不想花几千美元买一张 JPEG？当然，如果你想的话，可以免费下载并使用。而对于那些想通过购买和拥有他们喜欢的创造者的作品来支持他们的人，他们现在有能力以一种比现有方法更公平的方式来做到这一点。从 OpenSea 上的交易量来看 (2021 年的 NFT 销售额约为 170 亿美元)，我们可以清楚地看到，有相当多的资本对这一领域感兴趣。

  • “NFT 对环境有害。”

虽然像以太坊这样的 PoW (工作量证明) 区块链确实是能源密集型的，但许多较新的区块链，甚至以太坊本身都在转向 PoS (权益证明)。像 Solana 这样的 PoS 区块链消耗的能源要少得多，而且可能是碳中性的。除此之外，NFT 不太可能导致额外的能源使用，因为如果不存在 NFT，其他应用程序 (如 DeFi 应用) 可能会占用区块链上的任何多余的交易容量。

NFT 的未来

未来的 NFT 可能与我们今天看到的 NFT 完全不同。虽然 NFT 已经成为艺术家们的一种可行且有价值的工具，但我们在使用这项技术方面还处于初级阶段。

考虑到 NFT 的主要功能是作为所有权代币 ，NFT 可以是任何资产 (包括物理资产和数字资产) 的代币化表征 。房契、学校证书、身份证等都是可以通过代币化为 NFT 来进行改进的潜在项目。由于区块链是不可篡改的，数据很容易验证，如果一个人的学历证书和过去的工作经历都存储在区块链上，那么机构可以无缝地进行验证了。

NFT 将不会作为一种孤岛式资产而存在，而是可以与 DeFi 协议集成，以创造一个流动性加密货币系统 。不妨想象一下这样的世界，我们可以从使用 NFT 作为抵押品，从诸如 Aave 或 JPEG’D 等 DeFi 协议中进行借贷。

当前，NFT 是一种流动性相对不足的资产，且其较高的价格使得小型玩家无法参与到某些 NFT 收藏品的增值中。NFT 的碎片化 将有助于增加该领域的流动性，并允许较小的参与者共享收益。一个追踪 NFT 总市值的 NFT 指数 可以为投资者创造新的投资渠道，并为那些希望涉足该行业但不想持有特定 NFT 的投资者打开空间。

一些受欢迎的 NFT 收藏品可能会开始成为「元宇宙 」中被高度重视的资产，而且在元宇宙中可能会出现为了广告目的而使用 NFT 的需求 。在现实世界中，品牌倾向于使用名人做广告，利用他们的品牌和影响力来推广他们的产品。而特定的 NFT 可能成为宇宙中有影响力的东西，品牌通过租用这些 NFT 来达到其推广目的。

随着时间的推移，成功的 NFT 项目本身可以成长为大品牌。一旦一个 NFT 项目在流行文化中巩固了自己的地位，获得了足够的关注和文化意义，它们就可能与 LVMH 这样的传统公司竞争。拥有这样的 NFT 项目中的 NFT 成为一种强烈的社会财富信号，使其变得非常令人向往，就像拥有一块劳力士手表一样。

总结

NFT 为我们提供了一种利用实体和数字世界的优势，为创作者们创造了新的分销渠道和盈利方式。它为我们提供了一种新的方式来参与和激励志同道合的人组成的社区，并有可能改变我们协调社会群体的方式。NFT 允许能够更好地界定和确定数字资产的价值，并让相关创作者从自己创造的价值中获得更大的份额。“右击鼠标并保存”将始终是互联网的一个特征，但 NFT 让创作者和用户可以选择拥有所有权，并对其资产拥有更大程度的分配控制。“右击鼠标并保存”也会增长初始版本的价值，而不是降低其价值。

目前 NFT 技术的实现是有限的，也不是完美的。然而，这并不会影响 NFT 长期的潜力和将来会出现的用例。我们只是处于虚拟经济崛起的第一个阶段，我们需要找到新的、令人兴奋的方式来利用 NFT 带来的好处。
