全球首个以古典音乐领域为依托所创建的NFT平台KOLO.Market，正式宣布与深耕于NFT分布式数据存储平台NFT.Storage达成战略合作关系。 此次战略合作关系的确立，标志着双方将携手共建一个数据安全、交易可信、去中心化存储保护NFT资产及相关元数据的优质平台。

拥有众多优质音乐版权内容的KOLO.Market在专注于开发音乐NFT丰富而多元的应用场景的同时，也致力于不断优化用户在平台上的所有关联体验，选择与NFT.Storage的合作，正是为了保障NFT本身的去中心化、资产安全与合理维护费用等高质量服务。如何提供更优质的内容与更亲和优越的使用者体验去联通并吸引更多的全球音乐爱好者与专业人士参与到整个元宇宙音乐生态体系中，正是KOLO.Market与NFT.Storage开启合作的共同目标。

有别于其他仅仅具备各项平台服务功能的NFT平台，KOLO.Market凭借其强大的合作联盟优势，不但具备良好的全球艺术家资源，在全球古典音乐版权的市场占有率更位居领先地位，因而数据资产安全、数据检索便捷与去中心化存储对于平台而言，都是极为重要的基础保障。NFT.Storage聚焦NFT领域，凭借“去中心化储存”和“内容寻址”等技术，为用户提供简单、高性能的存储和检索体验，极大降低了NFT维护成本，提高了数据安全。可以说NFT.Storage的特点优势对KOLO.Market来说，无疑是最佳的选择。双方将在业务契合的基础上，进一步开展深度合作。

关于  NFT.Storage

NFT.Storage 是一项长期存储服务，专为链下 NFT 数据（如元数据、图像和其他资产）而设计，每次上传的大小高达 31GiB。数据是使用 IPFS 寻址的内容，这意味着指向一段数据的 URI（“ipfs://…”）对于该数据是完全唯一的（使用内容标识符或 CID）。 IPFS URL 和 CID 可用于 NFT 和元数据，以确保 NFT 永远关联相应数据。

NFT.Storage 以两种主要方式在公共 IPFS 网络上存储上传数据的副本，即在由 NFT.Storage 管理的专用 IPFS 服务器中，以及在去中心化的 Filecoin 上。由于 IPFS 是许多不同存储服务使用的标准，因此很容易将上传到 NFT.Storage 的数据存储在任何其他与 IPFS 兼容的存储解决方案上，从固定服务（Pinning Services）到本地 IPFS 节点，再到其他存储网络（如 Arweave 或 Storj）。随着时间的推移，作为公共产品， NFT.Storage将逐步扩大去中心化程度。

关于 KOLO.Market

KOLO.Market是首家由纽交所上市公司投资孵化的古典音乐NFT平台，与包括全球最大的独立古典音乐集团拿索斯（Naxos）在内的全球上百家知名唱片公司建立长期合作关系。这些知名唱片公司录制的作品屡获殊荣，包括格莱美奖、留声机古典音乐奖、BBC音乐杂志奖等重量级国际奖项。

KOLO.Market通过区块链去中心化、可追溯且不可篡改的属性，解决版权确权、维权和管理等问题，保护并提高版税支付效率，以确保音乐人的合法权益，借以实现古典音乐版税的分权，打造数字音乐资产基于区块链技术应用的长期性透明平台。此外，KOLO.Market同时融入了web3的理念，让粉丝不再只是给予者，在支持自己喜欢的音乐人的同时还能获得回报，这无疑创造了一种可持续的、更健康的新模式。

随着Web3的广泛应用与元宇宙世界的来临，KOLO.Market作为全球领先的古典音乐NFT平台，数据存储安全与优化用户体验是KOLO.Market长期坚持的追求。未来，KOLO.Market将持续借助NFT.Storage以内容寻址、可证明储存与弹性检索的工作原理所成就的出色存储技术，持续开发更多利于用户的各式服务与产品，包括基于区块链的去中心化高音质流媒体平台、KOLO元宇宙艺术论坛、国际古典音乐赛事等，力求最大限度地满足全球音乐NFT用户的需求，共同在元宇宙世界中体验以音乐为核心的各式场景，以及智识与乐趣兼具的活动与社交。
