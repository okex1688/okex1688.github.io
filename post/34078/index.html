当谈到优秀的 NFT 项目时，我们将才华横溢的创造者、创新机制以及充满活力的社区视为重要因素。

那糟糕的NFT项目具有哪些特征呢？

下面，我列出了十三个危险信号，当你在探索新的NFT项目时，你需要将它们牢记在心，它们的范围从疏忽到彻底操纵的问题，让我们保护好自己的钱包。

危险信号1: 站不住脚的铸造（mint）价格

关于什么是“合理”的NFT mint价格，有很大的空间，这里没有单一的正确答案。但是，如果一个新团队的收费要远远高于它的“前辈”或同类项目，这表明他们可能更关注的是拿走你的ETH，而不是其他。

危险信号2:可疑的社交媒体活动

一个 NFT 项目方突然向你发送消息，以将你的注意力引向即将到来的NFT售卖活动，比如你在大约几个小时前就看到了一条广告推文。现在你单击该项目的个人资料，然后发现（1）你关注的人都没有关注该项目， 以及（2）这个新项目拥有大量（异常）的关注者。这些都是危险信号！好的项目不会通过私信或使用 Web2 广告的方式来引起个人的注意，它们会有自然的流量，不需要购买粉丝。

危险信号3:刻意维护地板价

如果你看到项目的领头人在讨论或努力维持某个NFT的地板价，那你应该离开这个地方，他们只是在运行一个财务计划，而对文化价值或社区几乎没有真正的兴趣。这种不真诚，会影响他们在未来遇到的所有麻烦，所以要小心。

危险信号4:滥用Discord的行为

一个令人担忧的情况是，当社区成员提出合理的问题时，NFT团队选择禁止了他的Discord账户。你希望看到项目是包容和直率的，而不是好斗和隐秘的。团队没有充分的理由滥用他们的审核职责。

危险信号5:没有审计

安全公司的审计报告并不完美，但它们确实提供了一些基本的安全检查。如果你在 Web3 中，你希望使用的是经过审计的 DeFi 和 NFT 项目，因为未经审计的基础设施更容易被攻击或利用。问题是每个审计公司最近都被需求淹没了，一些新的 NFT 项目不想永远等待，因此通过启动未经审计的基础设施来抓住机会。而我们，应相应地谨慎对待此类项目。

危险信号6: 糟糕的设计

一个设计不佳的项目可能会导致漏洞、低效的gas、可利用的mint等等。对于新手和大量非技术专家来说，衡量糟糕的代码等是很困难的，所以跟随那些懂行的人，并从他们的判断中学习会是关键。

危险信号7:低质量艺术

如果你遇到一个艺术项目，它看起来像是由Fiverr内容平台在一个小时内构思出来并完成的，那么它大概率就是这样的东西。所以，如果NFT团队本身没有认真对待他们的内容，那你为何要认真对待呢？请记住，有品位的/迷人的/滑稽的/怪异的极简主义作品和庸俗的垃圾作品之间存在巨大的鸿沟。你可以做出判断，但根据我的经验，在这个范围内的项目，往往是糟糕的。

危险信号8:出售白名单

我最近看到的一个新事物是，NFT 团队在出售他们的NFT铸造白名单，哇！这只是一种攫取现金的行为，简单明了，我个人看不出它有什么正当性。

危险信号9:过于乐观

新 NFT 收藏品的创造者是否说过他们的作品是“蓝筹质量”，并会成为下一个无聊猿俱乐部（Bored Ape Yacht Club）？该项目的路线图是否隐喻会奔向月球，或者是声称了一些古怪的东西，比如声称它将成为第一个真正的月球NFT市场？要避免纯粹的花言巧语。

危险信号10: 完全匿名的团队

拥有一个完全匿名的团队本身并没有问题，人们有隐私权，在许多情况下，项目和代码库等的质量将不言而喻。然而，在最坏的情况下，完全匿名的团队可能会造成严重的损害，比如破坏流动性，然后几乎没有责任。

危险信号11: 缺乏业绩记录

如果一个NFT项目团队缺乏经验丰富的NFT老兵，那么这个项目更有可能被淘汰。

危险信号12: 没有通过Etherscan验证

通过Etherscan验证你的代码，可以向公众确保它在以太坊上正常工作。而使用未经验证的智能合约，意味着你没有基本的履约担保，所以这是你在NFT活动过程中一定要注意的事情。例如，你可以看到Cool Pets智能合约是如何通过其选中状态进行验证的。一定要小心那些没有类似勾选标记的项目。

危险信号13: 糟糕的所有权分布

假设有一个总量1万的PFP头像项目引起了你的注意，但当你看到它的OpenSea集合页面，并注意到该项目的所有权分布在几百个地址当中时，那你一定要小心，这代表这些人可以对这个收藏品市场产生重大影响。

结论

上面描述的一些危险信号影响会很大，例如滥用Discord的行为。而其他一些信号，例如完全匿名的团队，并不总是重要的，但至少要引起注意。养成发现这些危险信号的习惯，这样你就能更精明地进行你的NFT活动。
