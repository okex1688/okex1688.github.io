今年夏天，NFT (非同质化代币) 的价格飙升，加密货币爱好者在从数字交易卡到虚拟财产等所有领域都投入了巨额资金。

现在，他们又将「数字石头」添加到了投资组合中。

NFTs 的批评者将今年的 NFT 市场繁荣称为一时流行的狂热，或是一个即将破裂的短暂泡沫。最近突然火爆的 NFT 收藏品 EtherRocks 同样也是如此，但这些 NFT 持有者们正在赚很多钱。

EtherRock 在以太坊区块链上创建，是一组以「石头」为主题的 NFT 收藏品，限量 100 个 。每个 EtherRock 石头图像除了颜色不同，它们_在设计图样上都是相同的_，灵感来自于 1975 年末流行的经典 Pet Rock (宠物石) 玩具。然而，它们不是刚推出不久的，也不是为了迎合今年的 NFT 市场狂热而设计的：EtherRock 早在 2017 年底便已经推出 。

然而，当时这些「石头」并不受欢迎，而且这些 NFTs 中的大多数多年来一直无人认领。但这个被忽视的早期 NFT 项目_最近被重新发现_，随着以太坊用户铸造了剩余的「石头」，这些 NFTs 从上周开始获得很大关注。甚至连 Coinbase 投资者、社交媒体影响者和 NFT 企业家 Gary Vaynerchuk 也在推特上发布了这一消息，同时讨论了其作为早期 NFT 项目具有的_增值效益_。

所有这 100 块「石头」都已售出，现在这些 NFTs 的_二手价格_正在飞涨 。上周末，两枚 EtherRocks NFT 的售价都超过 10 万美元，其中编号为 #21 的 EtherRock 在上周日的售价达到了 45 ETH (约合 135,240 美元)，见下图：

EtherRocks 的价格在上周末翻了一倍，而本周一又翻了一倍 ！根据跟踪 EtherRocks 价格的推特账户 @EtherRockPrice 的数据，截止撰文时，目前市场上最便宜的一块 EtherRock 石头已经达到了 78 ETH，约合 246,722 美元！ 有些石头现在的售价甚至已经达到了 99 ETH (撰文时约合 32 万美元)。而就在几天前，这些石头的售价仅约为 5 万美元。

NFT 就像是代表了数字产品的所有权的契约，这些数字产品可以是一个视频片段，一幅数字绘画，或者是一张石头的图像等等。虽然一些 NFTs 可能具有额外的功能性用途，比如作为交互式电子游戏中的游戏物品，但 EtherRocks 并不具有这些额外用途。EtherRock 官网写道：

“这些虚拟石头除了能够用于买卖，并让你因拥有了仅有的 100 块石头中的 1 块而产生一种强烈的自豪感之外，没有其他用途 。”

一位自称持有 EtherRock NFT 的人发推文声称，上周末他用了 15 ETH (当时价值 46,300 美元) 购买了编号为 #74 的一块 EtherRock「石头」(见下图)，并阐述为何他认为这将是他在加密领域中做出的“最好的投资之一”。他写道：

“随着我们进入数字收藏品的时代，(EtherRock) 作为最早的 (数字收藏品) 艺术形式之一为它们带来了价值来源。也有怀旧的一面，收藏者们会寻找具有历史意义的早期作品。” 

尽管 NFTs 已经存在多年，但在今年早些时候疫情期间，以及数字艺术家 Beeple 在佳士得拍卖行以 6930 万美元的价格拍出一幅 NFT 拼贴画之后，NFT 突然大受欢迎。

对 EtherRocks 需求的突然上升正值 NFT 市场在夏末出现激增之际。2021 年上半年，NFT 市场产生了 25 亿美元的交易量，尽管在今年春末夏初，围绕 NFT 的炒作似乎消退了。然而，最近，NFT 交易量飙升，其高昂的售价再次成为头条新闻。

如前所述，EtherRocks 自 2017 年以来一直存在，并与 CryptoPunks (以太坊区块链上最早的 NFTs 之一) 差不多同时创建。

CryptoPunks (加密朋克) 是今年 NFT 热潮的最大受益者之一，该系列的 NFT 像素头像价值大幅上涨。上周，CryptoPunks 的市场最低价首次突破了价值 100,000 美元 (39 ETH)，且根据其背后团队 Larva Labs 的官方追踪器，截至撰文时，当前 CryptoPunks 最低售价为 47.7 ETH (约合 15.34 万美元) 。见下图：

截图：https://www.larvalabs.com/cryptopunks/forsale

最近几周，CryptoPunk 的销售价格也达到了**数百万美元** ，其中包括在 7 月底，编号为 #5217 的 CryptoPunk NFT 头像以 2,250 ETH (当时价值约 540 万美元) 的价格被售出 (见下图)；几乎同时，Gary Vaynerchuk 以 1,600 ETH (当时价值约 370 万美元) 的价格购买购买了编号为 #2140 的 CryptoPunk NFT 头像。上周，一个在 2018 年仅以 443 美元的价格购买到的 CryptoPunk NFT 头像以 1,500 ETH (当时价值约 438 万美元) 的价格被售出。

NFTs 可以在不同的第三方交易平台购买，其中最流行的平台是 OpenSea.io。至于 EtherRocks，只能在该项目的网站上购买。

据 CryptoBriefing 报道，这 100 个 EtherRocks 中的**一些已经丢失** ，数量有限导致了这些「石头」的价格飙升。收藏家们特别热衷于不同颜色的「石头」，包括四种罕见的蓝色石头，与其他大多数带有灰色阴影的石头形成鲜明对比。

一些 EtherRocks 的买家们认为，他们购买这些「石头」是因为 EtherRocks 的价值较之 CryptoPunks 有过之而无不及。

比如 EtherRock 持有者 LilMoonLambo 表示，CryptoPunks 的 NFT 头像目前有数千个，与 CryptoPunks 相比，这些 EtherRocks「石头」更有价值，因为它们的**稀缺性更强** 。他表示：“我们生活在模因 (meme) 经济中，现在这些石头的表现超过了黄金。在通货膨胀的世界里，数字稀缺性是一种趋势。”

LilMoonLambo 目前准备将自己持有的一块 EtherRock「石头」以 500 ETH (撰文时价值超 160 万美元) 的价格售出，他表示：“在加密艺术领域，500 ETH 只是九牛一毛。”
