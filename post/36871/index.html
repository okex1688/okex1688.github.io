原文标题：《Berkeley Blockchain Xcelerator: 2022 Cohort Announcement and Ecosystem News》

导读：近日，加州大学伯克利分校区块链加速器公布 2022 年批次孵化项目，Anorra、Auth3 Network 等 20 个项目从 300 份申请中入围。

此前，Blockchain Xcelerator 在前四批活动中孵化 65 个团队，后续融资超过 4 亿美元，其中包括 Acala、Automata、Astar、Manta Network 等知名项目，是美国一流的高校区块链加速器。

以下是本批次项目的具体名单：

AlfredPay

AlfredPay 是一个拉丁美洲的汇款平台，旨在为用户提供一种方便且经济实惠的汇款和收款方式。我们与受监管的金融机构和现金网络合作，提供即时获取当地货币或美元的功能，以换取 USDC。

Anifie, Inc.

后疫情时代的数字化趋势迫使音乐和时尚品牌寻找新的盈利方式。Anifie 使用我们的专有软件帮助音乐和时尚品牌创建、销售和提升 NFT 的价值。我们能使不同品牌的内容 IP，以 NFT 的形势出售。同步 NFT 的销售，我们的客户在我们的元宇宙中举办活动，从而使其社区更强大，进一步利好 NFT。

Anorra

Anorra 是一款建立在 Solana 上，使用虚幻引擎打造的，基于浏览器的 Free-to-Play 移动端 MMO 游戏。Anorra 为用户提供了方便、有趣且简化的参与 DeFi 的方式。我们的目标是通过史诗级游戏或 GameFi 扩大 DeFi 在不同人群中的应用。我们特别致力于打破这一领域的障碍并简化 Web3 的复杂性，为所有人提供可访问的、用户友好型服务。

Auth3 Network

凭借 Oasis Network 独有的隐私保护区块链计算技术，我们构建了第一个可以将数据价值从 Web 2.0 流式传输到 Web 3.0 以及元宇宙的「桥梁」。它为当前的 Web 3.0 带来了一个扩展范式。这个桥的核心是一个名为 MBuddy 的由用户所控制的 Web 3.0+身份。

用户可以独立控制自己的数据资产，并使用它们来服务于自己的利益，这些利益可能是货币、社交或其他方面的需求。例如，DAO 可能要求其成员具有某些个人特征。尽管这些需求目前还无法实现，但 MBuddy 可以作为一个解决方案来实现。拉取所需数据后，用户可以发起一个保密计算作业，为其链上身份生成一个经过身份验证的个人特征，并获得 DAO 成员角色。

Avatree

Avatree 是「Headspace for Climate」，其使命是大规模去除大气中的碳含量。我们销售 3D 树木版的 NFT，当用户订阅月购买碳封存时，这些树木就会根据算法生长 (我们与碳市场合作，以获得高质量的碳封存)。

每个月，树上的「果实」加密货币可以用于品牌奖励。Avatree 是互联网上气候行动的一个标志，同时也授予所有者进入 Avatribe 的权利，他们可以在我们的社区门户享受独家体验，并可以与热衷于创造可持续未来的其他人联系。

Avem

Avem 是建立在 Polkadot 之上的跨链 DeFi 基础设施，作为 Layer1 平行链，支持证券和实用型 Token 的交易和托管。我们正计划建立一个不可追踪型区块链，因为大型机构喜欢在不受社区其他人监视的情况下转移资金。我们的区块链将使用 KYC 和 KYB（零知识证明）进行监管，以促进机构和散户之间的资金混合。

Bitgreen

Bitgreen 旨在为与保护、清洁能源和社会企业相关的现实世界资产和有影响的项目提供资金。投资者将能够确定优质项目以及辛迪加，并投资于加密货币，以在原生 CEX 上持有或交易。

Campground

Campground 是用于内容创作、策划和货币化的现代媒体基础设施。我们为创作者开发了一种创新工具，用于构建名为 Trails 的多媒体内容导览，并附带社区音频空间，然后将他们的 Trails 作为 NFT 访问 Token 进行铸造和销售，以有效地利用内容价值获利并吸引其受众。Campground 由 Emoji 协议和 EMOJIToken 提供支持，这是一个底层 Token 系统，可将「情绪预测市场」中的反应游戏化，并通过 Expert（专家）审查网络对内容质量进行去中心化验证。Emoji 旨在最终扩展为一个可互操作的层，并作为一个强大的用户参与杠杆接入到所有内容/媒体平台。

Curio

Curio 是一款链上 RTS 游戏。支持玩家在丰富的 2D 世界中收获资源、建立基地，以进行防御、指挥军队作战并夺取对方的塔楼。其类似于 Sid Meier 的《Civilization》和 Markus Persson 的 MiniCraft。与专注于链上资产的加密游戏不同，Curio 将所有游戏逻辑和状态变化存储在区块链上，取代了传统的游戏服务器。因此，游戏实例不仅完全透明，而且对「modding（改装）」开放，任何玩家都可以通过智能合约与核心游戏逻辑进行交互。

传统游戏往往开发周期长、玩家参与度低并且会出现玩家作弊状况。在 Curio，我们利用区块链作为天然的合法性层来推广用户生成的内容。我们的目标是将游戏的围墙乐园转变为一个开放的世界，任何人都可以参与其中并获得公平的贡献奖励。我们将其视为 Steam 模组、Minecraft 和 Roblox 等「赋予玩家权力」实验的悠久历史的延续，在这些实验中，玩家将更多地参与塑造他们所重视的虚拟世界。

Dtravel

Dtravel 是一个原生 web3 旅游生态系统，为房东和酒店企业家提供接受链上预订的基础设施。其融合了直接网站预订的自由和在线旅行社的方便，为用户的提供整个预订过程无比的所有权和控制权。

Dtravel 支持房主和客人直接进行交易，而不是依靠通常收取高额费用并制定禁止政策的中介机构来促进交易。这种点对点系统使房东能够设置自己的预订条款，自由决定他们展示的内容，并存留大部分利润。

Fairmint

Fairmint 使社区所有权变得更加容易获取。创始人在他们的网站上启动了 Fairmint 的去中心化门户，允许贡献者获得 Token 化股权，以交换其投资的时间或金钱。我们的技术由创始人为创始人打造，利用区块链和智能合约将股权可编程化。Fairmint 是现代初创企业堆栈的 Web3 层，增强所有制经济的参与度。

MetaEngine

MetaEngine 为元宇宙之间的资产互操作性提供了一个优质平台。该平台上超过 3000 万活跃玩家使其成为独一无二的解决方案，其为游戏开发者社区提供了一套完整的、功能强大的、通过区块链开发的最先进的功能。

Moon

Moon 是一家加密行业的挑战者银行和 BaaS（区块链即服务）平台，特别适用于那些比特币持有者、加密原住民和无银行/资金不足的以加密为生的人。Moon 的隐私保护技术使其能够在不需要 KYC 的情况下提供类似银行的服务，维护比特币和更广泛的加密社区的核心价值观，并为全球数十亿被排除在现代金融之外的无银行/资金不足的人提供服务系统。

Muusic

Muusic 支持任何人都可以拥有和投资自己喜欢的音乐。我们正在通过处理从元数据标准化到多方版税支付的去信任化基础设施等一切事务，以创建一个开放的数字唱片市场来改变音乐的估值方式。

基于 Muusic，任何人都可以创建自己的唱片公司，投资自己喜欢的艺术家。我们正在颠覆音乐产业，它的运作依赖于过多的法律费用和中间商之间的协调，这让艺术家们损失了他们绝大部分的收入。通过构建所需的基础设施、工具和技术，艺术家们的整个职业生涯都可以在链上发展。

NilDao

即将推出的 Nil DAO V2 是一个运行在 Arbitrum（L2）上的 NFT 市场，并为生成型艺术家提供了一个集成的 launchpad。基于具有新的功能和优质 UI/UX 的 Nil V2，任何人只需支付一小部分以太坊 L1 的 gas 费，就都可以展出和收集艺术作品。其 launchpad 将帮助 NFT 无经验创造者自主在 NFT 空间展示其艺术作品。

OAK Network

OAK（链上自治内核）网络是用于 Defi 和定期支付的自动化区块链。我们的使命是构建跨协议的 Web3 Stripe 支付平台。区块链建立在具有事件驱动执行模型的 Parity Substrate 之上。与以太坊基于交易的模型（交易需要私钥签名）不同，OAK 允许交易由时间、价格和智能合约状态变化等事件信号触发。

OAK Network 旨在成为 Layer2 跨协议的首要实用程序，提供目前无法实现的链上自动化，例如自动做市 DEX 上的限价和止损订单以及定期付款功能。凭借 OAK 的链上最终性和网络安全性，它可以赋予任何区块链自主功能，释放产品的无限潜力。

Overnight

Overnight 是支持 USD+收益生成 Stablecoin 的协议。它完全由 DeFi 产生收益的 Stablecoin 资产支持——最适合用于个人和国库的 Stablecoin 流动性管理、流动性池中的资本效率以及作为杠杆收益流动性挖矿的抵押品。

Pendulum

Pendulum 的愿景是通过法币优化的智能合约区块链在法币和 DeFi 之间建立连接。其支持外汇、贸易融资或储蓄账户等传统金融服务与 DeFi 应用程序无缝集成，例如专门的外汇 AMM、借贷协议或收益耕作机会。通过这种连接，Pendulum 为 DeFi 生态系统带来强大流动性，并反过来为传统金融服务带来效率提升、互操作性最大化和全自动 24/7 运营的优势。

Pendulum 基于 Substrate 框架，旨在成为 Polkadot 平行链，以实现最大化安全性和互操作性。定制型的去信任化跨链桥将把它连接到其他主要的 DeFi 或支付生态系统中，如以太坊、Cosmos 和 Stellar。

Rampp

Rampp 创建了创建者或社区管理者在不编写任何代码的情况下启动和扩展 NFT 项目所需的所有无代码工具。从 IPFS 到合约，再到铸币等等，Rampp 都能提供相应服务。随着 Web3 的发展，Rampp 将始终提供无代码解决方案，为那些希望在 Web3 中探索、试验和构建的人们打破技术和成本的屏障。

ViteX

ViteX 是一个高性能的订单簿 DEX，在用户体验方面与中心化交易平台相当，但在安全性方面更强。其 DEX 支持由社区运营的运营商驱动的 60 多种 Token 的跨链交易（以无许可方式上币），很快将升级为由社区管理的去信任化桥。ViteX 建立在高速 Vite 链上，其交易无需 gas 费。

使用 ViteX 的交易者可获得 VXToken 奖励，这使质押者有权从 DEX 的收入池中去分配。VXToken 没有预挖矿，没有私募/预/公售，只能通过包括交易和流动性挖矿在内的挖矿方式获得。我们的下一个里程碑包括从网关到跨链桥的转换，通过各种赞助项目持续吸引社区运营商，开发 AMM 以提高流动性，以及发展整个 Vite 生态系统以吸引更多用户。
