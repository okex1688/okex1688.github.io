美国运通 (American Express) 所推出的黑卡

现实中的黑卡，通常是指美国运通公司于 1999 年起发行的最顶级黑金信用卡百夫长卡 (Centurion Card)。它定位于顶级群体，无额度上限，持卡人多为各国政要、亿万富豪及社会名流，并由美国运通邀请办理，不接受办卡申请。关于黑卡的传说很多，如让飞机掉头、火车停驶等等，但大多并未得到过证实，这也是发卡组织对客户的隐私保密。但可以确定的是，持卡人可以享受全球最顶级的会员专属礼遇、权益和服务，这就是为什么它能够受到权贵们追捧的最大原因。

从现实转向 NFT 的世界，购买某个社区发行 PASS 卡，持卡人享受社区带来的权益早已不是什么新鲜事。但各个社区为持卡人带来的权益参差不齐，NFT 玩家们渴望着顶级权益和优质讯息来让自己的投资效率最大化，但又苦于 NFT PASS 这条赛道上蓝筹项目的稀缺而观望。

事实上，结合本月的 Moonbirds、0xOG PASS、Premint Collector Pass 发售，我们再仔细观察 NFT PASS 这条赛道时，发现它其实早已暗流涌动。

Moonbirds：「巨鲸俱乐部」的扩张

Moonbirds 头像

自 4 月 16 日发售，短短一周时间，Moonbirds 这个以像素卡通猫头鹰为形象的项目的在 OpenSea 上的交易量已经超过了 110k ETH，它的地板价也从 2.5 ETH 的发售价暴涨至 37 ETH 的高价。Moonbirds NFT 能在这么短时间内取得巨大的成功，要归功于它背后的 PROOF 团队强大的背景及运营能力。

PROOF 背景

在 True Ventures 的官网可以查到 Kevin Rose 的相关信息

PROOF Collective 拥有一个具有丰富经验的团队，团队的创始人Kevin Rose、Ryan Carson 和 Justin Mezzell 都是 Web2 和 Web3 领域的知名人物，他们的信息在 LinkedIn 上都是公开的。

其中 Kevin Rose 是风险投资机构 True Ventures 的合伙人，还主持过很多与科技和 Crypto 相关的播客，他的 PROOF 播客促成了后来的 Proof Collective 俱乐部。Ryan Carson 是三家公司和一个非营利社区的创始人。Justin Mezzell 在加入 PROOF 前是一名自由插画师/设计师，客户包括 Google、Facebook、Twitter、PayPal、NASA、纽约时报等。

Proof Collective 俱乐部是一个由 1000 名 NFT 收藏家组成的团体，其成员包括了圈内知名的艺术家 Beeple（Mike Winkelmann）和投资人 Gary Vaynerchuk 等等。

PROOF Collective 俱乐部会员持有的蓝筹数量十分惊人

而从 PROOF Collective 官网公布的数据来看，俱乐部会员共计拥有超过 15 万个 NFT、148 个 CryptoPunks、817 个 BAYC、499 个 Meebits、5288 个 Art Blocks、326 个 SuperRare 1:1 NFT 作品。这些数字足以证明 PROOF 社区是名副其实的「巨鲸俱乐部」。

Moonbirds 启动了 PROOF Collective NFT 的暴涨（来源：NFTGo.io）

PROOF Collective NFT 是 Kevin Rose 创办的社区通证，持有者可以享有社区内的各项权益。它的地板价自 2 月初以来一直在横盘整理，维持在 30 ETH 左右。3 月 19 日，Moonbirds 的官方 Twitter 发布了它的第一条推文，这也按下了 PROOF Collective NFT 价格暴涨的启动键。在随后一个月的时间里价格一路飞涨，现在的地板价格已经达到了 130 ETH 的天价。

然而，价格暴涨也没有阻止投资者们继续购买 PROOF Collective NFT，因为他们相信它能够带来十分优质的权益，其中就包括 Moonbirds 的白名单认购。

优质权益

首先，在这次 Moonbirds 的发售中，有 2000 个 Moonbirds 是作为 Proof Collective NFT 的持有者的「福利」 ，每名 Proof Collective NFT 的持有者都可以直接认购 2 个，其余另有 7875 个分配给赢得抽奖的人，剩余的 125 个为 Moonbirds 团队预留。

相信是有一部分投资者奔着这个「福利」而来的，他们在参与这些 NFT 的交易后就快速获利出局。 简单算一笔账，假如你在 3 月 19 日以当时 37 ETH 左右的地板价购入一个 PROOF Collective NFT，除去铸造 2 个 Moonbirds 所需的 5 ETH 的成本，目前净盈利为 157 ETH 左右。

还有一部分投资者则是更关注 Moonbirds 的长期权益：

1. Moonbirds 持有者可以进入 PROOF 的私人 Discord 参加封闭式会议，但目前尚不清楚封闭式会议的真实收益以及两个 NFT 之间是否存在访问权限的差异。

2. Moonbirds 的持有者可以通过「筑巢（Nesting）」来获得奖励。这里和传统的「质押」有所不同，只要持有者不转移或卖出 Moonbirds，保持在持有者的钱包中，系统会根据持有时长来分配奖励。

3. Moonbirds 持有者将可以获得 PROOF 接下来将要推出的代号为「Project Highrise」的元宇宙项目的优先访问资格。

做市与操盘

推特账号 NFT Ethics 爆料 PROOF Collective NFT 可能存在「交易清洗」和「裙带关系」

随着 Moonbirds 的发售日期临近，背后团队的强大背景、高昂的发售价都是各个 NFT 社群讨论的热点话题。在它发售的前一天，曾爆料过 HypeBears、CerealClub 等多个 NFT 项目内幕的知名推特账号 NFT Ethics 发布了多条推文，质疑 Moonbirds 背后的团队曾对 PROOF Collective NFT 进行过「交易清洗」：很多钱包只有 PROOF Collective 这一种 NFT，有人在这些钱包之间来回出售 NFT，营造了「虚假繁荣」，进而推高这一系列 NFT 的价格。

NFT Ethics 表示：「交易清洗在 NFT 项目中如此普遍，以至于它甚至不再值得注意。」

事实也确实如此，这次爆料似乎并没有影响到后续 Moonbirds 的发售，公开发售的每个 Moonbirds 都以 2.5 ETH 的价格被认购，随后价格一路飙升。

Kevin Rose 表示有机器人会自动保护地板价

暴涨的背后，是投资者们对 Moonbirds 的价值认同还是 PROOF 团队仍在通过「交易清洗」做市来拉升价格呢？PROOF 的联合创始人 Kevin Rose 的一条推文似乎也回应了这件事：

1. 一旦将手中持有的 Moonbirds 在交易市场上挂单，无论是否成交，都会取消「筑巢（Nesting）」状态。这意味着，如果持有者有售出的倾向，则不能获得「筑巢」带来的奖励。 团队设置了机器人，如果将手中的 Moonbirds 以低于目前地板价的价格挂单，则立即会被机器人自动购买。

2. 团队希望通过这些机制来激励和操盘，将持有者培养成「钻石手」。以低价购入 Moonbirds 的玩家确实可以将它 Flip 后快速获利，但 PROOF 团队更希望他们能成为长期持有者。

PASS+PFP 的「铲子」组合

在 DeFi 领域中，如果要参与流动性挖矿，首先要提供单币如 USDT 或币对 ETH-USDT 的 LP 才能进行挖矿，这里的 USDT 或 LP 就是「铲子」，用户使用它来进行挖矿获得代币奖励。

延伸到 NFT 领域内，如 BAYC 的持有者可以获得 ApeCoin 和土地的空投，那 BAYC 就是「铲子」。通过持有某一系列的 NFT，可以获得相关空投或后续其他系列 NFT 的优先权益，则原本持有的 NFT 就是「铲子」。

PROOF 团队将 PROOF Collective NFT 打造成「铲子」，将 Moonbirds 的优先购买权益作为奖励，通过这种方式扩招「巨鲸俱乐部」的中阶玩家 ，同时 Moonbirds 作为 PFP 可以让原本的持有者更有社区归属感，大大加强了项目之间的联动性 。

0xOG：NFT 的顶级孵化器

打开 0xStudio 官网，看到它的自我介绍是**「以艺术家为先的、服务于 Web3、成长于 Web3 的品牌孵化器」** 。如果你了解 3landers，那你应该对 0xStudio 并不陌生。

直接获得孵化项目的优先权益

0xOG PASS 由艺术家 SeerLight 设计

0xStudio 最开始是作为 3landers 的孵化器而为人们所知，在 4 月 8 日团队官宣将推出 0xOG PASS，它是一个以艺术为中心的会员通行证，允许持有者直接获得 0xStudio 孵化的项目优先权益，0xOG PASS 的持有者在 0xStudio 后续孵化的所有项目的「第一阶段」中都可以直接铸造 1 个 NFT 。

专业的孵化服务

0xStudio 提供的服务

0xStudio 提供的服务包括智能合约定制、端到端 NFT 解决方案、代币经济学设计等，同时它十分重视安全性和透明度，推出的两套智能合约标准「0xStandard」和「Wagyu」已经通过了 Inspex 的审计，后续孵化的项目都将建立在这些经过审计的智能合约上。

1 月 14 日，3landers 官宣其将由 0xStudio 孵化

目前无论是从市场价格还是影响力来说，3landers 已经跻身二线蓝筹，除了本身的艺术价值以外，团队的能力为它打造了很好的口碑。当我们回看 3landers 的整个发售过程，是 0xStudio 提供的服务始终为其保驾护航，从 Twitter 宣发到 Discord 社区的组建，再到最后的官网上线发售，所有的步骤都是有条不紊的。0xStudio 专业的孵化能力是 3landers 成功的必要一环，而 3landers 的成功也为 0xStudio 树立了很好的口碑，他们是相辅相成的。

同样也是「铲子」，与 PROOF 为它的子系列提供「一条龙服务」不同，0xStudio 与它孵化的项目是合作关系而不是隶属关系，这让它推出的社区通证 0xOG PASS 有了更多的兼容性 。

新颖的发售方式

1000 张 0xOG PASS 的分配方式

0xOG PASS 采用一种 Two-Step Drop 的发售方式 ，发售价格为 1.8 ETH，在初始阶段仅流通 280 个（100 个公售，80 个空投给 0xStudio 的早期支持者，100 个保留给未来能与 0xStudio 一起 BUIDL 的 0xAlumni），其余的 720 个将在后续阶段售出，每 6 小时将有一个 0xOG PASS 可供铸造，直至 1000 个 PASS 完全被铸造完。

0xOG PASS 的地板价已升至 10.5 ETH

「我们相信优秀的团队必须随着时间的推移证明自己。」0xStudio 团队希望通过这种新颖的 NFT 发售方式来让投资者自行决定是否愿意随着时间的推移来加入他们，而不是强迫人们现在就为 0xStudio 未来的成功承担风险。「我们不急于销售，而是更关注有机增长。」通过这种方式来让社区缓慢而健康地发展，社区与团队一起成长。

0xStudio 孵化的项目

项目与孵化器之间始终是相辅相成的，前有 3landers 的成功印证了 0xStudio 有强大的孵化能力，投资者们选择购买 0xOG PASS 是因为相信这个团队仍旧能够孵化很多像 3landers 一样优秀的项目，让这把「铲子」不断创造收益。0xStudio 正在孵化另一个优秀的 NFT 项目「LonelyPop」，在这个时间节点选择同步将自己的 0xOG PASS 推出是十分聪明的做法，这更能够激发投资者对 PASS 的购买欲，不断扩大影响力。

Premint：抽奖机器与广告位

关于 Premint 的第一条推文

去年 10 月 16 日，Premint 的创始人 Brenden Mulligan 在他的个人推特上发布了关于 Premint 的第一条推文，「它只有基础功能，但比 Google 文档好得多。」原本产品的定位就是一个 Web3.0 抽奖工具，它解决了当时市场上 NFT 玩家和项目方抽奖/验证不方便的痛点，

从 NFT 玩家的角度来说，Premint 是一个引导用户完成抽奖流程的平台，用户可以在一个页面完成关注项目方 Twitter、进入官方 Discord、填写地址等抽奖步骤。从 NFT 项目方的角度来说，项目方不需要再独自开发抽奖系统，节约了时间成本和技术成本，Premint 可以帮助项目方完成收集地址、验资、自动开奖、引流粉丝获得关注等步骤。

Premint 积累了大量用户

短短半年时间，Premint 已积累了大量用户：超过 2,300 个项目已使用 PREMINT 管理它们的白名单抽奖，各个项目总计注册了超过 180 万个条目，超过 50 万个钱包已连接过 PREMINT 参与抽奖。

Premint 在这个时候，不满足于原有的功能，选择推出了拥有更高端的服务和功能的 PASS 通行证，PASS 分为两类，一种面向 NFT 玩家，一种面向项目方。

优秀的抽奖工具

简约的抽奖界面

面向 NFT 玩家的是 Premint Collector Pass，总量为 1 万个，现在价格已经从 0.25 ETH 的发售价上涨至 2.5 ETH 左右。持有者可以享受以下权益：

1. 追踪热门的项目

2. 会根据持有的 NFT 自动推荐可以参与的抽奖活动

3. 优先参与 Premint 网站新功能的测试

4. 定制个性化日历，可以显示平台上所有项目的注册和铸造时间

5. 订阅收藏家周报，了解最近 NFT 项目的趋势

6. 用交易版税补贴持有者，让持有者以折扣价格购买 NFT

7. 获得独家发售的购买权

自带流量的「广告位」

面向项目方的是 Premint Creator Pass，它更像是一个消耗品 。Premint 平台有免费抽奖的功能，但十分鸡肋，只适合小型抽奖活动。项目方如果需要在 Premint 平台上发布一个新的较为大型的项目抽奖活动，则需要向 Premint 官方购买一个 Premint Creator Pass 并消耗掉它。它的发售价格也随着 NFT 市场的火热而水涨船高，由最初的 0.25 ETH 涨到了 1 ETH，这意味着现在的项目方如果想在 Premint 上发布一次抽奖活动，则需要支付 1 ETH 来作为「广告费」，进而可以享受 Premint 平台提供的更加高级的抽奖功能，例如验证抽奖用户的 Discord 身份、设置抽奖用户钱包余额门槛等等。

潜在的问题

作为一个工具类的项目的 PASS 通证，目前的功能可能对用户的留存能力比较低 。首先，对于 NFT 玩家来说，他们最需要的是平台便捷的抽奖功能，而这个功能是没有壁垒的。去使用新的抽奖平台工具，对 Premint 的用户来说迁移成本很低。其次，对于每一个项目方来说，他们绝大多数要间隔很久才会再次发售一次项目，随着后续竞争者的出现、相关 NFT 项目趋于饱和以及潜在的流量流失，「广告位」为平台带来的收入会持续降低。因此，如何将低门槛的「护城河」建深是一个值得思考的问题。

总结

最近的这几个 NFT PASS 的发售都是十分成功的，它们发挥着自己在资源、讯息、资金、技术、流量等方面的优势，在各自的细分赛道探索着。它们看似没有竞争关系，但又不断争夺着市场内的热度与资金。在这波云诡谲的 NFT 市场里，既有「巨鲸俱乐部」PASS 的向下兼容，也有低门槛的工具 PASS 自下而上野蛮生长，但殊途同归，它们的野心都是想要成为加密世界的「百夫长卡」。
