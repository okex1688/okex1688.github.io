原标题：《Altered State Machine, $ASTO and the Next Great Evolution of NFTs》

导读：3月初，AI 元宇宙平台 Altered State Machine 宣布完成种子轮融资，参投方包括华纳音乐集团、Coinbase Ventures、Delphi Digital、Flamingo DAO、Outlier Ventures 等诸多知名机构。

根据公开资料，这是一个结合 NFT、人工智能和机器学习的创新项目，近期市场热度颇高，本文对该项目的基本特点、运作模式以及参与方式进行了消息介绍。

什么是Altered State Machine (ASM)？

Altered State Machine是一个 NFT 平台，旨在通过引入 AI 和机器学习来推进 NFT 平台的创新，探索更多可能性的元宇宙世界。

Altered State Machine有大量的潜在用例，而且随着元宇宙基础设施在连接性方面的完善，使用场景将继续增长。

Altered State Machine 团队最近在兴奋地宣传和发布 $ASTO 代币，许多聪明人早已提前关注这个项目，该项目还与目前最有价值的 NFT 系列建立了明确合作伙伴关系。

Altered State Machine 的愿景是成为使用 AI 技术实现 NFT 所有权和货币化的协议。

该项目处于技术前沿，理解其中的知识点可能会遇到障碍，目前还处于起步阶段。可能有很多未开发的价值洼地，对于那些愿意承担一些风险的投资者来说，价值可能会逐步凸显。

该项目的某些功能很难理解，因此本文尝试分开解读。

值得一提的是，团队正在申请一项技术专利，虽然有些人可能会认为这是反对 web3 的开源精神，但有时可能有必要建立护城河，尤其像这样的项目需要亮点来吸引注意力，而且专利的作用是小型项目防止“科技巨头”吞并。

Altered State Machine 正在申请使用 NFT 的 AI ML 模型所有权的专利。

什么是 AI 代理？

AI 代理有3个组件：Brains、Memories、Form。

Brains

Brains由 AI 代理的核心属性组成，称为“基因组矩阵”。

基因组矩阵值对应于强度、速度和大小（在游戏中）或风险承受能力和随机性（在金融中）等统计数据。

最重要的是，Brains通过 AI /机器学习 (AI/ML) 训练，可以学习特定目标（游戏、机器人）所需的属性，但可能需要更长的时间，以不同的方式训练，类似于为不同先天能力的人训练成拥有相同技能的专家。

这种结构将会引入虚拟世界某些类型的随机性（从而增强其深度），这类似于基因如何产生具有不同能力的人类。

2021 年 10 月团队在创世纪盒（4个人物和1个Brains）中推出的 AIFA（ASM 团队制作的足球 NFT 游戏），Brains是其中的一部分。

目前可以在Opensea上购买Brains，并在alpha 农场检查稀有度/智商。很多高智商/稀有Brains正在以底价出售（自行研究潜在的价值NFT）。Opensea 上已出售8500 只BrainsNFT，还有 1500 只待售。

第二季度，用户将能够质押 $ASTO 来铸造新Brains（Gen2），质押的越多成功率就越高。目前尚不清楚质押机制将如何运作。

Gen2 Brains将不具备繁殖等功能，也不会接受未来的空投。而 Gen1 Brains拥有这些功能。

Memories

Memories是指 AI 代理在机器学习“健身房”中完成的“训练”策略。

健身房将由 GPU 云计算提供商（类似于矿工）支持，为每个代理提供行为策略，费用以 $ASTO 代币支付。

每个代理的培训结果都是独一无二的，每次完成训练就会创建一个输出并铸造为单独的 NFT ，然后标记到Brains上，实际上是“升级”技能、能力等。

健身房主人通常是该 NFT 系列创造者，而健身房主机是该元宇宙生态系统。

比如用户可以在Passage 元宇宙的 Strange Clan NFT系列中训练 NFT 角色，以提高 NFT 的属性，从而在战斗中表现得更好。

整个元宇宙生态的 NFT 持有者将激励健身房的主人来帮助改进这些 AI 代理。

Form

这主要是指视觉外观。这一点将根据代理的功能而有所不同。例如，如果代理是一个交易机器人，它不会训练其皮肤，但如果它是一个格斗游戏，那么它们的大小或外观可能很重要。

已经存在的传统 NFT（例如 Punks、Bored Apes 或 Azukis）可以视为“Form”。

以上的三个 NFT 元素将组成一个功能齐全的“AI 代理”。

如上所述，“Memories”是单独铸造的 NFT ，因此一个代理可以有许多Memories NFT，而只有1个Brains和1个Form NFT。

所有代理都需要Brains，并且Brains会交叉用于不同的项目（游戏、元宇宙、DeFi ），但有些代理可能更适合 DApp ，而不是踢足球或者做数学方程式）。

Memories和Form将与类似项目有一些交叉，例如同一虚拟世界中的两款类似游戏，或者不同虚拟世界中的类似游戏。随着越来越多的项目使用这些技术，ASM 也会得到飞速发展。

验证概念的首款游戏 AIFA（ AI 足球协会）

AIFA是第一个使用由 ASM 创建的 NFT 代理的游戏。Form是Brains拆箱过程的一部分。每个盲盒有1个Brains和4个球员。

该游戏尚未上线，但本质上将是一场足球游戏，用户将在与其他四人足球队竞技。球员Form的各个方面都可以修改，训练将提高 NFT 属性，后续将引入各种其他变量。

Discord 有一个深入讨论游戏的社区频道，包括团队的说明和关于游戏将如何运作的提示。

$ASTO 代币

为 ASM 生态系统提供动力的代币称为 $ASTO，代币总供应量为 2,384,000,000。

代币功能：

  • 治理

  • 为代理购买训练时间

  • 铸造（通过质押）新代理/新Brains

  • P2E

  • 流动性挖矿/激励

  • 游戏/生态系统内的交易（充能、升级物品和皮肤）

  • DAO 将包含的进一步实用程序

请查看白皮书的代币经济学部分以获取更多有关循环供应和通货膨胀的详细信息，下面是 Brian 对于代币的解读。

ASM 还发布了一个常见问题FAQ，涵盖了有关代币的详细信息。

如何获得 $ASTO

  • 拥有一个创世Brains并领取空投。用户可以在Brain Claim Check上检查 Opensea 上出售的 Brains，以查看它们是否已经领取空投。

  • AIFA 角色未来将有可能赚取 $ASTO。但是目前还不清楚细节，用户可以通过 Opensea 购买 AIFA 的 NFT。

  • 参加格林威治标准时间 6 月 4 日 23:00 的Discovery 拍卖。ASM 已就此次拍卖发布了一个FAQ解答。简单来说，discovery 池中的 $ASTO 被锁定，直到拍卖结束只能往里添加 USDC。代币将被锁定 12 周，期间用户会获得 $ASTO 奖励。

  • 拍卖后购买。discovery 池将转换为 uniswap 池 ASTO-USDC。

为什么这一切令人兴奋？

我认为 NFT 将被用于多种用途，并在未来十年会得到飞速发展，生活里的一切都将被数字化。当前最有价值的 NFT 是头像（PFP）头像系列，它有效地结合了艺术和社会来创建强大的社区，例如 Bored Ape Yacht Club、CryptoPunks 和最近火热的 Azuki。

这些的 NFT 系列很可能会在历史上留下其成功的事例，但鉴于 BAYC 对 Punks 的“翻转”，很明显，先行者并不意味着能永远保持第一。BAYC 的成功可能是因为将其 Apes 的 IP 提供给所有者，当扩张到其他领域时，团队意识到在数字 web3 世界中取得成功是第一位的，因此最近宣布了元宇宙。

我认为未来将有更多此类项目允许其 NFT 的所有者能够对其进行进化和进一步开发，同时还允许所有者保留其原始Form。

目前团队公开的最大合作伙伴是fluf.world，还包括正在构建的元宇宙Flufs、Party Bears、Seekers 和Thingies。

通过推文、白皮书和 Discord 信息，一些特定的 NFT 系列，例如 BAYC、Punks 和 Meebits 也正在考虑使用 ASM 功能来使其 NFT（“Form”）成为 AI 代理（添加Brains和“Memories’），然后进化并衍生出更多的东西。

从本质上讲，我认为这个项目是一个 NFT 基础设施。将其与团队正在申请的专利相结合，无论哪个 NFT PFP 系列占据主导地位，或者哪个 web3 游戏脱颖而出，都可能成为这个领域最重要的中坚力量。对于任何人来说，为 NFT 引入 AI 机器学习是一件轻而易举的事，而且 ASM 团队目前在这方面做得最好。

Delphi Digital 在加密领域以擅长把握趋势而闻名，之前投资了包括在 AXS 在内的千倍代币，并参与 Axie Infinity 宣传和帮助其改进其代币经济学，目前 Delphi Digital 也在讨论和推广该项目，并且部分个人进行了投资。

另外，团队最新一轮的融资不仅来自 Coinbase 和其他 web3 基金等，还有传统 VC 和华纳音乐等媒体公司。

如何参与ASM？

从广义上讲，目前只有三种方法参与 ASM：

  • 购买Brains

  • 购买 AIFA 角色

  • 购买 $ASTO（通过正在进行的拍卖锁定 $ASTO 并获得收益，或者 6 月 4 日后在 uniswap 购买）

在 $ASTO 空投之后，Brains价格从 6-7 ETH 的高位下降到 3 ETH 左右（对于那些领取了空投的NFT）。

我认为 Gen1 Brains目前是“最安全”的价位，未来会随着生态系统的实用性加强，需求量会加大，并且具有超越 Gen2 Brains的显著优势，是受到团队重视的第一个 ASM NFT 。

AIFA 角色底价在 0.14 ETH 左右，相比Brains要便宜得多。空投领取情况目前尚不清楚。我认为它们的长期价值不会太稳定，更有可能在短期内表现良好，尤其是在 AIFA 推出完整版本之后。

发行时 $ASTO 代币有 5500 万的流通供应量。目前拍卖价格为 0.36 USDC（预计结束前几分钟价格会出现一些波动）。考虑到项目是这个领域的先行者，拥有技术护城河，ASM 将提供成为许多 NFT 的基础设施，加上团队和信念，我认为是这是一个严重低估的价格。
