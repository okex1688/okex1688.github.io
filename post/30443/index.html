原文：Messari

虽然 Solana 最初的梦想是在 DeFi 领域与以太坊竞争，但它已经开始将其可扩展的触角延伸到其他以太坊主导的领域，特别是 NFT。

随着 Solana 生态系统的不断扩展，Solana 上的 NFT 作为一个类别同样见证了惊人的增长。Solana 上的 NFT 二级总销售额已正式达到 5 亿美元。

Solana 的 NFT 累积销售额正在迅速接近 Flow，如果现有趋势持续下去，其销量可能会超过Flow生态最大的NFT项目NBA Top Shot。

属于Solana的九月

在启动NFT 交易的第二个月，Solana上的NFT项目数量和各自的交易量都出现了相当大的增长。Solana九月的总成交量达到近 1.9 亿美元，主要靠 Degen Ape Academy、Aurory 和 Galactic Gecko Space Garage 的支持。

随着九月之后热度消退和10月转入平淡，Solana上NFT的销售额出现了小幅下降，然而整体二级成交量仍然相对较高，成交量略高于 1.75 亿美元。

作为 Solana 上最早流行的 NFT 之一，Degen Ape Academy 受益于作为先行者和猿作为资产类别（例如 BAYC）的流行。然而，Degen Apes 的最大销售额是从它的第一个月开始，所以值得关注的是，这个备受喜爱的 Solana pfp（个人资料图片）NFT 能否保持其在Solana上的领先地位。

像以太坊一样，少数项目拥有大部分的二级交易量，进一步表明即使在 Solana 上，NFT 项目的最终命运也将类似于长尾分布。

以太坊 NFT 项目有多种类型，包括游戏、交易卡、pfp、虚拟土地和其他（例如 Loot）。然而，Solana NFT 项目主要由 pfp 风格的项目组成，这可以说是市场上最具投机性的 NFT 类型。值得注意的是，各种游戏项目都已经筹集了资金，打算在 Solana 上进行建设，例如 Aurory，它是最早进行 NFT 销售的公司之一。如果 Solana 希望真正争夺 NFT 的主导地位，它必须支持可收藏（和可爱的）pfp NFT 之外的其他类型的项目。

旭日东升：Solana NFT

Solana 已经开始在几个 NFT 指标上与以太坊竞争。

有趣的是，10 月份以太坊 NFT 的平均销售价格为 3,147 美元，而同月 Solana NFT 的平均销售价格为 1,081 美元。Solana 较低的交易成本可能有助于 pfp 翻转市场，该市场引起了 ETH 和 SOL 投机者的注意。

此外，以太坊的 NFT 生态系统有 132,879 个独立买家和 101,920 个独立卖家，而 Solana 拥有以太坊独立买家数量的 35% 和超过 50% 的独立卖家。这些数字是 Solana 用户在更主流市场 (NFT) 中采用的积极迹象。

值得注意的是，以太坊在 NFT 二级销售总量上仍然让 Solana 相形见绌，在过去三个月中的每一个月都记录了一个数量级的交易量。

然而，交易量并不是值得关注的规范指标。相反，这些指标和市场背景的结合对于确定生态系统的 NFT 市场的实力很重要。因此，尽管以太坊今天仍然是 NFT 市场的主导者，但 Solana 已经清楚地为 pfp / avatar NFT 项目开辟了自己的位置。Solana 能否将这种初始能量体现为长期 NFT 增长尚待确定，但目前的迹象表明，有充分的理由对 Solana 蓬勃发展的 NFT 生态系统持乐观态度。
