原文标题：《DAO Governance Contributor Toolkit》  

DAO 治理可能是可怕的。我们都知道的。我们的一些建议得到了鼓掌，而一些则遭到了激烈的反对，但有一点是肯定的，那就是没有比 DAO 治理的深度更好的东西可以直接影响我们领域前进的方向。

所以，亲爱的读者，如果你对于是否要跳进去为 DAO 治理做出贡献还在犹豫不决，那么让我们以热烈的态度鼓励你：「LFG!」

这篇文章将介绍这个行业的工具，以及如何用好这些工具。作为一个专业的治理小组，我们参与了十多个 DAO。治理贡献是我们的日常。这篇文章代表了我们的最佳实践，而目的就是为新的贡献者提供一个起点，并为有经验的专家提供了讨论的地方。让我们从头开始吧。如果你已经有了在 DAO 的经验，请跳过第一章。

什么是 DAO 治理？

DAO 治理是引导一个去中心化自治组织的过程。由于其去中心化的性质，DAO 不应该有中央领导（至少在理论上是这样的）。在实践中，创始人和最活跃的成员决定了议程的大部分内容，从而决定了所需要做的事情。尤其在早期启动阶段，创始人将会负责做所有的事情。

随着 DAO 的发展，越来越多的贡献者承担起了任务，并开始塑造整整体的运作。DAO 比起传统公司更具流动性，并且影响力更容易获得，但也容易丧失。与传统组织的一个主要区别就是，我们不能强迫其他成员或子单位执行某些任务。

DAO 的检查和平衡机制与传统组织不同。除了在链上组成 DAO 的智能合约外，它并没有任何负责指挥的结构。没有可以命令他人的高管。相反，优秀且论证充分的建议能说服其他人采取行动。

DAO 治理希望创造和支持一个繁荣的网络，以完成其使命。网络是如何实现其目标，以及其治理是如何运作的，在各个 DAO 之间是有很大不同的。如果你将 SushiSwap 与 MakerDAO 进行比较，你可以看到两种极其不同的运作模式，但它们却是两个非常成功的项目。

你应该参与 DAO 治理吗？

由于我们是在 DAO 治理中工作的，那么我们只能强调说：“应该！"。DAO 治理是一个令人兴奋、多样化、具有影响力的职业，它允许贡献者扩大他们的网络，处于发展的前沿，并为自己扬名。

然而，实际的活动和治理的整体速度并不适合所有人。那么更好的问题应该是：“谁应该参与 DAO 治理？”

在为 Flipside 招募贡献者时，我们寻找这些品质：

强大的沟通能力，包括书面和口头沟通

能迅速掌握情况，有能力在压力和极大不确定性下采取行动

快速的学习者，对生态系统有深刻的理解

有一些历史、哲学或行为科学的背景

关于治理，特别是在后期阶段的 DAO 中，涵盖了巨大的主题范围，贡献者有时需要对每个问题进行投票。虽然 “弃权 “是一种有效的投票，但如果你有自己的意见可以表达，那就更好了。重要的是要能够迅速掌握一个问题，然后进行必要的研究并得出结论。良好的沟通技巧需要的是清楚你到底知道什么和不知道什么的真诚心态。这样，你的意见就会在准确的情况下被理解。

很多治理是关于关系的，而人类与其他人类的关系是基于信任和尊重。一些贡献者使用匿名账户，并发表了伟大的作品，而另一些人则以他们的真实姓名发表。假设你想保持匿名，很好。但请不要以你的匿名身份，用冷嘲热讽和仇恨来打压大家。这在 Web3 中是一个负面因素，不要成为那个人。

良好的、透明的、揭示自身优势和劣势的沟通有助于他人建立信任，并提供大量的对话接触点。我们怎么强调这种品质的重要性都不为过。诚实、清晰和善意的沟通的概念很简单，但很难维持，特别是在面对逆境或批评时。理想的贡献者是 “基于 “某样事情的，这意味着她或他有坚实的自我价值感，不会轻易感到被攻击。

工具推荐

我们将介绍我们日常使用的工具：用来沟通我们的想法，找到最好的话题来切入探讨，以及我们如何制作吸引人的帖子来确保我们具有最大的影响力。

沟通工具——如何表达你的观点

贡献者的两个主要工具是 Discord 和 Discourse。Discord 是 DAO 群组闲聊的事实标准，而 Discourse 会是 DAO 讨论正事的地方。DAO 将他们的 Discourse 网站称为 “论坛”，这个词的灵感来自于古希腊的（agora），翻译为论坛。agora 是贵族们聚集，讨论他们国家命运的地方。

Discord 是非正式闲聊的地方。它是了解 DAO 整体氛围的地方，也是交朋友和用以建立联系最简单的方式。讨论区是提出、剖析、讨论、修正提案，然后付诸表决的地方。我们将在后面详细介绍投票的情况。我只想说，实际的投票在大部分情况下都是在 Discourse 之外进行的。

在本章的尾端，我们将讨论最有用的聚合网站，帮助你找到你所喜爱的 DAO 中最热门的话题。

研究工具——如何找到最热门的话题

Messari Governor 是一个不错的工具，因为它可以跟踪许多 DAO 并标记出那些有趣的提议。如果你想本身参与的 DAO 有被这个工具覆盖，你就会有真正的收获。

然后是 Tally.xyz，一个自称是 DAO 的运营工具，你可以在这里跟踪重要的投票和提案。如果没有提案监控系统 Boardroom 和 Sybil，这个列表就不完整了。

Boardroom 在你关注的 DAO 有新的提案需要投票时，会向你的邮箱发送通知。Sybil 目前还不支持很多 DAO，但有 Uniswap、Radicle 和 ENS，这几个最有趣的地方都在那里。Sybil 允许你关注各个 DAO 的代表的活动资讯，这是相当不错的。

提示和窍门

如何浏览论坛和聊天室

当你加入一个新的 Discord 服务器或在 DAO 论坛上打探消息时，大量的频道和并行的对话可能会让你不知所措。别担心，我们会带领你的。

Flipside 的 Francis ‘Fig’ Gowen 喜欢每天查看正在发生的事情，以确保他保持在发展的最前沿。他观察那些保持在某一类别顶端的主题。这些是参与度最高的话题，也有可能会产生最大的影响。有时，他还会关注关键贡献者的活动。这就像让一个当地人引导你穿过一个你刚开始了解的城市迷宫。

他的同事 Ian ‘d0bby’ Dobbins 使用 “自你上次访问以来 “的功能来让他了解论坛上的新问题。然后，他会访问象征性的事务和治理类别，因为这是他觉得被吸引的地方，也是他想产生最重大影响的地方。论坛的帖子通常都有标签，你可以搜索这些标签。但是检查类别本身有助于捕捉有趣但被误标的提案。

在 Discord，最好的事情就是关注那些你被吸引的频道的聊天记录，然后进行跳跃，直接写一个回复或帖子。在大多数服务器上，人们都很热情和友好。当 DAO 能够吸引和留住新的贡献者和人才时，它们就会茁壮成长。对成员来说，有一个内在的激励机制，鼓励他们对新手友好。你可以把它看成是人才培养。

菲格和伊恩都会追踪他们帖子的参与情况。费格保留了一个谷歌表格，记录他的回复和建议的点赞情况，以了解哪些是有效的，哪些是无效的，而伊恩则在 Discourse 讨论区的通知中查看点赞情况。

让你的帖子大放异彩

本章将给你一个快速的概述，以确保你的帖子能在竞争中脱颖而出。由于 Discourse 和 Discord 是两种截然不同的动物，我们将把本章分成两个小节。这一章是这篇部落格中最实用的，也是动手能力最强的。我们鼓励你立即付诸实践，在你选择的 DAO 中活跃起来。

在 Discourse 变得优秀

一个帖子最令人喜爱的品质是原创性和简明的好文章。你不需要成为莎士比亚来引起别人注意。基本的拼写检查、写作和在发帖前重读论坛帖子，以及拥有明确、积极的意图，都会有很大的帮助。

Fig’Gowen 使用 Google Docs 来制作提案或较长的回复，因为他可以在那里使用 Google 的拼写检查和 Grammarly。Ian Dobbins 在 Discourse 的帖子编辑器中直接输入他的回复，并充分使用右边的预览功能。Grammarly 也可以在 Discourse 中使用，远离拼写错误。

但有一个最重要的问题要问自己，那就是：“我想通过这个帖子达到什么目的？我实际上想说什么？” 好吧，这是两个问题，但又密切相关。

你可以使用基本的 HTML 标签，如你的文本在这里或使用 Markdown 语言来格式化你的 Discourse 帖子。这篇部落格有你想知道的所有细节。

我们最常使用的 Markdown 代码是：

## 代表 H2 标题。一个 # 号代表 H1，三个代表 H3，…. 接下来你懂的。

**表示粗体**，*表示斜体*。

以连字符 - 开始的字行，会成为一个圆点列表

选择其他帖子中的段落，并点击 “引用”，以插入另一篇帖子的部分内容供参考

[链接描述](https://hyperlink.here) 在帖子中插入描述性的链接

良好的结构有助于将长篇提案分解成更容易消化和理解的块状。

简洁、有表现力的写作是一种需要不断磨练和发展的技能，也是区分优秀和伟大的贡献者的原因。一份结构良好、目标明确、性格积极的提案，比起随意的乱说一通，更容易通过。给予你的提案所有的爱，使其有成功的最佳机会，这绝对是值得的。这里有一个最好的例子，是 MakerDAO 的 TJ Ragsdale 提供的一篇论述文章，写得非常好，显示了他深刻的研究。

大量的点赞并不总是衡量一个伟大提案的最佳标准。有时必须要表达一些不方便表达的事实。这不太可能获得大量的赞，但却可以产生巨大的影响。

在 Discord 中闪闪发光

相比之下，Discord 是一个更松散、更快速的媒介。拼写检查是由 Discord 即时完成的，我们需要留意这点。当然，拼写检查也会标记出很多有趣的口语，如 “HODL “或 “jeez”。

最重要的是问自己你想说什么，以及你想在互动中达到什么目的。我们刚刚就说过，但这值得再重复一次。每个人都会收到数以百计的通知和信息需要阅读。让它值得他们花时间来阅读。

推进工作——提案如何通过（或不通过）

如果你遵循上一节的内容，你应该上手了如何浏览各个频道。那让我们来讨论一下如何把事情做好。为了让 DAO 做一些事情，必须提出一个建议并被社区接受。只有这样才能提供必要的资源。

提案通常先在 Discord 浮出水面，通常是以非正式的「温度检查」的形式，来试探群体的情绪。如果情绪发生了压倒性的负面，那么这些担忧必须首先得到解决。在这种情况下，提出一个提案只会导致负面的评论，而且肯定不会通过。

你可以跳进这些对话，开始引导 DAO。谦虚和公开你的知识差距有助于避免尴尬的局面，并将确保你的问题得到解答。开始治理真的就这么简单。

如果提案的作者觉得他们的想法得到了积极的反馈，他们会把它带到另一个层次。在一个提案被付诸表决之前，它有时会在 DAO 的 Discourse 论坛上进行正式的「温度检查」。这些被称为「温度检查」或信号请求。在这个阶段，提案已经有了一定程度的完整和制定。注册用户可以投赞成票、反对票或有时投弃权票。有些论坛要求用户参与一段时间后才允许投票，以避免机器人和其他邪恶的活动。

在发出信号请求后，下一步是对提案进行正式投票。有时投票在 Discourse 中进行，或在采用 Token 加权投票时，在自定义网站上进行。偶尔也会在 Discord 中借助聊天机器人举行投票。其用于投票的工具有 Snapshot 和用于分发奖励的 Coordinape。后两者在链外工作，所以在那里的活动不会产生交易费用，使持有少数的 Token 的持有者可以参与。

让我们先仔细看看 Snapshot，然后快速浏览一下 Coordinape。

进一步了解 Snapshot 和 Coordinape

Snapshot 允许用户用他们的 Token 进行投票。要参与投票，用户需要在他的钱包里持有 Token ，也就是 MetaMask、Coinbase Wallet、Portis 以及其他所有支持 WalletConnect 的钱包。

如果你没有持有任何 Token ，你就不能投票。这是特别设计的机制，因为大多数 DAO 认为，只有那些受投票影响的人才能够参与。

下面是 BanklessDAO 上一个有些许争议的投票的截图。我们可以看到 icecool.eth，BanklessDAO 的核心贡献者和开发者之一是主持人，一个拥有 Discord handle links#7868 的用户支持该提案。

该提案要求提供资金，以便第四季的 BanklessDAO 能够得到本次讨论的资金。

导致该提案的实际讨论已经在 Discord 和论坛上进行。Snapshot 只是投票站。投票进行了一周，而留意一下时间就会发现，投票是以东部标准时间为中心的。作者在格林威治时间，也就是美国东部时间+5，投票从 2022 年 5 月 3 日美国东部时间上午 12:00 到 2022 年 5 月 10 日美国东部时间上午 12:00。

最终的投票结果被写入以太坊区块链，区块高度为 14,702,090。Snapshot 不在以太坊区块链上存储投票，因为那里的存储是非常昂贵的。

相反，它将投票，即由各自用户的钱包签名的信息，存储到星际文件系统，一个去中心化的云服务。

我们可以看到，投票通过了，~82% 的 Token 被委托给赞成票，~15% 的 Token 被委托给弃权票，3% 的 Token 被委托给反对票。如果投票允许， Token 持有人有时可以对多个选项进行投票。然后，持有人可以将其 Token 的一定比例分别分配给他们赞成的选项。

我们想提到的是，有些 DAO 的提案必须以智能合约的形式出现，以使自动提案的执行顺利进行，这也是最接近于自治组织的真正定义，其执行不依赖于任何一个成员或团体。通过调用相应的智能合约，投票可以在没有人的情况下发生，有些 DAO 需要这种程度的技术专长。

Coordinape 在严格意义上不是投票工具，而是让社区成员以一种很酷、很有趣的方式向贡献者分发奖励或报酬的工具。它有一个很好的用户体验，成员可以选择其他人，并决定他们想分配给其他人多少份额。Coordinape 是一个有趣的游戏理论实验：成员决定是否尽可能地分散他们的 Token ，以最大限度自身的接触点和宣传自己，或将他们的 Token 分配给少数提供了最多价值的关键成员。我们不想在这里讨论更多细节。如果你有兴趣，这篇部落格有更多的深度探讨。让我们用一个完毕的 Coordinape 轮次的截图来结束这一部分。线代表着分配。圆圈代表着贡献者。

是什么让贡献者变得杰出

衡量一个优秀贡献者的标准是他们对 DAO 的影响。Ian Dobbins 建议优先考虑在社群有存在感，展示自己，并提出好的想法，也不要害怕提出不同意见。作为一个不断向别人献殷勤的 “好人”，不可能使你有很多朋友，也不会产生影响。关注问题并发表意见是至关重要的。

遵循投票要求是确保你的参与度达到标准的另一个标志。一些 DAO 会衡量参与度，常规贡献者会因其努力而得到奖赏。把投票活动记在你的日程表上，并在论坛上活跃起来，因为有时候，错过一次投票就意味着你放弃了该月的奖励。

‘Fig’ Gowen 认为对更广泛的生态的良好理解是作为一个杰出贡献者的标志。许多原始的形态都会在多个生态进行流传的，而这可以帮助掌握 DAO 和协议之间的互动。例如，了解 Curve 和 Convex 的动态有助于理解 Saber 和 Sunny 在 Solana 上的情况。拥有这种基础性理解的贡献者可以指出最佳的做法，并能够展示已做了哪些类似的实验以及取得了哪些成果。

Fig 和 Ian 都认为，能够组织自己的意见并以简洁的文字表达出来，是贡献者最重要的一项技能。拥有独特的声音可以使帖子具有亲和力和识别度。

总的来说，DAO 治理最先是一项社群活动。能够通过领导力和研究带来价值，且有亲和力的贡献者总是具有影响力的，特别是如果他们能够很好地表达他们的意见。

总结

我们希望这篇部落格能给你提供你在这领域串游的工具和理解。DAO 治理是一个认识聪明人和影响你关注的项目的发展与方向的好地方。

在传统公司中获得影响力意味着被雇用担任一个具有影响力的全职职位。DAO 降低了入门门槛。即使是新的贡献者，当他们提交经过深入研究和精心编写的提案，为当前的问题提供解决方案时，也可以产生一定的影响。
