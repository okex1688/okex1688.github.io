原文标题：为何 NFT 能抗跌？

最近几周，一系列的宏观经济事件在加密市场掀起了轩然大波，也在提醒大家这个新兴行业无处不在的的潜在风险和波动性，比如：

1、哈萨克斯坦境内围绕比特币挖矿问题引发了争议；

2、新一轮新冠疫情的爆发；

3、对美联储加息的预期；

4、乌克兰不稳定的市场局势。

所有这些原因似乎都成为拖累资本市场的因素，也造成了加密市场出现阶段性下跌。

由于加密货币与传统市场之间存在着一定程度的相关性，因此在区块链领域也强烈感受到了这种影响，比特币和以太坊自 2021 年 11 月创下历史新高以来，目前价格已经下跌了一半，BNB、ADA、SOL、AVAX、SAND、MANA、GALA 以及其他几种历史表现良好的加密货币也无一幸免。在此期间，加密货币总市值也已经从 2.9 万亿美元缩水至 1.6 万亿美元。 

资料来源: DappRadar

毫无疑问，目前的加密市场情绪恐慌，处处面临着挑战。然而在区块链市场里，在一个特殊的垂直行业却又是另外一副景象，它就是：NFT。

了解 NFT 宏观经济学

尽管一系列事件的发生阻碍了加密市场前进的步伐，但以下几大因素在宏观经济层面上对 NFT 未来的发展产生了积极影响。

首先，各界名人和各大品牌越来越多地参与到了 NFT 领域。拥有广泛社交影响力的足球明星内马尔（Neymar Jr.）（他在 Twitter 和 Instagram 上拥有超过 2 亿粉丝）和知名影星凯文哈特（Kevin Hart）（在 Twitter 和 Instagram 上拥有超 1.92 亿粉丝）就于近日公开宣布他们已入手「无聊猿」Bored Ape Yacht Club (BAYC)（注：该项目也是加密市场中最早一批出现的 NFT 项目）。

与此同时，作为最受加密货币、NFT 爱好者欢迎的社交媒体平台，Twitter 为了扩大影响力也启用了第一个 Web 3 功能。接下来，社交媒体平台 Instagram 和 Facebook 也将会紧随其后推出了类似的功能。与此同时，就连零售巨头沃尔玛也紧跟潮流，为 NFT 用例申请了好几个商标。

资料来源: DappRadar

现阶段，NFT 资产的受欢迎程度绝对比以往任何时候都要高。数据不会说谎：「NFT」一词的 Google 搜索量最近首次超过了「Crypto」，而来自亚洲的 NFT 爱好者数量也在与日俱增，逐渐占领了这个曾经由北美和欧洲用户主导的市场。

NFT 链上指标讲述了一个牛市故事

NFT 是去年区块链行业的一匹黑马，表现相当出色。仅在 2021 年一年，NFT 市场就总共完成了 250 亿美元的交易量，比前四年的交易总和高出了 18,414%。

尽管加密货币正处于水生火热之中，但 NFT 却似乎丝毫不受影响，反而发展得欣欣向荣。接下来，我们再缩小分析范围，聚焦以太坊。以太坊上的 NFT 同样表现出了非常积极的趋势：其交易量占到了去年总交易量的 75%；以太坊上的 NFT 销售量和用户数量也在持续增涨；唯一活跃钱包（UAW）被连接到了收藏品去中心化市场 NFT Dapps。据统计，自 2021 年 12 月以来，平均每天有超过 53,300 个唯一活跃钱包连接到以太坊 NFT Dapps，这一数字要比去年第三季度高出 43%。

资料来源: DappRadar

尽管加密货币的表现不尽如人意，但受积极的宏观经济事件影响，以及 NFT 在「边玩边赚」（Play-to-Earn）和元宇宙领域中发挥的核心作用，链上指标节节攀升。建设去中心化、可互操作的元宇宙对 NFT 市场发展来说是绝对的利好。

资料来源: DappRadar

此外，由于支持 NFT 的基础资产——加密货币价值下跌，人们可能会将现在视为一个买入加密货币的绝好机会。尽管 2022 年刚开年，但独立交易者的数量比以往任何时候都要多——达到了创纪录的 160 万。在这些独立交易者的推动下，以太坊 NFT 产生了超过 37 亿美元的交易额，而且这还不包括 LooksRare 上的交易额，照此趋势下去将很有希望打破 2021 年 8 月创下的 45 亿美元的交易记录。

资料来源: DappRadar

上述提到的一系列链上指标反映出了市场对于新资产类别的认可以及积极的市场情绪。不仅如此，还有另一个指标也能很明显地反映出市场对于 NFT 资产的追捧——地板价。

地板价分析

地板价（Floor Price）是评估 NFT 收藏品最重要的指标之一，尤其是从投资者的角度来看。NFT 收藏品的地板价就是该藏品的最低要价，代表着最低准入门槛。

最近，我们对一些重要的以太坊收藏品的地板价进行了分析，结果表明，NFT 更像是一种存储价值的资产，而且重要的是，这一类别的资产表现要优于加密货币，甚至是优于黄金以及及标准普尔 500 指数等一些传统资产。

资料来源: DappRadar

不难看出，NFT 领域的整体价值正在稳步提升。通过分析排名前 100 的 NFT 收藏品的地板价，我们发现目前 NFT 的价值约为 148 亿美元，比 11 月减少了 24 亿美元。尽管在这段时间里 ETH 价格受到重创，下跌了 50%，但交易量最高的 NFT 收藏品的价格下跌幅度还未超过 15%，这也反映出了 NFT 资产十分抗跌。

在这里不得不提对 NFT 产生直接积极影响的 NFT 收藏品之一——BAYC。11 月初，当比特币和 ETH 达到历史高位时，BAYC 的地板价在 30 ETH 左右浮动。一周之后，尽管 ETH 的价格下跌了 15%，但 BAYC 的地板价却上涨了 60% 以上，超过 50 ETH。到 2021 年年底，最便宜的 BAYC 价格已经达到了 60 ETH，大部分都已超过 90 ETH。这意味着如果以当前 ETH 价格计算的话，入手最便宜的 BAYC 必须要花费 225,000 多美元。

资料来源: DappRadar

尽管在过去的两个月中，备受瞩目的加密货币蒸发了大约一半的市值，但从 11 月 10 日起，BAYC 却上涨了 207%（以 ETH 衡量），地板价上涨了 14%（以美元来衡量）。换句话说，假如你从 11 月 10 日到今天为止手中持续持有 BAYC 的话，那么你可以获得了 14% 的资金收益，与此同时那些持有任何加密货币的人的净亏损额却可以达到 50%。很明显，BAYC 已经成为了一种保值资产。  

资料来源: DappRadar

然而，BAYC 并不是唯一上涨的头像类 NFT。自 11 月 10 日以来，另一个以太坊头像 NFT——World of Women 上涨了 383%（以 ETH 衡量），同期的实际价格上涨了 185%。此外，自去年 12 月以来，CyberKongz 及其 voxel (VX) 版本的地板价也实际分别上涨了 27% 和 68%。Doodles 也是如此，其地板价自 11 月以来已上涨了 224%。

资料来源: DappRadar

CryptoPunks 作为最有「年代感」的 NFT 收藏品，分析起来比较棘手。尽管该系列 NFT 的表现不如上文中提到的其它系列，但不得不说 CryptoPunks 的表现仍然优于其它几种加密资产。Cryptopunks 地板价在 11 月 2 日冲上了 100 ETH，一周后又下跌至 83 ETH，就在同一天比特币和 ETH 创下了历史新高。然而从那天开始，CryptoPunks 的地板价下降了 9.6%（以 ETH 衡量），但实际价格仅仅从 254,000 美元缩水到了 229,000 美元。由此可见，CryptoPunks 也是相当抗跌的。

资料来源: DappRadar

就在头像类 NFT 的价值显著上涨时，与虚拟世界和游戏相关的 NFT 也毫不逊色。Forgotten Rune Wizards 就是典型的例子，其地板价从 11 月 10 日起上涨了 210%（以 ETH 衡量），实际价格比 12 月高出了 132%。同样，自 11 月以来，Gala Games VOX NFT 的地板价也上涨了 145%（以 ETH 衡量）。

资料来源: DappRadar

另一方面，作为虚拟世界底层架构的 NFT 从元宇宙的炒作中受益匪浅，迅速增值。例如，The Sandbox 和 Decentraland 中虚拟土地的地板价在 Facebook 宣布更名后就一直保持着较高的价值（尽管以美元衡量的实际价值有所下降）。与此同时，尽管加密货币出现了下跌趋势，但 Worldwide Webb 中虚拟公寓的楼面价格自 11 月以来上涨了 242%（以 ETH 衡量），而 Cryptovoxels 中的虚拟地块比 11 月高出了 17%（以 ETH 衡量）。

资料来源: DappRadar

我们认为，分析收藏品的地板价可以从投资角度了解该项目的稳定性。通过最近的分析证实，在整个市场呈现出下跌趋势的同时，一些 NFT 却站稳了脚跟。为什么会出现这种情况呢？为什么 NFT 不受加密货币价格下跌的影响，如此抗跌呢？

为什么 NFT 能够如此抗跌？

这个问题回答起来可没那么简单。但可以肯定的是，在一些特定因素的影响下，NFT 吸引了不少理性的投资者。

有一种观点认为，某些 NFT 应该被视为文化作品而不是数字资产。然而和某些艺术品一样，那些 NFT 也被一些人视为具有真正的投资价值。就如加密数字艺术家 Beeple 创作的「The First 5000 days」一样，这副作品通过佳士拍卖了 6900 万美元，并将非同质化Token带到了世人面前。此外，像 Fidenza、Ringers 以及 Autoglyphs 这样的生成艺术收藏品也正在作为一种新的表现形式不断涌现出来，且其价值也达到了数千美元。

资料来源: DappRadar

与此同时，BAYC 和 CryptoPunks 这类 NFT 也逐渐成为了市场主流。2021 年 8 月 23 日，VISA 宣布以 150,000 美元的价格购买了 CryptoPunk #7610，进一步巩固了 CryptoPunks 的保值地位。此外，BAYC NFT 系列中的 101 件作品在苏富比成功拍卖了 2,400 万美元。

尽管目前只有少数收藏品能和上述两类 NFT 相提并论，但相信在不久的将来，NFT 将会在音乐、票务、体育和时尚领域大放异彩。总而言之，人们对 NFT 寄托了美好的期待，但只有时间才能告诉我们 NFT 是否会产生超越世代的文化影响。

实用性

一个成功的 NFT 项目最重要的特点之一就是授予其社区成员奖励：持有者通过持有这些项目的 NFT 可以获得附加价值或是实用性。

例如，CryptoPunks 持有者在后期又收到了 Larva Labs 团队推出的免费 Meebit，增加了初始投资的实际资金收益。其它项目比如：Cool Cats（以 ETH 衡量的地板价从 11 月 10 日起上涨了 150%）、The Meta Key（上涨了 370%）、Doodles 以及 BAYC 等也是如此。

而实用性则可以在产生收益的 NFT 中体现出来，如每天都会发放原生实用Token的项目： CyberKongz 及其 BANANA Token 。Genesis Kongz 持有者每天都会收到十个 BANANA Token ，每个价值 24 美元。此外，在 SupDucks (35%) 和 VOLT Token 以及 Cool Cats 和他们的 MILK Token 中也存在着类似情况。

BANANA 实用性框架，上图资料来源: CyberKongz 

此外，NFT 还融入进了 DeFi 和游戏等其他类别的项目中。其中，DeFi 与 NFT 进行融合的典型例子就是像 NFTfi 这样的平台，用户可以在此类平台上使用 NFT 作为抵押品。而在像 Pixel Vault 这样的项目中，MetaHero Universe (57%) 中的 NFT 可以被质押来，也可以对基于工作量证明（PoW）共识机制的 Token 进行「挖矿」。

当然，NFT 还可以更好地融入到游戏里，成为必不可少的游戏元素。尽管 ETH、SOL、GALA 和 AURY 的价格大幅下跌，但你会发现 Mirandus NFT 和 Aurory Villagers 都保持了地板价值。我们之前还观察了 Forgotten Rune Wizards，这是另一个融入游戏实用性机制的 NFT 集合，其地板价值同样得到了不错的发展。尽管保持无数游戏替代品的地板价值具有挑战性，但可以肯定地说，对区块链游戏的需求正在上升，为这些类型的 NFT 基本上都增加了价值。

名人明星效应和 NFT

还有一个对 NFT 产生积极影响的相关趋势，那就是名人明星和知名品牌的参与，随着美国橄榄球联盟 NFL 巨星 Dez Bryant、NBA 球星 Stephen Curry、知名歌手 Post Malone、Snoop Dogg 和 Eminem 等名人加入「无聊猿」BAYC，这个 NFT 项目开始崭露头角。之后随着 Jimmy Fallon、Paris Hilton、Neymar 和 Kevin Hart 等更多娱乐明星入场，BAYC 社区变得更加独特。

名人的积极影响力还体现在另一个 NFT 项目上，它就是：World of Women，该系列在 Eva Longoria 宣布购买几小时后地板价上涨了 250%，Doodles 则是另一个受益于 Pranksy、Loopify 和 Steve Aoki 等著名 NFT 角色的项目。

上图：名人 NFT，资料来源：DappRadar

同样，传统大企业的加持给带来了不俗的影响力，比如，阿迪达斯、可口可乐、百事可乐、百威和其他几个品牌要么推出了他们的收藏品，要么与重要的 NFT 团队合作，在该领域占据一席之地。时尚巨头 Gucci、Dolce & Gabbana、Burberry 和其他探索元宇宙的公司也是如此。

NFT 项目团队正转型打造「品牌」

NFT 收藏背后的团队对其项目的命运负有高度责任。一些团队发现，让才华横溢的艺术家和经验丰富的开发人员伴随 NFT 项目成长也是一个成功的秘诀。一旦团队成功，他们就可以成为该领域里真正具有影响力的品牌。

资料来源: RTFKT

RTFKT 就是一个典型示例，他们是 Clone-X 背后的创作团队，负责与 Jeff Staple 和 Fewocious 合作开发独特的 NFT 作品。最终，这个数字设计和时尚品牌依靠这种商业模式达到了预期效果，并妥善应对了任何挑战。如今，RTFKT 已经成为 Web 3 领域里最具影响力的品牌之一，并于去年 12 月被耐克收购，收购金额未披露。

稀缺资产

最后，我们来谈谈 NFT 领域里的稀缺性因素。

实际上，稀缺性非常重要，因为它直接关系到人类的心理，错过某些东西的焦虑，也称为 FOMO，这种心态与稀缺资产和投资市场息息相关，而 NFT 将两者结合了起来。

大多数收藏品都会推出数量有限的 NFT，这些 NFT 将永远保存在区块链上。无论是 10,000 份 NFT 还是 20,000 份 NFT，特定版本的限量供应将始终保持不变。这就是区块链的魅力。

结果就是——NFT 逐渐变成了一种「自成一派」的资产类别。虽然由于这项技术的性质，与加密货币的相关性仍然存在，但 NFT 正在缓慢地为自己创造一种经济，其宏观事件和市场因素对该领域产生积极或消极的影响。

当文化价值、附加效用、名人参与、品牌知名度和这些资产的稀缺性等因素结合在一起之后，推动了最新一波 NFT 采用。

坦率地说，我们看到的也许只是 NFT 领域里的冰山一角。Larva Labs、BAYC、Cool Cats、RTFKT、CyberKongz 和 Doodles 等项目将继续高度参与元宇宙叙事。随着越来越多用户体验更好的产品推出，我们会看到更多非加密用户参与到这一领域，举个例子，据称 Coinbase 的 NFT 市场预计将整合万事达卡支付功能，未来肯定也会有很多潜在用例将 NFT 嵌入到传统商业模式中。

目前尚不清楚加密货币市场当前的看跌趋势是否会持续多久，也许是两周、也许是两年。尽管如此，在当前市场环境下，NFT 实际价值不但没有受到影响，而且还在持续增加。毫无疑问，NFT 已经证明它绝对算得上是一个能够存储价值的数字资产。
