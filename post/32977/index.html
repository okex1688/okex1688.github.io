我拥有的唯一加密货币是ExcelCoin。ExcelCoin是下一代加密货币，其核心创新在于其分类账的维护不是通过工作量证明（这会造成环境问题）或权益证明（存在自身问题），而是由我来维护。我在 Microsoft Excel 中维护帐本。这是截至今天上午9点的账本副本：

如果你想购买ExcelCoins，请给我发电子邮件；我将以 0.99 美元的出价 / 1.01 美元的报价做市，当然这会随着市场条件的变化而变化。

当然，ExcelCoin最初是作为一种货币，一种“数字现金”的形式，但这个概念的强大之处在于，原则上你可以使用相同的基础技术（我在Excel中输入）跟踪任何类型的资产。 例如，您可以扩展ExcelCoin技术，不仅可以跟踪像硬币这样的可替代代币，还可以跟踪不可替代的代币，其中分类帐中的每个条目都代表一种独特的资产。例如：

你为什么要我在名单上你的名字旁边写“一张猫的照片”？这只是一个玩具例子，但只要有一点想象力，你就会看到它有多么强大。 例如，我们可以通过这种方式跟踪房地产。我可以写下我镇上所有房屋的清单以及是谁所有，我们可以更新我列表中的条目，转移所有权。到目前为止，我在Excel中的列表不具有任何合法所有权，它不会与合法财产登记处同步或取代。但我认为你会赞同，它比旧的纸质财产登记法律制度效率更高，技术先进得多，所以我认为从长远来看它很可能会胜出。

这是愚蠢的，为什么这么说呢？简单地说，它在技术上是愚蠢的。实际的加密货币（以及 NFT 的区块链系统、提议的房地产区块链等）具有分布式共识机制，用于更新其共享分类账、股权证明或工作证明或其他一些机制，让参与者在系统对账本进行验证和集体控制。 而我可以在Excel中输入我想要的任何内容。如果你从我这里购买了一些ExcelCoins，而我在我的Excel分类帐中删除了你的条目，那么你也就没有了ExcelCoins，那是什么类型的系统？

一个更好的答案是一个更好的答案是它在社会上是愚蠢的，因为维护ExcelCoin分类账的人是我，我显然是在开玩笑。你有一个更好的答案是它在社会上是愚蠢的，因为维护 ExcelCoin 分类账的人是我，我显然是在开玩笑。你不会相信我会维护 ExcelCoin 账本，因为我发明 ExcelCoin是为了拿账本开玩笑。 另一方面，数以百万计的人拥有美元，这恰恰意味着某家银行有一个清单，并且在该清单上有一个条目。它可能不在 Excel 中，但基本思想相同：银行在其账户中有一张美元清单，该清单不是通过任何类型的共识机制或区块链或其他方式保存的，银行只是使用Cobol保存清单列表。 但是你（很大程度上）信任银行来维护该列表，因为除其他外**（1）它在维护这个名单的良好记录，（2）它具有维护这个名单的商业动机，（3）它有维护这个名单的强大监管和法律动机，等等。** 银行的名单在技术上没有任何大的震动，100年前，银行用笔和纸保存了非常类似的名单。这个名单系统并不是完美无缺的，有时银行会把名单弄得一团糟！但它并不是愚蠢至极。这是非常正常和标准的，数百万人不假思索地依赖它。不是因为ExcelCoin是愚蠢的，我才把他保存在Excel中，而是因为我把它保存在Excel中，才显得它很愚蠢。

上周五，Moxie Marlinspike写了一篇关于 “我对web3的第一印象 “的博文。Marlinspike是著名的密码学家和计算机安全专家，Signal应用程序的发明者，等等。这篇文章在Web3（和对Web3持怀疑态度的人）世界中得到了广泛关注；它很好，我强烈推荐它，我在这里只谈其表面。一个相关的观点是，分布式账本不一定是资产所有权的完整技术解决方案。

大多数人在想到 NFT 时会想到图像和数字艺术，但NFT通常不会将这些数据存储在链上。对于大多数图像成为NFT，这成本太高了。

NFT不在链上存储数据，而是包含一个指向数据的URL。这些标准让我感到惊讶的是，位于URL的数据没有哈希承诺。看看流行市场上的许多 NFT 以数十、数百或数百万美元的价格出售，这个URL往往只是指向某个运行Apache的VPS。任何有权访问该机器的人、将来购买该域名的任何人或任何破坏该机器的人都可以随时将 NFT 的图像、标题、描述等更改为他们想要的任何内容（无论是否不是他们“拥有”）。在NFT规范中，没有任何东西告诉你图像 “应该 “是什么，或者甚至允许你确认某些东西是否是 “正确的 “图像。

所以作为一个实验，我做了一个NFT，根据谁在看它而改变，因为提供图像的网络服务器可以根据请求者的IP或用户代理选择提供不同的图像。例如，它在OpenSea上看起来是一种方式，在Rarible上是另一种方式，但当你购买它并从你的加密货币钱包查看它时，它将始终显示为一个大的[大便]表情符号。你出价的东西并不是你得到的东西。这个NFT没有什么不寻常的地方，这就是NFT规格的构建方式。许多价格最高的NFT随时都可能变成[大便]表情符号，我只是把它变得更明确。

技术也是完全集中的吗？马林斯派克继续说：

NFT本身并没有在法律或实际意义上传达对相关事物的所有权。它在某种更形而上的意义上传达了所有权：如果你买了一个Bored Ape Yacht Club NFT，那么作为BAYC NFT社区一部分的人将把你视为猿人的所有者。这本质上是一个社会事实，即使区块链的不可更改的代码说你不拥有这一只，因为你被黑客攻击或其他什么原因，这也可能是真的。技术是悬挂社会系统的脚手架，但社会系统是在有意义的意义上传达或不传达 “所有权 “的东西。

技术也是完全集中的吗？Marlinspike继续说：

几天后，在没有任何警告或解释的情况下，我制作的 NFT 被从 OpenSea（一个 NFT 市场）中删除。

不过，我发现最有趣的是，在OpenSea删除了我的NFT后，它也不再出现在我设备上的任何加密钱包中。这是 web3，但是，这怎么可能？

MetaMask、Rainbow等加密钱包是“非托管”的（密钥保存在客户端），但它与我上面的 dApp 存在相同的问题：钱包必须在移动设备或浏览器中运行。与此同时，以太坊和其他区块链的设计理念是它是一个对等点网络，但并不是这样设计的，即你的移动设备或浏览器真的有可能成为这些对等点之一。

像MetaMask这样的钱包需要做一些基本的事情，比如显示你的余额、最近的交易和你的NFT，以及更复杂的事情，比如构建交易，与智能合约交互，等等。简而言之，MetaMask需要与区块链互动，但区块链的构建使MetaMask这样的客户端无法与之互动。因此，像我的DApp一样，MetaMask通过调用API给三家在这个领域合并的公司来完成这个任务。

例如，MetaMask……通过调用OpenSea的API来显示你的NFT…

这一切意味着，如果你的NFT从OpenSea被删除，它也会从你的钱包中消失。从功能上讲，我的NFT在区块链的某个地方不可磨灭并不重要，因为钱包（以及生态系统中越来越多的其他东西）只是使用OpenSea API来显示NFT，它开始为我的地址所拥有的NFT的查询返回304 No Content!

“OpenSea在Excel 中保留了一份清单，列出了谁拥有哪些NFT。”底层的列表保存在区块链上，只是OpenSea可以按照自己的意愿修改其版本的列表，而其修改实际上是非常有约束力的。事实上，经常会有这样的故事：（ 1 ）一些易受骗的加密人向任何索要他的私钥的人发放私钥，（ 2 ）他的NFT被盗取，（ 3 ）他在 Twitter 上抱怨，（ 4 ）他让一个主要的NFT市场锁定他被盗的NFT的交易，（5）每个人都写文章，如 “哈哈哈，你现在不喜欢去中心化了吧？”

Web3的支持者认为，这很好，因为OpenSea任意行动的能力受到社区标准的限制。开放的不可改变的区块链仍然存在，所以你可以把OpenSea说的和区块链说的进行比较。（也许 MetaMask 没有，但这种可能性是存在的。）如果OpenSea的列表不符合其客户的期望，例如它任意忽视区块链，它让人们声称不是他们的NFT，等等那么每个人都会把他们的猿猴带到其他地方。但这也是人们信任银行的基本原因! 基本的保护措施是社会性的，而不是技术性的。

Marlinspike:

当你想一想，如果所有的web3部分都消失了，OpenSea实际上会在直接意义上“更好”得多。它会更快，对每个人来说更便宜，并且更容易使用。例如，要接受对我的NFT的出价，我将不得不支付超过80至150美元以上的以太坊交易费用。这为所有出价设置了一个人为的底线，否则你会因为接受低于汽油费的出价而赔钱。与信用卡相比，信用卡支付费用通常让人觉得是敲诈勒索的，但看起来便宜。如果人们想要交易、报价、出价等的公共记录来验证他们的账目，OpenSea甚至可以发布一个简单的透明度日志。

但是，如果他们建立了一个平台来买卖名义上不基于加密的图像，我认为它不会起步。不是因为它不是分布式的，因为正如我们所看到的，使其运作所需的许多东西已经不是分布式的。我之所以不看好，因为这是一个淘金热。人们通过加密货币投机赚了钱，这些人有兴趣以支持他们的投资同时提供额外回报的方式花费这些加密货币，因此这就定义了财富转移市场的设定。

处于末端的炒作NFT的人，从根本上来说并不关心分布式信任模型或支付机制，但他们关心的是钱在哪里。因此，钱吸引人们进入OpenSea，他们通过建立一个在web2空间迭代底层web3协议的平台来改善体验，他们最终提供了通过OpenSea本身而不是通过你自己的智能合约来 “铸造 “NFT的能力，最终这一切为Coinbase打开了大门，通过你的借记卡，用他们自己的平台提供进入验证的NFT市场。这为Coinbase通过Coinbase持有的暗池自行管理代币本身打开了大门，这有助于消除交易费用，并使其有可能完全避免与智能合约交互。最终，所有的web3部分都消失了，你有一个用你的借记卡买卖JPEG的网站。由于市场动态，该项目不能以web2平台开始，但同样的市场动态和中心化的基本力量很可能会促使它最终达到这个目的。

同样，在技术上，ExcelCoin在某些方面优于其他依赖分布式共识机制的加密形式。但在社会上情况要糟糕得多，不仅因为我在开玩笑，没有做任何事情来建立对 ExcelCoin 的信任，而且还因为加密货币存在巨大的投机泡沫，不适用于透明的非加密货币事物，比如我的愚蠢 Excel 笑话.分布式共识和不可变区块链的力量并不在于它们是比你知道的一些大型可信网站保存列表更好的技术，这就是Web3的工作方式。分布式共识和不可变区块链的强大之处在于它们可以吸引资金，这正是Web3的工作原理。

国会股票交易

有时我了解到一些夫妻，其中一个在银行工作，另一位在对冲基金工作，我想“啊，真是一场合规噩梦。”许多受监管的金融公司（以及律师事务所、咨询公司等相关公司）对个人交易有非常严格的规定，以防止其员工不当使用内幕信息。他们要求预先批准交易，禁止以员工正在工作的名义进行交易，有时甚至禁止所有个股交易。并且为了避免明显地玩弄这些规则，它们往往会延伸到员工的家庭。如果你在银行工作，而你的配偶是一名教师，这很好，尽管如果你的配偶也是一名业余股票交易员，你可能不得不做出一些艰难的选择，但如果你在银行工作而你的配偶在对冲公司工作，似乎更难。你不能指望对冲基金能得到银行的预先批准。我想人们会解决这个问题，但从概念上看，这似乎是一个挑战。

快速浏览一下彭博社，也许你不会注意到，AeroCentury Corp的股票周五收于47.99美元。它在周末完成了5比1的股票分割，所以彭博社的历史价格页面追溯调整，显示周五的收盘价为9.598美元。然后人们今天早上醒来进行盘前交易，然后……呃……

纽约证券交易所Arca Equities与其他UTP交易所一起，裁定取消所有错误的ACY - AeroCentury Corp的交易，这些交易在今天美东时间4:00:00-4:04:00之间执行，价格或高于11.52美元。该裁决不符合上诉条件。

基本上这里似乎发生的事情是你有一个计算机程序来买卖 AeroCentury 股票，该程序正在运行中并在周五关闭，周五收盘时认为AeroCentury的交易价格是47.99美元，然后根据这个价格和周末的发展，它认为 “我将在47美元买入AeroCentury或在49美元卖出 “或其他（实际数字不重要），然后在周一早上之前，你要么做要么不更新程序，像 “别忘了把所有东西除以5。“有些人做了，有些人没有做。不这样做的人他们的程序在凌晨4点醒来时，AeroCentury的市场价格为47.00美元/49.00美元，而这样做的人，他们的程序在周一凌晨4点醒来时，AeroCentury的市场价格为9.40美元/9.80美元，而那些多想了一会儿的人醒来时，AeroCentury的市场价格为9.40美元/45.00美元，并以可笑的高价向那些忘记的人出售一些股票。然后证券交易所就会破坏这些交易，因为这其实更像是一种加密货币。
