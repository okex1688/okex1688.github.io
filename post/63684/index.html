经历超过72个小时后，OpenAI的这场“权利游戏”闹剧还未结束。

钛媒体App11月21日消息，据最新报道，OpenAI董事会就两家公司合并的可能性，曾接洽了其竞争对手、谷歌和亚马逊投资的AI大模型开发商Anthropic的联合创始人兼CEO达里奥·阿莫代伊（Dario Amodei）。董事会此举的部分原因是试图说服阿莫代伊接替奥特曼（Sam Altman）担任OpenAI CEO。

报道指，目前尚不清楚合并提议是否引发了任何认真的讨论。由于阿莫代伊在Anthropic的职位，其很快拒绝了担任CEO的邀请。

换句话说，此次合并最终未得到Anthropic的肯定，双方合并可能已经“凉了”。据悉，阿莫代伊也曾是OpenAI员工，由于担心该公司将其技术商业化的速度太快，他与其他部分团队成员于2020年底离职，创办了Anthropic。

早前，北京时间11月20日下午2点左右，OpenAI董事会突然宣布，“ChatGPT之父”、OpenAI联合创始人奥特曼确认被免去CEO职务并离开公司，同时聘请视频流媒体网站Twitch联合创始人米特·希尔（Emmett Shear）接任OpenAI临时CEO。

据悉，米特·希尔16年前与他人共同创立了游戏流媒体平台Twitch。2014年，亚马逊以近10亿美元收购了Twitch，今年6月，希尔宣布辞去Twitch首席执行官的职务，称想花更多的时间陪伴刚出生的儿子。

有消息称，希尔赢得了OpenAI董事们的青睐，是因为董事会成员认为，希尔多次对外公开演讲中提到，他似乎已认识到AI带来的生存威胁。报道补充称，作为一名计算机科学家，希尔花了十多年时间将Twitch打造成世界上最成功的视频平台之一，因此他被认为有能力领导一个大型工程团队。

希尔随后发推文表示，作为上任头30天工作的一部分，他计划改革这家人工智能(AI)初创公司的管理，并对其联合创始人和前CEO遭罢免一事展开调查。希尔将任命一名“独立调查员”调查围绕奥特曼被免职的过程和沟通策略，“（这件事）严重损害了我们的信誉”。

实际上，在奥特曼被罢免之后，董事会还联系了其他高管，邀请他们担任OpenAI临时CEO，其中包括GitHub首席执行官Nat Friedman，Scale AI首席执行官Alex Wang和阿莫代伊，除希尔之外，剩下三位均对此邀请表示拒绝。

消息公布两个多小时后，北京时间11月20日下午4点左右，微软CEO纳德拉（Satya Nadella）突然宣布，奥特曼和OpenAI原总裁布罗克曼（Greg Brockman）将加入微软，领导新的微软AI团队。奥特曼则转发该帖并表示：“任务继续。”

随后，OpenAI CTO米拉·穆拉蒂（Mira Murati）等多位员工发统一推文称，“没有了员工，OpenAI什么都不是”文字，奥特曼转发了超过10条该内容并配以“心”的表情图。

据报道，有超过700多名（占共计约770名员工比重95%）的OpenAI员工已签署联名信，以辞职为名逼迫董事会恢复奥特曼与布罗克曼的职位，并要求更换董事会成员。

更令人吃惊的是，此前被认为是对立面的OpenAI联合创始人、首席科学家伊利亚(Ilya Sutskever)也签了名。他是投票罢免奥特曼的四人董事会成员之一。但这份名单主要系电子签名，目前暂无法明确真假，OpenAI未对外确认此事。但是伊利亚有在社交媒体上表示，非常懊悔参与了董事会的这一罢免行动，将尽最大努力让OpenAI团队重聚。奥特曼和米拉都有给他的这条帖子贴心心符号支持。

值得一提的是，OpenAI董事会成员、Quora的联合创始人兼CEO Adam D’Angelo这次投了奥特曼罢免赞成票。Adam此前对外界的表态一直都是比较赞成伊利亚的观点，即AI是存在巨大安全问题，没想过OpenAI会成为一家全球科技巨头。因此他成为此轮奥特曼罢免事件的关键人物之一。有外界传闻奥特曼发布GPTs和Quora发布的产品有冲突，导致他认为奥特曼没有充分和董事会沟通而拉拢伊利亚发起了罢免行动。但这个事暂无定论，也无人证明，真相暂时无人知晓。

11月21日，微软CEO纳德拉对外表示，微软对奥特曼的走向持“开放态度”，无论奥特曼加入微软还是重返OpenAI，奥特曼都将与微软进行合作。

“无论Sam（奥特曼）在哪里，他都在与微软合作。我仍然对Sam及其领导力和能力充满信心，这就是为什么我们希望欢迎他加入微软。”纳德拉还透露，尽管作为OpenAI最大投资人，但截至目前，他还没有被告知奥特曼为什么被解雇。“除了沟通中断外，董事会和我没有谈论Sam所做的任何事情。”

整体来看，OpenAI的这场内部纷争已经影响到了整个AI行业，也同时让这家公司名誉受损。关于奥特曼是否会重返OpenAI，事件仍在不断变化中。

以下是钛媒体App梳理的奥特曼罢免事件大致脉络：

  • 当地时间11月16日晚，OpenAI前CEO奥特曼、前总裁布罗克曼收到首席科学家伊利亚（Ilya Sutskever）发来的短信，奥特曼要求17日中午谈话。参加了一次Google Meet会议，除了布罗克曼之外，整个董事会都在场。伊利亚告诉奥特曼他将被解雇，消息很快就会传出。
  • 当地时间17日中午12:19，布罗克曼收到伊利亚发来的短信，要求快速回电。
  • 当地时间17日中午12:23，伊利亚发送了Google Meet链接。布罗克曼被告知，他将被从董事会中除名（但他对公司至关重要，将保留他的职位），而奥特曼已被解雇。大约在同一时间（北京时间18日凌晨4点），OpenAI发表了一篇博文。
  • 管理团队不久后就意识到了这一点，除了新任临时CEO米拉（Mira Murati）前一天晚上发现了此事。

北京时间11月18日早些时候的公开消息：

  • 当地时间周五，OpenAI宣布奥特曼将辞去首席执行官一职并离开董事会，由技术主管米拉接任，而且是立即生效。
  • OpenAI董事会在推文中表示，奥特曼的离职是董事会经过审议后得出的结论，由于他在与董事会的沟通中始终不坦诚，阻碍了董事会履行职责的能力，董事会不再相信他有能力继续领导OpenAI。
  • 奥特曼发文称：“我很喜欢在OpenAI的时光，这对我自己来说是革命性的，希望也能对这个世界有所变革。最重要的是，我喜欢与这些才华横溢的人共事。之后我还有更多详细的东西要说。”
  • 随后，OpenAI联合创始人兼总裁格雷格·布罗克曼 (Greg Brockman) 也在社交平台上宣布，在辞去董事会主席一职后，他决定完全退出公司，辞去OpenAI的所有职务。

北京时间11月19日消息：

  • 奥特曼告诉投资者，他计划推出一家新的AI芯片企业，旨在构建张量处理单元（TPU）的AI芯片，与英伟达产生竞争。目前，其新公司仍处于发展阶段，布罗克曼或将加入其中，而包括红杉资本在内的多位投资者已表达支持奥特曼的意愿。
  • OpenAI董事会正在与奥特曼讨论重返CEO岗位。他对回归感到“矛盾”，并希望对治理进行重大改变。
  • 纳德拉已与奥特曼交谈，确保他可以在当地时间周一回归OpenAI 工作。

北京时间11月20日消息：

  • 奥特曼以“访客”身份与前总裁布罗克曼在公司旧金山总部与高管们会面；
  • 后续，有消息称微软希望扶持奥特曼重新上位，并且考虑在 OpenAI新董事会中布下一个重要棋子；
  • 20日中午，OpenAI宣布奥特曼回归失败，董事会成员选择前 Twitch 首席执行官Emmett Shear担任临时首席执行官。由于奥特曼确定离开，OpenAI数十名员工也选择跟随辞职。
  • 20日下午，微软CEO纳德拉宣布奥特曼、布罗克曼及其同事们将加入微软。
  • 20日下午17点50左右开始，包括OpenAI CTO米拉·穆拉蒂（Mira Murati）等多位员工发统一推文称，“没有了员工，OpenAI什么都不是”文字，奥特曼转发了超过10条该内容并配以“心”的表情图。
  • 20日晚上，490名OpenAI员工集体写信抗议，威胁董事会要辞职，除非董事会辞职，并让奥特曼重新担任首席执行官。信中写道：“你们解雇奥特曼、将布罗克曼从董事会除名的过程已经危及了公司所有工作，破坏了我们的使命。你们的行为已经清楚地表明，你们没有能力监管OpenAI。”值得注意的是，这封信的签名者包括该公司董事会董事兼首席科学家伊利亚（Ilya Sutskever）伊利亚也在推特上对之前的罢免行为表示悔恨。
  • 20日晚，新任临时CEO希尔发推文表示，他的管理团队将调查董事会罢免奥特曼事件，因为此事影响了OpenAI公司的声誉。

北京时间11月21日消息：

  • 联名信人数已超过700人，占OpenAI员工总数约770名的95%。信中还称，董事会必须要答应员工的所有条件，否则员工将集体跳槽到微软。
  • 21日凌晨，微软CEO纳德拉对彭博电视台表示，微软对奥特曼的走向持“开放态度”，无论奥特曼加入微软还是重返OpenAI，奥特曼都将与微软合作。“无论Sam（奥特曼）在哪里，他都在与微软合作。”
  • 据The information报道，除了阿莫代伊之外，董事会在解雇奥特曼后还联系了其他高管担任临时首席执行官，其中包括 GitHub 首席执行官Nat Friedman，Scale AI首席执行官Alex Wang。最终，董事会于11月20日任命希尔为临时CEO。
  • 罢免事件之后，有超过100名OpenAI客户对奥特曼被罢免事件感到恐慌，联系了其竞争对手Anthropic、谷歌云和Cohere。据透露，许多OpenAI客户也在考虑转向微软Azure云服务，以提供OpenAI的GPT模型技术。OpenAI曾透露，目前有超过200万开发者和客户在ChatGPT API上进行开发。世界财富500强公司中，有92%的企业在使用其产品。
  • 21日上午消息称，OpenAI董事会曾就与其竞争对手、谷歌和亚马逊投资的 AI大模型开发商Anthropic两家公司合并的可能性进行潜在的谈判，希望Anthropic联合创始人兼CEO加入公司取代奥特曼，但最终失败了。
