在区块链修复社交媒体之前，必须有人先修复区块链。弗兰克·麦考特（Frank McCourt）为去中心化社交媒体未来的愿景投入了大量资金，他认为 Gavin Wood 可能是一个关键存在 。

亿万富翁弗兰克·麦考特 (Frank McCourt) 捐赠了 1 亿美元，帮助建立社交媒体巨头的去中心化替代方案，他选择了一个合作伙伴，让区块链在 Facebook 上发挥作用！ 这个合作伙伴就是以太坊联合创始人、Polkadot 创始人 Gavin Wood。

上周二，McCourt 的 Project Liberty 与 Web3 基金会的 Polkadot 达成合作。 Gavin Wood 离开以太坊后在 2020 年推出了 Polkadot。Project Liberty 有一个技术提案，允许用户在未来使用不同社交媒体服务时将他们的数据保留在区块链上。Gavin Wood 的参与是为了加速这个想法。

McCourt 在一份声明中说：“我们很高兴与 Gavin 及其同事合作，将我们对更健康网络的共同愿景变为现实。他们不仅是开创性的创新者，而且我们的价值观和如何以赋予用户权力、造福社会和加强民主的方式来重组互联网是一致的。”

对于那些希望通过去中心化来打击当前科技巨头、隐私甚至公民话语权的人来说，可拓展性和可靠性的问题是最大的绊脚石。

例如，Meta 的 Facebook 和 Instagram 通过将大家与他们的朋友联系起来、大家可以发布状态更新，从而喂养了其他内容和广告。这些服务根据他们从所有这些交互和连接中储存的信息，以更加精确的方式定制商业内容。Facebook 举报人弗朗西斯·豪根 (Frances Haugen) 等批评者表示，该模型会激励最具社会腐蚀性的内容，从而产生大量评论、点赞和分享。

McCourt、Gavin 和其他支持者认为这种情况对民主很危险。他们认为，如果公司无法控制数据，就会扰乱整个系统。

McCourt 也认为，赋予消费者权力的最佳方式是将他们的社交关系图谱放在公共的、去中心化的区块链上。这确保了公司控制的没有单一的数据存储；人们可以将他们的数据和数字关系从一个服务器带到另一个服务器。

Project Liberty 和 Polkadot 之间的合作伙伴关系，是在 Marseille 敲定的，然后前往达沃斯讨论他们的初步计划，但他们知道实现 Web3 需要的不仅仅是存储和转移用户信息的技术方法。还有一方面是，区块链在处理大型数据集时是非常缓慢和不优雅的。Wood 的 Polkadot 旨在连接多个区块链，并行利用它们并加速它们的交互。此次合作将 Wood 对 Web3 的愿景从金融领域扩展到社交网络。

“Web 3.0 需要一个普遍可访问的社交图谱才能成功，因为 Web 2.0 社交巨头目前对用户可以在其平台上说什么或做什么具有绝对控制权，” Wood 在一份声明中说。他补充说，他的团队希望“其他人能够加入我们的努力，为互联网创造一个更公平的基础。”

Wood 以帮助创建以太坊应对比特币区块链的局限性而闻名。Wood 和由 Vitalik Buterin 领导的七位联合创始人用自己的加密货币以太币创建了一个更广泛的生态系统，并为用户提供了创建和交换产品和服务的能力，包括 NFT。Wood 随后离开以太坊建立 Polkadot，他将其设想为一个多链“无需信任或无信任”的生态系统，避免“通过中介和机构的必要性”。

去中心化社交媒体的想法吸引了相当一部分不关心细节的小角色或大思想家，但它也有像 Jack Dorsey 这样的追随者，他发起了 Project Bluesky，其既定目标是“一个用于公开对话，开放且持久的去中心化协议。” 2018 年，W3C 发布了其 ActivityPub 协议，用于将社交媒体上常见的各种内容转移到去中心化网络上。

McCourt 是洛杉矶道奇队的前老板，也是乔治城大学的主要捐助者，他与该机构和巴黎政治学院建立了一个研究所，以推进他对更多道德技术的愿景。而 Gavin 现在通过他的基金会在资助对新的去中心化网络架构的研究。

Gavin 和 McCourt 的项目仍将面临其他挑战。去中心化的环境将需要吸引开发人员，他们的应用程序可以克服困难并变得足够流行以挑战当前的主导者，至少在总体上是这样。即便如此，用户仍有强烈的社交倾向去寻找或留在他们朋友所在的服务上，而市场一直青睐极少数特别强大的玩家。尽管 McCourt 渴望创新而不是规范，但成功的 Web3 社交媒体可能仍需要政策制定者来提高用户跨服务交流的能力，并且需要克服“去中心化”的模糊概念。

对于 Gavin 来说，McCourt 的合作关系强调了技术的重要性，他认为，“技术定义了政治的界限。技术定义了一切”，“我的基本信念是技术控制政治 —— 而不是相反”。
