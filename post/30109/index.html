Coinbase 和 FTX 有可能建立具有独特价值主张的 NFT 交易所。

随着 Coinbase 和 FTX 公布自己的 NFT 平台和交易服务，NFT 市场格局正在升温。 一般来说，中心化交易所（CEX）在寻求成为主要的 NFT 交易场所时，既有一些关键优势，也有一些劣势。

中心化交易所（CEX）的 NFT 竞争优势

鉴于其规模和资源，中心化交易所（CEX）具有一些关键优势。 其中一些优势将是决定这些现有庞然大物是否在 NFT 市场上取得成功的关键。

托管服务

资产托管是中心化交易所（CEX）的主要优势。 Coinbase 确保零售交易者不会错放他们的资产或因忘记密码而被锁定在帐户之外。 从本质上讲，CEX 有助于防止在转移加密货币时犯错。

更重要的是，Coinbase 和其他交易所将能够找到与高端 NFT 托管（例如 Punks、BAYC）及其 NFT 平台的协同效应。大多数高净值人士会选择在 Coinbase 托管而不是自行托管其高价值的 NFT。 Coinbase 将能够列出并让个人对存储在 Coinbase Custody 中的 NFT 进行投标。这将有助于为 Coinbase 带来 NFT 流动性，因为他们自然会在他们的平台上提供一些更高端的 NFT。

中心化交易所（CEX）可以利用其垂直集成服务（节点基础设施、抵押服务等）为其平台增加更多价值，这还有其他潜在好处，但是，托管似乎是最大的直接优势。

品牌与分销

大型交易所拥有庞大的品牌、可花费的资金和强大的现有受众。例如，Robinhood 将成为 Coinbase 的强大竞争对手，因为它们现有的用户群更大。同样，Coinbase 将与 OpenSea 和其他 NFT 平台竞争。 Coinbase 或 FTX 将其现有客户群的一小部分转化为 NFT 交易者的潜力将意味着数十万客户。作为参考，最大的 NFT 市场 OpenSea 已支持超过 500,000 个唯一地址（不一定是不同的交易者）的 NFT 交易。

除了分销之外，Coinbase 和 FTX 拥有大品牌，可以让他们找到最好的 NFT 艺术家和项目，并让他们在各自的平台上发布（投放）。这是中心化交易所能够引导流动性的另一种方式。

合作伙伴关系和知识产权

与 Coinbase 现有的用户群相结合，这家著名的美国交易所也将拥有更强大的与大品牌和影响者合作的能力。 FTX 已经获得了您最喜欢的所有品牌授权的冠名权，并且有传言说他们接下来要攻占您的高中足球场。 Coinbase 最近宣布与 WNBA 和 NBA 合作，将为篮球迷提供独特的内容、创新和体验。

像迪士尼这样的大型组织更有可能与 OpenSea 或 Coinbase 等老牌公司合作，这些公司可以提供合法合同而不是无许可协议。此外，中心化交易所将具有与现有知识产权合作并确保版税在其平台上传递的竞争优势。

潜在的缺点

无许可的 NFT 市场（即 Rarible、NFTX、Genie）仍将比其中心化竞争对手具有优势。

监管

总的来说，像 Coinbase 这样的美国交易所的主要劣势是监管。许多 NFT 类似于证券，或者可能会在消费品和金融产品之间徘徊。 这个 NFT 是否提供对 DAO 的治理能力，是否从交易量中收取品牌授权使用费？你直接购买的炫酷 NFT 从歌曲、内容或视频中赚取版税？拥有 NFT 可以获取项目发布的其他 NFT，这些 NFT 可能具有货币价值或需要交换资金才能购买？碎片化 NFT？。请注意，这可能适用于美国公司，而 FTX 或其他非以美国为中心的公司可能没有相同的限制。

资产广度

流动性可以说是任何市场最重要的品质。 币安最初获得市场份额是因为它提供了对未在其他地方列出的交易资产的访问权限，从而吸引了资金和用户（Binance 也拥有出色的用户体验和进入体验）。同样，Uniswap 获得了吸引力，因为它提供资产的速度甚至比像币安这样快速发展的交易所还要快。同样，无需许可的协议甚至其他交易所可能能够以更快的速度列出更多种类的 NFT 资产。再加上对特定 NFT 和无需许可的 NFT 协议的潜在监管担忧，无论是来自产生现金流的 NFT 还是碎片化 NFT，都可能会发现积累流动性的独特优势。

集成

以太坊的标志性特征之一是智能合约和软件的可组合性。 Alchemix 可以无缝地利用 Yearn 金库来产生收益，而 InstaDapp 能够促进一键式流程，将抵押品从一种借贷协议转移到另一种协议（例如 Aave 到 Compound）。这些类型的集成和二次应用程序推动了平台的流动性。与 Coinbase、FTX 或其他中心化 NFT 市场相比，无许可市场由于其开放性，将更容易与其他协议集成。

最后的想法

最终，Coinbase 和 FTX 有可能建立具有独特价值主张的强大 NFT 交易所。与 Coinbase 和 Uniswap 如何都能保持成功类似，Coinbase 和 OpenSea 或去中心化 NFT 市场也是如此。重要的是，像 Coinbase、OpenSea、Rarible、NFTX 等 NFT 交易所也在争夺不同区块链和各种资产的流动性。 NFT 即将在各个行业的增长意味着将有很多机会。
