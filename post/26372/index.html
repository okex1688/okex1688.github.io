前言：

本文翻译整理来源于区块链新闻网Sedaily，Newsis，Cryptonews，介绍了韩国交易所Korbit 为著名电视剧“文森佐”制作并出售NFT的情况。

韩国首个虚拟资产交易所Korbit 开始为热门的韩国电视剧制作并上架售卖NFT(Non-Fungible Token，非同质化代币)。近日，Korbit表示，将销售韩国最大的电视剧工作室Studio Dragon所出品的电视剧《文森佐》的NFT。该电视剧由韩国著名演员宋仲基主演，是一部以黑手党为主题的电视剧。

Korbit决定通过运营中的NFT Market上架售卖今年5月完结的tvN电视剧《文森佐》中出现的刻有卡萨诺图案的打火机NFT周边，这个打火机是剧中男主角使用的标志性道具。想要购买 NFT 的用户可以访问预申请页面并输入个人信息。之后，Korbit 将向预注册者发送官方 NFT 售卖通知短信，并将向前 100 人售卖 NFT。

CJ娱乐公司旗下的Studio Dragon是制作《德鲁纳酒店》、《爱的迫降》、《文森佐》等多部爆款电视剧的工作室。《文森佐》于今年2 月在韩国开始播出，并在全球在线视频服务（OTT）上同步播出。Studio Dragon 选择了 《文森佐》 的“Casano Pattern Writer”作为其第一个NFT 。在大多数情况下，NFT 仅将作品的所有权授予以投标方式出价最高的人。不过，Korbit NFT Market的卡萨诺花纹打火机已改为限量100支，并且价格合理，能让更多的喜爱这部电视剧的观众感受到剧中的余辉。

Korbit 于今年 5 月首次在韩国虚拟资产交易所开设了 NFT 交易市场，并一直在寻求与拥有优质知识产权 (IP) 的 NFT 运营商合作，寻找高质量的 NFT 内容。Studio Dragon 正在考虑各种方式来利用其 180 个优质 IP。两家公司的需求不谋而合，电视剧的NFT数字商品项目由此拉开序幕。

Korbit的 CEO Oh Se-jin 表示：“很高兴能够销售 Studio Dragon 制作精良的电视剧 NFT，最近消费者不仅单纯地消费电视剧内容，还会消费电视剧衍生出的周边等，因此我们希望将通过Kobit NFT市场，为包括Studio Dragon在内的拥有优质IP的企业最大限度的提供为IP创造价值的最佳交易环境。以此次销售为开端，未来，Korbit NFT Market 将继续为 Studio Dragon 制作的其他热门电视剧相关商品制作 NFT 产品并呈现给观众。”
NFT 是一种加密技术，将数字图片或音乐记录在区块链上，并赋予每个内容一个独特的标记。目前，NFT 在艺术领域的应用最为活跃。由于交易记录是自动保存的，不可伪造或篡改，因此可以准确识别特定资产的所有者，并包含作品的作者是谁、作品的时间和出售对象等所有详细信息。它的另一个特点是在转售作品时不断向原作者支付版税的结构。
近年来，韩国的NFT 市场正在快速扩大，NFT 市场分析公司 Nonfusible称，今年第一季度全球 NFT 交易额达到 20 亿美元（约合 22364 亿韩元）。2018年为4,090万美元，2019年为1.4亿美元的NFT市场规模在去年增加到3.4亿美元（约3800亿韩元）后，今年进一步增长。

NFT是将数码图片或音乐等记录在区块链上，赋予各内容固有的标识的加密技术。目前，NFT在美术品领域最为活跃。由于交易记录自动储存，无法伪造和篡改，因此不仅能够准确掌握特定资产的所有者，还能包含该作品的原作者是谁、何时卖给哪些人的详细信息。另外，每当作品重新出售时，持续向原著者支付专利费也是一大特点。 
NFT市场正在迅速扩大。NFT市场分析企业NONFUNJABLE表示，今年第一季度全球NFT交易量达到20亿美元(约22364亿韩元)。2018年为4,090万美元，2019年为1.4亿美元的NFT市场规模在去年增加到3.4亿美元（约3800亿韩元）后，今年进一步增长。

结语

随着NFT市场的增长，越来越多的领域开始进入这个火爆的市场，目前韩国的NFT领域大多集中在美术、游戏、音乐行业，现在影视行业也瞄准了这一具有前景的市场。韩剧在全世界都有着很大的影响力，可以说是韩剧打开了众多国家迎接韩流的大门，韩剧的剧迷们也非常愿意购买自己喜爱的电视剧的周边，可见这个消费市场的规模是相当庞大的。拥有众多影视IP的公司无疑将成为NFT领域中最有发展前景的黑马。Korbit这次与CJ娱乐公司的合作开启了影视行业成功跨界区块链行业的重要先河，期待日后影视行业能生产出更多优秀的NFT作品，带动韩国的NFT市场快速成长。
