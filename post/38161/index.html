加密资产市场在过去一年的快速增长离不开传统支付巨头们的助攻。

拥有 3.77 亿用户的 Paypal 已经为美国和英国的用户开放了交易和持有比特币（BTC）、以太坊（ETH）、比特现金（BCH）和莱特币（LTC）功能，同时支持用户使用这些加密资产进行支付；Visa 和 Mastercard 则在全球多个洲发行了加密信用卡和借记卡，使比特币和以太坊在购买商品和服务方面变得更加容易。

很显然，这几个支付巨头“迷上”了加密资产作为支付手段的点对点、更隐私、更快的交易和更低的手续费等优势。

然而，由 Twitter 前首席执行官 Jack Dorsey 领导的 Square 却另辟蹊径，先大举购买 8027 枚比特币，现价值约 2.4 亿美元，又为其拥有 4000 万用户的拳头产品 Cash APP 整合了加密资产交易功能。后来甚至直接将公司更名为 Block（区块），主力开发比特币产品矩阵，将重心放在比特币上。

「持有比特币并为它赋能」成为了 Block 的宗旨，「比特币是最好的互联网原生货币」成为了 Block 的奋斗目标。

以下内容翻译自 Block 2022 投资者大会的演讲，一起听听 Jack Dorsey 如何带领 Block 公司在比特币生态系统中掘金。

Jack Dorsey：

我在开幕上说过，我认为比特币是全球货币传输开发标准的一个非凡趋势，这意味着什么呢？

我们成立 Block（原Square）这家公司的原因是帮助人们进入金融系统，作为从信用卡向前迈进的一步，也是从面对面交易转向数字商务的一步。虽然信用卡更加全球化，但它仍然是排他性的，由少数公司控制。

可以肯定的是，这个星球上的每个人都会以某种方式进入互联网，但能否参与全球金融系统却不然。为了实现这一点，我们需要一个原生于互联网的协议，一个具有包容性、可信赖且可扩展到整个人类的标准。

你可能听说过区块链和加密资产就是这样做的，但是对于存储和价值交换这样重要的事情，我们必须要求系统具有某些属性——开放的、安全的和可扩展的。开放可以确保系统在其开发和运营中都具有最大程度的透明性，并且任何希望使用的人都可以访问。安全意味着个人掌握解锁其资产的密钥，并且它能够抵御来自其他个人、公司的所有形式的攻击。可扩展意味着这些属性会随着越来越多的人使用它而变得更强大，同时最大限度地提高效率和可用性。

互联网需要一个原生货币，并且这个货币需要有完整的技术生态系统，比特币是目前最适合的解决方案。

为什么我们选择比特币作为公司的重点呢？专注于比特币，迫使我们改变了构建方式，我们必须同样开放的寻找加强它的开放性、安全性和可扩展性的每一个机会。

我们第一次真正推动这一点是创建了一个开源工程师团队 Spiral，现在他们正在专注于比特币的闪电网络。在团队建立的两年后，他们推出了 LDK（Lighting Decelopment Kit，闪电开发套件），这是一个软件开发工具包，可以帮助开发人员以最简单和最快的方式使用闪电网络创建应用程序。

虽然 Block 公司不会使用 Spiral 的工具包，但我们这么做的目的只是想让比特币网络更加强大。但是在对比特币所有技术进行了详尽的探索之后，我们的支付应用程序 Cash APP 希望整合比特币的闪电网络支付，使用了 Spiral 开发的 LDK，因为它的技术卓越。这是一个成功的故事，我为在比特币生态系统的构建感到自豪，从长远来看，我们的行为也在帮助我们公司。

所以我们希望继续以这种方式填补比特币生态系统的其他空白，例如我们正在研发个人硬件钱包，以增加用户对比特币的访问；比特币挖矿系统，以进一步分散和保护网络；TBD，一个中心化身份系统，在此基础上的 tbDEX 协议是一个没有中心化中介的可信交易所，能够成为连接比特币和法定货币的桥梁。

所有这些产品都是在比特币生态系统中开发，默认情况下都是开源的，并且产品之间都有互补的商业模式。也就是说，Block 公司正在为新客户构建一个围绕比特币的“宇宙”。

我们希望比特币可以成为互联网的标准，并通过帮助它，成为我们帮助人们进入全球金融体系的新机会。

<开发者工具>Spiral 领导人，Steve Lee：

我们在 2019 年创立了 Spiral，这是一个不受 Block 因商业利益而指导和干预的小型独立团队。相反，你可以将更广泛的比特币生态系统视为我们的“客户”。

我们相信比特币不仅仅是一种投资，它更是我们理想中的“最佳货币”。因此，我们建立和资助旨在改善比特币用户体验、安全、隐私和可扩展的免费开源项目。

比特币的价值与 Block 公司的金融赋权使命一致。Block 的核心愿景之一是让世界上的任何人都应该有能力、轻松的参与金融系统，而比特币可以通过让世界各地的人更高效地支付来实现这一愿景。

然而要实现它并不容易。Spiral 已经确定了比特币生态系统中的几个差距：

首先，比特币开发人员可用的开发工具一直很少，而且体验感很差。

其次，在设计方面，对用户需求的关注度不高。

对此，我们提出了两个计划来解决：LDK 和比特币设计社区。

比特币设计社区是一个开放的比特币开发者社区，成员可以提出新点子，大家共同付出实践。

而我们的团队成员正在专注于 LDK，使越来越多的开发人员更容易地在闪电网络上构建出色的应用程序。20 年前，一个开发团队可能需要数年时间才能开发出一款应用程序，今天， 开发者工具可以让一个人在几周内就可以创造出令人信服的应用程序。这就是 LDK 为比特币生态系统所做的事情，我们已经看到 LDK 被流行的应用程序所采用，例如 Blue Wallet。

那么我们的工作将如何影响 Block 公司呢？

正如 Jack 刚刚提到的，Cash APP 使用 LDK 集成了比特币闪电网络，用户可以即时发送、接收和交易比特币，集成的时间只有几个月。所以有了 LDK，Block 公司可以更快的将闪电网络集成到产品中，节省了大量的时间和资源。

此外，Spiral 开发基金已经资助了超过 15 个国家和地区的 40 个比特币项目开发者，可以说，这是所有比特币开发基金中最具影响力的。

我们还在研究女性如何看待和使用比特币，以确保所有人都可以使用比特币。

建立信誉并帮助扩展生态系统，我们在影响比特币的未来。 Spiral 将继续促进整个世界对比特币的采用，让其最终成为“互联网货币”。

<钱包+采矿>硬件领导人，Jesse Dorogusker：

钱包篇：

比特币的资产所有权由密钥定义，如果你没有密钥，你就没有资产。 在去中心化的未来，全球金融基础设施都将建立在这个简单的想法之上。

如今，全球估计有 1.75 亿比特币持有者，其中很多位于新兴经济体的人实际都没有密钥，而是依赖于代表他们持有密钥的托管机构，就像如今大多数人通过银行管理资金的方式。

选择托管服务的人是为了方便，担心密钥的意外丢失。但加密资产交易平台、钱包等托管服务商并不全是所有人都可以信赖的托管人。

一个值得信任的托管服务取决于：

- 什么时候可以取钱？

- 你的资金如何得到保障？

- 你是否可以随时访问他们的服务？

如今，保存好自己的密钥并不像听起来那么简单。早期比特币用户使用复杂的应用程序或硬件设备，技术含量极高，难以使用。而且，如果你不小心丢失手机或者硬件钱包，通常唯一的找回方法是依靠客户的 12 或 24 个私密短语，它们要么会在很长一段时间内丢失，要么把它写在便利贴上以防忘记。即使是如今对加密资产比较熟练的用户也不得不屏住呼吸，希望他们不会犯错误而导致意外损失，或者成为黑客的受害者。

Block 公司的机会是为普通人持有和管理比特币建立一种安全、简单的方式，以让更多人能够拥有比特币所有权。

我们正在构建一个钱包，将功能分为三个部分：

- 手机应用程序：用户用来管理资金的钱包部分，使他们能够利用智能手机进行更频繁的活动，例如发送、付款和接收。如今全球有 30 亿人以上都拥有智能手机，通过手机管理金融服务是非常吸引人的，所以我们在不断打磨产品体验。

- 安全硬件设备：作为大宗交易的第二层保护，对于较大的、不频繁的用户来说，安全更进一步。如果你丢失或更换手机，硬件设备还可以让用户将密钥安全地引入新手机中。

自助恢复工具：可在用户丢失部分密钥时提供帮助。

自我管理的友好型钱包对于尚未参与但肯定会参与比特币的主流客户来说，是一个更简单的选择。

我们正在公开开发此产品，并且展示出了我们的代码，任何人都可以检查和验证，以赢得比特币社区的信任。

采矿篇：

在传统金融体系中，银行被信任以保护客户的资产。而比特币是分布式的账本，通过比特币矿工贡献计算换取经济回报，以确保交易的安全结算，无需信任。任何个人和企业都可以使用具有计算能力的采矿计算机参与其中。

但是如今参与采矿的门槛非常高，矿机难以获得维护和运营，而且成本高昂。因此，采矿非常集中，该领域也是排他性的。

我们相信我们可以建立高性能的采矿组件和一个易于使用的成本效益系统，使更多的矿工参与，从而形成一个更高效和更分布式的网络。

当我们观察比特币采矿的当前格局时，我们可以看到几个改进它并使其更容易获得核心的机会。

专业的采矿计算机背后的技术是一种专用集成电路（ASIC），针对最低功耗和最低成本的比特币计算进行了优化。而如今比特币 ASIC 的制造和销售集中在少数几家公司，这种集中引入了供应和成本风险。

所以我们看到了构建自己的比特币 ASIC 的机会，它针对成本、性能和智能系统集成进行了优化，简单易用，并且将与参考数据表和配套软件一起出售，以使其适用于其他区块链网络。

如今，所有开源采矿系统中存在相同的市场集中度，包括计算、电源、管理、冷却和产品，这显著的提高了少数矿工的计算能力，因此采矿本身也是高度集中的。哈希率是比特币网络中每秒计算量的衡量标准，根据麻省理工学院和伦敦经济学院最近的一项研究，当前哈希率的 50%，掌握在前 0.1%的矿工手中，它们仅仅是 50 家公司。

通过运用我们在采矿系统设计、开发和 Block 公司的全球供应链方面的专业知识，使我们的采矿系统更容易获得成功。

我们还看到，目前采矿系统尚未通过智能系统集成对清洁能源的使用进行优化，特别是太阳能和其他的可再生能源，我们可以通过正在构建的采矿系统来提高清洁能源的使用效率。

我们还在探索创造一种完全依赖清洁能源的分布式采矿云服务。我们相信有些消费者和企业对采矿感兴趣，但不想承担自己维护矿机的负担。所以我们将构建一种“客户购买远程矿机，并通过 Block 公司连接到清洁能源”的模式。这将允许客户在更公平的竞争环境中与对能源和设备拥有更多控制权和访问权的大公司共同参与比特币采矿。

解决所有这些缺点，这将创建一个更持久和更去中心化的比特币生态系统。

对于 Block 公司来说，这也是一个可观的商机。我们预计随着比特币网络需求的增加，与采矿相关的资本支出将达到数十亿美元；比特币采矿市场巨大，采矿总收入从 2020 年的约 50 亿增长到了 2021 年的约 170 亿美元。随着采矿收入的增长，矿工将继续寻求各种途径来占领这个不断增长的市场的一部分。所以我们相信 Block 公司可以运用其硬件产品专业知识和设计理念在这一领域产生重大影响。

然而，采矿需要大量资本投资，收入状况却不完全可预测，我们可以通过借贷来降低准入门槛，帮助他们畅通现金流。最终，矿工可以将他们的采矿奖励存储在我们的比特币钱包和 Cash APP 中，便于他们随时提取。

我们很高兴有机会使比特币挖矿更高效和更分散，这就是我们在 Block 公司进行的比特币采矿工作。

<去中心化身份+交易>TBD 负责人，Mike Brock：

在 Block 公司，我们十年来一直在建立能够人们与金融联系起来的支付系统。通过 Square Reader（POS机），我们将数以百万计的小企业连接到支付系统，使他们能够通过 Cash APP 在日益无现金的社会中进行销售。可以说，我们在增加金融接入方面取得了长足进步。

但我们的努力受到传统金融体系的限制，如今全球仍有超过 10 亿人无法使用银行账户，在这其中的三分之二拥有互联网连接的智能手机，但为什么我们如法连接到他们？互联网和无线技术为我们提供了需要的信息基础设施，但资金和支付却没有跟上步伐，它们仍是旧模式、慢的、支付系统。

在新的数字经济时代，我们相信像比特币这样的技术可以从根本上改善全球金融体系，所以现在我们拥有了传统的支付系统和新的支付系统，我们如何从旧的过渡到新的？这就是 TBD 的使命，通过简化 API 和 SDK 来连接这两种支付系统。

我们建立了一家新的开源公司，专注于构建开放协议、开放标准和开放的开发者社区，以创建一个工具生态系统，所有参与者都可以从个人、企业、机构甚至政府中受益。

可以简单理解为：Red Hat 专注于 Linux，Mulesoft 专注于 SaaS，Databricks 专注于数据科学，Elastic 专注于搜索引擎，而 TBD 专注于去中心化身份系统。

互联网的巨大力量和潜力一直是信息访问的民主化，这种力量释放了一股创新的洪流，从根本上改变了我们的世界。但互联网的账户模式打破了我们验证和验证自己的方式，互联网实际上变得越来越中心化。

我们在互联网上的身份由少数几家非常大的公司代表，这种模式服务于平台的利益，在他们必须给予的隐私和控制方面不符合用户的利益。我们需要解决这个问题，去中心化身份是建立更具包容性的数字经济的答案。

身份验证是获得金融边缘化的访问权限的最大棘手问题之一，在美国的新移民中，即使是那些受过良好教育的人，没有信用记录，第一次尝试获得金融服务通常是“痛苦”的。

然而，金融服务与信用档案有关不是很奇怪吗？我们如何验证身份真的有意义吗？

这个问题的范围是巨大的，让全世界 11 亿人无法证明他们是谁，这也是可笑的。

去中心化身份将你在传统平台上的账号密码替换为你个人所拥有的 Token，合法合规，保护隐私。该 Token 可在任何服务类基础设施中使用，例如用于 KYC、驾驶执照或任何类似的证明。

这不仅与个人有关，对于无法解决问题的企业也很有意义，由于使用安全性较低的身份数据系统给用户所带来的风险，这些企业最终可能会选择使用更安全的去中心化身份系统，这将使企业能够建立更简化的客户体验，降低成本，并最终为他们自己受益。

以去中心化身份系统为“地基”，Block 公司就拥有了构建真正去中心化协议和去中心化网络的基础，例如用于将传统货币世界连接到加密资产世界，这就是 tbDEX 所要做的。

tbDEX 将互联网的信息民主化、比特币和去中心化身份系统这三个要素整合在一起，用户可以以一种开放的、无需许可和去中心化的方式进出加密资产世界，它将有效的降低每个人的交易成本，同时可以让世界各地的人安全的、以合规的方式转移资金，提高加密资产世界的可访问性和包容性。

当去中心化身份系统大规模应用时，我们就有了重新发明整个金融基础设施的可能性，更接近于我们每个人都希望获得的一键式体验。 谁不喜欢自己的钱包创建在自己的身份之上，并最终直接从“旧钱”到“新钱”呢？

相信大家已经理解了这些抽象的概念，但对我们来说，去中心化身份系统归结为一个重要的目的，那就是「金融系统的可访问性」。
