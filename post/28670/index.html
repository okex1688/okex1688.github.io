Facebook正在其Novi–Diem系统中探索NFT功能，但这种科技方案如何与现有的平台相匹配呢？

随着最大的品牌、企业和个体希望在创建自己的数字收藏品方面获得先发优势，NFT在娱乐、媒体和体育领域迅速广泛使用。

这些有多种形式，就像区块链技术被吹捧为可以应对任何行业的需求和问题一样，对于在世界各地铸造和交易的NFT，想象力是唯一的限制。

社交媒体使人们的联系比以往任何时候都更加紧密，这些不同的平台对向不同观众介绍和传播NFT发挥着重要作用。

然而，像Facebook和Twitter这样的行业领先巨头还没有完全将区块链技术纳入其基础系统的内部结构。用户可以发布令人兴奋的推出新NFT的信息，但终端用户不能点击链接转至数字收藏品所在平台，无法购买或交易这些项目。

尤其是Facebook，在过去两年中受到了大量监管审查，因为它计划的加密货币钱包和原始代币以及全球支付系统对金融领域产生了深远的影响。 以它约29亿用户群的基础，这家社交媒体巨头很可能会影响数亿人每天交易的方式。

随着其Novi钱包和Diem加密货币临近正式发布，公司也没有忽略火爆异常的NFT。Facebook金融主管David Marcus在8月承认说Novi钱包主要是帮助其用户获得NFT支持。

NFT越来越受欢迎，Facebook等社交媒体平台正在迅速采取行动跟上这个新兴领域的发展。还有待观察的是由Facebook驱动的NFT会是什么样子，以及该公司进入这个领域是否会对社交媒体巨头和加密行业互惠互利。

融合在所难免

行业参与者似乎有一个共识，即NFT与社交媒体平台的融合只是时间问题。Golem Network的董事会顾问María Paula Fernandez告诉Cointelegraph，这既有积极的一面，也有消极的一面：

“我对Facebook加入NFT的想法感到不安，NFT市场在很大程度上是交易性的和利益驱动的。我会更关心与Facebook融合的其他web3 APP。”

NFT多链基础设施平台Unifty的首席培训官兼创始人Markus Bopp也谈到了这个问题，他告诉Cointelegraph，对社交媒体平台来说，这个新兴领域可能会是有趣的使用案例。

除了数字收藏品的交易之外，Bopp认为NFT的功能可以为各种社交媒体平台提供一些棘手问题的解决方案，例如身份证明和验证需求：

“NFT系统可以毫不费力地解决这个问题，还可以结合个人资料图片。但身份证明只是一件很小的事情。NFT真正擅长的是游戏化和增值所有权。社交媒体对象将能够使用这些强大的工具来调查无数新的发展可能。”

随着NFT与社交媒体平台的融合有望成为现实，下一个考虑的是这是否会对更广泛的加密货币领域和区块链创新的运用产生根本性影响。

Bopp认为，最大的社交媒体平台的影响力对于推动加密货币的使用非常重要，尤其是当功能变得更广时：“用户对加密货币大量运用可能达到他甚至不知道他们正在使用区块链的程度。”

Fernandez提出了另一种看法，她强调了Facebook计划Diem时面临的困难，最初Diem被命名为Libra，然后被各种监管机构拦截。在那段时间里，NFT得到了广泛运用：“Facebook已经尝试过与加密货币融合，但失败了。尽管如此，没有Facebook，加密货币、区块链和NFT已经成为了主流。” 她补充说：“我们正在建立一个不需要 Facebook开发平台就能覆盖全世界主流受众的行业。”

与Facebook融合的NFT会是什么样子？

Libra基金会正在持续推出 Novi钱包和Diem加密货币，除非主要监管执法，否则该平台应该会将Facebook的用户群引入原生数字钱包，该钱包最初将提供代表主权货币（如美元、英镑、欧元）的稳定币。

目前尚不清楚Diem钱包是否以及何时会包含对比特币 ( BTC ) 和以太币 ( ETH ) 等的支持。这意味着它自己的NFT平台很可能会直接融合到Diem区块链中。可能不支持与ETH等去中心化代币的互通性，因此需要连接Novi钱包自己的NFT系统。

Bopp认可Novi钱包和Diem加密货币将成为支持NFT的重要组成部分，因为“在这个水平上的运用对加密领域来说非常棒”，但对于一些用户交易和更广领域的不同影响来说：

“Facebook 很可能会对NFT进行自己的调整，就像他们在大多数事情上所做的那样。我们应该期待所有权更改方案，因为他们试图单独为他们的用户群收集技术。这可能在很大程度上对他们有用，但同时它会推动发现更多公开的和去中心化的资源。”

在Facebook驱动的NFT由Novi钱包和Diem加密货币提供支持的假设前提下，Fernandez认为中心化系统无法与以太坊区块链相媲美，后者创造了智能合约功能和去中心化APP开发：“社区、游戏、赋予NFT属性的可定制智能合约、围绕某些NFT构建的系统、金融工具（如部分所有权和流动性池）。我认为Facebook 无法超越这一点。”

普及NFT的加密社区关键

鉴于不同司法管辖区的持续监管考虑，Novi和Diem的发布日期仍未确定，Facebook用户要实现购买在其订阅源中弹出的NFT还有很长的路要走。

与此同时，由构建基础设施并交易代币为其系统提供动力的社区驱动着，去中心化的NFT市场正在蓬勃发展。Fernandez提出了一个尖锐的分离想法，他告诉Cointelegraph，Facebook没有一个可以与以太坊相比的社区：

“NFT不仅是文化物品或收藏品——它们还是金融工具。以太坊社区已经掌握了构建金融工具的艺术，并且现已在其中融合了NFT。Facebook可以随心所欲地出售一堆jpeg，但如果它的交易层不在以太坊上，就构不成竞争。”
