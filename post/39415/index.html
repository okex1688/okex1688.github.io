回顾过去

在展望未来之前，先让我们回顾一下过去的一年吧。在过去的一年中，飞速发展的 NFT 市场主要有着以下焦点：

基于 FOMO、FUD、关注度与大 V 的价格投机

与由官方实体支持的法币或由商业基本面支持的股票相比，加密货币/NFT 完全依赖于市场情绪。因此，加密货币/NFT 是世界上最不稳定的资产之一。

被市场情绪驱动的比特币

情绪驱动的不稳定性在 NFT 市场尤甚。OpenSea、 LooksRare 等 NFT 交易平台不断向我们展示一周赚十倍、百倍从而改变人生的暴富神话，他们已然成了「数字赌场」。「投机推文」拥有点石成金的能力，它们能让某个 NFT 系列的价值快速翻倍，至于其真假与否，并不重要。

YugaLabs 联合创始人 @GordonGoner 说他身旁的这个男人开始发出「哥布林」的声音，goblintown.wtf 的地板价瞬间起飞

NFT 在很大程度上是一种「注意力经济」——某些大 V 的每条推文都有极高的关注度，这种力量在 NFT 市场中强大得让人难以置信。有些大 V 会负责任地使用这种力量，而有些则不会。

Meme + CC0

Meme 是文化的一种体现。它难以形容，但每个人都懂——简直与 NFT 是天作之合。

在「Kevin」Meme 最火的时候，下面这个玩意卖出了 8.88ETH(2.64 万美元) 的价格。Meme 文化会永存，但组成这永存文化的一个个独立符号并不一定，它们可能将被遗忘。

8.88ETH 的 Kevin

大多数 Meme 项目也都是 CC0 项目，如 CrypToadz、mfers、goblintown.wtf…。作为这些项目持有者的你甚至都不拥有这些艺术品，但无所谓，这些项目的价值是 Meme 与社区氛围而不是所有权。这也令人悲观——创始人可以不负责任地打一枪换一个地方，快速赚钱并持续得到版税收入。不幸的是，我们的注意力常会转移到项目的价格波动等方面而把这一点抛之脑后。

以 NFT 为载体的 Meme 还会不断浮现。能够从了解某个 Meme 文化中获取收益，这在过去是难以想象的。其实，在 10K NFT 系列流行起来之前，Meme 就已经被货币化了——你，是一个重要文化时刻的所有者。而 Meme 也早已成为 NFT 可收藏性中的重要元素之一。

不需要 Web3 的产品

这可能是最有争议的一点。

许多 NFT 项目作为「Alpha」的邮件列表或付费订阅服务会更好。大多数 P2E 游戏首先应该是个游戏，再考虑将 NFT 引入游戏中。

如果一个 NFT 项目将「技术」放在首位，需要攻克各种技术开发难题以实现其愿景，那么该项目将高度依赖炒作与版税收入。大多数只筹集了几万至几十万美元资金的项目难以兑现它们写在路线图中的承诺。

路线图必须符合实际。愿景是必要的，但必须有明确的规划与强有力的执行，才能将愿景全盘变为现实。不少 NFT 项目声称它们将「打造下一个迪斯尼」，但切记，数十年的努力与超过 20 万名的员工，才让迪斯尼成长为如今这般模样。

每个 NFT 项目都相当于一家初创公司，这家初创公司得有一个目标客户群，得有一个营销渠道，得有一个产品要卖——这家公司得赚钱。NFT 应该是这家公司成长的催化剂，而不该是这家公司唯一的商业模式。

到此，您应该了解了一些 NFT 市场的现状。那么，在未来半年里，我们会看到 NFT 在应用方面出现什么样的变化呢？

展望未来

IP 许可

当您购买了一个赋予您完整商业权利的 NFT，尤其是它属于某个 10K NFT 系列时，您实际上购买了一份 IP 许可，与一个现有的社区一起创造各种商品（咖啡豆、酒、帽子、桌游…）。这个社区中的其他人与您一样，为这个 IP 的现在与未来感到兴奋。

当这样的情况真正出现时，您就不再是单纯地对 NFT 的价格玩着猜大小的游戏，而是对 NFT 进行了价值创造，就像利用一块商业地产来壮大您的生意。这笔投资的对象将不再是 NFT，而是您自身，您可以控制这笔投资的财务收益。

与此同时，品牌代理商们的商机也将出现。品牌代理商拥有围绕 NFT IP 创造必要的品牌资产与故事的技能与网络。NFT 持有者可通过将其 IP 授权给品牌代理商来获取被动收入，不再只能仅凭自己的力量来做庞大的运营工作为 IP 输入价值。

社区/会员权益

社区是不能分叉的。在 NFT 出现前，许多不同的行业已经为我们展示了独家社区的重要性——法拉利只允许当前的法拉利车主购买他们的下一个系列车型，如果这种会员独享权变成了可转让的 NFT，会发生什么？迪士尼的 33 号俱乐部允许会员进入独家餐厅并向会员发放许多免费的主题乐园门票，如果这种会员独享权变成了可出租的 NFT，又会发生什么？

会员制（铸造费用）促使团队为会员（NFT 持有者）带来各种各样的实用权益。拥有一个组织良好的社区的商业网络是无价的，甚至超越了商业网络这个概念本身。设想一些有趣的案例吧？通过铸造 NFT 聚集起一帮支持某项慈善事业的人，组建起一个捐助者社区，让这项慈善事业可持续发展下去。以 NFT 作为高中社团的参与证明，并在大学录取时起到验证作用… 诸如此类有趣的用例，还有很多。

NFT 是将用户反馈、资本与市场营销融为一体的妙招。如今，许多 NFT 正与实际软件或其他商业产品一起发行。但是，NFT 也可以（并且应该）在已发布的产品上发挥作用。通过发行 NFT 以解锁高级功能，您可以精确定位您的优质客户，并筹集到额外的资金。例如，Netflix 可以发行 100 张终身会员证，激发市场热情的同时弄清楚优质客户想要拥有的独家权益到底都有些什么。

当然，如果以这样的方式去发行 NFT，就意味着将牺牲长期利润以换取短期的狂热。为这样的 NFT 支付了 X 美元的投资者，获得的权益价值应超过 X 美元。

结语

说一千道一万，我们最终要问的，也是最重要的问题——除了通过炒作获取利润外，我们买卖一个 NFT，到底是为了什么？

对于卖方（发行方）而言，发行一个 NFT 系列是为了引起轰动、筹集资金并组织起一个真正关心产品的社区。

对于买方（投资者）而言，购买一个 NFT 是为了获得独有的商业网络访问权，以及各种优质的产品权益。

当然，您也许会提到映射现实房地产的 NFT 凭证与灵魂绑定 NFT，但它们依然存在于遥远的未来。
