Key Takeaway

  • 近日，BAYC、MAYC、ApeCoin等Yuga Labs旗下项目都有较大程度涨幅，为Otherside营造了不错的市场开局。Otherside中的Koda 3D PFP 将有潜力成为新蓝筹。

  • Otherside 具有独立SDK软件的辅助，提升了MetaRPG内部的自定义想象空间。着眼于将基础设施，经济模型以及跨游戏间布局，未来Otherside或将串联其他各种不同的元宇宙产品，打造 NFT 世界的“公链”。

  • 与Animoca和OpenSea的合作将会为Yuga Labs的元宇宙和游戏布局带来加成。此外，Otherside也在重点关注土地再利用价值和实用性，而不是将土地作为闲置资产。

  • Yuga Labs拥有的头部优质Collection占据整个NFT市场超过20%的市值，这些IP的话语权将作为Yuga Labs的武器，构筑一个基于身份认同的世界。

市场节奏和新蓝筹

从去年4月份发布至今，BAYC已走过整整一年。在此期间，Yuga Labs一直把握着市场节奏，不断突破人们的预期。通过多次创新的尝试，不仅满足了明星对“Web3社交新贵”身份的需求，也给其他普通持有者带来巨大的红利：会员特权、社区周边福利、知名潮流品牌联动等。社区从不同人群汲取共识，最终形成了核心精神——开放与共创。

社区文化传播展示

在BAYC取得阶段性成功后，Yuga Labs开始了它的版图扩张，在三月初成功收购CryptoPunks和Meebits系列IP，紧随其后发行了其用于MetaRPG经济系统代币ApeCoin，为Otherside发布进行了一系列铺垫和预热。

2022年3月19日，Yuga Labs在官方推特中公布了 Otherside 预告片。伴随着话题热度的上升，BAYC、MAYC、ApeCoin价格也在不断飙升。自3月19日Otherside宣传片发布以来，BAYC地板价从100ETH涨至145ETH，涨幅达45%。MAYC价格涨幅更大，地板价从19ETH涨至39ETH，涨幅达100%。

BAYC价格走势图（3.19-4.29）；

同样，ApeCoin从3月17发行后便不断上涨，并且每次官方公布Otherside相关细节时，ApeCoin的价格都会有短时的联动效应，截至4月29日，Otherside官推宣布每块地以固定价格305ApeCoin的发售方式后，ApeCoin也随之创下历史新高突破$27。

与之联动的ApeCoin的价格，来源：AICoin

结合首次发售土地流通量和ApeCoin价格进行粗略计算，本次Otherside的发行结束时，将达到接近7.5亿美金市值，这将为之后的MetaRPG的布局打下经济基础。

从2D到3D

在Otherside发布的预告片中出现了一只外星生物，名叫“Koda”，本次将会有10000只Koda随机投放到Otherside地块中。在影片剧情中，Koda带着Bored Ape进入了Otherside，并见到了CryptoPunks、Meebits等角色，这可能暗示了未来项目运营的一些走向：BAYC会在Koda的指引下，在Otherside中带领着其它NFT IP走向成功。

Koda资料图

根据官方图，我们看到Koda与Yuga Labs的系列NFT都不同，只有它是以3D整身的形象出现，所以可以推测，Koda是Yuga Labs打开Metaverse的钥匙，同时意味着“2D→3D”的转变。而作为Yuga Labs的第一个3D NFT，Koda将有可能成为新的蓝筹。

更具拓展性Metaverse

代币模型和土地应用对比

元宇宙所囊括和代表其实是一个生态，而不是割裂的独立的游戏孤岛。代币则是岛屿之间的突破口和连接点。在元宇宙概念代币市值上，目前，$APE 排名第一，相比于排名第二的$SAND，$APE总市值一度接近两倍于$SAND，某种程度上说明了目前市场人们对于ApeCoin未来的预期。

代币是元宇宙经济治理的关键因素，不同代币设计模型将会影响用户的交易和使用体验。设计功能上的不同也在一定程度上说明每种元宇宙定位的不同之处。

对比来看，同类别的$SAND和$MANA主要具有购买，交易，游戏和治理特性，$WRLD具有交易，内部生态P2E等作用；而ApeCoin在此基础上，除了支持自身生态的游戏经济系统，还可以充当其他的游戏应用的代币，比如：Benji Bananas开发了针对$APE支付的接口。当有越来越多的项目用$APE代币来作为其效用代币时，基于ApeCoin所延伸的生态也会愈发趋向丰富。

SDK 和打造 NFT “公链”

值得一提的是，Otherside 开放了SDK，提供给所有开发者们使用。这样一来，Yuga Labs不局限于只能为自身的元宇宙产品提供拓展方案，同时也可以像搭建积木一样，先提供基础设施和规则，未来再串联其他各种不同的元宇宙产品，打造 NFT 世界的“公链”。

此外，在Flex文化遍布的Web3里，原生PFP IP是立足的根本因素之一。在虚拟世界中，通过NFT身份和对方交流，相比于Twitter Spaces单纯2D的头像展示，将更具传播力。

SteamVR游戏中VRChat内部会议参考图

在此之上，再给予一些想象空间——如果配合语音系统和VR设备的加持，在多元化的角色扮演中，用户将拥有无与伦比的体验感和沉浸感。此外，如果其他游戏也使用$APE的经济系统，跨游戏和应用间身份互动也将是一大亮点。

从Otherside看Yuga Labs的布局

BAYC仅仅是个开始，而要想在Web3领域进行布局，基于JPG的单纯的图片文化对于NFT项目发展并不长久，NFT 项目的后路何走？新的游戏和元宇宙领域（MetaRPG）和娱乐经济生态或许是方案之一。

强强联手

以投资元宇宙项目为名的Animoca Brands参与了Yuga Labs最新一轮的融资，另一家Animoca Brands子公司nWay也宣布将在与BAYC 合作开发新游戏，并把ApeCoin作为游戏结算货币。

两个团队包含了metaverse、GameFi、NFT等多个主题，技术和资源的共享能够让Yuga Labs在metaverse市场迅速后来居上。

人、财、地

从“一切是为了构建metaverse”的角度，可以比较清晰地理解Yuga Labs之前的运营路线。先建立社群，再构建经济，拥有了构建metaverse的最重要的两个元素，最后提供场景。这样便可以依靠UGC生产出大量优质内容。

更进一步说，为了使MetaRPG经济系统的运转，需要有用作游戏商城的货币，于是推出了原生代币ApeCoin。为了打造更完整的世界组成，需要在metaverse中涉及各类领域，于是收购了许多优质IP项目，而Otherside可以作为metaverse中的实际应用场景，供Yuga Labs进行链游开发、DAO社区等领域的探索，从而形成一个完整闭环。

此外，据消息，Otherside会根据位置层（KYC）为持有者提供一定比例的收入，这从而也能够盘活土地本身价值，而不是将土地作为闲置资产。

IP=话语权？

头部筹码

NFT总市值排名中，前五名中的CryptoPunks、Bored Ape Yacht Club、Meebits、Mutant Ape Yacht Club均是Yuga Labs的旗下项目，这些优质IP占据NFT市场绝对优势的交易量，如果后续这些项目均在Otherside内进行交易，则对OpenSea也可能形成一定的威胁。

因此，谁拥有这些IP，谁拥有流量，谁就拥有NFT的市场的话语权。

Yuga Labs四个头部NFT总市值占比

以IP为武器

这些优质IP无数次被验证过，已经成为NFT主流文化的代表，它们将是Yuga Labs以后在Web3开疆拓土的武器，可被应用于游戏开发、动画制作、潮牌联名等领域。

而在Otherside本次的预告片中许多头部IP联动，也暗示了Yuga Labs正在以BAYC为中心打造的打造IP矩阵，并将为这些IP制定发展策略。

元宇宙只是概念和工具

当前，The Sandbox、NFT Worlds 和 Decentraland 等多个项目通过发行土地的模式打入元宇宙地产市场先机，并带动了市场对于数字土地的重视。综合来看，很多元宇宙概念项目存在相似的构建框架：

如果说，Web3的基础是共识，那么，元宇宙的设立应该主要基于社区和共识，自下而上，而不是自上而下。在基础的结构下，Otherside的推出某种程度上革新了元宇宙的概念：身份认同和连接。这一点是很多元宇宙项目所不具备的，同时也是Otherside的优势之一。这一切的源头，是Yuga Labs所建立的BAYC社区。

此前，Yuga Labs 在宣传片中写道，“这就是我们不建立元宇宙的原因。我们正在构建可互操作的游戏元世界——一个 MetaRPG ”。说到底，元宇宙仅仅是个概念，概念背后需要技术和实质的需求支撑，以及最终的表现形式。很多人看过2018年上映的《头号玩家》电影，通过虚拟现实设备体验到电影内所呈现的画面，一个囊括社交、娱乐和工作的人造虚拟世界。但在Web3，我们所建立的不再是一个个信息的孤岛，而是一个基于身份认同的自我构筑的世界。

去中心化的道路仍然望不到头，沉浸感和交互性仍然有待软硬件的升级加持，虚拟货币支撑的经济系统也有待验证，一个真正意义上的元宇宙离我们或许还很远，但随着一系列新尝试和创新，我们也有望看到一个更好的市场模式。
