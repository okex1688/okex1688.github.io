当前 NFT 市场以去中心化平台为主，且主要集中在以太坊生态。

NFT 是一种与数字内容（比如艺术品、音乐或视频）相关联的独特加密代币，可以代表数字内容所有权。2021 年 3 月，著名加密艺术家 Beeple 创作的 NFT 作品《Everydays:The First 5000 Days》在佳士得以 6935 万美元的高价出售，随后一波又一波的 NFT 热潮涌现，吸引了越来越多人投资购买 CryptoPunks、Bored Apes 和 Loots 这些 NFT 作品。而在市场热度不断升温的情况下，提供 NFT 交易的市场和平台也如雨后春笋般出现，旨在为用户提供更便捷的数字艺术和收藏品出售和 NFT 交易服务。

下面，就让我们来盘点下现阶段规模最大的 13 个 NFT 市场和平台吧。

OpenSea

OpenSea 是最早一批 Peer-to-Peer NFT 市场，而且一直是规模最大的 NFT 交易平台，目前总交易量超过 95 亿美元（数据来自 DappRader）。在该平台上，你可以交易游戏内置道具、收藏品、艺术品、音乐、GIF 等几乎所有类型的 NFT。

注册 OpenSea 非常简单，你可以连接到 MetaMask 和其他钱包，比如 Coinbase Wallet、Bitski、Formic 等。登陆平台之后，购买 NFT 的过程同样非常简单，在浏览了自己喜欢的 NFT 系列之后，只需提交报价然后查看这个价格是否被卖家接受即可。

此外，卖家在 OpenSea 上传自己的 NFT 作品也是「小菜一碟」，只需前往「Create」（创建）选项，就能以内容创作者的身份连接钱包就能上传自己的 NFT，在提交出售之前你还需要对作品编写一个简单描述，然后就坐等买家出价了。

Axie Marketplace

「边玩边赚」游戏 Axie Infinity 是目前市场上规模第二大的 NFT 市场，目前交易总额超过 27 亿美元（数据来自 DappRader），不过该平台只能交易 Axie NFT，这是一种游戏中的可爱数字宠物，与 Pokémon 非常相似，玩家可以在 Axie 市场上购买并交易。

此外，玩家还可以通过游戏内置的「繁殖机制」（breeding mechanics）创建新的 Axie，然后在交易平台上进行出售。与收藏艺术 NFT 不同，Axie Infinity NFT 有一个目的：玩家可以在游戏中使用这些 NFT 与怪物和其他玩家战斗，然后赚取可用于培育新生物的代币。Axie Infinity 的 NFT 已经被证明能为玩家带来不错的收益，在菲律宾和印度尼西亚甚至有人专门靠玩这款游戏来谋生。

不过，Axie Infinity 对新玩家用户可能不太「友好」，其服务非常复杂，你可能需要完成一系列「障碍」才能真正开启游戏之旅。除了设置一个以太坊钱包（比如 MetaMask）之外，玩家还需要设置一个 Ronin 钱包，然后将 ETH 存入 Ronin 侧链并购买至少三个 Axie 宠物（可能会耗费你数百美元）——毫无疑问，对于部分「路人」玩家而言，游戏门槛的确有点高，不过在经验丰富的加密玩家看来，这点成本也许微不足道。

CryptoPunks/Larva Labs

作为以太坊网络上最早一批 NFT，CryptoPunks 是一个加密朋克 NFT 收藏品系列，由 Larva Labs 创建，是一个 10,000 个随即生成的像素化头像的集合，这些头像包括人类、僵尸猿和外星人，每个都具有像素艺术美学和独特属性。CryptoPunks 在刚刚发布的时候可以免费获取，但现在拥有它们的唯一方法就是花钱购买，这意味着你必须到 Larva Labs 市场进行交易，目前大部分 CryptoPunks 销售都是在该平台上进行的。本文撰写时（10 月 24 日） Punks 地板价为 94.95 ETH （约合 388,375.89 美元），另据 DappRader 数据显示，CryptoPunks 的交易总额目前为 15.2 亿美元。

CryptoPunks 交易较为便捷，你只需要在 Larva Labs 市场上连接自己的 MetaMask 钱包就能搜索 Punks 列表，然后输入出价即可。

NBA Top Shot

NBA Top Shot 是第一个获得广泛公众关注的 NFT 系列，它是一套包含 NBA 球星「高光时刻」（Moment）视频集锦剪辑的数字交易卡。一旦购买了一个 Top Shot 包，剪辑内容就会存储在经过区块链验证的安全加密钱包中，您可以在其中查看视频或是在 NBA Top Shot Marketplace 上转售它们。

NBA Top Shot 创作者 Dapper Labs 之所以能够获得成功，是因为主流用户很容易上手，注册 NBA Top Shot 非常简单，只需将自己的 Google 帐户链接到 Dapper，之后按要求输入电话号码以进行短信验证就能完成。

完成注册之后，你还需要执行一些帐户设置步骤，比如选择最喜欢的 NBA 球队等，完成后就可以加入新 Top Shot 空投并加入虚拟等待队列。NBA Top Shot 支付需要通过短信（再次）验证，用户可以选择使用现有加密钱包或 Flow 钱包，也可以使用自己的 Dapper 余额或信用卡进行购买支付。

Rarible

作为以太坊上领先的 NFT 市场之一，Rarible 平台最大的特点是社区拥有，上面展示了许多数字艺术和收藏品。 根据 Dappradar 的数据，Rarible 交易总量排名第十，约为 1.7 亿美元。

与 OpenSea 类似，用户可以在 Rarible 平台上买卖各种 NFT，卖家还可以选择为一张图片创建多个 NFT 进行多次销售。

Rarible 用户界面十分简单，加密「小白」也能轻松使用，如果你已经非常熟悉 OpenSea 等类似 NFT 市场，那么在 Rarible 上肯定会有一种宾至如归的感觉。

用户可以使用各种不同的加密钱包登录 Rarible，包括 MetaMask、Coinbase Wallet、MyEtherWallet、或是其他任何 WalletConnect 的手机钱包。 登录平台后即可可以使用钱包中的资金购买 NFT，甚至可以通过银行转账或借记卡中的法定货币为钱包充值。

SuperRare

SuperRare 的定位是成为「高端版 Rarible」，Rarible 的操作界面上充斥着各种令人眼花缭乱的闪烁 GIF 和现场拍卖活动，相比之下，SuperRare 则拥有更加简洁、简约的外观。

此外，SuperRare 在 NFT 活动策划方面的要求也更加严格，他们将自己「标榜」为一个鼓励加密艺术创作和收藏的社交平台，与艺术家合作时会要求作品在上架之前必须先提交到平台并获得批准；换句话说，SuperRare 更看重 NFT 的质量，而非数量。

注册 SuperRare 需要一个受平台支持的钱包，如 MetaMask 或 Formatic，此外用户还需要执行一个额外步骤，即创建一个与钱包地址相关联的用户名和密码，但这个步骤不会耗费太长时间，只需要几秒钟，之后就能访问平台上精选的独家 NFT 了。

KnownOrigin

KnownOrigin 的交易总量其实并不高，大约只有数百万美元，而且与 SuperRare 一样，旨在通过精选 NFT 作品为用户提供画廊式的使用体验。KnownOrigin 平台上的所有作品都保存在 IPFS 上，这其实是一种基础资产保护手段，此外该平台上的作品专注于数字艺术领域，因此不会提供到古怪头像或可爱宠物之类的 NFT。

KnownOrigin 可能是本文介绍平台中使用最简单的一个，用户所要做的就是连接一个 MetaMask 或 Formatic 这样的钱包，然后就可以竞标自己梦寐以求的 NFT 资产了。

Foundation

Foundation 平台自诩为艺术家的「创意乐园」，目前的交易总量逼近 1 亿美元。该平台举办了许多著名的销售活动，其中就包括互联网标志性 meme——Nyan Cat 的 NFT、Edward Snowden 的第一个 NFT、以及由制片人 Richard D. James （现已改名为 Aphex Twin）创作的视听数字收藏品。

一旦您使用 WalletConnect 连接了自己的 MetaMask 或是其他软件钱包，就能很方便地在 Foundation 市场上进行操作，您还可以像在常规拍卖网站上一样在计时拍卖中执行出价。

MakersPlace

MakersPlace 是另一个精选 NFT 市场，该平台因为拥有众多独家数字美术收藏而被人熟知。虽然目前销量总量可能不大（撰写本文时为 2350 万美元），但却拥有大量独家创作作品，这也使得 MakersPlace 显得非常与众不同。

2021 年 2 月，当著名加密艺术家 Beeple 以 1 美元的价格提供了一系列 NFT 时，由于活动火爆一度导致 MakersPlace 平台宕机。值得一提的是，该平台还邀请了一些音乐家主持了许多 NFT 活动，T-Pain、Shakira 和 Rage Against the Machine 的 Tom Morello 等人都已经在该平台上进行了注册。

假如您想要在 MakersPlac 上购买 NFT，注册过程也非常容易，甚至还可以使用自己 Google 或是 Facebook 帐户登录，继而节省了大量注册时间。

不过，作为平台注册过程的一部分，您必须选择五位艺术家进行关注，然后哎可以使 MetaMask ETH 余额或信用卡购买 NFT 作品（支持信用卡交易还是比较用户友好的）。 如果您是创作者，则必须通过邀请并填写在线表格申请才能加入该平台，申请需要获得平台策展人批准。

Nifty Gateway

2019 年，加密货币交易平台 Gemini 收购了 Nifty Gateway 平台。作为第一批大型的 NFT 市场之一，Nifty Gateway 通过这笔收购交易获得了强大的支持，事实证明，Nifty Gateway 具有先见之明，拥有 Gemini 这样的投资方的确能获得不少好处。在 NFT 热潮中，Nifty Gateway 凭借首批价值数百万美元的 NFT 销售成为了市场焦点：2021 年 2 月，Nifty Gateway 以 660 万美元的价格促成了 Beeple 的 CROSSROAD 销售；5 月，该平台上的商品总价值（GMV）已达到 3 亿美元。

Nifty Gateway 平台上的 NFT 作品都是通过精心策划挑选后才能上架，每隔三周，该平台会邀请 The Weeknd、Grimes 和 Eminem 等艺术家来研讨作品。此外，他们还拥有另一个 NFT 市场，其中可以浏览到精选收藏品，这些作品都经过艺术家的验证，而且还能欣赏到不少暂未验证的艺术家预发作品。

由于 Nifty Gateway 与 Gemini 彼此整合非常好，因此入门相对比较简单。首先，您需要在该网站上进行注册（如果您打算出售 NFT，则需要使用 Stripe 来验证您的身份），然后就能通过关联信用卡或向存款地址发送一些 ETH 来为您的帐户充值。完成后即可在市场购买 NFT 了。

此外，用户也可以选择将自己的 Gemini 交易账户与 Nifty Gateway 账户相关联，这样您就能够使用您的 Gemini 交易账户余额来进行购买和取款了（您的 Gemini 账户可以通过银行转账、电汇、加密存款交易进行充值） .

BakerySwap

与 OpenSea 相比，BakerySwap 算是规模相对较小的 NFT 市场，但其实也不奇怪，要知道大多数 NFT 市场都是建立在以太坊之上的，BakerySwap 则是首批在币安智能链 (BSC) 上推出的 NFT 市场，该平台除了支持交易 NFT 之外还允许用户之间直接进行流动性资产交易和兑换。

BakerySwap NFT 市场并不复杂，只需要连接开源以太坊钱包 MetaMask 即可使用，但需要注意的是，由于 BakerySwap 平台是在币安智能链上运行的，因此 NFT 资产只能使用 BNB 进行购买，而不是常用的 ETH。

Binance NFT Marketplace

Binance 是全球最大的加密货币交易平台之一，因此顾名思义，Binance NFT Marketplace 就是一个由币安提供的 NFT 平台，该平台旨在为 NFT 市场提供独家产品并利用其合作伙伴关系将交易平台用户资源吸引到 NFT 平台上。

如果您已经拥有了 Binance 帐户，那么恭喜你，这个账号会自动与 Binance NFT Marketplace 关联，继而无需重新注册。进入平台后，竞标 NFT 资产操作与其他同类平台一样简单，买家可以在该平台上使用 ETH、BNB 和 BUSD，具体取决于创建者可接受哪个数字资产。

Solanart

Solanar 区块链是以太坊的竞争对手，最近一股 mini NFT 热潮席卷 Solana，包括 Degenerate Ape Academy、Aurory 和 SolPunks 在内的许多收藏品均获得了数亿美元交易量，而这些交易都是在交易平台 Solanart 上完成的。

与 OpenSea 不同的是，Solanart 平台上的 NFT 藏品都经过仔细筛选，但正是因为如此，导致平台上可供选择的 NFT 数量相对较少。

如果用户想要在 Solanart 平台上购买 NFT，必须下载支持 Solana 区块链的钱包，比如 Phantom 或 Solflare，然后加载到 Solanart 平台上。值得一提的是，Phantom 支持与加密货币交易平台 FTX 是直接整合，因此如果在 FTX 上存入资金会变得十分便捷。连接钱包后，你只需点击自己选择的 NFT，然后连接钱包就能进行出价交易了。
