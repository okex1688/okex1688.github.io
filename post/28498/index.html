DeFi热点

1. 波卡DeFi项目Acala将集成Chainlink喂价

金色财经报道，波卡DeFi项目Acala宣布将集成Chainlink喂价。Acala透露，已经与Chainlink团队合作进行了一段时间的整合。Chainlink Oracle Pallet的推出推动了这一过程。据悉，Chainlink Oracle Pallet是一种即插即用的解决方案，于4月在Polkadot生态系统中可用。Acala还将Chainlink喂价集成到其在Kusama上的姊妹网络Karura中。

2. Algorand基金会宣布推出约合3亿美金DeFi支持基金

9月10日消息，据Algorand基金会官方网站公告，Algorand基金会宣布推出Viridis DeFi基金。该基金将提供1.5亿个ALGO代币，以推动Algorand上DeFi生态系统的增长，目前已发布前两个SupaGrants（巨额捐赠），它旨在支持关键DeFi基础设施的创建。其中包括500万美元的Oracle SupaGrant和500万美元的Bridge SupaGrant，后续Algorand基金会将发布关于该基金的更多详细信息。

3. Solana生态DeFi平台Apricot Finance发布测试网

9月10日消息，Solana生态DeFi平台Apricot Finance已发布测试网，将向提供反馈的用户发放NFT作为奖励。据悉，Apricot由Lend、X-Farm和Assist三个产品组成。8月下旬，基于Solana网络的去中心化货币市场平台Apricot Finance宣布完成400万美元融资，以推进其成为个性化DeFi解决方案首选平台的目标。 此前报道，Apricot Finance完成80万美元种子轮融资。

NFT

1. Art Blocks NFT艺术家8月份向慈善机构捐赠2350万美元

9月10日消息，8月份，NFT平台Art Blocks上的艺术家向慈善机构捐赠2350万美元，通过该项目筹集的总资金超过2600万美元。其中，艺术家Rich Lord在8月27日推出的项目Geometry Runners筹集的资金最多，捐款超过340万美元。 8月，共有25位艺术家在该平台上推出NFT。在每个项目上线之前，Art Blocks和艺术家就所筹资金中捐给慈善机构的比例达成一致。对于大多数项目，艺术家捐赠艺术品总销售额的10%至25%。随着新生成艺术继续获得更高的价格，捐赠给慈善机构的金额也将增加。

2. 印度摇滚乐队Euphoria将通过NFT形式发行其新专辑“SALE”

9月10日消息，印度摇滚乐队Euphoria将通过NFT形式发行其新专辑“SALE”，乐队将把音乐专辑NFT和9张幕后照片通过Token Runway在公开发行前进行拍卖。拍卖会于9月9日开始，9月12日结束，随后，专辑将于9月14日发行。

3. NFT碎片化交易协议Unicly上线uLoot

9月10日消息，NFT碎片化交易协议Unicly通过投票，上线uLoot，当前uLoot价格约为667美元。

4. NFT数据：CryptoPunks 24小时成交量涨幅677.05%

DappRadar最新数据显示，过去24小时成交量排名前十的NFT项目及其成交量涨跌幅如下：Axie Infinity（+6.02%）、CrypToadz by GREMPLIN（-33.42%）、Bored Ape Yacht Club（+144.66%）、888 inner circle、Inertial Moment（+16.38%）、Loot（+29.01%）、CryptoPunks（+677.05%）、The Sevens (-81.10%)、VeeFriends（+228.08%）、PUNKS Comic（-50.88%）。 其中，CryptoPunks 24小时成交量领涨（+677.05%）；The Sevens 24小时成交量领跌（-81.10%）。

5. 阿里巴巴推出NFT数字月饼

金色财经报道，日前，天猫超级品牌日推出超写实数字人AYAYI定制月饼礼盒及定制NFT数字月饼，中签用户可获得一枚NFT定制月饼。这款NFT定制月饼每一枚都配有专属序列号，不可篡改、不可分割。据官方介绍，AYAYI本次送出的NFT数字月饼采用多面体和酸性金属物质作为两个主要设计元素。其中多面体形态代表当下的现实世界，而具有流动性的酸性金属物质则象征元宇宙。两种元素交融、对撞，表现了现实生活与未来数字生活的融合过程。这次AYAYI定制的NFT数字月饼活动是由蚂蚁链平台提供技术支持，限定50份，中签用户只需出1元可得定制月饼礼盒+定制NFT数字月饼1枚。

6.首部NFT电影将于9月24日上映，奥斯卡影帝安东尼·霍普金斯领衔主演

9月10日消息，电影《ZeroContact》公布了首个宣传片，并宣布将于9月24日在NFT平台VUELE首映。该电影将全片铸造成NFT，并于9月24日在OpenSea上进行拍卖。《ZeroContact》由刚刚斩获奥斯卡影帝的安东尼·霍普金斯主演，讲述了5位角色试图关闭已故的科技巨头芬利·哈特（安东尼·霍普金斯饰）可能导致世界末日的发明的故事。VUELE是CurrencyWorks和EnderbyEntertainment共同建立的世界首个以NFT形式提供全长影片和数字收藏娱乐内容并直接面向消费者的平台。

7.NFT碎片化服务SZNS完成400万美元融资

金色财经报道，SZNS宣布完成了由Framework Ventures和Dragonfly Capital领投的400万美元融资，Pear.vc和Baller Ventures等公司参投。据悉，SZNS允许用户创建NFT“专辑”或集合，可以将其细分为治理代币并由去中心化自治组织(DAO)管理。
