2022年1月19日，横琴数字金融研究院学术与技术委员会主任朱嘉明教授接受第一财经记者曲莉莉专访，就元宇宙的商业前景、技术路径等问题进行了解答和治理规则。朱教授提出，“无论如何，对于绝大多数人来说，还是要与时俱进，正视元宇宙对现实世界的重构。”

以下为采访全文：

元宇宙的商业前景、技术路径和治理规则

一、元宇宙的历史地位

《第一财经》： 您如何看待元宇宙在互联网发展史上的地位？元宇宙的哪些特点决定了它对未来世界的深远影响？

朱嘉明： 互联网的历史可以追溯到1950年代，已经有六十、七十年了。就算算上互联网的里程碑，万维网的诞生，也已经有30年了。在这个过程中，人们看到了互联网的演进，即互联网经历了从Web 1.0到Web 2.0再到Web 3.0的过程。这是对互联网演进和进步的共识。

对于Web 3.0，在2006年就提出了，至今还没有非常强劲和鲜明的落地表现。人们更多地谈论Web 3.0的概念和趋势，其最佳应用集中在与以太坊相关的应用上。

在我看来，正是在这样的背景下，元宇宙的出现才算是Web 3.0演进的最重要成果。我们可以用两句话来理解元宇宙与互联网的关系。第一句话：元宇宙是互联网从 Web 1.0 到 Web 3.0 演进的结果。下一个更重要的句子：元宇宙本身也是一个突变——就像一个进化和变异的新物种。由于它是互联网与众多其他新技术相结合的产物，这个新物种使得元宇宙突破了互联网的极限。

互联网历史 资料来源：克里斯菲尔波特；thereboot.com

也就是说，没有互联网，没有 Web 3.0 的演进，就不会有元宇宙。元宇宙是互联网演进的结果。同时也是物种的突破——元宇宙不仅包括互联网技术，还包括虚拟现实技术、数字孪生技术等。元宇宙是互联网进化和变异的结果。可以预见，因为元宇宙，Web3.0将进入高速增长期。

至于元宇宙的特点，众说纷纭，有很多总结。但元宇宙的这些特点需要结合元宇宙带来的历史意义。在我看来，元宇宙有三个基本方面：

元宇宙的第一个特点：技术的融合。元宇宙的技术，甚至可以追溯到图灵和冯诺依曼，以及二战以来所有信息革命的历史。图灵机和冯诺依曼架构本身就包含了元宇宙的基因。因为它是技术大师，所以在这个世界上，没有一个新物种能够超越元宇宙，能够承载、吸收和容纳信息革命的所有技术。

元宇宙的第二个特点：展示了一种新的经济体系。元宇宙需要一个经济系统的支撑，而这个系统并不是地球经济系统的翻译。要避免和消除受资本主义制度影响或支配的市场经济的诸多弊端，消除垄断和贫富差距。也就是说，元宇宙的经济体系也应该有强大的共享和平等基因。元宇宙的代理人可以以一种新的平等主义方式组织新型的经济活动。这是元宇宙的一个非常显著的潜力，看起来有点过于理想化，但这是可能的。

例如，在元宇宙中，某些人垄断和控制他人的劳动、思想和时间的可能性很小。因为元宇宙无法真正“封闭”，所以没办法用地球村的“地牢当监狱”的方式。在任何元宇宙中，如果有人不开心，他们可以“转生”，从一个元宇宙跑到另一个元宇宙。

元宇宙中的对象会接受这样的概念，“我不必和你在一起，因为我可以随时随地在不同的地方。” 在元宇宙中，不会有城市的元宇宙，也不会有农村的元宇宙；也不会有发达国家的元宇宙，也不会出现新兴市场国家的元宇宙。因此，元宇宙有可能为人类甚至数字身份提供自由选择。元宇宙很难被资本束缚。在这个元宇宙中你不能在另一个元宇宙工作是没有问题的。

跨海移民工作人口资料来源：parisschoolofeconomics.eu

发展经济学里有一个词：“迁移”。因为元宇宙提供了自由移动的可能性。

元宇宙的第三个特点：人类存在方式的改变。元宇宙与现实世界之间存在明显的“分工”。在元宇宙中，没有必要解决地球村中人类的物质和物质需求。比如，没有人会去思考和讨论如何在元宇宙中实现“厕所革命”，因为元宇宙中没有人的排泄问题。

因此，在元宇宙的新维度中，主体的生存方式、生存方式、生存内涵都会发生非常大的变化。在物质世界中，人类的时间充满了柴米油盐醋茶生子等复杂的程序，而元宇宙的主题将从一堆琐碎的日常生活中解放出来。

元宇宙自然会有人类存在的模式。很多人试图将平凡的事物转化为元宇宙，但这一切都将失败。元宇宙无法承受现实世界的重量。因为元宇宙，人类将拥有两个家园：留下传统的地方，以及将美带到未来的地方。例如，一个人可以在白天继续在元宇宙中做梦，而不是在晚上完成，但在现实世界中却做不到。元宇宙不是分享灾难、痛苦和眼泪的地方。

对于这个目标，能不能达到和能达到什么程度都不是一回事。人们很难找到一个阳光普照的地方。

特别强调人类需要这样一个新型世界，元宇宙提供了这样一种可能。元宇宙是一个很大的地方：它不是乌托邦，也不是各种宗教的幸福或神秘圣地。后工业社会以来，人类最大的需求是心理需求，现实世界中即使是精神科医生也无法满足。元宇宙提供了改善人类心理需求的一面，满足了人们失去的童年和少年时代的幸福。元宇宙最重要的属性都在这里，提供了人们想象的东西。如果人们认为他们可以在元宇宙中再次在地球上玩同样的东西，他们会被吹走的。

元宇宙本质上是一个公共空间，正是因为元宇宙具备这些特点，它当然会对未来世界产生深远而全面的影响。从技术的深入发展，到人类经济体系的演进，再到人类的生活模式，尤其是新的人类心理模式。

人机交互 来源：army.mil

同时，在元宇宙中，人们将重建人机关系，以及各种人工智能之间的关系，在元宇宙中得到全面发展。目前，在现实世界中，人机关系还没有特别大的突破。“人”是碳基的，具有生理特性和生物学特性；“机器”是物理的和机械的。他们之间有互动。在元宇宙世界中，它们的物理载体不复存在，它们以虚拟的方式交互，双方更容易产生更大的公约数，不会有界限。这对未来具有重要意义。

所以，去年我多次说过，元宇宙时代是兼容“后人类社会”的。元宇宙将是一个新的存在。虽然元宇宙是由碳基人类创立的，但未来元宇宙的主体将逐渐向数字化、虚拟化的人类转移，也就是科幻电影《银翼2049》中的所谓“人类”。因为是数字化和虚拟化的，所以可以同时参与N个元宇宙。否则一个元宇宙上不去。目前的碳基人类只能通过数字化和虚拟化人类表达自己的愿望，实现自己的愿望。

如果一定要解释目前碳基人、数字人、虚拟人之间的关系，那么我们可以解释为：只有一个人，因为人必须生、老、病、死；但可以有N个数字人和虚拟人，前者不能长生，后者可以长生。也就是说，任何个体碳基人在出生、老、病、死后，其数字图像和虚拟形式仍然可以留在这个世界上。这需要两项技术：一是如何以真人的身份存储数据和信息；另一个是如何像碳基人类个体一样模拟数字人类并延长他的寿命。这应该是一个没有悬念的未来。

二、元宇宙的商业前景

《第一财经》： 在可预见的未来，元宇宙会带来哪些商业应用前景？当今投资者和行业参与者有哪些明显的机会？从长远的战略来看，哪些产业布局会变得越来越重要？

朱嘉明： 这种思维方式需要改变。我要强调的是：发生在元宇宙的“生意”并不是现实世界中的“生意”。元宇宙世界中的商业很难像现实世界中的商业一样存在、发展和发展。退一步说，虽然元宇宙的业务和现实世界的业务有着血缘和基因的关联，但它们已经是不同类型的业务，属于完全“分叉”的两种不同的商业模式。

在我看来，元宇宙的业务是一个更强调非物质状态、更多时间形式、更多虚拟业务、更多消费者参与的业务的业务。这些与现实世界的业务不同。例如，百事可乐是现实世界中的一种饮料，这种饮料的结果最终会导致你去厕所小便；而百事要在元宇宙中展示的是百事的非同质化代币（NFT），这是一个概念。因此，百事可乐有两种商业模式：一种属于物质和生理层面，满足食欲，另一种是与可口可乐竞争的饮料行业；另一种是基于灵性和美学的商业模式，基于想象进入赛道。

百事可乐发布 NFT资料来源：nftplazas.com

那么现实世界的企业如何努力在元宇宙建立商业前景呢？或者以百事可乐为例。百事可乐是一个经济实体，它将有两种存在形式：物理世界和元宇宙世界。但是，这并不意味着百事可以将现实世界的东西移植到元宇宙中，而元宇宙不是一个地方为百事可乐做广告。元宇宙广告也将是一种基于高度创造力和想象力的新模式。此外，耐克和阿迪达斯也开始进军元宇宙，同时也在打造实体和虚拟“双”的产品模式。

在元宇宙，公司必须创造新产品。企业在元宇宙布局商机的一个重要方向是实现艺术、想象、美学和新型数字金融工具的融合。前面提到的百事可乐、耐克、阿迪达斯都曾尝试打造元宇宙早期的商业模式，值得肯定。

对于面向未来的布局，三个产业至关重要：一是智能制造，二是教育，三是文化艺术。

三、元宇宙的技术路径

《第一财经》： 要真正打开元宇宙之门，需要突破哪些核心技术？元宇宙会有技术路线图吗？中小企业如何在这里找到自己的机会和定位？

朱嘉明： 核心技术有三：一是新能源，二是新算力，三是人工智能3.0。因为元宇宙最终是由能源支撑的，所以元宇宙的运行需要大量的能源，而目前的传统能源不足以支撑未来爆发性的元宇宙发展。同时，目前的计算能力也跟不上。只有量子计算的算力才能支撑元宇宙。最终，运行元宇宙的主体将不是人类，而是人工智能。因此，新能源、新算力和新人工智能的结合，最终可以全面激活元宇宙，支撑元宇宙的发展壮大。

完全激活元宇宙 来源：作者的绘图

在我看来，元宇宙至少需要10年才能确立目标，30年发展和奠定基础，50年左右才能使元宇宙成为与现实世界平行的新人类生存的理想空间。现在的问题是，人们开发元宇宙时间长，缺乏理性认知，过于渴望成功，过于焦虑，过于功利。这样，就会破坏元宇宙发展的环境和生态。

需要注意的是，技术本身具有生命力。新能源、新算力、新人工智能等技术既有差异，又有互动。他们每个人都有自己的路线图，这三个路线图的叠加就是元宇宙技术路线图。例如，量子技术应该在未来3到5年内有根本性的突破，实现量子计算机与经典计算机的并行，但要全面普及和大规模使用还需要10到15年。

同理，AI 3.0 也需要 10 年左右的时间，而且有不同的发展方向。比如人工智能本身的发展方向和人工智能直接改变人类生存方式的方向，比如脑机融合技术。这些都需要时间。简而言之，元宇宙的路线图是能源、算力、人工智能这三个十年发展图谱的叠加，并找出它们之间存在的关系。

在元宇宙浪潮中，中小企业不可能或极难成为元宇宙赛道的领头羊。因为中小企业无法适应元宇宙赛道的基本特点：元宇宙需要高科技投资相结合，高科技人才相结合，资本投资相结合。中小企业很难理解和完成这些。

因此，中小企业要想获得机会，最重要的是了解元宇宙产业链，然后认清自己在元宇宙产业链中可能的位置。

4. 元宇宙治理规则

《第一财经》： 在您看来，元宇宙世界需要构建什么样的治理规则？这些治理规则如何影响元宇宙世界的轨迹？

朱嘉明： 要回答这个问题，首先要明白一个重要前提，谁将是影响元宇宙世界的主体？答案是三大主体：一是世界级企业。这包括游戏公司，包括微软在昨天的新闻中收购的暴雪；其次，元宇宙的玩家，那些在数字经济中冲浪的年轻人，最后是政府。

元宇宙治理 资料来源：washington.edu

制定规则的是政府，但政府并不是元宇宙的鼻祖和第一推动力。现在，政府需要做四件事：一是从缺席到到位，准确定位。现在对于元宇宙众说纷纭，现在需要学习、观察、分析和辨别，才能对元宇宙做出判断和决策。二是建立政府协作、共治的制度。元宇宙本质上是跨区域的，甚至是跨主权的，很难进行传统的物理控制。除非所有游戏宇宙都关闭，否则A解决，大家都去B。三是创造符合元宇宙特点的规律。现在在法律法规方面，它基本上处于空白状态。四是形成元宇宙的税收模型。元宇宙会产生生意，政府应该有贡献，应该有税收奖励。

元宇宙世界的发展轨迹是对人类社会的历史考验，因此元宇宙的治理涉及到对人类基本价值观和人类基本规则的一些认知。只有在此基础上，元宇宙和人类社区的目标才能一致。

《第一财经》： 在元宇宙世界里，金融规则和数据规则将如何与现实世界的规则联系起来，又有何异同？

朱嘉明： 这和我对上述业务的看法是一样的：元宇宙的业务是一个“业务”，而不是另一个“业务”；同样，元宇宙的货币和金融系统也不是地球上的翻译。以目前对地球村货币金融体系的理解，人们无法想象元宇宙的货币金融体系。元宇宙中没有央行和IMF是常识，否则会有相当大的混乱和误导。比如在元宇宙，很难开某家私人银行或某家商业银行；也很难出现这样一种情况，一种法币在这个元宇宙流通，另一种法币在另一种元。宇宙旅行。因此，未来，元宇宙很可能会逐渐演变，演变成所谓元宇宙的货币，与现实世界中的银行、法律、金融体系息息相关。如果有这样的概念，你会发现 NFT 仍有巨大的潜力。

5. 元宇宙投资布局需要新理念

《第一财经》： 元宇宙世界的到来将如何影响公司的战略、投资和产业布局？企业家和政策制定者如何为元宇宙的到来做准备？

朱嘉明：元宇宙 带来了新的领域、新的市场理念、新的经济实体。新领域、新市场、新主题，这是支撑元宇宙的三大引擎。因此，投资和部署元宇宙需要新概念、新方法和新团队。

对于企业家和政策制定者来说，三个方面很重要：

首先，要学习。我是元宇宙的早期研究员，但我认为我对它的理解还不够到位，所以我每天都在思考它。许多人认为他们了解元宇宙，但事实并非如此。今天绝大多数人的头脑都跟不上元宇宙本身发展的速度。所以，学习吧。

二是企业管理拥抱“素养”，实现数字化转型。未来的企业高管如果不能理解和实现数字化转型，就无法参与元宇宙——这就是游戏的入场券。

三是建立与时俱进的新技术队伍。未来，企业必须有一个元宇宙团队，包括元宇宙工程师、框架工程师，以及区块链、人工智能、量子技术方面的专业人士。为什么目前部署的公司都是互联网巨头或IT行业的领导者？因为他们有相当的技术积累。这是一个技术积累比资本积累更重要的时代。因为有技术找资金不难，但有资金就不一定找技术。

最后一个问题是解放你的思想，知道你不知道什么。所有人，在元宇宙面前，在数字革命面前，都是小学生——就算现在领先，明天也会被淘汰。

元宇宙提供了前所未有的好学校；与时俱进的元宇宙就是与时俱进。因此，是时候举办元宇宙学习班来普及技术了。我支持和主张的所有公司都应该设置两个职位，一个是“未来战略官”，一个是“数字技术官”。否则，发达国家和发展中国家之间、数字之间的所谓“数字鸿沟”而传统企业，将因数字经济转型的速度而雪上加霜。
