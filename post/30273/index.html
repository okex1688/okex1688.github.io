DeFi热点

1.Curve锁仓量突破150亿美元创历史新高

金色财经报道，根据最新数据显示，基于以太坊区块链的DeFi协议Curve锁仓量已突破150亿美元，创下历史新高，本文撰写时达到158亿美元，总用户数为88626。目前以太坊链上DeFi协议总锁仓量达到1083.2亿美元，同样创下历史新高。

2.DeFi协议Project Serum启动流动性挖矿计划，初始拨款为1亿美元

10月29日消息，基于 Solana 区块链的流动性基础设施协议 Project Serum 周四宣布，Serum 去中心化自治组织（ DAO）已投票批准启动流动性挖矿计划，初始拨款为1 亿美元。这笔款项以 Serum 的原生 SRM 代币支付，将用作直接与 Serum 的链上订单簿合作的自动做市商的奖励机制。

3.BSC链上AutoShark Finance协议遭遇攻击，攻击者获利200万美元

10月29日，据PeckShield消息，BSC链上DeFi协议AutoShark Finance遭遇黑客攻击，攻击者获利200万美元，AutoShark协议遭遇的损失则可能高于200万美元。 此前消息，AutoShark于今年5月遭闪电贷攻击，Token闪崩，AutoShark回应将发行新Token JAWS补偿受损用户。此后，AutoShark于今年10月再次遭遇闪电贷攻击，黑客获利约58万美元。

4.去中心化数据索引工具 SubQuery 将支持 Moonbeam 和 Moonriver 的 EVM 测试版

10月29日消息，去中心化数据索引工具 SubQuery 宣布将支持 Moonbeam 和 Moonriver 的 EVM 测试版，允许波卡和以太坊的开发者将 Substrate 和 EVM 数据无缝集成到一处，并使用 GraphQL 查询这个单一数据源。此外，SubQuery 引入的过滤器可以过滤非合约交易、交易发送者、合约和索引日志参数，因此开发人员可以构建各种满足其特定数据需求的项目。

5.Synthetix将在Optimism上进行Avior版升级，支持包装合约部署

10月29日消息，合成资产项目Synthetix将于今日晚些时候在Optimism上进行Avior版本升级。本次升级为Optimism上的版本升级，Synthetix在Optimism上的版本将会进行两至三小时的停机，L1版本不受影响。该新版本添加了一个新的WrapperFactory合约，可以部署新的包装合约来支持任何ERC20代币，最初支持以太坊，未来将增加更多的包装资产。

NFT热点

1.Facebook将公司名称更改为“META”

金色财经报道，Facebook周四宣布将公司名称更改为“META”。该公司在其更名公告中表示，公司股票代码将从12月1日起变更为“MVRS”。扎克伯格周四在一封信中写道：“我们希望在未来十年内，元宇宙将覆盖10亿人，承载数千亿美元的数字商务，并为数百万创作者和开发者提供工作支持。”扎克伯格预测，元宇宙的元素可能会在5到10年内成为主流。他补充说，该公司预计“在元宇宙达到规模之前，将在未来几年内投资数十亿美元”。

2.麦当劳美国将从11月1日起免费送出10枚McRib NFT

10月29日消息，麦当劳美国为纪念烤汁猪排三明治（McRib）诞生40周年，决定在下周免费送出10枚McRib NFT。该公司在声明中称：我们的McRib NFT是粉丝最喜爱的三明治的数字版本，它几乎和McRib本身一样美味，我们将从11月1日起在推特上将它们送给一些幸运粉丝。关注麦当劳推特账号，并在11月1-7日间转发抽奖推文者，均有机会赢得该奖励。

3.CoinMarketCap推出惊魂之夜NFT

10月29日消息，CoinMarketCap宣布推出惊魂之夜NFT，以庆祝万圣节。据悉，该独家NFT由NFTb平台的艺术家Pixel Queen创建，为CoinMarketCap忠诚度奖励计划相关奖品之一，用户只可通过CoinMarketCap平台为用户提供的钻石道具购买。

4.红枣科技将推出部署NFT的基础设施

10月29日消息，BSN背后的“建筑师”红枣科技将推出支持在国内部署NFT的基础设施。该项目名为BSN分布式数字证书（DDC），项目为周五在BSN开放准入峰会上揭晓。红枣科技CEO何亦凡表示，在接下来的5年，NFT将在国内广泛使用，但是国内不希望该技术和加密货币相关，也不希望其跑在公链和联盟链上。“如果我们看看国内环境，这是唯一推出NFT的方式。” 据悉，在BSN-DDC上，开发者可以在BSN上的10条开放准入链上构建NFT平台。该基础设施也可以被部署在BSN以外的节点上。该项目将在2022年1月底推出。

5.CryptoPunks销售总量突破20亿美元创历史新高

金色财经报道，随着NFT 收藏家，推特用户 Farokh.eth（@farokh）以 124,457 ETH（约合 5.32 亿美元）的价格购买了CryptoPunk #9998 NFT作品之后，CryptoPunks销售总量突破20亿美元，创下历史新高。据官方数据显示，本文撰写时CryptoPunks销售总量为68.721万枚ETH，约合20.8亿美元。当前CryptoPunks地板价为94.99 ETH（约合406,357.71美元），过去12个月销售量为11630笔。

6.BTFS已上线NFT分布式存储解决方案NFT Hub最新版本V1.1

据官方消息，BitTorrent File System（BTFS）已上线NFT分布式存储解决方案NFT Hub最新版本V1.1。该版本新增Justin@BTFS页面，用户通过NFT hub平台可一键查看波场TRON创始人兼BitTorrent CEO 孙宇晨在BTFS上分享的收藏、作品、报道等精彩内容。同时，该版本可支持NFT铸造，进一步为全网用户提供铸造NFT的平台级服务。 NFT Hub为BitTorrent File System（BTFS）近期上线的NFT分布式存储解决方案，旨在进一步打通BTFS去中心化网络与波场TRON生态, 为NFT免费提供分布式存储支持。

7.大卫·林奇旧作将作为 NFT 拍卖

10月29日消息，数字艺术市场 SuperRare 宣布将拍卖 David Lynch 与纽约乐队 Interpol 的联合作品。双方合作衍生出一个由七部分组成的 NFT 系列，其中首个将拍卖的是一部名为“我触摸一个红色按钮“的短片。
