前言

Nansen NFT 指数旨在衡量投资 NFT 市场时的主动策略，它为寻求跟踪代表市场或特定细分市场指数的投资者设置了基础指数。或者，这些指数作为一种机制来衡量 NFT 市场随时间的表现。这篇研究文章的目的有三个：第一部分确定了投资者寻找 NFT 敞口的原因以及他们如何通过 NFT 指数实现这一目标；第二部分概述了 Nansen NFT 指数；第三部分说明了 Nansen NFT 指数所暗示的市场趋势。

通过指数为投资者提供 NFT 敞口

市场指数使投资者能够衡量具有特定市场特征的持股组合的价值。例如，追踪美国市场表现的三个最受欢迎的股票指数是道琼斯工业平均指数 (DJIS)、标准普尔 500 指数和纳斯达克综合指数。在传统金融领域，现有文献表明，随着时间的推移，指数投资的表现可以超越主动管理型基金。此外，市场指数为投资者提供历史数据，有助于投资者做出合理的投资决策。

将其应用于 NFT 市场，Nansen 设计了一组指数，由代表不同 NFT 细分市场（例如社交、游戏、艺术和元宇宙）的不同 NFT 集合组成。投资者可以跟随不同的市场指数来衡量市场走势和表现。

Nansen NFT 指数

Nansen NFT 指数允许投资者追踪特定的 NFT 细分市场，这些指数作为一种机制来衡量 NFT 市场随时间的表现。对于要包含在市场基准指数中的市场，它们必须满足特定的资格标准。

该组指数采用市值加权策略，市值以以太坊（ETH）计价， Nansen NFT 指数组合包括：

  • Nansen NFT-500

  • Nansen蓝筹10

  • Nansen 社交-100

  • Nansen 游戏-50

  • Nansen 艺术-20

  • Nansen 元宇宙-20

Nansen NFT-500

该基准指数由 500 个 NFT 集合组成，自 2022 年 1 月 1 日以来平均占每日市场交易量的 85.4%。该指数旨在跟踪 NFT 市场的活动和走势。

NFT-500 指数每天计算一次，每 30 天重新平衡一次，其中指数成分会相应地重新评估和重新平衡。

Nansen 蓝筹10

蓝筹 NFT 指数是著名和经典 NFT 的集合。这也是Nansen 构建的第一个指数。旧指数由 20 个成分组成，而修订后的 Blue Chip-10 指数由 10 个按市值加权的成熟 NFT 集合组成。

Blue Chip-10 指数每天计算一次，每 90 天重新平衡一次，其中指数成分会相应地重新评估和重新平衡。

Nansen社交-100

社交 NFT 收藏在 NFT 市场中占据最大的市场份额。社交-100 NFTs 索引由 100 个 NFT 集合组成，这些集合进一步分为子类别：个人资料图片 (PFP)、访问和成员资格以及实用程序（例如去中心化自治组织DAO）。

社交-100 NFTs 指数每天计算一次，每 30 天重新平衡一次，其中指数成分会相应地重新评估和重新平衡。

Nansen 游戏-50

游戏 NFT 收藏是 NFT 市场中增长最快的部分。Gaming NFTs 指数由 50 个 NFT 集合组成，这些集合进一步分为子类别：Play-to-Earn (P2E) 游戏、角色扮演游戏 (RPG)、DeFi 相关游戏 (GameFi)。

Nansen 艺术-20

艺术 NFTs 索引由 20 个 NFT 集合组成，涵盖 ERC-1155 和 ERC-721 上的 NFT 集合。该指数的目的是展示在市场上交易的著名 NFT 艺术品。该领域内的 NFT 收藏可以进一步分为子类别，例如生成艺术、物理艺术和音乐。

例如，XCOPY 的“TRAITORS”系列是一种数字艺术形式，有些人可能将其视为新媒体艺术，艺术家利用计算机或电子设备创作作品。另一方面，生成艺术是一种数字艺术的衍生形式，这种艺术形式通常涉及在其创作中使用编程，一个例子是 Autoglyphs。然后是实物艺术作品，比如FEWOCiOUS的作品。最后，音乐也被归类为 NFT 艺术的一部分。

Nansen 元宇宙-20

NFT 市场的另一个增长势头是元界领域。元宇宙 NFTs 索引由 20 个 NFT 集合组成，这些集合进一步分为子类别：土地和房地产、头像、资产和公用事业。

用 Nansen NFT 指数呈现市场趋势

1. NFT 市场以 68.5% 的年初至今（YTD）表现跑赢了加密货币市场。

虽然大多数市场年初至今都经历了修正，但 NFT 市场的表现一直优于加密货币市场，以 ETH 计价的 NFT市场年初至今表现为 68.5%，以美元计价的年初至今表现为 20.9%。加密货币市场方面，根据比推终端数据，自2022年 1 月 1 日以来，比特币下跌了大约17%，以太坊下跌了31%以上。

然而，在过去 30 天（截至 2022 年 3 月 9 日），NFT 市场经历了一次调整，以 ETH 计价时下跌 -28.8%，以美元计价时下跌 -38.5%。

2. NFT 与 DeFi 成反比

NFT 与其他资产之间的相关性（2022 年 1 月 1 日至 3 月 9 日）：

年初至今，Nansen NFT-500 (ETH) 指数与加密货币市场呈负相关。上图显示， NFT 500 指数（以 ETH 计价）与比特币的相关系数为 -0.46。同一指数与以太坊的相关性为 -0.6（也以 ETH 计价）。然而，当指数以美元计价时（Nansen NFT-500 (USD)），则与加密货币市场的相关性似乎呈正相关，尽管相关性相对较弱。以 ETH 计价的 NFT 与去中心化金融 (DeFi) 加密货币领域的反比关系稍微显著。

当测量相关系数时，“1”表示直接的正相关，这意味着两个变量的走势是完全趋同的。相反，“-1”表示这两个变量正朝着相反的方向移动。

造成这种差异的主要原因是波动性。Nansen 研究分析师 Louisa Choe 表示：“当 NFT 以美元计价时，它们的表现也有很大差异。这是由于 ETH/USD 汇率的波动。因此，由此产生的相关系数也不同。这可能是投资者在决定其投资组合的面额应该是加密货币还是法定货币时的一个重要因素。”

3. Shiba Inu 比 NFT 年初至今的波动性更大

在使用 Nansen NFT-500 (ETH) 和 NFT-500 (USD) 指数分析 NFT 市场的波动性时，我们观察到（meme coin）Shiba Inu 比 NFT 年初至今的波动性更大。此外，在探索以太坊的历史价格波动时，以太坊在 2016 年的价格走势似乎比当前的 NFT 市场波动更大。而在 2018 年，以太坊的波动性与 Nansen NFT-500（美元指数）的波动性相似。然而，重要的是，用于计算年初至今波动率的数据点可能不足以得出具体结论。

4. 年初至今社交 NFT 引领 NFT 市场

社交 NFT 的市值估计为 95 亿美元（357 万 ETH），约占 NFT 市值的 79%。该指数的表现突出了投资于社交 NFT（如个人资料图片）时多元化的好处。对于社交 NFT，我们观察到 NFT 的品牌认知度和感知建立是推动其绩效的重要因素。

截至 2022 年 3 月 9 日，Nansen 艺术 NFT 指数的回报率为 30.8%（以 ETH 计价）和 -6.1%（以美元计价）。同一时期，社交-100 指数的回报率为 93.7%（以 ETH 计价）和 39 %（以美元计价）。

5. 元宇宙和游戏 NFT 的波动性最小，而蓝筹和艺术 NFT 是 YTD 波动性最大的。

不同 NFT 类别的每日波动率：

NFT 类别之间的波动性不同。该指数的表现显示，元宇宙和游戏 NFT 是年初至今波动最小的，而蓝筹和艺术 NFT 在此期间波动最大。

具体来看，艺术 NFT 的每日波动率为 36%（ETH）和 37.3%（USD），元宇宙和社交代币的波动性最小，远低于 10%。

尽管将 NFT 整合到游戏和元宇宙中是一个新的趋势，但这些市场已经成熟并且拥有不断扩大的用户群。假设这些市场的成熟度，以及支持这些领域的其他公司（例如 Facebook 更名为 Meta、灰度提供 Decentraland (MANA) 信托，以及 a16z 对 Yield Guild Games 的投资）的动作，导致这些类别的 NFT 波动性相对较小。另一方面，蓝筹和艺术 NFT 波动最大的一个可能原因是，著名的蓝筹收藏和独特的艺术 NFT 作品相对缺乏流动性；造成价格信息不对称。简而言之，评估这些 NFT 收藏品更难，尤其是那些稀有的作品。因此，这些市场参与者中有许多行为表现得像投机者。
