金色财经 区块链12月28日讯    自 2021 年以来,NFT热潮席卷全球。与此同时，越来越多的传统艺术家开始公开抗议自己的作品被窃取。抗议声称，他们的作品遭窃取之后被铸造成了 NFT，并在 OpenSea 上进行出售。

但有些人认为，也许会有一种解决方法——这种方法能迫使用户认真地思索下关于 Web 2 和 Web 3 的争论。

是时候作出改变了

Twitter用户Batsy [@batbandage]  声称，他创作的艺术品被人放在了 OpenSea 上并铸造NFT，在过去很长一段时间以来，他被盗的艺术作品都没有被下架，因此他填写了谷歌的事件报告表进行投诉，但谷歌在后续的回应中，仅仅是删除了他们被盗作品的图像，而 OpenSea 页面本身却没有被删除。

这篇帖子被迅速传播开来，其他在 OpenSea 上看到自己作品被盗的艺术家表示他们也将尝试同样的做法。

@NFTtheft @martinmcneil 成功了！你可以打开这个网址，通过谷歌的自动化系统删除NFT https://t.co/QA8wPw9pdw（无需电话号码或是地址），但他们删除的是你的实际图像内容，而不是页面本身！? pic.twitter.com/6LwFAbxqQc

— Batsy (@batbandage) 2021 年 12 月 20 日

虽然 OpenSea 上也有自己的程序来供用户投诉欺诈行为，不过用户表示对该功能非常失望，因为投诉过程需要提供大量的个人信息。

下一战：Web 2 vs Web 3

也许，对大多数人来说，那些被盗的数字图片似乎没什么大不了的，但要知道，OpenSea 今年通过加密货币赚到了 150 多亿美元。此外，根据 ConsenSys有关Web 3的报告显示，上一季度，NFT 交易量占据了整个市场交易量的 97%。

然而，上文的Twitter帖子也提出了另一个有用且令人相当担忧的问题——谷歌真的能“删除”去中心化加密资产吗？随着越来越多的加密货币观察者和分析师加入到 Web 2 vs Web 3 的口水战中，这类例子起到了至关重要的作用。

也许，加密开发者和建设者们还需要扪心自问一下，区块链项目能否在国际法和搜索引擎监管下的 Web 2 环境中成功发展。

Visa现在变成了数字艺术评论家？

实际上，支付巨头 Visa 也一直表现出对加密领域的兴趣，并打算投身其中。该公司的加密部的负责人 Cuy Sheffield 说：“NFT 有可能成为创作者经济强有力的加速器，并降低个人创作者通过数字商务谋生的准入门槛。未来，用户的加密货币地址和电子邮件地址将具有同样的价值和重要性。”

根据Cuy Sheffield的说法，Visa 目前正在密切关注 NFT 领域，因为随着越来越多的用户加入这一狂潮，NFT领域已呈现出一种快速发展和增长的态势。 他认为，非同质化代币给企业提供了前所未有的新思路，而中小型企业 (SMB) 是最有可能从中受益的。通过NFT，公司可以以较低的前期成本生产和销售数字商品，并且这些商品可以被立即发送到用户的数字钱包。

值得一提的是，此前 Visa 公司还购买了价值 150,000 美元的 NFT。该公司认为，非同质化代币将在零售业、社交媒体、商业和娱乐领域发挥起重要的作用。

然而，那些自己作品被盗的传统艺术家可能不会答应。所以，如果 NFT 市场想要获得主流认可，可能还需要加强安全监管措施，严厉打击盗窃艺术品的行为。
