NFT，近两年最有趣，最独特的资产类别之一。

NFT为加密市场带来了技术、金融、文化和社区，然而，一系列广泛的扩张也带来了可扩张性的问题。

基于此，一个有趣的项目诞生——Gem.xyz，一个NFT 聚合器。

什么是NFT 聚合器？

传统上，聚合器用于数据聚合，这是指跨各个网站收集数据，允许将信息以不同类别呈现在一个地方，以供不同用户使用。

它本质上是一个搜索引擎，收集和编译有用的信息，并删除不相关的信息。使用机器学习，他们还可以创建有价值的指标，这些指标是通过处理后的数据计算出来的。 

专业的 NFT 聚合器只需从不同的链收集所有 NFT 交易信息，并将它们整合到一个地方。这提供了优化的交易体验并提高了效率和有效性。 

NFT 聚合器的一些主要好处包括： 

  1. 一页搞定—— 用户可以在所有市场上查看、交易和购买 NFT。用户可以查看所有内容，从当前的 NFT 数据到特定类型 NFT 的交易量。 

  2. 提高搜索效率—— 用户可以轻松找到所有交易所的最佳交易，包括 OpenSea、Rarible、Larva Labs 等。 

  3. NFT 分析—— 用户可以在一个地方找到他们需要的所有分析。他们可以找到诸如销售量、底价、顶级持有者、顶级买家等信息

  4. 优化—— 用户可以在 jpeg 而不是 gas 上花费更多的代币。用户还可以使用他们想要的任何代币进行支付。 

那么什么是 Gem.xyz？  

Gem 希望从以太坊区块链开始为用户提供内含多个 NFT 市场平台中的大量供应量以及最优价格的 NFT 聚合平台。

Gem 在去年 12 月 28 日上线 Beta 版，能帮助用户通过 Uniswap V2 和 V3、SushiSwap 以及通过 0x 的任何流动性，也因此支持用户在交易 NFT 时使用持有的多种 ERC-20 代币进行支付，也可以组合不同代币支付。

目前，GEM 直接集合了多个平台 (其中包括 OpenSea、Rarible、LooksRare、X2Y2、NFTX 和 NFT20) 上的 NFT 挂单。其最终目标是整合所有 NFT 市场的挂单，用户再不需要平台间反复横跳的比较价格。

自从 Gem 以跨资产互换的身份起步以来，它已经完善了产品以提供更多好处。以下是他们推出的一些最新和最新功能： 

  • Gem 与 Dune Analytics 合作，为专业的 NFT 交易者提供出色的 NFT 分析。这些范围从现场销售和铸造活动的各种利基统计数据，甚至是顶级持有者分布。以Bored Ape Yacht Club为例，可以轻松拉取销量、底价等信息。 

  • 还可以看到诸如 Live Activity 之类的销售活动。这些只是众多可用功能中的一部分。 

  • Gem 集成 Rarity Sniper，以实现在 Gem 平台显示 NFT 稀有度排名等详细信息。

  • Gem 通过提供多个市场的聚合，与直接从 OpenSea 市场购买相比，用户使用 Gem 可节省高达 39% 的gas费。用户在购物车中购买的每件额外商品都会节省gas费。 这是使用有趣的编程语言Yui完成的。通过以太坊的低级编程语言优化他们的智能合约有助于创新的智能合约，使更多人能够访问NFT。

  • 用户还可以批量购买/销售NFT，并从单个仪表板管理其所有列表。

  • 专业NFT交易员也可以使用扫描模式。用户可以通过选择扫描的项目数量或ETH预算来自动扫描NFT集合。为了快速进行交易，用户甚至可以使用“立即购买”按钮立即使用以太坊结账，并跳过整个付款流程。

  • 新增安全模式，该模式使用 mixtX Flashbots API，旨在保护用户在批量扫货时遭遇三明治攻击，或避免发生因单独 NFT 交易失败导致批量扫货失败并浪费 gas 费的情况。

路线图

虽然 Gem.xyz 目前没有路线图，但它正在开发中。有趣的功能之一是它确实具有反馈系统。

用户可以请求平台上的功能，提供错误报告，还可以请求稀有度排名请求。由于社区在该项目中有发言权，因此这一直是许多人关注的焦点。 

用户还可以查看计划中和进行中的活动。社区经常以交互方式更新，并且 Discord 内部的支持也非常出色。 

另外值得一提的是，用户还可以为其他用户的建议投票。

另有消息称Gem或将推出IC0，代币持有者或将获得更大的治理权，同时还可用来支付gas费。不过，目前没有具体的发行时间。

结语

由于其优化和改造 NFT 行业的能力， NFT聚合器Gem.xyz绝对是一个值得关注的项目。

从长远来看，它计划构建其他很酷的分析，以使 NFT 生态系统的整个格局受益和改变，包括个性化和共享您的收藏的方式、智能发现以及简单且无需许可的交易。

但是，由于它相对较新，因此存在一些限制。例如，一个问题是交易失败。当用户尝试使用 Gem 进行购买时，他们可能正在尝试购买其他人也在争夺的 NFT。

有时多人试图购买相同的资产意味着只有一个人最终会得到它，这取决于谁使用更多的gas费。 

Gem.xyz 在社区的大力支持下具有巨大的潜力。虽然这是一项正在进行的工作，但随着 NFT 格局和行业的改善，监管更加清晰，我们或将看到一个更好的 Gem.xyz 。
