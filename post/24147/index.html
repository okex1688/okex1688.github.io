Plasm Network 和 Shiden Network 背后的核心公司 Stake Technologies 宣布获得 1000 万美元资金的支持，由分布式资本领投，其他知名投资者也参与其中，包括 Hypersphere Ventures、Gumi Cryptos、IOSG Ventures、TRG Capital、AU21 Capital、Digital Strategies、SNZ、 Digital Strategies、Sub0 Capital、Altnomy、East Ventures 等。这轮融资还包括著名的天使投资人，索尼前首席执行官兼董事长 Nobuyuki Idei。

Stake Technologies 的首要任务是帮助 Shiden Network 在 Kusama 上赢得平行链插槽和帮助 Plasm Network 在 Polkadot 获得平行链插槽！随着 Kusama 的租赁拍卖即将到来，最新一轮筹集的资金将用于强有力的支持和确保赢得平行链插槽。

Polkadot 生态系统中的项目正在采取两种主要策略来来赢得平行链插槽：直销和 crowdloan。在前者中，新网络的代币直接出售以换取相应平行链网络的原生代币，即 DOT 或 KSM。在后者中，代币持有者“借出”他们的 DOT 或 KSM 作为平行链租赁的抵押品。在租赁期（从 6 个月到两年不等）结束时，代币将返还给其原始持有人。在 Kusama 上，租赁期的上限约为 11 个月。

Stake Technologies 采取了以 crowdloan 为中心的策略去赢得 Shiden 平行链租赁，由于平行链拍卖的性质，需要激励 KSM 持有者尽早做出贡献。高达 22% 的 SDN 总供应量将在第一次 crowdloan 中分配，其中 2% 保留用于早鸟奖金。只要平行链拍卖处于活跃状态并且还没有得到插槽的时候，KSM 持有者就一直可以将他们的代币提交到网络。

“自 2019 年与 Plasm 团队见面以来，当时 Stake Technologies 是最早在 Longhash 加速器中基于 Substrate 构建的团队之一，我们对团队的迭代和执行能力，对他们将可扩展的智能合约平台引入 Polkadot 生态系统的愿景有深刻的印象。”分布式资本合伙人 Remington Ong 说。“当我们临近第一次 Kusama 插槽拍卖时，Fenbushi 很高兴支持 Shiden 和 Plasm Networks 竞标成为首批 Kusama 和 Polkadot 平行链之一。成为平行链只是一个开始，我们期待团队继续为 Web3 生态系统的发展做出贡献。”

Plasm Network 及其表亲 Shiden 旨在成为 Polkadot 和 Kusama 的多链 DApp 中心。这两个网络将成为其他网络、二层平台和平行链跨链资产的网关。Plasm 和 Shiden 支持 Wasm 和 EVM 上的智能合约，为开发人员提供最大的灵活性。Plasm 和 Shiden 的原始实现之一是 DApps Staking，它允许智能合约开发人员获得基本收入。每个区块奖励的 50% 以上会根据 dApps 的开发者的表现和受欢迎程度分配给他们，开发者通过创建应用程序来赚取代币。我们团队将在 6 月底之前发布 Plasm Network Portal。这允许开发人员使用 GUI 部署和管理智能合约。

Plasm 团队最近在测试网上完成了 PoS 的迁移，并在 Cosmos 和 Polkadot 之间建立了第一个桥链。一旦 Shiden 成为 Kusama 平行链，核心团队将与微软一起启动 Shiden Azure 计划和 Shiden 加速计划。

Hypersphere Ventures 合伙人 Jack Platts 表示：“我们一直对 Plasm 团队的技术实力印象深刻，并且很高兴能以平行链拍卖方式支持他们。” “多链意味着 Plasm 和 Shiden 将成为整个平行链生态系统的关键贡献者，他们在扩展方面的工作将惠及所有其他未来的平行链。”

“gCC 寻找世界级的建设者和企业家，我们称之为 gumicorn 创始人，”gumi Cryptos Capital 的普通合伙人 Miko Matsumura 说。“ Plasm Network 团队代表了我们很荣幸能够支持的那种勇气、不懈的努力、远见和卓越的技术。”

关于 Plasm

Plasm Network 是 Polkadot 上的 DApps 中心，支持以太坊和 ZK Rollups 等 Layer2 解决方案。Plasm 旨在成为一个多链智能合约平台，将支持多个区块链和虚拟机，如 EVM 和 WASM。
关于分布式资本

分布式资本是亚洲第一家也是最活跃的专注于区块链的风险投资公司。它由区块链和传统金融行业的资深人士于 2015 年在上海创立，迄今为止，它已经支持了 6 大洲的 150 多个领先项目，利用区块链技术重塑了金融、医疗保健、供应链和消费品等众多行业。分布式的使命是推动全球 Web 3.0 生态系统的健康和可持续增长，并积极成为其投资组合项目的长期战略合作伙伴。
