一战成名NFT

什么是 NFT 头像？

NFT 头像是通过算法生成的卡通或像素化角色的数字图像，通常从肩膀向上，采用“个人资料图片”格式。每个 NFT 化身都是完全独特的，具有一系列不同的特征（即服装、发型、配饰等）。化身可以是卡通动物、人类或抽象生物，具有奇幻和科幻元素。 

为什么 NFT 头像如此受欢迎？

以近期的一个新项目“猿猴头像”为例，OpenSea数据显示该其销售额已突破4亿美元，交易人数达5400人，最低价达16万美元。也就是说，4个月时间其价格暴涨超500倍。

今年4月份推出的猿猴头像

2020年，NFT的市场规模仅为3.38亿美元。到了2021年第一季度，这个数字变成15亿美元**。Coingecko数据显示，目前NFT的市场规模已超200亿美元（约合人民币1300亿元）。**

OpenSea最新数据显示，过去1个月里销售额TOP10产生了约69万ETH的销售总额折合约27亿美元，其中销售额排名第1、3、4、6、7、9的6个项目均为加密头像NFT项目，销售额占TOP10总额的64.5%，约合17.4亿美元。

加密头像NFT以稀缺性+朋克文化+叙事+社区+新商业价值的代表+创新玩法而持续受到欢迎。

区块链投资机构Mechanism capital创始合伙人Andrew kang曾撰文，将NFT头像比作奢侈品、艺术品和高端房产，并暗示未来NFT市场规模将超过400亿美元。

其实一直以来，不少人都在思考类似的问题，为什么几百个像素组成的简单头像却能值那么多钱？那些花天价购买像素头像的人到底在想什么？ 目前我们能看到的，比较显著的理由有以下几点：

a. 加密朋克精神

像素头像其实最早起源于加密朋克精神。加密像素头像的早期创作者们曾表示，这其实是想要表达一种“朋克精神”，他们想在Punk的外观中反映出这种特殊的文化。20世纪70年代的伦敦Punk运动感觉就像是正确的审美观。

我们所熟知的许多加密货币如比特币等，实际上也是在早期加密朋克精神的影响下诞生的，由一群富有加密朋克精神的极客们推动而来。

b. 元宇宙身份的象征

在人类社会不知不觉的飞跃发展中，现代社会逐渐进入了数字化时代。元宇宙，作为当今科技界最火的概念之一，引发了跨越行业、跨领域、多学科的的多元探讨。

未来，进入以加密技术作为底层支撑的元宇宙的第一步就是需要进身份识别”以将虚拟世界和现时世界的身份进行对应。

支付巨头Visa斥资万美元购入加密像素头像的同时宣布入局NFT领域，一石激起千层浪掀起了一阵阵NFT热潮，Visa这一直接拿出实际行动”下场收购”的动作。

这不仅代表了Visa对NFT领域和加密像素头像背后朋克精神的认可，也可谓是其最好的入局宣言。就像孙宇晨天价收购Tpunks头像一样，对其创立的波场大生态也是意义深远。

c. 社区影响力的象征

在新商业价值中，社区影响力尤为重要。 加密头像现在已经成为了加密领域头部领袖、大V等爱好者们的收藏对象，把头像悬挂在社交平台上更是彰显了一种身份的象征，还能增强其在社区和业内的影响力。

许多人还将其视为了加密“俱乐部”的通行证，过去加入一些顶级俱乐部需要各种门槛以证明你由资格进入。

而现在只需要可被验证的加密头像就可以了，因为只有认知到位的加密圈顶层人士才会认可其价值并重金收购 ，而这一圈层的人开始以建立相同兴趣爱好的群组，而加密头像就是进入这个群组、这个圈子的最好通行证。

总的说来，加密头像NFT的价值和受欢迎的主要因素是：稀缺+朋克文化+叙事+新商业价值+社区作用+创新玩法+金融工具等等。

除了作为独特资产的可收藏性和内在价值之外，NFT 之所以受欢迎，原因如下： 

  • 社区：NFT 收藏通常有围绕它们建立的社区。拥有一个头像通常是进入这些社区的门票，其中一些是非常独特的。例如，Bored Ape NFT 授予持有者访问 Bored Ape 游艇俱乐部“浴室”和 Discord 频道的权限，以及许多其他特权和独家福利。 

  • 状态：投资者收藏中的 NFT 头像反映了该收藏家的数字状态。例如，OG CryptoPunks NFT 化身自 2017 年首次亮相以来就广为人知，拥有其中一个并公开展示它在加密社区中是一种严重的灵活性。

  • 稀缺性：每个单独的 NFT 都是完全独特的。您可以随意复制和粘贴头像，但只有代币持有者才能声明真正的所有权。稀缺性和数量有限有助于推高 NFT 的价格和需求。NFT 越老越稀有，它就越令人垂涎和昂贵。如果一个系列中只有一个头像升值了，那么整个系列通常都会效仿。许多收藏家和爱好者将 NFT 头像视为一种长期投资。 

NFT 头像有什么用途？

NFT 化身可能看起来像卡通或像素化角色的简单数字头像——但它们远不止这些。除了 NFT 头像的典型特征（例如从肩膀以上显示角色头像的头像格式）之外，每个头像都是完全独特且不可复制的。 

化身是使用程序生成的特征创建的，以保证它们的唯一性，无论是在表面级别的视觉上还是在代码级别的内部。NFT 的独特性使其可用于超越简单美学的多种用途。NFT 头像的一些突出用例示例包括： 

  • 游戏：2021 年发布的几款游戏即赚钱加密游戏使用 NFT 来创建玩家可以在 NFT 市场和游戏内虚拟世界中拥有和交易的收藏品和物品。 

  • 筹款：一些非营利组织已将 NFT 作为创收的一种形式。一些人正在铸造 NFT 作为对捐赠者的奖励，而另一些人正在创建 NFT 并将其拍卖作为筹款活动的一部分。

  • 赌博：赌博业也在利用 NFT 趋势获利。例如，Gambling Apes最近向赌博爱好者出售了 7,777 个 NFT 头像的集合。持有者可以参加独家在线赌场，参加比赛、派对等活动。 

  • 加密艺术：数字艺术世界可以从 NFT 中获益良多，作为一种追踪商业权利所有权和确定真实性的新方法。围绕 NFT 的热潮大部分源于加密艺术的销售。例如，2021 年初，数字艺术家 Beeple在佳士得拍卖行以 6930 万美元的加密货币价格售出了一件 NFT 艺术品。

NFT 头像示例

几乎每天都有新的 NFT 集合推出，但一些值得注意的例子在加密社区引起了巨大的轰动。这些包括： 

无聊猿游艇俱乐部

Bored Ape Yacht Club (BAYC) 是一个于 2021 年 5 月启动的 NFT 项目。它拥有 10,000 个独特的猿化身，每个化身的价格为 0.08 ETH（当时约为 270 美元）。 

拥有 BoredApe NFT 的持有者可以访问一个特殊的“浴室”，这是 BAYC 官方网站上的一个虚拟环境，用户可以在公共墙上写数字涂鸦。 

加密朋克

CryptoPunks是最初的 NFT 化身项目，也是Larva Labs于 2017 年 6 月在以太坊上发布的首批 NFT 化身项目之一。任何拥有以太坊钱包的人都可以免费领取这 10,000 个独特的像素化头像。如今，CryptoPunks 已经取得了传奇般的地位，进入了主流艺术界——甚至在苏富比和佳士得等主要拍卖行的拍卖会上成为头条新闻。 

CryptoPunks 已成为 NFT 世界的劳斯莱斯，即使是价格最低的 CryptoPunk 也能卖到数万美元。最稀有的——猿类、外星人和僵尸——价值数百万美元。 

哈宝头像

Habbo Avatar是一个包含 11,600 个像素化头像的 NFT 集合。这个 NFT 头像项目是由一群像素艺术家设计的，他们是两个在线游戏的创造者：Habbo和Hotel Hideaway。团队的长期计划是让每个化身在未来都可用于游戏中的虚拟世界。 

酷猫

Cool Cats是一个 NFT 项目，拥有 10,000 个以猫为特色的独特 NFT 化身。这些猫很受欢迎，最昂贵的 Cool Cats NFT（如上图）售价为 19 ETH。Cool Cats 背后的团队计划在未来加入一种繁殖机制，例如CryptoKitties上的繁殖机制。 

米比特

Meebits是 Larva Labs 的另一个作品。他们的化身是 3D 角色，旨在用于元节。Meebits 于 2021 年 5 月推出，受到更广泛的 NFT 社区的大量批评，因为铸造的 20,000 个 NFT 中有一半以上被保留为CryptoPunk 持有者的免费奖励。其余的收藏品在荷兰拍卖会上以 2.5 ETH 的铸币价格向公众出售，当时大约相当于 10,000 美元。 

如何获得 NFT 头像？

获取 NFT 头像的方法有很多种，从使用头像生成器制作自己的头像到从 NFT 市场购买现有头像。下面，我们来看看一些流行的选项。 

NFT 头像生成器

一些网站已经出现，为用户提供了一种使用不同艺术风格快速生成各种新头像的简单方法。流行的 NFT 头像制作者的例子包括8biticon和NFT Avatar Maker。

流行的 NFT 头像市场  

有许多NFT 市场，您可以在其中购买现有的 NFT，或列出您自己的 NFT 进行销售。一些最受欢迎的包括OpenSea、AtomicMarket、Async Art、SuperRare、Rarible和Decentraland Marketplace。 

创建和出售您自己的头像

创建和销售 NFT 的过程类似于在 eBay 等传统市场上列出和销售商品。首先，您设置一个以太坊帐户，然后在 OpenSea 等 NFT 市场上创建一个帐户。 

一切设置好后，您就可以“铸造”您拥有的数字媒体文件的 NFT。这可以包括图像、GIF、视频或音频文件。您必须向以太坊支付汽油费才能完成铸造过程，但是一旦创建了 NFT，您就可以立即将其挂牌出售。 

NFT 头像是一项好的投资吗？

NFT 提供了一个独特但高风险的机会来赚取巨额利润，但在加密世界中没有任何保证。作为一种投资工具，NFT 更适合风险偏好较大的投资者。 

NFT 虚拟化身除了市场赋予它们的价值之外并没有真正的内在价值，这就是它们的价值不断波动的原因。一旦你获得了 NFT，这是一个等待游戏，看看另一个投资者愿意为你支付多少费用。
