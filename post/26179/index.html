要说最近在加密世界最火爆的概念，NFT绝对是当仁不让。上个月末（6月23日），阿里巴巴与敦煌美术研究所联合发布了两款 NFT非同质化Token，分别为敦煌飞天和九色鹿皮肤 ，全球限量发行共16000 个，两款各限量8000个。

由于购买门槛很低，只需要10支付宝积分+9.9元就可以进行兑换，所以一经推出，就遭到了网友的疯抢，迅速抢光。而在24日，继敦煌美术研究所·支付宝联名款NFT皮肤推出后，支付宝再次推出知名动漫作品《伍六七》支付宝付款码皮肤 ，也再次遭到了疯抢。

就这样，支付宝推出的这几款NFT付款皮肤形成了一定的买卖需求。当天就有人将得到的皮肤在闲鱼平台上挂卖，进行二手交易，有很多根本还不清楚NFT究竟是什么的用户也加入了这场“战争”。

价格从刚开始的几十几百元，到后来的甚至出现了几十万的价格，而敦煌飞天 NFT甚至被炒到了150万元一个。

最后，由于价格过高市场过于疯狂，甚至影响到了正常的市场秩序。 很快闲鱼就出手制止，紧急下架了相关的NFT商品，部分用户曾在闲鱼发布将数字艺术品代理上传至 OpenSea 等 NFT 交易平台的服务也被迫下架了。

虽然这场“战争”已经落下了帷幕，但“后劲”和影响可不小。 这次支付宝推出的限量NFT皮肤，算是让NFT真正地走到了大众面前，让其在国内汇聚行业内部人员的目光后，也吸引了越来越多行业外的关注。

NFT是什么？

NFT 其实就是 Non-Fungible Token「非同质化代币」，最简单的理解就是“不同样的代币”，其意义为区块链凭证，它的主要特性为不可分割性、不可替代性、稀缺性以及独一无二。

NFT最常见于文化艺术品领域，发挥知识产权的链上发行、流转、确权等作用，能有效保护知识产权，防止篡改、造假等，是区块链技术的一类重要落地场景应用。             

它的起源可以追溯到2012年到2013年，当时出现了小面额的彩色比特币，这些彩色比特币内嵌了比特币的脚本语言，给元数据中授予了特定的属性编码。

这样，这些彩色币就可以代表任何不同的资产，无论是像美元、欧元的货币，还是股票、债券这样的金融凭证，还是艺术收藏品。

在2015年到2016年，交易卡片和MEME开始火了起来，各种电子游戏也让存储在区块链技术的数字资产开始普及起来。

2017年，加密猫（CryptoKitties）的出现，让大众开始注意到NFT，虽然当时NFT的体量并不显眼，但适逢牛市，加上加密猫爱好者的狂热推动，让一些稀缺的加密猫甚至上涨至了10万美元，市场的疯狂，甚至一度导致以太坊网络严重拥堵。

由此，大众开始注意到了NFT， 虽然当时NFT的体量非常小，但是快速发展的势头仍然让大批区块链领域从业者趋之若鹜。

到了2018年到2020年，NFT早期的生态初成，在这三年间NFT的市场规模增长了825%，活跃地址数增长了201%，买家增长了 144%，卖家增长了 113%。虽与其他加密货币市场相比，NFT 市场交易量还很小，可能只有1%都不到，但其发展趋势显著。

2020年，基于 Flow 开发的 NFT 数字收藏品 NBA TOP Shot 上线仅半年，成交量便逾 4200 万美金；2021年，越来越多的NFT数字艺术品被高价卖出，这些NFT数字艺术品、数字收藏品的高价拍卖或售卖，让NFT逐渐出圈，从小众逐渐走向大众。

NFT也被认为是2021年续DeFi的下一个最热风口，目前来看事实也是如此的。

支付宝的特殊NFT

回到我们开头提到的“支付宝NFT”，虽然支付宝推出的限量皮肤NFT已被下架，这意味着这些NFT在购买后是无法交易和转手买卖的，也就大大减少了其增值的可能性 ，这和我们常说的区块链当中的NFT是有明显的差异的。

另一方面，由于支付宝的NFT数字作品版权是由发行方或者是原创作者所拥有的，除非取得版权书面同意，否则是不可以用作商业用途的。

与此同时，蚂蚁链的NFT似乎是并不直接与用户产生关联的，因为合约是由支付宝进行管理的，也就是说用户购买了这个NFT，只是拥有了它的使用权，而无法拥有它的所有权，也就无法随意进行交易转卖。

这与区块链中的NFT一旦用户买下，这件NFT商品的所有权便完全属于买方，包括流通、转让、商用等权益有着本质的区别。

所以，支付宝这些NFT作品虽上了蚂蚁链（区块链平台），但就目前来看，这不能等同于我们常说的区块链行业中的NFT，因为它们之间有本质上的不同。

尽管如此，支付宝的“特殊”NFT在很大的程度上推动了整个NFT生态的发展，这是无疑的。 它让NFT更多地让普通的大众知道或是去了解，而不只局限于行业内的人员。

各路“大厂”纷纷布局NFT

其实除了支付宝，还有许多公司都布局了NFT领域，只不过在国内是由于支付宝蚂蚁链发行的这几个NFT才让大家开始真正注意到这个板块。

阿里

早在去年7月23日，蚂蚁集团就举行了蚂蚁链全新品牌发布会，将蚂蚁区块链正式升级为“蚂蚁链”了，并且宣布他们的愿景是“要让区块链像移动支付一样改变生产生活”。甚至在马云第一次作为淘宝主播现身主播间时，也不忘顺手“带货”阿里的蚂蚁链。

另外，在与支付宝合作推出限量付款皮肤NFT前，阿里就已经在其淘宝阿里拍卖平台的聚好玩520拍卖节推出NFT数字艺术公益拍卖专场。 在6月中旬又再次推出了“6月青年艺术家数字作品”拍卖活动，只不过这些活动还是比较小众，没有多少用户知道，更别说NFT的拍卖了。       

据资料显示，阿里拍卖的“NFT数字艺术”也已于7月3号上线，上线拍卖的作品以绘画作品为主，价格从一元人民币到几千不等，参与的人不是太多。另外，区块链游戏和NFT开发商Animoca Brands在上星期宣布新获得了一笔高达5000万美元的融资。

蓝池资本

有很多人注意到，除了Coinbase等传统加密行业巨头外，一个低调的投资机构—Blue Pool Capital(蓝池资本）也对其进行了投资。

蓝池资本成立于2014年，它是由马云以及阿里巴巴的核心创始人蔡崇信等联合设立的家族财富基金，被誉为是马云巨额财富背后的“神秘大管家”，团队只不过十几人，但却管理运营着数十亿美元的资产。

从其过往的投资行业来看，医疗赛道一直都是其重仓的赛道，所投资的8家企业中，医疗健康达到了5家，其余为互联网消费行业。

这次投资NFT开发商Animoca Brands,是蓝池资本时隔三年之后，今年至今第二次公开披露的投资，而且也是蓝池资本的第一次涉足区块链领域，就选择了NFT板块。

网易

继支付宝推出联名NFT后，网易也联合了Nervina Labs 在淘宝发行他们的首个NFT作品。该作品通过网易文创旗下的《三三工作室》在6月26日下午13点33分发行其首个IP向NFT作品——小羊驼三三纪念金币。

该NFT全球限量333枚，售价133元，在淘宝的「王三三周边品牌店」进行发售。

从外观上来看，这个NFT纪念币还是挺可爱的，全球限量333枚，也具有很大的稀缺性。据客服介绍，购买的用户可以通过注册基于 Nervos 的 NFT 平台「秘宝」接收该NFT。

而其实早在五月份，网易文创三三工作室就已经与基于Nervos公链开发的NFT平台展开了IP合作，在六月份下旬的这个时候推出，明显是想借支付宝的东风。另外，网易游戏授权澳洲NFT发行商MetaList Labs发行的《永劫无间》NFT，也即将上线币安。

当然，除了国内“大厂”，国外“大厂”也不会错过这个巨大的商机，也纷纷布局了NFT，甚至可能还早的多。

推特（twitter）

在6月30 号，全球领先的头部社交网络媒体 Twitter 推出了免费送 NFT 的活动。将免费送出140个新设计的NFT（非同质化代币）。

这140个NFT，由7款不同的NFT组成，每款NFT会有20个版本，推特通过在第三方数字资产交易平台Rarible上发布这些NFT。

Twitter发言人表示，Twitter将根据在Twitter或Rarible上对NFT表现出兴趣的人来选择这些NFT的受赠者。 Twitter不会出售这些数字图片，也不会从中收费。目前，该条信息，已经得到了近3万条评论，近两万次的转推。

说起推特进军NFT，不得不提到，推特首席执行官Jack Dorsey 在今年3月份以令人震惊的290万美元高价卖出了他在推特的2006 年的第一条推文的NFT，从这来看，推特应该是早就有进军NFT的想法了。

不过除此次的项目之外，推特暂时没有其他与NFT相关的项目或新功能。 但这不意味推特不会更近一步地去探索NFT，要知道Jack Dorsey 可是数字资产，比特币等的坚定信徒。

Instagram

据开发人员Alessandro Paluzzi在推特上爆料称，Instagram似乎正致力于在其应用程序上部署NFT功能，以及端对端的加密聊天功能。届时提供NFT的Instagram帖子将带有“收藏品（Collectible）”标签，这将向用户显示帖子中有可购买的NFT。

目前，Instagram尚未公开宣布在其平台上推出NFT产品。不过，该公司已经提到可能会在平台上增加支付功能。 此前，在今年的五月中旬，加密艺术家Sean Williams 也曾在其推特上透露类似消息。

他表示Instagram或将推出NFT平台，目前正与艺术家沟通签订相关协议。SeanWilliams称其收到了海外社交平台Instagram官方发来的邮件，Instagram在邮件中表示希望得到他和一些新兴加密艺术家的帮助来建立Instagram的NFT平台。

Instagram会给每一位提供帮助的艺术家1000美元的酬金，但同时也要求签署保密协议，禁止向外界公开他们在幕后帮助Instagram时付出的努力。关于消息的真假，只有Instagram公司知道结果了。

除了互联网“大厂”正在布局NFT，还有其他的体育文娱行业甚至一些传统行业也开始布局NFT。

比如像NIKE,它甚至比很多互联网“大厂”都要早进入NFT领域。早在2019年底，运动品牌巨头Nike就公布了一项名为「CryptoKicks」的专利。引发了对于「加密鞋」的一波探讨。

Nike的灵感思路来源于CryptoKitties，这项名叫 CryptoKicks 的专利，它会生成唯一的 ID 并为鞋子创建以 ERC-721 为标准生成的 NFT。这颗 NFT 除了代表虚拟球鞋，还可以记录它的 DNA，包括特定的属性、颜色、样式、背景。

并且，用户还可以对购买的NFT鞋子进行“饲养”，既把虚拟球鞋储存在加密钱包或数码鞋柜里，将球鞋配种或「繁殖」其他的球鞋生成「球鞋后代」，只要厂商可生产，这些繁殖出的球鞋后代将成为全新的实体球鞋。除了NIKE,另一运动品牌巨头Addidas也在布局NFT。

除了运动品牌，不少的体育明星也都涉及到了NFT领域。 如NBA顶级球星勒布朗·詹姆斯（Lebron James）和其他NBA球员的比赛“高光时刻”视频NFT就在NBA Top Shot上创下了超过2.3亿美元的销售额。

除了体育领域，NFT在文娱领域也广受欢迎。 如在5月18日，作为世界娱乐业的巨头，美国福克斯娱乐集团（Fox Entertainment Group）就宣布将成立名为「区块链创意实验室（Blockchain Creative Labs）」的 NFT 工作室，正式进入区块链NFT。

该工作室将会以策划和销售数字商品为主，包括 NFT、GIF 以及提供社交体验以吸引和奖励粉丝的代币。接下来，工作室将会推出首部完全在链上进行的动画《Krapoplis》。

在传统行业中，奢侈品牌毫无疑问是NFT领域的先行者，比如奢饰品巨头之一，GUCCI就已经正式宣布他们将在近期推出NFT的信息。 在这之前，GUCCI就已经利用AR技术推出了售价为11.99美元的虚拟鞋,这应该是GUCCI最便宜的鞋子了。

小结

毫无疑问NFT已经彻底出圈了，但是NFT真的如那句在业内广为流传一样“万物皆可NFT”吗？

从整体来看，NFT现在只是一项建立在区块链上的新兴技术，目前并不能完全保证物品版权的安全性以及唯一性等，也不能保证每一次的引用或转卖都能让创作者获得利润，另外它的内在价值不太好确定，有许多影响它的因素。

无论现在的NFT如何炒作造势，作为技术本身依旧是中立的，其实际的价值和命运取决于如何运用，哪些NFT能被留下是需要经过市场的考验的。

而对于NFT应该持有的态度，在这里引用国内知名篮球解说员苏群对于NFT是否有前景的回答：“对于新生事物，既不要直接否定，也不要梭哈”。

各行业领域“大厂”都在纷纷进入NFT领域，进行布局。我们又该如何去看待NFT的火爆呢？欢迎在留言区分享你的观点。
