6 月 2 日，NFT 项目 CheersUP 开发团队 Baselabs 在社区进行了一场 AMA。对于这款与 B 站紧密结合的 NFT 项目，团队解答了「后续玩法」、「对 NFT 和 Web3 的思考」等等问题。以下为本场 AMA 整理。

关于 CheersUP

首先请 Baselabs 做个简单的自我介绍

Baselabs： 大家好，我是 Baselabs 团队的运营负责人，感谢大家项目的支持，很荣幸今天能在这里和大家分享更多信息。

很多用户不太明白 CheersUP，bilibili，Cryptonatty 和我们 Baselabs 的关系，简单来说Cheers UP 是 bilibili 在 NFT 领域重点打造的 IP，也是海外发行的首个 NFT 项目。CryptoNatty 是 B 站官方授权合作的平台，我们 Baselabs 则是 CryptoNatty 与 Cheers UP 的运营团队。

「相遇就是同好，值得干杯」。干杯是 B 站社区文化与理念，也是我们社区的社区文化。因为同好，很荣幸我们能在这里相遇，一起建立共识。

大家目前最关心的一件事，就是什么时间开图

Baselabs： 确实这段时间大家都在期待开图，我们也已经筹备完毕。目前计划在国内时间 6 月 5 日晚上开图，并且定 6 月 6 日为 CheersDay，作为一个庆典来操办。

至于为什么选择 6 月 5 日晚上这样的时间，也是多方面综合考虑，正巧明天是端午节，我们经过商议之后还是决定节日就留给大家安心过节吧，所以决定放在 6 月 5 日开图，刚好与 6 月 6 的 ChaeersDay 活动衔接起来。

Cheers DAY 听起来很有趣，什么活动呢？

Baselabs： 哈哈，这个活动其实是社区用户最早在 #suggestion 频道提议的，首先感谢大家的建议，我们都会仔细阅读每一条建议并且研究其可行性和考虑采纳。

说回 CheersDay，我们更多是抱着将它作为一个每年一度的干杯文化庆典来举办，而不仅仅是为开图做的开图日活动。刚才也提到了，「干杯」是我们社区文化的精神内核，这个庆典会围绕着"干杯"的概念举办活动，还会给最早期就加入 Cheers 的支持者一些福利。

今年是第一届干杯节，也算是 CheersUP 或者说是 B 站迈向新世界的一个起点，我们希望通过在每年的这一天举杯欢庆的形式，来带动更多的人来了解我们的文化，加入到我们的世界里来。

可以聊一下 Cheers UP 的 Utility 吗？我看到有社交属性和 Pass 属性，可以和大家详细说一下嘛？

Baselabs： 好的, 我先从社交说起，在去年有很多关于 PFP 价值的讨论，讨论为什么 PFP 这个门类会在 NFT 领域脱颖而出。

如很多人所讨论的，我们认为社交属性是成就 PFP 的一个重要原因。再挑其中一个重要的点来说，PFP 能够「跨平台使用」对其早期的推广非常有利，即可以作为很多 Web2 平台的头像。这种以人为核心，跨 Web2 的产品功能，成为了 PFP 社交的一个基础。

所以对于 Cheers UP 来说，自身 IP 的品牌建设是后面重点工作。他具有 PFP 的属性，同时 B 站的生态会有头像位的认证来加持，增强其社交的曝光度；对于已经支持了 PFP 的生态来说也自带认证，比如推特的六边形头像。

同时关于 Pass 的一些属性，在 roadmap 上我们也做了一些描述。

关于 Pass 我还有挺多想了解的。Pass 在 NFT 领域也是不久前才开始被 FOMO 的，很多人认为 NFT 从社交资本迭代到了门票的故事，你们是如何看待所谓 Pass 在 NFT 中的应用的？

Baselabs： 嗯, 能做 Pass 的前提是背后有一个平台级的生态或组织，Cheers UP NFT 背后是 B 站平台生态，所以具备这个条件，这是给 Cheers UP 最直接的赋能。

而站在平台的视角来看，发行具备 Pass 能力的 NFT 又会多一个有效的治理工具，Pass 与平台相辅相成。比如与其他项目或平台合作，同时可以从 B 站平台生态与 Cheers UP NFT 两个角度展开，以及基于 NFT 的投票机制等。

具体如路线图所说，在平台层面，会建立两套名单体系，结合 NFT 项目的入驻机制，Cheers UP 持有者享有一定的特权，这一方面保证了 CUP 主的权益价值，另一方面也有拉新的作用，通过 CUP 主们的力量，引导优质的项目方入驻 B 站，形成一个良性的循环。我们相信与优质的项目方及社区一起，才能完成 Web3 的共建。

接下来我想问一下 PASS 的细节。一级名单是如何选择的？会不会与名单之外的蓝筹项目达成合作？

Baselabs： 第一期的一级名单已经昨天已经在社区公布了，初期一级名单的收录主要依靠 Hodler 推荐, 我们会重点尊重 holder 的提议，体现 holder 的提名权。同时我们项目方也选了一些公认的蓝筹项目, 以吸引更多蓝筹 holder 的关注。

同时, 一级名单的建设会持续进行，按期收集和更新 App 显示功能。随项目的落地实践，逐步迭代形成平台化的管理机制。

此外，我们也在主动地积极引入优质的项目，进一步完成二级专属名单的建设。

接下来也是个社区近来比较关心的话题，PASS 能否解锁 NFT 对应账号终身大会员呢？有空投计划或者分红计划吗？

Baselabs： 这些都是有可能的。大家合约也可以看到，里面包含了酿造功能，这将是对长期持有者友好的一个功能，同时也会有一定的周期性，以兼顾新用户的加入，未来如果有可能空投将会做一定的结合，增加 NFT 价值的同时，激励 holder 长期持有。

路线图上也提到过，我们会有一个大会员卡池，我们会制定月度免费领取的机制，保证不浪费的情况给到真正需要的人，卡池会周期性的扩充。

另外，发掘新的更大的价值会更重要，比如 pass 机制、与项目方联名合作等，这些工作我们都在进行当中。

其实国内已经有一套干杯的数藏了，和海外的 CheersUP 有什么区别吗？这也是不少人的疑问。

Baselabs： CheersUP 与国内的「干杯」，首先来说，在 IP 角度来说会有一定的关系，大的风格是一致的，这是为了可以更加集中国内外的资源联合打造品牌，对 CheersUP 本身的品牌宣传是有利的。

但整体上还是有很大的区别，首先权益完全不同，可流通的范围与自由度也不同，以及内容上也是完全不同的系列。

我了解到目前有许多用户希望调整认证的 CheersUP 显示角标样式，这方面官方是怎么考虑呢？

Baselabs： 当前深蓝色风格的动态图标设计其实是有多方面综合的考虑，包括和其他图标颜色尽量区分（比如大会员的图标是红色的）。

不过考虑到大家的反馈意见，我们已经向设计反映了需求。

图标更新涉及全站各处的头像资源位的变动也需要结合 APP 的发版，计划是再制定几套方案做一定考量。

另外，后面还会有其他特效和资源位，来进一步丰富展示的个性化权益，这个大家可以期待一下。

CheersUP 有没有可能做成动图 gif 呢？动图做头像应该会很 COOL

Baselabs： 这个问题之前在建议反馈频道里也解答过，从产品体验上的综合考虑，动态头像会干扰正常的内容浏览，所以全站都没有动态头像的实现（早年残留的极少历史遗留问题也会逐步下线）。

但从另外的角度，我们鼓励二创，甚至未来为优质的二创提供资源位展示。在这些展示空间内可以支持动图，会带来与头像不同的新的体验。像刚刚结束的 CCC 大赛，就是我们二创的第一步尝试，之后这个体系会逐渐建立完善起来

关于项目方对 NFT 发展的想法

我想很多人想知道 B 站是如何看待 NFT 的。这个 NFT 项目在 B 站规划里处在一个什么战略地位，支持力度有多大，项目领导话语权如何，能争取多大的资源，有多大的决心？

Baselabs： 作为 B 站的官方合作伙伴，以我们现在了解到的信息来看，在 B 站海外版 APP 内开放给 NFT 的各资源位是非常宝贵的流量入口，这种支持力度是非常大的，对比其他应用来说一般都是广告位才会用，与营收直接相关，这也是最难争取到的资源，由此可见战略位置是很高的。

而且对于 CUP 来说，我们是 B 站第一个海外 NFT 项目，其各种权益福利，自然也是首先安排、尽量满足。所以争取资源和决心方面，大家尽可以放心。

CC0 是 NFT 里非常重要的概念，就连龙头 CryptoPunk 都因为不是 CC0 而被 NFT 玩家吐槽，但 CheersUP 似乎不是 CC0，为什么？

Baselabs： 这方面的主要原因是，CheersUP 这套 PFP 内包含 B 站的小电视等已有 IP 元素，所以很难做到 CC0 的标准。但我们也会更主动的去打造品牌本身。

以及尽可能放宽约束，鼓励大家的二创及传播。二创是我们未来重点发展的一个方向，所以虽然做不到 CC0，但我们会在可操作的范围内尽量放宽限制。

CheersUP 是一个 Web3 项目，虽然 Web3 没有边界，但现在 CheersUP 还是国内用户居多，可以这么认为吗？我们的目标用户群体是? 有考虑海外宣发吗？

Baselabs： 如果从赋能的角度来说，B 站的用户群体确实体验会更方便一些，与项目持有者与 B 站用户的重合会有直接的关系。

事实上 B 站的简体/繁体版 APP 是 4 月底才开始陆续上架除了中国大陆/港澳台以外国家和地区的应用市场，不到一个月的时间，对全球华语群体的覆盖增长迅速，其中对 NFT 的支持是首发功能。

所以从用户吸引力来看，简体/繁体版 APP 的目标是服务于 Web3 的国人/华人群体，这是需要非常努力才能争取到的优质用户，在全球 Web3 社区，国人/华人拥有巨大的影响力。

另外在未来，B 站东南亚版/英语版也会推进，以覆盖更多的用户。

我们也看到了很多非华语的用户参与了项目，在此我们表示由衷的感谢，由于前期的社区还在建设，忽略了一些大家的感受，我们相关的权益会覆盖服务到全球用户，以及完善多语言的社区建设，同时 B 站国际化的进程也在加速。

CheersUP 在海外市场拓展方面有何计划，会不会与海外 up 主、大 v 等进行合作宣传，会是什么样子的策略呢？

Baselabs： 会的，这是肯定的，除了 Web3 方面的 Kol，我们还会持续吸引 Web2 领域的大 V、UP 主，让更多新用户加入我们。

对于 Web2 的用户群体来说也要做好基本概念普及，同时介绍 Cheers UP 的品牌理念与用户权益。

对 Web3 社群宣传平台的资源价值，吸引更多 Web3 用户社区使用 B 站的平台功能。

目前已经在推进，持续挑选合适的人选。

关于 B 站对 Web3 的想法

前面说过，Web3 没有边界，但与 B 站生态绑定紧密，是否违背 Web3 的精神？该如何与外部 Web3 生态结合？

Baselabs： 作为 B 站的官方合作伙伴，以下仅代表我个人的一些看法，根据目前了解的信息来看的。

首先从 Cheers UP（CUP）的视角来看，B 站是其背后已经存在的一个生态系统与社区，这是本项目一个重要的优势，我们不应该回避，而该积极融合。

这也是这个项目与其他项目发展路径上最大的不同，我们不会从零开始去创造一个 Web3 生态，而是在原有的 Web2 生态上孕育、生长出一个 Web3 生态。

另一方面，CUP 本身与 B 站是解耦的关系，资产是属于用户的，与其他 Web3 平台是开放合作、积极接入的态度和推进方向，如果有其他生态愿意开放，CUP 也会积极的接入，成为彼此生态的一部分。

所以再站在 B 站生态的视角来看，对于一个活跃用户过亿的平台来说，相较于其他平台已是足够开放，甚至比很多 Web3 原生的平台还要开放。我们欢迎项目方、平台的合作，创造新的可能。

B 站在 Web3 方面还做了哪些工作?

Baselabs： 先从 NFT 说起，成功的 NFT 项目包含的要素有很多，其中最重要的可能有几点：优质的内容、充沛的流动性、强共识的社群、丰富的应用场景等等。

有应用场景，同样可以决定价值的高低，这是 B 站未来重点要探索的领域，也是目前市场上比较缺乏的。

B 站海外版在 Web3 领域会从为 NFT 提供资源位和应用场景、为社群提供产品功能开始探索。这方面我们已经看到了一些动作（比如一级名单显示），相信 B 站以此作为切入点，进行更多探索。

事实上 B 站的行动比其他大型 Web2 公司都要早，早在 2021 年 12 月在中国大陆基于高能链就第一次上线了头像认证功能（《鸽德》），这比推特支持头像认证的上线时间还早一个月，未来开放的流量与资源位也会更多。而项目方的入驻机制，这也要比简单的面相持有者的会员收费模式更有层次。

用户提问：

1. 开图具体的稀有度比例是啥？

Baselabs： 提前透露下，「原始部落」这个等级为稀缺款，然后再上一个的未知等级只有 12 个 1of1。另外还有个神秘的 0of0。

2.Cheers up 后续是否有发币计划？

Baselabs： 目前项目还是非常早期的阶段，短期还没有这个计划。发币需要具备很多成熟的条件才有可能，需要建立在项目稳定且有突破发展的基础上。发币不是这个项目的目的，但有可能是一个水到渠成的结果。

3. 我们的社区文化建设后续怎么去做，会受重视吗？

Baselabs： 社区文化建设确实是我们非常重视的一个部分，目前是刚举办完第一期的 CCC 创作者活动，接下来社区会更多的鼓励大家进行各式创作，强化社区文化内核，同时会有专门的「创作者积分」作为激励，积分的应用也会逐步上线，这是个比较大的体系，大家可以保持期待和信心。

4. 酿造玩法是指酿造大会员吗？会酿造其他东西吗？

Baselabs： 大会员是之前很多用户提出的要求，为此准备了大量的资源来给有需要的用户，并会结合酿造来满足，这是一个最基础的权益。随着酿造时长的增加，还会结合空投的玩法，同时也会有一定的周期性，对新用户友好也至关重要。
