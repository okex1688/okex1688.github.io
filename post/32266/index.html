据报道，加密货币交易所 Kraken 正在开发自己的 NFT 市场，用户可以在那里买卖各种 JPEG 图片。

Kraken 交易所首席执行官杰西·鲍威尔（Jesse Powell）表示，该市场将提供托管服务，其交易所正在考虑建立一个系统，允许客户使用其  NFT 进行抵押借贷。

鲍威尔在接受彭博社采访时表示，“如果你在 Kraken 存入了一个 CryptoPunk，我们希望能够在你的账户中反映出它的价值。”

他补充道，“如果你想以此为抵押借入资金”，并暗示一个 CryptoPunk 可以被用作抵押品来借入其他加密货币或稳定币。

鲍威尔说，NFT 的金融化构成了该领域发展的“第三阶段”。他表示，“第一阶段是投机，第二阶段是购买艺术品和支持艺术家，第三阶段将是 NFT 的功能性用途。”使用存放的 NFT 作为 Kraken 的抵押品可能是其中一种用途。

2021 年和 NFT 热潮

2021 年对于非同质化代币（NFT）来说是重要的一年，该技术从加密小众领域发展到主流领域，并在此过程中被评为柯林斯词典2021年年度词汇。

在佳士得拍卖数字艺术家 Beeple 的NFT“EVERYDAYS: The First 5,000 Days”之后，NFT 在 2021 年首次成为头条新闻。在 3 月初，此次拍卖销售额超过 6900 万美元，成为在世艺术家的第三大单笔销售额。

此外，OpenSea 等现有 NFT 市场出现了破纪录的交易量，比如 Bored Ape Yacht Club 和 CryptoPunks。

然而，Kraken 入场有点晚了。其竞争对手币安和 FTX 已经在今年推出各自的 NFT 市场，希望从中分一杯羹。Coinbase 的产品也在开发中，其候补名单的注册人数已经超过数百万，这些人都有望成为其未来的用户。

加密借贷平台 Nexo 上周也宣布推出 NFT 抵押贷款产品。Nexo联合创始人 Antoni Trenchev 表示，“像 Nexo 借贷这样的服务将会有很高的需求，以释放 NFT 的潜在价值，同时让用户保留所有权。”

该公司表示，Bored Ape Yacht Club 和 CryptoPunks 将是首批两个符合条件的收藏品，并补充说将很快宣布支持的更多收藏品。
