模块化区块链和扩容执行层的创新

资料来源：IOSG 2021 Summit

通俗的方式理解 ZK Rollups 和 Optimistic Rollups

首先我们聊一下Layer 2。Layer 2其实没有区块的概念，也没有节点，也不需要共识机制，所以在 Layer 2 方案的区块浏览器上，是看不到区块的。Layer 2 的运营者只负责把交易排个顺序然后拿去处理。而二层网络的关键之处就在于对交易的排序工作。

在2021年我们见证了Layer2的市场的崛起以及更多流动性/项目的出现。目前在以太坊上已经出现了超过19个通用或者单种应用的Layer 2。其中，StarkWare 和 ZkSync 在ZK Rollups方面领先，他们通过零知识证明的方式实现扩容。Arbitrum和Optimism是在Optimistic Rollups方面领先的项目。

ZK头部链StarkWare 的TVL 有$1.2B且处理超过5500万笔交易，目前利用 StarkEx 技术方案的项目比如知名的爆款 dYdX，ImmutableX，DeversiFi，Sorare。OP方面，Arbitrum一共处理了三百多万笔交易,目前TVL超过了Optimism一倍，达到了$2.25B（而 Optimism 则只处理了二十多万笔交易,TVL为$1.1B）同时，我们也不能忽视在跨链流动性方向有进展的项目，比如Celer, Hop Protocol 以及 Connext。

简单来说，关于两个 Layer 2（ZK 和 OP） 方案的区别，我们可以这么分类：

ZK 是把结果和对结果的证明加密后小部分数据传到主网，来证明结果没问题；OP 也是要把结果和原始的记录传到主网，如果有挑战能够有证据可查。

当Layer 2 基建更加成熟的时候，我们可以预见Layer 2将会呈现更丰富的生态。我们预计在未来的一年，Layer 1主链上超过80%的流量将转移到Layer 2上，现在只是Layer 2大迁徙的开始。

模块化区块链的三个Layer

资料来源：IOSG 2021 Summit

模块化区块链(Modular Blockchain)的概念为扩容的“不可能三角”提供了更好的解决方案。 通过将执行层Execution与安全层Security以及数据可用性层(可获得性Data Availability）分开，以太坊主链将会提供更好的底层兼容设计，而创新的L2执行层将会提供更高的TPS以及更低的手续费。

简单来说，一个区块链有三个职责，执行、安全 、 以及数据可用性。 长久以来，这些职责都由一个区块链自己负责，这也导致了扩容性方面的问题。在模块区块链的概念下，执行层、安全层以及数据可用性层将会独立出来。

在Modular区块链的概念下，Execution, Security 以及DA分别独立出来，其中执行层Execution layer的发展是2021年最引人注目的。 Security 方面，因为竞争难度，这个layer仅限于BTC/ETH/Polkadot。而且短时间内BTC也没有与ETH竞争的计划。DA方面，ETH依旧领先，主要是源于大量的分片链带来的大量数据可用性。同时也不能忽视潜在的DA chains。随着ETH2.0的发展，当越来越多的data availability 提供给Rollups时，主链的TPS 将会大大提高，手续费也会降低。

将三个层分开来看，在安全层的角度来说 ，BTC和ETH是唯二从节点出发就将去中心化作为目标的链。安全层是最重要的，因为反审查制度、以及去中心化程度是一个链最为重要的品质。目前短期内BTC没有提供比ETH更高数据可用性的计划，所以以太坊还是Layer 2安全层的主要提供者。

在数据可用性方面 ，足够多数量的区块才能保证更多数据存储的可能性。ETH目前领先，因为它的分片链计划，所以能够为Layer 2提供足够多的数据可用性。然而，像Celestia, Polygon, Near以及Tezos 一样的竞争者也是有和ETH一样的计划。随着ETH2.0的发展（这个词汇最近被Ethereum Foundation放弃了），当更多的数据可用性被提供给rollup这个过程时，主链的TPS将会大幅提高并且手续费将会被降低。

最后，交易和智能合约发生在执行层 。所有智能合约、dApps和用户也在执行层进行交流。执行层应该专注于处理尽可能多的执行过程，同时将安全性和数据可用性外包给项目。目前的Layer 2，如Arbitrum，Optimism，StarkWare和ZkSync都在聚焦开发更有效的执行层。

目前执行层的情况一览

Layer 2在2021年飞升

资料来源：IOSG 2021 Summit

尽管Layer 2只上线了几个月，但是他们的TVL已经达到了70亿美元。在现有的Layer 2中，Arbitrum有着最高的TVL 。因为Arbitrum有EVM，所以开发者更加容易在上面部署应用。币安最近宣布开始支持在Arbitrum Layer 2上面存取资金。这也会大大增加Arbitrum Layer 2的运用。然后ZK Rollups（StarkWare）在过去7天当中拥有更多的交易量，相比于Optimistic Rollups显示出了更加强大的扩容优势。此外，DeFi大部分的应用如Uniswap、Sushiswap等也已经在L2上开始进行部署。

一些Layer 2上的应用如dYdX，已经达到了和CEX以及L1应用相当的TPS表现。dYdX在上线后2个月的时间里，曾经出现比Coinbase和Uniswap加起来的交易量还要多的日子！由此看来，Layer 2确实潜力无穷。

Layer 2备受争议的一点是交易费用 。Layer 2交易费并非比Layer 1更便宜，但是在开发者优化网络之后，Layer 2的费用应该会比现在低很多。Layer 2在交易费用的竞争优势会慢慢显示出来。目前，Layer 2的交易费用至少比L1的交易费用低10倍。将来，每项交易费甚至会低于0.1美元。我们会看见不久的将来，以太坊Layer 2不仅有着低于0.1的交易费，而且会比BSC、Polygon等Layer 1拥有更多的市场份额。所以，在下个3-5年，我们希望大部分应用和用户迁移至Layer 2。

下一个是谁？Layer 3 To Da Moon！

资料来源：IOSG Ventures

如果你觉得Layer 2是扩容最好的解决办法，那你就错了！尽管Layer 2能够扩容并且降低每笔交易的费用，有些应用还是需要更加定制化的、更加符合他们需求的服务。Layer 2在去中心化、其他普适目的方面保持了自己的优势，但是Layer 3会进一步增加应用的工作效率。

Layer 3之于Layer 2，正如Layer 2之于Layer 1。Layer 3会在Layer 2上发布有效性证明，正如Layer 2在Layer 1上执行批量交易，发布有效性证明那样。当Layer 2能够支持验证检验者智能合约之后（比如StarkWare发布StarkNet之后），这种多次打包结构能够增加有效性证明的数据利用率。除了扩容，Layer 3还会有更多的发展方向，比如隐私。这种情况下，有效性证明会被零知识证明的代替。

结论

我们最近几年都在密切关注Layer 2赛道的发展。我们很高兴和许多L2的领先项目合作，比如做Optimistic Rollups的Arbitrum，做ZK Rollups的StarkWare，以及跨链桥流动性解决方案Celer。

我们相信在不久的将来，Layer 2网络会在交易费用/TPS方面会比Layer 1变得更加有竞争力。

除了通用Layer 2，定制化Layer 2和Layer 3会进一步扩展区块链，比如基础设施提供商StarkWare会发展SaaS业务。

在基础设施层，还有一些在新技术方面比较诱人的投资机会，比如数据可用性赛道的Celestia和跨链流动性转移赛道的Movr，都是很好的例子。

我们还希望看到应用侧有重大的创新，在Layer 2技术驱动的新技术应用设计。Layer 2的特点如扩容和隐私一定会在dApp设计方面带来更多的创新！
