作者：Mason Nystrom, Senior Research Analyst at Messari

译者：lester@oct.network

什么是 Web 3.0？

一句话——它是互联网的下一个时代。Web 3.0 是互联网朝更加民主化发展的范式转变——互联网由集体而非公司和大型组织治理。

Web 3.0 通过新技术表现出来，例如加密货币、虚拟和增强现实、人工智能等。然而，Web 3.0 运动并不是由新技术引领，而是由我们人类如何看待和重视互联网的转变而来。Web 3.0 是围绕为人们创建一个互联网，并由人们创建。

以前没有任何东西，然后有了互联网

正如我们所知，互联网从根本上改变了世界。这不是有争议的声明。虽然由于它被低估而值得强调——互联网改变了一切。

找到你今天关心的问题或话题——无论是经济、政治还是社交——快速的谷歌搜索将显示互联网已经从根本上重塑或转变了问题。

经济——互联网是全球化和电子商务兴起的直接原因，这两种趋势对世界产生了深远的影响。更具体地说，在过去 3 年（2017 年、2018 年和 2019 年），标准普尔 500 指数科技板块的价值翻了一番，如今科技板块股票占整个标准普尔 500 指数的 25% 左右。

社交——这个很明显，连“社交”这个词相比二十年前都有了新的含义。Facebook、谷歌、Instagram 和 Twitter 等曾经的年轻创业公司现在风险投资的 40% 以上是从新创业公司那里以广告支出的形式获得的。TikTok 等新平台的潜在未来影响尚未确定，但对社交媒体影响者来说却是有希望的。

政治——提供精选新闻并旨在煽动而非凝聚的算法已成为常态。互联网影响了世界各地的政治选举，并催生了新一代互联网原住民的官僚。

互联网改变了一切——金融市场、文化、选举，一切。这条规则几乎没有例外，以至于它也可能是一条自然法则。一两十年后，源于或直接由互联网引起的事件将多次塑造世界。这就是为什么互联网的命运变得比以往任何时候都更加重要的原因。

为什么我们需要 Web 3.0

互联网早期本质上是探索性的。类似于 Manifest Destiny 的意识形态——相信美国在整个美洲大陆的扩张既合理又不可避免——早期的技术先驱探索、隔离、围堵和商品化互联网。随着时间的流逝，互联网一直保持“免费”状态，但已被败坏。

当你尝试购买公共汽车、火车或机票时，你可能会在生活中看到这一点。你的所有搜索都被记录、出售，并被操纵来对付你。互联网是你的，可以去用，但你不是它的主人。世界各地的个体面临更大的挑战，特别是在互联网被部分封锁、限制或完全封锁的国家。

互联网的标志性特征是信息的民主化，但今天，信息越来越不可靠，在某些情况下甚至是完全有害的。由机器人控制的虚假账户正在操纵儿童。由人工智能创造的逼真人脸将给社会带来更大的问题。2019年假新闻的成本超过 700 亿美元，并且会随着时间的推移而增加。病毒传播式的人类状态催生了数字病毒——虚假新闻报道——传播速度是真实新闻报道的六倍。这种趋势具有极端的影响。虚假信息的传播对冠状病毒等大流行病产生了负面影响，这增加了公共安全风险。

错误信息的传播并不是唯一的问题。在接下来的 20 年内，世界大部分都将在线上，这带来了新的挑战。

  • 如何处理来自智能手表的数据，以便当用户们尝试获得健康保险时，个人信息不会被用来对付用户吗？

  • 我们如何确保 Facebook 或 Twitter 等平台将用户视为价值创造者而不是产品？

  • 当一切都数字化时，保护个人和系统免受网络威胁和攻击的最佳方法是什么？

  • 在网络效应的力量导致单一企业控制互联网服务的世界中，我们如何设计推动创新和保护用户的法规？例如，试图拆分谷歌和 Facebook 是治标不治本。

  • 随着世界越来越依赖人工智能算法，社会如何防止单点故障，未来会怎样？

  • 更重要的是，我们如何资助和激励用于加密网络、人工智能算法和其他关键互联网服务的开源架构的开发？

这些只是一些存在的问题，社会目前正在解决。

迈向 Web 3.0 世界

现在，有些人让你相信 Web 3.0 是一个没有中间人的世界，每个协议都有自己的代币，公司都在各种区块链上运行。在我看来，这种信念在很大程度上脱离了现实，虽然这些人的意图是真诚的，但本质上并不实用。

Web 3.0 是围绕重新构建互联网的现有服务和产品，使它们造福于人而不是实体。加密网络（比特币和以太坊）和人工智能将成为实现这一目标的手段。虽然无法预测未来，但我相信一个实现 Web 3.0 的世界将以开源协议为基础，而企业则充当提供便捷访问和附加功能的接口。数据仍将用于推动决策制定，但不会用于对抗消费者。数据权利将受到保护，而不是为了谋取利益。激励机制和市场机制将有助于确保信息的可信度和可验证性。

Web 3.0 世界将优先考虑主权个人，而不是世界上富有的精英和寻租者。系统和协议的重新架构将侧重于民主化和权力分散。

为什么是现在？

实现 Web 3.0 的第一个先决条件是创造互联网原生货币。比特币和以太坊已经成为未来的互联网货币，因此第一波创新和实验正在发生也就不足为奇了。

开放金融——或去中心化金融 (DeFi)——是从当今封闭的金融体系转向基于可互操作、可编程和可组合的开放协议的范式转变。这是迈向由无许可开源协议定义的 Web 3.0 世界的第一阶段。

一旦 Web 3.0 的金融轨道得到保障，新的创新和商业模式将有助于重塑互联网服务和产品。互联网改变了一切，互联网发展的下一阶段——Web 3.0 将催化更大的变化。

结语：推进 Web 3.0

当然，人们容易对企业精英、政府或任何操纵互联网以谋取利益的人感到不满和愤怒。虽然这可以理解，但我认为这不是正确的情绪。

最初，互联网需要先驱者和企业来开发工具来使互联网易于访问，他们得到了应有的回报。只要现有的互联网设计激增，我们就会继续向大型科技公司放弃我们的数据、时间和金钱，以换取承诺免费访问的便捷产品。老实说，没有一种解决方案可以作为解决所有这些问题的灵丹妙药。

当我们接近一个越来越受技术控制的世界时，我们设计系统来重新调整激励措施以支持集体，这一点至关重要。从现有互联网（Web 2.0 到 Web 3.0）的过渡是一个长达数十年的过程，它将从根本上改变我们与互联网交互的方式。企业家、工程师、企业、设计师和互联网公民今天做出的决定将传递并影响后代。我乐观地认为，Web 3.0 之旅将是令人兴奋和成功的。
