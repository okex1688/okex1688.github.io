近日，腾讯推出 NFT 交易 APP 的新闻引发了 NFT 藏家的广泛关注。值得注意的，腾讯并不是第一个涉足 NFT 的互联网大厂。早在今年六月，支付宝就曾基于蚂蚁链推出付款码 NFT 皮肤，并引起了业界的广泛关注。这一发售价仅 9.9 元的 NFT 被收藏者炒至数千元的高价在闲鱼出售。

收藏市场始终是狂热的、沸腾的，投资者、收藏者和文人雅士们历来不吝于一掷千金购买高价藏品。今年 5 月，在 iBox 平台发行的《徐冰天书号》首件 NFT 太空艺术作品以 200 万人民币最终售价成交，创下了国内最高单价数字藏品的记录。在传统互联网大厂陆续入局 NFT 后，NFT 愈发被人们重视，这一新生事物正在重新定义「收藏」的概念。

NFT 是如何重新定义「收藏」的？

NFT 的出现挑战着人们的神经。他以创新性的所有权证明，创造了改变艺术圈的潜在可能性。艺术品的价值是来自于艺术审美在精神上的享受，而载体只是艺术家传达审美意象的物质手段或媒介系统。随着技术的进步，新人工载体的出现丰富了人们以艺术方式掌握世界的手段。

在创作更多地依赖于天然载体的时期，创作语言相应地显得较为贫乏，而人工载体出现后，极大地丰富了艺术语言。例如，人工颜料可以更丰富、更细腻地表现色彩。就此意义而言，NFT 的出现是一次突破，一次革命。

NFT 带来的改变并不止于创作，跻身 NFT 浪潮，还可以刺激艺术的流通。NFT 的出现正在改变着艺术生态，一定程度上挑战了画廊和艺术家之间的代理关系。也让买家可以跳过画廊、中间商去交易平台购买，摆脱不透明问题。

在传统的艺术品市场，一级市场（画廊）和二级市场（拍卖行）控制了大部分的艺术品流通。而 NFT 则是去中心化的，在一定程度上挑战了画廊和艺术家之间的代理关系。也让买家可以跳过画廊、中间商去交易平台购买，摆脱不透明问题。「NFT 推崇的去中心化，不存在传统艺术市场中你必须是美院毕业、作品被某个画廊挑中才可以靠作品谋生的现象，而是每一个创作者都可以靠自己的才华赚到钱。」有藏家这样向律动解释。

去中心化并不代表没有门槛，更准确来说，NFT 把藏品定价权去中心化了。没有人有权力凭借自己的喜好、认知决定一件作品的价值。收藏应是建立在自己的认知、对作品的理解以及和艺术家的沟通交流后产生的看法之上。当 NFT 艺术出现后，去中心化的艺术作品的自发生成，为创作者和作品带来了真正意义上的主体性。

以兴趣藏品为例，7 月 30 日，iBox 平台曾上线一款小黄鸭盲盒。这一盲盒由著名的 Pancoat 公司授权业内知名游戏 MEGA HEROES 发行，首期 1996 份盲盒上线即全部售馨，SSR 款小黄鸭在交易市场在交易市场已溢价逾万元，超过发行价百倍。

技术中立：人们对 NFT 的误解亟待澄清

NFT 作品作为采用 NFT 区块链技术的作品，其在链上给数字作品生成一个无法篡改的独特编码，确保其唯一性和真实性。技术本身是中立的，并无明确价值导向。但很多人将 NFT 用来做某种类别的虚拟货币的代称，存在误导之嫌。

需要注意的是，NFT 本身并不是交易的对象，通过 NFT 技术来实现上述作用的数字商品才是被交易的对象，相应价值也是由数字商品来实现的。NFT 是对数字作品权利内容和历史交易流转信息进行记载的非同质化电子凭证。NFT 具有唯一性和不可拆分性，不以融资为目的发行。

通过 NFT 交易的藏品首先是出于展现的收藏行为，这为用户提供了社交价值。当人们对藏品的价值形成共识后，藏家的认知不止局限于藏品，也对收藏同类藏品的其他藏家有了认知。所以市场价格体现的是群体的价值共识，而不是某个个体的价值认知。艺术品、兴趣藏品、多媒体作品等事物，其收藏者均会出于对其背后的故事、情感、叙事、社群的认可或共鸣等原因而形成社区共识。NFT 的出现，让这种社区共识以最低的成本维系，NFT 有权属、唯一、不可拆分，它不止是藏品，更是共识的载体。

以 iBox 为例，该平台一直致力于赋予数字藏品更多的社交价值，这一方向或将为数字藏品带来新的想象力。 iBox 曾联合多位知名艺人发行其 NFT 作品。相较实体藏品，这一藏品更可为维系粉丝的共同价值，同时为收藏者提供社交价值。

目前 NFT 作为新的领域还需要被更多人去知道并被认可。作为一种新生的创作媒介，NFT 不仅可为创作赋能，基于其不可篡改性，NFT 更是数字作品版权保护的最佳方案选择，它可让 ip 真正掌握话语权。

以电影周边为例，此前，电影周边开发往往面临周期较长、产品品控不确定、收藏价值较低等多方因素的制约，而 NFT 藏品则不然。NFT 平台 iBox 联合电影《真·三国无双》出品方共同发行了国内首套电影 NFT 系列作品。系列 NFT 作品共涉及电影《真三国无双》中 30 款不同形式的道具与海报等，包括古力娜扎、古天乐、刘嘉玲、王凯、韩庚等人在影片中扮演的角色形象剧照及戏服等，是国内数字藏品电商平台与头部电影 IP 合作的首次尝试。影片周边衍生品也多结合经典场景的重现与复刻，配合众多一线实力影星的加持，整套 NFT 作品均具备较强的观赏性。

NFT 应用前景广阔，远不止于收藏

毕加索曾说：「艺术本身不会演化，艺术的表现形式不过是随着人类的观念变化而变化。」当产业实践走在理论边界之前，艺术家们不得不进行大量的创新尝试与思考，以期获得更多的反馈。没有什么比艺术家的思想更重要。一个艺术品可以通过绘画、摄影、雕塑、行为、语言等多种方式表达，而艺术家需要选择的，只是哪种表达更适宜。而区块链技术为艺术创作者们提供了更丰富、更具革命性的媒介和手段。NFT 的诞生将艺术转换为一种新的价值取向，以及呈现和承载这种价值取向的载体。

但这并不代表加密艺术当下就没有问题。目前 NFT 行业还处于早期探索阶段。艺术家们也很忌讳被看作追逐短期利益的新概念跟风者，进入这个领域的艺术家的数量还不到 1%。目前市场上也存在一些乱象，例如侵权、剽窃等问题时有发生，而这一问题的解决就取决于平台的监管。目前来看，平台在 NFT 交易市场的重要性仍然举足轻重，一个良性发展的平台，对整个市场都会产生积极作用。

在未来，当 NFT 的应用普及之后，或将形成一个创作者、藏家、社区共同受益的生态。在传统收藏市场，作品一旦售出，无论在二级市场以何种价格流通，创作者均无法获得额外的收益；而目前绝大部分 NFT 艺术平台，都将艺术家的分成写入底层协议，艺术家首次售出作品后，依然可以从后续每次转卖中直接获得分成。

此外，iBox 平台曾发行国内首套公益 NFT 作品，开创了利用区块链凭证技术助力公益事业的先河。艺术品、兴趣藏品、公益、版权保护……NFT 的应用范围正在不断开拓。作为一种诞生时间并不久远的新生事物，NFT 距离普罗大众愈走愈近。在未来，随着 NFT 的普及和更多应用的落地，这一赛道的前景将远不止步于收藏，或将成为 IP 运作的主要战场。
