我们的全仓保证金永续合约的全新 Layer 2 主网已经上线，向所有符合我们的使用条款的交易者开放。为了扩大交易量，dYdX 和 StarkWare 在 StarkEx 可扩展性引擎（“Layer 2”）和 dYdX 的 Perpetual 智能合约的基础上携手构建了全仓保证金永续合约 Layer 2 协议。该协议不仅能为用户节省 gas 费，还能降低交易费和最低交易限额。

2 月，我们在以太坊区块链上启动了 Alpha 封测版，仅供用户、流动性提供者和战略合作伙伴体验。在接下来的几周，我们全面测试了该系统的稳定性，收集了用户反馈并推出了很多新功能。当时，注册人数达 11 万，共计完成 2.5 万笔交易，交易额高达 9000 万美元。今天，怀着激动的心情，我们郑重宣布，新的 Layer 2 协议主网上线。

去中心化永续合约交易用户的热情令我们感到震撼。密码学货币是全球性的，我们的国际用户人数急剧增长。完成 B 轮融资后，我们宣布了投资海外增长型市场（如亚洲）的战略目标，并将重点放在大中华地区。在我们的轮候名单上，亚洲用户的占比约为 78%。登录用户人数最多的是中国，然后是香港、俄罗斯、新加坡、日本、越南、乌克兰、英国、台湾、韩国和印度。

为便于向我们的 Layer 2 协议过渡，我们已经从头开始打造了一个全新的全仓保证金永续合约产品和交易系统。该产品采用全新的改良版用户界面，支持完全的即时交易以及永续合约交易者所需的一切高级功能。在 Alpha 版本中，我们还额外构建了以下功能：

更多代币对交易

有了全仓保证金模式和更强的可扩展性，我们可以在 dYdX 上创建更多市场。我们的 Alpha 版本支持 BTC-USD、ETH-USD 和 LINK-USD 代币对的永续合约。我们计划在不久的将来支持 AAVE-USD 和 UNI-USD 代币对，以及其它许多代币对。我们会重点考虑 DeFi 代币，以及交易量高的代币对。点击此处，查看我们的永续合约已支持代币对的完整名单。

快速提现

快速提现功能可以让用户立即收到资金，无需等待 Layer 2 交易被打包到以太坊区块链上。用户需为快速体现支付一笔较高的 gas 费和提现金额的 0.1%。快速提现的最高金额为 2.5 万美元。

0x API 整合

目前，我们的所有永续合约均使用 USDC 作为保证金和结算货币。但是，等到 dYdX 整合 0x API 后，用户就可以在他们的永续合约账户中存取其它资产。

FAQ

Layer 1 上的永续合约会怎么样？

Layer 1 上的永续合约将持续至 2021 年 4 月 20 日。届时，用户需要将 Layer 1 协议上的永续合约平仓，然后在我们的 Layer 2 协议上重新建仓。

什么是 Layer 2？

StarkWare zkSTARKS 技术是一种 ZK-Rollup 技术，能够极大提高 dYdX 的交易结算能力，同时依靠底层以太坊区块链提供安全性。整合 StarkWare 之后，dYdX 可以将 STARK 证明与链上数据可得性相结合，从而彻底实现免托管。Layer 2 系统会对交易进行结算，并定期发送零知识证明至以太坊上的智能合约，来证明 StarkWare Layer 2 系统中的状态转换是有效的。

为什么 dYdX 要转移到 Layer 2 上？

以太坊每秒可处理大约 15 笔交易，远不足以支持 DeFi 的超高速增长。过去几个月来，以太坊上的 gas 费暴涨，极大提高了交易者的交易成本。有了 Layer 2 网络，dYdX 的交易费可以降低至中心化交易所的水平。

除了降低交易成本外，Layer 2 还能带来很多其它好处。有了 StarkWare 的 Layer 2 网络，dYdX 现在可以极大提升交易体验，远远超过其它 DeFi 项目。

Layer 2 对交易者来说意味着什么？

更低的交易费

由于 Layer 2 能够大幅提高可扩展性，我们可以通过全面降低交易费的方式让利给交易者。最重要的是，用户在交易时无需再支付 gas 费了！

我们将发布一份媲美中心化交易所的交易额加权型交易收费表。点击此处，查看我们最新的 Layer 2 永续合约收费表。

更低的交易限额

鉴于交易成本大幅降低，我们放宽了最低交易额的限制，便于 dYdX 用户进行小额交易。点击此处，查看我们的最低交易额。

全仓保证金模式

交易者可以使用一个保证金账户在多个永续合约市场上交易，在实现多代币对交易的同时极大提高资本效率、简化交易体验。交易者能够通过 0x API 将绝大多数 ERC 20 资产兑换成 USDC，然后使用同一个账户在 dYdX 提供的所有市场上进行交易。点击此处，了解更多关于全仓保证金模式的信息。

即时交易结算

借助 StarkWare 的 Layer 2 网络和 dYdX 的链下撮合引擎，交易可以即刻执行。在 dYdX 上交易会变得像在中心化交易所上那样快。

更快捷的价格输入机制

价格输入机制使用兼容 STARK 证明的签名来验证价格。这样一来，智能合约可以直接使用已签署过的价格，无需等待交易上链。在以太坊上，价格输入机制的更新延迟高达数分钟，然而在 StarkWare 的 Layer 2 网络上，延迟可以降至 1 秒内。

我们目前使用 Chainlink 的价格输入机制网络来为我们的 Layer 2 协议提供安全的喂价。Chainlink 已经整合 Starkware，让 Chainlink Network 的价格输入功能可以服务于更多高性能的 Layer 2 解决方案。很快，MakerDAO 等其它价格输入机制提供商或将在部分市场投入使用。

更高的杠杆率 & 更低的清算惩罚

由于价格输入机制的性能提升，清算会变得更加安全快捷。这意味着，我们可以在提高杠杆率的同时降低清算惩罚。在 dYdX 的某些永续合约市场上，用户能以高达 25 倍的杠杆率进行交易。点击此处，了解 dYdX 的最高杠杆率。

更强的隐私性

在 ZK-Rollup 帮助下，并非所有交易细节都会发布到链上，余额变化除外。因此，隐私性大幅增强，交易者无需担心有人会模仿其交易策略，或监控其交易活动。
