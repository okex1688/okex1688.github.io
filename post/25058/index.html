结语

当前，很多人会将 NFT 与艺术、金融等其他看似远离日常生活的事物联系起来。NFT 看似离我们很远，但其实，也离我们很近。在可预见的未来，NFT 或许会有更多 “现实世界” 的功能和价值。

NFT 的社会用例

1. 购买和出售艺术品

NFT 在数字艺术品买卖和所有权方面发挥了较大的作用。我们都知道，实物艺术品的所有权通常是通过所有者实际持有而证明。如果是古代或高价值的艺术品，该作品可能有一份实物文件（称为 “出处”），显示谁在什么时候拥有该艺术品以及它是如何转手的。然而，一些艺术品市场鱼龙混杂，很多艺术品爱好者一是无法辨别艺术品真假，二是无法以合适的价格进行购买，有些平台通过将艺术品实物配以 NFT 所有权凭证，可以更好地实现实物艺术品的流通和版权归属问题。

相比较于实物艺术品，NFT 在数字艺术品中的应用和价值更加广泛和深入。知识产权保护并不适合所有权和财产转移的方式。通过将数字商品与区块链开发上的特定点联系起来，NFT 本质上创造了一个与艺术品本身相关的不可改变的数字出处。这让收藏家以一种新的方式拥有数字艺术，同时也允许艺术家以一种新的方式出售数字艺术。

2. 证明奖杯和稀有商品的真实性

2020年，第一个 WebXR 奖以 NFT 的形式 “交付“ 给了接受者。虽然物理上不是一个可触摸的奖杯，但这种奖项的设计可以作为一个三维的数字工艺品来作更多的延伸。虽然任何人都可以提交和上传 XR 奖的图片，但只有真正的获奖者才能通过 NFT 地址和钱包向所有人证明他们获得了这个荣誉。

如今几乎大部分的奖状和奖杯都是实物的，获奖的人需要通过实地到达或邮寄的方式拿到奖杯。然而，空间和地理的局限性或许会影响获奖人到场领奖。例如，疫情期间，跨国领奖并不是非常现实。随着全球化和数字化的发展，会有越来越多的比赛或提名出现。那么，如果机构通过官方发送 NFT 奖杯，且链上可溯源，并通过 NFT 钱包向大众展示。不仅解决了异地领取奖杯问题，还一并解决了奖杯真实性查验问题，例如奖项造假等。

奖杯和 NFT 属性十分一致，限量，稀有。类似的应用机会数不胜数，如文凭，甚至财产。这些想法也有希望与 “智能合约” 的发展联系起来，区块链支持的协议可以促进个人之间的交易，不再像过去需要第三方的参与。

3.在数字复制的时代保护稀缺性

人们在理解 NFT 的价值时，常常会遇到一个难以理解的事实：在 NFT 出售后，NFT 所代表的图像或数字工艺品仍可公开获得。然而，情况并非总是如此。最近爆火的 The Bored Ape Yacht Club 结合了 NFT + 潮玩品牌的方式，让 NFT 的稀缺性成为品牌宣发的渠道和特点，或许，这能够成为社会上一些默默无闻的小品牌诉说自己故事和品牌理念的顺风车。

此外，Byte Agency 和 Dept Apparel 此前出售了一件虚拟外套的 NFT。当买家收到NFT时，他们也收到了允许他们在 Snapchat 上 “穿” 这件外套的文件。现在还有一些 NFT 发行商，可以让拥有 NFT 收藏品的人们进行二次创作，比如发行周边等，这就保持了原始购买的稀缺性和价值。

4.虚拟物品的可证明属性

NFT 通过创造 “价值互联网” 来改变互联网。我们从来没有能够真正拥有我们的互联网原生数字物品，但现在我们可以。人们拥有的大多数物品在物理世界中都有价值，但人们很少在虚拟世界中拥有有价值的物品。在区块链发明之初，人们终于可以拥有自己的 FT。通过NFTs，人们可以拥有自己的虚拟物品。

如果你是一个游戏玩家，或许你在你最喜欢的游戏角色中投入了很多时间和精力。其实，游戏资产背后都承载着物质价值。稀有物品的高昂价格并没有办法证明其稀缺性，换句话说，你并不知道游戏厂商在全网中发行的稀有资产有多少，或许是一万个，或许是无数个。

随着人们开始在虚拟环境中花费越来越多的时间，虚拟外观也变得更加重要。2021 年 5 月，在 Roblox 中，一个 Gucci 包的售价超过了实体包的市场价值。在虚拟世界中，人们同样需要社交，消费以及个性化定义，需要团体归属和身份认同。而这些都离不开资产，而 NFT 也是资产价值证明的必要关键。

5.新工作

NFT 将改变我们已知的工作。NFTs 不仅将推动远程工作，还将使人们能够在虚拟环境中谋生。今天，大多数靠视频游戏和虚拟世界谋生的人都是流媒体或内容创作者，他们从实际应用的 “外部” 获得收入。他们通常将游戏或虚拟环境作为一种娱乐形式或内容创作工具。

NFT 实现了人们称之为 “Play to Earn” 的商业模式，用户可以玩游戏，赚取 NFT，然后将这些 NFT 卖出换取收益。每个用户都能设计游戏内的游戏，或是元宇宙的元素，并通过完成任务或者完成游戏获得收益。比如, 疫情期间, Axie Infinity 成为了菲律宾一些家庭的主要经济来源，通过打游戏，一些人一周可以赚取 200 美金。 

6.新经济

我们生活在一个以国内生产总值（GDP）来衡量的世界，这意味着人们对增长和消费的重视高于一切。在一个资源有限的世界里，我们最好将资源有效地合理分配，最大化运用。

如果全球经济的很大一部分是基于互联网的，那么我们可以围绕数字资产NFT 创建经济，这些资产几乎不需要任何生产成本。不需要开采铁矿石或砍伐树木来创造更多的 NFTs，它只是一些网络代码。如果世界经济的一大部分成为虚拟的，那么我们可以开采更少的资源，消耗更少的资源，并在虚拟世界中建造我们想要的东西，同时继续为 GDP 作出贡献。

当然，我们并不确定 NFT 的这些潜在用途是否会真正实现。鉴于其他技术和趋势的融合，很多可能性仍然很值得期待。

7. 平行世界

Metaverse 这个概念已被很多人提及，但在变革游戏行业之外，它也具有一定的社会价值。互联网给很多人开了一扇窗户，足不出户就能够看到外面的世界。而多数人也仅仅只是互联网的消费者，而非创造者，我们所看到的是网络上的一个个用户名，而不是用户名背后鲜活的人。

这两天看了一部纪录片 “Born Different”，这个世界上有许许多多的人无法出远门，或许是因为身体原因，或许是因为资金不足。或许，未来，在 Metaverse 这个平行世界中，人们可以做许多在现实世界做不了的事情，比如运用 Avatar 建立一个自己的虚拟身份，展现自己的个人特色，参与各种活动，随时去想去的地方，想做什么就做什么，结识来自全世界各地的朋友。用户可以远程与朋友闲逛，创造艺术，消费艺术，畅玩游戏和纵情购物，同时还可以带着自己的虚拟身份访问其他领域。

NFT 如何助力慈善事业

除了上述社会功用和场景，NFT 自身属性也决定了它可以发挥独特优势，助力慈善事业，发挥社会价值。

刺激捐赠

非营利组织帮助解决的是世界上一些最具挑战性和紧迫性的问题，从拯救热带雨林到为最需要的人提供教育资源，再到捍卫网络自由，以及许多其他重要事业。但是，他们如何能说服人们捐钱来支持这些事业？行为科学研究结果显示：

品牌和身份

捐赠者会被与自己价值观相似的慈善机构所吸引，并对其保持忠诚。他们将慈善机构比作品牌，并建议他们使用自己的品牌来加强捐赠者围绕捐赠所建立的身份。他们还建议慈善机构针对特定的捐赠者身份来调整他们的信息传递方式。

互惠性

互惠是指用更多的积极行动来回应一个好的社会规范。例如，如果有人帮了你的忙，你会更倾向于在他的生日时给他一个礼物。世界银行的行为科学家发现，互惠可以使人们捐赠更多，他们提到了一项研究，在这项研究中，发放糖果可以增加捐赠的数量。在美国，NPR 这样的组织向他们的捐赠者发放了手提袋和其他礼品，这正是出于互惠性考虑。每次捐赠者看到手提袋，他们都会产生积极的心理反应，更有可能再次捐赠。

那么，NFT 如何助力慈善机构收到更多的捐款呢？

比方说，在捐赠过后，人们会收到官方发放的不可伪造的 “影响力代币”，以表彰他们对慈善机构的捐赠。影响力代币将是慈善机构品牌的延伸，加强了捐赠者的身份和慈善机构的价值之间的联系。影响力代币还可以成为捐赠者社交媒体身份的一部分，从而增加捐赠者的社会影响力和知名度。最后，影响力代币还可以成为授予捐赠者的额外礼物，使捐赠者将捐赠行为视为一种积极的行为，并使他或她更有可能在未来的捐赠中互惠互利。

可见，NFTs 可以通过行为科学研究设计来激励人们进行更多的捐赠。比如，动物保护慈善机构可以为他们想要帮助的野生动物制作 NFT 图标，发放给那些帮助动物的人们。

慈善拍卖

在慈善拍卖中，NFT 也能够高效、双赢地支持慈善事业发展。在这里介绍两个线上机构如何将 NFT 与慈善事业联系起来：

BeNeFiT

BeNeFiT是一个 NFT 拍卖会，助力链上的慈善事业。该市场有两个明显的特点，使其有别于其他 NFT 市场：

1）作品可以 “捐赠”，所有收益都将捐给所选择的慈善机构；

2）超过设定底价的收益将直接捐给所选择的慈善机构进行拍卖，不仅给所有者了一些回报，同时也使慈善机构受益。

这些作品的价值本质上是上升的（因为作品已经交易，必须高于底价），而慈善机构则从通过拍卖创造的多余价值中受益。

NFTb

NFTb 通过收入分享功能将艺术家与慈善机构联系起来。NFTb 的慈善捐赠 “收入分享”功能，让艺术家能够以可承受的价格拍卖他们的艺术品，并将部分收益捐给他们选择的慈善机构。

慈善机构在我们的社会中发挥着重要作用。NFT 和慈善应该携手并进。因此与那些致力于创造持久影响的慈善机构联合起来。通过 NFT，让任何人都可以以合理的价格拍卖他们的艺术品，并将部分收益捐给他们选择的慈善机构。

在结账过程中建立慈善捐赠功能。卖家可以选择与选定的慈善机构分享销售的部分收入。直接向喜欢的慈善机构捐款，而不需要通过银行和额外的文书工作。只需在出售NFT时，勾选 “与慈善基金会分享收入”。

除了创立慈善平台外，还有很多非营利组织运用 NFT 更好让人们参与其中，以及很多区块链机构也设立了自己的非营利组织。例如：

Alex’s Lemonade Stand

Alex’s Lemonade Stand 计划使用 NFT，这是一个非营利组织，专注于资助儿童癌症研究和计划，以帮助受儿童癌症影响的家庭。

Alex’s Lemonade Stand 的联合执行董事 Jay Scott 表示，他们计划使用来自儿童癌症患者的艺术品来制作 NFT，并且很可能会在 9 月的儿童癌症宣传月期间将它们拍卖。

Binance 慈善基金

Binance 慈善基金是一个非营利组织，致力于通过采用区块链技术的力量实现全球可持续发展目标。Binance 慈善及其合作伙伴共同创立了 NFT for Good，愿景是希望每个人都能将他们的艺术和创造力转化为针对社会和人道主义问题有意义的全球行动。

问题与挑战

NFT 是由人们创造的独特、可验证、独一无二的数字品，它被记录在区块链账本上，从GIF动画到3D雕塑，它可以呈现出种种形态。

艺术家可以创造独特项目，并保证自己的创造者身份被永恒标记，当项目在世界各地销售时，每次转手艺术家也都能得到一笔版税。一些巨鲸也正在涌入这个领域。但就艺术而言，现在，赚取绝大部分利润的人是那些已经非常富有的人。明星卖数字商品，亿万富翁卖GIF，似乎任何人都可以把他们的作品放在上面，进行市场营销，建立一个社群。

由于区块链上的每一个条目都需要计算机解决一个独特的加密数学问题，因此，每一笔交易，每一个进入这个账本的条目，所耗费的能量足以为一个家庭提供多年的电力。而 NFT 的买卖的财富流向也多是巨鲸持有，现在看来，这依旧是一个富人的游乐场。

以太坊联合创始人 Vitalik Buterin 曾说道，NFT 可以应用于社会相关的事业，如慈善机构和资助公共产品，但在该技术仍被视为 “主要惠及富豪名人的赌场”时，就无法发挥这种作用。

他指出，通过出售 NFT 让类似埃隆·马斯克这样的名人增加 100 万美元的银行存款，这没有什么切实的社会价值。但他也相信，在支持和协调下，NFT 可以对社会的其他领域产生重大影响。

在一篇题为 “最重要的稀缺资源是合法性” 的博文中，作者布特林指出，公众的注意力和资源往往被分配给大多数人认为具有合法性的东西，他把这个博弈论术语定义为：一种高阶的接受模式。如果某种社会环境中，人们能够在制定结果时发挥自己的作用，并被广泛接受，那么该社会环境中产生的结果就是合法的，每个人这样做是因为他们期望社会中其他人也这样做。

简而言之，如果人们认为其他人也会这样做，特别是在对他们个人有利的情况下，他们就会做一样的事情。“人们认为哪些 NFT 具有购买的吸引力，哪些不具有，这是一个合法性的问题，“布特林说。“如果大家都认为一个NFT很有趣，而另一个 NFT 很逊色，那么人们会强烈倾向购买第一个，因为它既具有较高的吹嘘价值，又具有持有它的个人自豪感，还可以更高的价格转售，每个人都在以同样的方式思考，“他补充道。

这依旧是一个流量聚集和注意力聚集的经济模式，像埃隆·马斯克这样的名人所产生的影响和吸引力是巨大的。作为特斯拉首席执行官，他曾多次推动了更广泛的数字资产市场，Twitter 首席执行官杰克-多尔西(Jack Dorsey)也对 NFT 投以关注，将他的“第一条推文”NFT以 290 万美元拍卖给慈善机构。

但是，如果仍然聚焦在这些人身上，那么 NFT 可能会失去真正的社会影响力。

布特林也提出了两种潜在的解决方法，可以帮助使 NFTs 更加“合法”，在某种程度上促进社会公益事业的筹资。比如可以建立分散的自治组织，在其分散的治理社区的集体批准下，保证部分销售收入转入慈善事业，并“批准”某些 NFT。另一种方法是与社交媒体平台合作，将 NFT 的展示整合到用户的个人资料中，让用户展示他们投资的东西。与第一个想法相结合，这种方法可以起到 “鼓励用户使用为有价值的社会事业做出贡献的 NFT"的作用。

结语

区块链技术和 NFT，就像互联网本身一样，我们可以想象用这种技术进一步变革人类的生活方式。随着越来越多的人参与到 NFT 领域，共同改变领域生态。相信 NFT 对于现实和未来的塑造，值得我们期待。
