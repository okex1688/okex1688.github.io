据 Decrypt 报道，拥有3000万月活用户的 MetaMask 母公司完成巨额融资后表示，很快将推出去中心化自治组织或 DAO，“它不会管理 MetaMask，但它将为创建新的 MetaMask 提供资金”。CEO Lubin 没有提供有关 DAO 的更多细节或发布日期，但确实确认 MetaMask 打算发布 Token。上个月在以太坊丹佛，MetaMask 运营负责人表示，Token 的发布将谨慎进行。

DAO 的发布一定程度上也是为了 MetaMask 发行 Token 的前置步骤。MetaMask 公司与主要用户均位于美国，加上其巨大的用户体量，必然会受到极其严格的监管审视，甚至不能排除类似 Libra 那样遭到临时叫停。 因此它第一步就需要避免证券代币的指责，转为类似 DAO 的治理 Token 的模式。一旦发布 Token，对于3000万甚至更多用户的空投可能成为史上最大规模的一次空投。

MetaMask 对美国的监管非常敏感。 此前注明为前 CoinDesk 员工的 José Rafael Peña 表示，在 Metamask 使用 Infura 禁止了委内瑞拉的 IP，原因是“为了遵循法律在一些特定地区无法使用”，但更换节点后仍可继续使用。后续据 Decrypt 报道，MetaMask 封锁事件原来是一场“乌龙”，为了遵守美国新的制裁计划，Infura 正在阻止克里米亚和乌东两个地区的所有 IP 地址的访问，但因为配置错误导致封锁了委内瑞拉（现已解除）。目前封锁的地区还包括伊朗、朝鲜、古巴和叙利亚。这也表明 MetaMask 在配合美国监管时的积极态度，其他类似节点服务商并未明确阻止乌东地区的 IP 访问。

此外，母公司 ConsenSys 之前遭到 35 名前雇员的控告并提出特别审计，控告指在2020年时公司转移到新实体，以换取摩根大通等传统金融机构投资了10%的股权，并且抵消了 3900 万美元的贷款，此举损害了小股东的利益。

3月15日 Metamask Infura 等的母公司 ConsenSys 宣布完成 4.5 亿美元的融资，估值超过 70 亿美元，接受的资金将马上转换为 ETH。ParaFi Capital 领投，淡马锡、软银愿景基金、微软、Anthos Capital、Sound Ventures 和 C Ventures 等跟投。Metamask 目前拥有 3000 万月活用户。值得关注的是参与投资的基本是传统行业的 VC。ConsenSys 一直也是以太坊背后的重要支持力量，MetaMask 的爆发也得益于以太坊与 EVM 的发展。这也是它罕见表示“这一轮接受数字资产和法定货币，立即转换为 ETH”的原因。

官方还透露了目前旗下加密世界第一大应用 MetaMask 小狐狸的数据：该钱包现在支持超过 3000 万月活跃用户 (MAU)，仅在四个月内增长了 42%。全球用户群依靠 MetaMask 来铸造和收集 NFT、加入 DAO 并参与 DeFi 协议。 美国、菲律宾、巴西、德国和尼日利亚是 MetaMask 最活跃的一些市场。不过在中文下沉地区，TP 钱包市场竞争力相对较高；在 Solana 上超过120万用户的 Phantom 占据绝对领先；Coinbase Wallet 也在持续发力。

官方强调，资金还将支持 MetaMask 的快速扩展，重新的设计计划于 2022 年晚些时候发布，以及推出插件可扩展系统，该系统将允许与各种区块链协议和帐户安全方案集成。这意味着 MetaMask 后续可能会集成其他非 EVM 例如 Solana 波卡等公链。

MetaMask 通过内置的聚合交易，也开始获得一定的利润。 Decrypt 报道称，它的以太坊链上日收入约为 85,000 至 220,000 美元。此前也有数据称其2021年的收入约为2亿美金。值得注意的是，这一收入与交易所、矿机厂商等相比仍然非常低。这也说明目前行业最获利的、以及最成功的的商业模式，仍然是中心化交易所（合约）以及比特币矿机厂商。虽然巨额融资以后 MetaMask 没有资金成本上的压力，但通过 Token 经济学似乎也是不得不进行的选择，否则在商业变现模式匮乏的情况下难以更进一步。
