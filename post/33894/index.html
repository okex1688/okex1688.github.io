前言

近几年 NFT 赛道引起了投资者的极大关注，一些 NFT 的售价对于很多人来说是不可想象也无法理解的。其中爆火的 NFT 收藏品系列，尤其是头像类收藏品，可以说是让大家又爱又恨，大部分用户张口就提及加密朋克，BAYC 等动辄几十上百倍价格拉升的 NFT 系列。上线几小时就翻了 70 倍的冷兔 NFT，以及持续火了一个多星期的杰伦熊 PhantaBear，一举点燃了华人圈 NFT 投资者的热情。大量巨头机构、明星的入场，让 NFT 市场中源源不断地涌入 New money。NFT 市场如此火爆，那么我们作为普通圈内用户，又该如何看待呢？

本文将从 NFT 的起源与发展开始讨论，汇总了 NFT 赛道的历史里程碑事件，再结合当下的热门 NFT 收藏品项目，分析整个赛道的价值与其中的风险。

1. NFT 的概念、起源及发展

1.1 概念

非同质化Token (Non-Fungible Token)，缩写为 NFT，被称为区块链数字账本上的数据单位，也属于加密货币中的一类。早前，「NFT」一词仅在 ERC-721 标准中流行，该标准于 2017 年通过以太坊 GitHub 首次提出。每个 NFT 都可以代表一份独一无二的数字资料，作为虚拟商品所有权的电子认证或证书。

由于其独特，不可拆分，不可互换的特性，NFT 可以代表数字文件（如画作、声音、视频、游戏中的项目等）或其他形式的创意作品。虽然 NFT 本身可以被无限复制的，但 NFT 数据可以在其底层区块链上被追踪，并提供所有权证明。

1.2 起源

谈及 NFT，大家都知道它和以太坊的发展密不可分，但其实早在 2015 年 7 月以太坊主网启动前，NFT 的概念就已问世。

1993 年，Hal Finney（第一位经由交易机制得到比特币的人）在提出的加密交易卡（Crypto trading cards）时说表示：「可以以数字资产的形式向他人展示其收藏价值」。Hal Finney 依托加密学和数学，随机排列组成一个系列的套卡，并将其定义为加密卡。

而后，其概念落地，初代 NFT Dapp：彩色币（Colored Coin）在 2012 年应运而生。彩色币是建立在现有比特币数据块上的挖矿和交易应用协议，由小面额的比特币组成，且最小单位为一聪（比特币最小单位 1/100,000,000）。它以去中心化的方式代表了不同的资产，包括股票，债券，虚拟加密货币等。尽管彩色币存在不少缺陷，但彩色币同样也实现了资产上链的可塑性，以及巨大的发展潜力，为早期 NFT 的发展奠定了基础。

图 1-1 彩色币宣传图，图源：Google 图片搜索

2014 年 Counterparty（点对点交易平台）的「RarePepes」活动，旨在收集悲伤蛙（Pepe the forg，2015 年最受欢迎虚拟角色）的全套表情包。基于比特币的 CounterParty 协议标准，每张悲伤蛙的表情包都跟一个独特的 Token 绑定，用户通过交换 Token 来交换不同的表情包，或者使用比特币购买其他表情包。每一张被创作的表情包都是独特的 NFT，通过网络用户之间的传播，悲伤蛙 NFT 逐渐成为一个 IP（知识产权），以「悲伤的」表情走入大众的视野，并史无前例的让用户感受到虚拟物品 NFT 也可以变得有价值。

图 1-2 Pepe 蛙的表情包之一，图源：Google 图片搜索

1.3 发展

NFT 正式进入人们的视野是在 2017 年，由 CryptoKitties 的创始人提出。BTC 翻了几倍后在加密市场走红，加密猫游戏诞生。随着加密猫的火爆，NFT 这个概念很快走进了大众视野。由此，数字资产不再只是指代加密货币，而是由同质化的加密货币与非同质化的 NFT 代表的加密资产组合而成。

图 1-3 有关 NFT 的 Google 搜索量折线图

根据谷歌趋势显示，NFT 一词在 2020 年以前搜索量一般，2021 年起激增到现在一直处于上升的状态，意味着人们的关注度对此越来越高，NFT 的市场容量也越来越大。

现状 从 2021 年 8 月以来，NFT 市场爆火，Facebook 高管表示正在考虑推出数字钱包的同时构建 NFT 功能；全球支付巨头 Visa 宣布以 15 万美元购买并收藏 NFT 数字艺术品 CryptoPunks。另一方面，NFT 交易价格也在不断刷新记录：一副虚拟的石头画像在 8 月 23 日以 400 枚以太坊（约合 130 万美元）的价格售出，创下了 NFT 艺术品系列 EtherRock 的最新价格纪录。

图 1-4NFT 艺术品系列 EtherRock

大多数人对于 NFT 的影响力依旧有很大的争议，有人认为，NFT 本身是没有任何价值的，高价售出主要原因是「价值认同」。还有人认为 NFT 本身的潜在价值巨大，比如在加密货币市场中，使用 NFT 能够确保用户在虚拟世界完成身份构建和社交工作，相当于把现实世界中物品独特性和稀缺性较好的复制到元宇宙内。随着加密市场的发展，NFT 的发展也渐渐形成了它独有的市场，从众多区块链项目中脱颖而出，成为当前热门赛道之一。

相信随着越来越有创意的 NFT 项目进入市场，人们对于该数字艺术品的追求也会越来越狂热，市场份额会越来越大。总体来看，NFT 市场供需格局需要未来更多项目驻入市场来带动资金的流入，让它占有一席之地。

2. NFT 从小众到出圈

2.1 年份大事记

在 1993 年 Hal Finney 提出首个 NFT 概念并将其定义为加密交易卡（Crypto trading cards）。

2012 年第一个类似 NFT 的 Token -彩色币诞生。

2014-2017 年间是 NFT 的发展期，2014 年的「RarePepes」活动，令 NFT 逐渐走入用户视野。

2017 年，加密朋克（CryptoPunks）, 以太猫（CryptoKitties）引发了第一波 NFT 热潮，提高公众对 NFT 的认知度。

2018 年随着 ERC721，ERC1155，ERC998 协议逐步完善，NFT 也变得成熟。

2020 年现象级产品 NBATopshot 横空出世，并迅速攻占市场，此外 NFT 交易市场层出不穷。

2021 年，NFT 以一系列大事件刷屏，占据了各大新闻头条，刷新了用户对区块链的认知，甚至有媒体称之为是 NFT 元年。

2021 年 3 月，Beeple 以 6903.4 万美元出售 NFT，的出售可能是最具标志性的 NFT 时刻，艺术家 Beeple 以 6900 万美元的价格将他的 NFT 艺术品卖给了 Metakovan。这无疑是加密艺术未来的决定性时刻，并向全世界宣告 NFT 是一股不可忽视的力量。

图 3-1 Beeple 拍卖作品，图源：佳士得拍卖

同年 3 月，Sandbox 推出 Beta NFT 市场，向元宇宙的创建迈进。用户通过游戏平台出售以此赚取 $SAND。

2021 年 4 月，佳士得拍卖 CryptoPunks，9 枚 CryptoPunks NFT 共拍得了 16,962,500 美元。

2021 年 6 月，Decentraland 一块数字地皮以 91.3 万美元出售给虚拟房地产开发商 Republic Realm，这块地皮代表了 259 块数字土地，以 129.5 万枚 MANA 结算（MANA 是 Decentraland 的原生 Token ）。

2021 年 7 月，NFT 交易市场 OpenSea 完成 1 亿美元 B 轮融资，由 a16z 领投。据 DuneAnalytics 数据统计，OpenSea 6 月成交额达 1.49 亿美元，成交量达 21 万件，均创历史新高，此外，OpenSea 总用户数量突破 14 万，环比增长 19%。

2021 年 8 月，知名 NBA 球星库里花费 18 万美元购买一个 BAYC NFT 头像，一系列明星经过舆论的炒作，名人效应使得 NFT 赛道变得炙手可热。

2021 年 9 月，DC 漫画将与 Palm NFT Studio 合作推出首个官方 NFT 系列。

图 3-3 Plam NFT Studio 示例

2021 年 11 月，NFT 再次拍出天价，Beeple 新作以 2500 万美元成交。11 月 10 日，艺术家 Beeple 的在佳士得以 6103 枚 ETH（约合 2898.5 万美元）的价格成交，为 CryptoArt.Ai 网站统计的史上第二贵 NFT 作品。

2.2 里程碑式项目简括

2.2.1 CryptoPunks-加密朋克

CryptoPunks 是建立在以太坊上的最早期图像类 (JEPG) NFT 项目，这些极富「朋克风格」的像素化头像由美国手游开发公司 Larva Lab 于 2017 年使用计算机代码创建，没有任何两张图像是完全相同的。因此 CryptoPunks 共发行了 1 万 个 2424、8bit 样式的不规则像素 NFT 头像，不会再生成新的 NFT。每个 NFT 通过简化的小像素块鲜活的展示了曾经风靡世界的独特「朋克风」。

在创建早期，拥有以太坊钱包的用户可以免费领取 CryptoPunks 头像。如今，用户只能在 Larva Lab 和 OpenSea 等 NFT 二级交市场购买。2017-08-19，Larva Labs 宣布 CryptoPunks 已经完全链上以太坊网络，用户不仅可以免费在 Etherscan 上查询 CryptoPunks 的属性（发型、眼镜、胡须、帽子等）还可以在 NFT 市场进行交易。

图 3-4 机密朋克 NFT 价格排行榜，图源：Larva Labs 官网

2021 年 7 月，一位匿名收藏家一小时内连续买入 102 枚 CryptoPunks，共计花费高达 2650 ETH，约合 657 万美元。这名巨鲸玩家行动时，其它玩家也豪掷千金，同年 8 月，「CryptoPunk 3100」挂出报价 4200 枚 ETH，约合 758 万美元，成为目前世界上单价最高的 NFT 作品。此后，通过不断地宣传炒作，CryptoPunks 很快打响了知名度，以马赛克般的怀旧风格以及不断成交的高昂价格刷新了区块链用户的观念，用户开始逐渐接受 NFT 这个新兴热门赛道并踊跃地参与其中，成为 NFT 收藏中的「疯狂」一员。

2.2.2 Crypto Kitties-以太猫/谜恋猫

感受到了 NFT 市场的强大发展潜力，再加上受到」前辈」CryptoPunks 的启发，Dapper Labs 团队立即推出了一款叫做 CryptoKitties 的以太猫游戏。其开发者 Dieter 用 NFT 图像表示每只小猫拥有的不同花色、基因、代际等信息。

Crypto Kitties 于 2017 年 11 月 28 正式上线营运，是一款基于以太坊 (Ethereum) 上的智能合约游戏。由温哥华 Axiom Zen 的工作室 Venture Studio 制作完成。它是一款类卡牌游戏，每张卡牌上的小猫都不同，玩家可以购买、出售甚至是繁育其后代。当玩家购买小猫后，这只小猫就是完完全全属于购买玩家的。其火热程度甚至一度让以太坊网络崩溃，也成功的在加密货币领域迅速刮起了一阵养猫风，用小猫可爱呆萌的画风将 NFT 带入公众的视野。

图 3-5 以太猫的部分示例，图源：CryptoKitties 官网

当时以太坊上最流行的通证发行标准是 ERC-20。尤其 2017 年夏天，ERC20 尤为火爆，很多用户问到加密猫和 ERC20（ERC20 是同质化通证，即 FT）如何交互？为此 Dieter 提出了 NFT，即非同质化通证。FT 的主要特点是各通证之间是没有区别的，可以随意交换，且可以拆分整合。如：BTC、ETH 等加密货币都是属于 FT，每一枚 BTC 之间本质是没有任何区别的，可以互换和分割。Dieter 所做的创新是将稀缺性引入到加密货币概念中，NFT 的最主要的特点是唯一性和不可拆分性。每一枚非同质化通证各不相同，都是单独存在的、独一无二的，且无法互相替代。

CryptoPunks 和 CryptoKitties 两个项目将 NFT 概念正式引入大众视野并带进市场，其被炒作的价格也让市场看到了此类 NFT 的无限潜力。并且 CryptoPunks 首次让人们将头像类 NFT 作为一种文化代表，达成了用 NFT 来彰显身份象征与社交地位的共识。

3.1 CryptoSkulls-

图 4-4 CryptoSkulls 基本信息介绍，截至 2022-01-19

CryptoSkulls 是存储在区块链上的 10,000 个可收藏的 ERC-721 Token 。每个 CryptoSkull 角色都是一个独特的像素艺术图像。从 2019 年开始的 OG NFT 项目，每个 CryptoSkull 字符都是具有唯一性索引属性的唯一像素艺术。

最便宜的 CryptoSkulls 入手价其实不过 0.05ETH，而如今地板价都已经 2.75ETH。交易量暴增，更曾一度打败 Bored Ape Yacht Club 成为 TOP 级别的 NFT。无论是交易额以至升幅都是极之惊人，因此成为最近 NFT 圈子的热门话题。

图 4-5 部分 CryptoSkulls NFT 交易界面展示

CryptoSkulls 在其庞大的社区和名人投资者方面与 CryptoPunks 和 Bored Ape 等其他成功项目有很多共同之处，该项目还赋予持有者完全许可权，这是它与其他成功的 NFT 项目的一个相似之处。如今，有些 NFT 玩家出售他们的 CryptoPunks 以获得 CryptoSkulls。

许多人预测 NFT 空间将成为消费者日常使用的场所，音乐会门票、数字商品和其他收藏品等作为 NFT 出售，具有即时认证和消费者奖励的能力作为主要激励措施大规模采用。

3.2 Bored Ape Yacht Club-

图 4-6 BAYC 基本信息介绍，截至 2022-01-19

Bored Ape Yacht Club 也被戏称为「无聊猿」，是一个收集了 10,000 个独特的 Bored Ape NFTs 俱乐部，不仅是基于以太坊区块链上所建立的独特数字收藏品，而现在 BAYC 的地位，是仅次于 CryptoPunks 的热门项目。

BAYC 由美国 Web3 公司 Yuga Labs 于 2021-04-29 推出，由创作者使用演算法编程混合并匹配猿猴，创作出 10,000 个不同的猿猴图像。这些由算法呈现出的猿猴或出现在色彩斑斓的背景下，或穿着新奇古怪的服装，或佩戴着可爱搞怪的饰品。色彩鲜明且独一无二的设计感再加上高昂的价格，让不少 NFT 用户为其疯狂。

图 4-7 部分 BAYC 图像 交易界面展示

名人效应一直以来都是 NFT 市场销售不可或缺的一部分，而 BAYC 所附带的名人效应更是再一次证明了 NFT 市场的疯狂和能量。随着知名的 NFT 收藏家 Pranksy 的收购，」无聊猿「的热度不断发酵，不仅受到明星，说唱歌手，球星，主持人等一系列名人的青睐，各种各样的售卖新闻接踵而至。大批大批的 NFT 用户源源不断地涌入，都在为拥有一张「无聊猿」头像而庆幸。当需求远远大于供给时，BAYC 的价格也变得水涨船高，从最初发售价 0.08 ETH 到目前 Opensea 上的最低价格 82 ETH（约等于数十万美元）渐渐的也发展成了一个潮流 IP。

BAYC 的演变发展无疑是令人震撼且唏嘘的：

1）最初发起人基于开放社群的初衷，希望促进一个积极向上的社群。

2）Bored Ape NFT 头像作为用户进入其俱乐部的基本门槛，可以享受俱乐部会员的专属福利，解锁未来的区域和特权。

3）名人效应使其变成了供不应求的价格高昂的 VIP 象征，成为炫耀的资本。

4）再到现在没有实际应用而被挂上「名不副实」的潮牌商品。

可尽管 NFT 用户抨击此类的 NFT，在下一次发售中，还是会有大量的用户为它「争破头脑」。相对的，BAYC 作为 NFT 赛道中成功的产品，其短短 9 个月内爆发出的力量，不得不让人承认其强大的影响力。在 NFT 赛道中，总是会有像加密朋克（CryptoPunks）和 BAYC 这样的项目不仅刷新用户认知还展现出 NFT 的发展潜力。

3.3 PhantaBear-

图 4-8 PhantaBear 基本信息介绍，截至 2022-01-19

PhantaBear 项目由周杰伦的 PHANTACi 和 Ezek 联合发起。PhantaBear 是 10,000 个通过算法生成的数字收藏品的集合，可兼作 Ezek 俱乐部的会员卡。该项目于 2022 年 1 月 1 日 11 时在 Ezek 平台发行。预售价格为 0.26ETH。每个 PhantaBear 都有一套独特的特征，并为其所有者解锁不同的、独特的访问权限和特权。

做为周杰伦间接代言的 NFT 头像，PhantaBear 在发展时就自带吸粉效应。

但是尽管如此，其发展也并非一帆风顺，在诞生初期也曾遭受了极大的阻碍。其昂贵的发售价格促使用户前期并不买账，加上后期周杰伦经纪公司的公开澄清并未参与 NFT 制作和销售，导致其价格大跌和滞销。

然而其转折点是，周杰伦仍然十分支持 PhantaBear 发行平台 Ezek 创始人刘畊宏的创作项目，换上该 NFT 头像表示支持，并随后在海外社交媒体上开始了对 PhantaBear 的大力宣传。

同时也包括许多其他明星例如林俊杰，陈冠希等纷纷晒出自己的 PhantaBear，这一系列举动引爆了国内玩家对 PhantaBear 的追捧，并且还是在中国用户在 NFT 领域中处于弱势地位情况之下，使得 PhantaBear 价格飞速上涨，达到「一熊难求」的局面。这也是粉丝经济与名人效应的代表之一。

粉丝流量和名人效益在 NFT 圈子来说是非常重要的，任何一个项目的发展离不开人气的累计，而明星所带来的市场高度满足这一条件。

但是随着时间的推移，短暂的名人效应会消失殆尽，因为其本质上技术层次并未带来升级，这也侧面证明该方式不属于长久之计。截止到 2022 年 1 月 21 日，在其他热门 NFT 项目层出不穷的情况下，PhantaBear 的热度和交易量仍然在持续下降，其 Floorprice（最低价）也从最高峰 10ETH 跌回了 [2-3 个 ETH] 左右。不过最近项目方准备了一系列空投活动给予那些拥有 PhantaBear 的用户们，这也进一步说明其共识效应仍在持续，至于未来更多线路发展和机会，还需耐心等待观察。

PANews 文章称，Azuki 走红与其三种 NFT「新玩法」有关。

4. NFT 价值论述

4.1 NFT 的价值特性

不同于现实世界中的传统行业，基于互联网发展的区块链技术在信息传递以及传播方式上有着明确且高效的改变。而 NFT 不仅做为区块链中的一大发展板块同时也是经济学中非必需品种类之一，其价值来源取决于群众对该物品的附加属性。

对于非必需品来说，改变其总价值的方式包括通过增加群体基数来扩大物品需求市场，和通过增强物品的稀缺性 (减少数量)、附加值等来间接加强其原本具备的身份和权力认证。NFT 就刚好具有这样的特性：

另外，除去 NFT 最基础的特性，部分额外的因素维度通常能赋能一系列独特的 NFT 收藏品

1）基础价值

NFT 收藏品具备一定的收藏价值和观赏价值，给收藏该作品的用户带来心里上的满足感、和对外虚荣感的体现等。团队通过自己的特色、用心设计的独特 NFT 序列是其发展的基础，不过对于实际应用类的 NFT 而言，比如使用权凭证、财产凭证等，其审美价值则显得不那么重要。

2）实用价值

部分 NFT 收藏品不仅用于收藏，项目团队为了该项目持续发展，必然会持续的赋能这些 NFT，让它们在特殊的场景中具有独特的功能。从本质上来说，项目方希望通过旗下 NFT 项目来铸造一定的 IP 形象，产生从中心向外围产生辐射效应。

1）参与游戏。卡牌类，元宇宙资产类的 NFT 收藏品，部分可以当作参与游戏的「门票」，另外一部分则相对复杂，用于扩充游戏之中的额外功能和体验，增加用户可玩性，例如 SANDBOX 中土地可以用于开展官方或私人举办的活动等。

2）会员特权。例如 CryptoPunks 项目，热爱该项目的人们自然而言的组成了社区，任何人都可以参与，但拥有 NFT 的用户可以不定期获得部分福利（如抽奖，空投等）

  1. 数字身份象征。随着数字社会的发展，网络交流时间的增加，身份象征和形象具备了更多的适用场景，逐渐成为一种真实且不断扩大的需求。最能体现该特性就是 NFT 头像类收藏，目前 CryptoPunks 以及成为身份、地位的象征。犹如加入」上流社会「的」门票「，拥有该「门票」即可进入该圈子，获得与其他同样拥有该 NFT 用户交流探讨的机会等。

  2. 衍生活动。目前 Bored Ape Yacht Club 社区衍生出动漫制作、游戏等内容。尽管项目方在最开始并未规划该区域，但是随着社区自治的发展，开始逐渐向新方向拓宽道路，也同时推动该品牌下一系列产品活动等。

3）历史价值

NFT 收藏品创作的历史背景，反映出所对应的时代以及蕴含的历史意义。其发展通常伴随着社区的建立再到认知的沉淀，最后达成共识的完善即可成为独特社区文化，同时也会成为该 NFT 所映射的历史价值之一。

4.2 NFT 的价值困境

自于 2021 年 NFT 领域开始爆发，在 2021 上半年，其整体市值就达到 127 亿美元，相较 2018 年增长近 310 倍。同时也在那个时候，各大 NFT 平台完成了千万美元级别以上的 A 轮融资包括 OpenSea、SuperRare、Bitski、Rarible 等专注于不同领域的 NFT 交易市场。另外基于 NFT 的经济系统，国内外互联网大厂也开始布局 NFT。例如腾讯和阿里所构建的蚂蚁链、腾讯至信链等，都是在制度的前提下开始寻求热度与功能性。表面上国内外 NFT 市场全面繁荣，但是其背后潜在的危机却从未离开。

1）当今 NFT 赛道发展中存在大量的投机现象，过度炒作、盲目追捧和疯狂创造短期热点，这使得 NFT 市场中蕴含着大量泡沫，一旦市场低迷的时候就会发现绝大部分 NFT 流动性约等于无。

最初火爆整个币圈的 NFT 竞拍《Everydays：The First 5000 Days》买家并不是正统的收藏家，而是一位资深的币圈玩家，也是 NFT 基金 Metapurse 的创始人 Metakovan。他打着「让艺术品收藏更民主」的口号，发布了 1000 万枚 Token 。随后这些 Token 的价格从 0.36 美元一路飙升到了 28 美元。然后他在这场竞拍中不仅赚到了噱头，扣除拍卖支付费用后，还大赚一笔。

从市场目前投机氛围与 Fomo 情绪来看，市面上绝大部分 NFT 存在「噱头拉高价格」，或者通过大 V，明星等来参与项目宣传炒作，或者对敲交易（左手倒右手），市场价格远大于市场价值，而一旦市场泡沫破裂，NFT 资产价值总量将会进行大幅度缩水，超过 90%NFT 藏品将因为缺乏流动属性成为空气。

2）迄今为止，全球监管中尚未出台关于 NFT 法律规范，所以在这里不受任何监管的 NFT 成为践踏法律的重灾区

目前在全球范围内，NFT 实体商品属性仍难以界定，未来真正在元宇宙中发挥支撑经济体系作用的 NFT 虚拟商品也难以被定性。对于包括 NFT 在内的各种数字资产属于何种财产权利，目前也尚无定论。也正因为缺乏官方概念的指导，所以该领域必然存在各种」庞氏骗局」的犯罪活动。同时部分自媒体网红并未承担好正确引领大众的责任义务，反而通过夸大价值和思想误导，给一系列毫无艺术价值的 NFT「价值」包装，谎称艺术品投资，收藏品众筹等，运用区块链新型技术创造骗局。

3）NFT 的安全保障。NFT 做为数字产品其产生都是在区块链上执行的一段标准化软件代码，高度依赖于底层区块链协议的属性

NFT 的铸造是基于以太坊网络，但目前从以太坊网络面临拓展性的难题，这也势必会影响其衍生品 NFT 的发展。另外在 NFT 创造途径与过程中也不是一帆风顺，目前的区块链技术处于高速发展时期但其安全隐私方面并不完善，所以并没有绝对的安全保障。这也势必会导致部分 NFT 资产遭受黑客等不法分子的攻击和盗取。

除了外部因素的隐患，许多 NFT 交易市场的安全制度是否达到标准？正如前段时间 Opensea 交易市场直接冻结了被盗 NFT 资产的行为，同样引起了社区的质疑，在交易 NFT 的过程中是否达到足够的去中心化，可追溯性等来保证交易的完整性? 尽管目前暂未发生大规模安全隐私泄露事件，但我们仍需要关注这些隐蔽的角落，防止它们有机可乘。

5. NFT 赛道总结与提示

在微观层面上，对单个 NFT 销售的研究揭示了加密货币艺术市场与传统艺术市场之间的巨大相似性，尽管两者之间存在根本差异（例如，交易透明度，无中间人参与等）。

最近加拿大概念艺术家 Kimberly Parker 的一项研究，分析了 OpenSea 的数据以及其他市场的销售信息，发现近 80% 的 NFT 售价低于 500 美元，而只有 1.8% 的初级销售（即 NFT 的首次销售）售价超过了 0.5 ETH（据研究时汇率约 894 美元）。顶级稀缺资产从不缺市场，而大量长尾资产却一直滞销。这与因「头重脚轻」、「赢家通吃」而臭名昭著的传统艺术市场现状非常相似。去中心化并不意味着机会平等。

从宏观层面来看，NFT 整个赛道正在兴起，这是毋庸置疑的。

但其长期价值其实仍然不够明朗。从长远来看，这些 NFT 被赋予的价值，关键取决于它们与线下世界的联系是什么。这个联系可能很简单，就是一个支持其创造和交流的社区。如果这个社区缺乏参与，那么与 NFT 相关的大部分价值也就流失了。所以 NFT 或许会成为一种价值的存储形式，但在此之前，我们会经历一个阶段，即现实世界的社会关系和社区价值转移或反映至链上。

我们目前正处于 NFT 发展的起步阶段，无法预测 NFT 未来将怎样改变区块链上的经济行为和模式。

NFT 凭借铸造数字资产所有权的能力，以及易存储、受保护的特性，改变了不少艺术家和创作者的谋生方式，并带来了财富和机会。但在 NFT 赛道的发展过程中，也将面临比加密资产更多的不确定性和更大的风险。绝大多数 NFT 项目的生命周期非常短，过了火爆期之后，流动性变得不容乐观，极有可能就变成了钱包里的 JPG 图片。

NFT 赛道 FAQ

Q1：如何确定 NFT 的稀缺性和价值？

NFT 的价值取决于其所代表的内容。如果是数字货币艺术品，则价值衡量标准与其他任何类型的艺术品非常相似。我们需要考虑创作者、作品的艺术价值以及各路收藏家的需求度。如果 NFT 限量发行或属于系列收藏品的一部分，拥有特定编号的 NFT 通常更具收藏价值。

Q2：在哪里可以获得 NFT？

获得 NFT 的途径一般有市场购买、项目方赠送以及自行铸造。NFT 市场出售众多非同质化 Token ，创作者既有著名艺术家，也有业余爱好者。在众多产品中，规模最大的是基于以太坊的 NFT 平台 OpenSea。新的市场平台一直在不断上线，如果有兴趣购买著名艺术家的作品，请务必验证市场的真实性。

Q3：如何验证 NFT 的真实性？

一般核验都需要通过区块链浏览器来查看 NFT。有效信息包括铸造日期和铸造相应 NFT 的钱包地址。您还可以根据交易历史记录 ID 查看 NFT 是否匹配。这种方法比单纯检查与收藏品相关的照片或文件更可靠。涉及到资产时，请一定拒绝轻信，亲自核实。
