NFT如何定价、NFT如何选择购买？这是目前市面上比较关注的方向之一，本文通过从NFT的不同属性出发，在说明NFT价值来源的同时，也为NFT价值的增持提供了独特的思路。

该如何精准地诠释NFT的价值来源？本文通过社交、艺术、应用和资产四个维度进行分析，对NFT进行了解构与组合，相信对于当下的NFT创作者和投资玩家而言，这将显得尤为重要。

临渊羡鱼不如有的放矢，知其然还要知其所以然 。这是本文对NFT最大的贡献。

NFT赋予的图片独特性与持有者在社区权益的相等认同。

早在两年前，没人能想象如今的NFT市场会如此蓬勃的发展，每天都有无数新的NFT项目问世。可证明的唯一令牌，即使简化为简单的JPEG格式（这也是NFT出现在用户屏幕上的格式），也代表其持有者的可追溯价值。

在NFT的“欺骗性”JPEG外观之下，隐藏着各种技术进步。新技术总是被创造出来，然后被采用来满足我们的基本需求、需求和愿望。

NFT已成为主流媒体、大家争相讨论，但现在依然有些人对NFT的惊人表现感到反感。然而，正如一句名言所言：“否认我们的冲动就是否认使我们成为人类的东西。”

 Web 2.0降低了互联网用户之间的社交互动成本，并很快对用户的在线/离线互动产生了影响。

而Web 3.0通过将验证过程包含在智能合约中，解决了信任问题。由于区块链技术的发展，NFT像野火一样蔓延，并通过社交媒体、娱乐等落地应用，其价值发现市场以及相应的游戏行业得到广泛的普及。

NFT跨链应用可能是下一个NFT使用场景。但是，我们将在下一篇文章中讨论这个问题，并深度探讨NFT的未来及其在跨链环境中的各种应用。

NFT的底层着力点在哪里？

持有者者认为Axie Infinity NFT提供了游戏参与的效用、参与社区治理以及通过P2E奖励产生收入的能力。

另一个例子是BAYC NFT。BAYC因其向其所有者提供的广阔、著名和富有的网络而拥有了它的大部分效用。在每个可能的效用中，主题便是NFT如何为更广泛的受众提供效用。

NFT效用是NFT的购买者和持有者将自己视为特定项目的HODLER/所有者的好处。

社交赋能

2021年春天，数字身份发生了快速的转变。Twitter用户首先注意到了这一点：越来越多的Twitter账户将他们的头像变成了卡通形象的猿猴。

当NBA球员和众多A级名人进入NFT市场时，BAYC在NFT二级市场平台Opensea上聚集了数百万的ETH交易量。

BAYC和其他生成NFT项目的成功可以归功于它们的产品顺利落地并满足了社区对社会化效用的需求。在区块链技术的支持下，NFT坚不可摧的独特性为满足个人对社交化数字身份的需求奠定了基础。

NFT社区的目的是在内部和外部建立、达成和聚合共识：

  • 内部是社区成员的社会化效用如何满足。

  • 外部是市场价格在过去、现在和未来（预期）的表现。

由于难以为NFT找到新的使用场景，NFT的价值很大程度上（不完全是）基于社区共享的共识。NFT用户倾向于将NFT价值与其市场价格联系起来，即Opensea上项目的地板价。

艺术赋能

我认为NFT成功的艺术属性在于艺术家灵感的独特性、他们在区块链技术方面的工作以及其内在的艺术价值水平。

NFT艺术品的价值基于其文化影响力和意义。

一件“好”的艺术品将展示主流价值观、道德标准，并对社区中的成员产生激励。从根本上具有积极的主流价值观、社区愿景和提升社会的有影响力的愿望。

与社区或可用性相比，这种特定类型的NFT观众会更关注NFT的艺术品。因此，价值也放在值得信赖的艺术家或值得信赖的公司身上。

市场行为类似于有收藏爱好的传统艺术品收藏家。这些收藏家欣赏艺术，不会经常交易他们的作品，因此他们对应用价值不感兴趣。

功能赋能

游戏化的NFT

实际游戏体验，玩家可以选择为游戏功能付费，对未来回报的期望不高，游戏内NFT是用于游戏活动而非金融产品的印象，为玩家提供了情感投射，尤其是当玩家花费资源升级NFT，花时间与NFT角色一起战斗时。

这样的预测进一步降低了用户对未来回报的期望。

音乐化的NFT

音乐NFT允许粉丝购买歌曲、专辑和活动门票。它们还为艺术家提供了更多关于他们的名字、肖像、形象和艺术作品的盈利能力。

DAO治理的NFT

DAO被推测为下一代公司（的表现形式）。它们为协调全球投资和社区提供了一种新的组织形式。如今，许多DAO组织正在兴起。在治理方面，将使用多种形式的NFT，功能NFT就是一个可能的例子。

DAO成员可以通过其持有的NFT为DAO的发展做出治理贡献。做出突出贡献的会员也可以获DAO Token的空投。同时，DAO NFT也会作为DAO系统的通行证（来发挥其治理作用）

NFT对现实资产的重新定义

在加密货币的金融性质下，NFT具有特定的资产属性。

它们放大的市场价格波动是影响其实际价值的一个因素。NFT源自交易、贷款、拆分和ETF等金融应用的许多变体，可以整合复杂的金融合同关系。

因此，NFT已成为可信的金融投资，代表了加密货币世界的所有权，并在智能合约中存储了有保障的权利。

以NFT为载体的金融行为将会越来越多，甚至成为未来NFT的主要价值来源。

区块链技术和智能合约为艺术家和粉丝提供了实用性。艺术家将从他们的作品中保留更多的利润，想要表达支持他们最喜欢的艺术家的粉丝可以通过购买他们的NFT来做到这一点。

门票和独家通行证是音乐行业NFT的未来元素。

元宇宙的概念引起了对加密世界的极大关注。但是，如果没有NFT不同效用的集成，元宇宙的构想就无法实现。因此，社会化效用与艺术效用的耦合是必要的。

社交属性+艺术

通过社区的形成，用户基于对艺术作品的热爱这一共同共识而相互联系，并由此表现出强烈的社交愿望。此外，通过社区推广艺术品进一步扩大并有助于社区对项目的共识。

以白名单机制为例。在项目的预售阶段之前，最忠诚和最有影响力的社区成员将获得白名单。这使他们能够在公开发售之前铸造一定数量的NFT。

因此，受到激励的艺术品信徒受到激励和鼓励，将项目传播给社区以外的人。热情的社区成员加强了对项目的共同期望，邀请更多的人加入社区，从而创造了一个向上的螺旋。

艺术风格

NFT是通过区块链技术和前卫艺术的结合产生的。两者都受到主流媒体的质疑，这些NFT的核心是高度抽象的，需要市场参与者更深入的参与和理解。

艺术家通过绘画包装他们的自我身份，并通过不同的区块链将他们的作品展示给公众。在参与和理解的过程中，观众会形成“理解和认同”的共同共识，最终导致社区的出现。

通过社区发展，NFTs将被解构并赋予更多新的含义。从而吸引更多的玩家进入，促进一个积极而有价值的社区循环。

与社区融合的风格

事实上，社区和艺术成为文化的具体表现。文化影响可以影响世界各地的NFT观众，并为艺术家提供一个富有想象力的空间。

NFT是艺术和社会化效用的完美结合。以太坊网络固有的高gas费用带来的低流动性给NFT社区带来了净正面的市场预期。

另一方面，社区可以由艺术需求驱动。将社区效应与市场投机相结合，生成的NFT面向一群受众，包括艺术爱好者、KOL、投机者和社区建设者。随着社区产生的价值增加，新成员将流入，强化社区价值观，从而产生螺旋上升效应。

我们可以在游戏、音乐、域名、保险、收藏品、实物资产和身份等领域看到NFT。此外，NFT充当展示资产和映射虚拟资产的容器。

用户的持有不是基于对NFT内容的喜爱，也不是基于NFT的社交表达，而是基于获取回报等真实需求。

总结一下NFT的价值表现

每个单独的NFT的性能各不相同。但是，通过结合各种应用程序，NFT的价值会增长。

NFT在过去一年中发展迅速。Cryptopunks、BAYC等主流项目已经实现了数亿美元的交易额。根据Messari的数据，在过去的一年里，NFT二级市值已经从1亿美元上升到50亿美元。整个NFT二级市场销售已达220亿美元。该数据强调了NFT市场正在蓬勃发展。

随着Web 3.0的兴起和元宇宙的引入，未来的NFT将对我们的生活、社交和投资方式产生更深远的影响。集成不同NFT效用的能力将为NFT和区块链采用带来难以想象的价值。

我们见证了集成NFT不同效用的各种项目的成功，然后以单一产品的形式呈现。最终，产品有效地解决了一系列问题，即对人类基本需求的不满足。所述产品的非凡性能归功于解决问题的有效性。

国外NFT市场的发展历程较长，NFT应用场景呈现多元化发展趋势，头像、游戏、票务、音乐各类新项目层出不穷。

目前大多数NFT的展现形式只能满足视听感受（如音乐、视频、图片），但也有平台将其作为一种权益证明或者引流工具使用，比如采用实物绑定、票务凭证这样的方式规定持有NFT的用户所享有的权益（如商业使用权、投票权、会员权益等）。

也有IP方通过与NFT游戏平台合作发行NFT游戏装备，尝试颠覆传统游戏中玩家仅享有装备使用权而非所有权的固有范式，赋予玩家所有权，玩家可以在NFT平台创建和定制虚拟展厅，并接入元宇宙社区共创想象空间。

或像是近期大热的头像类项目（如无聊猿BAYC），其流动性与自身设计、明星效应强相关，用户对于头部项目的价值共识越来越强，虽然这类头像项目在加密社区很火热，可以看作满足了部分群体社区身份认同的需求，但仅仅是静态jpeg格式的展示头像持久性存疑。

延伸一下，回过头来我们聊聊国内现状

目前，NFT行业仍存在较多待探讨的问题。

首先是炒作问题。 从源头来说NFT的价格已经脱离了实物的发行定价逻辑，在NFT自由流转市场，买家容易在NFT的巨大价格泡沫里陷入狂热，把NFT视为金融投资产品进行投资炒作；

其次是权责问题。 一是虽然区块链技术能够对已上链的数字内容进行信息溯源，保证数字内容上链后的真实性和所有权归属，但数字形态的内容本身很容易被低成本复制，平台无法追溯链下的版权归属。

二是部分平台发 行的NFT用户协议认为用户仅有使用权而非所有权，发行后的权属纠纷、权责问题（发行方、发行平台、用户、创作者角色之间的权利义务以及应承担的法律责任）少有典型案例和正确答案可以参考。

第三是存储问题。 受成本影响，NFT通常不会直接把图片、视频之类的数据存储在链上，而是储存于其他中心化或非中心化的存储系统中，并通过哈希或URL映射上链，这意味着部分NFT项目内容存在内容资源丢失或被篡改的风险；

第四是监管问题。 2021年以来NFT热度急速上升，市场开发的NFT平台和发行NFT的项目不胜枚举，利用NFT噱头融资洗钱、平台跑路事件频发，相关监管机构较难控制这类事件的发生，同时行业也需要既能保障参与者权益、促进NFT有序发展，又不损害创造性的监管边界。
