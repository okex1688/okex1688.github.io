无论是名人、大品牌还是企业，都会有更多的目光投向 NFT 领域，从而导致其被大规模采用。

NFT在 2021 年迅速普及，但仍有很大一部分人不知道 NFT 是什么。回顾下图，NFT 要真正成为主流还需要做很多工作。

名人

名人一直是 NFT 兴起的关键支持者。最初，NFT 始于艺术家、创作者和交易者，他们为今天的 NFT 奠定了基础。著名人物也进入了这个领域，无论是通过创建自己的项目还是简单的收藏。

名人无疑是 NFT 增长的主要催化剂，毫无疑问，他们的影响将有助于将加密文化巩固。

Adidas（阿迪达斯）

也许迄今为止最重要的主流跨界合作是阿迪达斯与 Bored Ape Yacht Club、Gmoney 和 PUNKS Comics 的合作。阿迪达斯推出了一个新的 Twitter 帐户Adidas Originals，与这些新合作伙伴讨论他们的计划，此举令 NFT 世界兴奋不已。

Adidas Originals宣布了一些重要的事情，首先，他们计划进入元宇宙，一个允许用户在未来进行虚拟交互的数字世界。为了说明他们的想法，他们发布了一个预告片，描绘了阿迪达斯的无聊猿及其新合作伙伴无缝穿越天空并进入元宇宙的画面。

Adidas Originals 账户还将其个人资料图片更改为新购买的 Bored Ape Yacht Club NFT #8774，这是一种非常特殊的蓝猿，他们命名为 Indigo Herz，戴着心形眼镜和渔夫帽，将成为其品牌在元宇宙中的形象。

Adidas Originals为什么如此具有开创性？

到目前为止，NFT 一直由个人组成。无论是普通交易者、创作者还是名人，推动NFT向前发展的都是个人。Adidas，这是一个传统的主流品牌，在 Twitter 上拥有 400 万粉丝，在 Instagram 上拥有 2600 万粉丝。这个品牌的存在有能力为这个空间带来比所有个人加起来更多的关注，从而刺激 NFT 的增长。

为了进一步实现这一愿景，阿迪达斯还与领先的加密货币交易平台 Coinbase合作。此外，它还与最受欢迎的元宇宙游戏之一 The Sandbox 合作。在一条推文中，阿迪达斯展示了自己的The Sandbox图，这是一块数字土地，正式巩固了其在这个虚拟世界中的地位。

育碧

育碧本周宣布 Ubisoft Quartz将在 Tezos 区块链上可用，是下一家遵循 Web3 路线的大公司。该平台的前提是用户将能够使用加密货币买卖名为“数字”的游戏内物品。这很重要，因为它将授予玩家游戏内物品的所有权，而在常规游戏中，这些物品只能在这些游戏自己的生态系统中使用并依赖于这些游戏自己的生态系统，而不是区块链，后者允许所有权完全独立于游戏世界。

为什么这如此重要？

育碧以超过 58 亿美元的资本位列最大的 25 家游戏公司之一。如果这只是进入 NFT 的第一步，那么它在未来肯定能够做更复杂的事情。与阿迪达斯和其他深耕该领域的大品牌相比，它们都成功地让人们更加关注 NFT 。

尽管育碧的进入将 NFT 带入主流，但他们并没有受到欢迎。事实上，他们的 YouTube 视频在第一个小时内就收到了 20,000 多个不喜欢，该公司在其社交媒体上遭到了强烈反对，迫使其将视频下架。

在界定评论的范围时，很明显仇恨源于多个来源，其中包括：

  • 不管 NFT 是什么，都不喜欢这家公司

  • 不喜欢付款；对于许多人来说，成本可能是无法承受的

  • 不喜欢 NFT 所需的能源消耗

  • 纯粹不喜欢 NFT

这反映了一个事实，即 NFT 需要大量工作才能被更广泛的主流受众接受和采用。一些领先的游戏平台，例如 Steam，已经完全禁止在他们的商店中销售区块链游戏，育碧是否会继续其计划还有待观察。

是什么阻碍了 NFT进入主流，可以进行哪些改进？

污名化和缺乏信任是 NFT 被阻止进入主流的两个最根本原因。确实，该领域存在不良玩家，就像现实生活中也有骗子一样。但这并没有考虑到整个空间。

为了进一步防止诈骗，无疑必须为 NFT 用户引入更好的通信模式和更安全的平台，以确保使用它们的人的安全。

主流采用的另一个障碍是，如果您是该领域的新手，如果您不知道从哪里开始，设置 Coinbase 帐户和 MetaMask 钱包可能会非常混乱。每一步都涉及费用、转移和各种问题，这使其成为进入的严重障碍。对于普通人来说，这甚至会阻止他们考虑进入该空间。因此，如果 NFT 想要吸引主流，就必须有一个更简单的入门过程。

入门的复杂性是进入的严重障碍，虽然指南很有帮助，但需要更全面的东西，让每个参与者的操作更轻松。在此之前，很难说 NFT 何时会被主流采用。

NFT将不断发展

NFT 在被主流采用之前还有很长的路要走。无论是名人、大品牌还是公司，都会有更多的目光投向这个领域，随着时间的推移，消费者会认识到这一点，思想可能会发生转变。但是，在此之前，需要在教育和可访问性方面做很多工作。

监管和权力下放显然各有利弊。集中式平台允许框架和安全。相反，权力下放提供了创新和自由统治。但不利的一面是，法规会阻止所有权和自由统治，而权力下放会滋生骗子和恶意行事的人。

我们所知道的是，NFT 是一项仍处于试验阶段的新技术，但我们已经看到了主流公司和品牌的吸引力，NFT 的采用是不可避免的，无论是缓慢的还是快速的。
