沉浸在 NFT 领域的玩家，已经习惯了「盲盒」这种 NFT 发行方式。

和泡泡玛特的盲盒一样，投资者事先并不知道自己获得的 NFT 是什么样子。自 NBA Top Shot 风靡后，盲盒占据了 NFT 的主要玩法。就像当年链上博彩游戏出现导致大家对链上真随机数追捧一样，投资者对于链上盲盒有绝对信任，似乎在合约写死下，链上盲盒不会出现任何意外。然而除去链上合约以外，开发团队的上传方式，创作团队的内容把控都是决定 NFT 盲盒安全的重要因素。

最近几款倍受关注的盲盒 NFT 出现了严重不随机、内容提前泄露、内容侵权等重大事故，导致地板价破发，社区怨声载道，有的甚至直接下架。

BoredMummyWakingUp——木乃伊跌下地板重新沉睡

「无聊的木乃伊」，因为出色的设计，融合了各个国家的传统元素而做成的木乃伊 NFT 系列，在盲盒开盒前有超过 2700 个拥有者。

不过，开盒当天出现了接近 3 个小时的推迟，在开盒之后又遇到重大问题，开发者在上传到云服务器时出现失误，导致所有稀有的木乃伊 NFT 出现在了 6000-6444 号之间。这意味着，在 6000-6444 区间以外的投资者获得稀有物品的概率为百分之零。

随后的几个小时里面，开发团队将所有 8888 个 NFT 召回重新洗牌，让获得稀有 NFT 的投资者大为不满。事故的几个小时后开发团队在推特和 discord 社区进行迅速补救方案。

1.以收藏家的购买价格回收 NFT

2.继续持有，直到事件结束也许会给每一个人发放空投

事后，根据开发者在 discord 的公告称「这次事件的补救方案目前花费了整个团队将近 100ETH。」而且支出还在不断攀升。

另外因为数据出错和显示错误，还导致有些珍稀度很高的木乃伊系列，在 rarity tools 等网站的排名很低，到目前为止这个错误还没有被更正。

虽然 BoredMummyWakingUp 整个开发团队从事发到解决问题的反应都非常迅速，而却在积极的解决每个收藏家的问题，但是即使如此还是避免不了部分收藏家们的热情丧失，地板价也一路从 0.19ETH 跌到 0.03ETH（截止日期到 2021 年 8 月 18 日）。

The Wanderers——我在太空什么都不知道，所以什么都没发生

「The Wanderers」，具有迷幻赛博风格的飞船驾驶舱，因为简约独特的设计，在上线 27 分钟就被抢购一空。开发团队还承诺给每个拥有者发放制作高清图片的工具，可以用来作为社交头像和背景。

The Wanderers 出现的问题和 BoredMummyWakingUp 相近，在盲盒揭示之前因为提前上传了超链接，导致很多人可以提前从底层的超链接中提取 NFT 数据，造成即使盲盒没有开盒也能知道里面内容的事故。对于买家来说，摆在面前的就是透明包装的「盲」盒。因此在开合过后 The Wanderers 系列在二级市场 OpenSea 上充斥了大量价格不等的未开盲盒。

截止到撰稿时间为止，The Wanderers 的开发者团队并没有就此事发表歉意，并提出任何补救措施。相比于木乃伊项目，The Wanderers 不提不问的方式并没有阻止二级市场崩盘。地板价跌到了在 0.05ETH 左右（截止到 2021 年 8 月 18 日）。

Sad Frogs District——比悲伤蛙更悲伤的是它的 Owners

悲伤蛙因为以风靡社交网络的悲伤蛙 Pepe 为基础进行再创作，短时间内在 raity tools 的 7 日价值榜单飙升。虽然同样采取盲盒形式，但是悲伤蛙的开发团队并没有像上述两个团队一样出现数据失误。而问题出在了设计团队，因为涉嫌抄袭，悲伤蛙 NFT 被原作者警告并投递律师函，OpenSea 在同一时间下架了所有 Sad Frogs District NFT 的商品。

对于悲伤蛙下架，开发团队虽然在 dicord 社区内部道歉，同时也在聚集社区群众努力控评，反对负面消息。

而对于下架及解决办法，开发团队在社区中表示：「我们对 DMCA（美国颁布的数字千年版权法）投诉的回应已经提交到了 OpenSea。我们对侵权的指控提出异议，我们认为悲伤蛙只是互联网 meme 文化的简单延伸，绝对不会侵犯 Matt Furie（原作者）和他的 Pepe 作品的版权。我们只是在模仿青蛙这个物种，就像 Doge、Shiba 一样。Matt 有 10 天的时间聘请法律顾问反驳我们的回应，如果他这样做了，他就是在破坏他创作 meme 的初衷，更重要的是破坏了 1900 名 Sad Frogs District NFT 持有者的经济。1900 名普通人生活毁灭。选择权在你的手里 Matte，你选择做恶魔还是救世主。

虽然开发团队据理力争否认侵权行为，至于什么时候能够重新上架，官方给出消息，将在删除与 Pepe 相关元素（主要是眼睛和嘴巴）之后重新上架。

上述所说的三个项目并不能让爱好盲盒的收藏家们丧失热情，但是一周之内事故频发也不得不让我们提高警觉。

NFT 虽然拥有着独一无二性、可证明稀缺性、可流动性、可交易性并且能证明拥者是谁的极高价值，但是与此同时也带来了市场的鱼龙混杂。许多并不成熟的项目，并不成熟的团队为了能提早分一杯羹，而忽略了对项目最基本的打磨就冲入市场。出现了很多山寨「创意」、抄袭「创意」，他们粗糙的包一层「致敬」的口号，简单做一个「诱人」的发展路线图（Roadmap），就能在 Crypto 社区炒作营销，妄图获得成功。

NFT 的本质归根结底还是艺术家的创作，无论是单一的作品，还是批量生产的系列作品都是艺术家们本身的创意和表达诉求。

社区头像模式的成功，无疑让更多投资者看到机会。在盲盒 NFT 安全性无法保证的情况下，跟风炒作非常容易让投资者亏损。Crypto Punks 只有一个，Bored Ape Yacht Club 只有一个，Pudgy Penguins 也只有一个。

对于开发者来说，现象级的作品是为了让更多的人关注到 NFT 领域，并不能成为你跟风投机的参考目标。对于收藏家来说，不能太过专注于社交属性和升值空间而忽略了艺术价值和审美导向。一个真正好的的 NFT 项目永远是基于大众审美的需求，而非推特领袖的审美。
