在《 zkSync 2.0 大冒险：哪些项目值得体验？》一文中，我们分享了 zkSync 2.0 体验教程，本期将汇总已经支持或将要支持 zkSync 的 70 多个项目。

时至今日，Layer2 将成为区块链行业的重要基础设施层这一趋势已经被市场认可，而其中将计算和状态存储移至链下的 Rollups 有非常大的潜力成为以太坊关键扩容解决方案。

Vitalik Buterin 曾表示，在短期之内，Optimistic Rollups 可能会在通用 EVM 计算上胜出，而 ZK Rollups 可能会在简单的支付、交易和其他特定于应用程序的用例中胜出，但从中长期来看，随着 ZK-SNARK 技术的改进，ZK Rollups 将在所有用例中胜出。

Matter Labs 开发的 zkSync 作为基于 ZK Rollups 架构的扩容方案，已经上线主要聚焦于支付目的的 1.0 主网和完全兼容 EVM 的通用 2.0 测试网，在 zkSync 2.0 中，L2 状态分为具有链上数据可用性的 zkRollup 和具有链下数据可用性的 zkPorter，而且两者可组合和互操作。本月 zkSync 还升级了其 2.0 门户，支持以任何代币支付网络费用，极大地提升了用户使用灵活性。

zkSync 扩容方案不仅获得了大量资本的青睐，截止去年 11 月份 Matter Labs 从 a16z、Union Square Ventures、Placeholder 共融到 5800 万美元，当然，也吸引着其他生态的项目或基础设施。

根据 zkSync 官网生态全景和此前公布的信息，zkSync 生态目前共有 70 多个项目，其中基础设施占据过半，包括 10 来个钱包、15 个桥或跨链解决方案、支付网关等，还有多家交易所也将支持 zkSync 存取。另外，已经或将要支持 zkSync 的 DeFi 类项目有 30 个。

不过，由于 zkSync 1.0 尚不兼容 EVM，因此目前已经支持或集成 zkSync 的项目仅有 10 来个，多为钱包和支付网关类项目，包括 Argent、ZigZag、Ramp Network、Banxa、钱包 Numio、imToken、Orbiter Finance、MathWallet、支付网关 Sprintcheckout、The Graph。 AMM 协议 Mute.io 已经上线 zkSync 2.0 测试网。另外，目前还可以通过 zkSync 参与 Gitcoin 捐助活动。预计 zkSync 2.0 正式上线后才会在 DeFi 和 NFT 领域推动生态发展。

基础设施

钱包

Argent（已上线）

本月上旬，Argent 全面开放 Layer2 账户注册，现在任何人都可以访问其以太坊 zkSync 网络 Layer2 钱包。

Numio（已上线）

Numio 是一个基于以太坊和 zkSync 的 Layer2 移动钱包应用，该应用还集成了加密货币购买和兑换功能。根据路线图，Numio 之后会集成其他 Layer2 解决方案，以及整合 NFT、DeFi 功能，包括挖矿和借贷等。

imToken 和 MathWallet 也已集成 zkSync。Zerion、D’CENT Wallet 以及 ONTO Wallet 应该会在之后集成。

另外，zkSync 在其钱包官方页面也支持通过 Wallet Connect 或以太坊钱包 MetaMask、Formatic、Trezor 和 Ledger 来登陆 zkSync，之后 MetaMask、1inch Wallet 和 MyKey 也会支持 zkSync。

桥

目前共有 15 个跨链解决方案或桥已经或将要集成 zkSync，其中 ZigZag 和 Orbiter Finance 已支持 zkSync 与其他区块链间的跨链。

ZigZag Bridge（已上线）

ZigZag Exchange 是一个基于 ZK Rollup 的订单簿 DEX。另外，ZigZag 桥功能还支持以太坊和 zkSync 的资产兑换，也上线 zkSync 到 Polygon 的跨链桥，不过 zkSync - Polygon 跨链桥尚处于 alpha 模式，还未集成到 UI 页面，目前只能手动发送到地址。

Orbiter Finance（已上线）

Orbiter Finance 是一个跨 Rollup 桥，目前支持在以太坊、zkSync、Arbitrum、Polygon、Optimism 之间转移 ETH 和 USDC，近期还增加了对 StarkNet Goerli 测试网的支持。

Ferrum Network

Ferrum Network 是一家跨链区块链即服务 DeFi 公司，核心产品之一是跨链桥。Ferrum Network 还是 Matter Labs 5000 万美元 B 轮融资的投资方。本月初，Ferrum Network 表示，会将其产品套件与 zkSync 扩容方案集成，包括传统质押、VIP 质押、NFT 质押、多资产质押、LP 质押等，还将在 zkSync 整合其跨链聚合器 InfinitySwap。

Via Protocol  

Via Protocol 是一个跨链流动性聚合协议，目前支持的网络包括以太坊、BNB Chain、Fantom、Avalanche、Polygon、Arbitrum、Harmony、HECO、Optimism、OEC、Gnosis Chain 和 Moonriver，之后也会支持 zkSync。

根据官网，Via.Exchange 的合作伙伴包括 NEAR、Biconomy 等，此前完成 120 万美元前种子轮融资，投资方包括 Ravikant Capital 和 Shima Capital 等。

其他准备或正在集成 zkSync 的桥基础设施包括 Automata Network、Connext、Celer、Composable Finance、Poly Network、Multichain、Nomad（跨链通信解决方案，代币桥已上线以太坊和 Moonbeam）和 deBridge，这里将不再作进一步介绍。

此外，根据 zkSync 上个月更新的内容显示，支持 zkSync 的桥还将包括 DAI Wormhole（由 Maker 构建，将允许用户在 Layer2 间转移 DAI）、Hop Exchange 以及 LayerZero 等。

支付/网关/CEX（5 个）

提供法币到加密的网关解决方案的 Banxa、MoonPay、Ramp Network、Simplex、UTORG 都会通过各种支付方式直接在 zkSync 提供上购买加密货币的能力，其中，Banxa 和 Ramp Network 已经开始支持 zkSync。

基于 ZkSync 的 Layer2 支付网关 Sprintcheckout 也已上线。Sprintcheckout 面向商家推出了支付网关，可让商家客户通过 ZkSync 支付，接受 ETH 或 ERC-20 代币支付形式。每笔交易中，客户不需要支付费用，商家需要支付 0.5% 的 Sprintcheckout 交易费用和 ZkSync 费用。

此外，可将加密货币从 CEX 转移到 Layer2 的项目 LayerSwap 也支持将 ETH 或 USDC 从 Coinbase、FTX、币安、OKX、KuCoin 和火币转移到 zkSync。

区块浏览器（3）

Matter Labs 官方此前已经推出 zkSync 区块浏览器和分析平台 zkScan。除此之外，Matter Labs 还表示，之后还将确保用户能够通过 Etherscan 和 Blockscout 访问 zkSync 数据。

其他（8）

区块链开发平台 Tenderly 将集成 zkSync，允许开发人员监控和测试智能合约。以太坊开发工具 Hardhat 将允许开发人员轻松编译合约并在开发网络上运行。

The Graph 和 Covalent 都会与 zkSync 2.0 集成，以提供查询区块链数据的有效方式。

Chainlink 此前表示，在 zkSync 2.0 上构建的开发人员将可以通过 Chainlink 预言机访问链下数据。

数字资产管理工具 Gnosis Safe 将允许用户托管资产，并提供多重签名功能。

zkSync 列出的基础设施生态中，还有比如质押服务提供商 stakefish、去中心化存储项目 Arweave 等。

交易所

包括 Crypto.com、Bybit、Huobi、Blockchain.com、OKEx 和 Ripio 在内的主要交易所也在为其用户构建使用 zkSync 的直接存款和取款功能。

DeFi（30 个）

截止目前，zkSync 网站列出的 DeFi 生态系统中有 30 个项目，部分与前面列出的项目存在重复情况，不过目前上线 zkSync 主网的仅有基于 ZK Rollup 的订单簿 DEX ZigZag，AMM 协议 Mute.io 已上线 zkSync 2.0 Goerli 测试网。

其他将要构建于 zkSync 上的项目还包括：

  • Babylon Finance（资产管理协议 ）

  • Dank Protocol（固定利率协议 ）

  • CVI（由 COTI 构建的加密波动率指数协议）

  • Hedgey（DeFi 衍生协议）

  • Angle Protocol（稳定币协议 ）

  • Gro Protocol（稳定币收益率聚合器）

  • Increment（基于 zkSync 并利用 Curve V2 的 AMM 执行交易的的永续合约协议，还在 2 月份获得过 Aave Grant 捐赠）

  • Yearn Finance

  • Mute.io（基于 ZK-Rollup 的 AMM 和 IDO 平台，已上线 zkSync Goerli 测试网）

  • Frax（算法稳定币协议）

  • WOO Network

  • Taker Protocol（针对 NFT 和合成资产等的流动性协议 ）

  • 1inch Network

  • 88mph

  • Tracer DAO（去中心化衍生品协议）

  • Hashflow

  • Alchemix

  • Vovo Finance（结构化产品协议，此前已经在 Arbitrum 上线）

  • OlympusDAO

NFT/游戏

由于 zkSync 1.0 不兼容 EVM，所以当前唯一可用的 NFT 功能是内置的钱包铸造功能。其他正探索在 zkSync 2.0 上构建的 NFT 项目、游戏或者市场包括：

Mint Square（基于 Layer 2 ZK Rollup 的 NFT 市场，此前已在 StarkNet Goerli 测试网上支持 NFT 铸造，之后还会支持 zkSync 2.0）

tofuNFT （专注于 GameFi 和收藏品的多链 NFT 市场）

OpenStars（P2E 太空策略游戏）

其他

其他探索在 zkSync 上构建的生态项目还包括 Snapshot、允许任何人创建和管理去中心化组织的 Aragon、Forward Protocol 以及 clr.fund 等。

Forward Protocol 采用易于使用的类似 WordPress 的模型来促进无代码环境，即使没有技术知识，用户也可以与之交互。协议的模块化设计允许任何组织选择适合其平台的模块并对其进行修改以使其适用于他们的用例。

clr.fund 是一个将匹配资金有效分配给以太坊生态系统公共产品的、基于 Gnosis Chain 的二次融资协议，结合了二次方融资、zk-SNARKs、最小化反共谋基础设施（MACI）以及防 Sybil 攻击，致力于为公共产品提供更加民主化且更多的资金支持。

最后

zkSync 此前已确认转向去中心化和发布，在最近更新的内容中，zkSync 表示，当前 zkSync 完全由 zkSync 团队的服务器运行，是中心化的，目前接近收支平衡，当然也会不时根据 Gas 费用调整其费用。不过 zkSync 会在之后迈向去中心化和发币，目前的优先事项还是解决以太坊的可扩展性。

当 zkSync 完全去中心化时，会引入一个独立的共识机制和两个不同的角色，其中，验证者需要质押代币，还有一个角色是 Guardian，即，质押其代币份额提名验证者的大部分持有者。

风险提示：基于 zkSync 的原生项目绝大多数处于测试阶段，且未过多披露项目团队和安全审计等信息，因此提醒用户自行甄别风险。
