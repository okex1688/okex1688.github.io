Web 3无法脱离中心化服务

最近很多人批评web3 的点在于，它实际上并不是去中心化的，因为这里混合了一些中心化服务，例如OpenSea等NFT市场，以及Alchemy 等数据可用性服务。（注：Celestia联合创始人Nick White指出，Alchemy、the Graph和所有其他区块链索引项目并不是数据可用性服务，Filecoin 和 Arweave等文件存储协议也不是。）

这种批评是基于对 web3 倡导者所说的「去中心化」的错误理解， 我会尽力解释这一点。

web3中会存在中心化服务，就像web1中也存在一样。web3 中的关键问题是网络效应是作为私人物品（如 web2 中的）还是公共物品（如 web1 中的那样）产生的。

网络效应是留住用户和开发者的主要原因。如果我试图离开Twitter，我就会失去多年来积累的粉丝。这是因为在 web2 中，像Twitter这样的私人公司会形成网络效应。

但我可以离开我的网络托管服务提供商，通过切换我的 DNS 记录来保留我的入站链接、搜索排名等。

这是因为在 web1 中，由于 HTTP 和 SMTP 等开放协议以及 DNS 等社区拥有的服务，网络效应是作为一种公共资源累积起来。

互联网的杀手级应用是网络。web和电子邮件是网络，Instagram 和 Twitter 等社交应用程序是网络，优步和 Airbnb 等市场也是网络。

网络效应使企业拥有的网络，如Facebook和Twitter，能够占据主导地位，并获得极高的接受率（web2接受率在30%-100%之间）。

区块链提供了一种强大的新方法来构建网络，网络效应作为公共物品累积，就像它们在web1中所做的那样。

例如，以太坊NFT可以看作是一个用户和NFT互操作并形成连接的网络。该网络建立在以太坊区块链上。用户可以完全控制他们的数据。

有多种中心化的服务可以让你访问这个网络，包括OpenSea、Zora、LooksRare等。

（我上面提到的Zora，是指它的网站，请注意，Zora本身是一个去中心化协议）。

但是这些服务不会产生网络效应，正如我在这里解释的那样。

web3 中将会有中心化的服务。事实上，正如我们在 web1 中看到的那样，公有网络开启了围绕它们的大量创新和创业浪潮。

90 年代的企业家和投资者知道他们可以自由地建设和投资，而不必担心网络会改变他们的经济状况或贬低他们的访问权限，就像私有网络经常发生的那样。

城市中的公园吸引路人，而路人又反过来帮助附近的餐馆和其他企业。同样，公有网络释放了原本不存在的创业精神和机会。

Web2由在位者主导，他们花费巨资从他们控制的网络中提取数据和资金。互联网存在一种严重的风险，例如，广播电视被一些占市场主导地位的公司拥有，并且没有为初创公司和新想法提供空间。

好消息是我们仍处于互联网发展的早期阶段，未来几十年可能会建立许多新的网络。

web3 提供了一种构建网络的新方法，将 web1 的公有网络效应与 web2 的高级功能相结合。
