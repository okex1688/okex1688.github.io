粉丝经济革命的前夜：如何用 NFT 和代币经济玩转娱乐业 IP 运营？

撰文：Cuy Sheffield，Visa 全球加密产品团队负责人；编译：Perry Wang

我们每天都在消费以「角色」为中心的流行娱乐产品，一组成功的角色（链闻注：中国人更喜欢称之为「IP」）可以成为特许经营授权的基石，例如《星球大战》、漫威、哈利波特。这些角色可以带来长达几十年的稳定收入，并被纳入跨平台和跨媒体类型的成功产品之中。

但是今天，大多数成功的角色的背后，都是由一家公司掌握知识产权。这意味着粉丝对这些角色没有任何管理权，更不用说直接的所有权了，导致粉丝们只是公司所生产的产品和叙事的被动消费者。 即使粉丝购买公司公开上市的股权以表示支持或忠诚，也很难对单个角色或特许经营实体的成功进行有针对性的下注，因为许多公司都是大型、多元化、垂直整合的企业，拥有和经营多条产品线。无论是过去还是将来，通过股东投票，是不可能让粉丝和投资者选择哪个演员应该在系列电影的下一部影片中扮演他们最喜欢的角色，或者做出其他此类有影响力的决策。  

可能有些人会说，提出这个要求就很荒诞。但现实是，最热情的粉丝已经建立了在线社区、组织活动，甚至在网上发布了自己创作的同人小说。与其对这些社区不予理睬，角色（或IP）的持有者可以选择将他们带入创作过程，提供一个途径来测试和扩展想法，而使用传统工具和技术，很难大规模管理这些途径。

如今，像去中心化自治组织(DAO)和非同质代币(NFT)等加密技术实现了一种新的IP开发和所有权模式，不仅可以将创意媒体细分，还可以降低在线社区介入「角色」或 IP 的门槛，为现实世界带来新的角色。它还可能催生出能更充分地代表粉丝社区的 IP 。

打破「IP治理」的闭环

企业内外已经发生了更广泛的文化转变，如，环境、社会和公司治理(ESG)投资、激进型投资或各种创作者粉丝社区的兴起。这些不同参与方式的共同点是：利益相关者或社区寻求新的方式来组织和投入时间和金钱，以支持吸引他们的人和事业。 而围绕文化类产品的知识产权方面，我们正处于同样的风口浪尖之上。

我个人供职于电子商务支付和广告平台TrialPay（已被 Visa 收购）的职业生涯早期，曾与一些最成功的免费游戏开发商合作，帮助他们拓展虚拟经济并将其变现。其中一款游戏可以作为一个非常典型的早期案例研究，让粉丝感受到可以利用自己的手机控制一个高人气IP的「独享」感觉。 

这款游戏是「金·卡戴珊:好莱坞」（ Kim Kardashian: Hollywood，链闻注：是一款明星养成游戏，由 Glu Mobile 于 2014 年推出，曾长期占据美国 App Store 榜首）（https://studybreaks.com/tvfilm/kim-kardashian-hollywood-is-still-shockingly-popular-why/），这是一款可以选择你自己冒险风格的简单游戏，玩家可以假装自己是金·卡戴珊，并为游戏中的化身角色敲定服装、虚构电影拍摄和外观。

尽管这款游戏成为高人气爆款，但该游戏中的「货币」（https://kimkardashianhollywood.fandom.com/wiki/Stars），以及粉丝做出的无数选择、所决定的所有输出，都被限制在一个闭环世界内——这些早期的游戏体验充分演示了对现实中存在的人气「角色」进行模拟治理所拥有的壮丽前景：粉丝们个人可以享受这些「个性」，但他们从未得到机会，以更有意义的创造性方式与社区联手合作。

而如果粉丝可以将一个新角色引入现实世界，让任何人都有机会参与其成功带来的乐趣和经济收益，那会怎样？

DAO 和代币经济激励

DAO 闪亮登场。这些去中心化的自治组织为世界各地的创意人士提供了一种投入真金白银、创建社区和新角色的机制——就像梦幻（Fantasy）系列体育游戏迎合了体育迷对挚爱球队的所有权（以及潜在经济收益）的潜在渴望一样。

去中心化版的「梦幻好莱坞」也很有市场——只是尚未得到满足。

DAO 将如何实现这一目标？简而言之，DAO 是由智能合约或区块链上自动执行代码运行的网络，如果您是该 DAO 的成员，它可以对某些权利和责任做出承诺，同时很少或没有执行主动监督的中心人物。世界上任何拥有手机和互联网连接的人都可以参与到这样一个网络中，该网络可以根据参与者的贡献（或基于协议创建者决定的任何其他因素）向参与者发行代币。这些代币可以带来一定的投票权或治理权，如果越来越多的人希望参与该网络，这些代币可能会升值（不仅仅是指货币层面的升值）。由于代币本身的经济激励，网络效用最大化符合所有参与者的利益，包括使用他们的「股权」对正在进行的运营做出良好的集体决策。

成功 DAO 的初次迭代主要是围绕金融协议形成的，例如通过提议和投票对协议进行特定的更改，使代币持有者社区能够管理去中心化借贷。虽然这种结构意味着去中心化金融（DeFi）生态系统中新协议的增长和运行成为可能，但大多数主流消费者不具备所必需的金融知识或兴趣，来衡量评判类似抵押比率等特定治理决策。

但是 DAO 的概念具备更广泛的用途，并且也可以应用于共享激励和共同利益者网络的其他用例。因此，下一次 DAO 迭代将围绕创意社区出现，催生出众包创意和协调（又被称为「创作者 DAO」等）。

但是社区可以围绕 IP 、而不仅仅是创作者形成，并且这些社区可以进行协作，并将这些角色推广至主流观众视野中。 

围绕IP构建社区

目前至少有两条路径可以围绕角色构建社区，在其 IP 上进行创新，围绕角色创建数字身份，并从 IP 的市场成功中获得经济收益。未来几年肯定会涌现出更多路径，不过我会使用当前范例来演示每个模型。 

CryptoPunks

CryptoPunks 展示了开发者 Larva Labs 为 10,000 个角色创建数字艺术品的一种途径，其中每个角色都作为具有独特属性的 NFT 存在。

围绕 CryptoPunk NFT 所开发产品形成一个去中心化收藏家社区，该社区拥有自己的文化行为和规范，例如使用 CryptoPunk 作为各个社交媒体平台的个人资料头像图片。尽管 CryptoPunks（https://www.larvalabs.com/cryptopunks#getapunk）最初于 2017 年免费发布，但在社区引导下，该项目产品的终身销售额已超过 6.8 亿美元（最稀有的个别 CryptoPunk 售价超过 700 万美元）。

不过，收藏家们不仅是被动地持有 CryptoPunk，还开始相互合作，创作由 CryptoPunk 角色艺术启发出的新故事，使他们的CryptoPunk「拥有生命」。

NFT 艺术就像一个数字乐高积木，或者一种具有创造性的干细胞——无论哪种方式，都为丰富的想象力提供了向各个方向、以各种方式发展的基础。例如，一群收藏家创作了一部包含 16 位朋克人物的 PUNKS 漫画（https://punkscomic.com/），并附有背景故事和叙事线，将为这些朋克创造完整的 IP 身份。 

然而，这类社区计划完全不涉及 CryptoPunks 背后原始开发人员的参与。该社区仅将原始朋克作为灵感来创作全新的艺术，扩展到超出 Larva Labs创意控制和监督的新形式。

这些受朋克启发的新故事本身可以成为新的 NFT，社区成员可以将其货币化并作为粉丝艺术品出售，而无需获得 IP 持有者的许可协议。当涉及到由传统公司所拥有角色的同人小说时，粉丝的创造力可能会增加这些角色的价值和曝光率，但公司不允许这些粉丝分享这种价值提升。不过对于 PUNKS Comic，粉丝们已经对核心 CryptoPunk NFT 拥有所有权。因此他们以朋克为基础创造的角色和故事所取得的成功，推动了原有朋克的曝光率及对其的需求——使 PUNKS 漫画创作者、原创开发人员和更广泛的朋克社区共同受益。

Aku

通过粉丝社区掌控角色的另一条途径始于某一个特定的角色，这一角色具有启发 NFT 创造的背景故事和身份，但随后向收藏家社区开放，使其能够参与角色的演变。这方面的一个例子是 Aku，一个年轻的黑人宇航员角色，由前职棒大联盟（MLB）球员（同时是一位自学成才的艺术家）Micah Johnson 创作而成。

Johnson 无意中听到他的侄子问他母亲：宇航员是否可以是黑人。这带给他灵感，他开始画他的侄子戴着宇航员头盔的作品——鉴于现实中缺乏真实案例和叙事角色，以鼓励他，希望他拥有能够实现这个梦想的信心。然后 Johnson 创作了一个新的基于 NFT 的角色，这个角色是一个戴着宇航员头盔的黑人男孩。

当我看到 Johnson 的作品并与其周边社区互动时，对于可以由数亿互联网用户（而不是私人机构）确定艺术品价值、并轻松拥有代表该艺术品的代币，这一构想所具备的巨大潜力令我感到震惊。此外，他们可以利用代表其社区成员身份的这个代币，来建立数字文化身份。

对于像  Johnson 这样的艺术家，以及所有建立社区或拥有粉丝的艺术家来说，这意义重大。横跨音乐、文学和视觉艺术的黑人文化和创造性表达形式已经蓬勃发展，并推动了几代主流全球文化，但从历史上看，黑人艺术家和创意工作人员很难捕捉到文化的价值，更别说拥有，即使是部分价值。看到一位才华横溢的黑人艺术家将代表鼓舞人心黑人角色的 NFT 出售给以黑人为主的收藏家群体，这种力量，就像我之前写过的那样，感觉就像是一场新的文化运动（黑人数字复兴）的萌芽阶段。

就在几个月前， Johnson 宣布他将通过创造十部短视频形式的动画片，每次以一个 Aku 限定版 NFT 为主角，讲述 Aku 故事的一章，与不同的人和不同的地方互动。Aku NFT 不仅通过前两章推动了超过 200 万美元的销售额，而且 NFT 拥有了热情的粉丝社区，粉丝们希望看到 Aku 作为黑人孩子的正能量代表，以及黑人经济赋权的象征而取得成功。而通过拥有这些 NFT，早期的 Aku 收藏家从 Aku 的成功中获得了经济收益，这反过来又会提升 Aku 数字纪念品的需求和价值。

粉丝社区拥有和运营角色 IP 的潜力在传统媒体渠道也收获了共鸣：Johnson 不仅与 3D 雕塑家合作创造了 Aku 的实体雕像，而且它显然将成为第一个被选中拍摄电影和电视剧的 NFT。

「IP能创造经济利益」

在上述两种途径中，无论是通过创建还是扩展角色故事，NFT 都可以成为艺术家测试市场对其角色需求的强大工具。

从某种意义上说，这是一次试镜，或者被看作最小可行性产品(MVP)，是他们创作愿景的「角色与市场契合度」测试——其中 NFT 的早期收藏家是艺术风格、特质和艺术信息传递的风向标，有助于日后可以吸引更广泛的受众。

然而，相比与游戏本身没有任何经济瓜葛的传统粉丝小组，NFT 这种模式向艺术家和其他收藏家提供了更强的信号。如果 NFT 的角色有足够的市场需求，足以让个人愿意花费数百到数千美元来收集这些 NFT，那么创作者就会知道「IP 能创造经济利益」，就像游戏能创造经济利益一样。  

一旦某个 NFT 能够维持一个充满热情的早期收藏家社区，而这些收藏家与IP的经济利益密切挂钩，接下来的挑战就变成了如何发展壮大该社区、进一步发展IP，并通过主流媒体平台将其分发给广泛的受众。

事实上，我相信这种模式也可以让大公司受益，因为在当前的 IP 系统下，很难协调和实现由不同公司所拥有的多个角色IP之间的梦幻联动，因为这些角色IP将彼此视为直接竞争对手。即使两家独立的公司拥有的两个角色的粉丝群存在明显的重叠，要实现这种合作也是极具挑战性的。偶尔可能会有交叉（就像 DC 和漫威偶尔为之），但很难做到既为两家公司提供足够的经济收益，同时又最大限度地减少两家产品的潜在竞争问题。NFT 和 DAO 指向了一条光明的道路。

但首先，这种进行创造性工作的全新方式会带来哪些不同？  

在与 Micah Johnson 这样的艺术家交谈时，他的去中心化讲故事模式创造了一种「选择你自己的冒险」实践方法，社区成员可以为他们的IP推荐许多不同的故事或经历。

但这将如何具体实现呢？社区可以将治理代币分发给IP NFT的持有者，后者又可以利用这些代币，对关键的创意决策进行投票。创作者和社区之间的这种协作关系已经成真，并且随着参与者探索创意（包括治理和财务）的可能性，只会越来越多。

这是不是众包的另一种版本，而众包可是创造优质角色 IP 的最糟糕方式，它会不会导致创作者的愿景仅仅被打折兑现？  

不会，因为这种全新的、加密原生的经济激励使得一切变得不同：实际上创造了一个广阔的新模式领域，创作者在其中更多地成为去中心化粉丝社区的领导者。我们很快就会开始看到，社区成员一起开启来自艺术家和机构的公开征求建议书(RFP)，针对围绕 IP 构建的特定数字内容类型展开讨论，就像人们在开源项目中开发软件（已经做了几十年）一样的模式 。

IP DAO 的崛起

  • 但这项技术还可以实现更多功能。鉴于目前的技术，下面是一个完全可能兑现的模式：、社区成员为短片或系列动画片起草草案。代币持有者——已经获得 NFT 或赋予多个持有者同等管理权重的可替代「社交代币」的社区成员——可以对其进行投票，决定是否批准，并为其分配初始预算。

  • 然后，制作公司可以对此做出回应，制作出计划如何具体兑现草案愿景的预告片。社区审核所有提交的预告片，并用其代币对要资助哪个预告片进行投票。

  • 社区创始艺术家作为创意总监为代币持有者工作，帮助管理RFP流程，然后与竞标成功的公司密切合作，以具体执行。

  • 收藏家购买初始 NFT 以及持续空投的资金，可以再投资到社区金库，用于围绕该 NFT IP 众筹创作其它数字内容，以提高品牌知名度，并发展周边社区。 

这听起来可能有些遥不可及，但事实并非如此。考虑一下各类艺术家多么热切地探索 NFT 和其他与粉丝社区互动的新方式。加密的互操作性只是给出了创新空间，使得想法可以快速建立在彼此之上。这样的模是有什么好处？下面可以介绍几个：

提升创新迭代速度。通过社区拥有该角色 IP ——但将该IP的剧集、电影或视频游戏的创意执行外包，社区可能会创造出全新的全球特许经营实体，而且执行速度比单一公司更快。

「IP DAO」可以资助由瞄准不同市场的独立团队执行的多个节目、电影、视频游戏和商品（实体和数字商品），而不是每次只针对一个市场执行一个内容激活，上述行动可以在同一时间同步完成。小型、模块化社区催生了更多实验，而加密货币将担任协调所有激励措施的角色。

简化协作。围绕不同 IP 出现多个独立的、创造性的 DAO，它们之间的所有权将出现重叠。但这不是系统性漏洞，而是一个特色：这种重叠为跨 DAO 角色之间的协作创造了机会，例如通过创建包含两个角色的数字内容，使其能够推动跨不同社区的分发，并创建共同的受众（和品牌推广大使），而无需担心竞争问题。 

各 DAO 之间一致的经济激励。 社区成员将被激励以客户身份支持协作，并作为品牌大使进行推广**。IP 每次成功被媒体激活，都会有新的观众转化为社区成员，他们也想参与其中，帮助拥有和管理刚刚在自己喜欢的电影中看到的角色。**  

更出色的文化展示。除了创造具有不同文化（例如少数民族和宗教少数群体）的正能量角色之外，这类模式还使得所讲述的故事类型实现大众化。与其试图寻找现有的角色，具有相似价值观的父母实际上可以合作并汇集创意和财务资源，来创建自己的IP，真正匹配他们希望自己孩子拥有的价值观。

捕捉价值，提升收入。这不仅仅是一家自我感觉良好的企业——其中存在着真正的做市潜力。随着全球社区和观众不断发展壮大，以及对治理代币和 NFT 的需求增加，它可以带来额外的收入，可用于资助角色相关的更多故事和媒体作品。 

治理相关问题

当然，这里面也不全是乐趣和游戏——毕竟，DAO 不仅仅是一个抽象的加密概念，而是由人组成的系统。在围绕共同原则或目标创建新的人类协调模式的同时，在有效治理、日常执行和扩容方面也提出了新的挑战。其中包括以下一些问题，这将需要大量实验，但我愿意在这里分享一些想法：

DAO 应该围绕哪些类型的决策进行优化？  

如果社区需要对角色的每一个小细节进行投票，具体体验就会变得不那么有趣，效率也会大大降低，参与度也可能会随之降低。但是，如果投票决定的级别太高，社区成员可能会觉得他们没有足够的控制权和所有权。

谁将管理日常行政和社区管理职能？ 

我认为 DAO 作为「创意委员会」会更具效率，对关键的高级战略决策和角色进行投票，同时通过 RFP 将产品管理和创意开发外包给第三方。 

DAO 如何围绕 IP 实施品控？  

主要角色特许经营实体对角色 IP 可以做什么、不能做什么、说什么有严格的规定，以确立一致性、身份，以及质量。社区需要为角色 IP 特征确立自己的指南或原则，成员可以使用这些指南何原则来评估新提案。最终，如果社区寻求在不同形式媒体上同时实现IP的不同激活，其中一些作品会比其他作品更成功、有更好的体验。这里的关键是，所有这些实验都可以以在公司内部无法实现的方式进行。 

DAO 如何将角色 IP 链下产生的收入转换回链上金库？  

NFT 的销售催生了一种简单的方法，代币持有者管理的链上金库可以回笼资金。但是 DAO 可能需要第三方管理员，这些管理员可以在 DAO 的指导下提供支付和合约服务，用桥连接链下（例如，现实世界）和链上的收入、费用和资金管理。 

将新角色带入现实世界、测试它是否与特定观众产生共鸣，以及围绕该角色构建各种形式的媒体和故事，做这些事情的成本和障碍正在急剧下降——这要归功于加密技术，具体得益于 NFT 和 DAO 的出现。

不仅仅是艺术家个人将拥有巨大的机会，围绕自己的作品构建粉丝社区，消费者还能够从被动参与者转变为主动参与者——与他们支持的艺术家共同讲故事，之前被冷落的利益相关者，如父母、代表性不足的创作者/消费者 ，以及其他人士，也都找到了新的舞台和发声途径。
