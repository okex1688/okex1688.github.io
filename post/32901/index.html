去年 12 月，我对育碧印象最深的就是它的 NFT 项目「Ubisoft Quartz」。育碧称 Ubisoft Quartz 能为玩家提供一种全新的体验，一些游戏皮肤和装备将以NFT形式限量发放，它们不仅是游戏中的道具，还可以在指定的第三方市场上进行交易。

在 Ubisoft Quartz 的官网上育碧花了大篇幅介绍 NFT 的优势，描绘了去中心化、元宇宙等美好愿景，但大部分玩家都不看好这个 NFT 项目，甚至有的人还大骂育碧。Ubisoft Quartz 的项目宣传片目前在 YouTube 上处于隐藏状态，视频评论区里见不到几句好话。

（YouTube现在隐藏了踩和赞的具体数量，但从评论区还是能看到玩家的愤怒）

可能有读者不理解，在玩家不看好的情况下，为什么育碧仍要推进 NFT 项目，这好好的游戏厂商怎么就掉到 NFT 的圈子里了？

其实不只是育碧，在你不知道的地方，包括世嘉和卡普空在内的很多传统游戏厂商，已经在 NFT 领域摸索了一段时间。在他们看来，NFT 就如“魔戒”一般诱人。

业界在行动

我们从第一个站出来拿自家 3A 游戏开刀的育碧说起。育碧可以说是所有传统游戏厂商里对 NFT 表现得最积极的，并且已经在这个领域摸索了一段时间。只是 Ubisoft Quartz 来得太突然，让大家有种阿育急着进场捞钱的错觉。

育碧有个名为「Ubisoft Entrepreneurs Lab」的扶持计划，这个项目由育碧战略创新实验室领导，为一些有潜力改变娱乐业的公司提供帮助和支持。现在 Ubisoft Entrepreneurs Lab 已经来到了第 6 期，里面就有不少与 NFT 有关的扶持项目。

（Ubisoft Entrepreneurs Lab，直译过来就是「育碧企业家实验室」）

第 6 期项目里有 NFT 手游《GUILD OF GUARDIANS》，玩家可以把游戏中的道具转换成现实世界的货币；Horizon Blockchain Games 不仅推出了 NFT 游戏《Skyweaver》，自己也提供区块链相关的服务；NonFungible 是一个 NFT 数据平台，能为客户提供专门定制的市场报告。

（《GUILD OF GUARDIANS》的首页，一大卖点就是「玩家能将自己的热情转化为NFT」）

Ubisoft Entrepreneurs Lab 扶持的 NFT 项目里还有Aleph.im，育碧的 Ubisoft Quartz 也与它达成了合作。要说大家对 Aleph.im 印象更深的，应该是他们创始人在接受采访时表示：「未来大多数的大型游戏厂商都会给游戏引入 NFT 支持，这将使玩家拥有更愉快的游戏体验。」

育碧在 21 年 10 月还投资了热门 NFT 游戏开发公司 Animoca Brands。在接受 GamesBeat 采访时 Animoca Brands 董事长萧逸表示：「就像大家知道的那样，育碧已经观察区块链领域一段时间了。很多 NFT 项目都是通过育碧的扶持计划或加速计划取得成功。育碧真的很懂区块链，对 Animoca Brands 的投资就是一个信号。」

不难看出育碧对 NFT 市场确实抱有兴趣，并且也做了很多准备，只是这些准备都发生在玩家不容易接触的地方。

盯上 NFT 这块香饽饽的游戏厂商可不止育碧，还有很多我们熟悉的游戏厂商都推出过 NFT 项目，或是对 NFT 表现出极大兴趣。

比如世嘉，他们在今年 4 月就宣布投资 NFT 游戏开发公司 double jump .tokyo，并与其达成合作，销售利用区块链技术的 NFT 数字内容。同时世嘉在声明中还表示，将以这次投资为契机，给手里的各种 IP 作品开发 NFT 内容。

而且世嘉似乎很看好 NFT 市场的发展。在世嘉的财报里有一份关于在公司低谷期如何扩大收入的说明，「投资 NFT 等新兴产业」就是其中一环。

（财报里提到 NFT 的就这一个地方，但你能从内容中看出它的重要性）

对于很多只关注单机游戏的读者来说，SE 似乎和 NFT 没有什么关系，但别忘了 SE 在坊间有一个外号叫“手游大厂”。SE 在 21 年 6 月 9 日宣布为自家的《百万亚瑟王》手游系列推出一个 NFT 项目，名字叫《资产性百万亚瑟王》。

（《资产性百万亚瑟王》这名字就起得很有灵性）

《资产性百万亚瑟王》为用户提供的是限量的 NFT 数字贴纸，贴纸图案由四格漫画《弱酸性百万亚瑟王》的作者 ちょぼらうにょぽみ 老师绘制。除了能记载物品的购买和拥有历史等信息外，每张 NFT 数字贴纸的图案都不一样，确保拥有者拿到的是独一无二的 NFT 物品。

目前我们提到的 NFT 物品都是仅用于收藏或是与其他人进行交易，相比之下卡普空的 NFT 项目在这之外还具备一定的“可玩性”。

卡普空进场比较早，20 年年末就与区块链平台 WAX 达成合作，推出《街头霸王》主题的 NFT 数字交易卡牌。这些卡牌分为「基本卡」和「构建卡」（Build cards）两种，收藏家可以将两张「构建卡」合并来解锁新的卡。「构建卡」在用于合成后就会销毁，进一步增加了卡牌的稀有性，好处是收藏家有机会通过合并获得特殊的卡片。

在柏青哥等业务上赚得盆满钵满的 KONAMI 也看到了 NFT 这个全新市场，趁着《恶魔城》系列迎来 35 周年推出了「恶魔城35周年纪念NFT」。我知道有的朋友一看到《恶魔城》就走不动道了，嘴里念叨着什么：“虽然好像没什么用，但它是《恶魔城》啊。”但 KONAMI 表示这些 NFT 商品不一定会升值，如果你抱着「我喜欢《恶魔城》，正好搞个投资」的态度想要入手这个 NFT 商品，那我劝你三思。

有意思的是，这个商品属于「KONAMI 纪念NFT」系列，也许我们在未来还能在 NFT 领域见到 KONAMI 的身影。

EA 对待 NFT 倒是很谨慎，目前还没看到他们有推出 NFT 项目，不过倒是对外表示自己对 NFT 很感兴趣。在 21 年底的一次电话会议上，EA 的 CEO Andrew Wilson 表示“可收藏的数字内容”将在未来发挥重要作用，EA 应该在前沿基础上创造性地思考问题。但他也没有把话说死，在最后还补了一句：“现在弄清其运行模式还为时尚早。”

Take-Two 的 CEO Strauss Zelnick 曾公开表示他是 NFT 的信徒，他不明白大家都信赖实体收藏品，却对 NFT 持怀疑态度。他觉得 NFT 现在唯一的问题，就是很多人都以为 NFT 藏品的价值只会往上涨，而不会往下跌。这点倒是和 KONAMI 的声明一样，也许已经有不少投资人在 NFT 上栽了跟头。

传统游戏厂商布局的布局，表态的表态，但 NFT 并非无情的吐钱机器。在这条新赛道上，有人欢喜有人愁。

不是谁都能吃NFT这碗饭

这部分我们先来介绍一下目前传统游戏厂商在 NFT 市场上的赢家。对于 SE 和卡普空来说，NFT 项目可以用两个字来形容——真香！

不知道是《百万亚瑟王》系列迎来了新的上升期，还是 NFT 确实和手游相性很好，SE 对《资产性百万亚瑟王》的市场表现相当满意。

SE 在财报里表示，先前推出的两期《资产性百万亚瑟王》NFT 贴纸已经全部卖光，NFT 确实能与手里的 IP 相结合，计划未来将积极拓展相关市场。同时 SE 认为概念性的实验阶段已经结束，NFT 与自家 IP 的亲和力很高，未来将过渡到完全商业化阶段。

财报没有说明这个「完全商业化阶段」是怎么回事，是要继续深耕《资产性百万亚瑟王》，还是掏出更多 IP 制作 NFT 商品。年前我认为前者的可能性较大，毕竟他们那时正开心地卖着第 3 期《资产性百万亚瑟王》贴纸。没想到 SE 的社长在新年贺词里表示，NFT 的「Play To Earn」模式能够激发玩家的创作欲望，涌现更多优质的玩家自制内容。既然社长都站出来表态了，那说明 SE 对 NFT 的态度更倾向于后者。

（出了就能赚，那就继续出呗）

来到卡普空这边，他们的《街头霸王》NFT 卡牌也是销售火热。据外媒 DappRadar 报道，上架仅 24 小时卡普空就卖出了 6 万 5 千包 NFT 卡牌，总交易金额超 200 万美元，足见其欢迎程度。

在 NFT 交易市场上，街霸的卡牌也是热门商品。根据稀有度的不同有的卡牌售价不到一美元，但也有卡牌能突破 60 美元大关，转手就是一份全价游戏。

（街霸NFT卡牌的市场表现非常优秀）

说完最有赚头的两家公司，接着就来说说 NFT 项目表现不行的游戏厂商吧。

虽然世嘉在财报里把 NFT 当作救命良药，但他们的 NFT 项目表现真的很一般。

在宣布推出 NFT 商品时，世嘉就被索尼克粉丝狂喷。至于为什么是索尼克粉丝出来喷这个项目，这也是件趣事。我们都知道索尼克是世嘉的吉祥物，世嘉官推的头像也是索尼克，很容易让粉丝们误以为是世嘉要出索尼克 NFT 项目。

（你看这推文内容哪提到索尼克了，这波确实是冤）

索尼克粉丝们反对的理由也很简单，NFT 项目不够环保。有玩家在官方声明下表示「当年索尼克游戏里向玩家展现了过度发展，环境被完全破坏的城市，你现在是在开玩笑呢」「索尼克可是环保小达人，你拿索尼克来搞 NFT，我觉得不行」。

（图上说的是地球日宣传活动，世嘉选择索尼克作为代言人）

可能是受此影响，世嘉的 NFT 艺术品在市场上普遍价值不高。如果世嘉想靠 NFT 赚钱的话，可能还得多下点功夫，现在这情况就是既挨了骂，还捞不到多少钱，太惨了。

（索尼克的NFT物品）

让我感到意外的是，世嘉 CEO 在一场会议上表示，愿意在 NFT 领域进行各种实验，但如果 NFT 被视为简单的挣钱工具，就绝不会去推进这件事。虽然他们的 NFT 项目市场表现不算好，但公司高层能在人人都想进军 NFT 市场的时期说出这种话，我觉得挺酷的。

世嘉的情况和育碧比起来都不算什么，育碧这边才是惨到没地方说了。常关注 NFT 新闻的朋友肯定知道，育碧 NFT 项目宣传片被观众骂到隐藏，NFT 物品交易量低，自家员工都看不懂这个项目要干什么，出来表态要继续探索 NFT 项目又被大家取笑。不过大家可能不知道，阿育还被家乡的公会逮着狂喷。

法国公会 Solidaires Informatique 在育碧官宣 NFT 项目不久后，就发了一篇声明痛批育碧的决定，并称区块链技术是「一种有害、没有价值、毫无生态可言的技术」。该公会在声明中写道：

我们并非对 NFT 一无所知，相反我们是因为理解 NFT 才反对它。NFT 能带来的是金融诈骗，它让我们远离电子游戏的乐趣。区块链所谓的“创新”只是用更多的电力做同样的事情。

虽然观点比较尖锐，但这个公会确实指出了一个问题：从目前的表现来看，它和饰品商店并没有什么区别，这就是 Ubisoft Quartz 目前最尴尬的地方。

整理完目前传统游戏厂商 NFT 项目的口碑和市场表现，有的读者还是摸不着头脑，明知道玩家对 NFT 不感冒，为什么游戏厂商对 NFT 还那么感兴趣，是为了赶潮流吗？还是真如他们宣传项目时说的，NFT 能拓展游戏玩法？

可能原因并没有那么复杂，已经有游戏厂商正面回答了这个问题，他们推出 NFT 项目就是为了赚钱。

NFT的魔力

不久前开发商 GSC Game World 宣布将在自家游戏《潜行者2》里加入 NFT，结果在玩家间引起巨大争议，最后官方决定撤回往游戏中加入 NFT 的决定。事后官方非常诚实地表示，他们出 NFT 就是为了赚钱。

开发商在公告中称：「《潜行者2》是我们有史以来最庞大、最复杂的游戏。在开发过程中获得的所有资金（包括NFT的潜在收入），我们都将用于改进玩家期待已久的游戏，让它变得更好。」简单来说，他们希望从 NFT 获取资金，帮助完成游戏开发。

我们再回看《潜行者2》推出的 NFT 商品：独占的手套、纹身、皮肤、徽章、收集卡；游戏道具和建筑的冠名权；将一名现实玩家以 NPC 的形式做进游戏里，但制作组会淡化这名 NPC 的重要性，也不会参与到《潜行者2》的主要故事中。能看出开发商在尽量避免 NFT 对游戏内容的影响，他们卖 NFT 就是为了赚点奶粉钱继续做游戏，但依旧有很多人不买账。

游戏不该与 NFT 扯上关系，这是绝大多数玩家对 NFT 的态度。

可能有的读者奇怪，《潜行者2》的 NFT 我觉得可以接受啊，开发商拿到了钱继续开发游戏，玩家得到了质量更好的游戏，这不是双赢吗？没错，但我很理解玩家们在担心什么。

以《刺客信条 奥德赛》为例，育碧给游戏加入了等级加速包，只要少许钞票就能让主角升级得更快。如果玩家觉得升级慢，是游戏本就如此设计，还是开发人员故意让玩家觉得自己升级得慢，创造了等级加速包这一需求？没人说得清。

那些单独推出 NFT 收藏品的还好，但是直接往游戏本体加入 NFT 的，无论是边玩边赚（Play To Earn）模式，还是像 Ubisoft Quartz 这样与区块链市场结合，都是围绕一种新的经济系统设计的。在这个经济系统中，玩家被鼓励互相买卖数字物品，最重要的是尽可能多地赚钱，玩得开不开心反倒是次要的。

（边玩边赚游戏的代表《Axie Infinity》，有不少人靠它养家糊口）

这便是玩家们在担心的问题，游戏厂商可能会为了推广 NFT，改变游戏原本的运行规则，甚至像微交易系统一样有打破游戏平衡的可能性。所以大家才会如此排斥 NFT，想要把它扼杀在萌芽中。

不止是玩家，也有游戏开发者提出了类似的看法。游戏开发者 @maxnichols 连发 20 条推文分析 NFT 对游戏的影响，总结下来就是：

  • NFT 对环境和社会都有不良影响。

  • 去掉 NFT 内容的游戏，游戏表现要么差到不行，要么同样出色。

  • 往游戏中加入 NFT，会扭曲游戏的设计初衷。

  • 大多数玩家不喜欢边玩边赚模式。

这不是孤例，育碧内部社交中心 MANA 就有游戏开发者表示，他搞不懂这项新技术究竟解决了什么老问题，有的人则开始担心会被上级要求把 NFT 整合到自己的项目中。

尽管在玩家和部分游戏开发者看来，为游戏加入 NFT 是个愚蠢的决定，但如果我们从厂商的角度出发，单机游戏的开发时间变得越来越长，开发成本也逐年增高，哪怕投入巨大也不能保证游戏一定大卖。为了收回成本或是盈利，他们尝试过出氪金皮肤、出季票、把游戏做成服务型游戏，NFT 便是摆在游戏厂商面前的一个新选择，有再多反对的声音也值得一试，这便是 NFT 的魔力。

哪怕现在谁都不知道单机游戏与 NFT 的契合度究竟如何 ，但只要它有赚大钱的可能性，还会有游戏厂商如着了魔的咕噜，追着 NFT 这个“魔戒”不放。
