编者按：铭文市场热度不断攀升，除传统的比特币铭文市场外，其他公链的铭文也备受关注。近日，SOL 链在 Libreplex 平台上也出现铭文系统，而且铸造方式与目前现有的铭文铸造方式都不一样，KOL CG（X：@CG_BRC 20）发文梳理了 SOL 链铭文的铸造教程以及双锁教学，Odaily星球日报整理如下：

SOL 链铭文代币 $sols 的创始人推文称，必须要在今晚凌晨 1 点之前完成双锁否则将不会被录入索引范围内，很多小伙伴一脸懵不知道如何锁，CG 哥这就安排上双锁的教程以及 SOL 链的铭文实操铸造过程。

注意：当前 $sols 已经铸造结束，本教程的 sols 铸造流程 CG 哥只是用来演示，方便大家看，大家请勿浪费 GAS 去铸造。

前言

前两天 SOL 链 在 libreplex 平台上也出现铭文系统，而且该铭文的铸造方式与目前现有的铭文铸造方式都不一样，需要自己先部署 NFT，然后铭刻铭文，并在铭刻的 NFT 上可添加各类文件。

经国外一名老哥研究了下，把代币文本刻入到 NFT 铭文内，于是诞生了 $sols！当时引发全网 fomo 铸造，一度把 libreplex 平台网站冲崩。目前 $sols 以及全部铸造完毕。

由于该铭文存在可变性，目前还没有一个统一索引标准，目前场外价格维持在 200 U 附近。

昨晚凌晨创始人推文发布，在有效范围内的铭文编号，限制 24 小时内更改铭文参数并锁定，否则无法将添加索引范围内！所以小伙伴们赶紧去更改并锁起来吧（可直接跳过前面教程看最底部的《修改铭文参数并上锁》）。

《SOL 钱包下载安装与充值》

1、打开下载钱包 phantom 插件网页，并点击 添加至 Chrome

2、点击添加扩展程序

3、然后创建自己的钱包，或者导入钱包（我以创建新钱包为例）

4、设置好密码，并点击继续

5、把助记词备份下来，切记！并点击继续

6、点击开始

7、点击插件栏的钱包展开，并点存入

8、选择充值代币 Solana

9、复制这个地址，然后去交易所，把 sol 充值进去这个地址即可（这里就不演示了）

《SOL 链铭刻铭文》

一、准备代币图片与代币文本

1、点击该图片链接，鼠标右击图片，并点击图片另存为

pbs.twimg.com

2、点击保存

3、复制下面文本代码

{p:spl-20,op:mint,tick:sols,amt:1000}

4、电脑桌面，鼠标右击任意空白位置，选择创建 文本文档

5、把文本名称改为 sols 方便记，并把文本上面代币的文本粘贴进去

并把文本保存

二、NFT 部署

1、打开部署网页，并点击右上角的 Select Wallet

sol-tools.tonyboyle.io

2、选择 Phantom

3、点击第一行的选择文件，并打开刚保存的代币图片

4、下拉，把名称描述特征等，全部改为 sols ，并把版税改为 0 

5、继续下拉，勾选高级功能， IsMutable 栏 选择 False，并点击 Create NFT

6、钱包弹窗，会弹窗好几次，一直点击确认即可

7、显示打钩即可

三、铭文铭刻铸造

1、打开 libreplex 官网，并点击右上角链接下钱包 LibrePlex

2、上方导航栏选择 Inscriptions ，并找到刚部署的 NFT，点击后面的 Create Inscription

3、点击 Initialise

4、钱包弹窗，点击确认即可

5、点击 Custom，并点击 Choose

6、选择刚保存的代币文本，并点击打开

7、确保字节为 53 。并点击 Resize

8、钱包弹窗点击确认

9、点击 INSCRIBE NOW!

10、钱包弹窗，点击确认

11、铭刻完成后，点击放大镜

12、铭文信息核对无误后（按照我的截图中为准），多核对几次，出错就进不了有效索引范围内！并点击 Make immutable

核对内容如下：

  • Name：sols

  • Mutable：NO

  • Royalties： 0.00% 

  • 铭文 图片、文本

13、钱包弹窗点击确认

14、显示上锁即可

《修改铭文参数并上锁》

必须要在 2023 年 11 月 21 日凌晨 1 点之前完成双锁否则将不会被录入索引范围内

双锁的要求如下：

  • NFT 的税费必须是 0 ，且上锁

  • Solscription 必须文本正确，且上锁

1、打开 libreplex 官网，并找到需要修改的铭文，并点击后面的放大镜

2、复制 Mint ID （这个页面先别关）

3、打开 dandies 官网，并点击右上角链接钱包

4、点击导航栏的 UPDATE NFT，并粘贴上面复制的 Mint ID

5、把名称描述特征等，全部改为 sols ，并把版税改为 0 

（这里要注意，一定要等铭文图片加载出来后在操作，CG 哥有个就没等出来就操作，导致改铭文变成无图片显示，而且永久不能再更改了）

6、下拉，把 Mutable 关掉，并点击 UPDATE

7、钱包会弹窗几次，全部点击确认

8、返回刚复制 MINT DI 的页面，铭文信息核对无误后（按照我的截图中为准），多核对几次，出错就进不了有效索引范围内！并点击 Make immutable

核对内容如下：

  • Name：sols

  • Mutable：NO

  • Royalties： 0.00% 

  • 铭文 图片、文本

9、钱包弹窗，点击确认

10、这里显示上锁即可，后续就等待创始人加入索引了
