Web3隐私保护协议 Manta Network 先行网 Calamari Network 宣布与 Acala 先行网 Karura 成功完成跨链资产转账， HRMP 通道的开通以及跨共识通讯的部署将使两条平行链间实现互操作性以及资产隐私化。而资产隐私化将对 Calamari 和 Karura 未来的进一步整合以及波卡生态的发展都有着重要的意义。

Manta Network 核心贡献者 Kenny Li 表示，“与 Acala 的合作将会把更多的隐私带入 Web3世界中，这对于 Manta 和更广泛的生态来说是一个里程碑。互操作性一直以来都是我们关注的重点，此次合作后，Acala 网络中的稳定币 $aUSD 以及 Karura 的原生代币 $KAR 将可以实现完全隐私化的转账。”

Karura 网络以及网络上的各种项目也将支持 Calamari 的原生资产$KMA，为$KMA赋予更多的应用场景。Calamari 与 Karura 的 XCM 整合将允许两条平行链资产的跨链转移。$KMA 持有者将可以通过 Karura 生态系统参与各种 DeFi 产品，进行 staking、借贷及其它产品的使用。Karura 原生资产将可以转账到 Calamari 网络上进行隐私化，并实现隐私资产的端对端转账。

Acala 首席增长官 Dan Reecer 表示，“很欣慰可以看到 aUSD 已经在波卡生态取得了不少进展。希望双方可以继续在 DeFi 方面合作，为社区提供重要的隐私服务。”

今年3月，Acala 联合 Manta 等波卡平行链团队共同推出2.5亿美元 aUSD 生态基金以支持和孵化在 Polkadot 与 Kusama 生态平行链中利用 aUSD 来构建应用程序的初创团队。Kenny Li 表示，“非常开心可以支持 aUSD 生态基金，期待将隐私带入更多的项目和应用场景中。凭借波卡的原生互操作性，生态中的每一个新想法都可以为所有的平行链带来更多的价值。”

随着 Web3 向全球经济渗透，生态基金中的平行链团队将会一起助力 aUSD 成为波卡生态的支柱，并在未来数十年与数百个应用一起持续成长繁荣。

关于 Acala

Acala 是波卡 DeFi 中心，也是一条多链的 DeFi 中转枢纽。Acala 为波卡生态带来一系列开箱即用的 DeFi 基础设施产品，比如超额抵押的稳定币 aUSD，基于 AMM 的 DEX，为 DOT Staking 释放流动性的 LDOT 以及为平行链插槽竞拍释放流动性的 lcDOT。另外，Acala 作为一个兼容以太坊智能合约的高性能 Layer1 的 DeFi 跨链网络，不仅能100%兼容以太坊，使用诸如 Metamask、truffle 等以太坊的原生工具，还能100%保留 Substrate 框架可定制化的优势，并直接与波卡生态原生资产进行交互，无需搭建跨链桥等额外操作。

关于 Manta Network

Manta Network 致力于通过隐私保护构建一个更好的 Web3 世界。Manta 的产品设计从第一性原理出发，通过 zkSNARK 等领先的密码学架构为区块链用户提供端对端的隐私保护。在保障隐私的同时，Manta 兼具互操作性、便捷性、高性能以及可审计性，允许用户进行任意平行链资产间的隐私转账和交易。Manta 的愿景是为整个区块链世界提供更便捷的隐私保护服务。

Manta 的创始团队由多位加密货币资深人士，教授和学者组成，他们的经验包括哈佛大学，麻省理工学院和 Algorand。Manta 的投资机构包括 Polychain、ParaFi、Binance Labs、Multicoin、CoinFund、 Alameda、DeFiance 以及 Hypersphere 等。Manta 也是波卡官方 Web3 基金会资助获得者，Substrate Builder Program 成员，伯克利大学区块链加速器成员。
