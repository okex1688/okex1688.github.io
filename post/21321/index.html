Rococo

1. 目前 Rococo 上有四个平行链测试网 Statemint、Tick、Trick 和 Track。

2. Statemint 测试链正在 Rococo 上运行。请查看 Matrix 上的 Rococo 频道，了解更多细节和讨论，也可以在这里参与测试：https://polkadot.js.org/apps/?rpc=wss%3A%2F%2Fstatemint-rococo-rpc.parity.io#/explorer。中文版查看《Rococo 最新进展：已上线公益平行链 Statemint 测试网！》

3. 为了准备 Kusama 的平行链，Rococo 目前正在扩展到一个类似于 Kusama 的网络拓扑和等效参数。

4. 一个新的测试网，Wococo（Rococo 的邪恶双胞胎），正在计划试用 BEEFY 和 MMRs。https://github.com/paritytech/polkadot/pull/2918

  • BEEFY 通过在 GRANDPA 之上添加一个辅助协议，允许更有效地桥接外部的非 Substrate 的链。https://github.com/paritytech/grandpa-bridge-gadget/blob/master/docs/walkthrough.md
  • MMR (Merkle Mountain Ranges)是对 Merkle 树的修改，使它们只添加，并允许更快的更新。https://docs.rs/merklemountainrange/0.0.1/merklemountainrange/

Acala 联合创始人陈锡亮解释：Wococo 是一个新的测试网，与 Rococo 主要的区别是支持以太智能合约验证 grandpa，为以后的 snowfork 以太桥做准备！

请注意

最近很多小伙伴发现使用 Parity 节点连接 polkadot.js.org 非常慢，或者有的连接不上。你可以通过切换节点来连接到 Polkadot 的生态。

查看这个视频学习如何切换：https://www.youtube.com/watch?v=btqfURuOUmI

文字版可以查看这里：https://support.polkadot.network/support/solutions/articles/65000169778-how-to-switch-network-nodes
