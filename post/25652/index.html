目前的NFT在圈内毫无争议是一个炙手可热的词儿，除非你是一个刚刚入圈的萌新，否则对于NFT概念的定义和解释，相信你早已烂熟于心。

经历了年初充满泡沫化的巅峰到后来的沉寂，最近以AXIE为代表的NFT游戏再度火热，似乎又把人们的视线拉回了这里。

这次，NFT带来了哪些新叙事？

区块链的演进

在说NFT的前世今生之前，我们可以先来想想区块链的演化史：

1、电子现金： 最早的鼻祖比特币白皮书，是要做电子现金的。

2、 各路电子现金： 在比特币诞生后几年，以莱特币为首，出现了一大批比特币的仿盘，当然现在绝大多数已然销声匿迹。

3、 电子黄金： 慢慢的，比特币电子现金这个词儿已经没人在提，区块链演化着演化着，大家就把比特币“电子黄金”这个事儿给实锤了，电子现金只有LTC，BCH等在“负隅抵抗” ，然而我们都心知肚明，这活儿早就被USDT给抢了。

4、 去中心化一切： 2017年ETH的爆发，让无数人把区块链，或者说带有智能合约的区块链当成了锤子，到处找钉子。

一时间，去中心化微博、去中心化打车、去中心化社交、去中心化XX，防伪、溯源、产权保护……因此诞生了大量的爱系欧，毫无疑问现在看，这些项目和所谓的应用场景也都基本归零了。

5、 DeFi： 虽说18年便开始萌芽，然而直到2020年，人们才真的意识到 - 区块链，原来是拿来革金融市场命的，Defi才是王道。

6、 游戏 NFT： 2021年，区块链在这两个方向上进行着积极主动的探索，并与DeFi进行着越来越深入的融合。

所以你看在比特币或是区块链诞生之初，我们对他的设想只是电子现金，经历了各种脑洞大开的场景适配和无数次的失败之后，目前只确定了两大应用 ：BTC的电子黄金与以太坊的DeFi。

NFT、游戏、元宇宙、ICP带来的各类中心化DAPP等都还在努力探索和试错中，所以其实大概率你今天看到的NFT应用和发展方向，和未来的NFT真正使用案例会有不小的差距。

然而这并不妨碍我们去梳理NFT的脉络，审视NFT的价值和探索NFT的未来。

NFT的发展

提到NFT，许多人的第一印象是加密猫。然而熟不知加密猫只是ERC721 NFT标准的第一个作品，也是把NFT带火的招牌，却不是第一个NFT。

CripytoPunk比加密猫更早，然而Punk也不是第一个NFT，严格意义来说，区块链上第一个NFT应该是2012年比特币上面弄的彩色币。

这也可以大体看出一个简单的脉络：彩色币 - NFT概念形态 → Crpytopunk - NFT第一代收藏品，NFT界的BTC →加密猫 - 游戏NFT→当前百花齐放的NFT形态。

1、加密艺术+NFT 代表：Beeple佳士得7000万美元成交的画

加密艺术与NFT的结合，有那么几个有意思的点值得关注：

•   原生的加密创作与传统画作NFT化，哪个才是正确的道路？
•   传统画作NFT化之后，原作是否应该“烧掉”？
•   B20这种Beeple画作碎片化Token是一个探索方向。
•   加密艺术是否需要DID来配合，证明作者是作者本人？

2、IP+NFT 代表：NBA Topshot

传统IP与NFT的结合可以说是当前最为出圈的一种方式，玩过小浣熊干脆面水浒卡的你，一定知道这个世界上有多少人对于这种收藏有着近乎疯狂的热爱。

这也是为什么NBA Topshot可以长久霸占NFT销量榜首的秘密，因为在这种顶级IP的加持下，买卡片的人可不仅仅是圈内投资人，而是有着实实在在圈外一批NBA的粉丝们。

而随着欧洲各大豪门俱乐部纷纷“发币”，传统IP与NFT的结合已然成为一个不可阻挡的趋势，大家熟悉的漫威，迪士尼等大IP的拥有者也必然不会放过使用这个新技术的机会。

对于收藏玩家而言，NFT无疑更加安全，摩擦更小，市场更大，选择更多。

可能唯独缺少的，便是一张“真正”的NBA球星卡，或是小浣熊水浒卡拿在手里时的那种质感，正如电子书永远无法提供纸质书相同质感的阅读体验。

然而也正如纸质书逐步被电子书淘汰，这个世界终究是在向着“虚拟化”与“元宇宙”的方向不断向前的。

3、DeFi+NFT 代表： Whale、Meme、yinsure.finance,NFTX和Uniswap V3

NFT与DeFi的结合在过去一年里探索出大量的使用场景：

Whale - 抵押NFT收藏品发社交Token
Meme - Defi流动性挖矿奖励NFT卡片
yinsure.finance - 保单NFT化+挖矿
NFTX - 创造NFT流动性的指数基金
Uniswap V3 - 可定制化的LP NFT化实现更加负责的LP策略

NFT与DeFi的结合是区块链2020与2021年两大风口的强强联合，未来1-2年，我们一定会看到无数的NFT+DeFi结合用例。

或许DeFi+NFT会是未来区块链放飞想象力最大的一个方向，将来的元宇宙、全球结算层等一系列美好未来都一定离不开Defi+NFT的各种组合。

4、更深层的游戏+NFT 代表： AXIE ，Dark Forest（黑暗森林）

如果说上一代“链游”是以加密猫和Decentraland为代表的话，那么现在最火的“链游”无疑是AXIE和Dark Forest。

Axie 简直可以算是当前区块链游戏中的《魔兽世界》，不少人甚至以此为生，月入过万，越来越多的专业工作室也开始加入其中，让我恍惚看到了当年WOW专业打金团的盛况。

更可怕的是营收数据，过去7天AXIE收入达1252万美元 ，这个数据有多恐怖你知道么？几个DeFi扛把子项目过去7天的收入是这样的：PancakeSwap 190万美元，Synthetix 130万美元，MakerDAO 120万美元， Curve 80万美元……

目前整个DeFi界，只有Uniswap能压得住AXIE了……

Dark Forest则是目前一帮ETH硬核粉最爱的游戏，没有之一，用一个朋友的话说：为啥人在元宇宙中一定要做人呢？我就不能变成一类星体，或者仅仅是一股控制这些星体的力量，为什么要盖房子呢，去创造一个小宇宙不是更酷吗？

当然，就目前来看，最具有NFT游戏代表性，也是最接近“元宇宙”概念的链游依旧是老牌的Dencentraland，毕竟那一块块不同坐标的土地NFT，对于房价敏感的我们来说实在是太熟悉了，现实里买不起地，游戏里买买还不行么？！

NFT的未来

正如之前所说，我们现在想象的NFT未来很有可能并不存在，然而这不妨碍我们去“憧憬”一下。

1、NFT流动性

相信每个NFT玩家都会有“烂在手里”的一堆NFT卖不出去，比如本人就有两只初代加密猫挂在Opensea上无人问津。

没错，无论是加密艺术品还是收藏品，或是游戏道具等，NFT的流动性与主流Token种类完全不在一个量级，买一卖一经常几倍差价，有价无市或是有市无价都是家常便饭。

在促进NFT流动性这方面，诸多项目做出了一系列尝试。 有上面提到过的B20与NFTX利用NFT碎片化提供流动性。

前段时间的Unicly，则是把NFT碎片化流动性挖矿，AMM Dex 和 DAO彻底结合起来，给当前主流的NFT资产都提供了流动性支持，下一季度甚至会有借代和Layer2的探索。

而另一个项目Topbidder，则是使用“自动竞价机制，任何竞拍者只要出价不低于上位竞标者 10%，就可以获得该资产拥有权”的模式，通过使用激进市场理论与哈伯格税，把NFT变成了一件“非永久占有型资产形态”（也称维克里公物），而带来的结果则是：更好的价格发现机制与更好的流动性。

对于NFT流动性的拓展和流动性的使用，一定是NFT未来发展的一个重要方向之一。

2、链下真实资产抵押借代

这可以说是DeFi的终极梦想的基石，要做全球结算层，首先得把非区块链原生资产弄上链。

好在我们已经开了一个先例，几个月前，MakerDAO发行了世界上首笔基于DeFi的现实资产贷款。 使用了基于Centrifuge协议的借代平台Tinlake。New Silver作为资产发起方，使用 MakerDAO作为信贷工具获得了首笔贷款。

本质上来讲，Tinlake就是一组可以将现实资产（如汽车贷款、商业发票或音乐流版税）转换为带有必要法律文件的NFT的智能合约。

俗话说，完事开头难，我们已经开了一个好头。相信在NFT与DeFi技术成熟之后，这种链下资产通过NFT化上链抵押借代，打开流动性的方式，才是DeFi与NFT，或者说区块链真实改造世界的大工程。

也只有这一步实现了，区块链才能真的自信的说自己是价值互联网的基石。

3、任何独一无二的东西，理论上都可以用NFT来表示

当然别误会，并不是所有独一无二的东西都适合用NFT，一件事物也并不会因为NFT化而改变其内在价值。

然而，你可能可以找到许多贴切的、符合NFT使用场景的东西。 比如域名，这个世界上肯定不会有两个相同的域名，以太坊的ENS域名就是一个非常好的NFT场景。

当然，与Cryptopunk一样，ENS也是诞生于ERC721协议之前，所以2019年5月ENS迁移之后，每个域名才变成了符合ERC721协议标准的ENS NFT。

以后DID系统一定会有，每个人的链上身份证，包含你个人信息的身份Token是不是也一定会是NFT？

依次顺着想，等到类似《头号玩家》那种级别的元宇宙实现之后，这个世界上千千万万的东西都会被NFT化，有时候仅需要发挥你的想象力。

比如之前有Twitter创始人把第一条Twitter给NFT化了，卖了250万美元最近互联网之父把万维网的源代码给NFT化了，卖了540万美元。

一幅画、一首歌、一张照片、一段视频、一段文字、一件装备、一次行为、你的身份、你的版权和你的数据……在未来，也许都是NFT。
