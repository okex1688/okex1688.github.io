**一文告诉你头像类NFT凉多快。 **

今年以来，NFT 这把火越烧越旺。

在过去几周时间内，NFT市场热度已经攀升到了前所未有的高度，包括 Opensea 在内的 NFT 平台交易量不断创下新高；各大公链、侧链/Layer2 也相继开始入局 NFT 赛道，各类 NFT 新项目层出不穷。

围绕 NFT 价值的讨论，正反双方都在不断发声：有观点认为「共识赋予 NFT 价值」，CryptoPunks 就是 NFT 领域的比特币；也有人认为 NFT 只是短时炒作，泡沫终将破灭……

今天这篇文章，我们试着用数据说话，告诉你热门图片类 NFT 项目们的生命周期到底有多久，并在此基础上，探究那些长盛不衰的 NFT 究竟有哪些成功秘诀。

先亮核心结论：

  • 被统计项目在创下巅峰记录后的两天到三天内交易量全部腰斩；近一周平均交易量、平均买家数，大约相当于巅峰时刻的 20%。

  • 被统计项目从巅峰交易量滑落至均值水平，平均用时 5.3 天，中位数是 4 天；15 个项目从巅峰买家数滑落至均值水平，平均用时 6.5 天，中位数是 5.5 天。

  • 长盛不衰的项目取胜之道有三种：一是项目自身资历较老；二是差异审美以及名人加持；三是不断升级玩法。

本文的研究方法及选取样本的标准

市面上常见的 NFT，按其功能可大致分为展示与收藏类（如 NFT 艺术品或自动算法生成头像等）、游戏道具类（MANA、ZEST、REVV等，加密猫勉强也算）、金融产品类（如 Uniswap V3 的 LP NFT）等；其中的展示与收藏类，按呈现形式又大致分为图像类（CryptoPunks）、视频类（NBA Top Shot）以及音频类（腾讯幻核的“有声《十三邀》数字艺术收藏品NFT”）。

在过去几周，图像类 NFT 最为火热。加密圈不少人士都将自己的社交头像改成企鹅、狗狗、木乃伊等各类图片，这种 NFT MEME 社交文化也被戏称为「JPEG文化」。

本文章，我们将研究对象聚焦在近期最火的图像类（头像类） NFT。此外需说明下，之所以未将涉及游戏道具属性的 NFT 囊括进来，主要原因有二：

  • 目前游戏类 NFT 代表性项目屈指可数，样本可选范围小，为行业提供的参考性不高；

  • 图像类 NFT 使用场景单一，生产成本较低，交易情况更能反映市场情绪，容易暴涨暴跌；而游戏类 NFT（Axie Infinity、Farmers World、Crypto Raiders Characters、The Wonderquest、Galaxy Fight Club 等）有一定的使用场景，项目本身遵循游戏产业规律稳步发展迭代，短期币价受炒作影响相对较小。

本文所用数据，主要来自 Cryptoslam、Opensea；最终选出的 15 个项目为过去一周交易量下跌接近或超过 50% 、且最高成交量高于 1 万美元的图像类 NFT 项目。这些项目的成交量、用户数、买家数以及地板价各不相同，跨度较大，使得样本更具普遍性。

关于项目热度降低，我们以「换手指标」作为主要标准，包含项目在过去一周平均交易量与巅峰交易量的比较，交易量从巅峰回落到均值需要多久；项目在过去一周平均买家与巅峰买家数的比较，买家数从巅峰回落到均值需要多久。

需要注意的是，本文并没有将「地板价」作为项目热度的主要参考标准。原因之一是不同存续时长、发行总量的项目间的地板价及其变化不具备可比性，且跨度较大；原因之二是地板价数据具有一定的单边性，即项目热度上升，地板价往往上涨，但当项目热度下降甚至无人问津时，藏家可能出于惜售（不割肉）的心态，拒绝抛售，使地板价维持在相对高位。

除了研究项目热度下降速度，我们也统计了历史上排名前十五的 NFT 头像，试图剖析这些项目能够维持热度的内因，为 NFT 发行方和运营者提供参考。

NFT的「热」与「冷」

无论你是一名 NFT 头像爱好者、NFT 项目开发者，还是纯粹的投机客，下面这些数据或许会对你有一定启发，帮助你更好地运营项目，或者及时出手获利。

（1）热门NFT头像转冷

数据显示，15 个项目在 8 月中旬左右达到交易高巅峰（与比特币的走势和 NFT 市场整体热度密切相关）。虽然此前的 6、7 月份，NFT 在加密市场遇冷的情况下依然火热，但比特币的复苏无疑再次助燃了 NFT 交易热情。

然而，从 8 月下旬开始，上述 15 个项目的交易量开始走低，在创下巅峰记录后的两天到三天内交易量全部腰斩；近一周平均交易量，大约相当于交易量高峰时刻的 20%。此外，近一周平均买家数，也相当于巅峰买家数量的 20%-30%，有的项目一天只有几个人购买。绝大多数项目目前一天买家数量不到 50 人，NFT 头像最终成了少数人的狂欢。

以 My Fking Pickle 为例，在  8 月 14 日创下巅峰交易量 10.54 万美元成交量，但近一周平均成交量只有 2000 美元，近乎归零；从用户量来看，My Fking Pickle 总用户为 3000 人，巅峰买家为 81 人，近一周平均买家只有 9 人，着实惨淡。

当然，并不是所有 NFT 项目交易都遇冷。Incognito、Pudgy Penguins 这种用户基本广泛、共识更强的项目目前每天依然有 150 人左右的买家，近一周平均交易量也维持在近百万美元，只是相比其巅峰时刻略显暗淡。

将这个 15 个项目放在一起分析，得到结论如下：

  • 绝大多数项目从巅峰时刻的成交量、买家数，下跌至均值水平，基本都在一周以内；Reb3l Bots、Purrnelopes Country Club、CryptoZunks 这三个项目则在两周左右才滑落至均值水平，表现较为坚挺。

  • 15 个项目从巅峰交易量滑落至均值水平，平均用时 5.3 天，中位数是 4 天；15 个项目从巅峰买家数滑落至均值水平，平均用时 6.5 天，中位数是 5.5 天。

总结上述项目热度下降的原因，基本可以概括为：创新不足+疏于运营。

比如 Yat、Wrapped Strikers，由于缺乏有效的运营手段，用户增长停滞，目前累计用户不足 700 人，缺乏共识；再比如 Slumdoge Billionaires 这个项目，其实就是把狗头做成各种样式，这种蹭热度的玩法在 MEME 币盛行时已经收割了一波用户，注定无法长久。

Slumdoge Billionaires头像

在翻看这些项目时，给我最大的感觉就是「简陋」。很多 NFT 项目官网只是简单介绍项目，没有设计理念，也无未来规划，多是为了 NFT 而 NFT。此外，一些掺杂名人 IP 类的 NFT 之所以转冷，根本原因还是合法性存疑，用户无法确定自己买的究竟是官方授权还是盗版，再加上这类 NFT 普遍价格较高，也导致“路人党”难以对其产生兴趣。

（2）长期热门的项目靠什么维持热度？

在加密市场中，失败的项目大抵是相似的，成功的项目却各有各的法子。

下面我们将介绍 NFT 头像排名前十五的项目，试着揭示这些长盛不衰的 NFT 究竟有哪些成功秘诀。

我们统计的 15 个项目分别是：CryptoPunks、Art Blocks、Bored Ape Yacht Club、0N1 Force、Parallel Alpha、Pudgy Penguins、Hashmasks、Curio Cards、Bored Ape Kennel Club、Cool Cats、Veefriends、PUNKS Comic、World Of Women、CyberKongz VX、Mooncats。

数据显示，上述 15 个项目「累计成交量」范围从 3千万美元到 9.1 亿美元不等，其平均数为 1.6 亿美元，中位数是 5865 万美元；「累计买家」平均数为 5030 人，中位数为 4601 人。而且，上述 15 个项目与前文项目存在很大的不同在于，其累计用户量基本都达到了 4000 人左右，有了足够强大的共识——「所有者」平均数为 4329 人，中位数为 4437 人。 这些数据，对于有志于冲击头部 NFT 项目开发者而言，可以作为一个有效的参考指标。

关于这些项目为什么可以取得成功，我们也归纳了几个要点：

一是项目自身资历较老 。CryptoPunks、Curio Cards 都是最早一批 NFT 化石级项目，诞生于 2017 年，为 ERC-721 标准做过贡献。在 NFT 进行「文艺复兴」时，老一批的项目成为焦点，自带流量，圈外资本也更青睐（Visa 购买 CryptoPunks），因而能够保持热度。

二是差异审美 以及名人加持 。以 World of Women 为例，其是第一批面向女性的收藏品项目之一，背后为女艺术家中的超级明星 Yam，这为项目带来了充分的曝光。

三是不断升级玩法 。单一静态的 NFT 头像不足以满足用户需求，不少项目开始别出心裁地创造新的玩法，比如空投和质押挖矿（金融向玩法）、NFT 头像生育（游戏向玩法）……

例如，Cryptopunks 开发团队 Larva Labs 就推出了 3D 虚拟世界游戏 Meebits，Cryptopunks 持有者将可免费获得 Meebits，弥补了自身在游戏领域的空白。近期新品 0N1 Force 通过社区运营跻身 OpenSea 交易榜前三——0N1 Force 会对表现良好的社区成员，进行为期四天每天一轮的预售，并且需要用户积极在 Discord 以及推特互动，从而维持热度。

总结

对于 NFT 头像来说，从盛极一时到无人问津，可能只需要一周时间，生命周期太短。

一个重要原因在于，当前绝大多数 NFT 头像主要用于基于喜好的收藏、社交展示和坐等升值的投机心理。如果能扩展 NFT 的功能性，不失为一种“延寿”的有效手段。具体来说，NFT 可以与游戏结合，提升可玩性；也可以结合抵押借贷，成为 DeFi 的基础资产，等等。

再者，NFT 头像的所有权“合法性”、存储方式以及与 JPEG 的差异，也在时刻动摇整个领域的合理性和信念根基。

最后一个重要原因在于，NFT 缺乏成熟的定价机制，价格发现与价值流转不足。一旦共识会快速流失，NFT 价格可能一路雪崩，戳破泡沫。因此，设计更成熟合理的定价方式也是让 NFT 走向更大圈层的一种有效手段。

Odaily星球日报曾与多位 NFT 赛道创业者进行交流，MixMarvel 创始人 Jade Zhang 的观点或许可以破解当前难题，她提出了三重价值理论：

  • 第一重是基础定价。每一个 NFT 是由多少个块（MixMarvel 的设定中“链上乐高”的基础单位）组成的，决定了其有多少价值筑底。因为每个块都是质押代币换来的，而这些代币在市场中存在公允价（市场博弈价），也为每个块赋予了价格。

  • 第二重是功用性价值。玩家创造的 NFT 作品（成品或者半成品），在版权中心被多少人引用，别人愿意付多少钱，也决定了该作品本身的功用性价值。

  • 第三重是世界观溢价。比如知名艺术家创作的作品，溢价高；普通人创作的作品如果很有意思，被广泛认可，也可以具有高溢价。

目前绝大部分作品只有第三重价值，没有前两部分，并且第三重价值更容易被炒作。在热钱涌入的当下，绝大多数 NFT 的「世界观溢价」被严重高估，NFT 价格不断抬升。一旦市场转熊，经常出现有价无市的情况，好的 NFT 作品或许可以等到下一个牛市，而二流 NFT 作品最终会被市场淘汰，永无出头之日。
