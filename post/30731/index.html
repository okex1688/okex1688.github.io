金色财经报道，11月11日消息，波卡首批平行链插槽拍卖已正式开始，目前参与的项目共有 8 个 Crowdloan，分别为 Acala、Moonbeam、Parallel Finance、Astar、Manta Network、Clover Finance、Litentry、SubDAO，目前Acala 的支持DOT最多，已锁定超过1200万枚DOT 。

第一批参与竞拍的平行链策略总结如下：

Acala Network

Acala Network拿出了ACA总量的17%作为竞拍奖励，即1.7亿枚ACA，以求获得96周的平行链插槽租赁期。100%的ACA会在网络上线支持转账后进行分发，其中20%的ACA将在上线后支持转账，剩余的ACA在96周内以区块为单位线性释放。同时还有额外奖励和限量版的NFT吸引着社区用户的积极参与。

同时通过IcDOT进行Crowdloan的用户，100%的IcDOT将与ACA同时在上线后发放，100%的IcDOT将是流动的并可以在Acala DeFi协议内使用。让用户在享有Crowdloan激励的同时，最大程度上获取流动性。在96周平行链租赁期结束之后，IcDOT可以1：1赎回DOT。

Moonbeam

Moonbeam将初始供应量的10%作为总的竞拍奖励，即1亿枚Glimmer。30%在平行链上线时解锁，70%在96周的租赁期内以区块为单位线性分配。

Astar Network

Astar将总供应量的20%用于Crowdloan，15%作为基础奖励，5%作为额外奖励池。Astar会在启动转账功能后分发10% ，剩下90%会在22个月内线性释放。除此之外，10ASTR/DOT的推荐人奖励、20%的早鸟奖励和10%的Shiden支持者奖励等等额外奖励也非常的诱人。

Clover Finance

使用代币 CLV 总供应量的 20% （2 亿枚 CLV）建立代币奖励池奖励众贷支持者，当拍卖成功、项目平行链启动时，立即发放 28% 的众贷奖励，其余的 72% 将在接下来的 23 个月里按月解锁。若 Clover 在第一批拍卖中未成功竞拍，将用额外的 CLV 补偿众贷活动中的贡献者。

Parallel Finance

Parallel将总量（100亿PARA）的10-15%用于Crowdloan贡献（15亿枚PARA通证），Parallel’s 平行链租期：96 weeks，PARA解锁周期：24个月线性释放，Parallel提供5种奖励方式，通过Auction Loan产品贡献Parallel Crowdloan，用户可以获取所有奖励种类。基础奖励：1 DOT = 33 PARA，推荐奖励：10%推荐人获得5%；被推荐人获得5%，社区早鸟奖励：40%，NFT奖励：Parallel将会为参与Parallel Crowdloan的用户分配1000个NFT作为激励。这些NFT在未来将可以在Parallel的平行链进行使用，cDOT作为流动性资产：通过Auction Loan贡献任何参与Crowdloan项目，用户都会1DOT可以得到1cDOT作为在2年DOT质押过程中的流动性资产。在Parallel的平行链上线后，可以在Parallel的借贷市场和AMM市场中使用。

Menta Network

Manta Network最新Crowdloan策略：不再采用1DOT：5MANTA固定奖励比例，而是将15.6% MANTA奖励全部按贡献比例分配给首轮波卡平行链Crowdloan参与者，其中包括12%基础奖励、3%流动性挖矿奖励和0.6%的Binance贡献奖。

计算公式为：基础奖励=（您的 DOT 贡献 / 对 Manta 的 DOT 总贡献总额） * 120,000,000 MANTA；流动性激励奖励= （您的 DOT 贡献 / 对 Manta 的 DOT 总贡献总额） * 30,000,000 MANTA。其中，流动性激励奖励在去中心化平台参与均可获得。

SubDAO

SubDAO提供多种奖励：

GOV竞拍总奖励：GOV首次发行总量的30%，即3亿GOV。

基础奖励：基础奖励为20% ，即2亿GOV。每支持质押1枚DOT获初始奖励20GOV，预期筹集10,000,000 DOT投票。除了2亿的基础奖励，还有1亿GOV作为其他奖励。

早鸟奖励5%：SubDAO对前1000名质押用户，将会奖励额外5%的早鸟加成福利。）

邀请奖励5%：SubDAO官方Crowdloan页面可获取邀请链接与专属推荐ID，使用推荐码成功邀请用户，推荐人与被推荐人均将获得被推荐人的 5% 邀请奖励。）

“Strong Supporter” 最高10%：投票贡献100 DOT或以上的用户，将获额外的加成奖励

“冲刺计划” 最高6%：当支持竞拍锁定的DOT总数达到某个数量级，所有竞拍支持者都将获得奖励加成

在Starfish Network成功拍得插槽且作为平行链顺利运行后，GOV奖励将立刻释放34%；剩余 66% 的GOV奖励将分为 96 周线性释放。

目前，确定支持SubDAO竞拍的渠道包括Polkadot-JS、Bifrost、Zenlink、Math钱包等去中心化平台，以及Huobi、Gate、MEXC等中心化交易平台。

Litentry

Litentry会拿出总供应量的20%来激励那些帮助Litentry在拍卖中获得优势的支持者。所有拍卖的支持者可以瓜分总共两千万个LIT的奖励池。每锁定1DOT，参与者将获得至少2.5LIT的奖励。最终奖励将以每位参与者对Crowdloan池贡献的百分比计算。

除此之外，用户还可以获得最高20%的额外奖励，11月11日前，在Binance参与可以瓜分额外50万LIT，同时用户还可以通过Parallel的Auciton loan来解锁DOT流动性并获得额外的PARA奖励。
