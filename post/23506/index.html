6 月 2 日，波卡测试网 Westend 已经将 Shell 链升级为资产平行链 Westmint（即 Westend 版的 Statemint），目前运行稳定！

根据此前 Gavin 和波卡团队透露的计划，下一步将会在 Kusama 上把 Shell 升级到 Statemine。在 Statemine 运行稳定后，Kusama 插槽拍卖就将开始。让我们拭目以待吧！

Statemine 的上线，除了预示着拍卖即将来临之外，Statemine 作为资产平行链，对于波卡生态和用户来说也有很重要的作用。

Statemine 将是 Kusama 上的第一条平行链，也是一条无需竞拍插槽的公共利益平行链，专门用于在 Kusama 上发行资产。

下面的内容来自波卡开发方 Parity 对于 Statemint 的描述文件，可以帮助你进一步了解它们（下文提到的 Statemint 相关内容，同样适用于 Kusama 的 Statemine 链）。

什么是 Statemint/Statemine？

Statemint 是支持 Polkadot 和 Kusama 网络中通用资产的区块链。

Statemint 将允许用户：

  • 使用 DOT/KSM 存款，部署由承诺背书的资产。

  • 设置 AssetId 的管理员角色，来铸造、冻结、解冻和销毁代币。

  • 如果中继链同意，可以将资产注册为 “自给自足”。即只要某帐户保持最低代币余额，就可以在没有 DOT/KSM 的情况下存在。

  • 使用资产余额支付费用。

  • 转让（并批准转让）资产

    Statemint 必须与其连接的中继链保持完全一致。因此，它将接受中继链的治理起源。

关于 Statemint 的常见问题

由于 Statemint 很可能成为首批公共利益平行链（Common-good parachain）之一，我们收到了很多关于它将如何运作，及其在生态系统中的位置的问题。

Statemint 会上线到 Kusama 和 Polkadot 上吗

是的，Statemint 是 Polkadot 平行链，而 Statemine 是 Kusama 平行链。所有其他答案同样适用于 Statemine（但为了简洁起见，下文我们都使用 Statemint/Polkadot）。

Statemint 将如何获得平行链插槽？

我们正在构建 Statemint，将会提出平行链插槽的治理提案。该提案将进入全民公投，由 Polkadot 的利益相关者决定。有关公共利益平行链的更多信息，请参阅此博客文章：《公共平行链：通过治理分配平行链插槽的介绍》。

我将如何使用 Statemint？

Statemint 将是一条平行链，它使用 DOT 代币作为其原生代币，即在它的 Balance 模块实例中表示。为了在 Statemint 上进行交易，你需要首先使用跨链消息从中继链帐户向你的 Statemint 帐户发送一些 DOT。Statemint 上的地址将使用与其中继链相同的 SS58 前缀。请注意，在某些用户界面处理好跨链消息之前，这个过程可能对终端用户不太友好。

一种方法是让后端连接到中继链和 Statemint。当用户尝试在 Statemint 上进行交易时，应用程序会意识到它需要一些余额，并处理跨链消息的发送来进行余额转账，等待其成功，然后广播 Statemint 交易，对于用户来说只用一键操作即可。如果你想做出贡献，构建可以处理具有多链后端的应用的基础设施和 UI 将是一个巨大的贡献，不仅对 Statemint 而言。

Statemint 的费用是怎样的？

Statemint runtime 的存款和费用设置为中继链级别的 10%。也就是说，一般来说，交易费用应该是中继链上的费用的 1/10 左右（对于代理和多重签名等存款也是如此）。账户存续的最低金额也将是其中继链金额的 1/10。

注：这些值不会正好是 1/10。平行链的每个区块的 Weight 限制低于中继链，费用会根据区块的饱和度而变化。因此，如果 Statemint 区块在一段时间内比中继链区块更满，则费用将高于中继链的 1/10。

我可以运行收集人吗？

是的，Statemint 将有非常简单的逻辑，可以让一个人成为收集人，只需要绑定固定的金额。请注意，收集人没有通胀奖励，他们只收取一部分交易费用。在撰写本文时，Aura 尚未用于平行链，因此请耐心等待，因为我们会使用此新功能来扩大收集人插槽的数量。

Statemint 会支持智能合约吗？

不，Statemint 支持处理资产的专用逻辑。它不会提供任何智能合约接口。

Statemint 会支持 NFT 吗？

最终会支持，但可能不会出现其在 Kusama 和 Polkadot 的第一个版本中。

有关支持同质化和非同质化代币的路线图，请参考https://github.com/paritytech/substrate/issues/8453。

Statemint 会与智能合约链、NFT 链等竞争吗？

Statemint 是一条非常基础的链，它只提供一个表示资产的接口，和一些处理资产的原语函数。代币往往会使具有已知复杂性的合约存储和计量交易的智能合约链膨胀。通过将此逻辑直接编码到 Statemint runtime 中，代币存储和操作可以更快、更便宜。

这让其他链可以专注于它们擅长的领域，而不会被代币执行所拖累。智能合约链专门让任何人将非置信代码部署到全球系统。NFT 平台专注于培养社区、市场和画廊的能力。Statemint 可以存储代币的底层接口，而其他系统可以编写与它们交互的逻辑。

例如，合约执行的操作可以触发跨链消息，让 Statemint 处理作为合约执行结果的代币交易。这将减少 gas 的浪费，并降低智能合约链上的费用。同样，NFT 链或平台可以专注于其业务逻辑，也就是代表社区成员的交互，以及转让 NFT 的条件；但 NFT 的数据结构和原语函数可以放在 Statemint 上，供多个社区使用。

Statemint 将如何进行治理？

作为一条公共物品平行链，Statemint 必须与中继链保持完全一致。升级到 Statemint 将需要中继链的 “根起源（root origin）”，即公投。其他一些逻辑（如特权资产功能）将服从中继链的理事会，并可以通过 root 替换。
