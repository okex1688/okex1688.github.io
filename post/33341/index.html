Web 3.0 需要更多用户 而非更多投资者

项目需要专注于开发吸引非加密用户的产品，从而推动Web 3.0的创新和成熟。

加密货币的世界一直是由那些对金钱非常了解的人组成的。毕竟，加密货币本身就是金融产品。

Web 3.0的范式转变总是能够引起投资者的兴趣，在过去的几年里，我们已经看到有大量的资金流入这个行业。 这些资金既包括直接投资 — 风险投资（VC）和现在的去中心化自治组织（DAO），也包括间接投资 — 机构对加密货币的收购推动了代币价格的上涨，增强了Web 3.0项目基于加密货币的财政。

随着时间的推移，我们可以期待看到越来越多的资源涌入Web 3.0领域，但一个越来越明显的事实是，这个领域并不存在（货币）资源短缺的问题。它所面对的，是对主流受众来说可用、合适和有吸引力的产品的短缺。

到目前为止，Web 3.0和加密货币主要是由典型的早期用户人群使用 — 年轻人和具有高技术水平的人。对于任何极具颠覆性的技术来说，这都是可以预期的。在过去，社交媒体（Web 2.0）甚至计算机也是如此。当Meta（Facebook的前身）刚开始的时候，只有大学生被邀请使用这个平台。当“网络”初露头角时，必须保持一份已知网站的清单，只是为了让人们能够使用它。

加密DNA测序：金融、去中心化和民主化

目前，我们大多数人仍然生活和工作在这样一个世界里，用加密货币交换法币依旧是其功能的一个重要组成部分。这种模式很可能在未来发生转变，但就目前而言，大多数人愿意持有任何特定的代币，归根结底是他们相信其在数字经济和其他经济中超越其他资产的能力 — 激励因素是金融。

这些激励措施导致了一大批以前没有成为普通投资者的人涌向加密货币。进入加密货币市场的资金门槛相对较低，再加上关于巨额利润和加密货币亿万富翁的故事的诱惑，在加密货币游戏中催生了一个巨大的零售投资者阶层。这些散户投资者往往是那些为许多Web 3.0产品提供用户数量的人。

在大多数情况下，这些产品本身与使用加密货币作为投资和投机的工具直接相关。Uniswap（UNI）、OpenSea和Aave（AAVE）是一些最知名、最受欢迎的Web 3.0产品，它们都直接与收购和交易以及获得加密货币资产的杠杆有关。当然，这些产品的流行是有道理的，它们的流行促使它们成为整个领域中最精良的产品。

然而，“投资者”始终是整体人口中的一个相当小的子集，因此，能够在像Uniswap这样的东西中找到价值的人的数量也将始终是一个上限。为了使Web3.0继续增长 — 规模、采用率和价值–需要更加关注创造普通人每天都能找到价值的产品。尽管这是一个令人难以置信的创新，但只关注金融的Web 3.0永远不会成为普遍的技术。

取代Web 2.0：模仿 vs. 创新

很多Web 3.0项目通过成为“Web 2.0平台的替代品”而取得了大规模的进展。与Web 2.0相比，Web 3.0有一些明显的优势，这使得它们对广大受众更有吸引力。

一些应用程序，如Odysee，相较YouTube，为创作者和用户提供了一个更公平的平台。通过消除与中介和中间商有关的成本和开销，Odysee能够为其平台上的创作者提供比YouTube和Twitch等竞争性Web 2.0平台大得多的利润分成。

与Odysee一样，Brave浏览器也试图为其用户创造一个更公平的平台 — 这次是通过用一个更私密、更透明和更公平的替代方案来彻底改革基于广告的Web 2.0收入模式。

Web 2.0的广告往往是侵入性的，无论是在注入我们在线消费的内容方面，还是在用于提供目标广告的隐私侵入性做法方面。Brave广告根本不出现在网页上，而是出现在系统通知托盘中。广告不会使用你的个人信息的巨大数据库，而是在本地选择 — 因此你的个人信息可以保持隐私。不仅如此，如果你愿意，你可以完全选择退出Brave广告。如果你选择加入，用户可以得到广告商所支付的70%的费用 — 创造一个更加公平、私密和透明的广告系统。

其他应用程序，如Session Messenger，利用去中心化的好处，与中心化通信软件相比，提供了极大的隐私。通过使用它的服务节点网络，Session能够以完全去信任的方式提供比任何中心化竞争者更多的隐私和匿名性。

像这样的应用程序在将主流用户纳入Web 3.0的未来方面发挥着重要作用。2021年，仅会话就增长了500%，因为用户放弃了WhatsApp，而选择了更私密的替代品。Brave正在慢慢建立自己作为一个主要浏览器的地位，目前有超过4000万用户。

这些东西都有几个共同点：它们很容易使用，无论你对加密货币的兴趣如何，它们都有明显的优势，而且它们不需要任何重大的金融投资。

为什么我们需要新用户：改善产品，提高价值

在技术领域，价值最终是由用户驱动的。最后，如果没有人使用你的技术，那还有什么意义？如果Web 3.0要保持整个世界上最大的增长行业之一，它需要用户。

就其提供的所有好处而言，创建真正的去中心化的应用程序要比其中心化的替代品复杂得多。更多的用户将有助于解决这个问题，因为有用户的产品可以更快成熟。随着Session的发展，来自其社区的额外反馈、报告和支持极大地帮助了实际应用的改进。相信其他项目也发现了同样的情况，随着他们的受欢迎程度增加，一切都进展得更快。

加密行业的下一步应该是尽快达到用户的临界质量。这不仅会继续推动应用本身的增长和改进，也会吸引新的投资进入这个领域 — 更多的资源，更多的增长。Web 3.0的列车比以往任何时候都更快，但我们仍在加速前进。如果我们想让车轮继续滚动，我们应该支持和鼓励那些将新的主流受众引入令人激动的Web 3.0世界的项目。
