4 月 23 日，日本著名潮流艺术家村上隆在 Instagram 发表长文，阐述了他对 NFT 的看法。村上隆此前曾在 NFT 交易平台 OpenSea 拍卖像素版小太阳花 NFT，不过后来取消了拍卖活动，村上隆给出的理由是对 NFT 缺乏了解。

而本文是村上隆在参与了一场在 Clubhouse 举办的 NFT 分享会后所写，也是他在取消 NFT 拍卖后首次谈论 NFT。本文中，村上隆探讨了加密艺术家为何会对加密艺术社区十分忠诚、人们为何会去收藏 CryptoPunks 等 NFT 收藏品、NFT 对饱受新冠疫情折磨的人们意味着什么以及亚文化崛起的原因。

律动 BlockBeats 将全文翻译如下。

原文作者：村上隆

我在 Clubhouse 里听了一场由 Jehan Chu 和 Loic Gouzer 主持的讨论会，著名 NFT 藏家 Pablo 也加入了这场讨论，对于 NFT 我也有一些我自己的想法想写出来与大家分享。

首先，让我们来看看 Beeple 的作品，他因为在佳士得的拍卖一举成名。他的作品极具科幻感，并且往往都充斥着对社会的批评，Beeple 在 Instagram 上的地位似乎可以和班克斯在线下的地位相提并论。他已经加入了 NFT 领域去售卖他的动画几静态作品，他也在选择来到 NFT 的世界后以至于那些早已活跃在 NFT 生态系统中的人的交流。

而谈到加密艺术，目前它正在和加密世界一同成长，而在我看来，加密艺术的科幻感与兼容性正在推动着 NFT 领域飞速发展。我相信所有的加密艺术家都会为他们生活在一个由自己创造、建设的新领域而感到自豪，并且会向更多人分享他们对于未来无数发展可能性的信念与敬畏，而每一个艺术家个体的感受终将会升华成为集体的自豪感。而这可以说是属于加密艺术家的英雄主义与浪漫主义。

而另一方面，诸如 CryptoPunks、CryptoKitties 这样的系列收藏品，与收藏卡有着类似的原理，即通过唤起人们的占有欲来促使人们购买。CryptoPunks 总量由 1 万枚，而其中仅有的几枚蓝色的 Alien Punks 由于极其稀有，所以屡次卖出天价。而这种花天价购买一个像素头像的行为是收藏家心里的「渴望」来推动的。

艺术品收藏需要一个「动机」，万事开头难，但一旦你开始了收藏并完全沉浸其中时，收藏就会让你上瘾。有时候人们会觉得，当我花光了所有钱之后，这个「瘾」自然会被戒掉，然而并不是，之所以被称为「瘾」是因为即使你知道自己已经无法承担，但你还是想去购买更多艺术品。

所以我也认为，从某种程度上说，Beeple、CryptoPunks 和 CryptoKitties 等一众 NFT 作品中潜伏着很多成瘾因素。

下文中，我将会和大家分享我自己收藏成「瘾」的故事，并尝试从这一点解读如今的 NFT 热潮。

在 2008 年的时候，我的雕塑作品《My Lonesome Cowboy》在苏富比拍卖行排出了 1500 万美元的高价，这让我大吃一惊，并想亲自探索、体验购买艺术品、收藏艺术品。于是我开始疯狂收藏艺术品，几乎花掉了我赚到的三分之二的钱，我也真切地体会到了一个人在收藏的时候是怎样的心态。

艺术收藏、拍卖真的会令人上瘾，我买了很多作品，有时候即使我没有多少钱、好几次濒临破产，但我还是停不下来。

我认为在 NFT 的世界里，收藏也有着相同的成瘾性。一旦一件作品在某个特定的环境变成了可收藏的物品，它的成瘾性就会促使人们前来购买，而随着一次天价成交，人们会开始不停地谈论着它，开始探寻它背后的价值，而这又将会反过来推动人们的购买欲，而卖家也会借此机会继续推销他们的作品。

NFT 世界曾经是相对闭塞的，而这一次加密艺术热潮的开始几乎与加密货币牛市的开始是同一时间，这也许也与成瘾性有着密不可分的关系。

当人们因为新冠疫情全球大流行而被困在家里或陷入十分窘迫的境地时，往往需要对某些东西产生依赖，来让那个自己的心理状态维持在一个正常、健康的水平。在过去，这可能是宗教需要做的事情。

在疫情中，人们会对游戏，加密资产和金钱游戏之类的东西更感兴趣。这时，人们自然而然地会喜欢上那些带有奇妙元素的东西，无论是可爱的还是科幻的，都可以帮助他们把灵魂从这个残酷的现实之中解救出来。

我认为这种转变很像当年《星球大战》改变了电影行业的未来，曾经不被人们所接受的科幻电影如今已经成为了世界票房的中流砥柱，人们也会在科幻电影中讨论哲学与意识。现在大家也都了解漫威和 DC 电影在电影史的重要地位。

而在日本，动漫是日本文化中的重要组成部分。日本的动漫和游戏对与日本文化和金融的影响大概已经有 50 年的历史，而现在它们正在走向世界。 也就是说，曾经被认为是幼稚和不成熟的亚文化现在正在占领主流市场。

从这个角度上看，那些可爱的或者是极具科幻感的 NFT 作品将会是这个时代的潮流，它们与当代艺术完全不同，而我一直以来的创作风格，超扁平，则十分适合这个时代。因此，我对 NFT 艺术产生了浓厚的兴趣，并且通过沉浸在这个时代潮流中进行研究，我认识了超扁平化的意义，以及超扁平化后续发展的方向。
