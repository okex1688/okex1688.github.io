北京时间9月1凌晨，以太坊L2解决方案Arbitrum正式启动主网Arbitrum One，同时宣布完成了一轮1.2亿美元的B轮融资。

那作为一名对其感兴趣的潜在用户，我们该如何去使用Arbitrum One二层网络呢？

在学习这个教程之前，你需要安装一个类似MetaMask 这样的 web3 钱包，并且你在自己的以太坊主网钱包中已经拥有了ETH或一种token。

步骤1:添加Arbitrum One网络

第一种方式：通过在此处访问Arbitrum的资产桥，并单击页面中间的“添加/切换到 Arbitrum 网络”链接来自动添加它。

第二种方式：或者，你可以在metamask手动添加，依次点击“metamask扩展”>点击“以太坊主网络”>点击菜单末尾的“自定义RPC”，然后输入如下信息：

Network Name: Arb1
RPC: https://arb1.arbitrum.io/rpc
Chain ID: 42161
Currency Symbol: ETH

Block Explorer URL: https://arbiscan.io

如果你是一名开发人员，你还可以使用 Alchemy 或 Infura 作为连接到 Arbitrum One 的替代方式。

步骤2:将以太坊主网的资产通过桥（bridge）转移到Arbitrum One二层网络

要使用Arbitrum One（L2），你需要首先将自己的资产从以太坊主网通过桥转移到Arbitrum One，你可以通过http://bridge.arbitrum.io/访问这个资产桥。

1、确保你连接到了以太坊主网

2、在token下拉菜单中选择要转移的代币。

3、在“L1”字段中输入你想要转移的ETH/代币数量，然后点击“deposit”存款，然后按照Metamask 上的提示进行操作。

4、在你通过Metamask 提交交易后，在等待大约10分钟-1个小时（具体取决于拥堵情况）的时间，你的资产就会到达Arbitrum One 二层网络。

此时你要确保自己的MetaMask 钱包已切换到Arbitrum One 网络，这样你才能看到自己的资金。

步骤三：选择你想要使用的dapp应用，并进行相关操作

Arbitrum 生态系统充满了各种dapp、钱包以及工具，你可以通过Arbitrum One Portal浏览完整列表。

http://portal.arbitrum.one/

步骤四：将资金从Arbitrum One二层网络提取到以太坊主网

注意：由于Arbitrum One采用的是Optimism Rollup方案，因此存在欺诈证明挑战期，一旦你通过资产桥提取你的资金，你需要等待至少7天的时间，然后才能在以太坊主网收到你的资金（这与Optimism的情况是相同的）！

1、确保你已连接至Arbitrum One 网络

2、在token下拉菜单中选择你要提取的代币；

3、在“L2”字段中输入要提取的ETH/代币数量，然后点击“withdraw”提款，按照Metamask 上的提示进行操作。

一旦你通过资产桥提取了资金，你将至少等待7天的挑战期后，才能在主网上拿到你提取的资产。

4、欺诈证明挑战期结束后，你需要点击“claim”按钮，然后你提取的资产才会出现在你的以太坊主网钱包中。

需要注意的潜在风险

1、智能合约风险

任何智能合约中都可能存在可能导致用户资金丢失的漏洞。

2、价 格波动风险

将资产（ETH或任何代币）从L1转移到L2，或者从L2转移到L1的过程中，它们的价格会因市场波动而发生变化（尤其是从L2转移到L1的过程，至少需要等待7天的时间）。

提示

由于用户体验上存在的障碍（主要是提款7天的挑战等待期），Arbitrum One在初期阶段可能会劝退一部分用户，但其生态应用的多样性或正统性，可能是其他L2网络或竞争L1网络无法比拟的，此外，Offchain Labs的联合创始人Steven Goldfelder提到称，尽管Arbitrum在短期之内不会发行代币，但DAO可能会决定在未来进行代币化。
