Loot 在几周前诞生，行业对它的争议非常大：

喜欢的人特别的喜欢，因为他只保留了 8 个非常简短的元数据，可以在各种场景拓展，所以很多人认为 Loot 是一个有颠覆性的 NFT 的范式。

讨厌他的人则不看好，且质疑 Loot 就是一个图片，在上面放了一些文字而已。

因此关于如何理解 Loot ，BeWater 发起人 BMAN 分享了一些个人的思考。

大家好，我是 BMAN， 我 13 年购入比特币，开始接触区块链，目前跟朋友一起发起了 BeWater 开发者社区。BeWater 是一个非常硬核的开发者社区，有很多国内外顶尖开发者，会一起讨论前沿的区块链技术（ 点击 链接 了解 BeWater ：一个不焦虑的开发者社区）。

以下是我们对 Loot 的一些看法。

热媒介和冷媒介

有人认为像 CryptoPunk 这样的 NFT，实际上是个图片，而 Loot 是文字，这是它的一种区别，但它不是本质的区别。

我也在思考，Loot 的出现意味着什么？对于 NFT 来说有什么样的改变？

我思考了很久，直到我看到了最伟大的传播学的大师麦克卢汉，在「Understanding Media」的一句话 「热媒介比冷媒介让受众更多的理解、更少的参与」。

他在 「Understanding Media」里面定义了两种媒介：热媒介和冷媒介。

热媒介指的是解析度很高的，具象的，对受众参与度要求较低的媒介。这种媒介不需要解释，因为他提供了所有的必需要的信息。

冷媒介指的是解析度低的，抽象的，要求受众积极活跃参与的媒介，他需要有足够的认知和理解的投入。

举个例子，像照片跟漫画，照片就是一种热媒介，漫画就是冷媒介。因为照片它包含了 1 万个字以上的信息量，所以它的解析度很高，信息是非常具象的；但漫画的解析度很低，它是很抽象的，你需要投入很多的参与度才能看懂漫画。

再比如，象形文字和拼音文字，象形文字是热媒介，因为它具有象形，就像汉字；英文是一种拼音文字，是一个高解析度的热媒介，因为他更具象一些。

再比如讲座和研讨会，讲座只需要听，比如你今天听一下 BMAN 讲 Loot 其实就是一个热媒介；但研讨会不一样，研讨会就是一种冷媒介，因为你需要高度的参与，然后需要投入很多的认知，所以这就是热冷媒介的区别。

我认为热媒介和冷媒介的区别，就是古典 NFT 和 Loot 的本质区别。

这是两副梵高的星空，一幅是原画，另外一幅是后面的先锋艺术家做的一个装置艺术。

大家认为哪个是热媒介，哪个是冷媒介呢？左边包含了非常具象的信息，是个热媒介；右边这个装置艺术把它抽象化了，更多的让受众参与进去了，使用户成为了这个艺术的一部分，所以它是参与度更高的一个媒介，也就是冷媒介。

也就是说，如 CryptoPunk 等古典 NFT 和 Loot ，最本质的区别在于热媒介跟冷媒介区别。

像 CryptoPunk 这样一个图像，它其实非常的具象，它把所有的内容、所有需要表达的东西都定义完整，就在这样的一个图里，不需要太多的受众参与。

但是 Loot 不一样，它只定义了 8 个最基础的元数据，它是非常高度抽象的，且需要受众、开发者、社区，甚至是各种各样的项目方一起共同参与设计的，这就是本质区别。

古典的 NFT 是一个热媒介，Loot 是一个冷媒介。我们就去看 NFT 的发展的历程，其实就是一个从热媒介到冷媒介的发展的过程。

最早的 CryptoPunk 从 17 年的时候，他最早定义 NFT。

然后到了 NFT 2.0 的时候， Art Blocks 是一个算法生成艺术的平台，说艺术家他把一段算法上传到这个平台上面，算法生成出一幅艺术品，所以看到这幅画其实是算法生成的，相当于是他的创造者创造了一半，然后另外一半是从算法生成。

NFT 3.0 ，即 Loot 的时候，创作者干脆只创造了一个开端，他只做了一个开头，剩下的所有的都由后面的社区和受众一起来创造。

由此可见，NFT 从定义完整的，定义一半，到了 Loot 这样只定义了一个开头的，它是越来越抽象的，这个抽象到只有文字了。就像麦克罗汉所说的，就是我们看 NFT 的发展历程，它其实走过的是热媒接到冷媒介的一个过程。Loot 的一个最本质的创举就是把 NFT 带到了一个冷媒介的时代。

科技创新来源于媒介的创新

大家觉得 DeFi 是热媒介，还是冷媒介？

DeFi 是冷媒介，基础层有以太坊，有 MakerdDAO 到提供稳定币，有 Compound 提供借贷，有 Uniswap 提供各种交易，所有的组在一起，才完整的组成了 DeFi。

每一个都是一个需要受众参与，更抽象的更颗粒化了，更加的可具组合性的一个组件，所以整体组成了一个 DeFi 的生态，每一个单独开来都是一个冷媒介。

所以麦克罗卢汉还说了一个非常重要的点——所有媒介都是其他媒介的媒介，媒介等于媒体的一个科技，所以我们看到其实人类历史上是所很多的那种科技的进步和发展其实都来自于媒介的进步和发展。

比如说我们从电脑发展到手机，就出现了从 PC 互联网到移动互联网的发展，它本质上是一个媒介的发展，有了手机这个载体，就诞生了大量的科技创新。

区块链再到 NFT 它其实上也是媒介，区块链本质也是一个媒介，它是去中心化物联网的一个媒介。

那么今天到了 Web 3.0 互联网，然后到了 NFT 现在这个时代 , 我们看到其实正如麦克卢汉所说的，所有媒介都是其他媒介的媒介。

我们看是今天的 DeFi ，也是其他媒介的媒介，是冷媒介。

区块链

我们看区块链的历史，我们看到同样的历史正在押韵。

13 年的时候，比特币出现了第一个浪潮。比特币是一个热媒介还是冷媒介？相比较 17 年的以太坊，它其实是一个热媒介，以太坊是一个冷媒介，因为比特币它非常具象了，它就是一个价值存储的一个工具媒介。以太坊它定义了一个抽象的去中心化互联网，且以太坊需要更多的受众的参与，他需要更多的受众的投入，比如说我们可以去上面发 ICO，但在比特币上面，我们可能做什么都做不了。

那到了今天，我们的以太坊更加的媒介化了，他比 17 年的时候更需要受众参与，这个时候你需要掌控自己的 MetaMask 去掌控自己的私钥和钱包和去各种挖 DeFi， 以及收菜。

我们会发现，在这个周期里面， 2021 年我们所在这个行业非常的懂媒介化，我们需要大量的活跃的积极的参与，不像是在 13 年只要买币就行。

举个例子，13 年的时候，我们买比特币是在比特币中国，可以用法币、人民币入金去买比特币，这是一个非常具象的热媒介的途径。

到了 17 年，用的是更复杂的币安，它是一个用 币币交易了，用 USDT 去交易，它是一个逐渐的冷媒介化。

到了今天，我们要用 Uniswap ，一个更加去中心化，需要大量受众参与和理解，需要在这个过程中去各种组合，才能运用的一个工具的媒介。

所以我们看到不仅仅是 NFT，NFT 从 CryptoPunk 到 Art Blocks 到 Loot，他走过了热媒介到了冷媒介的过程。区块链也一样，从比特币到以太坊到 DeFi， 也从热媒介到冷媒介的过程，越来越越抽象，越来越需要我们投入认知和参与。

其实我们看到了一个非常共性的点，像麦克卢汉说的，冷媒介确实是一个发展到今天，需要大量用户参与和投入的一个趋势。

互联网

我们再拔高一点，到一个更宏观的视角去看整个互联网，会发现也是在一个这样的趋势当中 。

  • Web 1.0，所有的网站都是门户网站，它是单向的信息输出，不需要用户怎么参与。

  • Web 2.0，它成为了一个双向的互联网，用户可以输出也可以输入，可以去发布信息和内容，需要用户更多的参与；同时网络也变得更抽象了，更具拓展性。

  • Web 3.0 更加的抽象，更加需要我们每个受众的高度参与。我们需要保管钱包、私钥和资产，然后在各种各样的组合里使用我们的应用。

不仅仅是区块链，包括互联网，它也是从热媒介到冷媒介的发展过程中。Web 3.0 的所有应用其实都是冷媒介，它每一个都是可以互相组合成一个整体的。

所以，总结下 NFT 区块链和互联网的发展：

我们正身处一个冷媒介范式转移的浪潮之中。

Web 3.0 的本质其实是冷媒介的互联网。我们看到所有的今天包括未来的 Web 上的应用，它其实都是一个个可组合的颗粒状，需要大量受众参与的抽象的组件，所以我们未来构建的是一些 Web 3.0 互联网里面的冷媒介。

Loot 的出现意味着开启了 NFT 冷媒介的新纪元。在 Loot 之前所有 NFT 其实都是热媒介，是非常具象的表达完整的。在 Loot 之后，它出现了一个全新的冷媒介，所以这其实是我认为 Loot 真正的意义。

Web2.0 跟 Web3.0 的区别？

我们带着这个认知我去看了 Loot 创始人 Dom Hofmann 所有的 Twitter 和在 Loot 社区里面的帖子。

我发现他有一个非常深刻的思考，他思考了 Web 2.0 跟 Web 3.0 的关键的区别：

他认为 Web 2.0 就像我们所说的热媒介，用户内容数据都存储在应用中，这些应用程序很大程度是割裂的，而且是互相孤立的权限。应用程序之间的访问程序需要非常孤立地去申请，所以你看到的 web 2.0 是块状的，它是一个非常具象的已经定义完整的东西，而且需要受众参与的程度非常低。Web 3.0 它是一个典型的每一条数据和内容都列在一个开放的空间中，应用程序像颗粒状，它们之间可以互相组合权限，其他任意的应用程序也可以互相访问。

Web 3.0，把内容、数据和用户都变成颗粒状，这就是典型的一个热媒介。Dom Hofmann 说了一句话， Web 3.0 最有价值的事情是数据和内容的 Itemization。

Itemization 什么意思？就是颗粒化，就是把用户数据和内容变成颗粒化，变得可组合化、抽象化，然后元数据化。Itemization 其实本质也是麦克卢汉所说的冷媒介化。

所以我们今天在做的 Web 3.0 互联网是在做什么？最有价值的事情是做数据和内容的媒介。今天我们在 BeWater 社区跟开发者聊天，也在构建未来一个个的冷媒介，去拼接成一个未来的世界的版图。

同时，也可以发现，Dom Hofmann 是有很深度的思考。

他在做 Loot 之前，并不是随便发了一个图片，上面有 8 段字母，就是一个 NFT 了，他背后有自己的哲学：未来是一种 Itemization 的，是一种冷媒介的。

Dom Hofmann 的一个项目叫 Blitmap，它总共有 1700 件的 NFT 项目，其中 100 件是原创的，剩下 1600 件都是从这 100 件里面排列组合拼接而成。

从这些作品能看到什么？这 100 件其实是最原始的冷媒介，它是可组合的，可拼接的，需要受众去参与，去投入的。

过去 NFT 是完全是设计好的，是没法玩的，但 Dom Hofmann 认为未来是冷媒介化的，所以他做的第一个项目就是这种的冷媒介，1+1 等于一个新的东西的项目。

上述案例中，我们可以看到他做的第一个项目就是源于这样的一个哲学，后来到了 Loot 也延续了这个哲学：未来的 NFT 也是要像 DeFi 那样可以互相组合，每一个都可以拼接在一起互相调用组合。

他用这样的哲学做了 Loot ，所以我们把 Loot 跟过去的一些 NFT 对比，它需要社区，需要用户的高度参与去构建这样的一个生态，它更加抽象，是低解析度的，具有高组合性。

我们能看到，因为这样的一个哲学，他一夜之间唤起了全世界的开源社区，去一起构建 Loot 的一个世界和宇宙。

现在 Loot 的一个生态已经出现了，叫 LOOTVERSE，就是 Loot 宇宙的意思。

那么已经有三层的 Loot 宇宙：第一层是资产层；第二层是由很多开发者基于 Loot 做的游戏，还有开发者做 Loot 的交易；第三层是 Loot 的 一些技能、资源，还有人格，甚至有一些开发者自发的发起了 Loot 黑客马拉松，大家来一起做一些好玩的 Loot 生态宇宙。

所以 Loot 是一个原宇宙的第一个组件，他就啪的一声，开启了宇宙大爆炸的原点。他就像是编剧，只写了一个开头：

比如说，普通的少年走出了新手村……后面所有的游戏、剧情、情节都交由社区共同创造，这有无限的可能，可能 100 个编剧，写出 100 个故事， 100 个社区，做出 100 个游戏，这就大大的拓展了一个 NFT 的可能性和扩展性。

从过去的已经写死了的 NFT 变成了未来可以不断扩展，不断定义的 NFT；从一个封闭的 NFT 变成一个开放的 NFT；从一个热媒介 NFT 变成一个冷媒介的 NFT ，所以能看到现在社区已经把 Loot 跟一个 ROLE 的一个项目拼接在一起，并且视觉化了。

甚至还有一些社区在个人物属性基础上又加了宠物装备，就变成一个更加丰满的属性。还有的社区已经把 Loot 所有的装备视觉化，变得可组合，变成一整套视觉体系，这都是开源社区去创造的，他们已经在用自己的理解去构建了一个想要的宇宙。

更甚者，只要把 Loot 放在这样的一个工具里面，它就能够用 AI 生成所有工具的视觉化，这完全都是社区的一些创造。

我觉得这是非常酷的事情。Dom Hofmann 其实非常像一个在构建一个宇宙的首席工程师，首席架构师。他在搭建一个宇宙，把这个世界的骨肉骨架给搭起来，其他的血肉都交由社区、其他的开发者，让他们创造一个想要的世界。我看他发的一些帖子，是有非常宏大的格局的，比如说他发的这个关于 Loot 要去做一个映射，然后让一些 Loot 的持有者可以免 Gas 或者获得空投。

比如这个帖子，他提出一个提案：要销毁他持有的 Loot 合约的私钥。这样的话，他就失去了对 Loot 生杀的大权，因为他非常担心未来 Loot 宇宙会出现一个像教会中心的事情，这样会伤害 Loot 宇宙的未来发展。

此外，他又提出了 ERC-1155，一个以物换物的 token，因为他认为 Loot 宇宙必须要有一个交易的冷媒介。可以看到他有一个非常宏观、高瞻远瞩的设想，他会看到一个宏观的宇宙需要什么样的骨架，然后就去提案号召社区一起创造，所以他像是一个非常宏观的架构师。

总结

Loot 太新了，它只有 20 天的时间，他绝对代表了一个新的哲学和理念，但并不意味着就能成功，很可能会倒在黎明之前，但我觉得 Loot 至少代表了一个非常新的理念，至于后面能不能成功，需要更多时间的考验。
