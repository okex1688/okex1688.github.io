今天，我们很兴奋地宣布元宇宙（Metaverse) 的新时代到来了。Bit.Country（小元世界）团队在第一轮融资中获得了400万美元的资金，并得到了60家出色的元宇宙（Metaverse) 信仰机构支持。

// Bit.Country投资者与合作伙伴//

Bit.Country Investors & Partners

作为强劲的社交网络影响者，沃尔什财富创投公司(Walsh Wealth Ventures) 凭借其强大的KOL人脉网络在社交媒体平台上吸引了2亿粉丝 ，联合领投了Bit.Country本轮投资。

作为全球区块链游戏的领导者 ，Animoca Brands 也加入了这一轮的联合领投，并为元宇宙（Metaverse) 游戏带来了丰富的知识和联系。作为区块链众多热门游戏之一的所有者，我们也期待着将来与Animoca Brands在这一领域的一起合作。

作为Polkadot生态系统的顶级基金会 、加密生态基金Hypersphere Ventures 和加密数字资产与区块链技术投资企业Digital Financial Group (DFG) 是来自Polkadot社区的联合领投者，他们为我们带来了一个清晰的链上未来，以及Polkadot/Kusama上的链组成。

作为远见卓见者和潮流引领者 ，由杰弗里·伍 (Geoffrey Woo)和杰克·保罗 (Jake Paul)创建的Anti Fund (此基金会在YouTube上有2000万订阅用户)， 以及数字文艺复兴基金会DRF.EE（Digital Renaissance）的创始人曹寅 (中国第一个加密艺术品收藏家) 都是这一轮投资的有强力支持者。

作为硅谷的加密和数字融资平台 ，Republic团队对我们的工作表示了强烈的信心。其投资部门Republic Labs和数字房地产NFT基金Republic Realm都加入了此轮融资，让我们能够接触到从加密零售投资到数字房地产开发的最精英社区。

//  关于Bit.Country小元世界//  

小元世界（Bit.Country）让每个人都可以用代币经济和非功能性货币启动他们自己的元宇宙（Metaverse)，并将社区互动带到web3.0的一个新维度。

你的小元世界（Bit.Country）将有一个本地社区代币，由我们的代币$NUUM铸造 、本地市场 和治理社区的本地DAO 支持，并对资产供应等问题做出决策。

我们也正在将社交活动过渡到3D虚拟世界，同时允许用文本和多媒体，访问传统的网络社交时间轴.

// 我们的使命 //

虽然我们的团队对web3.0的开发感到兴奋，但我们的使命是创建一个能给现实世界带来影响的网络。我们希望在这种模式转变中为普通人提供产生价值的机会 。

当大众群体能够通过贡献他们的小元世界（Bit.Country）而获得收入，当小元世界（Bit.Country）的所有者能够通过网络扩张产生更大的影响力，从而将价值分配给他们的追随者时，我们的愿景和梦想就会实现。

// 我们的起源 //

2018年初，我们的联合创始人兼首席执行官Ray Lu, 在旧金山出席中美区块链大会时，已为虚拟资产和虚拟生活描绘了未来的数字世界。

通过一个去中心的高速价值网络，让虚拟世界变得有意义和真实，这就是元宇宙（Metaverse) 和现实之间的强大联系。自此以后，Ray购买了域名bit.country, 然后开始勾画了一个新的冒险旅程。

和他们一起开始你的小元世界

从那时起，他的团队成员Justin、Shannon和Daniel以他们的师徒文化，开始了对此概念的验证。

2018年底，他的前同事也是朋友、来自Acala Network的CEO兼联合创始人Ruitao，向Bit.Country团队分享了Substrate/Polkadot技术。

这项新技术让Bit.Country团队兴奋不已，因为他们发现这是市场上唯一可以实现Bit.Country最终愿景的框架, 同时具有环境的可持续性。

“在目前的市场环境下，我们本可以提前8个月筹集投资，但我们选择先建立里程碑，并创建一种长期思考的文化。这不仅是为了社区，更重要的是为了我们自己，也就是共同创建团队的成员。

我们的种子轮估值证明了我们的长远想法，所以我们的目标是策略性估值，给我们的用户一个增长的空间。”

——Ray Lu

创始人Ray Lu是一位经验丰富的企业家，他在科技和房地产领域成功创立了3家价值数百万美元的初创公司，其集团目前拥有5000万美元的资产管理规模。

“现在，我致力于在未来10年里建造一个元宇宙（Metaverse) 的未来。能够把web3.0带给像我们这样的普通人，这是一种很棒的感觉。

——Ray Lu
