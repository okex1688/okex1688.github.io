发端于金融行业

区块链技术最早应用于金融。

2008年，金融危机肆虐全球，150年历史之久的雷曼兄弟轰然倒塌，现代金融体系面临前所未有的信任危机。如何去中心化，不让金融数据掌握在少数金融机构手中是重建金融秩序的重要方向。

2008年底，中本聪提出了比特币概念，并在次年一月相继创造了第一个序号为0的创世区块和序号为1的创世区块，两者链接成链，标志着第一个区块链应用的诞生。

此后，以太币等各种加密数字货币、ICO（首次代币发行）、支付、比特币交易所等新金融玩法被各路玩家开创出来。以比特币为典型代表，在塞浦路斯政府破产危机（2013年）、美国停摆危机（2013年）、希腊政府债务危机（2015年）、英国脱欧公投（2016年）、新冠肺炎疫情（2020年）等历次全球性、地区性或国家性危机中逐渐担当起避险功能，并陆续得到德国、美国等政府承认，更在币圈炒家们的助推下价值水涨船高，一度达到六万八千多美元的高价。

如果说，最初的区块链技术是技术极客们热衷追捧的新事物，其发展到后来则成为资金盘的天堂。在我国，区块链技术在金融领域的过度利用受到严格监管。早在2017年，ICO融资就已被叫停，比特币在我国不受承认。2020年5月，银保监会等五部门发布提示：不法分子通过发行所谓“虚拟货币”“虚拟资产”“数字资产”等方式吸收资金，以“金融创新”为噱头，实质是“借新还旧”的庞氏骗局，资金运转难以长期维系。2021年，中国以强硬态度全面清理挖矿产业，明确了对加密货币的态度。

“区块链+”的创新应用

但这不代表我们否定区块链技术本身。习近平总书记在2019年10月24日的中央政治局第十八次集体学习时强调，我们要把区块链作为核心技术自主创新的重要突破口，明确主攻方向，加大投入力度，着力攻克一批关键核心技术，加快推动区块链技术和产业创新发展。

代表了“第四次工业革命”的区块链技术不仅仅适用于金融行业。2016年6月，工业和信息化部、中央网络安全和信息化委员会办公室联合发布《关于加快推动区块链技术应用和产业发展的指导意见》（以下简称“意见”），指出，到2025年区块链产业综合实力达到世界先进水平，产业初具规模。区块链应用渗透到经济社会多个领域，在产品溯源、数据流通、供应链管理等领域培育一批知名产品，形成场景化示范应用。继而开启了区块链创新应用和生态建设的进程。

2021年“十四五”规划中也明确提出，区块链作为新兴数字产业之一，“以联盟链为重点发展区块链服务平台和金融科技、供应链金融、政务服务等领域应用方案”等。

区块链，根据“意见”的定义，是指：新一代信息技术的重要组成部分，是分布式网络、加密技术、智能合约等多种技术集成的新型数据库软件，通过数据透明、不易篡改、可追溯，有望解决网络空间的信任和安全问题，推动互联网从传递信息向传递价值变革，重构信息产业体系。

2021年12月公布的《国家区块链创新应用试点入选名单》中将区块链的创新应用分为了综合性、特色两大领域，其中后者包含了十六个细分领域（“区块链+制造”“区块链+能源”“区块链+政务服务”“区块链+政务数据共享”“区块链+法治”“区块链+税费服务”“区块链+审判”“区块链+检察”“区块链+版权”“区块链+民政”“区块链+人社”“区块链+教育”“区块链+卫生健康”“区块链+贸易金融”“区块链+风控管理”“区块链+股权市场”“区块链+跨境金融”），可见，区块链技术正以“区块链+”的广度深入融合各产业，推动建立区块链为核心的生态，最终成为新兴产业中的重要一支。

Web 3.0、ABCDI、NFT、元宇宙，是时候翻开互联网的下一页

全球互联时代，创意理念和创新技术呈指数级增长。这主要得利于信息传播和信息交互的空前频繁，进而快速推动了科技实践和创新融合对全球生活生产面貌的迭代。

  • Web 3.0

不同于web 1.0 的只可获取信息，不能编辑信息；web 2.0的信息可读写，人人都可成为创作者，但个人行为受到平台规制限制，如黑客攻击、操作、审查、丢失、封杀等；web 3.0，是指可读写，同时可拥有（Bitfury Group首席执行官Brian Brook语）。这里面，去中心化、平权、隐私等正是通过区块链技术实现的，从而向分布式自组织形式DAO（Decentralized Autonomous Organization）演进。

早在2014年，以太坊联合创始人Gavin Wood就曾对Web 3.0做出过探讨，在垄断问题如此严峻的今天，实现去中心化也就实现了真正的革命。尽管那时或许将出现新的“组织”形式。

  • ABCDI

ABCDI（A人工智能，B区块链，C云计算，D大数据，I物联网），是新一代信息技术的集合。2019年的《听黄奇帆讲“区块链”》一文中形象地描述了ABCD的融合机理：

神经系统（互联网、移动互联网以及物联网）、脊梁（云计算）、五脏六腑（大数据）、皮肤和器官（大数据）加上相当于灵魂的人工智能——人的大脑和神经末梢系统，基础的数字化平台就已经成型了。而区块链技术，可以从DNA层面提升人的大脑反应速度、骨骼健壮程度、四肢操控灵活性，数字化平台在区块链技术的改造之下，基础功能和应用将得到大幅度扩展。

“从具体功能上讲，云计算的核心功能在于计算能力、存储能力和通道能力；大数据的核心功能在于静态数据之大、动态数据之大以及数据被使用后新生的叠加数据之大；人工智能的核心功能在于将数据通过滤波去除垃圾后变成信息，把信息经过挖掘推送后形成知识，把知识通过智能算法形成决策性判断；而区块链则是一种特殊的互联网技术，是对‘共识人群的一种管理方式’”。

ABCD加上物联网（链接物与物），任一项技术的突破都将带来巨大增长效应，五大技术间的融合应用将为未来世界带来无限遐想空间。

  • NFT

NFT，Non-Fungible Token，指非同质化通证，是用于表示数字资产（包括jpg和视频剪辑形式）的唯一加密货币令牌，可以买卖。2021年入选《柯林斯词典》2021年度热词第一名。

尽管当下NFT存在大量炒作行为，但其作为一项创新技术本身来说，不同于比特币、美元、以太坊、数字人民币等同质化代币（FT），其不可替代、不可分割特性具有极大的应用价值。

从数字艺术品《每一天：前5000天》3月被拍卖出6934万美元的天价以来，NFT强势出圈，受到各方注目，仅在2021年，NFT市场就完成了250亿美元的交易。在后期元宇宙概念的推波助澜下，NFT作为虚拟世界底层架构进一步受到热捧。

在我国，大厂们也很快进入这项火起来的区块链垂直应用。蚂蚁集团的“蚂蚁链粉丝粒”（今“鲸探”）一经发布就遭到疯抢，签约艺术创作者、推出数字拍卖等显示出蚂蚁的长远布局。腾讯推出国内首个NFT交易平台APP“幻核”，小红书除了NFT数字藏品外还赠送给藏家一份实物商品，百度旗下百信银行推出国内首个银行业的数字藏品，字节跳动旗下TikTok推出首个NFT系列——TikTok Top Moments，京东旗下NFT发行平台“灵稀”上线小程序……据说NFT是未来最可能达到千万日活用户的区块链APP。

  • 元宇宙

2021年3月10日，沙盒游戏平台Roblox首次将元宇宙概念写进招股书，上市首日市值破400亿美元，引起广泛关注和想象。10月，Facebook改名Meta（元宇宙）的新闻成为各大媒体和社交圈的热点话题，元宇宙概念热度再度升温。

不同于1992年《雪崩》小说里对这一概念的解读，今天的元宇宙已经具有了更为具象的技术和行业内涵。经济学家朱嘉明认为，元宇宙融合吸纳5G/6G、Web 3.0、AI、VR、AR、MR等前沿数字技术，以游戏入口和突破面，在技术成果上表现为“内容系统、区块链系统、显示系统、操作系统，最终展现为超越屏幕限制的3D界面，所代表的是继PC时代、移动时代之后的全平息时代。”

从意义层面上讲，元宇宙代表了通往新世界的窗口，这扇窗口会如古雅典遭遇斯巴达般带来新的伯罗奔尼撒战争史诗？还是如潘多拉的盒子是宙斯神的不怀好意礼物？还是代表了人类命运的新的希望？它会带来时空的扭曲进而将人类的生存方式带入一个前所未有的新世界吗？如果“宇宙便是吾心，吾心便是宇宙”，那么可以肯定的是，在任何时刻，古往今来，无论西东，人类都在向下一个充满不确定的命运奔赴，在现实与虚拟的反复切换中，如果觉得奔波于现在、过去、未来是一种消耗，就更多关注当下即时所绽放的力量吧。在那一时刻，人已经解放了。

结语

听起来都像乌托邦，迷人又浪漫。理想主义者已在奋楫击水，如西西弗斯抱石前行。
