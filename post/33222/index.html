如果区块链是城市？

我们将生活在一个多链的世界中，还是会有 “ 一条链来统治一切 ” ？这取决于您对区块链的心理模型。

人们通常将 L1 区块链描述为网络，例如以太坊网络、Solana 网络。这意味着区块链可以像互联网、Telegram 或 Facebook 一样无限扩展。如果区块链是网络，那么网络效应将占主导地位，一个区块链将获胜。

但是网络是区块链的错误类比， 区块链受到物理限制。区块链无法扩展到无限的区块空间，因为区块链需要许多独立的小型验证者；如果区块任意大，区块链将不再是去中心化的。

智能合约链更像城市。如果你接受这种心智模型，那么围绕 L1 区块链的动态就变得不那么神秘了。

每个人都喜欢抱怨以太坊。

它的价格昂贵。它很拥挤，很慢，它建于很久以前，没有任何东西能像它应该的那样工作，而且似乎没有任何改进。它是如此愚蠢的昂贵，只有富人才能负担得起在那里交易。

以太坊是纽约市。

纽约

当然，纽约是一个正在发生的地方！它拥有所有最大的银行、最多的亿万富翁、最炙手可热的品牌和名人。同样，以太坊拥有所有最大的 DeFi 协议、最多的 TVL、最热门的 DAO 和 NFT。

但它很贵，如果你是一个后起之秀，你的价格就被淘汰了。也许如果你早点购买资产，你就可以致富。但是今天，价格会吃掉你，而且没有足够的空间容纳每个人。亿万富翁可能没问题，但下一代将不得不去别处。

那么你如何扩展纽约？

扩展城市的三种途径：

路径＃1： 建立，土地可能是有限的，但你总是可以垂直的。通过建造越来越高的城市，你可以在同一块土地上容纳更多的人。

但建立起来并不是一个完整的答案，摩天大楼的高度是有限度的，即使是摩天大楼也无法摆脱底层城市的拥堵。如果我住在曼哈顿的高层，而你住在不同的高层，如果我想拜访你，我必须下到一楼，叫一辆昂贵的出租车，并与曼哈顿的交通作斗争，我们没有摆脱基本的约束——曼哈顿很拥挤。

L2 和汇总是摩天大楼的区块链等价物。每个 rollup 就像一个从地面 L1 延伸出来的垂直区块链。汇总中有很多空间！但是要从另一个访问一个汇总，您必须退出到下方的以太坊并处理其底层流量。

向上建设有帮助——它让更多人进入城市——但这并不是一个完整的答案。如果以太坊现在很拥挤，那么在汇总之后也会很拥挤（亿万富翁有能力留在 L1 并支付费用）。那么你还能如何扩展区块链呢？

路径 #2 ： 是 “ 互操作性网络 ”，如 Polkadot 或 Cosmos，Polkadot 和 Cosmos 为开发人员提供 SDK 以启动特定于应用程序的区块链——专用于单个应用程序的小型区块链。所有这些区块链都通过路由系统连接——Polkadot 的中继链，Cosmos 的 Cosmos Hub。

在城市比喻中，这就像创建一个只做一件事的小城镇网络。这里有一个采矿小镇，那里有一堆工厂，然后是一个农业小镇，然后是一个只是直销店的小镇。它们中的每一个都沿着一个巨大的高速公路系统连接起来。

辛城的一个工厂城镇

这样可行，有些地方会以这种方式建造。工厂城镇和农业城镇是一回事，但它们并不是人们生活和经商的最大份额。你需要的不仅仅是少数几个小镇来吸收不断增长的人口。

路径＃3： 剩下的关于如何扩展城市的最后一种方法：建造另一个城市。这就是 Solana、Avalanche 和 NEAR 各自所做的。

当你建造一座新城市时，你必须首先复制大量的基础设施。似乎是多余的。每个新城市都需要另一条道路、另一个警察局、一所学校、一所医院。同样，每一个新的 L1 都需要另一个区块浏览器、另一个法定入口、一个原生 AMM、一个 NFT 市场。这是多余的，但每个 L1 都需要这些基础知识才能起步。

但是建造一座新城市的好处是每个城市都可以以不同的方式建造。

以 Solana 为例——Solana 是 LA 。与曼哈顿相比，它又大又宽又便宜。你可以成为一个挨饿的演员，在洛杉矶过得去！忽略东海岸对去中心化的执着——将你的应用程序移至 Solana，启动你的 NFT，并获得 10 分钟的完成。

曼哈顿

当然，Solana 并不是最去中心化的 。但是游戏和 NFT 一开始就不需要那么多的去中心化。天气很好，费用很低，没有人把自己看得太重。

那什么是雪崩？我会说雪崩就是芝加哥： 试图成为下一个华尔街，但更新、更便宜、更激进。那里很冷，但 Avalanche 在金融和贸易方面的专长让这座城市充满活力和自信，很难不押注它会上涨。

芝加哥

NEAR 是旧金山 ——专为 web3 技术人员打造。这是一个理想主义的城市，到处都是想要实现以太坊 3.0 梦想的人。在他们看来，分片是长期前进的唯一途径。

旧金山

这些城市的重要之处不仅在于它们规模大而且对商业开放，每个人对城市应该是什么以及应该如何治理都有不同的看法。它们各自接受不同的权衡，采用独特的价值观，并吸引不同的行业。

那么我们将生活在一个多链的世界中，还是会有 “ 一条链来统治一切 ” ？这是重新构建的问题：我们将生活在一个多城市的世界中，还是会有一个城市来统治所有城市？

答案很明显，城市统治有幂律（另一种解释：马太效应）分布，但有很多城市很重要，没有什么比喻是完美的。但我发现这种心智模型对于预测 L1 将如何演变很有用。

我会给你留下这个模型预测的六件事：

未来将是多链。

  1. 以太坊可能是最有价值的链，因为引用一位著名的银行劫匪的话：那是钱的所在。

  2. 其他 L1 也很有价值，但它们将继续分化。纽约、洛杉矶、芝加哥和休斯顿是经久不衰的城市，因为它们的机构和文化彼此不同。

  3. L2 很重要——摩天大楼技术对于扩展任何城市都是必不可少的——但它们并不是故事的结局。L2 是“和”，而不是“或”。

  4. 特定于应用程序的区块链将仍然是利基市场。

  5. 在现实世界中，交通几乎占住房 GDP 的一半。如果我们在加密货币中看到类似的东西，跨链桥将变得非常有价值。
