2021 年标志着 NFT 的兴起——这是多么美好的一年！在这很短的一年时间内，我们见证了艺术创造力、技术创新和资金流入等在该空间的爆炸式增长。

我很高兴看到许多开发人员、创新者和艺术家将这项技术作为他们创作的新媒介进行试验。每个以有趣的新方式利用 NFT 的项目都让我不寒而栗。

导致这种高速增长的一个关键因素是NFT 使全球文化的金融化成为可能。文化是我们日常生活的体现。NFT对于普通人的参与门槛比较低，不像DeFi需要最低的技术和金融知识。购买一个axis去玩一款类似于《Pokemon》的纸牌战斗游戏比购买并将USDC存入Curve池去创造治理代币要直观100倍。

越来越多的主流名人如 Jay-Z、Snoop Dogg、Stephen Curry、Paris Hilton、Steve Aoki 将 NFT 作为他们的数字身份并点燃了数百万粉丝的效仿欲望，进一步推动了这个趋势。 

Steve Aoki 喜欢他的 CryptoPunks

NFT 狂热

在过去的几个月里，我们见证了很多投机行为，一些 NFT 价格在几周内上涨了 10 到 100 倍并暴跌了 90%。

NFT 价格的巨大波动以及难以估值的原因是：  

  1. NFT 的流动性相对较差。发生在边际上的交易非常迅速地改变了地板价 。

  2. 由于价格历史有限，它处于不断的价格发现中 。

  3. NFT 的许多内在价值（例如文化）是无形的，难以量化。

  4. 没有采用一致的估值框架。

  5. NFT 是高度叙事驱动的。（最重要的一点，后面我会详细展开）

我购买的第一个 NFT 是 2021 年 1 月的 Non-Fungible Ape，它一种将各种角色比喻成“猿”的像素艺术。在他们做第一次隐形空投时我偶然发现了discord，但是我不知道自己当时在做什么。浏览 OpenSea 上的每一个图片，它在我心中点燃了购买和收集具有最稀有特征的猿类的强烈愿望。如今它们的价值是 0：没有了买家。这个经理给我了深刻的教训。

图表 A：Donald Trump Non-Fungible Ape，一种个价值0.7E 的稀有猿

从事后看来，我应该预料到这一点，因为市场完全靠投机维持：它没有历史价值，刚加入推特的匿名艺术家，小社区，零效用，平庸的艺术。它试图在我稍后将概述的 5 个 NFT 领域中的任何一个领域中创建叙述，但未能成功。

然而，其他 NFT 的价格已经飙升至令人眼花缭乱的高度。2017 年仅以 35 美元购买的CryptoPunk今天可能价值高达 950 万美元。

看看 Board Ape Yacht Club、Fidenzas、XCOPY、mystic axies 等等。是什么让它们现在如此有价值，我们应该如何早期发现这些NFT的价值机会？

为什么NFT 是不同的？

我对 NFT 很着迷，因为它们本质上代表了可替代代币所没有的独特复杂层。这一层包括无形的（例如文化）和有形的，使它成为一个有趣的智力练习。

在最高级别，NFT 存在并与所有其他加密应用程序共享相同的基础层即区块链基础设施，以太坊是最常用的区块链。这包括特定于 NFT 应用的应用链和 L2 解决方案，例如 Flow 和Immutable X。

下一层——平台层——为 NFT 提供金融服务：想想交易所、流动性提供、贷款、分权化等。这些是平台，因为它们通常服务于多种不同的代币，而不是单一的代币。

与可替代代币不同，NFT 是另一层，其中一个单独的“应用程序”被打包到代币本身中。NFT 标准（通常为 ERC-721）使代币能够通过其元数据链接到链上或链下资产，同时保留区块链的出处、不变性和网络安全性。

让我用一个 Web2 类比进行简化：

（第一级）底层区块链=互联网+手机。

例如：TCP/IP，iPhone。广泛且公开可用

（第 2 级）应用程序/基础设施 = 互联网平台。

例如：苹果的应用商店

（第 3 级）NFT = 个人移动应用程序。

例如：脸书应用

平台非常强大且并通过其网络效应和广阔的市场覆盖面获取巨大价值。好的平台需要花费大量时间来构建、发展并在采用曲线上达到高速增长。单个应用程序通常代表具有最高水平的多样性、创新性和创造性投入的层——因为现在任何有想法的人都可以相当容易地创建和启动（基本）移动应用程序。这里的 web3 等价物是 NFT。

为了进一步说明这一点，让我们考虑 Aave（平台）和 CryptoPunks（NFT）：

1AAVE 的价值与整个 Aave 应用程序/协议的成功息息相关：锁定的总价值主要包括不同的 LP 代币、产生的收入、用户增长。 

1CryptoPunk 的价值是……它看起来有多酷，有多稀有以及有多少人真正想要它。不依赖于任何外部应用程序。

NFT 的简单的本体论框架

所以我开始探索一个思考 NFT 的框架：

  1. 您如何识别具有可能传播开来的叙述的 NFT？

  2. 您如何确定 NFT 是被低估还是被高估？

这样做的目的不是量化 NFT 的特定公允价值——NFT没有贴现现金流模型。相反，我为 NFT 项目创建者和 NFT 投资者制定了一个动态框架，以便他们做出更好的决策。

我深入剖析了赋予 NFT 价值的因素，并将其提炼为 5 个主要领域。这与我们都熟悉的 NFT 的典型分类不同：收藏品、生成艺术、虚拟土地、游戏等。原因是这些类别之间经常有很大的重叠。这 5 个领域适用于所有 NFT，尽管它们的相对重要性因项目而异。

与可替代的同类产品相比，NFT 具有文化和美学优势。从无形到有形的范围内，这 5 个领域是：

文化（无形）

  1. 这是 NFT 中最有趣的部分，也是最难估值的部分。 

  2. 文化是广泛的，包括一个社会的共同经验、看法和信仰。 

  3. 文化成分包括但不限于：

品牌价值

历史意义

模仿欲望：我们渴望别人的欲望，因为我们模仿他们的欲望

弹性价值：我是否被视为展示财富、文化和阶级？

美学（无形）

  1. 指与 NFT 相关的媒体（例如艺术品、音乐）的审美吸引力

  2. 你喜欢它？有多少其他人，尤其是时尚达人，喜欢它？

  3. 所有权是否表明您对美有很好的洞察力？ 

社区（有形-无形）

  1. 指 NFT 集合的现有所有者的总和

  2. 与 Web3 中的大多数事物一样，社区就是一切。

  3. 业主的人口统计资料、心理、财富和愿景是什么？他们更可能是钻石手还是纸手？它们是响亮而粗鲁，还是安静而复杂？

  4. 可衡量的指标：社区是增长、停滞还是衰退？有多少贡献者？

实用（有形）

  1. 指直接从拥有 NFT 获得的利益，例如私人会员团体、早期获得铸币厂、代币种植、实物商品

  2. 游戏和 Metaverse NFT 的主要组成部分：例如 Axies、Sandbox Land、Star Atlas 宇宙飞船、Ember 剑徽章。在玩赚钱游戏中，这些也可以产生收益。

  3. NFTfi（NFT + DeFi）非常有趣。

资产（有形）

  1. 指除 NFT 本身之外，NFT 所有者有权拥有的数字或现实世界资产

  2. 例如：项目金库、其他 NFT、知识产权、房地产

在接下来的文章中，我将通过示例更深入地探索这 5 个领域中的每一个。

简而言之，NFT = 文化 + 美学 + 社区 + 资产 + 效用

(NFT = Culture + Aesthetics + Community + Assets + Utility，CACAU)

这是一个求和 (+) 方程：这意味着所有打算最大化其价值并成为可投资资产的 NFT 项目都应该考虑它们在这 5 个领域中的每一个领域的表现。即使他们可能只重点聚焦其中一个领域。

例子：

一个具有创造能力的生成艺术家，他的天赋是创造出世界需要的精彩艺术品的算法，还应该考虑：  

他/她如何与收藏家互动并建立一个积极的社区，而不是完全消失“为我的艺术工作”

他/她如何通过会员资格、早期访问新作品等增加拥有艺术品的效用。

尽管美学是这里的主要领域。美学可能很难判断，但社区和效用更容易衡量。

笔记

与区块链相关的几个因素未在此框架中捕获，但可能有助于 NFT 的叙述和价值：

NFT 的基础链

以太坊目前是 NFT 的首选网络，尤其是高价值 NFT。这部分是由于网络上的安全保证。

以太坊之外的链上或L2的Rollup上的 NFT（例如 Polygon、Solana、Avax、Arbitrum）在今天的市场中被大幅打折，尽管未来可能不会如此。

基础数据的持久性

Arweave > IFPS > 中心化网站

(Gen Art) 链上 vs 链下

完全存储在链上的艺术品似乎比存储在链下位置的艺术品更具优势。区块空间非常昂贵，这使得链上存储通常不切实际。

例如 Autoglyphs 作为最古老的链上生成艺术项目。

衍生品与原始品

成功的项目将被多次克隆和分叉，这是加密法则。正如我们多次看到的那样，由于投机，衍生品的价格在短期内可能会急剧上涨，但从长期来看，价值通常会恢复到原来的水平。

例如 Cryptopunks ← Phunks、Fast Food Punks、Picasso Punks、Bastard Gan Punks……

总结

为什么要为此烦恼？NFT 不应该很简单：只要我喜欢就买？是的，这也有效。

我将这个框架放在一起，以帮助我更好地理解和系统地识别可以推动 NFT 价值的因素。并避免在叙事薄弱或临时性的项目上出现 FOMO。我希望它也对你有用。

我认为，要让 NFT 作为一种真正可投资的资产成为主流，我们需要开始与传统投资世界中的人们一样思考和使用相同的语言。非crypto native可以理解并揭开加密技术和概念的神秘面纱的框架。

类似于“价值投资”的方法是：

  1. 在 NFT 的背景下，文化是什么？

  2. 我们如何确定需要关注的实质性文化叙事？

  3. 我们可以为文化的不同组成部分赋予价值吗？

  4. 艺术家（原艺术家、摄影师、音乐家等）如何为他/她的作品定价？

  5. CryptoPunks vs Bored Apes：案例研究。
