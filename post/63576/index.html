就在刚刚，在拥有微软支持和大量公司员工支持情况下，ChatGPT 之父 Sam Altman 最终还是没能夺回 OpenAI 的控制权，正式“出局”。

这对微软来说是一个极其糟糕的局面。谁能想到就在微软凭借 AI 领域的一系列成果实现历史最高的市值时，却发生了如此难以想象的事情。曾经一片大好的光明前景，好像一夜崩溃。受 OpenAI 内斗的影响，微软股价当天就出现了大跌。

尽管微软明确支持 Altman 回归并重新担任 CEO，但 OpenAI 董事会最终拒绝了自己的最大金主，还是选择了一位新的 CEO——Twitch的联合创始人Emmett Shear。

最符合微软利益的 Altman 出局了，公司被不听话的董事会完全掌握，OpenAI 与微软的关系似乎正急转直下。如果微软今天宣布不再为 OpenAI 提供资金，那么我不会感到惊讶。

在 Altman 正式在 OpenAI 出局之后不久，微软董事长兼 CEO 萨蒂亚·纳德拉 (Satya Nadella) 迅速宣布 Altman 与一起离职的 OpenAI 前总裁加入该公司并将领导其新的 AI 团队。

“我们仍然致力于与 OpenAI 的合作伙伴关系，并对我们的产品路线图、我们在 Microsoft Ignite 上宣布的一切继续创新的能力以及继续为我们的客户和合作伙伴提供支持的能力充满信心。 我们期待了解 Emmett Shear 和 OAI 的新领导团队并与他们合作。 我们非常高兴地分享这样的消息：Sam Altman 和 Greg Brockman 及其同事将加入 Microsoft，领导一个新的高级人工智能研究团队。 我们期待迅速采取行动，为他们提供成功所需的资源。”

Sam Altman 转发并回复：”使命继续。“

微软早已最好最坏打算

就在 OpenAI 内斗爆发的前一天，微软董事长兼 CEO 萨蒂亚·纳德拉 (Satya Nadella) 谈到了与 OpenAI 的关系。他说，如果与 OpenAI 之间数十亿美元的关系破裂，微软可以自力更生。

就在一天后，一场震惊整个科技界的斗争公开爆发了。这让微软原本火热的 AI 进程突增变数。

事实上，微软已经与 OpenAI 变得如此“相互依赖”，它与这家 ChatGPT 背后的公司的关系现在可以比作微软与英特尔的“Wintel”合作关系。 纳德拉表示，正是这种关系造就了微软的今天，但这并不意味着这家软件巨头缺乏自己的 AI 软件可以依靠。

OpenAI 拥有着行业内最奇怪的董事会结构，微软作为最大金主（据信拥有 49% 的OpenAI股份）却在董事会没有任何职位，也无法对董事会施加压力，更重要的是董事会成员大部分不在 OpenAI 任职，且不持有 OpenAI 股份。当 OpenAI 的这场内斗爆发时，微软似乎只能旁观。

微软的 AI 布局恐被打乱

就在上周，微软发布了一系列新的 AI 战略，对多达上百项业务进行 AI 改革，全面进入 Copilot 时代，All in AI。纳德拉当时表示，最终将看到 OpenAI 的人工智能模型 GPT 嵌入到每个“Copilot”产品中。该公司还宣布正在构建一款名为 Maia 的人工智能芯片，该芯片将用于在运行微软 Azure 云服务的数据中心内运行 OpenAI 的模型以及其他机器学习模型。

纳德拉上周告诉 Financial Review 说，“没有我们的系统，OpenAI 不可能成功构建 ChatGPT。我们的系统为它们提供动力，我们在它们之上构建，它们也在我们之上构建。 因此，所有这一切都存在很大的相互依赖性。”

不过，他表示，如果与 OpenAI 关系出现问题，这家软件巨头将能够自行解决问题。“在这种情况下，这很容易。 就像你总是知道如何用两只脚走路。”

微软已经在研究另一种被称为“graphormer”的机器学习技术，以及自己的大型语言模型“Phi”，因此“我们有很多能力，如果需要，我们将继续前进” 。

大量员工追随 Altman，OpenAI 900 亿美元市值将灰飞烟灭？

知情人士透露，就在 OpenAI 董事会董事兼首席科学家Ilya Sutskever称已被罢免的Sam Altman将不会回归不久，数十名OpenAI员工在公司内部宣布辞职，此前不久已经有三名高级研究人员宣布辞职。有OpenAI员工表示，如果Altman创办一家新的AI企业，他们将有兴趣加入。

根据纳德拉所说，与 Altman 和 Greg Brockman 一起加入微软新 AI 团队的还有他们的其他同事，这意味着大量 OpenAI 人员将离开，前不久估值超过 900 亿美元的 OpenAI 恐怕价值要打折扣了。尽管纳德拉表示会继续支持 OpenAI 新的领导团队，但如果 OpenAI 无法继续为微软创作利润，那么毫无疑问，距离他们被放弃不会太遥远。
