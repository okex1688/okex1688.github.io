周末两天，圈内都在关注一个项目——Moonbirds。

4月16日以 2.5ETH 的价格正式公开发售，共7875枚，几小时内售罄，按照 ETH 美元价格计算，本次销售募集到的资金约为 6600 万美元，据悉，100% 的收益直接捐给True Ventures支持的 web3 媒体公司PROOF Holdings Inc；

4月17日，24 小时交易量已超过 1.1 亿美元，同时地板价逐渐攀升，截至4月18日11时为19.69ETH；

4月18日，Moonbirds #3904 以 150 ETH 价格成交，约合 45.7 万 美元，创该 NFT 系列交易额最高记录。

诚然，NFT市场高度内卷，但Moonbirds 似乎轻松赢得了市场的关注。

Moonbirds 是由播客节目 PROOF（PROOF是由 True Ventures 合伙人 Kevin Rose 创办，是一家聚焦于 NFT 领域的高质量播客）推出的一组以鸟类为基础形象的 NFT 头像系列，共计10,000枚。在这 10,000 个中，有 2,000 个通过免费铸币提供给 PROOF Collective NFT 持有者，125 个由 Moonbirds 团队分发，其余 7,875 个由通过公开抽奖获得许可名单的人以每人 2.5 ETH 的价格铸造。

Moonbirds 创建在以太坊网络上，每只鸟都有不同的特征，表明它们的稀有性。它最初是作为荷兰式拍卖从 2.5 ETH 开始推出的，但由于需求旺盛，团队选择了抽奖和允许名单销售（允许名单通常用于防止机器人攻击需求下降）。NFT 千千万，Moonbirds 为何如此受欢迎？

据悉，Moonbirds 承诺比许多其他 NFT 项目更多的东西，这就解释了为什么尽管Mint价格很高，但需求如此强劲。Moonbirds 表面上类似于可爱的小鸟角色，但实际上它们是 PROOF 的**“实用 NFT”** 。

PROOF 在推出 PROOF Collective 时取得了巨大成功，这是一个只有 1,000 名 NFT 收藏家和艺术家的私人会员俱乐部。PROOF Collective 的第一款名为 Grails 的产品展示了 Larva Labs 和 Tyler Hobbs 等公司的超稀有作品，以及其在二级市场上的地板价达到100 ETH。该权限类 NFT持有者拥有着访问 PROOF 私人 Discord、提前收听最新播客、参加线下活动、等特殊权限。在往期节目中，PROOF 还曾邀请过 Beeple、GaryVee、John Crain（SuperRare 首席执行官）、Gabby Dizon（YGG 联合创始人）、 Alexis Ohanian（Reddit 联合创始人）、Tyler Hobbs（Fidenza 创作者）等诸多 NFT 领域最知名的艺术家、从业者以及其他各行各业的大咖就 NFT 领域最前沿的话题展开探讨，其内容在业内有着相当大的影响力。

根据其CEO Kevin Rose的叙述以及官方内容显示，Moonbirds 的持有者大致会拥有以下三种权限：其一，Moonbirds 持有者可以解锁一些私人俱乐部的会员资格，包括进入 PROOF 的私人 Discord 参加闭门讨论会等等。其二，Moonbirds 持有者将可以获得 PROOF 接下来将要推出的元宇宙项目Project Highrise的先行访问资格。其三，Moonbirds 本身也会有一些特殊玩法，持有者可以让自己的 Moonbird 进入“筑巢（Nesting）”状态。

官方表示，Nesting将在拍卖完成后几天启动。官方将计算当前筑巢周期的长度和累积筑巢时间。两者都会有不同的奖励，PROOF 官方尚未宣布筑巢奖励的具体规则。进入筑巢的锁定状态后，Moonbirds 将无法正常通过 OpenSea 等二级市场交易（会在买方下单时提示），但依然可以在 PROOF 所提供的网站内通过亲手签名的形式转移至其他地址，这一设置的目的是为了防止一些常见的远程函数调用钓鱼手段。

另外，PROOF 将获得所有二级销售额的 5%。这些收益将用于继续雇用更多的全职员工、资助运营并为社区创造更多价值。
