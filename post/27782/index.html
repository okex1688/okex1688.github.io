曾经有人告诉我非同质化代币（NFT）最大的问题是缺乏流动性，我的回答是「DeFi 会解决这个问题」。

确实如此，而今天我们将探讨如何使用碎片化工具来将 NFT 拆分成一堆碎片（通常是以 ERC20 代币的形式），然后在 DeFi 中使用它们。

在 Uniswap 上交易 NFT 的碎片？当然没有问题。

将它们放到 Aave 的货币市场？也可以（尚未发生，但它即将会发生）。

碎片化意味着我们可以通过有效的定价在 NFT 上创造流动性。

名为 Fractional 的协议是解决这个利基市场的新协议之一，其团队刚刚拿到了 Paradigm 领投的 790 万美元种子轮融资。

  • 技能：中级

  • 努力：至少要花费 30 分钟来学习 Fractional 的基础知识

这里的内容都不构成财务建议。 碎片化 NFT 可能涉及法律问题，具体取决于你所在的司法管辖区，包括美国。 请咨询律师并自行承担使用风险！

通过 Fractional 解锁 NFT 的流动性

随着 DeFi 和 NFT 之间的交叉点继续蓬勃发展，NFT 碎片化协议正在兴起。它们促进了 NFT 的流动性并改善了价格发现，并通过集体所有权实现 NFT 访问的民主化。

它的想法？持有者可以使用一个碎片化协议将一个理想但流动性差的 NFT 拆分铸造成诸多 ERC20 代币。这些 ERC20 代币是可互换（因此很容易交易）和可分割的，这意味着它们共同代表基础资产，并且可以在多个所有者之间分配。

迄今为止，我们已经看到了一些 NFT 碎片化协议的兴起，例如 NIFTEX 和 Unicly。

而 Fractional 是最新出现在以太坊主网上的 NFT 碎片化协议，它于 2021 年 7 月推出，并在短期内成为解锁 NFT 流动性的流行协议。

Fractional 的简要指南

Fractional 协议的核心有两种主要类型的利益相关者：

  1. 买家：购买碎片化 NFT 以接触这些文化资产的用户；

  2. 策展人：通过协议将 NFT 铸造成碎片化的 ERC20 代币，并赚取策展人费用的用户；

为了促进这些过程，Fractional 围绕着三个主要的智能合约：一个用于单个碎片化 NFT 的金库，另一个用于多个碎片化 NFT 的金库，最后一个用于促进金库治理参数。

此外，请记住，由于这些智能合约是开放且无需许可的，因此任何人都可以构建自己的 Fractional 前端或使用 Web3 钱包直接与协议交互，也就是说，用户无需访问 fractional.art 网站也可以使用该协议。

这就是以太坊的魅力所在。

如何碎片化 NFT

图：Fractional 的碎片化界面

如果你有兴趣通过 Fractional 拆分一个 NFT，你的第一步是确定要使用哪个 NFT。

一旦你确认了目标，整个过程就是这样的：

  1. 前往 fractional.art，连接你的 Web3 钱包，然后单击网站顶部的「Fractionalize」按钮。

  2. 你将进入一个界面，你可以在其中选择要拆分的 NFT，然后输入你的金库名称、代币供应量、代币符号、底价以及管理费。

  3. 按「Continue」继续，此时系统会提示你批准 Fractional 转移相关的 NFT。确认钱包中的批准，然后根据需要进行转账。最后，按「Fractionalize」按钮并确认最终的铸造步骤。（请记住，每个 NFT 都需要经过批准+单独转移的过程，然后才能铸造 ERC20 代币）

  4. 一旦你的金库上线，你就可以使用「Settings & Actions」仪表板来编辑其描述、更新其拍卖参数等。

设定金库的底价

面对未来潜在的购买，Fractional 的金库依赖于一个价格系统，在该系统中，NFT 碎片持有者持续对底价进行投票，而这次投票的加权平均值决定了发起收购所需的价格。

这听起来可能有点令人困惑，而在最近的一篇博客文章中，Fractional 团队提供了两个非常简单的示例来说明这个规则。

示例如下：

「如果你拥有 100% 的所有权代币，并将底价设置为 100 ETH，那么底价就是 100 ETH。但如果你拥有 75% 的代币，而拥有另外 25% 投票权的人以 50ETH 的底价进行了投票，则最后的底价结果将是 81.25 ETH。

如果你只拥有代币供应量的 49% 并就底价投票，但其他拥有剩余 51% 代币的人没有对底价进行投票，则不会为金库设置底价。
」

也就是说，当个人或团体存入足够的 ETH 以达到或超过金库确定的底价时，Fractional 收购就启动了。 然后就开始拍卖过程，如果这个收购成功，则该金库的 NFT 碎片持有者可以用他们的 ERC20 代币兑换成 ETH。

搭配使用 PartyBid

我们认为，Fractional 是处于 DeFi 和 NFT 十字路口的新文化乐高积木。这种动态开辟了各种新颖的 Web3 可能性，例如 PartyDAO 最近发布的 PartyBid 工具。

PartyBid 是一个去中心化的应用程序，它用于促进对 NFT 的集体竞标。这个应用允许一群朋友和 / 或陌生人一起对选择的 NFT 进行突袭（游戏意义上的「突袭」），这在 NFT 生态系统中首次给普通投资者带来了优势。

长话短说，PartyBid 帮助团体通过 Foundation 或 Zora 拍卖获得 NFT，然后通过 Fractional 对竞拍品进行无需信任的分割，然后将随后的 ERC20 代币按比例分配给派对参与者。

这个 Dapp 已经被投资者用于收购 CryptoPunks、Nouns 以及其他主要的 NFT，而这一切都归功于 Fractional 的开放基础设施。

提醒

Fractional 正在为新的 NFT 流动性和代币效用可能性铺平道路，而上述 PartyBid 的例子只是对未来可能会出现在 Fractional 协议周围的创新的一种尝试。

同时，作为 DeFi+NFT 的粉丝，我个人很痛苦地说，Fractional 并不是每个人现在都应该去尝试的协议。这是因为美国等一些司法管辖区会将碎片化的艺术品视为证券。

换句话说，在这些司法管辖区内，碎片化的 NFT 似乎属于证券。在发行碎片化的 NFT 之前，请确保你了解自己所在司法管辖区的相关法律。

最后，值得注意的是，Fractional 团队此前曾表示，他们计划逐步去中心化该协议，以确保协议长期的弹性和可升级性。因此，未来 Fractional 完全有可能发行治理 token，你可以现在使用它，并累积潜在的追溯性空投奖励。
