“在一个充斥着无关信息的世界里，清晰就是力量。”——尤瓦尔·诺亚·哈拉利

记得曾经一部老电影中的一个场景。我忘记电影叫什么名字了，但那个场景一直刻在我的记忆中。讲的是一位顶级华尔街交易员的老司机透露，他一直在根据偷听到的后座谈话进行复制交易。最后他不再需要这份工作了。因为他已是百万富翁。

信息想要自由

交易者可以凭借第一手信息的优势直接使自己受益，但从大局来看，这可以创造一个更有效的市场。简而言之，信息想要自由。传递信息的基础设施越开放，知识的传递就越快、越公平。非法窃听政府新闻稿进行交易、将金融交易所的服务器托管到前端零售，以及分析师根据停在麦当劳的汽车的卫星图像预测季度收益，这都是交易者寻找智慧优势——新知识的例子。

信任需要去赢取

加密和区块链的出现加快了信息传输的速度。“成为你自己的银行”的个人隐私来自于所有交易被公开广播的权衡。这颠倒了我们所知道的一切。交易不是由有司机驾驶的豪华轿车背后的阴暗管理人员进行的，数字也不会隐藏在受欺诈评级机构保护的资产负债表后面——一切都在网上。我们称之为“链上”。在这个新经济中，信任是通过验证获得的，而不是隐藏在机构和凭证之后。

透明度为王

加密提供的隐私是有代价的。透明度。区块链上的透明交易意味着每个人都是倾听交易的驱动力。我们可以倾听世界上发生的每一笔交易，并窥探每个钱包和银行账户。但是，这对普通人来说也是可以做到的，真的吗？作为一名研究人员，我对这个新的开放经济中编码在链上的人类行为着迷。但是在实践中倾听区块链是很困难的，因此在大数据分析中，让清晰的信号浮出水面成为了一项复杂的工作。这是我第一次偶然发现Nansen的地方。

未知领域

Nansen 的团队在构建将代码转换为有意义的交易的开源工具方面发挥了重要作用。你可以查看由Nansen创始人之一 Evgeny Medvedev 开发的以太坊 ETL‌。Nansen 继续以专有的商业模式完善这些开源工具，从而获得所需的资金，以便更深入地挖掘这些未知领域。

名字很重要

有了标签，数据才会真实。Nansen 的标签是对各个区块链钱包的描述性和行为性命名。他们不会人肉搜索或透露这些钱包的私人身份。他们做了一些更有趣的事情：他们 doxx 的行为。Nansen 拥有超过 1 亿个带有标签的钱包，这些标签可归因于用户行为的一个方面。

Nansen价值创造的核心来自于看似被简单命名为“Attribution”的团队。在这个团队中，您会发现世界上一些最聪明的人不断提出并改进“启发式（heuristics）”以跟踪行为趋势。标签范围从简单的属性（如“代币百万富翁”）到更复杂的属性（如“聪明的流动性提供者”）。

地址“0xd387a6e4e84a6c86bd90c158c6028a58cc8ac459”的标签是被称为有影响力的 NFT 投资者“Pranksy”。

聪明的钱（smart money）很重要

Nansen标签的最顶部是聪明的钱（smart money）。这些是加密世界的精英所拥有的钱包。这些钱包在做出精明的投资决策和领先一步方面有着悠久的历史。Nansen 允许用户设置高度定制的智能警报，以便在这些钱包进行新的交易和投资时实时通知他们。

智能警报允许用户以高度可定制的方式来跟踪加密活动

 “聪明的钱”（smart money）释放信号

单独跟踪smart money钱包是了解成功投资者采用的交易策略的有力工具。Nansen 的新监视列表一直在不断更新，以允许用户轻松制作和跟踪个人制作的钱包列表。但是，当我们将smart money视为一组钱包时，它们提供了一个更基本的市场概览。以 Nansen 的smart money flow为例，它跟踪跨加密代币的smart money存款和取款的总流动。

自定义监视列表允许手动策划的钱包列表及其最新活动。

今年早些时候，当smart money开始投资柴犬币时，很多人问为什么Nansen显示的smart money正在投资一个看似毫无意义的模因币。几个月后，柴犬币已经位居顶级加密货币之列。smart money的流动提供了资金进入和退出信号，从而可以发现新投资并重新评估现有投资。

Nansen主页利用smart money标签跟踪重要的资金流动。

DeFi 和聪明的流动性提供商

加密创新的快速步伐使得许多创新让外界无法触及。对于那些涉足加密货币的人来说，被称为去中心化金融 (DeFi) 的金融创新彻底改变了我们可以用金钱做的事情。奖励提供流动性的智能合约，为通常更普遍地称为可编程货币的货币提供智能功能。智能手机解锁了新型应用程序，而智能合约解锁了金钱。

Nansen 的“聪明的流动性提供商标签（smart liquidity provider label）”通过结合精确的定价来源和先进的系统来评估模糊价值（如无常损失），从而吸引了最熟练的 DeFi 用户。在此处阅读有关这些更新的更多信息。

NFT 非常适合smart money

最近 NFT 的繁荣使钱包标签比以往任何时候都更加相关。NFT 是密切的社交市场。当钱包‘0xd387a6e4e84a6c86bd90c158c6028a58cc8ac459’（被打上‘Pranksy’标签）进行交易时，社区会很快跟进并进行交易。Nansen很早就发现smart money通常只在他们最擅长的方面聪明。DeFi 流动性提供者则通常对购买哪个 NFT 一无所知。

尤其是，NFT 市场的社会层往往使其不受外界影响。有些NFT 经常以数百万美元的价格转手。对 Bored Ape Yacht Club (BAYC) 猿猴皮毛的纹理稍作调整，对CryptoPunk 中以不同的方式排列几个像素，这都可以大幅度增加或减少这些NFT的价值。

为体现NFT市场的深度和成熟度，Nansen将其smart money进一步细化为细分市场。除了流行的“Legendary NFT Collector”标签等现有标签外，Nansen 现在提供改进的 NFT smart money标签。这些新细分市场反映了构成 NFT 市场的微妙之处和子社区。

Smart NFT 早期采用者

让我们从早起的鸟儿开始。这些是 OG（Original Gangsters），他们在项目火爆起来之前就购买了最好的项目。然而，不仅仅是早起的鸟儿，他们还是那些没有在第一时间出售自己的 NFT 的人——尽管有令人眼花缭乱的投资回报的诱惑，但他们始终坚持持有。这些先行者是 NFT 社区中真正的做市商、长期投资者和具有影响力的人。

Smart NFT 交易者

接下来讲交易者。与早鸟儿者和经验丰富的收藏家不同，NFT 交易者经常预订他们的利润。他们的 ETH 净余额为正。他们做出精明投资决策已经得到了证明。他们控制gas成本并重新平衡投资组合，以确保它们具有足够的流动性以进行再投资。他们采取战略来获取有价值的 NFT 并最大限度地降低风险。使用 Nansen 研究这些策略就像是获得加密界的哈弗大学学位，而成本只是学业内容的一小部分。

Smart NFT铸造者

从合约或网站中铸造 NFT 收藏品并不容易。NFT 项目之间存在安全风险、抢跑、gas 成本以及各种各样的选择。除了铸造方面的难题，项目寿命的短暂性使得选择合适的退出时间非常关键。我们的Smart NFT铸造者是最好的。我们计算了过去 60 天内跨多个项目最赚钱的钱包。

Smart NFT 持有者

对你和我来说，Hodlers 的构建方式不同。早在 2013 年，当比特币价格低于 1,000 美元并且经常上涨和暴跌超过 30% 时，HODL 一词似乎是一个喝醉酒后拼错的字，意思是催促人们持有而不是出售。通过这次事件，HODL 心态诞生了。尽管损失惨重，但 Hodlers 是最强大的手，他们可以继续赢得 NFT。Nansen 开发了一种“秘方”来寻找 NFT 持有者，与影响力者和交易者不同，他们在低谷和高点之间持有大量 NFT。#hodlgang

总结

Nansen Smart Money不仅仅是投资者和交易者的阿尔法工具。这些标签代表了一种新型经济，其中知识是开放的，强大的参与者无法隐藏他们的行为。我们的金融未来不再掌握在我们被迫信任的机构手中。相反，所有行动都是公开和公开广播以供验证。信任不是因缺乏替代品而盲目承担的风险，而是不断验证和赚取的宝贵资产。

这种新经济使信息在网络中的传播速度更快，从而造福于所有人。为了使加密变得可获取，像 Nansen 这样的工具必须继续推动前沿，将数据转化为有意义的信号。在加密货币中，追随智能货币比以往任何时候都更重要。
