概要

在对 BAYC 土地——Otherside NFT 的旺盛需求中，Yuga Labs 提出 ApeCoin 将需要自己的区块链、第二层网络/Subnet 或其他侧链解决方案。

Messari 分析师研究了 Yuga Labs 推出的扩展解决方案的后果，以及它对 BAYC 生态系统内其他资产的价值累积的影响。

BAYC 的土地 NFT（Otherside）销售是近期历史上最严重的销毁 ETH 的行为之一，也是无聊猿 NFT 的一个不朽的成就。从需求角度来看——考虑到用户对 BAYC 土地 Otherside NFT 的旺盛需求，Yuga Labs 提出 ApeCoin 将需要自己的区块链、第二层网络/Subnet 或其他侧链解决方案。虽然 Yuga Labs 没有提供任何其他明确的信息，但他们已经开始鼓励 DAO 开始朝这个方向思考。

随着 Otherside NFT 的推出，BAYC 品牌的总价值达到了约为 250 亿美元，或约占迪士尼市值的 12%。

关于 BAYC 的分析

这产生了一个重要的问题清单，其中一些问题是 Messari 分析师心目中的首要问题：

  1. 你对 BAYC 宣布自己的区块链有什么看法？

  2. 这对 BAYC 生态圈的价值有什么影响？

  3. 围绕 BAYC 战略的其他想法或意见？

由于不缺乏意见，Messari 里的分析师们根据这些问题分别对 BAYC 的生态系统进行了评估。

Mason Nystrom：是迪士尼化还是抢夺现金？

NFT 的迪斯尼化正在实时发生，但与迪斯尼不同的是，其基础并不是建立在爱情故事上。

在沃尔特 - 迪斯尼的早期职业生涯中，他的动画工作室一直在财政崩溃的边缘徘徊，因为该工作室相对成功的动画片都是赔钱货。在 20 世纪 30 年代中期，动画片价格昂贵，电影业的需求受到大萧条带来的长期失业的严重影响。可以说，新成立的动画工作室的前景很暗淡，也就是说，直到《白雪公主》的发行才催化了我们今天所知的迪斯尼帝国。

NFT 正处于类似的位置，早期的品牌如 Axie Infinity 和 BAYC 试图在 NFT 系列盈利的基础上建立经济帝国。Sky Mavis（Axie Infinity 的开发者）有效地将 AXS 代币与 Axie 世界隔离开来，推出了 Ronin 侧链并伴随着自己的原生资产 RON。虽然开发 Sky Mavis 仍有即将推出的 Axie，如自己的数字土地销售，但 Sky Mavis 显然是想通过创建垂直化游戏基础设施来加强其商业模式。另一个著名的 NFT 游戏开发商 Immutable Labs 采取了同样的方法，为其旗舰交易卡游戏 Gods Unchained 建立了自己的第二层网络（IMX），同时也为 Gods Unchained 的生态系统推出了 GODS 代币。

AXS 在 Axie Infinity 的世界里有其确定的功能，而 ApeCoin 没有同样的限制，推出一些新的链（ApeChain？）与自己的代币可能会出现机会主义，即使 Yuga 只是采用其他竞争对手的玩法。

由于 ApeCoin 的效用有限，新链的推出提供了为 ApeCoin 赋予更多价值的机会，尽管 Yuga Labs 更有可能为他们的扩展解决方案推出另一种原生代币（鉴于这是迄今为止所有扩展解决方案所做的）。

我们不去争论这是否是一个必要的转变（有优点也有缺点），这是一个将有意义地影响 BAYC 生态系统内有价值的资产累积的转变。更具体地说，鉴于 Yuga Labs 最近与 Animoca 和 a16 z 等合作伙伴进行的 40 亿美元融资是基于股权而不是代币的，竞争的激励措施可能会开始渗透到 Yuba Labs 团队和其机构投资者。

BAYC 已经很好地驾驭了发行的蓝图，它发布了狗伴侣、变异猿游艇俱乐部，以及现在的 BAYC 土地，从财务角度来看，这些都表现得相当不错。但这些收藏品的长期价值仍然是基于它们的奢侈品地位和潜在的空投奖励。甚至稀缺的数字土地的商业可行性也还没有得到证实。

如果 Yuga 想创造 Web3 的迪斯尼，他们就需要资本和可持续产生现金流的知识产权。 Yuga 显然有资本，虽然 BAYC 的交易费和 NFT 发布创造了大量的回报，但它们不一定是经常性的或可持续的。Yuga Labs 和无聊猿的未来将取决于创造一个消费者愿意付费的平台或建立在有价值的知识产权上的业务的能力。

Chase Devens：有的时候少即是多

过去半个世纪不乏雄心勃勃的公司，其早期的成功使他们相信自己可以（而且应该）继续以光速发展。在短短一年多的时间里，BAYC 已经渗透到了全球的思想领域，并将自己嵌入到加密货币的文化基石中。其最初的产品（一个基于社区的 NFT，同时也是一个 PFP）是如此成功，以至于它的任何后续版本几乎不可能达到同样的成功水平。

为什么 BAYC 会变得如此有价值？在我看来，这是一场完美的时机，以及来自加密货币以外的高知名度人士的采用。2021 年 4 月下旬的铸币比夏季的 NFT 爆炸还要早几周。当 NFT 的价格真正起飞时，像 j1 mmyeth 和 Pranksy 这样的早期 BAYC 鲸鱼成功地向第一波名人采用者如 Steve Aoki 和 Josh Hart 推广该项目。作为一个对 NFT 知之甚少的局外人，如果你能跟着鲸鱼走，为什么还要自己做研究？

随着 2021 年更多的文化注意力转移无聊猿身上，BAYC 的网络效应加强了。像任何其他固定供应的 NFT 收藏品一样，需求的增加推动了价格越来越高。业主们开始在物理世界中组织与无聊猿同伴的聚会。Yuga Labs 可能创造了令人向往的卡通画，但社区负责推动它的价值。

当 Yuga Labs 选择发布与 BAYC 有关的系列，推出他们自己的硬币，并创建虚拟土地时，他们不知不觉地对他们的品牌定位进行了一场巨大的赌博。正如我在最近的一份报告中所详述的，今天成功的 NFT 系列看起来与奢侈品牌非常相似，并且很可能遭受同样的低端市场品牌拉伸的威胁。从本质上讲，一个成熟的品牌如果试图创造更多的产品来迎合更大的受众，实际上可能会稀释其现有资产的价值。虽然变种猿、Kennel Club 和 ApeCoin 的推出取得了巨大的成功，但围绕着 Otherside 土地推出的负面情绪可能会给原来的无聊猿造成永久性的伤害。

如果你上周问我关于无聊猿的问题，我很难找到任何证据来反对它们被认为是不朽的。然而，现在他们的墙壁上出现了一个小裂缝。这是否会导致持有人或潜在买家重新评估该品牌 250 亿美元的集体价值？

我们可以让市场来决定这个问题。现在，我有兴趣看看 Yuga Labs 的反应。我个人认为，对于品牌将他们的成功押在潜在的元宇宙游戏上，我们还为时尚早。据我所知，无聊猿通过出现在篮球场和咖啡袋上创造了 FOMO。当元宇宙还没有准备好让大众居住时，为什么人们要从物理世界的采用转向虚拟世界呢？

即使 ApeChain 成为现实，在从以太坊迁移的过程中，又有多少无聊猿持有者将会流失？无聊猿因持有者点击诈骗链接而使他们的 Ape 永远被盗而臭名昭著。这样的迁移可以说让诈骗者以临时持有人为目标。

加密货币的发展速度很快，它很容易被热潮所淹没。虽然这通常是给新进入者的建议，但所有行为者都必须记住这一点。如果我是 Yuga Labs，我会退一步，重新评估我的优先事项，以保持我一直处在王位上的席位。

Tom Dumleavy：这是一个大胆的战略，让我们看看它是否有回报

无聊猿为什么不启动自己的区块链？以太坊能给 BAYC 提供什么？BAYC 在以太坊上开始，因为当他们推出 NFT 时，那是 NFT 开始的地方。NFT 在很大程度上仍然是以太坊的故事，但无聊猿的生态系统已经超越了正常的 NFT 地位。如果 NFT 项目在今天推出，以太坊上没有什么是他们不能在 Solana、Avalanche 子网或他们自己的链上做的。

BAYC 是一个品牌，BAYC 是一个奢侈品，BAYC 肯定会被你的表弟认可，有可能被你的妈妈和奶奶认可。离开以太坊并不能改变这一点。

实际上，我们已经看到有品牌知名度的 NFT 从传统的主链中退出，开始自己的链。我想到的最成功的例子是 Dapper Labs 团队和他们创建的 Flow 区块链。Dapper Labs 团队利用 NBA、NFL、UFC 和其他他们拥有知识产权合作权的品牌，推出了他们自己的定制链，你猜怎么着，这对他们的用例来说比以太坊的效果更好。

这里有一个问题？很好，将链上的数据回滚。我们不是在解决去中心化的问题。人们在这些上面花了很多钱，部分原因是他们对领导层的未来愿景有信心。

有东西放错了地方或寄错了地址？很好，中心化的机构可以把事情解决。

现有的品牌担心知识产权侵权？没问题，Flow 已经合作，确保他们的知识产权合作伙伴不会在其区块链上被滥用。

担心你的新元宇宙的收费问题？忘了它吧，这里的规则是你定的。让我们把这些小狗放倒在地上。

对于计划中的 BAYC 的元宇宙来说，一条新链解决了许多现有的问题。

生态系统的价值累积是不变的。对于我们正在拉拢的无聊猿生态系统来说，基本上是没有边际用户的。那些想加入并能负担得起价格的少数人将寻求途径来这样做。无论谁想在无聊猿生态系统中进行交易，都需要 Ape 代币，无论代币是否在以太坊上。有了新的链，他们就不用担心 gas 费了。

突出的缺点是新链的持续开发和维护，但是，嘿，Yuga Labs 又不是买不起。

Kunal Goel：我不买他们的账（我也买不起）

虽然遵循 Dapper Labs 和 Sky Mavis 设定的道路似乎很诱人，但在我看来，2022 年推出山寨区块链已经太晚了。虽然 Yuga Labs 在其品牌和 NFT 方面取得了巨大的成功，但他们并没有表现出任何技能来表明他们可以建立一个新的和更好的区块链。相反，在 Otherdeeds NFT 的推出中缺乏 gas 优化是一个负面的指标。也许像 Avalanche 子网或波卡平行连这样的中间解决方案可能效果最好。尽管如此，他们不太可能选择它，因为 BAYC 也许和这些备用平台一样是一个大品牌。与数据库资产相比，公链上的 NFT 的好处是它们是独立的安全和可组合的；启动一个新的链将对这些属性产生负面影响。

独家 NFT 有一个内置的飞轮机制。它们是维布伦的商品，随着价格的增加而变得更加可取和排他。Otherdeed NFT 的推出不仅通过增加 2.5 倍的供应量稀释了排他性，而且推出时的问题甚至可能伤害了品牌。不幸的是，飞轮是双向旋转的，Yuga Labs 所有资产的价格都受到了打击。

Yuga Labs 可能还能实现。如果他们缺乏区块链或游戏开发方面的专业知识，他们有办法获得这些知识。Otherdeed NFT 是一场灾难，只是因为它是多么受欢迎。加密货币投资者的记忆往往很短，未来的成功推出会冲淡过去的问题。

作者：Messari
