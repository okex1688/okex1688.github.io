随着加密货币和 NFT 的兴起，玩赚钱游戏变得越来越流行。以前，人们花钱玩电子游戏，但现在，游戏玩家有机会享受最刺激的电子游戏，同时通过这样做赚很多钱。

在区块链技术影响世界的所有方式中，区块链游戏可能是最有利可图的，它催生了 GameFi 和强大的游戏赚钱游戏经济。但在我们深入探讨该主题之前，让我们快速回顾一下 GameFi 的历史。

GameFi 的历史

“GameFi”是游戏金融的简称，可以追溯到 2019年11月，区块链游戏发行平台MixMarvel创始人在中国_乌镇世界区块链大会上发表演讲。_

虽然 Mary Ma 的演讲是第一次使用 GameFi 这个术语，标志着围绕该技术的轰动效应的开始，但 GameFi 的基本概念早在演讲之前就已经存在（标题为“ GameFi-游戏行业技术驱动创新的路径”对于任何有兴趣查看它的人）。

事实上，GameFi 背后的概念几乎与比特币本身一样古老，可以追溯到 2013 年，当时像“ _Minecraft”_和“ _Bombermine”_这样的游戏正在尝试与 BTC 集成。更具体地说，通过在游戏中利用点对点服务，这些游戏使游戏玩家能够将他们的游戏体验货币化，使用 BTC 作为交换媒介。 

多年后，我们在以太坊网络上构建的 “ CryptoKitties ”等项目中看到了对 GameFi 的更精确探索。

_CryptoKitties_具有以不可替代代币 (NFT) 为代表的游戏资产。玩家获得这些 NFT 是为了培育独特的猫，然后可以在专门的市场上进行交易。 

有趣的是，虽然 CryptoKitties 由于网络阻塞、开发者实现复杂游戏功能的能力受限、游戏体验不佳以及区块链行业缺乏竞争市场等多种因素，并没有取得太大的成功，它确实为构成我们今天所知的 GameFi 生态系统的一波新项目铺平了道路。

那么，什么是 GameFi？

对于不熟悉游戏的人，让我们这样说：GameFi 之于传统游戏行业，DeFi 之于传统金融。从本质上讲，GameFi 旨在通过结合和整合区块链技术和去中心化金融来彻底改变传统的游戏基础设施。 

与传统游戏不同，GameFi 使玩家能够通过两个步骤将他们的游戏体验货币化——首先，通过游戏内资产的代币化，其次，通过利用游戏内即玩即赚 (P2E) 经济。

例如，在典型的视频游戏中，获胜会获得某种形式的排名积分、积分、评级、星级和游戏内货币——所有这些都起到精神刺激的作用，提供一种胜利感。但关键是你不能在游戏之外拿走这些奖励，也不能将它们用于任何有价值的交换形式。

使用 GameFi，叙事发生了变化。因为它们基于区块链、GameFi 协议或游戏标题，为玩家提供可以在现实世界中轻松货币化的奖励。这些奖励通常以加密货币和 NFT 的形式出现，当玩家获得这些奖励时，可以兑换成真正的法定货币。

已经有很多人通过GameFi致富了，他们以赚钱的同时也玩得开心而自豪。由于经济利益，过去曾告诫孩子玩电子游戏的父母开始支持他们。

GameFi 是如何工作的？

GameFi 与大多数基于区块链的项目一样，通过分布式账本技术生存，分布式账本技术是区块链技术的另一个术语。然而，每个 GameFi 项目在技术元素和正在实施的奖励系统类型方面都有其独特的机制。

无论如何，GameFi 通常使用区块链作为底层基础设施，为玩家提供经过验证的资产所有权，并建立一个去中心化的市场，交易者可以在该市场上交换游戏内和游戏外的代币化物品 (NFT)。

例如，在每个 GameFi 协议中，诸如武器、皮肤、金币、宠物、文物等游戏内物品都被标记化并表示为 NFT。 

因此，当玩家竞争并获得任何这些游戏内物品时，他们会自动声明对这些物品的完全所有权，并有能力进行交易，无论是通过协议的专用市场还是在游戏世界之外。

Hoard Exchange 弥合 NFT 与常规游戏行业之间的差距

也就是说，如果您不急于获得经济收益，玩家可以通过拒绝出售来继续积累胜利和游戏内资产。事实上，这样做甚至是有益的，因为大多数游戏都有系统，允许将游戏内的资产换成更好的工具、更强大的物品和其他提升，从而更容易赢得更多。

游戏协议的金融化以激励玩家产生了“游戏赚钱”游戏模式。如今，每一个真正的 GameFi 协议几乎都必须有一个激励模型或奖励系统。

同样，DeFi 功能也被纳入大多数 GameFi 项目中，作为二级奖励系统。这允许整合来自 DeFi 领域的已知概念，例如收益农业、流动性挖矿和质押，从而提供一种在玩游戏时产生收入的被动方式。

一些著名的区块链游戏已经融入了上述 DeFi 概念，包括“ Axie Infinity ”、“ Decentraland ”、“ Aavegotchi ”、“ DeFi Kingdoms ”、“ Nine Chronicles ”等等。

了解游戏赚钱游戏经济

由于玩家可以将他们从区块链游戏中获得的奖金货币化，这意味着除了获得乐趣之外，玩家还可以为他们的时间付费。因此，特定视频游戏的游戏内经济将决定玩家花多少时间玩它。

这是因为游戏循环旨在让玩家保持参与和快乐，同时也支持其核心情节和自然动态。

从经济学的角度来看，为了保持玩家的参与度、延长在游戏中花费的时间并增加玩家可以从游戏中赚到的钱的数量，游戏赚钱经济必须具有以下五个特征。

玩家基础优先于治理

如果一款游戏没有完美的设计，也没有足够多的玩家在玩，那么就不需要治理系统。

在这种情况下，治理指的是治理代币——一种加密货币，它赋予项目周围的社区投票或为区块链项目的未来做出重大决策的权利，在这种情况下，就是游戏的方向。

必须在任何重大治理生效之前实现忠诚的玩家群，否则将被浪费。 

考虑多种经济主体

在考虑传统游戏时，游戏开发者和玩家通常是游戏成功的两个主要驱动因素。事实上，其他有影响力的代理人的想法经常被忽视。 

但是，当涉及到通过游戏赚钱的游戏时，还必须考虑其他一些因素，即推动游戏内经济的投资者、建设者和收藏家。

投资者是游戏的财务支持者。尽管他们中的许多人可能是投机购买的，但他们的钱对于使游戏开发取得成功大有帮助。 

收藏家可能不会积极参与游戏，但他们参与了 NFT 方面的工作，产生了使 P2E 系统运行的需求。相反，建设者可能会在游戏内外产生价值供应。

一款即使没有“赚钱”方面也能站得住脚的游戏

虽然区块链游戏确实是赚钱的游戏，并吸引了他们的玩家群，但部分是为了他们可以产生的价值，游戏必须以这样一种方式构建，即其生存不仅仅取决于这些收入。游戏应该始终能够在娱乐层面吸引玩家，而财务方面不应该放在首位。

如果视频游戏更多的是赚钱而不是体验的快感和乐趣，那么它将是不可持续的，因为用户群将迅速减少。这不是 P2E 游戏独有的，因为即使在传统游戏中，缺乏“最终游戏”的游戏也会大幅失去人气。这很好地表明了单靠一个方面是不够的，必须在娱乐和价值之间取得平衡。

更高的被动收入 = 更低的玩家活跃度

如上所述，如果整个视频游戏都专注于财务方面，那么“玩赚钱”经济将会消亡；如果唯一的吸引力是金钱，那么就没有参与。 

同样，如果人们在不积极参与完成诸如抢劫银行和获得战利品、杀死怪物赚取积分、赛车和赢得奖品等任务的情况下，能够从游戏中赚钱，那么 P2P 交易等经济活动就会下降。

寻求以被动收入吸引玩家的 GameFi 项目必须包括租赁等模式，该模式允许人们通过借用诸如安全屋或武器等物品来赚钱，这些物品将被其他人积极用于游戏本身。

必须保护玩家生成的 NFT 的价值

如果游戏设计的游戏经济围绕着玩家创建自己的 NFT 内容，那么必须进行足够的投资来保护他们的利益以及所产生内容的价值。

如何开始使用 GameFi

那些已经熟悉加密货币和 NFT 交易的人会发现这非常容易做到。当然，作为一名游戏玩家，你也比任何想开始为钱玩的新手更有优势。

但是，如果您不属于上述任何类别，则无需担心。整个过程很简单，区块链上种类繁多的视频游戏多种多样且令人兴奋，足以让任何人参与其中，同时获得丰厚的回报。以下步骤将指导您开始您的 GameFi 之旅：

第一步：创建一个新的加密钱包

即使您已经拥有一个，为您的 GameFi 帐户创建一个钱包也很重要，因为您的 GameFi 体验应该被视为一项新投资。不过要小心，并且一如既往地进行研究，因为不同的游戏可能需要不同的钱包来连接区块链网络，因此请确保选择您选择的游戏支持的钱包。

第二步：将您的新钱包连接到您想玩的游戏

访问游戏的官方网站，然后点击“连接你的钱包”或类似内容的选项。这将使您能够赚取数字资产，这些资产将被发送到您的钱包，让您可以随意交易或持有它们

第三步：满足启动要求

在 GameFi 中，每款游戏都有不同的启动要求，但有一点是一致的，那就是你需要购买他们的初始代币——通常是 NFT。

这些代币将作为您的初始投资，这就是为什么创建一个新钱包很重要，并且只用您可以承受损失的资金为其提供资金。

最后，谨防虚假网站！那里有许多虚假的 GameFi 网站，这就是为什么我们总是敦促我们的读者仔细检查您收到的任何链接，并且只有在您确定其真实性后才提交项目。研究是重中之重。
