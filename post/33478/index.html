这一期的灵感来自与@anonalyx的几次谈话，已经很久没有一个话题能让我如此兴奋了。

本周我们讨论的是Web 3的社交体验。这些想法非常新奇，我相信在后面几期里还会继续完善很多内容。下面是背景介绍：

Web 2社交体验（又称社交媒体）指的是把线下生活带到线上，即用照片或视频的方式记录真实生活瞬间，然后放到社交媒体平台上，供他人消费、点赞和评论。 这就像一个个人真人秀，素材来自于我们的真实生活。

随着Web 3和Metaverse的发展，未来我们生活的时代会有所变化：我们的虚拟生活（我们在网上做的事情）与我们的现实生活会一样有趣、记录下来之后可以进行分享，甚至会比我们的现实生活更有趣。Web 3社交体验就是基于此。

剧透：NFT抢了风头。

让我们把镜头拉近一点……

Web 2 社交 → Web 3社交

想想Instagram、Youtube和TikTok等平台之间的相似之处。归纳总结之后可以发现它们都是分享真实生活的数字平台。我们可以在Instagram上分享度假时的自拍；在Youtube上上传一个录制好的恶搞视频；在TikTok上发布一个和朋友一起编排的舞蹈视频。

所有这些内容都是基于发生在现实生活中的事情。社交媒体让那些不在现场，没有亲身经历过的人也能体验到我们的真实生活。换句话说，我们的真实生活就是Web 2社交媒体内容的素材。

社交媒体另一个有意思的地方是视觉资料的运用。除了Twitter之外，每个主流社交媒体平台都离不开视觉资料。这是为什么呢？

照片和视频是我们用来捕捉真实生活瞬间的工具，这样我们就可以与他人分享。这是我们向不在我们身边的人展示生活的唯一渠道。

视觉资料也是我们证明事情发生的方式。它们是证据。毋庸置疑，我们的整个社交媒体内容构成了我们网络声誉的基础。只要我给你看了一堆我生活中的精彩图片和视频，你就会相信这些事情确实发生过。你对我的看法完全是基于这些图片和视频。

我们先选好可以分享的时刻，把它们做成幻灯片，然后在自己的Instagram主页或Youtube频道上向世界展示。这些内容汇集的都是我们精心策划之后的生活。

现在我们来概括一下Web 2社交媒体：

  • 我们的真实生活作为素材

  • 基于视觉资料（照片、视频）

  • 精心策划，呈现我们希望别人看到的自己生活的版本

现在，我们正在朝下一个时代过渡：我们的虚拟生活——我们在网上做的事情——和我们的真实生活一样有趣、值得记录。我们可以拥有虚拟物品；可以获得虚拟证书；可以为DAO工作；还可以完全基于自己的虚拟活动创立一个品牌和观众。下一代的社交平台将是分享我们数字生活的数字工具。

如果说真实的生活瞬间、经历和财产是Web 2社交的素材，那么虚拟瞬间、虚拟经历和虚拟财产将是Web 3社交的素材。

那么，你该如何记录这些数字时刻、经历和财产呢？

答案是NFT 。

NFT，Web 3社交的基础

如果你最近读过一篇关于Metaverse的文章，你很可能发现其中绘制了一个未来主义的、类似视频游戏的世界形象，基本上和《头号玩家》差不多。我去年写的关于Metaverse的那期文章中，用了下面这张图片。其他几十家杂志和网站在他们的Metaverse文章中使用的也是这张图片。

问题是，当今世界的Metaverse并不长这样。现在的Metaverse体验还是互联网体验，其中掺杂了一些Dapp和NFT。我们仍然需要盯着一个13英寸的屏幕，依靠2D图形和网络应用程序互动来完成所有事情。我们还没有实现一个完全沉浸式的虚拟现实。

但是，即使没有完全沉浸式体验，我们已经可以开始创建填充这些虚拟世界的资产。NFT及其相关图片就是第一代Metaverse视觉资料。

如果你想知道向Web 3社交的过渡什么时候才能开始，那应该是NFT成为Twitter上的PFP（头像）的时候。 一个看似愚蠢、无关紧要的行为，标志着社交新阶段的开始。

这是人们第一次通过社交媒体平台展示自己的虚拟生活，而非真实生活。NFT的视觉资料会对观众说“我拥有这个东西”，就像一张新跑车的照片表示的也是“我拥有这个东西”一样。

在结束这一趋势的介绍之前，我们要记住一个假设——越来越多的网络行为和成就未来会以NFT的形式表现。我在前几期中把它们称为微证书，但更广泛地说，它们是记录我们虚拟生活的一种方式。想一想，很多生活中的碎片记忆随着时间的推移都慢慢消逝了。但是，当你看到一张照片时，你的思绪会立刻闪回到当初的地点和时间。NFT不止能证明我拥有这个，它还可以表示“我做了这件事”或“我在那里”。

就像我在Instagram上发布照片一样，未来我也会自豪地在我的钱包里展示NFT。

这就是NFT的未来。它们不仅是记录数字财产的工具，还是记录数字行为的工具。Web 3的社交功能相当于一张照片

每个网络社区中最有价值的成员都会以NFT作为他们声誉的标志。如果你是一家新创业公司的种子投资者，你可以获得一个NFT；如果你是一个独家DAO的成员，你可以获得一个NFT。你所做的一切，尤其是你比别人做得好的事情，都会由一个NFT代表。然后在这些社区中，你就会得到认可，你的地位会高于其他人。这就是网络地位游戏在Web 3中的表现。

但是，如果我不希望自己虚拟生活的每一个方面都被公开访问呢？目前来说，这是默认情况。任何人都可以查看你的钱包地址以及里面的内容。在Web 2社交网络中，我必须选择分享什么。有了Web 3，所有一切都已经被分享到在网上了。所以策划就显得尤为重要了。

策划

有这样一个有趣的思想实验：想象一下Metaverse版本的 Instagram会是什么样子。你不能分享现实生活中的照片，那么你能分享什么呢？

要不试试分享你的头像、数字艺术和获得的其他NFT（可以代表任何东西）？

你可以通过把Metamask钱包和Opensea连接起来然后查看你的“个人资料”来完成这个概念证明。你所有的NFT都显示在一个画廊视图中，并且“我认为”是按照获取日期的时间顺序排列的。NFT画廊是Web 3/Metaverse的第一代社交媒体档案。

目前，NFT画廊会显示钱包中的每个NFT。然而，当我们的钱包中有了上百个NFT时，问题就变成了“我想展示所有NFT吗？”既然我们能花几个小时精心挑选和编辑在Instagram上分享的内容，那么自然也可以假设我们对NFT画廊存在同样的挑选意识和自我意识。

下一代的NFT画廊在策划功能上会得到改进——能够进行隐藏、排列、按社区分类等操作。请密切关注Islands.xyz。

不过，真正的进步应该是当你能发现并与他人取得联系的时候。也就是真正的社交互动。目前还没有一个比较好的案例，但像Context和Zapper这样的项目未来一定会成为顶尖，前者以“新闻提要”的形式更新NFT转让和销售活动的消息，后者也正在取得新的进展（见截图）。

最后

基于图像的NFT仅仅是一个开始。我们现在还没有完全生活在虚拟世界中。但你可以想象未来某一天，我们可能会发布自己的头像跳舞的视频。
