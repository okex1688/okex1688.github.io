第六章 NFT 项目盘点

NFTGo

众所周知，NFT 有许多分类，艺术品，收藏品，虚拟世界，体育等等。在浏览各类 NFT 项目前，最重要的是对 NFT 行业有一个整体的认知。对于投资者来说，需要把握 NFT 资产的宏观情况与最新动向；对于 NFT 小白来说，会想要知道 NFT 是什么？为什么要持有 NFT 资产？对于 NFT 爱好者来说，哪些项目有 NFT 资产？如何可以低门槛地购买到 NFT？我持有哪些 NFT 资产？这些资产估价多少？

NFTGo 是什么？

正如在数字货币诞生以来，Coinmarketcap 占住了最有利的 “生态位置”，即提供了所有对数字货币相关的数据、知识及信息入口，逐渐成长为加密货币领域的生态入口。而 NFT 生态入口的空缺也被 NFTGo 完美填补—— NFTGo 是一个 NFT 数据聚合平台，旨在展示、分析全网多链 NFT 数据，包括宏观的市场数据、资产数据、巨鲸持仓数据等，并将为 NFT 生态提供 NFT 钱包、NFT 聚合交易、NFT 估价器等服务 。

这个平台可以为用户提供最低门槛的 NFT 数据、信息聚合服务，帮助科普、推广、交易 NFT 资产，深度整合各种 NFT 的使用场景，让用户以最佳的体验与 NFT 交互，最终成为全行业的 NFT 生态入口。

NFTGo 会提供什么？

全网 NFT 市场数据的可视化

一是数据可视化，NFTGo 拥有专业丰富的 NFT 市场数据指标，并且实时采集全网的 NFT 交易与资产数据，以适应当下用户需求的方式进行可视化，帮助用户洞察当前 NFT 市场趋势，优化购买与投资决策。

在数据可视化外，NFTGo 还拥有最全面的排行榜功能，涵盖多个维度。从而可以帮助用户发现高价值项目与资产，追踪 NFT 巨鲸，评估 NFT 与 NFT Token 投资价值，探索全网用户的 NFT 持仓情况。

二是资产搜索功能。NFTGo 提供了触达全网 NFT 的搜索引擎，相比于 NFT 的 “亚马逊”，很多人反映更需要一个 NFT 的” 谷歌 “。NFTGo 聚合了全网优质的 NFT 资产，并根据元信息构建索引，让用户可以在一个平台搜索到全网的 NFT 资产，以更高效的方式探索 NFT 世界。

除此之外，NFTGo 未来还会推出聚合交易以及基于数据的估价器功能，以及多样的 NFT 指标。

OpenSea

1. 项目简介

成立于 2018 年的 OpenSea 是一个基于区块链的全球数字物品买卖市场。OpenSea 的定位是业界领先的去中心化交易所（Decentralized Exchange，DEX），提供点对点（peer-to-peer）的以太坊非同质化代币（NFT）交易。该综合平台售卖的数字商品多种多样，包括数字艺术品、加密收藏品、游戏物品等。在 OpenSea 平台上，用户可以与世界上任何一个人进行代币买卖和交换。

2. 项目特点

目前，OpenSea 是最大的去中心化数字商品市场，商品数量 700 多件，包括 Gods Unchained 和 CryptoSpells 等交易卡牌游戏和 Axie Infinity 和加密猫（CryptoKitties）等收藏游戏。商品具有多个类别，完全实现代币化和数字化。由于所有权证明和交易记录会在区块链上永久存储，所以用户一经支付便可以成为合法所有者。与实物物品不同的是，其中物品不会被盗且卖往别处，因为所有权记录和以及所有后续所有权记录都会永久记录在区块链上。

3. 商品种类

OpenSea 涵盖最广泛的类别、最多的可转让数字藏品，以及最优惠的价格，对于特定区块链资产来说，是一个优秀且安全的平台。该平台通过吸引了充满热情的用户，建立了极其活跃的开发者社区。OpenSea 主要售卖数字艺术品、加密收藏品、游戏物品和其他建立在以太坊 ERC-721 和 ERC-1155 标准上的数字资产，包括 Gods Unchained 和 CryptoSpells 等交易卡牌游戏和 Axie Infinity 和加密猫（CryptoKitties）等收藏游戏。OpenSea 也是 My Crypto Heroes、Etheremon Adventure、CryptoVoxels、ChainBreakers、CryptoBeasties、Ether Kingdoms 等大型游戏的主要市场。

4. 项目代币

未发行项目代币

5. 佣金收取

目前，OpenSea 收费较其他传统平台相比，佣金收取相对较高，买家交易费约 2.5%，其他佣金收费，如游戏道具，需收取游戏项目方 7.5% 费用。苏富比、佳士得拍卖艺术收藏品收取买家佣金约为 12-25%，收取卖家佣金约为 2%-9%。因此，相比较于传统艺术品拍卖行而言，这个佣金价格也是相对较低的。

6. 开发团队

2018 年 1 月，Alex Atallah 和 Devin Finzer 打下了 OpenSea 平台的基础。两人拥有 Palantir、加利福尼亚大学伯克利分校、谷歌、斯坦福大学、Facebook 和 Pinterest 的背景，专业知识精湛，组成了一支经验丰富的强大团队。

7. 参与方式

Opensea 的平台交易是通过智能合约进行的，也就意味着用户的 NFT 绝不会受任何中央机构照管。这也意味着，交易各方可以相信交易会按协议进行，无需中间人。NFT 可以存储在软件钱包中，如 Enjin Wallet、Coinbase Wallet、Opera Touch、甚至是浏览器的 chrome 插件，如 MetaMask。

8. 合作伙伴

OpenSea 主要合作伙伴有 Quantstamp、BlockStack、Blockchain Capital、Trust Wallet、Combinator、Coinbase、Founders Fund 和 1C。OpenSea 曾与游戏开发商合作，为用户创建可自定义的线上店铺，以自动购买、销售他们的加密藏品。

Nifty Gateway

1. 项目简介

Nifty Gateway 是一个受监管的、非同质化代币（NFT）的货币市场，利用 Gemini 的后端基础设施建立。在平台上，用户可以用信用卡购买 NFT，出售时可直接兑现到银行帐户中。

Nifty Gateway 是一个极其活跃、勇于创新的平台，平台所有服务现今都可供国际用户使用。用户只需注册电子邮件地址即可开始进行 NFT 交易。近期，Nifty Gateway 还更新了隐秘功能——Prepaid ETH，希望以太坊用户与信用卡用户处于同一个交互速度的环境，只需一秒就可以看到是否接收到艺术品。

2. 项目特点

Nifty Gateway 善于在名人合作中发现热点，创作并发布独家代币化收藏品。该公司大约每三周推出一次新的 NFT。比较有代表性的作品包括该平台与前 UFC 女子轻量级冠军 Cris Cyborg 和摄影师 Lyle Owerko 合作的收藏品。今年，Nifty Gateway 发布推文宣布《瑞克与莫蒂（Rick & Morty）》的创作者也将在以太坊上出售其 NFT 艺术品。

3. 商品种类

Nifty Gateway 平台可以用于购买和管理 NFT，目前支持的加密货币游戏和应用包括加密猫（CryptoKitties）、OpenSea、Gods Unchained 等项目。过去一年里，Nifty Gateway 无疑在加密货币掀起了波澜。

Nifty Gateway 平台上见证了加密艺术家 Beeple 的艺术作品 THE COMPLETE MF COLLECTION 的成功拍卖，成交价为 777,777 美元，是目前为止最贵的加密艺术品；艺术家 Trevor Jones 的作品《毕加索的公牛（Picasso’s Bull）》以 55,555.55 美元的价格卖给了加密艺术博物馆（MOCA）的 Pablo Rodriguez-Fraile，打破了以往所有以 NFT 形式出售数字艺术作品的记录。此类成功拍卖昭示了 NFT 的光明前景，也使得 Nifty 的发展态势更加强劲。

4. 项目代币

5. 开发团队

Nifty Gateway 于 2019 年经 Gemini 加密货币交易所的 “比特币亿万富翁” 双胞胎 Tyler 和 Cameron Winklevoss 收购，这也是该交易所的首次收购。巧合的是，Nifty Gateway 的创始人同样是一对孪生兄弟——Duncan 和 Griffin Cock Foster。

6. 参与方式

目前，Nifty Gateway、SuperRare 等平台也允许用户使用信用卡或借记卡购买商品。

Rarible (RARI)

1. 项目简介

Rarible 是 NFT 发行与交易平台，创建和出售由区块链保护的数字藏品。该平台是一个供用户铸造、销售和创建收藏品的开源的、非托管的平台。任何用户都可以进入 Rarible，创作和展示自己的艺术品，也可以用以太坊代币购买艺术品来收藏。用户拥有代币的所有权以及知识产权。买家和卖家可以在 Rarible NFT 市场上以几乎可以忽略不计的成本进行收藏品交易。

Rarible 还于 2020 年发布了属于自己的原生治理代币 ERC-20，改善了该市场的销售流程和销售条件，将用户的成本降至几乎为零。

2. 项目特点

Rarible 的去中心化特征非常明显。去中心化交易平台会将治理权下放给所有持有治理代币的人，而赚取这些治理代币的唯一方法就是在加密交易平台购买。虽然这些代币名义上用于治理，但实际上很多持有者用它们来盈利。去中心化模式加上治理代币允许用户对平台的发展和决策进行管理，让最为活跃的玩家们为任何平台升级投票并参与管理和审核。

细数 Rarible 的特性，我们不难发现该平台极大地方便了用户创建和铸造属于自己的数字代币的过程，门槛低，为非编码人员提供方便的用户体验，而且点对点的模式中交易成本低。

3. 商品种类

Rarible 上的艺术品创作活动十分活跃，例如，亿万富翁、达拉斯独行侠老板 Mark Cuban 曾在其发售其创作的加密艺术品 NFT “The RollUp 2021”。Cointelegraph 也曾在其平台上进行知名艺术作品的单版 NFT 拍卖，包括受意大利画家达芬奇作品《最后的晚餐》启发的 “最后的 ICO（The Last ICO）”、根据法国画家亨利·马蒂斯的作品《舞蹈（La Danse）》改编的 “无声迪斯科（Silent Disco）” 等。

4. 项目代币

RARI 是 Rarible 于 2020 年 7 月发布的 ERC-20 治理代币。RARI 将 2020 年 DeFi + NFT 几个热点进行整合，将数字藏品与农耕收益与流动挖矿相结合。使用平台用户便可得到 RARI 代理治币作为奖励。RARI 主要侧重用户生成的艺术作品，运用 NFT 提供出处证明，保证用户新艺术品的知识产权。此外，RARI 改善了该市场的销售流程和销售条件，将用户的成本降至几乎为零，激励用户参与。

RARI 的用例包括治理、管理和特色艺术品投票，交易者们也可以时刻关注其他平台的平台代币，或添加作者 Sophia(ID:lovebit98) 获取相关最新资讯。

5. 佣金收取

Rarible 市场建议创作者将版权费金额设置在 10%、20% 和 30% 左右。

6. 开发团队

Rarible 的联合创始人为 Alexei Falin 和 Alexander Salnikov。Alexander Salinkov 拥有超过 7 年的技术背景，也曾参与过一些著名的加密项目，包括 Humaniq。

7. 参与方式

在 Rarible 交易也仍然需要提前注册 MetaMask。

8. 合作伙伴

Rarible 受到了众多投资者青睐，也开展了很多合作项目。Rarible 曾与 DeFi 项目 Yearn Finance 合作以代币的形式发行保险，Yearn Finance 的 yInsure 将允许用户通过 Rarible 市场为各种 DeFi 活动购买代币化保险。

MakersPlace

1. 项目简介

发布于 2016 年的 MakersPlace 是一个基于区块链技术的、为艺术家和创作者提供全方位服务的平台，用于铸造代表其作品的 NFT。借助区块链技术，收藏者通过 MakersPlace 获得的每个数字创作都是由创作者签名和发行的独特的数字创作。即使其他人复制了该数字作品，得到的也不会是正品的或原始签名版本。该平台通过市场和合作伙伴网络，将艺术家与艺术爱好者、收藏家联系起来。通过 MakersPlace，艺术家们能够更好地捕捉数字作品的全部潜在价值。

去年，MakersPlace 进行了为期一年的 Beta 测试后，正式推出。该平台创始人 Dannie Chu 和 Yash Nelapati 曾在 Pinterest 有任职经历，两人发现了发现 Pinterest 在为创作者提供商业化途径方面的短板，因此创立了 MakersPlace ，以作为一个数字作品的交易平台。

2. 项目特点

MakersPlace 为艺术家们提供了线上出售自己的艺术作品的平台，艺术家无需对区块链技术有深入了解，只需要提供一张证件照便可在 MakersPlace 中获得生成的 ERC-20 代币，进行作品交易。艺术家可以在自己的数字加密 “钱包” 中保存货币，也可以选择使用 MakersPlace 提供的钱包。

3. 商品种类

MakersPlace 是搭载的区块链技术为数字艺术家们提供例如真实性、所有权和稀缺性证明。

通过 MakersPlace，创作者能够更好地获取数字作品的全部底层价值。MakersPlace 已经和数千名出色的数字艺术家合作，帮助他们交易他们独特的艺术作品。该平台简便好用，交易流程也非常顺畅，为购物者和卖家都提供了一站式的全面体验。

4. 项目代币

暂未发行项目代币

5. 佣金收取

Makersplace 鼓励艺术家在该平台上出售自己的作品，在数字艺术品交易中抽取 15% 的佣金进行获利。

6. 开发团队

MakersPlace 的主要创始人 Dannie Chu 和 Yash Nelapati 此前均为 Pinterest 的工程师。另一位联合创始人 Ryoma Ito 则曾在多家初创企业担任产品 / 营销相关的重要职位。

7. 参与方式

MakersPlace 为了在区块链购买数字作品，用户需要数字钱包和加密货币分别用于存储和购买作品。然而，世界上大多数人没有数字钱包和以太币，这使得创作者很难向粉丝和收藏家销售他们的作品。为了解决这个问题， MakersPlace 与 Stripe 合作，支持全球信用卡支付购买数字作品。

8. 合作伙伴

MakersPlace 合伙伙伴众多，合作大受欢迎，引发热点的伙伴之一有 Decentraland。MakersPlace 将其全部作品加到了 Decentraland 的市场板块中，以真实且独特的数字作品为主题，通过区块链进行数字签名和验证，当之无愧是世界上最优质、最稀缺的数字艺术作品的聚集地。感兴趣的读者可以到 MakersPlace VR Gallery 逛一逛。

Super Rare

1. 项目简介

成立于 2017 年的 SuperRare 被誉为拥有全球数字艺术家网络的互联网数字艺术市场，拥有近 700 位艺术家。SuperRare 基于区块链技术，遵循 ERC721 标准，艺术品的整个历史和起源都可以在分布式账本上追踪，任何人都无法作弊造假。

在该平台上可以轻松创建、销售、收集稀有数字艺术品。该平台让艺术家能够获得新的收入来源，并为收藏家提供了储存或交易艺术品资产的场所，保证作品的稀缺性、原创性，并解决版权溯源等问题。艺术家可以拍卖自己的创作，也可以以固定的价格出售。NFT 一经出售，可以在各种 NFT 交易平台上以任何价格转卖。自成立以来，该平台已取得长足进步。

2. 项目特点

SuperRare 的独特之处在于，尽管任何人都可以购买和持有以太坊代币，但是只有受邀的艺术家才能在平台上创建艺术品。要申请以艺术家身份加入网络，须得填写申请表。SuperRare 要求所有基于平台创作的艺术品都必须为原创，并且不得在互联网上的其他地方进行代币化。对艺术家入驻的高门槛要求很大程度上保证了艺术品质量。

3. 商品种类

SuperRare 的平台艺术品定位高端，对艺术家入驻门槛要求高，一定程度上保证了艺术品的高质量。同时，SuperRare 良好的社区建设和定期高质量的数字艺术展览有效地扩大了数字艺术覆盖人群，逐渐成为数字艺术行业的基石和标准。SuperRare 链上艺术馆具备线下艺术馆的展览、拍卖等功能，并使用 NFT 保证了作品原创性，保护了艺术家们的知识产权。

SuperRare 曾与 ArtGee、ChainNews 以及 Digital Renaissance 在上海举办了中国首秀，邀请了十位风格多元，创作手法各异的主流加密艺术家进行展览，包括 Pak、Esteban Diacono 等。该活动致力于推广和提高大众对 NFT 艺术品的认知，弥合艺术家和区块链领域人士之间的鸿沟。

4. 项目代币

暂未发行项目代币。

5. 佣金抽取

在 SuperRare 测试平台上，初次销售没有佣金，二次销售佣金 3％。其使命是帮助艺术家在互联网上谋生，同时为世界贡献更多的艺术作品，相信艺术家能够以他们所爱的方式谋生。

6. 开发团队

SuperRare 由 John Crain（CEO）、Charles Crain（CTO）和 Johnathan Perkins（CPO）于 2017 年创立。他们同时也是 Pixura 的创始人（Pixura 是一个用于创建、跟踪和交换加密数字藏品的平台）。

7. 参与方式

目前，Nifty Gateway、SuperRare 等平台也允许用户使用信用卡或借记卡购买商品。

8. 合作伙伴

Decentraland (MANA)

Decentraland 是最老牌的 NFT 项目之一。该平台是一个虚拟世界，玩家可以购买、构建、探索虚拟结构，也可进行社交活动。 Decentraland 是基于以太坊的去中心化 VR 平台，用户能够完全控制自己创建的内容环境和应用，其范围可以从任何静态 3D 场景到具有更多交互功能的应用或游戏。Decentraland 上有限、可流通的 3D 虚拟空间被称为 LAND，都是使用 ERC-721 协议的 NFT，LAND 通过坐标标识，社区成员可使用 MANA 购买后可永久拥有。此外，LAND 还可以进一步组织形成主题社区，社区是一个具有共同兴趣和用途的共享空间。在 Decentraland 中，NFT 商店遍布，有些 NFT 只供展示，而其它 NFT 则可用 MANA 购买。用户可在 Decentraland 市场上购买可穿戴设备，市场也会售卖帽子、鞋、衬衫和地块等。此外，去中心化自治组织（DAO）可让用户参与虚拟世界的创建及运行工作。为解决扩容问题，降低交易成本，提高交易速度，Decentraland 用户可以利用 Matic 的股权证明（Proof of Stake，PoS）侧链，该侧链是为以太坊构建的 Layer 2 扩容方案（之前名为 Plasma）。

Decentraland 核心团队在加密技术、软件工程和风险投资领域享有盛誉。 该团队由倍受赞誉的 BitPay 软件工程师 Esteban Ordano 创立，注册地为 Metaverse Holdings Ltd，总部位于中国北京。Ordano 与项目主管 Ariel Meilich、董事会成员 Manuel Araoz 和 Yemel Jardi 等其他 Decentraland 团队成员创建了面向区块链的黑客空间，Voltaire，总部位于阿根廷。

Decentraland 与多个区块链项目建成了合作关系，如 Aragon、Coral、district0x 等。 Decentraland 由于诞生的时机和背后的支持机构助力， 在 ICO 热潮中获得了投资者和市场的疯狂追捧，成功筹集 2500 万美金。Craft 的数据表明，截至 2020 年 10 月，Decentraland 团队即便按照每年 300 万美金的成本计算，筹集的资金也几乎能覆盖团队 10 年的人员开支。

MANA 是 Decentraland 于 2017 年发布的以太坊 ERC-20 实用性代币，可用于购买 LAND 以及 Decentraland 中的虚拟商品和服务。 MANA 还可用于支付化身，物品，收藏品以及赌博等娱乐活动。MANA 的赚取机制是对游戏开发人员创建和管理优质内容的一种激励。MANA 供应量固定，MANA 用于购买 LAND 后即被烧毁。

Cryptovoxels

Cryptovoxels 是以太坊最炫酷的虚拟世界项目之一，最近迅速增长，让我们一探究竟。

1. Cryptovoxels 是什么？

简单来说， CV 是一个以太坊虚拟现实（VR）世界。简而言之，这个项目让用户能够在 CV 世界中买卖虚拟地皮和其他数字资产。

然而，类似我的世界（Minecraft）的 CV 绝非仅仅是部分的总和。该项目涵盖许多内容，如房地产、VR、超数字化、加密技术等等。CV 迄今建立的社区也不是单一性的，而是多元化的：目前为止，CV 中既有投机者，也有建造者，还有介于两种角色之间的人。

这些基于 CV 地皮总共带来了超 7,300 ETH 的交易额。CV 土地的需求量增大，价格也比以往任何时候都高，在可见的未来，CV 的交易额必然会增加。

2. 创造者之地

CV 为数字化建造者提供了类似 Minecraft 的体验。通过这种方式，CV 中已经出现很多才华出众的建造者，他们在数字化世界的出色地皮上炫耀自己的物品。

但数字化建筑师只是 CV 的主要用户群之一。实际上，数字世界的效用性远远超出用户的想象力。比如，最近几个月，CV 成为了著名加密艺术家的主要画廊。

也就是说，很多创建以太坊同质化代币（NFT）的艺术家现在都在 CV 中建造画廊，展示作品，说明总体来说，CV 这个年轻平台对于内容创造者十分友好。

3. 为什么？“数字稀缺”

Andrew Steinwold 是以太坊最著名的 NFT 分析家之一，Zima Red 新闻通讯创始人。

这个月，Steinwold 在 Zima Red 发表了一篇名为《稀缺网络（The Scarce Web）》的文章。总结来说，Steinwold 认为，CV 类似的虚拟世界项目是 " 稀缺网络平台 “。如果想了解 CV 土地为什么如此具有价值，Steinwold 的论点能为您解答：

“当今，稀缺网络最好的例证便是区块链的虚拟世界，如 Decentraland、Cryptovoxels、Somnium Space、The Sandbox。这篇博文中使用 " 稀缺网络平台 “，这个词指的便是基于区块链的虚拟世界 …… 进一步解释就是这些虚拟世界向用户出售地皮，用户可以建造任何喜欢的东西。用户可以创建一个社交媒体档案，链接自己的照片、发布自己的简历，甚至可以创建电子商务商店。简单来说，虚拟世界是一个具有原生货币化和稀缺性的特点的 3D 网站。”

4. 原始地皮拍卖

Cryptovoxels 的标志性虚拟地皮叫做原始地皮（Origin City）。以太坊中最缺少充分报道的事情之一便是 2020 年 4 月 8 日，最后的原始地皮刚刚拍卖售罄。

也就是说，原始地皮再也不能通过拍卖获得——只能通过点对点（peer-to-peer，P2P）交易。

从这个意义上讲，CV 已经实现了初步重要的去中心化，项目未来发展方向主要来自需求上升。目前需求很高。去中心化市场 OpenSea 上，现价格最低的 CV 地皮为 2.99 ETH，约合 520 美元。

但此价格下限并不是定数。曾经较低，未来可能会更高。但目前这个下限可以说明 CV 的受欢迎程度，因为如今买家须得付出一定价钱以锁定自己的数字土地。

相邻地皮（Island Properties）即将推出

CV 中的原始地皮现在可能已经售罄。但并不是说 CV 拍卖永远告别。

实际上，虚拟世界的建造者最近提供了一种可能性：在原始地皮周围的边缘新增、拍卖相邻地皮。

因此，未来几周会有更多 CV 拍卖，虽然仅针对于新增的相邻地皮。原始地皮只能在二级市场中购入。

5. CV 不断改进

加密货币项目属于软件项目。也就是说排名靠前的加密项目处于不断更新之中，通常由出色的专业人士完成。

CV 团队是整个加密经济领域最为多产勤奋的团队之一。为什么？关注 CV 的 Twitter 账户便会发现开发者的更新速度着实惊人。

有些加密项目几个月来从不更新，但 CV 恰恰相反：几乎按天更新，CV 开发团队规模虽小，但素质极高，活力非凡。

6. VR 崛起

新冠疫情大大提高了远程办公和远程生活可能性。未来几年，这个焦点会持续下去，至少取决于从此事情的演变进程，而 VR 产业必定从中获益。

重燃对数字空间分享的兴趣只会促使 CV 提高在加密经济中的地位。现在 CV 还在初期，增长空间巨大，不可否认，未来几个月里，CV 的影响力和受欢迎程度会大大增加。

因此，CV 看起来是加密领域活跃的黑马之一。随着 VR 在加密领域内外不断发展，CV 的影响必定会持续上升。

也就是说，CV 不仅构建精良，也精准地把握了时机。更广义的来看，CV 表明以太坊不仅仅是一个去中心化金融平台——远不止于此，如数字化房地产交易结算服务等。还有很多可能性有待探索，而炫酷的 CV 便是探索中的项目之一。

Alien World (TLM)

DeFi 和 NFT 是两大区块链热词。 Alien Worlds 将两者与代币抵押、挖矿和虚拟土地销售相结合，创造了全新的游戏体验。 继 2020 年 10 月中旬完成首场 NFT“跨链” 拍卖活动后，Alien Worlds 又于 12 月在 “卡牌包预售” 中成功筹集 25 万美元。涉足 DeFi 领域的人对其机制和功能肯定会对 Alien Worlds 感到熟悉。

Alien Worlds 的团队 Dacoco 向来敢于创新。该团队协助成功推进了 2018 年 eosDAC 火热空投，市值暴涨，还推动了 EOS（Enterprise Operation System）公链的发布。新兴技术自然能征服新维度。

对一些人来说，加密猫已经是最为伟大的游戏。用户和 DeFi degen（degenerate gambler，堕落的赌徒）每日计算着自己的加密投资组合，希望找到下一个爆点和财运。 Alien Worlds 本着同样的冒险精神，创造了一个太空远征的故事情节，最终的目的地远在太阳系之外。 2055 年，流行病肆虐地球，但人类并没有像 2020 年那样封锁呆在家里，而是去探索其他星球。这种解决方式可能不太实际，但听起来要刺激得多。

加密货币领域中，游戏化是一个大趋势。Alien Worlds 利用以太坊和 WAX 区块链为玩家提供了购买，出售和赚取数字收藏品的新方式，是第一个也是唯一一个跨领域项目。

什么是 Alien Worlds

Alien Worlds 背景设定在 2055 年，探险者能够在 6 个不同虚拟星球中战斗、采矿，玩家还有权利参与自治星球上 DAO（去中心化自治组织）的管理和决策。 这种去中心化分权模式可以激发其创造空间内的创造力和增长。在 Alien Worlds 中，用户可以创造自己的游戏和 NFT。

该游戏的 6 个自治星球由 3343 个土地包组成。每个土地包都包含一个土地 NFT 以及其他 7 个配套 NFT。 与 Decentraland（9 万个土地包）或 Sandbox（12 万个土地包）等项目相比，就会发现 Alien Worlds 中土地有多稀缺。空间可能是无限的，但 Alien Worlds 的土地 NFT 却是有限的。

Alien Worlds 拥有两种货币：ERC-20 协议的游戏币 Trilium，以及代表武器、土地等物品的 NFT 代币。 土地所有带来的利益会以游戏中的原生代币—— Trilium（TLM）的形式呈现。

冒险者可以选择自己开采土地，也可以允许他人开采自己的土地并支付佣金。因此，土地所有者可以选择积极参与，也可以让其他人在自己的土地挖矿。游戏的内置收益功能可赚取佣金，玩家还能够将佣金率按喜好设置高低，赚取属于自己的被动收入。 在 Alien Worlds 中，拥有土地是赚取 TLM 最稳妥、最轻松的途径。该游戏的 NFT 卡牌拥有受智能合约驱动的实际作用，与用于收藏的 NFT 不同。

Alien Worlds 也有其他形式的 NFT，如采矿工具等，但只有拥有者才能使用。而土地具有类似 DeFi 的收益属性，将会鼓励土地所有者向太空探险者开放自己的土地。

在加密空间中，虚拟土地购买狂潮是数字化的，在舒适的椅子上或在电脑屏幕后便可发生。曾经，土地所有权一直是属于上层阶级的特权；如今，在虚拟世界里，每位玩家都有机会拥有一块属于自己的土地 NFT。

DIGITALAX

1. 项目简介

初创公司 DIGITALAX 推出了世界上第一个数字时尚产业 NFT 协议——而且它有信心能够将这些前沿数字资产推向主流。DIGITALAX 是第一个专门的纯数字时尚拍卖交易平台和开源的数字时尚工具包。未来将进一步提高可编程的稀有性、去中心化的数字时尚供应链、原生的数字时尚定价框架、设计人员和开发人员协作通道等。

该项目基于以太坊平台。DIGITALAX 的主要目标包括但不仅限于围绕数字商品、开放数字许可证和供应链自动化实现新行业标准，并解决有关非同质化代币流动性和稳定性的核心问题。

该项目自主筹集资金，解决了围绕可编程稀缺性和去中心化数字时尚供应链的核心问题。其采用的工具和技术可使设计者和开发人员相互协作，开发数字化材料、图案和可互操的游戏和 VR 时尚资产。

2. 平台特色

DIGITALAX 市场为设计师们提供了拍卖作品的界面——这些虚拟服装甚至可以进行抵押。

30 多位才华横溢的专业人士聚首，组成了全球设计师网络（Global Designer Network），其成员遍布欧洲、美国和非洲。他们创造了出超逼真的 3D 皮肤和虚拟化妆品，面向所有用户出售。

这种创新模式下，每当设计的衣服经人购买时，设计师都可获得版税——NFT 技术还融合了 DIGITALAX 的开放数字时尚授权（Open Digital Fashion License），保护知识产权，买家购买任何物品，都可拥有完整的知识产权转让证明。

DIGITALAX 还希望其 NFT 能够重振游戏和虚拟现实世界——总市值达数万亿美元——并且为热情的创作者赋能。

平台采用三种代币——ERC-721、ERC-1155 和 ERC-20——可用于参与拍卖、对治理事项进行投票、从市场平台获得收入。它们奠定了部分衣服所有权等新商业模式的基础。

数字时尚是由 CLO3D、Zbrush 等电脑软件制作并呈现的礼服、衣物和配饰等，现还处于萌芽期。设计师可以使用数字时尚创作软件进行设计，最终到处创作产品，大大减少了服装材料浪费。数字时尚可以创建和构建虚拟空间，在游戏、数字音乐会或演出中提供数字限量版系列服装 , 带来一种全新的商业模式。

Meetbits

像素肖像游戏 CryptoPunks 是网络上最受欢迎的 NFT 项目之一，其创作团队 Larva Labs 已净赚了超 5.5 亿美元。Larva Labs 最近推出了一个新项目——Meebits。Meebits 拥有 20,000 个程序生成的 3D 字符，可以在以太坊上交易。

过去几个月里，出现了数百个 3D 角色 NFT 平台，吸引了非常多的加密买家的注意力，Meebits 的到来正逢 NFT 炒作高峰，几乎每天和 NFT 有关的数据都在创下历史新高。

Meebit 发行量总共有 2 万个，每个 Meebit 都有自己的风格和特征。许多 Meebit 的分发是为了奖励早期的 Larva Labs 支持者以及区块链社区。在 2 万个 Meebits 中，10,512 个将保留给之前的 Larva Labs 资产持有人，其中有 10,000 个 CryptoPunks 和 512 个 glyph 是 Meebit 的兑换 “门票”，允许免费兑换 Meebit（以太坊铸币的 Gas 费用除外）。

剩下 9488 个 Meebit 用于投放给其他用户，以荷兰拍卖机制迅速卖光。Meebit 铸造完全是随机的，没人确切知道会收到哪一个，但许多人仍然花费 2.5 ETH ，即近 8500 美元，去生成罕见角色。

从 2D 到 3D

我们所熟知的 CryptoPunks 是像素化的 2D 图标，Meebits 是用体素渲染的 3D 角色（体积像素——“体素（voxel）”）。Meebit 通过算法生成，是以太坊上的 NFT。

3D 角色

Meebits 是 20,000 个独特的 3D 体素字符，由自定义生成算法创建，在以太坊上注册。

ERC-721

Meebits 管理所有权的 NFT 协议是 ERC-721，适用于任何兼容的服务或交易所。

综合无费用市场

Meebits 拥有定制市场，支持每笔交易最多 100 笔 Meebit 交易，以及标准的买入、出价和要价交易。Meebits 作为一种可下载软件将涵盖多个范畴，包括查看加密藏品、NFT 和其他代币，还可以对加密收藏品和 NFT 进行交易、存储、发送、接收和传递等。Meebits 看起来像一个全新市场，也可以通过手机轻松访问。

Meebits 可能会让人想起我的世界或 Roblox 中的 3D 角色，这是有意为之的。开发团队写道：“我们喜欢低聚和体素 3D 图形。在过去十年左右的时间里，这种风格一直受到我的世界和 Roblox 等游戏的欢迎。我们认为，就像 8 位像素艺术一样，体素艺术的极简主义和可接受性将会永远存在，受到欢迎。”

在某些方面，Meebits 与 CryptoPunks 非常相似。每个 Meebit 角色都有不同的属性，表示一定程度的稀缺性。例如，骷髅、访客和机器人比人类要稀缺得多。

编写体素字符生成器比编写二维生成器更具挑战性。几年来，团队一直在改进体素，而最终呈现的结果也让人感到自豪。

许多小细节值得注意：某些 Meebit 上有纹身图案，还有一些很酷的稀有 Meebit 等待大家在社区中探索。

Meebits 还没有开发完全，但后续会为每个 Meebit 提供资产包，允许所有者使用 “T 姿势” 版本的角色，并将其用于任何游戏引擎、3D 工作室或虚拟世界。

与 Cryptopunks 一样，智能合约中存储着文件哈希值。20,000 个 Meebit 铸造出来后，IPFS 上同时进行了发布，包含了每个 Meebit 的所有属性和体素数据，因此它们可以永久存活。

什么是 “体素”？

在计算机图形学中，术语 “体积元素”，简称 “体素”，英文为 “Volume Pixel”，即 “voxel”，是用于表示三维空间中矩形网格上的值。一如其名，体素是数字数据于三维空间分割上的最小单位，体素用于三维成像、科学数据与医学影像等领域。二维空间的最小单位——像素，是二维图像上像素的三维类比，体素在概念上类似像素。

在虚拟环境和游戏中，体素图形的引入十分引人注目，因为它可以构建完整的、可交互的世界。基于像素的世界通常具有许多用户无法产生影响的环境元素，以二维平面的形式存在，类似于一张纸，没有深度和灵活性。而体素可以将超现实主义和可交互性融合起来，带来卓越的性能提升，为用户带来更好的游戏体验。

Larva Labs 将 Meebit 创建为 3D 体素字符，至少能使它们与基于体素的图形环境兼容。Larva Labs 还将为 Meebit 所有者提供资产包，其中包括高分辨率、无损渲染的 T-pose OBJ 文件。T-pose OBJ 文件是一种标准文件类型，适用于导入大多数标准 3D 建模和动画软件。

决定 Meebit 价值的基本因素

一是特征，不同的特征能够使得某些 Meebit 比其他的更具稀缺性，这些特征包括：

·头发颜色

·衬衫

·裤装

·鞋类

·帽子、眼镜、珠宝、外套或纹身等

决定 Meebit 价值的另一个重要的因素是其铸造顺序，早期的 Meebit 通常比后来的更有价值（尽管每个角色的特征扮演着重要作用）。

目前，有史以来第一次铸造的 Meebit 在 Opensea 上已经出售。大多数 Meebit 属于人类，一种最为常见的变体，但也有很多更罕见的变体，如猪、访客、大象、解剖物（类似于僵尸）等。

决定 Meebit 价值的玩法因素

用途驱动潜力

Meebit 被设计为一个 3D 角色，可用于各种应用程序、服务器、游戏和平台，具有独特的可验证所有权，易于识别。理论上，用户将能够在任何与基于 ERC721 协议的 NFT 兼容的可互操作区块链应用程序中使用 Meebit。因此，Meebit 的使用机会几乎无穷无尽。

尽管目前 Meebit 发行数量是固定的，但 Larva Labs 计划推出资产包，使用户能够在游戏引擎、3D 工作室或虚拟世界中摆出 Meebit 的独特姿势——这些功能的实现将为 Meebit 带来更多活力。

无 gas 费的创新型交易市场

Cryptopunks 推出时，ERC-721 协议和 NFT 市场仍未推出，因此 Cryptopunks 拥有内嵌在智能合同中的自己的市场。NFT 标准和市场出现后，原本认为这可能会导致 Cryptopunks 落后。但恰恰相反，Wrapped Cryptopunks 能够与标准市场连接起来，同时保持一个定制的专用市场，为收藏家提供了良好的用户体验。

因此，Meebits 也想复制同样的体验，并做出升级。Meebit 是标准的 721 代币，但也有自己的市场。这个市场不仅允许用以太币进行买卖，还可以进行复杂交易。

用 Meebit 换 Meebit 将是非常有趣的体验，让人想起小时候的收集游戏，不涉及金钱。

此外，提议交易只需用钱包签署消息即可，不需要以太坊交易，因此不需要 Gas 费用（只有交易的 “taker” 才用支付 Gas 费用）。随着交易可以在链下轻松进行，这应该会提高流动性。

与 Cryptopunks 市场一样，Meebits 市场是去中心化的，不收取任何费用。更多市场功能即将推出。

去中心化、简单化、定制化

NFT 引入的最重要的功能之一是大多数 NFT 资产的交易都十分简单。Meebits 带来了 NFT 之间定制化的交易，允许比以前更为复杂的交易。Meebits 市场中，用户不仅可以自由交换资产，还可以进行同质化代币的交易。

举例来说，如果用户有两个总价值为 1 万美元的 Meebit，想与价值为 15,000 美元的 Meebit 的所有者进行交易，他们现在有机会用自己的 2 个 Meebit 加上 1.5 ETH（价值约 5000 美元）与后者进行交换，以换取直接的等额交易。以前，用户必须经历多个步骤和更长的过程才能完成类似的交易，Meebits 的创新在于允许 NFT 之间进行最简便的复杂交易。

“NFT + 我的世界” 构筑多样化玩法

我的世界，Minecraft，是一款由 Mojang AB 和 4J Studios 开发的高自由度的沙盒游戏，于 2009 年 5 月发行。该游戏让玩家在三维空间中自由地创造和破坏不同种类的方块，玩家可以在单人或多人模式中摧毁或创造方块，来创造精美的建筑物和艺术，或者收集物品探索地图以完成游戏主线。

自 2010 年起，我的世界这款游戏便有着悠久的 modding 历史。“mod” 是 “modifications” 的缩写，也叫模组, 是指任何基于我的世界原本游戏内容所作出的任何修改。许多 Mod 向游戏添加更多的额外内容来改变游戏的方式和可塑性, 或给予玩家更大的刺激, 或给予玩家与我的世界更多交互的方式。

Meebit 在未来或将会复制我的世界中的技术能力。我的世界强大的粉丝群体加上 Meebit 角色的巧妙技术，似乎能让 NFT 系列爆发，赢得大众追随。

决定 Meebit 价值的流量因素

社区为王

为了感谢用户对于 Cryptopunks 和 Autoglyphs 社区的支持，吸引之前的 NFT 收藏品粉丝参与进来，Meebits 允许所有者免费铸造新项目中的 Meebit。每个 Cryptopunk 和 Autoglyph 都可以铸造 1 个免费的 Meebit。用户只需支付与铸币交易相关的以太坊交易费用（Gas 费用）。从公开销售开始一周后用户便可以进行铸造。

但铸造 Meebit 必须由拥有 Cryptopunk 或 Autoglyph 的帐户来进行交易。铸币合同将根据申请铸币的账户验证用户所有权。

需要注意的是，每个 punk 或 glyph 都可被视为 1 张免费的 Meebit 门票，但这张票一旦使用，就不能二次使用。用户持有 punk 或 glyph 的数量不会发生改变，销售合同仅根据拥有的 punk 或 glyph 的数量来明确允许铸造 Meebit 的数量。

抓住未来趋势——Metaverse

除了基本角色设计外，Larva Labs 还注意到，CryptoPunks 被广泛用作 Discord、Twitter 和其他社交媒体的理想 2D 头像。因此，Larva Labs 希望 Meebits 成为领先的 “虚拟世界、游戏和虚拟现实的 3D 头像”。为了实现这一目标，该团队很快将发布资产包，以支持虚拟世界中的角色动画。换句话说，Meebits 的设计是为了让其成为元界中的首选身份。

在完全沉浸式、基于体素的虚拟现实中，Meebits 会成为未来趋势吗？我们还不能盖棺定论。但是这个项目的确值得关注，Meebits 拥有 NFT 投资者追求的特质——计算机随机生成的特征，精妙的细节以及独特性和稀缺性。此外，其体素技术方法以及创新 3D 资产会带来前所未有的游戏体验。

收益驱动增长

CryptoPunks 的部分吸引力源于其稀缺性：CryptoPunks 的总供应量只有 1 万只。但即使 CryptoPunks 价格不断攀高，将 Meebits 的总供应量翻一番（2 万个 Meebit），Meebits 的价格仍然会保持高价。

该公司正在通过拍卖分发 Meebit 角色，这意味着那一周里 Meebit 购买和铸造的价格可以低至零 ETH（外加以太坊 Gas 费用）。目前，用户随机铸造一个 Meebit 需要支付 2.49 Eth。但尽管如此，大约 2000 个 Meebit 已经售出。尽管荷兰拍卖的形式导致价格在特定时间内下降，但上线不到六小时，所有 Meebit 公开发行就一抢而空。这意味着 Larva Labs 在上市两小时多后已经获得了近 8000 万美元的收益。

同时，二级卖家也将迅速获得巨额收益。例如，第二天，OpenSea 报道称，目前排名前三名的 Meebit 中，最初售价低于 3 ETH 的 Meebits #8598 打破了其最高单件商品销售记录，达到 420 ETH，在本文撰写之时相当于超 143 万美元。Meebit#2948 达到 400 ETH，Meebit #776 达到 390 ETH。

DappRadar 的数据显示，在不到一周的时间里，根据二级销售额来看，Meebits 是有史以来最为成功的第八个 NFT 项目。

当然，在某些方面，Meebits 的巨大成功并不令人惊讶。NFT 收藏品游戏 CryptoPunks 自 2017 年 6 月推出以来，价值就保持稳步增长。2021 年第一季度，CryptoPunks 同样蓬勃发展，市场交易额仅次于 NBA Top Shot。

解锁红人效应

很多 NFT 网络红人的参与也进一步提高了 Meebits 及其社区的知名度和参与度。如 NFT 记者 William M. Peaster 为 Meebits 撰写了详细文章。Megacollector Pranksy 宣布在销售当天将在 Meebits 上花费 100 万美元。随着二级市场不断发展，越来越多的收藏家也在社交媒体上分享了自己的购买清单。

一位网名为 Sillytuna 的游戏制作人在接受采访时分享了对 Meebits 未来价值的看法。Sillytuna 目前正在资助并开发 Claymatics 的 Clodhoppers，在游戏行业已经有 15 年的从业历史。Sillytuna 已经拥有 15 个 Meebits，其中最为昂贵的一个购买价格为 100 ETH。

Sillytuna 预计，最为稀缺的 Meebit 将实现增值，但长期收益将取决于数字角色对虚拟现实空间的适应程度。他说，“绝大多数收藏品都是垃圾”，但 “好的设计加上好的发布和好的社区将走向成功”，而收集 Meebits“从某种意义上来说，是理所当然的事情”。

Sillytuna 还认为，游戏工作室和元界开发者需要在角色融入第三方平台之前，对技术问题进行评估。Sillytuna 初期购买表明他的确认为 Meebit 具有短期价值。但其长期价值仍然取决于技术问题是否得到充分评估和考量。

决定 Meebit 价值的团队因素

创始团队背景

Larva Labs 团队是一个 2017 年成立的、由 Matt Hall 和 John Watkinson 两人组成的小团队，总部位于纽约。他们两人最初本以为自己有能力研究研究手机应用和游戏，但最终却成功地为数字艺术市场带来了改变范式，对 “所有权” 概念本身提出了挑战。

Matt 和 John 是创意技术人员，几乎参与了各种软件的开发，包括大规模的网络基础设施、基因组分析软件（John 拥有博士学位）、8 位角色扮演游戏、《纽约时报》涵盖的 “可能会改变我们对数字艺术的看法” 的区块链艺术项目。两人还创建了最大的法律文档开源存储库，一个无疾而终的驾驶游戏（也有 VR 版本），一个名为 AppChat 的安卓应用程序等等。

在多种多样的工作经历中，两人有机会与众多著名机构进行广泛交流，包括麻省理工学院媒体实验室、哈佛商学院、斯坦福法学院、迈阿密艺术学院、哈佛伯克曼互联网与社会中心、纽约技术聚会和伦敦佳士得拍卖行等。Larva Labs 还与谷歌和微软等世界上巨头公司建立了长期的客户关系。

团队其他 NFT 项目

截止 Meebits 发布为止，Larva Labs 主要发布了三个 NFT 项目，也就是我们上文提到的 CryptoPunks，Autoglyphs 以及本文的主角 Meebits。

CryptoPunks

Cryptopunks 是第一批基于以太坊上的 “非同质化代币”，也是 ERC-721 协议的灵感来源，正是 ERC-721 协议推动了大多数数字艺术和收藏品行业的蓬勃发展。2017 年 5 月，Larva Labs 团队向加密社区成员免费赠送 1 万枚 PUNK 代币，每一个 PUNK 都是一副独一无二的像素艺术图像，是算法生成的 24×24 像素大小的数字艺术作品。代币发布后一抢而空。

大多数朋克都是人类角色，其中有 6039 个男性 NFT，3840 个女性 NFT。一些特殊类别，如僵尸（88）、猿类（24）和外星人（9）也存在。每个 punk 都有自己的页面，其中包含有关其功能、所有者、可用性状态和交易历史记录的详细信息。

每个 Punk 独一无二，两两之间并不相同，持有者在以太坊区块链上永久持有。它们都有不同的特征，如肤色、发型、配饰等，使它们彼此完全可以区分，其中还有一些具有极其罕见的属性。编号 #7804 和 #3100 的角色是迄今为止最昂贵的 punk。

Punk 也可以根据属性进行分类。属性对于稀缺性有着至关重要的作用。这些不同属性类型的排列和组合有助于创建独特的数字项目。

Autoglyphs

Autoglyphs 是以太坊区块链上第一个 “链上” 生成艺术项目，让用户可以自主创作和拥有艺术品。

Glyph 是一种生成艺术，每个 glyph 都是独一无二的，由以太坊区块链上的代码创建的。任何愿意将 0.20 亿分（703 美元）的创作费捐赠给 Larva Labs 选择的慈善机构 350.org 的用户都可以创建 glyph。Glyph 的创建者也将成为其第一个所有者。但在创建 512 个 glyph 后，生成器将永远关闭。此外，glyph 只能在二级市场上使用。

Autoglyphs 将项目部署到以太坊，所以平台不再控制生成艺术的代码，也不再控制管理 glyph 所有权的代码。这与传统艺术家和画廊出版销售的艺术有一个至关重要的区别：Autoglyphs 带来了数字物品长期所有权，独立于任何中央机构。

复制过去？还是引领未来？

Larva Labs 团队想要借助 Meebits，在 NFT 市场复制过去的成功。该团队希望 Meebits 最终能成为 “虚拟世界、游戏和虚拟现实” 的化身。Meebits 拥有精细的艺术风格，面向免费市场，具有定制交易的能力，允许用户以更复杂的方式相互交换 Meebits，降低外界进入门槛。未来会将属于 3D 虚拟现实吗？下一个机会爆发点又在哪里？

LoserChick

游戏介绍

LoserChick 是属于社区的开放性游戏平台，玩家可参与到游戏的治理与建设中，实现不同协议的接入，共同打造链上元宇宙世界。

Phantasma (SOUL)

Phantasma 是一个去中心化的内容分发系统，强调隐私和安全，其智能 NFT 有望颠覆我们与艺术、音乐和游戏的互动方式。

Q1：NFT Labs：能否介绍一下你们目前取得的成就？

Phantasma：Phantasma 区块链已经开发了三年多，专注于开发，不那么注重市场份额。我们的主网在 2019 年 10 月推出后，链上推出了十多个 dapp。

Q2：NFT Labs：你们都引入了双代币系统，为什么，这种系统有什么优势？能为大家介绍一下吗？

Phantasma：大多数链都采用单一代币模式，只能用一种代币来支付交易，这给主流应用造成了阻碍。Phantasma 引入了双代币——一个是产生 KCAL 的 SOUL，一个是支付交易的 KCAL。这样一来，用户只需购买一次 SOUL 就能永久持有它，可以生成链上使用所需的所有 KCAL，将系统从"每次支付"变成"一次性支付”。

Q3：NFT Labs：你们认为 NFT 的最终形态会是什么？NFT 应该发展到什么程度才能被大多数普通人接受？你们将为此做出怎样的贡献和努力？

Phantasma：我们不知道 5 到 10 年后 NFT 会变成什么样子，市场会给我们答案。然而这项技术可以在全球范围内解决很多问题。 NFT 势必将颠覆艺术、音乐、电影行业，有望会颠覆房地产和一些基础服务。

Q4：NFT Labs：你们所期待的最终形态是什么？未来会倾向于向哪些领域发展？

Phantasma：Phantasma 是一个社区高度驱动的区块链，正在逐渐认识到应该为企业提供可扩展的定制化区块链。随着世界资产走向数字化，Phantasma 正处于有利位置，推动代币游戏化与智能 NFT 技术的发展。这使得 Phantasma 有望在游戏和媒体领域提供独特的 NFT 社交体验。

NFT Labs 对话 Phantasma

Q1：NFT Labs：Phantasma 这个名字不错，它的含义是什么呢？Phantasma 最终的理想形态和目标是什么呢？

Phantasma：我们的名字意味着现实与幻想的交织。我们的发展目标是将金融模式与数字世界连接起来。我们进行连接，你们进行创造和控制。

Q3：NFT Labs：有一句话是"如果大脑以数字形式传输，是否能代表永生”？NFT 的无缝传输会是 NFT 映射现实的第一步吗？

Phantasma：NFT 代表的是一种体验，而区块链则无限期地承载着这种体验。 NFT 可以永恒地代表一种体验，可以成为映射现实和数字化的基础。

Pepemon (PPBLZ)

同样是游戏相关的 Pepemon 为玩家带来极为有趣的数字卡牌收藏游戏，十分重视艺术质量。自从推出以来，Pepemon 已经推出了很多 NFT，为玩家带来了有趣的游戏体验。此外，Pepemon 的稀缺卡牌极有可能升值，为玩家赚取更多收益。

Q1：NFT Labs：能否介绍一下你们目前取得的成就？Pepemon：我们从 0 开始，头半年有 1500 多个 NFT 持有者和 3000 多个代币持有者，每天都有越来越多的人加入我们，我们的代币和 NFT 的价值也越来越高，这超出了我们开始的预期。尤其考虑到零营销预算推广的情况，只是靠用户口口相传。Pepemon 目前的营收、用户活跃度和发展情况已经超出了我们团队的预期。随着游戏的推出，接下来，我们将专注于项目推广，和合作伙伴一起参加样品会和网游大会。

Q2：NFT Labs：你们都引入了双代币系统，为什么，这种系统有什么优势？能为大家介绍一下吗？

Pepemon：Pepemon 有两种代币，PPBLZ 和 PPDEX，这两种代币有各自的功能，可以出售或购买。PPBLZ 还可以通过抵押产生 PPDEX，用 PPDEX 可以购买游戏道具。之所以引入双代币，是因为我们希望带来平衡，让代币持有者不会因为耕种/抵押/押金而减少其持有量。14，000 个 PPBLZ 的固定供应量十分稳定。代币不会再产生，所以持有者可以放心持有代币。

虽然 PPDEX 只通过抵押 PPBLZ 产生，但在 NFT 生态系统中和即将推出的游戏中，其实用性很强。通过抵押，一个 PPBLZ 可以产生 19 个 PPDEX。购买 NFT 会销毁 90% 用于购买 NFT 的 PPDEX(50% 的 PPDEX 供应已经被销毁)，PPDEX 将来还将用于增援包、游戏中的机制、奖励等用途。

Q3：NFT Labs：你们认为 NFT 的最终形态会是什么？NFT 应该发展到什么程度才能被大多数普通人接受？你们将为此做出怎样的贡献和努力？

Pepemon：我们选择了游戏的道路，在这个领域进行创新，同时也要关注其他 NFT 的情况。就在前几周，我们看到 Uniswap 要用 NFT 作为显示交易对流动性的方式。无论是科技、艺术、游戏还是主流领域，NFT 的潜力都很大。我们预计 NFT 将在几年后成为我们生活的一部分，我们已经习惯于它——如 NFT 音乐，房产证明等，NFT 可以是拥有的任何有价值的、可以在区块链上验证的商品。

Q4：NFT Labs：你们所期待的最终形态是什么？未来会倾向于向哪些领域发展？

Pepemon：我们希望，大家可以在上下班或者睡觉的时候，上 Pepemon 网站对决其他玩家。本质上，我们简单易懂又难以掌握，类似万智牌和炉石这样的游戏，我们知道这肯定是个新游戏，我们会尽量让新玩家和老玩家都能接受。

NFT Labs 对话 Pepemon

Q1：NFT Labs：基于区块链的数字收藏卡游戏，能解释一下为什么选择这个赛道吗？

Pepemon：自 2015 年开始，Pepemon 就在加密领域发展，和一群充满激情的游戏玩家创建了这个项目。我们想把游戏和 DeFi+NFT 技术结合起来，证明区块链可以成为优质的去中心化游戏平台。

Q2：NFT Labs：我们看到了用区块链和 NFT 技术制作 “数字收藏卡游戏” 的优势，但与互联网技术相比，区块链游戏技术有什么缺点？怎样解决呢？

Pepemon：缺少拥有从事区块链游戏工作相关知识的人! 另外，这个领域还很年轻，用于构建这些游戏的技术还很少。我们预计这种情况在未来会有所改变，目前，有很多东西需要我们自己去构建。

还有一个缺点是，区块链交易十分昂贵——比如，玩游戏时需要与区块链进行交互，做 2-3 次交易——如向服务器做 2-3 次请求——在区块链上，这些操作十分昂贵。我们开始放弃速度慢、价格高的以太坊，转而探索像 Matic 和 Tron 等替代区块链来驱动游戏。

Q3：NFT Labs：互联网和区块链存在技术限制。那么，目前互联网和区块链最大的局限性是什么？如何改善？

Pepemon：Steam 等中心化平台使得只有自己能够从生态系统中获利。而区块链使得每个参与代币/生态系统的人都有可能成为平台的一部分。拥有 PPBLZ 和 PPDEX 的玩家在 Pepemon 平台上可以分得一杯羹，但在 Steam 上消费的人并不能从平台上获得任何收益。这就是区块链的优势：平台用户越多，越多支持的用户便能获利。

Q4：NFT Labs：继续上一个问题，如果这些限制和障碍得以解决的话，游戏会如何扩张？

Pepemon：如果有了一个去中心化平台，像区块链一样快速、便宜、易用，游戏玩家就可以直接从 play2earn 机制中获利。

Q5：NFT Labs：大胆猜测一下，如果这些局限性得以解决，它会给 NFT、区块链，甚至是我们的生活带来哪些变化呢？

Pepemon：游戏中的 play2earn 机制已经在改变人们的生活，让人们可以通过玩游戏获得收入，游戏打得越好，收益便会更高。我们已经迫不及待地想看到第一个 Pepe 百万富翁和第一个千元物品的出现。这些物品实际上可以在现实生活中交换东西。

Q6：NFT Labs：肯定会有用户对有稀缺卡牌收集比玩游戏更感兴趣。但从游戏的角度来看，随着游戏的更新发展，老一代游戏卡牌往往会出现弱化折旧的情况，对此有什么保护机制？

Pepemon：随着玩家用卡牌玩的游戏越来越多，他们能够对卡牌进行等级提升，我们鼓励玩家创建多个战斗卡牌，平等地对卡牌进行等级提升，这样他们就能随时备战对决其他玩家，赢得他们手中自己有缺点的卡牌或卡牌组。这种机制非常有趣，会让喜欢胜利的人将卡牌多样化，并提升登记，这样他们就能时时做好反击准备。

Q7：NFT Labs：除了 Pepemon，您认为哪个 NFT 应用最好？

Pepemon：显然，我们非常喜欢其他 NFT 游戏。我们与 Chainguardians 和 DEGO 有密切合作，他们游戏很棒，如果作为玩家，想看看这个领域，它们值得一看。
