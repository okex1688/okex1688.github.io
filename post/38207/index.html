据 Footprint Analytics 不完全统计，目前 GameFi 已有 1400+ 款游戏。在近一年 GameFi 领域中，市场看到了巨大的爆发力，融资不断。

但以 Axie Infinity 为首的 P2E 游戏在经历过疯狂增长后，已经变得索然无味。如今游戏大多都毫无创新，用户也只是机械地挖矿，看到币价下降后再疯狂撤离寻找下一个投机计划。而项目捞了钱后任由币价螺旋下降，大不了再重新新开一个盘。

整个 GameFi 风向逐渐飘向为跟庞氏骗局一样的投机游戏。Play-to-earn 看起来更像是 “play for earn”，是充斥着铜臭的 FOMO 情绪。用户没能像 DeFi 一样干脆直接地获得收益，在 Play-to-earn 上花费了更多的时间，不但没有享受乐趣，反而还跟着死亡螺旋一起沉到海底。

直到在 X to earn 概念下以 StepN 为代表的 Move to earn 模式似乎让行业看到了一个新的方向。更多场景的 X to Earn 可以解救目前的困境，逃出死亡螺旋吗？

X to earn 是什么？

X to earn 指一切能在 Web3 中获得收益的行为。其中 X 指得是用户的某种行为，用户执行了这种行为后可获得一定 token 的收益。与 DeFi 不同的是，X to earn 更强调用户时间的投入，而非资本的投入。

目前 X to earn 概念火热，除了早已熟知的 P2E，在出行、运动、生活、创造方面都应运而生了各种“to earn”的项目：

1.运动和出行相关

  • Move-to-earn

最出名的为 StepN，它把 X to Earn 推到了大众视野。用户使用购买的跑鞋行走、慢跑或跑步获得 token。StepN 将健康与收益、社区联系在一起，获得更多用户的青睐与传播。根据 Footprint Analytics 的数据显示，StepN 上线不到半年价格也呈现不错表现。

Footprint Analytics - StepN Token Price

此外还有 StepN 的各种仿盘，但由于先发优势发展与 StepN 相差甚远。其中 Genopets 将运动与游戏中的宠物升级结合起来，更具有游戏属性。其他项目还有 Step、OliveX、Dotmoovs、GENO、WIRTUA、Movey。

  • Drive-to-earn

HiveMapper 是第一个使用加密技术的行车记录仪，用户可以在开车的同时通过拓展地图挖取 token HONEY。我们生活中使用的地图多来自谷歌这种大企业，在免费使用的同时也在被企业免费获取私人数据。并且这种企业每年需要高昂的成本维护，因为成本问题限制了更新频率和覆盖率。而 HiveMapper 将于今年上线测试网试图解决这一问题。

CPLE（Carpools Life Economy） 想要做一款真正的顺风车 APP，利用去中心化的特点防止控制价格，减低信息发布成本从而减少各方费用。

  • Bike / Ride-to-earn

有  Sweetgum Labs 推出的 MOTE，是世界上第一个让用户可以通过可持续交通方式直接参与碳中和的游戏化项目，用户在乘坐可持续交通工具时将获得 $OCO 代币。目前项目还未上线，官网的介绍不多。

此外还有 BikeRush，用户配备 NFT 后通过户外的骑行获得游戏 token $BST。BikeRush 也以健康生活和绿色环保为理念，联合创始人也都是骑行爱好者。目前项目仍处于基础功能的设计阶段。

  • Exercise-to-earn

同时还有一些综合的运动项目统称为 exercise-to-earn，与上面的 move-to-earn 和 bike-to-earn 有些相似。但 exercise-to-earn 的范围更加广泛，它可能包含以上两种甚至更多的其他运动。

如 Wirtual 中可以通过跑步、游泳、骑车、跳舞和锻炼活动赚取 $WIRTUAL token，获得 token 后还能在商店购买或享受折扣。用户只需要简单的授权手机上常用的健身设备（如 Apple Health 和 Huawei Health）就能获得运动奖励。据官网统计目前月活跃用户超 4 万人。

VICMOVE 也集合了走步、跑步和骑行。目前体验与 StepN 类似，用户需要先再 BSC、以太坊或 Polygon 购买 box，通过运动获得 box 的碎片或 token $VIM。Cycling 2 Earn 的功能在今年 Q2 上线。

  • Meditation-to-earn

冥想作为一项新型运动虽然还不被很多人接受，但这并没有阻止一些人的尝试。它的目的是将参与者从世俗中抽离出来，从而获得头脑的清醒。 

Proof of Meditation 每月制定 21 天的学习周期，用户会先为这个目标质押 token。如果错过一次冥想，token 的一部分将被锁定在池子中。21 天后没有错过冥想的人将收回质押的代币并分配池子中的 token。

ZenCats 也将在今年 9 月上线区块链第一个使用 $AWAKE 和 ZenCat NFTS 的冥想和正念 DApp。

2.生活相关

  • Eat / Drink- to-earn

Poppin 是在 Solana 上的第一款 eat-to-earn 游戏，用户可以收集、对战和养殖 Poppin，预计在今年 9 月上线公开测试版。用户可以通过拍下食物的照片计算一天中总摄入的卡路里和营养均衡情况，越接近标准获得份数越高，以此来升级 Poppin。

Class 属于 drink- to-earn，是一家连锁咖啡店，最早起源于 2013 年的泰国呵叻，如今已有 19 家连锁店。$CLASS 是它发行的数字基础，用户可以使用 token 购买产品和服务。$CLASS 存在于名为 Velaverse 的社区，在此用户可以进行 NFT 和 Land 的交换，也可用户参加各种线上活动。

  • Sleep-to-earn

全世界大量人忍受着睡眠不足、睡眠质量差，这影响着他们在白天中的学习和工作。Sleep Future 旨在改善全球的睡眠质量和健康，根据用户睡眠质量发放 token $SLEEPEE，同时 token 可以在其商店和小镇中使用。项目计划在今年下半年上线 beta 版本的 APP。

3.创作相关

  • Write-to-earn

CrypNote 是一个 Web3 的永久储存笔记和协作平台。它适合 DAO 组织的协作创造、激励、分享、同步，鼓励通过激励池来高效的创造价值和分析见解。同时它没有 gas 费，还将免费获得 Wiki 个人域名，创建 Web 3 主页。但目前他还处于测试阶段。

  • Rap-to-earn

Cipher 为创作者提供了制作 RAP NFT 的工具，未被发现的艺术家可以通过制作 rap 的 NFT 出售获得收入，平台将收取 10% 的佣金。买家可以在试听后通过 ETH 购买，但目前受众还比较小，销量看起来并不高。

  • Sing-to-earn

Seoul Stars 中有两款游戏——在线节奏游戏（DJ-to-earn）和卡拉 OK 游戏（Sing-to-earn）。K-pop 在线节奏游戏通过在合适的时间点击下降物获得分数，卡拉 OK 游戏中玩家将使用麦克风唱歌，评分根据玩家的音高、节奏和音调的准确性。游戏中使用 NFT 将获得更高的收益比率，但游戏还未发布试玩版 。

据 Footprint Analytics 已收录的 X-to-earn 币价上来看，StepN 的币价目前最高，此外最高的为 exercise-to-earn 的 Wirtual，其他 token 都还在早起阶段，币价不超 1 美元。

Footprint Analytics - X-to-earn Token Price

结语

各种 X-to-earn 的概念兴起，但大多项目都还在初始阶段，有些甚至只有概念还未上线。有些其实还是 P2E 只是还了一层衣服，但新鲜的事物往往能吸引人的眼球，用户需要谨防只是炒作概念的项目。

 X-to-earn 将更多的场景与现实生活想结合，让用户愿意开始接触到 Web3 的世界中去。不论这场风潮最终结局如何，在推动 Web3 的进程上  X-to-earn 功不可没。
