在全球范围内， NFT 正越来越多地进入日常使用。从2021年2月开始， NFT开始爆炸式增长，每周交易量超过200万美元，并在数月的时间里的时间里，大型 NFT 项目的总市值增长高达2000%。

自然，由于此类物品的市场正在迅速增长，并且可以成为相当有利可图的收入来源，因此许多投资者寻求从这种热潮中牟利。对许多人来说，这也是一种另类投资，有许多不同的项目可供选择。游戏规则与传统的做事方式大不相同。

我们不排除 Fomo 情绪，但在此之前，也许热潮起飞的部分原因是 NFT 的用例得到了体现。

对于艺术家来说，NFT 带来了表达自由。由于它们不遵循与实体艺术相同的惯例，艺术家以数字方式创作的作品可能更容易实现其心中所想，从而更具吸引力。

Fungible 是指任何可以交换的单位，例如货币。

10 美元纸币可被视为相当于两张 5 美元纸币或五张 2 美元纸币。就我们的日常而言，不可替代的物品不能换成类似价值的东西。就像每个人都会同意蒙娜丽莎是无价的。

任何对象的“价值”都不是由资源数量和创造和生产它的劳动时间来定义的，而是根据其背景和用户的基本原理或观点而变化。简单来说，任何物品的价值都取决于个人购买或出售它。

NFT 最初只是一项边缘活动，现在已经一跃成为主流的区块链应用，成为巨鲸们的另一个游乐场。这些 NFT 可以通过区块链网络进行数字转移，每一个都是独一无二的、独特的物品。

NFT 在艺术家和艺术收藏家、游戏玩家和大型商业品牌中很受欢迎，可以是从艺术品游戏角色或装备、音乐、视频、社交媒体帖子或 GIF 的任何东西。可能性和范围是无穷无尽的。

它为那些精妙绝伦的数字艺术作品赋予了新的意义和目的，并为艺术家和投资者打开了很多新的路径——甚至其中许多数字艺术品的成本与画廊或博物馆中的实体艺术品一样高。

那么，究竟是什么让这些数字物品如此价值不菲？

   什么是价值？

价值的含义有两个主要定义。

首先，价值是一种原则或标准，就行为而言，它是重要或可取的；其次，根据对价值或可取性的相对估计对某事物进行评价或评估。特别是在经济学中，一个对象或服务的价值通常被定义为它会在一个开放、公平和竞争的市场上带来的价格。

市场价格的确定一般是基于对象在社会中的相对供求关系确定的。

   如何评估 NFT 的价值？

NFT 的价值基于三个不同的变量。这些被称为主观情绪、效用和出处。

1. 主观情绪

通常，价值是根据项目的主观或客观价值来定义的。

客观价值是指某人的观点、看法或偏好不会影响真正重要的地方。客观地说，无论你说什么或做什么，价值仍然是价值，蒙娜丽莎不会因为我们少数人的批判而掉价。

相反，主观价值可以根据个人的一时兴起而改变和改变。主观是指一件物品的价值完全取决于某个人的信念、偏好、选择或想法。

例如，观看足球比赛的人可能会花费数千美元购买前排座位，因为他们是足球迷，且认为这场比赛值得。对于那些不是足球迷的人来说，数千美元的前排座位是一种巨大的浪费，对他们来说毫无价值。球鞋爱好者可能愿意花费数万元来买一双限量版球鞋，甚至放在柜子中观赏而不是穿在脚上；但对于多数非球鞋爱好者来说可能无法理解。  

在任何市场上，无论是线上还是线下，所有的物品和商品都根据目标消费者、消费喜好和消费目标设定了一个价值。这个价值是基于主观的，就像一些进入美术馆的人会对某些艺术品印象深刻，并对它们进行主观评价，一些人人则不会这么认为。

这与 NFT 艺术和物品相同。

佳士得 6900 万美元的 Beeple’s 拍卖是 NFT 历史上最昂贵的 NFT 之一 —— “Everydays: The First 5000 Days NFT”。客户知道这件作品价值高达 6900 万美元，并愿意支付，背后的逻辑则是有一位理解并能够感知这件作品的主观价值的客户。

因此，这种主观性和主观价值是定义 NFT 所拥有的实际价值的基础。每个人的价值感知都不同，可能你觉得一文不值的作品有人就视如珍宝。

2. 出处

关于 NFT 经常提出的另一个问题是出处问题。

大多数 NFT 都以数字和在线形式存在，这在某些方面与房屋等东西相比非常困难。当然，这不是指流通和交易记录，而是指它出自谁手；虽然如 OpenSea 等应用已经有打标的功能，但多数 NFT 还是不知从何而来。

出处很重要，因为它可以最终证明一件艺术品的真实性，可以大大增加其价值。在历史上，出处被视为验证物品真实性的最佳方法。出处一直是昂贵物品市场上的价值锚定基础，适用于艺术品、奢侈品和收藏品市场。同样，出处也适用于 NFT。

在 NFT 世界中，数字物品、艺术品或收藏品的价值与创作者是谁或发起者、个人艺术家、主要奢侈品牌或主要体育品牌直接相关——Moonbirds 成功的很大一部分原因就是其创始人强大的 VC 背景。

熟悉 NFT 市场的人，可以看到许多艺术家与全球大品 牌公司合作创作和发行 NFT 艺术品；以及一些非常有影响力的明星、球星等通过 NFT 创造了全新的财富类别。

基于区块链技术，所有这些数字物品的出处、品牌以及它们所来自的 KOL 都可以被准确地定位和追踪；区块链技术上的数字物品附带的区块链证书保证了它们的出处、先前的所有权以及作为可销售物品的完全真实性。

它可以在不失去设计或艺术品所有权的情况下分发，并且其价值得到保证。

    3.实际应用

NFT 的效用价值完全取决于 NFT 的实际应用。

Gamefi 平台、元宇宙和实物结合的支撑是 NFT 实用和真实的示例之一。在整个 NFT 市场中，NFT 的应用用例有很多，从游戏到数字财产所有权再到头像所有权等，选择是无穷无尽的。

4、数字稀缺性

我们不得不承认，现在对于许多将工作发布到互联网上的人来说，任何可以访问计算机或手机的人，都可以免费访问他们的作品。但 NFT （区块链）的到来或将改变我们对开放互联网的概念和理解。

开放互联网的想法对我们影响十分深刻。从某种意义上说，NFT 确实会为本不稀缺的东西带来稀缺性。

有人可能立即反驳说，为什么既然人们想要，还会想让一些东西变得更为稀缺呢？诺贝尔经济学奖得主 Elinor Ostrom 曾经开展过一个有趣的实验，研究了人们在有或没有政府干预下对于自己资产管理的方式，得出了公用品悲剧（tragedy of the commons）的结论——过度消费会给社会中的每个人带来伤害。

互联网和社交媒体带来的自由可能会耗尽内容的库存以及价值。任何作品——耗费了大量时间和精力——一旦放到互联网上，复制它的边际成本为零，作品的价格基本上为零。这意味着，无论付出多少精力，任何人都不会为该作品付钱买单。考虑到这种情况，更多的稀缺性的确会带来益处。

NFT 所具有的二元性正好为这个问题提供了解决方案。有人愿意支付大笔金钱，想要建立与某件艺术品或音乐的联系，但该作品仍然可以公开免费浏览（就像即使可以免费听到歌曲，但还是有人愿意付费购买专辑）。这种二元性既能保留互联网的公开性，又能为原作者带来物质上的奖励。在未来，这或将颠覆人们对于传统互联网公开性的认知，构建人们对于数字稀缺性的更多需求。

结论

正如我们所见，NFT 是真正有价值的商品。

它们将稀缺性强加于本质上并不稀缺的物品，因此可以在市场上获得难以置信的价值。与许多物品一样，NFT 的定价非常主观；它们的价格取决于购买它们的人。

这些物品的来源很容易追踪，可以识别以前的所有者并更好地增加这些物品的价值。

对于所有 NFT 来说，它们的效用只会增加它们的价值；作为艺术品、设计成品、游戏物品、权益证明甚至数字身份，NFT 将成为未来所有加密生态系统的一部分。

当然，NFT 的用例远不止如此，它的最终形态会是什么样子？或许只有时间才知道答案。
