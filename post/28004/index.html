推特名称为niftynaut的创业者联系到我们，想谈谈他的经历。他是最早的Cryptopunks持有者，目前拥有50个以上的Cryptopunks，他认为自己持有的稀有“猿猴PUNK”价值超过12000个以太坊。

问：请介绍下自己，以及你在2017年-2018年是怎么注意到Cryptopunks？

答：嗨，我是@niftynaut（推特）。我是一个早期的 Cryptopunks 持有者。我在 2017 年认领了 50 多个。当时 Cryptopunks 可以自由认领。你唯一需要支付的就是交易的gas费用。过了几天，他们才全部被申请完，这在今天可能有点无法置信。

（niftynaut 的 Opensea 账户）

是什么让我注意到它们：

第一，他们让我意识到以太坊生态带来的实用价值。

第二，我在收藏品（足球卡、小雕像等）中长大。Cryptopunks 对我来说是终极收藏品。当时，这是我所知道的唯一一个做过类似事情的项目。我想成为其中的一部分。

当然，2017年的时候，我不会想到它的底价会是 40 万美元。但我相信了这个项目并看到了它的创新性。

问：虽然PUNK已经非常火爆了，你能否再向中国读者介绍一下。

答：Cryptopunks 是最早存在的 NFT 项目之一。Cryptopunks 是由 larvalabs 发布的，只有 10.000 个可收藏的NFT，所有权证明存储在可用的以太坊区块链上。在区块链方面，Cryptopunks 是创新的先驱。它们是 NFT 宇宙的大爆炸。它们最初创建于 2017 年，是当今 NFT (ERC-721) 标准的灵感的一部分，并且它们仍然是许多加密艺术家和项目的灵感来源。还有很多拥有Cryptopunk的人正在为 NFT 社区构建服务、平台和基础设施。

关于 PUNK 是 NFT 中的比特币。我知道这很难理解，但 1 PUNK = 1 PUNK。从未有过与 Cryptopunks 相似或可比的东西。

问：你会出售还是继续持有？

答：我的头像是 Cryptopunks 猴子。现存仅有的 24 个之一。以目前飙升的价格，我估计仅这个 PUNK 就值 12K ETH 以上。这听起来很诱人——但我不会放弃。我自 2017 年以来一直保留它，我将保留它很长一段时间。

（niftynaut的头像）

问：现在也有人在把 PUNK 碎片化，你怎么看这个趋势？

答：碎片化是与其他人共享高质量 NFT 的一种好方法。我甚至认为，根据某些 NFT 的当前价格路径，碎片化是人们买得起 Cryptopunks 的唯一途径。而且，碎片化后的的 PUNK 基本上是一种货币。我已经在 2020/2021 年碎片化了我的一个僵尸并买回了所有份额。

（新闻背景：推特用户@0x650d 表示决定将其拥有的 104 个 PUNK 通过 NFT 碎片化协议进行碎片化，碎片化后的通证为 FLOOR （地板），总量为十亿，其中有 30%（3 亿）FLOOR 将放入 Uniswap v3 流动性池）

问：你认为目前的 PUNK 潮流是泡沫吗？

答：很多恐惧贩子都称 NFT 泡沫很快就会破灭。但我仍然认为还早。PUNK 在很多时候都被低估了。他们才刚刚开始展翅高飞。我们已经看到像佳士得和苏富比这样的拍卖行以数百万美元的价格拍卖 Cryptopunks 。Cryptopunks 现在不仅仅是收藏品。它们是当代艺术，它们是围绕区块链文化的一部分。它们具有历史意义。

问：你怎么看NFT与游戏的结合？

答：游戏将在 NFT 中发挥重要作用，反之亦然。人们购买一些游戏内物品，想象一下这些物品在多个平台上交易。这是开发者、应用程序商店、公司等的新收入来源。NFT 允许数字项目不仅仅是存在，而是赋予他们价值。

问：你参与了加密猿猴（Bored Ape）的投资吗？你怎么看它？它们似乎抛售了2.8万个以太坊，导致ETH价格下跌？

答：是的，我参与了 BAYC 的铸造，并给自己买了一些 BAYC。我喜欢这个项目，因为它是一个奖励持有者的项目。好点子与伟大的执行。

成功在于社区。与 Cryptopunks 相同但不同。如果您没有足够的财力购买 PUNK，那么 PUNK 是无法实现的。BAYC 最初以 0.08 ETH 的铸币价格出现。人们负担得起。我们看到新一轮的虚拟形象项目不断涌现，因为人们现在也买不起 BAYC了。

我对赚钱的人没有任何意见。这是一个自由市场。

（新闻背景：8月29日早上九点左右，Bored Ape Yacht Club（BAYC）无聊猿开启Mutant Ape Yacht Club（MAYC）变异猿的荷兰式拍卖，每只猿猴的最后成交价约为 2.7 ETH，BAYC 团队也因此获得了 2.8 万 ETH。然而根据链上数据显示，在早上10点半左右，BAYC 团队将约 2.8 万 ETH 转入了标识为 Blockfolio 交易所的地址，Blockfolio 已被 FTX 收购合并，因此本质上立马进行了出售砸盘行为，或许也是当时 ETH 从日内高点 3285 美元闪跌 4% 的主要推手。）

问：中国社区有两种观点：一种是嘲讽，认为很多有钱人，花了几百万美元，买了一张JPG图片，另一种观点认为 PUNK 是 NFT 行业的比特币，BAYC 则是以太坊，你怎么看这两种观点？

答：我们往往被困在自己经验的范围内。当面对未来时，我们不理会它。我完全理解这一点。但出于同样的原因，人们购买 NFT，他们也会购买帆布艺术品、劳力士、一辆漂亮的汽车或其他东西，它具有深厚的文化根源。Y 世代（Generation Y，一般指1980年代和1990年代出生的人）之后的所有世代现在都在数字化成长。那就是我们的生活。我们与朋友数字互动。我们阅读、玩耍、热爱数字产品。在 NFT 之前，基本上不可能证明数字物品的所有权。

我的收藏中既有 PUNK 也有 BAYC。但我不同意比特币与以太坊的比较。我认为说这话的人不了解 NFT。这只是FUD。这总是归结为让社区相互对抗。当 BAYC 进入这个领域时，我们已经看到一些人反对 PUNK 的行为，而且每次都有新项目进入时都会发生这种情况。我认为 larvalabs（PUNK的发起者） 仍然非常具有创新性。他们做了三个项目，这三个项目都在树立标杆，或者至少是对未来项目有所启发。我们看到了 2017 年的朋克、2019 年的 autoglyphs 和 2021 年的 meebits。期待他们能够不断发布这个领域的新项目。

问：您现在在做什么？

答：我辞掉了工作，成立了自己的公司，并在 2 个月前发布了一个 NFT 交换平台。您可以与其他人交换您的 NFT，而不会有遇到骗子和丢失资产的风险。
