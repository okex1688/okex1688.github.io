▌越南成立区块链协会助力国内该领域发展

金色财经报道，作为国家数字化转型计划的一部分，越南已经成立区块链协会，该协会隶属于越南内政部。越南科技部副部长TranVan Tung表示，在全球排名前200位的区块链公司中，有7家是由越南人创办。“我们在区块链领域有10家企业，资本总额超过1亿美元。越南的区块链独角兽也提高了它们的全球影响力。”区块链技术是突破性的。随着工程师和企业数量的增加，他们对联系和共享的需求也在增长。因此，成立区块链协会是必要的。据悉，越南区块链协会旨在发展其成员，建立成员标准，告知公众并提供政策意见等。

▌美国众议员提出法案，要求国会议员披露加密货币交易信息

金色财经报道，美国众议员Elissa Slotkin、Dusty Johnson提出一项法案，要求国会议员披露加密货币相关财务利益。立法者于5月20日提出该法案，当时《加密货币问责法案》（Cryptocurrency Accountability Act）被提交给众议院管理委员会。 该法案将修改《政府道德法》，后者要求公职人员披露其财务和就业历史以及其直系亲属的历史。如果获得通过，它将修改描述必要披露的部分，以包括“该成员或其配偶或受抚养子女在上一个日历年中购买、出售或兑换的任何价值超过1000美元的加密货币相关简要说明、日期和价值类型。”它设定从资产购买通知到披露的45天期限。如果故意隐瞒，可能会导致小额罚款或提起民事诉讼。

▌纽约州比特币矿企由于监管不确定性开始选择撤离

金色财经报道，纽约的比特币矿企正越来越多地考虑放弃在此地拓展业务，因为纽约州立法机构正在考虑通过一项法案，在对该行业的环境影响进行评估之前，禁止使用碳基能源的新挖矿项目。据悉，纽约州参议院正在研究一项法案，该法案呼吁将使用天然气、煤炭或其他不可再生能源的新加密挖矿项目暂停两年。 “由于政治和监管的模糊性，所有加密货币公司都有在纽约开展业务的永久权利。但如果该法案获得通过，纽约将成为该行业永久的‘后来者’，”美国最大矿企之一Foundry公共政策主管Kyle Schneps表示。

▌百慕大制定明确监管框架以吸引更多加密货币项目和公司

6月3日消息，尽管加密市场最近出现动荡，但百慕大正在押注其数字资产监管的透明度能够帮助吸引更多的加密货币项目和公司。 百慕大经济和劳工部长Jason Hayward表示，“我们意识到加密货币价格最近下跌，并仍然相信这不会威胁到百慕大成为加密中心的能力。这次行业低迷可能会推进我们的目标，并对我们在该领域的长期增长和作用产生积极影响。” 据行业专家称，百慕大是全球少数几个制定监管加密货币的全面框架的司法管辖区之一。百慕大金融管理局（BMA）首席执行官Craig Swan表示，BMA已向14家数字资产公司颁发许可证，其中包括2022年获得许可的四家加密公司。 百慕大的许可证要求包括审慎监管，即审查风险管理、治理和网络安全，以及遵守反洗钱和反恐融资法规，如制裁筛选和交易监控。监管机构还使用区块链分析公司Chainalysis Inc.的监控工具来帮助其基于风险的监管。

▌日本通过稳定币法案，未来数月内将推出管理稳定币发行方的法规

6月3日消息，日本议会周五通过了一项法案，澄清了稳定币的法律地位，将其本质上定义为数字货币。根据新法律，稳定币必须与日元或其他法定货币挂钩，并保证持有者有权按面值赎回。至此，日本成为首批围绕稳定币引入法律框架的主要经济体之一。 该法律上的定义实际上意味着稳定币只能由持牌银行、注册转账代理机构和信托公司发行，注重投资者保护。该法案没有涉及Tether等海外发行方发行的稳定币或算法稳定币。据悉，日本的加密交易所没有上线稳定币。 新的法律框架将在一年内生效。日本金融厅表示，将在未来几个月推出管理稳定币发行方的法规。（彭博社）

最早出现在计算机世界中时，VM 的定义是真实计算机的副本。

但VM实际上有不同的种类。第一种称为 System VM ，系统虚拟机（也称为完全虚拟化虚拟机），它可以替代硬件机器。我们可以在上面运行未经修改的操作系统，如 Linux 或 Windows 。VirtualBox 和 VMware Server 就属于这类虚拟机，也是大家比较熟悉的。第二种是 Process VM，进程 VM ，它被设计为在独立于平台的环境中执行计算机程序。众所周知的 JVM 就是这种类型，我们可以在 JVM 上运行 Java 程序。
