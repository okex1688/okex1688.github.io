在本文中，链捕手将对近期（4月1-15日）头部加密风投机构押注的早期项目进行介绍，帮助读者们进一步了解到未来的潜力项目。

Fractal

4月2日，由 Twitch 联合创始人 Justin Kan 创立的游戏 NFT 市场 Fractal 完成 3500 万美元种子轮融资，Multicoin Capital 和 Paradigm 领投，a16z、Animoca 和 Coinbase 等机构参投。

Fractal 是一个基于 Solana 的市场，将作为游戏公司首次发行 NFT 的主要市场，以及用于游戏玩家与玩家之间交易的二级市场，并计划在未来集成其他区块链网络。第一批游戏合作伙伴包括十多个 Web3 游戏，例如 The Sandbox 以及 P2E 游戏 Genopets 和 SynCity。

Fractal 的使命是为数字商品的自由交易创造一个开放的平台。该项目还计划为 NFT 的新兴使用案例创建基础设施，例如一些游戏中的借贷/奖学金模型。

Kikitrade

4月4日，总部位于香港的加密货币社交投资平台 Kikitrade 完成 600 万美元融资，AppWorks 和 Media Asia 联合领投，Audeo Ventures 也参与了本轮融资。

据了解，Kikitrade 提供加密货币投资平台和具有社交和游戏元素的“互动和教育”应用内社区，由亚洲区块链集团 Everest Ventures Group（EVG）孵化，Animoca Brands 为第二大股东。自2020年10月的试行阶段起，该项目用户人数已突破十万量级。

该平台支持用户低门槛地购买、持有与出售加密货币，同时还建立了一个教育社区，拥有大量志同道合的人来分享他们的交易策略和经验。

Polemos

4月4日，P2E 游戏公会 Polemos 完成 1400 万美元种子轮融资，Framework Ventures 和 Delphi Digital 领投，QCP、GoldenTree、IOSG 参投，投后估值达到 1 亿美元。

据了解，Polemos 专注于 NFT 游戏的 P2E 游戏公会，计划开发一个 GameFi 学习管理系统、一个去中心化的公会资产交易所，以及一个用于教育、跟踪、分析和管理玩家的玩家分析平台。该公司旨在通过教授游戏技能或向他们提供代币和游戏物品等初始资源，为发展中国家的玩家打下基础。

Polemos 采用 Discord 服务器的作为公会主体，在Discord 频道中进行日常活动、互动讨论和定期赠品。即将推出的类似学习的进度管理系统，从新手以上级别的 Hoplite 开始。Hoplite 目前占整个社区的 15-20%，并且将能够享受普通会员无法获得的福利。

Strata

4月6日，代币发行和管理协议 Strata 完成150万美元种子轮融资，Multicoin Capital领投，Solana Ventures、Alameda Research参投。

该协议由软件开发者Noah Prince创立，为用户提供了一个无代码的解决方案，可以发行和销售代币，也可以提供代币以换取捐款。Strata Bounties 是一种在链上为事件或任务众筹资金的简单方法。任何人都可以在 Bounties 上存入资金。如果代币是为 SOL 创建的，则需要用户存入 SOL。存款后，用户将收到与其存款等量的代币，代表他们的贡献。

Strata 旨在让开发者更容易在Solana上发行代币，并成为社交代币和联合曲线的标准，将围绕社交代币网络集中资源和构建激励模型。这些可组合的代币机制不仅将推动创造者经济，而且将开启一种新的合作经济，使社交代币网络的价值更加复合。此外，Strata 还提供软件开发工具包，这样用户就可以灵活调整代币的属性。

Bridge Network

4月7日，跨链通信协议Bridge Network完成 380 万美元的融资，FTX Ventures、MEXC Global等参投。

Bridge Network 正在构建一个跨链超级 dApp，以实现区块链生态系统之间的无缝交易。跨链协议（CCP）旨在通过提供用户在多链世界中进行交易可能需要的工具，成为所有主要跨链活动的一站式商店。

BridgeNetwork平台有代币桥接、NFT桥接、跨链交换等功能，正在解决的难题包括破碎的用户体验、集中式跨链协议及其资产扩展。该项目由TDeFi与一系列战略合作伙伴、投资者和顾问共同孵化。该项目主网计划于2022年第二季度初推出。

Taki

4月7日，Solana 生态社交网络 Taki 完成 345 万美元种子轮融资，Coinbase Ventures、FTX等参投。

Taki 是是一个以代币为动力的社交网络，将用户代币与社交媒体功能相结合，以激励和奖励有益于 Taki 社区的活动，内容创建者和用户可以通过参与社区对话获得每日收入。

用户通过评论和喜欢帖子获得加密代币 TAKI，并为那些创建其他人参与的帖子的人提供更多奖励。所有用户都可以在 Taki 平台上创建自己的加密货币，以促进更深入的参与，并随着其受欢迎程度的增长可能增加收入。

Lovo

4月7日，AI语音NFT初创公司 Lovo 宣布完成共计650万美元的Pre-A轮融资，Hashed领投，PKO Investments等机构以及Yat Siu（Animoca Brands创始人）等天使投资人参投。

Lovo 旨在为Web3用户提供NFT形式的AI语音包，以供用户在游戏、应用，甚至Discord中使用。首席执行官 Choi 表示，他将语音 NFT 视为游戏中的物品，类似于角色现在如何改变他们虚拟世界服装。虽然语音转换器在游戏圈中已经存在多年，但 Lovo 提供了一种更人性化、独特的替代方案。

此前1 月，Lovo 推出 8888个 Voiceverse NFT 并迅速售罄，其中每个 NFT 都包含一个独特的声音，该声音由 100 个真实配音演员授权的录音种子创建，每次交易 NFT 时创作者都会获得版税收益。

Joyn

4月8日，Web3 创作者协作平台 Joyn 在其种子轮融资中筹集了350万美元，IOSG Ventures和GBV Capital领投，ConsenSys、Axia8 Ventures等参投。此轮融资将用于扩大平台规模和资助社区发展。Joyn目前正在准备推出其封闭式alpha，首个合作项目将在这里进行孵化。

由 Chestr Labs 开发的Joyn是一个创意中心，可以通过开放式的共同创造来分享、完善和重新混合想法。其核心是为创作者提供从概念到发布的项目所需的工具和社区支持，任何创作者都可以成为平台上的项目生产者。

Joyn为创作者提供项目管理蓝图、Web3技术解决方案以及人才、团队和社区孵化服务。创作者和社区成员可以在Joyn平台上共创、分享和发展他们的作品。除此之外，平台还提供铸造、营销和项目支持等解决方案。

Shrapnel

4月11日，基于 Avalanche 的射击类链游 Shrapnel 通过 Token 销售完成 700 万美元融资，Dragonfly 和三箭资本等参投。

据悉，Shrapnel 是一个 AAA FPS 游戏，其中基于技能的竞争、创意改装工具和真正的数字所有权都由区块链技术提供支持，旨在为玩家带来具有竞争力的第一人称多人游戏。 Shrapnel 提供丰富的玩家创建工具，将战斗、创建、管理和连接结合一起。

Vovo Finance

4月12日，以太坊二层结构化产品协议Vovo Finance获得200万美元种子轮融资，由IOSG Ventures领投，imToken，Kyber Network等参投。

Vovo Finance旨在建立Arbitrum上的 DeFi 结构化产品，解决了期权金库中心化或流动性短缺的风险、选项仅限于比特币 ( BTC )、以太币 ( ETH ) 以及结构性产品种类有限的问题，同时拓展的其他功能，包括风险水平、市场偏见、锁定期、代币存款等。

Vovo Finance 推出了DeFi 领域的首个 PPP：受本金保护产品（Principal Protected Product）。这款产品可以从 Curve 上获得收益，在 GMX 上进行高杠杆交易。后续将推出基于 GMX 代币GLP 的产品以及一系列增加收益的产品。

Blast Royale

4月13日，NFT链游Blast Royale宣布完成500万美元种子轮融资，Animoca Brands和Mechanism Capital领投，Dragonfly Capital、Polygon等参投。

该项目是新一代电子游戏，玩家在“最后一个站立”的大逃杀生存比赛中相互对抗。此外，玩家的元库存会影响比赛中携带的物品的质量。玩家根据他们的表现、投入的时间和整体努力获得奖励。

该平台将在今年二季度末推出测试网，用户将能够为角色装备 NFT 并参加大逃杀，可以获得“Craft Spice”来提升装备。

Core Loop

4月13日，沙盒 MMO Core Loop 宣布完成 400 万美元种子轮融资，本轮融资由 a16z 领投，Alchemy Ventures、CoinList 等参投。

Core Loop 是一个具有玩家驱动经济和区块链集成的下一代沙盒 MMO，玩家可以在这个充满活力、不断变化的世界中杀死巨大的突袭首领并征服领土。该团队正在许多不同领域进行创新——包括设计一个以人工智能为导向的动态世界，游戏的经济将由玩家驱动，参与者能够真正拥有他们制作/赚取的资产，并有可能长期在游戏中谋生。

Elumia

4月14日，Solana 生态 P2E 链游 Elumia 宣布完成新一轮融资，Alameda Research 和 Animoca Brands 领投，Guilds YGG SEA、GuildFi等参投。

Elumia 是一款通P2E大型多人在线角色扮演游戏 (MMORPG)，通过多个不同的世界区域、深度多人地下城以及玩家对玩家的战斗，玩家将体验经典 MMORPG 游戏的多种挑战。拥有角色、设备、土地和建筑物会极大地改变动态和参与度。

Elumia 还推出了一个虚拟世界 Elumia 城市，允许玩家拥有土地并放置和定制自己的建筑物，并在 Elumia 市场上买卖。Elumia 将于 2022 年第 2 季度推出首个 Alpha 版本。
