NFT，可能是今年最具争议的科技创业项目之一。

很多人说是庞氏骗局，是21世纪的郁金香泡沫，也有人认为在疫情期间，NFT成为了一些线下销售IP的“救星”。如果控制了它的炒作属性，就可以成为助推企业发展的新媒介。
争议之下，全球多家明星企业纷纷选择参与其中：媒体行业的纽约时报、新华社；互联网行业的阿里、腾讯；餐饮业的肯德基、麦当劳、奈雪的茶；汽车行业的阿迪达斯和耐克；汽车行业的宝马和法拉利；甚至故宫博物院、湖南省博物馆、俄罗斯冬宫博物馆等数十家博物馆。
需要指出的是，国内的 NFT 和国外（主要是美国）的 NFT 完全是“两个物种”。

纪念阿姆斯特朗登月,美太空部队发售ar太空主题nft商品

国内 NFT 也称为“数字收藏”，不能交易，禁止投机。不论颜值，大部分价格定位在0-100元；国外NFTs，全称不可替代代币（Non-Fungible Tokens），是一种允许投机的可交易数字资产，部分作品甚至被拍卖到数千万美元。
在 NFT 项目中，有些是“打鼓游戏”，比如 NFT 画作《Everydays: The First 5000 Days》，卖出了近 7000 万美元。就连作者本人 Beeple 也直言 NFT 是个泡沫。但也有一些公益项目或品牌推广项目，不以营利为目的，试图为用户或粉丝提供更好的体验。比如新华社以0的价格推出了11万个NFT作品；传奇时刻，开启奥运NFT。

NFT 都是骗局吗？在 NFT 吹起的泡沫中，是否有一些真正有价值的创新尝试？2021 年寻找能够在一定程度上推动传统产业发展的 NFT 项目及其企业。

媒体行业：NFT是一种新媒体
单个NFT参考价：0元
十二月，让中国 NFT 参与者非常惊讶和兴奋的一则消息来自新华社。
24日晚，新华社推出基于区块链NFT技术的“新闻数字收藏”，全球限量11万枚。一位业内人士透露：“新华社发行的NFT是基于腾讯智信链的，本月底或明年初，还将与蚂蚁合作，发布基于蚂蚁链的NFT作品。”
此前，国内 NFT 项目的参与者大多为企业或个人，没有相关国家机构参与。新华社此次的参与，在一定程度上代表了中国的一种“新趋势”——支持和鼓励NFT相关产业的发展。这也将使许多因监管问题而犹豫不决的公司在 2022 年参与 NFT 初创公司。无独有偶，同样是权威媒体的《纽约时报》不久前也发布了自己的 NFT。

新华社的NFT作品

2021 年 3 月，《纽约时报》记者 Kevin Roose 写了一篇专栏文章，以 NFT 的形式被拍卖，价值约 56 万美元（约合人民币 350 万元）。拍卖所得将由记者捐赠给纽约时报扶贫基金会Neediest Cases Fund。
每个国家都有一批历史悠久的主流媒体机构，拥有重大事件报道中的图像、音频、重要人物访谈等资源。比如基辛格秘密访华期间与周总理握手的照片，比如莫言获得中国首位诺贝尔文学奖获得者的专访，比如披头士乐队生前的录音……做成NFT，通过新媒体传播。

以上不是幻想，有些已经成为现实。

目前，披头士生前的采访录音已被一家名为 VOCR（Voice of Classic Rock Archives）的媒体制作成 NFT。VOCR 档案馆收录了 650 个历史性采访和广播节目，其中包括“从 1964 年到 2008 年几乎所有主要摇滚音乐偶像”。他们拥有几十年来没有被公众听到的四次披头士采访的独家音频剪辑。
VOCR总经理Jonathan Furstberg 表示：“Beatles 采访录制 NFT 作为一种新媒体可能会继续存在，我们对这种新媒体将如何发展非常感兴趣。”
体育产业：NFT成为俱乐部生态的“补充”
单个NFT参考价：3179元
体育是一个能够凝聚全球共识的行业，这也是 NFT 能够广泛应用于体育行业的原因之一。当刘翔以12.88秒打破110米栏世界纪录，当科比在一场篮球赛中砍下81分时，人们从心底为他们欢呼。

而体育明星也是拥抱 NFT 最活跃的人群。例如，拳皇泰森的个人 NFT 以 98.7 万美元的价格售出；勇士队球星斯蒂芬库里推出了新的 NFT 系列来庆祝他的 2974 个三分球，每个独特的 NFT 售价为 499 美元（约合人民币 3179 元），库里计划将 100% 的利润捐赠给基金会。利润总额约为 150 万美元。

除了放大个别球星的品牌价值，NFT还可以为俱乐部和球迷提供更好的连接。比如顶级足球俱乐部AC米兰，通过推出NFT，让所有者享有俱乐部的部分决策权。巴塞罗那和尤文图斯等顶级俱乐部也利用这种方式与铁杆球迷建立更好的联系。
除了俱乐部，NBA、CBA、中国足协等赛事主办方也可以通过发行NFT来更好地传播体育赛事。这或许有利于体育赛事和整个行业生态的健康运行。
在国家层面，英国在2021年7月参加东京奥运会时宣布推出以奥运会为主题的NFT，以记录英国队在东京奥运会上的传奇时刻。此前，国际组织国际奥委会宣布将推出以历届奥运会艺术设计为主题的NFT奥运徽章。
影视行业：NFT正在成为电影公告的“标准”
单个NFT参考价：318元
受疫情影响，影视行业损失惨重。与星球大战“比如，它的衍生品占电影总收入的70%以上，主要是线下销售。但疫情让他们的收入大减。

漫威推出蜘蛛侠美国队长NFT

NFT作为数字收藏品，不需要线下销售，成为疫情期间电影人青睐的宣传渠道和创收方式之一。
以即将上映的《黑客帝国4：重生》为例，制作方华纳兄弟与NFT平台Niftys合作推出《黑客帝国》系列NFT头像盲盒进行宣传。单个 NFT 头像价格为 50 美元（约合人民币 318 元）。凭借前三季带来的口碑，粉丝们纷纷为其买单，为制作方带来了500万美元（约合人民币3185万元）的收入。而且成本几乎为0。这不是影视界第一次尝试 NFT。电影《沙丘》、《罪恶之城》、《教父》、王家卫电影《花样年华》、国产动画《精神牢笼》、香港电影《狂怒重罪》等。他们相继推出了 NFT。
在影视行业，NFT正在逐渐成为作品公布的“标配”，让部分IP挺过了疫情的寒冬。
音乐产业：NFT，版权收入的“一部分”
单个NFT参考价：555元
如果说电影业给人视觉上的享受，音乐给人听觉上的享受。巴赫、披头士、周杰伦五月天……音乐有治愈一代又一代的力量。但是，如果一首好音乐只以它的流行度和名气来定义，那就有点不公平了。
许多独立音乐人或唱片公司，他们的风格只吸引一小部分人并赋予他们力量。但是，音乐创作需要物质支撑，不能完全“用爱产生”。比如新裤子乐队的彭磊，作为知名摇滚乐队的主唱，他也曾直言自己靠音乐无法生存，需要跨行业去支撑自己的音乐梦想，背后有是更多的音乐家被现实打败的局面。

NFT 可能是他们增加收入的渠道之一。
《滚石》杂志今年 3 月报道称，Kings of Leon 以 NFT 的形式发行了新专辑，销量超过200万美元。12月21日，为《末代皇帝》作曲的日本音乐大师坂本龙一发布了自己的NFT作品《Merry Christmas Mr. Lawrence》，售价1万日元（约合人民币555元），引起支持的音乐迷。.

美国摇滚乐队King of Leon已把新专辑《When You See Yourself》做成NFT发售

中国近几年才开始重视版权保护的内容。时间短，使我们在版权保护方面还缺乏经验，对创新创造不利。NFT 不是灵丹妙药，但它们是一种互补的解决方案。
服装业：打造“虚拟球鞋帝国”
单个NFT参考价：5000元
从线下场景，到线上场景，再到元宇宙，Nike Adi 一对永远跟风同行。

今年11月，耐克打造了虚拟世界“Nikeland”，成为首批进入“元宇宙”的知名品牌。阿迪达斯紧随其后，12 月中旬推出的 3 万件“走进元界”系列 NFT 作品，每件售价 800 美元（约合人民币 5000 元），在短时间内被狂热的消费者一扫而空。第一次尝试并赢得了近 2200 万美元的收入，这对于阿迪达斯来说也是一笔不小的收入，阿迪达斯上个季度实现了约 5 亿美元的利润。
此外，阿迪还有三个合作伙伴：无聊猿游艇NFT俱乐部；流氓漫画 NFT；AR公司G-Money。他们将融入阿迪达斯的元宇宙世界。

耐克似乎领先阿迪达斯一步，其“虚拟球鞋帝国”已初具规模。耐克不仅成立了自己的元宇宙工作室，还在区块链等领域频繁申请相关专利和商标。据《纽约时报》分析，它还以2亿美元的天价收购了虚拟球鞋公司RTFKT。
盈利能力方面，这家 2020 年 1 月诞生于英国的初创公司，年初仅 7 分钟就卖出了 600 多双 NFT 运动鞋，收入超过 300 万美元。
国内也有一些球鞋设计师也开始做鞋服NFT的生意。他们组成工作室，白天在安踏、字节、宝马等公司工作，工作日休息。或者在周末设计作品并在国内的 NFT 平台上发布。
目前还没有国内运动品牌正式宣布进军NFT，但有业内人士说道，一些服装供应商和平台已经在NFT领域展开了尝试。
食品饮料行业：打造新IP，培养消费者“心智”
单个 NFT 参考价：59 元
在瞬息万变的市场中，人气就是“钱”，这也是国内外企业纷纷入局NFT的原因之一。

今年12月，国产茶饮品牌“奈雪”在品牌6岁生日之际正式公布了元界IP形象：“奈雪”我”，短短1秒就卖光了。

值得一提的是，中国发行的NFT大部分是互联网巨头，奈雪成为国内第一个自主推出NFT数字收藏的消费品牌。
在国外，百事可乐、可口可乐、肯德基、麦当劳也推出了NFT作品。这些消费巨头的玩法也是层出不穷，盲盒、图片、视频、实物……同样的一点是，在百年品牌形象的背书下，他们的 NFT 总能在短时间内被抢购一空。
每个品牌的 NFT 值也间接表明了它们在消费者心目中的分量。
汽车行业：客户不能要，我不能没有
单个NFT参考价：0元
汽车行业也在掀起一股 NFT 热潮。
某区块链公司CEO表示：“NFT更像是一种数据库技术，目前车主的驾照信息等数据都在车管所，行车数据掌握在车企手中。普通人是没有数据的，未来相关单位可以将所有个人数据以 NFT 的形式放到区块链上，NFT 会在汽车出厂时产生，让各方都能很好地利用数据。”
目前，劳斯莱斯、法拉利、保时捷、宝马等主流汽车品牌已经进入区块链，推出了自己的NFT作品。为了在今年 10 月底推广其下一款 Black Badge 车型，劳斯莱斯将同样的 NFT 作为礼物送给了车主。
法拉利更进一步，与瑞士 NFT 公司 Velas Network 签署了一项多年协议，为其粉丝创建数字内容。法拉利敏锐地发现，未来为用户提供 NFT 服务已经成为必然。
在中国，汽车行业也在积极与互联网巨头合作，推出汽车相关的 NFT 数字收藏。不久前，宝马在腾讯NFT平台魔核上发布了500款数字模型合集，已有近3万人报名参与。中国造车新势力智极汽车、长安欧尚也在NFT领域布局。
从目前大部分汽车品牌的动作来看，出售 NFT 收藏品不是他们关心的问题。以 NFT 为纽带，构建自己的生态，吸引更多的消费者，是他们的目标。
房地产行业：一万美元的虚拟房子是天然的“广告牌”
单个NFT参考价：6万元
当房地产被年轻人压得喘不过气来的时候，已经有一大群人在买卖虚拟房地产。
2021年7月，一对1995年代出生的上海夫妇买了一套虚拟房产作为婚房。这种事情越来越多。根据 NonFungible 的数据，在虚拟世界平台 Decentraland 上，仅上周就售出了 443 套虚拟房屋，销售额达 512 万美元。而如果你想在这个平台上买房子，平均要花费10,000美元。目前，国内上市公司天下秀今年推出彩虹宇宙，推出不同档次的虚拟地产。公司董事长李萌今年11月18日介绍，彩虹宇宙拥有超过13万预订用户。

目前，一些传统地产公司也开始布局。12 月 9 日，香港房地产巨头、新世界发展集团首席执行官郑志刚宣布以 500 万美元的价格购买沙盒中最大的数字地块之一。郑志刚将在这块虚拟土地上展示10家特色公司，包括诊断和基因检测开发商Prenetics、物流独角兽Lalamove和科技配件品牌Casetify。
博物馆：NFT成为传播历史文化的“新平台”
单个 NFT 参考价：19.9 元
疫情期间，博物馆冷清，但线上博物馆通过 NFTs Rate 曝光率更高。
目前，国内已有 17 家博物馆通过蚂蚁链 NFT 技术测试了数字馆藏。比如湖南省博物馆的珍品“T字绢画”设计的四件数字馆藏就通过NFT平台引起了人们的关注。这让旧艺术在新技术平台上重获新生.

同样在今年下半年，俄罗斯冬宫博物馆也开启了 NFT 拍卖，拍卖其收藏的 5 幅世界名画，包括达芬奇、莫奈、梵高等大师的作品。
艺术产业：改变传统的创作、营销、拍卖等环节
单个NFT参考价格：6亿元
传统艺术家在纸上作画，随着 NFT 影响力的扩大，越来越多的艺术创作者开始在电脑上设计自己的作品。
这与当前“疯狂”的 NFT 价格有关。数字艺术家 Beeple 的 NFT 画作在今年 3 月以近 7000 万美元的价格售出，而艺术家 Pak 的艺术项目 Merge 以近 1 亿美元的价格售出，与毕加索名画《拿着烟斗的男孩》的拍卖价格相近。毕加索的画作已经走过了 50 多年，而 NFT 的创作仅仅在短短几年内就达到了与世界名画相近的价格，就事件本身而言，这是非常神奇的。
目前，蚂蚁、腾讯等互联网公司仍是 NFT 的“主流玩家”。据虎秀最新统计，国内已经有腾讯、字节跳动、小红书、百度、B站、网易……。至少有9家互联网巨头入局。
一批艺术创作者已与互联网巨头签约，并在其平台上发布NFT作品。其中有传统画家、艺术院校学生、平面设计师、摄影师等。
通过NFT，艺术行业的从业者从创作、营销、拍卖等环节入手，试图改变传统的艺术产业形态。比如顶级拍卖行佳士得2021年售出超过100件NFT作品，总成交额近1.5亿美元，占佳士得2021年当代艺术总成交额的8%。目前，全球三大拍卖行已经售出的 NFT 艺术品被列入拍卖类别。

2021 年，世界的变化让很多人印象最深刻的是，NFT 的出现改变了他们的生活。在柯林斯词典的 2021 年最新报告中，NFT 击败了元界和加密货币，成为唯一的“年度词汇”。
不知不觉中，NFT已经渗透到我们生活的方方面面，衣着（耐克阿迪的虚拟鞋）、食物（可口可乐麦当劳的NFT收藏品）、生活（香港地产巨头新世界的虚拟房子）、旅行（宝马法拉利）等虚拟汽车的推出）……NFT以不可思议的速度在传统行业传播，并试图改变传统行业固有的创造和营销模式，甚至在体育、艺术等方面产生了很多颠覆行业。
目前，国内 NFT 行业整体落后美国两到三年。国内的 NFT 项目大多在今年下半年才开工，这也是国内对它们的认知较弱的原因。如果政策不干预，预计明年NFT项目数量将快速增加并引燃市场，这也成为国内众多创业公司的机会。目前，A股市场的远景中国、天下秀等NFT概念股已经提前感受到了市场情绪。
不可否认，国外的 NFT 已经吹起了一个巨大的泡沫，一些数字收藏品很可能一夜之间从千万美元变为零。但是，并非所有气泡都是郁金香式气泡。还有互联网式的泡沫、房地产式的泡沫、大规模的基础设施式的泡沫。当这些泡沫破灭时，有价值的东西就会被抛在脑后。
在中国，NFT 更像是邮票和纪念币。不同的是，发行人不再是机构，而是拥有独立品牌的企业甚至个人。音乐家、运动员、书法家、记者、小说家……未来，只要人们在某个行业做到极致，真正受到人们的喜爱，就可以通过NFT实现粉丝的回馈。
这种新的激励机制在现实中会遇到巨大的阻力，但当超过这个阻力时，就会带动各个行业乃至整个社会的效率提升。
