我们在互联网上构建的金融创新是变革性的，我们正在释放一种无需许可的超级金融化，它与有许可的过度金融化和政府对货币的垄断是相冲突的，而这种超级金融化是社会经济革命的基础。

下面，我试图在Web3所代表的社会经济变化以及婴儿潮一代与技术创新之间更大的冲突的背景下，来描述Web3的流行语。

婴儿潮一代与信息革命

除了 meme（模因），理解“Ok boomers”（老年人）很重要。婴儿潮一代不仅仅是一代人，相反，它们是一种成长为世界秩序的多代意识形态，它超越了婴儿潮一代。

由沉默世代（注：指出生在二十世纪二十年代中期至四十年代初期的人）建立的信仰体系，在婴儿潮一代的统治下逐渐成熟，一直延伸到X一代和千禧一代。Boomerism 是一种以机构组织为最终目标的意识形态框架。

它已经成为一种痴迷于维持权力的文化，专注于扩大嵌入这种权力的组织结构。飞轮是通过宣传创造的，然后是制度化，冲洗，重复，有机地构建结构。

就像任何曾经存在的既定权力结构一样，婴儿潮一代也陷入了近视。该系统愿意用长期健康来换取今天的权力和财富维护。婴儿潮一代是一个与19世纪欧洲君主制地位相似的世界秩序，其不可持续但愿意与未来发生冲突。

婴儿潮一代的世界秩序不是为了应对信息技术革命而建立的，他们一直希望互联网是假的。逐渐地，到了20世纪10年代，婴儿潮一代开始逐渐理解互联网的变革本质。

我们生活在一个大转变的时代，网络对社会可扩展性的影响是变革性的，而旧的体制正在挣扎。

Web 3 意味着什么？

互联网采用的开端（即 Web 1 时代），主要建立在社区管理的开放协议之上。但90 年代的开放网络并不成功，因为它被 Web 2 的中心化和封闭源代码公司垄断地占领了。

网络的破坏性一开始对婴儿潮一代来说并不明显。早期，它被视为一种愚蠢且无利可图的冒险。当商业模式在 2010 年代初开始发挥作用时，它仍然被误解并被视为经济的边缘部分。

由于 Web 1 时代的吸引力较低，而 Web 2 时代是由公司运营的，直到网络成为政治战场，冲突才浮出水面。Web3在试图治理时与婴儿潮一代的制度存在固有的分歧，其削弱了婴儿潮一代的合法性。

我们生活在 Web 3 区块链原生时代的边缘，其不仅引入了新颖的商业模式，而且引入了一个全新的社会经济组织。但从根本上讲，区块链是什么？

从高纬度来看，区块链主要是“状态机”。它们记录下每个人（通过共识）都同意的“最后状态”，我们正在将（民族）国家与货币分开。

这种新的状态保持机制是价值转移机制的支柱，它不需要银行等第三方，网络的未来不是一个民族国家，而是由运营自己的银行和资助自己的努力的代码管理的在线部落。

婴儿潮一代正在将互联网硬塞进现有的世界秩序，而不是适应不断变化的技术环境。将婴儿潮一代的制度与互联网协调起来是徒劳的，因为Web3与当前的制度结构存在固有的矛盾。

有些人开始投机取巧，他们意识到 Web 3 上的资金流可以成为收入来源。然而，对某些东西征税需要了解它是如何运作的。为了无情地攫取收入，婴儿潮一代的掌权者们对这项技术的了解越来越少。

监管机构正在将 Web 3 塞进方洞，因为对他们来说，神奇的互联网货币是一个古怪的概念。

Web 2 对信息分发（社交媒体）的拆分已经破坏了婴儿潮一代的控制。想象一下，如果神奇的互联网货币成为现实，他们的控制会受到多大程度的削弱。这是对现有货币和金融秩序生存的一种威胁。

但 Web 3 不仅仅是神奇的互联网货币：它是嵌入神奇互联网土地的神奇互联网货币，其中充满了神奇的互联网人。可以肯定地说，Web 3仍然是一个流行词，就像人工智能一样，它仍是水中月，却依旧令人向往。

Web 3 与在线部落

Web 1 发现了网络，Web 2 已填充了网络，而Web 3 将对网络进行分区。也就是说，Web 3将允许不同的社区创建自己拥有主权的网络角落，即Web 3 允许在线部落的分离。

过去是部落的，未来也不例外。Chris Dixon 用一个简单的短语描述了 Web 2：

“为工具而来，为网络而留。”

Dixon 名言中的“网络”是指一个更广泛的社交图，它与传统的Web2社交网络（如Facebook、Instagram或Twitter）相连。Web2提供了一个全球化的基础设施，从中产生了一个通用的社交图。

而Web 3稍微修改了上述内容：

“为获利（Tendies）而来，为部落（tribe）留下。”

部落是网络中的一组节点，在这种情况下，部落应被理解为具有共同性格、职业或利益的一群人。部落的大多数定义是指共享信仰体系或其他某种归属形式的人的网络。不同部落的分离描述了这些联网的集群将如何在他们的网络角落进行交互。

Tendies 是chicken tenders的缩写，这是 Reddit 华尔街投注 (WSB) 交易员用来描述金钱或任何经济收益的术语。“为tendies而来”部分是指这些假名部落背后的经济动机。

当一群互联网巨魔针对机构投资者的空单组织了一场名为 “Gamestop事件"的逼空交易时，巴士底狱风暴再次上演了。

Gamestop事件是新的巴士底狱吗？

这是一个互联网论坛部落对持有空头头寸的金融机构的协同攻击。这是一种反抗的表现，也是对未来的展示。但这只是一个非常原始的部落形式。

加密资产或代币是直接嵌入这些无需许可网络的新型货币，其改变了货币与互联网之间的关系。在Web2中，钱是从互联网上赚来的。而在Web3中，金钱是在互联网上产生的。用户成为了创造者、销售人员、所有者、运营商和客户。

代币是形成 Web 3 的关键因素，并且有可能增强在线部落的力量，代币使联盟正式化并调整部落的激励措施。

财务自由很棒，但这并不一定会带来成就感，但目的可以带来成就感。一个由一群有动机、经济上自由的人组成的部落，由一个共同确定的目标所驱动，这是一种不可忽视的力量。

Bitcoiners、LINK Marines、Ethearans、SNX 持有者、OHMies 等等都是原始的在线部落。

但未来不是每个部落都有一种代币，部落可能会演变成类似于ChaosDAO（不是真正的DAO）或Egirl Capital 的在线投资组织。为了超越邓巴数字，他们可能会通过实际的DAO进行协调，形成规范和规则。

部落不一定数量庞大，但他们将控制大量的金融资本。许多人会在部落之间跳跃，使部落成为一个非常流畅的概念。但是会有障碍和边界——成功的部落可能会有准入制。

此外，还有一个持续存在的公地悲剧问题困扰着区块链协议。为了确保他们的生存，在线部落必须设计可持续的资助创新方法。部落要高效、协调地运作，还有很多困难需要克服。但基础就在那里。

市场正在吞噬网络

初创公司是 Web 2 时代的标志，而token网络似乎正在取代 Web 3 中的初创公司，或者至少变得与“初创公司中的股权”竞争。这是网络部落时代的基础，也是 Web 2 和 Web 3之间主要区别的总结。

Twitter 或 Facebook 新闻供应建立在充满数字噪音的封闭源全球化社交图谱上。Web 3 将允许各种部落从这个全球化的社交图谱中分离出来，通过私人聊天室（今天主要是 Discord 或 Telegram）和放大信号的开源金融网络进行连接。

新闻供应是一个 24/7 的全球迷你才艺秀，而加密网络是一个 24/7 的世代财富追求。关注者和喜欢的数量与你的代币余额无关。首先，我们将媒体游戏化，现在我们将金融游戏化。

在线部落围绕两种货币出现：

  1. 具有横向相关性的上下文无关货币，例如ETH、OHM或Solana，这代表了财务图。

  2. 上下文相关的货币又名 NFT，这意味着它们可能是代表特定在线部落文化所有权的身份象征。但也代表了一些不那么抽象的东西，比如 IP。

NFT 也基于财务图，但带有社交元素。它们似乎也是 Web 2 和 Web 3 之间的连接部分，因为它们可能代表一个原始社会层，并附加了访问经济。

很快，用户将能够对 NFT 发表评论并向所有者发送消息。对于很多人来说，NFT 将成为他们伪匿名的象征，并将他们与他人联系起来。围绕这些独特的数字资产正在形成一种经济，这些资产不会因为稀缺而有价值，但主要是因为它们代表了一种身份。

如果 NFT 成为 IP 会怎样？想象一下，在一个没有许可的研究世界里，在线集体聚在一起解决一个特定的问题。针对特定医学主题的reddit论坛是无害的。如果你让这些集体有能力拥有并资助这项研究，我们就可以拆分生物技术和医药分销。

Web 2初创公司是按照“先用户，后收入，然后股价上涨”的原则建立的，但代币网络似乎建立在不同的前提下：

首先是“数字上升”，然后是用户和收入。

这也可以翻译为“将投机者变成用户”，这描述了新兴网络经济的过程。引导流动性也是所有权获取阶段，这是部落形成的时候。最有成效的对话不会发生在 Twitter 的开放领域。

我们正在从巨大的聊天室转移到更加私密的 Discord环境中。新的社交媒体将更像一套“部落管理工具”（例如，私密聊天室和DAO财政库之间的无缝互操作）。未来的网络将是无许可的，但可能是分区的。

面对最后的boss

灵感来自 The Squad Wealth

Web 2 彻底改变了通讯方式，而 Web 3 增加了无需许可的金融，为部落提供了一种经济组织方式。现在在线集体可以做的不仅仅是聊天，他们可以过滤噪音，因为通过底层状态机的财务图增强了可发现性。资金流动成为了信号。

有趣的是，一旦部落占领了物质世界，它与zf的竞争将变得更为明显。通过 Web 3，在线部落可能会从数字化中重新出现，并做只有zf（和埃隆·马斯克）才应该做的事情。

我预计这些运动将致力于向火星发射火箭、资助核聚变研究、投资生物技术，以及做婴儿潮一代认为不可想象或愚蠢的事情。但这需要时间。

有组织的在线部落时代才刚刚起步，建立一个强大的部落需要的不仅仅是高价。
