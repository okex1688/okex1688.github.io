NFT 用例将会爆发式增长，以下这些方向不容忽略。

NFT 艺术品今年 3 月在一些最大交易市场上的成交额激增突破 2 亿美元，但 4 月的交易额大幅缩水了 50%。在批评者眼中，这意味着「NFT 时代的终结」。要知道，艺术品仅仅是 NFT 用例的冰山一角，我们正迅速看到涌现出 NFT 的新用例。

我们正从创造 NFT 转变为使用 NFT

NFT 创建者的工具已经变得更强大，现在需要所有开发团队实施的是 NFT 铸造平台上的商品化功能 。各个平台现在为创作者提供了交互界面，以确定 NFT 的稀缺性、拍卖机制，以及在以太坊、Flow、Tezos、NEAR 等区块链上进行二级 销售 的版权费百分比。由于创作者工具简单易用，NFT供应明显增加 。

现在除了创建 NFT 变得简便以外，还出现了一些有趣的使用 NFT 的新方法。让我通过这篇文章，重点介绍一下 NFT 覆盖的五个新领域：

赞助人的新福利

NFT 是获取独家内容和机遇的绝妙工具

现在，创作者使用 NFT 和社交代币直接通过其粉丝群变现 。持有这些 NFT 和社交代币的粉丝可以访问独家内容、获得专有的待遇，并可以直接投资于他 / 她喜爱的创作者。

将来，我们将看到 NFT 与社交代币交互的实验，加密原生平台 Patreon、Substack 和 OnlyFans 将会崛起。

当前的具体例子 :

  • 对早期支持者予以物质回馈： Jack Butcher 通过「InfinitePlayers」创建了一种 NFT，与支持他以前 NFT 的每个人分享收入。

  • 奖励早期支持者可以获取独家内容： 美国 DJ  3LAU 为其用户提供可获取他未发行歌曲的专有访问权 ，或与他互动的新机会。想要获得资格购买 3LAU 最新销售的某些作品，用户需要拥有 3LAU 之前发行的 NFT 。与此类似，Micah Johnson 创造的宇航员角色 Aku (ht__tps://aku-dreams.com/aku) 也进行了 NFT 空投，只有以前接受过空投的赞助者 才能获得新空投的 NFT。创作者使用这些机制来奖励早期支持者，并为其 NFT 创建实用性。

预测：

  • 去中心化 Patreon： 去中心化的、加密原生的 Patreon 会与今天的 Patreon 面目大不相同，粉丝们如今都在 Patreon 订阅内容。创作者将使用 NFT 和社交代币，围绕其粉丝社区打造出一个经济生态 。使用新平台的创作者可以无缝地：（1）出售 NFT，（2）为独家内容和图片设立访问门槛 ，与 NFT 或社交代币所有权绑定（3）根据其粉丝拥有的访问代币的类型，可以实时获取流支付（4）根据活跃成员的贡献，将 NFT 销售收入向社区进行分配。我们将看到更多有趣的 NFT 与社交代币交互实验 ，例如，只能使用创建者的社交代币购买 NFT 空投币，或 NFT 赋予持币者稍后可以让粉丝获取社交代币。Roll  等社交代币平台，Collab.Land  _(https://collab.land/)_和 Mintgate (https://www.mintgate.app/) 等基于代币的社区管理平台 ，以及 Unlock _(https://unlock-protocol.com/)_和 Superfluid (https://www.superfluid.finance/) 等流支付服务都将在新的去中心化 Patreon 用例中获得参与机会。

  • NFT 版 Shopify： 当前的 NFT 铸币平台遵循亚马逊模式 ，重视产品，不重视创作者。新型 NFT 平台可能会遵循创造者驱动模式 ，让创作者打造自己的品牌店铺，销售其 NFT。近期涌现出部分允许创作者创建自己店铺的平台，例如 Mintbase _(https://www.mintbase.io/)_和 DAORecords (https://www.daorecords.org/)。

  • 大品牌的企业 NFT： 当今世界部分最响亮的品牌_（例如，阿迪达斯）和艺术家（例如，TheWeeknd）使用 NFT 铸币平台（例如 NiftyGateway）_来销售其 NFT。随着 NFT 成为其战略的重要组成部分，企业会需要自有品牌的 NFT 平台。

联名创作和合作的所有权

粉丝现在还可以创建内容，而不只是被动地消费内容

粉丝和创作者之间的界线将会模糊。粉丝将为 NFT 角色创造丰富的世界，并根据他们希望单独看到的内容或通过去中心化自治组织_（DAO）_来交换 NFT。粉丝将不再是被动的消费者。最大的粉丝将成为创作过程的一部分。

当前的具体例子：

  •  为 NFT 藏品打造工具的社区 NFT： Meebits_(https://meebits.larvalabs.com/)_ 是 Cryptopunk 团队创建的、可收藏的 3D 角色，他们已经有社区成员打造了相应工具，类似如何将 Meebits 改造为 Metaverse 变形产品。部分社区成员甚至形成了 MeebitsDAO 之类的 DAO，创建了工具和管道，使 Meebits 成为 metaverse 中的数字化身。与此类似，Axie Infinity _（https://axieinfinity.com/）_创建了称为 Axies 的可收藏角色，社区可以创建自己的游戏，将这些角色纳入这些游戏中。

  • 收藏者可以对其拥有的 NFT 进行二次创作： AsyncArt （https://async.art/） 是一个 NFT铸造平台，用于创建可编程的 NFT 和音乐。通过拥有一段 NFT 音乐或一部分 NFT 音乐，收藏者可以对其进行二次创作，以使其绝无仅有。

预测：

  • NFT remixing： 粉丝可以将 NFT 音乐、艺术品、视频、虚构产品以及更多文化类产品，通过「remix」创建新的 NFT，同时向原创作者支付版权费。

  • 粉丝创造的世界： 最受欢迎的角色将成为 NFT，可以被放置进粉丝创建的世界中。粉丝将能够为自己喜欢的角色创建 游戏、实现 metaverse 支持，使角色作为他们的化身一样四处走动，甚至可以将角色放到电影、小说、音乐、视频等新的创意作品中，同时会将一定收益回馈 给创作者。艺术家已经在创建 NFT 时考虑到了世界构建的可能性。PUNKS comic （https://punkscomic.com/）  利用 Cryptopunks 创建了漫画。  NFT 持有者可以对 NFT 进行质押 ，或将其焚毁以获得 PUNKS 代币，该代币占到漫画中使用的 Cryptopunks 的一小部分。

  • 粉丝可以对 NFT 进行二次创作： 粉丝能够单独或集体确定某些 NFT 的外观和声音。例如，歌迷可以拥有歌星 Taylor Swift 的专辑封面并更改其背景。随着 NFT 成为游戏、电影或漫画世界的一部分，粉丝可以集体投票 ，能够对故事情节或某些基于 NFT 的角色的举止上进行投票。

  • 所有权和收入纠纷解决： 随着 NFT 创作越来越普遍，且复杂程度不断升高，链上 纠纷 自然会随之出现。协作创建 NFT 的创作者可能会对自己所占的收入分配和所有权比例提出异议。一些创作者可能会在多个平台上创建相同的 NFT。还会有人篡改 NFT 上的元数据。Upshot _（https://upshot.io/）_是评估 NFT 的平台，未来还会涌现出更多协议来验证 NFT 的真实性、调查 NFT 铸造欺诈，以及解决其他链上纠纷。

链上声誉和身份

NFT 可以作为过去行为和忠诚度的唯一标识

人们将根据他们所采取的行动而获得「徽章」等不可转让的 NFT。

人们拥有的 NFT 将类似一个独特的指纹，可以根据他们过去的活动对其进行识别。这一信誉历史记录将用作解锁新机遇的链上身份 。

当前的具体例子：

  • 过往行为的凭证： Uniswap 现在向用户提供 NFT，以证明其头寸。Reality Cards 是一个预测市场，可以让多空两侧的最大持仓者将结果铸成 NFT，使用户能够构建投注历史。POAP _（https://www.poap.xyz/）_是一种协议，使活动创建者可以发出「出勤证明」徽章。Bankless 的社区会议发出 NFT 以证明成员的出席。

  •  成员资格证明： Orca Protocol 向工作组_（如治理协议的补贴发放委员会）_授予基于 NFT 的成员资格。这些 NFT 可以用作解锁链上权限或活动的凭据，类似于预算委员会成员可以执行的活动。

预测：

  • 链上声誉： 链应用将 NFT 所有权作为用户的质量信号 。拥有特定 NFT 的用户将被视为高质量用户，并且可以早期或访问专属的最新应用。例如，拥有某些 Uniswap NFT 的用户可能会意味着他们是流动性提供者_（LP）_。某些 DeFi 协议将为拥有某些 NFT 的用户提供提早访问或更高的支出限额。

  • 根据贡献奖励 NFT 的社区： 可以根据社区成员的贡献向其提供 NFT。例如，可以将特定的 NFT 授予 在治理中最活跃的社区成员、顶尖代码贡献者或在评论区最活跃的社区成员等等。    

  • NFT 所有权决定治理席位： 如果用户具有 NFT 能够证明他们过去投入过资金，或者已经积极参与治理协议，才能有资格加入补贴发放委员会。Discord 服务器中的「角色」也可能基于 NFT 所有权分配。

  • NFT 决定信用评分： 如果用户因在借贷协议中行为良好而获得 NFT，则借贷协议可能会为他们提供更好的借贷利率。如果用户行为不良，则他们可能会收到标有此类标志的徽章。NFT 所有权可用于确定信用度。

NFT 发现和策展平台

寻找、发现和组织 NFT 的平台

类似于 Google 出现之前的 1990 年代网站，NFT 是一种新型数据 ，无法被轻松搜索和发现。未来将会出现强大的平台来策展、组织和推荐 NFT。

此外，由于 NFT 构成了链上声誉和身份的基础，因此我们将基于 NFT 所有权创建社交网络。社交信号将成为我们用于组织 NFT 的功能的重要组成部分。

当前的具体例子：

  • 基于 NFT 所有权的社交网络： Showtime _（https://tryshowtime.com/）_是类似于 Instagram 的平台，服务于 NFT 创作者和持币者。

  • NFT 策展和搜索平台： 像 eBay 这样的传统网络平台现在都在其平台上支持 NFT，这将成为我们搜索和查找 NFT 的一种途径。从 Opensea 等一般市场到 Foundation （https://foundation.app/） 等精选体验平台 ，到以太坊之外的 hicet nunc 、Mintbase 和 Paras.id ，存在大量的加密原生 NFT 市场。现在有大量资源提供更全面的 NFT 市场列表，例如 Sean Bonner 提供的这一市场列表或 Tech Optimist 所提供这一市场列表。

预测：

  • NFT 版 LinkedIn  (该设想贡献者：CuySheffield)： 当我们会因为参加活动、参加 DAO 投票或提供 DeFi 流动性等过去的行为获得不可转让的 NFT「徽章」，社交媒体平台将基于我们积累的徽章而发展。我们可能会看到 NFT 版的 LinkedIn，其中我们的简历只是过去贡献声誉的视觉显示。可以根据其不可转让的 NFT 来识别和招募人员。

  • NFT 搜索引擎 （亚马逊、Etsy 等）： 我们可能会看到更多 NFT 的产品搜索引擎 。亚马逊和 Etsy 都已经有了许多出售收藏品的店主，未来这些平台可能也会支持 NFT 产品搜索。像前文总结的那样，可能会演变出复杂的 NFT 产品搜索、发现和推荐功能。

NFT 金融化

NFT 是金融资产

NFT 代表可用于金融交易资产的所有权 。NFT 有许多新的机制，例如收入「拆分」或 NFT 碎片化，使 NFT 成为更有用的金融资产。将来我们会看到 NFT 作为金融资产的指数级增长的。

当前的具体例子：

  • 为慈善事业和公共利益筹集资金 ： 创作者和艺术家可以通过拍卖 NFT 来为慈善事业筹集资金。多家组织联手创建链上基金，以资助公益事业。NFT 创作者可以承诺将其 NFT 拍卖收益的一部分捐赠给此类基金，当 NFT 出售时，收益将自动支付。现在当 NFT 创作者想为某个特定原因筹集资金时，背后资金的去向便是完全透明的。

  • NFT 可通过碎片化进行互换： NFTX _（https://nftx.org/）使用户可以将 NFT 存入资金池中并铸造 ERC20 代币。B.20 是 Beeple 推出的、将碎片化 NFT 艺术品中的一部分进行「捆绑」。WHALE （http://whale.me/）是一个代币，代表最初由名为 Whaleshark 化名藏家拥有的 13,000+个 NFT 组成的碎片化资金池。Rats Vaults （https://rats.art/）允许用户存入任何 NFT 并获得代币作为回报（提示：特定社交代币 FWB 的持币者有专门的资金库）。以 NFT 为基础铸造可互换代币是一个 解锁功能，因为代币可用于通过 DeFi 协议进行流动性挖矿、改善 NFT 项目的价格发现，并使 NFT 更具流动性，因为它们可以通过自动做市商（AMM）_进行兑换。这些碎片化资金池的成员可能具有特殊权利，例如对资金池中的资产进行买卖的投票权，租赁池中  NFT 的能力，利用资金池所产生收益的 NFT 来收取股息的能力，等等。

  • NFT 发行「股息」： 收藏者可以根据其拥有的 NFT 获得新资产。这些「股息」可以非正式的方式发行。例如，一些收藏者因为拥有著名的 NFT，而从艺术家那里获得了空投的新 NFT，而艺术家希望向这些收藏者推广他们的作品。  要获得「股息」，也可能需要积极参与，例如当 Meebits 推出挖矿流程时，允许 Cryptopunk 持有人赎回新的 Meebits。「股息」可以是将 NFT 赚到的收益进行直接发放。例如，Yieldguild 是 DAO 的集合，这些 DAO 购买 NFT 游戏资产_（例如 AxieInfinity 中的 Axies）_，然后将其借给玩家，这样玩家就无需预先购买资产，并可以从玩这些游戏中获得收入。更多产生收益的 NFT 正在酝酿之中，例如 Aito 宣布与 Aave 集成的 NFT。

  • DAO 购买 NFT 作为投资 ： PleasrDAO  近期以 500 ETH 价格购买了 Tor 的 NFT 。该 DAO 最初创建是为了购买艺术家 pplpleasr 的作品。FlamingoDAO 、 Whaleshark ，以及其它很多 DAO 也都进行了 NFT 投资。

预测：

  • 利用 NFT 产生现金流： NFT 很快将可以赚取收入。NFT 可以产生收入的一种途径是通过土地权利 租金。例如，像在 Cryptovoxels 等区块链平台中的 metaverse 地主可能会向使用其土地的人收取租金 。如今 Cryptovoxels 平台上的地主可以添加允许在其土地上建设但不拥有土地的「合作伙伴」。将来会出现智能合约，地主和租户可以用来确定租金、租赁期限和其他条款。NFT 可以产生收入的另一种途径是通过拥有音乐版权 。例如，music master 可以是每次播放歌曲都可赚取收入的 NFT。粉丝社区可以联手购买他们最喜欢的音乐家的 music master。这种用例已经开始存在：个人正在出租其 Crytopunk，像 reNFT （https://twitter.com/renftlabs） 这样的平台正在试验让个人租赁 NFT，Varda _（https://www.varda.vision/）_正在试验质押用于购买 NFT 的资金，以产生 NFT 收益，Charged Particles 让用户投入一篮子 ERC20 作为 NFT，并在篮子中赚取利息。

  • 碎片化 NFT 插入 DeFi 生态系统 ： 一旦我们将 NFT 碎片化为可互换代币_（例如 ERC20）_，就可以将这些代币投入现有的 DeFi 协议进行流动性挖矿。这些代币可以通过 AMM 进行兑换，投入借贷机枪池等。例如，用户可以通过发行 ERC20 代币实现 NFT 艺术藏品的碎片化，然后通过 Uniswap 交易这些代币。DeFi 协议可能会演变出一些新功能，以优化碎片化 NFT，或者可能会涌现出新的 DeFi 协议。像 NFTfi _（https://www.varda.vision/）_这样的平台已经允许用户以 NFT 作为抵押来获取贷款。

  • 影响力 NFT： 艺术家可以永久性将 NFT 的百分之一销售额投入某种事由，甚至在过世后依然有效。创作者可以创造出「影响力 NFT」，特别支持环保或社会主题。

  • 代币出炉前集资 : 尚未推出代币的项目可以发行 NFT，对早期支持者予以奖励。当项目代币一旦推出，可以将代币空投给 NFT 持币者。

当然，还有很多前沿领域期待发掘。

上面提到的这些设想不过是 NFT 可实现的可能性的冰山一角。近期如果在下面两个方面取得关键突破，将会带来 NFT 用例的爆发式增长，带来我们从未见过甚至未想象过的用例：

  • NFT 所代表的新型资产 ：NFT 的核心是一种通用机制 ，可以表达对任何事物的所有权。到目前为止，我们主要使用它来代表对视觉图像、Gif、加密原生游戏和音乐等的所有权。而新型资产将创造出新的用例。这些资产可能包括来自传统游戏开发商的游戏资产、写作作品_（已通过 Mirror 平台出现）、电影和视频、体育人物、实物商品、房地产、诸如哈利波特或迪斯尼等经典品牌中的人物、资产池（已通过 Charged Particles 出现）_等等。

  • 交易成本低： 如今 NFT 用例主要存在于以太坊上，因此它们的运行受到高昂交易成本的掣肘。当铸造 NFT 或进行交易的成本可能超过 50 美元时，用例可能会受到限制。导致我们多数时候将 NFT 视为奢侈艺术品 。但是，第 1 层_（如 NEAR、Tezos 等）_可以以不到 1 美分的成本价格进行 NFT 铸币和交易。没有高昂交易成本约束的情况下，开发者和创作者将为我们创建我们甚至没有想到过的 NFT 新用例。
