Chainlink旨在为智能合约和Web2系统提供一系列丰富的去中心化服务，创造围绕加密事实运行的世界。 Chainlink独特的预言机共识机制最初用于提供去中心化的数据，而目前正扩展至安全的链下计算和跨链互操作性。

随着智能合约的功能不断发展，以及大众对于品牌背书和机构的信任逐渐崩塌，有越来越多应用开始接入Chainlink去中心化预言机网络（DON），实现信任最小化。这种需求为节点运营商创造了巨大的费用收入机会，并且最终让质押者可以通过为Chainlink预言机网络及其应用保障安全来获得奖励。

Chainlink即将推出权益质押机制，并开启可持续增长的新纪元。这个新的发展阶段将实现更高的透明性以及更强大的加密经济安全性。Chainlink 2.0经济模式将通过一系列服务来加速应用，并持续提升网络的经济效益。本文分析了信任最小化应用的总体潜在市场（TAM）以及预言机的价值；然后阐述了Chainlink将采用哪些增长策略在这个不断扩张的新经济模式中捕获价值。

信任最小化应用的总体潜在市场（TAM）

信任最小化的应用是达到最终理想状态的过渡阶段。交易对手方完全按照协议来执行，参与者无法操纵协议条款或反悔。在大范围内实现信任最小化可以缓解大众对机构的不信任，使用加密手段作为保障，而不是只依靠机构的品牌背书或高成本且反应迟缓的法律体系。这种新型的加密保障基于区块链和预言机网络等去中心化基础架构，通常被称为加密事实。它也是构建信任最小化应用的核心要素。

信任最小化通过技术手段，直接为应用的关键模块提供安全保障，消除差异和不对称性，使交易对手方处于平等的地位。这将有效恢复用户对应用的信任。由于主流聊天软件无法保护用户隐私，因此催生出了端到端加密技术（E2EE）以及Signal等新型聊天软件。E2EE通过加密手段为用户保障隐私，不依赖通信服务商未经验证的简单声明。

要在金融等更加复杂的数字协议中实现信任最小化，通常需要采用混合型智能合约基础架构，将智能合约作为应用层，并将预言机作为服务层。应用层包含链上逻辑，定义了协议条件以及价值交换方式；而服务层则负责连接至各种接口和外部依赖，最终使应用能够运行起来。

Chainlink为信任最小化应用提供了不可或缺的服务基础架构，将应用连接至关键的外部资源，其中包括经过验证的现实世界数据、链下计算以及跨链互操作性等。这些服务为十几条区块链上的几百个应用一共保障了数百万美元的价值，最终推动了混合型智能合约生态的增长。

去中心化金融 （DeFi）就是其中一个垂直领域，利用了Chainlink和混合型智能合约设计模式，快速从小众用例发展成了规模可观的平行金融行业。在2022年，DeFi的总锁仓量（TVL）峰值约为2500亿美元。然而，DeFi在价值数万亿美元的cryptocurrency市场中仅占很小一部分，而区块链和预言机相关技术将有潜力撬动传统金融市场中867万亿美元的价值。

目前，DeFi以及混合型智能合约经济仍处于发展初期，还有很多真实世界的应用场景有待开发，其中包括国际贸易、保险、房地产以及衍生品等。另外，混合型智能合约只是信任最小化行业的一个细分市场，区块链和智能合约技术还可以为链下运行的后端流程保障客观性、去中心化水平以及透明性。

Chainlink作为通用的去中心化链下计算协议，为Web3智能合约以及现有Web2系统提供去中心化服务，实现超高的可靠性、可验证的安全性以及去中心化的准确性，打造信任最小化的应用，并捕获市场价值。Chainlink可以实现以下功能：

  • 让任何Web2系统无需重建就可以升级成信任最小化的版本。

  • 开展安全的链下计算，为链上智能合约实现隐私保护、可扩展性以及可验证性。

  • 提供去中心化的数据，为高价值智能合约触发结算。

  • 为链下抵押品提供证明，提高其透明性。

  • 为智能合约输出关键数据，触发链下系统流程。

  • 在发布跨链互操作性协议（CCIP）后，基于用户自定义逻辑在异构区块链之间传输数据和资产。

由于Chainlink预言机具有通用性，因此Chainlink的目标是打造一个预言机服务层，不仅覆盖所有链上的智能合约，还为传统后端系统提供越来越多的去中心化服务。那么关键问题就是：如何开发预言机服务？以及如何让预言机服务的经济模式变得可持续？

制定行业标准的常见做法

无论是传统行业还是Web3行业，要制定行业标准，常见的做法都是投入资金来支持关键开发工作。项目提供资金来开发创新服务并拓展现有服务的功能，可以很好地满足当前和未来的市场需求。除此之外，项目通常还要对早期用户和服务提供商进行补贴，以聚集网络效应，整合当前的市场份额，并通过推出各种具有竞争力的服务来扩大收入来源。

在Web3行业中，区块链会采用区块奖励作为经济激励机制来启动项目，对网络做出贡献的miner/验证者（即：区块创建者）会获得网络铸造的cryptocurrency作为奖励。这样一来，区块链就可以建立一个安全的区块创建者网络，并防止网络遭受攻击。如果没有区块奖励，区块链网络就会陷入“先有鸡还是先有蛋”的问题，即：

如果区块链没有安全的节点网络，用户就不愿意使用区块链服务；但如果没有付费用户或经济激励，区块链又无法建立安全的节点网络。 

然而，一旦建立了区块奖励机制，并能够正常运作，那么就会形成经济激励机制，为安全的节点网络提供支撑，并为网络最终转型至可持续的经济模式争取时间。最终，网络将可以完全基于用户交易费维持运行。

预言机奖励如何加速Chainlink的应用

与区块链的经济激励模式相同，Chainlink预言机网络也通过预言机奖励在早期提供经济激励。Chainlink的预言机奖励指除了用户费之外付给节点运营商的经济激励。比如，Chainlink Data Feeds是去中心化的预言机网络，将DeFi应用接入高质量的金融市场数据。Chainlink Data Feeds可以基于用户需求，为某一区块链提供具体喂价（如：为以太坊区块链提供BTC/USD价格）。Data Feeds在发布初期是通过用户费以及预言机奖励来维持运行的。

由于Chainlink Data Feeds是公共服务，因此任意数量的用户都可以出资并接入同一个数据。Data Feeds每增加一个新的付费用户，用户付费的总金额就会增加，因此就可以抵消更多成本。最终，一旦用户费用超过了网络的运行成本，运行Data Feeds就不用再依靠预言机奖励，因此就会在经济上完全实现可持续性。实际上，BNB和Polygon等高吞吐量区块链上已经有好几个Chainlink Data Feeds实现了经济可持续性。

为了加速Chainlink Data Feeds以及其他Chainlink服务实现经济可持续性，我们已经推出了许多降低成本的方案，并提高预言机奖励的使用效率。比如，我们发布了OCR（链下报告），有效将Data Feeds的运营成本降低了90%，将传输到链上的数据量提升了10倍，因此极大促进了Data Feeds的数量。随着区块链不断扩容，我们还在开发更多的降低成本方案，进一步推动Chainlink生态的增长，其中包括：

  • OCR 2.0 ：对链下报告协议进行了升级，驱动Chainlink Data Feeds提升数据传输的成本效益，可以部署更多预言机网络并产生更高的用户费。

  • 快速通道（Fast-Lanes） ：区块链会为预言机网络等公共服务分配一部分带宽，或为其建立优先交易排序。这将大幅降低或消除预言机更新相关的链上成本，在区块链网络拥堵期间可以缩短响应时间并提升可靠性。

  • 区块链gas费奖励 ：区块链项目会承诺补贴预言机网络的链上gas成本。这个方案可以向后兼容，不需要对底层区块链网络做任何技术调整。近期会有许多顶尖的区块链生态宣布发布gas奖金。

随着越来越多Chainlink预言机网络实现经济可持续性，预言机奖励将被重新分配，用来创建新的Data Feeds和其他Chainlink服务，为节点运营商提供新的费用收入机会，并为社区的权益质押者在未来提供奖励。这种预言机网络的经济模式保障了用户无需负担Data Feeds的全部运行成本，并围绕Chainlink建立了强大的网络效应。

在这种成熟的经济模式的支持下，十几条区块链上已经成功部署了1000多个Chainlink预言机网络，以满足用户需求。据近期美国银行的研究报告称，Chainlink对DeFi的发展起到了至关重要的推动作用，帮助DeFi实现了2030亿美元的总锁仓量，并将加速区块链未来在金融、保险以及供应链等各个领域的应用。

Chainlink将推出增长计划和权益质押，打造2.0经济模式

随着Chainlink不断实现各种关键的里程碑（比如链下计算服务得到广泛应用、多条区块链上的预言机网络实现经济可持续性、以及在众多区块链上部署超过1000个预言机网络），我们将步入Chainlink网络的下一个发展阶段，那就是Chainlink 2.0经济时代。

为了实现这一重大转型，我们成立了Chainlink基金会，负责发起一系列综合性的增长计划，其中许多计划都会获得新发行的LINK通证作为支撑。Chainlink基金会的核心目标是通过以下举措支持网络的长期可持续增长：

  • 制定LINK通证发行时间表，支持关键增长计划，包括为节点运营商提供预言机奖励，并为权益质押者提供权益质押奖励。

  • 为开源开发者和节点运营商社区进一步制定技术标准、参数以及最佳实践，帮助他们更好地建设和支持Chainlink。

  • 通过GitHub和npm等平台为开发者托管、审核并维护Chainlink的开源软件。

  • 组织并支持Chainlink黑客松活动，鼓励开发者使用Chainlink和Web3技术进行开发。

  • 通过Chainlink奖金计划来推动预言机、Chainlink网络以及相关工具的开发工作。

Chainlink基金会预期在未来9个月中，一直到2023年第一季度，总共发行不超过货币总量5%的LINK通证（即：5000万个LINK通证），以支持网络增长计划，其中包括今年即将推出的权益质押机制。这将是Chainlink 2.0经济模式的重要组成部分。预计权益质押机制将间接抵消一部分通证发行量，这将部分取决于质押金额是否会达到初步计划的7500万个通证上限。Chainlink基金会预期近期内发行的通证量会比较小，并将主要聚焦已经发布的预言机奖励。从2022年第三季度开始到2023年第一季度如果要落地新的关键增长计划，会相应进行调整。

钱包迁移后首次发行的LINK通证将用于已经发布的预言机奖励，以支持现有预言机网络持续运行。之后发行的LINK通证将逐步扩展至以下领域，目的是支持Chainlink网络不断扩张，为更多区块链推出更多去中心化服务，并通过推出权益质押来强化加密经济安全性：

  • 发放预言机奖励 ，降低初始用户成本，并跨所有关键垂直领域实现强大的网络效应。预言机奖励与用户费相结合，将推动节点运营商提供更安全的Chainlink服务。

  • 为节点运营商和参与Chainlink权益质押的社区成员发放权益质押奖励 。这将进一步提高Chainlink服务的加密经济安全和用户保障。

  • 发布新的预言机网络，进一步为开发者提升链上数据集的可用性，以丰富现有的Chainlink服务 。这将有助于提高新型智能合约应用开发的安全性和成本效益。

  • 覆盖更多区块链 ，让Chainlink成为多链生态中主流的预言机解决方案，覆盖L1和L2网络，包括EVM以及非EVM的区块链网络。

  • 开发新的Chainlink预言机服务 ，扩展智能合约经济的功能。开发跨链互操作性协议（CCIP）、公允排序服务（FSS）、DECO以及企业抽象层。

  • 推出其他生态增长计划 ，提升Chainlink的应用范围、安全性、服务种类以及用户体验。增长计划包括Chainlink奖金计划、黑客松赞助、研发计划、代码审计以及bug赏金计划。

Chainlink网络可以通过这些增长计划实现经济可持续性，打造出安全、丰富的去中心化服务生态，助力开发下一代智能合约以及信任最小化的Web2应用。

总结

Chainlink目前已经成长为了区块链行业中保障价值最高、应用范围最广且集成数量最多的系统之一。 Chainlink之所以能实现如此惊人的增长，是因为其具有极高的安全性，并为Web3和Web2开发者提供了各种关键的去中心化服务。Chainlink网络目前已达成了许多关键里程碑，其中包括在多个区块链上实现经济可持续性，以及发布了1000多个预言机网络。随着Chainlink推出2.0经济模式，将迈入全新的发展阶段。

我们非常高兴看到Chainlink网络能够为所有Web3开发者以及希望实现信任最小化的Web2系统提供所需服务。所有的发展都离不开Chainlink社区和生态的不懈努力。我们一直以来都非常感谢各位社区成员的帮助和支持，与我们共同打造基于加密事实的新世界。
