什么是NFT

NFT（Non-Fungible Token），翻译为：非同质化代币，具有不可分割、不可替代、独一无二等特点。在区块链上，加密货币分为原生币和代币两大类。

原生币有大家熟知的比特币、以太币等；代币则是依附于现有的区块链，使用智能合约来进行账本的记录，如依附于以太坊上而发布的Token。

代币又可分为同质化和非同质化两种。

同质化代币（FT），互相可以替代、可接近无限拆分的Token。比如:A有一个比特币与B有一个比特币，本质上没有任何区别 ，这就是同质化，就是同质化币。

而非同质化代币（NFT）是唯一的、不可拆分的Token，如加密猫、token化的数字门票等。也就相当于带有编号的人民币，这个世界上不会有两张编号一样的人民币，也不会有两个完全一样的NFT。常见的票据（发票、网络域名等都是非同质化）。

NFT的关键创新之处在于提供了一种标记原生数字资产所有权（即存在于数字世界，或发源于数字世界的资产）的方法，且该所有权可以存在于中心化服务或中心化库之外。

NFT的所有权并不阻止其他人阅读它，NFT并不是捕获信息然后把它藏起来，只是捕捉信息然后发现该信息与链上所有其它信息的关系和价值。

NFT由于其非同质化、不可拆分的特性，使得它可以和现实世界中的一些商品绑定。

换言之，其实就是发行在区块链上的数字资产，这个资产可以是游戏道具、数字艺术品、门票等，并且具有唯一性和不可复制性。由于NFT具备天然的收藏属性和便于交易，加密艺术家们可以利用NFT创造出独一无二的数字艺术品。

NFT是基于2017年以太坊中一个叫做CryptoPunks的像素头像项目，这些像素头像总量上限为1万，任何两个人物都不能相同，拥有以太坊钱包的人都可以免费领取，领完可以放到二级市场交易。

NFT诈骗类型

随着NFT继续的持续走红，诈骗数量和范围出现爆炸性增长。主要有：仿冒NFT 平台、虚假 NFT 平台、NFT 造假、虚假 NFT 空投和免费赠送的社交媒体骗局。

仿冒NFT 平台

仿冒NFT 平台是常见的网络诈骗手段，用户看起来该网站与官方平台网站一样，诈骗者会试图获取用户的登陆信息或信用卡信息。

虚假 NFT 空投

 虚假NFT就是冒牌，假限量真增发。由于NFT的内容载体是公开的，伪造基本是没有成本的，导致造假者不断。大家一定要仔细确认好发行方账号和NFT记录。

NFT社交媒体骗局

NFT社交媒体骗局鉴于班克斯风格的 NFT（非班克斯本人创作）以高价出售。诈骗分子利用人性的弱点（天下有免费午餐），在社交媒体Tiwtter、微信群、QQ群、贴吧、各个加密货币项目成员的聚集地，分享信息、表达观点、答疑等形式进行信息投放，并声称“官方”、“权威”等字眼。

NFT诈骗防范

NFT诈骗层出不穷，那么如何进行防范呢？知帆科技安全专家提醒大家注意以下几点，提高防范意识：

1. 投资者在投资之前一定要做足功课，冷静的分析，理性投资。可以在相关NFT媒体、社群冷静分析和观察，以免被诈骗份子（托儿）套路。

NFT诈骗的包装套路是上链包装，编造一个故事（比如某名人生前的签名），让参与者形成情感共鸣，推高价格、炒作。对NFT进行反向图像搜索，确保自己要交易的NFT没有出现在其他NFT交易所/市场上。

2. 购买NFT时，尽量选择在大型NFT 交易平台上购买，同时仔细辨别所访问网站是否为交易平台的官方网址，防止访问到假冒的钓鱼网站造成钱财损失和信息泄露。

3. 不要轻易点击陌生人、陌生邮件、社群、社交软件等发送或推送的不明链接，在需要输入账户和银行卡信息的时候要仔细辨认真伪。

4. 使用二元认证或物理代币生成器或基于硬件设备的身份认证应用保护自己的密码/密钥 。

NFT诈骗的打击防范建议

1. 可疑网站或域名的监控、信息收集，建议加强此类网站的信息自动化监控和人工反馈渠道。特别关注数字艺术品、球星电子联名卡、数字宠物等信息监控。

2. 提供官方的举报、检举、鉴别服务，将该类NFT鉴定纳入侦破的线索库，并持续更新、批露给民众。

3. NFT从业经营者网站域名进行强备案监管，与备案相关部门进行合作。网站备案，从业者会产生敬畏感和可追踪性。

4. 加强对NFT从业技术团队进行监管，案情侦破和线索提供进行匿名悬赏，针对新兴领域加大众包技术服务力度。

面对NFT火爆的增长势头，知帆科技认为：一方面建议投资者理性投资，保持冷静，权衡利弊后再行动；另一方面希望相关监管机构尽快为NFT及其他区块链业务建立风险监管体系，规范其发展。
