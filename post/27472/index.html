在过去两周，NFT的销售量激增，远远超过了3月份的NFT热潮。最大的NFT市场OpenSea的交易笔数在上周达到每天超过6万的峰值，接近3月份峰值的8倍。OpenSea的交易量在8月份已经突破了10 亿美元，每日独立买家的数量也创下了历史新高。 

来源：Coin Metrics Network Data Pro

7月底的一些事件似乎引发了繁荣。7月30日，一位匿名买家花费700万美元在一个区块中收购了104个CryptoPunk 。他们有选择地购买了市场上最便宜的104个CryptoPunk，大大提高了底价（“底价”是指集合中最便宜的可用NFT的价格）。他们继续解释说，他们将CryptoPunks视为一种长期的价值储存手段，并作为他们优化流动性和多元化战略的一部分。  

为什么我花700万美元底价买入104个CryptoPunks？很简单：因为我选择财富。

买家还使用MiningDAO将所有104笔交易直接发送给矿工。这意味着在交易被添加到一个区块之前是不公开的，这可以防止抢跑者在交易完成之前买走这些CryptoPunks。在扫购之前，（统计之前的500次销售）CryptoPunk的平均售价约为6万美元。但自购买以来，CryptoPunk的平均售价已增加至接近20万美元。 

来源：Coin Metrics Reference Rates & OpenSea API

自8月初以来，CryptoPunks的交易额呈爆炸式增长。但它之外，还有大量基于动物的NFT项目接管了Twitter个人资料图片。  

来源：Coin Metrics Reference Rates & OpenSea API

Bored Ape Yacht Club (BAYC) 的交易量在宣布其将在佳士得拍卖会上展出后猛增。从那以后，BAYC的交易额达到了百万美元，底价达到了15 ETH以上。除了BAYC，其他个人资料图片集也登上了OpenSea排行榜。在与BAYC的Twitter大战之后，Pudgy Penguins的交易量猛增，在8月13日达到了1350 万美元。

大约在CryptoPunks大扫购的同时，新加坡的对冲基金三箭资本开始疯狂购买NFT。在过去两周，该公司花费了数千个ETH购买 NFT，主要集中在Art Blocks和CryptoPunks。 

Art Blocks是一个生成性艺术平台，它制作了流行的收藏品，包括Ringers、Chromie Squiggles和Fidenzas。在三箭资本开始投资之后，Art Blocks市场出现了爆炸式增长。 

来源：Coin Metrics Reference Rates & OpenSea API

8月9日，Ringers系列中的两个 NFT分别以400ETH和370ETH的价格售出，这是该系列迄今为止的两个最高售价。8月14日，Chromie Squiggle以750 ETH的价格售出，约合240万美元。 

Art Blocks的价格在过去两周飙升。Art Blocks项目有几十个，价格各不相同，但平均售价已高达29,500美元，8月10日达到顶峰。 

加密艺术市场在3月份的高峰之后再次卷土重来。领先的点对点加密艺术市场SuperRare的每日交易量在3月份创下历史新高后正在回升。8月3日，SuperRare上最早的艺术家之一 XCOPY的Summer.jpg以336 ETH售出，约合83.8万美元。

在3月NFT高峰月，加密艺术受到了媒体的大量关注，其中Beeple的作品在3月11日以6900万美元的价格出售。但在今年夏天，主流媒体的关注度并没有达到类似水平，至少到目前为止是这样。事后看来，价值6900万美元的Beeple作品标志着3月份繁荣的顶峰。 

但除了Beeple之外，ETH的价格在3月中旬开始上涨，在4月创下4100美元的历史新高。随着 ETH价格超过4000美元，NFT销售额下降。在ETH大跌之后，NFT的销售开始再次起飞。

来源：Coin Metrics Network数据和Reference Rates

这可能显示了NFT市场和ETH价格之间的关系。NFT通常以ETH定价，因此如果ETH价格突然上涨，新进入者可能会被排除在外。例如，一个定价为2 ETH的NFT可能会突然从大约3000 美元的成本上升到超过6000 美元。对于一些尚未拥有ETH并需要使用法定货币购买加密货币市场的潜在买家来说，这些类型的价格上涨可能会使其变得过于昂贵。 

但NFT市场和ETH价格之间也可能存在一些反身性。人们购买ETH来购买 NFT，然后以更多的 ETH出售他们的NFT，然后经常将这些ETH再投资到更多的NFT。高NFT销售额为ETH带来了新的关注，并有助于将更多用户带入以太坊生态系统。 

在3月NFT繁荣期间，NFT销售额和ETH价格同时上涨，直到市场因供大于求而突然放缓。NFT销售额和ETH价格现在都再次上涨。 

尽管今年夏天NFT的销量猛增，但全球谷歌对“NFT”的搜索量仍远低于3月份的峰值。如果 NFT开始像今年早些时候那样受到外界的关注，那么未来可能会出现更大的繁荣。
