月光鸟：来看看 Moonbirds 真的能够成为下一个蓝筹 NFT 吗？Moonbirds 都干了什么？2022 年，今年 NFT“破圈”与“合规”是 NFT 的大方向，现阶段 NFT 内卷严重，尤其要注意入场的姿势。而 PROOF 推出的 Moonbirds 新贵 NFT 集合，真的可以吗？

“Moonbird 和「PROOF Collective」”NFT 家族

Moonbird：来看看 Moonbird，它究竟是什么？Moonbirds 是由 PROOF 推出的一组以鸟类为基础形象的 NFT 集合。PROOF 则是由 True Ventures 合伙人 Kevin Rose 创办的，是一家聚焦于 NFT 领域的高质量播客。这次 PROOF 再次推出 Moonbird，也许真的可以期待一下。

Moonbirds 拥有四大看点：价值背书、技术创新、社区基础、和财富效应。

1）Moonbirds 的发行人 PROOF 其高质量播客，在 NFT 领域有着相当大的影响力，这就是“价值背书”。

2）持有者还可以让自己的 Moonbird 进入“筑巢（Nesting）”状态，实现“技术创新”。

3）PROOF 还曾发行过另一个 NFT 集合「PROOF Collective」，营建了扎实的“社区基础”。

4）Moonbirds 能以 2.5 ETH 高价公售，就是过往的「PROOF Collective」底板价也有近 20 倍的涨幅，实现了“财富效应”。

Moonbirds 作为 PROOF 推出的首个 pfp 类项目，在通往元宇宙的布局，有着这些令人印象深刻的表现，我们已经对 Moonbirds 满怀期待了。

价值背书：前面提到 PROOF 的内容播客在 NFT 领域有着相当大的影响力，Moonbirds 以鸟类为基础形象。

作为 PROOF 合伙人 Kevin Rose 创办的这家聚焦于 NFT 领域的高质量播客，还铸造过一款「PROOF Collective」NFT 集合。该款权限类 NFT 在 OpenSea 的地板价已涨至 92.5 ETH 的天价，距离当年 5 ETH 的“荷兰拍”起售价已有了近 20 倍的涨幅，而从 PROOF Collective 的持有地址上来看，大多持有者在 NFT 领域也都斩获颇丰。显然，巨鲸们是认可 PROOF Collective 的价值的。现在，Moonbirds 是时候再次得到巨鲸持有者们的青睐。

社区基础 ：这是 1000 个「PROOF Collective」的 NFT 集合，形成的朋克俱乐部。

在 Moonbirds 之前，PROOF 发行过另一个 NFT 集合就是「PROOF Collective」，该 NFT 总数量为 1000 个，二级市场很活跃。该权限类 NFT 持有者拥有着访问 PROOF 私人 Discord、提前收听最新播客、参加线下活动、等特殊权限。在往期节目中，PROOF 还曾邀请过 Beeple、GaryVee、John Crain（SuperRare 首席执行官）、Gabby Dizon（YGG 联合创始人）、Alexis Ohanian（Reddit 联合创始人）、Tyler Hobbs（Fidenza 创作者）等诸多 NFT 领域最知名的艺术家、从业者以及其他各行各业的大咖就 NFT 领域最前沿的话题展开探讨，其内容在业内有着相当大的影响力。这些持有者拥有着访问 PROOF 私人 Discord、提前收听最新播客、参加线下活动、等特殊权限。该款权限类 NFT 在 OpenSea 的地板价已有了近 20 倍的涨幅，现价 92.5 ETH 的天价。从 PROOF Collective 的持有地址上来看，大多持有者在 NFT 领域也都斩获颇丰。巨鲸们是认可 PROOF Collective 的价值的，他们之间建立了紧密的俱乐部。

玩法 ：Moonbirds 也是一款权限类 NFT（当然自身也有 pfp 属性，这点与前者不同）。Moonbirds 在参与抽奖的时候，是有门槛的，要求除了认证推特、Discord 之外，还要在钱包内至少拥有 2.5 ETH。

根据官网描述，Moonbirds 的持有者大致会拥有以下三种权限：其一，Moonbirds 持有者可以解锁一些私人俱乐部的会员资格，包括进入 PROOF 的私人 Discord 参加闭门讨论会等等。其二，Moonbirds 持有者将可以获得 PROOF 接下来将要推出的元宇宙项目「Project Highrise」的先行访问资格。其三，Moonbirds 本身也会有一些特殊玩法，持有者可以让自己的 Moonbird 进入“筑巢（Nesting）”状态。

Moonbirds 是通向财富的另一把财富钥匙，与 PROOF Collective 类似。Moonbirds 的特殊玩法，允许 NFT 在不离开持有者钱包的情况下进入“锁定”（并非完全锁定）状态。当进入筑巢状态之后，Moonbirds 将开始自动积累额外收益，除此之外，随着时间的推移 NFT 本身的形态也会发生相应变化，属于一把财富钥匙。

如何获取：Moonbirds 如何获取氪金 or 赌脸？

由抽签决定！

根据规划，7875 只将通过 Premint 平台以抽签（raffle）形式分发给 7875 个受许可地址，每个地址仅可铸造一只，如果出现卖不完的情况，将会继续抽签。2000 只（20%）会分配给 PROOF Collective 的持有者，每人将有权免费铸造两只。Moonbirds 总量合计 10000 只，其中 PROOF 官方会自留 125 只（1.25%），用于未来的合作、营销。

Moonbird 破圈真实表现

破圈 ：你之前听说过「PROOF Collective」这个 NFT 集合吗，哪怕仅有一丝印象？

“破圈”与“合规”属于 2022 年 NFT 的大方向。当初 20 位匿名艺术家加持下，Proof 的成员每人可以挑选一件来铸造自己的 NFT。在所有作品都被铸造后，艺术家的名字才会被公布。作品背后的故事和艺术家的面纱，让所有人都震惊了。这样的安排，当时还是吸引了普通收藏者、艺术爱好者的广泛关注。

现在吃瓜群众，贪欲游戏的逻辑，低位买进，高位卖出。再加上现阶段 NFT 内卷严重，Moonbirds 真的能够吸引更多缺乏艺术鉴赏的普通玩家吗？

恐怕还是需要注意入场姿势哦。

要说“会面”破圈，虽然 PROOF 一直在提供与创作者的当面交流的真实空间，但它的门槛对普通玩家的不友好，会相对的“小众”。不过，这个属于与流行创作者进行真实世界体验（例如：会面、会议）的重要途径，朋克俱乐部的聚集效应还是相当不错。

安全 ：新玩法体现出来技术安全创新点——Moonbirds 的特殊玩法，允许 NFT 在不离开持有者钱包的情况下进入“锁定”（并非完全锁定）状态。

当进入筑巢状态之后，Moonbirds 将开始自动积累额外收益，除此之外，随着时间的推移 NFT 本身的形态也会发生相应变化。持有者可以让自己的 Moonbird 进入“筑巢（Nesting）”状态。进入筑巢的锁定状态后，Moonbirds NFT 将无法正常通过 OpenSea 等二级市场交易（会在买方下单时提示），但依然可以在 PROOF 所提供的网站内通过亲手签名的形式转移至其他地址，这一设置的目的是为了防止一些常见的远程函数调用钓鱼手段。

notDAO 技术社区会为你分析：“利用远程函数钓鱼”，实现安全转移。NFT 转移跟 ERC20 token 一样需要先授权这就意味着有些钓鱼手段可以骗取用户先授权，然后转走用户的资产，黑客在这里的做法，就是用户可以关闭 transferFrom 这种转移方法，用非 NFT 的接口方法实现 NFT 资产的非法转移。

当前 NFT 资产安全，颇有点当年传奇游戏的时代，玩家装备经常被盗取的网络技术安全初级时代。notDAO 技术社区分析认为，此次技术创新最大这亮点，就是加强对已有 1000 个高价值 NFT 资产的保护，面向 PROOF 整个现有社区，是一只脚立足强化原有「PROOF Collective」项目的已有资产安全，另一只脚为 Moonbirds 等新项目做整体安全升级。

pfp 类项目 ：这个 PFP NFT，指封装用户个人资料头像等元数据的 NFT。拥有未来元宇宙项目「Project Highrise」的先行访问资格。在新领域 GameFi 和元宇宙中拥有的多种应用和延伸。

里面的社区热度，采用的都是相对成熟的 NFT 应用技术，主要聚焦在拼社区热度。从“筑巢玩法”作为一大创新点，强化了防远程函数钓鱼，就能够窥到这是加强对原社区持有者的资产保护，和资产价值巩固。且可玩性自娱性强，notDAO 认为这算是区块链时代很细微的一次创新，基本是社区的自娱自乐。

不过新进玩家，还是取决于对于其社区长期建设的认可度，和短期财富效应。Moonbirds 是 PROOF 向元宇宙发展过程中，有着承前启后的作用，前后衔接的项目，为后续发展蓄力，而这次项目总数较上次放大十倍已达 1 万个。

元宇宙 ：PROOF 是把元宇宙拆分成了 N 个项目，一个项目又拆成 N 个阶段，然后每个阶段卖一批 NFT 出来，每一批 NFT 在技术创新上解决部分问题，用 NFT 融资和拉热度，巩固基本社区后，又发下一批……不断循环。循环产生的新旧 NFT，也就是在新旧 NFT 的中间，不但一直要有热度，需要人出来讲故事，给内容，团结社区，实现技术创新。还需要在 2022 年 NFT 破圈，才有机会走向元宇宙的道路。

Moonbird 开发现状和未知规则

开发进度 ：PROOF 设置了未知规则，一是 PROOF 官方尚未宣布筑巢奖励的具体规则。二是进入筑巢的锁定状态后，Moonbirds NFT 将无法正常通过 OpenSea 等二级市场交易（会在买方下单时提示），但依然可以在 PROOF 所提供的网站内通过亲手签名的形式转移至其他地址，这一设置的目的是为了防止一些常见的远程函数调用钓鱼手段，不过它们的开发进度还没有看到官方公布，现在只知道做了一个功能，Premint 已上线了相关抽奖界面，参与抽奖的要求除了认证推特、Discord 之外，还要在钱包内至少拥有 2.5 ETH。每个地址仅可铸造一只，如果出现卖不完的情况，将会继续抽签。推测将要推出的元宇宙项目「Project Highrise」的先行访问资格，还没有看到相关开发进度，但是技术保障是有的。

新人 fomo ：提醒大家，Moonbirds 是否就是下一个 NFT 蓝筹？

它的主要玩法其实还是拼原有社区热度阶段。新进玩家，还是取决于对其社区长期建设的认可度，和短期财富效应。

还想要提醒大家，NFT 仍是个高风险的行业，鉴于 Moonbirds 2.5 ETH 起售价并不低，且临时改变公售机制的举措（主要矛盾在 2.5 ETH 从顶价变成了固定价格）已招来了一些不满情绪，而且能够制定抽奖玩法，代表已经拥有扎实的社区基础，大家在投资决策之前请务谨慎。

关于风险，纵观市场，除了暂时还看不出苗头如何的全链式（ominchain）NFT 外，近期足够亮眼的新项目可以说是少之又少。而它的社区热度，采用的都是相对成熟的 NFT 应用技术，主要聚焦在拼社区热度。从“筑巢玩法”作为一大创新点，强化防远程函数钓鱼，能够窥就到这是加强对原社区持有者的资产保护，和资产价值巩固。（notDAO 技术社区认为这算是区块链时代很细微的一次创新，而且基本是社区的自娱自乐，参与时要有所了解。）

结束 ：PROOF 拥有 NFT 领域最知名的艺术家、从业者以及其他各行各业大咖的高质量播客，新推出的 Moonbirds 也是属于 pfp 属性的权限 NFT。

其创造的 NFT，与拥有者的相似度、和美学观感双重吸引下，相信它前景可期。
