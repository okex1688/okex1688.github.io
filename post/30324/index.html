1.《福布斯》发文谈NFT税收问题

10月30日消息，《福布斯》于10月28日发文谈NFT税收问题。其中提到：尽管美国国税局（IRS）尚未发布任何NFT的具体税收指导，但大多数基于艺术的NFT（如CryptoPunks）都可能根据IRS§408(m)(2)(A)被归类为收藏品。此外，文中提到，用户持有一项资产不到12个月即卖掉会产生短期资本收益。对于最高纳税等级而言，短期收益最高纳税比例为37%，年收入超过一定门槛的还需缴纳3.8%的净投资所得税。用户持有一项资产超过12个月后卖掉会产生长期资本收益。加密货币的长期资本利得税最高为20%。同样，年收入超过一定门槛的需要缴纳3.8%的净投资所得税。高收入者的长期NFT收入最高征税比例为28%。

不仅是NFT，虚拟货币的应税问题也由来已久，随着虚拟货币市值的扩大，该问题已引起了世界各国的普遍重视。不少国家也建立了专门的税收监管政策，如德国是第一个对虚拟资产予以课税的国家、美国是税收相对完善的国家。

以美国为例，美国国内收入局（Internal Revenue Service，以下简称“IRS”）将比特币一类虚拟货币视为财产，规定利用虚拟货币支付适用于其他物品交易的一般税务原则。2018年， IRS再次提醒纳税人，加密货币交易应当和其他形式的财产一样，是需要纳税的。如未进行纳税申报，不仅可能对罚金及利息承担责任，甚至受到刑事起诉。任何被判有逃税罪的人最高可面临五年监禁和25万美元的罚款。

2.NFT税收的必须性

2-1.NFT公司希望通过收税激励创作者

今年上半年 NFT 的迅速蹿红顺利接过了去年 DeFi 浪潮的接力棒，在 NFT 市场今年已经刷新各种记录的背景下，今年取得的现象级爆发，NFT 藏品天价成交的消息频现，顶级机构乃至传统行业巨头纷纷布局甚至下场参与其中，而作为 NFT 市场重要基础设施的 OpenSea 平台的数据，也很好地描绘出了这波颇具震撼力的增长。

随着新的浪潮推动 NFT 概念的不断出圈，并吸引越来越多的投资者真正参与其中，如何在众多 NFT 市场平台应用中选择最适合自己的那一个，无疑是众多 NFT 市场新人必然会面对的一个问题。随着 NFT 市场的不断发展，也有部分平台开始寻求优化版税体系，为 NFT 创建者带来源源不断的收入的同时，在后续转售中也为经手的所有藏家提供收入，以此持续激励 NFT 的流动性和驱动市场达到供需平衡。

2-2.NFT市场希望税务部门征税

NFT市场蓬勃发展，但是随着天价艺术品频繁推出后，很多人眼红市场开始出现很多混乱现象，这些“虚假空”项目的出现严重的影响了NFT市场的正常发展。

NFT目前集中于知识产权、艺术品等领域，且热度居高不下，一众名人骚客纷纷打造自己的NFT资产。因为NFT没有明确法律约束和监管规定，所以NFT交易存在交易双方权利义务无法明确、保障的窘境，利用新技术、新应用从事违法犯罪活动等风险。

如果美国各个监管机构开始对NFT行业收税，会进一步规范市场乱象，有助于市场的长期发展。随着NFT市场规模越来越大，为其时代到来提供了坚实的法律基础。

3.NFT税收顺应时代发展趋势  

NFT 的未来在于把全世界亿万个社区的共识价值化，并且它正在以不可抗拒的步伐前进。艺术、游戏、版权等都只是这个大的未来中的一小片图景。很快，有影响力的人，IP 方，组织就会开始铸造和分发自己的 NFT。

美好的愿景需要相应的制度作为基石，而收税是其第一步。NFT赋能实体产业，必须得到政策的支持与鼓励。未来各国随着NFT市场发展都将陆续出台相关政策，让NFT市场生态能够健康的发展壮大。
