不同城市因其不同的特点、独特的优势吸引着有不同需求的人们前来观光旅游以及定居。

这与不同的区块链十分相似。以太坊，第一个能够以智能合约形式实现顶层应用的平台，他的先发优势为其积累了首批 Web3.0 创新开发者。 所以 DeFi，第一个创造了财富效应且突破了关键规模的去中心化应用在以太坊台落地也是水到渠成。城市经济体量的攀升，经济活动频次增速的迅猛导致城市内资源短缺，供给端的缺失让过剩的需求无法得到满足，城市居民消费价格指数居高不下。虽然这里的搭建者和原住民们大多享受到了这座城市增长所带来的红利，也选择了留下，但其他人都选择去哪了呢？

许多后期进入到以太坊城市中的搭建者及新打工人的首选会是类似于 BSC 这类的中心化以太坊分叉链，虽然这里并不是这么民主，但这里拥有着与以太坊一样的地质条件且更广阔的土地面积。开发者们在这片土地上建设的得心应手，打工人们也在这片沃土上以极低的生活成本做着与以太坊上相同的工种。这类城市往往很快就能完成自举，并进入到成熟期。

然而看看那些在海边以及丘陵地区的城市及村庄，他们的发展仿佛就要缓慢许多。虽然环境宜人，地质条件优越，但在这些地区进行搭建的试错成本过高。正如 NEAR、Solana 以及 Avalanche 这一批新公链，技术上的优势让他们能够为用户提供极佳的使用体验，但开发者数量的稀缺让生态的搭建速度十分缓慢，太多的不确定性让开发者们望而却步。

特色小镇一直是人们选择出游度假的首选地，因为他们能够将特色做到极致，给人们带来无法取代的感受。应用链就是这样，若想把将某种应用做到极致，不向通用框架妥协，那么也只有应用链能做到这一点了，有许多东西是智能合约平台永远无法实现的。若想真的步入 Web3.0，应用链的地位将不容小视。

那些以太坊原住民们会时不时到不同城市里游玩，也可能出差一段时间，甚至有一天搬迁到另一个城市生活，因为这里能够满足他所有的需求，他属于这个城市。

本文出自加密顶级风投 Dragonfly Capital 的 Haseeb，文中将区块链比喻为城市，并就城市的扩建方案进行了综述，最后将不同区块链与不同城市进行了类比。

未来将会是一个多链的世界，还是会形成「一链治天下」的局面？

这取决于你对区块链的思考方式。

人们通常将 L1 区块链称为网络，如 Ethereum 网络、Solana 网络等等。这也就意味着，区块链和互联网、Telegram 或 Facebook 一样，可以无限化的扩展。不过，如果区块链是网络的话，那么其所带来的网络效应将最终只能让一条链胜出。

但事实上，将区块链视作网络并不准确。区块链会受到物理限制，而且需要大量独立的小型验证者，因此不能无限扩展其区块空间；而如果区块变得异常庞大的话，那么区块链将不再是去中心化的。

智能合约链和我们的城市非常相似。一旦你接受了这个设定，就不会再觉得 L1 区块链的原理有多神秘了。

人人都在谈论 Ethereum 的各种不足——它要价高昂、容易堵塞、延迟缓慢。自很久之前创建以来，它就一直都没达到人们的预期，而它也没做出任何调整和改变。Ethereum 的交易费用之高，让除了富人以外的交易者都望而却步。

Ethereum 和纽约这座城市甚为相似。

图片来源：IntroducingNewYork

纽约作为美国的经济和金融中心，汇集了全世界所有规模最大的银行、数量最多的亿万富翁、最热门的品牌以及各大明星。同样，Ethereum 也拥有所有主流的 DeFi 协议、最高的总锁仓价值、最热门的 DAO 和 NFT。

但它仍然相当昂贵，所以如果你是一名新人，那你很容易就会被淘汰出局。不过但凡你在早期购置了资产，那你可能已经发财了。但如果你现在才入局，那你不仅负担不起上面高昂的费用，还会发现上面没有能容纳所有人的空间。这对于亿万富翁们来说可能不算什么，但我们下一代的年轻人将不得不另寻去处。

所以，你打算如何扩大纽约市的规模？

扩大城市规模主要可以通过以下三种方式：

方法一：建造高楼大厦。虽然土地面积是有限的，但你可以在有限的空间上加高、建高。一个高楼林立的城市一定可以在同样的土地面积上容纳更多的人。

图片来源：Erik Pendzich/Rex

但是，建造高楼大厦并不是一个一劳永逸的解决方案，因为摩天大楼的高度也有其限度，而且即使是摩天大楼也无法摆脱底层城市的拥堵。假设你我都是曼哈顿的高层住户，但分别住在两座高楼里，那如果我想去拜访你的话，我就必须下到一楼，打一辆贵价的出租车，并在曼哈顿拥挤的车流中穿梭之后才能到达你家。尽管我们在曼哈顿有了住所，但还是无法逃避其堵塞的交通。

L2 和 Rollup 就相当于区块链世界里的摩天大楼，而每个 Rollup 也都是在底层 L1 链的基础上搭建起来的区块链。Rollup 有着很大的容量，但如果要从一个 Rollup 访问另一个 Rollup，你也必须退出到下层的 Ethereum 上，并和其底层流通量打交道。

虽说建造高楼确实能让更多的人住进城市，但这并不是一个完美的方案。如果 Ethereum 现在已经非常拥挤了，那就算有了 Rollup 它也依然会很拥挤，因为那群亿万富翁们有钱支付 L1 上高额的交易费用并留在 L1 链上。

所以说，还有没有别的办法来扩展区块链？

方法二：利用 Polkadot 或 Cosmos 这样的「可互操作网络」 。

Polkadot 和 Cosmos 会为开发者提供 SDK，以便他们推出特定用途的区块链，也就是一个专门用于单一应用的小型区块链。所有这些区块链都由一个路由系统所连接，比如 Polkadot 链上的 Relay Chain，或者 Cosmos 上的 Cosmos Hub。

类似的，这就好比在城市中搭建功能性城镇网络，有的负责采矿，有的负责加工，有的负责种植，有的则负责出售，而它们则都是由城市的大型公路系统所连接起来的。

Sim City中的一个工厂小镇

这种做法确实奏效，也的确会有一些地方按照这种模式进行建设。但是，加工和种植的功能分区并不能成为人们生活和做交易的主要场所。也就是说，要想容纳不断增长的人口，你不能只光顾着建造这些城镇。

所以我们只剩下第三种，也是最后一种扩大城市规模方法：再建一座城市。

这也是 Solana、Avalanche 和 NEAR 所做的事情。

当你在建设一个新的城市时，你首先需要做的是借鉴以往的基础设施。这种做法看起来似乎很是累赘，毕竟每座新的城市都需要建造一批新的道路，全新的警察局、学校和医院。同样，每一个新的 L1 也需要新的一批区块浏览器、法币入金通道、原生 AMM 和 NFT 市场。所以说，虽然它看起来有些多余，但每一个 L1 都需要这些基础设施才能加以发展。

但建造一个新城市的好处是，每个城市都可以用不同的方式进行建设。

以 Solana 为例，它就像是洛杉矶。与曼哈顿相比，它不仅面积广阔，且消费较低。即便你是一名底层演员，也可以在洛杉矶过得很好。你不用太在意去中心化，只需要把你的应用程序移到 Solana 链上，发布自己的 NFT，然后就能小火一把了。

当然，Solana 不是最为去中心化的链，但是去中心化也不是游戏和 NFT 发布的必需条件。既然这里气候宜人、消费不高，没人会真的斤斤计较。

那 Avalanche 又像是那座城市呢？在我看来，Avalanche 就像是芝加哥：它意欲成为下一个华尔街，但要比原先的更新颖、更便宜、更进取。虽然这座城市气候寒冷，但 Avalanche 在金融和贸易方面的专业性能够赋予它更多的活力与信心，让它快速地发展起来。

图片来源：ChooseChicago

至于 NEAR，我觉得它更像是旧金山——一座为 web3 技术员而建造的城市。这是一座理想化的城市，汇集了很多想要打造 Ethereum3.0 的人。在他们看来，分片（sharding）是唯一一个长期方案。

图片来源：Rough Guides

这些城市的重要性不仅体现在它们的规模和商业开放程度，还在于它们对于城市建设和治理模式的不同思考。每座城市都有着自己的权衡，采取着自己认可的价值观，并由此吸引来不同的行业。

所以，回到开头那个问题，未来将会是一个多链的世界，还是会形成「一链治天下」的局面呢？

现在我们也可以重新阐述一下这个问题：我们将生活在一个城市众多的世界，还是只会由一个城市掌握统治权？

图片来源：ThoughtCo

答案很明显，城市的主导权会符合幂律分布，而且重要的城市也会不止一个。

虽然世界上不存在完美的比喻，但我认为，这种思考模式对于预测 L1 的发展方向非常适用。

以下是基于这一模式的六大预测：

1.未来世界将多链并存。  

2.Ethereum 可能成为最有价值的链，因为，用一位著名的银行抢劫犯的话说：「那是存钱的地方」。

3.其他 L1 也将很有价值，但它们还会继续分化。纽约、洛杉矶、芝加哥和休斯顿将成为经久不衰的城市，因为它们各自有着不同的制度和文化。  

4.L2 也将不可或缺。虽然搭建摩天大楼对于城市扩张必不可少，但也并不是故事的结束。L2 是一个必然选择，而非一个可选选项。  

5.特定用途的区块链将继续作为小众链。  

6.在现实世界中，交通开销几乎占到了住房 GDP 的一半。如果我们在 Crypto 中看到类似的情况，跨链桥的优势就将极大地显现出来。
