布局NFT平台 加密货币交易所的差异化竞争

2021-08-09
约 5860 字 - 预计阅读 12 分钟

欧易OKX 注册链接 欧易是全球著名的数字资产交易平台之一，主要面向全球用户提供比特币、莱特币、以太币等数字资产的币币和衍生品交易服务。
HTX火币 注册链接 火币全球专业站，是火币集团旗下服务于全球专业交易用户的创新数字资产国际站，致力于发现优质的创新数字资产投资机会。

从 NBA 球星勒布朗·詹姆斯的经典扣篮镜头，到推特创始人杰克·多西的历史第一条推文的 NFT 拍卖，再到加密艺术家 Beeple 的数字作品《每一天：前 5000 天》拍出 6934 万美元的高价，支付宝联合敦煌美术研究所发行的两款基于蚂蚁链的付款码 NFT 皮肤刷爆朋友圈。“万物皆可 NFT”成为加密社区里一个标志性的 Slogan。

面对已经在全球市场上攻城略地的 Opensea、Nfity Gateway、Rarible 等 NFT 交易平台，加密货币交易所也纷纷在年初开始布局旗下 NFT 平台。

5月17日，由 Huobi X Center 孵化的 NFT 平台——iBox 正式上线。

6月3日，FTX 宣布推出 NFT 交易市场 NFTs。

6月22日，Gate.io 推出基于自身平台的 Gate.io NFT 魔盒。

6月24日，币安 NFT 平台 BINANCE NFT正式上线。同年8月1日，币安将NFT 平台接入到 App 中。

与国外已经沉淀了两三年的 NFT 交易平台相比，加密货币交易所作为行业最大的一环，虽然起步晚，却也具备天然的优势：

1. 具有大量持有数字货币的用户，币安、FTX 这样的交易所甚至拥有大量的全球用户。

2. 可以做到简化交易过程以及保存过程，相当一部分交易所的用户并没有数字钱包。

3. 交易所有更多的资源和市场运营能力。头部的品牌、艺术家、画廊美术馆、经纪公司更愿意和这样的天然销售平台合作。从运营角度上容易造势，形成一定的 NFT 升值效应后，市场更加容易 fomo。

4. 作为资金实力本就雄厚的交易所，可以做到更多的让利并加速其市场占有率。

当然，要竞争就要做选择，交易所在布局 NFT 交易平台中的共性和差异化又如何？

「 iBox 」

作为交易所里第一个上线的 NFT 平台，iBox 定位于高端 NFT 的发行和流通，并确定了服务亚太地区，联合知名 IP、艺术家和 KOL 的路线。iBox 部署在多条公链上（首发 HECO），是交易所里唯一一家去中心化的平台。支持 ERC-721 和 ERC-1155 标准的数字资产买卖。

iBox 上线首日，即联合影星陈小春发行首个 NFT 著作 “iBox 001号” ，这也是国内较早的明星 NFT 作品。

在艺术家方面，iBox 平台联合北京万户创世文化传媒有限公司共同发行《徐冰天书号》首件 NFT 太空艺术作品。
了解艺术的人都知道徐冰是中国当代艺术里最重要的艺术家。徐冰代表作《天书》（Book From The Sky）于 1987－1991 年创作完成，首次展出即受到全球当代艺术界的关注，并进入多部欧美《世界艺术史》教科书。作品最终拍卖价高达 200 万人民币，上线即被某神秘藏家抢购。一举创下国内单件 NFT 最高成交价。

《徐冰天书号》NFT 太空艺术作品由三段影像构成，完美呈现了载入世界太空艺术史的重要时刻。

火币集团副总裁七爷此前曾在社交媒体透露，单 ibox.com 这个域名就花费了千万元，在某种程度上这反映出火币在 NFT 市场发力的决心。

首发 NFT 后，iBox 相继推出了《真三国无双》的 NFT 版，包括古天乐、
刘嘉玲、古力娜扎、韩庚、王凯等人的角色 NFT。

此外，还上线了陶喆等音乐人的音乐 NFT，与上影合作的大闹天宫系列，水浒传的 IP 收藏卡 NFT 等。
这些动作应证了 iBox 第一阶段探索影视、娱乐明星相关 IP 的 NFT 化产品，通过粉丝群体的关注，及对产品的赋能引起行业内用户的关注，从而获得首批种子用户的目标。

也许是看到盲盒 NFT 市场的火热，iBox 在 7 月 26 日上线了第一期盲盒，7月30日上线第一期 IP 小黄鸭盲盒并迅速卖空。

火币作为占据国内市场半壁江山的交易所，在资源、渠道等诸多方面都有很大优势。除了现有的用户，iBox 更期望能够拓展圈层，通过 IP、明星效应吸引更多的圈外流量。这也是 NFT 走向大众的必经之路。

「 FTX NFTs 」

FTX 在上线 NFT 平台后并没有做太多的动作。

成功竞拍该 NFT 的用户可与 FTX 创始人 SBF 共进一次午餐或进行一次 30 分钟的 Zoom 对话。该 NFT 最高出价达到 11 万美元。

除了 SBF NFT 午餐拍卖，就是官方发布的对于 FTX NFTs 平台的唯一一篇“另类”的介绍。文章中用了大量的篇幅来提示投资 NFT 由于流动性和本身的价值考量而存在的潜在风险。
这是文章的结尾风趣幽默的Q&A：

- 对于 NFT 我很困惑。我错过了什么吗？它们有什么作用？

- 您可能没有遗漏任何东西，但您可以在此处探索 FTX 提供的 NFT 。有些人喜欢 NFT，有些人不喜欢。各有各的！

当然，从 FTX 上线的 NFT 品类——潮玩、潮服以及 FTX 的文化衫等可以看出 FTX NFTs 的大致定位。FTX NFTs 目前是中心化的，未来会允许用户从 FTX Gallary 中提取到去中心化钱包。

FTX 相继与一线潮玩品牌 PoP Mart、MEDICOMTOY 推出了NFT

8月2日，FTX 宣布与美国上市公司 Dolphin Entertainment 合作 NFT 市场。Dolphin 的子公司包括几家不同行业的著名公关公司，包括 42West、Shore Fire Media 和 The Door。  

FTX 发推文称，其 NFT 市场将专注于电影、电视、音乐、游戏、电子竞技、烹饪、生活方式和慈善行业。

「 Gate.io NFT 魔盒 」

Gate.io NFT 魔盒致力于打造一个平民化的，人人艺术家的中心化NFT交易平台。其平台上NFT均为免费铸造。目标是服务于有创作热情的用户和创作者。

从 Gate.io NFT 魔盒的平台以及活动宣传中， 我们可以看到逆袭、素人、是金子总能发光、最接地气、下沉、零手续费等亲民又鸡血的词汇。这反映出 Gate.io 想要走大众化 NFT 路线的决心。

上线之际，Gate.io NFT 魔盒特别推出了“一拍即盒”系列拍卖周活动。拍品均为当代新锐艺术家创作的限量版画作。

第一天的拍卖作品，系毕业于中央美院的 80 后青年艺术家胡桂东创作的两幅画作：《梦乡之一》和《梦乡之二》。6 月 28 日下午 18 点，两幅画作分别以 259 美元和 271 美元成交。二者的起拍价均为 1 美元。

相较于BINANCE NFT、iBox 的大手笔，Gate.io NFT 魔盒无论在活动造势上，还是拍卖结果上，都显得“朴素”了很多。

除了「一拍即“盒”」系列拍卖周，以及两场 Gate.io NFT 魔盒「魔力欧洲杯2021」主题活动，1 美金抢限量周边活动，Gate.io NFT 魔盒不论从活动或者销售拍卖上都没有给市场带来太大的惊喜。

「 BINANCE NFT 」

币安在3月就开始为 BINANCE NFT 平台造势，策划了 MyFirstNFT 大赛，4月宣布与新加坡出版商 Media Publishares（Vogue新加坡），VIDY 合作共同搭建 NFT 平台。
4月27日正式宣布 6 月上线。同时，与币安相关的 NFT 板块的币种，如 Bake 迎来了一波超过 500% 的大涨。

作为全球交易量最大的加密货币交易平台，币安有国际化的团队和国际化运营的经验。
在官网上也可以看到该平台的大体布局：通过交易市场铸造、购买和竞标全球顶尖的NFT；独一无二的NFT盲盒；独特收藏品展览等高端活动。

在 6 月 24 日平台上线的当日，BINANCE NFT 举行了首场 “ Genesis 创世纪” 拍卖活动，推出其 “盲盒” 功能，以及最近宣布的 “100位创作者” 的交易市场。用户可以参与竞拍，申请盲盒并在市场上开展交易。

1. 拍卖： BINANCE NFT 联合波场 TRON、APENFT 基金会举行了主题为“ Genesis ”的首次拍卖会，并选择安迪·沃霍尔《三幅自画像》的NFT作品为本次拍卖会主要拍品，在开拍的10分钟之内，以252万美元的竞价迎来开门红，并最终以280万美元成交。

**
2. 盲盒：** 首个亮相的盲盒系列是“ tokidoki ”，日语解作“ sometimes(有时) ”之意 ，是国际知名的生活方式品牌 。自 2005 年首次亮相以来，tokidoki 凭借其超凡脱俗的角色积累了一批狂热的追随者，在全球销售了超过数百万的实物 tokidoki，最近三期总数11万个数字化 tokidoki 产品也在短短几分钟内售罄。

第一期发售时，几乎不到1秒，总数4万个，每个 20 BUSD的盲盒被一扫而空。后来据官方解释是因为科学家的介入让系统出错了，系统修正后，盲盒依旧在极短的时间内一扫而空。很多只抢到N类盲盒（最普通，数量相对于「R 稀有」「SR 超级稀有」「SSR 极度稀有」最多的类型）的人依然按照3、4倍的价格在二级市场上挂出售卖。其热度可见一斑。

3.  “ 100 位创作者”——来自世界各地的名人和创作者： 这 100 位创作者作品的多样性让用户将能够交易各个领域优秀创作者的 NFT。在名单里我们甚至可以看到炎亚纶，唐朝乐队主唱丁武这些国内比较熟知的名字。

用户可以从他们喜欢的百位创作者那里购买和竞拍，还可以转售 NFT。币安将收取 1% 的手续费，创作者或藏品提供人将持续获得 1% 的版税——从而将大部分利润分配给创作者和用户。

相较于传统的艺术品买卖画廊占据销售额大头的行业状况，NFT 交易平台在发展初期都做出了极大的让利，不急于吃下这块蛋糕。这从一定程度上提升了创作者尝试的热情，价格的下降和版税机制也起到了提高流动性的效果。

从 6 月 24 日至今不到四十天的时间，币安已经连续推出了14 期盲盒，合作的 IP 包括网易旗下游戏《永劫无间》IP 授权的《NARAKA HERO》系列盲盒，游戏生态系统平台 Seascape 的僵尸系列盲盒。以及币安 NFT 市场推出的独家盲盒系列——《Big Bang for NFT》，聚焦币安智能链（BSC）热门 NFT 项目，如 BunnyPark 等。

从销售状况来看，盲盒 NFT 几乎都是上线即售罄，市场热度极高。

其中 BunnyPark 的 NFT 作品「The KING」的拍卖初始价格为 1000 BNB，但最终以 3100 BNB 被币安用户 Anonymous_udau 拍得，按当前BNB 340 美元计算的话，这笔交易金额达685万美元，创下 Collectible 类别的拍卖历史最高纪录。

此外，国民老师苍井空也通过 BINANCE NFT 发行了自己创作的第一套 NFT 作品。

7月27日，BINANCE NFT 市场宣布，通过与位于圣彼得堡的俄罗斯国家冬宫博物馆合作，达芬奇、文森特·梵高、克劳德·莫奈和乔尔乔内的画作将在8 月通过币安的 NFT 市场进行代币化拍卖。这也是币安 NFT 平台和国家性质的博物馆机构的合作。

丰富的资源以及与品牌、头部艺术家的合作，让币安得以持续输出优质的活动并上线质量较高的NFT数字艺术品。从 BINANCE NFT 强劲的发展势头和推特上持续的曝光情况上看，BINANCE NFT 有能力在未来获得更长远的发展。

「 用户体验如何 」

在对比了平台不同的定位和落地项目外，平台本身的用户体验又怎样呢？

从创作端来说，币安和火币都限制了 NFT 的铸币权，只有通过认证的艺术家个人和机构才可以铸造 NFT。虽然 BINANCE NFT 的路线图显示未来会允许所有人上传 NFT 作品，但并未披露具体的时间。
对于创作者而言，这些平台存在一定的门槛和限制。

Gate.io NFT 魔盒允许所有人免费铸 NFT，不过这样开放式的打法并没有带来特别好的效果：除了官方发布的 NFT，平台上充斥着大量有价无市、无人问津的作品。由于上链没有手续费门槛，甚至出现了一些打色情擦边球以及侵犯版权的 NFT 作品。

从购买端来说，币安、FTX 和 Gate 的 NFT 平台，依托于交易所本身的账号体系，方便了平台用户参与。但一定程度上也意味着其它想要参与的人必须先成为该加密货币交易所的用户。

在用户购买方面，iBox 接入了微信商城用人民币支付很好地解决了圈外人想要购买 NFT 但无从下手的痛点。但对于了解 NFT 机制的人来说，在微信端购买的 NFT 没有接入到任何自己创建的去中心化数字钱包，从这一层面来讲，手里的数字资产依然没有保障。

另外，相较于 BINANCE NFT 和 Gate.io NFT 魔盒，iBox 上的所有 NFT 没有归类，甚至不具备检索功能，这也影响了用户的使用体验。

「 问题：定价、版权纠纷 」

NFT 的生产必然是跨行业的合作，这对于以做交易、金融起家的交易所来说是比较新的领域。所以在将数字作品 NFT 化的过程中会出现定位或者价格设定的模糊，一个拆分为 88 份的电影人物海报 888 USDT 的售价真的会被接受吗？这是留给创作者和经销者的问题。

不以实物为依托的 NFT 艺术品超过一定的价格之后就变得很难获客，现阶段依然没有出现对于 NFT 艺术品来说更好的应用场景。

Gate.io NFT 魔盒上大量出现廉价的 NFT 实际的意义在哪里需要时间的检验，与其上币风格相似，Gate 确实没什么架子，但是这次能不能乱拳打死老师傅要打上问号。

iBox 与 IP 明星的合作涉及到了更多的版权归属和纠纷问题 。两个重磅明星 NFT 项目均遭到侵权争议。

7月9日，伟大文化官方微博称：我司梦达灵文化（北京）有限公司对于“ iBoxNFT ”微信公众号中非法冒用陶喆姓名、肖像制售 NFT 的侵权行为表示强烈谴责。

5月18日 iBox 上线，联合陈小春推出了首个售价199 USDT 的 1000 份 NFT图片。随后与相关公司出现侵权争议，iBox 与营销渠道上陈小春 NFT 相关信息已经全部删除。iBox 微信平台上关于陶喆的首条宣发《猜歌手，赢取华语乐坛首个教父级音乐 NFT 》文章也已经删除。

这也是以前很多已经存在的影视，音乐 IP 等不太容易 NFT 化的原因。有些影视的 IP 版权归属大大小小加起来可以有几十家公司。单纯协调一个影视 IP 的 NFT 过程中就面临着极大的挑战。
但是随着NFT带动数字资产市场的兴起，机构和公司会重新梳理对于创作端和发行端以及 NFT 所有者在链上版权解决方案，对之后想要参与到 NFT 领域的公司以及个人都将提供一个新的版权划分和确权方式。

「 市场展望 」

由于 NFT 与艺术品和稀缺数字商品的属性不谋而合，所以稀有数字艺术品交易依托在非同质化代币 NFT 的技术协议上得以快速地发展。

全球最大的 NFT 交易市场 OpenSea 7 月交易量 3.25 亿美元，环比历史前高的 6 月交易量（1.49 亿美元）激增 118%，续创历史新高，且 7 月 NFT 交易数量和交易费用分别为 45.77 万个、2178 万美元，同步打破此前 21.14 万（6 月）、1050 万美元（3 月）的历史记录。

这些数据显示出，NFT 市场依旧表现出强劲的增长趋势。大量的新平台涌出，诸如腾讯等传统互联网企业的入局，都将为 NFT 在未来的发展中提供更多的可能性。
