本文梳理自 Color Capital 合伙人 cantino.eth 在个人社交媒体平台上发布的观点，律动 BlockBeats 对其整理翻译如下：

读懂智能合约可以让你更深入地了解项目的受欢迎程度、其所有权的分布情况以及它的能力。此外，你还可以查看每笔交易的详细信息。这些都是 web3.0 的强大优势，有一天，它们可能会成为学校里教授的内容。

首先要弄清楚，什么是智能合约？

它是一个在满足特定条件时在区块链上执行代码的程序，各方以数字签署合同的方式准许并维护它的其运行。这些代码可以是向朋友汇款、买卖 NFT 虚拟商品等一系列复杂的内容。

那么，如何找到智能合约呢？

以太坊合约、交易和钱包均可在 Etherscan 上浏览。

要查找 NFT 合约，首先导航到 OpenSea 上的列表，然后点击「Details」，这里会显示一个「Contract address」链接。

点击它。

这里有很多东西要看，所以让我们分解一下。

1.「Balance」和「Value」：这里能显示合约持有多少 Token。

2.「My Name Tag」：如果已经登录，你可以在此处为自己创建私人笔记。

3.「Creator」：这里显示创建合约的钱包，以及原始交易的链接。让我们点击它。

在这里，我们能看到关于创建合约的交易的详细信息。

我们可以看到：

1. 交易确认（Transaction confirmation）

2. 时间戳（Timestamp）

3. 每枚 Token 的铸造数量（Quantity minted of each token）

4. 合约创建者（Contract creator）

5. 支付的费用（Fees paid）

这些都是很有价值的内容。 

现在，让我们回到主合同，看一下「Tracker」

点击以后我们可以了解到：

1. 最低 Token 价值（minimum token value）

2. Token 的总供应量（max total supply of the token）

3. 有多少个钱包地址持有 Token（how many wallets hold the token）

4. 发生了多少笔交易（how many transfers have occurred）

此外，这里还有一个大多数人不知道，但却很酷的东西。

单击选项卡列表中的「Holders」。

这里显示了合约中持有最多 Token 的地址排名，如果你点击「Token Holders Chart」，你甚至可以看到一个图表化的地址信息。

OK，回到主合约页面。让我们深入了解一下「Transactions」选项卡。

第一列是所有交易按时间顺序排列的列表，每个都可以点击进入。

接下来是「Method」，这些是通过合约执行的功能。每个合约都是不同的，但常见的方法包括铸币、转账和批准设置。

现在，让我们来研究「Contracts」选项卡。

Code：显示的是合约的源代码。

Read Contract：显示的是合约的功能。

Write Contract：这允许你启动特定功能（如果您的钱包有权限），例如销毁 Token、授予角色、暂停等。

最后，让我们再看一下「Analytics」。

在这里，我们可以找到展示交易、费用、ETH 余额等的图表。比如下图就展现了 Cryptopunks 的崛起之路。

另外，我比较推荐数据网站 nansen，这里有关于数量、项目钱包重叠、著名 Token 持有者等的更多详细信息。
