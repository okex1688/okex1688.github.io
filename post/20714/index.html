关注区块链行业的人或多或少都听说过 Web 3.0 这个名词，Web 3.0 旨在解决目前 Web 2.0 的互联网世界中存在的种种问题。

虽然 Web 2.0 中的互联网把每个人链接在了一起，使得信息的流通速度和信息量指数级上升，每个人都可以获取信息并享受互联网带来的各种服务，好像很公平，但其实互联网的管理是非常复杂的。

平台、运营商都对各自地盘上发生的事情有管理权和监控权，比如推特可以擅自封杀美国前总统的账号，比如携程可以根据用户的行为进行大数据杀熟获取商业利益。

Web 2.0 发展至今，用户数据成为了每家公司最重要的资产，而这资产的拥有者——用户自己却没有任何选择的余地。Web3.0 引领互联网未来的趋势，将随着用户的需求和使用而发展，最终把权益都归于用户。

Web 3.0一个更自由开放的世界

那么，Web 3.0 究竟是如何解决这些问题的呢？我们先来看一下 Web 1.0 和  Web 2.0 的发展。

Web 1.0 时代是早期的互联网，此时台式机诞生，由网站运营者编辑发布信息，用户大部分都是被动输入信息。

而随着手机端的移动互联网的诞生，Web 2.0 产生允许用户与平台进行大量的互动，同时像微信、微博、抖音的诞生，使得用户可以自己上传发布内容，让互联网上的信息量得到了空前的增长。

然而，无论是 Web 1.0 和还是 Web 2.0 都十分依赖于特定的“信息中介”，它可以是应用层面的平台，比如推特、facebook，也可以是中心化的服务器，比如阿里云、AWS 等。而也正是这样的特性，导致了我们开篇所述的种种负面现象。

而 Web3.0 具有无需信任、去中心化、抗审查等特点，用户对于自己的数据、隐私。为了实现 Web 3.0，则需要各种技术的加持，包括区块链、人工智能、物联网等，这其中区块链技术又与 Web 3.0 的愿景尤为的契合，这也是区块链被称为“价值互联网”的原因之一。

包括智能合约在内的区块链技术是基石，代码即法律，代码即规定。 一切规则被写进协议，一旦开启，便不可篡改。代码开源，公开透明，接受所有人的检查。同时区块链节点分布式的协同共识，把信任的需求降到最低。无需信任，只需验证代码。

又由于各种区块链协议像乐高积木一样，可以让开发者随意调用，并组合成神奇的城堡，也就是去中心化应用 DApp，碰撞出各种火花。而这一切将带给我们一个与 Web 2.0 完全不同的世界。

另外，由于 Web 2.0 中，数据都被存储在一台或指定的中心化服务器内，因此只要黑客对目标服务器进行攻击，就可以获得大量用户数据，比如某平台的账号密码、甚至信用卡信息。如果你曾经注册过谷歌账号，不妨现在登陆上去看一下，谷歌将会告诉你你的哪些密码已经被泄露。

而来到 Web 3.0 的世界时，用户数据可以存储在去中心化的数据库上，不需要再去租用特定的服务器。 因此，每次数据的授权都需要经过你的同意才，而且由于分布式的记录方式，也不存在单点攻破后的数据泄露问题，更好地保护了用户隐私。

而用户发布信息，也不再将收到平台“一言堂”式的审查，真正实现了言论自由。在这一轮牛市中，市场也感知到了这一赛道的潜力，fil的价格因此也水涨船高。

Web 2.0 发展到现在，相信大家没少在平时上网的时候被广告轰炸。虽然每个人都获得了免费的互联网服务，但代价就是我们的基本信息和行为数据被巨头占为己有，巨头以此为卖点，向广告方收费。

各大互联网平台之间对于用户的争夺也总是不休，Web 2.0 时代，用户就是互联网巨头最大的资产，巨头可以从为平台付出内容、活跃贡献的用户中获取利益，却不支付用户任何费用，也不经过用户同意。

Web 3.0 可以把数据和数据带来的收益回归个人，私钥掌握在用户自己手里 ，也就是你把自己数据的使用权都掌握在自己手中，你可以选择保护它不然它被利用，也可以选择将其出售获得利益。

真正的自由不是想做什么就做什么，而是不想做什么就可以不做什么。Web 2.0给我们带来了无限的可能性的同时，却没有把数据自由给我们。而Web 3.0大大提升了用户的自由度。

人人都将从Web 3.0通证经济中获得收益

区块链离不开通证经济，而这也将会成为 Web 3.0 世界中重要的一环。想象一下，如果你在早期参与了某个 Web 3.0 公链项目，并投资了它的具有治理权益的 Token，那么你就有权在对于项目的各种规划进行投票，发表自己的意见。就如同传统公司董事会的股东，有权对公司决策进行一定的干预。

除了普通用户的权力变大之外，每个用户都可以获得切实的收益。如果有任何生态上的应用需要进行一些交易，那么 Token 将会作为手续费，付给各大节点。

如果是 PoS 权益证明或者其衍生出来的其他机制，那么任何将 Token 进行质押 Staking 的用户不仅参与了维护网络安全的光荣实名，使得网络更加去中心化、更加健壮，而用户自己也将会从中收到来自节点的分红。

更值得一提的是，如果涉及到任何需要出售用户数据的请求，只要用户同意，那么用户就可以从中获得收益。 巨头凭借海量用户数据收取广告收益，污染用户体验的事情，将不复存在。而用户也多了一种“赚外快”的方式。

Web 3.0 通证经济的玩法肯定不仅仅局限于上面几种，而这一切都必在长期来看，必然会使得生态内的代币价值提升，一切都会形成良性循环。

可以看到，通证经济将大大提升 Web 3.0 生态的网络效应，每一位参与 Web 3.0 的用户价值将远远高于 Web2.0，而这价值不仅赋予给了生态代币，更是传递给了每一位参与者。

 Web 3.0各大赛道项目介绍

那么普通人除了等待 Web 3.0 到来，享受更智能、安全、透明的互联网服务之外，还有什么办法参与这场红利呢？也许参与早期 Web 3.0 项目也是一个选择，尤其是头部项目，下面是一些项目介绍。（非投资建议）

公链板块 ——波卡Polkadot（DOT）：一上线就空降市值前十的波卡，解决了区块链 2.0 智能合约时代各种公链的跨链问题，波卡不仅可以实现资产跨链，也可以信息跨链，创造 Web 3.0 的基础设施。只有当万链互联之时，所有的 Web 3.0 应用才能跑在畅通无阻的信息高速公路上，用户的切换成本和使用体验才会达到极致。

存储板块 ——Filecoin（FIL）：今年的大牛市中，FIL 表现可谓是抢眼，从年初底部一路上涨十倍。从 15 年开始酝酿了多年的大项目一上线便一鸣惊人，而它所讲述的故事也是 Web 3.0 必不可少的一个模块——存储。

说到 Filecoin，就不得不提 IPFS 协议。IPFS 协议也是一种点对点传输协议，类似于古典互联网时期的 BT 下载网络，对标的是目前我们耳熟能详的 HTTP 协议。而 Filecoin 是运行在 ipfs 上面的一个激励层，带着巨大的分布式存储空间，通过挖矿，激励更多的节点参与，把云存储转换为算法市场。

浏览器板块 ——Brave（BAT）：Brave 可以说是行业里为数不多拥有大量真实圈内圈外用户的项目，2020 年就拥有 2540 万的月活用户。它最大的特色莫过于其广告拦截功能，用户使用 Brave 浏览器上网，不会收到广告的困扰。

如果有商家想在 Brave 浏览器上打广告，则不得不向浏览器使用者付费，而用户也可以根据自己的意愿选择是否收看广告并赚取 Token。同时，Brave 以用户隐私为中心，拒绝一切试图跟踪用户个人数据的行为。

域名板块 ——Handshake（HNS）：去年 Github 上的空投获得使得 Handshake 这个去中心化域名项目一炮走红。传统的域名解析系统被称为 DNS，但是非常中心化，也就是交易、租赁都要问特定几个主要的域名供应商。

普通用户无法真正拥有自己的域名，并且由于中心化机构的维护，容易造成一系列的网络故障。而通过 Handshake，用户可以使用 HNS 支付注册费用，并获得加密密钥作为域名所有权的凭证，所有通过 HNS 的域名交易都会被记录在公链上。

小结

Web 3.0 的愿景看似美好无比，但目前也存在一些问题，比如区块链网络拥堵导致效率低下，网络费用高企转嫁到普通用户身上，智能合约漏洞被黑客利用攻击等……

但任何新技术的大规模应用不会一蹴而就，无疑我们现在是处于 Web 3.0 爆发的前夜，新技术、新概念和新项目层出不穷，或许会存在泡沫，而未来的大规模落地却指日可待。
