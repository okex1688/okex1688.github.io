原文：《The Creator Economy Will Slowly (Then Quickly) Migrate to Web 3.0》

凯莉·克拉克森(Kelly Clarkson)配合《美国之声》(The Voice)简直就是一场视听盛宴。

因为我不会唱歌，所以我喜欢这甜美的味道。有一天，我看到一些在节目中唱歌的创作者。他们穿着最好的周日服装来到演播室。他们擦鞋子，做头发，穿上ck或香奈儿5号香水，然后唱出了自己的心声：“灵感的哭喊”。

我仔细地看着每个参赛者唱歌，大约演出过半的时候，电视屏幕下方弹出一个小盒子。

看到了吗? 《美国之声》为其最有价值歌手们的社交媒体账户进行了大约3秒钟的推广。

这是第一次发生这样的事。

以前，每位歌手的Youtube视频下面只有一个描述，没有可以链接到这位歌手的网站链接。在节目的网站上，以前每个参赛者都有一份歌手简介。即便如此，也没有链接到这位歌手的网站。现在这些个人资料已经不放在节目网站上了。

创作者们已经被封存了好几年了，我们让垄断对他们造成了伤害。

Web 3.0转型

  • Web 1.0:互联网

  • Web 2.0:社交媒体/移动互联网

  • Web 3.0:区块链/ AI

我们正处于互联网的第三阶段。许多Web 3.0怀疑者认为去中心化不会改变任何事情。我不同意。

Coinbase的首席执行官布莱恩·阿姆斯特朗(Brian Armstrong)认为，Web 3.0并不是什么新鲜事。他说，我们正在重新对互联网去中心化，即互联网最初是什么样子，为什么?

他说我们需要“检验大型科技公司的力量”，还因为许多国家被排除在创造者经济繁荣之外。问问像我在非洲的朋友Chinedu这样的创作者吧。他们很难通过PayPal和Stripe获得盈利。然而，像我这样的人一直认为这是理所当然的。你出生地的运气不应该影响你是否能加入创造者经济并以此谋生。

创造力不受来源国的限制。Web 2.0却让我们认为它是。

“转换率”解释了对创作者的剥削。

像Twitter、Instagram和TikTok这样的社交媒体平台都有100%的转换率——它们根本不与创作者分享任何收入!这对他们来说是好事，但对用户来说却是坏事。——克里斯·迪克森

这里我来进一步纠迪克森。在首席执行官杰克•多尔西看到Web 3.0的力量之前，Twitter从未与其创建者分享收入，现在他重新起航。他公开宣布他将去中心化Twitter，把它变成一个Web 3.0产品。此外，Twitter刚刚推出了“超级关注”功能，允许创作者从内容中赚钱。

此外，还有新的evue newsletterR功能，允许创作者为付费的时事通讯收费。这还不是全部。最近推出了最好的功能。Twitter现在开放了加密货币服务。你可以用比特币在推特上转账。例如，如果你曾经试图从美国寄钱到萨尔瓦多，这是新的游戏规则。Twitter证明了向Web 3.0的迁移将100%发生，虽然是缓慢地。

不过，Facebook、Instagram和TikTok还没有理解这种转变。它们让内容创造者像奴隶一样工作。创造者建立追随者，并获得由算法控制的视图，该算法可以决定明天是否喜欢你的连帽衫颜色。他们利用这些内容吸引消费者，并向他们展示广告。广告收入中的0美元将与创作者分享。 请再读一遍。

当内容平台刚刚出现时，我们只是乐于使用它们，而不需要花一分钱。现在我们已经习惯了。我们的标准已经改变了。创作者希望公平地分享平台从我们的内容中获得的收益。这是合理的。但是，除非扎克伯格被迫出售广告并分享收入，而不是购买另一套房子，否则变革不会发生。

如果创作者不再免费提供内容，那么像TikTok这样的平台就破产了。请想像一下。

平台所有权被打破

Web 3.0有什么不同？ 所有权。创作者希望拥有他们所创建的平台，并拥有投票权。当平台赚钱时，他们也会赚钱。

一个内容审核政策是通过投票权形成的，它是通过区块链共识和世界各地成千上万的计算机强制信任的过程来完成的(即使是一个橙色总统也不能影响区块链选举)。

此外还有该平台自身的功能。你是否曾经在一个美好的周六下午打开你最喜欢的应用程序，发现一切都变了，看起来像是由一个大一的UI/UX孩子设计的?你感到你没有看到设计给你脸上打来的一拳?这不是孩子的错。他们离创造者的工作太远了。他们是局外人。

Web 3.0则不同。 功能是由用户通过民主程序决定的。如果一组用户不喜欢它，他们就会创建一个分叉。分叉是指运行同一平台的两个具有不同的特性的版本。

当所有权是固定的和透明的，动机就会改变。所有权会导致用户民主，而不是类似于web2.0的独裁哲学“这是最新的更新，你这个肮脏的动物。”

智能合约促成了这场革命

  • 21世纪初:做一个网站。

  • 21世纪10年代:开发应用程序。

  • 21世纪20年代:部署智能合约。

许多人误解了以太坊在新创造者经济中的角色。 以太坊允许机器而不是人类执行智能合约。

企业家迈克·诺沃格拉茨解释说:“区块链将允许信息不应该是免费的，而不是免费的。”还记得Napster吗?你可以免费下载音乐，没有人能阻止你，即使是金属乐队的傻瓜。

我们现在使用的Web 2.0版本的互联网不是为内容所有权而构建的。如果你想的话，你可以从Youtube上拿走任何视频。这很难被察觉。见鬼，这篇文章很容易被复制粘贴偷走。

智能合约改变了这一点。智能合约告诉所有人，这篇内容的主人是谁。NFT的创作性在于使你的内容变成便携的数字资产，以便你可以在互联网的任何地方发表。

根据专家埃里克·约根索（Eric Jorgenson）的说法，现在web3.0时代的到来仅仅是因为这三个方面的分歧:

Web1 =免费出版

Web2 =免费交流

Web3=免费交易

更多盈利方式

既然Web 3.0解决了内容所有权问题，并使我们的数字资产便携化，我们作为创作者可以找到更多赚钱的方法。

迈克•诺沃格拉茨(Mike Novogratz)表示，我们将“看到从商人到创造力的转变”。创作者将有办法将他们的创造力货币化。不仅仅是艺术上的创造力，而是各种各样的创造力。

他甚至说，我们会在T恤上佩戴NFTs，只有戴上AR/VR眼镜，其他人才能看到它们。

你可以和多方分享钱。你可以根据那个平台更好，将你的内容从一个平台传输到另一个平台。来自世界各地的创作者将能够合作，想出疯狂的方法从我们的工作中赚钱。

在扎克富起来的时候，再也不用在Facebook上赚0美元了。根据内容的表现，金钱可以按秒流进你的腰包。

创作者将加速Web 3.0的发展，因为我们只是想要得到报酬，从而我们就可以全职从事创造了。一旦内容捕捉者释放了我们，更多的赚钱方式将淹没我们的创造性大脑。我等不及了。

Web 3.0改变了每个主要行业

ETH：去中心化的硅谷

DeFi：去中心化的华尔街

NFTs：去中心化的好莱坞

这种转变并非创造者经济所独有。所有行业都将慢慢改变，以反映人类对民主的痴迷。Web 3.0首先瞄准的是金融，并成为一个超过2万亿美元的产业。这种转变已经转向了创造者经济，例如Bitclout等平台发明了硬币，让创造者成为他们的用户，从而可以投资股票。

下一波去中心化的浪潮将是Youtube、Spotify和Facebook。现在是成为创造者经济的一部分的最佳时机。观察在您眼前发生的转变，并将您喜欢的任何内容类型移植到Web 3.0中。这就是你作为创造者如何利用这个绝佳机会的方法。

作者：Tim Denning
