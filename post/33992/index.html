NFT在韩国的地位

不可替代代币 (NFT) 在韩国越来越受欢迎。该国的高科技产业，加上其在 K-pop 和视频游戏等流行文化产品方面的实力，使其成为 NFT 业务的沃土。根据谷歌趋势，截至 2021 年 11 月 1 日，韩国在购买 NFT 的兴趣方面排名世界第四。

NFT 的搜索量以 74% 比 26% 的比例超过了加密货币，突显了 NFT 在虚拟资产市场上的明星地位。

最近的 NFT 销售也说明了这种巨大的受欢迎程度。韩国著名艺术家 Jang Koal 的数字动画作品《Mirage Cat 3》以 3.5 比特币成交，约合 2 亿韩元（168,314 美元），Mr Unknown 的作品《Money Factory》以 200 以太坊成交4 月，或超过 11 亿韩元。

尽管 NFT 很受欢迎，但根据韩国法律，NFT 的法律地位仍然不确定。尽管最近通过的《特定财务信息报告和使用法》修正案为加密货币监管提供了很大的明确性，但该法对 NFT 的适用尚不清楚，主要是关于 NFT 是否会被视为由法案。

鉴于韩国作为 NFT 市场主要参与者的潜力，有关 NFT 的监管发展值得密切关注。

NFT的人气

Samsung Next 和 CJ OliveNetworks（CJ 集团的 IT 分支机构）等韩国大公司已采取重大措施投资 NFT 并开发基于 NFT 的票务解决方案。

Ground X（Kakao 的区块链子公司）、未来资产证券、新韩银行和韩华资产管理等成熟的金融机构都认可将 NFT 整合到其产品或业务模式中的想法。

韩国在流行文化方面的实力，拥有全球流行的音乐团体、电视节目和视频游戏，使该国成为 NFT 未来的沃土。

韩国第一个 NFT 市场（于 2021 年 3 月 31 日开业）包括一个“带有 ‘casano’ 图案的打火机”，该打火机出现在热门韩剧Vincenzo中，并以 0.13 以太坊的价格作为 NFT 限量出售 100 个（250 美元）每件。

三大 K-pop 机构——SM 娱乐、YG 娱乐和 JYP 娱乐——已宣布计划投资、合作或发行其艺人的 NFT。世界上最大的音乐团体防弹少年团也加入了这场竞争。2021 年 11 月 4 日，HYBE（BTS 的代理机构）宣布已与区块链公司 Dunamu 合作建立合资企业，以进军 NFT 市场。

尽管 K-pop 和韩剧往往会吸引更多的头条新闻，但韩国流行文化机器的真正引擎是其视频游戏产业。

2020 年，韩国内容产业出口的 66.9% 是视频游戏，使 K-pop（6.4%）和电视节目（4.5%）的销售额相形见绌。 韩国主要游戏公司已宣布计划将 NFT 集成到其产品和平台中。

公告发布后，NCSoft 股价上涨 30%（韩国股市涨停），Krafton 股价上涨 22%。另一家玩家 Kakao Games 宣布与今年最受欢迎的在线游戏《奥丁：英灵殿崛起》的开发商 Lion Heart Studio 合作，进军全球 NFT 市场。

然而，尽管允许开发 NFT 集成游戏，但韩国游戏评级和管理委员会已禁止在该国境内进行可兑现和可交易的 NFT 集成游戏，因此这些游戏只能在韩国境外使用。

监管虚拟资产

根据修正案，虚拟资产被广泛定义为“具有经济价值的电子证书，可以通过电子方式交易或转移”，从而允许未来的发展和创新被纳入这些法规。尽管 NFT 广受欢迎且前景广阔，但在韩国法律下 NFT 的法律地位尚不清楚。

争论的焦点是 NFT 是否是韩国法律定义的虚拟资产。2020 年 3 月 5 日，国民议会通过了《特定财务信息报告和使用法》的一系列修正案，极大地明确了韩国蓬勃发展的加密货币行业的规定。

然而，即使是这个广义的定义也有几个例外：(1) 不能与货币、商品或服务交换的电子证书，或发行人限制与货币、商品或服务交换的证书；(2) 先前存在的资产类别的电子版本，或不打算用作现实货币的电子证书；（3）股票、本票或信用证的电子版；(4) 电子游戏中使用的货币和物品。

虚拟资产企业必须在韩国金融服务委员会 (FSC) 的金融情报部门 (FIU) 进行注册，这涉及证明其拥有足够的信息安全管理系统并以其真实姓名开展银行业务。

虚拟资产业务还必须保持法定要求的记录保存水平，以满足修正案的反洗钱和了解客户要求。虚拟资产企业将需要在全球监管不足的行业中投资合规基础设施，以满足这些要求。

虚拟资产框架

NFT 是否会被包含在这个总括定义中仍未得到答复，韩国政府尚未做出统一回应。尽管该国金融监管机构 FSC 宣布可能通过将 NFT 视为虚拟资产来对其征税，但战略和财政部更加犹豫不决，称 NFT 作为虚拟资产的地位尚未确定。

韩国法律对虚拟资产的广泛而包容的定义为 NFT 被视为虚拟资产提供了足够的空间。如上所述，虚拟资产是一种“具有经济价值的电子凭证，可以通过电子方式进行交易或转让”。

NFT 满足这一定义，因为它们具有经济价值，以电子方式进行交易和转移，并起到电子证书的作用。

下一个问题是 NFT 是否属于该定义的例外之一。不能与货币、商品或服务交换或限制此类交换的电子凭证不能被视为虚拟资产。

根据虚拟资产金融行动特别工作组 (FATF) 的指导——该国际组织致力于打击洗钱和恐怖主义融资——例如航空里程、信用卡奖励或类似的忠诚度计划奖励或积分。

由于 NFT 可以用金钱交换（正如我们在其高价值销售中所见）并且通常不被视为忠诚度计划奖励，它们可能不属于此例外。

更接近的问题是视频游戏例外。韩国法律还将从视频游戏中收集的有形和无形物品排除在虚拟资产的定义之外，例如在视频游戏中使用的货币和物品，如总统令更具体定义的那样。

总统令禁止从电子游戏中获得的物品进行投注或鼓励投注，以及非法获得的物品（例如黑客攻击）。

然而，电子游戏中的 NFT 是否属于这一例外可能取决于每个 NFT 的特性。例如，如果一个特定的 NFT 更像是一种需要用户购买的数字艺术品，而不是游戏内的货币，或者通过赌博获得的物品，那么这样的 NFT 可以说不属于这个例外。

最终，国内政治以外的因素可能会影响韩国政府的决定。作为 FATF 的成员，韩国必须遵守其作为“反洗钱和反恐融资义务的行业标准”的指导方针。根据 FATF 的新指南，NFT 是虚拟资产。

今年早些时候，FATF 修订了他们的指导方针，将虚拟资产定义为可转换或可互换的，取代了之前将虚拟资产定义为可替代的定义。10 月 28 日，FATF 建议将 NFT 纳入虚拟资产的定义中，从而受到虚拟资产监管。韩国可能会效仿 FATF，将 NFT 视为虚拟资产。这样做会带来实际好处，例如征税和防止 NFT 被用于洗钱和其他犯罪目的。

如果韩国决定将 NFT 视为虚拟资产，则需要进一步澄清法律。例如，必须注册虚拟资产业务。经营未注册虚拟资产业务的个人可能面临最高五年的监禁，或不超过 5000 万韩元的罚款。

注册要求是否会扩展到以点对点的方式以相对较低的金额提供其数字艺术作品的 NFT 的个人艺术家？确切的税收类别也仍然是一个问题。FSC 表示，NFT 交易应作为其他收入征税，但艺术品销售产生的收入通常被视为资本收益。

如果韩国开始将 NFT 视为虚拟资产，市场参与者应熟悉有关虚拟资产和虚拟资产业务的法规，尤其是有关认证和反洗钱安全要求的法规。
