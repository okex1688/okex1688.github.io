2021年对Web3领域来说是关键的一年，它从一个新生的网络形式逐渐成为一个有潜力的产业。

在过去的一年里，我们看到了人才和资本大量流入Web3生态系统，在底层去中心化技术栈中出现了大规模的创新，出现了面向消费者的应用，这些应用正在颠覆货币、金融甚至互联网本身。

机会

Web3广义上是指由其建设者、用户和创造者拥有的互联网形式。通过利用代币和去中心化技术，旨在颠覆中心化的中介机构。加密货币、DeFi和NFT是第一批找到产品和市场契合点并获得消费者采用的应用。

然而， Web 3 技术的大规模应用才刚刚开始。加密货币仍由不到全球人数 10 %的一小部分人拥有。去中心化的金融系统目前拥有约1000亿美元的资产，但相对于传统金融系统仍然是一个小小的跌幅。Web 3 应用程序已经达到了数千万用户，但与使用 Web 2 应用程序的数十亿用户相比仍显苍白，仍然相形见绌。

正如Soundwise的Tascha最近所阐述的那样，前两波指数技术、互联网和移动技术的经验表明，一旦底层技术达到10亿用户，大众市场应用程序开始获得主要的牵引力。相比之下，目前仅有1.8亿个以太坊地址。以此作为Web3应用的代表，按照目前的增长速度，还需要5年时间才能达到10亿用户。

我们可能正处于互联网出现以来最大的技术革新的早期阶段，为风投提供了近年来最不对称的上升机会之一。

2021年市场的相关高级衡量指标：

  • 2021年，风险投资公司在全球范围内向加密货币初创企业部署了300亿美元以上的资金

  • 有65个以上的加密货币独角兽公司，其中40多个在2021年创建。2021年有近50家加密货币初创企业融资超过1亿美元

  • 加密货币总市值超过3亿美元

风险投资格局

2021年，风险投资公司对加密货币初创公司下了大赌注，截至11月底，全球投资约300亿美元，达成1278笔交易（根据Pitchbook数据）。2021年，区块链相关的初创公司平均每天融资2000万美元，平均种子融资从2020年的150万美元上升到330万美元。

我们看到以加密货币为重点的大型风险投资基金正在筹集。a16z在6月筹集了22亿美元的加密货币基金。Paradigm在10月筹集了25亿美元的加密货币基金。此外，像老虎和红杉这样规模较大、历史悠久的基金，历来对加密货币避之不及，也正在通过后期股权投资积极进入市场。

随着越来越多的机构资本流入Web3生态系统，预计估值将被抬高，风险投资基金之间的竞争将加剧。随着资金分配的竞争越来越激烈，不同垂直领域的差异化，如阶段（早期与晚期）、地域（西部与东部）、附加值（tokenomics、治理、开发者访问、流动性等）将成为基金的关键。

创业公司布局

后期阶段融资的频率越来越高，使得加密货币/区块链领域至少有65家初创公司达到10亿美元以上的估值，2021年有40多家公司达到独角兽地位。

2021年，接近50家加密货币初创公司筹集了1亿美元以上的资金。从历史上看，加密货币的估值和区块链融资一直由交易所、金融技术公司和机构服务平台主导。今年的情况仍然如此，FTX、Celsius、Gemini和Fireblocks是一些最重要的融资项目。

与此同时，我们也看到了对消费者应用的日益关注，Sorare、Moonpay、Forte和Dapper Labs的大规模融资就是证明。他们都专注于将下一个10亿用户纳入数字资产生态系统。

我们该何去何从？Web3生态系统目前是一个人才（开发者和企业家）和资本（风险投资公司和机构）的大熔炉。结合一个新的技术原语（区块链/加密货币），你就拥有了创新和价值创造爆炸所需的所有要素。随着去中心化技术和应用的成熟，以及元数据采用等后力的出现，主流消费者将不可避免地采用Web3。
