DeFi热点

1.Uniswap周交易量达到100亿美元历史新高

金色财经报道，去中心化交易所Uniswap在周二创造了新的每周交易量记录，累计交易额超过100亿美元。

2. Uniswap V3 版核心和外围合约已部署到以太坊主要测试网

据官方消息，Uniswap 表示Uniswap V3 版本的核心和偏外围（periphery）合约现已部署到主要的以太坊测试网，包括 Ropsten、Rinkeby、Kovan 和 Goerli。Uniswap 智能合约代码由两个 github 项目组成。一个是 core，一个是 periphery。Uniswap 表示，这些合约地址非最终地址，当对外围存储库进行最终更新时，将会被更改。

3.DeFiance Capital创始人：将继续买入SUSHI

三箭资本旗下DeFi风投基金DeFiance Capital创始人Arthur发布推特称，今天是有史以来交易量最高的日子之一 ，SUSHI价格却下跌了25%，与此同时他表示自己将继续买入。据DeBank数据显示，SUSHI 24H成交量突破10亿美元。

4.DeFi基准利率持续下跌至7.93%

金色财经报道，据同伴客数据显示，04月22日DeFi去中心化金融基准利率为7.93%，较前一日下跌0.4%。同期美国国债抵押回购率（Repo Rate）为0.02%，二者利率差为7.91%。 注：DeFi基准利率代表了DeFi融资难易程度，利率越高说明融资成本越高，利率越低说明融资成本越低。其与Repo Rate的利率差则便于DeFi与传统市场作进行同类比较。

5.报告：第一季度调整后DeFi TVL增长50％ NFT交易量增长超25倍

CoinDesk今日发布了2021年第一季度行业趋势报告。报告称，经过一年的指数增长和活动之后，去中心化金融（DeFi）已开始降温。在2021年第一季度，DeFi应用程序的加密资产管理总规模的美元价值似乎增长了150％，从大约200亿美元增至500亿美元。但是，大部分活动反映了以太坊和其他基于以太坊的代币的美元价值不断上升，而不是所管理的加密资产实际数量的增长。在调整此“价格效应”后，可以看到DeFi的总锁定价值（TVL）在本季度增长了50％，并在3月底逐步修正。此外，去中心化交易所（DEX）的交易量也出现了类似的稳定增长模式，并且在第一季度末略有下降。抢走DeFi风头的是NFT行业，交易量增长了25倍以上，某些市场的估值超过20亿美元。

6.YFI 创始人 Andre Cronje：有四个新产品待发布，但担心在推出后被分叉

4月19日消息，yearn.finance （YFI）创始人 Andre Cronje 撰文称“这个市场奖励了骗子和模仿者。在这个市场中，发布没有代币的产品是没有意义的，比如无法吸引资金、无法建立贡献者社区并且产品也不能自立，另外，匿名团队也会在 24 小时之内对该项目的代码进行分叉，并发布代币。目前我有四个已经完成的新产品，但担心在推出后会被分叉。我们已经开始看到版权和营业执照权开始发挥作用。”

7.以太坊智能合约自动执行工具Gelato发布V2版本

4月19日消息，以太坊智能合约自动执行工具Gelato Network发布 V2 版本。Gelato Network V2 采用钻石标准（EIP-2535），新功能包括：每个 DApp 都可以定义自己的自定义界面、无需代理合同、数据链下生成节省 Gas、无需预存款支持任何代币支付交易费用、能通过脱机签名完成交易、协调机器人以免彼此争抢。 Gelato Network 表示，作为一个经过审核的智能合约协议，支持开发人员无需运行任何底层服务器基础架构，自动执行用户交易。Gelato 的设计目标是激励中继器的开放和去中心化，以使智能合约自动化执行并使其不受审查。

8.dYdX Layer2版本总交易量突破3亿美金

4月19日，去中心化衍生品交易平台dYdX Layer2版本总交易量突破3亿美金。此前报道，dYdX Layer2版本于4月6日上线，目前支持BTC、ETH、LINK、UNI和AAVE五种代币的永续合约交易。

HECO

1.DeFiBox安全提醒：警惕Heco链上APN假币风险

DeFi门户网站DeFiBox项目监测发现，目前Heco上搜索到的APN代币为假冒APN项目假币，且买入假币的用户无法卖出，安全风险极高。据了解，目前APN项目Token并未在Heco发行。 DeFiBox提醒广大用户全面了解项目信息，注意查看Token合约辨别真伪，杜绝假币风险。

2.SUSHI已成功跨链至Heco

4月22日，Heco宣布SUSHI现已成功跨链至Heco，用户可通过Huobi Global充提入口直接跨链到Heco，未来SUSHI将作为DeFi资产，陆续打通Heco链上包括DEX交易、借贷、流动性挖矿在内的各个应用场景。 SushiSwap是一个在以太坊区块链上运行的自动做市商(AMM)的去中心化交易所，由技术极客Chef Nomi创建。 Heco是完全兼容EVM（虚拟机）的去中心化高效节能公链，可为开发人员提供高效且低成本的链上环境。 为了丰富资产多样性，方便用户使用Heco网络，Heco目前已经上线AAVE、ARPA、BAL、YFI等多个币种的Heco网络锚定Token。

3.去中心化交易所MyTrade正式登陆火币HECO Chain

据官方消息，MyTrade V1版DEX正式登陆HECO Chain, 主打挂单交易0手续费。交易者可以通过支持HECO网络的去中心化钱包里搜索MyTrade登陆移动端产品，比如火币钱包，Token Pocket，Bitkeep等。 相较于各类Swap产品，MyTrade在链上实现了订单薄与AMM高效精准的融合，这意味着交易者在市价交易的时候，交易者的订单会自动去订单簿里或者流动性池里取最优的价格来立即成交。同时，低买高卖的交易者也可以选择限价挂单的模式来交易，被成交后无交易手续费。 MyTrade V2版会在今年6月份上线，届时会空投平台币。

4.Clubhouse Media计划推出代币化数字艺术品和NFT内容之家

Clubhouse Media Group新闻稿称，该公司打算建立一个新的内容之家，聚集顶尖的艺术家、加密理财专家、行业影响者、加密投资者和NFT爱好者。Clubhouse Media计划对创作者制作的内容进行代币化，在平台上销售，同时对数字艺术品进行代币化，以备出售。

5.孙宇晨：NFT并不是泡沫，是新的艺术流派，是概念性的革命

据最新消息，波场TRON创始人兼BitTorrent CEO孙宇晨正在火星财经进行视频直播，解析NFT行业发展机遇，他在直播中表示：“NFT并不是泡沫，是新的艺术流派，NFT看的人越多，越有价值，这是艺术品革命，概念性的革命。我创办的JUST NFT基金将只专注于顶级艺术家与艺术品，原则上JUST NFT基金收藏的艺术品单价不低于100万美金，中位数价格在1000万美金左右。 NFT领域艺术品交易市场良莠不齐，频被诟病，JUST NFT基金设置入选艺术品门槛，一方面是基于优化市场秩序的考量，另一方面，我始终坚信，艺术领域的价值来自于头部效应，只有金字塔顶尖的艺术家才具有超越时间的价值。我们不完全拒绝其他类别的NFT，但我们始终以艺术品本身的价值作为考量标准，我们欢迎更多优质的作品入驻，百花齐放才是春。”

NFT

1.Taurus 将推出支持加密货币和 NFT 的交易平台 TDX

瑞士合规数字资产机构 Taurus 获得了瑞士金融市场监管局 FINMA 颁发的证券公司许可，将于 5 月 10 日推出其 Taurus Digital eXchange （TDX）证券交易平台。该平台支持投资者和银行交易多种资产，包括代币化证券、私人资产、房地产、艺术品、NFT 和加密货币。Taurus 表示，TDX 可以同时进行首次发行和直接挂牌交易，能够根据基础资产和监管框架来管理中央限价订单簿、基于拍卖的报价系统或报价系统，此外该平台还能处理在以太坊或 Tezos 上发布的智能合约。

2.基于波卡生态的新一代NFT平台SubGift获红杉资本投资

4月20日消息，基于新一代智能合约语言Subscript的NFT生态园SubGift已获得红杉资本战略投资。 据悉，SubGift底层采用Subscript语言开发，可以支持任何与Substrate框架兼容的智能合约平台，将常驻波卡生态并共享与波卡任期内所有平行链的互操作性，为用户提供可跨链、跨平台的NFT流动性服务。SubGift支持加密艺术家申请、NFT铸造、买卖、空投、拍卖、挖矿、借贷和社交服务，致力于提供与Web3.0网络适配的NFT基础设施，并增强与现实世界的价值链接。

3.数据：NFT市场从顶峰下滑，价格下降70%，购买者减少了24%

数据表明，NFT市场可能已经从顶峰开始下滑。Nonfungible.com的数据显示，与NFT销售相关的几乎所有可衡量指标都已大幅下降。在过去的一个月中，NFT的销售频率下降了30％，从179,656下降至124,371。同时，以NFT销售总额也下降了15%。上个月的平均总额为229,945,000美元，本月为193,261,000美元。同期活跃的NFT钱包数量下降了24％，而NFT的购买者数量也下降了24％。彭博社最近的一份报告显示，到4月，平均每个购买的NFT的价格下降了70％，2月份NFT平均价格达到高点4300美元，4月平均价格降至1400美元。

4.李威廉：NFT或需要度过一段非常困难的“挤泡沫”时期

欧易OKEx研究院首席研究员李威廉表示，NFT的火爆是数字经济发展到一定阶段的产物。NFT提供了独一无二的版权证明。由于NFT非同质化、不可拆分的特性，使得拥有NFT也就拥有了排他性的占有权，有力地保障了创作者的著作权，可以说NFT突然走红，是近年来数字艺术的快速发展与滞后产权保护矛盾下的结果。现阶段的NFT存在投机和泡沫，这一点是毋庸置疑的。李威廉表示，尽管目前NFT市场十分火爆，但NFT想要在未来更健康地发展，可能需要度过一段非常困难的“挤泡沫”时期。

5.NFT游戏平台CryptoFIFA获得百万美金级私募轮投资

据官方消息，NFT游戏平台CryptoFIFA 宣布获得FBG、YBB、Kryptos Research、Kernel Ventures、Horizon Capital、499 Block、7 O’Clock Capital、ChainUP等机构的私募轮投资。 CryptoFIFA 是一款基于NFT的加密足球明星卡牌Gamefi平台，将传统足球游戏中的经典玩法引入加密世界，并融入NFT、Defi元素，发展出一套更加公平、友好、可持续的游戏机制，玩法包括盲盒抽卡、赛事挖矿、阵容挑战等。6月份，体育界最盛大的赛事之一欧洲杯即将开幕，届时CryptoFIFA的赛事挖矿活动也将同步开启。

6.数据：NFT一季度销售额突破20亿美元

NFT数据统计网站Nonfungible的数据显示，截至2020年年底，NFT市场交易量超过2.5亿美元。今年第一季度NFT销售额更是突破20亿美元，环比增长至少20倍。

7.去中心化流动性协议Tokemak筹集400万美元

去中心化流动性协议Tokemak获得400万美元的投资，由DeFi投资基金Framework Ventures领投，Electric Capital、Coinbase Ventures、North Island Ventures、Delphi Digital和ConsenSys等参投。这笔资金将为Tokemak于2021年第二季度末的推出做准备，其将在以太坊主网上部署其流动性网络。

8.NFT平台 NFTb将推出新版本，支持社会公益事业

NFT平台NFTb正在推出其市场的新版本，该版本专注于支持社会公益事业，并为艺术家提供更实惠的作品上线选项。新的市场将艺术家和主要慈善机构联系起来，允许任何人以负担得起的价格拍卖其艺术品，并将部分收益捐赠给自己选择的慈善机构。 NFTb声称已有120多位艺术家加入了候补名单。
