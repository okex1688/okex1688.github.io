不可替代代币 (NFT) 已经走过了漫长的道路，并在过去几年中占据了时代精神。主流焦点主要集中在个人NFT头像（PFP）项目上，这得益于 CryptoPunks 和 Bored Ape Yacht Club（无聊猿猴）日益增长的关注（和价值）。

CryptoPunks 和 BAYC 的价格飞涨……

引发了NFT头像的热潮。。。。

这些项目说明了 NFT 启用的一些新兴属性和行为，首先 ，NFT 提供了独特资产的真正数字所有权。因为人们确信他们实际上拥有这些资产，所以他们可以（并且经常）将它们用作其数字身份的表示。甚至像 Twitter 这样的 Web2 平台也将NFT 作为个人资料图片。而且，就像在物理世界中一样，这些化身产生社会属性，例如状态信号、封闭社区、社会等级和其他相关行为，所有权也超越了原始文件本身。例如，无聊猿游艇俱乐部的成员拥有猿猴的知识产权，允许他们使用猿来推出啤酒品牌、漫画，甚至虚拟乐队。

其次 ，这些资产的可编程特性支持广泛的用例，这些用例通常在现实世界的对象中不可用。可以对 NFT 进行编程，以响应所有者采取的行动、对外部触发做出反应或随时间变化。他们可以嵌入版税，使创作者能够与他们的社区一起参与未来的收益。而且由于所有这些都发生在区块链上，创建者可以通过空投或早期访问未来项目来识别和奖励所有者。

所有这些功能结合在一起可以引发良性循环，社区成员从草根倡议中创造价值，推动对其资产的需求，通过销售版税为创作者提供更高的收入，他们可以利用这些收入进一步扩大项目范围。

事实上，PFP（图片资料）一直处于 NFT 采用和实验的最前沿，并且是迄今为止最大的 NFT 垂直领域。但在它们之外，NFT 实验出现了寒武纪大爆发，包括生成艺术、不可替代的虚拟世界、交互式 NFT、不同形式的媒体（如 NFT）、NFT 的金融化等等。

NFT 市值（按类别）

在 Amber，我们相信元宇宙 —— 一个定义松散的概念，指的是实时、可互操作和身临其境的数字体验——是不可避免的，并且 NFT 是这些虚拟世界的关键原语。在本报告中，我们谈到了一些有趣的项目，这些项目在 NFT 可以做什么方面进行了创新，以及读者可以用来导航不断增长的 NFT 空间的工具和框架。由于跨多个区块链的庞大且快速增长的 NFT 项目，我们主要关注基于以太坊的项目。

土地掠夺

新兴元宇宙的一个关键问题是：用户将在哪里？随着人们涌入拥有人们聚集的空间，许多平台的虚拟土地销售在过去一年中有所上升。也许这场虚拟土地热潮的两个最大受益者是 Decentraland 和 Sandbox，这两个都是免费游戏平台，拥有有限数量的代币化土地。

数字土地抢夺

购买土地背后的理由之一是举办元宇宙活动和体验，然而，Decentraland 和 Sandbox 都是相对较新的平台，尚未构建基础设施以适应高流量负载。例如，三星试图在 Decentraland 举办 Galaxy 智能手机发布会，但在人们试图获得访问权限时遇到了技术难题。Decentraland 联合创始人建议该平台的最大并发量仅为1~2,500。即使是现有的具有成熟服务器基础设施的开放世界平台仍然难以处理拥有数百名玩家的环境——Free Fire（大逃杀游戏），一款仅限移动设备的大逃杀游戏，每个竞技场最多只能容纳 50 人，而Roblox 仅在 beta 测试中最多可以处理 700人。

因此，一些项目没有重建整个技术堆栈，而是决定在现有基础设施上迭代开放元宇宙的概念，NFT Worlds 就是这样一个在加密社区中获得越来越多关注的项目的一个例子。

NFT 世界

NFT Worlds 是一个社区驱动的元宇宙平台，建立在 Minecraft 的开源生态系统之上。通过在 Minecraft 之上引导一个开放世界，NFT Worlds 可以利用跨平台支持和现有的世界构建工具。

共有 10,000 个独特的世界，元宇宙数据涵盖 39 个类别，包括土地面积、水域、年降雨量、木材和金属等资源。这些特征决定了每个世界的稀有度，可用于不同的游戏元素。这些世界的持有者可以使用现有的 Minecraft 启动器或即将推出的 NFT Worlds Launcher 来构建和编辑他们拥有的世界。

世界所有者可以自定义他们的土地并同时托管约 600 名用户。此外，NFT Worlds 的目标是在 2022 年第二季度推出多人语音聊天、用户生成的游戏，并支持自定义头像。由于这些功能，该项目吸引了广泛的 NFT 社区越来越多的兴趣，希望举办元节活动。自 2021 年 10 月以来，NFT Worlds 的土地销售总值显著增长，近几个月甚至超过了 Decentraland 和 Sandbox。

NFT Worlds 过去三个月的月交易量超过 2000 万美元

截至目前，已有超过100 个 NFT 社区公开宣布收购 NFT Worlds，包括RTFKT、Zipcy 的 Supernormal和WVRPS。任何人都可以在 Web 浏览器中查看特定的世界，例如World #9856，它已经被自定义。

NFT Worlds 所有者将收到两轮 WRLD 代币的空投。他们还可以抵押和/或出租他们的土地来获得额外的代币。WRLD 的最大供应量为 50 亿个代币，其中约 85% 的代币在五年内分发。

WRLD 分发时间表

该团队最近推出了一个以 WLRD 为中心的支付层，将用作生态系统内的主要交换媒介，包括即将到来的游戏赚取和奖励机制。交易费用也以 WLRD 支付，让玩家无需使用其他代币管理成本。

尽管最近对 NFT Worlds 进行了大肆宣传，但其土地的隐含价值和 WLRD 代币的完全稀释估值仍然比前两个加密原生开放世界平台低几个数量级。

NFT Worlds 与其他元宇宙平台

当然，NFT World 的方法也有一些缺点。例如，微软（Minecraft 的所有者）可以关闭该项目。尽管 NFT Worlds 团队表示，他们一直与微软的 IP 团队保持密切联系，迄今为止没有遇到任何问题，但开放世界经常冒着创始团队无法控制的不当行为的风险。

尽管如此，我们正在密切关注该项目，作为我们论文的一个指标，即 web3 的未来必须在短期内在 web2 rails（Rails 是使用Ruby 语言编写的Web 应用开发框架）上构建和迭代。虽然我们相信开放的元宇宙平台最终将拥有更多去中心化的技术堆栈（例如，内容分发、服务器和存储），但这些部分尚未完全构建。与此同时，像 NFT Worlds 这样的项目可以更好地反映实时、交互式和社区驱动的开放元宇宙的样子。

支持 NFT 的 DAO

去中心化自治组织 (DAO) 是由共同的使命和价值观组织成的团体，通常，这些 DAO 是通过可替代的代币形成，例如 BitDAO (BIT)、ConstitutionDAO (PEOPLE) 和 Friends with Benefits DAO (FWB)。

NounsDAO 是在 NFT 的背景下形成 DAO 的案例研究。每天随机生成和拍卖的名词是一个 32x32 像素的人物、地点和物体的字符。名词的主要定义特征是彩色方形眼镜和像素化艺术品。

愚蠢的Nouns画廊 

NounsDAO 是一个开源项目——创建名词的代码和艺术品本身都放在公共领域。拍卖收益的 100% 直接进入 NounDAO 国库，每个 Noun NFT 所有者对国库资金的使用有一票。

第一个名词 NFT 于 2021 年 8 月 9 日以 613.37 ETH 的价格售出，当时约为 190 万美元。从那时起，价格从 24.2 波动到 313.69 ETH，自今年年初以来每个Nouns稳定在 80 ETH 左右。该国库目前拥有超过 20k ETH（在撰写本文时约为 6000 万美元），DAO 用这些资金来资助创造性（有时是疯狂的）提案，以进一步巩固 Nouns 的文化相关性，包括在 Superbowl 商业广告中展示 Noun 眼镜并发送国际空间站的名词。

初始炒作之后的Nouns价格稳定

由于许可证可在公共领域获得，因此生态系统中涌现出广泛的社区驱动计划和衍生艺术作品，包括一个 “ 通知 ” 您现有个人资料图片的应用程序和一个试图生成可能是下一个Nouns。

NounsDAO 是一个令人兴奋的社区驱动品牌建设实验，反映了 NFT 可以带来社会作用。Nouns 项目拥有大量资金和参与度高、联系紧密的社区，是一个值得关注的项目。

NFT 作为实用程序

主流 NFT 话语围绕社区 (BAYC)、艺术 (Fidenza收藏品) 或游戏 (Axie Infinity) 项目展开。但是，我们看到无数团队希望利用 NFT 的独特属性为用户提供实用的应用程序。

点燃特斯拉

Chris Cassano（区块链黑客）通过将他的 Tesla Model 3 连接到 NFT展示了 NFT 的有趣用途，使用Lit 协议——允许任何人通过区块链对象授予对内容、软件和数据的访问权限——克里斯允许任何拥有特定 NFT 的人控制他的特斯拉，包括闪烁灯、锁定和解锁汽车以及启动引擎。

用 NFT 控制特斯拉

使用 NFT 作为车钥匙的利弊可以争论，但这个实验说明了 NFT 可以带来的实际作用。例如，像 Airbnb 这样的平台可以通过使用过期的 NFT 授予客人访问主人家的权限——客人可以确保访问权限，并且主人可以省去在每位客人入住后更改密码的麻烦。

Superfluid（流动支付）

Superfluid支持可编程的资金流，例如订阅、工资和经常性奖励。它的最新应用之一是将代币流式传输到 NFT，而不是以太坊地址。因此，如果 NFT 易手，新所有者会自动接收资金流（在一定的时间线程内，资金会自动逐渐转入你的账户）。此功能为可编程现金流打开了广阔的设计空间，例如，Superfluid 团队建议可以对元宇宙财产收取租金（在流中），如果该房产被出售，来自租金的现金流会自动流向新业主。人们还可以取出由这些 NFT 支持的贷款，并通过基础现金流随着时间的推移自动偿还贷款。

会员资格和访问权限

一些组织正在尝试使用 NFT 作为会员通行证的门控访问。LinksDAO的成员包括 NBA 球员斯蒂芬库里和 JJ 雷迪克，他们正试图创建一个通过 NFT 认证的会员资格的高尔夫和休闲俱乐部。Flyfish Club同样旨在打造一个专属于 NFT 持有者的私人餐饮俱乐部。Coachella 的 10 个 NFT 密钥授予终身访问 Coachella 活动的权限，拍卖价格为 55,000 美元。

通过 NFT 对成员身份进行身份验证和门控为传统方法提供了一些优势。首先 ，与大多数私人俱乐部模式相比，会员可以轻松找到并识别其他会员（或钱包地址，如果某些会员选择保持匿名）。其次 ，现有会员可以在二级市场上转售他们的 NFT，这是大多数传统会员无法实现的壮举。由于转售价值，人们可能愿意为这些通行证支付更高的价格。这些通行证甚至可以在二级市场上出租，将通常的消费转变为产生现金的资产。最后，因为这一切都发生在区块链上，所以成员身份验证和身份验证是透明的。

NFT交易

蓝海和Opensea

大多数用户首次涉足 NFT 是通过 OpenSea，根据交易量，它拥有超过 95% 的市场份额。然而，许多新平台已经进入市场，针对特定的子行业（例如，生成艺术）、不同的用例（例如，NFT 交换），或者仅仅是 OpenSea 的市场份额的一部分。

今年1月，LooksRare在OpenSea推出其平台时，通过向OpenSea用户空投本地令牌来 “ 吸血鬼攻击 ” OpenSea。LooksRare成功地在加密获得了广泛的用户，利用了OpenSea可能会首次公开募股(IPO)而不是为用户生成令牌的失望情绪。

乍一看，新市场似乎从 OpenSea 手中夺走了市场份额，但 LooksRare 推出了一项积极的质押和奖励计划，用户在平台上的刷交易量奖励代币。在过滤掉大部分的刷交易量活动后，其平台有活动也难以显得生机盎然。此外，在 2 月 9 日 LooksRare 的奖励减半后，交易量大幅收缩。

LooksRare 的交易量以有限的有机活动为平台刷交易量

OpenSea 继续占据主导地位

LooksRare Filtered 减去两个钱包之间的 “ 来回 ” 交易活动和异常高价的交易。

Opensea尽管偶尔会出现交易暂停、网络钓鱼攻击和审查，但由于其网络效应，OpenSea继续主导着NFT的交易量-用户将汇聚到其他用户所在的地方。但NFT的采用仍处于初级阶段，新的市场将不可避免地出现，挑战OpenSea在市场上的据点。就在本月，另一家NFT平台X2Y2向用户空投代币，并提供奖励制度，鼓励高质量的新项目上市交易，而不是提振交易量，从而对OpenSea发起了吸血鬼般的攻击。宇宙，一个DAO驱动的NFT市场，预计也将在未来几个月的某个时候推出。Coinbase的NFT交易平台预计也将于今年晚些时候推出，目前约有380万人在等待上市。未来几年，OpenSea的主导地位将继续受到考验。

NFT交易聚合器

随着新市场平台的推出，我们正在密切关注这个领域的NFT交易聚合器。这些聚合器具有多种功能，包括找到最优惠的价格以及在多个平台上无缝列出要出售的 NFT。它们还允许批量购买，允许用户轻松 “ 扫荡地板价 ”（通过购买列出的最便宜的系列来提高系列的底价）并在多个系列中购物。

目前有两个主要的 NFT 聚合器：Genie和Gem。Genie 于去年 7 月首次推出，允许用户在 OpenSea 和 Rarible 上轻松交易，自撰写本文时推出以来，交易量已接近 109k ETH。Gem 于今年年初推出，由于节省了更多的气体和更快的产品迭代（例如，与 LooksRare 集成、推出 Sweep Mode），其销量仅在几周内就超过了 Genie。

Gem 在短短几周内就超过了 Genie

Gem 为用户节省更多的手续费

这两个交易聚合器的主要目标都是JPG Flipper，但我们预计它们的用例会扩展得更广。例如，我们看到交易聚合器在NFT游戏资产的环境中发挥着重要作用，这些资产既可以在游戏的专有市场上市，也可以在开放平台上上市。在这种情况下，聚合器可以帮助玩家轻松地跨多个市场找到他们想要的物品，或者发现可以在多个游戏之间互操作的资产。

NFT金融化

随着某些 NFT 收藏品的价格上涨，某些所有者的新痛点是将其资产的代币化。一些平台现在提供由 NFT 抵押品支持的贷款，模仿现实世界不可替代商品（如房屋、汽车和奢侈品）的抵押品支持贷款。

NFTfi 是最大的 NFT 去中心化借贷的点对点平台。借款人可以在平台上列出和抵押选定的 NFT，以向贷方征求贷款条款。这些以 WETH 或 DAI 为主的贷款可以持续长达 90 天，平均年利率约为 70%。如果借款人拖欠贷款，贷款人可以扣押底层的 NFT 抵押品。

NFTfi 于 2020 年 5 月推出，并得到越来越多的采用，迄今已促成约 7200 万美元的贷款。Bored Ape Yacht Club、Art Blocks 和 CryptoPunks 占贷款总额的近 50%。目前，像 Metastreet 这样的大型贷方对蓝筹资产的最高贷款价值比率为 45% 。

NFTFi 统计

NFTfi 预计将在本季度推出 “ NFTfi V2 ”，其中包括几个新功能，包括无限期贷款、贷款延期和贷款重新谈判。该团队还计划通过空投最终的 NFTfi 代币来奖励以前的用户。

NFTfi 成功利用了对 NFT 抵押贷款不断增长的需求。然而，它的点对点模型也有一些缺点。借款人无法立即获得流动性；他们必须等待贷方提供可接受的条款。另一方面，贷方必须手动为平台上列出的每项资产提供贷款条款。贷方还必须在钱包中存放足够的 WETH/DAI，直到贷款被接受，面临闲置、非生产性资本的风险。

我们看到 NFT 金融与 DeFi 的早期阶段之间存在相似之处，EtherDelta（OpenDAO创始人瞿佳炜，后来跑路到英国，被人维权）提供点对点交易所交易。

Uniswap 通过其点对池模型获得市场采用，同样，在 NFT 垂直领域，我们正在密切关注寻求使用流动性池为 NFT 支持的抵押品提供即时流动性的平台，对我们来说突出的一个平台是 Pine。

Pine

Pine 是一个早期项目，旨在解决两个问题：为拥有的 NFT 提供即时流动性，以及为有抱负的高价 NFT 买家提供 “ 先买后付 ” 的融资。他们计划在本月为少数列入白名单的 NFT 推出即时流动性。

机制相对简单，贷方将他们的代币（例如，ETH、USDC）质押到选定的保险库中，称为“ Pine Pools ”。每个保险库都支持一个集合（例如，BAYC、Doodles），其中包含最大借款金额（基于特定的抵押因子）、利率和可用期限。借款人可以访问这些金库并立即针对他们的 NFT 进行贷款。为提高资金效率，流动性提供者存放在池中的资产可以用于产生收益的协议。

Pine 协议设计

Pine 的借用功能目前处于 alpha（超额回报）阶段，因为团队测试了平台并扩展了他们支持的集合套件，Pine 的目标是在 2022 年第二季度发布其 “ 先买后付 ”（BNPL）解决方案。

诚然，Pine 的解决方案是新颖且未经测试的。也许无需许可的点对池模型不能很好地从可替代代币转换为不可替代代币。尽管如此，即时流动性和 BNPL 解决方案对于增强 NFT 的可访问性和功能都至关重要。在与 Pine 团队就其设计和潜在挑战进行了交谈后，我们对他们实现这一愿景的能力印象深刻并充满信心。[披露：Amber（金融智能服务提供商）是 Pine 的投资者。]

超越炒作

目前以高于40%的利率借入ETH几周的用例似乎有限。我们推测，大部分需求来自杠杆NFT交易，以一种NFT为抵押购买另一种NFT，目的是为了快速获利。

现在还为时尚早，在现实世界中，不可替代商品的金融化明显改善了可获得性，并创造了新的资产类别(例如，抵押贷款和汽车贷款扩大了住房和汽车拥有量)。

我们预计NFT也会出现类似的情况。NFT不仅仅是图像-它们可以反映流动性(Uniswap v3流动性提供者头寸)、知识产权(如音乐和专利)、甚至现实世界资产的所有权。例如，就在几周前，佛罗里达州的一套房子被拍卖为NFT。在未来，或许所有者可以在未经许可的情况下，以这种抵押品为担保的贷款，而不必与多家银行谈判，核实信用评分，并处理文件工作。

NFT 是否属于投资者的投资组合？

将 NFT 纳入投资组合有一些优点，NFT 和相关技术是元宇宙的关键组成部分，并将在未来十年内呈指数级增长，资产类别的表现已经开始偏离整个加密货币市场。尽管历史数据有限，但来自多个 来源的分析表明，NFT 和加密货币之间的相关性很低，这是投资组合管理中非常理想的属性。不同类型的 NFT（例如收藏品与 1-of-1 艺术品）之间的相关性也很低。最后，考虑到资产类别的新生，可以说获得阿尔法（超高回报率）的空间很大。

另一方面，虽然我们相信 NFT 作为一种广泛的资产类别将在未来有显著扩展，但我们怀疑大多数 NFT 项目最终将几乎没有价值。此外，市场仍然由大玩家主导。根据NFTGo的数据，只有约 1,100 头鲸鱼（大户）持有价值近 40 亿美元的 NFT，约占 NFT 总市值的 23%。更不用说刷交易、内幕交易、黑客攻击、诈骗、未公开广告、快速致富 NFT 项目等问题。

因此，买家要小心。我们建议，对于大多数人来说，购买 NFT 主要是为了实用性、艺术品、社区和其他无形利益，而不是为了潜在的价格升值。尽管如此，对于那些愿意在风险范围内进一步冒险的人来说，有选择地投资 NFT 可以提供多元化收益并有助于提高回报。

因此，下面我们提供了一些指标和框架来帮助读者在 NFT 项目之间导航和评估。它主要关注 PFP （NFT头像项目）和收藏品，这是目前占主导地位的 NFT 类别。与物理世界中的不可替代商品一样，NFT 的估值仍然不精确，并且很大程度上依赖于叙述。对于艺术品和收藏品，社区、文化和艺术品都在感知价值中发挥作用。尽管如此，数字和指标可以帮助衡量项目的优点并揭示危险信号。

评估 NFT

谁是持有者？

使用Etherscan等开源工具或社区创建的 Dune 仪表板（如@rantum 的），人们可以验证谁是项目的主要持有者。使用手动挖掘或更专业的工具（如 Nansen），他们还可以检查 “ 钻石手（获利就卖出的人） ” 的余额——那些在购买/铸造资产后从未出售过的所有者。

猿猴和企鹅

这些练习提供了几个好处。首先，如果顶级钱包（通常是创始人和内部人士）控制着不成比例的资产，NFT 所有者就有这些钱包在市场上 “ 倾销 ” 资产的风险。其次，越来越多的 “ 钻石之手 ” 体现了社区的力量和信念。此外，这些指标表明项目资产的分布，项目在广泛的参与者中的均匀分布减少了销售压力并鼓励更多的社区参与。

价格

NFT 项目最常见的定价衡量标准是 “ 底价 ” ——目前市场上最便宜的资产的价格。NFT 项目中的资产通常具有不同的稀有性特征，地板价物品通常是最稀有但流动性最强的，相比之下，稀有性较高的资产通常流动性较低。

底价通常被用作衡量藏品整体价值的晴雨表，但高稀有资产也很重要，稀有资产的高价出售可以引起市场对收藏品的关注，特别是如果搭配一个可识别的名字，并引发更高的需求。[请注意，有时，稀有物品也可能是刷量交易的结果。]

名人购买和品牌代言可以引发价格上涨

如果一个项目没有给出官方稀有度指南，Rarity Sniffer 、Rarity Tools 这两个项目可以评估项目的稀有度。

潜在买家还可以评估上市资产之间的价格列表，类似于加密货币的订单簿。下面，Azuki 的底价为 11.5 ETH，但只需购买 12 次即可达到 13.0 ETH。相比之下，考虑到 5.0 ETH 的上市集中，mfer 的底价在 12 次购买后并未从 4.8 ETH 的底价大幅上涨。

Azuki 地板价格

列表数量占总数的百分比是读者可以用来衡量情绪的另一个指标，数字越低，当前的卖家越少，价格上涨的可能性就越大。Flips（数据工具）是一个有用的工具，可以将这些指标汇总到各种集合中。

仪表板以显示 NFT 集合范围

请注意，NFT 的销售可以而且经常确实发生在场外交易平台上，以避免像 OpenSea 的 2.5% 平台费用这样的市场收取率。这种方式的 “ 买/卖 ” ，通常会在 Discord 洽谈他们的交易，这也可以为潜在买家和卖家提供一个粗略的情绪衡量标准。

NFT：仍处于第一局

随着 NFT 流行语进入主流话语和某些系列看似荒谬的价格，一些评论家将整个行业视为泡沫或时尚。当然，有一些炒作和过度兴奋，这个空间已经充满了崇高的承诺和未实现的路线图。

然而，与百万美元的朋克和衍生猿猴相比，NFT 只是简单地封装了不可替代和独特资产的数字所有权。在现实世界中，我们购买和消费的大多数物品都是不可替代的，从房屋和汽车等高价值商品到音乐和衣服等低价商品。以数字方式启用这些动态极大地扩展了我们在网上可以做的事情。因此，我们重申我们的信念：NFT原生数字经济是广泛元宇宙中的重要组成部分 。

从这个角度来看，NFT 的增长之路很长。NFT 的全球总市值估计约为 160 亿美元，与目前超过1 万亿美元的 TAM（总可寻址市场）估计值相去甚远。TAM 的估计也可能低估了最终的市场规模，因为 NFT 通过将所有权授予一系列数字资产，实现了以前无法在线实现的一系列类别。

本报告仅触及快速发展空间的表面，我们期待在未来的报告中涵盖将 NFT 推向新领域的新项目和新想法。
