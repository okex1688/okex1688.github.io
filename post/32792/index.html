什么是Web 3.0，它和元宇宙是什么关系？

我们先说答案：

1、Web 1.0->Web 2.0->Web3.0 其实就是 只读 ->读写 ->可拥有（包括可读可写）；

2、元宇宙是Web 3.0的一个大的应用场景，也是5G、VR/AR、区块链、NFT等的应用场景。

下面先以菜农的故事来比喻，帮助大家理解。相信读者您，会有更好的理解和比喻。

下面有关互联网的阐述，主要是指，针对绝大多数用户而言。需要注意的是，1.0, 2.0, 3.0 甚至到未来的4.0,……，是逐渐演化而成的，并没有泾渭分明的界线 。例如，国内Web1.0的典型代表新浪较为敏锐地，快速增加了Web2.0的功能新浪博客；2003年Google收购了blogger.com等事件屡见不鲜，包括Facebook进军元宇宙甚至更名，能看到一些IT巨头也在不断演化、迭代自己。 

一、Web 1.0

个体户卖菜，在自家农田旁边，或者自家门口。

优点是：距离近、能自主；缺点是：菜的种类极其有限，辅助卖菜的工具和环境都得自己准备、维护，还要准备大量的零钱；缺少流量，人来人往较少。

图片摘自：百家号

对应到互联网，Web 1.0的用户逛一逛、看一看网络信息（早期多是精英创建的，独立的个人网站），也就是用户 参与的这个Web是“只读”的互联网。

此时的互联网，对绝大多数用户而言，主要是信息的聚集地，包括早期命令行形式的BBS（电子公告板，Bulletin Board System的简称）。

下图是1991年在中国，罗伊用Telnet （ 一种电话线路 ）搭建了国内最早的拨号连接的 BBS 站点 —— 长城站，是按FidoNet协议建立的。

图片摘自：百家号

1994年出现了包括文字和图片的YAHOO门户网站。如下图：

图片摘自：https://www.sohu.com/a/412070632_114819

中国出现了搜狐、新浪、网易、腾讯、TOM、China.com等门户网站。后面出现了轻微互动（也即键入关键词）的搜索网站，如Google、百度、中搜网等，用户获取信息的手段开始朝着个性化程度更高的趋势发展。

二、Web 2.0

像赶集，或菜市场一样，有专门的人或组织来管理，包括搭建场地、提供保鲜、切削剁砍等工具、整钱换零、清理和准备环境等，收管理费或手续费。

对于卖菜人而言，优点是：人流量大大增加，“拎包入住”交租金即可；卖菜的，也可以是买菜的，买到自家没有种植，但又想品尝，或者喜欢的菜。赶集的人，除了买菜，也可以卖菜。

缺点是：自主性减弱，要办营业执照或获取某种资格；菜市场逐渐形成垄断地位，例如靠科技、管理水平、市场配置能力、资本和市场地位等把其他竞争的菜市场都扼杀在萌芽之中，只要愿意菜市场管理机构可以坐地涨价，收30%甚至更高的手续费。

图片摘自：https://www.sohu.com/a/485846633_100049550

对应到互联网，网站从媒体属性（新闻,信息等单向传输）走向交互属性（出现UGC，User Generated Content，即用户创造内容，互动，双向传输）；从文本走向图片、音频、视频兼具的多媒体形态。

Web 2.0演化为平台，用户可以看，也可以发表信息，是“可读可写”的互联网。

国外国内的代表有：1999年8月，Pyra Labs创立的blogger.com；新浪博客，如下图。到后面的Facebook（现更名为meta）、Twitter、Youtube、Musical.ly、优酷土豆、今日头条、微信公众号、快手、抖音等。

上图是摘自寒的新浪博客

三、Web 3.0：价值重新分配

问题来了。有个卖菜的姑娘，闲时用菜市场的切削工具，将白萝卜等雕刻出美丽的花朵。这个花朵的所有权算谁的？可以自主出售吗？卖的钱算谁的？

买主A拿着美丽的萝卜花出去炫耀（相当于宣传），被爱不释手的买主B用更高的价格买去，买主A获得的差价如何分配？

买主B在萝卜花上再次加工和创造，更精细更形象了，买主C用买主B的十倍百倍的价格买走，买主B的收益又该如何分配呢？

下一代“菜市场”，或者说理想的，应该是怎样的？生产者，甚至传播者、消费者也应具备一定的管理权、话语权，参与到规则的制定。

例如，未来是否可能出现，菜农甚至包括买菜的，通过众筹的方式来建设和管理菜市场，管理规则大家一起讨论，最终推选经理班子来管理？虽然效率低但能达成共识，手续费也比较合理，能够可持续的健康发展。

这样，有可能尽力保障生产者的收益大部分归自己。这类虽然讲的是菜市场，也许不一定合适，各行各业有没有哪个可能适合，欢迎留言？

说回互联网。上面比喻可能不恰当，但类似的问题不要小看，当人类的生活从时长、深度和广度在元宇宙或数字世界里逐渐增加时，因为身份、社交、信用、情感、精神需求、数字资产等方面的归属甚至依赖，会变得越来越重要，不可忽视。

例如：2020年12月，韩国游戏玩家因花费近3351万韩元升级的游戏角色无故消失，将谷歌韩国等公司告上法庭；2020年1月， Facebook、、Twitter、谷歌等互联网巨头封锁了时任总统特朗普的账号；2021年，国内有一场官司引起了大家的关注，焦点在于游戏玩家的金币、道具、装备等到底归游戏公司，还是玩家？

核心在于用户的数据谁做主？以往互联网巨头利用用户隐私数据，通过大数据和AI实现精准营销，在用户为了获取便利并不那么看重隐私的前提下，不断发展。 然而随着政府、组织、个人越来越重视隐私、数据确权、数字资产，这类红利逐渐见顶。

Web 3.0的出现和引起大家重视，正是为了尝试解决上面的挑战（用户和平台之间的冲突）。互联网巨头也逐渐意识到以往模式的不可持续，例如2021年11月初，Facebook 宣布将会关闭面部识别系统，因为来自用户和监管者的担忧越渐增强，Facebook 将会删除超过 10 亿用户的个人面部识别版本。

不过，如同元宇宙一样，大家对Web 3.0也有自己的不同理解，早期Web 3.0更多是指语义网，区块链和DAO出现后，渐渐地更多人把Web 3.0视为一种去中心化的网络，有时会简称为Web3。

例如：2006年5月，蒂姆·伯纳斯-李（发明万维网、第一个浏览器）：“在Web 2.0的基础上大面积使用——所有东西都起波纹、被折叠并且看起来没有菱角——以及一整张语义网涵盖着大量的数据，你就可以访问这难以置信的数据资源”。语义网是一种可以通过机器处理语义的内容网络。

2006年11月的Technet峰会上，Yahoo杨致远：“你不一定得是计算机科学家才能创作出一个程序。这种现象在Web 2.0里初现端倪，而3.0将更加深化，是一个真正的公共载体……专业、半专业和消费者的界线越来越模糊，创造出一种商业和应用程序的网络效应。”

而Web3D联盟提出3D化构想：“将整个网络转化为一系列3D空间，采用第二人生（Second Life）启发的概念。同时也提供新的方式在3D共享空间连接和协同”。

有的人认为Web 3.0是：将互联网转化为数据库；有的强调这是一条最终通向人工智能的网络进化的道路；有的着重在 Web 3.0无处不在，也即无处不联网。

笔者更偏向于去中心化的理解。我在2021年9月23日参加WEB3.0云响力峰会时曾发表过：区块链IT基础设施的建设，是一种新的运营模式。非中心化云计算平台将所有权、运营权、使用权三权分置，是一种从“使用它，不一定要拥有它”到“运营它，也不一定要拥有它”的转变。

去中心化（或叫非中心化）也好，新一代云计算也好，分布式云也好，在笔者看来，核心就是在平台层次的三权分置。需要注意的是，在不同层次的三权分置，又会演化出不同的商业模式或业态，丰富多彩。例如物理世界实物，如Airbnb，滴滴顺风车；再如运营平台，如ETH、Filicoin、CityDAO等；虚拟商品如NFT的三权分置，又能探讨出多种模式。

1、如果谈网络平台，这个平台不只是由几个大股东操控的，是利益相关者共同所有的，这里的利益相关者包括了生产者、消费者、运营者还有传播者。类似以太坊、IPFS Filicoin，大家共创共治，推进生态壮大，共享发展带来的收益。我曾经在分享过类似的逻辑。当然，这里指的所有，是部分拥有，或者是拥有部分，哪怕是很小的一部分，具有共识的人群汇聚起来，也是一股不可忽视的力量。

2、如果谈内容和创意，刚开始一定是生产者所有，但吸引流量、靠人宣传，自然会使得生产者通过合理的激励机制，分配部分利润给运营者、消费者和传播者。

我比较认同孟岩在原创文章，他说Web 3.0 的范式是： 用户创造、用户所有、用户控制、协议分配。 如果进一步简练，我愿意说成：Web 3.0是用户“可部分拥有”的互联网。

笔者认为，在此基础之上，如果能够满足用户多种感官，如眼耳鼻舌意身，也即除了视觉听觉之外，还能调动触觉、嗅觉、味觉、体感、甚至意识（如催眠、冥想）等，就更完善了，也即从Web 2.0的平面跃升到多维立体的网络空间。

这也是部分专家，将其比喻为3D互联网的原因，不过基于已经或者即将实现的多种感官的实现，个人仍觉得含有定语“多维”或许更持续一些。

四、Web 3.0和元宇宙的关系

首先，笔者认为在当下：元宇宙是以区块链为基础，虚实融合的（可迭代为虚实相生的），由创作者驱动的，共创、共治、共享的数字新世界，简称多维共创互信网。其中互信对应的是以区块链为基础，虚实融合或虚实相生，包括了AR/VR、脑机接口、全息投影等交互技术和展示技术。

在2021年中国数据与存储峰会，11月25日下午的【区块链技术与实践】论坛，笔者在题为“元宇宙与区块链IT基础设施”的演讲中曾分享过，“演讲文档和视频《元宇宙与区块链IT基础设施》”，并在12月16日下午新浪的“2021科技风云榜”再次提及。

受万向区块链董事长肖风老师的启发，在笔者看来，元宇宙是下一代网络。是从物质->能量->信息->价值，再到创意和意识的网络。

之前的有物质的网络，如交通运输网；能量的网络，如电网；通讯和媒体的网络，如电报电话和收音机电视；信息的网络，如互联网和移动互联网。

在2018年7~8月间，应日知录、比特大咖的分享，提到：“能量、信息、人才、资本或价值，任何能促进以上四要素高效流动的方式都能促进社会的进步”。

在2019年12月7日在公司内部分享《区块链和存储》系列讲座时，采用了下图 - 解耦和自由流动。

区块链是价值的网络，信任的网络；元宇宙会不会成长为创意和意识的分享、交换的网络？ 

毫无疑问，元宇宙是一个数字新世界。其实是大家对未来数字新世界的想象共同体。

早在30年前左右，东方西方都对这个想象共同体有了阐述，也即钱学森将VR翻译成的灵境（1990年提出，可视为元宇宙初级版），和大成智慧（1992年提出，meta-synthetic，可视为元宇宙终极版）；美国科幻小说家尼尔森1992年在《雪崩》里提到的metaverse。再往前数，盘古、女娲、山海经、天宫、地府、宙斯、圣经等对于的意识世界，也可以看成是“元宇宙的雏形”。

在当下，我们也可以把元宇宙看成是一个场景共同体。16日下午也提到，“把元宇宙看成是web3.0的一个大的应用场景”，其实不只如此，元宇宙也可以看成是5G、VR/AR、AI、IoT、区块链、NFT、DeFi、GameFi、SocialFi等的场景。

很高兴的是，就在昨天（2021年12月17日），看到《广义虚拟经济：二元价值容介态的经济》作者林左鸣老师转了一篇经济学家、万博新经济研究院院长的好文《滕泰：元宇宙是投机炒作，还是人类未来？》，系统深刻地表达了类似的观点：元宇宙是场景创新的大爆炸。包括文中他对5G的看法，我也非常认同，我曾在《为什么中国要不遗余力的推进5G？》有类似观点。

元宇宙现在确实太热，我们要当心热过头，糊了！需要警惕借此行骗、传销、割韭菜的情况。

当然也要看到，国外巨头已经纷纷行动了，不只是meta、英伟达、微软等。还有如《Nike为何天价收购虚拟潮牌RTFKT？》，这篇文章彭顺丰（大西洲创始人）的评论非常精到：“不是工具和渠道，而是【场】，乃至【主场】”，这和肖风所提，从“在线”到“在场”，异曲同工。BTW，前些天，彭顺丰老师在CC讲坛的演讲视频《让生命在数字化世界永生！》广为热传。

毋庸置疑，元宇宙还在其非常早期的阶段，如果理想状态为100%，可能现在只有1%，2%；但因此否定元宇宙的深远意义可能过于草率。我比较喜欢朱嘉明老师曾说过的一个比喻，原话记不得了，加上我的联想，描绘如下：元宇宙是一个正在建设的通天塔，外围是脚手架，脚手架卸下后，我们才能看清元宇宙的全貌。

笔者认为，在此之前，构成这个通天塔的几个重要支柱自身在不断的发展，并且和另外的一个或者几个支柱交叉融合，正在或即将形成沿途的美丽风景线！

比如，交互技术(AR/VR)或展示技术和AI的融合，区块链和文学艺术的结合，区块链和游戏的结合，例如：基于区块链以太坊之上的SANDBOX。

最后总结一下：

1、Web 1.0->Web 2.0->Web3.0 其实就是  

1）只读 ->读写 ->可拥有（包括可读可写）；

2）命令行 ->平面 ->多维立体

3）信息 -> 自媒体 -> 创意平台(去中心化)

4）信息关联 -> 信息与人的关联 -> 信息、人和物的全方位连接(包含人与人的关联, 人与物的关联, 物与物的关联)

2、元宇宙是Web 3.0的一个大的应用场景，也是5G、VR/AR、区块链、NFT、DeFi等的应用场景。
