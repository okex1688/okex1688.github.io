金色财经 区块链12月27日讯   从 JPEG 到体育文化再到加密生物，NFT 占据了今年的各大新闻头条，其火爆程度超过了加密领域里的任何一个类别。

2021 年，比特币价格比历史最高价翻了一倍多，以太坊生态系统快速发展，机构投资者纷纷投身加密货币领域，而狗狗币也莫名地火了一把，成为了主流潮词。但令人惊讶的是，似乎身边的每个人都在谈论“JPEG”。

去年的这个时候，NFT 还只是加密领域的一个细分市场，而如今却已成为了一种潮流。在 过去的一年时间里，“NFT”这个概念不仅渗透到了广泛的文化圈，还使数字资产所有权成为了广为人知的概念，最令人吃惊的是该行业的月交易额已飙升至数十亿美元。

实际上，许多人对于 NFT 的感觉是又爱又恨。因为他们不确定NFT到底是代表着所有权的未来，还只是一个灾难性的大骗局。

那么，到底什么是NFT（非同质化代币）呢？与较为流行的观点相反的是，NFT并不是一种JPEG。它既不是冷酷猿猴或是像素化的头像，也不是体育文化数字收藏品，更不是在电子游戏中代表用户战斗的卡通怪兽。所有这些都只是 NFT 代表的东西。

从本质上来说，NFT 本身是一种基于区块链的代币，它可以证明你是一个独一无二的数字项目的唯一持有者——无论这个项目是什么。通过智能合约，NFT可以被出售或是转让、艺术家们可以获得永久性版税、虚拟资产可以在游戏和元宇宙中被使用。

突然爆发： 2021NFT元年

2021 年是NFT走入大众视线的一年。尽管NFT的概念在 2014 年就已经出现了，但直到今年才开始大放异彩。仅今年一年，NFT领域就完成了超过 230 亿美元的交易量。随机生成的NFT藏品、主流名人在该领域的参与、以及边玩边赚游戏的兴起等都是 NFT 繁荣背后的主要驱动力。而最新的元宇宙热潮更是锦上添花。年初，体育圈名人效应一下子将 NFT 从一个细分市场推向了真正的主流市场。仅 Dapper Labs 上的 NBA Top Shot 在 2 月和 3 月的月交易量就超过了 2 亿美元，超过了整个 NFT 市场在 2020 年的总交易量。

今年的早些时候，名人和艺术家们也在 NFT 领域掀起了不小的波澜，从Grimes 到Rob Gronkowski，再到 Pak 和 3LAU 这类加密创作者。值得一提的是，数字艺术家 Beeple 在 3 月初的佳士得拍卖会上以 6,930 万美元的价格售出了一件 NFT，在目前的在世艺术家单项拍卖规模中排名第三，轰动了整个加密行业，进一步推动了 NFT 热潮。

不过从现在看来，这次拍卖似乎成为了 NFT 繁荣初期的天花板：狂热逐渐褪去，市场开始趋于饱和。很明显，Top Shot 的销量已经开始逐渐减少，越来越多的名人没有获得预期的回报。许多人开始怀疑 NFT 的繁荣是否只是昙花一现。 

总体来说，在 2021 年上半年，NFT 领域完成了创纪录的 25 亿美元的交易额，平均每季度的交易量都比较均衡，打破了该领域在以往取得的成绩。然而随着加密货币价格在7月开始一路高歌猛进，NFT 市场像是被注入了火箭燃料似的，在8月份一下子飙升到了惊人的新高度。NFT 市场在经历了几个月的发展后，社区、团队逐渐强大起来、效用逐步扩大，奠定了其“蓝筹”的地位，交易额更是达到了 107 亿美元。 

NFT 藏品的价值在 2021 年得到了高度体现。收藏品已不再是简简单单的 NFT 项目了，有些已经成为了独家品牌，而随着元宇宙变得越发完善成熟，这些品牌将变得与元宇宙越发息息相关。通过对历史交易量排名前 100 的 NFT 统计，在撰写本文时 ，NFT 市值约为 167 亿美元。

OpenSea 是目前业内较为领先的 NFT 平台，其 6 月份在以太坊上的交易量为 1.5 亿美元，到了 7 月份跃升至 3.29 亿美元，而8 月份则飙升至 34 亿美元。不难发现，这一次推动其激增的是更加原始、完全加密的原生项目，也充分展示了数字资产的创造力以及社区、功能效用的潜力。另一方面，名人效应在NFT市场发展过程中也起到了非常大的作用，一系列名人如SouljaBoy、Steve Aoki 和 Rob Gronkowski 在今年年初纷纷踏入 NFT 领域，在NFT最关键的发展期助了一把力。在接下来的几个月里，名媛Paris Hilton、饶舌巨星Eminem和NBA球星Stephen Curry也加入了 NFT 收藏队伍，此后关于 NFT 的头条新闻就开始频繁出现在主流媒体上。 随着Beeple 的艺术作品“Everydays: the First 5000 Days”以 6900 万美元的价格售出、NFT 在著名的佳士得拍卖行首次亮相，NFT的这种流行趋势变得越发明显。在接下来的几个月里，其他一些艺术 NFT 以及CryptoPunks 和 BAYC藏品的一部分在佳士得和苏富比都拍出了天价。

其它一些值得关注的NFT

1、The Bored Ape Yacht Club从 O.G. 手中接过接力棒。 

2、CryptoPunks将NFT 个人头像视作一个通行证，有了这个通行证你就可以畅通无阻地进入这个独一无二的特权俱乐部了，与此同时，NFT持有者还可以对这些图片进行商业化操作。 

3、Art Blocks 使用以太坊区块链铸造出算法生成的艺术品，就像 EulerBeats 的生成音乐NFT项目一样。

4、Axie Infinity 是目前最大的NFT项目，这个项目体现了 NFT 在视频游戏中的巨大潜力。数以百万的玩家购买了类似神奇宝贝一样的卡通怪物，并用它们进行战斗。甚至还有一部分人靠此来谋生。

5、另一个NFT项目——Loot，不同于我们常见的JPEG或GIF等形式存在，它只是一个TXT文本，这8行文本写的是8种随机生成的冒险者装备。 

如今，如雨后春笋般涌现的去中心化社区都在围绕着 NFT 展开游戏体验的开发。

随着元宇宙的逐渐形成，像The Sandbox 和 Decentraland 这样的沉浸式世界引领着数字潮流，它们可以定制、共享甚至货币化虚拟土地的所有权。假如Facebook和其他公司雄心勃勃的元宇宙计划最终能付诸实施，那么我们可能很快将会生活在其中。

NFT 市场反弹

令人无法忽视的是，NFT 市场正在蓬勃发展。根据 DappRadar 统计的数据显示，NFT市场在第三季度的交易量达到了 106.7 亿美元，而今年迄今为止已经达到了约 220 亿美元。 不仅如此，这个市场似乎正在变得越来越成熟，因为我们发现越来越多的NFT用例正在向实用性转变。而NFT 被认为是构建 Web3 的重要模块——换句话说，它也是构建元宇宙的关键。

一直以来，NFT 都在引领着人们走近加密货币和区块链技术。它们的形式多种多样，并且与人们喜爱的艺术作品和文化息息相关。不仅如此，NFT 还使加密领域变得不再那么神秘，而是更加平易近人。正如 Nansen 首席执行官 Alex Svanevik 说的那样：

“DeFi 将资本带进了加密领域，而 NFT 则将人们带入了加密领域。”

当然，也许有人会为一个 Twitter 头像而花费数百万美元，这使我们大多数人都倍感困惑。 但要知道，这只是个例，也是这个市场需要逐步解决的众多问题中的冰山一角。

不得不说，以太坊上还存在着一些漏洞，即使是在能源密集度较低的区块链平台（如 Solana）上开采到的收藏品和资产也经常会被卷入骗局，这种情况在整个加密行业依然很普遍。

而社交媒体对于 NFT则普遍持批评态度，想要涉足该领域的公司和项目也常常遭到强烈抵制。不过，这并没有阻挡住大品牌进军NFT市场的脚步，正如麦当劳和百威最近的一系列动作。品牌营销人员声称，Budverse（百威啤酒发布的第一个系列产品的NFTs）即将发布，而Tinderverse 和 White Castle 的 Sliderverse 也即将发布。

想必，即使是最初的那批对 NFT 持乐观态度的支持者也没有料到代币化数字收藏品在今年会如此火爆。然而，这种狂热会持续下去吗？

一些业内人士认为，大部分 NFT 将无法长期维持这样的价值。甚至著名的 NFT 收藏家和创作者 Gary Vaynerchuk 也警告说“NFT 冬季”即将到来，市场将会进行回调，他认为只有经过筛选的“蓝筹”项目才会蓬勃发展。

如果 NFT 真的是下一代互联网基石的话，那么它们最终可能会趋于平凡。然而，令怀疑论者大为失望的是，在不久的将来，NFT 很可能会无处不在，因为它会被用来代表我们数字生活中对事物的所有权。

最终NFT将会完全融入我们的生活，我们甚至可能都不会再称它们为NFT，也意识不到它们的特殊存在，一切都表现得那么顺其自然。想想看，你还会记得最后一次称自己的数字音乐收藏为MP3是什么时候吗？

2021 年，是人们被NFT深深吸引并为之疯狂的一年。2022 年的钟声即将敲响，市场还将继续为之狂热下去吗？就让我们拭目以待吧。
