“NFT 碎片：大势所趋还是流动性透支？”

前日，这张图片刷屏了加密社区：

23 日， path.eth@Cryptopathic 创建了 Feisty Doge 的 NFT 代币 NFD，这是以狗狗币头像为原型的碎片化 NFT 。

NFD 将原名为 Kabosu 的柴犬（狗狗币头像原型）一张照片所有权进行分割，总量为 1000 亿份，发行当日，NFD 最高涨至 0.0009 美元，近 24 小时交易量超 1.4 亿美元。

发行当日，这张图片一跃成为了估值最昂贵的 NFT。

倘若将该 NFT 的碎片可视化展现，那么投资人所持的一枚 NFD 大约是这样的：

随之而来还有该事件的杠杆效应。一些加密货币衍生品平台很快提供了针对 NFD 的借入、借出和做空的操作工具，投资人可通过 Uniswap V3 和 Sushiswap 建立空头头寸。

很快，加密社区 将目光 聚焦到 NFT 碎片化后暴增的流动性上。狗狗图片 NFD 成为了  NFT 「碎片化」久违的爆发契机。

但是， 市场对于 NFT 「碎片化」的观点两极分化，是福是祸？

NFT 版「拼单」现象已「燎原」？

NFT 碎片化是如何演变，并因这张狗狗照片突然走红的？

早期的 NFT 碎片化标志性案例就是《Everydays: The First 5000 Days》这幅画了。这幅画也是 NFD 打破记录之前的、最昂贵 NFT 的** 保持者。**

2020 年底，名为 Metapurse 的 NFT 基金收购了《Everydays: The First 5000 Days》中的 20 件作品，并在以太坊上发行了总量为 1000 万枚的 ERC20 代币 B20（Beeple 20 Collection），总共分两期发售。通过这些代币，该基金所持有的 NFT 资产被分割了。

B20 代币总量的 59% 归 Metapurse 所有，剩余的 41% 留给了艺术家、合作伙伴和投资者。在这样的分配下，B20 被指高度中心化，存在弊端。

但无论如何，该作品的所有权已经被「碎片化」重新分配了。

这样「部分所有权」代币为 NFT 的流动性问题提出了解决方案，允许用户将一个完整的 NFT 拆分成若干 ERC20 标准的碎片化代币，这些代币通过二级市场交易，降低了 NFT 的流通难度，也降低了投资者参与 NFT 等收藏品的投资门槛。

2021 年上半年，「碎片化」NFT 在慢慢渗透着加密社区。

NFT 热潮下，大量受欢迎的 NFT 藏品动辄几百几千 ETH，价格居高不下，提高了加密投资新人的参与门槛。

相当长的一段时间里，NFT 的碎片化一直被视为解决 NFT 流动性的最好方法，NFTX、NIFTEX、FractionalArt、Unicly 等业内平台，一直在为投资者「撕碎」 NFT 提供着容易上手的工具。

NFT 碎片化平台 NFTX 中，可以看到很多已经「被碎片化」的知名的 NFT 项目。其中，CryptoPUNK 占了很大的比重。

（图片来源：NFTX）

该平台上目前共有超过 123 个 CryptoPUNK。这些  CryptoPUNK 的持有者们都选择了将自己的头像碎片化处理，将每个 CryptoPUNK 转化为 ERC-20 代币「PUNK」。投资者可以以 PUNK 为统一计价单位来购买这些头像。

（图片来源：NFTX）

加密社区的同样也有着在 NFT 拆分上的积极探索，出现了「NFT DAOS」的新型社区，以社区为单位提供了小众范围内的 NFT 管理服务。

以较早期出现的 Jenny Metaverse DAO 为例，该社区由包括 NFT 收藏家、艺术家、创作者、项目、基金、有影响力的人组成，于 Unicly 平台上发行了 ERC20 代币 uJenny，代表着该组织所购买的所有 NFT 的部分所有权。

此外，uJenny 代币还被赋予了一些新的功能，比如在 Unicly 平台上参与挖矿。

在拆分 CryptoPunk 头像上，「圈子文化」似乎找到了合适的土壤。在海外社区应用平台 Discord 上，一些群主充当了牵头人的角色，通过 Dao 组织发起头像「众筹」。

这些组织的代表会发起某一个 CryptoPunk 众筹，竞拍成功后，参与者都会拿到与自己出资额相对应数量的新代币。凭借钱包里的「拆分代币」，即可直接在 Discord 中参与之后的 Dao 治理。

某种程度上，这里的 CryptoPunk 碎片更像一种专属社区的「通行证」。

（图片：社交平台上，一位投资者展示通过 DAO 来众筹购买 PUNK）

通过这些方式，投资者只需付出很少的费用，即可部分持有「NFT 中的兰博基尼」之称的头像。

有趣的是，在海外社区，一些 NFT 投资者还分享了自己用这种方式来「定投」 PUNK 的投资逻辑，并认为这对自己来说「力所能及」。

在海外社区，对于 NFD 的讨论将碎片化 NFT 的热度推至高点：极具 Meme 特征的柴犬图片及 NFD 出现，行业重新燃起了由 NFT 碎片带来的、对于「流动性」和「增量」的期待。

受 NFD 的影响，短短几日，NFT 投资者的注意力，快速转移至 CryptoPunk 碎片上。

NFD 的案例给花了大价钱购买一张图片的人提供了案例：将自己具有价值的 NFT 碎片化并出售，自己拥有了该 NFT 的基础上，甚至还有进一步获利的可能。

NFT 碎片化发展至今，主流思路在于：

将 NFT 本体碎⽚化： 如一幅名贵的画。

将 NFT 权益碎⽚化： 权益既可以是质押权益，也可以是工作权益，也可以是流动性分红权益。

将 NFT 与其他方式联动： 如流动性挖矿、NFTFi、GameFi 等等 能够看到，碎片化 NFT 还有很多空间等待着挖掘。

社区观点严重分歧

NFT 碎片化是大势所趋还是流动性透支？

NFT 碎片化的蹿红，让市场看到了 DeFi 时代 NFT 的更多可能性。

乐观的投资人从 NFD 的案例中看到，NFT 碎片化对资本流动性和资本效率做出了补充。 在二级市场，NFD 最近 24 小时交易量仍在 1 千万美金以上。

(数据来源：Coinmarketcap)

此外，NFT 抵押借贷是 NFT 赛道上长期以来的另一个难点，倘若 NFT 碎片能够持续获得更高的流动性，充当抵押品，那么抵押问题迎刃而解。

对于大部分投资者来讲，远超 6900 万的《Everydays: The First 5000 Days》难以企及，顶级 NFT 资产拆分代币或许代表了一种发展的趋势。碎片化的 NFT 与普通的代币一样，甚至更容易被理解和接受。在 NFT 碎片化降低投资者的参与金额门槛之后，存在着反哺 NFT 生态的可能性，这是一个充满想象力的发展方向。

不过，也有更为理性的投资者认为这是一种「套娃」策略，将投资风险转移给了更多个人或散户投资者。 这种观点认为，「NFT 即代币 」的模式太过魔幻，NFT 碎片化中，所有权的完整性、决策机制的有效性仍存在巨大问题，NFT 的价值泡沫或将再度放大。

另一方面，虽然最近碎片化 NFT 项目数量在增多，但从宣传、运营、推广等方式来看，与普通意义上的项目「发币」区别甚微，在将 FT 换为 NFT 之后，甚至发行者在无形中避开了许多风险，这些风险毫无疑问被转向了市场投资者。

这种担忧并非没有道理。行业主流观点认为，本轮加密市场回调，NFTFi、GameFi 等概念是主要推动因素之一。而比特币「5 万美金」的位置恰恰被视为整个市场的「临界值」，BTC 能否在 5 万美金位置突破无法预测，这种情况下，能够支持高额的 NFT 产品价值上涨的资金有限，碎片化 NFT 便很可能成为 NFT 巨鲸们转移风险、加速收益的一种捷径。

此外，也有分析从不同角度的观点讨论 NFT 碎片：倘若强调这些高价值 NFT 的奢侈品属性，那么 NFT 碎片可以类比于「二手奢侈品」 ，仍有收藏、交易和流通的价值，并无不妥。

能够看到的是，虽然此前也有 NFT 碎片项目不断出现，但 NFD 的狗狗 Meme 图片成为了最容易传播的方式，在短期内快速获取流动性。极具 Meme 属性色彩的 NFD ，从某种程度上进一步打开了 NFT 碎片化认知大门。

NFT 碎片化对于市场的或正面或负面的影响还在继续。有更多的人在为这些碎片「买单」，二级市场内上的数据还在不断更新。
