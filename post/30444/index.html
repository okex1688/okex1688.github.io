本文汇集适合Arweave初学者的基本工具和资源。这些工具和资源将有助于你了解Arweave生态系统的基本构建模块。

ArConnect

ArConnect是一个浏览器扩展，用作Arweave的钱包管理和应用程序登录。它可以让你查看和转移资产、管理账户余额并查看钱包的最新交易。它还允许你与许多Arweave应用程序（或 dApp）进行交互。 

由于交易签名和加密是通过扩展程序而不是应用程序在后台进行的，因此你的密钥文件不会被窃取。这是比上传密钥文件更安全的解决方案，因为交易发生在客户端本地。 

它们不对扩展程序内的AR或PST传输收取费用，但在与dApp交互时会收取少量费用。当第三方应用程序使用ArConnect时，收取费用的一小部分会给随机选择的VRT代币持有者。

你可以从他们的网站arconnect.io下载浏览器扩展。

ArDrive

ArDrive是一个去中心化、社区拥有的数据存储平台，提供永久性存储。它提供NFT、个人照片和视频、文档甚至是网页档案的存储。ArDrive提供安全和公开的多文件上传，以及抗审查的文件共享。人们使用ArDrive共享文章、书籍、录音、照片和代码。

ArDrive提供“一次性付费”（pay-per-life）的付款计划，而不是基于订阅，这意味着你无需经常支付费用来存储和访问你的数据。它还确保无论公司的条款和服务如何例如你的帐户长时间处于非活动状态，所有数据都会留给你掌控。

由于ArDrive允许你将数据存储在永存网络（permaweb）上，因此你的文件无论如何都会被存储，即使 ArDrive停业，你也可以完全访问和控制它们。 

ArDrive注册过程：https://ardrive.io/start/

Arweave Dev Discord

为了支持永存网络（permaweb）开发，值得加入Arweave开发社区。它拥有Arweave团队和生态系统的核心开发人员。它是开发人员提供和获得有关开发问题的帮助的有用服务器。还有应用程序创意、SmartWeave帮助、网关、testweave等渠道。

这里加入Discord：https://discord.gg/BXk8tq7。

Arweavers Telegram社区

另一个参与社区的工具是Telegram，因此你可以加入Arweaver Telegram 社区。这是一个技术含量较低的聊天社区，更多地关注Arweave协议的价格和一般应用。社区今年甚至在里斯本举行了一次聚会。

除了投机聊天，你还可以询问有关如何购买AR、如何上传数据、矿工所在位置等类似信息的信息。

这里加入：http://t.me/arweavers。

Arweave网络扩展

Arweave Web Extension既可以用作钱包，也可以用作Web存档器。使用Arweave Web Extension，你可以存档网页和在线pdf、视频以及任何可能被404的内容。 

它还允许你直接在浏览器中管理自己的AR代币钱包，导出钱包历史记录并将其下载为CSV文件。它具有多钱包支持，你可以轻松切换。可以发送和接收AR代币并生成新的AR钱包密钥。

这里非别提供了存档和使用钱包的指南：https://docs.arweave.org/info/archiving/archiving-web-pages

https://docs.arweave.org/info/wallets/arweave-web-extension-wallet 

arweave-js

用于与Arweave交互的JavaScript库——arweave-js——是在永存网（permaweb） 上构建自定义应用程序的主要方式。由于同时支持客户端JS 和Node服务器，arweave-js是JavaScript 开发人员的宝库。

该库可以轻松生成新钱包、AR转账、签名并将数据发布到网络等。

项目的GitHub地址：https://github.com/ArweaveTeam/arweave-js

everPay

everPay是Arweave和以太坊的实时支付协议。它的目标是为每个人提供一个可信的、去中心化的支付应用程序，并为开发人员提供一个SDK，使构建DEX变得简单。

由于everPay交易没有打包，交易会在几分钟内上传到链上。everPay协议目前支持Ethereum 和Arweave，即将支持Arweave利润分享代币 ( profit-sharing tokens，PST)。

everPay地址：https://app.everpay.io/

Finnie

Finnie是Koii钱包浏览器扩展。你可以使用它来创建NFT并将它们永久存储（例如可以使用 Finnie来获取不断发展的atomic Zombies）。还可以使用它来存储你的NFT并从参与中获取KOII。

下载Finnie浏览器扩展程序后，你可以制作一个新钱包或添加你已有的钱包，然后可以获得少量或 AR和KOII代币开始。

Finnie地址：https://chrome.google.com/webstore/detail/finnie/cjmkndjhnagcfbpiemnkdpomccnjblmj

Verto

Verto是Arweave利润分享代币 ( profit-sharing tokens，PST)和NFT的通用交易所，可以将AR交换为生态系统的各种PST中的任何一个，这些PST代表ArDrive、denefit.land、PermaBot 和Verto本身等产品。Verto Space还充当所有Verto铸造的NFT的画廊，包括articulate.eth的 Bark Blocks系列。

Verto地址：http://verto.exchange/

ViewBlock

每个区块链都需要一种让公众监控交易、地址、区块和网络统计数据的方式。ViewBlock为是Arweave的主要区块浏览器。监控交易是否已被挖入区块，或者SmartWeave合约交互是否失败，ViewBlock非常方便。还可以获取有关blockweave总大小、每日交易数量、节点位置等的统计信息。

ViewBlock地址：http://viewblock.io/

Wrapped AR

Wrapped AR或wAR是由everFinance创建的ERC20代币。每个wAR代币与AR代币1:1锚定，它们是Arweave代币持有者与以太坊生态系统互动的一种方式。

如果你持有AR，你可以把它换成wAR，你也可以把wAR换成AR。有了这个，everFinance创造了一种以去中心化方式获取AR代币的方法。
