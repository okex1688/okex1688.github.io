NFT在2021年得到了迅速发展，成为这一年最大热点之一。Cryptopunks、无聊猿等主流项目彻底爆发，实现了亿万美元交易量。NFT接下来要怎么发展？投资机构Color Capital合伙人Chris Cantino今日在推特上分享了NFT的下一步。

我们都听说过 NFT讲述的故事。

  1. 艺术家享受无限版税

  2. 社区拥有的所有权

  3. 24/7 交易市场

  4. 以代币作为门槛的解锁方式

这些特点都很好，但 NFT 下一步要做的才是你要记住的。

散户

从 7-11 到 Gamestop 的散户将使用 NFT 来激励 IRL 交易，利用创作者为商店带来流量。

例如，只有从 Gap 商店购买才能解锁的 Yeezy NFT，或解锁特斯拉汽车专属功能的 NFT。

服务

NFT 将解锁对服务和爱好者社区的使用权限。与有影响力的厨师、摄影师、医生和小众热情社区的问答和教程将会蓬勃发展。

这也将扩展到 IRL 服务，如交通、酒店和按摩。

激进主义

随着数百万人流入社区钱包，NFT 所有权支持政治事业的社区将获得关注。

想象一个气候变化 NFT收藏品，其中的销售针对游说工作，而这些工作由社区 DAO 协调。

社交feeds市场

将出现基于 NFT 收藏的社交feeds，提供对顶级收藏家策略的洞察，以及收藏家互动的平台。

这些feeds将演变为社交商务市场，具有评论、分析等功能。

多人游戏

NFT 将通过激励群体行为来创造蜂群活动。一旦达到 1 万用户就解锁关卡的 MMORPG。向已签署合约以合并其基础层 NFT 的用户空投稀有 NFT。

收藏将成为一项团队运动。

NFT借贷平台

由于 NFT 可以解锁临时访问权限——比如参加会议——短期借贷市场正在兴起。例如，我亲自向朋友推荐 $FWB，以便他们可以参加活动。

把它想象成一个市场：StubHub，但适用于 NFT。

NFT 指数

随着市场推动者利用收藏家获取资金，通过指数接触数百个 NFT 的机会将会激增。

想象一下佳士得的 NFT ETF。它们具有使资产流动性的影响力——投资者将效仿这一点。

它还将扩大对 NFT 社区的支持。

抵押品

迄今为止，贷方一直反对 NFT 持有者以其资产借款。

但随着机构资金流入，资产成为更好的价值储存手段，抵押品市场将为所有 NFT 持有者提供灵活性和流动性，而不仅仅是鲸鱼。

碎片化

将 NFT 拆分为个体份额，可以让更多人接触到这些蓝筹资产，例如拥有一个Cryptopunk作品的一部分。

结果？随着资金进入市场，持有者的流动性增加，蓝筹资产的升值幅度更大。

DeFi

随着狂热者变得更愿意抵押和耕种他们的资产以获得收益，NFT 承诺提供除了 APY%之外的激励。

NFT 将成为基础 DeFi 资产，包括社区访问。这些不会是简单的储蓄游戏——它们也将是对社区的投资。

忠诚度交易所

随着品牌和创作者寻求增强忠诚度，社区行为将获得 NFT 奖励。成员是否贡献了内容？奖励NFT。完成调查？奖励NFT。进行购买？奖励NFT。

代币比传统的打折码更具流动性和更值得夸耀。

研发

品牌和创作者将交换 NFT 以获取对产品开发的见解：痛点、营销主张、路线图等。

NFT 将解锁对产品发布和潜在利润分享的早期访问。品牌将跟踪和维护这些关键关系。

内容提交

用户将提交短视频、评论和教程等内容以换取 NFT。这是一个营销飞轮。

权利可以编入合约，以便在广告中使用贡献者的内容可以为他们带来未来的利润。

客户群组 NFT

想象一下，因为成为 Air Jordan 的首批客户之一而收到 NFT，这在今天是多么有价值。多年后，耐克将如何通过特殊渠道和产品奖励您。

群组 NFT 证明您在特定时间采取了特定行动。

教育和客户支持

在展示对品牌产品的非凡了解后，用户可以获得 NFT，以换取将新手引入社区或提供支持。

这可能比品牌员工做同样的事情更有影响力。

赏金

创作者和品牌将在社区内下放独特的任务，完成任务则奖励 NFT。这可以是任何事情，从完成问卷调查到推进新成员，再到参加活动。

赏金可以是高度竞争的，也可以是开放的。

排行榜

NFT 社区将通过突出顶级持有者的表现和参与来游戏化。

顶级贡献者将获得奖励和影响力，从而激励社区持有和参与，以确保他们的投资获得更好的回报。

我们还处于NFT的早期阶段，未来那么光明。NFT 技术对社区参与民主化和奖励的承诺每天都在增长，前景是非常看涨的。
