6 月 9 日，Gitcoin 官方宣布，现已开启第 14 轮捐赠活动（GR14），共有超 300 万美元匹配资金。

具体分配如下：

GR14-主奖金池：100万美元；气候解决方案：50万美元；支持web3 DEI项目：15万美元；倡导良好加密政策的项目：35万美元；ETH基础设施：50万美元；推进 ZK 工具和社区的项目：14.5万枚美元；Loot生态：13万美元；Polygon生态：10万美元；Aurora生态：10万美元；Web3游戏：10万美元；Web3社交：5万美元；Celo生态：5万美元；Protocol Labs：5万美元；Subgraph生态：5万美元；ENS生态：4.2万美元；Unlock Protocol：2.5万美元；Radicle：2万美元；奥克兰当地：1.5万美元。

相较于上一轮，本轮捐赠中特别增加了细分化的web3生态，包括DEI（多样性、公平性和包容性）、游戏、社交；另外，Loot生态以及Aurora生态也占有一定侧重。

Gitcoin发文强调称，Web3 不仅有机会重建我们世界的金融和技术基础设施，还可以根据首要原则重新创建一个包容、有代表性的工作环境。考虑到这一点，DEI旨在展示和授权致力于确保来自不同背景的个人有平等机会参与 Web3 的倡议和组织 ——具有性别、种族、地理位置和其他方面的多样性身份。

本文中我们除了挑选不同类别的有趣项目，虽然未必有投机价值，但却对用户真正具有使用价值的产品。

JediSwap

官网：https://app.testnet.jediswap.xyz/

JediSwap，是StarkNet 上的第一个完全可组合且无需许可的 AMM，可实现资产的即时无gas费交易。

zkRollups.xyz

官网：https://www.zkrollups.xyz/

zkRollups.xyz聚合所有构建 zkRollups、基于 zkRollups 以及为 zkRollups 构建的项目。通过直观的目录使这些项目易于查找。Jason Garland 和 Joel Kite 是 Numio 的联合创始人，自 2020 年以来一直在构建 zkRollup 技术。目前，页面已集成了16个项目，涵盖NFT、钱包、基础设施、DApp、游戏等项目。

Lenster

官网：https://lenster.xyz/

Lenster 是使用 Lens 协议构建的去中心化且无需许可的社交媒体应用。Lenster 基于Polygon 网络，用户可以以零Gas费与 DApp 无缝交互。用户可以创建、评论、分享帖子，可以创建众筹、社区、发行/收藏NFT等。

接下来，Lenster将启动移动应用程序的开发。 2022年第四季度启动 Lenster DAO 并使用 Lens follow NFT 作为治理代币。

Connext Network

官网：https://www.connext.network/

Connext 位于以太坊区块链和 HTTP 等现有网络协议之上。该协议使用网络状态通道系统，这种技术允许对等方彼此做出牢不可破的承诺，如果发生争议，所有这些都可以在以太坊区块链上进行裁决和解决。这使系统用户可以使用任何货币进行近乎免费、即时、可编程的转账，而无需依赖第三方来存储资金。Connext 的技术可用于互连 EVM 兼容的Layer2解决方案，例如跨 Connext 节点的汇总链。

Connext还与 ExchangeUnion 合作发布了Layer3解决方案，用于在 Connext 的 ETH 主网和 BTC的闪电网络之间进行交换。

Defillama

DefiLlama 是著名的 DeFi TVL 数据仪表板，目前以其支持市场上绝大多数主流公链与生态应用数据而著称，相信绝大多数加密圈的用户都使用过其产品。

Umbra  

官网：https://app.umbra.cash/

Umbra 是一种在 EVM 区块链网络上进行隐形支付的协议。它支持隐私保护交易，其中接收者的身份只有发送者和接收者知道 。Umbra 基于普通的椭圆曲线密码学。它适用于两个实体之间的支付，并带有一组不同的隐私权衡。Umbra 并没有像 Tornado 那样破坏发送和接收地址之间的链接，而是使该链接变得毫无意义。每个人都可以看到谁发送了资金，每个人都可以看到资金发送到的地址，但是接收地址从未在链上看到过，因此任何外部观察者都不可能知道谁在控制它。

Tally Ho

官网：https://tally.cash/

Tally Ho，是一个社区拥有和运营的 Web3钱包，将在不久的将来推出 DAO，以全面实现社区拥有的梦想。目前，Tally Ho已通过Least Authority 审计。

PoolTogether

官网：https://pooltogether.com/zh

PoolTogether 是一个以Premium Bonds为基础的加密货币驱动的储蓄协议。在储蓄的同时，还能获得每周的中奖机会。当向 PoolTogether 存入一笔存款时，该存款会自动路由到其他去中心化金融协议（如 Aave）以开始赚取利息。PoolTogether 是一个非托管协议。这意味着没有人有能力控制存入的资金。所有存款和取款均由构成 PoolTogether 协议的智能合约自动进行。POOL 是 PoolTogether 协议的治理代币。对协议的控制权完全掌握在持有 POOL 代币的社区手中。POOL 持有者治理的主要职责是通过适当管理协议参数来确保协议的增长。

Via Protocol  

官网：https://via.exchange/

Via Protocol是一种跨链流动性聚合协议，旨在创建跨不同区块链的快速、高效和无缝交换体验。Via 协议 可以被认为是加密货币的Visa和万事达卡，它是一个管理跨各种网络和代币的转账和支付的单一基础设施，克服了与多个 DEX 和桥交互的障碍。

Via Protocol 打算通过两种产品来克服这些限制：作为独立解决方案的跨链聚合协议，以及可以轻松集成到市场和游戏等去中心化应用程序中的跨链结账系统。

SheFi

官网：https://www.shefi.org/

SheFi 将教育、实验和社区结合在一起，在以区块链技术为基础的新金融和技术范式中赋予不同背景和经验的女性权力。通过基于队列的学习，女性将探索并获得使用顶级区块链协议和应用程序的经验。每门课程包含 1 - 2 节深度教育课程，其中包含演示以及共享录音、材料和详细的操作指南。

Zk Block  

官网：https://zkblock.app/

zkblock.app 是零知识证明、智能合约、以太坊（和 L2）、web3 应用程序和密码学的一站式工具目录。它最初是用于 snarks 的样板代码，目前正在添加更多功能以帮助社区并使工具信息更易于访问。

Punk Domains

官网：https://punk.domains/#/

Punk Domains 是 ENS 的 L2 替代品。它的不同之处在于对多个 TLD（顶级域，如 .web3、.wagmi、.ape）的支持，这些域可以由不同的实体（如 DAO、项目或其他社区）拥有。例如，.agave TLD 可以归 Agave DAO 所有，出售 .agave 域名的收入将直接进入 Agave 国库。此外，社区成员可以在社交网络上使用域名作为用户名，建立更强的团结感和归属感。

DeFrag

官网：https://defrag.fi/

DeFrag 是一种去中心化的流动性市场协议，借款人可以使用 NFT 资产作为抵押品来获得即时贷款。该协议使用称为看跌期权的金融工具自动为用户的 NFT 资产提供保险。承销商提供流动性以通过出售看跌期权赚取溢价费用，而无需与同行或交易对手协商行使价或到期等条款。借款人抵押 NFT 资产以申请贷款，协议自动计算从承销商处购买所需的看跌期权。DeFrag 将分阶段推出，随着协议的成熟逐渐增加功能。Metamaticians 是基于著名数学家随机生成的 NFT。通过拥有 Metamatician，您将拥有第一个承保池的股份以赚取保费。Metamatician NFT 作为进入 NFT 抵押贷款 DeFrag 领域的入场券。在铸币过程中，会生成一个随机的 8 位数字和一个与金融工具定价风险相关的希腊符号，并分配给每个 Metamatician。

所有铸币收益都会自动存入 DeFrag DAO 国库。一旦社区投票决定将初始 NFT 项目用作抵押品，ETH 将从财政部转移到初始承销池中。当借款人开始贷款时，协议将自动发行所需数量的 NFT 看跌期权并锁定所需的 ETH 以承销它们。

目前，已经有超过2500个项目申请了捐助。整个捐赠过程将持续到6月23日。
