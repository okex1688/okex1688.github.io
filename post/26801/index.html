Kusama 生态正在酝酿一场艺术和 NFT 的热潮。

NFT 是 “Non-Fungible Token 非同质化代币” 的简称，指在某些区块链上持有的，易于交换且具有个体唯一性的数字资产。

今年，NFT 频频出圈，从科技界名人，到球星、说唱歌手，都在争相发行 NFT，蚂蚁等一些大厂也开始试水 NFT。根据 Coingecko 数据，目前 NFT 市值已经达到 187 亿，已经成为加密世界不可忽视的一股力量。以 “Expect Chaos” 作为信条的 Kusama 当然也没有理由错过这场热潮。

波卡/Kusama 创始人 Gavin Wood 在一次采访中称 “Kusama 正在大力支持艺术界。Kusama 资助了一些艺术项目，来试图让艺术界对区块链有一些了解。”

今年 3 月，Kusama 国库资助的 “虚拟生境 —— 镜中迷因可曾见” 展览在北京尤伦斯当代艺术中心 UCCA Lab 展出，这被称为全球首个加密艺术展。

在 Kusama 生态中，我们已经看到了不少有趣的 NFT 实践，或许有潜力在 NFT 领域掀起一番创新的浪潮。在这篇文章中我们总结了一部分，希望帮助大家注意到这个正在蓬勃发展的领域，并参与进来。

NFT 协议

NFT 协议至关重要，它让 NFT 能够标准化，从而让创建 NFT 变得更加简单，也促进了 NFT 在不同的 DAPP 和区块链之间的流转。

在了解波卡生态的 NFT 协议前，我们可以先复习一下目前比较常用的一些 NFT 协议。

目前比较流行的以太坊 NFT 协议有以下两种：

  • ERC-721 是最早的以太坊 NFT 标准，每一种代币都需要单独的智能合约。CryptoKitties 和 Decentraland 等都采用了 ERC 721 标准。

  • ERC-1155 是一种 “多种代币标准” ，让一个智能合约能处理多种类型的代币，其中可以包括同质化代币、非同质化代币、半同质化代币等。

RMRK 标准

RMRK 是 Kusama 上的第一个 NFT 标准，可以用它来在 Kusama 上铸造和交易 NFT。很多大家很眼熟的 Kusama 艺术品就是使用 RMRK 标准铸造的。

RMRK 由 Bruno Skvorc 创造，他曾经是 Web3 的基金会技术教育者，也是波卡生态中很积极的 NFT 布道者。

和其他一些 NFT 协议相比，RMRK 最大的特点就是，使用它生成的 NFT 在铸造之后还可以对外界的互动做出反应，从而不断进化。

RMRK 1.0 目前已经实现了，具备以下功能：

  • 最核心的就是互动相关的 EMOTE 功能，它让 NFT 可以在链上接受表情图标。EMOTE 功能是一种价格发现机制，有点类似于点赞的效果，也是一种和 NFT 互动的机制。

  • 基本功能有发送、铸造和销毁（SEND、MINT、CONSUME）。

  • 交易相关的功能有挂出和购买。LIST 功能，让用户可以把 NFT 挂到链上去销售，销售 NFT 不需要通过某种 UI 或者拍卖平台就能做，没有中间商赚差价。BUY 功能，可以买入挂出的物品，并马上把 KSM 转给卖家。

RMRK 2.0 正在开发中，比较有亮点的有以下功能：

  • 嵌套 NFT： 让 NFT 可以拥有其他的 NFT，而且拥有之后还可以让原 NFT 获得不同的渲染效果，比如一个游戏角色 NFT 可以拥有各种装备、服装的 NFT。这个功能还可以把一堆 NFT 打包成一个集合来出售。

  • 条件化渲染： 让特定的 NFT 可以根据特定条件显示出不同的视觉效果。NFT 可以对一些链上的值做出反应，改变其外观。

  • 多资源 NFT： NFT 可以由多种资源组成。比如一个游戏 NFT，其中可以包含一段 JS 代码来在浏览器中显示，一张高清艺术图片作为封面，一份 PDF 手册来解释如何玩游戏。在不同的平台中会展示不同的资源。

Bruno 将 RMRK 描述为 “艺术乐高” 。用 Bruno 自己的话说，RMRK 是 “世界上最先进的 NFT 系统，甚至不需要智能合约就能运行。它创造了一个超越 ERC721 和 ERC1155 的世界，在这里你能做的不只是收集一些数字资产，躺在你的钱包里吃灰等死，或者等待下一波炒作周期。”

NFT Pallet（Unique Network）

NFT Pallet  是 Substrate 中的一个 NFT 模块，由 Unique Network 开发。它的基本功能包括创建 NFT 集合、铸造代币并管理其所有权等。除了 NFT Pallet 之外，Unique 链还包含了智能合约模块，开发者可以实现一些自定义的逻辑。Unique Network 计划成为 Kusama 和波卡的 NFT 平行链。

在 Unique 中创造的 NFT 具有一些独特的特点。

更加灵活的经济模型

我想任何一个游戏玩家，刚开始玩区块链游戏时，应该都会深受折磨。每进行一次操作都要付一次 gas fee，确认一次交易，这不仅费钱，也相当破坏游戏体验。

相比之下，Unique 提供了多种 gas 收费模式，开发者可以自行选择。

比如你开发了一款 NFT 游戏，你可以选择收费模式为 “用户模式”，让用户自己付费，就像目前的大多数 NFT 游戏一样。

你也可以选择 “赞助模式”，先帮你的用户付费，勾引他们来玩你的游戏，等他们上手之后，再引导用户心甘情愿地氪金，就像我们现在玩的各种网游、手游一样。

“重新同质化” 模式

我们都知道 NFT 的特征就是独一无二、不可分割，但是在现实中，确实存在 “共同拥有某个物品”  的需求，比如说对某件艺术品的共同所有权等。

Unique 提供了一种特殊的集合模式：重新同质化（Re-fungible），简单地说，就是把本身唯一的 NFT 进行所有权的分割，让它重新变得同质化，可以部分转移给多个所有者。

NFT 基础设施

PolkadotAssetJS

PolkadotAssetJS 主要包含一个 Web 应用，可以直接显示 Kusama 钱包中的艺术品和收藏品。该应用可以显示 NFT 余额、NFT 交易记录。

除此之外还包含一个开源的 Typescript 库，让开发者可以展示其 NFT 和余额。

Unique NFT 交易市场

Unique 建立了一个 NFT 交易市场 ，可以在里面浏览、出售、购买 NFT。

网址：https://unqnft.io/#/market

Chiba Gallery 加密画廊

Chiba Gallery 是 Kusama 上的加密画廊，你可以在其中浏览 NFT 作品。目前我看到画廊中暂时还没有展出任何作品。

Chiba Gallery 主要由 Bit.Country 团队和 Unique 团队开发。

NFT 应用

Bit.Country

Bit.Country 是基于 Substrate 的元宇宙（Metaverse）项目，计划竞拍 Kusama 和波卡平行链。

Bit.Country 让每个人都可以创建自己的元宇宙，元宇宙是虚拟空间中的另一个世界，用户可以在其中购买土地、建造建筑、参加活动、工作赚钱等。

你可以建立自己的 Bit Country 虚拟社区，为自己的社区成员创造一个世界。你在元宇宙中的资产，比如土地、服饰等都是 NFT，还可以在虚拟画廊中欣赏 NFT 艺术品等。目前 Bit.Country 测试网还未上线，你可以在这篇 PolkaWorld 专访中了解 Bit.Country 的更多信息。

Kanaria

Kanaria（https://kanaria.rmrk.app/）是一款基于 Kusama，使用 RMRK 标准开发的 NFT 游戏，由 RMRK 团队开发。同时它也作为 RMRK 的第一个用例，展示了各种独特的功能。

Kanaria 的玩法是出售固定数量的 Kanaria 鸟蛋 NFT，这些蛋可以孵出不同的 Kanaria 鸟，在规定时间内未出售的蛋会被销毁。

用户可以对这些 NFT 发送表情符号（Emote），来让其表现出不同的外观。之前 Kanaria 做了个活动，收到 Emote 最多的 NFT 会有奖励，期间交易量大增，甚至把 Kusama 网络挤爆了两次。

Kanaria 鸟 NFT 还可以装备别的 NFT，这些 NFT 装备可能来自用户在其他合作链上的行为，比如铸造 vksm 参与 Bifrost 的 Crowdloan 就能获得 Kanaria 的稀有装备。

SubstraPunk

SubstraPunk 由 Unique 团队开发的基于 Substrate 的虚拟头像 NFT 收藏品项目，致敬了著名的 NFT 项目 CryptoPunks。

这个项目获得了 Hackusama 开发大赛区块链赛道的冠军。目前在 Unique 的 NFT 交易市场上，最贵的 Punk 挂了 149.6 KSM 的价格，按照目前的 KSM 价格来算接近 3 万美金。

部分 SubstraPunk 在 https://unqnft.io/#/market 上的价格

ComingID NFT

ComingChat 是基于 Substrate 的加密通讯工具，由 Coming 基金会开发。每一个用户在注册 ComingChat 时都会得到一个 ComingID（简称 CID），每个 CID 都是一个 NFT。

CID NFT 有不同的 ID 编号、志号、水印和多种卡面风格，来保证其稀缺性。CID NFT 可编程，可以实现各种锻造、制作和设计。

CID NFT

在流通方面，CID NFT 可以直接在 ComingChat 应用中转赠，可以直接进行交易，未来计划上线一些主流 NFT 交易平台。

进化星球

进化星球是一款虚拟经营类区块链游戏，由达尔文团队开发。

在游戏中，用户可以买卖土地 NFT，购买 “使徒” 和 “钻头” NFT 来挖掘土地中蕴藏的元素资源，搭建建筑、制造道具等。游戏中有内置的各类 NFT 市场可以交易，玩家还可以在游戏中的 DeFi 场所 “古灵阁” 进行 Swap、流动性挖矿、Farming 等。

值得注意的一点是，进化星球是多链部署的。进化星球游戏中有不同的大陆，目前已开放的大陆有 3 个，分别部署在以太坊、波场、Heco 上。还有一个大陆将会部署在 Crab 上，Crab 基于 Substrate 开发，是达尔文网络的金丝雀网络，计划竞拍 Kusama 平行链。

达尔文团队还在使用 Substrate 开发一系列跨链桥，来将不同的大陆打通，让用户可以在不同大陆之间交互。

总结

目前 Kusama 的 NFT 生态还处于较早期的阶段，一些基础设施还不是特别完善，应用也不是很多，但还是能看得出不少创新的点。期待经过一段时间的发展，Kusama 未来能成为区块链艺术和 NFT 发展的大本营。
