用 NFT 价值链来理顺该领域投资逻辑。

撰文 : Jasmine Zhang 与 Fiona He，分别为 A&T Capital 合伙人与投资经理。

6 月下旬，我们发布了一篇在 NFT 赛道领域的研究并阐述了我们重点关注的 NFT 领域的投资机会。时隔至今，NFT 方兴未艾，我们看到更多的行业龙头在涉猎 NFT。我们希望通过如下这篇文章，分享我们对于NFT 产业价值链的方法论。

6 月 21 日，支付宝 在「蚂蚁链粉丝粒」小程序上限量发售了两款「敦煌飞天，九色鹿」付款码 NFT 皮肤，上线即被迅速抢光。

6 月 24 日，以新闻聚合和评论为特点的美国社交网络、号称「互联网头版」的 Reddit 发行 NFT CryptoSnoo 进行为期七天的拍卖。用户可以购买 CryptoSnoo NFT 并将其链接到 Reddit 帐户后作为 Reddit 头像，并且一张可收藏的 NFT 卡会出现在个人资料中。

6 月 25 日，漫威 宣布与 VeVe Digital Collectibles APP 合作推出官方 NFT 数字产品、3D 塑像和数字漫画书。 

7 月 1 日，美国有线电视新闻网 CNN 推出两则可收藏的新闻历史时刻 NFT，分别为「1980 年 6 月 1 日，世界上第一个 24 小时电视新闻网络 CNN 的诞生」和「1991 年 1 月，CNN 记者在伊拉克战争爆发的独家报道」。

7 月 6 日，区块链游戏和 NFT 开发商 Animoca Brands 宣布获得马云，蔡崇信的家族管理基金 Blue Pool Capital 的一笔投资。

近一个月来，虽然大盘仍低迷，但「圈」外继续保持着对于 NFT 的探索。究其原因，当然是 NFT 的价值逐渐被认可。我们认为，NFT 的创新在于其提供了一种标记 原生数字资产所有权 （即存在于数字世界，或发源于数字世界的资产）的方法，且该所有权可以存在于中心化服务之外。

而究 NFT 之价值，则在于它拓展了上链资产的类别，使原先无法表示的资产提供了可能。与此同时，被拓展的资产类别与大众群体息息相关且更易传播，这导致 NFT 背后的逻辑与商业模式更容易被理解和接受。

我们认为 NFT 赛道还拥有巨大的潜力，NFT 的版图才刚刚开始拼凑，因此我们按照 NFT 的铸造到流通梳理了 NFT 产业价值链。一方面，我们将利用这条链来帮助理顺投资逻辑；另一方面，也欢迎同行或项目方与我们交流心得。

下图 NFT 产业价值链分层逻辑如下 : 整体按照 NFT 流动分成三层，由下至上依次为基础设施层 、协议层 以及应用层 。

1. 基础设施层 ，主要涵盖底层公链、侧链/Layer2、开发工具、代币标准、存储以及钱包。这一层捕获价值基于 NFT 的铸造，比如底层公链和侧链/Layer 2捕获了铸造产生的 Gas Fee。NFT 数量越多，这一层捕获价值越大。

2. 协议层 ，包括：

  • NFT 铸造协议（Mint & Launch）兼一级流通市场（Sell & Trade） ，大部分协议都兼顾上述两者以便进一步捕获价值。捕获价值基于 NFT 的一级交易，因为二级交易更多的是看平台流量，而目前 OpenSea 是最大流量的二级平台，因此这一层的铸造协议只能靠一级交易捕获价值，这也是为什么 Mintable 会引入 Gasless item 的概念，吸引用户在同一平台上铸造并进行后续交易，只有交易后 NFT 才上链从而捕获一级交易价值。

  • 流动性协议 ，主要用于对 NFT 进行价值发现。这一类协议并不铸造 NFT，而是通过各种方式对 NFT 进行定价，价值捕获未来很可能类似 DEX 和预言机，基于提供流动性基础金融服务。

  • DeFi+NFT ，主要是 DeFi 活动铸造的 NFT，与其他铸造协议不同，主要通过铸造活动捕获价值。

  1. 应用层 ，主要由基于协议层而产生的代币而衍生出来的应用为主，比如泛二级市场代表项目 OpenSea 中交易的 NFT，主要由协议层的各铸造协议平台构成。这一层级的价值捕获主要基于流量和需求变现，比如社交和策展，如何聚集流量以及变现是核心的价值点。

下面我们将逐一介绍我们重点关注的协议层以及应用层的项目。

协议层 - NFT 铸造协议及一级市场 – 「综合」

这一类代表可交易多种类实物的综合性交易市场 。NFT 走向主流与 NFT 铸造平台兼一级市场息息相关，因为平台直接连接撮合创作者与用户。一方面，平台需要进一步降低用户的接受门槛，提升用户使用感，增加用户粘性与参与度，可参考闲鱼类 APP；另一方面，平台需要为创作者提供容易上手的 NFT 创建体验，将创作 NFT 的门槛降到足够低，吸引更多样的创作者进行展示。

  • Mintable （https://mintable.app/）：为用户提供包括私信、推广、打分等功能的交易平台，并引入 ERC-721 作为投票治理代币，引入 gasless item 吸引用户铸造后留在平台交易。

  • Mintbase （https://www.mintbase.io/zh）：基于 NEAR 生态的交易市场，用户可以创造和销售音乐、艺术品、门票、摄影等NFT。

  • VIV3 （https://viv3.com/）：首个基于 Flow 的交易市场。

  • Zora （https://zora.co/）：由 Coinbase 前产品经理组建，主要产品是一个为创作者或艺术家提供发行作品的平台，产品形式以媒体为主包括视频，音频，文件等。

  • Cargo （https://cargo.build/）：允许用户在平台上铸造、买卖 NFT，支持 Ethereum，xDai 以及 Polygon

  • TerraVirtua （https://terravirtua.io/）：艺术、收藏品、游戏 IP 等铸造交易平台。操作环境具有高度兼容性，使用户能够以沉浸式的方式展示、交易和与 NFT 互动；同时，其基于 3D Fancave、Terradome 和 Galleries 的虚拟界面以及移动 AR 应用程序，用户体验好，IP 资源强。

协议层 - NFT 铸造协议及一级市场 – 「艺术 / 收藏」

这一类代表艺术 / 收藏类为主的铸造以及一级市场 。艺术类项目看重强 IP 资源以及艺术品味，而收藏类项目看重未来的埋线与彩蛋。其中，艺术类项目大多有较高门槛，需要创造者带有一定的艺术背景或是在社交平台上有足够的粉丝，才可以获得准入入驻平台，而这反而与加密世界所倡导的无准入，平等观念相悖。

  • SuperRare （https://superrare.co/）：2018 年成立的，最早的 NFT 艺术品铸造平台，对艺术品以及艺术家有一定的门槛。

  • CryptoKitties （https://www.cryptokitties.co/ ）：2017 年上线的收款区块链技术上的数字猫收集与繁殖游戏，曾占据以太坊网络 16% 以上的交易流量，导致以太坊网络不堪重负，出现严重拥堵，转账交易延迟、无法转账。

  • Makersplace （https://makersplace.com/）：稀有艺术品铸造交易平台。

  • CryptoPunks （https://www.larvalabs.com/cryptopunks）：2017 年成立的构建于 Ethereum 上的 NFT，并最早启发了 ERC721 标准。它通过改造 ERC20 合约发行代币，生成了 10,000 个完全不同的 24x24、8 位元风格像素的艺术图像。

  • Rarible （https://rarible.com/）：2019 年底成立，以趣味艺术作品为主，由俄罗斯裔连续创业者创立，是第一个推出治理代币的 NFT 交易平台。

  • HashMasks （https://www.thehashmasks.com/）：2021 年由全球超过 70 位艺术家创作的 16,384 张独一无二的肖像画，盲盒形式且用户可为每张画命名，增强用户参与感并赋予其代币更多价值。

  • NBA Top Shot （https://www.nbatopshot.com/）：基于 flow 公链打造的收藏类游戏。NBA Top Shot 通过和 NBA 达成官方 IP 授权合作，从而让收藏者通过游戏化的方式来获得正版 NBA 球星高光集锦，并可以在平台内的市场进行交易。

  • Nifty Gateway （https://niftygateway.com/）：一个受监管的 NFT 的货币市场，于 2019 年经 「比特币亿万富翁」双胞胎 Tyler 和 Cameron Winklevoss 收购，这也是该交易所的首次收购。该平台利用 Gemini 的后端基础设施建立了美元兑换平台，使用户可以用信用卡购买 NFT，出售时可直接兑现到银行帐户中。

  • Bored Ape Yacht Club （https://boredapeyachtclub.com/）：NFT 潮玩+会员俱乐部。

  • Autoglyphs （https://larvalabs.com/autoglyphs）：CryptoPunks 开发团队 Larva Labs 第二个 NFT 项目，生成艺术 Generative Art 平台。

  • Meebits （https://meebits.larvalabs.com/）：CryptoPunks 开发团队 Larva Labs 第三个 NFT 项目，由 2 万个独特的 3D 体素字符组成，可被开发者使用。其所有者可以访问包含完整 3D 模型的资料包，自由渲染 Meebit 并为其设置动画，让它可以被运用在任何游戏引擎、3D 工作室或用作 Metaverse 中的化身。同时，Meebits 可以直接交换 Meebit。

  • Foundation （https://withfoundation.com/）： 社区驱动型，高门槛铸造交易平台。

  • Async Art （https://async.art/）：可编程艺术（主要），音乐铸造交易平台。

  • KnownOrigin （https://knownorigin.io/）：高端艺术铸造交易（一级，二级）平台。

  • Avastas ：生成艺术作品交易平台。

  • Art Blocks （https://artblocks.io/）： 生成艺术 Generative Art 铸造交易平台。

协议层 – NFT 铸造协议及一级市场 – 「游戏」  

这一类应用列举了目前链游开发者们使用 NFT 为游戏内玩家经济赋能的案例。以前 NFT 游戏项目吸引玩家的战略还处在于 P2E (Play-to-earn)模式，主要利用 NFT 道具独特可交易的特性以期待创造贴合的游戏经济，但是在游戏本身的设计上缺少关注，并不能吸引到币圈长期玩家更别说要吸引币圈之外的玩家放弃中心化游戏加入链游。经过近几年的沉淀，开发者们也逐渐意识到成功的链游需要扎实的游玩体验做支撑，再加上 P2E 的加持才能完全打开链游的潜力。

  • Axie Infinity （https://axieinfinity.com/）：Axie Infinity 是一款收集、饲养各式各样稀奇古怪小生物的链游。除了普通的收集属性，Axie 还增加了独有的战斗属性，成为了最早能战斗的 NFT 宠物之一。Axie 团队在 2 月份还为游戏推出了定制化以太坊侧链 Ronin，通过 POA 共识来提高验证速度以缓解以太坊网络拥挤和玩家 gas fee 的压力。目前 Axie 已然成为了链游的代表， 6 月份销售额突破 1.21 亿美元，环比增长 351%，是第二名 NBA Top Shot （4543 万美元）销售额的近 3 倍。

  • Gods Unchained （https://godsunchained.com/）：基于以太坊的链上首个电竞类卡牌游戏，类似《炉石传说》，用户能够自由地交易和出售他们的卡片，并且拥有卡牌的所有权。

  • Zed （https://www.zed.run/）：基于以太坊 Layer2 的区块链上首个 AR 赛马类游戏，形成了养马、赛马、观看比赛等各类玩法充值的一站式 NFT 赛马体验。而其背后推手 Atari 已经向 Zed Run 授权《Asteroids》、《Centipede》、《Missile Command》、《Pong》等多个游戏 IP 内容。未来，随着更多赛马游戏人士的不断加入，玩法和内容边界也将随之不断拓展。

  • Skyweaver （https://www.skyweaver.net/）：类似 Gods Unchained，是一款具有实际交易功能的数字交易卡游戏。玩家根据他们在不同游戏模式中锻造的卡牌数量、商店上对某些卡牌的需求以及在商店上提供卡牌销售的玩家数量来控制经济和 NFT 卡牌的价值。

  • Alien Worlds （https://alienworlds.io/）：基于 WAX 公链的 DeFi 类游戏，玩家可以通过挖矿赚取 Trilium （TLM），每次挖矿时还有一定的几率挖到 NFT。Trilium 可以用来质押到游戏内的六个星球中的一个或多个上面，在报名成为星球理事候选人后，玩家将有可能统治该星球。NFT 游戏卡会随着时间的推移推出创新玩法，包括通过「闪耀」来升级卡牌以及在格斗游戏中作战

  • Sorare （https://sorare.com/）：基于以太坊、由初创公司 Sorare 开发的梦幻足球游戏。玩家可以购买和交易经过俱乐部或联盟认证的 NFT 球员卡来组建自己的球队，且都有其稀缺性和唯一性等级，每张球员卡的评分都是根据他们现实比赛中的表现评定的。玩家搭配完整阵容后可以参加平台上的梦幻足球比赛并获得卡牌和代币奖励。

  • Mega Crypto Polis （https://mcp3d.com/）: 基于以太坊和 TRON 的城市经营管理游戏，玩家可以开始在分散式的城市地图上获得一块土地，建造一系列的建筑类型。每一块土地连同在建造的建筑物都是以 NFT 存在的，可以在市场被转让或交易。游戏中的每一个动作都需要为地区和联邦预算征收一些税收，然后根据他们在游戏周期内 (24 小时）产生的总影响力点中所占的份额，将其分配给所有玩家。

协议层 – NFT 铸造协议及一级市场 – 「元宇宙」

这一类应用代表目前元宇宙使用 NFT 作为数字资产的确权方式的案例 。目前 NFT 内的元宇宙生态相比起艺术和收藏品等早期应用还有相当大的发展空间，当下需要解决的问题有以下几点：

1. 应用硬件门槛较高 ，优化不完善。用户的电脑需要达到一定的配置才能够流畅运行开放世界的渲染画面。未来项目方还是需要在画面表现和流畅游玩体验中做出取舍。

2. 相比起其他区块链领域，元宇宙目前缺乏贴合自身优势的玩法和造富效应 。当下元宇宙生态内大多内容依旧是围绕着 NFT 收藏品所搭建的，收藏家们在元宇宙的各个角落展出自己的藏品，犹如一个 3D 化的 Opensea，针对开放世界玩家的吸引点不多。

3. 玩家群体还是局限于圈内的用户，触及不到不关注区块链的大众群体 。这一点需要强力 IP+流畅使用体验+创新的 NFT 元宇宙玩法来结合出圈。

  • Cryptovoxels （https://www.cryptovoxels.com/）：建立以太坊上的虚拟世界。在加密的像素街区上进行建造、开发和出售房产 parcel，用户会以 NFT 的形式拥有自己的土地，并且可以在建筑物中挂出不同的 NFT 收藏品。

  • Decentraland （https://decentraland.org/）：建立在以太坊上的虚拟世界，于 cryptovoxel 类似，提供 NFT 形式的土地，avatar 穿搭物、内容服务器。除此之外 Decentraland 利用其专属的加密货币 MANA，可为元宇宙的各项运营和计划提供补贴，并且作为用户间交易的货币。

  • The Sandbox （https://www.sandbox.game/）：类似于 Minecraft 和 Roblox 的像素沙盒类搭建游戏，玩家可以在 The Sandbox VoxEdit 中创建 NFT 形式的数字资产并将其上传到商店、通过 Game Maker 简单的拖拉方式来在自己的地产上创建游戏吸引其他玩家参与。

  • Somnium Space （https://somniumspace.com/）:  Somnium Space VR 是基于以太坊的 VR 社交应用，用户可购买土地、构建或导入对象以创建共享的虚拟世界。提供本机 NFT 集成，并支持主流头戴式头显以完整 VR 形式进行探索，使用 CUBE 作为应用内的交互代币。

协议层 – NFT 铸造协议及一级市场 – 「粉丝经济」

传统粉丝经济下，无论是粉丝还是偶像，都不是这个生态中的最大受益方。反而是中心化社交平台作为创作者和粉丝建立联系的媒介，通过在创作者和用户之间插入广告和算法推荐，获取大部分收入。另外一方面，偶像和粉丝之间的互动往往是单调和单向的。NFT 的优势在于更高效的连接粉丝和偶像，鼓励偶像与粉丝双向互动。而围绕 NFT 生态的粉丝经济平台则需要给粉丝和创作者双方设立贴合的经济激励与机制，方便「平民」成为偶像从而鼓励更多原创。

  • BitClout （https://bitclout.com/）：可以让用户使用加密货币来炒作 Creator Coin，是基于名人名气的 NFT。其建立在自己的自定义区块链上，支持复杂的社交网络活动，如发帖、简介、关注、以及其他推测功能等。是专为运行社交媒体而设计的 POW 区块链，每个人可以拥有属于自己的 Creator Coin，并可以进行随意买卖。

  • Rally （https://www.rally.io/）：可以让创作者推出自己的社交代币。Rally 由其本土代币 RLY 管理，粉丝可以购买创作者代币来支持他们喜爱的创作者，并获得由创作者决定的特殊利益。开发者能够基于自己的社交代币驱动的商业模型，并与粉丝共享经济。平台还可以将其原生治理代币 RLY 与创作者的社交代币相结合并铸造成一款新代币，同时设计了 Layer2 方案，允许即时交易、美元购买和使用粘合曲线的固有价值获取机制。

  • Roll （https://tryroll.com/）：代表用户创建 ERC-20 代币，同时提供存储和交易场所。发行模式是每个社交代币的供应量均为 1000 万枚，其中 200 万枚直接发放给内容创作者，120 万枚归属于 Roll 平台，剩余部分会在三年内每个月线性解锁。社交代币发行后，创作者可以将其出售，也可以通过 Roll 应用将其分发给粉丝，也以将其挂牌出售或严格进行点对点交易，或者在 Uniswap 上交换其他代币。

  • Mirror （https://mirror.xyz/）：Mirror 的平台代币 $WRITE （ERC-20）代表了作者们在平台上发布内容的权限，内容创作者需要销毁 $WRITE 才能开启自己的专栏，Mirror 团队首选了一批创作者作为平台的首发作家并每周举行 $WRITE Race，向社区用户们投票选出的作者们投放代币让他们开启专栏，作者们在作品初期可以通过向粉丝发行作品代币进行众筹，作品完成后会以 NFT 的形式上链，文章转载、打赏、以及拍卖产生的收益会根据粉丝在众筹中的贡献分成。

  • Chiliz （https://www.chiliz.io/）：球迷网站 Socios.com 的使用代币，以投票的形式为体育和电竞粉丝提供了联合管理球队、赛事、联赛等活动的权利，参与到团队管理和战略决策中。体育和电竞领域的团队、联赛等各类组织也能通过赋予粉丝一定的决策权而实现粉丝经济化。

应用层 – 金融 – 「流动性协议」

这一类主要代表 NFT 作为底层资产，提供 NFT 定价 的协议。目前三种解决方法：一是买方定价 ，比如 TopBidder 通过竞拍定价；二是卖方定价 ，比如 NiftEx/NFT20/NFTX 通过卖家质押 NFT 并产生 ERC-20 提供初始流动性产生定价；三是第三方定价 ，比如 Upshot 通过随机用户的预测行为产生定价。这些流动性协议是必要的金融底层设施，为上层复杂应用提供 NFT 价值发现。

  • NiftEx （https://www.niftex.com/）：2020 年 5 月上线，允许质押 NFT 并将其所有权分割为 ERC-20，以促进流动性，截止 7 月 7 日，所有质押 NFT 的市值为 $4.4M。

  • Upshot （https://upshot.io/）：2021 年 5 月上线 Open Beta，用预测市场来提供 NFT 的价格发现，不必输入精确价格，只用完成一系列简单选择题，随机两幅画中选择你认为价值更高的。

  • NFT20 （https://nft20.io/）：2020 年 11 月上线，允许将 NFT 质押并产生其 ERC-20 代币衍生品，可去 Uniswap 等 dex 交易以促进流动性。截止 7 月 7 日，所有质押 NFT 的市值为 $1.4M。

  • NFTX （https://nftx.org/）：2021 年 1 月上线，允许质押 NFT 并将其所有权分割为 ERC-20，截止 7 月 7 日，所有质押 NFT 的市值为 $3.2M。

  • TopBidder （https://bidder.top/）：2021 年 4 月上线，是一个加密艺术与收藏品竞拍协议，为 NFT 带来价格发现。

应用层 – 金融 – 「DeFi+NFT」

这一类主要代表利用 NFT 作为 资产表示工具 的协议。NFT 可以看作是 Token 的进化，从只能表示无差别资产到可以定义更多独特价值。这一类会产生**「DeFi」类 NFT** ，比如 Uniswap LP token 和 yinsure 保单，这些 NFTs 和艺术及收藏品铸造协议铸造出的 NFTs 相比，主要体现在铸造方式不同，是通过金融活动来铸造。

  • Uniswap V3 （https://uniswap.org/）：2021 年 5 月上线，将 LP Token 以 NFT 表示 ，定义了不同做市策略下的流动性价值。截止 7 月 7 日，总锁仓量为 $1.8B。

  • Aavegotchi （http://www.aavegotchi.com/）：2021 年 3 月上线，通过抵押 Aave 借贷平台上的付息存款凭证 aToken 获得，向游戏场景拓展了 NFT 的用途。

  • Dego （https://dego.finance/）：2020 年 9 月上线，项目通过盲盒的方式空投拥有不同稀有度的 NFT GEGO，质押后获得 DEGO 代币奖励，也可以通过持有 NFT 参与到社区治理。未来用户还可以通过 DEGO 代币铸造 NFT。截止 7 月 7 日，所有质押 NFT 的市值为 $9M。

  • Yinsure.finance （https://yinsure.finance/）：将独特的保单以 NFT 代币化 ，可在二级市场流动，增加了保单的流动性。

  • Meme （https://dontbuymeme.com/）：2020 年 9 月上线，通过 DeFi 和 NFT 结合的玩法（Stake Meme 到特定抵押池获得相应分数，用此分数可兑换 NFT）。

应用层 – 金融 – 「抵押借贷」

这一类主要代表 NFT 作为底层资产实现抵押借贷 的协议。目前由于 NFT 流动性缺乏和定价问题，导致抵押借贷只能以 P2P 的形式实现。但是参考 DeFi 迭代路径，从 ETHLend 到 Aave，P2Pool 可能是下一个升级路径。

  • NFTfi （https://nftfi.com/）：2020 年 2 月上线，以 P2P 的形式，借款人将自己的 NFT 资产作为抵押品投放到市场上，然后由市场上的放贷人来报价。截止 7 月 7 日，放款量为 $4.8m。

  • Pawnhouse （https://pawnhouse.io/）：2021 年 7 月上线测试版，同样是 P2P 的形式，借款人将自己的 NFT 资产作为抵押品投放到市场上，然后由市场上的放贷人来报价。

  • Taker Protocol （https://taker.org/）：暂未上线。同样是 P2P 的形式，引入 ERC-20 代币 Tai 作为付息存款凭证，创造借贷池成为 DeFi 和 NFT 间的桥梁。

应用层 – 金融 – 「资管工具」

这一类主要是 NFT 产业链上层的资管工具 ，类似 DeFi 里的 Zapper、DeBank 等资管平台。目前项目较少。

  • NFT Bank （https://nftbank.ai/）：2020 年 6 月上线，针对 NFT 的资产管理平台。

应用层 – 泛二级市场

这一类主要是 NFT 经过铸造后，形成的二级市场，不同于一级市场，这一类平台会承担二级市场的职责，大多包括了主要的铸造平台产出的 NFTs。

  • OpenSea （https://opensea.io/）：2017 年 12 月上线，是目前最大的 NFT 交易平台。6 月销售额近 $150m。

  • Auctionity （https://www.auctionity.com/）：2018 年 1 月上线测试版，是加密收藏品的链上拍卖行。

  • TokenTrove （https://tokentrove.com/）：2019 年 10 月上线。主要针对游戏特别是 Gods Unchained 的二级市场。

应用层 – 垂直领域 – 「数据」

NFT 数据领域还未出现全面的数据提供商 。现有的数据要么在 NFT 覆盖面上不够全面，要么在数据维度上不够完整。除此之外，也还缺乏一个功能全面的 NFT 搜索引擎，这是未来重要的流量入口。

  • Cryptoart.io （https://cryptoart.io/）：专注于加密艺术领域的数据平台。提供有关加密艺术品，加密艺术家以及宏观加密艺术数据分析等信息。

  • Nonfungible （https://nonfungible.com/）：覆盖以太坊 NFT 的数据平台，包括市场上主流 NFT 项目，历史及实时 NFT 行业数据以及 NFT 相关行业研究等。

  • NFTGuru （https://nftguru.io/）: 使用 Opensea APIs 追踪数据，特点在于全面的搜索功能。

  • CryptoSlam! （https://cryptoslam.io/）：囊括 NBA Top Shot 等不同链上主流 NFT 收藏品的交易额等历史数据。但是艺术品领域较少数据。

应用层 – 垂直领域 – 「策展与社区」

社区是由内在价值 (如社会地位) 和外在价值 (如未来发展) 所驱动的。策展人的工作就是为这两种动机创造一个和谐生活的生态系统。对于一般 NFT 创作者来说，痛点之一是无法为自己的作品获得初始流量。我们现在看到策展人通过捆绑优质服务去与他们的超级粉丝建立更深层次的关系。因此策展类项目在帮助 NFT 创作者自我营销的同时，可以为用户发现更多潜在价值。

  • Whale DAO （https://whale.me/）：策展社区，通过发行一种社区代币，基于质押「具有价值的」NFT 发行 FT 代币 Whale，截至 7 月 7 日，Whale 资产池估值为 $4.5m。

  • Flamingo （https://flamingo.finance/）: Flamingo 是一个专注于加密艺术投资的 DAO 组织，其成员有权利共同决定所要购买的 NFT 艺术品，并可以将 DAO 购买的 NFT 进行分拆，借贷，抵押，策展等活动。成为 Flamingo DAO 有一定的门槛要求，需捐赠一些 ETH 作为入会条件，其次成员人数上限为 100 人。

  • MetaPurse （https://www.metapurse.fund/）：MetaPurse 是一个专注于投资加密艺术，虚拟资产和资助早期项目的由 MetaKoven 资助的加密社区原生风投基金。区别于其他以 ROI 驱动的加密基金，MetaPurse 以促进 NFT 生态为其主要目标。其近期通过策划 NFT VR 展览展示其购买的加密艺术品。

  • Friends with benefits （https://www.fwb.help/）：FWB 是一个基于的 Discord 私人服务器的私人社交平台，聚集了一批加密领域创作者与思想者。参与者需要持有一定量的原生代币 $FWB 才能加入这个社交圈。$FWB 具有恒定的 100 万供应量。

  • Collab.land （https://collab.land/）：Collab.land 是一个基于 Telegram Bot 的帮助加密初创项目建立早期社区的解决方案。针对于 NFT 创作者，Collab.land 会协助他们策展，创作者的社区中不同层级的参与者展示专属内容。

  • Ark （http://ark.io/）：Ark 是一个针对于 Cryptopunk 以投资为导向的 DAO 组织，成员可以创建一个 ARK 来召集其他成员一同购买特定的 Cryptopunk，也可以参与一个 ARK 获得合力购买的 Cryptopunk 的份额。

应用层 – 垂直领域 – 「社交」

文初提到的上一篇文章中，曾详细介绍了我们对于社交平台的期待，我们认为社交平台是 NFT 出圈的重要媒介。

  • Nifty’s （https://www.niftys.com/）：Nifty’s 是一个专注于 NFT 的社交媒体平台，它将优质出版商、品牌和创作者与收藏家、策展人和他们周围出现的粉丝社区联系在一起。该创新平台提供易于使用的界面，允许会员创建、收集、发现和策划 NFT 世界中的数字艺术和其他收藏品。利用 MEME 协议的技术，Nifty’s 将为创作者提供一个优质、强大、灵活和安全的平台来启动他们的 NFT 。

  • Showtime （https://tryshowtime.com/）：Showtime 提供了一个为加密艺术创作者展示自己作品的社交平台，帮助创作者与加密艺术投资者发现价值。

  • Nafter （https://nafter.io/）：Nafter 是一个将短内容 NFT 化的社交平台。类似于 Instagram，创作者可以将创作的 post 转化为 NFT，并从售卖该 post 查看权中获利。同时用户也可以通过向喜欢的内容创作的个人账户质押该平台原生代币 NAFT 或者该内容创作者的收益分成。

  • Only1 （https://only1.io/）：Only1 是一个专注于 NFT 形式内容的社交媒体平台。不同于 Nafter 上的内容创作者可以同时将一个 NFT 形式内容出售给多人，Only1 通过拍卖方式使得只有一个人可以获得这个 NFT 形式的所有权。而通过拍卖获得这个 NFT 形式内容所有权的用户可以通过收取原生代币 $LIKE 发放该内容的查看权。

我们近期之所以对 NFT 赛道进行频繁的研究，是因为无论从项目方的交易增量，用户增量还是圈外各行各业在 NFT 上的探索，看到了 NFT 的潜力。

未来，几乎一切事物都可以通证化，而 NFT 可以被视为开放的 API 接口，开发者和用户通过接入释放创造性，从而创造无限未来。我们期待成为构造这个未来的重要一环。
