昨晚，随着以太坊伦敦硬分叉升级的正式完成，EIP-1559的效用已经开始发挥。

对于以太坊社区而言，这是一个特殊的时刻，它足足酝酿了2年的时间，而EIP-1559的激活更新了以太坊的费用市场系统，使其更具可预测性并对用户更加友好。

在这个新的费用系统下，每一笔以太坊交易的基础部分费用会被燃烧销毁。随着时间的推移，这种“燃烧”会使 ETH 出现通缩并对其价格施加上行压力，这反过来又会提高以太坊网络的安全性。

截至发稿时，我们距第一次EIP-1559 ETH燃烧已经过去大约13个小时，根据数据跟踪网站ultrasound.money显示，以太坊全网燃烧的ETH已超过了3200 ETH。

年初至今，以太坊的日均发行量大约为10,400 ETH，如果我们保持目前6000 ETH的日燃烧速率，则以太坊相当于减产了近60%。

这一数据确实令人印象深刻，但观察哪些项目在早期成为最大的“ETH燃烧器”也是非常有趣的。

正如你在上图中看到的，迄今为止排名前5的“ETH燃烧器”中有2个明确是NFT项目，一个是目前最大的NFT交易市场OpenSea（目前已销毁358 ETH），另一个则是近期大热的NFT游戏 Axie Infinity（目前已销毁141 ETH）。

这两个项目目前呈现出令人难以置信的活动水平，这对 NFT 观察者来说并不奇怪，因为 OpenSea 是目前世界上收入最高的在线市场之一，而 Axie Infinity 目前的日活用户数已达到了100 万的级别。

（译者注：通过查看etherscan，我们发现目前燃烧ETH最多的地址同样也是一个NFT项目，不过它似乎只是一个CryptoPunk的抄袭品。）

那么，从这里开始，需要考虑的一些早期要点包括：

  1. EIP-1559 的到来进一步加强了以太坊和 NFT 之间的共生关系——以太坊为 NFT 提供了可编程环境且本质上永恒的基础设施，而 NFT 现在更直接地为以太坊的安全服务。

  2. 随着顶级 NFT 项目的不断增长以及更多NFT 项目的到来，ETH燃烧排行榜呈现出的变化将会十分有趣。

  3. 以太坊社区可以共同完成这种规模和重要性的升级，这表明以太坊及其 NFT 生态系统已准备好持续繁荣。
