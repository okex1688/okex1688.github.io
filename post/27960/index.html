如何在众多 NFT 市场平台应用中选择最适合自己的那一个。

今年上半年 NFT 的迅速蹿红顺利接过了去年 DeFi 浪潮的接力棒，不过当我们把时钟调回到大约两个月前，真的又有多少人领会到了**「2021 年是属于 NFT 的一年」** 这句话真正的力量？

在 NFT 市场上半年已经刷新各种记录的背景下，在过去两个月时间里取得了「更进一步」的现象级爆发，NFT 藏品天价成交的消息频现，顶级机构乃至传统行业巨头纷纷布局甚至下场参与其中，而作为 NFT 市场重要基础设施的 OpenSea 平台的数据，也很好地描绘出了这波颇具震撼力的增长。

Dune Analytic 数据显示，OpenSea在 8 月 29 日的单日交易量超过 3 亿美元 ，月内交易量达到近 30 亿美元 ，环比激增 8 倍，本月平台活跃用户数量也达到了 20 万人。

随着新的浪潮推动 NFT 概念的不断出圈，并吸引越来越多的投资者真正参与其中，如何在众多 NFT 市场平台应用中选择最适合自己的那一个，无疑是众多 NFT 市场新人必然会面对的一个问题。为了更直观地对比主流 NFT 市场平台的交互成本，以及各平台对于创作者的激励方案，本文盘点了 OpenSea 、SuperRare 、Rarible 、MakersPlace 、Nifty Gateway 、币安 NFT 、Foundation 和 Zora 八个主流 NFT 市场平台的佣金和版税体系 ，希望能够解决这一疑问。

服务费 很容易理解，简单来说就是平台在提供交易服务时收取的**「手续费」** ，也是市场参与者在买卖 NFT 时所需要支付的主要成本之一。而版税 主要是用来激励创造者和艺术家的，NFT 创建者在进行首次销售之后的所有转售交易中，都能收到一定比例的版税收入。

随着 NFT 市场的不断发展，也有部分平台开始寻求优化版税体系，为 NFT 创建者带来源源不断的收入的同时，在后续转售中也为经手的所有藏家提供收入，以此持续激励 NFT 的流动性和驱动市场达到供需平衡。

那么，在如此大规模的 NFT 市场中，这类平台的服务费和版税体系是怎样的？

通过对比可以发现，不同 NFT 市场平台的服务费和版税体系差别较大，其中，OpenSea、Rarible、币安 NFT 市场以及 Zora 的服务费较低，更多的是让利普通用户和创造者，而 SuperRare、MakersPlace 以及 Nifty Gateway 的首次销售服务费均达到 15%，不过 SuperRare 在版税体系方面存在很大优势，在此前首创 NFT 版税的基础上，上个月新增了收藏者版税计划，在激励二级市场交易的同时也为收藏者带去了额外的版税收益。

另外，Rarible 还在近期推出可简化 NFT 项目上线流程的协议 Rarible Protocol ，其中实施的一套版税标准使得在协议应用程序上出售的 NFT 能够遵守创建者和平台的版税，而不管其出处。

以下是各个 NFT 市场平台的服务费和版税体系的详细介绍。

OpenSea

作为目前市场份额占绝对主导地位的 NFT 交易平台，成立于 2018 年年初的 OpenSea 在交易量和用户数量上都遥遥领先于其他平台。截至目前 OpenSea 已经完成了总额 超 1.2 亿美元的融资 ，投资方包括 a16z 、对冲基金 Coatue 以及长期活跃于加密领域的亿万富翁 Mark Cuban 等。截至 8 月 23 日，OpenSea 于本月在 SuperRare、Foundation、Rarible 等 5 家 NFT 市场平台中的交易量占比高达98%，总用户数也达到超 25 万人。

那 OpenSea 的版税机制和交易费用是怎样的呢？OpenSea 会抽取交易额_（包括首次或二级销售）_的 2.5% 作为服务费用。而关于购买者支付给 NFT 创建者的版税，由 NFT 创建者自行设置 ，不过最高不得超过 10% 。OpenSea 会将这些 NFT 版税收入每两周转至 NFT 创建者指定的收款地址。

另外，OpenSea 无法将版税分配到多个地址，所以接收版税的地址只能制定一个，这也意味着，NFT 创建者无法将其部分版税费用通过 OpenSea 直接用于其他目的。值得一提的是，OpenSea 上免费转让或赠送 NFT 不会产生版税，但若该获赠方再次进行出售，还需要支付版税。

SuperRare

本月完成了代币发行并且又因为**「巨额空投」** 火了一把的 SuperRare 近期正在不断优化其版税体系。SuperRare 与 OpenSea 于同一年成立，目前还处于抢先体验阶段，仅招募了少数艺术家，会在明年全面推出。今年 3 月底，SuperRare 完成由 Velvet Sea Ventures 和 1confirmation 领投的 900 万美元 A 轮融资，Mark Cuban、Chamath Palihapitiya 和 Marc Benioff 等参投。

SuperRare 声称是首个引入自动化艺术家版税 的平台，艺术家在 SuperRare 售出艺术品后将获得销售总交易额的 85%，SuperRare 会收取剩余 15% 作为服务费，当艺术品二次销售时，创作者会收到 10% 的版税收入。而对于 SuperRare 而言，除了会在艺术家首次销售时收取的销售额的 15% 的服务费，还会在所有购买交易中向买方收取 3% 的市场交易费用。

而在上个月，SuperRare 为持续激励艺术品二级销售，在数字艺术品版税方面取得突破性进展，推出了为期 12 个月的收藏家版税体系试点计划，SuperRare 平台上所有未事先销售的任何艺术品都将有资格参与该收藏家版税。

顾名思义，所有收藏版税会支付给收藏家，这部分费用由 SuperRare 收取的 3% 的市场费用来支付。对于符合条件的艺术品的第一位藏家而言，会在该作品再次转售时获得 1% 的收藏版税，若该件藏品再度转手，会获得交易额的 0.5%，三次转售时获得 0.25%，此后依次递减一半，分别为 0.25%、0.125%、0.0625% 等。

SuperRare 中单件作品中的版税和收藏版税分配

而该件藏品的第二位收藏家，将在作品再度转售时获得 0.5% 的收藏版税，之后若再次转手也依次递减一半。同理第三位藏家在作品转售时获得 0.25% 的收藏版税，以此类推。

时隔一个月之后，SuperRare 宣布扩大收藏家版税覆盖范围，从而涵盖到所有 SuperRare 艺术品，这意味着，不管铸造或首次出售的日期如何，SuperRare 上收集的每件艺术品现在都能够参与 SuperRare 的收藏家版税计划。

Rarible

Rarible 在 2019 年年底至 2020 年年初由 Alexander Salnikov 等人成立，共融资到约 1600 万美元，其中在今年 2 月份完成 1kx 领投 的 175 万美元种子轮融资，ParaFi Capital、Coinbase Ventures、CoinFund、MetaCartel Ventures 以及 Bollinger Investment Group 参投，今年 6 月份完成由洛克菲勒家族风险基金 Venrock 和区块链风险投资公司 CoinFund 领投的 1420 万美元 A 轮融资。

与 OpenSea 类似，Rarible 的版税体系也相对简单，且版税 可由 NFT 创建者设置到最大 50% ，不过官方推荐的范围在 0 至 30% 之间 。Rarible 提示称，目前 OpenSea 上的版税不会转移到 Rarible，不过团队目前正在研究跨平台版税计划。另外，Rarible 会在初次销售收取销售额的 2.5% 作为服务费，转售中额外从购买者抽取 2.5% 的佣金，不过后续可由代币持有者进行治理决定。

虽然 Rarible 成立时间较晚，但其可以在平台买卖 NFT 资产来获得原生代币 RARI 奖励的机制，吸引了一批用户。另外，本月 Rarible 还推出了可简化 NFT 项目上线流程的协议 Rarible Protocol，具有可在兑换过程中分摊费用、懒人铸造各类 NFT、新的版税标准以及共享订单簿和流动性等特点。

其中，Rarible Protocol 为协议铸造的 NFT 以及外部铸造的 NFT 实施了一套版税标准，使得在协议应用程序上出售的 NFT 能够遵守创建者和平台的版税，而不管其出处。

MakersPlace

MakersPlace 创立于 2018 年初，在其平台上的每个作品都会有创作者在区块链上进行数字签名，目前仅供艺术家申请入驻。MakersPlace 在本月初完成一笔由 Bessemer Venture Partners 和 Pantera Capital 领投的 3000 万美元的融资 。

MakersPlace 的知名度在今年得到大幅提升，一定程度上得益于数字艺术家 Beeple 耗时 13 年创造的作品《Everydays: The First 5000 Days》，MakersPlace 通过与佳士得合作，在佳士得上将该作品以近 7000 万美元的价格成交，创下 NFT 最高销售记录。

关于费用和版税问题，当创作者出售作品时，MakersPlace 会以 ETH 形式收取作品最终销售额的 15% 作为服务费，以及额外通过信用卡出售的交易费用。当作品被转售时，创作者可以获得销售额的 10% 的版税，MakersPlace 会收取 2.5%，剩余 87.5% 归出售方所有。

Nifty Gateway

Nifty Gateway 成立于 2018 年，在 2019 年 11 月份被加密货币交易所 Gemini 收购，会对申请入驻的艺术家进行审核，也支持借记卡和信用卡购买。

在 Nifty Gateway 上，艺术家可以自行设置二级市场中的版税 ，平台会对每笔二级市场的交易收取销售额的 5% 以及 0.3 美元 的服务费，用于支付信用卡处理费并为平台运营提供资金。

币安 NFT

作为由中心化交易所推出的 NFT 平台，币安 NFT 市场在今年 6 月份正式上线，铸造功能目前仅对获得授权的创作者开放，支持卖家以 BNB、BUSD 和 ETH 进行出售。

NFT 费用设置将支持创作者和将作品存入币安进行售卖的创造者_（后续支持转入功能）_。无论是在币安铸造 NFT 的创作者还是将外部 NFT 存入币安 NFT 市场的创作者将从后续成交的交易中获得 1% 的版税 。

而关于平台佣金，在作品首次销售时会根据不同市场设置不同层级的费用 ，如在 Premium 市场会收取不超过 10% 的费用，在 Marketplace 市场收取最低 1% 的费用，在后续的转售中收取 1% 的费用。

Foundation

Foundation 成立于 2020 年，艺术家需要获得邀请才能入驻，也可以申请成为创作者。在费用和版税方面，Foundation 会在艺术家或创作者首次销售时，收取总销售额的 15% ，剩余 85% 归创作者；二次销售中，创作者能够获得销售额的 10% 作为版税收入，Foundation 会收取 5%，剩余 85% 归卖方所有。

Zora

Zora 是一个基于联合曲线进行动态定价的艺术平台，采取邀请制和申请验证机制，该平台由 Coinbase 前产品经理 Jacob Horne 等人成立创建，在去年 10 月份完成由 Kindred Ventures 领投的 200 万美元种子轮融资， Coinbase Ventures 也参与了投资。

根据 Zora 的服务条款，NFT 创造者可以设置其作品在二级市场转售时的版税收入百分比，范围在 0 至 100% 之间。Zora 平台不会从 NFT 创造、出售等活动中收取任何费用。
