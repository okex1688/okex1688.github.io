Mintbase是一个基于NEAR的NFT平台，曝光度并不算高，但奇妙的是，在此前的许多研究报告中，都可以看到这个项目的身影。

这个项目的进展直到上周才正式上线。该项目背后的团队位于艺术之都葡萄牙里斯本，有意思的是，创始人Nate Geier曾经是个消防员，2008年金融危机期间自学成才变成了一名程序员。

该平台上线后，有推特用户在体验后表示其功能基本上打败了他尝试过的任何其他平台，audioNFTs是一个令人兴奋的前沿领域。Audio NFT是融合音频片段与底层图像或视频来实现多维呈现的NFT，可以帮助音乐、视频创作者拓展一个新的收入来源。

铸造费低廉 让批量制造NFT成为可能

以太坊实在太贵了，另外以太坊的拥堵也限制了NFT的发展，因此在NFT市场我们看到很多不少项目索性自己出来做公链或者选择其他公链搭建应用。

Mintbase选择的NEAR也被称作末代公链。NEAR在去年发布了一个新的跨链工具叫做彩虹桥（Rainbow Bridge），十分有针对性地用来连接以太坊和 NEAR 网络。所以Mintbase平台上线就能支持NEAR测试网和以太坊，任何支持 NFT (ERC-721) 的钱包在Mintbase都可以使用。

而且，在Mintbase上使用 Minting 工具创建完代币后，也可以提到OpenSea等平台去交易，或者选择在Mintbase交易。这是由于Mintbase 与 NEAR所制定的新标准NEP-171所决定，在该标准下，NFT 的版税被内嵌在通证之内，创作者可以通过多个不同平台赚取使用费。

Mintbase 与 NEAR 共同定义了一种新的通证标准——NEP-171，在该标准下，NFT 的版税被内嵌在通证之内，创作者可以通过多个不同平台赚取使用费。Mintbase只在出售价格大于0的商品销售时收取2%的费用，用于发展、运行传统基础设施，而若转移资产至其他平台交易时不收费。

Mintbase当前整个 UI 由 IPFS 上的TheGraph 驱动，因此Mintbase支持开发者API，也就是说只要程序部署，人可以在Mintbase完成的操作，也可以用代码自动完成……这让人回到了在DeFi世界被科学家统治的“恐惧”中是不是？

该平台旨在降低玩家参与NFT的门槛，例如，在以太坊上用NFT铸造一张门票可能需要60美元-100美元，而在Naer上费用低于0.01美元。

创作者经济？

Mintbase推特称，希望驱动创作者经济至一个新水平，使得创作者享受永久的版税、收入分成和负担得起的铸币费。

创作者经济？其实就是NFT 版 Shopify，当前的NFT铸币平台是亚马逊模式，而创作者经济即创造者驱动模式，让创作者打造自我的品牌店铺（这不就是淘宝吗）。

Mintbase的展示板公布了现在入驻的情况，的确重视“商店”。目前有超过1000家商店入驻，超过4万个令牌被铸造，累计有2000多个NFT被出售，而Mintbase用户总数是1.5万人左右。

Mintbase上可以售卖的东西展示形态很杂，可以是数字艺术品，可以是音乐创作，也可以是门票等等，几乎除了钱，都可以上去卖，并且允任何人（商家、作者、经纪人，甚至其他创作者等）都可以对某件 NFT 作品出售收益和版税（royalties）进行一次性分成，参与分成的人数最高可达 40人——目前这一机制还不清楚，因为其竞拍功能还未上线，我们可以拭目以待。
