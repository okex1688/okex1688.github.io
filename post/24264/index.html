自2020 年底以来，NFT 在音乐行业掀起了一场大风暴。随着疫情让很多演出搁置，音乐人越发迫切地希望从流媒体服务中赚取更多版税。随着需求的增加和新项目的开发，NFT 音乐市场正在不断扩大。

和许多行业一样，在利益驱动的音乐行业中，稳坐金字塔顶层的企业没有动力改善音乐人的生活条件。那么在金字塔底层的音乐人，就必须自己想办法，改善音乐生态。

区块链的出现，带动了一波全球性的平权运动，用户能真正掌握自己的区块链账户，信息的传递与价值的转移，无需第三方的审核或准许。区块链已经为许多第三世界的居民实现了金融自由及金融自主的生活。

区块链可以授权音乐人通过私钥持有其音乐或视频，通过 NFT 分发，买家将有权获得其独特的副本。

今年NFT生态与多个知名 IP 进行结合，掀起新一波浪潮，看准了消费者市场对明星的追随。

Snapback

音乐编号：BLE528-3

音乐描述：回响贝斯，电子舞曲。

多位音乐人纷纷加入到NFT的热潮中。2月，摇滚乐队Linkin Park的创办者联合主唱Mike Shinoda将自己创作的一段音乐作为NFT（One Hundredth Stream）在平台发行拍卖，最终以3万美元成交。

2月底，知名DJ、制作人3LAU发行了33张限量版NFT专辑《Ultraviolet》，拍卖总价高达1160万美元；

3月，摇滚乐队Kings Of Leon发行NFT专辑《When You See Yourself》，销售收入超200万美元。

No Aces

音乐编号：AD2217-1

音乐描述：充满活力的原声吉他推动了这首经典的流行歌曲。

5月25日，歌手阿朵在个人微博发布了新歌《WATER KNOW》，全曲采用NFT技术进行数字加密，并将封面和歌曲的署名权公益拍卖，这是国内第一首NFT数字歌曲公益拍卖。据悉，上链仅10分钟，竞拍金额就突破1万元人民币，最终作品以304271元人民币的价格成交。

NFT与音乐的结合正成为一种新的流行趋势，吸引着更多的音乐人参与到NFT领域。它为音乐产业的发展带来了一种全新的模式，NFT和区块链技术改变了传统的音乐发行方式，让音乐版权管理变得更加透明清晰。它解决了数字音乐所有权的难题，也为音乐创作者带来了新的收入来源。

NFT的实时真伪识别及可交易属性有效提升了传统收藏市场的模式，不但得到了加密资产用户的认可，也吸引了更广泛的大众消费者。NFT 将成为独立的新娱乐媒介品类。

NFT无法解决所有艺术家在疫情期间已经或正在面临的生计问题，但仍然给他们提供了新的机遇。

NFT+音乐行业发展潜力

众所周知，在艺术界，非同质化代币已掀起巨大波澜。而在音乐界，NFT 的飓风效应也是如期而至。Water & Music 收集的数据显示，仅在 2021 年 2 月，NFT 音乐行业盈利就已经达到近 2200 万美元。

了解NFT音乐行业最好的方法之一是关注该领域出现的一些关键早期玩家，随着音乐NFT 迅速扩张，他们也引领着其发展方向。Water & Music 统计的数据显示，过去半年多的时间里，音乐 NFT 的销售额增长了150倍。似乎每个音乐人和音乐公司都在试图参与一把。

音乐 NFT 可以采取多种形式，包括但不限于音乐会（虚拟和实体）门票销售、采样包、未发行歌曲预览、艺术品等。随着音乐 NFT 的不断增长，该技术的潜在用途将迅速扩展，远远超出如今的 NFT 使用方式。

为什么购买NFT版本音乐

很多人会问，既然传统平台可以找到音源，为什么还要购买 NFT 版本？

新奇效应

对一些人来说，毋庸置疑，NFT 新奇且炫酷，除了提供电子专辑下载权限外，还有限量版实体版本唱片，以及独一无二的NFT专辑收藏品。

NFT 专辑在音乐行业仍还是一个相对较新的概念， 对于不熟悉NFT 行业的人来说，这很像90 年代从未见过互联网的人解释互联网。NFT 所具有的高度概念性和前所未有的创新性对于很多人来说还新奇，通过购买自己喜欢的音乐人发行的NFT，在某种程度上可以以最直接的方式了解这个新兴概念。

限量稀有

NFT，「非同质化代币」，往往数量有限或独一无二，这恰恰是粉丝经济的关键"来自偶像，限量珍贵"。大多数音乐人会以有限的模式发布歌曲或专辑，为粉丝创造独特、可销售的数字商品、艺术品，带来独特的音乐体验。

虽然 NFT 是一种数字商品，但购买 NFT 的底层逻辑其实和购买实物商品相似。例如，即使一张专辑可以在网上可以免费或付少量费用播放，许多粉丝仍然会想要购买限量版黑胶唱片。

NFT 的走红和那些依靠营销驱动的品牌有共通之处。Supreme T恤没有什么特别之处，但仍然十分特别，因为它只供少数人拥有。

独特权益

一些音乐人正在使用NFT来给他们的支持者提供前所未有的体验，包括前排门票、个人纪念品等等总的来说，这是一次丰富且独特的尝试。因此，从音乐家那里购买 NFT 购买到的是独一无二的物品。

投资目的

有些人购买 NFT 资产可能是出于投资的目的，希望其持有的 NFT 会在未来实现保值或升值，可以从转售中获利。NFT 的波动性和不确定性使得 NFT 持有风险比一些传统投资要高一些，但对于许多投资者来说，他们愿意承担这种风险。

简单来说，NFT 可以简单看作是一种「数字资产+身份验证证书」混合体，对于任何能够将现实世界与 NFT 联系起来的应用都具有很大的发展前景。由于 NFT 可以进行验证且极具安全，音乐人也可以更好地留存粉丝，创造新的互动方式，不仅获得更多的版权收益，也能减少传统固有成本开支。

NFT如何赋能音乐人

对于音乐人来说，NFT 提供了令人兴奋的可能性，可以去除行业的一些中间商和第三方，以全新的方式将他们的作品代币化。

歌曲出售给唱片公司，唱片公司抽成加上营销费用后，大多数音乐人获得的收入远远不可能达到预期。对于大多数音乐人来说，公司经理、唱片公司、出版商等从他们的音乐中获取的利润远远超过音乐人们，留给他们的只有一小部分。

格莱美提名音乐人Illmind 最近以NFT的形式发布了世界上第一个节拍采样包。任何人都可以收听，但所有权和使用权只归出价最高的人所有。这种销售模式可能会对音乐的授权和销售方式产生重大影响。

目前，对唱片公司、出版商及其律师来说，音乐授权和所有权转让是一个相当繁琐的过程，手动清除样本使用或所有权转让甚至会推迟音乐的发布日期。但使用 NFT 可以让音乐授权和转让像线上购物一样方便快捷，文档传输高效且安全。

由于新冠病毒疫情给音乐行业带来的沉重打击，越来越多人认为 NFT 可以作为减少数字作品盗版、帮助为艺术家创收的一种新方式。

NFT音乐平台  ROCKI+AUDIUS

如果上文的介绍已经让你对 NFT 音乐行业有了基本了解，那么接下来将盘点一下两个相对有名的 NFT 音乐平台——ROCKI 和 Audius 帮助大家进一步了解 NFT 音乐平台如何为粉丝和音乐家赋能。

ROCKI

ROCKI 是一个数字音乐流媒体服务和支付网络，是币安智能链上最大的音乐平台，旨在通过其应用程序内原生令牌 $ROCKS直接奖励艺术家。该平台采用混合型以用户为中心的运行模式，艺术家可以获得货币作为奖励。

ROCKI 为艺术家开辟了新的收入流，同时带来了新粉丝参与度和价值。在 ROCKI 上，无论音乐家初始受众规模大小，都可以利用其现有粉丝群，引入音乐 NFT 以及独特的支付模式，即使在疫情仍然肆虐的不确定时期，也能让音乐家有收入可得。

此外，ROCKI 通过在 ROCKI 平台上用 ROCKS 代币奖励听众，感谢他们收听赞助和策划的音乐，扩展了「以粉丝为中心的」的定义。听众在创建播放列表、提供反馈和主持社交活动时都可以获得 ROCKS 代币，这是该流媒体平台创下的先例。

ROCKI 试图解决以下问题 :

1.省去中间商和第三方，让粉丝和艺术家直接建立关系

2.与完全拥有自己音乐的艺术家，即独立艺术家合作，这也是迄今为止增长最快的音乐家类别。

3.利用区块链的力量来处理运作：透明度、安全性、无需信任的合同、金融。

4.用混合模式支付流媒体版税，并将大部分订阅收入分享给音乐家。

5.奖励听众，吸引更多用户参与。

Audius

Audius 是一个去中心化流媒体平台，在多个节点上运行，以确保音乐完全归艺术家自己所有。该平台每月用户超 300 万，旨在替代 Spotify 和 SoundCloud。该平台开启了新的可能性：将自己的 NFT 图书馆纳入网络。

该平台自称是「创作者经济的基石」，是艺术家 / 粉丝互动的枢纽。该平台还启用了 Audius 护照，允许用户连接到 Web 3.0 生态系统。该网络无需权限。因此，创作者能够利用整个图书馆，利用现有音乐呈现出带有新外观和新感觉的新作品。此外，网络参与者可以运行节点，做出贡献并获得奖励。

Audius 并不出售 NFT，而是推出了「收藏品」功能，允许参与艺术家和一定级别的用户（音乐家和用户都必须持有至少 100 个 Audio 代币才能拥有 Silver Tier 帐户，并有资格使用 Collectibles 功能。）展示其已经拥有的 NFT。这是一个音乐家营销、出售 NFT 藏品的好机会，也是用户发现并购买中意的数字物品的好机会。

Audius 允许音乐家向粉丝发行并直接获得收益，特点是：

1.由 Audius 平台代币（$AUDIO）、第三方稳定币和艺术家代币驱动的高效代币经济。

2.用于分享音频和元数据的去中心化存储解决方案和分类账户。

3.可编程的机制。

4.供用户有效查询元数据的发现协议。

5.去中心化的管理协议，艺术家、节点运营商和粉丝在平台变更和升级的决策中都可以拥有或个人或集体的权利。

NFT对音乐的赋能方案

知识产权

众所周知，数字媒体可复制、可共享、可窃取：种子下载和音源泄漏无疑削减了整个音乐行业的利润，威胁到传统的分销和货币化模式。通过投资 NFT，音乐界的创作者、策划人、画廊家和其他利益相关者正在试图创造和捕捉「数字稀缺」的价值。一首 NFT 歌曲的价值不一定只是它的声音，而是让消费者可以拥有对该歌曲文件的独特所有权，将所有权通过区块链直接从艺术家本人手中转移到买家手中。

对于超级粉丝来说，这份荣光、炫耀与权利可能就足够使之倾倒；而对于那数百名涌入 NFT 市场，推高 NFT资产，如 GIF 和 JPEG价格的投机者来说，他们希望，NFT 音乐媒体的新颖性将随着时间的推移在二级市场上不断积累价值，让他们通过转卖获得更多利润。

方便灵活

NFT 最为重要的一个特点就是其便利性。粉丝只需要创建数字钱包，就可以访问并接收内容，参加拍卖，有机会获胜竞品。这使得购买独家音乐作品像网上购物一样简单。音乐家也只需注册上传作品，设置意愿价格等条目，然后等待买家拍卖即可。

此外，艺术家对想要拍卖的物品有很大的灵活性，可以是数字的，但也可以是实物的。专辑、数字作品、声音片段、商品和音乐会门票都是音乐家可以发布的非同质化代币形式。粉丝们乐意为这种新奇的体验付款，音乐家们自然可以「坐享其成」。

安全且去中心化

数字交易是安全即时的，没有中间商，NFT直接链接了音乐家和粉丝。但音乐家们交换的不是数字货币，而是NFT，包括音乐会门票、限量版体验、数字内容、商品和音乐。粉丝们参与这些代币的拍卖，出价最高的人通过安全交易直接存放在数字钱包。

在这些不涉及第三方唱片公司交易所中，音乐家可以为他们拍卖的内容筹集资金并保留所有内容。粉丝们为获得独特的内容而花费资金，所得物安全地存放在自己的数字钱包中，而拍卖资金几乎可以归音乐家所有。

此外，区块链内编码的使得 NFT 音乐行业具有持久性，存在于任何媒体平台之外。一旦文件被编码为 NFT，就很难被窃取或进行其他违规操作。只要区块链正常运行，NFT 音乐就几乎会持续下去。

票务活动

长期以来，现场和虚拟活动的票务一直面临着许多困境：低效率、机器人程序、黄牛、隐藏费用、假冒伪劣票等等。得到区块链技术支持的 NFT 使得诈骗可能性降到最低，所有权转让记录一目了然，票务验证轻松完成。NFT 票务（或「智能票务」）也可以革新二级销售市场。

未来面临的挑战

新兴音乐家

市场上对 NFT 音乐代币的需求往往来自对特定音乐家作品的需求。流行或老牌的音乐家可以毫不费力地吸引粉丝，参与竞拍。但对于新兴艺术家来说，这种需求还需要花费很多时间和精力创造。新艺术家没有那么多机会从这种新音乐形式中受益。

到目前为止，进入 NFT 市场的新兴艺术家数量仍然很少。NFT的销售通常由地位和炒作驱动，而不是由水平驱动，因此新兴艺术家没有爆红也是情有可原的。

制造稀缺性

NFT 可以激励新的艺术形式，但如果艺术的价值纯粹是其稀缺性，那么有限的发行和艺术家访问权限就是重中之重。

数字稀缺性和物理稀缺性大不相同。座位有限是现场表演稀缺的一个例子，这也是物理条件带来的限制，但 NFT 直播节目门票出售的数量同样需要制造稀缺性，门票出售数量可由艺术家自行决定。NFT 依赖于人造概念：让本不稀缺的东西稀缺起来，价值自然也就随之出现。但价格过高可能也会使很多粉丝产生被排斥感，把很多粉丝拒之门外。稀缺性高低的决定仍然需要视尺度而定。

市场泡沫

这会是新一轮的泡沫吗？我们是否因为 NFT 当前的火爆程度而高估了什么？如果出售 NFT 的价格是基于炒作和稀缺性，那么对于那些购买 NFT 作为投资产品的人来说，人们一旦兴趣下降，他们面临的风险将是不可估量的。

拥有一首带有 NFT 标签的歌曲或专辑可能需要花费数百美元。NFT 音乐物品已经不是简单的音乐或 MP3，而更多意义上已经进入了收藏品行列，售价往往要高得多。此外，还有一些音乐家正在创作拥有 NFT 标签的作品副本。理论上，它确实能让每件作品都独一无二，但其价值也会在现实世界中大打折扣。如果作为一名收藏家，希望购买的资产能在未来实现保值、增值，可以尝试购买那些真正独一无二的作品，而不仅仅是附加 NFT 标签的副本。

知识产权

使用区块链转让知识产权所有权的好处一目了然，但这其中仍然存在许多不确定性。鉴于该技术刚刚推出，而全球版权法又极为复杂，去中心化系统中的智能合约很有可能遇到问题，这都需要在前进的途中不断发现，不断解决。

出售模式

NFT 可以在市场出售。绝大多数音乐 NFT 使用拍卖模式，但有些则以固定价格出售。哪种方法优先取决于卖方意愿。大多数市场不接受使用信用卡或传统现金付款，这一定程度上阻碍了很多粉丝的进入。

未来已来

随着越来越多的企业认识到 NFT 音乐行业的价值，这个新兴市场逐步开始得到认可。

如果你是一位试图进入 NFT 行业的音乐人，可以设计限量且有意义的 NFT，让支持者购买到的是正宗且独特的物品。这不仅能为他们带来更多价值，也能建立双方之间的亲密和信任关系。

总而言之，NFT 对音乐家以及音乐行业都会带来颠覆。它让音乐家享有更多控制权和决定权，也能为他们提供了新的收入流，这在疫情尚未结束的今天尤为重要。同时，它让粉丝能有机会享受到拥有爱豆全球单品的快感，也让更多粉丝能够更近距离的接触音乐家，也能更加安全、更有保障地获得可验证的 NFT 资产。

区块链行业一直存在降低使用门槛的“破圈”问题，浏览器、钱包、交易平台、游戏等纷纷提升体验，简化流程，但似乎成效有限，还在小圈子里转圈，主流的“流量人群”始终没有进入。

NFT提供了一个工具和载体，以音乐行业为代表的文化娱乐产业，基于“粉丝经济”的巨大的能量，让“破圈”轻而易举，未来预计将会有更多影视明星、音乐人、设计师等加入 NFT 热潮中，一些垂直发行 NFT 功能的区块链、传统行业的知名 IP 以及区块链与产业的结合深度都将受益于此。

尽管NFT音乐现在尚处于起步阶段，其商业运作模式和用户体量都无法与主流音乐市场相比，但它在培养为数字音乐付费的观念，建设美好的音乐生态环境，推动音乐产业发展所起的作用还是具有深刻意义的。
