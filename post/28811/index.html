对NFT感兴趣的朋友们，看看下面这些月饼你喜欢哪一个？限量发行，给我地址，发送给你。按照目前的行情，数字月饼可比真实的月饼值钱多了。怎么样？

告诉你一个秘密，这样的月饼我想发多少就发多少。各种组合，各种图案。但是呢？发多了就不值钱了，所以，我就发100个。 有五仁的，有蛋黄的，有咸肉的，各式各样。最稀缺的就是火腿肉，还有鹅蛋黄的了，每样只有一个。 

这个NFT发行模式大家是不是非常熟悉，其中有一些创作，但更多是组合，有发行方来制定规则和稀缺性。在目前发行方不多的情况，在NFT的发展早期，这种NFT更多的是这个时期的历史价值。与发行的什么实际的东西（就是数据）没有多大关系。

所以说，这是NFT的极早期阶段，也是很多人看不懂的阶段，难以理解从传统的价值判断来如何解读。泡沫有多大，什么时候破灭。什么时候真正迎来更大的发展，都是要拷问的。

NFT 发展的下一步，价值必然需要回归到数据本身，无论是画作、音乐、视频、文学，还是其他任何数据，NFT的价值体现一定回归到数据本身， 真正有价值的NFT一定包含创作者的心血和积淀，在实际的世界中（无论是传统的现实世界，还是虚拟世界）中发挥其功用和艺术价值，并通过交易价格反映出来。

也就是说，NFT必然有一个创作过程，NFT的价值也是创作这劳动价值的体现。前面讲到的月饼NFT是一种创作，其实不是整个NFT的创作，它是元素创作后的根据算法规则的一种组合；而另一种NFT是链下艺术转移到链上，比如本身就是艺术家的作品，球星的射门视频等等，目前这些链下的创作与传统创作方式没有区别，这只是一种链下与链上的一种链接。

由此，我们可以看到，其实，在目前的区块链世界里，还没有真正的由广大创作人员在链上参与的NFT艺术，也没有这样的创作平台，这与当前区块链的属性和局限性有很大的关系：  

  • 其一，NFT是数据在链上的价值化呈现。但是传统的区块链对于数据的存储效率极低。IPFS/Filecoin 等去中心化存储平台的诞生就是瞄准了这一类问题；

  • 其二，区块链的处理效率较低，就是我们常说的 TPS 问题需要突破。近年来有许多创新来推进这个问题的解决。比如 DFinity 采用子网的模式，Solana采用 POH共识。

这些发展为NFT的链上创作提供了可能性。但是，不要忘了，几年前的 EOS 也曾经做过这样的努力，有一些创作类的游戏，但是EOS在整个生态中并不十分成功。其重要原因在于，采用 DPOS 的EOS并不是一个可扩展的平台，对应用的支持难以满足 Web 级的应用，同时EOS的生态相对比较封闭，成本也比较高。

IPFS/Filecoin 是一个更开发的存储平台，目前与众多的其他公链都在建立链接，目标就是提供存储的底层，并能够支持 Web2 和 Web3 的应用。现在的区块俩生态更加开放，Composibility 是区块链的一个重要特性。DFinity 也在前几天刚刚通过的与 BTC 和 Ethereum 链建立桥接的提案，以实现把区块链的智能合约以及价值导入到 Web 级的应用之中。

因此，可以说，NFT的创作将迎来新的机遇。  

那么为什么要链上的 NFT 创作呢？很简单，去中心化的、去信任的、广泛的合作的威力远远比传统的合作更具想象力和破坏性。这与互联网电商链接小商家到全球某个角落的需求者，以及开源社区的跨地域合作一样，这将带来合作的新范式，没有繁琐的法律条款，无需以公司的形式存在，大家就可以广泛合作，但，与当前互联网的协作不同的是，区块链为基础的协作被成为 DAO，它是通过算法和规则链接合作各方的，它是通过 Token 以及与之关联的激励规则来约束和推进协作的。也就是说，是建立在智能合约的基础上的协作。

区块链建立的信任关系将极大地降低这种协作的成本，同时区块链链接全球的属性将完全打破地域，意识形态、种族等等限制。这会给协作带来无限的可能性。 

在NFT的发展过程中，我们将看到不同的相关项目会推出来，这种协作的模式、激励和约束方式将被不同的项目进行创新、实践、改进。 最终我们将看见一个全新的世界。在这个世界里，链上的创作如繁星般璀璨，NFT自身不仅仅是NFT，也将是创作新的NFT的素材，NFT的组合和加工将形成新的产业，构建更大的世界，这个世界由大家一起创建，一起建立规则，这里面，没有寡头的垄断，因为开放是这个世界的主题。这个世界，我们就成为元宇宙。
