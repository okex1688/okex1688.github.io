迈入 NFT 领域有点像冒险进入飓风中心。

DeFi、加密、Web3 等等一系列缩略词从四面八方涌来。

更糟糕的是，不怀好意的诈骗者正在利用这些新事物，窃取秘钥、洗劫钱包的虚假网站、钓鱼链接等是加密采用的严重障碍。

话虽如此，但毫无疑问 NFT 已成为一种新兴的媒介，有可能演变为一种全球现象（如果它们还没出现的话）。

大多数 NFT 项目是一种众筹形式或投资于与房地产、交易卡、奢侈品手包和葡萄酒相关的新型资产。作为一家风险投资公司（Intudo Ventures）的投资者，我将把传统金融世界与虚拟世界进行比较，以向您展示进入 NFT 投资的安全途径。

您想成为什么样的投资者？

加密和 NFT 领域很有吸引力，因为它打破了传统上阻止散户投资者进行交易的障碍。

最常见的问题是：您想当一个短线炒作者（当天买卖，迅速进出的 NFT 日内交易者）吗？

如果您的回答是肯定的：媒体都在关注那些成功的 NFT 短线交易者统计数据，这些人通过加入白名单+二次销售一夜之间成为百万富翁。但是，以下是经常被忽略的部分：

被列入白名单并非易事。 白名单意味着您可以保证以项目的铸造价格获得 NFT，这通常是对早期支持者和活跃社区成员的奖励。除非你有幸赢得赠品，否则典型的方法是花很长时间在 Discord 上保持对话或邀请新成员。甚至在某些情况下，人们会花钱外包他们的 Discord 活动，以便将大肆宣传的 NFT 项目列入白名单。

“广撒网，重点捞鱼（spray and pray）”的投资策略，往往仅适用于那些资本充裕的人 ，大多数普通投资者并非如此。作为一个短线操作者，参与每个项目的社区可能不是明智的选择，因为时间至关重要。与投资初创公司类似，如果资本是你唯一能提供的价值，那么失败的机会就会大大增加；采用这种策略的投资者指望靠运气“中彩票”，以弥补其整体投资组合中的损失。

如果您的愿望是成为一名全职的日内交易者，请考虑您需要投入多少时间和研究来实现翻盘或者一夜暴富。

“孵化”您的藏品

如果您的答案是否定的：NFT 在交易采购和获得令人垂涎的投资机会方面改变了游戏规则。从本质上讲，您投资的是一项类似于运行风险投资策略的长期业务。

传统上，风险投资公司从高净值个人或机构筹集资金，并使用该资金对早期公司进行投资。这需要购买股份并将其培养孵化至成熟，通常需要 8 到 10 年。然而，通过 NFT，创作者网络的覆盖范围已从他们身边的朋友和熟人群体扩展到 Discord、Twitter 和 NFT 市场；任何人都可以选择通过铸造或购买二级资产来成为该项目的投资者。

铸造一个项目类似于众包一轮融资。从创作者的角度来看，最终目标是筹集资金。

对于投资者来说，问题是如何理清噪音。

购买时，社区、艺术和路线图都是常见的答案。然而，如果没有强大的创始人或团队到位，这些项目将会失败。如果将其与早期创业公司进行比较，团队是最强大的驱动因素，因为他们是推动愿景、鼓舞士气和凝聚内部和外部利益相关者的领头人。如果团队选择继续匿名，请重新评估您的风险偏好以及您如何权衡产品与团队。

将 NFT 的 Discord 频道视为数据中心，一个包含尽职调查过程相关信息的渠道。您将获得产品市场契合度和客户服务，而不是财务和运营计划。在成名之前，团队如何处理喷子/水军、缓和矛盾并脚踏实地的执行路线图计划，这是衡量 NFT 成功潜力的重要指标。

对于这种类型的投资策略，耐心是关键。给自己一个更长的时间线可以让你作为投资者与项目一起成长并建立你的股权价值。

这将我们引向下一个问题——投资组合构建。

就投资而言，Less Is More 适用吗？

错失恐惧症（FOMO）是 NFT 领域的一大推动力，万一擦身而过的项目就是下一个风口怎么办？

但是，一旦您摆脱了对地板价的痴迷，您可能会发现您可以对项目的增长产生积极影响。类似于初创公司的 A 轮或 B 轮融资，如果您的种子轮投资者试图立即兑现，这将是向所有未来投资者发出的极其糟糕的信号。

一个首选的投资论点是采用集中投资组合方法，这样就可以保持积极和高互动的投资组合管理。这是通过花费尽可能多的时间通过产品建议、参与路线图项目和教育新成员以构建 NFT 项目来完成的。

直观地说，忠诚的成员越多，项目的议价能力就越强。成功的 NFT 路线图为吸纳新成员和货币化创造了机会。

Bored Ape Yacht Club (BAYC) 执行了最伟大的路线图剧本之一。 BAYC 的创建者能够为 NFT 持有者提供通过转售或知识产权建立新收入来源的机会。通过提供“采用”模仿其 BAYC 化身的狗类 NFT 和创建其化身的突变版本的免费数字血清的机会，最初的 BAYC NFT 所有者只需持有即可获得两个额外的 NFT。除了几个 BAYC 持有者使用他们的头像创建的音乐、咖啡和商品品牌之外，我们如何处理创作的新视角也出现了。

Writer’s Room

专注于创建去中心化的内容创作媒体业务，来自 BAYC #1798 的 Jenkins the Valet 推出了 Writer’s Room。在 Jenkins The Valet 的知识产权获得成功后，团队搭建了一个平台，让 Writer’s Room NFT 持有者可以享受相同的资源，例如将角色授权给专业的作者，与各个平台的内容创作者合作，对创意方向进行投票，以及收取版税。

以知识产权（IP）作为 NFT 的基础，Writer’s Room 将 IP 提升到了只有 Web3 公司才能做到的水平；利用两个去中心化自治组织 (DAO)，Writer’s Room 提供了一个基础设施，允许成员协作、开发、推销并最终产生加密原生知识产权。创作者和观众不再相互排斥，能够参与制作和享受他们关心的作品，因此您可以在不收购新项目的情况下实现多元化的投资组合。

总体而言，虽然 NFT 似乎是一个令人生畏和奇怪的概念，但可以通过现实世界的案例来驳斥那些反对它的人。与投资早期创业公司类似，多年历史业绩之间的权衡是股权成本；例如，作为对早期支持者的奖励，投资一家公司的成本低于你尝试购买一家财富 500 强公司的成本。

与所有开创性运动一样，NFT也存在一系列独特的障碍，尤其是教育，仅以本文献给那些好奇并渴望探索 NFT 世界的新人。
