Bespin 新版本介绍

我们怀着激动的心情向大家宣布，RaidenNetwork（雷电网络）的 Bespin 版本已上线主网！Bespin 版本的发布标志着我们离成熟的 RaidenNetwork 又近了一步。Bespin 版本聚焦于稳定性，尤其是传输层的稳定性。过去一年来，我们团队因为传输层的问题已经进行了几次很长的调试会话。因此，我们很高兴终于有了一个稳定的传输层。

还有一个好消息是，继 Bespin 版本上线主网后，Raiden 轻客户端很快也会上线主网！

最后，Bespin 通道并不兼容 Alderaan 通道。因此，用户需要关闭并结算所有 Alderaan 通道。

点击此处，查看 Bespin 版本的完整更新日志。

Bespin 版本的目标

Bespin 是 RaidenNetwork 在以太坊主网上发布的第三个主要版本。该版本旨在实现一个稳定且没有漏洞的传输层。我们已经下了很大功夫来重构传输层。主要的区别在于，Raiden 不再使用 Matrix room。这大幅降低了 Matrix 服务器的负载，并提高了传输层的可靠性。

另外，Bespin 版本与即将亮相的 Raiden 轻客户端兼容。因此，Bespin 不仅是新的 Raiden Python 客户端主网版本，也是迈向 Raiden 轻客户端主网上线的重要一步。

Bespin 版本可以为以太坊用户提供快速、低价且可扩展的支付，我们鼓励任何有兴趣的人都来试试。尤其是在近期 gas 费高涨且 rollup 吞吐量受限的背景下，人们对可扩展方案的需求早已今非昔比！我们非常愿意与想要使用 Raiden Network 支持付款功能的项目合作，并对所有反馈和功能建议表示感谢。如果你想要在侧链或 rollup 上部署 Raiden Network，欢迎联系我们！请随时通过 contact@raiden.network 与我们联系。此外，如果你正在构建有益于 Raiden 生态的项目，可以关注下 Raiden 信托的赠款计划。如果你发现了严重漏洞，请立即上报至漏洞赏金网站。

新功能

如上文所述，Bespin 的绝大多数改进都发生在后台。下面我们将简要介绍一些比较重要的更新。

  • 新增 UDC 存/取款端点 ——用户可以更轻松地向 UDC 合约存/取款，这是使用 Raiden 服务包所必不可少的。

  • 支持最新的以太坊客户端

  • 支持 Python 3.9

  • 缩短同步时间 —— Raiden 启动时需要与区块链同步，以免遗漏任何相关的链上事件。这个过程一向都很缓慢，但是 Bespin 版本大幅缩短了同步时间。

  • 新增 “快速连接” 功能 —— WebUI 已经引入了 “快速连接” 功能，可以为用户提供一份开通通道的推荐节点名单。该功能可以让用户快速连接至网络，并保持最优拓扑结构。

  • 去掉了 “加入代币网络”  端点 —— 该功能已经从 API 中移除。建议使用 “快速连接” 功能。

  • 去掉了转账拦截功能 —— 该功能原本是一种在转账未完成的情况下快速解锁资金的方式。结果证明，这个功能非常低效，而且大幅增加了代码库的复杂性，因此再次被去除。

  • 去掉了 Matrix room —— Bespin 版本不再使用 Matrixroom。这有助于减少 Matrix 服务器上的负载，并增加了传输层的可靠性。此外，这还提高了 Raiden 的隐私性，因为通道余额只对 RSB（Raiden 服务包）和通道伙伴可见。

Bespin 版本的安全措施

Bespin 版本依然使用 Alderaan 版本的智能合约。这就意味着我们采取了相同的安全措施和限制。我们已经决定保留这些措施，因为 Bespin 版本只是 Raiden Network 的 beta 部署。我们希望通过降低风险来减少因漏洞或软件滥用而造成的潜在损害，并为这项新兴技术提供可靠的实验环境。请注意，Bespin 版本还没有经过第三方审计。

正如 Alderaan 的博客文章解释的那样，每条通道最多可存入价值 1000 美元的 ETH 或 DAI，而每个代币网络最多可存入价值 100 万美元的代币。但是，由于 ETH 的价格相比 Alderaan 版本发布时上涨了约 15 倍，WETH 代币网络的价值也随之上涨了 15 倍。因为 DAI 是稳定币，所以 DAI 网络的价值不变。

由于 Bespin 使用与 Alderaan 版本相同的智能合约，只有 WETH 和 DAI 代币网络依然可用。

漏洞赏金计划

针对 Alderaan 智能合约的漏洞赏金计划仍在进行中。Bespin 依然使用同样的合约，也就是说赏金池的价值高达20 万美元。如果你想了解更多有关漏洞赏金计划的信息，可以阅读这篇文章。

弃用 Alderaan

正如上文所述，虽然 Bespin 使用的还是之前为 Alderaan 部署的智能合约，但是 Bespin 并不兼容 Alderaan 的通道。这就意味着，用户应该在升级至 Bespin 之前关闭并结算通道。

Bespin 版本的功能局限性

Bespin 版本的功能局限性包括：

  • 代币网络不可升级：Bespin 版本不支持智能合约升级。换言之，升级网络的唯一方法是重新部署新的合约并发布针对新合约的新的客户端版本。旧版网络中的所有通道都需要关闭并在新版网络中重新开启。
