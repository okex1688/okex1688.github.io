NFT的范式出现新的演化方向

NFT 的正式概念虽然是在2017 年提出的，但是基于 NFT 的类似概念和应用却是在更早之前就出现，了解 NFT 的历史演进将更有利于我们了解 NFT 的范式及其演化方向。

NFT最初的设想与尝试起源于2013年的Colored Coins方案，该方案的思路是对比特币进行标记，使得这类被标记的比特币在被追踪到之后可以有其他的用法，如优惠券、数字收藏品、数字凭证等。NFT的“雏形”以“会员权益”为内核开始展开。

在此之后，社区针对区块链在游戏领域的探索进行了大量讨论，其中一部分社区成员开始把目光集中到了艺术品身上。从此，区块链开始与链下的实物资产开始产生了密切的联系，“实物资产所有权”上链，成为了NFT发展的一个重要分支。这其实也与“会员权益”的理念密不可分。英国的一个匿名行为艺术家把他的一幅作品烧掉了，然后在链上的同时锻造了这个作品的数字化代表去拍卖链上的数字化代表。《Beeple | The First 5000 Days》把过去5千天的作品创造拼接在一起，变成完整的一个画作，拍出了7千万美金的高价。这样的形式本质上是一种传统的逻辑和新的NFT的逻辑的结合。

“链上原生”、“去中心化”则是另一个NFT的核心要素。在2013年在2015年期间，区块链社区针对链上的“卡牌资产”进行了探索，当中涌现出了不少与游戏相结合的方案。不过游戏并不是当时区块链核心发展方向，而且对于通证的“唯一性标准”还未推出，这些项目并没有造成巨大的影响力。

2017年推出的CryptoPunks是社区共识下的收藏品。艺术品上链是一种弱形态，真正强形态是完全原生的在链上发生的并且审美和链下的审美是完全独立的强形态，比如CryptoPunks，比如Bored Ape Yacht Club完全是链上算法生成的图片，很难说它是美还是不美，甚至无法界定它是不是艺术品，但是是一种新的价值体系。有很多人愿意为这样的价值体系付出非常高昂的成本。，

以上的NFT都是“自上到下”的模式，即项目方设计NFT的形式或权益，社区成员在这个主题之下进行创作。用户想获取这类型的NFT往往也需要支付一定的资金。

第三种是自下而上的社会性实验。像是《头号玩家》或者《失控玩家》电影里的场景，但是那样的场景中都是美伦美奂的视觉动画，很可能元宇宙的搭建不是由一个超级聪明的程序员最后开发出一个美伦美奂的游戏，而是全世界各种各样的聪明人用像素化或者单词型的元素开发，逐渐像乐高一样把它拼接起来。Loot就有这样的尝试，每个NFT只有8个词，比如一个巨人，一个丝绸的手套，龙的王冠等，每个人可以自己锻造一个，会有人把这些道具具像化描绘出来，有人做地图和故事情节，甚至最后把它变成一个完整的游戏。现在有不少的视觉化版本，如果把NFT地址输入到某个网站会有一个诠释，你会看到一个具像的东西。未来的元宇宙很有可能是这样的模式，在链上是非常基础的元素的表示，但是在链下在每一个软件中会有具体的诠释方式。

NFT的发展史是逐渐把创造力和自由度交还给用户的一个过程。

NFT的演化是区块链生态更成熟的标志

在金融资产出现之前，现实世界中所有的资产都是非标准化资产。为了方便交易，人类经历了用相对统一的贝壳、金属制品、纸张作为货币（标准化资产）的演进历程。其他的资产交易也采用标准化资产：先将资产打包，然后切割成可分割的同质化资产后再放到市场中供投资者买卖。资产证券化是实现交易的重要方向，所以我们看到，在传统金融行业每一次金融创新，资产标准化是永恒的主题。资产的发展方向，遵从复杂到简单。

区块链行业的发展脉络与此恰好相反。最初诞生的数字货币之间其实没有区别，这种特质非常有利于交易，比特币就是为了去中心化支付而出现。区块链行业在短短10多年的时间内出现了好几轮泡沫，便利的交易性是助推泡沫形成的重要因素。

NFT资产的“非标准化”特质不利于交易，不过这仍然不妨碍NFT资产和范式的发展。随着用户在NFT创作链条中占据越来越重要的地位，NFT的“非标准化属性”会越发明显，因为会有大量长尾端的中小型社区会产生大量NFT资产，这些资产的出现在某种程度上增加了区块链行业中的交易层面的难度，但资产类型复杂维度的增加，反过来说明用户对于资产的接纳度提升。未来，针对各种NFT资产的“标准化”的探索可能会非常精彩，各种“标准化”创新很可能是DeFi领域具有里程碑的事件。

NFT是Web3.0多元宇宙的重要组件

回顾Web1.0至今的发展脉络，与NFT的发展历程有较多相似之处。在Web1.0时代，公司设定内容的主题，提供相关的内容，用户往往需要付费才可以使用部分有价值的内容，公司赚取收益；在Web2.0时代，用户的自由度提高，UGC的内容开始逐步兴起，互联网公司扮演“平台”的角色，平台开始与用户分享收益；

Web3.0以NFT作为价值载体，用户成为内容、价值产生和捕获的核心，平台在这个过程中捕获的利益和对于内容的控制力将大幅降低。

在这个阶段，会存在“中心化、有准入”的元宇宙，公司制不会消失；也会存在“去中心化、无准入”的元宇宙，社区成员会成为这类型元宇宙的建设者。第一种元宇宙会由公司编写“故事线”，搭建世界观，用户在里面更多的是体验，以及进行微创作。这类型的元宇宙在影视行业最多，《哈利波特》、《星球大战》等拥有大量粉丝的电影都构建了庞大的元宇宙。第二类元宇宙是原生于区块链的元宇宙，用户对于世界观、故事线可以有很大的影响和改动，单个社群中具有巨大影响力的KOL可以对世界观的最新进展按照他的意愿进行修改。

NFT作为Web3.0的重要组件，有机会自由穿梭在这两类型的元宇宙中。比如在“哈利波特”元宇宙的魔法杖，可以代入区块链原生的其他元宇宙中，其属性、功能都可以在不被修改的状态下使用。由于公司需要捕获元宇宙中的部分价值，（比如Facebook更名为Meta，聚焦建立元宇宙），部分元宇宙可能具有较强的“中心化”属性，那些装饰、社交属性的NFT可能会是最容易穿梭在两种元宇宙中的资产，有功能性的NFT可能会因影响“中心化、有准入”元宇宙的世界观而受到阻碍。

多元属性、以用户为核心将是NFT的未来

NFT归根到底是一种资产标准，站在现在来看NFT的发展，我认为未来NFT发展有2个关键词——“多元属性”和“以用户为核心”。

首先，NFT种类会更加丰富。不论是现实资产映射，还是原生于区块链的NFT，还是将各种权益通证化的NFT，未来NFT市场会变得非常多元。同时，NFT会拥有横跨多元宇宙的多元属性。试想哈利波特魔法棒在《权力的游戏》的宇宙中将碰撞出怎样的火花？我们所拥有的NFT将会更容易穿梭在多元宇宙的web3.0时代。

其次，NFT资产的“用户”属性会更加强烈，NFT资产基于社区，个人创作是未来的大势所趋，把创作权和收益权交还给用户，将回到互联网和区块链“去中心化”的本心。

如果把NFT看作是一场数字经济潜力验证的实验，也许任何参照物都不足以与之对照。随着人类对数字资产所有权和产权的需求不断上升和发展，我们将亲身见证由 NFT 引领的Web 3.0美丽新世界。
