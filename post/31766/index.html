一切都是从Facebook改名“ Meta ”掀起元宇宙热潮开始，国内外的大公司疯狂蹭热点，加密领域的元宇宙概念 token 也随之暴涨。

欢呼雀跃的背后，我们应该思考科技巨头的中心化元宇宙，真的能带领我们去往向往的元宇宙吗？

Facebook 虽然没有披露其元宇宙的细节，但是有一点可以肯定，他们一定会围绕自己发布的 Diem 货币为中心的经济体，应该会在 Diem 网络上发行 NFT 。

那Diem 是什么？Diem 前身为 Libra，是一种经许可的中心化公司网络运营的货币。

从某种意义上来说，Diem 网络上的每一笔资产都经过机构审核，并和现实身份进行绑定，也就意味着你在 meta 元宇宙中的资产也是中心化的。

举例来说，Facebook可以一秒冻结你的资产，并完成转移，这和目前的现实世界没有区别。

这种“围墙”的方式适合大公司，毫无疑问，短期会涌入大量的传统用户，数据肯定也比去中心化的项目好看很多。

但其实用户就已经对 Facebook 的中心化主导地位开始感到厌倦， NFT 市场 Ethernity Chain 的首席执行官兼创始人 Nick Rose Ntertsas 所指出的那样

“在由大流行推动的数字化转型中，加密货币的采用率增加了五倍。与此同时，全球的民意调查显示，人们对中心化技术平台的不信任感越来越强，并且对加密和区块链在保护隐私、实现点对点交易以及支持透明度和不变性方面所提供的本质的评价也越来越高。”

那去中心化的元宇宙项目的优势是什么？是可以在开放、无需许可的区块链架构上运行。

任何开发者都可以在开放、无需认可的区块链上构建项目，而且任何用户都可以获取自己的虚拟房地产并参与虚拟资产。

去中心化、开放式架构的最大好处之一是用户可以在不同的元宇宙之间无障碍地加入和移动，资产包括加密货币、稳定币、NFT或其他任何东西都可以跨链转移，所有权在自己的手里，这才是真正的资产。

但考虑到 Facebook 的用户群规模，它一开始肯定会对去中心化产生重大影响，毕竟中心化的机构有用户还有钱。

也许，去中心的平台可能会陷入与 Instagram 和 WhatsApp 相同的命运，并作为 Meta 收购狂潮的一部分被吞没？

Sandbox 的联合创始人兼首席运营官 Sebastien Borget 认为“通常情况下，大型科技公司会袖手旁观，而新进入者会为市场份额而战。尤其是这种策略只有在初创公司出售才有效。

所以必须有不同的经济激励，这正是 Web 3.0 如此强大的原因。它使平台和用户保持一致，以构建一个独立的平台，而用户会对其治理拥有所有权——并最终取得成功。 ”

正如 Libra 引起了世界的轩然大波，最终却被监管机构压制，Facebook 享受了中心化的红利，也被中心化世界拖累，而束手束脚。

总体而言，Libra 给去中心化带来了积极宣传，但中心化 VS 去中心化在未来必然一战。

因此对于去中心化的元宇宙项目来说，在中心化公司还被政策影响的时间里，提供比中心化更好的技术与服务，重中之重。
