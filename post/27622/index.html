NFT 保险库101

NFT 保险库是以太坊地址的比喻，该地址的设置是为了安全地促进 NFT 的长期存储。

创建自己的 NFT 保险库是没有一刀切的解决方案的，因为任何保险库都要依赖资源和防御策略的组合。但是，基本的 NFT 保险库有一些共享的核心元素。 这些元素包括：

多个备份

假设发生了难以言说的事情，你的房子被烧毁了，要带走你的 NFT 保险库私钥的最重要的纸质备份。确保在其他地方有多个备份，从而确保在面临灾难或更危险的情况时，你的 NFT 是可以找回来的。

隐私性

你的保险库只有当私钥是安全的时候才是安全的。因此，建立一个 NFT 保险库系统，最重要的的一点是只有你和你信任的人才能访问存储的私钥备份。

可访问性

可访问性带来了一定隐私方面的困难，是的，但在 NFT 生态系统里，5 天内可以发生很多事，5 年以上更是如此。尽管你的 NFT 保险库得到严谨的设置，有足够高的可访问性，随时可以交易和转账， 提供非常好的灵活性，但你可能不需要经常与你的 NFT 保险库互动。

创建 NFT 保险库的建议

如果在家存储 NFT 保险库私钥的备份，请用防火防水的存储容器。这将保护你的密钥备份不受基本的洪水、火灾事故的影响。这些容器也适用于其他备份，例如你在亲戚家的备份。

只将保险库密钥备份作离线存储，且置于有非常强的安全保障的地方。最起码，租用一个保险箱是一个最低限度的可行选项。

使用像 Ledger 和 GridPlus 这些供应商的硬件钱包作为你的保险库的安全终端。这些专门的“冷”钱包减少黑客使用的一系列攻击向量，以从在线“热”钱包窃取 NFT 的风险，也使得与你的金库进行安全交互变得更容易。

考虑使用多签解决方案，例如 Gnosis Safe，这样任何输出的 NFT 交易都需要多个地址的确认。

考虑使用一个 ENS 域名作为你的保险库地址，例如“yourvault.eth"。然后你可以进入 ENS App，把你的 ENS 域名与你的电子邮箱、推特个人信息等连接起来。这会使得你自己与别人在和你的保险库交互起来更方便，使得想要进行 NFT 交易的人梗荣联系到你 (请记住，仅通过 P2P dapp 进行交易。
