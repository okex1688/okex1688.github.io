GameFi 虽然 Game 在前，但紧随其后的 Finance 才是支撑其迅速发展的主要原因。

GameFi 仍处在早期，相比传统游戏可玩性相差甚远，但由于可以帮玩家提升盈利能力而受到青睐。

游戏中玩家的获利主要来自于 NFT 交易，以及对游戏 token 的挖矿和交易。因此了解 token 的经济模式对玩家尤为重要。

Token 的经济模式

GameFi 中玩家的目的为实现利益的最大化，游戏中 token 的模式将影响玩家的行为。可以说 token 将游戏、NFT 和金融属性链接起来。一个可持续、公平且具有吸引力的 token 模式将对游戏的成败有着重大的影响。

目前游戏中 token 模式主要分为单币和多币。

单币

单币的经济模式即使用一种 token 处理平台相关的所有事务，如在游戏内的活动、平台治理以及在二级市场的交易。

好处在于经济模式简单，玩家只需获得一种 token 便可体验游戏，获得的收益也为相同 token。它方便了玩家交易，降低了资源整合的难度。

但劣势也很明显，由于游戏中消耗的 token 也是玩家想要持有增值的资产，玩家想要持有的意愿与游戏中不得不消耗 token 才能继续推进的设定发生了矛盾。

玩家很容易因为币价的波动而影响在游戏中的操作。币价过低时，玩家会对游戏的收益性产生动摇，失去继续游戏的兴趣；币价较高时，继续游戏会消耗掉高价值的 token，损失玩家的资产。

此外，如平台采取不断增发 token 的模式，会造成通货膨胀。由于单币模式很容易对 token 自身生态造成影响，因此更多的游戏项目更愿意采取多币的经济模式。

多币

多币模式的游戏一般会区分平台治理 token 和游戏内流通的 token。

  • 平台治理 token，主要用于鼓励玩家对社区治理做出贡献从而提高玩家对平台价值的认可程度。

  • 游戏内使用的 token，主要用于游戏中的挖矿、交易、日常任务、升级等。通常不设置最高发行量，有的甚至不需要在链上成为虚拟货币，不会在二级市场上进行交易。

多币模式根据不同的游戏设定有多种形式，相对于单币的经济模式也有着明显的优点和缺点。

优点主要体现在可以稳定游戏内的经济系统。多币模式可以让玩家在游戏中不受主要 token 在二级市场价格变动的影响。与游戏中进行交易的 token 分离，也可以让游戏方更加灵活地调整不同 token 的经济模式。

多币模式的劣势在于提高了游戏方的整合难度，对于玩家也增加了学习成本高。

经济体系中需关注的维度

知道了游戏中 token 的形式后，更为重要的是要关注平台内经济体系如何运行。可以通过 token 的以下角度进行评估：

发行量与销毁机制

合理且有限的发行量可以防止 token 的通货膨胀，不限量的经济模式可能引起 token 贬值，让玩家失去长期持有的信心。

销毁机制同样对防止通货膨胀有着重要的作用，积极的销毁机制有助于稳定币价。如 2 月 4 日 Axie Infinity 宣布在每日任务和冒险模式下不再发放 SLP token 奖励，又增加了更多的 SLP 销毁机制（包括装饰皮肤、升级和繁殖 Axie、添加表情符号和重命名等）。这一机制的上线使一路下滑的 SLP 价格发生了回升。 

 Footprint Analytics - Axie Infinity Token Price

分配机制

团队对 token 分配的方案也值得关注，更多比重在社区的分配机制能够获得玩家更多的偏爱。

用例

更广泛的实用性对玩家有着更大的吸引力。大部分游戏的 token 都有在游戏中进行日常任务、升级、购买、对战等功能，有些还可以通过抵押 token 获得额外 NFT 或 token 奖励。延伸到游戏外部的用例将会为 token 增加更多赋能，如在二级市场的交易，在其他平台集成获得收益等。

解锁时间

设定获得 token 后的解锁时间有利于防止投机用户影响币价，也可以更多的留住用户。同时保持项目团队与玩家有相同的锁定期也有利于使两者利益保持一致。

Gaming Token 的数据表现

根据 Footprint Analytics 数据显示，GameFi token 按市值排名前 5 的项目分别是 Decentraland、Axie Infinity、The Sandbox、DeFi Kingdoms 和 Illuvium。其中从第四开始已与前三有近 10 倍的断层差距。

 Footprint Analytics - GameFi Token by MC

老牌 GameFi 游戏 Axie Infinity 仍在前三，但第一的地位已被撼动。随着元宇宙在各界讨论度的飙升，以元宇宙为概念的 Decentraland 和 The Sandbox 在 10 月迅速暴涨，到 2 月末时前三的差距可谓只差毫厘。

 Footprint Analytics - Top 5 Market Cap in Gaming

良好的用户体验与可玩性也至关重要，这体现在 Illuvium。虽未正式上线，但仅通过已公布的游戏展示预告片就让玩家震撼，精美的画面和丰富的游戏人设对于现在普遍粗糙的 GameFi 项目是降维打击。

Illuvium 虽在 12 月平台因遭受攻击和合约漏洞发生了损失但仍排名前 5，可以看出当前玩家对游戏质量仍有较大的渴求。

DeFi Kingdoms 不同于其他前 5 的平台，它的主要核心是 DeFi，游戏属性只是锦上添花。 这种像游戏一样有趣的投资体验让 DeFi Kingdoms 的 token JEWEL 上线后一路上升到市值第四名。

结语

随着越来越多的资金进入 GameFi 领域，现在并不缺少 GameFi 项目，缺少的是从游戏质量到 token 经济体系都优秀的项目。

与成熟的传统游戏相比，加密世界的游戏还处于“孩童”阶段。但其金融属性和高社区参与度让它在游戏领域的红海中开拓出一片新的蓝海。

游戏方通过 gaming token 完善游戏内经济体系，与提升游戏体验同样重要。不论是为了 P2E 还是单纯持币待涨的用户，进入前对 gaming token 有足够的了解也是一项必修课。
