以太坊2.0的到来应该会让以太坊区块链更实用：更好的安全性、更快和更便宜的交易，以及更少的关于这些交易是否完成的烦恼。简而言之，它将允许网络扩展。

几个月来，去中心化应用程序 ( dapp ) 用户一直依赖Polygon解决网络拥塞问题，这是一种位于以太坊区块链之上的所谓layer 2解决方案。早在4月和5月，该项目就以似乎无休止地攀登和合作伙伴关系占据了新闻头条，因为它为天文数字的交易需求和费用价格提供了解决方案。

但是现在以太坊的伦敦硬分叉出现在后视镜中，开发者正朝着权益证明Eth2网络前进，Polygon的最终目的地是什么？它能否证明不仅仅是堵塞网络的临时修复？

Polygon的作用

Polygon之前被称为 Matic Network，是以太坊的可互操作扩展解决方案，这意味着它可以与多个不同的区块链通信，并帮助缓解网络拥塞。它为dapp提供了多种扩展方法，以便它们可以更快地处理交易。这是去中心化金融 ( DeFi ) 应用程序的一个重要功能。虽然DeFi允许人们在没有中介的情况下进行借贷和交易，但不合时宜的交易可能会导致他们损失数千美元。

扩展解决方案可以分为两大类：rollup和侧链。Rollups位于以太坊区块链之上，共同处理一系列交易而不是独立处理，然后将它们作为一个交易提交回主链。不同版本的rollup——optimistic、plasma、validium和ZK-rollup——以不同的方式捆绑和验证这一系列交易，尽管它们都是将一系列交易“rollup”成一个以提交给主链。

相比之下，侧链本质上是以太坊主链的克隆，交易和资产在主链中移动和处理，并且可以在必要时回调到主链。

目前，Polygon提供了每种扩展解决方案之一：Plasma rollups或其Matic权益证明、与以太坊兼容的链。虽然rollup和侧链都是扩展解决方案，但它们提供了不同级别的安全性。前者利用以太坊网络的安全性，而独立侧链则使用自己的安全方法。

Polygon在以太坊2.0中的地位

对于一些人来说，诸如Polygon之类的rollup和扩展解决方案是以太坊2.0即将到来之前的临时解决方案，以太坊2.0是备受期待的网络从工作证明到权益证明的升级，这应该同时减少网络拥塞并提高安全性。它使用一种完全不同的扩展方式：分片。

简而言之，分片是一种扩展解决方案，它将区块链分成一系列不同的部分，以允许网络划分工作。一个分片验证一种动作，另一个分片验证另一种动作。这种结构将为网络提供管理不同行为的能力，例如游戏的链上交易。

但仅仅因为分片即将到来并不意味着没有其他扩展解决方案的空间。

就其本身而言，分片权益证明可以将以太坊从目前的每秒15笔交易(TPS)的速度提高到大约3,000 TPS。（对于那些想要进行数学计算的人，仅权益证明就可以带来50 TPS的速度；将其乘以64个分片。）这非常好，超过了Visa通常处理的1,700 TPS左右。

但是通过结合其他扩展解决方案，以太坊可以达到每秒100,000 笔交易——不仅可以用于点对点交易，还可以用于基于区块链的视频游戏、NFT拍卖、和DeFi协议中更密集的交易。

Polygon的下一步是什么？

Polygon本身也没有停滞不前。该网络有一个雄心勃勃的路线图，包括optimistic rollup、ZK-rollup和Validum链的实现。

ZK-rollups将许多链下转账捆绑到单笔交易中，并使用ZK-rollup作为以太坊主链上的最终公共记录。Validium链也使用ZK-rollup，但将数据存储在链下。与此同时，Optimistic rollup使用“欺诈证明”，其中在以太坊链上提出索赔，并有一个挑战期，使其他运营商能够对链上的索赔提出异议。

Polygon正在迅速采取行动，将这些解决方案上线。在2021年8 月签署的2.5亿美元交易中，Polygon与开源 ZK-rollup 扩展项目Hermez合并，将其纳入Polygon保护伞，这是“一个区块链网络首次全面合并到另一个区块链网络”。Hermez的HEZ代币持有者可以选择将他们的HEZ换成MATIC。

以太坊创建者Vitalik Buterin多次表示网络将需要plasma、optimistic rollups和ZK-rollups，如果Polygon能够履行其“即将推出”列表的承诺，那么在ETH 2.0中找到一席之地应该没有问题，甚至可能是它的核心。

让以太坊负担得起

这将使以太坊区块链更实惠。在他7月8日的文章《我担心没人会关心Rollups》中，Dragonfly Capital合伙人Haseeb Qureshi比较了价格。他写道，“现在在Polygon上，一个简单的Uniswap式交易成本为0.0001美元。在币安智能链上，成本为0.20美元。在以太坊上，它的成本约为7美元。而在Optimism（以太坊的Rollup扩展解决方案）上，它的成本约为 0.68 美元。”

这表明，在BSC或Polygon等evm兼容链上，交易成本将低几个数量级，尽管这种权衡将需要牺牲安全性。即使在更具安全意识的Rollup上进行交易，其成本也应该只是今天的一小部分。

随着未来几个月ETH 2.0的到来，Polygon计划扩展其工具集，应该会继续在DeFi的前沿占有一席之地。
