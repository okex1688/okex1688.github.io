原文标题：《好的稳定币、货币协议和数字钱包：修复我们损坏的支付系统的公式》

为什么货币流动的方式必须改变

最近在美国有很多关于稳定币、加密货币和其他数字资产的对话和辩论。我很高兴看到讨论获得了动力，尤其是现在。因为改造我们破碎的支付基础设施从未像现在这样紧迫。

疫情的大流行加速了全球数字经济的扩张。它引发了人们购买方式、购买地点以及他们发现企业和与企业互动的方式的变化。它促使许多家庭更加依赖海外汇款作为重要的经济命脉。而且这种趋势将继续下去——全球数字交易的百分比预计将从疫情爆发之前的 57% 上升到 2025 年的 67%。这一切都使得企业和政策制定者比以往任何时候都更加重要，以帮助确保人们不会落后。

然而，在解决这些挑战时，仍然存在许多障碍。在这个关键时刻，美国应该引领变革——但现在我们处于次要地位，让中国等国家带头。

在领导 Facebook 团队的过程中，我也遇到了这些障碍，团队在过去两年多的时间里一直在开发 Novi 钱包——一个可互操作的数字钱包，它将使人们，最终是小企业，能够转移资金以快速且负担得起的方式在国内和国际上销售。

在这段时间里，我就我们的选择进行了多次讨论，以及为什么我们如此关心用稳定币来构建 Novi，而不是用法定货币，或者换句话说，我们今天都熟悉和使用的政府发行的货币. 我想花时间解释为什么这对我们很重要。

稳定币如何帮助解决问题

如果我们只在 Novi 提供法定货币，它会给人们带来很多价值。我们可以提供国内和跨境支付，这将使 Facebook 受益，因为我们将创建大量的钱包账户。反过来，这些将使我们能够通过向商家收取有竞争力的费率以接受来自 Novi 客户的付款，从而建立稳固的商家服务业务。那么为什么不这样做并且制作成功呢？好吧，我们可能会。但在我们这样做之前，我坚信如果有机会在互联网上创建一个开放的、可互操作的货币协议，并真正改变世界各地的人们和企业的游戏规则，那就是现在。我们作为Diem 协会的成员参与的事实，以及在其他方面，可以帮助更多公司围绕一个标准，我不希望我们浪费我们的精力。

虽然最近有很多关于稳定币及其在支付中的作用的讨论， 稳定币本身并不能解决任何问题。为了释放它们的潜力，它们需要与更便宜、更快、更安全、可互操作和可编程的底层支付网络相结合。一种比当前陈旧的系统更开放的系统，后者将太多人抛在后面，并且已被证明对创新具有抵触情绪。事实上，自 60 多年前创建第一个信用卡网络以来，当前的支付轨道很少能看到有创新的新技术了。

重要的是要了解并非所有稳定币都是平等的。一个设计良好的稳定币的定义是其储备的设计和管理方式，这些储备对消费者和监管机构的透明度，以及发行人提供的消费者保护和合规模式。一种设计精良的稳定币需要在美国银行和短期的国债中始终持有 1:1 的现金储备，发行人持有资本作为缓冲。可以说比美国任何钱包中持有的法定余额提供更好的消费者保护。事实上，如果银行的所有客户都想在短时间内提取所有余额，即使是在有偿付能力的银行，银行将其长期资产转换为现金的提取也会延迟。有了设计良好的稳定币，这种风险就不存在了。

在反洗钱、反恐融资、制裁和税收合规方面，精心设计的稳定币及其钱包生态系统有可能改进这些控制的传统实施方式。根据定义，数字资产是匿名的，这是一种误解。以正确的方式构建和配置稳定币和钱包，将客户尽职调查置于其合规性方法的中心。当稳定币的控制被设计为与支持它们的单个钱包的控制协同工作时，更有效地检测和报告非法活动的潜力将被证明优于通常对交易进行后处理、筛选的现有系统在他们完成后很长一段时间。这种方法为更有效地阻止金融犯罪释放了开创性的潜力。与许多观点相反，在网络和钱包层面具有强大控制的稳定币为该领域的创新提供了巨大的机会。

什么时候让最好的想法引领潮流

然而，在美国本土，我们的支付基础设施可以说是世界上任何发达国家中最差的，而且越来越落后，而中国正在坚定地建设支付渠道的基础设施，使数字人民币成为挑战者美元作为世界储备货币。根据2019 年美联储的支付研究报告，2018 年，自动清算所 (ACH) 支付占美国非现金支付价值的 66.1%。ACH 是在 70 年代初构思的，付款仍然需要最多三天的时间才能完成。与此同时，商家在美国接受消费者支付的交易费用是欧洲的十倍。

在我们构建 Novi 的整个过程中，我们与许多被当前系统排斥或服务不足的人进行了交谈。听到许多人目前没有资格拥有银行账户，我感到震惊——事实上，他们中的许多人以前都有过账户。他们认为自己负担不起额外的费用，宁愿多支付 10% 来兑现支票，也不愿被他们无法支付的费用所蒙蔽。这怎么能被所有人接受？如果我们想让人们振作起来，让他们有机会茁壮成长，那么帮助他们保护和转移资金的系统就应该是可访问的，并以低得多的成本、以非歧视性的方式为他们服务。

这就是我们每天来 Novi 工作的原因。我们相信，我们可以帮助许多值得更好的人改变事情。为了产生最大的影响，仅使用法币构建封闭系统并不会削减它。我们需要抓住机会帮助建立一个开放、低成本、可访问、近实时的生态系统和网络。这是我们在 2019 年夏天为 Libra 制定的愿景。

在我们宣布 Libra 之后，我们公开承诺，如果没有获得必要的监管批准，我们不会在 Libra（现在的 Diem）上发布 Novi，并且我们将在开发产品时与监管机构、政策制定者和专家进行接触。我们一直恪守这些承诺，并与世界各地的监管机构和政策制定者进行了建设性的磋商。在美国，我们几乎在每个州都获得了 Novi 的许可或批准，我们不会在尚未获得此类许可的地方启动。Diem 协会已成为一个独立实体，与美国和全球监管机构进行自己的对话。它解决了在设计和构建具有广泛消费者保护的高质量稳定币的过程中提出的所有合理问题，

尽管如此，在这段旅程中，我反复听到各种不同的论点，即支付和金融服务行业不应让 Facebook 成为这些创新的一部分。

首先，Facebook 已经是支付行业的参与者，而且已经有一段时间了。我们于 2009 年启动了 Facebook Payments，在过去四个季度中，我们实现了超过 1000 亿美元的支付额。我们在美国和国外拥有大量的蓬勃发展的点对点支付产品，并且当我们受到适当监管时可以做更多事情。160 多个国家/地区的人们和企业使用我们的平台以 55 种货币进行付款。人们可以在我们的应用系列中购物、进行游戏道具的购买、捐赠和购买活动。

其次，我发现这种思维方式非常的非美国。我来到这个国家，并因其价值体系成为一名自豪的公民，有机会制造改善许多人生活的产品，并为我的家人创造美好生活。我们国家价值体系的核心原则之一是对公平竞争环境的承诺。任何为问题提供可靠解决方案的人或公司都会有公平的机会。尽管 Facebook 在重建信任方面还有更多工作要做，但它也始终如一地为涉及类似重要服务的消费者提供了巨大的价值。

所以我相信我们应该得到公平的对待。我们应该改善对大多数人来说持续存在的一些状况方面发挥关键作用，而美国的做法是让更多的竞争和创新能够打破数十年来停滞不前的僵局。此外，在美元的相关性和我们的全球影响力受到前所未有的挑战的时候，我们应该利用我们一些最成功公司的技术和能力来赢得这场针对我们国家利益的无声战争。

让我也谈谈规模问题。由于我们的规模庞大，需要进行额外审查，这一点我表示理解并接受。鉴于 Facebook 的影响力，一些人认为 Novi 将立即被全球数亿人使用。正如从我们推出和扩展其他支付体验的经验中知道的那样，这不是它的发展方式。无论使用 Facebook 多久，人们都必须做出肯定的选择才能开设 Novi 帐户。作为创建新帐户的一部分，Novi 需要对客户进行全面的尽职调查，包括通过上传政府颁发的 ID 进行身份识别，因此新客户的入职需要时间。在获得适当的许可并遵守当地法规和标准之前，Novi 也无法在特定司法管辖区开展业务，因此我们将在少数国家/地区开展业务。

我也听说过对我们最终意图的担忧。论点是：_“显然你们是想从中赚钱，你不会出于好意而这样做！”_我不能责怪人们有这种想法。但这种说法源于对科技行业的误解，以及大多数非常成功的科技公司所拥护的精神。我们首先专注于为人们大规模解决重大问题。最终，找到了通过货币化维持工作的途径。在这种情况下，我们的商业模式非常清晰。我们是支付行业的挑战者，我们将为使用 Novi 钱包的人提供免费的国内和国际的点对点（个人对个人）支付。我们相信人们会更喜欢免费且更方便的服务，而不是那些价格更高且不以消费者为中心的服务。一旦我们拥有了一个有意义的客户群，我们就可以向世界各地的企业提供更便宜的商家付款，并且仍然可以从商家服务中获得利润。

货币的未来走向何方

最近，我一直在思考扎克伯格（Facebook的首席执行官）对元宇宙（Metaverse）的愿景。在 Metaverse 中思考金钱的流动实际上对我们如何设计我们的产品和支持它们的底层基础设施非常有帮助。毕竟，如果今天为纯数字世界发明了货币和支付系统，它们会是什么样子？它们肯定不会像我们当前的基础设施。我们需要一个全球性的、开放的、可互操作的、近实时的、廉价的、合规的全球货币协议，以使人们、创作者和企业能够无缝地转移资金并通过可编程货币进行创新。钱包需要支持 NFT。真正的合约和头衔将被智能合约取代。虽然这听起来可能与我们想要服务的人——那些被当前系统抛在后面的人——所经历的现实脱节，但实际上并非如此。

改变早该发生了。Novi 钱包目前已准备好上市。它受到监管，我们对我们的运营能力充满信心，以超越我们对合规性的高标准要求。我们认为延迟提供更便宜、可互操作、更易于访问的数字支付的好处是不合理的。我们将继续坚持并证明我们可以成为这个行业中值得信赖的参与者，并且通过参与其中，为全世界带来积极的变化。
