金色财经报道 区块链2月13日讯    NFT 作为一种新的数字资产类别，过去常常被当作嘲笑的对象，类似这样的嘲讽简直如雷贯耳：“你竟然花1万美元买了NFT？看看，我完全可以不花一分钱对其进行截图。” 为了回应这种嘲讽，人们不得不对区块链技术进行一番解释：区块链上的智能合约赋予了数字资产可编程性，使其超越了单纯的数字文件范畴。但现在，NFT已经成为加密行业里最重要的“一员”，那么从被嘲讽到不容忽视，NFT是如何完成这一逆袭的呢？                                           

尽管加密行业为解释NFT的巨大潜力做出了不少努力，但人们对于整个 NFT 市场的印象往往还停留在“一时的狂热“。然而，就在一年过后，人们担心的昙花一现却并没有发生。恰恰相反的是，2022 年首月，NFT以极为亮眼的成绩超越了此前的所有记录，如下图所示（资料来自The Block）：

 

令人大跌眼镜的是，在加密货币熊市中，比特币在上个月下跌了 39%，而NFT 交易量却节节攀升。所以问题来了，那些嘲讽NFT 的人是否代表了正确的观点，我们现在看到的只是一个周期较长的泡沫？还是他们的观点完全不正确，因为他们根本不了解 NFT 是什么？

揭开 NFT 价值之谜

2017 年，也就是NFT 出现之初，人们非常不看好NFT，认为这只是一时的“狂热”，持续不了多久。然而接下来，CryptoKitties的出现在NFT领域激起了不小的水花，当时该项目中的数字卡通猫的售价已超过了 2 万美元。不得不说，CryptoKitties 在 NFT 从一个笑话变成不容忽视的资产道路上起到了关键性作用。毕竟，如果有人肯为虚拟所有权花费重金的话，那么这个数字世界中的新门槛就被创造出来了。

由于 NFT 具有稀缺性、新颖性和增值性的特点，因此它们赋予了持有者一定的社会地位，这其实也是一笔无形的资产。然而，CryptoKitties 之所以能创下4750万美元的销售额不仅仅是因为其数字收藏品的属性，更重要的还兼具实用性。通过智能合约创建的罕见算法，CryptoKitty NFT可以进行繁殖，并在繁殖过程中随机产生某些稀缺性特征，区块链技术确保了这些特征是完全独特的，且无法复制。

换句话说，也就是NFT 的可编程性不仅仅体现在将Pokemon之类的卡片转变为数字版本，而是创建了一个动态的、自我进化的、以及自动化的生态系统。在这个多层次的领域中，CryptoKitties作为一个优质的“创始”项目，最初几代的CryptoKitty NFT必将会衍生出巨大的需求。

假如通过算法繁衍出的小猫具有独特的外观特征，那么即便是第 9 代的价格也会超出第1代，就像 CryptoKitty #896775，又叫做 Dragon，它的出售价格达到了惊人的 600 ETH 。 

最后结果证明，这些卡通小猫仅仅作为一件视觉作品来看其实并没有什么特别的，EtherRocks （一组以石头为主题的 NFT 收藏品，限量 100 个）就很好地证明了这一点。然而这些 2D 石头图片如今售价已达数百万美元。

这些石头曾以 400 ETH 的价格售出，在当时约合 130 万美元。然而EtherRocks 的实用性受到了质疑，因为它们除了用来交易之外没有任何实际功能。不过，由于它们总共只有 100 个，因此只能说实用性体现在了它们的稀缺性方面。稀缺性抬高了这款NFT的价格，此外社交媒体和其它媒体的大力宣传也起到了推波助澜的作用。

简而言之，人们通过NFT展示了自己的社交影响力，或者说是自己数字化的一面。当我们在审视人类文化时，很快就会意识到这其中大部分都来自于我们的核心属性——社交能力。从我们的衣服、发型，到审美偏好、道德价值观，这些无形的交流方式形成了如今这个社交网络。

实际上，NFT 是将现有社交网络的一部分进行了数字化和通证化。在由社会地位驱动的经济影响下，Twitter Blue 服务现在宣布推出NFT个人头像验证功能，用户将能够从其加密钱包中导入NFT，并将其设置为自己的Twitter头像。用户可以无缝验证所有权并发现关键的NFT信息，包括NFT名称、其原始创建者、其收藏描述和艺术品的其他属性。

同样，Meta公司旗下的 Facebook 和 Instagram 似乎也在推进这项功能，这将能有效地将NFT推广到占世界一半人口的市场中。

超越社会影响力的 NFT

去年 3 月，Twitter 创始人 Jack Dorsey以 290 万美元的价格出售了他的第一条 NFT 推文，与此同时其他 NFT 应用也加入了这场游戏。除了稀缺性、创新性、将历史记录通证化之外，NFT 还创造了数字经济。而电子游戏又为 NFT 搭起建了一个舞台，使其拥有自然进化的可能性。

从《World of Warcraft》到《Runescape》，特许经营费都高达数十亿美元，并且已经形成了自己的经济体系，包括游戏货币、制作、耕种、探索和举办锦标赛等。然而，这个经济体系似乎比较虚幻，因为这些数字资产并没有一个可以被提取出来兑换成真实的货币。

随着越来越多的免费存储应用被应用到加密货币甚至是memecoins中，在不久的将来，我们或许可以看到 NFT 也将具备存储功能。

就像 CryptoKitties NFT 引领了数字收藏品潮流一样，Axie Infinity 引领了区块链游戏（或者称为GameFi）的潮流。为了可以进行场外交易，Axie 将其重要的数字资产进行了通证化：

1、Axies，用于游戏中进行战斗的虚拟生物NFT。

2、Smooth Love Potion (SLP)，用于繁殖 Axies 的游戏货币。

3、Axie Infinity (AXS)，用于投票和质押的游戏原生货币。

4、土地、虚拟空间NFT，用于建立 Axie 繁殖点。

这个通证化的元宇宙推动 Axie Infinity 成为了排名第二的 NFT 平台，仅次于 OpenSea，总销售额达到了 39 亿美元。

回顾过去，只有电子竞技游戏和流媒体才能让游戏玩家赚到钱; 而如今，NFT将边玩边赚 (P2E) 模式带进了每款游戏。谁能想到起初不被看好的 NFT 现在竟然能大展宏图。

数十个由 NFT 驱动的虚拟空间项目正在参与这场竞争，如 Decentraland (MANA)、The Sandbox (SAND)、Aurory (AURY)、Illuvium (ILV) 等。它们争夺着元宇宙的市场份额，而企业则纷纷购买虚拟 NFT 土地作为 P2E 游戏的运营基地。

微软以 680 亿美元收购了动视暴雪（Activision Blizzard ）、育碧（Ubisoft）推出 Quartz NFT 平台。然而这些业界动态只是众多 NFT 热潮中的第一波动作，更多的静态 NFT 收藏品正在为其持有者带来可观的价值。 

1、Bored Ape Yacht Club NFT 的销售额达到了 11.6 亿美元，它为持有者提供了幕后商业交易和社交网络的机会，持有者可以参与该项目的独家活动。

2、职业体育队伍正在将他们的门票通证化，这类门票NFT既可以当作通行证还可以作为纪念品，例如 NBA Top Shots 和 Mark Cuban 的达拉斯小牛队正在做这件事情。

3、将于 3 月 18 日在拉斯维加斯的 Area15 举行的Afterparty使用 NFT 作为门票，这将是世界上第一次使用 NFT 作为艺术音乐节的门票。

以上市场活动反映出了区块链作为一种可编程式数字账本赋予了 NFT 价值，这是非常灵活的做法。如果一个系统可以被数字化，那么它也可以被通证化。无论这个社会中存在着什么活动，NFT 都可以将其从无形化、非货币化转变为可量化的经济单位。

尽管有些人可能会对如何量化、以及为什么要量化NFT 价值感到吃惊，但这并不影响NFT依然是一个非常有潜力和适用性的未来应用。不用怀疑，NFT时代已经到来了！
