「咖说V讲」系列，原汁原味地汇集国际咖级名流的数字经济与投资观点，阅读者尽可以智者见智，取法其上，健壮成长。本文为「咖说V讲」第33篇，分享原Facebook高管、2022年即将出任Meta首席技术官的安德鲁·博斯沃思（Andrew Bosworth），有关Meta元宇宙的重要观点，尤其是这个由中心化组织构架的元宇宙与区块链、NFT和去中心化DAO融合探索的初步设想。本文第2部分是安德鲁·博斯沃思的简介。

与Meta（原Facebook）老大马克·扎克伯格同为哈佛大学高材生，而且曾经在人工智能课程担任过小扎助教老师的安德鲁·博斯沃思，长期担任Meta（原Facebook）核心高管，目前负责领导公司的AR和VR板块，预计将在新年后出任Meta 首席技术官，成为 Meta转型元宇宙（Metaverse）的关键人物。

在扎克伯格宣布Facebook改名为Meta并向元宇宙转型后，博斯沃思曾经接受了Decoder的长篇访谈，披露了自己对于Meta未来元宇宙的一些初步设想：

“（未来元宇宙的部署）有一些将是全新的，在物理世界中没有类似物，这需要真正的发明。然后，我认为我们作为一个社会要考虑完整性、隐私，这是悬而未决的问题。我在想，做的完整性和隐私保证越强，互操作性就越弱。我们知道，这是我们必须权衡的艰难平衡。”

“如果整个元宇宙由一家公司控制，这对数据库来说并不难，但我们不希望它成为这个样子，我们也认为这不可能。另一种可能非常强大的方法是 NFT 和加密技术。”

“不必将数据存储在某个有其自身缺点的数据库中，而是将其存储在区块链上。那个空间非常令人兴奋，而且发展非常迅速。基础技术绝对是在这里（元宇宙）有用的东西之一。如果它们不是支撑，至少是一部分之一。我不确定元宇宙的每个部分都将得到加密技术的支持。但我认为它很重要，尤其是考虑到目前该领域的创业活力，这确实与创作者有关。”

在前面的访谈后，据《纽约时报》等媒体报道，资深技术大佬博斯沃思又发布了“内部帖子”，希望由Facebook改名而来的Meta，能够“锚定与区块链的深度兼容”，同时他还谈到了探索去中心化自治组织（DAO）和NFT的相关商业路径。

在贴文中，博斯沃思“呼吁谨慎行事”，但是又补充说Meta“应该先于其他组织采用这些技术”。他指出：区块链技术可能“在未来十年对我们的行业产生深远影响”。

博斯沃思在内部贴文中还表示，Meta应该“开发应用 NFT的方式”，并“寻求可能的投资领域，包括基于区块链的智能合约和 DAO。”

据报道，他写道:“我的总体想法是建立与区块链深度兼容的目标。我希望我们完全依赖它的地方并不多，但如果我们看到了与 Web 3.0领域企业家合作的机会，我期望值得去努力。”

不过，据称博斯沃思也警告员工们不要对 Web 3.0 过于狂热，并“敦促员工不要过于依靠去中心化技术。”

安德鲁·博斯沃思 简介

安德鲁·博斯沃思，美国知名的技术领域高级管理人员。 他目前是 Meta 负责AR / VR板块的副总裁。

在加利福尼亚州圣何塞郊外农场长大的博斯沃思，2004年毕业于哈佛大学计算机科学专业，在担任人工智能 (CS182) 助教期间，他结识了当时的本科生马克·扎克伯格。

从哈佛大学毕业后，博斯沃思先在Microsoft Visio担任了近两年的开发人员，之后响应扎克伯格的召唤，于2006年1月加入Facebook，从业至今。

对于博斯沃思加入早期Facebook的经历，媒体有过详细的披露。当时，博斯沃思接到邀请面试Facebook人工智能职位的AOL 即时消息时，大吃一惊，持怀疑态度参加了面试。

“我没有把它当回事，”他回忆，“直到面试后，我才意识到他们所做事情的雄心，以及它可能会变成什么样。在那之后，我感到非常兴奋”。

不过，博斯沃思仍然不愿意放弃在微软的职业生涯，进入一家初创公司的未知领域。

“我的经理给了我最好的礼物之一。当有机会加入Facebook 时，他对我说：你必须走了。我不想失去你，因为你在这里是一个伟大的天才，但你还年轻，你没有什么可失去的，你会在那种环境中学到很多东西。”博斯沃思回忆说。

他接受了经理的建议，成为这家刚刚起步公司的首批十多名员工之一。

作为Facebook的15年老将，博斯沃思先是作为后端工程师构建基础设施，开发了后来成为News Feed的原型，并共同发明了 News Feed、Messenger和Groups 等主要功能，并致力于早期的反垃圾邮件和反滥用系统。

博斯沃思加入Facebook的当年，平台拥有 500 万用户，但是不到一年时间，用户数量就增加了一倍多，而且增长速度呈指数级发展。团队在早期的工作节奏也非常快，工作和生活之间的界限也基本不存在。当时博斯沃思的6位室友都是 Facebook 员工，所以他们是全天候的工作状态。回忆当年，他说这真的不像是工作，他们是在享受冒险、创造力和尝试新事物的乐趣，有时甚至是疯狂的事务。

之后，博斯沃思领导创建了Facebook的广告业务。目前，他领导Reality Labs，构建消费硬件和软件，包括 Quest VR 耳机。他领导的团队拥有超过1万名员工，占全公司五分之一的员工。2022年，他将成为 Meta首席技术官，职权范围扩大到包括公司的人工智能和更广泛的工程团队。

博斯沃思可以说是世界上最有影响力的计算机科学专家，是一位非凡的社交媒体大V，拥有大量粉丝。他经常发布个人照片和视频，与庞大的粉丝群在社交媒体平台上互动。
