无论是通过创造、收集还是兜售，拥抱NFT的名人清单似乎每小时都在增长，并且涉及的范围越来越广。

NFT不再是边缘事物，具体来说，艺术和收藏品NFTs已经以惊人的速度成为主流，甚至比加密货币达人所期待的还要快。我们现在对原因有了更好的了解。正如我在这个周期开始时写的那样，NFT（尤其是艺术和收藏品）是有趣的，可视化的，与文化相关的，而且它们很容易理解，而许多区块链概念则不然。

他们可以解决现实世界的问题。问问NBA 76人费城队篮球运营总裁达Daryl Morey，几个月前他告诉我，“我刚从休斯顿搬到费城，这真是太痛苦了。我不得不搬动10,000部漫画。我不得不移动我所有的壁画。我不得不移动所有这些东西，而对于数字内容来说，只需鼠标移动它。这是绝对有优势的。这是一个主流趋势的开始。”

这个领域的许多人，包括我自己，可能大大低估的一个视角是: 社区视角 。

当你为 Bored Ape Yacht Club、CryptoPunk 或 World of Women NFT买单时，你就明白了。这表明你属于社区的一员。“人们在个人层面上认同他们，”专注于区块链的风险投资 (VC) 公司Electric Capital的合伙人Maria Shen 说。“所有权说明了有关他们身份的一些基本信息，它表明了他们的权益。”

仅在一年前，艺术品和收藏品NFT大多是具有“潜力”的想法，但加密货币领域之外没有人认真对待它们。今天, 哪些类别处于类似的早期投机阶段，哪些可能会爆发？2022年哪些听起来很傻的NFT将受到汤姆·布雷迪的认可？2023年哪些大胆的、颠覆行业的NFT将得到Ariana Grande的推广？这个狂野叙事的下一章是什么？

我咨询了NFT内部智囊团，有投资者、创始人、以NFT为生的人——让我们一瞥（可能的）未来。其中一些类别看起来很明显。有些看起来很牵强。有些人甚至会觉得很荒谬。然而，从整体来看，它们有可能改变我们消费内容的方式、我们花钱和赚钱的方式、我们如何证明自己的身份、我们如何参加活动、我们如何穿着，甚至我们在哪里以及如何度过大部分时间。

也许一切都有可能，现在让我们从小的案例开始。

链游

每天有 200 万人在玩Axie Infinity，现在公司的估值为30亿美元。“游戏真的很令人兴奋，已经有数十亿人在游戏内购买数字商品，”最大的 NFT 平台 OpenSea 的CEO Devin Finzer说。我们还没有看到更广泛采用的原因是“游戏的开发周期比简单的艺术和收藏项目的开发周期要长一些。还有一点延迟。” 他想象我们会在一两年内看到这些发展的成果。

Outlier Ventures（一家位于英国的区块链风险投资和加速器实验室）的CEO Jamie Burke 最初对“人们在区块链游戏中的花费是传统游戏的五倍”这一研究结果而感到鼓舞。他说怀疑论者最初对这项研究“嗤之以鼻”，但后来 Axie 出现了。Burke 说，Axie巨头证明了，如果玩家可以退出游戏并使用加密货币兑现，并且如果他们可以自由地“用钱做他们想做的任何事”，那么他们会花更多的钱。“现在感觉真的很明显，”他说，并预测 Axie 只是一个更大的游戏热潮的开始，这将是“未来十年中的重大事件”。

时尚可穿戴设备

“奢侈品牌正在进入 NFT 领域，” Laglasse一边说，一边在我们的 Zoom 通话中抽着烟。展览A：9月30日，杜嘉班纳 (Dolce & Gabbana)以560万美元的价格出售了其首个九件套NFT系列，名为“Collezione Genesi”，这是一种高级时装和区块链的超现实组合。该套装包括实体物品（如女装）及其NFT数字伴侣。

就在两周前，在伦敦时装周上，一个名为Auroboros的新品牌自称为“第一家将科技与实体时装相结合的时装公司”，推出了一系列数字服装，您可以使用增强现实（ AR）技术“穿”它。这不是在加密会议上。这发生在伦敦时装周。

看看The Bored Ape Yacht Club和The Millions（高端街头服饰）之间的合作，他们为社区创造了商品。“产品几个小时就卖光了，”Shen说。当时尚品牌看到这一现象时，他们就会知道有一群热切（且财力雄厚）的观众。“18 年来，我们从未在我们的网站上看到过像现在这样的事情，”The Hundreds的联合创始人Bobby Hundreds当时赞叹不绝: “哇哦。”

时尚可穿戴设备可能只是冰山一角。Shen 想象 NFT 可以创造新的方式来将时尚商业化并投资时尚。拿运动鞋举例, “为了周转运动鞋，你必须进行实物库存清点，”她说。“想象一下，如果你可以提高周转物品的速度。” 或者也许由 NFT 推动的新金融工具可以让人们投资时尚行业。“你可以创建金融产品，比如今天最畅销的运动鞋指数，”Shen说，例如它可以让你投资 100 款最热门的运动鞋，而不是把骰子扔在一双上。

DeFi+NFT

假设你在 CryptoPunk NFT 上花费了你最后的 500 万美元，但你忘记了你需要支付租金。没问题。您可以使用该 CryptoPunk 作为NFTfi贷款的抵押品。“NFT 支持资产的一些新行为，”Sfermion（一家 NFT 投资公司）的执行合伙人兼NFT 播客主持人 Andrew Steinwold 说。

想想那笔贷款。当你将 CryptoPunk 作为抵押品交出时，你会在还清债务时自动取回。由于智能合约的神奇之处，NFT 被转移到贷方，消除了收债和赏金猎人的需要。那只是开始。正如我的同事David Z. Morris 所写，NFT 正在被细分化（以提供更多流动性），它们充当准证券，并且变得足够金融化，以至于它们可能很快会引起美国证券交易委员会(SEC)的兴趣 。

Burke 预计创新将继续快速发展。他想象着二级市场甚至“衍生”市场的崛起。想象一下，无论出于何种原因，Epic Games 决定不将 NFT 集成到 Fortnite 中。也许一些衍生的 NFT 可以填补空白？“假设我有一把剑，它是一把非常有价值的剑，”Burke说。“而且我想从游戏中兑现。这把剑太值钱了，它有能力还清我的助学贷款，或买房。” 他继续说，显然很享受这个思想实验：“我爱我的剑，但我需要现金。”

现在，你对此无能为力。你刚刚有一把毫无用处的剑，它被困在 Fortnite 中。但也许这把剑的“双胞胎”— NFT—存在于 Fortnite 之外—可以在衍生品市场上买卖，然后某种机制将允许在交易完成后转移资金在堡垒之夜。这些是NFT+DeFi 可以释放的可能性。

在这种情况下，Burke认为，在我们看到传统金融中已经存在的事物（例如房屋契约）的 NFT 化之前，我们更有可能看到新的、创造性的、支持区块链的金融产品的增长。“DeFi 的增长从何而来？在这种监管环境中，它肯定不会是现实世界的资产，也不会是任何类似于证券的东西，”Burke说：“这将是一场长达十年的战斗。”

活动和票务

另一个假设：Drake 即将举办一场音乐会，而你想去。门票是100美元。现在想象你买的票实际上是一个 NFT，你的 NFT 解决以下六件事：

  • 上面印有您喜欢的设计师的一些艺术品。

  • 作为演唱会纪念品。所以有一天它可能是有价值的，比如披头士乐队演唱会的旧票根。

  • 在会场内你可以使用 NFT 作为获取小吃或啤酒的一种方式。

  • 再次感谢智能合约的魔力，你的 NFT 收入自动支付 Drake 40%，DJ 10%，灯光组 2%，看门人 1%，等等。无需中间人来安排付款，这意味着你几乎无需支付任何附加费用。

  • 如果你购买了 VIP 门票，NFT 会解锁您访问 Drake 后台的权限，或者它可能会为你提供对未来数字产品的独家访问权限。

  • 令人惊讶的是，NFT 赋予你权利，不仅可以成为活动的参与者，而且还可以成为 Drake 音乐会的股东。这个 NFT 充当音乐会股权，你将获得一小部分利润。也许你甚至可以通过参加节目来赚钱。

这个未来比你想象的更近。“让我们兴奋的是 NFT 在现实世界中的使用，”NFT 铸造平台 Mintbase 的联合创始人 Carolin Wend 说。以即将在里斯本举行的NEARCon节为例，它围绕NEAR Protocol 进行，这是 Mintbase 使用的智能合约系统，而不是以太坊。“音乐节创始人想要 100% NFT 票务，”Wend说，这可能（有一天）包括“抵押”你的门票，这意味着你可以从门票中获利。

其中一些功能在未来仍处于关闭状态，但即使在里斯本音乐节（10 月 26 日举行）上，Wend 也希望它们能够“从空间中提取二维码”，以便人们可以扫描二维码, 用NFT兑换墨西哥卷饼 。这一点并非假设的。在里斯本，Wend说，“我们将做墨西哥卷饼 NFT。

NFT 社区

还记得那些 Dolce & Gabbana NFT 吗？现在想象一下，你是幸运的人，买了一个。这是一件几乎可笑的华丽珠宝。你如何储存它？也许你将实体保存在你的私人金库中，或者你将它展示在米科诺斯别墅的王座室顶上，谦虚而有品位。

但是数字 NFT 呢？你用它做什么？或者正如Burke所说，“你在哪里使用它？” 从 Dolce & Gabbana 的角度考虑一下。“如果你是一个高端奢侈品牌，那就是控制零售体验。”Burke指出杜嘉班纳 (Dolce & Gabbana) 给 NFT 的买家两年时间，以弄清楚在哪里渲染它。

因此，Burke 预测了人们可以聚集、聊天和炫耀他们的 NFT 的某种环境。也许它是 AR 的一种形式，或者可能是虚拟现实 (VR)，或者可能是 Web 3.0 风格的东西，或者可能是其他完全不同的东西。关键是需要有一个良好的环境来体验 NFT。“网络平台有哪些？这将是大趋势，”Burke说。

Shen 有一个类似的想法，但有不同的转折。她设想“一个 NFT 社区的家”，或者更准确地说，“一个为 NFT 所有者打造的社交网络”。也许这个网络连接到你拥有 Bored Ape 的钱包，并且只有拥有某些 NFT 的经过验证的用户才能访问它。

元宇宙

Nonfungible.com的内容负责人Maxime Laglasse表示：“元宇宙将成为 NFT 领域未来的游戏规则改变者之一。” 简短得说：元宇宙可以是一个存储和欣赏 NFT 艺术的地方，它是游戏的中心，它让扎克伯格在生活中有了新的使命，也许它是我们如何在网上闲逛的进化版本。

虚拟土地

一位明智的投资者曾说过：“土地。这是他们无法制造更多的一件事。” 投资者是Lex Luthor，他的建议在元宇宙中听起来可能是正确的。像 Decentraland、The Sandbox 和 Cryptovoxels 这样的虚拟世界给他们的房地产一个上限，这理论上意味着如果需求飙升，有限的供应将变得更有价值。

“这是非常有希望的，” Matty “ DCLBlogger ”说，他是 NFT 社区中一个有影响力的声音，他是一位预言家，在 2020 年（一年多以前）在推特上发布了一个广泛共享的帖子, 概述了 NFT 的 25 个未来用例。这条线索看起来很有先见之明，正确地称其为艺术、收藏品和游戏的兴起。Matty也同样看好虚拟土地。“看看 Axie Infinity，”他说, “有一百万玩家，” 其中一些玩家会想，“也许拥有那块土地是有意义的。”

一旦你开始思考人们可以从数字房地产中获利的想法，可能性就会令人眼花缭乱。也许你拥有一块土地，你建造了一座虚拟办公楼，然后你把这栋建筑出租给虚拟会议。当会议参加者想要发泄一下时，也许他们会想在鸡尾酒时间赌博，所以你租了一个赌场游戏，让他们赢得和输掉加密资产。正如Matty所说，“这真的很疯狂。”

数字身份

自我主权身份 (SSID) 长期以来一直是区块链技术最有趣的应用之一，而 NFT 可能是打开大门的钥匙。（为什么 SSID 很重要？这是我去年的入门读物。） Laglasse 将以太坊名称服务（ENS）视为一个有用的例子。“现在你可以将你的 Instagram 帐户、你的 Twitter 帐户和几乎所有社交媒体帐户链接到 ENS，”Laglasse 说。“由于它是去中心化的，你可以真正拥有你向社区展示的名字。”

Laglasse 将这些归类为“实用型 NFT”，因为您通常不能将它们换成金钱（如艺术品和收藏品），但它们的存在是为了证明您拥有某些东西。与此类似的另一个想法：大学文凭的 NFT。例如，这将让一名越南学生前往美国，并快速证明她的文凭。Laglasse 预测：“总有一天会发生的。”

社交媒体和影响力

这个有点难以想象。但坚持下去。“想想通过社交媒体平台创造的所有数字价值，”Burke说。他指的是“在线影响力”这个难以捉摸且难以量化的概念。目前，Burke认为，将这种影响力货币化的唯一真正方法是通过广告，而影响者只能从收入蛋糕中分得一小部分。但是，如果这种影响可以量化、评分并作为 NFT 捕捉呢？

让我们称之为“影响力NFT”，它不会像奖励 Twitter 粉丝数量那么简单。“这是一种原子化的社交形式，”Burke解释说，“我可以将每一点内容原子化，并且可以量化我与社区的关系。” 他说，这种 NFT 可能会考虑诸如在线声誉、可信度甚至策划和创建优质内容的能力等变量。“这是最难解释的，”他笑着说。

还记得 Twitter 早期的“Klout”分数，其中一种算法试图量化用户的社交魔力？Klout 似乎从未流行起来，它在 2018 年就破产了，但你可以看到这个想法的（可以说是邪恶的）吸引力。“从理论上讲，任何形式的数字价值流都可以变成一种资产，”Burke说，他还设想将这种 NFT 插入 DeFi 生态系统，例如用作贷款的抵押品。

Shen 认为这个概念是信用评分的更精细版本，但它是为了信誉和影响力。“实际上，你可以拥有 NFT，构成你身份和声誉的一部分，”她说。她以Rabbithole.gg 为例，用户可以通过证明他们具有某些技能或完成某些任务（例如注册 ENS 名称）来赚取加密货币、代币或 NFT。

Steinwold 喜欢另一个例子：Galaxy.eco旨在使用链上数据来奖励人们进行某些可验证的活动。也许如果你在 Uniswap 上进行了 1,000 笔交易，你就可以获得 Master Trader 徽章。或者如果你是长期持有者，你将获得 Diamond Hands。如果你能证明你已经阅读了这篇文章，也许我应该给你铸造一个 NFT。

音乐

“整个行业肯定需要一些颠覆，” Matty说，他预计音乐最终会像艺术品和收藏品一样起飞。也许这已经开始了。Shawn Mendes、Grimes 和 Snoop Dogg 都涉足 NFT，发现了通过粉丝赚钱和与粉丝互动的新方法。Kings of Leon 并不满足于简单地以 NFT 的形式发行专辑；他们是如此热情，以至于他们在太空中玩了 NFT。

Live Nation 想要为音乐做一些 Top Shot 为 NBA 所做的事情——捕捉“瞬间”作为收藏品。“我们都从 NBA 的 Top Shot 中学到了东西，”CEO Michael Rapino 在财报电话会议上说，并补充说他希望 Live Nation 创造“我们可以创造并附加到我们正在进行的门票节和特殊时刻的神奇时刻。”

或者，也许 NFT、音乐和 DeFi 都可以合并。“如果音乐大师可以真正汇集，然后细分，那么你就可以获得收入作为流媒体的红利，”Shen 说。然后她提出了另一种可能性：想象一下，如果每首歌曲的版权都被捕捉为 NFT，然后你把当月的前 40 首歌曲集中在一起，你对音乐流进行分割，然后你可以购买该音乐流的一部分并获得流媒体收入。“那会很有趣，”她说，“我们只是还没到那个地步。”

快速描述另外四个领域：

医疗保健

 “在医疗保健行业，这将带来巨大影响，”Laglasse说，他认为 NFT 可用于保护健康记录的机密性，以及加强药物的供应链跟踪。

广告

这里是一个曲线球。“广告不是人们关注的东西，” Matty说。但他问道，“为什么网站不出售赋予人们访问某些媒体广告权利的 NFT？” 所有这些都在发挥作用。

体育

Sorare已经在全球足球比赛中站稳脚跟。不难想象，NFL 和梦幻足球这个大产业会火起来。

NFT检索

 “现在，NFT 是一个有趣但混乱的类别，”Shen 说，并补充说它们“看起来很像 90 年代谷歌出现之前的网站。” 她说的有道理。目前还没有针对 NFT 的干净搜索功能……。想象一下 NFT 的 Google。或者一个去中心化的 NFT 谷歌。

最后，最极端的场景：

一切物质的数字化

“很多人仍然没有意识到 NFT 会有多大，”Steinwold 说。“它们将被用于任何事情。所有形式的价值都将被表示为 NFT。” 这是一个大胆的声明。我问他我是否听错了。所有形式的价值都将由 NFT 表示？

他怎么这么自信？

“我们醒着的大部分时间都在网上度过，”Steinwold 解释道，并认为这种趋势将在未来 20 或 30 年内继续加速，而且很快“我们将拥有大量数字产品。” 未来，Steinwold 说，如果他买了一双很酷的 Air Jordans（可以穿在脚上的实物），他会想要它的 NFT 伴侣。他会想要 NFT，这样他就可以炫耀它、灵活使用它，或许还可以在元宇宙或play-to-earn游戏中使用它。

“也许不是这把椅子，”他说，指着 Zoom 上方的椅子，笑了起来。但他预计人们最终会想要他们的葡萄酒、棒球卡、稀有硬币、手表、汽车—你能说出的—的 NFT, “这就是世界将要走的路”。

也许这听起来很牵强。就像在一年前，对艺术和收藏品的主流 NFT 的想法也是如此。
