没有一家运营 IP 的公司是不想成为迪士尼的。

仅靠一个个创作出来的 IP 形象，迪士尼挣到了多少钱？据统计，2021 年全球最赚钱的 10 个 IP 中，迪士尼一家就占据了半壁江山，5 个 IP（维尼熊、米老鼠和他的朋友们、星球大战、迪士尼公主、漫威电影宇宙）总收入 3110 亿美元。

围绕 IP 为核心，迪士尼建立了闭环且全面的盈利手段，覆盖了消费者可以接触到的方方面面，主题乐园、电影制作、流媒体、零售……一个个由人所创作出来的虚拟形象，一旦到了迪士尼手中，他们就摇身一变成为了马力全开的印钞机，以其他公司艳羡的速度产出着源源不断的营收。

迪士尼的诞生和繁荣，离不开现代版权体系的建立，「地表最强法务部」的诨号也足以说明司法体系对维系迪士尼繁荣的重要性。当 Web3 的时代到来，无需许可、去中心化的链上世界，借助于 NFT、ERC20、DAO 等全新的技术和组织形式，链上世界的原住民们能否复刻一个去中心化的迪士尼、甚至超越迪士尼？

3 月 17 日，BAYC 地板价突破 100 ETH。这意味着一个「猴子头像」的价格已经高达约 30 万美元。有人从中看到了炒作和荒诞，而有的人却看到了团队对 IP 运营的强劲实力。估值 50 亿美元的 BAYC 团队，尽管距离迪士尼 2500 亿美元的市值仍相距甚远，但这个诞生尚不足一年的项目，已经让我们看到了 Web3 版本迪士尼诞生的可能性。

米老鼠与无聊猿猴

1923 年，华特·迪士尼卖掉了自己的摄影机带着未完成的《爱丽丝在卡通国》的影片胶卷购买了去往洛杉矶的单程车票，那一年他和哥哥洛伊·迪士尼成立了迪士尼兄弟工作室，5 年后在去往好莱坞的车上创造了以老鼠为原型的卡通形象莫蒂默，在妻子的建议下更名为米奇，至此之后这个传奇的卡通老鼠形象在迪士尼的笔下影响了一代又一代人。

生活在纽约的华裔插画师塞内卡，是受到迪士尼影响的一代人中最为普通的一个，她在曼哈顿公寓中特意留出一个角落，随意摆上画笔和纸张那便是属于她自己的「工作室」，杂乱的工作室中除了散落在地上的一堆水彩画以外，游戏《爱丽丝：疯狂回归》也夹杂其中，那是一百年前由迪士尼带上荧幕的故事，也是塞内卡开启了她的插画之旅掉进的「兔子洞」。

天马行空的想象力让塞内卡顺利成为了 Yuga Labs 的首席插画师，她的奇幻世界里居住着一群对生活感到厌倦的猿猴，他们的名字叫做 Bored Ape Yacht Club，这些猿猴们席卷了 Web3 的世界，就像那只吹着口哨开着蒸汽船的米老鼠曾经席卷了全世界每一个人的童年一样，

乐园与宇宙

每一个女孩儿都梦想成为公主，迪士尼给了她们圆梦方式，只要你走进来，你就是公主。

米奇的初次尝试打开了迪士尼乐园的大门，接下来的几十年里唐老鸭、白雪公主、小飞象、钢铁侠、纷纷加入到这个世界之中，文具、日用品、服饰，似乎你能生活中能接触到的一切都被印上过迪士尼的图案，那些记忆中的角色为乐园点燃的焰火绽放至今。他们誓死捍卫着每一个形象的归属，但这些概念显然在 Web3 的世界里并不那么受欢迎。

人们希望 BAYC 更为开放的，事实也是如此。

「BAYC membership costs the same for everyone」，在发售之前，BAYC 的团队在推特就喊出了「公平」的口号。公平发售并且给予拥有者完整的商用授权，即是头像也是俱乐部的会员卡，发展社区成为了每个成员自发的热血行动——只要你的头像是一只猿猴，那么其他的「猿猴」就会来关注你。

那时人们将真正属于自己的猿猴印在了衣服上、手机壳上甚至是面膜上。BAYC 给予他们的不是隔着荧幕无法触及的童年偶像，而是「你的邻居」或者说「你」。

这些 Web3 的年轻人远比想象的更为疯狂，卫生间涂鸦、线下聚会，他们凝结起来的力量丝毫不弱于沉淀百年的米奇，「猿猴们」很快拥有了自己的乐园，不仅仅是官方所空投的 BAKC 还是使用药水所变异而来的 MAYC，这些新加入的成员远不能构成现在的「猿」宇宙。商用版权的开放，让更多的 NFT 项目做起了以无聊猿为核心的二次创作，整个 NFT 的市场掀起「动物园」的热浪才是最有力的推手。

他始终在告诉我们：你不再需要仰慕公主而进入乐园，你在的地方就是乐园。

星战与朋克

2012 年，迪士尼开启了扩张之路，以 40.5 亿美元的价格收购乔治·卢卡斯创立的卢卡斯影业。这家影业对国内大部分读者也许都有些陌生，但他们的 IP 你一定知道：闻名全球的《星球大战》。

受限于文化背景的差异，《星球大战》对于中国观众来说或许仅仅只是一个「知名 IP」。而对于美国观众来说，这是一个家喻户晓的「经典」，在 35 年的时间里陪伴了数代人的成长。星战价值早已不止局限于电影，电影、玩具、游戏、衍生品，更是科幻题材 IP 综合运营的里程碑式开山之作。

让我们稍加修改，把这个故事放到 NFT 的世界里再讲一遍，你会联想到什么？

一个经典又悠久的 IP、一家强大的 IP 运营商，两个知名公司的强强联合。这听上去像不像 BAYC 和 Crypto Punks？

3 月 12 日，BAYC 母公司 Yuga Labs 宣布收购 Larva Labs 开发的 CryptoPunks 和 Meebits NFT 系列。Yuga Labs 计划将这两个 NFT 系列的全部商业权利提供给持有者。Yuga Labs 将致力于为 CryptoPunks 和 Meebits 系列带来额外效用，在决定下一步行动之前，将充分听取社区的意见。

CryptoPunks 这个「古董级」的 NFT，若论起知名度在 Web3 的世界可谓是人尽皆知了，甚至屡次破圈，其影响力一度外溢到 Web2 的世界。而 Yuga Labs 对 CryptoPunks 的收购，又和迪士尼对「星战」对收购有着异曲同工之妙。

近期以来，Larva Labs 的种种不顾及社区的做法一度引发网友们的争议。但 CryptoPunks 的地位却始终难以撼动——毕竟，CryptoPunks 拥有任何项目都难以取代的历史价值。而 Yuga Labs 无疑是受到社区所信任的。

一个经典且历史悠久的 IP、一个正在受到非议的团队、一个冉冉升起的后起之秀。Yuga Labs 对 CryptoPunks 的收购，无疑是对多方最好的结果之一。

正如卢卡斯影业 CEO 在提及星战收购时所讲述的那样，「在过去 35 年，我最大的喜悦之一是看着《星球大战》从一代传承到另一代。现在是时候让我将《星球大战》传承给新一代的制片人。」现在，CryptoPunks 也迎来了他的传承时刻。

当一个优秀的 IP 遇到了擅长「搞事情」的 Yuga Labs 团队，Yuga 运营 IP 的能力能否为 CryptoPunks 赋能？在最理想的情况下，或许就像迪士尼在收购「星战」之后那样：「最优秀的内容组合与迪士尼横跨多个平台、业务和市场、独特和无与伦比的创造力整合在一起，从而产生可持续的增长并推动巨大的长期价值。」

Web2 与 Web3

至此，关于迪士尼世界的故事暂告一段落。可 Yuga Labs 的企图远远没有这么简单。

2021 年 10 月份，面对新鲜血液的不断冲击，大部分资历较老 NFT 项目开始坐如针毡，空投 NFT 似乎无法解决当务之急，发 Token 似乎成了唯一的救命稻草。CoolCat、Pudgy Penguins 、Groupies、 DeadHeads、Animetas 等，都开始重新规划 RoadMap，而内容基本上也是围绕发放 Token 为核心融入 Defi 的玩法，有的可以质押获益，有的开放 P2E 游戏。

就在所有人期待着 BAYC 能出一个满意答案的时候，他们却给所有人泼了一盆冷水。

「什么时候发放 Token 是我们在社区常被问到的问题。首先，要知道，上传一个 ERC-20 的 Token 是非常简单的，只需要几分钟就能完成。但是如何构建一个合规合法的 Token，并且对其负责，让其可持续的发展，这才是最为困难的事情。更不要谈论说如何使用和治理 Token，我们需要为 BAYC 的社区成员带来收益的同时，也要将 BAYC 的经济体系带到更广泛的视野中。为了我们的社区成员和更为宽广的 NFT 社区，这才是我们要做的事情」。

3 个月后，BAYC 系列地板价超过 CryptoPunks 正式成为 NFT 市场中最大的龙头。

5 个月后，在距离 Yuga Labs 发布收购了 Crypto Punks 和 Meebits 的消息仅仅 5 天之后，ApeCoin 来了。

3 月 17 日，BAYC 母公司 Yuga Labs 正式宣布推出 ApeCoin，用于 APE 生态系统中，并为去中心化社区建设赋能。

ApeCoin 具有治理、统一支出、生态系统访问和激励等功能，总供应量为 10 亿枚，其中 62% 属于生态系统基金（1.5 亿枚将空投给 BAYC/MAYC NFT 持有者），16% 由 Yuga Labs 及其支持的慈善机构享有，14% 发放给项目贡献者，8% 留给项目四位联合创始人。

空投将于美国东部时间 3 月 17 日 8:30（即北京时间今日 20:30）开始发放。空投规则显示，仅持有 Bored Ape NFT 的可获得 10,094 枚 Token；仅持有 Mutant Ape 的获得 2,042 枚 Token；同时持有 Bored Ape 和 Kennel Club 将获得 10,950 枚 Token；同时持有 Mutant Ape 和 Kennel Club 将获得 2,898 枚 Token。

此外，Yuga Labs 同时推出 Ape Foundation。根据声明，ApeCoin 归属权属于 ApeCoin DAO，而 Ape Foundation 仅是 ApeCoin 的「管家」。ApeCoin DAO 成员可以对生态系统基金的治理和使用进行投票，持有 ApeCoin 是 ApeCoin DAO 成员的唯一要求。

至此，一直致力于建设社区的 Yuga Labs，终于完成了在 Web3 建设中最重要的一块版图，有人说 Yuga Labs 走了一条最为稳妥的道路，选择做 Web3 时代的迪士尼；有人说 BAYC 打开了当前市场的窘境，开辟了新的道路。无论哪种说法，我们不可否认的是一个庞大的世界即将浮现，它将改变现有的规则，建立新的秩序。

结尾

在浩荡的太空探索时代，人类第一次进入太空、登上月球，想象力向无垠的太空肆意的生长着，太空题材一时成为文艺作品的主旋律。面对浩瀚的太空，卢卡斯创作出了《星球大战》这样的经典 IP，延续四十余年在银幕、游戏、衍生品等领域不断的生长着。一个好的 IP，可以孵化出自己的文化、社群乃至圈层。尽管太空探索的时代已经走向终结，但《星球大战》的影响力却丝毫没有衰减。

而当我们身处一个加密技术深入改变人类社会的历史窗口期之时，加密世界的原生 IP 是否会像曾经的「星战」一样肆意生出自身的文化属性？在坐拥 BAYC、Cryptophunks 等众多顶级 IP 以后，Yuga Labs 至少已经在成为 Web3 迪士尼的路线上一路疾驰了。而如何用好这些高价值 IP，就考验 Yuga 团队的智慧了。
