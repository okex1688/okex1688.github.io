今天，令人印象深刻的收藏与未来的技术集成相结合，加密艺术博物馆 M○C△ （MOCA）代币持有者有可能获得并保持作为 NFT 空间中文化策展人的相关性。在分析原因之前，我们先来看看博物馆的历史。

MOCA 简介

M○C△ – 加密艺术博物馆 – 最初于 2020 年 4 月开始，是朋友和加密投资者 Colborn Bell 和 Pablo Rodriguez-Fraile 之间的合作。他们想展示一个通过独特的以艺术为重点的 NFT 收藏品策划的与加密货币文化相关的顶级收藏品。在艺术史学家 Shivani Mitra 和建筑师 Desiree Casoni 的帮助下，博物馆在 2020 年收集了来自 Beeple、Pak、Osinachi、Hackatao、Fewocious 等顶级加密艺术家的艺术作品。完整收藏品存储在 ENS 地址museumofcryptoart.eth。

仅是永久收藏中一些更显著的艺术品

2021 年 1 月，Bell 和 Rodriguez-Fraile 分手，Bell 继续与 Mitra 一起管理 M○C△ 品牌。不久之后，Bell 和 Mitra 负责了 M○C△ 的第一个也是最重要的支点：向社区运营项目的转变。作为转变的一部分，M○C△ 被重组为 DAO，以 $MOCA 代币作为其治理代币。博物馆 150 万美元的代币销售吸引了著名的私人投资者，如 Polygon、Metapurse、Coingecko 等。

Tokenomics：分发、效用、策展

$MOCA 是一种固定上限资产，总供应量为 10,000,000 枚代币，近期升级可能会使代币出现通货紧缩。MOCA 为生态基金预留 10% 的代币，空投 5%，为核心团队预留 18%，分配给私募投资者的 8.1%。最后，大部分代币分配给永久收藏品收购。

除了治理之外，MOCA 代币还计划在两个方面提供进一步的实用性：访问和管理：

  • 访问（Moca Points）：MOCA 团队计划使用代币门控 Discord 为代币带来更多价值。可以对不和谐频道进行分层并为博物馆成员提供独占权。抵押 MOCA 代币将为用户赚取“Moca 积分”，用于解锁特殊的 Discord 功能。

  • 社区策展：策展可用于多种功能，包括决定将哪些作品添加到博物馆的不同部分，以及赞助他们自己的“翼”和博物馆子展览的个人收藏家。

分发：“艺术证明”

博物馆转向以社区为导向的项目的核心是他们的“永久收藏”。这些是永久属于M○C△ DAO的艺术品。永久收藏的新颖之处在于分配 $MOCA 代币来购买这些艺术品——大约 50% 的供应保留用于永久收藏的收购。作为一种伪工作证明，这种“艺术证明”既是一种新颖的代币分配方法，也是发展 M○C△ 社区的一种手段。当然，艺术家总是可以出售他们的代币，但随着收藏的增加和 $MOCA 代币的实用性增加，社区保留率应该会增加。此外，作为一种治理机制，艺术家可能会发现将自己与 MOCA 等各种数字博物馆结盟以确保他们的艺术品被其他高质量作品所包围具有战略意义。

博物馆将这种分发方法与通证促进的分散式策展路线图相结合。因此，令牌可以被概念化为管理收藏的部分权利。为了使这种策展权获得价值，作为 M○C△ 展览一部分的艺术家作品的出处也必须获得价值——类似于让艺术品出处成为 MOMA 或卢浮宫展览的一部分。在这种情况下，博物馆可以成为加密艺术领域的领先文化实体，代币持有者将成为加密艺术领域的主要文化策展人。此外，博物馆的数字原生、固有的可扩展性使其比现有的博物馆具有明显的优势，随着时间推移，这些博物馆并且很大程度上受限于物理世界。

关于社区和“gm”测试的说明

分析 NFT 社区的一种方法是评估项目 Discord 中的活动。臭名昭著的是，Reddit 和 Initialized Capital 的联合创始人亚历克西斯·奥哈尼安 (Alexis Ohanian) 曾表示，原因之一他投资 Coinbase 是因为 r/Bitcoin 等 subreddits 中的强大社区活动。同样，Discord 服务器内的活动可以为任何公司或协议的新兴社区提供线索。一般来说，找到积极讨论与项目相关的加密话题、参与社区电话/讨论或小组活动的社区是社区一致的积极标志。社区活动的一项快速测试是“gm 测试”。例如，分析说“gm”（或早上好）的人数与谈论价格或普遍抱怨为什么项目还没有代币/NFT 的人数。如果这个比例非常高，意味着每天有几十甚至几百人在垃圾邮件“gm”，而没有人关心代币价格/项目底价，你可能已经找到了一个更真实的社区。一旦您确定最初的 gm 测试是成功的，那么您就可以更深入地探索各种不和谐的渠道，以确定社区的文化和真实活动。

创建实用程序：社交媒体集成和 Moca 积分

M○C△ 路线图的一个关键节点是前面提到的 Discord 集成。该团队正在构建一项功能，允许 Discord 用户展示他们可证明拥有的个性化 NFT 选择。此功能是通过与 Discord-ubiquitous Collab.land bot 的集成启用的。基本上，这个机器人允许用户验证钱包是否持有指定的令牌。这在传统上用于分配特殊角色，例如项目中的“持有者”，以及允许访问仅限持有一定数量和/或类型代币的人可用的受限渠道。

随着M○C△的Discord功能的加入，用户可以调用简单的功能来炫耀他们在博物馆应用程序上预先配置的各种藏品。例如，拥有 CryptoPunk 的人将能够调用 !punk 以立即显示其钱包拥有的一个或多个朋克的选择。在此之前，NFT 社区中的大多数人展示他们的收藏品/艺术品的方式是通过 Opensea 的链接或简单的屏幕截图。除非您已经建立了将您的 Discord 帐户链接到拥有 NFT 的地址/ENS 的声誉，否则没有人可以验证您是否拥有您所说的内容。即使您确实拥有这种声誉，其他用户也可能冒充您和您的个人资料。

基本功能将是免费的，增加的功能将提供给那些积累“MOCA 积分”的人，这些功能主要可以通过购买和抵押 $MOCA 或通过参与 M○C△ 应用程序来获得。解锁的功能和所需的点数如下所示。

考虑到许多人收集的 NFT 比他们在社交媒体上的个人资料图片和横幅插槽中显示的要多，因此显示个人投资组合中其他 NFT 的需求可能很高。对于具有一定感知影响力的 NFT 尤其如此，通常是由于高度稀有和/或估值。由于需要 Moca 点数来显示更多 NFT，Discord 集成有可能充当 $MOCA 代币的实用程序（和潜在需求）催化剂。

此外，未来的整合，包括 Telegram 和 Twitter 的整合、多资产化身等，可能会创造类似的需求侧催化剂来购买和质押 $MOCA。

通货紧缩供应

为了利用 M○C△ 平台的功能，用户必须在平台上“激活”他们的 NFT，并为此支付 0.1 美元的 MOCA 费用。然后该费用由协议烧毁。假设固定供应量没有变化，这意味着在每个代币被销毁之前，平台上总共可以激活 1 亿个 NFT。作为参考，仅 9 月份就在 OpenSea 上售出了 170 万个 NFT。(h/t rchen8 )。

现在，该费用要到 2022 年 1 月才会生效，但如果 M○C△以不和谐的方式捕获当前NFT 买家的很大一部分，更不用说未来的用户，那么这种模式似乎是不可持续的。似乎需要降低费用、摆脱销毁方法、代币通胀或动态发行模型中的一项或多项，以确保代币和平台可持续发展。尽管这是不可持续的，但这是一个很好的问题，因为它表明对博物馆服务的需求。

社区策展

就目前而言，核心 M○C△ 团队选择添加到永久收藏中的艺术品。随着代币在整个社区中的传播，以及协议逐渐去中心化，收藏收购似乎很可能由 DAO 或 MOCA DAO 内的委员会接管。对社区收购未来的乐观展望是一种类似于Party Bid的体验，在这种体验中，收藏家在有趣的在线环境中集体竞标不同的艺术品。在社区达到更成熟、更参与的点之前，这不太可能发生，鉴于加密市场的快速、不可预测的变化速度，这种情况发生的时间表未知。

竞争分析

由于该协议及其代币具有广泛的用例，因此 M○C△ 与数字艺术领域的众多团队展开竞争。我比较了博物馆上述代币的三个功能：分发、效用和社区策展。

分配：去中心化真的很重要吗？

如前所述，MOCA 代币不授予基础资产的任何分割所有权。分割的论点通常围绕 1) 获取高价值、非流动性资产并使其具有流动性，以及 2) 为那些不能或不会为“完整”NFT 付费的人提供财务优势。这一思路的一个流行例外是指纹 DAO。Fingerprints 是 Autoglyphs 以及其他具有重要文化意义的 NFT 的最大持有者，并孵化了几个以 NFT 为中心的 DAO。在这方面，Fingerprints 类似于博物馆——一群藏家试图策划他们认为对加密艺术空间有价值的东西。

目前，Fingerprints DAO 的一级会员在场外交易市场上的价格约为 25 美元。$PRINTS（Fingerprints 的原生代币）非常受欢迎的一个原因是它的排他性——一个 25 以太币的入口点意味着只有 OG 或鲸鱼可以访问。虽然 $MOCA 的目标是成为每个人的博物馆，但 $PRINTS 是精英精英的俱乐部，因此它要求更高的价格——尽管同样不授予任何所有权或使用权，但无法获得私人 Discord。

Fingerprints DAO 网站。20 $ ETH 或更多加入该 Discord。

总而言之，社区历来通过提供一种资产的排他性和所有权感而获胜。博物馆既不提供这些，也可能因此面临与其他类似组织的竞争劣势。

各种形式的实用程序

许多 PFP 或艺术 NFT 项目提供零有形效用，一个值得注意的例外是像gen.art这样的项目，其会员资格充当“铸币通行证”，允许所有者铸币新收藏，避免了天然气战争的需要。像 gen.art 这样的项目具有“垂直效用”，在其产品套件中为用户提供效用。M○C△ 通过提供任何 NFT 持有者都可以使用的服务——可验证的所有权——具有“横向效用”的独特优势。由于 Discord 集成需要用户销毁和/或质押 $MOCA 代币，这表明了 $MOCA 代币的实用性的明确途径。由于大多数其他 NFT 项目要么不打算提供效用，要么面临重大的执行风险（例如，游戏开发商在构建游戏之前筹集资金）。博物馆在其 Moca 点、访问和对博物馆资产库的管理控制方面的有形效用高于该领域。

在无形效用方面，M○C△处于独特的地位。一方面，永久收藏中的艺术品目前具有很高的估值。另一方面，没有任何代币持有者真正拥有它，因此代币持有者无法像其他组织或个人 NFT 所有者那样获得出售博物馆作品或将这些艺术品金融化的优势。如果 MOCA 能够利用其资产来赚取收入，那么拥有 MOCA 代币的价值可能会增加。最后，拥有资产的博物馆与个人对社会价值的感知也不同。这使得拥有博物馆的一部分可能比拥有特定的 NFT 在社会上更不受欢迎。如果使用/展示集体所有的 NFT 的文化兴起，这个问题就会得到缓解。

野外的无形效用——Fidenza 的一部分以 2500 ETH 的估值购买。

社区策展 Pt.2

由于博物馆的钥匙还没有交给代币持有者，M○C△ 发现自己与它的集中设计竞争对手处于相同的位置：集中策展。看看随着时间的推移，团队对社区提供更多的策展控制权是否会激发对策展的需求，从而为代币持有者带来价值，这将会很有趣。

关于 Web 2.0 风险的说明

我强调的一个主要优势是 $MOCA 令牌在 web2 社交媒体集成方面的实用性。在某种程度上，这些功能——Discord/Telegram 命令和经过验证的单资产和多资产化身来概括——充当我们最喜欢的 web2 平台和我们最喜欢的 web3 资产之间的桥梁。那么，如果这些公司建立自己的 2.0 <> 3.0 桥梁会怎样？我们已经看到 Twitter 梳理出了对验证头像和集成 Web3 钱包（如 MetaMask）的支持。由于社区的强烈反对，Discord在退出之前对原生 web3 集成进行了取代。Telegram 理论上可以实现自定义 NFT 配置文件，并使用自定义功能启动。此外，这些网络可以免费实现这些功能，就像他们提供给用户的所有其他功能一样。

然而，大公司通常较慢将这些类型的功能推出到他们的平台，这就是像 Collab.land 这样的初创公司被广泛使用的原因。此外，TikTok 等公司选择与 Audis 等现有平台合作，而不是在内部构建 Web3 功能。因此，虽然这些现有平台构成了一些竞争，但如果现有平台决定构建这些功能，M○C△ 可能会以比现有平台快得多的速度创新这些功能。

对 MOCA 的最终想法

加密艺术博物馆是加密艺术空间中一个非常独特的项目。它针对长期相关性进行了优化，建立了一系列极其相关且有价值的艺术品，在很大程度上避免了最近的流行趋势。如果该项目成功地分散了策展，社区将有一个令人印象深刻的收藏来管理，并有明确的进一步收购途径。在那之前，博物馆处于一个尴尬的中间地带：由一个小型、集中的团体管理，该团体无法保证其他小型收藏家 DAO 所具有的排他性。为代币提供实用性的近期升级很有可能为 M○C△ 生态系统带来新用户，增加代币需求并可能创造新的长期社区成员。如果出现这种需求，很有可能需要更新代币经济学，

最终，博物馆的最终目标似乎是可行的。他们有一个文化相关的、有价值的收藏品和一个在不久的将来有几个潜在用例的代币。缺少的部分是一个充满活力的社区，可以充分利用两者。在接下来的 3-6 个月内发布令牌的各种功能应该会大大改变这种状况。
