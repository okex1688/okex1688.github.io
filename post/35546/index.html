基于区块链技术的新一代网络正在赋予期望和快速发展。尽管这一趋势有反对者，但 Web 3.0 的元素已经在使用，包括新一批具有颠覆性的非银行竞争对手。

如果“Web 3.0”这个词已经引起您的注意，并且您想知道它是什么以及是否值得开始探索，我们有四个词给您：Eco、Valora、Compound 和 TrueFi。

Eco， 其宣称它不是银行也不受 FDIC 保险保护的事实，它提供以稳定币持有的账户，既可以作为支出账户，也可以作为储蓄账户，但回报更高。

Valora 是一款可通过 App Store 和 Google Play 获得的应用程序，它使用区块链从移动加密钱包中提供一个 P2P 服务版本。

Compound 将自己描述为“一种为开发人员构建的算法、自主利率协议，用于解锁一系列开放金融应用程序。” 使用它的服务包括 Compound Treasury，它承诺将美元存款转化为稳定币，年化回报率为 4%。

TrueFi 是一家市场贷款机构，为借款人提供无担保贷款，并为有资金的人提供投资贷款以获得更高回报的机会。TrueFi 是一种使用稳定币的去中心化金融形式——“DeFi ”。

许多技术和加密货币投资者支持这些新兴公司，所有这些公司都通过“Web 3.0”的某些方面开展业务，也称为“Web3”。一位共同的投资者是安德森霍洛维茨。

这家风险投资公司正在通过对冲基金大力投资加密货币。2021 年底，它发起了一场游说活动，以支持其关于万维网应该如何发展的想法，以及与之相关的许多业务和活动——包括金融服务。迄今为止，这一推动包括为塑造网络未来的世界领导人发布十项Web 3.0 原则，并发布主要的Web 3.0白皮书“如何赢得未来：第三代互联网的议程”。

关于金融服务，该文件指出：“我们的金融系统需要进行改造以实现快速支付。它还需要更具包容性；例如，通过整合更多数据源的更好的信用评分来改善获得信用的机会。Web3 展示了替代当前系统的潜力。去中心化金融或 DeFi 也在提供一波新的基础设施，以支持更复杂的金融产品……”

这篇论文中还有更多内容，但它远非华盛顿推广 Web 3.0 的唯一声音。反银行言论是其中的一部分。

在关于Web 3.0的国会证词中，前货币署审计长布莱恩布鲁克斯（Brian Brooks）放开了全国性银行对加密货币活动的处理，他表示：“我们是否相信用户控制的去中心化互联网优于主要由五家大型公司控制的互联网？……我们是更信任大银行，还是更信任开源软件，作为维护账户账本和分配信贷和资本的工具？” 布鲁克斯现在是加密货币业务供应商 Bitfury Group 的首席执行官。

显然，传统金融机构以及新银行和金融科技公司都必须了解 Web 3.0 是什么以及它如何发展。以下问答介绍了已经成为一个非常复杂的新领域的基础知识。

颠覆似曾相识：

人们意识到许多银行错过了 PayPal 和 Chime 的潜力，直到它们成为巨大的竞争威胁，这种将 Web 3.0 金融服务视为边缘工作的趋势应该有所缓和。

Web 3.0 还包括元宇宙，但远不止于此。

什么是 Web 3.0？（技术答案）

有两种方式来看待 Web 3.0。一是了解是什么让其开始阶段发挥作用。另一个是考虑它的支持者认为它将获得什么成就以及以什么方式。

让我们开始探索第一个。

Web 3.0 的支持者说这是万维网发展的下一步。下图显示了我们今天拥有的广泛使用的 Web 2.0，以及我们认为理所当然的包含音频、视频和更多内容的高度链接网页，是如何从现在原始的 Web 1.0 发展而来的。

Web 3.0 基于区块链技术。因此，它打算如何运行的一个关键因素是它将是去中心化的。区块链旨在独立于政府等中心化机构甚至谷歌等主要互联网平台独立运行。事实上，正如 Investopedia 所暗示的那样，Web 3.0 “将打破目前由 Facebook（现为 Meta）和谷歌等互联网巨头持有的海量数据库，并将更大的控制权交给用户。” Web 3.0 旨在基于任何开发人员都可以利用且“无需信任”的开源软件运行。根据 Investopedia 的说法，这意味着“任何人都可以在未经管理机构授权的情况下参与”。

一些现状：

Web 3.0 中进行的大部分交易都涉及加密货币。加密货币信徒和 Web 3.0 粉丝之间有很强的重叠。

在大多数情况下，交易的价值是“代币”，是加密货币的同义词。以太坊加密货币是使用的关键加密货币之一。

在某些方面，Web 3.0 的各个方面就像科幻迷所说的“另类现实”。例如，在当前的网络世界中，我们有“应用程序”。在 Web 3.0 中有“dapps”，它代表去中心化的应用程序。从理论上讲，这些都是由用户构建和拥有的，尽管有些人怀疑像现在的网络和应用程序世界这样的公司不会这样做。

虽然想法是没有中心，但治理在某种意义上确实存在，以“DAO”的形式存在。这代表去中心化的自治组织。这些机构的主要作用是开发和维护管理每个区块链的数字协议。

正如多位评论家所指出的，对于互联网上的普通人来说，他们是否会在没有第三方提供便捷访问的情况下参与 Web 3.0 的可能性是值得怀疑的。而且，正如本文开头的金融服务示例所强调的那样，可以赚钱。

什么是 Web 3.0？（哲学答案）

第一个答案从高层次上了解 Web 3.0 的具体细节，但对“为什么”没有太多了解。在大约二十年的时间里，我们逐渐接受了围绕网站和移动设备构建的生活，以及以前从未梦想过的连接。

NPR 将 Web 3.0 描述为“不同想法的总称，所有这些想法都指向消除互联网上的大型中间商。在这个新时代，浏览网络不再意味着登录 Facebook、Google 或 Twitter。……Web3 是为了夺回一些力量。”

或许。事实是，随着 Web 3.0 的构成更加明确，科技巨头们已经在为 Web 3.0 而战。2021 年底，随着比特币倡导者、Twitter 和 Square（现为 Block，来自区块链）的创始人 Jack Dorsey 开始批评 Web 3.0 如何融合在一起，其中一些问题浮出水面。

对于所有关于去中心化甚至“网络民主化”的讨论，多尔西通过 Twitter 表示，他相信这些目标已经结束。

“你不拥有’web3’。” 风险投资家 和他们的有限合伙人会这样做。它永远不会逃脱他们的激励。它最终是一个具有不同标签的中心化实体。知道你在做什么……”

——杰克·多尔西，布洛克和推特

包括 Andreessen Horowitz 在内的 支持Web 3.0 力量进行了反击，而社交媒体和其他地方的争论仍在继续。与此同时，随着时间的推移，消费者会有更多的问题。例如，如果我们担心在当前网络上暴露的信息通过区块链传输，这是否会使我们的个人数据更容易泄露给世界？

支持者对 Web 3.0 的愿景是什么？

愤世嫉俗的人会说，当风险资本家支持某件事时，那是因为他们认为他们会从中赚钱。然后是这群人认为你可以“把好事做好”。

Andreessen Horowitz 的论文将 Web 3.0 视为一个从 Web 发展至今的位置进行重置的机会。

“Web 2.0 改变了我们的经济和社会互动方式，使社会受益匪浅。与此同时，很少有人会争论 Web 2.0 社交媒体和当今的大型科技平台——在此过程中转向了错误的方向。”

——安德森霍洛维茨报告

更具体地说，该论文认为，在 Web 2.0 世界中，无论是公共部门还是私营部门，都无法解决诸如数据泄露、通过网络传播的彻头彻尾的谎言、垄断和带有偏见的算法等问题。

“与此同时，威权政府从未有过比现在还要多的数据可以用来监视、审查和操纵其公民和其他国家的公民，”该公司补充道。

随着 Web 3.0 的巩固，Andreessen Horowitz 倾向于看到以美元计价的稳定币和央行数字货币并存。虽然银行最近进入了稳定币市场，但银行政策研究所将 CBDC 视为破坏传统银行系统的目标。

该公司的论文并没有主张采取自由放任的方法，而是主张政府广泛参与 Web 3.0 的演变，并提出了新的监管方法。

Web 3.0 反对者的观点

潜伏在网络下一次迭代可能带来的所有好处背后的信念是，真正的动机是“我们如何将其货币化？”

纽约大学斯特恩分校营销学教授、博主 Scott Galloway 将 Web 3.0 称为“一个模糊/含糊的术语，描述了加密货币驱动的互联网，以及一种瑜伽语字体。”

他在同一篇博客中谈到了Web 3.0中的领导力发展：“新卫士看起来也更老，而且比老卫士更像是卫士。具体来说，来自斯坦福/哈佛的家伙们将运气与天赋混为一谈，并在一个任务中服务于拯救我们的错觉，同时意外地赚了数十亿美元。”

关于去中心化的促进好处，加洛韦指出，“中心化”听起来令人窒息和腐败。但他指出，历史给我们上了一课。

“狂野西部时代是去中心化的。我们很快发现城镇需要治安官和中心协议（即法律）。盗窃在 Web3 上猖獗。2021 年，加密货币诈骗和盗窃造成的损失总计 140 亿美元。”

— 纽约大学斯特恩分校的 Scott Galloway

Galloway 远非唯一的 Web 3.0 怀疑论者。例如，存在关于 Web 3.0 是否真的是去中心化的问题。

华盛顿有什么会影响银行业？

国会已经提出了大量关于加密货币等的立法，但可能会也可能不会进一步。可能影响事情的一个因素是一项未发布的总统行政命令，涉及加密货币、CBDC 和相关问题。文件泄露给人们留下的总体印象是，拜登政府可能希望 CBDC 的发展速度比美联储快一些。另一方面，政府已经表现出对金融服务加强控制的愿望，这与 Web 3.0 支持者所青睐的大部分内容背道而驰。
