作为Cosmos生态的EVM网络，Evmos是近期最受关注的项目之一，它将通过 IBC 与以太坊主网、EVM 兼容环境和其他 BFT 链互操作，使用户和开发人员可以轻松地在链之间进行无缝交互。Evmos 旨在成为 Cosmos 的 EVM 中心，使智能合约在 Cosmos 生态系统中的部署和通信变得容易。

Evmos在多次推出主网上线时间后，于今日正式上线主网并开启空投。 目前，多个知名加密项目已经宣布支持该网络，同时也有原生应用在DEX、借贷等领域诞生，可以预见其中将涌现诸多热门项目。

为了帮助大家进一步了解Evmos生态发展现状，链捕手将在本文对目前的Evmos生态项目进行全面的梳理与介绍。

一、原生应用：

Kinesis

Kinesis是建立在Evmos上的稳定币交易协议，允许用户以最低的费用和最低的滑点将稳定币与其他稳定币交换。Kinesis 不仅计划成为 Cosmos 上稳定币交易的首选目的地，而且还将成为将稳定币引入 Cosmos 的积极推动者。

Kinesis Labs 计划积极与流行的稳定币协议（如 Abracadabra、Alchemix、Fei 和 Frax）合作并参与治理，以将 EVM 的所有稳定币引入 Cosmos。该项目还将与稳定币发行方密切合作，以确保稳定币有足够的流动性。

官方推特：https://twitter.com/KinesisLabs

Cronus Finance

Cronus Finance 是基于 Evmos 构建的完全无需许可且可组合的 AMM DEX项目，将在 Evmos 主网上线后很快启动，大部分代币将通过空投发行。

官方推特：https://twitter.com/cronusfinance

Exswap

Exswap 是在 Evmos 上推出的 Uniswap v2 分叉，计划在 Evmos 主网上线后不久推出。该项目将是一个公平启动的以社区为中心的项目，计划在推出后几天内发行治理代币EX，大部分代币供应将通过空投、LP 激励和农场分配给社区，45%的代币计划向流动性提供者分发，20%的代币计划用于多次空投，没有预售。该项目还将推出矿池、借贷与启动板功能。

官方推特：https://twitter.com/Exswapdotxyz

FantasticSwap  

FantasticSwap 是基于Evmos的AMM DEX，该项目已宣布将于近日启动IDO，计划通过销售治理代币FANTA募集10万美元，没有私募，10% 代币用于开发资金，90%代币用于LP挖矿。

官方推特：https://twitter.com/Fantastic_Swap

Coslend  

Coslend 借贷协议是基于 Compound 的 Evmos 生态系统上的货币市场，旨在打造一个安全、公开、透明的去中心化金融货币市场，允许用户提供和借入资产，同时以完全无需许可和去中心化的方式接收或支付利息。

官方网站：https://coslend.com/

Diffusion

Diffusion 是 Evmos 的 Uniswap v2 分叉，预计3月中旬发行代币DIFF并进行空投，最大供应量约3.2亿枚，空投数量为2500万枚，空投条件包括在去年12月31日前持有超500个UNI或累计gas费超1个ETH的Uniswap用户、OSMO质押者、Evmos质押者及Osmosis 上的 Evmos LP等。

官方网站：https://diffusion.fi/

EvmOS Punks

使用 Cosmos SDK 构建的 EvmOS 链上的 Punks，计划在2月28日开放铸造，并在3月10日发行治理代币 EVMP，该代币是构建朋克未来的基本单元。然后它将允许在 Punk DAO 中进行投票，以决定 DAO 国库资金的用途，或为项目的继续进行做出决定。

Expanding space

Evmos 上的第一个 NFT 市场，用户可以创建、购买和出售 NFT，正在开发拍卖系统以及用于进行 NFT 收藏品初始销售的队列系统。该项目未来可能发行代币并空投。

官方推特：https://twitter.com/expanding_space

EVMOS Shiba

EVMOS Shiba是一个以社区为中心的 meme 代币，它将 Shiba 的热情和可爱带到了 EVMOS。

官方推特：https://twitter.com/EvmosShiba

2月初，Evmos 提议在 Evmos 区块链上部署 Aave v3，以扩大对 Evmos 和更大的 Cosmos 生态系统的借贷。Evmos 还将使跨链应用程序能够利用 AAVE 的流动性，同时部署到 Terra 等其他 Cosmos 链。该提案已获得 Aave 社区投票通过。

二、宣布集成的应用

Frax

Frax 是第一个部分算法稳定币协议。Frax 是开源的、无需许可的，并且完全在链上——目前在以太坊和其他链上实现。Frax 协议的最终目标是提供一种高度可扩展、去中心化的算法货币来代替 BTC 等固定供应的数字资产。

2月中旬，Frax Finance 宣布将在 Evmos 上部署 FRAX 稳定币。部署后，开发人员和最终用户将能够在 Evmos 上构建和使用 DeFi 协议时利用 FRAX。用户还可以通过 IBC 在整个 Cosmos 生态系统中发送 FRAX 和 FXS（Frax 的价值增值和治理令牌）。因此，用户可以使用 FRAX 与其他基于 Cosmos 的链上的 DeFi 协议进行交互。

Aave v3

2月初，Evmos 提议在 Evmos 区块链上部署 Aave v3，以扩大对 Evmos 和更大的 Cosmos 生态系统的借贷。Evmos 还将使跨链应用程序能够利用 AAVE 的流动性，同时部署到 Terra 等其他 Cosmos 链。该提案已获得 Aave 社区投票通过。

dForce  

3月初，DeFi 协议 dForce 宣布其全套 DeFi 产品即将上线 Cosmos EVM Hub Evmos，包括借贷协议、去中心化 Stablecoin USX、跨链桥 dForce Bridge 等。

SpaceFi

SpaceFi 是一个社区驱动的 DEX，包含 Farm、NFT、Starter 和 DAO等，此前已在 Celo 公链推出测试网1.0近一个月。2月下旬，该项目宣布将在在 Evmos 与 Celo 上推出测试网2.0。

Thorus

Thorus是一个建立在Avalanche与Moonbeam多合一的跨链DeFi平台，目前总TVL为730万美元，近日宣布将集成Evmos。

三、各类基础设施

Nomad与Connext

Nomad 是一种新颖的互操作性协议，旨在降低成本并提高跨链消息传递的安全性。与基于验证者的跨链桥不同，Nomad 不依赖大量外部方来验证跨链通信，而是通过利用一种optimistic机制，让用户可以安全地发送消息和桥接资产，并保证任何观看的人都可以标记欺诈并保护系统。Nomad 所做的主要权衡是引入一个争议窗口来解决欺诈证明。

Connext 解决方案允许用户快速、廉价地跨链转移小额到大额的价值。该解决方案旨在用于高频传输，并通过利用发送和接收链上的路由器提供的流动性池来工作，在 L2 或其他系统之间移动时不会触及以太坊 L1，也不会引入任何新的信任假设。

今年1月，Evmos宣布 Nomad 和 Connext 将把他们的桥接解决方案部署到 Evmos。借助 Nomad 和 Connext，机构和最终用户将能够在以太坊和 Evmos 生态系统之间发送价值和调用数据。这将允许以太坊和其他 EVM 链的 ERC-20 代币在 Evmos 上可用，随后通过 IBC 在整个 Cosmos 生态系统中可用。

Celer Network

Celer Network 的 cBridge 是 DeFi 中领先的多链桥之一，锁仓总价值超过 4 亿美元。3月初，该项目表示 cBridge 将部署 Evmos 主网上，允许任何人将 USDC 和 USDT 等资产连接到 Evmos 和整个 Cosmos 生态系统。

UMA

UMA 是一个optimistic 预言机，它使用经济担保来保护市场。与机械限制的价格供给预言机不同，乐观预言机能够为链上的任意数据提供服务。UMA 灵活的预言机为跨链桥、保险、定制衍生品和预测市场等用途提供数据。

今年1月，Evmos 表示 UMA 将为 Evmos 开发人员提供去中心化且经过实战考验的 optimistic 预言机，开发人员可以使用 UMA 乐观预言机构建一系列去中心化金融应用程序。

Pocket Network

Pocket Network是一种 RPC 中继协议，它激励全节点运营商为去中心化应用程序提供准确的数据和丰富的带宽。包括 Ethereum、Polygon、Solana 和 Harmony 在内的 20 多个区块链都利用了 Pocket Network 的去中心化基础设施。

2021年12月，Evmos 宣布与 Pocket Network 建立合作伙伴关系，为 Evmos 开发人员提供必要的去中心化节点基础设施，以便在第一个 EVM 兼容的 Cosmos 区块链上部署他们的去中心化应用程序。

Covalent

Covalent 是行业领先的区块链索引器，可让开发人员通过其统一的 API 访问市场上最详尽的区块链数据。开发人员具有独特的能力，可以通过单个 API 调用在所有受支持的区块链上使用丰富、结构良好的数据快速构建 dApp。

2021年12月，Evmos 宣布与 Covalent 集成，为 Evmos 开发人员和最终用户提供了一个统一的 API，该 API 可以跨多个领先的区块链访问索引数据，以便开发人员将能够快速且一致地访问索引区块链数据，以在 Evmos 区块链上开发和部署复杂的去中心化应用程序。

Blockdaemon

Blockdaemon 将企业连接到区块链，是全球领先的区块链基础设施提供商。他们的客户包括主要交易所、托管机构和金融机构。

2021年12月，Evmos 表示 Blockdaemon 将为 Evmos 提供可靠、安全、高可用性（HA）的 RPC 集群和节点基础设施平台，将允许 Evmos 代币持有者轻松运行自己的验证节点。

KYVE

KYVE是使数据提供者能够标准化、验证和永久存储数据流的下一代协议。通过利用 Arweave，该解决方案可确保这些资源随时间推移的可扩展性、不变性和可用性，并且通过使用 Moonbeam 智能合约，KYVE 与 EVM 兼容。

2021年12月下旬，KYVE 正在将 Evmos 集成到去中心化区块链数据流的 KYVE 网络中。KYVE 将帮助可靠、准确地存储来自 Evmos 区块链的区块链数据。Evmos 社区将能够利用这些数据在 Evmos 区块链上构建和部署复杂的应用程序。
