原文标题：《 重新解构 NFT：Token 化商业的哲学思维！ 》

原文整理：老雅痞 授权转载

内容来自对《Not Boring》访谈栏目的整理。

对话来自主持人 Packy McCormick 和 Alex Danco。Alex 就职于 Shopify，职位是 Systems & Crypto person。

希望可以给你带来新的视角和启发。

主持人 ：我们都为加拿大人工作——你在 Social Capital，我在 Breather——然后我们都写了时事通讯。从那以后，我们朝着不同的方向前进——我正在做一个时事通讯和一家基金，而你已经把时事通讯放慢了更新速度，并成为 Shopify 的全职运营商。

我记得当你接受这份工作时，我在想，a) 太酷了！b) 你将在 Shopify 做什么具体工作呢？c）Shopify 会到达哪里呢？对于那些没有使用 Shopify 的人，你能否定义 Shopify 今天是什么以及你认为未来十年它将成为什么？

Alex ： 我简单的描述 Shopify 吧：它让每个人都能更好地开展商务活动，具体而言，我们通过软件做到这一点：我们在建立电子商务平台。我们的工作是以一种独特的方式来构建该平台，为商家——以及许多其他人服务，如应用程序开发人员、代理商和现在的买家——我们会提供最好的互联网软件服务。

主持人：Alex Danco，你在 Shopify 做什么？当然，除了 Shopify wallet-aware。

让我们直接进入 Web3。几个月前你第一次发推特「gm」时，我以为你在讽刺。然后你改变了你的头像。现在你正在运营一个区块链团队——你在 LinkedIn 的头衔是「Systems & Crypto person」。你接触加密货币有一段时间了？你对它有什么兴趣？

Alex ： 哈，这真的很有趣，因为我甚至没有想到它可能会引起任何人的注意。我涉足加密货币已有一段时间了；几年来，我一直在我的时事通讯上写这件事。

至于，「我对它有什么兴趣？」我不知道。应该是一切都有兴趣。以下 是 我最喜欢思考和写作的一些例子，但我想最重要的是，加密世界正在寻找它的目的，这应该是你想知道的。通常大家可以在三种地方消磨时间：

  • 没有目的的人

  • 有既定目标的人

  • 那些正在发现他们的目的的人。

你告诉我哪个听起来最有趣。

主持人：我将开始使用它，而不是「我们来得太早了！」

对于很多人来说，「发现它的目的」意味着找到「真实的用例」。

为此，从加密世界的角度来看，Shopify 是贸易科技中更有趣的地方之一——加密和商业密切相关，你们 CEO Tobi 似乎对它真的很感兴趣，毕竟有超过 100 万家企业在 Shopify 上运行。鉴于它的规模和范围，感觉 Shopify 可以将其合法性借作为用例，即如果 Shopify 支持某些东西，它就是一个真正的用例，因为 Shopify 支持它。

拥有权利的同时也被赋予了重大的责任。你是如何开始考虑是否参与以及在该领域做什么的？

Alex ：  我在两年前加入 Shopify，发现了一个有趣的现象，即 

1）很多非常熟悉加密货币的人分散在公司中，但他们也 

2）没有「对齐」的驱动力，大家都不知道除了交易加密货币，还能做些什么。

几年来，Shopify 商家已经能够接受加密支付，这要归功于第三方应用程序开发人员构建的支付通道（而我们只是构建了整体商业平台！）。

但去年当 NFT 起飞时，我们真正点燃了寻找这个问题的答案：这是我们平台必须为商家提供的「最好的互联网版本」的一部分吗？答案显然是肯定的，尽管我们花了一分钟才发现原因。

主持人：确实。我认为，该公司发布的第一个加密新闻是芝加哥公牛队 NFT 商店。当我想到 Shopify 的角色从那里开始时，我认为你在该领域的大进军将是沿着这些路线——NFT mint、drop、销售。你帮助企业销售东西——包括实体的和数字的——所以也包括帮助他们销售高利润 NFT。那不是你走的路。为什么？

Alex ： 好吧，帕基，这就是我想和你谈谈的关于钱的问题。回到我们的主题，而不是直接给你一个简单的答案（「因为我们比任何人都意识到，Token 化商业将是比现有商业模式大 100 倍的机会，它对我们来说比其他任何东西都更有趣」）。

主持人：几周前，乔纳森·海特 ( Jonathan Haidt) 的那篇大文章中有一句话让我立即想到了加密货币：「历史是有方向的，它是朝着更大规模的合作方向发展的。」加密世界是对更大规模合作的赌注，而不用为自以为协调一切的公司支付费用。你怎么看？

Alex ：  我特别喜欢这个答案的「协调」部分；郑重声明，我认为这当然没有一个正确的答案——关于区块链和加密货币在这里为我们做什么，有很多不同的社区和思想流派。这是好事。但我会告诉你我的，我认为这涵盖了很多重要的事情。

当我们关注加密世界「正在发生」的变化时，我们关注两件事：

  • 钱包改变了人们的行为方式， 

  • 随着时间的推移，这种行为会获胜。

主持人：我喜欢这个答案，因为它融合了社交和技术，以及计算机和人的交互方式。「首先我们塑造我们的界面，然后我们的界面塑造我们。」

Alex ：  我认为它很好地回答了这个问题，你不觉得吗？它为每个人提供了一些东西。拥有 Sound Money 的人有一些东西（我们的钱包和加密密钥对改变了我们储蓄和消费的行为，从长远来看，这种行为将胜过旧行为），dapp 建设者有一些东西（钱包改变了人们创造的方式和消费软件），社区有一些东西（钱包改变了人们收集和展示他们身份的方式）。

我喜欢这个起点，因为它将我们聚集在我们有共识的事情上。加密从根本上说是一种行为，与它是一种技术的属性一样多，甚至可以说更多。

这种行为有很多方面，但它从密码学开始；它从我们的钱包开始，当我们签署一些东西时。它从某个时刻开始，那是我们的钱包打开一扇通往新世界的大门并踏入其中的神奇时刻。

我们正在研究那个世界里的一些很酷的东西。我们称之为 Token 化商业。

Token 化商业始于一个观察：世界上有一种新的互联网用户，称为有钱包的人。

主持人：我喜欢这个框架：有钱包的人。对加密让人们做什么的描述很常见，但我还没有看到任何描述加密让人们成为什么样的描述。

Alex ：  就是这样。暂时忘记你能做什么，让我们谈谈你能成为什么样的人吧。 

我会告诉你答案：你会是一个有钱包的人。

让我们来谈谈这些有钱包的人。他们是谁？他们是所有者。也许他们拥有一些表示社区成员身份的 NFT、一个 ENS 名称、一个 ETH 或 USDC 余额，也许是一个与他们相关的智能合约。他们可以在互联网上签名。

这些人的行为方式很有趣：他们真的想把钱包和东西联系起来。他们总是将他们的钱包连接到 dapps，或者将他们的钱包连接到 NFT 市场。

怎么会这样？因为我们通过钱包拥有的东西意味着我们本身是什么。当我们带着钱包出现时，我们就以我们的身份出现。当我们连接我们的钱包，并感受到世界突然改变的神奇时刻——它看到了我们，打开了——那是一种非常棒的感觉。感觉是这样的：

这张照片中发生了什么？这是去年 3 月在 SXSW 举行的 Doodles 活动，这是一个由 Web3 社区 Doodles 创建的弹出式世界。

该活动向所有人开放，但如果你是 Doodle 持有者，那么这个世界上的各种体验——从入口一直到 Shopify 支持的礼品店——都以令人愉快的方式向你「开放」。灯光闪烁，气泡吹响，特殊的通行证握在手上；你拥有的 Doodle 涂鸦出现在一个巨大的屏幕上，向全世界宣布：「我是 Doodle NFT 持有者 3125 号！」一个又一个，客人的脸上洋溢着喜悦；大家都成功的进入了这个新世界。

这一刻就是 Token 化商业的全部意义所在。你会意识到，「啊。我现在总算是了解 NFT 和商业之间关系的人了。」NFT 不是商业的输出；它们是商业的输入。NFT 不是一种产品。他们是一种买家的身份象征。

主持人：我喜欢你提到的两件事，

1. 体验可以对任何人开放，但对 NFT 持有者来说更特别。我认为音乐 NFT 真的很有代表性：任何人都可以听，少数人可以明显地成为鼎力支持者，并且可以访问特殊曲目和见面会等……

2. 我又说回来了。加密让人们成为什么。

Alex ：  成为不可替代的买家是件好事。你到达店面，连接你的钱包，突然店面根据你持有的 NFT 发生变化。也许一个特殊的产品被解锁，或者你可以提前获得一个 drop。也许你连接了你的钱包以解锁你的 NFT 对应的高质量艺术印刷品或 3D 模型。也许你可以访问一个特殊的合作：对于这个限量版的 drop，你的 NFT 为另一个社区打开了一扇新的大门。你探索的越多，你找到的门就越多。

具体而言，NFT 已被证明是这种行为的完美催化剂。你知道为什么？想想最成功的 NFT 品牌是什么样的。它看起来像一个主题和变化：这就是你适合的，这就是你脱颖而出的方式。

它们是那种成就感的身份象征：我在 Doodle 中，我有这个 Doodle。我在 Adam Bomb 小队中，我有这个 Adam Bomb。我在 Stapleverse，我有这只绑着头巾的鸽子。NFT 是互联网上「适应并脱颖而出」的新含义。

主持人：我认为，这又是规模点上的合作之一。在当地社区，我们自然是不可替代的买家。你去你最喜欢的社区商店和餐馆，那里的人认识你，在你说话之前就知道你要点什么。你是当地社区的一员（你「融入」进来了），但你就是你，有自己的喜好和明显的风格等等（你「脱颖而出」）。

Alex ： 是的。事实证明，「适应，然后脱颖而出」的行为是世界上如此多美好事物——文化、社区、商业——的最重要的原材料之一。现在我们有了钱包，里面有 NFT 清单，它代表了这种精确的原子行为单位：适应，然后脱颖而出。

当然，这种「融入，然后脱颖而出」的感觉并不是加密货币独有的。这是一种普遍的感觉，当你回报它时，它会变得更好。

我的同事妮可有一个轶事，关于带着大丝芙兰包（不是小包，当你买太多东西装不进小包时你得到的大包）在商场里走来走去， 并认识到别人也有一个大包。她们俩立即想到了相同的两件事：1，好棒你也背同款；2、你的包里有什么？当我们团队中的一位工程师看到其他人戴着工程师勋章戒指时，她也有类似的感受。我以前在乐队里巡演，现在每当我在街上看到有人穿着演唱会 T 恤时，我都会有那种感觉。

这种认可是有感染力的，因为你们俩都感觉很好。这是一种相互包容的感觉：我让你进来，你也让我进来。这是我们一起做的挑战：我们都适应并脱颖而出。你可以在每个社区看到它：运动员、收藏家、喜欢乐队和现场音乐的人，以及喜欢漫画书的人。在任何你看到这种感觉的地方，商业都指日可待。

伟大的品牌非常了解这个概念。当你去他们的旗舰店购物时，很明显在营造这种相互认可的成就环境中投入了多少心血和工艺：不仅在装饰和美学上，更重要的是，在你与商家的互动中。可以很简单，就是商家看到你的 T 恤，点点头，不错。

尤其是因为这种认同获得了一些额外的折扣资格后，「那件连帽衫是什么？18 年春季系列？好的。嘿，我们在后面为你准备了一些东西，我想你会喜欢的。」

Bobby Hundreds（来自洛杉矶街头服饰品牌 The Hundreds，当然还有 Adam Bomb Squad）写了一本书，名为《这不是一件 T 恤》，我强烈推荐这本书，它确实强调了这一点。

主持人：对！所以我猜这是第二层意义——在你生活的现实社区，人们可能会认出你是你。再往上一层，无论你在物理世界中走到哪里，其他喜欢你所做事情的人都可能会认出你是一个志同道合的人。但这在网上很难做到。

Alex ： 是的。在钱包出现之前，那种被人看到的神奇时刻很难在网上创造出来。在线店面很难对买家进行氛围检查，并以认可和庆祝社会契约的方式回应买家。客户帐户和忠诚度计划不会削减它；他们不是你，与你的钱包不同。客户帐户基本上是商店的一部分，而不是买家的一部分。相比之下，「连接你的钱包，得到认可」是一种特别的感觉。这是不一样的。

主持人：好的，这是有道理的，但对我来说还有一个很大的悬而未决的问题。当我读到「tokengated commerce」或「x-gated commerce」时，首先想到的是，「你到底为什么要结束你的商业行为？如果你在卖东西，为什么不尽可能接触最广泛的受众呢？为什么不直接一键结账？」

 我正要问你，你之前曾经讲过很多关于保持一些障碍，以建立对你归类为「高信任度商业」的事物的信任的重要性。 Token 化商业感觉像是一种故意为某些产品引入一点摩擦的方式。是对的吗？

Alex ：  100%，我认为解决这个问题的方法是将「摩擦」这个词替换为挑战这个词： Token 化就是为某些产品增加挑战。

世界的默认状态不是「每个人都想要你的产品」。创造需求是一门艺术：你创造一个故事和一个挑战，然后你将挑战包裹在买家和商家一起进行的冒险中。考虑限时抢购或限量发售；这些体验很有趣，而且它们都被包裹在一个故事中。你会成为一个成功获得 drop 的人，这就是一半的乐趣。

不过，这实际上是重新审视你关于加密和「协作」的好时机，因为我在 Token 化商业中最兴奋的事情之一就是合作。

合作是创意工作的命脉；询问音乐、街头服饰或任何场景中的任何人。但是在网上，你如何组织一个专为一群人提供的合作商品：我的粉丝和你的粉丝？没有传统的方法可以做到这一点，既排他又不笨拙。但是 Token 化商务使这变得如此简单。连接你的钱包，使用这些 NFT 中的任何一个访问 drop。我让你进来，你让我进来。就这么简单。

我对 Token 化商业如何帮助品牌相互认可的挑战特别感兴趣。我认为「代言」是合作的一个特殊子集，一位成熟的艺术家将与一位崭露头角的艺术家主持或联合开发品牌。（音乐产业在这方面做得特别好。）

我们应该问：一个社区如何支持互联网上另一个崭露头角的社区？通过 Token 合作：当两个品牌走到一起时，将排他性转变为互惠包容性。

什么是 Token 合作？当 The Hundreds 与 Deadfellaz 合作时，你可以使用他们任何一个系列中的 NFT 解锁限量版商品，这就是 Token 合作。当 Superplastic 与 Gucci 进行合作时，该合作仅限于秘密 SUPERGUCCI NFT 的持有者，这就是 Token 合作。

这就是魔法开始发生的地方：我属于我的社区，你属于你的社区，但是当我们一起合作时，我可以邀请你加入；你可以邀请我加入。Token 化的合作将排他性转变为互惠的包容性。像 Gucci 这样的百年品牌可以为 Superplastic 这样的非常火爆的品牌代言，他们的粉丝们都在邀请对方加入这种共享体验，以及他们现在拥有的共享社区。

主持人：所以有一个明显的问题，为什么 Token 化会使商业变得更好，为什么这与品牌忠诚度计划或促销代码不同？

Alex ： 所以，我可以给你一个逃避的答案，它再次强调了「挑战感觉很好」的想法——当你在珍贵的 NFT 上花费更多 ETH 时，用它解锁 Token 化产品感觉就像是一种成就。但是，要让 Token 化商业真正发挥其潜力，我们需要下一代更便宜的 Token（最终，便宜到可以忽略不计），其中许多 Token 将更加实用，而「挑战」更少是对它自己的奖励」，你知道吗？因此，从长远来看，我们需要一个更好的答案来说明令牌化将如何成功。

忠诚度计划、促销代码或任何其他类型的「基于帐户」的门槛系统都可以用于互惠包容机制——当然可以——但这里有一个问题。这是因为你的门槛规则将作为引用回帐户和权限 的依赖项存在，如果你尝试引入任何类型的有趣的多人游戏机制，它们会很快变得复杂。

想象一下，为四个不同品牌的粉丝开设一个弹出式店面，具有多层次的访问权限，让你尽早访问商品 drop。你在哪里「登录」，以及作为什么身份登陆？如果你是多个社区的成员怎么办 - 我需要每个社区的代码吗？你如何防止促销代码被滥用，你投入的变量越多，它就变得越复杂？我的意思是，你可以做到，但是建立在依赖关系上的社交门槛规则很快就会在复杂性下崩溃，当第四面墙被打破并且「独家社区氛围」被有缺陷的软件破坏时，这根本不是一个好的社区体验。

相比之下，Token 是一种不记名的工具—它更像是一个文件，或者你手里拿着的一张实体票。（这个文件类比以后会很重要。）NFT 不会「链接回」任何东西，它不是对一个或多个商家数据库上帐户条目的引用；这是票本身。

在我们称之为「单人模式」 （一个商家对一个单一的客户列表的门槛控制）的情况下，很难看出 Token 化的优势。但是，一旦你进入多人模式——多个商家通过一套社会规则与多组粉丝联系——门票是一种不记名工具，不会链接回任何东西，因此不会带来任何依赖——但仍然可以受到「门」的信任，即使它没有链接回任何东西（稍后会详细介绍，这是「为什么需要区块链」部分）是一个从根本上简化的机制。因为现在，Token 门槛控制规则可能变得超级复杂，但它们不会在依赖的重压下崩溃，因为没有什么可崩溃的。

主持人：聊到这儿，对我来说是一个小小的启发时刻。加密货币的一些杀手级用例可能很大、很狂野、难以想象，但也有一部分只是，「让同样的事情变得如此容易，以至于现在会发生更多的事情。」稳定币轨道就在这个阵营中。

Alex ： 我觉得这实际上抢占了我以后想主打的一项功能，所以我把它留在这里。但就目前而言——当我们回顾十年后的 2022 年并问：「那段时间里，什么是真正显而易见的并且摆在我们面前的答案呢？」我认为一个答案是，合作 Token 化门槛控制机制将极大地简化品牌之间的协作和相互认可的方式，而这种简化将意味着社区商务将获得更多的多人游戏机会。

此外，对于涉及重新混合社区能认可的任何事情，合作 Token 化门槛控制机制本质上是一种优越的形式：在我们的门槛控制合作 Token 化示例中，我们并不是要对一组特定的人进行门槛控制，而是要对一组特定的成就进行门控- holder，这是一种更加灵活和可重新混合的形式。

我想通过强调 NFT 是你的成就清单来加倍强调这个想法。NFT 代表你。

今天，我们看到 Web3 品牌和已经对其在线商店进行 Token 化的传统品牌非常适合这条路径，通过 Token 化应用程序合作伙伴将这些店面带入生活。这些应用程序合作伙伴的合作深度令人难以置信，我们正在与他们合作构建——稍后会更多。与此同时，我们刚刚发布了一个用于 Token 化商业的新移动应用程序的 Demo：Gated Merch Shop（简称「gm shop」），你应该去看看。在过去的几个月里，我们真的爱上了打造令人难以置信的、移动优先的、单一用途的、Token 化的商业体验。我为建立这个的团队感到骄傲；它会统治一切！

但这一切都只是可能性之门上的一道裂缝。后院发生的事情更有趣。

既然我们已经通过引入 Token 化商业设置了所有问题的注脚，以及这种融入和脱颖而出的想法对于商业的重要性，现在我们可以谈谈为什么这一切必须在 Web3 钱包和公共区块链上发生的重要性。

这里要记住的一句话是：「我们将使 Shopify  wallet-aware。」

要真正了解 Shopify，首先要意识到我们不仅仅是商家的在线商店服务；我们也是一个开发者平台。

每个店面都是独一无二的；如果你去任何一家优秀商家的店面，那不仅仅是开箱即用的 Shopify，他们可能正在运行来自不同开发人员（或他们自己构建）的十几个或更多应用程序，所有这些应用程序共同创造准确的业务体验（商家想要代表他们的业务）。当你注册订阅产品，或在店面浏览酷炫的内容，或连接你的钱包以解锁封闭式产品时，这一切都由代理商和应用程序开发人员提供支持——就像你在 iPhone 上所做的大部分事情都是由应用程序构建的一样开发人员，而不是 Apple。

好吧，我们想让钱包成为构建在 Shopify 平台之上的开发人员的新输入方式。你钱包中的 NFT 远不止「拥有或不拥有 NFT」。NFT 是信息；这是一个智能合约，这很重要。

这就是我们所说的「让 Shopify  wallet-aware」的意思。

当买家出现在店面时，如果他们带着钱包出现，他们的钱包可以签注所有东西，包括买家希望提出的所有公共、社会意义，都应该可供买家使用。

这些开发人员可以获取所有相关状态并构建可以将该状态作为输入的各种应用程序和机制。未来将会有一版非常成功的基于 Shopify 的应用程序为店面赋予新的权力，今天我们已经有四个令人惊叹的 Token 化应用程序（PERC）：Engage、Manifold、Shopthru 和 Lit Protocol——为商家提供 Token 化商业支持。

最好的 Web3 品牌不会等待。他们正在为 Token 持有者创造错综复杂的亲密体验——制作商品投放的新方法，品牌及其买家在多人模式下合作的新方法。拥有应用生态系统的美妙之处在于，开发人员可以重用、迭代、改进并最终与许多商家和品牌共享这些机制。

我，alexdanco.eth，适合 Adam Bomb Squad，我作为 Bomb #6233 的所有者脱颖而出，这是一款干净的蓝色迪士尼风格的陷阱设计。

这一切看起来可能真的有点愚蠢，但这里正在发生一些非常强大的事情。我们现在有了让计算机识别和理解这些社会成就单位的方法；这就是这些小块的意义，它几乎就像文件（它们很像文件！这就是我之前所说的！）以这种人类和计算机都可读的持久小格式存储社会意义。通过社会协调的一个小奇迹，它正在被广泛接受。

NFT（和区块链上的其他智能合约）是一种中介格式——位于建立这种社会意义的环境和部署这种社会意义的另一个环境（无论你连接钱包的地方）之间——这一事实很重要。这几乎是从 web2 倒退了一步，回到了一个应用程序直接通过 API 与另一个应用程序对话的世界，回到了更早的时候，我们得到了更深刻的正确性；我们重新发现的东西叫做可组合性。

这里的基本思想是，如果你希望一个应用程序成功地与另一个应用程序通信，那么这很容易；你只要告诉他们怎么做。如果你想让一个应用程序成功地与 100 个应用程序对话，那么你可以编写一个很好的 API，上面写着：「嘿，一百个应用程序，这是与我对话的方法。」

但是如果 100 个应用程序想与 100 个应用程序对话呢？需要维护很多连接。因此，我们可以使用其他东西：files。如果我想将信息从一个应用程序传递到另一个应用程序，我可以将其作为每个人都能理解的 CSV 文件传递；在制作文件的人和使用文件的人之间没有创建依赖关系。

所以这里要理解的一个重要原则是：严格来说，对于任何一项任务或任何一项要完成的工作，让 App 1 直接与 App 99 对话将比通过中间文件「更好」。但随着时间的推移，文件的力量会通过标准的力量显现出来。因为它们有一个共同的格式，他们就如何相互阅读和写入达成一致，并且这些内容存在一个共同的地方；在磁盘上，或在网络上，或在可访问和持久的地方。

所以如果我们回到现在，问：「什么是 Web3？」，对我来说，最实际的思考方式是「Web3」只是作为一个环境，围绕一套新标准建立共识：一切都是智能合约地址，一切都是 tokenID，一切都是钱包地址。

我认为这对于商业来说将是非常疯狂的。尤其是最特别和最有意义的社区商业，我们不断创造和重新组合新的方式来表达我们的身份：我们如何融入并脱颖而出。最好的部分是，这个想法是如此开放和可发展的，以至于没有任何一家公司或没有任何一个应用程序可以独立「解决」这个问题。这是不可能的。文化太有趣了；我们需要许多应用程序、许多开发人员、许多尝试来捕捉这些社区表达形式——从 Token 化开始，然后扩展到「Token 化实际上可以意味着很多东西」。
