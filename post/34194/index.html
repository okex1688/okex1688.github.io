这届冬奥会似乎比以往汇聚了更多的关注，冰墩墩就是最好的例子，一个看似简单的冰墩墩玩具，竟然一夜之间成为了社交资本。

不仅是实物冰墩墩被疯狂炒作，国际奥委会官方授权的「冰墩墩」NFT 更加夸张，99 美元的发售价在二级市场叫价已经到了 1000 美元。

国际奥委会不仅授权为 2022 年北京冬奥会发布了 NFT 版本的数字徽章，以本届冬奥会为 IP 开发的游戏《Olympic Games Jam: Beijing 2022》也于开幕式前一天正式上线。其中，链上 Token 与 NFT 功能，也被集成到了这款游戏中。

该游戏由总部位于香港的 Animoca Brands 公司旗下的区块链游戏工作室 nWay 基于 Flow 链开发**，在游戏中，玩家可以参与到一系列冬季冰雪体育赛事，在游戏竞技的过程中赢取国际奥委会授权的 NFT 数字徽章。** 接下来，律动 BlockBeats 将为大家详细介绍这款游戏，一起来感受下 NFT 游戏中的奥运氛围。

《Olympic Games Jam：Beijing 2022》

《Olympic Games Jam: Beijing 2022》由区块链游戏工作室 nWay 开发，在 Google Play 和苹果 App Store 推出，下载后，玩家可以使用 nWayPlay ID 登录，无需连接加密钱包。

通过该游戏，玩家在游戏竞技的过程中可将获得的游戏内 Gem 宝石，在 nWay 打造的 nWayPlay 市场中换取为国际奥委会官方授权的 NFT 数字徽章进行收藏或交易，也可在未来支持的 DEX 中转化为该游戏 Token 进行获利。

去年 6 月，nWay 与国际奥委会 (IOC) 正式签署了许可协议，宣布将推出以奥林匹克为主题的 NFT 数字徽章和多人在线区块链游戏，而 IOC 则从每次的 NFT 徽章销售中收取版税。

经过八个月的时间，该游戏已成功上线，并且官方并不打算在北京冬奥会结束后停止它，游戏的功能还将继续更新，包括新的内容、运动、模式和赛事，以保持比赛的新鲜感和吸引力。nWay 首席执行官 Taehoon Kim 表示，通过持续更新来支持这项运动，以让奥林匹克精神延续下去；并且《Olympic Games Jam: Beijing 2022》作为一款 Play to Earn 游戏，玩家在参与这场全球庆典的同时，还能获得一定的收益。

NFT 及游戏内货币系统

NFT BOX

NFT BOX（数字盲盒）是一些列 NFT PIN（数字徽章）的组合包。拿本届北京冬奥会的「冰墩墩」数字盲盒来说，官方于北京时间 2 月 12 日凌晨 1 时在 Flow 链上交易市场 nWayPlay 发售了总共 500 个 EPIC BOX，每个 99 美元，限购 5 个。不过上线后不久便销售一空，一「墩」难求现象再次在 NFT 市场上演。

该 EPIC BOX 内包含了 3 个官方授权的 NFT PIN：1 个 Epic 和 2 个 Rare，内容则包括 15 个版本的官方授权吉祥物「冰墩墩」表演各种冬季运动，如单板滑雪、自由式滑雪、俯式雪橇等，5 个版本的北京 2022 年冬奥会官方会徽和 1 个版本的海报。

NFT PIN

NFT PIN 是可收藏和交易的奥林匹克徽章的数字版本，于 2021 年 6 月在 nWayPlay 首次发售，其具有与物理徽章相关的各种属性，包括真实性、稀缺性和来源，稀有度分为了 Common、Rare、Epic、Legendary 四个等级，且每个都有独一无二的序列号，使它们成为真正的收藏品。

作为奥林匹克遗产收藏的一部分，NFT PIN 旨在庆祝过去奥运会的艺术和设计，并反映世界上最大的体育赛事的图形遗产，如海报、标志、象形图，以及有着 126 年历史的现代奥运会的吉祥物。

律动 BlockBeats 在 nWayPlay 市场购买的 2022 北京冬奥会 NFT PIN

如何获取 NFT PIN？

1）在 nWayPlay 市场购买 NFT BOX，获得 NFT PIN；

  1. 从 nWayPlay 市场上其他收藏家处购买特定的 NFT PIN；

  2. 从《Olympic Games Jam: Beijing 2022》游戏中获得奖励。

此外，NFT PIN 给《Olympic Games Jam: Beijing 2022》游戏玩家带来的增益 则包括：

1）获得参加高等级比赛所需要的「Ticket」

通过使用 Ticket，玩家可参加高等级比赛获得 Gem 奖励；

  1. 解锁独家游戏内物品

使用与特定 NFT 绑定的专属皮肤自定义用户的头像，解锁特定 NFT 独有的特殊运动装备；

  1. 获得额外奖励

所有玩家在参加比赛时都可以获得 Ticket 或 Gem，但 NFT PIN 持有者可以获得额外的奖励；

  1. 参加特别活动

将会有一些锦标赛提供独家奖励，这些奖励只对 NFT PIN 持有者开放；

  1. 抢先体验游戏

此前内测，NFT PIN 持有者可作为游戏的首批体验玩家。

Ticket

在更高级游戏模式中，玩家需要使用相应数量的 Ticket，才能开启对应的游戏模式。而在游戏中，玩家可通过 Free 模式获得 Ticket 奖励，以及装备 NFT PIN 来生成 Ticket。

装备 NFT PIN 后，每隔几分钟将会为玩家生成相应的 Ticket，生成的 Ticket 数和剩余生成量将显示在 NFT PIN 上方，生成总数则由其稀有度决定。一旦每个 NFT PIN 生成完它所有的 Ticket 后，就会转变成一个「Artifact」PIN，不能再生成任何 Ticket。

Gem

Gem 是该游戏内货币，玩家在获得 Gem 后，可在 nWayPlay 市场中换取为 NFT PIN 进行收藏或交易，也可在未来支持的 DEX 中转化为该游戏 Token 获利。

游戏 Token

该 Token 在 Uniswap、Pancake Swap 或任何其他 DEX 中还尚不可用，一旦可用，官方将发布 Token 智能合约地址。

游戏怎么玩？

该游戏分为 4 种模式，分别为 Free、Amateur、Pro 和 Champion。

1）Free（免费模式）

玩家不需要任何 Ticket，即可进行游戏；且最高可获得 12 个 Ticket 奖励。

2）Amateur（业余模式）

玩家需要 100 个 Ticket 才能开始游戏；最高可获得 600 个 Gem 奖励。

3）Pro（专业模式）

玩家需要 300 个 Ticket 可开始游戏；最高可获得 1860 个 Gem 奖励。

4）Champion（冠军模式）

玩家需要 1000 个 Ticket 可开始游戏；最高可获得 6600 个 Gem 奖励。

而在具体《Olympic Games Jam: Beijing 2022》游戏中，玩家则可参与多项冰雪体育赛事，包括 Snowboard - Slalom、Snowboard - Slopestyle、 Freestyle Skiing - Cross 和 Skeleton。每轮比赛，有多达 20 名玩家可一同参与进行角逐。

Snowboard - Slalom（单板滑雪 - 激流回转）

Snowboard - Slalom 在下坡比赛中往往设置几组闸门，运动员在门或杆之间滑行。闸门的间隔相距约 8-15 米，比大回转、超级大回转、滑降赛中的间隔更近，致使转弯极其紧凑和快速，因此需要平时大量的技能训练。

游戏开始倒计时，玩家可点击「Gain」按钮获得加速；通过旗帜附近时也可获得速度提升，但要避免撞到它们，撞到后玩家会因眩晕而减速。此外，在领先其他玩家完成之前应尽可能获得更多的积分。

Snowboard - Slopestyle（单板滑雪 - 坡面障碍技巧）

Snowboard - Slopestyle 起源于滑板和摩托车越野赛等极限运动，以赛道长、连续设置多组跳台和道具、充分展现单板动作技巧而成为最具观赏性的比赛项目。运动员将根据滑行过程中的翻转幅度、原创性以及质量得分，刚拿了银牌的苏翊鸣参加的就是这个项目。

Freestyle Skiing - Cross（自由式滑雪双板追逐赛）

自由式滑雪从上世纪中叶开始首先在美国发展起来，九十年代陆续被列入冬奥项目，由双板完成，运动员要借助滑雪板和雪杖于特定的雪场上做出指定和自选的动作，已经家喻户晓的谷爱凌和徐梦桃都是自由式滑雪的代表人物。

Freestyle Skiing - Cross 则是自由式滑雪中的一个小项，比赛在一个包含有不同回转、跳台、波浪和其他形式地貌的赛道中进行，场地上设有旗门，连续旗门有颜色间隔。

Skeleton（钢架雪车）

Skeleton 起源于北欧，是一项以雪橇为比赛工具的冬季运动项目，运动员需俯卧在雪橇上滑行，最高速度可达至每小时 130 公里以上。由于滑行时头部朝向前方，俯式雪橇具有极高的危险性，因此曾长期被排除在正式的比赛项目之外。

基于以上概述，律动 BlockBeats 通过下载试玩了《Olympic Games Jam: Beijing 2022》后，发现该游戏目前还处于非常早期阶段，玩家不是特别多，匹配的过程中会分配到一些「机器人选手」。在同时完成 4 项赛事后，律动 BlockBeats 获得了 3 个 Ticket 的奖励。

此外，据官方介绍，该游戏还将结合当下流行的 NFT 头像化身玩法，玩家在游戏中可使用与特定 NFT PIN 绑定的专属皮肤自定义用户的头像或化身，解锁特定 NFT PIN 独有的运动装备，甚至扩展到角色强化，如更快的速度、更严格的控制或更强的危险抵抗力等。

经济机制

《Olympic Games Jam: Beijing 2022》是一款 Play to Earn 游戏，玩家在其中参与北京冬奥会这场全球庆典的同时，还能获得一定的收益。

1）首先，玩家可通过 Free 模式获得 Ticket 奖励，以及装备 NFT PIN 来生成 Ticket。然后使用 Ticket 参加更高等级的游戏模式，获得 Gem 奖励。

2）获得 Gem 后，玩家可在 nWayPlay 市场中购买 NFT BOX，或将 Gem 转换为该游戏 Token，再进行购买 NFT PIN。

3）玩家还可将获得的游戏 Token，在未来支持的 DEX 中转化为其他加密货币。

项目背景

nWay / Animoca Brands 开发

nWay 是 Animoca Brands 旗下的区块链游戏工作室，这是一家总部位于旧金山的开发商、发行商和技术平台，提供跨移动、PC 和游戏机的 Play to Earn 游戏。

该公司在开发移动设备和控制台上的 AAA 质量游戏方面拥有良好的记录，如《Power Rangers》系列、《Battlepalooza》，并已推出十余款热门主机游戏和在线游戏，包括几款重新定义了各自类型的游戏：《侠盗猎车手》、《暗黑破坏神 II》和《天堂 II》。团队成员则由来自 Sony、LucasArts、EA、Ubisoft、Capcom、Bandai Namco、Xbox、Warner Bros.、Konami、Kabam、Pocket Gems、Zynga、Glu、Jam City 和 Machine Zone 的领先开发人员和高级管理人员组成。

Animoca Brands 则是今年 NFT、GameFi、元宇宙等概念热潮中最大的赢家之一，其成立于 2014 年，总部位于香港，旗下 The Sandbox 是当前最热门的元宇宙项目之一，此外也是 NFT 头部交易市场 OpenSea、GameFi 龙头项目 Axie Infinity 的开发商 Sky Mavis、链游公会 YGG 等明星项目的早期投资者，其 2021 年 Q3 季度报告数据显示，目前已累计投资超过 140 个项目。

国际奥委会 (IOC) 官方授权

《Olympic Games Jam: Beijing 2022》于去年 6 月获得了国际奥委会官方的正式许可。并且，nWay 积极响应 IOC 关于「Heritage Collection」（奥林匹克遗产收藏）的全球许可计划，以历届奥运会、生活方式和限量版高端产品为核心，旨在通过艺术和设计将观众与奥运历史联系起来。

基于 Flow 链构建

该游戏基于 Flow 链构建。Flow 是一个快速、去中心化、对开发者友好的区块链，由加拿大知名区块链服务商 Dapper Labs 推出，旨在为新一代游戏、娱乐应用程序提供数字资产的基础，创造的明星 DApp 包括：CryptoKitties、NBA Top shot、Dapper Wallets 等。

总结

目前，2022 年北京冬奥会正在如火如荼地进行之中。赛场上，中国奥运健儿不断地挑战自我，刷新一个又一个的记录，弘扬着「更高、更快、更强、更团结」的奥林匹克精神。赛场外，北京冬奥会也再次引发了全民的热切关注。本届冬奥会带来精彩比赛的同时，也将带动 3 亿人参加冰雪运动，让更多的人爱上冰雪运动，使得中国的冰雪产业进入蓬勃发展期，为建设体育强国奠定了坚实基础。

而对于众多的游戏玩家和 NFT 收藏者而言，除了观赛、为奥运健儿呐喊助威外，还有什么比收藏与畅玩国际奥委会官方授权的 NFT 数字徽章和游戏更好的庆祝方式了。而《Olympic Games Jam: Beijing 2022》这款游戏也将进一步扩大这些 NFT 数字徽章的受众，并让玩家在 Play to Earn 机制的 NFT 游戏中感受到浓烈的奥运氛围，以一种全新的方式体验奥运会的精彩。

此外，nWayPlay 平台获得授权的 NFT 数字徽章，也是国际奥委会首次公布的进入 NFT 领域的计划，并希望采用新的数字技术来宣传奥林匹克价值观并与全球体育迷们建立更直接的关系，这在向所有体育迷提供完整奥运赛事体验上也迈出了重要一步。
