在历史上大部分时期，“技术”和生产力总是紧密相连，而“观念”又是生产关系变革的驱动力，由此，通常人们会把生产力想象成技术进步，把生产关系联系到群体思维模式。虽然技术的变化有时也会促进或影响生产关系的组织形式，但大部分技术的直接目标或者直接影响的依然是生产力的提升。 但有一项技术显得有些另类，它一直以来提出的主张并不是解决某一个单一生产力问题，而一直强调构建新的协作组织方式，从这一点来说，它更像一种“技术”和“观念”的混合体，它的“观念”带给人希望甚至想象，而它的“技术”实践也不断面临希望与困扰重叠的困境，它就是在希冀狂热和误解争议中交替前行的区块链“技术”和它背后的 WEB3 “观念”。

技术和制度-永恒的话题

生产力的迷思

在常规的语境当中，很多日常讨论起生产力，总不由自主地联系到硬科技技术、大爆炸、机器设备等等。

的确，从直观来说，一项技术科技的进步，包括制造工艺的进步，生产能力的进步，的确直接促进了整个社会的发展，但不管从历史来看，还是全球范围来看，还有一个问题也同等重要，就是这些技术为什么而诞生，以及为什么在这里诞生，而不是在那里诞生，这个就又成为一个技术发展的重要问题。

更进一步，因为生产力的比较，既可以在宏观上讲，整个人类的生产能力，可以由人类最先进的地区，最先进的群体，最先进的国家或最先进的公司来代表人类生产，但是走到微观层面，大到每个人，每一个国家，每一个组织，小到每一个人，其实更关心的是自己的生产力，在大环境中的竞争优势，以及自己的生产力在竞争中的能力。

所以现在在常用的评价不同地区、不同组织和国家的生产力指标，用的是一个全要素生产力，它里面就不是日常，我们习惯性认为的科技的发展能力，还包括了组织和协作等等，更多的维度和评价。

“全要素生产率”是指“生产活动在一定时间内的效率”。是衡量单位总投入的总产量的生产率指标，即总产量与全部要素投入量之比。全要素生产率的增长率常常被视为科技进步的指标,它的来源包括技术进步、组织创新、专业化和生产创新等。产出增长率超出要素投入增长率的部分为全要素生产率（TFP，也称总和要素生产率）增长率。全要素生产率一般的含义为资源（包括人力、物力、财力）开发利用的效率。从经济增长的角度来说，生产率与资本、劳动等要素投入都贡献于经济的增长。从效率角度考察，生产率等同于一定时间内国民经济中产出与各种资源要素总投入的比值。从本质上讲，它反映的则是各国家（地区）为了摆脱贫困、落后和发展经济在一定时期里表现出来的能力和努力程度，是技术进步对经济发展作用的综合反映。 全要素生产率是用来衡量生产效率的指标，它有三个来源：一是效率的改善；二是技术进步；三是规模效应。在计算上它是除去劳动、资本、土地等要素投入之后的“余值”，由于“余值”还包括没有识别带来增长的因素和概念上的差异以及度量上的误差，它只能相对衡量效益改善技术进步的程度。”

从这一个评价维度，虽然有三个要素，但是可以把它看成是常讲的两种要素的这个共同协作，第一种是技术和科技的进步，也是我们日常熟悉的生产力进步方式，另外一种可以把它归纳成组织协作的效率。和规模的提升这两点，那后一种可以把它看成是生产关系的改进和进步，所以从全面来讲，一个组织的能力，是技术和生产关系能力的同时进步。

如果我们把这两个要素一个叫做技术，另外一个可以总结为制度。这里的制度并不是指狭义的政治制度或某些公司制度，而是指在各种活动当中的组织形式、组织协作形式。那对这两个要素构成了生产率的全要素生产率指标，那我们更深入到里要，要问另外一个问题，这两个要素是独立发展还是相互依赖的，是互相促进的，还是有先有后，这个问题也一直在所有的经济讨论当中争论不休。

技术和制度：谁是鸡/谁是蛋

本文讨论的制度不是指狭义的政治制度，而是指广义的各种协作机制。技术和制度重要性的争论，基本两派观点，一种是技术促进了制度的改变，第二种是制度孵化了技术。

比如历史上的工业革命，从直观来看是一次很明显的技术革命，蒸汽机为典型代表的技术革新，让工业生产达到了一个新的高度，但换一个角度，从更细的角度去看工业革命的过程，它并不是某一时点的一种技术突破，而是在很长一个周期里面，点点滴滴的小的技术变化，累积出的技术变革，而且当时身处工业文明中间的人，并没有感觉他们在经历一些突发性的突破性技术变革，只感觉很多东西慢慢叠加累积，顺其自然的来到了生活中。可以讲，即便没有瓦特，也会在那个时间点诞生出蒸汽机，只是可能是有另一个人或者另好几个人同时出现。事实是，当时很多基础科学和技术领域的新发明和新突破，都是在欧洲不同地方同时被不同的人发明，证明那个时候在欧洲的科学、技术、文化积累已经到了一定的程度，这些的新技术、新模式自然地涌现出来。

但还有另一个问题，这些新技术诞生在什么地区诞生和发展，是偶然还是必然，还是某些背后其他要素影响的结果？可以狭义的科学技术水准来做比较，虽然工业革命一直被认为出现在英国，但是以及科学技术的水准来横向比较，英国并不是绝对领先的。海峡对岸的法国同样拥有非常深厚的基础科学和技术人才，但大部分的工业革命的技术革新，最终以英国为主要的落脚点和发展源头，这背后的原因又是什么？

大量工业革命历史的研究发现，英国在制度建设和协作机制建设上的优势，帮助了孵化化了工业革命诞生，比如说他的专利制度，也比如说他的以工商业为主导的社会政治制度，都让更多的技术诞生、孵化。或者转移到英国来发扬光大。所以从这个角度来讲，协作机制或制度又是技术重要的催化剂和生长的土壤。虽然技术是一种全人类的共享的知识，但这个技术发生在哪，使用在哪，甚至控制在哪，是非常重要的。

但因为这一分析，又容易落到狭义的政治制度的比较优势当中，让后来的很多建设又把政治制度作为核心要素来强调，认为只要采取了某一种先进的政治制度，其他的一切发展进步自然而然的发生，但回过头去看。到底很多政治制度很类似的国家，也有很大的经济发展和技术发展差异，有些完全用不同政治制度的国家也发展出也也有很好的，不同的经济发展和技术发展迭代路径。

所以大家慢慢意识到制度是技术乃至经济发展很好的基础和孵化剂，但是很多的关注只聚集狭义的政治制度。对到组织来讲则是狭义的组织规章建设和结构划分，觉得采用了某种先进的政治制度或者公司治理结构，就自然而然能带来技术发展或经济发展。 

在历史上，这一过程又遇到了大量的矛盾，很多自上而下的良好规划和良好设计最终没有达到预期的效果，甚至带来了负面的灾难性效果。核心的原因是什么？是在技术的发展、生产力的发展、社会的发展当中，有大量的不确定要素和不确定环境，自上而下的单一规划会在某些时间点起到促进作用，但是无法达成长效的、持续的完美的掌控和设计。

所以又有新的理念提出制度或协作机制不是是设计出来的，而是自下而上由社会自己涌现出来的。

100多年前，法国年轻的政治学家托克维尔对美国民主的首次考察，写下了“论美国民主”一书，里面比较体系的介绍了美国的民主制度建设，但是着重提出的是，美国社会的社团精神、公民精神、社会协作精神和参与社会治理的精神，才是保证这套制度能完整、能较好持续运行的关键要素。

而另一本美国社会学的书，“独自打保龄”，又从另一个角度观察，发现因为新的技术，如电视、互联网带来的人的交流的更加的独立性，所以在实体的、社区的，组织身边社会治理的参与的情况越来越少，也忧虑的指出，这样的趋势是对美国政治制度和民主制度的一个威胁，因为它的底层的自下而上的土壤越来越薄弱和分裂，很难支撑这棵大树更好的繁茂的生长。

所以很快很多人发现，光按照范本设计一个良好的协作制度，并不能一劳永逸地解决问题。因为在实际的协作和社会发展当中，社会经济发展当中有大量的新的场景，新的协作方式，新的商业流程，新的社会治理需求，会不断的涌现产生，只靠一套设计好的所谓完美制度，很难一劳永逸的长解决长期问题，甚至在新的形势和环境变化中，这样的设计完美的制度，反而可能会形成发展的阻碍。

所以，这又引出了一个新的问题，一个可以自我迭代、自我调整甚至自我创新的自下而上的社会土壤，又该如何诞生呢？ 

社会资本：种参天大树的土地

上一节提到一个社会自下而上的孕育土壤，这方面有过很多的研究，包括前文提到的美国的公民社区、社会研究，在国内也是在社会学或人类学的交叉领域来研究人群的行为文化，等等。近年很火热的“社会资本”这个概念也是在这方面重要的观点，社会资本是一个社会土壤质量的衡量。

最早狭义的社会资本概念是对个人而言，在社会上可运用的资源。随后也有不少社会学家提出了社会资本是作为整个社会的一个基础设施的特性，是产生的组织协作、经济发展、社会治理发展所需要社会土壤和社会资本，以下是一个其中一个广义的社会资本定义。

“社会资本内涵的相似性体现在研究者要么将社会资本当作推动社会网络内行动主体之间合作的生产性资源，要么是作为促进个体在社会网络中获取资源和利益的能力。学者们将网络、信任和规范社会文化要义补充进入社会资本理论框架中，丰富了“社会资本”的内涵。”

从广义的社会资本定义可以看出，社会资本是一个综合性的评价指标，它包括了一个社会文化、组织、协作方式、公民的观念及各种基础设施的完善程度等等，简单的讲，就是这一个地区的社会土壤的肥力好不好。

就像种地，当你有一个肥力非常好的土地，或者生态化比较好的环境，你自然可以慢慢会长成一个生态多样性的地方，动植物的种类也会非常丰富，中间又会互相促进，互相。交叉影响，形成了正循环。放到现实社会中，这也最终带来了地区或国家公司组织的繁荣和发展，

从这一角度来讲，回到前文提到，在工业革命之间之中，英国首先先于欧洲大陆先发萌芽，也是因为在工革命之前的数百年间，英国在文化、政治发展，经济发展孕育了形成这样的社会土壤。

孕育了有利于技术创新、技术发展的制度的社会资本，不单是一个单一的制度因素，就像在英国会产生更好的这个专利制度，那也是因为他在之前的观念、文化和社会资本的这个发展当中，水到渠成地形成了这些观念。

从这个角度来讲，社会资本可以看成是制度的一个升级版本，是一个根源，也是长出参天大树的土壤构成要素。

之前林林总总说了这么多技术跟制度相辅相成的的关系，现在又引出关于WEB 3跟它们有什么关系？

WEB 3是一种特别的产物，在有些人眼里它是技术，在有些人眼里它是制度、观念，所以就有点像光的波粒二项性，从这里，我们更愿意把WEB 3定义成是技术和制度的综合产物。

Web3：技术和制度的合体

之前讲了这么多技术跟制度的融合和孵化关系，从WEB 3来讲，它更想创造一种可以自由迭代创新的土壤，社会资本或基层的土壤。但就像之前提到的，无论从英国的工商业资本推动的光荣革命到工业革命的服化，及美国社会制度的社会机制和公民精神的形成，多多少少都是一些自发的由历史、宗教、文化等等原因综合组成的诞生物，而WEB 3是想要为社会实现一个自下而上，自由协作的基础设施。

Web 3的核心要素是什么？有些人可能会提去中心化，当然去中心化当然也有程度之分，所以这单一的要素可能还不足完全概括web 3。从我个人的观点来看，web 3的核心要素是给了人不依赖于第三方自由协作的“工具”和“观念”。

在历史上，各种主要的商业社会的各类协作当中，都会对第三方产生一定的依赖性，新的协作制度、协作工具、协作平台都需要由不同的提供方来支持，而且这个提供方会像一个环节嵌入到协作流程当中去。

没有互联网公司，我们可能做不了社交，进行不了电子商务交易和大量的线上的工作，当没有金融公司服务，我们也没法做更多的规模性交易和资本协作等等，这些都是由第三方为你提供了协作的支持，当然这样的支持不是免费的，必然会成为交易成本的一部分。此外，因为第三方在中介环节对信息、数据、匹配权力等等的控制，又衍生出垄断、隐私等一系列问题。更进一步，你只能选择已有的第三方服务完成你的协作，如果你的协作任务由特殊性或还没有第三方支持，很多是否即便愿意付成本也无法获得有效支持。

所以WEB 3的这个理念，它也是从技术跟制度两个方面引起了人们的希望、热爱和愿景，在技术上的确可以通过一系列的技术让大家有通用的去中心化协作工具，虽然还在早期，但在不断的迭代中，让普通每一个人都可以去做各种各样的协作。从观念和理念上来讲，给大家带来了一个自由协作、自由组合、自我掌控新的观念，一个自由人的自由组合啊，这又形成了特别的理念。

当然，你也会看到在这一过程当中所遇到的困惑。其一，Web 3本身是一个提倡自下而上涌现的理念，自下而上的自由组合和去中心化协作的，但在web 3还没有规模性的应用到主流里，Web 3本身这个概念来自于大的理念的规划，web 3的愿景规划也是理念先行。那这个是自上而下的规划和自下而上的涌现之间，如何找到平衡的度？那条路径是web 3下一阶段的主要推动力。

此外，在自由协作和核心推动之间，当你只有自由协作，而不具备有效的执行能力，也不能为社会提供更有效的服务。观念的触动很多人，但很多时候到了真实的要使用服务甚至定义的时候，观念和利益产生矛盾，有些时候也会产生阻碍，并不是所有人都愿意为新观念买单。

WEB 3本身这个概念在现阶段，又更像是一个理念的提出，甚至有一点自上而下来设计出这样一个去中心化的协作理念，所以它的诞生又有又带有一定的自下而上的规划感，因为现在坦率讲，WEB 3还没有在大量的主流的应用场景当中被广泛使用，依然在发展的早期探索和理念阶段，所以在这个中间就会产生很多新的希望、矛盾、困惑和不确定性，期待这些问题在不断创新的进程中得到解决。
