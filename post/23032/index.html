DeFi热点

1. 加密期权交易协议Optyn已集成Chainlink喂价

2021年春季Chainlink线上黑客松亚军Optyn宣布，已成功集成Chainlink喂价，以准确地为基于Polygon的DeFi协议创建、交易和结算的期权合约和合成资产定价。 据悉，Optyn是建立在Polygon上的期权市场和合成资产协议，使用户能够接触合成资产并创建期权合约。

2. PeckShield：DeFi协议MerlinLabs遭到攻击，是PancakeBunny的又一同源攻击

5月26日消息，PeckShield“派盾”预警显示，DeFi收益聚合器MerlinLabs遭到攻击，此次攻击手法与5天前遭到闪电贷攻击的PancakeBunny的攻击手段相似，损失200ETH。

3. DeFi收益优化协议ETHA已成功在以太坊侧链上启动主网

近日，DeFi收益优化协议ETHA已成功在以太坊侧链上启动主网，以提升拓展性、交易性能与用户体验。同时，ETHA已于侧链上发行ETHA代币。此外，ETHALend与DODO流动性挖矿已于5月24日结束，社区活动正在逐步完善。

4.基于以太坊的去中心化FIL借贷平台DeFIL资产规模超5亿元

据Gasnow网站显示，截止目前2021.5.26 11:20 SGT，以太坊网络Gas费用大幅降低，每笔交易的Gas费用已低于20元。Gwei的降低促使基于以太坊链上的DeFi协议交易活跃，其中基于以太坊的去中心化FIL借贷平台DeFIL存入资产规模超￥500,000,000。 DeFIL是基于Ethereum的Filecoin去中心化借贷协议。用户参与存FIL可获取利息收益以及平台治理通证DFL，DFL可继续进行Staking挖矿.

HECO

1. 火币全球站已开通6个锚定代币在HECO网络的充值、提现

据官方公告，5月26日，Huobi Global宣布正式开放NFT、GTC、RING、STORJ、PHA和CRU的HECO网络锚定Token的充值和提现业务。HECO现已上线Heco-Peg NFT Token 、Heco-Peg GTC Token、Heco-Peg RING Token、Heco-Peg STORJ Token、Heco-Peg PHA Toke和Heco-Peg CRU Token。 需要注意的是，Heco-Peg NFT Token 、Heco-Peg GTC Token、Heco-Peg RING Token、Heco-Peg STORJ Token、Heco-Peg PHA Toke和Heco-Peg CRU Token为HECO网络代币，暂不支持与其他网络互转。 用户在充提上述五种锚定Token时，务必选择正确链名称“ HECO”，以免造成充值、提现不上账的情况。

2. Cash Flow单币抵押资产已超过1000万美元

据算法稳定币Cash Flow社区消息，Cash Flow5月24号已开启五天创世挖矿，其中单币挖矿区支持HT，HUSD，HBTC，USDT，MDX，BAG，TPT，ETH， COO；LP挖矿区支持COC/USDT，COB/USDT，COS/USDT，COC/COO，目前单币抵押资产已超过1000万美元，通过双稳定机制，有效地避免了算法稳定币死亡螺旋的循环。Cash Flow社区治理是Cash Flow生态发展最重要的一步，社区志愿者、社区代表、特别代表共同管理社区，从而打造HECO下一代算法稳定币。

NFT

1. Phoenix DAO在Polygon上推出NFT活动市场

Phoenix DAO在Polygon上推出NFT活动市场（NFT Event Marketplace）。使用MATIC以极低的费用进行快速交易，可以为PHNX带来诸多好处，而PHNX将用于应用内购买和创建活动。 据悉，PhoenixDAO Event Marketplace是一站式解决方案，用于与全球观众互动、寻找和购买门票。

2 . 波卡跨链NFT协议DNFT获得新一轮百万美金的融资

据官方消息，DNFT宣布完成获得百万美元的融资，本轮投资机构包括LD capital、Candaq Fintech Group、Waterdrip capital、7Star Capital、Worshipper capital 、GDCapital、1PAR、InnCube Crypto Capital、ChianUp等数家机构。 DNFT是基于波卡的新一代跨链NFT协议，致力于把数据资产跨平台变现，目标是发展成NFT领域的跨链基础设施。DNFT已获得Web3.0Grant、BSCGrant等支持。

3. 美联社拍卖一系列代表历史著名照片NFT作品

美联社正在拍卖一系列10个NFT作品，这些作品代表了一些历史上最著名的照片。该系列的第一个已在NFT市场OpenSea上投标。这是美联社1945年拍摄的著名照片，六名美国士兵在硫磺岛升起美国国旗。这是该系列名为“美联社175件藏品”(AP ARTiFACTS: the 175 Collection)的首个NFT作品。NFT还收录了小提琴家Nick Kennerly的配乐，以及摄影师Rosenthal的罕见图像，以及该机构档案中的硫磺岛电影和音频。

4. 华裔创作歌手陈奂仁将于5月27日发售第二件音乐NFT作品

据官方消息，华裔创作歌手陈奂仁将于5月27日在BSCNFT交易平台Treasureland发售第二件音乐NFT作品。

5.歌手阿朵发行NFT数字音乐作品 成交价超30万

5月26日消息，国内歌手阿朵发行NFT 数字音乐作品《WATER KNOW》，全曲采用 NFT 技术进行数字加密，并将封面和歌曲的署名权公益拍卖，成交价 30.4271 万元，拍卖收入将捐赠给音乐公益项目。

6.音乐版权服务平台火花音悦成立NFT厂牌Free Spark

火花音悦宣布成立NFT音乐厂牌Free Spark（自由火花），面向全球市场，围绕音乐创作、版权授权和作品的NFT化，进行策划、合作、风控等。
