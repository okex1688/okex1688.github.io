NFT 在 2021 年可谓风光无限，谁也没有想到 Crypto 在 2021 年的出圈之路会是从 NFT 开始的。从无人问津到人尽皆知，甚至当选《柯林斯词典》2021 年度词汇，据统计，在 2021 年一年的时间里，NFT 一词的使用率增长了 110,000%。

而在 2021 年，NFT 也经历了一件件出圈事件，也经历了一次次变革，今天，律动 BlockBeats 按照时间梳理，为大家盘点一下 2021 年的十大 NFT。

NBA Top Shot

2021 年初，NBA Top Shot 打响了 NFT 出圈的第一枪。

尽管 NBA Top Shot 在 2020 年中旬就开始测试，但在 2021 年 1 月才彻底爆发。这些数字版球星卡吸引到了众多 NFT 爱好者、NBA 球迷、实体球星卡收藏家的关注与积极参与，在 NBA Top Shot 热度最高的时候，单日交易额可达 4500 万美元。截止发稿时，NBA Top Shot 二级市场总交易额高达 8.11 亿美元。

其中，成交额最高的数字球星卡为著名球星勒布朗·詹姆斯在 2020 年总决赛的一记扣篮，成交额高达 23 万美元。

Hashmasks

2021 年 2 月，NBA Top Shot 之后下一个对 NFT 领域产生巨大影响的项目 Hashmasks 诞生了。

Hashmasks 的出现让人们惊呼盲盒的发售形式居然能用在 NFT 身上，发售价格的联合曲线居然会让人们产生如此之大的 FOMO 情绪。Hashmasks 也第一次让人们意识到可以通过为自己的藏品起名字的方式参与到作品共创的过程中，而 Name Change Token 也第一次让人们意识到 NFT 收藏品的玩法也可以与代币经济相结合。

可以说 Hashmasks 的诞生拉开了 NFT 系列收藏品狂潮的序幕，截止发稿时，Hashmasks 二级市场交易额达 8340 万美元。其中，第 9939 号 Hashmasks 创下了整个系列的最高成交额记录，约为 159 万美元。

Everydays: The First 5000 Days

紧接着，在 2021 年 3 月，艺术家 Beeple 的加密艺术作品《Everydays: The First 5000 Days》登陆顶级拍卖会佳士得，并以 6900 万美元的天价成交，在当时这件作品也成为了在世艺术家的作品中排名第三高的艺术品。

这件事在加密艺术圈、NFT 圈、传统艺术圈等多个领域均引起了极大的轰动。在当时，主流媒体对 NFT 的报道还停留在 NBA Top Shot 的财富效应以及 NFT 是否是一个新型骗局，Beeple 和其作品的出圈让人们开始认识到 NFT 作为一种新型媒介对数字艺术来说有多么重要。

这件由 5000 幅 Beeple 每日创作的作品拼接而成的巨作也让无数 NFT 的关注者高呼「这是一场伟大的 NFT 革命」、「NFT 的时代已经来临」。

Bored Ape Yacht Club

接下来到了 5 月初，Bored Ape Yacht Club（BAYC）诞生了。BAYC 绝对可以说是 2021 年表现最优秀的 NFT 系列，无论是价格涨幅还是知名度、社交资本等层面，BAYC 带领着一种 NFT 头像收藏品上升到了此前无法想象的高度。如今，Stephen Curry、Post Malone 等来自各界的知名人士纷纷买入 BAYC，这也让 BAYC 持续获得更高的热度，地板价也攀升至 51.9 ETH，约合 21 万美元。

而在年底，BAYC 与 CryptoPunks 的正面对抗同样引来了无数关注。BAYC 将版权送给藏家，藏家便可以使用猿猴的形象制作周边、衍生品等；而 CryptoPunks 的制作团队 Larva Labs 的选择恰恰相反，没有把版权送给藏家。这也引起了两大社区持续不断的争论，而在争论中，一些 CryptoPunks 的藏家倒戈，这也使得 CryptoPunks 的地板价一路下跌。

其实，如果把这些头像单纯视为「收藏品」，保留版权是完全可以理解的，然而在 Web3 时代中，这些头像已经成为了藏家们的「数字身份」，而无法掌握在自己手里的「身份」与 Web3 的宗旨相违背，更不用说 Larva Labs 还直接将一些「数字身份」授权给影视公司当作电影角色。而 BAYC 爆火的一大原因便是藏家使用自己的猿猴形象制作了众多周边产品，这种自发的、大面积的宣发也为 BAYC 积累了更大的知名度。

截止发稿时，Bored Ape Yacht Club 二级市场总交易额达 8.98 亿美元，最高成交额约为 243.76 万美元。

Art Blocks

Art Blocks 是一个生成艺术平台，2020 年推出，而直到 2021 年 6 月才开始逐渐为人熟知，并在八月创下了月度交易额的历史记录，达 6.27 亿美元。

生成艺术是加密艺术语境下成为主流的艺术类别，与我们熟知的油画、素描等不同，生成艺术的创作依赖于代码，且通常还会结合计算机的随机性，可以算是人类与计算、有序与随机、感性与理性结合而生的艺术。而生成艺术所使用的很多代码也都是开源的，任何人使用相同的代码都能生成出相同的作品，而在开源代码的基础上加入一些艺术家自己的思考便又能产生出一套新的艺术作品。生成艺术的创作模式、理念也与 Crypto、Web3 不谋而合。

截止发稿时，Art Blocks 二级市场总交易额达 11.2 亿美元，最高成交额纪录为 Ringer 系列第 109 号作品创下的 690.65 万美元。

Nouns

时间来到 2021 年 8 月，NFT 的发展又来到了一个新的阶段。而从 Hashmasks 到 BAYC 再到 Nouns，我们可以发现对于 NFT 系列而言，人们对于审美的追求逐渐降低，社交资本越来越受重视。社交资本的获取与增值与 NFT 外观的关系越来越小，重要的是都有谁在用相同系列的头像，社区里都有哪些人。

Nouns NFT 的发行与我们常见的一次性发行 1 万件不同，Nouns 采用的是每天拍卖一件的形式，并且成立了 Nouns DAO，Nouns NFT 是加入 DAO 并参与提案、治理的唯一方式。而拍卖所得也将全部归为 DAO 金库所有。与 BAYC 一样，Nouns 同样也鼓励周边项目的创作，Noadz、Nouns Vox、Noundles、HeadDAO 等项目的诞生也为 Nouns 带来了更多的关注。

Nouns 系列的 1 号 NFT 创下了成交额的历史最高纪录，在 8 月 8 日拍卖时以 613.37 ETH 成交，约合 248.51 万美元。

Loot (for Adventurers)

时间来到 9 月，一个绝对会被记录在 NFT 历史上的项目诞生了，它就是 Loot (for Adventurers)。

Loot (for Adventurers) 的创始人是大名鼎鼎的 Dom Hofmann，他曾于 2012 年推出了短视频 App Vine 并于 2016 年被推特以 3000 万美元的价格收购，而在被收购的同年，抖音才刚刚诞生。

Loot 的爆火是因为它打破了现有的模式。发行模式采用的是通过合约免费铸造而非官网发售，且二级市场版税比例设定为 0；在社区构建方面，其他的 NFT 都是先有一个产品然后吸引人们加入社区，而 Loot 则是先吸引人们加入社区然后再由社区成员共同创作一个产品。

此前，人们需要接受项目方给出的「形象」来当作自己在原宇宙中的数字化身，拥有了这个数字化身之后才可以加入到这些项目的社区当中，然后等待着项目方决定为 NFT 赋予哪些实用性或是为持有者发放哪些福利。而 Loot 所做的事情便是将这种模式调转过来，由自己、社区创造出大家认可的形象作为数字化身，一起决定如何赋能而不再简单听命于某个项目方。这种范式转变也让人们对这个项目的前景极为看好。

不过，由于 Loot 缺少捕获后续价值的方式，所以目前二级市场的表现没有达到人们的预期，不过 Loot 带来的范式转变让我们对于 NFT 与社区产生了全新的认知。

截止发稿时，Loot 二级市场交易额约为 2.68 亿美元，最高成交额约为 170.34 万美元。

Merge

12 月，加密艺术家 Pak 的作品 Merge 吸引了人们的眼球。

单看 NFT 本身，黑色背景加上一个小小的圆形，无数人质疑「这也算艺术？」，其实人们对于 Pak 一直有这样的「偏见」。

Pak 此次发售的「小圆球」会随着每一次交易与藏家本来拥有的「小圆球」融合，融合得越多，「小圆球」就会变得越大，而在之后，这些「小圆球」会按照规则兑换成 Pak 发行的 ASH token。而这样的融合机制既能让 NFT 与藏家产生互动，又能起到通缩的作用。并且 Merge 还打破了原有的「先买入，再碎片化」的流程，变成了「先买碎片，再融合」的玩法。此外，Merge 与 ASH 的链接也让艺术品拥有了金融属性，让 token 拥有了艺术属性。

从小正方体组成的 CUBE 到最初是一张白纸的 LostPoets，人们总是认为这些「自己也能做得出来」。的确，只从单个 NFT 的外观来说，几乎没有制作门槛，但 Pak 其实并不能算是「视觉艺术家」，相比于视觉，Pak 作品的机制设计才更加吸引人，可以说是那些 NFT 加上整套玩法机制共同构成了「一件艺术品」，Pak 是一位「数字观念艺术家」。

而 Merge 发售的总交易额高达 9200 万美元，一举超过 Beeple 在佳士得拍卖取得的成绩，而 Pak 作品的总交易额也超过 Beeple，在所有加密艺术家中排名第一。

RTFKT

紧接着令人震惊的消息便是数字潮牌、Web3 独角兽企业 RTFKT 被传统行业巨头 Nike 收购。这个消息一出，也让 RTFKT 发行的 CLONE X 头像系列 NFT 收藏品价格直接翻倍。

RTFKT 此前推出的各种 1/1 球鞋、与 FEWOCIOUS 等艺术家联名的潮玩以及为 CryptoPunks 持有者推出的球鞋系列等都广受好评，而与潮流艺术家村上隆合作的 CLONE X 也备受瞩目，所以 RTFKT 被 Nike 收购的消息引起了轰动与人们热烈的讨论。

有人认为这是一件好事，证明了 NFT 在全世界范围拥有着越来越大的影响力；而有人也认为这并非像大多数人们所认为的那样振奋人心，Web3 的独角兽企业最终还是被「招安」。

截止发稿时，CLONE X 盲盒二级市场总交易额 2.46 万 ETH，头像 NFT 二级市场总交易额 4.21 万 ETH，总计约 2.66 亿美元，其中，村上隆与 RTFKT 联名款 NFT 创下了最高成交额纪录，约合 117 万美元。

adidas Originals Into the Metaverse

同样是在 12 月，Nike 的竞争对手 adidas 也在 NFT 领域展开了大动作。

adidas 旗下三叶草品牌或将与 BAYC、G-Money 和 PUNKSComic 合作，在 12 月中旬发售了「Into the Metaverse」NFT，此次发售共计售出 29,620 枚 NFT，销售额达 5924 ETH，约合 2300 万美元。

与其他众多发售 NFT 的传统行业巨头不同，adidas 这次与 NFT 项目的合作不但没有受到质疑，反而大获好评。adidas 并不是发售赚一大笔钱后就又返回 Web2 做着传统的生意，而是选择在 Web3 长远发展，adidas 官方也说这一次发售只是开始，会在未来为 NFT 持续赋能。而目前已经公布的 NFT 持有者福利为可以在 2022 年免费领取 3 件与 NFT 相关的标志性实体服饰。

截止发稿时，在不到一周的时间里，adidas Originals Into the Metaverse NFT 二级市场交易额便已高达 4718 万美元，超越了 CryptoKitties 等一众元老级 NFT 项目。

2021 年是 NFT 飞速发展的一年，Crypto 的发展速度本就比传统行业更快，再加上 NFT 拥有着比 Crypto 其他赛道更容易出圈的文化属性，让此前从未关注 NFT 甚至从未关注 Crypto 的人们蜂拥而至，这也使得 NFT 能在短短一年中经历数次变革。

至于 NFT 未来会变成什么样子，谁也无法下定论，但可以预见的是，现有的优质项目会持续不断地沿着自己开辟的道路前进，并吸引更多人的加入，同时也会有更多的新项目会去尝试探索 NFT 的边界，探索在今天我们还没有办法想象的事情。
