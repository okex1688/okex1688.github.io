100 万美元、660 美元、6000 万美元……一张图片、一段 10 秒的视频，NFT 藏品不断刷新着价值上限，数字后面有几个 0 都得数上好一会。

对此，普通人除了啧啧称奇，似乎没有特别多的情绪，毕竟数字后再多的 0，再高的价格都和自己没有多大关系。

卖出 6900 万美元的天价 NFT 图片

但如果你是一位游戏玩家，尤其是主机游戏玩家，可能就不一样了，推出刺客信条系列游戏的育碧、最终幻想系列的缔造者 Square Enix，不断有游戏大厂表示要加入到这一潮流当中，推出 NFT 游戏藏品。

渐渐地，玩家们发现游戏好像因为 NFT，越来越贵了。

玩家讨厌 NFT

去年 12 月，育碧推出了平台 Quartz NFT 用以销售 NFT，游戏《幽灵行动：断点》中的枪械、头盔、服装被制作成了 NFT，玩家可以根据通过游玩时间获取，也可以直接在 Quartz NFT 平台中自由交易。

育碧首批推出了共 3000 个 NFT，按照它的说法，这些枪械、头盔、外套等游戏道具作为 NFT，具备唯一性和不可修改性，相当于一个限定的游戏道具，且所有交易都存储在区块链，公开且透明。

不过，玩家们并不买账，首先 NFT 道具获取难度极高，头盔道具需要游玩 600 小时才能获取、裤子则需要 100 小时、枪械更是要求玩家达到一定的游戏成就评级。

600 小时，以每人每周 10 小时游戏时间计算，一年都拿不到这款头盔 NFT 道具。

人们每周的游戏时间是有限的。

肝游戏拿不到，那有没有人直接买呢？

确实有人，但实际购买的人非常少，普通玩家对这种产品并不感兴趣，《幽灵行动：断点》本身的游戏评价也不是很好。

《幽灵行动：断点》

外媒 Vice 采访了几位购买《幽灵行动：断点》NFT 的玩家，其中一位表示他以 62 美元的价格购买了一款标准代号 #1436 的 NFT 枪械。

只是，这位玩家其实本身就是一位区块链从业人士，他仅仅是作为行业人士观察 NFT 游戏的发展罢了。

育碧之后，接连不断有游戏大厂表示对 NFT 感兴趣，游戏开发商 Square Enix、知名游戏 IP 潜行者系列的游戏开发商 GSC Game World 。

这两家游戏大厂同样遭到了玩家的批评，社交媒体、官方论坛、给官方宣传视频点踩，玩家们以各种各样的方式输出自己的不满：

不想着好好做游戏，就知道用 NFT 来骗钱？

玩家曾在《最终幻想 14》论坛中请愿 Square Enix 放弃推出 NFT 的计划

之前，玩家花费 300 元就可以获得一款不错的游戏，数十个小时的游戏时间，在不断地选择中体验不同的故事，又或是拿起一把把性能不同的枪械，在游戏中与朋友竞技。

游戏厂商推出 NFT 后，原本在游戏过程中可以获取的枪械道具等，都与区块链连接，被标好一个又一个不断变化的价格。

要体验一款完整的游戏，玩家需要付出的不止是 300 元，甚至会因为部分 NFT 道具数量太少，价格飙升到普通玩家无法接受的地步。

科乐美恶魔城游戏一张地图 NFT 的价格达到了 11.9 万美元，价格还在不断变化

游戏开发者中也有不少人厌恶 NFT，根据最近举办的游戏开发者大会年度调查，2700 多名开发者中有近 70% 的人表示不会在游戏中推出 NFT。

微软游戏首席执行官菲尔·斯宾塞（Phil Spencer）也曾在采访中表示：

对于 NFT 游戏，我认为很多实验和尝试正在进行中，我看到的一些创新项目更像是剥削而非娱乐。

菲尔·斯宾塞

当 NFT 作为一个技术概念进入各行各业，拥有过高的流行度，但普通玩家很难感受到其正面价值，反而会带来各种负面影响，自然会引发群体式愤怒与抵制。

游戏厂商为何看好 NFT？

被玩家疯狂抵制后，育碧还是不死心，表示看好 NFT 在游戏行业的发展，拥有侠盗猎车手 IP 的游戏公司 Take-Two Interactive 也表达了类似的看法，虽然存在争议，但仍然有一部分游戏大厂持续关注 NFT。

玩家大规模抵制后，为什么游戏公司还是这么坚持，NFT 游戏真的有这么好吗？

要了解答案，我们还是得先回过头看看 NFT，作为非同质化代币，从被创造的那一刻它就和区块链接在了一起，唯一且不可替代，所有转移都是公开的、非中心化的。

箱子开出的每一把枪械都是唯一的

就如同在 CS：GO 游戏中开箱子一样，玩家开到的每一把枪都是不一样的，颜色、磨损程度等，类比一下， NFT 其实就是将颜色、磨损度这些换成了数字化的唯一标识代号。

在游戏厂商们看来，未来玩家在 NFT 游戏中的游玩是可以创造价值的，与真实的金钱相互兑换，甚至超越游戏本身的游戏周期，只要需求存在，玩家就可以一直获得价值，正如 CS：GO 中因开箱而暴富一样。

实际上已经有游戏朝着这个方向发展了，以 NFT 为基础的游戏 Axie Infinity 就是其中之一，玩家中可以游戏中培育宠物，通关各种地图甚至互相竞技。

在 Axie Infinity 中宠物、道具都是 NFT，它们都和虚拟游戏货币相连接，要想玩这个游戏得先买虚拟货币，在玩游戏的过程中也可以通过刷图获取货币。

这些虚拟货币都和真实的金钱挂钩，只是价格在不断变化，因为玩家的需求在不断变化，游戏仿佛成为了一个新的经济系统。

据福布斯报道，截止 2021 年 10 月，Axie Infinity 的交易额超过 23 亿美元，每月有 250 万活跃用户，甚至有一部分玩家将游戏作为主要收入来源。

Axie Infinity 中的宠物价格

游戏大厂们之所以看好 NFT 游戏，是因为NFT 技术的唯一性和不可以代替性，让数字内容有机会成为资产，而游戏正是与 NFT 最融洽的数字内容之一，游戏币、宠物、装备，都可以数字化，都可以是数字货币。

基于这项技术的游戏模式，有机会改变现今游戏的主流运营模式，游戏产业是典型的高投入高风险行业，投入上亿美元开发、运营、推广一款游戏，收回成本主要看有多少玩家愿意买。

而在 NFT 游戏中，玩家可以靠 NFT 代币获取收益，游戏厂商作为 NFT 游戏的创建者，也可以通过发售新 NFT 等方式不断获取收益，只要需求还在 NFT 游戏的生命周期会大大延长。

玩家想要好游戏，而非在游戏里打工

作为一项技术，NFT 自然有其价值，只是目前的 NFT 游戏或是基于游戏发布的 NFT 藏品，都有一个大前提——需求，信则有不信则无。

玩家对于游戏的需求来自什么，是游戏质量，目前的 NFT 游戏却未在这点上有突出表现。

基于 NFT 技术构建的游戏 Axie Infinity ，身上有太多其他游戏的身影，宠物培育来自宝可梦系列游戏，刷图加玩家竞技的模式，国内玩家可能会想起网络游戏 DNF。

Axie Infinity 的游戏模式并没有太多创新，本身游戏质量实属普通，它能崛起的一大原因是疫情期间部分人失去了生计，这才转向游戏，游戏玩家也主要以菲律宾为主，大家抱团取暖，依靠游戏为生。

以 DNF 为例，游戏金币过多，超过需求，价格就会崩盘

DNF 等游戏遇到的问题 Aixe Infinity 也会遇到，当初 DNF 要打击游戏工作室，就是不想游戏币产出太多，让游戏经济系统崩溃，导致玩家流失，而现在 Axie Infinity 也因宠物繁育的产能和消耗不够平衡而被用户吐槽。

一旦代币价格下跌过大，Axie Infinity 就会失去大量用户。

任由玩家自由发展，Axie Infinity 供需体系会崩溃吗，一家游戏公司出手干预，能维护好一个经济体系吗，作为一款游戏质量一般，能长久的吸引用户吗？

这些都是 Axie Infinity 要走向更远需要回答的问题，或许还有监管。

NFT 游戏所指向的未来是不确定的，众多游戏厂商希望将其丰富，描绘成现实，但目前以 NFT 数字代币或 NFT 藏品为主而不关注游戏质量的模式，并不受玩家欢迎。

人们之所以热爱游戏，是游戏通过选择等交互方式，让玩家可以体验一种与现实完全不一样的生活；是因为游戏出众的战斗竞技系统，让玩家体验技巧提升，获取成就感；是因为出色的故事和角色设计，满足玩家的收藏喜好。

《勇者斗恶龙11》

《勇者斗恶龙》游戏 IP 能火 40 多年，《英雄联盟》走过 12 个赛季仍然被玩家喜爱，靠的是出众的游戏质量。

作为玩家，我们想要的是好创意、好故事、好游戏，而不是在一个套着游戏外壳的系统里打工赚「NFT」。
