近日推特上两条关于 Frog Nation 前 CFO 的信息传遍了 DeFi 乃至整个币圈。作为 CFO，0xSifu 不仅手握 Wonderland 这一最大 DeFi 2.0 协议的金库大权，还对其它 Frog Nation 生态协议的收入有着巨大的影响，涉及资金超十亿美金。而 0xSifu 曾涉嫌诈骗的消息传出后，也将处于发展关键阶段的 Frog Nation 推向了火坑。

凌晨时分的爆炸性消息：CFO 曾多次涉嫌诈骗

美东时间 27 日凌晨，Frog Nation 的创始人 Daniele 发布了一条推文：「今天有关我们团队成员 0xSifu 的指控将大面积流传开。我想让所有人知道，我对此事有所了解，但我认为一个人的过去并不能决定他的未来。我更加看重自己在了解 0xSifu 身份之前与他共处的时光。」

一分钟后，zachxbt 就在推特上公布了自己与 Daniele 的聊天记录，证实 0xSifu 曾多次参与诈骗犯罪。

2005 年，0xSifu 以 Omar Dhanani 的身份经营了一个名为 Shadowcrew 的身份盗窃团伙，专门从事信用卡盗窃、身份盗窃和其他庞氏骗局等网络犯罪行为。此外他自己还经营了另一个名为 Midas Gold 的庞氏骗局并卷钱跑路。不久后，他与其他 6 名 Shadowcrew 同伙在美国被捕入狱。

出狱后，0xSifu 再次化名 Michael Patryn 前往加拿大。在 2018 年，他与 Gerald Cotten 在加拿大一起创立了加密货币交易所 QuadrigaCX。这个交易所同样也是一个庞氏骗局，Cotten 和「Patryn」会用新用户的存款来支付老用户的提款，使得后来每个人的提款都会被困长达数周甚至数月。

没过多久，Cotten 就在印度旅行的途中意外「死亡」（其死亡证明和妻子的反应均被质疑有伪造嫌疑），使存储于交易所内价值 1.7 美元的加密资产无法被用户提出。许多加拿大人失去了毕生的积蓄，而「Patryn」却得以带着大量资产逍遥法外。

如今，Frog Nation 创始人 Daniele 却任命 0xSifu 为自己的 CFO，将 Wonderland 金库里近 8 亿美元的调度权交给了这位前科罪犯。当然，0xSifu 自己的资产也在此期间在此期间增长了 10 倍之多。

节骨眼上发生的事件

0xSifu 的消息传出后，所有关键的 Frog Nation Token 都受到了重创。Wonderland 的 TIME 下跌了 30%，交易价格从 500 美元附近跌至 370 美元；Abracadabra 的 SPELL 则下跌了 15%，从 0.007 美元跌至 0.006 美元。但此次曝光事件对 Frog Nation 生态的影响，远不仅是对 Token 价格的冲击。

Frog Nation 是 Daniele 创立的 DeFi 项目网络，它由多个项目构成，并旨在共同成长、相互促进，实现生态更大规模的发展。其生态内的协议在此前取得了巨大的成功：Abracadabra 平台稳定币 MIM 采用率在几个月内飞速提升，Wonderland 则超过 Olympus DAO 成为最大的 DeFi 2.0 协议。

最近 Frog Nation 更是处于关键的发展节点，几项关乎生态未来的计划都在同时推进。如果一切进展顺利，Frog Nation 将成为难以匹敌的「DeFi 黑帮」：

合并 Abracadabra 与 Wonderland

Abracadabra 和 Wonderland 是 Frog Nation 生态里最为成功的两个平台，前者的年收入高达 3 亿美元，后者则坐拥 8 亿美元的金库资产。但 Wonderland 近期一直在寻找新的出路，以取代 DeFi 2.0 的变基模型。在本周的大规模清仓事件后，为减轻 TIME 的抛售压力，Daniele 提议将 Abracadabra 和 Wonderland 合并，组建一个超大规模的去中心化 VC，以期让 Token 持有者通过来自协议金库的 Farming 以及所投资项目发放的空投获得收益。

这一新的 VC 机制已经在 Wonderland 的新 Token wMEMO 上展开了实验。就在前段时间，Wonderland 已经敲定了对跨链桥项目 Wormhole 的关键投资，这一投资将让 MIM 成为 Wormhole 的首选稳定币，目的是大幅增加 MIM 的采用率和流动性，助力 MIM 实现跨链第一稳定币的愿景。而 wMEMO 持有者将能获得 Wormhole 的空投。

合并 Sushi Swap

Daniele 在近期的博客文章中表示，Frog Nation 团队将接管 SushiSwap，帮助清理并恢复 Sushi 的效率，使其成为 Frog Nation 生态系统的一部分。

这项计划对于双方都有不少好处。Frog Nation 生态相当于添加了一个由自己完全掌控的 AMM，如果其发展能够步入正轨，将成为又一重要收入来源。而 Sushi 的社群之前确实相对较乱，原本就是依靠吸血鬼攻击起家，后期团队与社群又陷入了信任危机，导致 Token 过往表现一直不佳，所以的确需要一个强有力的新团队来重塑局面。

毋庸置疑，推动此次合并的重要因素之一，就是社群对 Daniele 和 Frog Nation 团队的信任。

抢占 ve(3,3) 发展先机

Daniele 在此前的 AMA 中透露，自己正和 Convex 团队谈判，为 MIM 争取在 Convex War 中的竞争优势。这就说通了为什么 Daniele 要和 AC 合作新的 AMM 项目 Solidly（此前称为 ve(3,3)），在律动《AC 哥拼命宣传的 ve(3,3) 究竟是什么？》一文中提到，Solidly 就好比 Fantom 上的 Curve，持有 Solidly Token 越多的协议激励投票权越大。

要知道，此前 Fantom 上的协议为了 ve(3,3) 快照可以说是大打出手。而 Daniele 与 AC 的合作却极有可能为 Frog Nation 生态内的协议赋予先天的竞争优势。

当生态发展处于这样的节骨眼时，团队的任何一个失误都有可能在生态内产生涟漪效应，带来难以难以想象的后果。本来一切似乎都进展顺利，但就是在这样的关键节点上，顶层团队的原则性错误却被曝光出来，直接撼动了 Frog Nation 的发展根基以及所有人对其前景的信仰。

MIM 脱铆

0xSifu 的消息被曝光后不久，涟漪就波及到了 Abracadabra。28 日早晨，大量用户开始将自己的 MIM 通过 Curve 平台上的 MIM-3CRV 流动性池换为 3CRV，使池中的资产比例严重倾斜。最终，整个池中一度只剩下不到 5% 左右的 3CRV，MIM 也开始脱铆。截止撰稿时，MIM 兑 DAI 的价格为 0.98 美元，但仍在脱铆状态。

更糟糕的是，很多人不能直接从 MIM-3CRV 中得到 3CRV，于是便先将 MIM 换成 UST，再用 UST 换 3CRV，这就形成了对 UST 的抛售压力，并牵连 LUNA 的价格开始下行。

尽管 Abracadabra 目前已启动了 mint 计划，激励用户用 3CRV 来 mint 出总计 43,881,642 枚新的 MIM，但还是难以扛住当前的抛售洪流，MIM 根本无法获得足够的市场需求。现在，Anchor 平台 Degenbox 的许多用户都会进行这样的操作：存入 UST 并借出 90% 的 MIM，再把 MIM 转为 UST 并重复操作，最终能获得 100% 以上的稳定币 APY。即使在 MIM 此次脱铆期间，很多人也不愿意为了归还折扣的 MIM 而放弃更高的 APY。

这对于意在成为采用率最高的跨链稳定币 MIM 而言打击巨大，尽管将来会有 Wormhole 作为支持，但用户对其锚定能力的信任和需求在短期内已经很难恢复了。

来自合作伙伴的担忧

此外，AC 也在推特上表达了自己的不满，他写道：「做事是要有原则的，一系列长期的欺诈、盗窃和恶意行为的过往不是「错误」，而是明确的意图… 让 0xSifu 管理资金，尤其是用户的资金，足以认定为玩忽职守… 我现在只想从 Willy Wonka 的魔法车上下来。」

尽管 AC 表示将来还会继续与 Daniele 保持合作，但可以明显地看出，为了减少此次事件对 Solidly 的影响，AC 正在与 Daniele 划清界限。Solidly 可以说是关乎 Fantom 生态未来的重要项目，结果在发行不久就出现这样的情况，对项目之后的发展的确带来不少麻烦。二者未来的合作深度、Frog Nation 未来能得到的福利多少都将受到影响。

现在，Frog Nation 团队在社群中的形象已经急转直下，被很多人称为「F(rug) Nation」。巧合的是，Yearn Finance 在此前也曾有合并 Sushi 的打算，在此次事件发酵后，Sushi 社群已经有成员提议放弃 Frog Nation 并重新投靠 Yearn。如果类似情况发生，Frog Nation 就真的是雪上加霜了。

DeFi 魔法师还是温水煮青蛙？

Daniele 作为 Frog Nation 甚至整个 DeFi 领域的灵魂人物之一，是享有极高声誉的。必须承认，他在此前的确是一个出色的 DeFi 魔法师：Abracadabra 平台很好地发挥了各类收益性资产的作用，促进了 Frog Nation 生态的发展。

但在 0xSifu 的消息被曝光后，后院的火也烧到了 Daniele 自己的身上，有关 Daniele 是否存在「温水煮青蛙」嫌疑的讨论也浮了出来。

在前不久，Wonderland 发生了一起大规模清算，导致原生 Token TIME 的价格在几个小时内从 800 美元暴跌至 360 美元。有人发现，这次清算的触发似乎与 Daniele 和 0xSifu 脱不了干系。

在大规模清算发生之前，Daniele 和 0xSifu 等团队成员使用 Wonderland 新 Token wMEMO 作为抵押借出了大量的 MIM，并将这些稳定币转至 Bitfinex 交易所，这实际意味着团队在不实际出售 wMEMO 的情况下进行了「套现」。

这次大规模的清算可能正是因为与团队相关的借款被清算而触发的。但在整个清算过程中，没有一个团队成员作出回应，本应到位的协议回购也没有发生，使 Token 价格最终跌破背书价格，抹去了许多用户的投资，实际上就是一次软性 rug pull。

更可怕的是，这种事情已经不是第一次发生了，在之前 DeFi 2.0 普遍暴跌时，TIME 就已经跌破了背书价格。同样，团队对此毫无作为，背书价格也随 Token 价格一起不断下跌。律动编者在《TVL 暴跌，DeFi 2.0 真的凉了吗？》一文对此也有所提及。

而现在，虽然消息真实性没有得到证实，但有关 Daniele 与 Bitfinex 关系的传闻也已开始发酵。

目前，0xSifu 本人也已就此事作出回应，表示愿意接受社群最终的决定。

虽然 Wonderland DAO 发起的投票也已经决定将 0xSifu 从团队中除名，Frog Nation 相关 Token 得的价格也有所回升，但我们不禁开始思考，在去中心化金融的世界里，究竟谁被赋予了的最大的权利？发生这样的事件究竟是个人还是机制原因？人们在 DeFi 中投入的资金是否真的安全？显然，Frog Nation 对此事的处理方式不仅关乎自己的生态，也影响了整个 DeFi 未来的发展。
