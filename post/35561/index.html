本文作者从技术、功能以及享乐价值三方面对NFT建立了一个基本模型，具象了NFT也可以被估值的概念。

以下为正文：

NFT（非同质化代币）的定价与BTC、ETH等同质化代币不同。NFT的价值取决于其类别、设计，以及在某些情况下的金融属性以及效用。随着越来越多的人开始了解NFT，仍然没有答案的问题是：

NFT 如何获得价值？

现实情况是，这背后没有确切的科学依据，也没有非黑即白的答案—即使不总是，也大多是灰色的。但是，与蒙娜丽莎或杰克逊·波洛克等不可伪造的实物艺术品不同，NFT有一些独特的数字可追溯的特征，可以客观地进行估值。我认为，在对NFT进行估值时，应该考虑到这些特征或价值系数。这些价值系数可以分为：

  • 技术价值： 链的安全性、去中心化程度以及稀缺性。

  • 功能性价值： NFT的实用性和流动性。

  • 享乐价值： 对NFT的期许性，NFT创造者+社区。

技术价值

链的安全性

NFT的意义在于它们是不可更改，是有保障的数字资产（建立在它们的底层区块链基础设施保持不可更改和有保障之上）。因此，以太坊是NFT网络的统治者，这在很大程度上要归功于它是当今最安全的智能合约平台，而且这种主导地位有望在可预见的未来延续下去。

以太坊上的 NFT 项目数量

换句话说，在可靠的链上铸造NFT有助于保证它的价值，这就是为什么以太坊上铸造的NFT目前比其他链铸造的NFT更有价值。

稀缺性

这是价值体现的一个主要原因，类似于同质化代币的总供应量。稀有度因素是NFT最核心的元素，收藏品中最稀有的NFT通常是市场上最有价值的。以下面的CryptoPunks为例子，是对稀缺性和售价之间关系的简单描述，提供了对市场价值的洞察，以及验证市场估值的快速公式模型。

资料来源：Cointelegraph

这并不是说，这个数学公式对CryptoPunks有效，它也会对其他NFT项目有效，它们的特征和属性是不同的。NFT创造者通过编程调整NFT的供应，以影响稀有性。例如，铸造1000个基本收藏品，只有10个黄金级收藏品。换句话说，除了稀有度之外，所有的变量都保持不变，比较稀缺的NFT的价格会更高。这构成了基于属性统计稀有性的收藏NFT的估价模型的基础。

去中心化的程度

完全在链上铸造的NFT，如Avastars、Aavegotchis和Art Blocksdrops，只依赖于它们各自的以太坊智能合约而存在。这意味着只要以太坊存在，它们就会一直存在，这可能会是一个非常长的时间。

另一方面，一些NFT项目使他们的NFT依靠外部的、非链上的供应商（如亚马逊）来获得便利和灵活性。这引入了一个信任的维度，所以你必须祈祷该项目会坚持下去并保持其服务器的运行。否则，你的NFT可能在几年内变成空白的。因此， NFT上链的价值不言而喻—它证明了自己，并且可以在任何时候证明自己。

功能性价值

实用性

这也许是NFT最开放的方面，NFT的效用描述了其功能和财务价值。

  1. NFT持有者拥有对功能性商品、服务或资产的权利。例如，通过NFT许可访问软件，保护知识产权或划定保险责任范围。

  2. NFT持有者拥有对未来现金流的权利，可以根据折现现金流模型进行估值。这方面的例子包括转售版税、数字房地产和有收益的NFT。

例如，一个更新颖的概念是NFT质押或挖矿。这指的是将NFT质押在一个协议上以获得收益。它给NFT带来被动的收益，可以认为是去DeFi与NFT的结合。

Onessus Blockchain Systems是一家位于美国的区块链游戏和NFT发行商。它的WhenStaking平台已经证明了收益NFT的概念越来越受欢迎。

另一种越来越受欢迎的形式是NFT的可赎回性。可赎回的NFT允许持有人将NFT换成实物或数字商品。虽然不是新事物，但它们的地位正在不断提高。例如，NiftyVille正在进一步探索可赎回的NFT，游戏中的这种可兑换元素创造了一个元宇宙和真实世界的跨界娱乐平台，将游戏和现实生活联系起来。

流动性

这也是吸引投资者溢价的原因。与 ‘链下 ‘收藏品相比，基于区块链的收藏品具有更高的流动性价值。考虑到易于销售和数量，投资者可能会重视在以太坊上发行的NFT。然而，其他几条公链正在逐渐占领以太坊的市场主导地位。

支持 NFT 的公链

享乐价值

期许性

这涉及到NFT给其买家带来的体验。

  1. 感知的社会文化价值，例如与艺术家、团体、事件或文化领域的关系。这也包括拥有一个NFT的社会效益—例如，访问独家数字社区，与团队认同或得到尊重和赞美。

  2. 购买者和NFT之间的个人、情感或审美羁绊。

  3. 这种享乐价值可以只是猎取稀有物品或完成收藏的乐趣。

创造者 + 社区

如果一个没有粉丝，也没创造历史的人在OpenSea或Rarible上投放了一个NFT，并且没有宣发它，它能卖出去吗？有时候能，但需要做更多的跑腿活来获得社区的关注。这就是为什么一个由主要艺术家或创作者出版的NFT赋予它价值。它被印上了发行者的数字指纹的魔力，当然这与签名没有区别。要记住的关键是，有参与的社区创造了需求。自然，创作者越受欢迎，社区越大，NFT就越有价值，这种动态对任何市场都是如此。

而社区在很大程度上存在在Twitter、Discord或Telegram上。有前途的NFT收藏品的标志是热情的社区，每天都有新成员加入。因此，必须注意以下几点：

  1. 该NFT的Discord或Telegram群组中的用户数量。

  2. Twitter粉丝的数量。

  3. Discord/Telegram用户和Twitter追随者的数量随时间增长。如果Discord是死的，或者Telegram充满了机器人，那就是一个危险信号。

  4. NFT是否有什么好处，比如独家内容或聊天室。

结论

当涉及到NFT时，别掉以轻心。我们都还在摸索如何准确评价它们，当然还没有正确的答案。从空间发展的速度来看，我们现在对它们的评价很可能与一年后的评价不同。底线是我们必须从多个角度来解决NFT的估值问题。但如果你能建立一个模型，你就会领先于市场。

译者笔记：

作者的观点归纳起来，一个NFT需要关注的维度有：

  1. 所在公链

  2. 该NFT在集合中的稀缺性

  3. 项目的去中心化程度

  4. 该NFT有什么特殊的用途或服务

  5. 文化属性

  6. 审美

  7. 社区热度

作者提到“以太坊是NFT网络的统治者，这在很大程度上要归功于它是当今最安全的智能合约平台，而且这种主导地位有望在可预见的未来延续下去”，这确实是NFT的现状，等同于L1公链之战的现状，在其他公链能叫出来名字的NFT屈指可数，但这也仅仅是现状。目前比较热门的NFT大多是PFP类型（个人头像类），其次是艺术类，这些类别蓝筹的项目也几乎都在以太链上。但是以太的高GAS低TPS的弊端，使更多的GameFi类NFT部署在了BSC链上，和体育、娱乐资本交好的Flow则有可能是体育类、音乐类NFT发展的温床…多链竞争发展，我觉得才能进步。

我相信NFT的终点不是市值是多少，而是NFT作为一个工具让人没有感知而又无处不在，烙印在crypto的血液里。
