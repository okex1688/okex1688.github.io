首先：浏览器钱包是旨在通过网络浏览器使用的加密钱包。

对我们大多数人来说，当我们看到「浏览器钱包」时，我们就会想到 MetaMask 这样的浏览器扩展项目，人们可以直接下载到 Chrome、Brave 等浏览器中。

但此外还有很多项目。除了钱包扩展，还有各种以浏览器为中心的服务，都属于「浏览器钱包」的范畴。

那么，哪一个能提供最大的价值呢？为了回答这个问题，我审查了 10 个不同的浏览器钱包项目，涉及四种不同的钱包风格。它们是：

浏览器扩展

  • MetaMask

  • Coinbase 钱包

  • GameStop 钱包

  • Rabby

  • Math Wallet

有内置钱包的浏览器

  • Brave

  • Opera

「任何浏览器」的钱包服务

  • WalletConnect

  • Portis

浏览器兼容的桌面钱包

  • Frame

今天，我们将讨论这些钱包的优缺点，并回答这个问题：「哪个是当今最好的浏览器钱包？」

让我们拭目以待……

MetaMask

网站：metamask.io

支持的浏览器：Chrome, Brave, Firefox, Edge

支持的链：所有以太坊虚拟机（EVM）区块链

MetaMask 扩展是加密经济中最广泛支持的浏览器钱包，已成为所有去中心化应用程序（dapp）的标准

优点：

  • 易于下载和使用

  • 很好地解释了钱包恢复短语 + 存储备份的过程

  • 易于导入现有账户或创建新账户

  • 与 Ledger、Trezor 和 GridPlus 硬件钱包兼容

  • 钱包内加密互换系统

缺点：

  • 不能像智能合约钱包那样找回丢失的账户

  • 由于 MetaMask 非常受欢迎，其用户经常成为社媒上钓鱼者的攻击目标

总体得分：9/10

Coinbase Wallet

网站：coinbase.com/wallet

支持的浏览器：Chrome

支持的链：所有以太坊虚拟机（EVM）区块链

加密交易所巨头 Coinbase 在 2021 年 11 月推出了 Coinbase Wallet 浏览器扩展。从那时起，Coinbase 迅速展开行动，使 Coinbase 钱包成为了 MetaMask 的可行挑战者。它的发展速度很快，但该钱包目前仍不及 MetaMask。

优点：

  • 自我托管

  • 易于下载和使用

  • 易于通过 Coinbase Pay 加载钱包

  • 提供一系列默认网络，如 Ethereum、Arbitrum、BNB Chain，另外还可以添加自定义网络

  • 提供自动锁定的计时器

  • 提供一个「登出」功能

  • 支持非 EVM 加密货币（BTC、BCH、DOGE、LTC、SOL、SPL 代币、XLM 和 XRP）

缺点：

  • 没有充分强调恢复短语 + 存储备份的过程

  • 不像 MetaMask 那样被普遍支持

  • 不能像智能合约钱包那样找回丢失的账户

  • 没有钱包内置代币互换

  • 暂时没有直接的硬件钱包支持

总体评分：8/10

GameStop Wallet

网站：wallet.gamestop.com

支持的浏览器：Chrome

支持的链：以太坊、Loopring L2

GameStop 是世界上最大的视频游戏零售商。最近几个月，该公司的 Web3 团队一直在建立 GameStop 钱包，以期待未来对加密经济的探索。该钱包因提供对 Loopring zk-rollup L2 的直接支持而引人注目。

优点：

  • 自托管

  • 易于下载

  • 清楚地说明了自托管的风险

  • 仔细解释了恢复短语的过程

  • 通过 Loopring 提供 L2 支持

  • 提供钱包内置互换系统

  • 提供对硬件钱包的直接支持

缺点：

  • 缺乏广泛的直接 dapp 支持

  • 不能像智能合约钱包那样找回丢失的账户

  • 目前只支持两条链

总体评分：7/10

Rabby

网站：rabby.io

支持的浏览器：Chrome, Brave - 即将支持 Firefox & Edge

支持的链：支持超 30 条链，包括 EVM 的主流，如 Ethereum，Polygon 和 Optimism，加上一些非 EVM 链。

由 DeBank 团队创建，Rabby 是一个浏览器扩展钱包，强调多链支持，有着清晰的交易签署体验，交易前风险扫描等。

优点：

  • 自托管

  • 易于下载

  • 开源，任何人都可检查或基于其构建

  • 广泛的硬件钱包 + 机构钱包 + 移动钱包支持

  • 很好解释了种子短语的过程

  • 提供钱包内小部件系统

  • 在用户界面中强调了安全审批系统

缺点：

  • UI 有点混乱

  • 缺乏广泛的直接 DApp 支持

  • 不能像智能合约钱包那样找回丢失的账户

总体评分：7/10

Math Wallet

网站：mathwallet.org

支持的浏览器：Chrome, Brave, and Edge

支持的链：少数 EVM 链加上比特币、Kusama、Solana、TRON 等。

Math Wallet 是一个加密钱包浏览器扩展，支持 100 多个不同的区块链。然而，与其他流行的加密钱包解决方案相比，该项目使用起来相对混乱。

优点：

  • 自托管

  • 易于下载

  • 支持广泛的区块链

缺点：

  • 钱包创建过程和钱包 UI 感觉不直观

  • 缺乏广泛的 DApp 支持

  • 不能像智能合约钱包那样找回丢失的账户

  • 没有强调恢复短语过程的风险和重要性

总体评分：5.5/10

Brave Wallet

网站：brave.com/wallet

支持的浏览器：Brave

支持的链：支持 EVM 链，因此支持 ETH，ERC20 s 等代币，此外还有 Solana 和 SPL 代币

2021 年 11 月，加密友好浏览器 Brave 推出了一个内置的加密货币钱包，称为 Brave Wallet。该应用程序可以让你直接从你的浏览器体验中与 Web3 互动，提供了对未来互联网的一瞥。它目前支持所有基于 EVM 的链和 Solana。

优点：

  • 自托管

  • 易于下载

  • 提供钱包内置互换系统

  • 与 Ramp 和 Wyre 兼容

缺点：

  • 缺乏广泛的直接 dapp 支持

  • 不能像智能合约钱包那样找回丢失的账户

总体评分：8/10

Opera Wallet

网站：opera.com/crypto/next

支持的浏览器：Opera

支持的链：以太坊，BNB 链，Polygon，和比特币（加上一些测试网）

2022 年 1 月，Opera 推出了一个专门的加密浏览器，内置了一个钱包系统。该浏览器还与 Discord 和 Telegram 直接集成，有着强烈的加密原生感。

优点：

  • 自托管

  • 易于下载

  • 提供钱包内置互换系统

缺点：

  • 缺乏广泛的直接 DApp 支持

  • 缺少对许多区块链的支持

  • 不能像智能合约钱包那样找回丢失的账户

总体评分：7/10

WalletConnect

网站：walletconnect.com

支持的浏览器：任何浏览器

支持的链：任何区块链

根据其网站，WalletConnect 是一个 「开源协议，用于通过二维码扫描或深度链接将 dapps 连接到移动钱包」。例如，Rainbow 钱包是一个流行的以太坊钱包，但还没有一个浏览器扩展。

WalletConnect 能使移动端的 Rainbow 用户（以及其他用户）很容易地通过任何浏览器接入任何 dapp。

优点：

  • 这是一个支持所有钱包的元钱包

  • 易于使用

  • 可在不同的浏览器上使用

  • 开源

缺点：

  • 目前没有发现 WalletConnect 有任何比较大的问题

总体评分：9.2/10

Portis

网站：wallet.portis.io/login

支持的浏览器：任何浏览器

支持的链：以太坊、比特币、Gnosis Chain、Polygon、EOS

Portis 为 dapp 提供了一个「简单的浏览器内电子邮件 / 密码登录方法」，对主流用户来说直观且熟悉。

优点：

  • 可在各种浏览器上使用

  • 钱包内备份功能

缺点：

  • 钱包的用户界面有点混乱（个人意见）

  • 缺乏广泛的直接 dapp 支持

  • 缺少对许多区块链的支持

总体评分：6.5/10

Frame

网站：frame.sh

支持的系统：Linux、MacOS 和 Windows，支持 Chrome 和 Firefox 扩展

支持的链：所有以太坊虚拟机（EVM）区块链

Frame 是一个桌面钱包，提供 Frame Companion 浏览器扩展。该连接器可以「将与 Frame（桌面本地以太坊钱包）的连接植入到无法直接连接 Frame 的基于浏览器的应用程序中。」

优点：

  • 适用于所有的 dapps

  • 以隐私为重点

  • 我的一些同僚非常喜欢这个项目

缺点：

  • 由于不断出现错误提示，我在新电脑上使用 Frame 桌面 + 扩展组合时遇到了麻烦，所以我很难直接探索这个项目

总体评分：5.5/10

结论

在审查了上面列出的 10 个浏览器钱包后，我目前认为 WalletConnect（9.2/10）是整体最好的产品，因为它是一个优秀的「元钱包」，它允许你在任何你想要的地方，以任何你想要的方式连接。

紧随其后的是 MetaMask (9/10)，它为加密经济中的直接钱包连接设定了标准，并且可能会继续如此。

以下是按类别排列的排名：

浏览器扩展

  • MetaMask - 9/10

  • Coinbase Wallet—8/10

  • GameStop Wallet—7/10

  • Rabby - 7/10

  • Math Wallet - 5.5/10

有内置钱包的浏览器

  • Brave—8/10

  • Opera—7/10

「任何浏览器」的钱包服务

  • WalletConnect - 9.2/10

  • Portis—6.5/10

浏览器兼容的桌面钱包

  • Frame—5.5/10

随着更多项目的涌入，预计浏览器钱包领域将在未来几年内大幅增长。

我们已经知道 Ledger、Rainbow 和 Argent 正在转向他们各自的本地浏览器钱包，而这些努力可能只是未来的冰山一角。
