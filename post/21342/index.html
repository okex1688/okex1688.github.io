撰文：Zoe Zhang

在 2019 年 1 月 3 日的《晓说》节目中，高晓松重回母校清华大学，来了一场「没有诗和远方，全是苟且」的演讲。本次演讲的主题是：《5G 与区块链：文创产业的重要赛道》。

演讲中，高晓松谈到：目前文创产业一家独大的垄断模式对市场造成不良影响，而区块链凭借去中心化、可溯源、不可篡改等特征，能够实现去中心化、碎片化的版权自动分发，是文创产业克服垄断问题的一大利器。

时隔 2 年，在区块链生态不断发展完善的前提下，有个团队希望通过构建一个基于区块链的全球化 NFT 音乐平台，尝试将高晓松在清华大学演讲中的设想变为现实，这个团队便是 Mozik。

去中心化综合音乐分享平台

根据国际唱片业联合会的《2019 年全球音乐报告》，全球音乐产业市场在 2019 年的收入为 191 亿美元，即 1337 亿人民币，与 2018 年相比增长 9.7%，这说明市场对于数字音乐商用版权的需求旺盛。

旺盛的市场需求下，在线音乐市场发展却饱受诟病。

一方面，在线音乐市场巨头垄断现象严重：超级平台占据了网络音乐生态的核心地位，在全球音乐版权营收的 191 亿美元中，环球、索尼及华纳三大唱片公司甚至占据了 68.6% 的份额。

另一方面，由于网络音乐作品极易被复制传播，多次传播之后版权确权成一大难题，导致侵权事件频发，无论是个人还是平台法律维权难度大。

而随着互联网的全面普及以及自媒体经济的蓬勃发展，越来越趋于碎片化的版权交易难以规范化，创作者的价值难以有效评估，IP 及衍生品市场孵化处于高风险状态…恶性循环下，在线音乐市场的每一位参与者都深受其害。

对于创作者而言：版权收益低，维权成本高，创作激情容易被消磨，独立音乐人难以生存。中国传媒大学音乐与录音艺术学院发布的《2018 年音乐人生存现况与版权认知状况调查》报告显示，中国有三成音乐人收入为零，近七成音乐人从事兼职工作，95% 的音乐人都无法仅靠音乐收益维持生计。

对于平台方而言：音乐版权支出过大，且维护成本高昂。

对于在线音乐消费者而言：乐迷仅作为消费方，无法参与到平台经济生态，参与积极性有限。

面对如上发展困境，Mozik 希望通过区块链技术建立点对点的利益分享机制，在更加公平合理的利益分享机制下，实现创作者、平台、消费者、投资者等生态圈内各方利益的平衡。上线运营 4 年来，已累计超千万粉丝。

作为一个去中心化的综合音乐分享平台，Mozik 的业务范围涵盖数字内容存储、版权确权、版权交易、粉丝经济等方面。

在 Mozik 平台：用户可以上架作品、可以参与作品交易、可以孵化发行优质 IP 以及可以通过搜索引擎和排名榜找到自己感兴趣的作品。除此之外，Mozik 还鼓励 DApp 项目方入驻 Mozik，打造音乐垂直数字创意内容平台。

在利用区块链技术保护并推动在线音乐经济生态发展方面，Mozik 提出如下解决思路：

建立版权保护系统

用户在线音乐创作和交易全过程的完整数据都记录在 Mozik 系统中。基于区块链不可更改的特性，作品可以被确认、被追踪、被交易、被监控。

版权 NFT 数字资产化

每一个音乐作品都是独一无二的，每一枚 NFT 也是独一无二的。在 Mozik，创作者可以将自己符合上链标准的作品发行为 NFT 数字资产。NFT 赋予了作品唯一性、可溯源、可交易等特征，让商品的估值过程更顺畅的同时，版权确权也不再是难题。

与此同时，Mozik 平台提供一键发行功能，并可根据用户发行需求，提供资产发行管理、资产评估、在线发行和交易以及精准用户推送等一系列技术支持和发行支持。发行之后，IP 拥有者可以继续发展和运营粉丝圈，也可以通过 IP 创造来激活社区，分享利益，最终形成可循环的微观经济生态。

版权交易系统

Mozik 设置了两种交易方式：版权交易市场、衍生品交易市场。

在版权交易模式下，上链作品默认登上 Mozik 内置的排行榜，购买者可以根据作品的知名度、风格、升值潜力等因素与创作者进行点对点的版权交易，交易过程由智能合约执行，做到公开透明。

另一方面，创作者也可以选择将作品完成后，不让作品上榜，而是直接将作品版权卖给购买商，购买商以发行人的身份帮助作品登上排行榜并取得更高排名。这样一来，创作者可以在初次交易的时候将自己的作品卖出一个符合自己心理预期的价格，购买商也有机会通过优化排行、孵化 IP 等方式，在作品发挥升值潜力时，获得升值回报。

IP 授权交易市场

衍生品交易市场是 Mozik 另一种交易方式，也是知识产权内容快速变现的一个重要环节。

Mozik 社区中存在非常多的优质原创 IP 项目，这些项目可以选择将 IP 授权给第三方，通过 Mozik 平台的知识产权保护体系，授权的整个过程中，版权归属将得到良好的保护。

获得授权的第三方可以围绕 IP 建立音乐周边拍卖、IP 联名等形式的衍生品交易市场，所有交易都在链上进行，IP 创作者与授权第三方共同分享衍生品交易市场中所获营收，实现 IP 创作者与授权第三方双赢的局面。

用户身份与账号体系

Mozik 会为所有参与 Mozik 生态的用户建立一个基于区块链的统一的身份系统。然后基于该身份系统，Mozik 建立服务于 Mozik 生态的账户体系。

该账号支持链上认证、钱包、统一授权等功能，且用户能够通过掌握相应的公私密钥对管理相应的虚拟资产，安全可靠的同时简单便捷。所有的用户都可以通过账号钱包进行 Mozik 平台虚拟资产管理。同时，统一身份系统可以实现圈子互通、数据互导、统一认证的功能，以及不同 APP 之间虚拟资产的交换。

身份系统与账号体系两层相互关联，保证安全的同时实现简单便捷。

交易排行榜

Mozik 系统设置了交易排行榜统计成交量，分为作品交易排行和买家交易排行两种。在早期阶段，排名中的每笔交易都有较高权重，也能获得更高的系统奖励。Mozik 鼓励交易者交易更多、更好的作品。在 Mozik 数字交易所成立后，交易排名的奖励系数将趋于正常，排名将影响作品进入数字交易所的权重，以及买家在数字交易所获得的权益。

Mozik 经济模型介绍

在 Mozik 经济体系，存在 MOS 和 MOZ 两个代币概念。

MOS 代币可视为 Mozik 平台发行的积分，其价值以 MOS:USDT=10:1 的方式保持相对稳定的状态，是平台经济活动的重要载体和工具，当用户面对购买平台内商品与服务、打赏创作者等需求时，用户需要使用 MOS 代币。

MOZ 代币是 Mozik 平台发行的权益代币，总发行量为 10 亿枚，根据用户参与生态的深度，Mozik 平台为投资者、粉丝、创作者等生态参与者发放 MOZ 分红权益。其分配规则如下：

  • 核心团队预留：12%，上主流交易所锁定 18 个月

  • 基金会预留： 20%，用于社区与项目运营

  • 生态激励：25% 用于生态建设，社区项目激励、粉丝激励和城市合伙人

  • 内容挖矿：25% 用于内容挖矿，激励创作人贡献内容以及粉丝贡献流量

  • 基石和天使投资： 18%

部分 MOZ 可以抵押给 Mozik 平台，部分 MOZ 可以逐步转化为 MOS，获得现金支持自己的音乐创作和运营，同时可以获得相应的平台分红。

Mozik 团队介绍

作为一个音乐类平台，Mozik 团队汇集独立唱片公司联合创始人、歌坛天后、当红流量众多音乐领域优秀人才。

核心团队中，Mozik 项目联合创始人兼 CEO 李杰出是前滚石移动数字音乐首席运营官、前新京报传媒新媒体中心副总经理，从事互联网、移动互联网行业十五年，深厚的从业经验让李杰出深刻理解音乐文化，充分了解用户音乐消费需求，是中国第一代数字音乐从业者；在技术实现方面，前滚石移动开发总监崔宇以及苹果认证技术主管韩金乌分别担任团队首席技术官和首席运营官，从产品设计与功能实现方面层层把关，推动产品优化发展；在版权内容方面，其负责人张苗是 2016 - 2018 全国十佳制作人。

战略顾问中，当红流量偶像欧阳娜娜、前滚石移动董事长李敬、现任索尼音乐艺人总监陈曦倾情加盟，其中欧阳娜娜为 Mozik 平台全球发言人。在音乐领域，Mozik 团队拥有深厚资源。

Mozik 发展规划

在已经到来的 2021 第二季度：Mozik 项目计划持续引入多个海外项目，初步建立国内和国外创作社区和粉丝社区，并计划不定期开展线下活动。

2021 年第三季度，Mozik 项目计划上线钱包功能、构建底层平台、实现 IP NFT 资产管理以及支持音乐平台功能。

2021 年第四季度，Mozik 项目计划初步建立入驻和发行的统一市场拓展机制、平台与应用平台共建模式以及社区化治理模式。

等到 2022 年第一季度，Mozik 项目将计划完善 IP 治理功能、基础平台和链社区移交共治以及完善资产、内容、用户社区功能。

结语

在中心化音乐平台占据主导的今天，音乐平台用户的痛点日益凸显，正如英国著名女歌手、作曲家 Imogen Heap 说的那样：「音乐创作者应该成为音乐的源头，而不是因为市场的其他原因在边缘挨饿。」

区块链技术应用于音乐行业虽然仍处于起步阶段，但它创造了一个人人平等的创作发展环境，给我们带来了音乐市场突破目前发展僵局的新思路。

通过 Mozik 项目的尝试，我们从音乐的角度看得到了区块链技术的价值，也希望未来，我们能在更多应用角度看到区块链技术的价值。
