金色财经 区块链8月23日讯   支付科技巨头 Visa 周一宣布正式进军非同质化代币（NFT）领域，而且已经购买了当前最火爆的NFT收藏品CryptoPunk。

按照 Visa 的说法，该公司上周以约 15 万美元的价格购买了 CryptoPunk 7610，CryptoPunk 总计有 3,840 名女性，而 CryptoPunk 7610 就是其中之一。CryptoPunks 被认为是历史最悠久的 NFT，由 Larva Labs 于 2017 年推出，总计包括 10,000 个像素艺术图像，每个图像看起来和正常图绘都不一样且非常古怪，但每个 CryptoPunk 都有自己的个性和独特的功能组合。

Visa 加密业务负责人库伊·谢菲尔德（Cuy Sheffield）在接受采访时说道：

“我们认为 CryptoPunks 将是 Visa在收藏领域的一个很好的补充，可以描绘和庆祝商业的过去、现在和未来，”

库伊·谢菲尔德透露，Visa 公司其实本身就拥有几件与商业相关的古董，作为其艺术收藏的一部分，包括早期的纸质信用卡和knuckle busters——一种商家在电子销售点终端出现之前用来记录信用卡交易的设备。

当被问及为什么 Visa 公司会在自己的收藏品系列中添加了一个 CryptoPunk 时，库伊·谢菲尔德解释说，CryptoPunks“开创了 NFT 技术和 NFT 商业浪潮”，所以 Visa 公司想拥有一个Punk。考虑到这是一个历史性的 NFT 项目，这个决定不是关于独立的punk，而是关于 CryptoPunks这个系列。

库伊·谢菲尔德表示，Visa 公司这次其实与 Anchorage Digital 合作购买了 CryptoPunk，这意味着 Anchorage 促进了交易并为 Visa 托管了 NFT。库伊·谢菲尔德补充说道： “我们使用法定货币从 Anchorage 购买了 NFT， Visa 公司在今年早些时候首次与 Anchorage 合作，在以太坊上以 USDC 稳定币结算付款。”

商业中的 NFT

Visa 公司相信 NFT 将在未来的商业中发挥重要作用。

按照库伊·谢菲尔德的分析，NFT 可以用一种全新的方式帮助个人内容创作者和中小型企业，甚至将会是文化和商业的交叉点。库伊·谢菲尔德将 NFT 与电子商务的早期发展进行了比较，称电子商务使小型企业能够在线销售并接触全球客户，但电商服务提供商仍然需要生产和运输实物，这可能会产生很高的前期成本。

库伊·谢菲尔德认为，相比之下，NFT 允许小型企业利用公共区块链来创建数字商品，这些商品可以立即交付到世界任何地方的加密钱包，他解释说：

“我们可以设想一个未来，加密地址变得和你的邮寄地址一样重要。”

Visa 公司看到了希望参与 NFT 领域的商家、品牌和内容平台数量越来越多，兴趣也越来越大， “所以现在我们致力于开始帮助客户探索他们可以参与 NFT 商业生态系统的方式。”——库伊·谢菲尔德说道。

Visa 公司还发布了一份关于 NFT 的白皮书，以帮助企业了解如何将 NFT 集成到他们的平台中，以及 Visa 公司如何为他们提供帮助。

库伊·谢菲尔德表示，作为 Visa 公司的核心角色，该公司希望帮助消费者购买 NFT，商家可以像对待传统商品和电子商务一样轻松接受 NFT。不仅如此，Visa 公司目前正在与其许多客户“积极互动”，帮助他们了解参与 NFT 生态系统的方式，但 Visa 公司拒绝提供具体客户名称。

早在 2019 年，Visa 公司就成立了内部加密团队，此后在该领域采取了多项举措，最近他们宣布成为第一个使用稳定币结算交易的主要支付网络，而且还着眼于促进央行数字货币（CBDC）支付。
