Yuga Labs杀死了Web 3

Web 3最大的神话是去中心化自治组织是获得创新的途径。在过去的一年里，我们看到Web 3的领头羊Yuga Labs，确凿地证明了Web 3并不是完美乌托邦。

有三件事经常被吹捧为Web 3的核心理念：

  • 去中心化组织的治理

  • 利益回归社区–不再有大公司控制一切

  • 资产和数据所有权归于个人

要评估Web 3的现实，最好的办法是看看Yuga Labs，即Bored Ape Yacht Club及其分支背后的公司，我将着眼于几个关键问题：

区块链的使用–它真的在使用区块链吗？

艺术家的补偿–艺术家能拿到钱吗？

社区控制–治理是否通过社区进行的？

企业所有权–是否远离了 “大科技”？

无聊猿游艇俱乐部

首先让我们来看看Yuga Labs的开端。

它存储了一个令牌，一个数字，合约用它来生成一个URI，这个URI指向一个指向图像的元数据的IPFS文件。它使用了IPFS，所以它可以被删除但不能被修改。不过，这仍然是加密顺势疗法。

那么它是否使用区块链？

它是否在区块链上存储了一些东西？是的，是一种代币。

它在区块链上存储数据吗？没有

它是否在区块链上存储对象（图像）？没有

Yuga Labs可以删除图片吗？可以

有一些人研究了它，声称它不是直接的IPFS，而是使用重定向，但我认为合约URI是到一个 “适当的 “IPFS地址。但是，当合约中的 “将此代币附加到此URI “只是指向一个文件时，声称这是 “使用区块链 “真的有点搞笑。

那么，艺术家们得到补偿了吗？这里有两件事，第一件事对我来说超级有趣，是否有人拥有这些图像。要拥有图像，你需要有版权，而BAYC生成的图像，是由电脑而不是人生成的，是否有版权，这是非常值得怀疑的。

因此，这里的第一点是，没有艺术家可以得到补偿，艺术家创造了猿猴的各个部分，以及整体的主题。所以第二点是，尽管如此，他们本可以从铸造中获得报酬，但这些人却与公司签订了合同，没有从铸造或后续销售中获得报酬，所以艺术家不赚钱。Yuga实验室从每笔销售中抽取一部分，钱则归公司所有。

因此，下一个部分是社区是否在控制之中。其中一部分是相当明显的问题，即Yuga Labs是否可以铸造更多的无聊猿….

我认为，如果公司想创造更多的东西就可以创造更多的东西，如果他们想的话，就可以把所有的东西都消灭掉，这就很难说社区是有控制权的。如果公司可以再印刷一批，而且你没有法律追索权，因为合同中写明他们可以这样做，你还会购买限量版的印刷品吗？或者花10万美元买一个东西，而公司可以远程销毁它？

“能够摧毁一件东西的人，他们就能控制它”

我不是说Yuga Labs会这样做，而是人们并不 “拥有 “数字资产，控制权完全在Yuga Labs手中，而且鉴于 “代码就是法律”，这都是公开的信息。人们可以说他们信任Yuga实验室，但这与 “无信任 “区块链或Web3的说法是对立的。

因此，这既是对社区的一个巨大的否定，也是对企业控制的否定。我不打算分解每一个NFT mint，但我要指出，Mutant Apes有setBaseURI。

让我们在这里也明确一下这些NFT和它们的合约是做什么的。

取一个模板图像

将图像解构为若干部分

在一个特定的部分上创造多种变化，但要在定义的尺寸/覆盖范围内，实现数百万/数十亿的可能组合

创建一个随机的变化组合（我假设他们会检查是否匹配）。

将图像保存到IPFS，然后将元数据+IPFS链接保存到IPFS，并附上一个令牌号码

出售NFT代币

他们改变步骤1、2和3，然后可以随意重复4、5和6。这就是为什么可能无法对图片进行版权保护。如果他们愿意，Yuga实验室可以重复数百万次，重新使用基础图片并创建一个新的 “Bored Ape Yacht Club 2.0 “系列，或者只是在现有系列中生成新的图片。这都是由Yuga实验室控制的。

现在开始第二部分，以及APE币是否可以作为公共货币。

如果无聊猿猴不是Web 3，那么APE币呢？深入了解Yuga实验室如何突出Web 3的缺陷。

APECoin

APECoin是Yuga Labs的货币，它目前是基于ETH的代币，所以听起来很像Web 3。

这里的答案其实很简单，与Otherside mint有关，他们很清楚，Web 3的方式对他们来说是不可行的。

他们是正确的，而且如果Yuga Labs愿意的话，他们有绝大部分的ape代币，这对现有的所有者来说更像是一种威胁。Yuga Labs可以完全强行通过DAO的任何决定。

或者他们可以完全无视ETH上的APE币，建立一个完全不同的、不在ETH上的APE币，并抛弃目前的所有者。这里的权力完全在Yuga Labs手中。所以它使用区块链，因为它只是一个ERC20代币，但他们已经有效地宣布了这种转变，如果要建造Otherside，它需要转移到一个单一的公司私有链。

Otherside 的铸造

最近的Otherside铸造在几乎所有明智的商业措施中都是一场灾难，虽然它筹集了大量的资金。

它还燃烧了大量的Gas

所以铸造基本上 “证明 “了ETH不能处理他们的负载。他们可以做一些更有效的事情，他们可以使用另一种铸造机制避免gas大战发生吗？但他们并不想这样做。

为一个游戏筹集3亿美元是令人印象深刻的，但早期迹象表明，除了钱的部分，Web 3并不是计划中的重要部分。

Otherside是一个游戏

“预告片 “是一个很好的开始

这是一个不存在的游戏，人们可以在其中花钱买地。这个游戏的独特之处在于：

抛开文字游戏，在这个游戏里你可以拥有土地，并使用与NFTs相关的皮肤。记得之前说过，皮肤是没有版权的，为了让皮肤在游戏中出现，你需要Yuga Labs为游戏创建一个3D版本的皮肤。你是否能够为你自己的NFT提供自己的3D模型，或者你只能使用Yuga Labs的NFT。换句话说，这是一个由创作者和艺术家拥有的开放系统，还是你必须在封闭的Yuga Labs NFT生态系统内工作？

我以前说过，Metaverses将建立在游戏而不是区块链上，他们已经与Improbable.io合作，帮助他们进行游戏创作。

Improbable.io不是独立的公司，也不是硅谷科技巨头，这是由同一家风险投资公司A16z支持的企业合作。

铸造的Otherside是什么？对我来说，这就是我们与Web3所宣称的雄心壮志相去甚远的地方，以至于很荒谬。使用与他们正常的NFTs相同的合约，这个铸造有一个相当大的区别。他们不再使用IPFS来存储元数据，现在是一个标准的REST端点，https://api.otherside.xyz/lands/65995 是一个例子

但在铸造的前几个小时，资产还没有被创建。Yuga实验室已经改变了令牌所指的内容。与 “无聊猿 “不同的是，他们可以删除它们，他们可以真正地做任何他们想做的事情，而且已经做了。图片只是作为资产存储在另一个URI上，还是从一组部件中生成的，以创造 “独特的 “资产。

所以这是传统web2.0网络，有了setBaseURI方法，他们将来可以把它重新指向更像Web 3的东西，或者至少假装是Web 3。

除了NFT皮肤之外，游戏的功能是超级通用的。

这些功能听起来并不那么令人印象深刻，有关于并发性的大胆主张，Improbable.io创造了一个单一的游戏，似乎没有抓住时代的潮流，但3亿美元可以买到大量的开发时间，像这样的游戏，如果他们要为这些投资者显示出回报，就必须要受欢迎，游戏需要快速迭代，他们正在谈论大规模的连接和并发。

Otherside将需要APECoin的链式转移

首先他们想用APE币作为游戏中的货币。这意味着它需要快速交易，人们不会想要在eth上等待5、10或15分钟来确认在游戏中的购物，所以ETH的处理速度是不可能承载这个体量。这给了我们两个选择：

  1. 去L2，把结算留给ETH

  2. 创建一个他们完全拥有和控制的链

他们已经说了他们想走第二条路，说实话，如果你正在创造你自己的游戏货币，需要快速结算交易，L1的交易速度和高gas导致你的游戏受到巨大的用户群影响。那么，这与《第二人生》中的虚拟币或任何其他游戏货币有什么不同？换句话说，他们想创造另一种L1货币来与ETH和BTC竞争，而且他们可以控制性能特征以确保它在游戏中发挥作用。

他们不能让一个游戏推出后，由于人们试图交易战利品而燃烧掉数百万的ETH。这些钱不仅会从游戏中流失，而且玩家们会慢慢放弃游戏。

Otherdeed将在游戏中出现

第二块是契约，那些图像和元数据显然是没有价值的，除非它代表了游戏中的某些东西。

因此，为了使它有价值，你需要控制游戏中的土地，让用户进入这个游戏，它需要有赚钱效应，我们需要几百万活跃的玩家，这可能意味着多个服务器，我们的契约是基础模型，如果我们在一个世界上建造，它会出现在所有的世界上，或者一个建筑只是暂时存在？

我们的资源是有限的还是无限的？是像《堡垒之夜》那样，每场比赛都是新的开始，还是像《Minecraft》那样，任意建设和破坏？如果它像Minecraft一样，每个契约指的是同一个固定的地方，还是在每个世界中生成的地方？

很明显，一个正常的deed最重要的东西在Otherside中是缺失的，在游戏推出之前，它只是另一个图像NFT。

有20万份otherdeed，地契是否包含关于空间状态的详细信息？那么，如果你在上面建造，是否记录为与契约相关的NFT？如果你开采一些资源，是否通过另一个NFT机制来减少？我们不知道。但我们知道的是，一个需要进行400,000次REST调用来获取地图的游戏不会有很快的启动时间。

我们知道它不需要这样做，因为Yuga实验室完全控制了API另一端的数据库，所以他们可以直接调用这些数据而不需要使用NFT。

他们创建 “otherside"的方式意味着所有这些资源都在他们的控制之下，所以他们可以将其作为一个世界进行优化，然后修改otherdeed以指代生成的世界的定义部分，有点像Minecraft的世界。

并不是说这是某种欺诈行为，我是说他们通过REST API给人们提供一些元数据和JPEG文件来进行Web 2的众筹，整个架构的实现，甚至整个游戏的设计，都还没有完成。整体世界100%在Yuga实验室和其同伴A16z公司的控制之下。

在大约12个月的时间里，Yuga Labs开创了Web 3，然后又将其扼杀。

Web 3已死，众筹万岁

Yuga Labs是一家典型的美国科技创业公司，他们结合了我们在以前的创业公司中看到的一切。改变世界的愿景，“社区 “的呼声，接着是中心化和控制的良好路径，我从来没有见过这么快的骑士变恶龙故事。

那些与合约有关的问题并不隐蔽，它们是公开的。图片可能不受版权保护的事实，也没有隐藏。图片没有 “存储在区块链上 “，虽然他们的一些支持者围绕着Web 3是 “为创作者和艺术家 “以及通过Web 3 “你拥有数字事物 “的说法发表了一些废话，但Yuga Labs的代码是诚实的：Otherside在他们的控制之中。

如果他们能够创造出一个现象级的游戏，重新定义这个类型，并真正使创作者能够独立挣钱，那么我们都会震惊。如果他们允许任何人带来他们自己的NFT皮肤并出售，即使他们从中抽取费用。如果它真的是一款击败《GTA V》、《堡垒之夜》、《使命召唤》和所有其他大型游戏的游戏，并且重新定义了这一类型，那么我是赞成的。但这并不意味着这将是一款Web 3游戏。

“去中心化，通过社区自治进行管理，人们对他们想要的东西进行投票，并在游戏中使用公共区块链，这样东西就可以在生态系统之外进行交换”这是Web 3中最大的废话，认为通过委员会设计是获得一个真正的好游戏、好产品的方法。

Yuga需要杀死Web 3

因此，对Yuga实验室来说，杀死Web 3是很公平的，因为它正是创新的对立面。要建立一个像样的游戏，你不可能有20万个 “创意总监”，他们纯粹是根据他们是否觉得它能赚钱来投票。你不能让20万人决定游戏中什么是和什么是不存在的。

这些otherdeed持有者能够有机会拿回他们的钱的唯一方法是完全信任Yuga实验室创造一个好游戏。没有其他办法，没有 “不受信任 “的游戏开发过程或基于 “DAO “的开发过程会创造一个好游戏。以前也有过做这种事情的尝试，但确实不顺利。游戏设计是一门生意，就像制作电影是一门生意一样。对每一步都进行投票并不能做出好的游戏、电影或贸易政策。

对于NFT，你可以说服自己有价值，而价值就在于这种市场认知，一切由市场决定，抛开大规模的清洗交易问题不谈，市场会支付NFT的价值，这个理论对艺术来说确实有效。

Otherside则不同，它是一个不存在的产品，这需要几年的时间才能建立，并且需要Yuga实验室找到一个游戏可玩性来吸引非otherdeed持有者加入。

没人在乎他们是否遵守NFT元数据的规定

最后一点是由某位硅谷领导人总结出来的：

「有些人说：“消费者想要什么就给他们什么。”但那不是我的方式。我们的责任是提前一步搞清楚他们将来想要什么。我记得亨利·福特曾说过，“如果我最初是问消费者他们想要什么，他们应该是会告诉我，‘要一匹更快的马！’”人们不知道想要什么，直到你把它摆在他们面前。正因如此，我从不依靠市场研究。」

那些otherdeed持有者并不关心最终的游戏是否像《GTA VI》或其他类似的游戏，只要最终的结果能使他们赚钱。如果他们关心游戏可玩性就太愚蠢了，如果游戏看起来像《使命召唤》之类，人们真的会抗议吗？如果只能使用Yuga实验室的皮肤而不是第三方皮肤，他们会在意吗？当然不会，除非这减少了收入的来源。

Yuga实验室完全控制着NFTs和游戏，它是一家由VC支持的巨型公司，他们刚刚为历史上最大的游戏开发预算提供了资金，并在这个过程中杀死了Web 3。
