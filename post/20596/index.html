NFT的火爆 到底是昙花一现还是下一个发展机会？

除了销售独一无二的 JPEG 和游戏中的物品外，NFT 还有其他的应用。那么NFT 究竟是什么？技术的下一步又是什么？

价值 560,000 美元的“纽约时报”专栏文章、价值 6900 万美元的 Beeple 的 JPEG、价值 700 万美元的加密朋克头像、Jack Dorsey（译者注：推特CEO）价值 290 万美元推特，都反映了当下 NFT 的大热。

NFT 火爆全球，达到了高昂的价格和爆炸性的炒作。与 2017 年的 lCO (首次公开发行)狂热相比较，截至 3 月底，NFT 的销售额超乎预测的突破了 5 亿美元。

简而言之，NFT 代表的是通过区块链技术进行验证的数字资产，从图像到歌曲、视频、再到推特，拥有 NFT 允许个人对这些数字资产拥有独一无二的所有权。

至关重要的是，NFT通过验证数字商品的稀缺性和来源，提供了一种将其通证化的方法。

在本报告中，我们深入探讨了什么是NFT，NFT的机会在哪里。

什么是NFT？

NFT 市场 OpenSea 将 NFT 定义为“独特的，拥有区块链管理其所有权的数字物品”。

例如一张罕见的棒球卡是有价值并且稀缺的，不能被任何其他的棒球卡所取代。数字文件也是如此。

Beeple 以 6900 万美元售出拼贴图像文件，同时 Nyan Cat 的创建者以 60 万美元的价格卖出了这款标志性的 gif，Grimes 以 600 万美元的价格出售了一系列短片，莱昂国王发行了一张新专辑并以 NFT 的形式发行了音乐会门票。

这些令人瞠目结舌的价格很可能是由投机推动的，但是，艺术的货币价值问题实际上是传统艺术界长期以来一直在努力解决的问题。

资料来源：Foundation

在 NFT 的背景下，已经出现了关于“所有权实际上意味着什么”的争论。在许多交易中，NFT 并不代表实际资产本身(也不代表 IP、复制品或版权)，而只是所有权的记录。

有些人把它比作拥有房子的地契而不是房子本身。或者把它比作拥有一份经核实的，亲笔签名的蒙娜丽莎，但是它不是原来的蒙娜丽莎也不是印刷品。交易的具体条款可能因 NFT 而异，导致了所有权定义的混乱。

尽管如此，在将 NFT 定义为“新所有权形式”时，NFT 有一些固有的特征，包括：

1、稀有而独特：这就是 NFT 的“不可替代”部分。NFT 是完全唯一的，没有两个 NFT 是相同的。

2、互操作性（译者注：也可以理解为跨链属性）和可交易性：由于许多区块链标准是具有开放性的，NFT 可以跨不同的生态系统、钱包、市场等进行移动。因此，用户可以使用 NFT 在其原始系统之外进行商品交易(例如，销售电子游戏皮肤)。

3、不变性：一旦 NFT 使用区块链技术进行编码，任何个人都不能更改其所有权数据/历史或元数据。NFT 允许持有者证明其资产的真实性和原创性，并通过其存储的任一区块链进行验证。

4、所有权 ：持有 NFT 通证可以让人们证明自己是该资产的唯一所有者。

其他功能包括将版税编入通证，允许艺术家在此后持续获利或者包括享有部分所有权。

目前，绝大部分 NFT 都是使用以太坊标准构建的，尽管技术上 NFT 可以在如 Algorand，Tezos 和 Polkadot 等公链上创建。

购买或出售 NFT 的标准过程如下：

1、建立一个数字钱包并购买加密货币

2、将钱包连接到 NFT 市场

3、铸造/展示你的 NFT 或者竞拍 NFT 作品

这项技术实际上并不是新技术。2017 年，人们花了数百万购买和销售一种可收藏的数字猫 CryptoKitties，其中一些售出价格超过了 10 万美元。

面对不断增长的需求，最近出现了越来越多的 NFT 市场。OpenSea 最近筹集了 2300 万美元，由风投公司 Andreessen Horowitz 领投（A16Z）。

Sorare，SuperRare，NiftyGateway，MakersPlace，Decentry aland，Rarible 等等 NFT 市场也如雨后春笋般蓬勃发展。这是继 2020 年末人们急于通过锁定他们的投机性通证来赚钱的 DeFi 和流动性挖矿热潮之后，又一热潮。

那么，为什么现在 NFT 引起了如此多的关注？

一些人引用财富效应来给予解释，当一个人的净资产增加时他会花更多的钱。

在 2021 年第一季度末，以太坊的价格同比飙升了 1000% 以上，比特币的价格在同一时期飙升了 600% 以上，突破了 5 万美元，这意味着早期加密货币买家获得了巨大的回报。

反过来，昂贵的 NFT 的购买者已经深深地扎根于加密货币领域，Beeple 价值 6900 万美元的艺术品被一位想要建造虚拟 NFT 博物馆的加拿大加密货币投资者买下。

机会在哪里

NFT 最早的一些应用是以游戏和数字艺术为代表的，同时也代表了生态系统中一些最大的资产类别。

游戏中的 NFT 体现在例如交易卡的数字收藏品中，例如区块链游戏 Gods Unchained、CryptoKitties，或者其他作为虚拟世界的资产。

在虚拟世界中，用户可以购买 Sandbox 中的虚拟地块, 可在 Decentraland 戴热狗帽，甚至可以获得 Enjin 的“通用”斧头。

“通用”斧头不是为一个特定的游戏开发的而是存在于更为广泛的生态中，表明了 NFT 交易项目的互操作性（跨链）的潜力。

想想视频游戏平台 Steam 的社区市场，在这里玩家可以从该平台的不同游戏中购买和出售游戏内部资产(如《军团要塞2》中的火焰喷火器)——但这是为了比 Steam 更广泛的生态系统。

资料来源：OpenSea。

对于艺术来说，大部分信息都是围绕着艺术行业的改革而展开的，一些艺术家想要绕过掮客，直接把他们的艺术品卖给大众。对于 NFT 的拥护者说，这项技术允许创作者在传统框架之外赚钱，从而使创作者和粉丝之间的联系更加紧密。

知名品牌也加入到了这一行动当中，标志着 NFT 进入主流，由市场 FOMO 和潜在的营销机会驱动。比如：

1、佳士得主持了价值 6900 万美元的 Beeple 艺术品的拍卖会；

1、NBA 的 NBA TOP SHOT 2 月份销售额约为 2.5 亿美元。

3、食品品牌也是如此，塔可钟的玉米饼主题 NFT 在美国 30 分钟内销售一空。普林格出售了一罐数码碎片作为NFT；加拿大必胜客出售像素化比萨片

4、Grimes、William Shatner、Paris Hilton和Lindsay Lohan 等名人都卖过数码艺术品的 NFT

5、音乐艺术家 3lau 和莱尔国王乐队都通过 NFT 出售了专辑

6、时代周刊《Time》拍卖了一些封面作为 NFT

挑战和缺点

当前的 NFT 交易和 Dapp 的问题在很大程度上是围绕着这些系统的效率产生的。

由于交易发生在“链上”的性质，目前的交易费可能昂贵得令人望而却步。一些公司正在开发潜在的解决方案，如侧链、rollups 等等（译者注：一些扩容技术来提高效率，降低费用）。

此外版权一直是围绕数字资产的噩梦。可以被编码成区块链交易的智能合约从技术角度看可能具有约束力，但在法律上的可执行性是难以回答的问题。

NFT 的支持者声称该技术将一劳永逸地证明所有权。但是我们还必须考虑数字所有权和知识产权的模糊定义。

例如：当你购买一条通证化的推特时，你实际上拥有什么？“拥有”的数字内容实际存储在哪里？如果NFT市场崩溃会发生什么？你如何区分模因（Meme）与混合模因的所有权？如果你买的东西的标记版本被证明是被盗的，会发生什么？

Castle Island 风险投资公司合伙人兼 Coinmetrics 创始人 Nic Carter 指出,“ NFT 是一个过程，而不是一个产品。对 NFT 来说，就是给它分配一个存在于公共区块链上的不同序列号。‘NFT’ 除了传递有关序列化内容的目的或性质的其他信息外，不传递其他信息。”

另一个常见的问题是工作证明（PoW）的区块链对环境的巨大影响（译者注：比如比特币、以太坊）。据估计，仅以太坊网络的年化碳排放就接近 16 兆吨二氧化碳，相当于立陶宛全国的碳排放，而它消耗的电力为 34 太瓦时，与丹麦的电力消耗类似。

资料来源：Digiconomist

一些人已经在研究如何提高 POW 区块链平台的效率。例如，大部分去中心化应用程序都是建立在以太坊上的，它最近推出了升级方案以太坊 2.0 的一部分，有望建立一个更环保的系统。

接下来是什么？

NFT 有可能对游戏和数字艺术以外的行业产生深远影响。

在内容上，像 Mirror 这样的公司允许作者拥有和出售他们的作品作为 NFT，开辟了内容分发和货币化的新途径。因此，分散自治组织（DAO）的想法又开始浮出水面，Ark Gallery 是 CryptoPunk 收藏家生态位群落的一个例子。

一些公司正把业务拓展到更深的金融服务领域当中，将 DeFi 和 NFT 结合。NFTfi 允许用户利用他们的 NFT 作为贷款抵押品，同时 NFTX 推出了 NFT 指数基金允许买家拥有多个 NFT 的一部分。而 Nori 公司表示，它已经将碳权通证化。

过去几年，随着基于区块链构建的应用程序和协议的不断发展，已经引入了一轮炒作，包括当天价需求淹没了生态系统时出现的一些泡沫。但随着创造者经济的概念和所有权的定义不断发展，NFT 可能会有更大的机会。
