NFT 的市场规模有多大？ 这个问题很难回答，因为 NFT 可以代表很多截然不同的资产类型。债券和房地产的所有权可以用 NFT 表示，艺术、游戏内物品和音乐也可以。

可以从下图看看不同类型 NFT 的广泛分类。

证券类 NFT 可能构成一个巨大的市场，但所需的基础设施和法规还处于发展的初期阶段。相比之下，收藏型 NFT 在今年迅速发展壮大成为一个以 10 亿美元为单位计算的市场，最近每天的交易额超过 1 亿美元。让我们先从关注后者开始。

收藏型 NFT 可分为多个类别。例如，CryptoPunks 可以被视为艺术和历史文物，因为它们是以太坊上最早的 NFT 收藏品之一——请注意，Visa 最近买进了一个 CryptoPunk，称其为历史性的商业文物。

NFT 一开始可能只是一种艺术或 Meme 玩梗，后来可能作为物品或化身集成到游戏中：想象一下用你买的的无聊猿（Bored Ape） NFT 作为一个新角色，去玩一个新版本的街头霸王（Street Fighter）游戏。

衡量 NFT 市场规模的一种方法是自下而上的方法——查看 Topshots、CryptoPunks、Artblocks 等的当前市值，并预测它们的增长。Nansen 估计，这些顶级 NFT 系列的市值超过 40 亿美元。

但考虑到某些子板块（meme、视频游戏时刻等）尚未取得突破性进展，仅基于当前市场增长的预测可能会大大低估 NFT 的市值。 例如，目前市值为 400 亿美元的狗狗币可以被认为是碎片化 NFT 的早期代表，但 meme NFT 的一级市场销售额几乎没有达到八位数；我们预计最近出现的碎片化（fractionalization）协议会加速这一趋势。

自上而下的估值方法主要手段是参考与可收藏 NFT 相似特征的现实世界物品市场，两者能满足相似的人类需求。这些特征归结为以下需求的组合 ：

  1. 不是普通人的日常需要

  2. 可以用来满足「有钱就是任性」的炫耀（请参阅 Arthur Hayes 最新的文章 ）

  3. 具备文化或历史意义

一些可以作为参考、拥有完全数字化对应物的现实世界市场包括：

  • 奢侈品 (时尚产品、豪车、名表、香槟)

  • 艺术品

  • 高端房地产

  • 文物

  • 其它收藏品 (邮票、体育收藏品、钱币)

奢侈品市场规模有多大？ 看一下几家公司的市值列表（按市值排序）：

  1. 3650 亿美元– 高盛

  2. 1530 亿美元–路易威登集团

  3. 1330 亿美元– Uber

  4. 750 亿美元– Walgreens

  5. 420 亿美元– 爱马仕

哈哈，我开了个玩笑。上面列举的市值数字是对的，但排序被我打乱了。事实是：路易威登集团和爱马仕分别位列第一和第二位，高盛、Uber、Walgreens 分别位列第 3、4、5 位。正确排序是这样：

  1. 3650 亿美元– 路易威登集团

  2. 1530 亿美元– 爱马仕

  3. 1330 亿美元– 高盛

  4. 750 亿美元– Uber

  5. 420 亿美元– Walgreens

艺术品市场呢？ 据报道，2020 年全球艺术品市场的销售额为 500 亿美元。但非数字艺术品的周转率并不高，买家购买的艺术品通常不会在同一年转售，所以 500 亿美元这一数字并不等同于所有艺术品的价值总和。

如果我们具体看一位艺术家：毕加索，他所有 50,000 件作品的估计「市值」在数百亿美元。如果再算上莫奈、伦勃朗、波洛克和巴斯奎特会怎样？

有些人认为 NFT 市场已经饱和。CryptoPunk、artblock、无聊猿等太多了。但与 NFT 能涉猎的市场数量相比，情况并非如此。

百达翡丽被认为是顶级奢侈手表品牌之一，从 19 世纪以来一直存在。他们每年生产 60,000 多只手表。看一下其他奢侈品牌的年产量： Richard Mille 手表每年大约生产 4,000 多只、兰博基尼跑车年产 8,000 多辆，爱马仕包年产 100,000 多只。年复一年，这些物品的稀缺性日益降低。而对于带有封顶上限的流行 NFT，情况正好相反，因为 NFT 随着时间推移自然会出现丢失。

最后同样重要的一点是，考虑一下目前估计 每年 400 亿美元的游戏皮肤市场。这个市场本身已经是数字化的，而且不久之后游戏开发商可能会开始抛弃割据收费的模式，会开始通过 NFT 实现更开放的市场。想象一下，随着通过游戏内资产赚取现金流这一有现实意义的方式引入，这个市场将会有多大规模的增长。

那么 NFT 市场到底有多大？ 比我们大多数人想象的要大得多。
