区块链应用

▌经济日报：利用区块链等现代信息科技提高供应链金融数字化水平

12月1日消息，保产业链供应链稳定，是畅通国民经济循环的一个重要基础。供应链金融模式下，金融机构所关注的原材料或产成品存货都是融资主体日常经营所必需的。通过利用人工智能、大数据、云计算、区块链、物联网等现代信息科技，可以在供应链金融发展中实时监控物流、商流、资金流，完善风险控制技术和模型，提高金融服务的精准度、覆盖面和便利性，从而在完善风险防控的同时推动业务更好发展。

▌工行首席技术官：建议国家从顶层设计方面加强区块链应用创新的规划布局

工行首席技术官吕仲涛近日在“数字金融创新应用与发展”研讨会上表示，近年来各行业的区块链应用也是创新活跃、百花齐放，在快速发展的同时面临着一些挑战，主要来自两个方面。第一个挑战是在技术方面，第二个挑战是在应用方面。对于进一步推动区块链创新发展，吕仲涛同时也给出了三点建议。

加密货币

▌印度前财政部长：内阁尚未考虑禁止私人加密货币的措施

金色财经报道，印度前财政部长Subhash Garg称：“声称私人加密货币将被禁止是一种误导”。为了缓解担忧，这位前财政部长强调，内阁尚未起草或考虑此类禁止加密货币的措施。与此同时，财政部长Nirmala Sitharaman今天在议会宣布，此前的加密法案正在修改，新法案可能很快就会出台。虽然加密广告不会被取缔，但正在尝试教育投资者。

▌灰度将面向机构投资者和高净值投资者推出Solana信托基金

11月30日消息，灰度投资将推出第十六个产品Solana信托基金，该产品面向机构投资者和高净值投资者。Grayscale首席执行官Michael Sonnenshein表示，提供该产品的一个主要原因是Solana的近期和长期潜力，尤其是在确定区块链技术的新用例方面。灰度Solana信托基金最低投资额度为25,000美元，收取2.5%的管理费。

▌灰度：SEC一再拒绝现货比特币ETF可能违反行政程序法

金色财经报道，在美国证券交易委员会（SEC）最近拒绝了比特币交易所交易基金(ETF)之后，灰度正在反驳该监管机构的论点。灰度致函SEC，声称该机构愿意批准基于期货的产品而非现货产品是“任意和反复无常的”，称SEC的一再拒绝可能违反《行政程序法》(APA)。据悉，APA管理联邦机构的决策过程。灰度在信中说：“根据1940年法案注册的比特币期货ETP和不需要或没有资格注册的现货比特币ETP在所有相关方面都是相同的，但根据2021年11月12日的不批准命令中的分析，SEC对它们的的处理不一样”。

▌Coinbase收购以色列多方计算技术公司Unbound Security

金色财经报道，Coinbase正在继续其收购狂潮，据The Block消息，Coinbase正准备收购Unbound Security，这是一家专门从事多方计算 (MPC) 技术的以色列加密托管基础设施服务提供商。Coinbase 发言人表示，Coinbase 计划将 MPC 技术整合到其零售和机构产品中，目前这笔收购交易的具体金额尚未对外披露，迄今为止，Unbound Security 已筹集了至少 4000 万美元的外部资金，员工人数超过 70 人，预计全部收购交易会在几个月后完成。收购交易完成后，Coinbase还打算拓展以色列业务并将在该国一个“卓越技术中心”。 Unbound Security 团队将成为这个新研究设施的核心工作。

▌2家基金公司Horizons和Evolve同时在加拿大推出元宇宙ETF

11月30日消息，两家基金公司Horizons和Evolve周一同时在加拿大推出元宇宙ETF，这是该国首批此类ETF。Evolve Metaverse ETF（MESH）和 Horizons Global Metaverse ETF（MTAV）都将在多伦多证券交易所交易。Evolve的ETF将积极投资于参与元宇宙发展的股票组合，包括动视暴雪、Roblox、微软及Meta。Horizons的ETF将复制Solactive Global元宇宙指数的表现，最高权重公司包括腾讯、S4 Capital及Qurate Retail。Evolve收取0.60%的管理费，Horizons收取0.55%的管理费。
