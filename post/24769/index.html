本期文章为大家介绍SubGame Gamma（Subgame先行网），Crab Network（达尔文先行网），MXC（DataHighway 项目）。

Subgame Gamma（Subgame）

SubGame基于Polkadot平行链研发，通过完整的技术架构搭建出多重应用，其中包含游戏类型模组、支付模型接口、通讯领域发展、数据上链应用、跨链结算与区块链存储空间等，并以中继站合约协议为外挂模组，可以开发出跨公链智能合约，最小化链与链之间的价值传递成本、最大化应用程序的可扩展性，进而开发出更多有利于生态正向发展的应用，完成对产业的变革。

  • 接入生态后的贡献：

未来提供搭建跨链Bridge提供用户进行GSGB转换至SGB，也让用户透过Bridge参与各类生态。

  • 已有商用成就：

  1. 获得Web 3基金会Grant支持

  2. 2021年3月SGB代币上线7家交易所，同时获得全球30多家机构VC投资。

  3. 2021年6月超越众多波卡生态项目开发进度，据官方统计显示，SubGame开发进度排名全生态第22名。

  • 已有技术成就:  

  1. 2021年4月-完成首期5个游戏结算模组与游戏大厅上线。

  2. 2021年5月-在Heco 火币生态链上支持SGB代币流通。

  3. 2021年5月-SubGame主网上线，并与全球最大去中心化钱包TokenPocket战略合作，成为继PolkaDot与Kusama后第一个上线TP钱包的波卡平行链。

  • 后期发展规划：  

  1. 游戏结算模组与游戏大厅上线

  2. 多元可插拔式的开发模组

  3. 多元支付模型引擎

  4. 建立双向且多链跨链服务

  5. 可订制可身份确认的NFT、支持大型游戏IP的NFT模组、NFT拍卖平台

Crab（达尔文）

达尔文第一批接入波卡平行链测试网，是生态中的第一个游戏链，也是一个开放的异构链跨链协议，其角色即成立在波卡生态和外部链之间的一座桥，它主要专注于游戏和应用方向的跨链和应用链业务，达尔文网络的

通过达尔文网络，区块链游戏或者DApp可以更便捷地进行游戏资产和游戏操作的跨链交互，为单链应用升级到跨链版本提供关键的基础设施（应用包括但不局限于 Defi，游戏，NFT 交易市场等），比如可以通过达尔文网络把以太坊上的加密猫转变成 EOS 上的加密猫或者以太坊上的玩家和 EOS上的玩家可以同时玩“进化星球”游戏。

  • 接入生态后的贡献：

  1. Crab是Kusama桥中心，提供Sustrate to Sustrate桥，和波卡生态外的桥接等；

  2. 达尔文虚拟机，兼容EVM已经部署在Crab上，提供智能合约解决方案，帮助以太坊项目轻松迁移到波卡；

  3. 跨链游戏进化星球建立Crab大陆，升级Defi+NFT玩法，Crab作为跨链NFT链，为进化星球打造Metaverse提供基础设施；

  4. 超低Gas fee，提供升级用户体验。

  • 已有商用成就：

  1. 被写入波卡轻白皮书；

  2. 被选中加入 Substrate Builder Program 和 Web3.0 训练营；

  3. 达尔文网络开发的工具和产品共获得了 3 个 W3F Grant。

  • 已有技术成就:  

  1. ParityLevel 2 高级勋章；

  2. 达尔文-以太坊双向桥已经成功开通，并支持NFT的跨链；

  3. 达尔文虚拟机，兼容以太坊虚拟机已经部署在Crab网络上，提供智能合约解决方案；

  4. 成功把以太坊Ropsten区块头relay到了波卡Rococo，标志着波卡以太坊通过达尔文实现连接的第一步；

  5. 通过达尔文完成了波卡-以太坊的首笔跨链转账；

  6. 跨链游戏进化星球已经开通以太坊，波场，Crab，Heco四块大陆。

  • 后期发展规划：  

打造区块链互联网，成为波卡生态的桥中心，连接波卡和更多公链生态，下一座桥的连接是BSC，打通多链资产。同时继续打造进化星球成为跨链Metaverse,波卡生态NFT的流量入口，Wwb3.0世界的实验场。进一步发展defi跨链用例，比如升级Uniswap可以支持原生波卡资产，NFT用例，支持波卡生态NFT项目进入以太坊获得更多流量，游戏用例，整合多链游戏NFT进入进化星球。

MXC（DataHighway）

MXC极域是在全球百分之80国家铺设的分布式物联网络。以LPWAN为技术基础全力打造广覆盖、低功耗、人人受益的智慧城市。MXC极域是第一个加入DataHighway的项目，且与DataHighway联合参与卡槽竞拍。

数据高速公路（DataHighway）是全球第一个基于Polkadot生态系统的跨链数据交易市场，其创建目的是充当MXC，IPFS，IOTA和HT等区块链项目之间的桥梁。MXC极域联合DataHightway的社群成员将会最终被激励来运营基于Polkadot 波卡的复杂物联网平行链。MXC极域作为第一个物联网Layer2扩容方案将会接入DHX，从以太坊扩容转移到波卡平行链。

  • 接入生态后的贡献：

DHX在物联网生态中扮演的是沟通不同的区块链、将数据进行跨链输送的角色，也就是数据交易介质，理论上所有项目都可以通过数据高速公路进行跨链数据传输，包括但不限于波卡生态项目、以太坊项目等等；DataHighway跨链传输的数据量越大，价值越高。

通过全球的、开放免费的智慧城市网络的部署，数据可通过DataHighway运输到不同的区块链和云端平台，DHX将成为数据传输的介质。

  • 已有商用成就：

  1. MXC极域作为唯一的物联网区块链项目加入阿里巴巴ICA联盟。

  2. 铺设了覆盖 160 多个国家的去中心化无线网络（LPWAN），是全球最大的开放免费智慧城市网络；

  3. MXC极域将LPWAN智慧城市技术带到上海市杨浦区，得到了欧盟地平线计划资助以及杭州市西湖区的注资；

  4.  与德国铁路、博世、德国柏林科技局、美国卡内基梅隆大学、加拿大 Canadian Tires、韩国 Enlink、韩国国家电网、纽约 Citiesense、日本 M2B、江苏爱康科技、美国Semtech等公共机构和企业合作，成功实现项目落地；

  5. 帮助采集人员流动、停车位、垃圾桶等智慧城市建设运营所需的物联网数据，并取得业务收入，实现了盈利。在这些落地项目的基础上，MXC 极域初步搭建了物联网数据跨链市场。

  • 已有技术成就:  

  1. layer-2物联网扩容方案正式投入商用,全球接入7个超级节点。

  2. 改进LoraWAN协议，利用区块链公私钥入网认证设备

  3. 接入ipfs等数据接口，将物联网数据存入去中心化存储

  • 后期发展规划：

未来团队希望建立平行链所组成的去中心化数据交易的市场。通过这样的市场把数据搬运到它应该去的地方，而不是留在大家的个人终端里，通过区块链，让数据发挥出更大的价值。

未来在波卡上面，希望和做存储、做Defi、做保险的公司合作，让这些项目和真实世界的数据联动起来。借助波卡挖掘物联网真实数据的价值。
