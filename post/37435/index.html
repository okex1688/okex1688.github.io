随着Covid-19大流行加速了经济的数字化转型，服务提供商争相推出相关技术解决方案，以提供新的客户体验并开辟创新的在线销售和营销渠道。许多公司已经进入了虚拟世界，在这个新的生态系统中进行营销，他们可以在其中与客户互动，创造新的体验和商机。

在香港贸发局研究部近期举办的题为“探索元宇宙：超越 NFT 炒作”的网络研讨会上，行业专家讨论了元宇宙的出现如何帮助传统供应链克服现实世界的限制，以及中小企业如何进行营销并在元宇宙中推广。网络研讨会由香港贸发局研究部经济学家（全球研究）Simeon Woo 主持。

身临其境的体验

香港贸发局研究部首席经济学家（全球研究）Louis Chan首先介绍了元宇宙概念和相关的商机。Chan 将元宇宙描述为一个新的生态系统，为用户和创作者带来新的体验（尤其是身临其境的 3D 互联网体验），接着列出了元宇宙的五个基本组成部分。

首先是内容。因为元宇宙是一个去中心化的生态系统，它的大部分新内容都是由用户创建的。第二个是硬件和设备，例如 VR（虚拟现实）和 AR（增强现实）眼镜，所有这些都需要强大的半导体产业的支持。第三是基础设施，包括互联网、服务器、基于云的系统以及 5G 和 6G 无线网络，这些都有助于增强互联网体验和元宇宙生态系统的发展。四是社区——跨平台、网络和应用程序的虚拟协作，可以产生新的社区和新的交流方式。五是货币或结算。新的货币系统，包括比特币等虚拟或加密货币可以为元宇宙带来新的生态系统和环境。

不可替代代币 (NFT) 是访问元宇宙的工具。它们赋予经典电影、经典漫画、艺术品、数字产品等产品新的价值。因此，元宇宙的发展也涉及新的市场。Chan 举的一个例子是一个价值数百万美元的 NFT 头像（头像），将不再通过在线购物平台或普通交易场所出售。相反，它将在 NFT 市场之一上出售，其中最大的是OpenSea。

在虚拟世界中，可以创建化身以用于产品照片或在线电影的制作。这些化身不仅限于在游戏中使用。它们还可以用于其他领域，例如娱乐、体育、在线购物、时尚和社交媒体；或担任品牌代言人和关键意见领袖 (KOL)。在社交媒体中，头像也可以有追随者。一些消费者还喜欢在网上购物时拥有自己的头像，以创造新的购物体验。

然而，Chan 警告观众，参与虚拟世界会带来一定的风险。根据香港电脑紧急应变小组协调中心（HKCERT）的数据，当有人沉浸在虚拟世界中时，他们很可能会遇到越来越多的网络攻击威胁，包括钓鱼、僵尸病毒和垃圾软件。因此，进入元宇宙的企业应考虑购买虚拟保险以增加保护。

从图片到 NFT

Vplus Studio Corporation Limited ( Vplus )的联合创始人兼业务总监 Roy Kwok解释了元宇宙与 NFT 之间的关系。他将元宇宙描述为真实空间之外的第二个空间。当人们戴上 VR/AR 眼镜时，他们进入了另一个世界，在这个世界中，他们可以与里面的社区互动和分享，甚至可以购物。元宇宙提供了不同的环境和设置，例如游戏场景、动物空间等。

NFT 是公众可以访问元宇宙的最简单的媒介。它可以从一张图片开始，例如球赛俱乐部标志、俱乐部会员卡或艺术家的画作。NFT 图片的交易必须使用区块链进行，因为该技术可以提供 NFT 所有权的可验证证明。实际上，图片只是一种呈现方式，而 NFT 代表的是图片的价值。目前，Solana和以太坊是两个最受欢迎的区块链平台。NFT 图片还赋予了购买者一个社区团体的象征，该团体拥有不同的授权和权利。因此，图片不仅仅是一个化身，也是买家对社区的依恋的象征。作为社区的一员，买家可以享受社区提供的所有现有和未来的特权。买家还可以与社区的其他成员互动，甚至参与决策。

Bored Apes：交易量最大的 NFT 收藏品之一。

Kwok 解释说，NFT 是以不同的方式呈现现实世界的东西。例如，各种会所的会员卡可以在区块链世界中进行交易，无需任何文书工作，因为有与各种规则和法规绑定的智能合约。因为 NFT 也是极好的推广渠道，艺术家可以通过 NFT 创造性地推广自己的作品。以MERGE by Pak 系列为例。这件艺术品中球体的大小根据公众参与的数量和相关拍卖的热情而有所不同。因此，该系列的每一件作品都是独一无二的。

虚实结合

Vplus联合创始人兼执行董事 Tony Lau解释了公司如何使用元宇宙和 NFT 进行营销和推广。他说 NFT 是元宇宙的组成部分之一。通过区块链技术，可以定义 NFT 的所有权并进行交易。物联网 (IoT) 和 6G 的广泛使用推动了虚拟世界中完美画面的创造，让来自世界各地的玩家进入同一个空间进行互动、接受挑战或交换道具，信息和图像。虚拟世界的推广也需要人工智能技术的支持，例如通过机器学习和受众定位与公众互动。

刘指出，元宇宙所涉及的交换技术可以帮助增加公众参与和增强虚拟体验。一个运动品牌最近推出的虚拟运动鞋就是一个很好的例子。购买了虚拟鞋的顾客可以根据自己的步行速度来判断哪种运动鞋最合适。此外，客户根据步行速度积累虚拟币，不仅可以在虚拟世界中使用，还可以在交易平台上兑换成现金。品牌及其客户可以通过运动获得收入，也可以通过参与游戏或商业活动获得收入。这种模式催生了全新的市场。

随着元宇宙的扩大，一些品牌开始举办新型元宇宙推广活动。例如，今年年初，移动巨头三星在元宇宙世界购买了一块土地，并要求一位游戏设计师按照其在美国的一家实体旗舰店的形象，在那里创建一个虚拟旗舰店。随后召开了虚拟发布会，新品发布，全球记者、粉丝、游戏玩家实时参与。产品发布会结束后，与会人员进行了虚拟任务，同时获得了可以在二手市场出售的虚拟服装和头盔。

一些品牌举办了虚拟活动，将参与者转移到互联网界面浏览其官方网站。这些参与者随后被邀请购买 NFT 并参与未来的体育活动。通过这种方式，他们建立了虚拟社区，然后可以将其作为进一步营销和推广的目标。
