NFT热点

1.OpenSea网络钓鱼攻击已造成640 ETH（约170万美元）资产损失

金色财经报道，2 月 21 日，OpenSea 首席执行官 Devin Finzer 对涉嫌网络钓鱼诈骗作出回应，本次网络钓鱼攻击波及32名用户的640 ETH（约合 170 万美元）资产，相关 NFT 已经在被标记为“Fake_Phishing5169”的钱包之中。据悉，黑客已经在LooksRare 和 Rarible NFT 市场上进行了交易，此前有消息称黑客退还了部分 NFT。

2.俄罗斯最大社交媒体VKontakte将为用户内容变现提供NFT支持

2月21日消息，俄罗斯最大的社交媒体VKontakte周日宣布，他们将为用户内容变现和版权保护提供NFT支持。 VKontakte的技术总监Alexander Tobol发表了一份声明。他宣布，VKontakte计划充当中介和助手，但不会创建自己的区块链，而是将使用其Content ID系统来验证上传的音乐和视频的作者身份。

3.NFT数据聚合平台NFTGo与RSS3达成合作伙伴关系

2月21日消息，据官方消息，NFT 数据聚合平台 NFTGo.io 与 Web3 信息分发基础设施 RSS3 达成合作伙伴关系，本次合作将为 NFTGo 和 RSS3 带来更多的可能性和应用场景，为其他应用程序或平台带来更多功能，NFTGo 也将在 RSS3 的支持下为用户提供更好的体验。

4.Aloha NFT正式与Polygon合作开发夏威夷首个 NFT

金色财经报道，在 Polygon 区块链的资助下，Aloha NFT 将于2022年2月22日启动他们的“Alohachella”项目，其中包括 22 位夏威夷艺术家的 NFT 艺术品和实物收藏卡。Aloha NFT是夏威夷的第一个 NFT 市场，于 2021 年 4 月推出。

5.巴西球星罗纳尔迪尼奥宣布将发行NFT

2月21日消息，著名巴西足球明星，两次世界足球先生得主罗纳尔迪尼奥宣布将发行其首个官方 NFT 系列。该系列与 Shirtum 合作开发，将发行在以太坊网络上。据 Shirtum 透露，持有 Shirtum Token SHI 将是确保获取铸造该 NFT 名额的唯一途径。

6.Azuki系列NFT24小时交易额为290.8万美元，增幅达70.86%

2月21日消息，据 NFTGo.io 数据显示，Azuki 系列 NFT 总交易额达 3.06 亿美元，在所有 NFT 项目总交易额排名中位列第 8；其 24 小时交易额为 290.8 万美元，增幅达 70.86%。截止发稿时，该系列 NFT 当前地板价为 13.64 ETH。

7.BOOMGALA上线欧易NFT平台发售

2月21日，NFT项目BOOMGALA与欧易（OKX）宣布达成官方合作；欧易NFT平台MetaX将会上架BOOMGALA的100个NFT，并于22日晚间发售，发售价格为0.2ETH。 据悉，BOOMGALA NFT项目将于北京时间2月22日凌晨1点开始白名单预售，预售价格为0.2ETH。

DeFi热点

1.派盾：WOLFI 为“貔貅盘”骗局，与此前MarkMeta Network骗局有直接关联

2月21日，PeckShield 检测到 WOLFI 存在欺诈，为“貔貅盘”骗局。未获取白名单用户仅能购买 WOLFI 而无法出售。目前，由于大量 Token 无法出售，WOLFI 短时上涨已达 5000%。此外，通过查询合约显示，WOLFI 与此前同为「貔貅盘」骗局的 MarkMeta Network 有直接关联。 此前报道，2 月 16 日，PeckShield 检测到 BNB Chain 链上项目 MarkMeta Network 存在欺诈，为「貔貅盘」骗局。未获取白名单用户仅能购买 MMT 而无法出售。

2.Helium将升级基础设施，提升网络性能

2月21日消息，去中心化无线网络 Helium 宣布将迎来重大基础设施升级，Light Hotspots。升级完成后，Helium 网络将更快地扩展覆盖范围，无需同步，并显着提高网络上运行的应用程序和设备的可靠性。

3.派盾：监测到Raptor2发生Rug Pull，损失855枚BNB

2月21日消息，派盾（PeckShield）监测到Raptor2发生Rug Pull，地址0xdaa5e5692f24b0284cbd9fb6fbe2ddc78bf4d34a已经抛售Raptor2并将855枚BNB存入以太坊隐私交易平台Tornado.cash。

游戏热点

1.FTX将成立游戏部门，旨在促进更多游戏工作室采用加密货币和NFT

2月21日消息，据彭博社报道，FTX 正在成立自己的游戏部门FTX Gaming，旨在促进更多游戏工作室采用加密货币和 NFT。FTX Gaming 将推出加密即服务平台，游戏公司可以通过该平台发行 Token 并为 NFT 提供支持。据悉，该部门于本月开始招聘一个完全远程的团队，寻找具有游戏引擎 Unity 编码经验的软件工程师。

2.Immutable X与游戏平台Kongregate合作，将共同开发Web3游戏

2月21日消息，据 Play to Earn Online Magazine 报道，NFT Layer2 解决方案 Immutable X 与 Flash 休闲游戏平台 Kongregate 合作，双方将共同开发 Web3 游戏，并将于今年晚些时候发布两款基于区块链技术的 NFT 游戏。 Kongregate 曾是 GameStop 旗下的游戏平台，2017 年被 Modern Times Group 收购。此前报道，2 月 3 日，GameStop 宣布与 Immutable X 达成合作，二者将共同开发 NFT 交易平台，并推出 1 亿美元游戏基金，用于支持其生态系统中的 Web 3 游戏开发。

3.NFT游戏公会UniX Gaming成立工作室并开发首款P2E游戏

2月21日消息，NFT游戏公会UniX Gaming已经开始扩展其生态系统，推出了其官方的视频游戏开发工作室。新推出的UniX Gaming Studios已经致力于推出公会的首款P2E视频游戏，这是第一款使用Unreal Engine平台开发的同类游戏之一。

4.Axie Infinity与Bit2Me签署合作协议，以促进西班牙、葡萄牙和拉美业务增长

2月21日消息，Axie Infinity与西班牙最大的加密货币金融技术服务平台之一Bit2Me签署合作协议，以发展其在葡萄牙、西班牙和拉丁美洲的业务。 通过此次战略合作，Bit2Me将成为AXS在南欧和拉丁美洲的交易所伙伴之一。Axie Infinity的玩家将获得Bit2Me平台提供的所有优势，例如本地代理提供西班牙语和葡萄牙语的电话支持等。

5.Gala Games 将推出自主研发的“P2E游戏专用”主网 GYRI

2月21日消息，全球链游工作室Gala Games 近日宣布了“Project GYRI”的发布计划及相关内容，表示目前正在研发在以太坊主网之外的“P2E游戏专用”的区块链主网 GYRI。Gala Games 表示，“GYRI是第一次为专为链游玩家设计”，并“提供高质量的游戏优先体验 ”。 Gala Games 成立于 2018 年，旨在成为一个去中心化的游戏生态系统，做链游领域的Steam，向用户提供他们真正想玩的区块链游戏，继而将传统的游戏用户带入链游、元宇宙的世界。Gala Games 代总市值在 P2E 游戏公司中排名第四，仅次于 Decentraland、The Sandbox 和 Axie Infinity。

DAO热点

1.资产配置治理组织PieDAO发布提案，将成立NFT封闭式投资基金

2月21日消息，资产配置治理组织 PieDAO 发布提案 PIP-72，提案目的为推动成立 NFT 专项封闭式投资基金。基金运作方式为：DAO 组织初始融资轮融资，投资者在特定时期内将 ETH 存入智能合约，并按其贡献比例获得代表基金份额的 Token。融资成功后，基金管理者根据 PieDAO 事先定义的一套执行标准去购买 NFT。在到达预定时间后，该基金卖出之前购买的 NFT，卖出收益将作为基金收益，供基金份额 Token 持有者分享。目前该提案正在社区讨论。

2.BuyTheBroncos DAO寻求募资超40亿美元收购NFL丹佛野马队

2月20日消息，美国国家橄榄球联盟丹佛野马队（Denver Broncos）正在出售。一个DAO正在寻求筹集超过40亿美元来收购该俱乐部，让球迷有机会部分拥有他们最喜欢的球队。 打算收购丹佛野马队的BuyTheBroncos DAO包括律师、会计师、软件开发人员、职业运动员和一名数学家。曾供职于思科法律部门的律师Sean O’Brien正在领导该计划。 ESPN表示，丹佛野马队预计将获得一笔相当可观的收购金额，或许是美国体育史上最大的一笔收购。球队的受托人表示，他们希望俱乐部在2022年的NFL赛季被出售。 O’Brien表示，BuyTheBroncos DAO的智能合约和加密钱包已经配置好，随时可以使用。DAO要到3月初才会上线。
