互联网建设者为何应该奔赴 Web3？

Web3 模式提供了独特优势，更利于产品启动。

互联网正在发生重大变化。 虽然占主导地位的大型平台凭借掌控用户数据和__网络效应__几乎立于不败之地，但是 Web3 模式正在建立新的价值主张。在 Web3 中，用户拥有他们创建的任何内容（例如发帖或视频）以及他们购买的数字产品，不再是平台控制底层数据，用户拥有且可以转移这些数字资产。 这种新模式有几个重要的优势：

  • 新公司更容易通过提供良好的用户体验与老牌公司竞争；

  • 系统不再是零和，留住用户不是主要目标；

  • 平台更容易获得用户信任，关键操作可以通过不可篡改的「智能合约」在区块链上进行编码。

这些都意味着，在 Web3 中启动一个产品会更加容易。

当今占主导地位的互联网平台是建立在聚合用户和用户数据的基础上的。随着这些平台发展，它们提供价值的能力也在增强，但这要归功于网络效应的力量从而使它们能够保持领先地位。例如，Facebook（现在的 Meta）关于用户行为的数据帮助其优化算法，使其内容和广告服务远远领先于竞争对手。与此同时，亚马逊利用其对客户需求的大数据来优化配送物流并开发自己的产品线。YouTube 已经建立了来自众多创作者的庞大视频库，使其能够向观众提供几乎任何主题的内容。

在这些商业模式中，锁定用户及其数据是竞争优势的关键来源。因此，传统的互联网平台通常不会共享数据，用户也无法使用这些数据创造价值。 因此，即使用户对特定平台不满意，通常也无法离开。

但这一切可能正在改变。虽然新来者很难以自己的方式挑战 Meta 这样的「Web 2.0」公司，但在「Web3」模式下工作，这个问题有望得到解决。尽管围绕元宇宙和各种超金融化 NFT 项目有着很多争议，但最重要的是，一些开发者已经达成了一种共识：它基于这样一个前提，即除了利用用户获取数据来赚钱之外，还有另一种选择，建立与用户直接共享价值的开放平台，从而为包括平台在内的每个人创造更多价值。

在 Web3 中，用户通常拥有他们创建的任何内容（例如文章或视频）以及他们购买的数字产品，而不是平台完全控制底层数据。此外，这些数字资产通常根据公共区块链上的可互操作标准创建，而不是托管在公司的服务器上。这使得资产「便携」，原则上，用户可以随时离开任何给定的平台，只要卸载该应用程序，并将他们的数据迁移到另一个平台。

这可能从根本上改变数字公司的运营方式：用户将数据从一个平台迁移到另一个平台的能力产生了新的竞争压力，迫使公司重新制定其业务战略。 如果一个平台没有为它的用户创造足够的价值，用户可以轻易离开。事实上，在 Web3 中，新平台会有更多的激励方式来吸引用户。例如，NFT 交易平台 LooksRare 最近通过推出「吸血鬼攻击」活动，激励用户离开 OpenSea 到 LooksRare 上交易。

与此同时，Web3 不再是一场零和的存量争夺游戏，平台的整体价值创造机会更大。建立在可互操作的基础设施层上，平台可以轻松插入更广泛的内容网络，从而扩大它们可以为用户提供的价值规模和类型。例如，Web3 艺术画廊可以引导用户在区块链上创建艺术品，而不是要求他们直接将艺术品上传到平台。

对于一些 Web2 平台，这也是一个获取内容的好方法。 Twitter 最近推出了一项功能，用户可以在其个人资料中展示他们拥有的 NFT； Instagram 也在开发类似的功能。于新平台而言，由于缺乏初始内容，平台很难在早期获得用户，整合已有数字资产数据可能是顺利实现「冷启动」的关键一步。

基础设施层意味着与创建用户信任相关的成本在 Web3 中要低得多。在公共分类账上管理数字资产可以更清楚地了解哪些资产存在以及谁拥有什么，这在 Web2 网络上是一个难题。例如，一位数字艺术家声称自己拥有一件新艺术品，那么所有者可以直接在区块链上验证这一点，无需信任艺术家本人，也无需让画廊或其他中介机构提供此类保证。

这种信任框架扩展到 Web3 平台：关键操作以可审计和不可变的「智能合约」在区块链上进行编码。这使得平台设计师可以预先承诺某些功能，例如定价规则、版税协议和用户奖励机制。

所有这一切意味着至少从理论上来说，在 Web3 世界中启动新项目会容易得多。即使是一个不知名的企业家也可以在未经既定平台许可的情况下推出自己的产品。事实上，在 Web3 中，用户有时不需要信任项目背后的公司（或人）。

相反，他们只需要信任代码本身。例如，最近一些支持乌克兰人道主义援助工作的筹款活动是通过智能合约进行的，该合约会自动将收到的所有资金转移给乌克兰政府或相关慈善机构；这意味着即使活动组织者完全匿名，捐助者也可以相信他们的资金会得到妥善使用。

当然，在 Web3 早期，出现了很多金融案例和大量非法交易，许多投机者利用炒作来策划诈骗。 而且许多 Web3 应用都是为精通技术的高级用户设计的，普通用户可能对应用程序或平台的实际功能了解有限，更不用说审查源代码以验证其功能是否符合描述。 Web3 技术要安全并可供普通消费者使用，还有很长的路要走。

在实践中一个新应用并不意味一定会受用户的喜欢。就像在所有创业企业中一样，构建能够满足真正用户需求的产品至关重要。一旦解决了用户需求，通过 Web3 利用已建立的网络可以更轻松地进行部署和进一步发展。

使平台后端开放且可互操作，可以促进复合创新，并激励开发者构建基础设施层。例如，koodos（一项 Web3 服务，可以让人们从互联网上收集自己喜欢的东西）正在建设任何网络都可以接入和改进的共享基础设施。 

共享基础架构意味着应用程序可以专注于构建出色的体验，从而更加重视平台设计，这将是重要的竞争优势。应用程序对其市场的了解体现在其用户体验和界面中。因此即使在 Web3 中，用户体验仍然是衡量产品好坏的重要标准。

Web3 平台还具有通过社区参与和社会凝聚力形成一种特别强大的网络效应。数字资产的所有权培养了一种心理归属感，这种心理归属感形成强烈的用户粘性且激励用户维护平台和自己的数字资产。

例如，流行的街头服饰品牌 The Hundreds 最近发售了以吉祥物「Adam Bombs」为主题的 NFT。持有该系列 NFT 可以享受参与社区活动的权益以及购买限量商品的资格，同时为品牌粉丝提供一种见面和互动的方式，从而增强他们的热情。 The Hundreds 还宣布，将向 Adam Bombs 相关的 NFT 所有者支付版税。这样持有者可以拥有 Ralph Lauren 标志的部分所有权，并且每件使用该标志的新 Polo 衫系列都会带来一定的分红。这种方式将品牌价值部分去中心化，导致 The Hundreds 的社区对 IP 更加依恋，并不遗余力地推广它，甚至一些社区成员把 Adam Bomb 纹在了身上。

另一个例子是 SushiSwap，它是去中心化金融平台 Uniswap 的「分支」—— 这意味着 SushiSwap 的底层算法是 Uniswap 代码的克隆。二者的主要区别在于 SushiSwap 建立了强大的品牌和社区，同时为用户提供了一个积极和持续的奖励系统，从而推动了更高的用户参与度；这使其迅速成为 Uniswap 的有力竞争对手。

一般地说，共享所有权可以激励每个人成为建设者和贡献者。底层技术标准可以使每个 Web3 公司都能够建立在其之上。这意味着围绕一个平台的社区可以以一种更加和谐的方式共同创造，使平台生态系统变得更加强大。

在短期内，这种模式将部分消费者剩余让给了建造者或创造者。但是因为建设者得到了更多，建设者会更加努力建设平台从而使蛋糕变得更大，反过来增加消费者剩余。

总而言之：Web3 可以为每个人创造更有价值的互联网。 与 Web2 相比，新公司可以更加容易地在 Web3 基础架构上构建围绕其品牌和产品概念的社区。甚至已建立的平台也可以通过增加基于区块链的内容网络并赋予其用户数据所有权。所有这一切意味着网络的下一个时代将会与我们今天生活的时代大不相同，而且更加开放。
