距离10月24日还剩不到11天，文本NFT 应用TronLoot的盲盒兑换活动正在进入倒计时，从9月24日开兑当日到现在，面向市场的8000个TronLoot盲盒还剩5787个。元宇宙「分子级」NFT应用Loot从以太坊区块链上诞生，其铸造热潮在波场链上得到了延续和发展。

以太坊（ETH）链上的产品性极强的NFT应用Loot带来了NFT赛道的进化，以文本框形式出现的非同质化代币Loot，给原创型的NFT带来了可编程性。

全球三大公链之一的波场TRON上的生态开发者快速抓住机会。这一次，TronLoot改变了分发方式，以市面上流行的「开盲盒」玩法打入市场。利用APENFT的治理通证NFT，玩家可以直接在波场链上开启盲盒，铸造TronLoot，随机得到不同的文本框。玩家既可以在TronLoot文本NFT上进行再开发，也可以直接在官方自带的交易市场中将铸造出的TronLoot卖出。

再开发、再创新是Loot概念项目的价值所在，相较当前的以太坊网络效率，波场TRON网络的低成本显然对开发者来说更为友好，这是TronLoot的优势所在。但相较以太坊Loot，TronLoot在生态上还是白纸一张。一个隐形「彩蛋」是该项目背后的合作伙伴APENFT基金会，其已经积累了多个超级NFT作品，都将为TronLoot生态带来铺垫。TronLoot的出现也昭示着波场TRON布局元宇宙的野心。

改造分发方式 TronLoot盲盒吸引玩家

NFT爱好者苏菲（化名）错过了8月底的ETH-Loot铸造，「得到消息的时候就已经抢完了，最后听说用于合成Loot的Gas费很高，有人甚至花了1ETH去拥堵的网络上『拿货』。」高费用消弭了苏菲没抢到Loot的遗憾。

9月27日，在TronLoot上线的第三天，苏菲抱着收藏的心态打开了TronLoot兑换页面，总量为8000个的文本NFT正在上面发售，已经有2000多个TronLoot已完成铸造，「和ETH-Loot不同，官方团队没有为自己预留，而是将NFT全部对外开放。 」

两个项目除了部署网络不一样之外，分发方式也有差异，「ETH-Loot允许用户直接在以太坊网络上通过钱包随机合成装备袋，成本是网络Gas费；而TronLoot冒险装备集合在Market页面中，以兑换盲盒、随机铸造的方式打开装备。 」

在TronLoot官页，链接钱包可开启文本NFT盲盒

收集TronLoot文本框需要APENFT基金会的治理通证NFT，1个TronLoot需要的地板价为2000万NFT通证，该通证可以在二级市场购买。将NFT通证转入波场支持的钱包后，即可连接TronLoot官网，开启盲盒。

苏菲体验后总结道，相比ETH-Loot的链上铸造、随机分发，TronLoot的开盲盒方式多了一些产品运营思维，「对于普通玩家来说，比较有意思，有期待性。」

用户的确Get到了项目的用意。TronLoot开发者告诉蜂巢财经，采用开盲盒的方式分发TronLoot主要是考虑该NFT的趣味性和可玩性，「这是线上、线下游戏市场中非常流行的游戏方式，较容易被年轻人接受，也符合TronLoot根据随机哈希值产生的方式。」

盲盒开出后，玩家可以直接放在TronLoot内置的交易市场去售卖， 「这也比ETH-Loot更直接，因为ETH-Loot的二级市场是第三方平台OpenSea，玩家交易Loot又多了一重流程，又要花费手续费，TronLoot的交易就比较便捷。」

TronLoot市场数据显示，截至10月8日，TronLoot已经售出2213个，其中9月27日，TronLoot的二级市场最高成交价达2.2亿NFT，时值572美元，较一级市场地板价的溢价率达1100%，未成交挂单的价格较地板价也有几十倍到上百倍的溢价率。

截至10月7日，挂单价格最高的TronLoot是3号文本框，挂单价为90亿NFT，价值31230美元，这个已经开出的盲盒内含的装备有巨人弯刀、巨人皮甲、大头盔、华丽的完美腰带、神圣拖鞋、手套、恶魔怒吼的脖子和力量银戒指，各装备的稀有度较高，这也是3号TronLoot挂单价较高的原因。

TRON链为TronLoot开发提供低成本环境

对于苏菲来说，玩法之外更重要的是「它便宜」。

苏菲开TronLoot盲盒时，用于兑换盲盒的通证NFT时值0.0000026美元，开启一个盲盒需要2000万NFT，这样算下来，一个TronLoot盲盒时值52美元；而她花费的Gas费约为40TRX，当时TRX的报价为0.08美元，即她开盲盒的Gas费为3.2美元左右。兑换一个TronLoot盲盒，苏菲总共花费了55.5美元的成本。

 兑换一个TronLoot装备袋盲盒需要2000万NFT

尽管TronLoot开盲盒另需付费，但总体成本远低于ETH-Loot。9月27日，一个ETH-Loot的二级市场价格已经在14 ETH，折合47180美元。苏菲听说，当时ETH-Loot正式发布时，有人花费了1个ETH的Gas费去合成，当时ETH价值3000多美元。「因为Loot抢购已经造成了网络拥堵，Gas费也就水涨船高了。」

除了费用低廉外，苏菲还发现，TronLoot并不给用户的地址设置开盲盒的数量限制，也就是说，一个用户地址可以开启多个盲盒。

TronLoot的开发者表示，考虑到一些生态玩家或者再创新的团队可能需要大量Loot，因此在地址上不做上限设置。

无论是ETH-Loot还是TronLoot，都采用了文本框式的可编程形态，这就让Loot充满了想象力，它以装备包、武器袋为包装，提供了创新的基本元素，头、颈、胸、四肢等等元素构成了可组合性，让NFT不再是自上而下、用户被动接受的收藏品、艺术品，而是支持装备包持有者再创造的产品级应用。

相对单一NFT藏品，Loot的可组合、可编程性决定了它的复杂性，只有利用好Loot的属性才能让这一文本型NFT发挥更大的价值。正如ETH-Loot的创始人Dom Hoffman所表达的那样，「Loot的价值取决于持有它的人如何对它进行解释。」

编程的复杂性决定了Loot类NFT需要一个低成本、高效的区块网络支撑。相较ETH-Loot，TronLoot 的链上环境对开发者更为友好。

对于「每操作一步都要花费Gas费用」的区块链网络来说，以太坊上每笔交易需要支付的Gas费平均为0.005-0.008ETH左右，这是网络通畅的时候，以ETH当前3500美元价格计算，每笔交易的美元价格在17.5-28美元左右；而波场TRON链上每笔交易Gas费大概在3-5 TRX，按照TRX0.09美元的价值计算，每笔交易的美元费用在0.27-0.45美元左右。

对于凝聚了开发复杂性的Loot产品，开发成本花费高下立判。

TronLoot还有哪些生态想象力？

尽管具有开发者环境优势的TronLoot目前仍处于分发阶段，生态的想象力仍在酝酿中。ETH-Loot已经产生了一些工具和社区生态。

当前，ETH-Loot已经形成了自身的社区，有自己的社区通证AGLD，衍生出了其他项目。社区成员们在Discord社交群组内探讨创新方向；它还有15个开发者工具，80多个衍生项目；8个公会，公会多以装备形态命名，比如神袍（Divine Roles）公会、皇冠（CrownDAO）公会等等；社区还开发出了10个追踪工具，以供用户查询流通中的Loot稀缺度。

有ETH-Loot在前，理论上，TronLoot早期也可以构建类似的衍生生态，该项目开发者介绍，团队和APENFT基金会已经在辅助玩家开发相关的工具及场景铺设，构建在波场链上的TronLoot是一个更加高效、便捷的试验场。

ETH-Loot的创始人Dom Hofmann曾在Discord里提示社区可能针对Loot建构的方向，包括和艺术家合作，构建艺术NFT；以Loot为基础，制作图片项目；解读Loot中装备的价值，提供装备升级方案；把整个装备包切分为独立装备，围绕新的主题改造原来的Loot；开发游戏等等。

这些方向正是TronLoot推广方APENFT基金会已经涉猎的方向。

APENFT基金会成立于2021年3月份，它基于以太坊和波场TRON为底层技术支持，结合分布式存储系统BTFS搭建了NFT运营平台。根据官网介绍，APENFT的主要业务包括投资和布局头部NFT平台与作品，致力于将全球顶级艺术品NFT化与区块链化，为传统艺术家进军NFT提供桥梁。

APENFT基金会致力于将全球顶级艺术品NFT化

APENFT基金会的首批NFT化艺术品主要来自波场TRON创始人孙宇晨的捐赠，包括世界知名传统艺术家毕加索的《戴项链的躺卧裸女》、加密艺术家Beeple的《OCEAN FRONT》系列作品等等。在以全球知名公链以太坊Ethereum及波场TRON底层技术的支持下，结合全球最大的分布式数据存储系统BitTorrent File System (BTFS)，APENFT基金会还率先将毕加索的《戴项链的躺卧裸女》和安迪·沃霍尔的《三幅自画像》进行了NFT化，将传统艺术家引入区块链世界。目前，APENFT基金会的藏品价值已经超过3亿元人民币。

从艺术品的NFT，到头像NFT，APENFT基金会几乎拥有了当前市面上最热门的NFT内容，它们也都被视作元宇宙在链上构建的基本元素。

使用包括互联网和 VR 在内的一系列技术而建立的虚拟世界被称为元宇宙，这个概念诞生在上世纪70年代，但迟迟没有落地。随着VR、AR、5G、区块链等技术的发展，元宇宙正在成为可能。

在元宇宙生态系统中，AI、5G、计算机硬件、软件系统都是必要的外在装备，而区块链被视作构建元宇宙生态的经济系统，有顶层设计之意，为元宇宙世界提供了一个理想的去中心化的环境和经济系统环境，而NFT 为元宇宙内产生的数字资产确权提供了可行路径，致力于将全球顶级艺术品NFT化与区块链化的APENFT基金会正沿着这条道路做尝试。

根据 Coingecko 的数据，NFT 市场总市值已经达到 229.7 亿美元，占全球加密货币市值的 1.2%，24 小时交易量 32.5 亿，仍在保持强劲的增长势头。

在这样的市场环境下，TronLoot再次为NFT的应用实验添加了一个场所，加上拥有的内容实力的APENFT基金会的加持，TronLoot为波场的元宇宙布局揭开了大幕。
