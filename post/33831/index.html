印度，NFT 最大的潜在市场之一，正面临着初期的困难。在过去八个月左右的时间里，印度的 NFT 实验发展迅速。 2021 年 6 月，印度最大的加密货币交易所 WazirX 推出了一个主要面向印度艺术家的 NFT 市场。在那之后，许多宝莱坞名人和体育明星都推出了NFT。迄今为止，全球最大的 NFT 销售，6900 万美元的 Beeple 作品拍卖，也是由一位印度裔买家Vignesh Sundaresan(又称MetaKovan)购买的。但奇怪的是，印度民众还没有开始购买 NFT。

供需缺口

印度 NFT 市场存在相当大的供需缺口。这里的 NFT 创作者正在激增，而收藏家仍然不存在。对于NFT在印度的机遇，双方描绘了一幅不同的图景。

从创作者的角度来看，机会比比皆是。印度的艺术和数字艺术人才市场充满活力——从平面设计师和插画家，到为好莱坞和宝莱坞工作的专业视觉效果 (VFX) 艺术家，再到涵盖多种风格的丰富传统艺术遗产。年轻而精明的人已经进入了 NFT 领域，而且人数还在增长。WazirX 有一个由 1300 名艺术家组成的活跃的印度创作者社区，还有2万名艺术家等待进入市场。

SamosaRani 是一位成功的创作者，她在 OpenSea 和其他市场拥有超过 18 次 NFT 销售和三个系列作品。作为一名工作20年的艺术家，她在 NFT 领域为她的艺术找到了一个很好的利基市场。然而，她的忠实收藏家都不是印度人，她的一次性买家中只有一小部分来自印度。 她说，“对大多数印度艺术家来说，寻找收藏家是一项艰巨的任务。”

有趣的是，印度 NFT 创作者已经转变为 NFT 收藏家和交易者。 CreatiWitty 是 Kalamint 平台上的佼佼者，他曾经是印度古吉拉特邦一个小镇的自由平面设计师，为公司设计标志和品牌。“以前我设计商标需要三天的时间，客户抱怨并需要不断修改。现在，在几分钟内，我可以转手一个 NFT 并赚取两倍的利润，三倍于我过去三天努力所赚的钱，”他在接受采访时说。

然而，从大规模采用的角度来看，NFT收藏品并不太适合印度的产品市场。少数印度收藏家利用NFT进行投机。这个小型的印度收藏家社区提到了对高流动性NFT的偏好，其中 PFP（个人资料照片）艺术和照片是表现最好的类别。

NFT 平台 Kalamint 的创始人 Sandeep Sangli 说：“从历史上看，印度人并不是收藏家。印度人还没有真正热衷于将艺术品作为一种资产类别。为了让 NFT 在印度发挥作用，它们需要具备实用性。”

Sangli 表示，Kalamint 表现较好的部分之一是品牌和企业家可以交叉推广和销售 NFT，这些 NFT 可以在他们的网站上兑换商品。WazirX 尝试在其市场上销售音乐节的 NFT 门票。这些商品在创纪录的时间内销售一空。

进入宝莱坞

NFT 收藏品没有流行起来并不是因为缺乏尝试。宝莱坞和板球是两个丰富的文化类别，这两个领域的名人都被NFT所吸引是很自然的。在 2021 年 6 月至 2021 年 12 月期间，几位电影明星和板球运动员发布了 NFT，但这些发布得到了相对冷淡的反应。

宝莱坞最大的超级巨星Amitabh Bachhan，被亲切地称为“Big B”，推出了一个 NFT 系列，其中包含几个收藏品，包括诗歌朗诵，甚至还有一个 Big B CryptoPunk。整个系列的售价不到 100 万美元。虽然这还不错，但它无法与我们在世界各地看到的高价 NFT 销售相比，尤其是考虑到 Big B 是印度最大的超级巨星，具有跨代吸引力。单个 CryptoPunks 已经卖出了许多倍于此的价格，从 300 万美元到 1200 万美元不等。与其他名人的 NFT 相比，这笔交易也相形见绌。

名人现在正在从纯粹的 NFT 扩展到元宇宙。最早对此进行深入实验的人之一是宝莱坞明星Kural Kapoor，他现在正在构建 LoveABot/LABverse 项目。他说，“我想做一些事情来改变围绕科技反乌托邦和我们担心的科技驱动未来的叙述，”他的使命是为技术人员建立一个创造性的元宇宙。

另一位演员 Rana Daggubati 正在通过 Ikonz 项目冒险进入一个雄心勃勃的印度元宇宙。Abhinav Kalidindi 认为，“我们需要推动思维方式的转变，但有了它，印度将成为世界上最大的元宇宙和 NFT 市场之一。”

根据其他印度项目的经验，Kalidindi 表示，“为印度人构建具有实用性的 NFT”将是他们成功的关键。

游戏

在印度蓬勃发展的NFT行业背景下，所有利益相关者现在都在将目光投向印度的游戏行业，以寻找与NFT相匹配的终极产品市场。印度的游戏行业在去年呈爆炸式增长，仅在 2020 年至 2021 年期间就实现了 170% 的增长，预计到 2025 年市场将增长两倍，拥有超过4.5亿游戏玩家，业内人士认为这将为 NFT 创造合适的增长市场。

印度创作者和收藏家已经开始研究游戏 NFT。一些受访者已经开始尝试游戏皮肤和相关的 NFT。他们经常预售NFT，作为一种筹款形式，这些NFT将在游戏中使用。Sangli说，“有了足够多的关于这些游戏的炒作，我们看到很多收藏家提前购买并转手这些 NFT”，他认为游戏和内容实用性可能是印度人最喜欢的。

Kalidindi表示：“NFT收藏品可能在全球范围内有价值，但对印度人来说却没有价值。加密游戏规模庞大，在印度，玩赚游戏做得非常好。这可能就是印度找到适合 NFT 产品市场的地方。”

随着印度NFT实验的加速，观察印度对NFT的态度如何演变将是一件有趣的事情。
