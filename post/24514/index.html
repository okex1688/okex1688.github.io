6月17日16:00，“Kusama卡槽竞拍”系列AMA请到Phala CMO Ruby做客「金色百家谈」直播间，探讨了Phala先行网Khala插槽竞拍方案。

直播中，Ruby表示，Kusama平行链上线代表着波卡的真正上线，波卡的价值将被平行链体现出来。首批5个平行链，既有 DeFi 基础设施，又有智能合约链、扩容、隐私、桥、存储，接下来我们会看待波卡生态的价值传递。

对于这场接入Kusama的竞拍“盛宴”，我们准备在6月22日开始参与拍卖，crowloan已经在6月11日正式启动，截止6月16日下午，Khala crowdloan 的 KSM 数量已经突破 1W，预计前五个卡槽会有 Phala 的席位。对于用户来说，每个支持 Khala 竞拍的 KSM，Khala 将给出100个 PHA 作为奖励，即固定比例PHA：KSM =  100 : 1，PHA 奖励池上限为 1500万 PHA。

以下为直播先请：

金色百家谈：我们先请Ruby和大家打个招呼，然后对Phala做一个简单的介绍。

Ruby： 大家好，我是 Phala Network 的 CMO Ruby，很高兴能做客《金色百家谈》。

Phala Network是一个无需许可和去信任化的隐私计算云。 Phala基于 Substrate 建造，作为波卡生态系统的平行链运行，以提供与现有云服务相当的计算能力并保护托管程序的隐私。 基于Phala的TEE-区块链混合架构，开发者可以在 CPU 的 TEE 区域内轻松部署运行 Phala 机密智能合约，我们希望可以成为下一代的计算云服务标杆。

金色百家谈： 平行链拍卖已经在15日正式开启，您认为这将生态会有哪些变化，带来什么机遇？

Ruby： 平行链上线代表着波卡的真正上线，意味着波卡的优势价值可以被平行链体现出来，这件事情已经拖了一段时间，后面一个月我们可以看到生态的价值传递。

首批5个平行链，既有 DeFi 基础设施，又有智能合约链、扩容、隐私、桥、存储，相比以太坊和 BSC 的应用多、设施少的情况，应该可以快速补齐。借着卡槽拍卖的市场热度，我们相信第一波波卡 DeFi 会赢来爆发。接下来就是各个协议的组合性用例，可能会比以太坊的 DeFi 组合更好玩。

金色百家谈： 目前 Phala Kusama 插槽竞拍的最新进展如何？

Ruby： Phala 预备主网 Khala 将代表 Phala 参与 Kusama 竞拍，此前已经成功注册平行链ID 2004，6月11日晚，Khala Crowdloan Dapp（地址： 

https://crowdloan.phala.network/ ）正式上线，这是我们团队为此次拍卖开发的工具，用户可以通过它直接支持 Khala。虽然我们并没有打算竞拍第一个槽位，但是根据最新链上数据（地址：

https://polkadot.js.org/apps/?rpc=wss%3A%2F%2Fkusama-rpc.polkadot.io#/parachains/crowdloan）显示，已有超过1万枚 KSM 支持 Khala，目前暂列竞拍项目第五。

另外，我们近日举办了首次海外社区官方 AMA，Phala CEO&联合创始人 Marvin 与波卡高级大使 Zoe 一起向海外用户解读了 Khala 的竞拍策略，并对 Khala Crowdloan Dapp 进行了演示。有兴趣的朋友可以看下直播回顾：

https://www.youtube.com/embed/5he43wikOiY?modestbranding=1

金色百家谈： 对于Kusama及波卡卡槽拍卖，Phala 做了哪些准备？能否介绍下 Phala 的竞拍方案？

Ruby： 我们很早就在去年就开始着手准备拍卖的相关事宜了，并且也一直紧跟相关动态、持续技术迭代。关注Phala 的朋友一定知道，Phala 是从波卡平行链测试网 Rococo 上线前一路实践过来的。

首先是于2020年末，和 Acala 联合发布基于 Rococo v0 的跨链 demo：

https://mp.weixin.qq.com/s/3CjEGCfvmy3chuOuNcy1Pg；

然后随着 Rococo v1上线，Phala 成功接入，并在模拟竞拍中成功拍的槽位；

后来 Khala 成为第一批次成功注册 Kusama平行链 ID 的项目。非技术部分，4月末我们公布了 Phala 的 Kusama 竞拍策略（链接：

https://mp.weixin.qq.com/s/1iaLE_8Q2LjlgUsxW6LtWA ）——Phala 的预备主网 Khala Network 将参加 Kusama 平行链插槽拍卖，并将在第二个槽位释放后加入竞拍。

产品层面，我们还为所有有意愿参与众贷支持 Khala 用户开发了 Khala Crowdloan DApp，方便用户操作，6月11日已经正式上线。通过 Crowdloan 邀请朋友参与质押的用户，还能额外获得邀请奖励。

同时，我们还与多家支持 Kusama 平行链插槽拍卖的机构进行合作，比如 OKEx、KuCoin、Kraken、Newland、Bounce、Hotbit 、MXC 、Gate等等。用户能接触并参与支持 Khala 竞拍平行链的渠道还是很多的。

关于策略，有几个关键词：100:1, 1500W, 早鸟奖励。

总结下有以下8点：

Phala Network 将在 Kusama 网络部署预备主网 Khala Network（主网将随后部署在 Polkadot）；

Khala Network 并不会发行独立于 PHA 之外的新代币，Khala Network 将会以 K-PHA 作为网络通证，并且 K-PHA 与 ERC20-PHA 和主网 PHA 可以进行 1:1:1兑换；

Khala Network 将从第 2 个插槽开始参与 Kusama 平行链插槽拍卖，将拍卖 8 个时间段的插槽，即租赁期为 6 周*8=48 周；

在本次 Khala 众贷中，对于每个支持 Khala 竞拍的 KSM，Khala 将给出100个 PHA 作为奖励。即固定比例PHA：KSM =  100 : 1 。 这样设计的好处是不论给 Khala 的 KSM 支持量是多少，都不会稀释每个用户的奖励；

PHA 奖励池上限为 1500万 PHA，即在 Khala 达到15万个 KSM 的支持量后，将不再继续给支持者奖励；

在 Khala Network 成功拍得插槽，且作为平行链顺利运行后，PHA 奖励将立刻释放 34% ，并发放至支持者的地址；剩余 66% 奖励将分11个月线性释放至每个支持者地址；

Khala 的众贷活动将有额外的“邀请奖励”，A 用户邀请 B 用户帮助 Khala 赢得插槽拍卖后，A、B 双方都会获得 B 用户支持奖励的 0.5% 作为额外邀请奖励；

Khala 的插槽拍卖方式：将包括由 Phala 团队开发的链上 dAapp，以及与多个顶级中心化交易所合作的通道。

另外就是6月22日之前支持 Khala 竞拍的朋友，可以获得 NFT 激励，这个 NFT 在 Khala 竞拍成功上线后，可以对应兑换1%的额外激励。同时 NFT 可以进行出售，出售即意味着这个1%会转移给其他人。这里有张图可以比较好的告诉大家我们的竞拍激励。

金色百家谈： 竞拍成功后，会对 Phala 带来哪些影响？

Ruby： 我们希望基于 Substrate 开发的 Phala Network 可以作为波卡生态下的平行链，连接到 Kusama 以及 Polkadot 的跨链协议并进行互相操作，向其他智能合约平台提供隐私合约能力，将隐私计算及保护落到实处。

https://mp.weixin.qq.com/s/3CjEGCfvmy3chuOuNcy1Pg 

比如在这个基于 Rococo 测试网的落地场景中，Phala 和 Acala 作为两个“L2”(实际上波卡叫作平行链)，几乎没做改动，就完成了 L2 协议之间的互相调用，原理就是使用波卡的 XCMP 协议，非常的低成本，可用性很高。这说明在技术设计上，异构链之间的交互，波卡已经实现了很大一部分能力。

如果 Phala 成功拍得插槽，未来 Phala 将向波卡全生态输送隐私保护并扩大自己的效用边界。

金色百家谈： 以区块链搭建的Web3逐步成形，Phala 未来定义下能为web3.0提供什么？进展到了什么阶段？

Ruby： 目前，我们有三个主打产品：

用户管理自己所有数据和信息的隐私杀手App：暗钱包。可以支持划转&暗转账功能。可以为波卡defi提供隐私保护的环境，防止MEV套利攻击、抵押借贷定向爆破等目前defi普遍存在的问题。

低成本的去信任化桥，实现锚定资产跨链：pBridge系列。其中 pDiem 获得过 W3F 的Grant，millstone3 已经提交W3F并完成了所有审核。后面还会有pBTG，pDOGE，pSolona等跨链桥，来用Phala的低成本信任方案来帮助长尾链完成跨链。

数据分析工具：W3A，和 pDiem 一样，也获得了 W3F 的Grant。它能够在保护用户隐私的前提下，向开发者、企业提供成本低廉、体验完备的数据分析产品。

下一步就是卡槽拍卖，目前Kusama的卡槽拍卖已经启动了，我们预计在6月22日开始参与拍卖，crowloan已经在6月11日正式启动，6月16日下午，Khala crowdloan 的 KSM 数量已经突破 1W，感谢来自社区的支持，我们预计前五个卡槽会有 Phala 的席位。

金色百家谈： Phala 接下来的工作重点有哪些？

Ruby： 今年有很多工作要做，具体地说，我们会逐步实现：

参加并赢得 Kusama 和 Polkadot 的拍卖；启动平行链主网；

在Phala TEE中运行 WASM；

上线保密智能合约和 Web3 Analytics，并实现社区的去中心化治理；

支持万级别对的 TEE 设备接入 Phala；

在 Kusama 上上线暗池；

Phala DeFi 生态的三个拳头产品；

隐私云平台上线；

还有很多，不过我们属于闷头苦干的团队，相信能达成今年的目标。也欢迎大家关注 Phala 的微信公众号，「Phala可信网络」了解项目最新信息。
