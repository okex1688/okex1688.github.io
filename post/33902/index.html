前言

近几年NFT赛道引起了投资者的极大关注，一些NFT的售价对于很多人来说是不可想象也无法理解的。其中爆火的NFT收藏品系列，尤其是头像类收藏品 ，可以说是让大家又爱又恨，大部分用户张口就提及加密朋克，BAYC 等动辄几十上百倍价格拉升的NFT系列。上线几小时就翻了70倍的冷兔NFT，以及持续火了一个多星期的杰伦熊PhantaBear，一举点燃了华人圈NFT投资者的热情。大量巨头机构、明星的入场，让NFT市场中源源不断地涌入New money。NFT市场如此火爆，那么我们作为普通圈内用户，又该如何看待呢？

本文将从NFT的起源与发展开始讨论，汇总了NFT赛道的历史里程碑事件，再结合当下的热门NFT收藏品项目，分析整个赛道的价值与其中的风险。

1. NFT的概念、起源及发展

1.1 概念

非同质化代币 (Non-Fungible Token)，缩写为 NFT，被称为区块链数字账本上的数据单位，也属于加密货币中的一类。早前，“NFT”一词仅在 ERC-721 标准中流行，该标准于 2017 年通过以太坊GitHub首次提出。每个 NFT 都可以代表一份独一无二的数字资料，作为虚拟商品所有权的电子认证或证书。

由于其独特，不可拆分，不可互换的特性，NFT 可以代表数字文件（如画作、声音、视频、游戏中的项目等）或其他形式的创意作品。虽然 NFT 本身可以被无限复制的，但 NFT 数据可以在其底层区块链上被追踪，并提供所有权证明。

1.2 起源

谈及NFT，大家都知道它和以太坊的发展密不可分，但其实早在2015年7月以太坊主网启动前，NFT的概念就已问世。

1993年 ，Hal Finney（第一位经由交易机制得到比特币的人）在提出的加密交易卡（Crypto trading cards）时说表示：“可以以数字资产的形式向他人展示其收藏价值”。Hal Finney 依托加密学和数学，随机排列组成一个系列的套卡，并将其定义为加密卡。

而后，其概念落地，初代NFT Dapp：彩色币（Colored Coin）在2012年应运而生。彩色币是建立在现有比特币数据块上的挖矿和交易应用协议，由小面额的比特币组成，且最小单位为一聪（比特币最小单位1/100,000,000）。它以去中心化的方式代表了不同的资产，包括股票，债券，虚拟加密货币等。尽管彩色币存在不少缺陷，但彩色币同样也实现了资产上链的可塑性，以及巨大的发展潜力，为早期NFT 的发展奠定了基础。

图1-1 彩色币宣传图，图源：Google 图片搜索

2014年Counterparty（点对点交易平台）的“RarePepes”活动，旨在收集悲伤蛙（Pepe the forg，2015年最受欢迎虚拟角色）的全套表情包。基于比特币的CounterParty协议标准，每张悲伤蛙的表情包都跟一个独特的代币绑定，用户通过交换代币来交换不同的表情包，或者使用比特币购买其他表情包。每一张被创作的表情包都是独特的NFT，通过网络用户之间的传播，悲伤蛙NFT逐渐成为一个IP（知识产权），以“悲伤的”表情走入大众的视野，并史无前例的让用户感受到虚拟物品NFT也可以变得有价值。

图1-2 Pepe 蛙的表情包之一，图源：Google 图片搜索

1.3 发展

NFT正式进入人们的视野是在2017年，由CryptoKitties 的创始人提出。BTC翻了几倍后在加密市场走红，加密猫游戏诞生。随着加密猫的火爆，NFT 这个概念很快走进了大众视野。由此，数字资产不再只是指代加密货币，而是由同质化的加密货币与非同质化的 NFT 代表的加密资产组合而成。

图1-3 有关 NFT 的 Google 搜索量折线图

根据谷歌趋势显示，NFT一词在2020年以前搜索量一般，2021年起激增到现在一直处于上升的状态，意味着人们的关注度对此越来越高，NFT的市场容量也越来越大。

现状 从2021年8月以来，NFT市场爆火，Facebook高管表示正在考虑推出数字钱包的同时构建NFT功能；全球支付巨头Visa宣布以15万美元购买并收藏NFT数字艺术品CryptoPunks。 另一方面，NFT交易价格也在不断刷新记录：一副虚拟的石头画像在8月23日以400枚以太坊（约合130万美元）的价格售出，创下了NFT艺术品系列 EtherRock 的最新价格纪录。

图1-4NFT艺术品系列 EtherRock

大多数人对于NFT的影响力依旧有很大的争议，有人认为，NFT本身是没有任何价值的，高价售出主要原因是“价值认同”。还有人认为 NFT本身的潜在价值巨大，比如在加密货币市场中，使用NFT能够确保用户在虚拟世界完成身份构建和社交工作，相当于把现实世界中物品独特性和稀缺性较好的复制到元宇宙内。随着加密市场的发展，NFT的发展也渐渐形成了它独有的市场，从众多区块链项目中脱颖而出，成为当前热门赛道之一。

相信随着越来越有创意的NFT项目进入市场，人们对于该数字艺术品的追求也会越来越狂热，市场份额会越来越大。总体来看，NFT市场供需格局需要未来更多项目驻入市场来带动资金的流入，让它占有一席之地。

2. NFT从小众到出圈

2.1 年份大事记

  • 在1993年 Hal Finney 提出首个NFT概念并将其定义为加密交易卡（Crypto trading cards）。

  • 2012年第一个类似 NFT 的代币-彩色币诞生。

  • 2014-2017年间是NFT的发展期，2014年的**“RarePepes”**活动，令NFT逐渐走入用户视野。

  • 2017年，加密朋克（CryptoPunks）, 以太猫（CryptoKitties）引发了第一波 NFT 热潮，提高公众对NFT的认知度。

  • 2018年随着ERC721，ERC1155，ERC998 协议逐步完善，NFT也变得成熟。

  • 2020年现象级产品 NBATopshot 横空出世，并迅速攻占市场，此外NFT交易市场层出不穷。

  • 2021年，NFT以一系列大事件刷屏，占据了各大新闻头条，刷新了用户对区块链的认知，甚至有媒体称之为是NFT元年。

  • 2021年3月，Beeple 以 6903.4万美元出售 NFT，<Everydays: The First 5000 Days> 的出售可能是最具标志性的 NFT 时刻，艺术家Beeple 以 6900 万美元的价格将他的 NFT 艺术品卖给了 Metakovan。这无疑是加密艺术未来的决定性时刻，并向全世界宣告 NFT 是一股不可忽视的力量。

图3-1 Beeple 拍卖作品，图源：佳士得拍卖

  • 同年3月，Sandbox 推出 Beta NFT 市场，向元宇宙的创建迈进。用户通过游戏平台出售以此赚取 $SAND。

  • 2021年4月，佳士得拍卖CryptoPunks，9 枚 CryptoPunks NFT 共拍得了 16,962,500 美元。

  • 2021年6月，Decentraland 一块数字地皮以91.3万美元出售给虚拟房地产开发商Republic Realm，这块地皮代表了259块数字土地，以129.5万枚MANA结算（MANA是Decentraland的原生代币）。

  • 2021年7月，NFT交易市场OpenSea完成1亿美元B轮融资，由a16z领投。据DuneAnalytics数据统计，OpenSea 6月成交额达1.49亿美元，成交量达21万件，均创历史新高，此外，OpenSea总用户数量突破14万，环比增长19%。

  • 2021年8月，知名NBA球星库里花费18万美元购买一个BAYC NFT头像，一系列明星经过舆论的炒作，名人效应使得NFT赛道变得炙手可热。

2.2 里程碑式项目简括

2.2.1 CryptoPunks-加密朋克

CryptoPunks 是建立在以太坊上的最早期图像类 (JEPG) NFT 项目，这些极富“朋克风格”的像素化头像由美国手游开发公司 Larva Lab 于2017年使用计算机代码创建，没有任何两张图像是完全相同的。因此CryptoPunks 共发行了 1 万 个 24×24、8bit 样式的不规则像素 NFT 头像，不会再生成新的 NFT。每个 NFT 通过简化的小像素块鲜活的展示了曾经风靡世界的独特“朋克风”。

在创建早期，拥有以太坊钱包的用户可以免费领取 CryptoPunks 头像。如今，用户只能在 Larva Lab 和 OpenSea 等NFT二级交市场购买。2017-08-19，Larva Labs 宣布 CryptoPunks 已经完全链上以太坊网络，用户不仅可以免费在 Etherscan 上查询CryptoPunks 的属性（发型、眼镜、胡须、帽子等）还可以在NFT市场进行交易。

图3-4 机密朋克NFT价格排行榜，图源：Larva Labs 官网

2021年7月，一位匿名收藏家一小时内连续买入 102 枚 CryptoPunks，共计花费高达 2650 ETH，约合657万美元。这名巨鲸玩家行动时，其它玩家也豪掷千金，同年8月，“CryptoPunk 3100”挂出报价4200枚ETH，约合758万美元，成为目前世界上单价最高的NFT作品。此后，通过不断地宣传炒作，CryptoPunks很快打响了知名度，以马赛克般的怀旧风格以及不断成交的高昂价格刷新了区块链用户的观念，用户开始逐渐接受NFT这个新兴热门赛道并踊跃地参与其中，成为NFT收藏中的”疯狂“一员。

2.2.2 Crypto Kitties-以太猫/谜恋猫

感受到了NFT市场的强大发展潜力，再加上受到”前辈”CryptoPunks 的启发，Dapper Labs 团队立即推出了一款叫做 CryptoKitties 的以太猫游戏。其开发者 Dieter 用 NFT 图像表示每只小猫拥有的不同花色、基因、代际等信息。

Crypto Kitties于2017年11月28正式上线营运，是一款基于以太坊(Ethereum)上的智能合约游戏。由温哥华Axiom Zen的工作室Venture Studio制作完成。它是一款类卡牌游戏，每张卡牌上的小猫都不同，玩家可以购买、出售甚至是繁育其后代。当玩家购买小猫后，这只小猫就是完完全全属于购买玩家的。其火热程度甚至一度让以太坊网络崩溃，也成功的在加密货币领域迅速刮起了一阵养猫风，用小猫可爱呆萌的画风将NFT带入公众的视野。

图3-5 以太猫的部分示例，图源：CryptoKitties 官网

当时以太坊上最流行的通证发行标准是ERC-20。尤其2017年夏天，ERC20尤为火爆，很多用户问到加密猫和ERC20（ERC20是同质化通证，即FT）如何交互？为此 Dieter 提出了NFT，即非同质化通证。FT的主要特点是各通证之间是没有区别的，可以随意交换，且可以拆分整合。如：BTC、ETH等加密货币都是属于FT，每一枚BTC之间本质是没有任何区别的，可以互换和分割。Dieter所做的创新是将稀缺性引入到加密货币概念中，NFT的最主要的特点是唯一性和不可拆分性。每一枚非同质化通证各不相同，都是单独存在的、独一无二的，且无法互相替代。

CryptoPunks和CryptoKitties两个项目将NFT概念正式引入大众视野并带进市场，其被炒作的价格也让市场看到了此类NFT的无限潜力。并且CryptoPunks首次让人们将头像类NFT作为一种文化代表，达成了用NFT来彰显身份象征与社交地位的共识。

3.1 CryptoSkulls-★★★☆☆

图4-4 CryptoSkulls 基本信息介绍，截至2022-01-19

CryptoSkulls是存储在区块链上的 10,000 个可收藏的 ERC-721代币。每个 CryptoSkull 角色都是一个独特的像素艺术图像。从2019 年开始的OG NFT 项目，每个 CryptoSkull 字符都是具有唯一性索引属性的唯一像素艺术。

最便宜的CryptoSkulls入手价其实不过0.05ETH，而如今地板价都已经2.75ETH。交易量暴增，更曾一度打败Bored Ape Yacht Club成为TOP级别的NFT。无论是交易额以至升幅都是极之惊人，因此成为最近NFT圈子的热门话题。

图4-5 部分CryptoSkulls NFT 交易界面展示

CryptoSkulls 在其庞大的社区和名人投资者方面与 CryptoPunks 和 Bored Ape 等其他成功项目有很多共同之处，该项目还赋予持有者完全许可权，这是它与其他成功的 NFT 项目的一个相似之处。如今，有些NFT玩家出售他们的 CryptoPunks 以获得 CryptoSkulls。

许多人预测 NFT 空间将成为消费者日常使用的场所，音乐会门票、数字商品和其他收藏品等作为 NFT 出售，具有即时认证和消费者奖励的能力作为主要激励措施大规模采用。

3.2 Bored Ape Yacht Club-★★★★★

图4-6 BAYC 基本信息介绍，截至2022-01-19

Bored Ape Yacht Club 也被戏称为“无聊猿”，是一个收集了10,000个独特的Bored Ape NFTs 俱乐部，不仅是基于以太坊区块链上所建立的独特数字收藏品，而现在BAYC的地位，是仅次于CryptoPunks的热门项目。

BAYC 由美国Web3公司Yuga Labs于2021-04-29推出，由创作者使用演算法编程混合并匹配猿猴，创作出10,000个不同的猿猴图像。这些由算法呈现出的猿猴或出现在色彩斑斓的背景下，或穿着新奇古怪的服装，或佩戴着可爱搞怪的饰品。色彩鲜明且独一无二的设计感再加上高昂的价格，让不少NFT用户为其疯狂。

图4-7 部分BAYC 图像 交易界面展示

名人效应一直以来都是NFT市场销售不可或缺的一部分，而BAYC所附带的名人效应更是再一次证明了NFT市场的疯狂和能量。随着知名的NFT收藏家Pranksy的收购，”无聊猿“的热度不断发酵，不仅受到明星，说唱歌手，球星，主持人等一系列名人的青睐，各种各样的售卖新闻接踵而至。大批大批的NFT用户源源不断地涌入，都在为拥有一张“无聊猿”头像而庆幸。当需求远远大于供给时，BAYC的价格也变得水涨船高，从最初发售价0.08 ETH 到目前Opensea上的最低价格 82 ETH（约等于数十万美元）渐渐的也发展成了一个潮流IP。

BAYC的演变发展无疑是令人震撼且唏嘘的：

  • 最初发起人基于开放社群的初衷，希望促进一个积极向上的社群。

  • Bored Ape NFT头像作为用户进入其俱乐部的基本门槛，可以享受俱乐部会员的专属福利，解锁未来的区域和特权。

  • 名人效应使其变成了供不应求的价格高昂的VIP象征，成为炫耀的资本。

  • 再到现在没有实际应用而被挂上“名不副实”的潮牌商品。

可尽管NFT用户抨击此类的NFT，在下一次发售中，还是会有大量的用户为它“争破头脑”。相对的，BAYC作为NFT赛道中成功的产品，其短短9个月内爆发出的力量，不得不让人承认其强大的影响力。在NFT赛道中，总是会有像加密朋克（CryptoPunks）和BAYC 这样的项目不仅刷新用户认知还展现出NFT 的发展潜力。

3.3 PhantaBear-★★★☆☆

图4-8 PhantaBear 基本信息介绍，截至2022-01-19

PhantaBear 项目由周杰伦的 PHANTACi 和 Ezek 联合发起。PhantaBear 是 10,000 个通过算法生成的数字收藏品的集合，可兼作 Ezek 俱乐部的会员卡。该项目于2022 年1 月1 日11 时在Ezek 平台发行。预售价格为0.26ETH。每个 PhantaBear 都有一套独特的特征，并为其所有者解锁不同的、独特的访问权限和特权。

做为周杰伦间接代言的NFT头像，PhantaBear在发展时就自带吸粉效应。

但是尽管如此，其发展也并非一帆风顺，在诞生初期也曾遭受了极大的阻碍。其昂贵的发售价格促使用户前期并不买账，加上后期周杰伦经纪公司的公开澄清并未参与NFT制作和销售，导致其价格大跌和滞销。

4.1 NFT的价值特性

不同于现实世界中的传统行业，基于互联网发展的区块链技术在信息传递以及传播方式上有着明确且高效的改变。而NFT不仅做为区块链中的一大发展板块同时也是经济学中非必需品种类之一，其价值来源取决于群众对该物品的附加属性。

对于非必需品来说，改变其总价值的方式包括通过增加群体基数来扩大物品需求市场，和通过增强物品的稀缺性(减少数量)、附加值等来间接加强其原本具备的身份和权力认证。NFT就刚好具有这样的特性：

另外，除去NFT最基础的特性，部分额外的因素维度通常能赋能一系列独特的NFT收藏品

1）基础价值

NFT收藏品具备一定的收藏价值和观赏价值，给收藏该作品的用户带来心里上的满足感、和对外虚荣感的体现等。团队通过自己的特色、用心设计的独特NFT序列是其发展的基础，不过对于实际应用类的 NFT 而言，比如使用权凭证、财产凭证等，其审美价值则显得不那么重要。

2）实用价值

部分NFT收藏品不仅用于收藏，项目团队为了该项目持续发展，必然会持续的赋能这些NFT，让它们在特殊的场景中具有独特的功能。从本质上来说，项目方希望通过旗下NFT项目来铸造一定的IP形象，产生从中心向外围产生辐射效应。

  • 参与游戏。卡牌类，元宇宙资产类的NFT收藏品，部分可以当作参与游戏的“门票”，另外一部分则相对复杂，用于扩充游戏之中的额外功能和体验，增加用户可玩性，例如SANDBOX中土地可以用于开展官方或私人举办的活动等。

  • 会员特权。例如CryptoPunks项目，热爱该项目的人们自然而言的组成了社区，任何人都可以参与，但拥有NFT的用户可以不定期获得部分福利（如抽奖，空投等）

  • 数字身份象征。随着数字社会的发展，网络交流时间的增加，身份象征和形象具备了更多的适用场景，逐渐成为一种真实且不断扩大的需求。最能体现该特性就是NFT头像类收藏，目前CryptoPunks以及成为身份、地位的象征。犹如加入”上流社会“的”门票“，拥有该“门票”即可进入该圈子，获得与其他同样拥有该NFT用户交流探讨的机会等。

  • 衍生活动。目前Bored Ape Yacht Club 社区衍生出动漫制作、游戏等内容。尽管项目方在最开始并未规划该区域，但是随着社区自治的发展，开始逐渐向新方向拓宽道路，也同时推动该品牌下一系列产品活动等。

3）历史价值

NFT收藏品创作的历史背景，反映出所对应的时代以及蕴含的历史意义。其发展通常伴随着社区的建立再到认知的沉淀，最后达成共识的完善即可成为独特社区文化，同时也会成为该NFT所映射的历史价值之一。

4.2 NFT的价值困境

自于2021年NFT领域开始爆发，在2021上半年，其整体市值就达到127亿美元，相较2018年增长近310倍。同时也在那个时候，各大NFT平台完成了千万美元级别以上的A轮融资包括OpenSea、SuperRare、Bitski、Rarible等专注于不同领域的NFT交易市场。另外基于NFT的经济系统，国内外互联网大厂也开始布局NFT。例如腾讯和阿里所构建的蚂蚁链、腾讯至信链等，都是在制度的前提下开始寻求热度与功能性。表面上国内外NFT市场全面繁荣，但是其背后潜在的危机却从未离开。

1）当今NFT赛道发展中存在大量的投机现象，过度炒作、盲目追捧和疯狂创造短期热点，这使得NFT市场中蕴含着大量泡沫，一旦市场低迷的时候就会发现绝大部分NFT流动性约等于无。

最初火爆整个币圈的NFT竞拍《Everydays：The First 5000 Days》买家并不是正统的收藏家，而是一位资深的币圈玩家，也是NFT基金Metapurse的创始人Metakovan。他打着“让艺术品收藏更民主”的口号，发布了1000万枚代币。随后这些代币的价格从0.36美元一路飙升到了28美元。然后他在这场竞拍中不仅赚到了噱头，扣除拍卖支付费用后，还大赚一笔。

从市场目前投机氛围与Fomo情绪来看，市面上绝大部分NFT存在“噱头拉高价格”，或者通过大V，明星等来参与项目宣传炒作，或者对敲交易（左手倒右手），市场价格远大于市场价值，而一旦市场泡沫破裂，NFT资产价值总量将会进行大幅度缩水，超过90%NFT藏品将因为缺乏流动属性成为空气。

2）迄今为止，全球监管中尚未出台关于NFT法律规范，所以在这里不受任何监管的NFT成为践踏法律的重灾区

目前在全球范围内，NFT实体商品属性仍难以界定，未来真正在元宇宙中发挥支撑经济体系作用的NFT虚拟商品也难以被定性。对于包括NFT在内的各种数字资产属于何种财产权利，目前也尚无定论。也正因为缺乏官方概念的指导，所以该领域必然存在各种”庞氏骗局”的犯罪活动。同时部分自媒体网红并未承担好正确引领大众的责任义务，反而通过夸大价值和思想误导，给一系列毫无艺术价值的NFT“价值”包装，谎称艺术品投资，收藏品众筹等，运用区块链新型技术创造骗局。

3）NFT的安全保障。NFT做为数字产品其产生都是在区块链上执行的一段标准化软件代码，高度依赖于底层区块链协议的属性

NFT的铸造是基于以太坊网络，但目前从以太坊网络面临拓展性的难题，这也势必会影响其衍生品NFT的发展。另外在NFT创造途径与过程中也不是一帆风顺，目前的区块链技术处于高速发展时期但其安全隐私方面并不完善，所以并没有绝对的安全保障。这也势必会导致部分NFT资产遭受黑客等不法分子的攻击和盗取。

除了外部因素的隐患，许多NFT交易市场的安全制度是否达到标准？正如前段时间Opensea交易市场直接冻结了被盗NFT资产的行为，同样引起了社区的质疑，在交易NFT的过程中是否达到足够的去中心化，可追溯性等来保证交易的完整性?尽管目前暂未发生大规模安全隐私泄露事件，但我们仍需要关注这些隐蔽的角落，防止它们有机可乘。

5. NFT赛道总结与提示

在微观层面上，对单个NFT销售的研究揭示了加密货币艺术市场与传统艺术市场之间的巨大相似性，尽管两者之间存在根本差异（例如，交易透明度，无中间人参与等）。

最近加拿大概念艺术家Kimberly Parker的一项研究，分析了OpenSea的数据以及其他市场的销售信息，发现近80%的NFT售价低于500美元，而只有1.8%的初级销售（即NFT的首次销售）售价超过了0.5 ETH（据研究时汇率约894美元）。顶级稀缺资产从不缺市场，而大量长尾资产却一直滞销。这与因“头重脚轻”、“赢家通吃”而臭名昭著的传统艺术市场现状非常相似。去中心化并不意味着机会平等。

从宏观层面来看，NFT整个赛道正在兴起，这是毋庸置疑的。

但其长期价值其实仍然不够明朗。从长远来看，这些NFT被赋予的价值，关键取决于它们与线下世界的联系是什么。这个联系可能很简单，就是一个支持其创造和交流的社区。如果这个社区缺乏参与，那么与NFT相关的大部分价值也就流失了。所以NFT或许会成为一种价值的存储形式，但在此之前，我们会经历一个阶段，即现实世界的社会关系和社区价值转移或反映至链上。

我们目前正处于NFT发展的起步阶段，无法预测NFT未来将怎样改变区块链上的经济行为和模式。

NFT凭借铸造数字资产所有权的能力，以及易存储、受保护的特性，改变了不少艺术家和创作者的谋生方式，并带来了财富和机会。但在NFT赛道的发展过程中，也将面临比加密资产更多的不确定性和更大的风险。绝大多数NFT项目的生命周期非常短，过了火爆期之后，流动性变得不容乐观，极有可能就变成了钱包里的JPG图片。

现阶段的NFT市场存在不容小觑的安全隐患，需要警惕当中的”庞氏骗局“与”幕后操作“。所以我们应意识到，NFT带来的财富效应可能是巨大的，但各种不确定性带来的风险同样也不能忽视！

NFT赛道FAQ

Q1：如何确定NFT的稀缺性和价值？

NFT的价值取决于其所代表的内容。如果是数字货币艺术品，则价值衡量标准与其他任何类型的艺术品非常相似。我们需要考虑创作者、作品的艺术价值以及各路收藏家的需求度。如果NFT限量发行或属于系列收藏品的一部分，拥有特定编号的NFT通常更具收藏价值。

Q2：在哪里可以获得NFT？

获得NFT的途径一般有市场购买、项目方赠送以及自行铸造。NFT市场出售众多非同质化代币，创作者既有著名艺术家，也有业余爱好者。在众多产品中，规模最大的是基于以太坊的NFT平台OpenSea。新的市场平台一直在不断上线，如果有兴趣购买著名艺术家的作品，请务必验证市场的真实性。

Q3：如何验证NFT的真实性？

一般核验都需要通过区块链浏览器来查看NFT。有效信息包括铸造日期和铸造相应NFT的钱包地址。您还可以根据交易历史记录ID查看NFT是否匹配。这种方法比单纯检查与收藏品相关的照片或文件更可靠。涉及到资产时，请一定拒绝轻信，亲自核实。
