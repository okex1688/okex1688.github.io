作为区块链技术变革的代表项目，波卡从诞生至今一直颇受关注。而插槽拍卖作为波卡的重要节点和里程碑，也几乎聚集了所有波卡支持者的关注度。

随着Rococo测试网再次升级成功，Kusama平行链插槽拍卖已经进入倒计时。在平行链完成在Rococo V1的测试，并且在Kusama上线之后，Polkadot的平行链将正式面世，多链时代将开启。

欧易OKEx情报局持续关注波卡生态，近期正举办插槽拍卖系列AMA，对话平行链候选项目，挖掘有价值信息，此处可回顾前两期内容：《对话Fuyao：Kusama插槽拍卖将带来爆发行情，首条平行链或达到50万锁仓量》；《对话Plasm宋明时：平行链上线后，波卡的价值会真正爆发》

对于波卡普通投资者有这么关心的几大问题：目前Rococo测试网和Kusama网络的进展如何？插槽拍卖到底什么时候开始和上线？哪些项目公布了平行链竞拍的细则？用户有哪些方式可以深度参与波卡生态？插槽拍卖对生态和市场的影响？本文将一一解读。

目前阶段和重要进展

3月25日，平行链测试网 Rococo V1 重启成功后，正式开始测试众筹及卡槽拍卖模块。

3月31日，平行链测试网 Rococo结束了第四轮插槽拍卖测试，Phala Network、Literntry、Darwinia、Biforst集体赢得第四个插槽位，目前共有7个波卡项目成功在测试网上竞拍，前三个赢得卡槽的项目是Acala、Plasm、Kilt。这对Rococo测试网来说算是一大进展，也意味着测试网插槽拍卖的测试期逐渐步入尾声。

但是，我们观察到，目前在 Rococo 上，除了测试平行链 Tick、Trick 和 Track之外只有公益平行链Statemint。原因是 Rococo 又进行重启了，Rococo 作为测试网，很可能以后还有很多的重启。

关于Rococo最近也有一些新的进展和更新：1）增加了系统平行链的概念，平行链ID小于 1000 的都是系统平行链。2）Parity 将拍卖和众贷的插槽改成8个时段。这意味着，如果 Kusama 上线平行链后，平行链项目最多可以租赁48周的 Kusama 插槽，同样，用户最多需要锁定48周的 KSM。（注：Kusama上的平行链租赁插槽租赁周期缩短至6-48周，而波卡的平行链插槽租赁期为6-24个月。）

此外，还有关于 XCM 的重构，目前 Gavin 正在进行一系列的修改。XCM 的更改将导致很多编译问题，对于平行链团队来说，在升级版本的时候需要更改很多关于 XCM 的代码。所以目前大部分的平行链项目都在等待 XCM 的重构结束和 Rococo 的最终升级，然后再进行平行链升级和测试。

众人期待的Kusama平行链竞拍

为什么平行链拍卖如此受到关注和期待？这就需要提及波卡的架构和获得卡槽的方式。

波卡是一条可扩展的异构多链区块链，由“中继链 + 平行链” 组成。成为平行链，能够共享波卡的安全性、跨链互操作性、链上治理、无分叉链上升级、扩容性、低手续费等诸多好处，而平行链插槽就是平行链和中继链的接口，自然也就成为了“香饽饽”。

由于波卡网络资源有限，只能存在有限数量（预期约100条）的平行链插槽，因此，便产生了插槽拍卖的方式，项目方竞拍者需质押DOT/KSM参与竞拍，竞拍成功可租赁一定租期范围的卡槽，成为波卡/Kusama的平行链。

Kusama是波卡的先行测试网，为了确保一切可以顺利进行，平行链拍卖的所有流程会先在Kusama进行测试和优化，据Gavin透露，Kusama第一年会上线30-40个平行链。因此在前期Kusama的插槽拍卖更加受到关注，KSM币价也一路攀升。

平行链竞拍到底何时开始？可以关注Github上的进度。4月19日，Gavin在Github上发布了启动 Kusama平行链功能的三大阶段以及一系列待完成事项。

在开启正式的拍卖之前，会先部署Shell chain 和公益平行链 Statemint。 需要了解issues进度的话可以点开 github （https://github.com/paritytech/polkadot/issues/2902）查看，完成的会显示√。

这里我们需要提到一下波卡整个平行链插槽竞拍的流程。此前，波卡创始人Gavin Wood公布了波卡平行链的上线顺序：1）上线 Rococo v1（测试网）并等待其稳定运行；2）上线 Kusama 的 system（公益性质）平行链；3）开启 Kusama 上的竞拍；4）上线 Kusama 上竞拍成功的平行链；5）在审计完成后，上线 Polkadot 的公共利益性质平行链；6）开启 Polkadot 上的竞拍；7）上线 Polkadot 上竞拍成功的平行链。

目前我们仍处于第一阶段，从目前节奏来看，随着 Rococo 上的出块稳定和 Crowdloan 功能完善，接下来的 Kusama 平行链插槽拍卖大戏即将揭开面纱。

（如果你想跟进平行链、拍卖的状态，可以在 polkadot.js查询）

普通用户如何参与这场红利？

插槽拍卖既是波卡的里程碑事件，也是区块链生态爆发的催化剂。不少交易所、项目方、投资方、用户、矿池等利益相关方都已经摩拳擦掌，打算深度参与到这场盛大的生态红利之中。

Acala、Plasm、Phala、ChainX等项目已经公布了插槽竞拍计划或细则，而投资方不断给看好的项目“加码”资金，交易所不仅推出了锁仓挖矿产品，也将大力支持插槽拍卖，帮助用户参与投票锁仓，那么普通投资者具体该如何去参与平行链拍卖？

作为普通用户，主要有四种方式可以参与到这场红利：

1）持有KSM/DOT代币可选择质押代币赚取staking收益。

可以自己下载波卡钱包参与，这里有一份欧易OKEx情报局之前出的钱包质押教程：《奶爸级教程：Polkadot/Kusama挖矿与投票，从此代币不再缩水》。当然，也可以通过交易所的通道一键参与，这里以欧易OKEx为例，在APP首页点击“赚币/DeFi”，选择币种“DOT”，有余币宝（活期）、锁仓挖矿（活期）、定期生息（30日）几大产品，可以根据需要自行选择赚取收益。

2）购买DOT/KSM，在二级市场抓住获利时机捕获收益。

在平行链拍卖的预期和波卡生态迅速扩张的推动下，DOT和KSM代币价格水涨船高。普通持币者可以选择合适的时机选择买入或者出货。

3）根据市场合适时机进行数字资产衍生品交易，比如做多做空。

较为专业的投资者可以根据平行链相关的进展，对行情中短期趋势做预期判断，借助衍生品工具进行获利。

4）参与优质波卡项目的插槽众贷获得项目方提供的Token奖励。

一些资金不太充裕的平行链团队可能会选择使用Kusama的crowdloan机制来吸引用户支持他们参与平行链插槽竞标。参与众贷的用户需要在平行链插槽租期内锁定KSM，待租赁期结束或竞拍失败之后可以解锁。如果代币正处于staking，需要在众贷活动之前进行解绑。

（不同项目分配给众贷的代币份额不同，这个网址可以对比到不同平行链项目给予奖励的情况：https://parachain.live/。）

这里需要注意，当心骗局，本地 crowdloan 是通过 Kusama 的活动索引（campaign index）进行交易的，千万不要将自己钱包或交易所账户的 KSM 转给第三方地址来参与众贷。

参与众贷的用户要谨慎选择基本面更好的项目，减少机会成本损失。因为如果支持的项目竞拍失败，则意味着锁定的资产在一个月内无收益。

插槽拍卖对生态和市场的影响

既然插槽拍卖是多方受益的事情，那么，它对整个生态和市场将产生哪些影响？

首先对区块链生态来说，作为跨链项目的代表，波卡被称为是能够比肩以太坊的区块链项目。平行链上线之后，意味着波卡的跨链愿景将开始落地。我们的区块链世界不再是一个个数据孤岛，取而代之的是，我们将迎来多链互通的区块链时代，资产互通、信息互通，以及互操作性，这对波卡生态乃至整个区块链生态而言，都将有巨大的意义。

与此同时，随着平行链上线，越多越多的项目将接入波卡，专注于不同业务和领域的项目将推动波卡生态迎来爆发。

其次，对市场而言，不论是成为波卡还是Kusama的平行链，都需要参与竞拍的项目和用户质押DOT/KSM来竞拍插槽。4月8日，Substrate生态布道者Fuyao在做客欧易OKEx情报局《大咖零距离》的访谈对话时称， Kusama首条平行链可能会有20万-50万的量锁进去，而波卡首槽可能达到上千万级别的DOT量。大量的锁仓将DOT/KSM在二级市场的流动性，对币价上行是一种利好因素。

此外，平行链插槽拍卖会影响DOT网络总锁定率和锁定结构，同样可能会对 DOT 的二级市场表现产生一定的积极影响。当然，以上不作为投资建议，具体投资需要结合市场环境进行判断。

总而言之，当资源变得稀缺时，市场供需就会对价格产生一定的支撑。可以预计在平行链插槽拍卖正式上线之际，波卡生态和行情将迎来爆发。而参与这场红利的时机，宜早不宜迟。

正如波卡的信念所描述，“没有一个单独的区块链可以做一切，更确切地说，未来是多链的”。基于Web3.0的愿景，我们将迎来一个新的区块链时代。
