Web 3.0 时代中替代 加速器的机制将为下一代企业家及其区块链项目赋能。

我的同事 Elias Simos 喜欢说我们正在经历数字资产、包括非数字资产价值的数字化的文艺复兴。针对无数金融和点对点障碍的去中心化解决方案正在蓬勃发展，但许多创新者和企业家仍然依赖于构建项目的旧模式 。

一些平台，例如 Polkadot 和 Cosmos，正在建立适合 Web 3.0 的新的创业模型。它们使创建和连接去中心化应用程序变得更加容易，并使创新者能够构建更好的解决方案。这些多链协议就像一个初创团队的加速器 ，提供了一种启动和自然扩展数百个新区块链的机制。它打破了孤岛，为新进入者提供了参与和支持新兴项目的机会。

未来十年最激动人心的项目将使用区块链的支持系统。这些替代加速器的机制将赋能下一代企业家及其区块链项目 。

我们以 Polkadot 为例。通常情况下，加速器通过教育、指导和融资支持早期、增长驱动的初创公司。他们提供这些资源以换取初创公司的股权。虽然大多数企业都未能成功，但少数企业会成长为大型公司。Polkadot 的扩展机制为小型但有前途的公司提供资源，具有硅谷加速器（例如 Y Combinator 或 Techstars）的许多特征。 但是在波卡的这个 Web 3.0 愿景中，整个社区都可以参与到项目中，而不是只选择一些人为他们提供资源。

引导一个权益证明的网络是复杂的。初创团队从区块链生态系统所拥有的网络、人才、治理和卓越运营中受益匪浅 —— 就像加速器为初创公司提供的一样。Polkadot 网络和社区为应用程序或网络开发者提供所有这些元素，在价值链中最具挑战性的地方帮助他们。企业家可以在满足强大的市场需求之前找到适合他们的问题/解决方案并连接必要的资源。

去中心化平行链的过程

Web3 基金会的旗舰项目 Polkadot 和 Polkadot 上的金丝雀网络 Kusama 运行两种类型的区块链：中继链和平行链。中继链 是负责达成共识和完成交易的枢纽。平行链 是独立且可互操作的 Layer1 区块链，通过中继链共享安全性，它们包含各种去中心化项目，例如身份、DeFi、桥链和智能合约。Polkadot 和 Kusama 将各自拥有最多 100 个平行链插槽，并且必须通过拍卖和治理来获得它们。获得一个插槽的过程旨在确保平行链以公平公正 的方式获得资金，它有两个阶段 —— Crowdloans 和 Auctions。

在第一阶段 ，去中心化 Crowdloan，“Polkadot 允许平行链为其平行链 [auction] 出价而去收集代币”，支持者将 DOT（Polkadot 协议的原生代币）锁定到他们支持的项目长达两年。支持者永远不会失去对代币的保管，但代币是被锁定且不能流动的，直到 crowdloan 到期。这种锁定表明支持者相信平行链会成功。每个潜在的平行链项目都需要争夺 DOT 和 Kusama (KSM) 代币持有者的支持，就像初创公司在加速器推介日所做的那样。

第二阶段 要求锁定在 crowdloans 中的所有代币都用于平行链插槽拍卖。团队使用锁定的代币来竞标一个插槽，并根据需要来提高竞标价，直到一个未知的时间之后。这种拍卖方式可以追溯到 17 世纪流行的蜡烛拍卖，蜡烛火焰的到期标志着拍卖的结束，以避免竞争对手在最后一刻以高价进入。获胜项目的支持者将获得平行链的原生代币，以补偿他们的锁定代币。

对于开发人员和支持者来说，这种去中心化的平行链过程可能比 Web 3.0 的传统加速器提供更多的希望。可用资源 对成功至关重要。

Polkadot 的 Web 3.0 加速器模型中的主要区别包括：

  • 支持者永远不会失去对其代币的保管。

  • 当项目在拍卖中失败或对平行链插槽的访问权到期时，支持者锁定的代币将被退还。

  • 支持者不分担机会成本。

  • 除了你自己的小型网络外，还可以访问去中心化且无需信任的合作伙伴生态系统。

  • 任何个人或实体都可以运行验证人来获得奖励。

  • 平行链具有内在的可扩展性。

  • 平行链能够使用共享的安全模型，与 Polkadot 生态系统内外的其他项目互操作

开发者目前正在为 Polkadot 构建平行链，例如 Acala、ChainX 和 Chainlink。Kusama 平行链插槽拍卖已经进行到第四次。

多链的未来

其他平台，如 Cosmos，也提供了多链未来的一瞥。Cosmos Hub 是主链和经济中心，通过跨链通信将称为 Zones 的区块链连接在一起。为了换取 Cosmos Hub 上的安全服务，交易费用和质押奖励会分配给原生代币 ATOM 的质押者。

Polygon 和 SKALE 是与以太坊兼容的网络的例子，它们为开发人员提供了构建侧链的机会，以支持生态系统去中心化应用程序的增长。截至 2020 年 12 月，以太坊、Polkadot 和 Cosmos，所有旨在使开发者能够构建应用程序的平台，都是增长最快的 Layer1 生态系统。我们可能会看到 Web 3.0 对 Y Combinator 和 Techstars 的挑战正在兴起 。

通过培养社区和促进与主要支持者的伙伴关系，团队可以测试解决方案、制定制胜策略并在真正发布之前展示相当大的吸引力。有替代支持机制的多链未来将为下一批企业家及其区块链项目赋能。像 Polkadot 这样的可互操作的生态系统将发挥关键作用。
