**NFT 把每个人都变成一块广告牌，并与其现有的社交网络结合。 **

星球日报编者按：在 web2 的世界里，我们已经看到了品牌经济的新模式：从通过给用户投喂他们沉迷的内容增加广告播发量，到通过 KOL 推荐的信赖模式销售产品，但当中也存在许多问题：获客成本越来越高、数据不精准，无法有效激励核心消费者等等。这篇文章叙述了 NFT 改变现状的可能。NFT 将不止是炒作投机的产品，而是成为未来所有品牌建立真正的社区、通过链上数据有效触达消费者、提高核心消费者粘性的工具，很有意思。

随着互联网的出现，我们从交易经济转向了注意力经济。注意力经济的发展，牢牢抓住公众的心理，大规模地改变了文化的叙事。

NFT 则代表着注意力的金融产品化。随着越来越多的品牌发布 NFT，这可能会改变未来十年的商业模式，品牌的成本可能会无限降低。

不过目前，我们仍处于 NFT 的实验阶段，品牌在积极进行 NFT 实验。阿迪达斯正为 Bored Ape Yacht Club 成员制作卫衣，FTX 正助 Coachella 音乐节设计 NFT 音乐节终身门票， POAP 在帮助活动组织者利用 NFT 制作出勤证明。NFT 的可能性是无穷的。

而要预判 NFT 未来走向何方，我们首先需要知道商业如何走到今天。

NFT的前生

早期，人类经济是资源密集型的，所有权不受重视。随着时间的流逝，人们对给予自己的工作付出所有权成为了一种常态。想想公司是如何从发工资到分红，再到现在的员工期权激励。

分析资源支出和所有权后，我们总结出经济模型演变的四个阶段：

  • 交易经济： 货币交换商品或服务。人类历史的大部分时间都是在交易经济的基础上进行的。想想物物交换，丝绸之路或者最终的殖民形态。交易经济是以上所有的支柱。在船舶和贸易路线等技术出现之前，他们的重点是当地市场，一直持续到印刷机的出现。

  • 注意力经济： 印刷机大幅降低了复制信息的成本。但直到 18 世纪，图书市场才出现，识字率才大幅上升。报纸是今天的平台的先驱。注意力经济是指你向第三方出售“注意力”而不是商品。仅去年一年，Youtube 就通过广告影响力赚了 290 亿美元。他们几乎没有生产成本，而且客户粘性很高。你可以像 Facebook 那样把注意力经济规模扩大到一个国家的规模，而不用担心成本会按比例上升。

  • 平台经济： 平台经济将卖家和服务提供商与潜在买家联系起来。亚马逊因此获得了近 4890 亿美元的收入。依循同样的模式，Uber 的营收从 2013 年的 1 亿美元增长到 2020 年的 130 亿美元。平台的价值在于发现并信任平台上的供应商。我们现在每天使用的大多数手机应用都有一个平台。成本在于对供应商的管理和对拥有足够大用户基础的用户的信任。

  • 社区经济： 社区经济是合作社的演变。互联网让我们接触到全球市场（2010 年），智能合约使我们无需中介就可以相互信任（2020 年）。我们一直处于探索替代经济模式的阶段。这就是 DAO 的意义所在，它们将个人自治与以前不存在的经济机会联系起来。

社区经济是用户管理他们使用的东西的实例。这些经济模型在链上留下经济相互作用的证据，使得人们可以公开获取这些经济体参与者的数据。社区经济与平台经济和注意力经济的不同之处在于，参与者的数据通常不会储存在企业拥有的服务器上。任何人都可以查询并在此基础上进行构建。Gitcoin、ENS、Covalent、Biconomy 和基于 NFT 的 DAO（如 LobsterDAO）就是这样的例子。

股东驱动的公司算社区经济吗？GameStop 或许勉强符合这个定义，但现在我们还是说回 NFT 主题。

NFT的今世

让我们用图表来剖析不同经济模式下，品牌价值量化的不同。x 轴表示关系，关系指的是交易中当事人之间的关系。关系越密切，就越难进行高度量化和充分谈判的交易。在 y 轴上，我们细化分解下经济互动。它指的是卖方在交易前确定价值的程度，以及客户接受交易的速度。想想亚马逊的送货服务，你知道你买的是什么，以及什么时候它会到达你的门口。

买家通常知道劳斯莱斯或古驰产品的价值和价格，它们很少需要反复确认，因为品牌信任已经建立，价值主张也很明确。注意力经济平台的信任度则往往很低。我们很少直接从那些平台购买品牌产品。注意力经济平台已经为 KOL 进行了优化，因为它们在个人内容和商业之间取得了恰当的平衡。

有足够影响力（和信任）的人以订阅的形式直接销售商品，代表例子有 Substack （星球日报注：类似美版公众号的自媒体平台，但消息接收是以付费订阅邮件的形式）和 Patreon（星球日报注：供内容创作者和艺术家对自己的作品和产品进行众筹募资的平台）。问题在于这些 KOL 的内容可能存在偏见，这就是为什么社区在现代商业中很重要。

像 Product Hunt （星球日报注：一个发现新产品的平台，开发者可以提交自己的作品，网站会依据大众投票产生每日榜单，可以在上面了解到新颖的服务、有趣的应用、好玩的硬件等）这样的社区改变了参与者之间的关系，把从在互联网上偶然相遇的陌生人变成了受共同愿景驱动的合作者。

而对于基于代币和 NFT 的社区，价格通常是将人们凝聚在一起的粘合剂，社区减少了交易中的摩擦。 由于信息不对称最小化，最终价格通常是出现在公开竞拍上。Web3 项目社区将这一点发挥到了极致。大多数团队只是简单地办一个治理委员会，而不是进行外包。对任务感兴趣的个人可以通过社区治理规则完成并提交，获得代币支付。

这打破了典型的雇佣模式。贡献者与多个团队合作，而团队可以从一个更大的人才库中挑选人才。由于奖励通常以该项目的代币支付，贡献者可以选择换为稳定币，或是持有该代币。拥有代币也使贡献者在精神上和经济上归属于网络，从而提高用户粘性。

NFT的未来

NFT 将这种模式发挥到了极致。由于 NFT 的平均成本高于同质化代币，普通的散户参与者没有足够的闲置资金购买多个 NFT，只能 all in，要么 all out。展望未来，各大品牌或许将利用 NFT 作为一种刺激高端消费群体的方式。

给 1000 个最活跃的用户空投 NFT？使用 NFT 鼓励友好行为或向最活跃的产品用户发出信号。我们可以将 NFT 看作是链上的参与认证。因为 NFT 可以解锁新的客户特权。从历史上看，品牌拥有与用户相关的所有数据，而独立第三方开发者却无法直接激励这些用户，使用 NFT 则可以实现这一点。

多个商业实体可以根据钱包的历史提供独特的折扣

当然，NFT 并不只是用来解锁客户的特权，现在的会员卡也能做到这点。NFT 在未来能够实现的是以增值的方式达到无受许可的社区目标。

查理·芒格有句名言：“给我看激励，我就会给你看结果。”

Web2 平台会让我们沉迷于屏幕上的内容，因为它们的动机是基于销售广告。你花越多时间盯着有趣的视频，平台向你展示广告的可能性就越大，这就是为什么它们建立在可变奖励的基础上。

NFT 可能会成为我们改变互联网动态的途径。 品牌可以利用链上数据来瞄准用户，为他们提供特殊待遇，并进入他们可能从中受益的社区，而不是缓慢地推动用户购买他们可能并不需要的东西。当然，这里也存在隐私问题。例如，一个在现实生活中参加特定地理位置的人能否在链上被跟踪？这就是零知识证明这样的解决方案变得越来越重要的原因。

NFT带动的增长

你可能会认为我提出我们将从靠广告来经营我们的经济，转向用户数据导向是疯狂的。但有几个因素可能促成这种转变。iOS 14 发布以来的硬件或操作系统变化可以减少广告。事实上，有几个平台已经尝试过 NFT。

时代周刊发行一个含有 4600 多个 NFT 收藏品的系列，持有者可以访问该杂志。著名 WWE 明星 John Cena 向 1620 万名 Instagram 粉丝发布了 500 张 NFT，只有大约 37 人购买了它们。梅拉尼娅·特朗普试图发行 NFT，但她自己却把它们买回了。育碧软件是我最喜欢的游戏之一，它进行 NFT 发行测试，销售额仅为 400 美元。与名人有关的 NFT，绝大多数自发行以来呈下降趋势。

我认为基于 NFT 的社区建设需要对用户定位进行大量思考。

当 LooksRare发布时，有 22000 个持币地址。目前约有 1.8 万个钱包仍保存着这些代币。其中约 80% 持有 LOOKS 的时间超过了一个月。之所以会出现这种情况是因为 LooksRare 的目标用户是那些在 NFT 平台上活跃的交易用户。

另一个例子是 LobsterDAO，其 DAO 的启动得益于“在社区聊天中按用户活动的比例提供 NFT”。社区已经发布了大约 7 个 DeFi 原始版本，使用 NFT 用于验证用户的身份，并奖励代币以交换执行某些操作，如质押或增加流动性。对于团队来说，赠送的代币是获取曾经参与过 DeFi 相关讨论的用户的成本。

初创公司应该看到 NFT 为他们的关键用户提供了社会资本。Twitter Blue 的最新更新允许用户验证他们是否拥有 NFT。随着用户偏好的演变，报纸的广告模式可能也会面临同样的命运。Twitter 允许用户将他们持有的 NFT 显示为个人资料图片，这有效地将每个人的个人身份变成了一个广告牌，并将其与现有的社交网络相结合。

初创公司很快就可以空投带有预定义特权的 NFT，如高级用户、贡献者和产品大使，有效地引起第三方用户的兴趣和好奇心。另一种方式是教育。随着学习模式从大学向数字媒体发展，我们将看到基于链上非强制性学习的认证变得越来越普遍。如果一个平台是数字化的，可以简单地通过检查用户在链上的资产来访问用户。Bankless 是另一个使用链上资产作为 DAO 运行的发布平台。

为NFT而来，为社区而留

还记得我在本文开头将 NFT 定义为你的注意力的一种经济代表吗？我认为这是当今生态系统中最被忽视的方面。大多数 NFT 都代表了某种亚文化或是 meme 的色彩，这让人们有一种归属感，例如 mfers。这就是当品牌=身份的表现。

在内心深处，我们都在寻找同样的东西，一个我们所属的部落，一种身份认同。

有时候，用户会竭尽全力去建立自己在“部落”中的统治地位（社交地位），有时候这需要花费大量金钱。而 NFT 这样的商业工具让你找到同类、找到部落，彼此交换信号的过程变得更高效。
