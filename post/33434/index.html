为什么要铸造音乐 NFT？

“独立艺术家可以在 NFT 领域靠自己赚钱，从根本上消除了被公司实体低估、欺骗以及依赖于公司的风险。” — Freddy Got Magic，艺术家、制作人和音频工程师

以太坊是社区运行的中立的公共基础设施，不管是应用程序、媒体，还是资金方面。

对于艺术家，这意味着创作者可以使用以太坊和 NFT 直接以自己的方式与他们的粉丝互动并向他们分发自己的作品（例如专辑、歌词、音乐视频、单曲、EP等）。

这种直接面向用户的模式使音乐家能够充分利用他们的创作作品，无需处理困扰了传统音乐行业和企业音乐服务很久的中间商寻租问题。

此外，NFT 还为艺术家提供了新的机会，例如自动化和自主确定的版税，以及为观众提供的新体验。

以太坊是 NFT 的最佳平台吗？

以太坊是领先的智能合约平台，已成为 NFT青睐的区块链。简而言之，以太坊可以提供：

最好的安全保证 （以太坊是迄今为止最去中心化和最安全的智能合约平台，这意味着它可以无限期地保护你的 NFT）

最大的 NFT 社区 （以太坊拥有最大的 NFT 社区，因此目前是结交新朋友和联系新粉丝的最佳场所）

最好的流动性 （以太坊拥有最多的 NFT 市场和 NFT 活动）

不过，艺术家们目前也有在其他区块链上试验 NFT。这些替代链可能无法提供以太坊那样卓越长期的保证，但如果你仔细研究并权衡利弊，你可能会考虑尝试一些项目。我亲自测试过的两个：

Hic et Nunc — 基于 Tezos 的独立 NFT 艺术平台，支持 MP3、MP4、WAV。

Mintbase‌ — 基于 NEAR 协议构建的 NFT 平台，支持 MP3、MP4。

NFT 对环境有害吗？

可以毫不夸张地说，区块链和 NFT 被普遍误解了。互联网刚出现时也是如此。不幸的是，这些误解最近导致一些人错误地描述了 NFT 对环境的影响。

无论如何，你只要知道这一点：区块链和 NFT 的未来是通过环保的权益证明（PoS）系统来实现的。PoS 链被认为是“绿色”的，因为它们使用的能源比比特币等非 PoS 项目少很多个数量级。以太坊正处于完全过渡到 PoS 的边缘，并且有许多绿色的以太坊扩展解决方案，随着时间的推移，这些解决方案对创作者来说将变得越来越强大。

此外，我基于功利主义的理由认为，为了未来的环境保护，我们反而应该去拥抱以太坊、NFT 。为什么？因为 NFT 提供了一种数字化、直接面向用户的分发途径，可以去实体化或为大量的实体活动（例如全球CD发行）提供强大的替代方案。在绿色区块链和绿色扩展解决方案的支持下，NFT 可以并且将会这样做。

需要哪些准备工作

在以太坊上，你至少需要做好两项准备工作才能开始制作音乐 NFT：

一个以太坊浏览器钱包（MetaMask 是使用最广泛的；其他选项包括 WalletConnect 和 Coinbase Wallet）

一些 ETH（支付“gas”，也就是网络费用；在以太坊上进行铸币交易，一般都需要支付gas）

巨变-音乐产业正在NFT化

音乐行业存在着"霸榜",裙带关系,流量至上等诸多问题,这些问题是世界性的.而现在这种问题开始有了被解决的曙光. 音乐从数字化到通证化的进程,正在悄然开始. 

NFT的出现,解决了艺术文化行业对于"专利"这个核心的隐忧. 因为NFT基于区块链技术,实现了数据上链、不可篡改的目标,并且因为不可同质化的特性,确保的NFT的独一无二. 这在根本上,解决了音乐人对于自己专利的担忧. 将自己的专辑铸造成NFT,可以让创作者与自己的作品永久绑定,并且链上可查,任何人都无法偷走你的创意,也没法盗版,如果有人对你的音乐进行拷贝,那么通过链上查询就可以证明谁先进行的上链,并且区块链是通过分布式原理进行维护,所以这种证明没有任何人可以篡改.

NFT的底层逻辑,尊重了文化的独特性,让其价值得到了认证. 也正是因此,不少惊艳的创作艺术品通过NFT的形式,一直刷新着交易的新高,而NFT对于音乐行业的影响,正在暗流涌动,蓄势待发. 

2月，摇滚乐队Linkin Park的创办者联合主唱Mike Shinoda将自己创作的一段音乐作为NFT（One Hundredth Stream）在平台发行拍卖，最终以3万美元成交。

5月，歌手阿朵在个人微博发布了新歌《WATER KNOW》，全曲采用NFT技术进行数字加密，并将封面和歌曲的署名权公益拍卖，这是国内第一首NFT数字歌曲公益拍卖。据悉，上链仅10分钟，竞拍金额就突破1万元人民币，最终作品以304271元人民币的价格成交。

9月,通过一首《Made In China 》红透Youtube的国内顶流嘻哈乐团 Higher Brothers成员,Psy.p,也在Instgram公告了自己将发行一首以NFT形式的MV. 

DMX、Snoop Dog 等一众嘻哈大佬,也正在开始NFT行业的布局,TikTok,阿里等知名大厂,也在加速音乐行业NFT的进程, 我们现在正在见证音乐市场NFT化.未来,我们将通过镶嵌了NFT的音乐播放器,收听我们喜欢的音乐.
