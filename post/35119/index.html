在大多人心里，现在的web 3.0相比较之前的仅一个概念，已经演变为要即将到来。但事实上，它不仅仅是即将到来，甚至已经在这里了，而且增长得非常快。

目前，Web3.0市场已经价值数万亿，预计到2030年这个数字将翻十倍。虽然Web3.0是在2014年出现的一个术语，用来描述能够实现去中心化共识的新型协议，但它现在已经开始描述公共区块链、应用程序甚至是近年来设计理念的整个生态系统。接下来我们就来详细的向大家解释为什么Web 3.0已经存在。

Web1.0非常简单，但它是只读的，只有少数人积极创建内容。绝大多数用户只是使用静态页面和内容。大多数网站归公司和机构所有。这些网站也只是传统印刷媒体的再造。值得庆幸的是，Web2.0（我们目前所处的位置）的发展比我们曾经的设想更加动态和令人兴奋。

JavaScript和PHP等前端和后端技术的引入，使开发人员能够对动态网站进行编程。用户生成的内容和社交网络创造了一个新的在线创作和协作时代。今天，我们可以在各种平台上阅读、编写、查看和上传内容。我们在全球范围内相互分享、联系和交流。然而，这也出现了一个问题，如今Web 2.0的大部分内容都由少数几家大型科技公司控制，并且所有用户活动都受到监控。

与其他伟大的深奥想法一样，“什么是 Web3.0？”这个问题引出了一连串的答案，这些答案与表达它们的人一样多变。

对于某些人来说，新的术语可能会让人感到疏远，因此在这篇文章中，我们将介绍一些描述 Web3.0的想法，并提供大量示例以在实践中理解这些想法。

Web 1.0为只读，Web 2.0为可读写，Web 3.0为可读写

很简单，Web 1.0是只读的，Web 2.0是可读写的，Web 3.0是可读写的。Web的初始版本建立在TCP、IP、SMTP，当然还有HTTP等开源协议之上。协议是多台计算机同意相互通信的标准方式。这些基础协议管理互联网上的信息和消息流，如果您想使用它们的规则构建应用程序或服务，则无需为访问付费。

Web 2.0描述了使用互联网的免费和开源协议构建的下一个演变。然而，一个重要的转变是，与Web 1.0网站的静态只读版本不同，个人可以向网络添加内容。最初在Digg留言板上的支持变成了微博，现在有超过20亿的Facebook个人资料。

另一个微妙的转变也发生了。Web 2.0公司没有维护自己的服务器，而是支付了账单。然而，作为交换，他们还创建了一堆用户数据和行为，以构建一个对广告商非常有价值的社交图谱。在Web 2.0中，个人用户就是产品。

Web 3.0的所有权意味着平台的建设者、运营者和用户拥有他们使用的一部分。比特币和以太坊是最早的例子：作为更新分布式记账和保持其他参与者诚实的回报，他们会获得BTC或ETH的奖励，以换取网络安全。建立在以太坊和其他智能合约区块链上的基于代币的网络甚至引入了新的所有权模型，这些模型不一定与合作或股东股权模型相同。例如，所有权可能以为服务提供的代币的形式提供，例如为交易提供流动性，同样的代币也可以用于管理网络的未来变化。

Web 3.0是去中心化网络的新潮流名称

当我们提到去中心化网络时，我们还指的是去中心化货币和身份之外的堆栈的其余部分。去中心化网络的其他方面，例如去中心化存储，正在成为持久存储文件（如IPFS和Arweave）、去中心化存储（Golem、W3BCloud等）和去中心化数据（图形协议）的重要组成部分。现在，Web 3.0是a16z和其他大型风险投资公司整个投资类别的流行语，这意味着它也是Twitter冗长话题、讽刺、嘲笑和混乱的主题。Web 3.0成为公共互联网话语中足够大的一部分，让小丑们为协议牧师而来只是时间问题。

Web 3.0是互联网的身份层

早期互联网协议中最大的遗漏之一是没有公共和开源的身份层。此后，Facebook和Twitter等 Web 2.0平台将该层垄断为闭源应用程序。Web 3.0的立场是，您应该在网上拥有自己的身份，并且只有在您决定时才公开该身份的一部分。在实践中，以太坊上的身份是非常基本的。将其视为一个允许声明与之关联的容器。政府可以证明您的出生日期和地点，而无需了解有关您不同意的数字身份的任何其他信息。您的身份可能包括金融服务可以查询的交易历史，而无需知道您出生在哪里。此外，您在社交网络上开发的数字身份可以移植到其他网络。

在今天的实践中，我们拥有最接近Web 3.0中通用身份层的是以太坊名称服务（ENS）。使用ENS，您可以购买一个唯一名称（例如James back.eth）作为ERC-721不可替代代币，然后让它指向一个以太坊地址。ENS创建了人类可读的以太坊地址，但此后它已被用作空投NFT、与他人炫耀您的代币或NFT收藏以及了解谁在为DAO治理投票的提案投票的便捷方式。发出“获得”Web 3.0的信号具有不可否认的吸引力，这也许就是Paris Hilton、Shaq和其他名人在Twitter上使用.eth 用户名的原因。然而，与早期的互联网协议一样，ENS没有早期的投资者，而且协议本身是去中心化的，并且基于开放标准。

Web 3.0是对社交网络不保护用户数据安全并出售其利润的一种反应

Facebook拥有您社交图谱上的大部分数据。即使您退出服务，数据仍然与Meta的服务器相关联。Gavin Wood在2014年提出了当代术语Web 3.0的观点，“Web 3.0，或者可以称为‘后斯诺登’的网络，是对我们已经在使用的那些东西的重新想象，用一种完全不同的模式来进行各方之间的互动。我们认为是公开的信息，我们就会发布。我们假设对这些信息达成了共识，就会把它们放在一个共识分类帐上。” 2018年的剑桥分析丑闻显示，一家试图创建选民心理档案并影响选举的公司收集了8700万人的个人信息。虽然这一发现成为头条新闻，但也发生了重大数据泄露事件一直影响着数以百万计的互联网用户，仅仅是因为我们委托公司存储我们的数据，并且当我们转移到不同的服务时不能撤销权限。

还记得我们将MetaMask描述为加密同意管理器的时候吗？Web 3.0应用程序设计的原则是个人将信息“推送”到可信来源，而不是应用程序从拥有您数据的来源中提取信息。例如，在 Web 2.0世界中，当您“使用Google登录”时，应用程序可能会提取您不同意的个人识别数据。您提供给网络上不同应用程序的数据的不透明性是社交媒体网络能够创造如此主导地位的原因之一——这些信息非常有价值，而且在大多数情况下，我们会立即放弃它签署平台的服务条款。

Web 3.0可以被认为是对当今互联网上用户和平台之间的榨取关系的一种反应，在这种关系中，用户总是可以选择分享什么以及保密什么。

Web 3.0是让艺术家和创作者拥有他们在一个平台上创作的作品以及这个平台本身的方式

根据DappRadar的数据，似乎每个人都在2021年推出NFT，总交易量超过230亿美元。对于许多数字艺术家来说，NFT代表了第一个全职支持他们的手艺的真正方式。由于NFT是一种通用代币格式，您可以通过部署代码或使用FT市场来创建一个NFT。与Web 2.0的社交网络不同，您的代币可以在一项服务上购买，在二级市场上出售，或用于其他游戏和应用程序。换句话说，通过继承以太坊的属性，NFT是可移植和可互操作的价值表示。

一些最初的NFT市场很快意识到，他们的定位只是充当市场，但要在创造者、用户和平台本身之间建立更多的一致性。2021年，SuperRare发布了RARE治理代币并创建了SuperRare DAO，以表彰其最早的艺术家和收藏家，并鼓励社区参与其艺术的策展，并解释说，“我们将Spaces 视为社区驱动的未来艺术策展——一个由策展人、艺术家集体、画廊和社区成员组成的充满活力的生态系统，在SuperRare的共享品牌和技术下，加入艺术家并在拍卖上合作。”

以太坊上的其他应用程序现在正在使用代币作为奖励对网络的贡献并帮助管理决策的一种方式。甚至像Reddit这样的Web 2.0社交网络也在测试被称为“社区积分”的代币的使用，以便活跃的subreddit贡献者可以“拥有”社交网络的一部分。像Uniswap这样的DeFi协议已经通过激励流动性提供者为以太坊上几乎所有资产的交易对提供资金来创造正和动态。

个人在他们在网络上使用的许多服务中拥有集体利益的好处可能是对Web 2.0社交网络的网络效应的最大威胁。正如Scott Belsky所问的那样，“如果这些企业的每个利益相关者都受到强烈的激励来帮助建立、改进、营销和光顾品牌，那会成为对抗大公司的竞争优势吗？”

Web 3.0是一种新的互联网赞助模式

越来越模糊的“创作者经济”是一个用来描述旨在帮助创作者以新方式获利的互联网空间的术语。OnlyFans、Twitch和其他平台承诺平台用户可以自由地直接从他们的粉丝那里赚钱，而不是依赖广告驱动、基于注意力的货币化模式。但是，与Web 3.0网络不同，创建者可以随意从网络中启动，并且不拥有他们共享的内容。

对于作家和记者而言，在过去一年中，通过Substack、Ghost和Lede等平台，直接从观众那里获得收入的吸引力得到了放大。然而，这些都不能让作家通过所有权与他们的粉丝建立直接的关系。Mirror，一个 Web 3.0博客网络，让用户将他们的作品作为NFT出售，并用“众筹”重新定义作家和赞助人模式。众筹功能允许顾客存入ETH来资助一个想法，以换取代表您的赞助证明的代币，并可进一步用于获得DAO的入口或从出版物中获得未来的奖励。该代币具有实用性，但也可以表明您对某个想法或作家的早期支持，并且随着更多人同样支持众筹，它的价值会更高。

正如通过Mirror资助Dirt.xyz的《纽约客》特约撰稿人Kyle Chayka所说, “订阅对于多种媒体形式来说无疑是一种可持续的商业模式，但它不一定适合所有形式的内容或实验作品，而收藏家和赞助人非常适合这些形式。NFT可以为收藏家和赞助人关系提供支持，Mirror支持的代币种类也可以，例如ESSAY 或Emily Segal的NOVEL。我希望看到由代币和NFT资助的博客、系列和智囊团，而不仅仅是读者和经常性付款，同时为创作者和赞助人提供奖励。”

将可替代或不可替代的代币作为赞助到艺术家关系的一部分，艺术家也可以与他们最早的支持者有直接的联系——以太坊地址或ENS 名称的集合，可用于邮件列表、入场证和支付系统无论他们使用什么平台，艺术家都可以吸引他们的粉丝。

Web 3.0使建立合作治理和所有权结构变得容易

如果你在2021年加入了任何DAO，你可能出于各种原因进入了Telegram或Discord群组，其中有一群互联网陌生人：对DeFi治理提案进行投票；决定资助哪个项目；访问Erykah Badu音乐会；加入艺术家和开发商的驻留计划；集体购买武当派2015年专辑《少林往事》的唯一副本，甚至联手购买《美国宪法》的副本。

DAO或“去中心化自治组织”是一个社区主导的实体，它使用以太坊智能合约来建立基本规则并执行商定的决策。以太坊中一些最大和最早的DAO管理着越来越多的去中心化金融协议。前20名DAO目前持有价值近105亿美元的数字资产。

DAO不仅限于DeFi。像Bankless这样的媒体组织和像Gitcoin这样的公共资助实体都利用DAO来协调、治理和管理他们的财务。现在Web3中有超过130万DAO代币持有者。一位《纽约时报》记者打趣道，DAO 是“带有银行账户的聊天室”。但Web 3.0的一个不可否认的吸引力在于，志同道合的人组成的无领导的在线团体如何能够快速聚集、共同集中资金并做出决策。

Web 3.0还没有在每一层都去中心化

任何在Web 3.0生态系统中待了足够长的人都知道工程师在尝试遵循最大分散的架构、易于使用的应用程序和可扩展的基础架构时遇到的设计权衡。正如Signal的创始人Moxie Marlinspike最近在他对Web 3.0的探索中所写的那样，“web 2.0的前提是互联网上的每个人都将成为内容的发布者和消费者，以及基础设施的发布者和消费者。我们的网站都会有我们的网络服务器，我们的电子邮件服务器有我们的邮件服务器……但是——我认为这一点不能再强调了——这不是人们想要的。人们不想运行他们的服务器。”

当他得知以太坊的大部分应用程序调用来自可信API源的数据时，他应该不会太惊讶。或者说目前以太坊全节点的数量是5,433，其中40%目前在美国运行。这也是MetaMask做出的第一个设计决策之一——不是要求每个用户运行一个自托管的以太坊节点，而是连接到Infura来提供以太坊数据，这样开发人员就可以更专注于他们正在构建的应用程序，而不是运行网络基础设施。

正如MetaMask创始人Dan Finlay所写, “这使用户可以立即开始使用，而不会经常耗尽笔记本电脑的电池。它改变了采用的游戏规则，它展示了Moxie在这里所说的：人们不想托管他们的服务器（当然不是一个旨在消耗整台笔记本电脑容量的服务器）。”

这并不是说以太坊和Web 3.0社区仍在考虑如何减少各个层面的信任，无论是通过W3BCloud等去中心化数据中心，还是将轻客户端引入 Eth2的实现。虽然MetaMask使用Infura作为默认设置，但他们始终允许用户选择他们的区块链连接，并且使用Snaps，用户将能够选择整个钱包-服务器关系的替代方案。正如Dan Finlay解释的那样，“快照可能会帮助用户运行轻客户端，选择替代运行时，如zk-STARK链或新的友好语言，或者它只是让用户连接到他们喜欢的集中式服务。”

以太坊社区有很多团队致力于改善去中心化，但一些更新的、更中心化的加密网络的成功证明用户可能不太关心。然而，没有理由认为Web 3.0基础设施的现状在未来会保持不变——尤其是随着更多人体验到完全去中心化网络的优势。

互联网最伟大的创新之一是让信息在全球范围内可访问、廉价、可复制和丰富。这些财产与有价值的东西直接对立，无论是金钱还是财产，根据定义，它们都是稀缺且难以获得的。比特币是第一个将稀缺性引入互联网的协议，部分原因是解决了困扰早期数字货币尝试的“双花”问题。双花问题是指您可以使用重复的数字货币并在两个或多个地方同时花费的想法。在主流金融世界中，银行、信用卡公司和支付处理商会自行验证交易，以最大程度地降低双重支出的风险。使用去中心化的加密货币，矿工或验证者网络负责确保账户不会双花。这具有深远的意义，因为验证不再依赖于受信任的中心化方。通过互联网连接，任何人都可以参与点对点网络并检查分类帐。社会共识可以防止阴谋集团试图扭转或审查交易。

稀缺性的另一个属性是单个单元是否是可替代的。可替代性意味着您可以将一个单元替换为另一个单元，并且仍然值得。例如，1 ETH价值1 ETH。不可替代意味着单独独特。NFT赋予用户拥有对象的能力，这些对象可以是艺术品、照片、音乐、文本、游戏对象、凭证、治理权限、访问通行证。

如果我可以右键单击图像并将其保存到我的桌面，NFT怎么会稀缺呢？从根本上说，重要的是区块链维护从一个帐户到另一个帐户的所有权记录。它允许艺术家或公司建立“原创”，就像区块链解决的基本问题一样，防止另一个人声称自己是其所有者或“双花”。” 许多人对NFT作为证明数字来源的一种方式感到兴奋的原因之一是，由于它们是以太坊代币（NFT现在也可以在其他智能合约区块链上使用），它们可以在Web 3.0生态系统的其他部分互操作。它们可以分成更小的部分，所以多人可以拥有它们，用作其他去中心化金融服务的抵押品，包括永久使用费，甚至被用作互联网上的身份基础。
