一些BUG导致NBA球迷们没有得到该有的NFT，但联盟表示将会补偿他们。

NBA向球迷投放18000个免费的NFT的计划出现了问题。

篮球联盟昨天推出了一个名为“The Association”的新 NFT 集合，旨在为NBA Discord 服务器的最早成员提供独家 NFT 。没有想到的是，集合的智能合约中出现了安全漏洞：在 NFT 能够被创建和交易的计算机代码，导致用户在不公平地铸造 NFT，并在大约一个小时内清理集合。

NBA昨天承认了这一漏洞，并表示将努力为其球迷解决问题。今天，联盟宣布将其 NFT 收藏的数量从 18,000 件增加到 30,000 件，以确保每个应该收到一件的人都能得到一件。

nft是“不可替代的代币”的缩写，是唯一的代币，用来表示对数字资产的所有权，如艺术品和其他类型的收藏品。在这种情况下，每个协会NFT代表一名NBA球员：来自16支不同球队的75个NFT，最初总计18000个NFT。nft是“动态的”，将会改变，并可能会增加或减少价值，这取决于与之相关的玩家的现实生活表现。

每一个NFT都将预留给上周五刚刚推出的NBA Discord服务器的早期成员。这些成员被允许访问一个“允许列表”（白名单的另一个术语），该列表将为每个注册在列表中的以太坊钱包地址保留一个免费的 NFT。

但是协会智能合约中的漏洞破坏了对这些粉丝的承诺。这个漏洞允许被列入白名单的用户可以访问原来不允许使用的其他钱包。

合同也没有正确地记录每个钱包的 NFT 数量。“如果签订了合同，他可以在一次交易中铸造整个NFT的系列。”一名推特用户在周三提供了对代码的概述。

这个BUG导致一些用户直接铸造出了了他们想要的NBA的Nft，一些用户铸造出超过100个，然后迅速在二级NFT交易市场OpenSea上以超过0.30ETH（当时大约1000美元）的价格出售。

NBA在意识到他们的智能合同被利用后，在发布了一个多小时后暂停了NFT的下跌。

然而，这并不是 NBA 第一次使用 NFT。NBA Top Shot以Flow 区块链上的 NFT 发布，结果去年声名狼藉，但是影响了体育 NFT 收藏品的主流关注度的上升。虽然在 NBA 最新的NFT中与 Top Shot 没有明确的关系，但联盟已经向 Top Shot 收藏家承诺了一些额外罕见的协会 NFT。

尽管昨天的BUG很让人苦恼，但是NBA 似乎已准备好推进其协会 NFT 收藏的计划。NBA 的一位代表在其官方 Discord 服务器中宣布：“我们已经在允许名单上确定了昨天无法铸造 NFT 的钱包，并将向这些球迷空投来自 The Association 收藏的 NFT。”
