资产的流动性

流动性是指一项资产在不影响其市场价格的情况下可以转化为现成现金的效率。最具流动性的资产是现金本身。换句话说，流动性描述了一项资产可以在市场上以反映其内在价值的价格快速购买或出售的程度。

什么决定了一项资产的流动性？

一项资产的流动性是其转化为现金的难易程度。在公司财务中，流动性资产是指那些可以在短时间内用来偿还债务的资产，现金是你能拥有的最具流动性的资产。在某种程度上，资产的交易量越大，其流动性就越强。这是因为较高的交易量表明，该资产可以以市场公允价格进行交易。

NFT 的流动性

经历过png summer后，不少人手上积攒了一堆NFTs。有些人很喜欢，想长久持有，但也有一些人面临着 NFTs 无法卖出的情况。

NFT收藏品，如头像和艺术品，普遍被认为流动性较低。如果这些物品的所有者遇到合适的买家，便可以按市场价值进行交易。然而，如果想尽快得到现金，有可能不得不以折扣价出售，显然降低了此类资产的流动性。然而，当对某一物品的需求突然增加时（例如，市场对某一 Collection 的热情高涨），流动性将被重新评估。

衡量一个NFT的价值，有这么几种方法。一是过去的成交数据，但一个周全的买家会考虑到许多其他因素，例如，与其他同类型的产品相比的流动性，简称，相对流动性。道理很简单，较高的交易数量某种程度上代表较高的流动性。这个数字不是从货币的角度，而是从生产力的角度来证明产品的市场规模。然而，一些交易量小的市场（比如知名艺术家推出的作品集）也可以被认为一个非常独特和有吸引力的流动性市场，因此要分别讨论。

优势与问题并存

与传统市场相比，NFT具有许多优势，如防止复制和延长版权。每个NFT都与证明数字艺术的真实性和所有权的区块链联系在一起。另外，当数字艺术被转售时，原作者会收到每个NFT销售的百分比。因此，创作者和艺术家从原始销售下游发生的所有经济活动中获益。这样一来，创作者就能得到更多的应得利益。

对于那些将NFT作为投资资产的购买者来说，相对较低的流动性是 NFT 市场中一个很自然的问题。当前，NFT的交易并不是一个即时的过程，通常需要经过钱包注册等程序，购买者还要面对高昂的gas交易成本。

当前，NFT市场正在蓬勃发展，而作为一个崭新而独特的投资领域，尽管有很多投资特质，但如果只是简单的收藏，而不进行金融化，NFT作为一种资产的流动性就会降低，成为进一步发展面临的挑战。与股票、黄金和其他传统资产相比，全球艺术品和收藏品市场的流动性匮乏得多。这个问题在新生的NFT市场中更加明显。买卖双方或贷款人与借款人可能需要相当长的时间才能快速进行最佳和互惠互利的交易。

当涉及到收藏品时，转售是一个重大挑战。这又可以归结为缺乏流动性的问题。例如，投资者很难通过交易一种非金融资产来购买另一种。由于缺乏流动性，投资者不能轻易地将NFT换成资金。这就引出了一个问题：NFT热潮能否持续？毕竟，相比之下，高流动性的资产会导致投资者更快地做出购买决定。

为何缺乏流动性？

现在NFT市场面临的最大障碍之一，至少在数字资产的世界里，就是缺乏流动性和在不出售NFT的情况下获得投资回报的能力。正如DeFi所证明的那样，投资者重视流动性，渴望收益。那么，为什么NFT缺乏流动性呢？

NFT中的 “无担保”

知名艺术家的作品天价拍卖之后，NFT见证了数字创作者的大量涌入，大家都希望在聚光灯下一夜成名，但不那么容易。新的艺术家们在真正售出他们的NFT之前，必须跳过几个绊脚石。首先是了解NFT和其运作体系。另一个问题是，在不知道艺术品是否会卖出去的情况下，创作者必须通过在社交媒体上宣传他们的作品，并积极参与社区活动，来进行推广工作。但即使如此，也不能保证艺术品能成功出售。

非透明数据

在 NFT 市场上，一些销售数据是不可知的，而且指数中包含的另一半数据，即拍卖市场的数据，也有选择偏差。同时，网上拍卖产生的许多销售价格没有公布，再加上越来越多的私人赠予，比较价格可能会变得更不容易，而不是更容易。可见，NFT 市场因为无法做到透明和全面，导致判断流动性的困难，在遇到相似困境的NFT市场，想要提高判断流动性的能力，底层的数据支撑可以助一臂之力。

为什么流动性对NFT投资很重要？

考虑NFT资产的投资者来说，自然会关注到NFT相对于其他大多数传统资产的低流动性。目前，NFT的交易不是一个即时的过程。它通常需要一个复杂的过程和高交易成本。

如果您投资了某一种NFT，现在在市场上受到欢迎。然而，如果一个市场的流动性低，这意味着没有足够的买家来支付这个价格。当这种情况发生时，可能要以较低的价格出售，或等待更多的人加入市场，然后再出售。如果选择在这时出售，记录显示有人以较低的价格进行了出售，这可能会影响您的NFT在市场上的价值。

由于每个NFT资产的独特性，一般来说，无法通过连续的交易模式来量化NFT资产的流动性。

在股票市场上，成交率通常被用来衡量股票的流动性。换手率越高，股票的交易就越活跃。计算公式如下：

股票周转率=特定时间段内的成交量/发行股票总数的100%。

在此基础上，NFT流动性指标可以是：特定类别的NFT资产流动性=该类别的NFT交易数量/该类别的NFT总流动性100%。

判断特定项目的流动性数据

对于那些不打算长期持有 NFT，并想要通过 NFT 买卖获得收益的朋友，流动性指标确实是必须要关注的数据之一。举个例子，30天内The Sandbox比Decentraland的流动性略高，但整体而言都是偏低的，并且呈下降趋势。

(Decentraland 流动性数据，数据来源：nftgo.io)

当然，随着行业的成熟，还会有更多流动性指标出现，比如类比于艺术品的领域的 BI Rate (Bought-In)，衡量 NFT 艺术品类未售出的平均份额，表示在出售艺术品的难度和风险水平。非常低的 BI 表明需求很高，证明行业流动性更强。

所有这些指标都是有用的资源，人们可能会关注前 25、50 或 1000 项目的结果。但最终也避免不了长尾效应，所看到的热潮，往往集中于拍卖市场中表现最好的部分。

未来，随着知识产权作为 NFT 转移到区块链上，数以万亿计的数字内容将转移到二级市场。这将释放出巨大的非流动价值，并成为区块链中最大的资产类别，并赋予 NFT 流动性更深的意义。借助标准和智能合约，NFT 或许可以被多个用户或社区分割、共同拥有和管理，并进行碎片化交易、众筹，并释放出更多价值。
