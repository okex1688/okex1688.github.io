随着各类加密产品的成熟度越来越高，其用户群体与影响力都在不断扩大，但目前严重制约用户体验的问题在于，用户仍然难以即时获取链上产品动态。

例如，DeFi借贷用户无法直接获得其仓位抵押率不足的通知，可能面临清算的风险；代币持有者无法及时获得重要治理提案开启投票以及结束的通知；无法及时获取链上合约或期权到期的通知。

因此，链上信息通知服务类协议正在大范围出现，它们可以帮助用户及时获得链上动态，其目标是在构建一个去中心化信息推送协议，允许用户（即钱包地址）接收来自各种 Dapp、智能合约甚至是中心化服务的信息通知。

下面链捕手将介绍几款链上信息通知服务协议，包以太坊推送通知服务 （EPNS）、Dialect、HAL、Notifi、XMTP等。

1.以太坊推送通知服务 （EPNS）

以太坊推送通知服务（EPNS）是帮助用户接收以太坊链上信息的通知协议，同时也让智能合约、服务商向用户发送通知。

2020年12月，EPNS完成了75万美元种子轮融资，Coinbase前首席技术官、Gitcoin联合创始人、MakerDAO负责人等多名业内人士和风投机构参与。2021年3月，EPNS又以1200万美元估值完成66万美元种子轮扩展融资，Binance Labs、LD Capital等均有参投。

据了解，EPNS平台由三个主要参与者组成——服务商、频道和订阅者。服务商是指想要发送推送通知的Dapps或者智能合约。订阅者是订阅频道并想要收到相关推送通知的用户。频道是指EPNS协议已激活的信息推送服务。服务商们会设立自己的频道，以便于在特定事件发生或者触发某项参数时发送通知，用户可以选择订阅自己感兴趣的频道。

用户可以通过下载该项目 APP 或在桌面浏览器下载其扩展应用，通过绑定链上地址接受订阅通知，并以弹窗的形式呈现。该项目主网于今年1月初上线，目前有超过50个频道以及超过1万名用户。

PUSH 是 EPNS 协议的原生治理代币，为持有者提供两个主要利益：治理权和应收奖励。治理权即代币所有者对协议决策进行投票的权利，包括质押结构、质押机制和奖励分配率等。应收奖励是指当服务商创建频道时，必须将至少 50 个 DAI的费用存入到一个共同的质押池，质押池将所有DAI质押到AAVE中产生持续的被动收益，收益按照订阅者加入EPNS频道的时间比例分配给PUSH持有者。

官网：http://epns.io/

2.Dialect

Dialect 是基于Solana主网的消息通知平台，帮助用户接受由链上事件自动触发的通知。除了推送通知之外，Dialect 还可以帮助开发人员在其 Web 3 应用程序中构建钱包之间的聊天功能。

3月2日，Dialect 宣布完成 410 万美元种子轮融资，由Multicoin Capital 和 Jump Capital 共同领投。

Dialect的特点是支持用户钱包之间的聊天功能，通过使用发布-订阅（pub-sub）消息传递功能处理链上信息。目前 Dialect v0 支持钱包之间的一对一消息传递，未来版本还将支持一对多和多对多消息传递。

团队在Github上开源了协议框架，其中有3个实用性的组件：

  • React组件可以让开发人员快速无缝地把Web 3通知和聊天功能集成到Dapp。

  • Saber monitoring service组件则用于构建推特机器人，实时监控链上的转账信息并发布推特。

  • Monitor组件是一个Dialect简化版通知中心集成框架，可以为开发人员提供 API 来提取链上数据、然后转换格式并生成通知，实现跟踪 dApp 订阅者链上资源的功能。

官方表示预计会出现大量基于Dialect的消息传递类应用程序，用于构建对话式的 DEX、链上聊天应用程序，甚至是功能齐全的 Web3 收件箱。

官网：https://www.dialect.to/

3.HAL

HAL是一个功能强大且易于使用的区块链数据侦听和自动化平台，帮助任何人自动化和查询区块链数据 。该项目主网上线于2020年6月，已集成以太坊、Polygon 、xDai、BSC与Avalanche，并支持用户订阅Aave、Compound等近40个DeFi协议的通知。

今年2月，HAL已获得由 CoinFund、Eden Block 和 Animoca Brands 领投的 300 万美元种子资金，Hashkey Capital、Wintermute、SkyVision、imToken Ventures、Bitcoin.com 等参投。

HAL的工作原理基于触发器实现，目前支持观察合约、观察交易、观看事件等三类触发器。当触发用户设置的触发器后，HAL 支持可以调用的几种不同类型的操作，向用户的邮箱、Discord、Telegram、Slack账户发送通知。

目前该项目支持的场景包括代币价格追踪、链上清算追踪、交易对创建追踪、借贷利率追踪等。以代币价格监控功能为例，用户根据网页流程使用简单的"IF…THEN…“的语句就能完成监控设置，例如用户想监控BNB Chain上Sushiswap的BNB价格，只需要按照下图点击下拉菜单就可以完成监控设置，链上代币价格达到设置条件时，用户就能收到HAL的通知。

值得注意的是，Aave、Bancor、Gitcoin等多个协议还在前端集成了该产品，例如Gitcoin部分提案界面支持用户通过HAL设置提案投票结束的提醒。

官网：https://www.hal.xyz/

4.Notifi

2022年3月3日，Notifi 完成250万美元种子轮融资，由Race Capital和Hashed领投，筹集的资金将用于发展团队和建设基础设施。Notifi 将采用简单的 API 设计，提供可轻松集成 dApp 和 Web3 应用程序的可定制高级功能。目前，用户可以在Notifi 官网申请参与早期产品的测试。

Notifi 创始团队来自Meta、Microsoft、Oracle、Amazon等科技巨头，拥有快速交付产品的能力。尤其是CEO和CTO，他们之前负责Oracle的云服务产品，因对Web3缺乏企业级通知感到沮丧而创建Notifi，旨在成为 Web3 的 Twilio，简化所有 Web3 消息传递渠道之间的通信。

据悉，Notifi将推出三款产品：

Notifi API 允许开发人员通过简单的 API 解锁复杂的通信基础设施，这些 API 可以为世界上所有的应用程序提供原生用户体验。

Notifi Center 为用户提供定制信息的通知体验，它将从移动端和 Web 端允许用户查看和管理 web3 世界中的所有信息。

Notifi Push 使营销人员能够创建有凝聚力的多渠道参与，从而推动业务增长和保留其用户群。使用钱包数据选择您的受众，并大规模推送您的消息或空投！

官网：https://notifi.network/

5.Tenderly

Tenderly是一个全面的以太坊开发者平台，用于实时监控、警报、调试和模拟智能合约。用户可以创建自定义场景，实时跟踪和分析链上数据。

目前，Tenderly支持包括ETH、BSC、Polygon、Fantom等10多条公链及其测试链的使用，最近也支持以太坊Layer2的Optimistic、Arbitrum、zkSync等公链。

相比与其他链上消息通知类协议，Tenderly拥有更多强大的功能，比如钱包监控、交易过滤、可视化调试、智能合约分析等，同时支持开发者使用Soldity和Vyper两种编程语言进行开发。

显然，Tenderly面向的用户更多是开发者，产品分为3个价位：免费版、开发版（80美元/月）、专业版（500美元/月）。免费版可以设置3个15分钟以内的警报通知，而开发版和专业版是实时通知。

Tenderly监控智能合约的函数调用，并作出反应的功能，让链上的操作可以实现自动化、智能化。比如NFT 开发人员可以监控合约铸造的时间并在 Discord 上自动发布公告。用户可以使用通知来关注钱包余额和合约操作，以监督开发人员是否正在做预期做的事情。

此前，Tenderly 完成 4000 万美元 的B 轮融资，加上之前种子轮和A轮融资，共筹集 5860 万美元，投资方包括 Accel 、Point Nine Capital、Version One 、Coinbase Ventures。

官网：https://tenderly.co/

6.XMTP

XMTP 是连接社区、协议、应用程序和用户的加密原生通信协议，将通过自己的独立协议和去中心化网络实现加密钱包之间的通信。

去年8月底，XMTP 完成 2000 万美元的 A 轮融资，由 a16z Crypto领投，Atelier Ventures、Betaworks、Alchemy、Coinbase Ventures、Scalar Capital、Not Boring Capial等参投。

XMTP 设想的服务场景包括：当协议、dapp 等出现问题时，可以直接向钱包发送消息；通过激励协调建立更好的方法来防止垃圾邮件和不请自来的消息；建立一个所有 Dapp 都可以使用的统一、去中心化的收件箱等。

目前 XMTP 公开的项目细节并不多，但可以看出该项目不仅仅希望成为链上事件通知协议，还希望成为链上钱包通信基础设施，打造下一代网络构建基础层通信协议。

官网：https://xmtp.com/
