Web3 没有蓝图，我们每个人都需要参与其中，并收获与自己做出的贡献对等的回报。

今天我们讲 Web3，大家都看到了，标题叫「为什么说所有帮你定义 Web3 的人都是骗子」，今天相信大家都看到或者听到了很多人关于 Web3 是割韭菜的批判，同时也有很多人给 Web3 下各种各样不同的定义。但其实所有的这些东西都是可能会有问题的。

第一点，我们知道现在 Web3 在资本市场上非常受追捧，我们知道任何一个有清晰的定义或者说有利可图的项目，其实都会有资本比你更早的进入，所以市场是能非常快速地反应关于所有已知信息的价格的。

Web3 之所以有利可图的本质，并非因为有人能清晰的定义跟预测，反而正是因为它不可定义跟预测。

这是资本市场的一个基本常识，最好的项目是什么样的项目？基本上有三条标准 ，第一条标准我们叫做项目本身的技术准备已经基本 ready，那么 Web3 毫无疑问符合这一条。

第二条是因为它有高度的不可预测性，所以它的状态和发展前景是模糊不清的。正是因为这一条阻止了很多资本追高进入。

第三条则是它有非常高的上限，一旦技术上准备好了，并且它的基本模式确立之后，这个概念就能够在资本市场上冲到一个非常高的价位上去。

我们都知道，Web3 意在重建整个互联网，所以说， Web3 之所以在资本上如此受追捧，并不是资本想割谁的韭菜，而是因为完美符合了上述三条标准。

我们如何重建互联网？为什么 Web3 要重建互联网？要搞清楚这个问题，我们就一定要搞清楚什么是 Web2，不清楚历史的人是无法预测未来的。

为了搞清楚 Web2，我们就要知道真正在过去 20 年的时间里面驱动整个 Web2 高速发展的两大基石 。

第一个是开源（Open Source），第二个是用户原创内容（UGC）。

Open Source 是什么？ Open Source 是全球的程序员，无论在哪里，都可以通过网络来参与一个他想支持的项目，成为这个项目的贡献者，为这个项目贡献代码，甚至影响这个项目未来的发展和走向。其实 Open Source 其实就是一种去中心化的开发模式。

而 UGC 则是用户层面的特征，用户在 Web2 的项目上不但可读也可写。比如 B 站如果没有大家上传的视频，如果没有大家生产的弹幕，其实它没有价值。正是因为大量的用户在抖音在 B 站在微博在推特上创造了大量的原创内容，Web2 整个网络才可以实现。

所以上述两点构成了 Web2 的两大基石，但是 Web2 还埋了两颗雷，这两颗雷是谁埋的，其实就是号称不作恶的谷歌为代表的大公司的。

第一颗雷还针对开发者层面，我们知道开发者可以在开源项目中做贡献，他能贡献代码，它能改变这个项目的走向，但是它不能分享 Open Source 的开源项目所产生的真实的商业利益。

但是我们知道的是 Open Source 项目支撑了整个 Web2 互联网的非常多应用的基本底层，它所支撑的这么高的价值，最后去到了哪里去？其实是到了 Facebook、谷歌，还有甲骨文等等这些互联网巨头口袋里，他们用一个非常巧妙的商业模式，掠取了全部 Open Source 项目的所有商业价值。

而在这之后他们发现如果 Open Source 项目不再继续发展下去，那么赚钱就不方便了，所以他们又雇大量的人，然后重新反哺开源社区，让真正的谷歌员工 Facebook 员工去开源社区贡献代码，然后他们美其名曰这是给开源社区做贡献，我应该说这种说法非常的不要脸，因为没有开源社区，其实他们都赚不到钱。

第二颗雷是什么？第二颗雷是在 UGC 层面的，你仔细分析一下你使用的所有互联网服务的用户需知，最终你会发现对其实你对你自己生产的内容没有对准控制权，这是为什么平台想删你的内容就删你的内容，因为你 UGC 的内容已经不是你完完全全可控的私有产权了，那个没人会去看的用户需知已经从你手里剥夺走了这个权利。

用户创造了内容，用户创造的内容促成了繁荣。最终这种繁荣为这些商业巨头带来了巨大的商业回报。而反过来这些商业巨头还跟用户说你在免费使用我的服务，所以你必须得遵守我的条款，你生产的内容最终实际上并不属于你自己。所以这个也应该说是非常不要脸的行为。

现在我们知道了 Web2 的两大基石以及两颗雷，你就大概能搞明白为什么今天那么多人都涌入 Web3，迫不及待的想建设新一代互联网，因为今天技术跟组织的发展让大家隐隐约约看到了一种可能性，就是把 Web2 两大基石下面谷歌这些不作恶的大公司埋的两颗雷彻底排掉的可能性，这就是 Web3 真正吸引人的地方 ，也是驱动了无数的开发者用户和骗子加入 Web3 大潮的关键。

第一颗雷就是开发者的雷，其实我们可以通过 NFT 为所有项目的贡献记账，让大家清晰的知道谁在这个项目里做出了什么贡献，而智能合约最终能把这个项目的利润分给这些贡献者。

第二颗雷 UGC 的雷，当区块链技术和不可撤销的记账结合 NFT 的时候，你会发现虚拟资产可以真正成为私有产权，比如说当 Twitter Web3 化之后也许你用起来是完全一样的，可是那时候的 Twitter 并没有权利经由欧盟的命令直接把你发布的内容删除，因为这是你的私有产权。

所以说 Web3 讲的这些故事，让我们真的有可能排掉 Web2 的这两颗大雷。

正是因为如此，Web3 打倒的对象就是 Facebook 跟谷歌这些 Web2 时代的互联网巨头，而这个时候如果 Facebook 跳出来说，我要拥抱 Web3，这事就很讽刺，所以我们如何分辨一个项目是否是真正的 Web3 可以从两个维度来进行判断。

第一个特征就是这个项目的贡献者是否能获得公允的回报，必须得经由不可撤销的记录和智能合约能分到这个项目的回报，它不再是用爱发电。第二点是 Web3 的项目会使用非常多的先进的技术手段，降低贡献者参与的门槛。因为 Web3 想要重建整个 Web2 的互联网的野心如果要达成的话，我们需要非常多的贡献者，这是为什么 Roblox 成为了 Web3 领域的第一个上市公司的原因，是因为这公司发布了一个非常简单的制作游戏的图形化工具，小孩都可以在上面制作自己的游戏，他们不再是一个用户，而成为了一个真正的贡献者，同时他能收到这个项目的回报，对这就是真正的 Web3 项目的两大特征，基本上能过了这两关的。那么你可以认为它是一个 Web3 项目。

如果我在最后做一个比喻帮助你理解的话，Web2 是已经一个建成的大厦，这个大厦真正的主人就是衣冠楚楚的谷歌、Facebook、甲骨文等等这些互联网巨头，他们已经在第五大道的帝国大厦里面住好了，他们当然希望所有人给他打工，而在这个帝国大厦的旁边。另一个街区有一个哄闹的大工地，里面汇集了各种野心勃勃的开发者，汇集了非常多很有才华的用户，而与此同时汇集了很多投机的骗子，大家在这个工地里面各种添砖加瓦。

这个时候如果有人这样，他说我们 Web3 的大工地，我们要建成这样一个结果，拿出了一个蓝图，对不起，没有人听他的，因为这里面没有权利层级的关系，大家都是各自建设，也有可能有些人在偷偷破坏，这就是一个 Web3 的真实状况。

正是因为如此，我们真的没办法定义所有这些人在大工地里做的事情会把这片工地最终建成什么样，Web3 根本没有蓝图，我们每个人都需要参与其中，并收获与自己做出的贡献对等的回报，因为 Web3 最关键的原则在于对于各种形式的「私有财产」的保护。
