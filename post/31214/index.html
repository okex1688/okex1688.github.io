 元宇宙  

加密云经济是下一个新兴市场的投资前沿，而元宇宙处于Web 3.0互联网发展的前沿。元宇宙是一组相互连接的、体验式的3D虚拟世界，在这里，任何人都可以实时社交，形成一个持久的、用户自主拥有的、跨越数字和物理世界的互联网经济。

像元宇宙这样的新兴应⽤还在不断涌现，但构建元宇宙的关键已经初具雏形，并正在彻底改变从电子商务到媒体和娱乐，以及房地产行业。我们的Grayscale Decentraland Report和Decentraland Tour通过引入领先的基于区块链的虚拟世界之一——Decentraland，使元宇宙这一概念更加具体。

Decentraland等项目是创建一个开放的元宇宙,用户可以登录到游戏,获得体力(本机Decentraland的象征,用户可以购买非功能性测试,包括土地或收藏品,并投票决定经济治理),或者创建非功能性测试,让他们真实世界的互操作性为他们的游戏时间创造价值。

Decentraland这样的项⽬正在创建⼀个开放元宇宙世界，⽤户可以登录玩游戏，赚取MANA（Decentraland的原⽣令牌，⽤户可以使⽤该令牌购买NFT，包括LAND或收藏品，并就经济治理进⾏投票），或创建NFT，以他们在游戏中花费的时间的 价值提供现实世界的互操作性。

这种互联⽹演变的潜⼒已经开始吸引像Facebook之类的Web 2.0的公司，Facebook正在转向Metaverse，并将其名称更改为“Meta”。⽽其他领先的 Web 2.0科技公司可能需要开始探索Metaverse以保持竞争⼒，这也引发了对元宇宙这类新兴加密类别的新⼀轮投资。

 元宇宙论点  

互联网的诞生主要是为了更好的链接人们。

在过去的三十年，互联网技术不断发展，我们与互联网交互的方式也随之改变，网络的发展被认为是以下三个关键时代:

  • Web 1.0—Netscape—将我们与互联网连接；

  • Web 2.0—Facebook—将我们连接到网络社区；

  • Web 3.0—去中心化—将我们连接到一个拥有虚拟世界的社区；

回顾过去的发展，我们的互动以及用来创造它们的媒介也在不断扩展，我们亲身见证了连接我们的组织架构以及计算是如何转变的，目前我们所依赖的基础设施已经成熟，对网络的控制也逐渐减弱，并在社区和大型科技公司之间流动。

图2:Web 1.0、2.0和3.0的关键特性

Web 2.0移动互联⽹改变了我们使⽤互联⽹的⽅式、地点、时间和原因。反过来⼜改变了我们使用的产品、服务和公司，改变了我们的商业模式、⽂化和政治——Web 3.0 Metaverse。

 元宇宙市场机遇  

我们越来越多的注意力都集中在数字活动上，尤其是年轻一代来说。目前，我们⼤约1/3的⽣命（每天约8⼩时）已经花在看电视、玩游戏或社交媒体上。随着我们花更多时间在这些数字世界体验上，我们也在这些数字领域花费了更多的钱来建⽴我们在这些在线社区中的社会地位。

我们的社交生活和游戏正在融合，并创造出一个庞大且快速增长的虚拟世界商品消费经济。据估计，虚拟游戏世界的收⼊可能会从2020年的约180亿美元增⻓到2025年的约400亿美元。

游戏开发商盈利的持续转变是这一增长的关键动力。越来越多的玩家放弃了付费游戏，转向免费游戏。开发者通过向玩家出售游戏内道具来盈利，以提高在这些虚拟世界中的游戏玩法和社会地位。

随着从Web 2.0封闭企业元宇宙的转变，进一步加速向Web 3.0开放加密元宇宙⽹络的过渡，它们是:

Web 2.0 Closed Corporate Metaverse: 由⼤型科技公司集中拥有和控制；

Web 3.0 Open Crypto Metaverse: 由全球⽤户⺠主拥有和控制。

如今，许多游戏玩家在Web 2.0封闭的企业元宇宙世界中花费了他们的⾦钱和时间来积累数字财富。Web 2.0的问题是，⼤多数游戏开发⼈员不允许玩家将他们的投资和努⼒货币化。 开发者禁⽌玩家与其他玩家交易物品，并关闭这些世界，使得玩家⽆法将其游戏内财富转移到实体经济中。

Web 3.0开放加密元宇宙⽹络通过消除Web 2.0平台对这些虚拟世界施加的资本控制来解决这个问题。这个新的范式允许用户拥有他们的数字资产作为不可替代通证(NFT)，与游戏中的其他用户进⾏交易，并将其带到其他数字体验中，创造⼀个全新的⾃由市场互联⽹原⽣经济，可以在物理世界中货币化。这种“创造者经济”的演变被称为“边玩边赚”。

在元宇宙中，⽤户指定这些类型的跨数字社区的⽆缝、现实世界的相邻互动。相反，与Web3.0开放加密元宇宙⽹络相⽐，Web 2.0企业Metaverse平台的封闭性可能会使用户处于劣势。

到Web 3.0开放加密Metaverse网络。已建立的Web 2.0公司将需要打破他们的商业模式，通过开放他们的生态系统，移除他们的竞争性护城河。我们还不知道Facebook将如何实现他们的元宇宙野心，但是他们——就像其他Web 2.0公司一样——将需要进行这种具有挑战性的转变面对满足股东季度业绩的压力。

成熟的Web 2.0公司需要通过开放⽣态系统和拆除竞争护城河来破坏其商业模式。我们对于Facebook⾛向Metaverse的野心尚未可知，但他们和其他Web 2.0公司⼀样，在⾯临满⾜股东季度业绩的压⼒时，需要做出具有挑战性的转变。

游戏只是价值已经开始⾃然转向Web 3.0的最直接解决的部分之⼀，但是元宇宙的机遇不仅仅局限于游戏领域。 元宇宙估计是数万亿美元的收入机会，涉及广告、社交商务和数字活动、硬件和开发者/开发者盈利。
