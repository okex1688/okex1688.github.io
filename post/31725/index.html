除了Roberts有明确提及上市计划外，其创始人兼CEO Devin Finzer此前也委婉地表示，公司虽然当前Roberts并不缺钱（前不久刚以15亿美元估值融资1亿美元），但是OpenSea筹集到的更多资金可以用来收购公司、建立合作伙伴关系并建立合资企业，以进一步将 NFT 的使用扩展到新行业，当前公司正在与包括a16z、Founders Fund、Coinbase和Blockchain Capital在内的投资方讨论新的融资问题。（补充：目前还没有一家公司在 IPO 之后还发行自己的 token）

不过需要注意的是，OpenSea目前尚未明确表示不会发行代币。一些乐观主义者认为，OpenSea可能会炮制一种混合模式，将传统的IPO与某种类型的社区代币相结合，不过随着IPO消息不断涌现，甚至连社区也认为：OpenSea背叛了Web3.0。

OpenSea：一家被抬上NFT龙头宝座的公司

OpenSea是一家怎样的公司？

首先，OpenSea是一个NFT交易平台，靠对收取卖方交易 2.5% 的交易手续费盈利。只不过，OpenSea是全球最大的NFT交易平台，自上线以来，其历史销售额超过了100亿美元，据悉在交易峰值的8月，OpenSea平台月交易额高达34亿美元，目前在11月的NFT交易中，OpenSea独占92.5%的市场份额。

（数据来源：Dune Analytics）

其实，OpenSea是一家被&amprdquo;抬&ampldquo;上NFT行业龙头的百亿独角兽，不过并非前文中空投用户的所言，只是被他们抬上去的，将OpenSea抬上龙头宝座的还有风口与资本两只手。

正所谓站在风口上猪也能飞起来，NFT的风口来临让OpenSea成功飞了起来。据OpenSea 创始人德文&ampmiddot;芬泽(Devin Finzer) 和亚历克斯&ampmiddot;阿特拉 (AlexAtallah) 回忆，他们在上线一个可创建、购买和出售各种不可替代的NFT平台&ampmdash;&ampmdash;OpenSea 之初，过得很惨。上线 26 个月后，他们只有 4000 名活跃用户每月进行 110 万美元的交易，这相当于（基于OpenSea 2.5% 的销售佣金）平台每月收入仅有28000美元。

而这一情况在2021年2月出现了转机，NFT市场开始苏醒了，大量用户涌入平台在平台交易NFT，此后几乎每一次现象级爆款 NFT 发售后或是 Beanie、Gary Vaynerchuk 等 KOL 同时宣传某个 NFT 系列时，OpenSea网站都会由于访问人数激增而崩溃。

OpenSea体量的急速扩大就像是亚马逊雨林中的蝴蝶扇动了一下翅膀，它会导致一连串反应。首先，投用户看到了财富入口，它们期待着 OpenSea 发行自己的原生 token 并认为如果 OpenSea 发行 token 一定会给用户发放空投（实际上 OpenSea也的确发行并治理过Token），所以有人不惜花费人力财力用数百个账号与 OpenSea 一一交互，希望能在发空投时大赚一笔。

然后当空投用户帮着OpenSea刷数据时，资本又看到了其背后的财富价值，纷纷对OpenSea 抛开资本的橄榄枝（红衫资本虽然没有投，但近期也表示要all in区块链）。最后我们看到的结果就是OpenSea 成为了全球最大的NFT交易平台，而且在2021年还获得了a16z、Coinbase在内的多家头部基金投资（2021年分别获得2300万美元A轮、1亿美元B轮融资，当前正在进行10亿美元的C轮融资）的多轮融资。

OpenSea：一家拔苗助长的公司

被抬上龙头之位的OpenSea还是一家拔苗助长、根基不稳的公司。

OpenSea的发展十分迅速，从月活跃用户4000人到月活跃20万人，OpenSea半年50倍增长的发展是奇迹，但也让OpenSea多次陷入危机。例如前文中提到的服务器崩溃问题，据统计，2021年OpenSea因NFT交易而引发的崩溃，大大小小次数超11次，还有无数次拥堵（甚至还出现过在一段时间内所有 NFT 均无法挂单的问题），陷入OpenSea目前还没有准备好迎接好快速增长带来的一系列问题。

另一个例子便是前不久令人大跌眼镜的&ampldquo;老鼠仓&amprdquo;事件，OpenSea员工利用职务之便，事先利用几个小号账号买入NFT，然后在头版头条上对其大肆渲染以使其价值飙升，最后再将其转入主账号完成交易。老鼠仓事件在推特被抖出来后引起了用户的强烈抵触，OpenSea不得不采取紧急措施，即禁止 OpenSea 员工在公司所宣传收藏家或艺术家的时候购买他们的 NFT、禁止 OpenSea 员工通过内幕信息买卖 NFT （不限于在 OpenSea 平台上），以便平息大众的怒火，但站在上帝视觉来看，OpenSea无疑是在拔苗助长。

同时OpenSea的根基也不稳。虽然OpenSea当然较行业第二名 Rarible具有压倒性优势（其他平台加起来交易量也没有OpenSea多），但是强敌正在来袭，近期包含Coinbase、FTX.us等交易所均已宣布计划推出NFT市场，而且随着红杉资本等机构加入战局助威，未来后来者的威胁将会越来越大。而且如果OpenSea真的无法刷空投，想必这部分以利益为驱动的核心用户并不介意换个地方刷空投，而他们的离去也将反噬其估值。

哈佛商学院副教授Scott Duke Kominers也表示，&ampldquo;加密领域的早期领导者面临着很大的风险，一个行业的第一批参与者不一定是最成功的，你的网络必须足够强大，才能战胜后来借用你商业模式成功部分并针对略有不同的受众的竞争对手。&amprdquo;

最后还是回到开始的问题，OpenSea是一家怎样的公司，我想我的答案是：OpenSea是一个较有影响力的NFT交易平台，仅此而已。
