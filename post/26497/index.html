铸造 BED 代币

未来，你会更多听到“DAO和DAO之间达成合作”的消息。请注意，新BED指数代币正是两个DAO合作的成果！

这是因为Bankless DAO和Index Coop达成合作并推出了BED代币，后者是促进DeFi脉冲指数等加密指数代币交易的去中心化资产管理协议。

正如其名，BED指数代币由BTC、ETH和DeFi组成，各占3成，即三分之一的WBTC、ETH和DPI，DPI由14个主流DeFi代币构成。

那么为什么投资BED代币呢？对于各类投资者而言，BED代币能够提供一种被动简单地投资加密经济顶级项目资产的方式，尤其对于新加密投资者。

BED代币结合了简单直接的加密投资策略，能够自动重新平衡代币权重，这意味着不需要人工操作，降低了费用，对任何人而言，投资成本都很低。对于想要获得加密货币基本面敞口的投资者来说，BED代币是一种非常方便省事的投资选择。

1. 通过TokenSets界面，提供相应比例的BTC、ETH和DPI等底层资产，创建BED代币。

下文将演示如何从零开始铸造BED代币，以及如何通过提供流动性来获得收益！

1） 获得 BED 代币

难度 ：新手难度

要求 ：Metamask等web3钱包

获得BED代币非常容易。首先，你前往Index Coop网站中的Bankless BED指数仪表盘。然后连接钱包，例如Metamask、WalletConnect和Coinbase，就可以拥有BED代币了。

Index Coop支持ETH, DAI和USDC购买BED代币，所以你有多种选择！

一旦你确定了想购买的金额，点击“购买”并签署交易。一旦交易在以太坊得到确认，恭喜你！你现在可以一边投资加密货币一边睡安稳觉了。

2 ） 成为 BED 代币流动性提供者

难度 ：中高级

要求 ：拥有Metamask钱包，理解Uniswap V3

一旦你获得了BED代币，不需要长期在钱包中持有，可以开始在DeFi中提供BED代币流动性，获得收益！虽然未来陆续会有更多项目会集成该代币，但现在可以为Uniswap V3 BED/WETH池提供流动性。

你如果非常热衷寻找DeFi机会，可以按照以下步骤提供流动性：

1. 前往Uniswap V3 BED/WETH Pool仪表板。

2. 输入你想提供的BED/WETH数量，并确定想要提供的流动性范围。

3. 点击“同意”，让Uniswap使用你的资金并完成后续交易。然后通过钱包选择“预览”、“添加”和“确认”。一旦保证金交易得到确认，你就能够为Uniswap V3 BED/WETH提供流动性！
