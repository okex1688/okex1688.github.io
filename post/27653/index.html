艺术是人类文明能量丰富的表现。 从净能量方面看，这是完全没有价值。但当被视为超越我们消耗卡路里、繁殖和消亡的基本功能的人类最纯粹的表达时，它是无价的。我们“工作”是为了享受休闲，休闲是一种个人的追求，通常涉及一些艺术形式。这包括音乐、电影、绘画、雕塑、体育等。所有这些活动都是能量消耗，但给参与者和观众带来了无尽的乐趣。

超级智能网络思维机器的出现将预示着一个时代，绝大多数人类的劳动在经济上都毫无价值。从工作的物理限制中解放出来，人类将转向他们的新数字世界和文明创造力和活力的完整表达。元节是未来。

那么，纯数字结构中的“艺术”概念是什么？只是一种能量抽象的金钱，怎么会被“浪费”在数字艺术的追求上？与所有其他“传统”艺术形式相似，基于 NFT 的艺术形式是否既无价值又无价？

从能源的角度来看，NFT 许可的艺术完全没有价值，但它将代表在纯数字世界中 Flex 社会地位的最终方式。虽然对于那些认为巴塞尔艺术展和威尼斯双年展是志同道合的文化人聚会的缩影的人来说，这似乎很愚蠢，但在区块链上交易的可无限复制的 JPEG 并不比画布上的波浪线更愚蠢。

你一文不值

与自学习智能机器相比，绝大多数人类劳动不值得为其维持所需的能量投入。不管你认为自己有多“聪明”或“有创造力”，在接下来的几十年里，机器都会比你更好。

现在要做什么？一旦我们厌倦了 COVID 的封锁，老年人就会进行体育运动、在海滩上闲逛、去实体夜总会等。（我指的是千禧一代和更年长的人。）年轻人将玩电子游戏并在各种数字元宇宙中创造全新的世界。

COVID-19冲击只是加速了这些趋势 。现在，世界上很大一部分人都被锁在他们的住所里；他们唯一的互动方式是通过联网的机器。无论您喜欢与否，您的在线头像只会变得越来越重要。元宇宙在进行中，你正在参与它的创造。

元宇宙将是人类头脑可以想象的任何东西，它不会受到我们在肉空间中认为理所当然的传统物理定律的阻碍。我们无法想象的整个新经济和职业都将出现在这些世界中。希望这些工作能创造出与传统工作相同的自我满足感，让人们对自己的生活感到满意。另一种选择是数十亿不安的灵魂会抨击感知和实际的不平等，尤其是当资本进一步集中在我们的技术霸主手中时。

世界上最有权势的人——根据他影响的用户数量来定义——是马克·扎克伯格。他打赌，未来真正支持 Facebook 的社区是元宇宙。允许人类在数字空间中创建社区的 AR 和 VR 设备是 Facebook 认为数字社区的下一次迭代所代表的东西。

中国正在打造一个移动优先的网络社会。所有数据和交互都由国家联网、监控和监管。中国元节是完整的社会信用评分，涵盖您的整个线上和线下生活。考虑到人工智能和 5G 的研发支出，北京认为未来是数字优先连接的人口。

无论您的政治如何，西方和东方资本都相信元宇宙。

玩家团结

根据 Statista 的数据，2020 年有 25.5 亿人玩过某种电子游戏。平均而言，游戏玩家每天花 54 分钟或每周 6.33 小时来做他们的游戏。

游戏玩家有 25.5 亿人，是最大的附属人群。他们跨越传统民族和宗教。一个有趣的数据点——我没有——是游戏玩家是否对游戏中的社区或他们的出生国/宗教信仰有更强的归属感。我敢打赌，随着游戏玩家在他们的虚拟世界中花费更多时间，对一个民族国家或宗教的亲和力逐渐减弱，而有利于游戏中的社区。

游戏不仅仅是射击游戏。我认为它本质上是虚拟世界中的一些社交活动。该构造将变形为虚拟世界中的任何活动。如果我们考虑人类现在在 Zoom 和 Microsoft Teams（它们只是虚拟工作空间）上花费的时间，社交/工作所花费的时间会高得多。

人们喜欢这些新世界 。在家工作 (WFH) 与在办公楼内工作的偏好在代际和收入水平之间划定了界限。婴儿潮一代的老板希望看到他们更年轻、更贫穷的动产坐在办公室里，在那里他们可以受到监管。劳动力，如果可以在线完成，宁愿坐在自己舒适的家中的运动休闲区，并从不再需要通勤的额外休闲时间中受益。随着我们在完全虚拟的环境中社会发展以形成牢固的联系，通过面对面会议来达成交易的概念将变得不合时宜。

如何弯曲

作为社会人，许多活动和购买的唯一目的是公开展示您可以浪费多少能量。夜总会经济是这个概念的极好提议。个人走进一个黑暗的房间，听嘈杂的音乐（艺术），跳舞（浪费能量，类似于求偶），并支付过高的钱喝液体。每个人都穿着非常漂亮的衣服，这些衣服除了向在场的其他俱乐部成员展示他们花了很多钱来展示他们的社会地位之外，没有任何用处。

如果你认为夜总会太俗气，那么全球艺术展怎么样？有钱有名的艺术爱好者、创作者和策展人浪费精力前往一处。他们聚集在一起“收集”无用的绘画、雕塑和其他装置。根据你代表的画廊和/或你囤积了多少无用的东西，有一个明确的社会啄食顺序。提供食物和饮料以与其他志同道合的开明艺术爱好者社交，完成后，每个人都收拾行装，浪费更多精力返回家园。

社会化和交流是人类与众不同的原因。这就是我们为我们的众神和统治者建造纪念碑的方式。我们就是这样把人送上月球的。这就是我们创建互联网和集成电路的方式。弯曲既是一种 100% 的能源浪费，也是建立社会纽带以促进文明进步的必要条件。

当我们想象元宇宙会变成什么样子时，它的居民将如何改变他们的社会地位？回到游戏，围绕您的在线头像所穿的皮肤已经蓬勃发展的经济指向代表大量价值的毫无价值的艺术数字对象的数字未来。

DMarket估计皮肤市场约为 $40B。

Newzoo 还对美国游戏玩家对皮肤交易的态度进行了报告，颇有见解。

报告要点

  • “在了解皮肤的美国核心游戏玩家中，81% 的人希望为他们的皮肤获得现实世界的金钱，因此对皮肤交易感兴趣。超过三分之二的人不知道美国有任何主要的皮肤交易平台（包括 Steam 的社区市场、DMarket、OpenSea 和 Bitskins）。”

  • “至关重要的是，75% 对皮肤交易感兴趣的玩家表示，如果他们在游戏之外拥有货币价值，他们会在皮肤上花费更多。”

  • 从人口统计来看，年轻男性最有可能接触游戏化妆品，但大手笔的鲸鱼年龄较大。

我们的在线头像需要灵活处理。与我们独特的在线身份相结合的数字物品的价值将呈爆炸式增长。问问自己，顽固的游戏玩家会愿意花 1,000 美元购买稀有皮肤，还是 1,000 美元购买路易威登包（您可能会问，“什么 LV 包只需 1,000 美元”）？这两个项目的目的相同。它们显示了您可以负担得起因富裕而浪费资源的程度，它们也是稀缺品。

Flex Good

您弯曲的商品应具有以下一些特征：

1.  好东西在本质上必须毫无价值，或者如果它像一件衣服一样有用途，那么有更便宜的替代品可以实现相同的功能。

2. 如有疑问，请检查该项目是否由流行歌星在歌词中提及。如果为真，那么它可能是 Flex Good 并且本质上已经破产。

3. 拥有物品应授予专属社区的成员资格。

4. 货物应该是数量有限的，换句话说，是稀缺的。

你有没有想过穿袜子运动鞋是什么感觉？好吧，Balenciaga 会以接近 1,000 美元的价格卖给你一双。

这是 Flex 好吗？让我们来看看清单。

属性 1：

您可以花更少的钱买一双运动鞋，以保护您的双脚，同时进行人类称之为步行的生物力学活动。是的，一位名叫 Cardi B 的说唱歌手对上述运动鞋有以下看法：

“我喜欢那些 Balenciagas，那些看起来像袜子的”。

属性 2：

如果您购买这些鞋子，则表明您是一个欣赏高级时装并负担得起 Balenciaga 品牌的小社区的一员。

属性 3：

这双鞋是半稀缺的。Balenciaga 生产了足够多的一双，任何人只要有 1,000 美元就可以购买一双。然而，它们在货币方面是稀缺的，因为大多数人负担不起在袜子运动鞋上浪费 1,000 美元。

Crypto Flexing——支持 NFT 的艺术

NFT，或不可替代的代币，目前只是所有权记录。NFT 告诉我们哪个加密公钥是特定数据的真正所有者。就目前的市场而言，参考艺术本身很容易复制，但原始所有权在密码学上是独一无二的。

由于当今的 NFT 艺术大多采用数字图像和视频的形式，反对者对为什么有人会为可以免费复制的数据支付金钱（能源）感到傻眼。无需购买任何东西即可享受互联艺术。也可以免费转让。然而，这些人没有理解的是，艺术本身与 Flex 相切。这是关于拥有加密唯一证书……NFT。

实物艺术品之所以有价值，是因为艺术品本身不能同时出现在两个地方。卢浮宫内只有一幅《蒙娜丽莎》。像素化岩石的 jpeg 可以同时出现在每个地方。那么为什么两者的值都大于零呢？

在 Metaverse 中，仅“佩戴”或“展示”您的数字项目是不够的。你还必须能够证明它的稀缺性。这可以通过拥有 NFT 以加密方式完成。

您的加密 Flex 有两个方面。首先，你有一双看起来很糟糕的虚拟运动鞋、可爱的企鹅、像素化的面孔或 jpeg 摇滚，你可以在各种社交平台上显示或作为你的头像。看看有多少加密人通过将他们的头像与 NFT 相关联的图像来表明他们在精英社区中的价值和成员资格。最后，也是最重要的一点，在因为它是你的 Twitter 头像而拥有一个 +1,000 ETH 价值的加密朋克之后，你实际上可以证明你在以太坊区块链上以加密方式拥有它。现在每个人都知道你为此付出了多少——你不必大喊，“嘿，世界，我他妈的很富有。” 我们人类喜欢将我们在商品中的财富展示抽象化，而不是通过直接显示我们的银行余额或链上持有的财富来过于新潮。

Ether Rocks 是一个有趣的 NFT 项目，用于评估与检查清单 。

属性 1：

EtherRock 实际上是一组摇滚 PNG 图片。我不会说它们在实现人类创造力的方式上花费了太多。它们本质上毫无价值。

属性 2：

如果您拥有 EtherRock，您就是一个不超过 100 人的专属社区的一部分。如果您愿意，您可以购买多个 EtherRock，但永远只会有 100 个。

属性 3：

EtherRocks 非常稀缺，因为只有 100 个铸造。

这是一个有趣的项目，因为 PNG 文件显然毫无价值且缺乏创造力，以至于引发许多人质疑为什么有人会浪费大量的 Ether 来购买它们。事实上，仅在 8 月份就交易了价值近 2,000 ETH 的岩石。但 NFT 越明显越不值钱和越昂贵，Flex 就越大。与所有事物一样，价值可能会波动。想象一下，拥有一块曾经价值很多 ETH 的岩石，现在一文不值。价值波动也增加了 Flex，因为它意味着所有者非常高兴他们的 Rock 在下一个以太坊区块的传播中价值 0。

交易 NFT

就像传统艺术一样，一个健康的投机者生态系统将会并且已经被 NFT 所吸引。在艺术界，他们称他们为经销商或收藏家，但他们在赌哪些毫无价值的东西将来会被人类“珍视”。随着时间的推移，做好并盈利是一件非常困难的事情。人们如何选择支持哪些当代艺术家，以便他们的作品集随着时间的推移继续升值？

在 NFT 领域同样困难。你怎么知道哪个项目的艺术作品会流行？铸造的物品数量应该非常稀少吗？它应该是一个旧项目，其中“年龄”是与比特币创世块的接近程度吗？对于特定的有钱人来说，它应该在美学上令人愉悦吗？它应该与流行的视频游戏直接相关吗？有很多问题，没有正确或错误的答案。但是会存在一群精通叙事构建和模因传播的交易员。他们将能够收集最受追捧的 NFT，并在此过程中获得可观的回报。

艺术界雇佣了成千上万的人，他们的目的是验证和验证待售作品的来源和真实性。考虑到它是由易犯错误且有时会发生冲突的人完成的，艺术世界中假货比比皆是。对于在公共区块链上交易的 NFT，这是不可能的。当您在 OpenSea 上购买某物时，您 100% 确定您购买的是 Number Every 特定项目的任何东西。

解雇他们有危险

扭曲是人类体验不可或缺的一部分。我们不质疑用于展示社会地位的物理人类空间物品的价值。我们了解并重视时尚、绘画、珠宝等。我们在工作中都穿着能说明我们属于哪个专业社区的服装。没有爱马仕领带或红底鞋的投资银行家算什么？服装是自我价值的一部分。

仅仅因为机器人接管了我们人类工作，并不意味着人类在元宇宙中不再是人类。社交信号将采用新形式，由支持区块链的 NFT“无价值”对象提供支持。那些认识到这些相似之处并尽早为数字 Flex Goods 创建新市场的人将获得天文数字的回报。那些对这种新的毫无价值的社交信号表示不满的内容可以继续走在街上，走进一家商店，并从一些营销良好的时装店购买价值 500 美元的白色 T 恤。适当地选择您的 Flex Good。
