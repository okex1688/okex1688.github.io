Web3是今年最具代表性的话题。

除了全球大量相关Web3融资消息的涌入，之前虎嗅发表的《Web3革命：逃离、信仰、大迁徙》中还甚至透露出了在国内不少互联网大厂的技术大神、工程师接连出走，投身Web3的热潮当中，吸引了一波又一波眼球。

根据业内人士的透露，Web3已超越硬科技和消费品，成为了整个投资圈公认的年度热门赛道。除此之外，Web3还被普遍认为是打破传统互联网平台经济格局的新兴力量，还有区块链用户都共同认知的一点—Web3就是互联网的未来雏形

那么，在Web3热潮风投正盛的当下，应该关注哪些必要的风口呢？

Web3是否正在渐行渐近？

在进入正题之前，相信很多人都带着同样的疑问—Web3是否真的正在渐行渐近？对于一个行业拥有足够的了解才能真正的找到投机点，那么接下来我们先把关注点放在这里，先了解清楚是否Web3正在渐行渐近。

对于这个问题，中国证监会科技监管局局长—姚前给出了很好的答案，他在《中国金融》撰文，结合国内外互联网发展实践和技术演变趋势，分析Web3的可能形态并进行相关思考。文章表示，Web3有望大幅改进现有的互联网生态系统，有效解决Web2时代存在的垄断、隐私保护缺失、算法作恶等问题，使互联网更加开放、普惠和安全，向更高阶的可信互联网、价值互联网、智能互联网、全系互联网创新发展。

并从建设高质量的分布式基础设施、推动治理良好的技术创新、建立通用标准，增进互操作性、建立清晰、公平的税收规则和建立针对DAO的法律框架这几个维度解析，Web3是渐行渐近的新一代互联网，或将引发新一轮的资讯革命，进一步深刻改变人们的生活、工作以及社会的方方面面。

Web3的融资动向

事实上，全球知名的风投机构早已开始关注Web3。尤其是以a16z为代表的全球顶级VC，一直在布局Web3、元宇宙、区块链等前沿领域。

在2018年的博客中，a16z就着重提到了Web3的重要性，声称该领域提供了一种全新的方式—它结合了Web2时代之前的所有最佳方面，并认为当时乃至现在（在最新的一篇加密报告中）还是这场运动的早期阶段，是参与其中的好时机，要积极的建设Web3领域。

不仅是a16z，在过去的两年时间内红杉、谷歌、Meta等知名机构和企业也纷纷投入巨资，切入新兴的Web3赛道，相关融资事件频出。近期红杉资本更是豪掷28.5亿美金成立新的Web3投研基金，进一步扩大对于Web3领域的投资范围。

Cryptopolitan数据显示，2022年第一季度，Web3.0初创企业已获得超1.73亿美元的投资，而2020年全年的融资数据仅为2,000万美元，由此可见VC们对Web3领域是多么的看好。

在Web2时代存在垄断的当下，Web3描绘的世界领所有人向往。

Web3 ≥ 区块链

Web3即Web3.0，指的是下一代互联网概念或模式。然而，不同层面对于Web3有不同的理解。

简单而言，Web3整合了Web1和Web2各自的特点，既实现了信息的分享，也具备交互的特点，更重要的是用户才是价值或相应资产的持有者，但Web3进一步突出了去中心化、开放、可信。

由此可以得知，Web3与区块链的概念类似。但现在大家比较认同的一种说法是，Web3是基于区块链技术的万维网新迭代的想法，它结合了去中心化和基于Token的经济学等概念。换句话来说，Web3就是基于区块链技术和理念的全新互联网模式，同时区块链其实就是Web3的底层技术或核心构架。

但是真的是这样的吗？

事实上，Web3不仅具备区块链的技术特征，也是互联网技术与区块链技术的融合，但两者的构架并不相通。虽然与区块链的经典应用重合，但Web3毕竟是全新的互联网模式，与偏于技术的区块链仍有明显的差异。此外，Web3的协议层包含了区块链中至关重要的智能合约、算法机制、激励机制、共识算法等组成部分。而基础设施则是其中最底层的数据结构、存储及计算，是整个Web3层级中最重要的基础和核心。

所以Web3并不等同于区块链，其既包含区块链技术，也离不开与传统互联网的互通和融合。

We3浪潮下，该关注哪些风口？

风口在哪？风投机构会准确的告诉你。只要你关注每日快讯，就能够清楚的知道每天风投机构的主要融资集中在哪。今年Web3领域融资依旧集中在NFT、游戏、社交、基础设施及金融五大板块，这些也是目前区块链中用户量最大的应用场景，以下是浅谈这几个板块为什么能够成为VC眼中的投机板块。

Web3 x NFT

随着互联网Web3的热潮兴起，NFT也随之火热，而Web2和Web3的世界在未来几年内融合在一起，显示出了链上的大众市场潜力。人们开始发现无法抗拒快速崛起的Web3数字市场。因此，尝试Web3数字产品的公司、企业家、艺术家和影响着的数量猛增。

现阶段，Web3获得全球认可并被大多数企业采用，围绕NFT的创新也将在这个新互联网中发挥关键作用，它也在慢慢改变核心领域的去中心化互联网的未来，是真正去中心化未来的工具之一，应用于包括虚拟房地产、内容创作、游戏、在线社区、音乐会和其他有趣的领域。

Web3 x 游戏

Web3游戏同时是休闲和金融工程的途径，激励因素不再仅仅是社区、娱乐或消遣，能够获取改变生活的资金或生计现在也融入其中。这改变了以往传统游戏的动态以及用户与游戏的交互方式。

Web3游戏被认可的另一原因是，它们解锁了完全不同的经济行为主题—交易者、投机者和公会可以帮助创造对Web3游戏发行商和玩家都有利的自我维持的生态系统。

Web3 x 社交

Web3社交赛道，可以说是所有具备与社交相关的Web3产品总称，是一个很宽泛、但比较容易理解的概念。Web3中常常被提及“创作者经济”和“SocialFi”，就是我们能够在Web3中创建价值人设的重要途径——“创作型”身份和“社交型”身份更容易创建，也更容易变现。

但目前，Web3社交是一个依然处于萌芽期的赛道，这样的经济需要建立在真实可信的、去中心化的、保护隐私且可自主管理的身份之下。并且在这样的模式中，普罗大众将能够释放自己的创作力，凭作品或者链上贡献创造社交影响力，从而收获更大的价值回报。

Web3 x 基础设施

基础设施是Web3的基石，虽然它目前在大规模应用上面临着一些挑战，也可能不会完全取代中心化基础设施，但它正在推动新的对话并获得大量投资者的兴趣，它的未来具有无限的潜力。

Web3 x 金融

Web3生态的稳健和安全性需要依赖像货币、区块链、DAO这样的基础。而为了维持市场的活性和生长潜力，另外一种“生长激素”也是不可或缺的，那就是让资产充分流动的金融体系。Web3正是要让用户享受自己财产所带来的收益。因为只有收益权，才真正体现了拥有权。

随着区块链和Web3的不断发展和应用，未来全球传统科技和互联网巨头们将会遭遇一大批难缠的新对手，以及更激烈的竞争。届时，传统互联网垄断和割裂的竞争模式将被打破，让用户真正拥有互联网的价值和红利，构建起全新的互联网模式和格局。
