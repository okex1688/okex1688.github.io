NFT 将渗透到我们的生活中

在 2021 年经历了巨大的突破之后，很明显，在消费内容和与应用程序（包括数码照片、图像、游戏、音乐、视频）以及可能在其他领域（如房地产），NFT 用例正在增长。

The Block 报道称，“流行音乐节 Coachella 最近与加密货币交易所 FTX US 合作推出了 NFT 系列。NFT 被称为 Coachella Collectibles，将为消费者提供解锁节日通行证、艺术印刷品、相册、数字收藏品、节日独特的现实生活体验和实体商品的机会。”

同样，NFL 宣布将为参加超级碗 LVI 的球迷提供“NFT 形式的虚拟纪念门票”。

因此，我们需要了解 NFT，并找到投资这一数字资产子类的方法。

什么是 NFT？

Coingecko 将 NFT 定义为“一种代表独特事物的数字所有权形式：照片、图像、游戏工件、音乐、视频等。NFT 利用创新的智能合约技术在区块链上存储和记录独特信息，这意味着每当 NFT被创建，只有其中一个可验证地存在。使用 NFT，人们可以以数字方式证明资产是真实的。NFT 可以在专门的市场上交易。”

资料来源：OpenSea.io 上的 NFT 市场

不断增长的用例

到目前为止， NFT 可以归纳为以下四大类：视觉艺术、个人资料图片项目（PFP：Cryptopunks、Bored Ape Yacht Club、CrypToadz）、文化收藏品（体育、音乐、照片、视频）和游戏。

NFT 为数字艺术家提供了一种通过他们的工作和才能获利的新方式。据 Cryptoart.io 称，通过区块链技术对原创作品的轻松认证使数字艺术家能够在 2021 年在以太坊上出售高价艺术品。

在游戏领域，动视暴雪（纳斯达克股票代码：ATVI）、Epic Games 和电子艺界（纳斯达克股票代码：EA）等公司通过购买或赚取游戏内物品的玩家赚取了数十亿美元，但玩家无法出售这些物品。

借助 NFT，玩家可以将他们在游戏中的成就转化为金钱，因为他们可以交易 NFT。Axie Infinity (AXS-USD) 是领先的游戏赚钱游戏平台的一个例子，玩家还可以在其中赚取和交易 NFT。

期待看到 NFT 被整合到游戏中。Ubisoft (OTCPK:UBSFY)已经将它们集成到 Ghost Recon Breakpoint 中，其他人可能会效仿。这可能会为游戏公司带来一些额外的增长。

截至 2021 年底，在 OpenSea 上交易的 NFT 集合超过 6,000 个，同比增长 20 倍，但大部分交易量集中在少数几个最受欢迎的集合中（见下图）。大约 250 个收藏品占二级 NFT 销售额的 80%。

CryptoPunks 成立于 2017 年，甚至早于 NFT 热潮，并且一直是领先的系列，直到最近才被 Bored Ape Yacht Club 超越。

NFT 是一种全球现象

与比特币 (BTC-USD) 和其他加密货币非常相似，NFT 由全球投资者交易。

2021 年是 NFT 标志性的一年

NFT 活动在 2021 年爆发式增长，并出现了主流采用的迹象。如果说 2020 年是 DeFi 年，那么 2021 年就是 NFT 年。

2021 年，NFT 总交易量达到 19.6B 美元，高于 2020 年的 8600 万美元。2021 年约有 200 万 NFT 交易者活跃。

2021 年夏季之后，炒作似乎已经结束，市场上的 NFT 收藏变得过饱和，活动从 9 月的每日交易量峰值 3.25 亿美元急剧下降到 2021 年 11 月的每日 5000 万美元。

然而，NFT 在 2021 年底和 2022 年初迎来了第二次风潮，销量再次回升。

我们使用 OpenSea 的交易量作为 NFT 市场的指标，因为它在 NFT 市场中拥有 90% 以上的市场份额。

竞争格局

OpenSea 拥有最大的市场份额，直到LooksRare (LOOKS-USD)于 1 月 10日推出“吸血鬼攻击”。

最大的 NFT 市场包括：

  • OpenSea – 2022 年 1 月在 C 轮融资中以 13.3B 美元的估值筹集了 3 亿美元

  • LooksRare – OpenSea 的直接竞争对手

  • SuperRare (RARE-USD) – 专注于高端视觉艺术

  • 基金会——许多文化资产类别

  • Rarible (RARI-USD) – 收藏品和艺术品

LooksRare 的“吸血鬼攻击”指的是一种试图引诱交易者和卖家远离 OpenSea 的市场战略。从本质上讲，LooksRare 为在其市场上列出和交易 NFT 的 NFT 交易者和卖家提供了丰厚的奖励。

这些交易者可以赚取 LOOKS 和 Weth (WETH-USD) 代币，目前可产生高达 400-600% 的年利率。显然，这是不可持续的，预计会下降。

LooksRare 对所有交易收取 2% 的费用（比 OpenSea 低 0.5%），但其中 100% 的费用用于质押 LOOKs 代币，这也使其比 OpenSea 更具吸引力（不分摊费用）。

就 LooksRare 市场的交易量而言，“吸血鬼攻击”似乎正在发挥作用，尽管也有一些洗牌交易正在进行。

LooksRare 的长期成功将取决于一旦当前的交易者激励减弱，它从 OpenSea 夺取市场份额的成功程度。

2022 年 NFT 的前景如何？

NFT 销量在 2022 年开局强劲，1 月份销量创下历史新高。这样的数量将吸引更多的玩家。

加密货币交易所是天作之合：Binance (seekingalpha.com/symbol/BNB-USDBNB-USD)、ByBit 和 Coinbase (COIN) 已经启动或宣布了启动 NFT 市场的计划。

截至 2021 年第四季度，Coinbase 拥有 8000 万用户，可能会成为一个巨大的参与者。事实上，他们的市场等待名单在短短两天内就达到了 150 万，超过了 OpenSea的活跃用户。

这些交易所还将提供法定入口，这将使加密生态系统之外的人们更容易购买 NFT。例如，Coinbase 将允许用户通过万事达卡（纽约证券交易所代码：MA）购买带有法定货币的 NFT ，这也应该有助于新用户的入职。

此外，如前所述，游戏公司可能会将 NFT 集成到他们的游戏中，从而使数千万玩家接触到 NFT。GameStop (GME) 计划在今年年底前为游戏内 NFT 推出 NFT 市场。

更多消费品牌和名人将推出 NFT 项目，为品类带来更多主流关注。

NFT 的使用将超越视觉艺术，发展为音乐、虚拟房地产、游戏中的人工制品，这将极大地扩展该类别。

投资者如何参与 NFT 增长？

仍在加密交易生态系统之外的投资者可以通过 Coinbase、GameStop 等代理参与，或者等待 OpenSea 的首次公开募股，这暗示了最近的首次公开募股。

玩 NFT 的其他可能方式是通过 Meta （纳斯达克股票代码：FB）和 Twitter （纽约证券交易所股票代码：TWTR），它们已宣布计划让其用户为个人资料图片选择 NFT。

在利用 NFT 时，这可能只是这些 Web 2.0 社交媒体巨头的冰山一角。这也是 Meta 的元宇宙计划的自然延伸，该公司每年向其中投资 10B 美元的虚拟世界。

加密原生投资者可以

  •  在领先市场之一上交易 NFT 收藏品

  • 投资于此类市场的代币：LooksRare、SuperRare、Rarible

  • 投资于一些 NFT 基础设施提供商，如 Immutable X (IMX- USD)、Wordwide Asset eXchange (WAX-USD) 和 Loopring (LRC-USD)

投资风险

处于全新资产子类别的早期阶段令人兴奋。提早有可能获得巨大收益，但也使投资者面临高风险。投资者应考虑将赌注分散在一篮子与 NFT 相关的投资理念中，以捕捉整体趋势，同时将非系统性的公司特定风险降至最低。

  • 法律和版权问题。关于艺术家是否应该从二级市场销售中获得许可费或版权费存在争议。创建看起来与原件非常相似的新 NFT 集合也很容易。

  • 竞争可能会升温。正如 LooksRare 与 OpenSea 所证明的那样，新 NFT 市场的进入门槛相对较低。我们预计今年会有很多新品上市，这可能会影响他们的收费率。

  • 分散的 NFT 市场在智能合约（基于程序代码）上运行，这些合约可能经过审计，也可能未经审计，但即使经过审计，也可能面临被黑客入侵的风险。

  • 有“内幕交易指控”，即使是在领先的市场（OpenSea），他们自己的高管也是如此。
