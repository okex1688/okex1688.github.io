PolkaWorld 在前段时间发布了一篇关于判断项目是否合法 必须满足的一些硬指标，今天是这个系列文章的下篇，除了硬指标，还需要考虑结合一些 “ 软指标 ” 来进行判断。

“软”指标

“硬指标”是你在评估项目时首先应该考虑的因素，并且应该在你的决定中占据大部分 权重。原因是它们很难伪造或操纵，当然，前提是你能够验证找到的信息。因此，我们称它们为“硬”指标。

但是还有其他一些因素需要注意，他们也可以用来判断项目的合法性，但更容易被人为操纵，所以这些不应该 严重影响你的决定。这些被称为“软”指标。

1. 网站质量

项目网站的质量有时候可以提供对项目合法性的判断。一个结构糟糕的网站，有拼写错误和语法错误，或者样式很差，一个没有认真改进或更改的模板网站，一个网站几乎没有关于项目的信息，没有指向他们的 GitHub 或其他资源的链接，通常你感觉一个网站不专业 ，这表明团队对这个项目可能并不认真 。

但这并不意味着所有精心设计的网站都是可靠的项目。毕竟，这是一个软指标。许多没有任何实质性计划的项目也拥有出色的，甚至是外观漂亮的网站，他们将精力用在如何提高网站的视觉来对社区进行误导。所以，一个优秀的网站并不一定代表一个合法的项目，一个很垃圾的网站也可能代表一个不合法的项目，仅凭网站的质量通常不足以 得出结论。

2. 社交媒体

拥有一个充满活力的社区是一个合法项目的良好标志。与社区互动的团队提供更新、回答问题、举行 AMA、发布文章，是一个有意愿让社区成员参与和了解情况的团队。

但与此同时，社交媒体的存在和参与很容易被伪造和操纵。创建一个 Telegram 群组或一个 Discord 群并用数千个机器人 填充它非常容易。尽管机器人需要根据 Discord 的条款在 Discord 上进行识别，但诈骗者并不在乎这些条款和条件。

一个每天发 5 次推文或每隔一天发布一篇 Medium 文章的团队不一定是在构建实质性的东西。

因此，请验证他们的社交媒体的真实性 。加入他们的频道，提出问题并直接了解社区和管理员的情况。如果你看到很多用户发布非常简短的评论，例如“ 好项目 ”、“ To the moon ”、“ 谢谢 ”等，但并没有真正吸引人，请保持怀疑并仔细观察，因为这些可能是机器人的评论 。此外，验证团队在社交媒体上分享的任何信息，并检查其他人在说什么。但是也要验证这些评论，无论它们是正面的还是负面的。

3. 媒体内容

验证社交媒体的真实性是媒体内容。第三方文章、YouTube 视频中的提及以及项目的一般宣传。

说到文章，首先要检查的是文章是真实报道还是付费新闻稿，尤其是当一个项目将此报道放在页面上的显着位置时。或者作者是否有任何既得利益来帮助营销项目。你可以查看这些作者之前的文章，看看他们是否一直在为这个项目进行营销服务。

同样的事情，也适用于一般的 YouTube 用户和一些在 YouTube 上有影响力的人。他们中的许多人这样做是为了谋生或作为“抽成”项目的一种方式。寻找那些尽可能提供客观信息 的优秀作者和 KOL，通常他们输出自己单独研究的成果。

这并不是说媒体曝光是一件不好的事情。它可能是项目本身之外最丰富的信息来源，但同时，它需要大量的信息去交叉检查和验证。

4.邮件沟通

如今，许多项目使用 Telegram、Discord 或类似应用程序进行社区发展，也作为唯一的沟通和支持渠道。但是，如果他们有一个他们域名的邮箱，这除了提供另一个通信渠道之外，还可以被视为一个额外的可信度标准。

此外，从项目的域名接收电子邮件可以轻松验证通信的真实性（但要注意欺骗性电子邮件！）。另一方面，通过个人电子邮件或使用公共电子邮件（如谷歌或雅虎）进行交流，可能会表示这是一个不太严肃的团队或一个比较分散，并且一个可能还在项目早期阶段的团队。

Crowdloans

随着最近在 Kusama 上推出平行链，许多旨在成为平行链的项目发起了 crowdloan，以收集必要的资金参与平行链 Auction。但是随着 Kusama 和 Polkadot 平行链的发布，许多骗局 也可能浮出水面。因此，crowdloan 可以确保你安全参与。

首先，只有旨在成为平行链的项目才应该有 crowloan。如果一个项目不是平行链候选者，则不会有与之相关的 crowloan。

参与平行链 crowloan 的最可靠方法是通过 Polkadot-JS 应用程序上的 crowloan 模块。这个模块会发出一个特殊的外在信息，在平行链插槽租赁期结束之前锁定你的资金，并保证在那之后将你的质押返还 。

许多平行链候选者也提供了一种通过他们的网站参与的方式。但是你应该确保他们在后台使用 crowloan 模块，并且他们只是将其包装在一个更用户友好的用户界面中。如果相反，他们的 crowloan 接口将资金转移到一个帐户，这些资金将完全在他们的控制之下，这意味着你需要完全相信团队的确会使用这些资金进行 crowloan，并在租赁期结束时将你的质押返还给你。或者如果他们没有赢得插槽，他们会妥善保护它们。如果他们正在做类似的事情，应该在他们的网站和文档中明确提及。

话虽如此，一些团队一直在进行 token 销售或交易，试图在为 auction 筹集资金方面取得先机，但这不是 crowdloan，仍然需要对团队的完全信任。因此，如果你打算参与这些 token 交换，请确保该项目信誉良好，并且你通过其官方网站和社交媒体渠道获得了正确的信息。

同样，一些中心化交易所也正在创建通过他们的平台参与 crowdloan 的方式，还有一些钱包也正在将 crowdloan 功能集成到他们的应用程序中。肯定的是会有更多人跟进。任何合法的努力都应该在后台使用原生的 crowdloan 模块并提供更简化的用户体验。在使用这些服务之前确保这种情况是必要的，但无论如何，它需要你信任交易所或服务提供商。

最后一条建议

一旦你阅读了所有这些材料并完成了研究并确定了一个项目是合法的，那么你需要确保的另一件事是，你需要了解该项目的用途 以及它可以为生态系统带来什么新事物 。

这不属于事实检查和验证声明的范畴，但这需要提到：充分了解某事的作用及其预期影响是做出明智决定不可或缺的一部分，因此不要忽视它。
