最近，在美国呼吁对俄罗斯实施更严格的制裁后，加密/Web3 平台或公司一直在禁止俄罗斯人使用他们的服务，这些行为虽然可以被认为是合理的，但区块链和 Web3 领域是否真正去中心化、是否具有真正的抗审查性也让人们在心里打了个问号。

由于俄罗斯对乌克兰发起入侵，美国等西方国家一直在对俄罗斯及其公民实施经济制裁。这些制裁实际上将俄罗斯从全球经济中剔除，旨在孤立该国并使普京发起的战争最终无利可图。

作为制裁的一部分，美国切断了俄罗斯寡头和企业与美国打交道的各个机会。两家著名的加密货币交易所 Coinbase 和 Binance 虽同意遵守制裁，但并没有采取进一步措施。一些人呼吁平台禁止所有俄罗斯用户使用他们的服务，但 Coinbase 和 Binance 拒绝了，理由是每个人都应该获得金融服务。话虽如此，如果在不久的将来监管变得更加严格，双方都可能遵守美国的制裁。

另外两个 Web3 平台 MetaMask 和 OpenSea 采取了相反的方法，并禁止所有来自受美国制裁国家的帐户，包括伊朗、委内瑞拉和俄罗斯等。这些地区的 OpenSea 用户已经发现他们的作品和帐户被删除，生活在受制裁国家的 MetaMask 钱包用户不能再使用该钱包访问 Web3。

对许多人来说，这违反了 Web3 的基本原则：所有人完全平等和访问，没有审查或限制的可能性。这一系列事件揭露了 Web3 的残酷现实 —— 它离真正的去中心化还有一段路要走。

与任何公司一样，OpenSea 和 ConsenSys（MetaMask 母公司）应该遵守美国政府的法律条款，否则将面临严厉处罚。此外，MetaMask 用于与以太坊区块链交互的服务 Infura 基于 Amazon Web Services（AWS），也必须遵守其条款和条件。

Web3 的最大参与者可以说与 Web 2.0 一样中心化，这似乎是一个重大问题，这些参与者与科技巨头一样受政府监管，希望这种情况在未来会有所改变。

尽管 MetaMask 和 OpenSea 必须遵守制裁，但还有其他更去中心化的替代方案，不必遵守美国政府。这些替代方案存在于区块链上，在某些情况下没有关联的公司实体，这意味着如果这些服务托管俄罗斯或伊朗用户，没有人会受到惩罚。此外，随着区块链和加密货币变得更加主流，将有更多的钱包和交易所供人们选择，所有这些都将创造一个更加去中心化和抗审查的环境。

当前形式的 Web3 还没有为主流采用做好准备。它需要变得更加去中心化、更加用户友好和更易于访问，然后才能声称自己是下一代互联网。在达到这个目标之前，早期采用者可以继续采用它的用例并支持有助于使 Web3 真正去中心化的替代方案。
