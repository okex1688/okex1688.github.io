必须指出，由于 Chia Network 涉及到硬盘挖矿概念，市面上已经出现大量出售所谓「Chia 算力」的团队，真假难辨。Chia 团队在反复强：

什么是 Chia？

Chia 成立于 2017 年 8 月，旨在开发一个能够优化区块链和智能交易的平台。我们正在建立 Chia Network，以改善全球金融和支付系统。

Chia 将会成为第一个企业级数字货币。

Chia 正在使用一个被称为「时空证明」（Proof of Space and Time）的全新中本聪共识算法，它是由当前世界上最优秀的网络协议工程师、BitTorrent 发明者 Bram Cohen 创建的。

Chialisp 是 Chia 的新型智能交易编程语言，功能强大、易于审核且安全，当前可支持的参考智能交易包括：原子兑换、授权收款人、可恢复钱包、多重签名钱包和限价钱包。

什么是时空证明？

空间证明是一种加密技术，证明人会证明他们将未使用的硬盘驱动器空间分配给存储空间。不过，如果要将「空间证明」用作为一种共识机制，就必须将其与时间证明联系起来，时间证明可以确保区块之间的时间具有一致性，继而提高了区块链的整体安全性。

空间证明和时间证明如何工作？

空间证明，可以视为一种「证明你在硬盘驱动器上保留了一些未使用存储」的方式。Chia 区块链用户需要通过安装软件来快照自己在硬盘驱动器上未使用的空间，这种操作也被称为「播种」（seed），这个软件会在磁盘上一组加密数字存储到「地块」（plot）中，而执行这些操作的人被称为「农民」（farmer）。当区块链广播下一个区块挑战时，农民可以扫描他们的「地块」以查找是否具有最接近挑战的哈希值，Chia 会根据农民所拥有的空间大小和整个网络空间大小比例分配赢得一个区块的概率。

时间证明要求区块之间传递「必须」有一小段时间。时间证明是通过可验证延迟函数（VDF：Verifiable Delay Function）实现的，该函数需要花费一定的时间来计算，但验证速度其实非常快。可验证延迟函数的核心思想是顺序计算，由于大量并行设备不会产生任何收益，因此可以最大程度地减少电力浪费。VDF 服务器（「Timelords」）数量可能会相对较少，因为验证速度最快的 VDF 服务器将会始终排在最前面，并且网络上仅需要一个快速且公平的 Timelord 即可完成一个区块并将整条区块链向前推进。

什么是 Chialisp？

Chia 拥有一种全新开发的创新型区块链编程语言，即 Chialisp。这个区块链编程语言的功能十分强大、易于审核且十分安全。Chialisp 也是一种出色的链上智能交易开发环境，能够释放加密货币所承诺的安全性、透明性和易用性。

Chia 优势何在？

Chia 采用全新的中本聪共识算法，该算法可以彻底消除区块链系统工作量证明的能耗需求。与其他加密货币相比，Chia 区块链更加去中心化，安全性也得到大幅提高。Chialisp 是 Chia 区块链的全新智能交易编程语言，它功能强大、易于审核且十分安全，能够更好地释放加密货币所承诺的安全性、透明性和易用性。

此外，Chia 还采用了更先进的加密工具来实现丰富的智能交易功能。通过一家公开、开源、营利性开发公司，Chia 正在采用一种全新的高级解决方案来构建和支持区块链，并向区块提供支持。Chia 将使用「预耕作 pre-farm」（战略储备）来解决代币波动性问题，旨在减轻市场泡沫或市场崩盘对 Chia 带来的影响，同时进一步推动 Chia 采用。

Chia 的发展策略是什么？

我们相信，区块链行业仍由开发人员领导，当他们打算成为部署新应用程序和服务时，我们希望 Chia 是这些开发人员优先选择的区块链，并成为与主权货币挂钩的稳定币等应用程序的安全首选。

Chia Network 业务将是第一个管理预挖代币（pre-farm）的营利性公司，我们打算成为第一个可公开交易的「类 ETF」加密货币。

由于 Redhat 和 MySQL AB 是企业采用 Linux 和 mysql 的背后推手——我们认为，当主权国家、金融机构和企业在日常交易中开始使用加密货币，Chia 能够提供支持和培训。

最后，我们相信 Chia 将能够利用存储生态系统来推动企业和终端用户的采用，因为硬盘驱动器制造商和存储服务器销售商很可能将「存储空间耕作」（space farming）捆绑到他们的产品中。

更多细节，可以参阅 商业白皮书。

Chia 的耕作（Farming）与传统的挖矿（Mining）有什么区别？

挖矿需要昂贵的一次性硬件设备，这些设备往往会消耗大量电力。我们希望通过一个公平、环保且更优质的区块链来解决这一问题，Chia 区块链使用「耕作」让分布在全球节点上的现有空硬盘空间得到充分利用。「耕作」是去中心化的，因为任何安装了 Chia 软件并拥有「地块」（Plot）的人都可以赢得下一个区块。挖矿需要两个条件：第一，定制昂贵一次性挖矿硬；第二，获得批发价或更低价格的电力供应。所以，只有专业挖矿公司才有能力负担得起这些成本。「耕作」更加分散，因为它依赖于空闲硬盘空间，任何拥有移动电话、笔记本电脑或公司网络的人目前都有许多没有使用的额外存储空间。

与挖矿不同的是，一旦你完成了存储「耕作」，就可以重新调整存储用途，例如，存储你的家庭照片。

为什么要耕作 Chia？

您可以在手机、笔记本电脑或公司网络中的未使用存储空间上「耕作」Chia，作为回报，您有机会获得 chia 代表奖励以帮助保护区块链。我们的软件允许用户分配一定数量的未使用磁盘空间来创建「地块」。由于唯一资源密集型的一步是初始地块，因此一旦下载了 Chia 节点软件，驱动器将在后台挖出一个「地块」。「地块」挖完之后，您的计算机就会代表您进行耕作，软件会完成所有工作并为您跟踪代表奖励。值得一提的是，耕作只会占用很少一部分网络带宽，而且除存储外几乎不占用任何资源，此外耕作过程对任何未使用磁盘空间的人都可用，Chia 正在朝着实现真正去中心化的区块链的目标迈进，该区块链也将作为一种存储和云行业的交叉补贴形式。

在将 chia / XCH 列入数字货币交易所之前，耕种将是获得 chia / XCH 的唯一方法。

什么是 Chia「战略储备」？

在网络启动时，Chia 将进行「预耕作」并挖出大量代币，这样就能通过 Chia 借贷商业计划实现 Chia 经济更加稳定，也有助于今后发展。

我们的业务白皮书中提供了详细信息。

为什么要进行「预耕作」？

Chia 具有一套新颖的商业模式，既可以降低代币波动性，又可以提高采用率。通过向 Chia 借贷并管理这些贷款利率以及其他工具（例如，使用 Chia 代币购买我们的股票），我们希望代币季度波动率可以得到进一步降低。

为了进一步推动应用普及，我们打算把 Chia 借给全球 5000 家公司，这些公司将使用 Chia 向全球供应商提供快速、廉价、安全地付款。

我们还打算利用战略储备来协助开发和采用工作，还有其他一些事情，比如在 Chia 生态系统中投资一些有前途的初创公司，并在有限时间内增加耕作奖励以刺激其他耕作活动。如果企业使用 Chia 而非法定货币进行国际付款，那么可以获得 105% 转账金额「福利」。

Chia 会进行 ICO 吗？

不，Chia 没有 ICO 计划。相反，我们的目标是在美国证券交易所公开上市。这样，股东可以透明和公开的方式与管理层共享风险和回报，我们还可以利用充分理解的公司控制权，就 Chia Network 打算如何使用战略储备做出有约束力的声明。Chia 数字货币是一种有用的付款工具，但不是投资机会。Chia 打算完成符合监管规定的、通过 SEC 注册的 IPO 上市，并将在主网启动择机上市。

我现在可以买 Chia 吗？

现在还不行，一旦主网启动，我们预计六个星期之后才会在网络上分配耕种代币奖励，并且不允许进行代币交易，这么做的目的是为了提供一段时间来帮助稳定分配给 Chia 区块链的存储空间数量，并优先考虑我们的存储空间农民。

此外，发布的候选耕作软件将能够生成「地块」，这些地块可以在测试网和主网上进行耕作，只要主网启动代币交易之后，你就能在全球大多数交易所上购买 Chia 了。

Chia 的通货膨胀率是多少？

在 Chia 进行预耕作之后，将会以每 10 分钟提供 64 Chia 的速度发布耕作代币奖励。在最初的 12 年中，每三年结束之后耕作奖励将会减半。从第 13 年到无限期，代币奖励将保持不变，即每 10 分钟 4 Chia，这意味着通货膨胀率将会不断下降。在主网启动 22 年后，Chia 的通货膨胀率会下降到 0.50％。

Chia 是否使用了比特币代码？

没有。Chia 代码都是自己原创的。

比特币不够好吗？

当比特币被开发时，市场还没有预见到专用挖矿硬件设备的算力会大大超过普通计算机的性能，导致原本希望形成的去中心化网络被少数矿工控制，他们可以使用芯片制造厂和批发电力采购（或者更糟）。当前控制比特币网络的矿工们认为，他们仍然具有竞争优势，并且明确反对改变协议——即使协议很明显应该被改变。Chia 花了十年时间研究新的数字货币生态系统，并相信我们可以使加密货币更分散、更安全、更易于使用。

Chia 的价值会像其他加密货币一样波动吗？

我们相信，由于我们的计划是上市，因此代币波动性将比其他加密货币更小，这将使金融机构可以利用代币和股票实现对冲。

最初，我们预计 Chia Network 的估值将主要基于公司将在其资产负债表资产负债表上对 Chia 的估值，Chia 价格在数字交易平台上的波动很可能反映在证券交易所的股票价格变动上。与传统的区块链项目相比，有更多的方法可以让人们了解 Chia 的成功，这也将使我们股票的常规期权和衍生产品可用作类似 Chia 代币价格的合成衍生产品。此外，我们使用 Chia 战略储备的能力也可能有助于减少市场中 Chia 的波动性。

在应用加密技术方面，Chia 的水平大概处于什么位置？

Chia 创建了三项新的核心发明，并推动了市场兴趣和采用。首先，Chia 创建了首个用于生产的 BLS Signatures 库。其次，Chia 将成为可验证延迟函数（VDF）的第一个生产用例。第三，Chia 为中本聪共识创建了时空证明。

普通人能否将 Chia 用作付款工具？

我们希望是，随着时间的推移，Chia 能够得到全球 PoS 系统和消费者支付应用程序的支持。例如，如果您使用泰国的 GrabPay 应用程序在英格兰 Tully』s 咖啡店购买咖啡，那么它就可以「正常工作」，消费者无需知道交易是用 Chia 支付的。

再举个例子，如果您居住的地方无法使用诸如 Venmo 或 Cash App 之类的汇款服务，那么相比于速度缓慢且价格昂贵的电汇，使用 Chia 的麻烦就会少得多。使用传统银行业务进行跨境汇款和货币兑换即麻烦又缓慢，成本也很高，虽然替代性汇款服务可能更快，但它们并不便宜：西联汇款会收取高达 10％的汇款费用。

Chia 技术将完全开源并且可访问，任何人都可以在没有我们许可或帮助的情况下构建新的钱包。不仅如此，我们还会帮助钱包、交易所和商户合作伙伴，为他们提供支持和集成服务、联合营销和流动性贷款。

为什么选择 Chia 这个名字？

我们是环保的货币。Chia 创始人 Bram Cohen 创立公司之初就想通过「绿色」来降低区块链的能源依赖性。耕作概念似乎是填充未使用磁盘空间并对其进行监视以达到环保目标的最佳隐喻，这导致我们希望为新网络寻找一种能够体现环保特性的「农作物」名字。此外，团队也想要一个简短而有影响力的名字，最终每个人都感到很开心，因为 Chia 这个名字是一个与异想天开 meme 有关的「农作物」——Chia Network 就这么诞生了。
