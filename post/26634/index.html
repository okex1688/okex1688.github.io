7月22日14:00，N.Fans CMO大岛巖 做客金色微访谈第31期直播间，本次直播由金色财经花花主持，主题为“NFT+元宇宙，一时热点还是发展必然 ? ”。

直播过程中，嘉宾大岛巖 分享了万物 是否 皆可NFT、国内外大厂为何纷纷进军元宇宙、NFT+元宇宙会产生什么化学反应、NFT 的投资建议 等热点话题。

以下为本次直播的核心要点：

如果比特币、以太坊这些加密资产更是用作交易用途的代币，那么NFT更像是对应了现实世界中的“物品”概念，而元宇宙，这个建立在区块链之上的虚拟世界，去中心化平台让玩家享有所有权和自治权；通过沉浸式的体验，让虚拟进一步接近现实。NFT让人们看到无限的可能，如果说比特币带来了一场金融主权的革命，那么NFT带来了一场数字资产所有权的革命，这场革命也许会把整个区块链生态发展推到难以置信的高度和维度。NFT与区块链加密领域中主要解决于数字的稀缺性以及唯一性、数字产权、跨虚拟环境的大规模协调以及保护用户隐私的系统。万物皆可NFT，在“元宇宙”中，NFT带来了数字唯一性以及可验证性，彻底颠覆了如艺术品、收藏品、游戏领域等的一系列物品；也为“元宇宙”增添了重要的独立性、唯一性质。NFT 让 “元宇宙”以开放、无需信任的形式存在，且可实现去中心化所有权；从投资角度来说，NFT目前还处于较为早期，都在不断的尝试所以导致群魔乱舞，安全方面考虑的话首选资金实力强，团队实力强，有着自我见解给与赋能的NFT团队进行投资……

以下为本次直播详情（在不改变原意的情况下略有删减）：

主持人 ：首先请您介绍一下自己和N.Fans

大岛巖 ：Hello，各位小伙伴好，主持人好，很高兴做客金色财经直播间，我是N.Fans的CMO大岛巖，我曾从事过商业咨询顾问、专业投资人。日本大型银行和科技金融公司的创始人，在IT行业拥有12年行业经验。N.Fans是基于区块链的 NFT 艺术品与潮玩交易聚合性平台，平台聚集数字艺术、明星影视、动漫游戏等内容，利用智能合约技术构建集IP、产品、数字权益于一体的版权，为流行、娱乐等亚文化实现非同质通证化，并服务粉丝，帮助粉丝的NFT进行流通、确权行权等，以此构建一个面向全球参与者的交易平台。

主持人 ：谈谈您对NFT和元宇宙的理解和看法？

大岛巖 ：可以这么说，如果比特币，以太坊这些加密货币更是用作交易用途的货币、代币，那么NFT更像是对应了现实世界中的“物品”概念。而元宇宙，这个建立在区块链之上的虚拟世界，去中心化平台让玩家享有所有权和自治权。通过沉浸式的体验，让虚拟进一步接近现实。

主持人 ：今年3月，艺术家 Beeple 创作的数字画作NFT《Everydays: The First 5000 Days》以6934万美元的成交价刷新了NFT艺术品的价格记录，NFT也成为了区块链行业热点，艺术品、游戏、明星、音乐、体育等也发布自己的NFT，有种观点是“万物皆可NFT”，您如何看待这种观点？

大岛巖 ：像游戏、艺术品、域名、收藏品、虚拟资产、现实资产、身份等方面都有 NFT 的应用。NFT让人们看到无限的可能，如果说比特币带来了一场金融主权的革命，那么NFT带来了一场数字资产所有权的革命，这场革命也许会把整个区块链生态发展推到难以置信的高度和维度。如今，NFT已经不是一种「单纯」的数字资产，而是变成了一种所有权象征，有 NFT 意味着拥有了基础资产所有权主张 。我们选择盲盒的形式来迎合全球客户。鉴于全球用户对 NFT 越来越感兴趣，可以采用 盲盒更有效地基于本地消费行为。未来，我们还将考虑加入团购来推广我们的 NFT 艺术品， 这类似拼多多的电子商务模式。随着我们继续发布盲盒，我们将将价格保持在合理水平，以 吸引新客户进入 NFT 世界。我们还在盲盒中包含 NFS 代币以奖励客户。用户收集了一件艺术品的所有 NFT 片段后，他们可以使用我们的将片段融合为一个完 整的 NFT 并接收 NFS 代币的平台奖励。用户还可以兑换此 NFT 以获得艺术家的实物艺术 品，包含用户的独特签名。赎回后，相应的 NFT 状态将更改为 “已赎回”。用户还可以选择 保留 NFT 而不是接收物理艺术品来维护艺术品的二级市场的经济价值。

主持人 ：元宇宙概念提出于20世纪90年的小说《雪崩》，在这部小说中，人们初步感知元宇宙的“面容”，那么今年在NFT重提元宇宙概念的意义是什么？NFT+元宇宙会产生什么样的化学反应？

大岛巖 ：在我看来，NFT与区块链加密领域中主要解决于数字的稀缺性以及唯一性、数字产权、跨虚拟环境的大规模协调以及保护用户隐私的系统。万物皆可NFT，在“元宇宙”中，NFT带来了数字唯一性以及可验证性，彻底颠覆了如艺术品、收藏品、游戏领域等的一系列物品；也为“元宇宙”增添了重要的独立性、唯一性质。NFT 让 “元宇宙”以开放、无需信任的形式存在，且可实现去中心化所有权。

主持人 ：海内外动漫迷都是一个庞大的市场，NFT与动漫的结合是否会改变动漫市场？对于动漫发展起到了什么作用？带来了哪些影响？

大岛巖 ：目前各类元宇宙产品趋之若鹜，开发者开始融入多样的元素和理念。而我们N.Fans与动漫界大IP《火影忍者》官方达成战略合作，融合盲盒、3D NFT、NFT收藏品等当下流行的玩法。用户可以通过购买盲盒、燃烧通证和收集NFT，以通证持续通缩和收集 NFT 进入经济系统。这是个可循环的经济系统，依附火影忍者的背景以及丰富的人物塑造，通过社交通证构造出元宇宙。N.Fans是区块链历史上，第一次成功尝试动漫人物素材，并大受瞩目的元宇宙与NFT项目，未来N.Fans还会开拓更多的领域与更多的IP达成合作；使得N.Fans在元宇宙的场景中使用更加广泛。

主持人 ：目前市场上有很多NFT和元宇宙项目，普通大众如何参与和筛选优质项目？参与在其中能获得什么呢？您作为投资人有什么投资建议方便分享一下吗？

大岛巖 ：从投资角度来说，NFT目前还处于较为早期，都在不断的尝试所以导致群魔乱舞，安全方面考虑的话首选资金实力强，团队实力强，有着自我见解给与赋能的NFT团队进行投资。 在虚拟世界中，NFT确实是一个较为吸引投资者的目光。

NFT虽说目前只存在币圈玩家，但是NFT出圈的效果及其强大，还包括游戏圈、艺术圈、影视圈、音乐圈等都是潜在的NFT玩家。在币圈市场热度逐渐低迷而NFT的概念不断向着圈外扩散，越来越多知名度玩家加入NFT。用户可以通过购买盲盒、燃烧通证和收集NFT，以通证持续通缩和收集 NFT 进入经济系统。这是个可循环的经济系统，依附火影忍者的背景以及丰富的人物塑造，通过社交通证构造出元宇宙。N.Fans是区块链历史上，第一次成功尝试动漫人物素材，并大受瞩目的元宇宙与NFT项目，未来N.Fans还会开拓更多的领域与更多的IP达成合作；使得N.Fans在元宇宙的场景中使用更加广泛。

主持人 ：NFT和元宇宙的发展前景展望

大岛巖 ：N.Fans 是一个基于区块链技术的 NFT+DEFI 结合的生态聚合平台。基于区块链技术构建了一个集 NFT 创作、推广、消费支付资源管理等为一体的自由化、透明化、去中心化的文化娱乐生态体系。汇集了世界区块链、文化娱乐、计算机、数字资产、游戏等领域的顶尖人才，链接多方资源，有着高端创新的项目运营方式与资本投资思路。旨在打造一个撬动 NFT与非同质化现实世界的入口，平台通过构建标准化协议连通多元化的 NFT 资产，为所有的NFT 资产创造一个更完善的生态环境，成为区块链技术映射现实世界的基础设施。未来N.Fans还会开拓更多的领域与更多的IP达成合作；使得N.Fans在元宇宙的场景中使用更加广泛。

主持人 ：据说火影忍者监修完成，接下来的发展规划是怎样的？

大岛巖 ：关于这个问题呢，很高兴的告诉大家，现在火影忍者NFT作品3D漩涡鸣人，昨天已经送审完毕，将在近期重新上架币安NFT平台。后续我们还将上线暗界神使、宸灵纪、相信很快就能通过上市。大家可以关注我们的官方资讯获取更多的消息。
