以太坊正以多种方式进行扩展。

以太坊就像一座经济活动繁忙的超级城市 ，它全天候运行，跨越全球，允许任何人访问。不幸的是，以太坊一直受到其自身成功带来的附带影响：由于其高昂的交易成本，导致许多用户流失。但别担心，以太坊扩展已至，而且其扩展不仅仅是以人们经常谈论的 L2 (第二层) 的形式出现——以太坊正在许多不同的战线上进行扩展 ！

一个很好的类比是，我们可以将以太坊的垂直扩展 (即 L2s) 想象成一座城市如何通过建造高层建筑 来扩大规模。如果一座城市仅由 1 层高的建筑组成，那么该城市要么需要对第 1 层 (L1) 进行大规模水平扩展 (尽管这依然会受限且很昂贵)，要么将限制其增长，只满足那些住得起的人住在那里。这正是以太坊 L1 目前所处的情况，也是我们搭建 L2 可扩展性技术 (比如 Rollups 、Validiums  甚至侧链) 的原因所在。

在 L1 扩展方面，也就是水平扩展，以太坊将以数据分片 (data sharding) 的方式实现。数据分片将为 L2s 提供额外的容量 (就好比在新土地上建造更多的高层建筑)。因此，我们不仅能够建造足够多的高层建筑 (L2 容量) 来容纳 10,000 个人，还能建造更多的高楼来容纳数百万人。就像你可以优化一座高楼的容量和优化一座城市的布局一样，你可以对以太坊 L1 及其 L2 生态系统做同样的事情 。我们实际上正处于模块化的区块链范式 的开端。

还有一件事不常被提及，那就是以太坊如何扩展其社交层 ，或者换句话说，以太坊如何随着其社区越来越大而扩展。我认为以太坊通过拥抱去中心化 的精神来实现这一点，这意味着不是通过自上而下的方法来构建社区 —— 没有规则手册，没有指导方针，没有一套法律 (除了代码)，当然在以太坊城市中没有统治者；相反，以太坊的运作更像是一个「集市 」——正如 Ben Edgington 所说，以太坊这个集市「嘈杂且看似混乱，但非常高效和充满活力——而且，至关重要的是，(对于开源软件开发) 会产生更好的结果」。

如你所见，以太坊处于非常有利的位置，将在所有重要的领域进行扩展 。事实上，我认为对于加密网络而言，社交可扩展性 (social scalability) 是最重要的，因为没有它，加密网络实际上将是一个中心化的网络，仅由一小群人控制，而这正是我们试图通过区块链技术来打破的东西！

因此，让我们继续扩展以太坊这座奇妙的超级城市，并确保让其保持去中心化。

以太坊的黄金时代

作为一个从 2017 年初就一直在以太坊生态圈的人，我可以肯定地说，我从来没有像今天这样对以太坊的未来如此乐观。经过多年的努力，我们正在看到以太坊 L1 方面 ( EIP-1559 和其他主要的升级)、L2 方面 ( Rollups 正大步前进) 以及应用层方面 ( DeFi、NFTs、DAOs 等等) 的成果。我们正处于以太坊的黄金时代 。

于我而言，最让我兴奋的是，以太坊生态系统真的像是一个「不夜城 」。一天中无论什么时候都有事情发生，且你可以随时使用以太坊网络。很明显，这与互联网长期以来的运作方式非常相似，但我们从来没有一个始终开放 和去中心化 的价值转移系统——至少对我来说，这是以太坊带给世界的最令人兴奋的创新。让我们深入探讨以太坊中正在发生的一些令人兴奋的事情。

让我们从「合并 」开始。这方面正在取得惊人的进展，参与以太坊共识层和执行层开发的大多数开发人员现在都在集中精力完成合并。据我所知，有关合并的大多数核心研究已经完成，各客户团队正在相关的功能构建方面取得进展 (针对合并的开发者测试网和公开测试网将在未来几个月上线)。当然，我们距离主网的合并还有一段时间 (2022 年第一季度或者第二季度 ) ，但我相信，研究人员和开发人员将花费尽可能多的时间来确保一切顺利和安全进行。毕竟，这将是以太坊历史上最大的一次升级——我们想把它做好。

在过去几个月里，对整个以太坊生态系统来说，有一件事变得更加清晰了，那就是以太坊将如何实现扩展 ：Rollups (和零知识技术) 是当前所有可扩展性研究人员 / 开发人员都在研究的谢林点，且他们似乎也有一个普遍共识，即**「 模块化的区块链 」 设计是以太坊应该努力的方向** ；在此基础上，未来我们还将实现数据分片 ( data sharding)，以增强这些需要大量数据的 Rollups，给予 Rollups 更大的扩展性 。尽管在数据分片实现之前，我们已经有了混合解决方案 ，比如 Validium 和 Volition，它们以一定的安全性为代价 (因为它们将数据放在链下)，从而实现大规模扩展。

与此同时，NFTs (非同质化代币) 已经成为非加密原生用户进入 Crypto 领域最好和最简单的工具 。NFTs 的特别之处在于，它们对于新的 Crypto 用户更加友好，因为 NFTs 生态系统不像 DeFi 那样充满了复杂的金融术语。最重要的是，对于普通人而言，NFTs 的收藏和交易要比金融活动更加有趣和直观 —— 我的意思是，在传统金融系统中，普通人什么时候会去向什么产品提供流动性呢？不要误会我的意思，我仍然认为 DeFi 将是一个巨大的市场，但它不会成为大多数人进入 Crypto 领域的「大门」。

最后，我们现在有许多不同形式的 DAOs (去中心化自治组织) 存在于以太坊生态系统中，从像 Maker 这样的老派 DAOs，到「社交代币」 (social token) 生态系统中的新时代 DAOs。不过，这里有一条贯穿线——所有这些 DAOs 都集中于 改善全球参与者之间的社会和经济协调 ——所有这些 DAOs 都不存在中心化的中介方。当涉及到 DAOs 时，我认为可以采取「百花好放」的方法，这允许人们进行快速实验、创新并最终取得进展。

以上是为何我认为以太坊正处于黄金时代的简单阐述，我甚至还没有谈及像 DeFi 和新兴的社交代币生态系统。当然，并不是每一个基于以太坊的用例都能获得长期成功 (1COs 就是其中之一)，但作为一个生态系统，我们会随着时间的推移了解哪些用例是有生命力的，哪些是转瞬即逝的。

因为毕竟，以太坊允许你在世界上任何地方建造任何你想要的东西。
