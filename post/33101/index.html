Crypto Punk很反叛…

无聊猿（BAYC）有点阴郁…

冷兔 Never back…

如果你是表情包玩家，可能此刻收藏夹里就躺着冷兔。不妨打开看看，或许“信冷兔，不中暑”，“信冷兔，不挂科”的暖心日子就回来了。

前两者我们都很熟悉，Crypto原生的头像NFT，因贵出圈，因社区开放性而闻名；冷兔相比而言显的有点不同，它是从Web2而来，正准备向Web3开炮。

措辞也许有些夸张，但资深的NFT玩家们可能有所耳闻，最近华语Crypto圈子似乎都在抢冷兔团队创作的NFT-XRC的白名单，创始人易水寒甚至去劝阻用户不要Fomo。

笔者蛮好奇他为什么会这样说？毕竟Fomo是价值“发现”的重要环节。

对此，易老师是这样回答的：“我们当然不拒绝Fomo，但不想大家纯粹从炒作和投机心理出发去跟风，大家可以先来了解下XRC，再决定要不要喜欢它，然后再使用并持有它。”

是的，了解很重要，而这也是本文的目的。

从冷兔到XRC

从漫画到表情包再到幽默短视频，冷兔近几年的表现形式不断升级，目前全网粉丝近6500万，同时还有不少在卖的周边，堪称Web2古早IP。

易水寒在一次采访中说：“冷兔的内容基于话题场景生产,有些内容是与用户互动中产生的,具有很强的共鸣,一且有了这种共鸣就会形成自发性的传播 ”。

虽然当时还没有Meme这个词，但这似乎就是Meme的内核了。

2017年，创始团队又在杭州举办了一场名为《时空回廊——冷兔遇见名画主题艺术展》。戴珍珠耳环的少女等不少国内外知名画作里的人物被冷兔大胆进行了替换和融入，让网络流行IP瞬间变得更有艺术性，让冷兔从线上走到了线下。

而今年它又从Web2走向了Web，让人不禁感叹，团队好能折腾！

采访下来发现，他们能折腾的背后其实是对打造全球性IP的追求。“我们的理想是希望冷兔有一天可以像迪士尼、皮克斯一样，通过好故事、好载体和好的商业运作把中国文化传向世界。”易水寒说。

很幸运，这个理想在遇到无国界的“Crypto”后开始发酵。

1月7日，Cobo 联合创始人神鱼在社交媒体上发文称，Cobo 与漫画 IP 冷兔已达成合作，持有Cobo四周年NFT的用户可优先获得由冷兔创作的 XRC 项目白名单。

随后几天，XRC的在推特、微博以及Discord等社交平台上火了。

XRC全称X Rabbits Club，虽然继承了冷兔最原始的基础外貌，但却是为Allin Crypto而生的全新“物种”。

易水寒开玩笑说，如果愿意，你可以把XRC理解为冷兔在Crypto的仿盘。

仿盘？乍一听有种把老IP拿到Crypto做流量二次变现的感觉。

但实际上XRC并没有面向Web2流量出击，而是将目标用户锚定在NFT玩家身上，此前XRC还和原生的加密社区合作，为诸如无聊猿、Cobo四周年NFT这样的原生NFT的持有者开放白名单。

该做法在吸引玩家关注的同时，也让冷兔团队有机会宣布自己Allin Crypto的决心—Never back！

从Web2-Web3

为什么Never back？

这可能得从冷兔团队曾经的Slogan“用心传递快乐”讲起。

易老师表示，经营冷兔是源于一份不仅让用户感到快乐，也让自己感受到快乐的初心，是正确的事情，对于正确的事，哪怕很难他也一定不会回头，当下 All in Crypto也是如此。

这听起来好像有点伟光正，我们就问这会不会不够流氓？不够丧？或者不够Cyrpto？不够Web3？甚至不够国际化？

易老师“冷兔”上身般作出回答，他说：“从Web2走向Web3这件事本身就是全球都在探索的，XRC不会编故事，我们只是用“自己”去叙事而已——在有限的规则下把自己作为例子，走出一条路给两个世界的人看，同时证明华人还有好项目和认真做事的人，这就够了。

冷兔本来就是一个中国人的IP，我们来到Crypto，创作了XRC，总得先让国人知道它是什么吧！出海很难吗？XRC在以太坊上发行就说明已经走出国门了，后面一旦在Opensea有靠前的排名，那自然就进入国际视野了。

突然想到一本叫《把自己作为方法》的书，主旨思想大概是号召“人最后要回到人本身”，和把“自己”作为叙事在逻辑上有异曲同工之妙，当然，有时候能回到事物本身，就已经很酷了。

加密世界本身就是开放的，文化氛围不一定只有丧、丑和非主流，也应该有冷兔的“Never back”。

而目前XRC的首要任务就是做好Crypto头像，按照易老师的说法，如果连头像都做不好，那还凭什么能去做后面那些更大的事情？而且加密行业有很强的金融属性，需要学习和研究的课题很多,最难的不是在产品设计，也不是能卖出多少白名单，更不是二级市场能不能卖完，而是当所有产品都售完后，那时我们的Web之旅才真正开始…

终点是XRC宇宙

引爆一个NFT头像的方式有很多种，比如以涨（贵）服人，但单纯以贵取胜总不是长久之计，人们需要故事，需要更广泛地渗透力，需要找到组织，需要为爱也能发电。

XRC就是要做后者。

以著名地无聊猿为例，该NFT来自于无聊猿游艇俱乐部（BAYC)，于2021年4月底启动，由10000 个不同特征和属性的猿猴NFT组成，这些猴子虽然服装、神态各异，但都有一副无聊的表情。

用户持有无聊猿 NFT 相当于拥有BAYC的门票，代表你是俱乐部的成员，同时意味着你将拥有专属于你的那只猴子的全部商业使用权，既可以做对外授权，也可以二次创作。任何成员均可以用任何方式给BAYC俱乐部及其品牌赋能，他们正在把猿猴形象和一切结合，创造出一系列衍生品和属于自己社区的文化。

XRC的 “C” 也是Club的缩写，冷兔团队第一步也会从建立俱乐部开始。

易老师认为 NFT 是Web3乃至元宇宙的载体，给了冷兔团队实现理想的机会，而Club则是让NFT得以发展的重要组织形式，是未来社区的原型，是一切 NFT 项目的基石。

虽然XRC-NFT还没发售，但官方已经在听取社区的意见后给出了定价，感兴趣的朋友可自行前往官方Discord查看。

此外，对于大家最关心的路线图，目前已经发布了V1版本，秉承打造自由丰富的元宇宙空间，持续给社区成员带来实际回报的初心，官方做出如下规划：

1. DAO  

成立 DAO，推动项目长期发展；

建立多签和投票机制，让 XRC 持有者参与社区发展和决议；

以 DAO 之名，帮助并联合更多艺术家和传统 IP，携手进入 Web3 新世界。

2. Airdrop  

向持有者空投 XRC 周边实体限量版产品，及中国特色产品；

所有 XRC 持有者，直接成为后续 NFT 系列的白名单；

联合其他 NFT 社区或艺术家，给 XRC 持有者空投 NFT 作品。

3. Branding  

与其他主流及特色品牌合作，推出联名款产品；

联合 XRC 持有者，一起探索分散版权集中运营，授权给其他产品。通过收益 token 化的机制让 holder 受益。

4． Metaverse  

购买元宇宙地块，持续播放 X Rabbits Club 的相关作品；

联合游戏厂商探索合作，NFT 持有者可凭此进入 GameFi 空间实现 Play-to-Earn；

让 XRC NFT 成为元宇宙中必备的头像资产和身份象征。

关于XRC现有的一切都已传达完毕，希望让大家看懂了。

最后，以小明同学（0xmin）的金句来收个尾，他说不管在原子世界还是比特世界，一个牛逼的NFT肯定是带有文化属性的。笔者在采访易老师时也问了类似问题：如果X Rabbits Club逐渐成熟，你希望社区传递出怎样的文化属性？

易老师说，“没法预测，但终局肯定是光伟正，可能过程会曲折，就像粉丝为冷兔创作的各种故事一样，有阳光睿智的，也有变异邪恶的。 ”
