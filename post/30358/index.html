Coinbase Ventures总结了三季度关键主题，包括监管深入、多链趋势和 NFT 崛起等。

Coinbase Ventures 已成长为加密领域最活跃的投资者之一。在第三季度，Coinbase Ventures 已经进行创纪录的 49 笔投资，平均每约 1.8 天就有一笔新交易。这比第二季度的 28 笔投资和第一季度的 24 笔有所增加。截至 2021 年第三季度，Coinbase Ventures 投资组合包括超过 200 家公司和项目。

累计来看，Coinbase Ventures 90% 的投资资金已于 2021 年年初至今投入使用，反映出 Coinbase Ventures 在其运营的第四年加快了步伐。其投资组合中 50% 的独特新「Logo」也是在 2021 年出现。

动机和哲学

Coinbase Ventures 的主要任务是支持不断增长的加密生态系统，因此广泛投资推动加密货币发展的强大企业家。Coinbase Ventures 希望加密生态系统蓬勃发展，而不是只关注特定的结果（这是典型的企业风险投资）。

最终，Coinbase Ventures 将加密货币视为一股上涨的浪潮，生态系统的发展推动了包括 Coinbase 在内的所有参与者「水涨船高」。传统的战略利益（如商业伙伴关系和潜在的并购）固然很好，但 Coinbase Ventures 认为它们是锦上添花。

投资类别

Coinbase Ventures 的投资范围从六位数的种子轮交易投资到数百万美元的增长轮投资。有许多方法可以划分投资类别，但 Coinbase Ventures 将市场细分为以下类别：协议 + Web3 基础设施、DeFi、CeFi、平台+开发工具、NFT/ 元宇宙（Metaverse）和其他。

具体而言，在 Coinbase Ventures 六大投资类别中，协议+Web3 基础设施（29%）占比最大，其他类别为 DeFi （24%）、CeFi （18%）、平台+开发工具（15%）、NFT+元宇宙（9%）和其他（5%）。

关键主题和心得

在其迄今为止最活跃的一个季度，Coinbase Ventures 看到了美国 CeFi、第 1 层 / 第 2 层（L1/L2）、跨链协议以及 Web3 工具的重大发展。以下是 Coinbase Ventures 观察到的一些主要主题：

监管机构和中心化参与者深入加密货币领域

监管机构在第三季度更加广为人知，因为美国证券交易委员会（SEC）和财政部以及国际金融行动特别工作组（FATF）都加强了在整个加密生态系统中的参与。这为早期协议和团队带来了某些形式的监管风险。另一方面，最大市值的加密资产 BTC 获得积极的支持，BTC 期货交易所交易基金（ETF）获得 SEC 批准。Coinbase Ventures 认为这将允许潜在资本进入加密市场，从而带来大量交易量、资金流入和兴趣。

Square、Twitter、Stripe 和 Tik Tok 等 Web 2.0 公司也在第三季度扩展其加密策略。Square 宣布了一个基于比特币的金融服务平台，Twitter 透露未来将整合 BTC 闪电网络和 NFT，Stripe 宣布将通过其专门的新加密团队回归加密领域。Tik Tok 宣布与 ImmutableX 合作，推出由创作者主导的 NFT 系列收藏品。

与此同时，在 Coinbase Prime、Coinbase Cloud 和其他第三方平台的支持下，银行、金融科技公司和经纪交易商开始进一步将加密货币集成到他们的产品中。总之，在过去的一个季度里，加密行业在成熟度和机构采用方面取得了巨大的进步。

多链生态系统发展迅速

经过多年开发旨在缓解以太坊瓶颈的解决方案，随着一系列第 1 层和第 2 层生态系统的兴起，扩容终于实现了。当前的主要动力在于利用 EVM （以太坊虚拟机）兼容性的解决方案，允许用户和开发人员以相对较低的切换成本迁移到新环境。用户可以使用其现有的钱包访问 EVM 兼容的 L1，比如 Avalanche；或者侧链 /L2，比如 Polygon、Arbitrum、Optimism。Solidity 智能合约通常也可以复制粘贴到任何与 EVM 兼容的 L1/L2，这促进了热门 DeFi 应用程序跨多个链部署。

由于 CeFi 交易所与这些新的 L1/L2 集成的速度很慢，Coinbase Ventures 看到了新推出的跨链桥的吸引力。这些跨链桥促进了数十亿资金从以太坊转移到各种 L1/L2。

虽然在第三季度用 Solidity 编写的 EVM 兼容应用程序在 L1 和 L2 上的吸引力最大，但其他生态系统正在为桌面程序带来更具表现力的可编程性。新的原语（Primitives）专注于人们更为熟悉的编程语言，如 Rust （Solana、Polkadot）、Golang （Cosmos）和 Move （Facebook Diem、Flow），可能会为该行业带来新一波 Web 2.0 开发人员。

更好的 Web3 UX 即将推出

在第三季度，Coinbase Ventures 看到 Web3 工具进一步开发，将简化 Web3 交互体验。XMTP 正在引领跨 Web3 地址的消息传递标准。Spruce 正在将「OAuth」（开放授权）标准化，这将允许用户与 Web 3 应用程序安全地共享数字凭证、私人文件和敏感媒体内容。Snapshot 使跨 DApp 访问治理论坛和决策变得简单。

同时，正在开展大量工作为 Web3 应用程序创建额外的安全性。OpenZeppelin 的去中心化项目 Forta 正在智能合约的实时安全监控方面取得进展，目标是在智能合约代码执行、错误检测以及最终实时防止黑客攻击方面提供更多透明度。同样，Certik 正在提供「快速简便」的自动化审计工具，以帮助 DApp 更快地在市场推出。

同时，随着链上社区的技术和法案创建模式开始奏效，DAO 技术栈不断发展。Syndicate （以及其他项目）旨在通过创建去中心化投资协议和社交网络成为「加密版 AngelList」。

NFT 2.0 和加密游戏蓬勃发展

第三季度还看到大量专注于 NFT 创建者工具的开发，这些工具最终将扩大 NFT 用例和受众的范围，同时创建新的社交功能。

与此同时，在 Axie Infinity 的引领下，基于 NFT 的游戏继续加速发展，因为其游戏边玩边赚（Play-to-Earn）模式在新兴市场（菲律宾、巴西、印度等）大受欢迎，吸引了 200 万 DAU （日活用户），并产生超过 20 万美元的收入。Loot Project 还通过一种反向的游戏开发模式吸引了整个行业。这是通过首先向公众发布基于 NFT 的游戏资产来完成的，以引导社区并激励进一步的发展。
