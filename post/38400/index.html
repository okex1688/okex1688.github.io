虽说区块链是开放的系统，但是现在准入门槛正在成为一种新常态。以NFT PASS为代表的准入式玩法正在受到追捧，背后主要有2个原因：

一是NFT项目需要NFT PASS。 NFT PASS将初代NFT和后代NFT连接到一起。以BAYC为例，用金铲子（BAYC）挖银铲子(MAYC)，用金银铲子(BAYC、MAYC）挖地（Otherdeed），从而衍生出副IP、开放性故事，乃至元宇宙生态。NFT PASS延长了PFP的热度和生命周期。

二是用户追求NFT PASS标的。 一方面，NFT PASS提供进入私人俱乐部的权利，参与测试和提前mining的资格，获取最新消息和专业工具的敞口等。NFT PASS打造的资源优势是用户所需要的。另一方面，优质NFT PASS承诺的空投、白名单等确实让用户获得了收益。

57个NFT PASS类项目，来源：哔哔News

伴随Moonbirds、0xOG PASS的热度，市场上已经有非常多的NFT PASS项目。它们可以被分为俱乐部型、合作型、alpha社区型、工具型等类型。整体来看，这些项目的模式和功能比较重合。NFT PASS对于NFT的赋能并不明显。反过来，NFT PASS的价格依赖于NFT IP本身。

NFT PASS提供哪些赋能？

NFT PASS可以理解为Web3世界的会员卡，由各大项目方推出。用户通过购买NFT PASS提前锁定一些权益，比如成为私人Discord社群的成员，获得后续子项目的空投或铸造白名单，以更低的成本参与合作项目，参与线上线下活动，使用一些内部工具或功能等。

哔哔News统计了57个NFT PASS项目，发现NFT PASS提供的权益重合度较高，基本上局限在上述功能场景中，没有太多的创新。而用户参与NFT PASS，主要是看中后期空投、白名单，以及通过社区发现、参与早期项目的机会。

对于项目方而言，NFT PASS是一种提前售卖预期的工具 ，也在某种程度上承诺了未来的责任，例如要与其他NFT项目合作，要推出后代NFT，要开发元宇宙等等。不过目前的NFT PASS玩法对项目方履行职责没有强制要求。这对于用户而言是最大的风险点。

NFT PASS也能帮助项目方锁定早期参与者和贡献者。 以BAYC为例，越早购入BAYC的用户获得了越丰厚的福利，他/她们陆续获得变异药水、BAKC、$APE、Otherdeed for Otherside空投。随着BAYC的发展，尽管大量新用户涌入，早期参与者、贡献者、长持者的“股份”并没有被稀释。相反，他/她们在每个阶段都是最核心的参与者、最大的受益方。

所以，NFT PASS对NFT的赋能主要是2点：一，连接初代和后代NFT，解锁叙事和玩法；二，标记早期参与者、贡献者、长持者。整体来看，NFT PASS对NFT的赋能并不明显，而NFT本身却为NFT PASS提供了重要的价格支撑。

哔哔News统计到的地板价较高的NFT PASS都有强IP的特点。例如地板价达到14.2 ETH、3.18 ETH的CLONE X和Element背后分别是知名艺术家村上隆、方力钧IP。由Adidas、X2Y2推出的NFT PASS地板价则分别达到1.24ETH、1.88 ETH。

Adidas推出的NFT PASS：Adidas Originals Into the Metaverse

5种NFT PASS类型

根据发起方式的不同，NFT PASS可以被分为5种类型，分别是俱乐部型、合作型、alpha社区型、工具型、其他。

俱乐部型

代表项目：BAYC、Proof Collective NFT、0xOG Pass

俱乐部型NFT PASS的整体质量最高，发展得最好。本轮NFT PASS类项目的代表Moonbirds就属于Proof Collective NFT系。而致力于打造“无聊猿游艇”俱乐部文化的BAYC则是现在NFT领域的第一IP。

俱乐部型NFT PASS往往有创始团队声誉加持，俱乐部NFT持仓雄厚的特点。以Proof Collective NFT为例，3个创始人都是Web2、Web3领域的知名人物。其中1个创始人的播客节目PROOF在NFT领域有相当大的影响力，曾邀请过Beeple、YGG联合创始人、Reddit联合创始人等参与。PROOF也直接促成了Proof Collective俱乐部的诞生。

在蓝筹持仓方面，Proof Collective俱乐部持有148个CryptoPunks、817个BAYC、499 个Meebits、5288个Art Blocks等。创始团队的声誉和俱乐部的经济实力是对NFT PASS价格的最大支撑。目前Proof Collective NFT和Moonbirds地板价分别达到82.5和24.99 ETH。

Proof Collective俱乐部持仓

合作型

代表项目：Adidas Originals Into the Metaverse、CLONE X - X TAKASHI MURAKAMI

合作型NFT PASS主要通过知名IP和平台、知名IP间合作推出。 例如，CLONE X - X TAKASHI MURAKAMI基于目前世界上最知名的艺术家之一村上隆推出。合作的另一方RTFKT是虚拟世界第一潮牌，曾经和雷克萨斯、Clarks、a16z等合作，为其设计虚拟球鞋。

RTFKT为a16z打造的虚拟球鞋

另一个例子是中国当代艺术家方力钧和新兴加密艺术平台Outland推出的Element。Element将作为Outland的mintPass。IP与平台合作的形式中，IP提供了声誉和粉丝基础，平台则提供了配套的NFT设施和服务。双方是互补的关系。

而Adidas Originals Into the Metaverse则是IP和IP间的强强联合。这款NFT PASS由阿迪达斯的NFT系列服装Adidas Originals和BYAC、NFT漫画Punks Comic、NFT收藏家G-Money合作推出。让我们看到了时尚品牌进军NFT的玩法——为NFT IP创建虚拟服饰，虚拟服饰和穿着虚拟服饰的NFT形象可用于The Sandbox等元宇宙。持有者还可以获得连帽卫衣等实物发放，从而获得虚拟和现实的双重体验。

Alpha社区型

代表项目：Lasercat NFT

Alpha社区型NFT PASS基于投资社群而产生，NFT PASS作为进入社群的通行证。这类PASS卡往往为用户提供社群内资源，比如NFT开图排行榜、研报、名人堂、仪表盘工具等，而且强调群策群力的运作方式。

部分alpha社区既是社群又是孵化器。其中一些会推出自己的NFT项目，NFT PASS的持有者可以获得空投或白名单。更多的Alpha社区兼孵化器则是通过与其他NFT项目合作为“会员”提供福利。

Alpha社区型NFT PASS质量参差不齐。有的PASS卡能提供高价值社群信息和福利，有的则被曝出一些负面消息。

工具型

代表项目：Premint、Trait Sniper、Coniun Pass、Dtools Pass

目前已经出现了工具网站把付费服务打包进NFT PASS的趋势。虽然对于大部分用户而言，一些进阶类的数据、查询功能并没有太大的吸引力，但是部分观点认为优质工具网站的NFT PASS会成为未来空投的依据，尤其是在NFT PASS有一定获取门槛的情况下。

以DexGuru推出的GURU Season Pass NFT为例（虽然不是工具类网站，但是能比较好地说明问题）。在2022年2月1日之前向DexGuru发送小费，或者是LobsterDAO NFT、DEXT、UniWhales、Chartex、AstroTools通证的持有者才能获得该NFT PASS。在这种情况下，NFT PASS帮助DexGuru标记出了符合特定要求的用户。

其他

代表项目：Big Time、Invisible Friends、Magic Ticket、Ajin Youth Rebels

其他NFT PASS主要是由NFT项目、交易平台等推出的PASS卡，往往具有“铲子”的功能。

在Big Time的案例中，NFT PASS作为内测通行证，用户可以提前体验游戏，通过play to earn获得掉落的NFT。Big Time也是典型的为NFT PASS设置分级的案例。等级越高的NFT PASS，对应的权益的和获得奖励的概率越高。哔哔News统计的57个NFT PASS中，被设计成不同级别的情况主要出现在其他类、工具类、alpha社区类NFT PASS中。

而在Magic Eden、X2Y2等平台上，官方推出的NFT PASS则可以用作mint PASS。作为聚合NFT的平台，这类PASS卡应该能为用户提供比较好的打新权益。这两个平台推出的NFT PASS分别是Magic Ticket和Ajin Youth Rebels，值得关注。

NFT PASS：Magic Ticket、Ajin Youth Rebels

结语

NFT PASS为NFT的使用场景提供了新的思路。在这一风潮下，越来越多的NFT PASS出现。 如前文所述，哔哔News不完全统计到的就有57个。这一赛道也一度受到关注。

NFT PASS帮助NFT解锁新的叙事和玩法，也能让早期参与者、贡献者、长持者提前锁定权益。NFT项目要做好NFT PASS不容易，因为他们将要构建的不再只是一张PFP，而是开放性的故事、玩法，甚至元宇宙。但项目要推出NFT PASS却是一件非常简单的事情，这对用户构成了风险。

我们期待NFT PASS赛道的发展，但同时也需警惕其中的风险。目前来看，优质俱乐部和NFT项目、大IP、头部平台推出的NFT PASS具有更稳定的价值。
