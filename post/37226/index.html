元宇宙（Metaverse）叙事作为人类想象力的天花板，它是由人类利用各种科技手段创造出来的与现实宇宙映射与交互的虚拟宇宙，而它自然会是包罗万象的，在我看来，元宇宙会是由成千上万公有链、私有链以及联盟链星系组成的虚拟宇宙。

而本文要讨论的，自然是公有链星系的范畴。

提醒：如果你认为元宇宙叙事是一种忽悠，那么请自动忽略以下内容。

今年二月份，顶级加密投资基金 Paradigm 的创始合伙人 Matt Huang 提出了一个非常有意思的问题：

“你从地球移民到一个经济规模小得多，但未来前景广阔的外星球。
着陆后，你必须将所有地球财富转换为：
1、外星球的土地；
2、外星球货币；
3、外星球股票市场的股票；
4、稀有金属；
你选择哪一个？为什么？
”

这一问题让我联想到了元宇宙（Web3）叙事和公链、L2、NFT、DeFi 以及 DAO 之间的关系。

接下来，请直接将你的想象力拉满，而不用考虑这些叙事是否离谱荒诞。

先画两张极丑的思维导图，然后展开描述这些子叙（da）事（bing）。

公链生态星系：由主链、L2、子网、平行链等组成

首先，元宇宙和 Web 3 接下来的子叙事就是公链生态星系。

其中，比特币是目前规模最大的一个星系，它也是元宇宙大爆炸的起点，但这个星系比较单一，它主要生产一种最通用和最稀有的元宇宙贵金属，而它会被所有其他星系接受。

然后是以太坊星系、BNB 星系、波卡星系、Solana 星系、Avalanche 星系、Cosmos 星系等大家耳熟能详的公链生态系统，这些星系里还会有恒星（主链）、行星和卫星（L2、子网、侧链等）。

比如，以太坊星系是我们目前最了解的一个公链星系，除了主链恒星，它还会有 Starknet、zksync、Optimism、Abitrum、Polygon、Aztec、Scroll、Loopring 等各大 L2 行星和卫星。

ETH 呢，则是以太坊星系的通用货币和燃料资源，而 L2 和子网的原生代币（比如 Optimism 刚宣布发行的 OP 代币），则是每个次级星球的通用货币和燃料资源。

公链生态星系的三大组成要素：DeFi、NFT 和 DAO

而一个星系，除了通用货币和燃料之外，还由 DeFi、NFT 和 DAO 这三个基本要素组成。

DeFi 相当于元宇宙星系里的金融系统，不同的 DeFi 应用对应不同公司的股票，这一点非常容易理解。

而 NFT 是星系的构建块，它可以代表任何事物，比如目前我们接触到最多的头像（化身）、Pass 卡、游戏资产、土地、收藏品、音乐、视频等等。

而 DAO 是由人组成的组织群体，它可以是元宇宙里的国家、公司、联盟等各种组织形式。

下面就举几个项目例子来简单说明：

Uniswap 和 Curve 作为 DeFi 当中 的 DEX，相当于以太坊等星系中的通用货币及燃料的交易所。

而 Opensea 等 NFT 交易平台，就相当于以太坊和 Polygon 星系中所有非同质化物体的交易所。

除此之外，还会有各种通用货币（token）和非同质化物体（NFT）的借贷、衍生品协议（或组织）。

除了这些通用协议（或组织）之外，还有像 Yuga labs 所创造的 NFT 国家，它是由文化、apecoin（国家通用货币）、otherside（土地）、BAYC 等头像 NFT（国家公民身份）、游戏（城市）及游戏物品（城市里的物品）组成的。而其他的一些 NFT 项目，还尚未形成国家的概念，但从元宇宙叙事的角度来看，这可能会成为一种发展趋势。

此外，我们并不能排除 Yuga labs 等 NFT 项目方推出自己的公链、L2 或侧链的可能性，这样它的叙事就从国家升级到了星球或星系，那么在这个星球上，还会出现各种 DEX、稳定币、衍生品等金融应用和 DAO 组织，甚至还会有其他的一些小型国家。

如何在元宇宙里投资？

回到 Matt Huang 的问题，我是否能理解成：当你从以太坊这个已经成熟的星球转移到另一个有发展潜力的星球，你应该选择投资什么？

首先，最容易的选择自然是这个星系/星球的通用货币（或燃料），如果这个星系具有潜力，那么你的投资就会带来回报，但这种方式的缺点是，回报并没有那么暴力。

选项 1（土地）可能也是很多人会选择的，这属于国家的子概念，但并不是所有土地都拥有投资价值，而是只有大城市的核心土地才具备投资价值，并且这类资源的稀缺性更明显，也更容易被炒作。

选项 3（股票市场的股票），则对应各种 DeFi 代币，这似乎很容易被大家理解，但选择就比较困难。

选项 4（贵金属），对应 BTC 等，硬通货，这种投资也非常稳。

除此之外，还会有各种元宇宙基础设施，比如跨链协议（星际航行公司）、预言机（数据服务公司）、去中心化社交应用等等，这里就不一而足了。

需要注意的是，由于元宇宙是由人类创造出来的虚拟宇宙，因此，不论是星系、星球、国家、公司股票等叙事，人都应该放在第一位来考虑。

以上这些零碎的想法还很幼稚，大家轻点拍砖。
