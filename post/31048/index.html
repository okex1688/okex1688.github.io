2021年大火的“元宇宙”概念将人们对世界的目光从对现实世界的探索转向虚拟世界的创建。元宇宙、NFT相关一系列衍生项目的火爆，也引发着人们的思考：当所有的情感价值脱离身体、来源于代码时，人们对真实的宇宙逐渐失去探索的好奇，人类又会否退化成尤瓦尔·赫拉利笔下的“无用之人”？也许时间才能评判一切。

2021年8月28日，一个平平无奇的炎热周六，NBA 金州勇士队的明星控卫、三届 NBA 总冠军、两届常规赛 MVP，同时也是慈善家和企业家的库里，悄然将自己的推特头像从偶像科比，换成了一只穿粗花西装、面无表情的卡通猿猴。这一举动引起了人们广泛的关注，更给不断向外扩圈的crypto市场又添了一把火。这只猿猴是什么来历？又是什么原因让库里愿意花费18万美元买下这副图像，甚至用它替换了自己的偶像作为头像？这一切都源自于NFT的独特魅力。2021成为NFT爆火的一年。

什么是NFT

Non-Fungible Token(NFT)是起源于2017年底Ethereum Implementation Proposal 721(EIP-721)的一种数字货币，一般译为非同质化代币。与比特币、以太币不同，相同金额的比特币是毫无差异的，但NFT包含了记录在其智能合约中的识别信息，这赋予了NFT唯一性和稀缺性，因此不能被另一种代币直接取代。同质化代币是一种能够替换、具有统一性、可接近无穷拆分的代币。货币就是一种典型的“同质化”资产，比如美元，其代表的价值量与钞票序列号无关。但现实生活中许多具有实际价值的事物是不可替代的，如一份合同、房屋产权、艺术作品、出生证明等。非同质化代币应运而生。当我们在区块链上表示这些NFT项目时，理解这一“非同质”概念尤为重要。

NFT的特性

以区块链为依托

我们在现实生活中接收与创造着各种数字资产：域名、活动门票、游戏装备资产，以及在Twitter、Facebook、微信等社交网络上处理的各类信息。但我们在多大程度上“拥有”这些资产呢？这些数字资产的确权问题不像物理意义上的资产权属那样明确，资产的转移也比较困难。例如你与某手游中的“皮肤”与“装备”的权属关系在游戏下线后便随之消失。

而这就是区块链发挥作用的地方！区块链为数字资产提供了一个协调层，赋予用户所有权和管理权限。区块链为NFT赋予了唯一性和可验证性，这些属性改变了用户和开发者与这些资产的关系。

NFT是唯一的。每个NFT都代表数字或现实世界中的某个对象，并且即使存在类似的对象也是唯一的，在以太坊上被单独追踪。

NFT是可验证的。NFT的历史所有权依托区块链，记录在公共分类帐中，所有人都可以看到。这使得每一个NFT难以“复制”，进而有利于真实性的证明。例如你随手录制的一则短视频就可以生成一个NFT，或自己拥有，或在opensea上进行交易。虽然它可以被无限复制粘贴，但正品只属于你或购买这个NFT的人。

标准化的表示方法

传统的数字资产没有统一的表示形式，而通过在公共区块链上表示NFT，开发人员可以构建与所有NFT相关的通用、可重复使用、可继承的标准。其中包括所有权、转让和简单访问控制等基本原始操作。一些附加的标准（例如如何显示 NFT 的规格）可以分层在顶部，以便在应用程序内部进行丰富的显示，相关数字资产的表示得以标准化。

项目互操作性

NFT标准允许NFT在多个生态系统中轻松移动。当开发人员启动新的 NFT 项目时，这些NFT 可以在OpenSea 等 NFT 交易平台上被查看与交易。在NFT收藏品领域，藏家们可以通过数十家不同的加密钱包轻松地查看、展示、炫耀自己的 NFT 收藏品，满足人们的情感需求。

资产可交易性与高流动性

NFT可依托区块链在公开市场上进行自由交易。用户可以利用拍卖、竞价、捆绑等货币具有的复杂的交易能力，将物品移出原始环境，进入市场。NFT市场可以迎合各种受众，从铁杆交易员到更多新手玩家，将资产更多地暴露在更广泛的买家群体中。

不可变性与稀缺性

智能合同允许开发人员对NFT的供应设置硬性上限，并强制实施NFT发布后无法修改的持久属性。例如，开发人员可以在程序上强制要求只能创建特定数量的特定稀有项目，同时保持更常见项目的无限供应。开发人员还可以强制要求特定属性不会通过在链条上进行编码而随时间而更改。这对于艺术品来说有重大意义，因为艺术在很大程度上依赖于原创作品的可探稀缺性。

NFT的前世今生

了解了NFT的基本特性，我们将目光再次转向文章开头的那只“猿猴”，它正是以太坊上“无聊猿猴游艇俱乐部”——BAYC（Bored Ape Yacht Club）发行的1万个猿猴NFT头像之一，也是区块链上无数NFT项目之一。让我们回溯NFT的前世今生，探寻NFT项目如何发展壮大，集合广大参与者成为社区。

NFT头像的首创——CryptoPunks

CryptoPunks是以太坊上的早期头像NFT项目。它由10,000个朋克头像组成，每个朋克都有一套独特的特征，由手游公司Larva Labs建造。CryptoPunks 拥有一个链上市场，可用于像MetaMask这样的钱包，降低了进入与 NFT 互动的门槛。CryptoPunks的NFT头像总数是固定的。在创建早期，拥有以太币钱包的用户都可以免费领取CryptoPunks头像。而如今，用户只能在LarvaLabs网站和Opensea等二级交易场所购买CryptoPunks的NFT头像。

NFT价值的发现——CryptoKitties

CryptoKitties是第一个将 NFT 纳入主流的项目。CryptoKitties 于 2017年底在ETH滑铁卢黑客马拉松上推出，其特点是一款原始的链上游戏，允许用户一起繁殖数字猫，以生产不同稀有的新猫。“0代"猫在价格不断下跌的荷兰式拍卖中被拍卖，新的猫也可以在二级市场上出售。

虽然一些游戏社区后来将CryptoKitties贴上了“不是真正的游戏”的标签，但考虑到区块链的设计限制，该团队实际上在开创链上游戏机制方面具有开创性意义。首先，他们建立了一个链上繁殖算法，隐藏在一个封闭源智能合约中，该合同确定了猫的遗传密码。其次，CryptoKitties 团队通过复杂的激励系统确保了繁殖的随机性，并有远见地保留某些低ID猫，供以后用作促销工具。最后，他们开创了荷兰式拍卖形式，后来成为NFT的主要价格发现机制之一。CryptoKitties团队非凡的远见卓识，为NFT的价值空间带来了巨大的提升。

CryptoKitties的育种和交易机制带来了一条清晰的盈利途径：买几只猫，繁殖它们来制造一只稀有的猫，转让交易猫，重复（或者干脆买一只稀有的猫，希望有人来买它）。这刺激了育种社区的成长：致力于繁殖和转让稀有猫的用户集合。只要有一组新的用户进来玩游戏，价格就会上涨。然而“CryptoKitties泡沫”在这种靠新奇博得关注的交易中不可避免地产生了：新的需求进入CryptoKitties世界，价格上涨，价格上涨带来新的需求。当然，所有的泡沫最终都必须破灭。小猫的平均价格开始下降，销量下降。许多人意识到，相对于“真实游戏”而言，CryptoKitties游戏是原始的，除了投机者之外，它不会留住更广泛的受众。一旦新奇事物消失，市场就遭受了破坏。

NFT的“第二层”游戏

在CryptoKitties之后，出现了创新的“第二层”游戏，即由不隶属于最初的CryptoKitties团队的第三方开发者在CryptoKitties之上构建的游戏。CryptoKitties的神奇之处在于，这些体验可以“不受许可”地开发：开发人员可以简单地将自己的应用程序放在公有的CryptoKitty智能合约之上。在某种意义上，CryptoKitties可以在原始环境之外拥有自己的生命。

例如，《Kitty Race》允许你让你的CryptoKitties与其他的CryptoKitties竞赛以赢得ETH，而《KittyHats》允许用户用帽子和绘画装饰他们的CryptoKitties。后来，Wrapped Kitties将Kitties和DeFi结合在一起，让你把你的CryptoKitties变成可替换的ERC20代币，在去中心化的交易所交易，这对CryptoKitty市场有各种有趣的分支。Dapper Labs （CryptoKitties背后的新成立公司）通过成立KittyVerse接纳了这些项目。

NFT与数字艺术

艺术品的保存消耗着大量的人力物力，顶级展览需要花费大量保险费用，名画修复动辄几十万的代价，这都在抑制着艺术市场的发展。NFT链上交易的特点为艺术市场提供了活水。

2021年3月，数字视觉艺术家Beeple的一套NFT艺术品引发广泛关注。从2007年开始，Beeple每天都创作一件艺术品，最后他将五千幅作品拼接成一张图片，并将其命名为《Everydays：The First 5000 Days》，并在佳士得拍卖行上线。3月11日，一个名叫“MetaKovan”的卖家以6934.8万美元的价格拍下了这幅作品。

同样在2021年3月，著名街头艺术家Banksy的作品《Mornos》（《白痴》）被以9.5万美元的价格买下，随后购买者直播烧毁了画作，并将其NFT版本挂上NFT交易平台Opensea。最终，《Mornos》的NFT版本被以4倍的价格售出，约合人民币247万。随着《Mornos》的熊熊燃烧，画作完成了从实体艺术品到NFT艺术品的转变，人们看到了NFT艺术的商业潜力，NFT艺术的时代正式来临。

《Mornos》实体作品的持有者——区块链公司Injective Protocol将烧画行为定义为“一种艺术表达”。NFT代币与数字艺术、行为艺术紧密结合，逐渐衍生出一种新的艺术形式——加密艺术。一条短视频、一条代码、一双潮鞋、一个游戏头盔、NBA球星的生涯经典瞬间……NFT艺术作品可以以各种形式存在，拥有无限可能性。

文章开头的Bored Ape Yacht Club（无聊猿游艇俱乐部）也是当前炙手可热的NFT之一，最初的约1000个像素头像在6个小时左右的时间里迅速售罄。

在BAYC中，每个Bored Ape都是独一无二的，并由170多种可能的特征以编程方式生成，包括表情、头饰、服装等。人们认同他们的猿，并期待拥有代表他们个性的人。猿现在是社交媒体上流行的数字化身。卡通猿在NFT社区引起了共鸣，它们是相互理解的象征，将人们聚集在一起。那些拥有猿猴NFT的人往往选择展示他们的猿，以表达对他们目前最珍贵的NFT的自豪感。

Bored Ape Yacht Club团队凭借其透明度和对社区的奉献精神，为俱乐部成员展现了严密具体的路线图和未来计划，激发着人们的创新意愿。NFT社区的真谛进一步明晰：共识是建立社区的基础，沟通是建立在信任基础上的强大社区的关键。

NFT的虚拟-现实延展

2021年大火的“元宇宙”概念将人们对世界的目光从对现实世界的探索转向虚拟世界的创建。元宇宙是一个承载虚拟活动的平台，为人们提供丰富的消费内容，公平的创作平台，可靠的经济体系，沉浸式的交互体验，寄托人的情感，给予人们心灵归属。人们可以在元宇宙体验不同的内容，结交数字世界的好友，甚至创造自己的作品，进行交易、教育、开会等社会性活动。

Decentraland是元宇宙概念应用的典型例子之一。Decentraland创立于2017年9月，是一个由以太坊区块链支持的去中心化虚拟现实平台，Decentraland中的一切产权和交易行为都在区块链上有迹可循，人们可以自主创建场景，前往市场（MarketPlace）中购买现成的建筑、装备等应用内物品，进行一系列博物馆展览、演唱会、赌场、万圣节等公开活动。

同时Decentraland是DeFi世界里第一批采用DAO社区治理模式的项目，用户能够通过集体投票成为其真正的主人和治理者，是第一个完全去中心化、由用户所拥有的虚拟世界。

在游戏中复刻一个能与现实世界相像的场景世界与人物画面，逐渐成为数字世界下人们的技术与情感需求。虚拟世界不再仅仅是现实世界的补充，它将与现实一起构成人类生存的世界。
关于NFT的审思

在独立于现实的虚拟世界里，我们创造、转让、交易一系列数字资产，产权确权、资产转移等一系列在现实中棘手的问题，在区块链上迎刃而解，给予人们无限便利与创新激励。

然而元宇宙、NFT相关一系列衍生项目的火爆，也引发着人们的思考：当所有的情感价值脱离身体、来源于代码时，人们对真实的宇宙逐渐失去探索的好奇，人类又会否退化成尤瓦尔·赫拉利笔下的“无用之人”？也许时间才能评判一切。
