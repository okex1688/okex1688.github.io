下面是31个免费的指标、网站、列表和工具，我们可以使用它们来发现、分析和研究加密货币和代币。不需要注册(就本篇介绍的功能)。

加密新闻，市场和货币更新

  • Coingecko Cryptocurrency Categories —— Coingecko.com为特定用例、行业、L1区块链等提供了硬币和代币的概述。

  • Open Data Crypto Library  —— cypherhunter.com是一个收集每个区块链项目信息的数据库，包括在加密空间工作的人。

  • Coinmarketcap ICO Calendar ——在coinmarketcap.com上获得即将开始的ICO列表，包括每个新加密项目的详细信息。

  • airdropalert.com  ——当前和或正在计划的硬币和代币空投的列表，包括如何参与的内容。

  • coinmarketcal.com  ——为我们概述加密领域的重要事件，如硬币列表、beta测试、加密会议、网站更新等。

  • cryptopanic.com  ——一个加密新闻聚合网站，实时列出不同来源的新闻。很好地了解了正在发生的事情。

  • lunarcrush.com  ——这个网站收集和分析社交媒体网络数据来衡量市场活动和用户情绪。还可以让我们跟踪哪个影响者是主要参与哪个代币。

  • Crypto Proposal Tracker ——messari.io 跟踪加密项目的提案过程，从初步讨论到完成投票。该平台还提供了许多其他免费的加密资源和列表。

  • Crypto Fundraising Stats ——dovenmetrics.com是一个收集加密融资信息的数据库。在这里我们可以了解到个人项目筹集了多少资金。该数据库还收集投资者、加密风投、融资轮等信息。

加密统计和分析

  • Cryptocurrency Stats Pie Charts  ——Bitinfocharts.com将主要加密货币的重要统计数据可视化为饼图。

  • Crypto Treemap Charts ——在coin360.com上，我们可以查看加密货币和代币的整洁的树图。我们可以深入并可视化加密生态系统的特定部分(例如，它允许我们只显示以太坊代币)

  • Mining Pool Stats ——列出数百种加密货币的矿池关键统计信息的网站。

  • Stablecoin Stats  ——一个收集稳定币数据的网站，如总流通，24小时的交易量，抵押品等。

  • Investigate & Monitor Crypto Addresses  ——Breadcrumbs.app提供了工具来分析比特币、Polygon、以太坊和任何ERC-20代币的个人地址。非常适合可视化交易流。

  • Github Activity Check  ——使用CryptoMiso，我们可以轻松检查Github超过300个加密货币和代币的活动。

  • Bitcoin Visuals ——一个网站致力于可视化每一种可以想象的比特币数据。

  • dune.xyz ——该网站允许用户提取几乎存在于以太坊区块链上的任何信息，并在以太坊生态系统中创建数百个项目指示板。

  • toketerminal.com  ——该网站为我们提供dapp和区块链的深入了解，如累积协议收入，并允许我们了解顶级表现平台。

  • ETH Gasstation  ——一个分享以太坊区块链和底层协议上的gas使用和成本统计数据的网站。

  • DApp Review  ——本网站收集运行在L1区块链上的协议数据，如ETH、NEO、BSC和TRON。在这里，我们可以看到诸如每个DApp的用户数、已部署的智能合约等统计数据。

  • DappRadar ——另一个专门聚合区块链DApps数据的站点。通过不同的类别浏览协议，并根据特定的数据点对它们进行排序。

  • Cryptocurrency Investment Fund Graphs & Charts  ——cryptofundresearch.com提供了一系列免费图表，可以可视化全球加密货币投资基金行业的状况。

加密指标

  • Bitcoin Fear & Green Index ——一个检查比特币市场情绪的指标(也适用于一般加密货币)。

  • Relative Strenght Index(RSI)  ——显示比特币是超卖(RSI < 30)还是超买(RSI > 70)的重要指标。

  • Market Value to Realized Value Ratio (MVRV)  ——比特币的另一个超卖(MVRV < 1)，超买(MVRV > 3.5)指标。在coinmetrics上，我们可以在一些大型加密货币上使用这个免费指标。

  • Bollinger Bands  ——包括3条线：一个简单的移动平均线，一个正的和一个负的标准差(SD)。价格越接近SD线上方，市场就越超买。价格越接近低SD线，市场就越超卖。在tradingview.com上，我们可以在不同的加密货币上使用该指标。

  • Bitcoin’s Never Look Back Price  ——一个交互式图表，显示了比特币的最后一次是在一个特定的价格水平。一旦达到这个价格，它只会继续上涨，再也不会回到那个价格。

  • Bitcoin’s Power Law Corridor  ——比特币的幂律走廊是比特币自成立以来一直在价格增长的通道，当记录在平方根时间尺度上的价格时。

  • Exchange Reserves  ——显示在交易所持有的硬币总数的图表。更低的数字意味着更多的人把硬币放在钱包里，这意味着更低的抛售压力，反之亦然。在cryptoquant.com上，我们可以访问以下图表，了解几种主要的加密货币。

  • Combined Orderbook  ——Coinlobster.com提供了一个实时联合订单簿，其跨越10个现货和永久期货交易所。该网站还有其他免费功能，如“清算观察”和“观看鲸鱼”。

  • Bitcoin Fair Value ——顾名思义，该指标显示当前比特币价格是否被高估或低估。它是根据网络价值与交易比率(NVT)和链上数据计算的。bytetree.com还提供其他免费的比特币指标和统计数据。例如，基金持有多少比特币。
