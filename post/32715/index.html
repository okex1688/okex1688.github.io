渴望成为“区块链互联网”的 Cosmos 在 2021 年已经产生了巨大的增长，拥有超过 260 个区块链项目，币安智能链、Terra 和 Crypto com 等知名项目都是使用 Cosmos SDK 软件开发工具包构建的。根据数据，Cosmos 生态系统已经拥有超过 1700 亿美元的资金。

Cosmos 是一个由可互操作但自治的区块链组成的生态系统，随着跨链通信协议 IBC 的推出，Cosmos 在 3 月份进行了重大升级。IBC 协议允许独立的区块链之间相互连接以交换数据和资产。过去一个月中，在 20 多个启用 IBC 的区块链（如 Osmosis、Cosmos Hub 和 Terra）之间发生了超过 170 万笔跨链交易。

Cosmos 生态系统的核心贡献者 Tendermint 的首席执行官钟鹏（Peng Zhong）在接受采访时表示，IBC 将营造一个全新的跨链行业。 “到 2021 年底，我们应该会看到大约 30 个启用 IBC 的分区（Cosmos 中的独立区块链），我预测到 2022 年年底将超过 200 个。这就是创新将发生的地方。”

根据 DeFi Llama 的数据，以太坊是托管大多数稳定币、去中心化金融（DeFi）和 NFT 协议的智能合约平台，目前锁定的总价值为 1700 亿美元。 

钟鹏认为：“这是一种令人眼花缭乱的复杂性，甚至超出了以太坊的智能合约框架，以太坊有数以万计的智能合约，但在 Cosmos 中，可能有数以万计的以太坊，而每一个都有数以万计的只能合约。”

钟鹏曾经是一名网络开发人员，于 2015 年底加入 Tendermint，担任公司的首席设计官。此后，Tendermint 持续开发 Cosmos 基础设施和工具，例如 Tendermint 核心引擎、Cosmos SDK 和 IBC 协议，以帮助开发人员和项目快速构建自己的区块链，并实现跨区块链的数据和资产交换。

以下为经过编辑的采访内容：

为什么 Cosmos 被称为“区块链互联网”？

Cosmos 最初是创始人 Jae Kwon 和 Ethan Buchman 在 2016 年提出的一个想法，在白皮书中概述了“区块链互联网”是什么。2017 年的 1C0 由 Interchain 基金会发起，Cosmos 生态系统中的第一个区块链 Cosmos Hub 于 2019 年初推出。 

到 2021 年，用于构建 Cosmos 区块链的框架变得非常流行，使用 Cosmos 软件开发工具包 (SDK) 构建的区块链超过 250 个。虽然使用 Cosmos SDK 构建了很多有价值的区块链，但自 2016 年白皮书撰写以来，Cosmos 生态一直无法相互转移资产。 

直到五年后的现在，区块链间通信（IBC）才得以启用。这是今年 3 月在 Cosmos 中的第一个区块链 Cosmos Hub 上推出的。已经有 20 多个 Cosmos 区块链启用 IBC，并且这些区块链已经有相当多的流量。

Cosmos 与 Polkadot  相比如何 ？你认为 Polkadot 是 Cosmos 的竞争对手吗？

目前，Cosmos 和 Polkadot 之间没有竞争，而且我认为“旨在提高区块链之间互操作性的项目”不会真正竞争。 

Polkadot 和 Cosmos 之间的主要区别在于初始方向。Cosmos 专注于无需许可的跨链数据交换，它允许任何 Cosmos 区块链将数据发送到任何其他的 Cosmos 链上。它允许开发人员构建一条新链并将其 Token 发送到 Cosmos Hub，而 Cosmos Hub 将诸如 ATOM 之类的 Token 发送到这个刚刚创建的新链，这在今天的 Cosmos 中都是可能的。 

Polkadot 首先开始构建的是共享安全性，这意味着它不是一个无需许可的生态系统，无法让任何人都可以构建新链并在这些链之间来回发送资产。相反，它是一个有更多秩序的系统，它确实允许中继链，能够连接到多达 100 个其他平行链，但要成为平行链，你将需要很多 DOT Token，并能够赢得拍卖成为平行链。这意味着在 Polkadot 生态系统中，作为 DOT 的持有者，您拥有很多权力，您希望允许哪些项目成为平行链，哪些项目会为 Polkadot 带来价值。

而在 Cosmos 中，诸如 ATOM 的持有者，目前对哪些链可以连接到 Cosmos 不具有发言权。因此，在新链启动的那一刻，它们就能够连接到 Cosmos。 

当然，对于 Polkadot 来说，成为 DOT 持有者更有价值，因为您可以保护所有平行链。但在 Cosmos 中，我们相信开发者可以自由地构建他们所想象的任何东西并将其连接到 Cosmos 网络，这将为 Cosmos 带来更多的创新和增长。

Cosmos 生态系统中的安全性是如何管理的？

现在，每个 Cosmos 区块链链都有自己的安全性，并拥有自己的验证节点。在 Cosmos Hub 上，有 150 个验证节点，这些节点通常是为特定区块链运行基础设施的中小型企业。每个验证节点运行完全相同的软件，以确保创建的区块是有效的。 

现在我们正在创新，正在构建的技术将允许类似于 Polkadot 的共享安全性，但并不完全相同。对于 Cosmos Hub，我们正在研究跨链安全性，将允许 Cosmos Hub 验证节点同时验证另一个链或多个链。

什么是 Tendermint，它如何融入 Cosmos 生态系统？

Tendermint 本质上是 Cosmos 生态系统中最古老的公司。Cosmos 的联合创始人 Jae Kwon 和 Ethan Buchman 也是 Tendermint Inc. 的联合创始人，Tendermint 成立于 2014 年，甚至在 Cosmos 出现之前。 

我于 2015 年底加入 Tendermint 作为第一位员工，当我加入时，我们专注于企业区块链的 Tendermint 共识协议。但是在我们决定专注于 Cosmos 之后，我们的使命转变为构建网络，创建尽可能多的主权区块链，并允许人们通过在线控制自己的网络，即区块链治理，但同时允许这些区块链相互之间转移资产。区块链具有真正重塑全球协作、连通性和决策的潜力。

Cosmos 是由 Tendermint 的创始人创造的一个想法，但与此同时，Cosmos 比任何其他公司的想法都要大得多，而且由于我们工具的免许可性质以及这些工具具有的开源许可证，现在有许多公司使用 Cosmos 技术构建，为 Cosmos 做出贡献。

Cosmos 生态系统的一些关键特性是什么？

在更高层面上，Cosmos 是关于自治、主权和可扩展性的。

开发者在 Cosmos 中启动区块链并能够无需许可地发送 Token 的能力确实是完全独一无二的。这意味着资金不足的开发人员和团队，可以使用 Cosmos 进行免费构建。我非常热衷于保持这个核心价值，因为我以前是一名 Web 开发人员，并且通常在 Web 上开发东西，你几乎可以免费做任何事情。Cosmos 的这种完全免费和无需许可的性质将是 Web 3.0 发展的方向。

无论是在一个生态系统中还是跨链，DeFi 都存在风险，正如黑客利用智能合约中的错误或操纵预言机所产生的后果。而 Cosmos 已经在 IBC 上研发了 5 年，我们的技术以高标准而闻名于业界。许多顶级区块链项目都是使用 Cosmos 技术构建的，包括 Crypto com、Terra 和币安智能链。我们所做的一切都经过全面审核，以确保额外的安全性。

IBC 旨在成为类似于传统互联网的消息传输协议 TCP/IP 的框架。所以就像今天一样，有许多不同的 Web 开发框架，Cosmos SDK 或许也不是最终解决方案。但只要其他区块链框架采用 IBC 协议，那么就可以实现所有连接 IBC 的区块链之间的互通互联。 

其他希望采用 IBC 的区块链生态系统正在进行大量开发和研究。所以你可以想象一下，如果 Polkadot 采用了 IBC，未来将会是什么样的。Polkadot 使用 Substrate 框架，如果 Substrate 采用 IBC，这意味着每个 Polkadot 的平行链甚至中继链都将在未经许可的情况下与 Cosmos 互相传输 Token。你甚至可以看到 Polkadot 链可以通过 IBC 与 Kusama 链对话的未来。

随着越来越多的区块链决定采用 IBC 协议，这将是一个非常强大的网络效应。下一个网络可能不仅仅是需要连接到 Cosmos 链，而是：“哦，如果我采用 IBC，那么我可以连接到 Cosmos 和 Polkadot ”，而下下一个网络可能是：“哦，我可以连接到 Cosmos，Polkadot 和 Avalanche  ”。

采用 IBC 协议可以使多链变得轻而易举。

您认为推动 Cosmos 增长的一些因素是什么？

Cosmos 获得更多知名度的主要原因是，今年是“多链互通”兴起的一年——流行的区块链生态系统之间需要存在跨链桥的想法兴起。我们在 2016 年时就预测了这个未来，那时会有许多有价值的区块链，它们之间将能够相互发送资产。如今，我们看到了 DEX 的兴起，也看到了中心化交易所的衰落，这都归功于跨链桥的发展。 

已经有了这么多区块链，您是否可以看到行业整合？

从 Web3.0 的角度而言，Cosmos 和 Cosmos SDK，可以让小团队非常容易地创建新项目，并使 Cosmos 生态系统中的所有其他区块链都可以直接访问它们。

在技术层面上，许多人希望能够在不同的 Cosmos 区块链中使用多种协议，并以一种方式将它们连接在一起。如果他们想进行某种自动化投资，多个 Cosmos 链可以通过类似于以太坊智能合约的方式对您的资产进行操作并将发送回给您。 

但在用户层面，我预计很多用户对在数千个区块链上管理资产的想法感到麻烦。他们如何跟踪他们的资产？所以我们会有一些整合，但可能是发生在 UI 上或钱包上。钱包是一个不错的办法，可以隐藏区块链后端并完全脱离区块链的抽象化。 

您对 2021 年的亮点和 2022 年的预测有何看法？

Cosmos 生态系统已经完成了启用 IBC 协议，将创建一个全新的跨链行业。Gravity DEX 和 Osmosis 等跨链 DEX 让人们了解了多链世界的价值，也真正为 Cosmos 带来了大量用户。

NFT 领域还有很多事情要做。我有点担心关于 NFT 的预测。人们担心匿名地址有能力购买、出售和夸大 NFT 的价值，使其超出一个人的实际支付能力。这将引起监管机构的注意。但与此同时，监管是为了保护消费者。所以我们很高兴与监管机构合作，以确保我们正在构建的东西真正适合所有用户，而不是骗局。
