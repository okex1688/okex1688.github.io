我们中的很多人应该都还记得，当初那些加密钱包没有太多选择的日子。现在，我们有了很多选择！

以下是Bankless的职业作家和Metaversal的创建者——William M. Peaster对基于浏览器的10个加密钱包作出的分析和评价，以探讨Metamask是否已经失去了浏览器钱包的主导地位？

十大浏览器钱包

先说重要的：浏览器钱包是通过网络浏览器使用的加密钱包。

对大多数人来说，看到 “浏览器钱包 “这个词，首先会想到MetaMask这样的浏览器扩展项目，它能直接下载到Chrome、Brave等浏览器中。

但这里还有很多的浏览器扩展项目。除了钱包扩展，还有各种以浏览器为中心的服务，都属于 “浏览器钱包 “的范畴。

那么哪一个提供了最大的价值呢？为了回答这个问题，我从四个不同的钱包类型中评估10个不同的浏览器钱包项目。它们分别是：

浏览器扩展

  • MetaMask

  • Coinbase Wallet

  • GameStop Wallet

  • Rabby

  • Math Wallet

有内置钱包的浏览器

  • Brave

  • Opera

” 支持所有 浏览器 “的钱包服务

  • WalletConnect

  • Portis

浏览器兼容的桌面钱包

  • Frame

我们将逐一分析这些钱包的优缺点，并回答你迫切想知道的问题：“哪个是当今最好的浏览器钱包？”  

MetaMask

Website: metamask.io

支持 的浏览器 ：Chrome, Brave, Firefox, Edge

支持 的链 ： 所有以太坊虚拟机（EVM）区块链

MetaMask扩展是加密经济中支持最广泛的浏览器钱包，并已成为所有去中心化应用（dapp）的标准。

优点

  • 易于下载和使用

  • 很好地引导钱包恢复密钥+存储备份过程

  • 轻松导入现有账户或创建新账户

  • 与Ledger、Trezor和GridPlus硬件钱包兼容

  • 钱包内置加密互换系统

缺点

  • 无法像智能合约钱包那样找回丢失的账户

  • MetaMask非常受欢迎，以致其用户经常成为社交媒体上钓鱼者的目标

总体得分： 9/10

Coinbase Wallet

网站：coinbase.com/wallet

可 ️支持的浏览器 ：Chrome

️ 支持 的链 ：所有以太坊虚拟机（EVM）区块链

2021年11月，加密货币交易所巨头Coinbase推出其Coinbase钱包浏览器扩展。从那时起，Coinbase钱包很快就成为与MetaMask旗鼓相当的对手。虽然它发展速度很快，但目前该钱包仍处于MetaMask的阴影之下。

优点

  • 自主托管

  • 易于下载和开始使用

  • 通过Coinbase Pay轻松加载钱包

  • 提供一系列默认网络，如Ethereum、Arbitrum、BNB Chain，以及能够添加自定义网络

  • 在几种网络（以太坊、Polygon、Avalanche C-Chain、BNB Chain）上支持钱包内代币兑换。

  • 支持与硬件钱包（如Ledger）集成

  • 提供自动锁定计时器

  • 提供 “账号退出"功能

  • 支持非EVM加密货币（BTC、BCH、DOGE、LTC、SOL、SPL代币、XLM和XRP）。

缺点

  • 不够强调恢复密钥+存储备份过程

  • 支持的功能没有MetaMask多

  • 不能像智能合约钱包那样找回丢失的账户

总体得分：8.5/10

GameStop Wallet

网站：wallet.gamestop.com

可 ️支持的浏览器 ：Chrome

️ 支持 的链 ：以太坊、Loopring L2

GameStop是世界上最大的视频游戏零售商。最近几个月，该公司的web3团队一直在构建GameStop钱包，以期待未来涉足加密经济。该钱包的显著特点是直接支持Loopring zk-rollup L2。

优点

  • 自主托管

  • 易于下载

  • 明确自主托管的风险

  • 很好地引导钱包恢复密钥

  • 通过Loopring提供L2支持

  • 提供钱包内置加密货币兑换系统

  • 提供对硬件钱包的直接支持

缺点

  • 对dapp缺乏广泛的直接支持

  • 不能像智能合约钱包那样找回丢失的账户

  • 目前只支持两条链

总体评分：7/10

Rabby

网站：rabby.io

可 ️支持的浏览器 ： Chrome, Brave —— 很快也能支持Firefox & Edge

️ 支持 的链 ：超过三十条链，包括EVM的主流，如Ethereum，Polygon和Optimism，以及一些非EVM链。

Rabby是一个由DeBank团队创建的浏览器扩展钱包，强调多链支持，清晰的交易签名体验，交易前风险扫描等。

优点

  • 自主托管

  • 易于下载

  • 开源，任何人都可以检测和构建

  • 支持大量的硬件钱包+机构钱包+移动钱包

  • 很好地引导钱包恢复密钥

  • 提供钱包内置小部件系统

  • 在UI（用户界面）中突出安全审批系统

缺点

  • UI有点混乱

  • 对dapp缺乏广泛的直接支持

  • 不能像智能合约钱包那样找回丢失的账户

总体评分：7/10

Math  Wallet

网站：mathwallet.org

可 ️支持的浏览器 ： Chrome, Brave, and Edge

️ 支持 的链 ： 少数EVM链，Bitcoin、Kusama、Solana、TRON等。

Math Wallet是一个浏览器扩展加密钱包，支持超过100种不同的区块链。然而，与当前其他流行的加密钱包相比，该钱包在使用上相对混乱。

优点

  • 自主托管

  • 易于下载

  • 支持大量的链

缺点

  • 钱包创建过程和钱包UI感觉上不直观

  • 对dapp缺乏广泛的直接支持

  • 不能像智能合约钱包那样找回丢失的账户

  • 没有突出恢复密钥过程的风险和重要性

总体评分：5.5/10

Brave Wallet

网站：brave.com/wallet

可 ️支持的浏览器 ： Brave

️ 支持 的链 ： 支持EVM链，因此支持ETH，ERC20s等，以及Solana和SPL代币

2021年11月，加密友好型浏览器Brave推出了一款名为Brave Wallet的内置加密钱包。该应用可以让你直接在浏览器的体验中与Web3交互，让你一窥互联网未来的样子。它目前支持所有基于EVM的链和Solana。

优点

  • 自主托管

  • 易于下载

  • 提供钱包内置加密互换系统

  • 与Ramp和Wyre兼容

缺点

  • 对dapp缺乏广泛的直接支持

  • 不能像智能合约钱包那样找回丢失的账户

总体评分：8/10

Opera Wallet

网站: opera.com/crypto/next

可 ️支持的浏览器 ： Opera

️ 支持 的链 ： 以太坊，BNB链，Polygon，和Bitcoin（以及一些测试网）

2022年1月，Opera推出了一款内置了钱包系统的专用加密浏览器。该浏览器还与Discord和Telegram直接集成，给人一种强烈的加密原生感觉。

优点。

  • 自主托管

  • 易于下载

  • 提供钱包内置加密互换系统

缺点。

  • 对dapp缺乏广泛的直接支持

  • 缺乏对许多区块链的支持

  • 不能像智能合约钱包那样找回丢失的账户

总体评分：7/10

WalletConnect

网站：walletconnect.com

可 ️支持的浏览器 ： 任何浏览器

️ 支持 的链 ： 任何链

根据其网站说明，WalletConnect是一种“通过二维码扫描或深度链接将 dapp 连接到移动钱包的开源协议”。例如，Rainbow钱包是一款流行的以太坊钱包，但还不支持浏览器扩展。

WalletConnect使移动Rainbow用户（以及其他用户）可以通过任何浏览器轻松访问所有的dapp。

优点

  • 一个支持所有钱包的元钱包

  • 易于使用

  • 可在各种浏览器上使用

  • 开源

缺点

目前没有发现WalletConnect有任何相重大问题。

总体评分：9.2/10

Portis

网站：wallet.portis.io/login

可 ️支持的浏览器 ： 任何浏览器

️ 支持 的链 ： 以太坊、Bitcoin、Gnosis链、Polygon、EOS

Portis为dapp提供了一个 “在浏览器内使用邮件/密码登录的简单方法”，对主流用户来说这是直观和熟悉的。

优点

  • 可在各种浏览器上使用

  • 钱包内置备份功能

缺点

  • 钱包的用户界面有点混乱

  • 对dapp缺乏广泛的直接支持

  • 不支持许多区块链

总体评分：6.5/10

Frame

网站： frame.sh

可 ️支持的 系统 ： Linux、MacOS和Windows，带有Chrome和Firefox扩展

️ 支持 的链 ： 所有以太坊虚拟机（EVM）区块链

Frame是一个提供Frame Companion浏览器扩展的桌面钱包。这种连接器将Frame（桌面上的原生以太坊钱包）与基于浏览器的应用连接起来，在此之前，这些应用不能直接与Frame连接。

优点

  • 适用于所有的dapps

  • 注重隐私

  • 很多人非常喜欢这个项目

缺点

由于不断出现错误提示，无法深入探索该项目

总体评分：5.5/10

Conclusion

在评估上面列出的10个浏览器钱包后，目前认为WalletConnect（9.2/10）是最好的产品。因为它能让你随时随地连接，所以它是一款出色的“元钱包”。

其次是MetaMask (9/10)，它为加密经济中钱包的直接连接设定了一般标准，并且以后还可能设定新的标准。 

以下是按类别 依次 排名。

浏览器扩展

  • MetaMask - 9/10

  • Coinbase Wallet - 8.5/10

  • GameStopWallet - 7/10

  • Rabby - 7/10

  • Math Wallet - 5.5/10

有内置钱包的浏览器

  • Brave - 8/10

  • Opera - 7/10

” 支持所有 浏览器 “的钱包服务

  • WalletConnect - 9.2/10

  • Portis - 6.5/10

浏览器兼容的桌面钱包

  • Frame - 5.5/10

随着更多项目的涌入，预计浏览器钱包领域将在未来几年内迅猛发展。

Ledger、Rainbow和Argent现在将重心放在各自的原生浏览器钱包上，而这些努力可能只是未来的冰山一角。
