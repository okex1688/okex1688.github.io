▌ 格林纳达委任孙宇晨为常驻WTO代表，孙宇晨宣布波场基金会将于明年解散

12月17日消息，孙宇晨在公开信表示，波场基金会将于明年解散，与他相关的三个超级节点将很快退出，波场区块链上的27位超级代表将由波场社区选举产生。他将在退出Tron基金会的日常运营的同时继续推广Tron的品牌。在目睹萨尔瓦多成功使比特币成为法定货币后，他将重点关注拉丁美洲国家的加密合法化，成为格林纳达政府驻世界贸易组织 (WTO) 的大使。

区块链应用

▌ 高盛：区块链技术是元宇宙和 Web 3发展的核心

12月17日消息，高盛在一份研究报告中表示，区块链技术是元宇宙和 Web 3发展的核心。Rod Hall 领导的分析师在 12 月发表的一份报告中写道，它是唯一可以“独立于中央机构唯一识别任何虚拟对象”的技术，这种识别和跟踪所有权的能力对于元节的运作至关重要。该说明称，对于 Web 3，区块链允许“部分消除集中控制”。未来用户将能够在不需要第三方（例如 Meta、Google 或 Apple）的情况下登录。Web 3 是通过去中心化网络实现的第三代互联网服务。

▌ 金华首家区块链技术检察应用中心在义乌市成立

12月17日消息，日前，金华市首家区块链技术检察应用中心在义乌市检察院揭牌。该中心旨在通过产、学、研相结合的全新模式，融合平台建设、机制探索、理论创新等多项职能，加强区块链技术在法律监督领域的应用，以数字赋能为法律监督提质增效。区块链技术是数字检察的创新动力引擎，积极运用区块链等新技术推进司法机关、行政机关跨部门间大数据协同办案，突破执法司法数据共享问题，为“行刑衔接闭环管理”应用场景在更广领域、更大范围的提档升级提供“加速”动力引擎；区块链技术是强化监督的公共基础设施，区块链技术的融入，有助于改变司法机关的协作行为与信息互动方式，能够提高司法协作的效率和快速应变能力。

▌ Nervos 将与印度区块链加速器达成合作

加密货币

▌ 格林纳达委任孙宇晨为常驻WTO代表，孙宇晨宣布波场基金会将于明年解散

12月17日消息，孙宇晨在公开信表示，波场基金会将于明年解散，与他相关的三个超级节点将很快退出，波场区块链上的27位超级代表将由波场社区选举产生。他将在退出Tron基金会的日常运营的同时继续推广Tron的品牌。在目睹萨尔瓦多成功使比特币成为法定货币后，他将重点关注拉丁美洲国家的加密合法化，成为格林纳达政府驻世界贸易组织 (WTO) 的大使。

12月17日消息，俄罗斯央行行长纳比乌琳娜：对加密货币持怀疑态度，加密货币经常被用于非法操作，不欢迎对此类资产的投资，我们担心散户投资者在股市上的权利，俄罗斯的基础设施不应该被用于使用加密货币。

▌ 彭博社：币安离开新加坡真正原因是不遵守MAS反洗钱规则

12月17日消息，加密货币交易平台币安此前称出于全球战略、商业和发展方面的考虑停止在新加坡的所有加密法币交易活动，但据彭博社报道，币安在 2022年2月停止在新加坡所有业务的真正原因是他们无法遵守当地监管机构新加坡金融管理局制定的反洗钱规则。新加坡金融管理局董事总经理Ravi Menon在今年秋季中旬曾表示，在当地获得加密货币交易平台许可的标准非常高，去年大约有170家公司向新加坡金融管理局申请许可，但目前已有大约100家公司因为潜在非法资金流动问题而不得不退出申请，迄今只有四家公司获得运营加密交易平台许可。

▌ 未来金融科技签署合作协议在美国建造加密货币矿场

12月17日消息，未来金融科技集团今天宣布，公司与APC Service Ltd. 签署合作协议在美国俄亥俄州设立合资企业建造加密货币矿场。该加密货币矿场总规模为300兆瓦，第一阶段项目规模为50兆瓦。当矿场第一阶段建成后可部署S19蚂蚁矿机12000台，预计算力可达1.3EH/s。 根据协议，公司将在美国俄亥俄州设立一家全资子公司“ FTFT 超算中心”用来开展投资和建设加密货币矿场业务。APC有权在矿场第一阶段完工后365天内购买30%的FTFT 超算中心股份。

▌ 华尔街金融顾问LionTree正为其服务探索加密支付选项

12月17日消息，知名金融顾问和投行LionTree正在研究将加密货币作为一种支付选项。在其年终信中，董事长兼首席执行官Aryeh Bourkoff谈到了加密货币作为一种赋权个人的潜在工具，并探讨了加密货币领域的一些主流趋势，如DeFi和NFT。Bourkoff写道：“2021年，加密货币让自己变得不可忽视，其市值一度超过3万亿美元。”Bourkoff还暗示，将在不久的将来为其服务探索加密支付选项。据悉，LionTree参与了一些最知名的华尔街收购和IPO，今年曾帮助亚马逊收购米高梅电影公司。
