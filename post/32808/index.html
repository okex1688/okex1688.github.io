▌经济日报：莫给虚拟货币“矿场”留空间

1月10日消息，经济日报发文表示，莫给虚拟货币“矿场”留空间。严监管之下，虚拟货币规模化“挖矿”在国内被全面禁止，但个人“挖矿”行为在某些地方依然存在。对此，必须零容忍、全覆盖，形成持续治理的社会合力和高压态势，不给比特币等虚拟货币违法违规行为留下任何可乘之机。

政策

▌土耳其议会将权衡新的加密法案，不包括征收40%的税，并希望建立中央托管银行以消除交易对手风险

1月9日消息，据一位高级政府官员称，土耳其议会即将审查的加密货币法规不包括对加密货币征收40%的税。据悉，一个旨在优先考虑加密交换平台的透明度、安全性和可审计性的法律框架，将在未来几周提交土耳其议会。该框架的第二个优先事项是为区块链业务的增长创造适当的金融环境。执政的正义与发展党 (AKP) 的议会领袖Mustafa Elitas称，这还不包括所谓的40%的税收。即将出台的法律将旨在规范 本地加密货币系统、防止恶意行为、保护投资者和消除不满情绪。议会将对任何提议的法规拥有最终发言权。政府希望建立一家中央托管银行，以消除交易对手风险。

区块链应用

▌北京市文物局：推动区块链等数字技术赋能交易各环节

金色财经报道，根据北京市文物局发布的《北京市“十四五”时期推进国际文物艺术品交易中心建设规划》，将推动区块链等数字技术赋能交易各环节，推广移动数字支付方式。瞄准区块链等国际前沿科技，完善文物艺术品交易等场景中的系统集成应用，加强《基于区块链技术的文物艺术品追溯系统元数据》等国际标准的推广应用，为国际标准提供中国构建的规范方案与数据交流的“通用语言”，确立中国在全球文物市场的话语权与引领作用。依托区块链等核心技术，重点突破文物身份唯一性验证及溯源、禁止交易文物智能识别与检索等关键共性技术创新。

▌上海市经信委：布局数字经济、元宇宙、绿色低碳等新赛道

据“上海经信委”微信公众号消息，1月8日，上海市经济信息化工作党委、市经济信息化委召开会议谋划2022年产业和信息化工作。会议强调，要聚焦赛道布局、优势发力、企业引育、产业成长等解题破题，拿出务实举措，制定“一赛道一方案”，牵住超级应用场景“牛鼻子”，培育壮大发展新动能。

加密货币

▌巴基斯坦调查币安数百万美元的加密骗局

金色财经报道，巴基斯坦联邦机构在收到大量针对持续骗局的投诉后开始刑事调查，该骗局涉及误导投资者将资金从币安钱包发送到未知的第三方钱包。据报道，巴基斯坦联邦调查局 (FIA) 已向加密货币交易所 Binance 发出正式通知，以查明与该地区数百万加密货币骗局有关的联系。 国际汽联网络犯罪部门已向币安巴基斯坦总经理 Hamza Khan发出出席命令，以查明该交易所与“欺诈性在线投资移动应用程序”的链接。币安还被要求提供详细信息，包括欺诈者用来连接币安服务的 API 的官方支持文件和集成机制。尽管 FIA 主动封锁了与可疑应用程序相关联的银行账户，但通知警告说： “如果不遵守规定，国际汽联网络犯罪将有理由通过巴基斯坦国家银行对币安提出经济处罚建议。”

▌ 以太坊分叉Bellatrix将把当前链与新的PoS信标链合并，预计将在2022年6月实施并在12月开放信标链质押ETH取款

1月9日消息，以太坊信标链社区安全顾问superphiz表示，以太坊分叉Bellatrix将把当前链与新的PoS信标链合并。该分叉不会启用从信标链中提款，预计在合并后大约六个月启用质押的ETH提款。合并后，传统的GPU挖矿将不再在以太坊上进行，取而代之的是，网络将由权益证明（PoS）验证节点保护，他们将在合并时开始接收交易费奖励。 此外，superphiz预计，Bellatrix将在2022年6月实施，在12月开放信标链质押ETH取款。 此前消息，以太坊将ETH1和ETH2合并的硬分叉命名从Merge改为Bellatrix。

▌报告：金价下跌韩国投资者开始转向加密货币

金色财经报道，在美联储宣布逐步缩减加密货币之后，韩国国内市场也出现了类似的情况。之后提高基准利率两次以1％的水平，去年，韩国央行今年也暗示了加息。韩国银行行长李柱烈表示，“我们要根据经济状况的好转，适当调整货币政策的宽松程度。”结果，金价一直在六年来的最低水平挣扎。随后，虚拟资产成为流行的投资工具。一位韩国银行家指出，“我还不能说虚拟资产比黄金好，但如果世界继续流向 NFT 和虚拟资产，它就有可能像股市一样大。”

▌以太坊开放者会议：或将在合并之前启动最后一个合并测试网

1月7日，以太坊核心开发者进行电话会议。以太坊开发者Tim Beiko更新会议内容表示，以太坊2.0测试网Kintsugi进行了大量测试，但以太坊开发者MariusVanDerWijden再次成功破解网络，网络约13小时没有产生区块。希望再启动一个合并测试网，这可能是合并之前的最后一个。此外，将于1月21日进行下次电话会议。

▌资深交易员Peter Brandt：比特币投资者不要盲目“抄底”

1月10日消息，资深交易员Peter Brandt建议投资者不要盲目“抄底”BTC。他表示，未来几个月，长期持有比特币的人可能会有更多机会购买更便宜的比特币，他写道：“我使用过的一个神圣的交易规则是：永远不要在亏损的交易中再加钱。”他解释说，1980年白银期货突破50美元后，人们也在说同样的话。并强调说：“然后它跌到了3.65美元的低点，24年来一直没有回升。”不过，这名交易员澄清说，他并不预测比特币将走上同样的道路。

▌旧金山一男子通过特斯拉汽车挖取加密货币

金色财经报道，旧金山的一名男子Siraj Raval发现从特斯拉 Model 3 汽车挖取比特币和以太坊获利的方法。Raval 将图形处理单元 (GPU) 直接连接到 Model 3 的电动机，同时在汽车 FSD 计算机的帮助下运行加密货币挖掘脚本。Raval 表示，即使在市场调整期间，采矿设置仍然有利可图。从长远来看，他计划将他的轿车变成机器人出租车，以便能够在停车时挖掘加密货币。Raval 打算用他的大部分收入来投资一系列加密货币。

▌科索沃在镇压中没收了数百台加密矿机

金色财经报道，科索沃警方缴获了另一批 200 多台采矿设备，作为在电力短缺情况下遏制加密采矿活动的努力的一部分。据法新社报道，科索沃警方发表的一份声明显示，当局在 Leposavic 市查获了 272 台用于生产加密货币的设备。财政部长赫库兰·穆拉蒂 (Hekuran Murati) 也在社交媒体平台上指出，采矿设备的估计每月消耗量与 500 户家庭使用的电力一样多，价值在 60,000 至 120,000 欧元之间。穆拉蒂还表示：我们不能允许某些人以牺牲纳税人的利益为代价进行非法致富。内政部的数据显示，自本周早些时候针对矿工的突袭行动开始以来，新缉获的矿机总数已达到 342 台。周二，普里什蒂纳政府以寒冷的冬季电力短缺日益严重为由，停止了所有采矿作业，随后开始了镇压行动。

▌中国金融四十人论坛：DeFi具有一定中心化特征存在“共谋”风险

1月9日消息，中国金融四十人论坛发文《为什么说“去中心化金融”蕴含巨大金融风险》表示，DeFi容易给人一种完全去中心化错觉，但事实恰恰相反。因为算法不能预判所有可能发生的情况，这就导致DeFi系统中不可避免存在一定程度的中心化特征。首先，所有DeFi平台都有中心化的治理框架。其次，在特定制度安排下，决策权可能会集中在那些大型稳定币持有者手中。最后，实际上存在的中心化安排导致可能持币量大的少数账户“共谋”。 DeFi仍处于起步阶段，提供与传统金融体系类似的金融服务，自然也存在类似的脆弱性。

重要经济动向

▌ 财政部原副部长朱光耀：预计美联储将在4月提高基准利率，首次提高0.25%

1月9日消息，今日，在第六届新金融论坛上，财政部原副部长、中央财经领导小组办公室原副主任朱光耀表示，预计在4月份美联储将提高基准利率，甚至最早可能在3月份执行，首次或提高基准利率0.25%，全年或提升基准利率三至四次。

▌海通宏观：当前美联储加息的条件都已具备，随时可以开始加息

1月9日消息，据报道，海通宏观认为，当前美联储加息的条件都已经具备，随时可以开始加息。首先，美国通胀早就已经达到加息门槛。其次，失业率降至低位，甚至比2015年加息的时候还要低，想工作的人已充分就业。 最后，美国经济已经修复至疫情前。无论从通胀角度，还是从就业角度，或者是从整体经济修复角度来看，美联储加息的条件都已经具备，随时都可以开启加息。
