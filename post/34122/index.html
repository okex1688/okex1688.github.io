NFT 分析平台NFTGo发布数据显示，目前加密产业总市值高达两万亿美元，其中 NFT 市值逾 150 亿美元，约占 1%。

除了为人熟知的头像和艺术类 NFT 外，还有各式各样的 NFT 类型，包括：

头像 - BAYC, Punks, VeeFriends

游戏资产 - Loot, Sorare, ASm AIFA Genesis

实用资产 - membership assets

元宇宙资产 - Sandbox assets (non-land)

数字土地 - Sandbox LAND, Decentraland

社交 - HeadDAO, LinksDAO, MetaHero Universe

知识产权 - Adidas, Budverse Cans, Pepsi Mic Drop, Twitter collection

音乐 - Deafbeef, Eulerbeats, Kloud

金融/去中心化金融 - Uniswap liquidity positions, yield bearing NFTs

目前，NFT 总市值达 180 亿美元，其中头像类 NFT 占比最大，市值高达 80 亿美元，超过排名第二至第四位的各类 NFT 市值总和。

游戏、艺术和收藏品类 NFT 紧随其后，市值在 23 至 33 亿美元之间，它们也是大部分消费者最为关注的领域。然而，由于部分 NFT 能被归为多个类型，因此如何对各个 NFT 进行合理归类也极具挑战性。比如说，如果 Nike 发行了头像类 NFT 项目，这些 NFT 算作 IP 还是头像？随着 NFT 市场日臻成熟，这些类型的界定将更为明确。

从百分比上看，头像类 NFT 的市值占比近 50%，而其他子类的市值占比大多仅为个位数。

头部 NFT 类型将随着 NFT 市场的发展持续扩张，例如艺术类 NFT。与市值高达 1.7 万亿美元的全球艺术市场相比，艺术类 NFT 市场宛如冰山一角。与此同时，收藏品等类型的 NFT 将随着时间的推移日渐沉寂，而具有实用性的 NFT 则将脱颖而出，包括成员资格和现金流等其他认购权益。

目前，NFT 行业的交易额累计已超过 280 亿美元。随着新人源源不断地涌入，越来越多的公司利用现有 IP 或推出新 IP 发行非同质化 Token，这一行业仍将持续繁荣。与上图相似，头像、艺术和收藏品类 NFT 占据交易量大头。但更重要的是，从根本上看，这些 NFT 都是静态 NFT，这意味着它们除了代表相应资产的所有权和 Discord 访问权外，绝大多数都无法提供额外的实用价值。尽管静态 NFT 十分珍贵，但是随着投资者逐渐倾向于追求通过所持资产获得更优质的体验，人们对非静态 NFT 的需求将不断攀升，主要包括游戏资产、实用资产和音乐类 NFT 等。

除游戏类 NFT 外，其他各类 NFT 的交易额的百分比分布图在很大程度上与其市值分布图相类似。值得一提的是，游戏资产的交易额占比要低于其市值占比，这意味着许多收藏中的游戏资产可能尚未进行交易。

尽管累计数据能反映许多信息，但它们无法提供更具前瞻性的观察视角。30 天内交易额统计图显示，游戏类 NFT 的交易量激增，这也揭示了其市值较高，而累计交易量较低的原因。由于市值计算在定价时通常以该 NFT 收藏的最近成交价为准，因此市值的增速往往快于交易量。

一方面，游戏类 NFT 交易额的增长可能标志着游戏行业的进一步复苏，这也表现为同质化游戏 Token 对游戏产业市值增长的推动。另一方面，NFT 也可能是一项滞后指标，而同质化游戏 Token 的激增则是助推游戏类 NFT 发展的催化剂。在未来，同质化与非同质化游戏 Token 市场表现的差异将是度量游戏产业发展轨迹的重要指标。

除游戏市场外，NFT 艺术市场的交易额与其累计总量间也存在巨大差异，其中部分交易额来自头像类 NFT 项目。加密艺术销售将继续由四大平台主导，即 Art Blocks、Nifty Gateway、Foundation 和 SuperRare。 

最终，无形资产的长尾将实现链上 Token 化，NFT 市场中的类型也会日益丰富。在过去几个月中，尽管加密市场整体表现出下行态势，NFT 市场仍屡创新高。NFT 交易者和投资者都应当牢记以下两个看似对立的观点&ampmdash;&ampmdash;从长远上看，NFT 市场将持续发展，成长为千亿美元级别的市场；但在某些时期，NFT 市场也可能被普遍高估了。
