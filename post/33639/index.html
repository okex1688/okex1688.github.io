比推消息，据CNBC报道，谷歌云（Google Cloud）正在招聘一个区块链专家团队，以利用向去中心化 Web 3 应用程序过渡的机会。谷歌云将围绕区块链应用开展业务，在零售、医疗保健和其他行业大展拳脚。

2021年第三季度，谷歌母公司 Alphabet 约 82% 的收入来自广告业务。如果该战略进展顺利，将帮助谷歌多元化其收入，并进一步从远程计算和存储服务市场中瓜分市场蛋糕。

新的数字资产战略团队负责人是 Rich Widmann，他之前曾在 Google 的法律部门担任产品顾问，Widmann 在个人推特中确认了这一消息。

去中心化努力

区块链倡导者经常谈论构建去中心化的应用程序，即消除大型中介对某种权力或信息的控制。DeFi（去中心化金融）已成为加密市场中一个飞速增长的细分赛道，旨在将银行等中间商从传统金融交易（如借贷）中剔除。

借助 DeFi，被称为智能合约的可编程代码将取代传统金融中的银行或律师，智能合约写在公共区块链上，如以太坊或 solana，当满足某些条件时自动执行程序，无需中介机构。

这种去中心化应用程序的概念在宣扬 Web3 的技术专家中变得越来越流行，Web3 是一个脱离 Web 2.0 的去中心化互联网版本，Web 2.0 见证了用户生成内容（如博客和社交网络）的爆炸式增长，其中一些服务最终被大型互联网公司控制，包括收购了 Blogger 和 YouTube（现在是其最强大的业务之一）的谷歌。

如今，亚马逊、谷歌等云计算提供商运营为数百万客户提供计算服务的庞大设施就代表着一种“Web 2.0 的高度集中化”。

Widmann表示：“我们认为，如果我们把工作做好，就会推动权力的去中心化”。

谷歌云未来或将接受加密支付

谷歌云市场已经为开发人员提供了可以开始构建区块链网络的工具，客户包括 Dapper Labs、Hedera 和 Theta Labs 等区块链公司和几家加密交易所，谷歌还提供数据集，人们可以使用 BigQuery 服务探索比特币和其他货币的交易历史。

Widmann 暗示，谷歌云未来或可接受加密货币支付。他说：“我们可以做一些事情来减少一些客户在使用加密货币支付集中式云计算方面的摩擦”。他表示，数字资产领域的基金会和其他实体主要以加密货币为资本。

谷歌云计算首席执行官Thomas Kurian已将零售、医疗保健和其他三个行业确定为目标领域。Widmann 说，随着这些行业的客户选择采用区块链技术，谷歌可以提供帮助。

传统领域对加密好奇心的爆炸式增长

其他传统云提供商也对加密货币感到好奇。

根据技术行业研究机构 Gartner 的数据，亚马逊网络服务（AWS）在 2020 年以 40.8% 的份额引领云基础设施市场，该公司在 2018 年宣布了一项托管区块链服务，AWS 服务的网站将埃森哲、AT&T 和雀巢确定为客户。比推此前报道，基础设施初创公司 Alchemy 正在寻求成为区块链市场的 AWS，最近在 C 轮融资中筹集了 2.5 亿美元，估值为 35 亿美元。

Gartner 表示，微软在 2020 年拥有 19.7% 的份额，该公司在 2019 年推出了完全托管的 Azure 区块链服务，但在 9 月将其停用，理由是“客户兴趣降低”。

较小的云提供商也意识到了这个机会。

专注于中小型企业的 DigitalOcean 首席产品官 Gabe Monroy 在CNBC采访中表示：“我们在平台上有很多区块链和加密客户，这是我们 2021 年最大的增长部分之一，我们肯定会密切关注这个领域。”

加密公司也开始向软件开发人员敞开大门。加密货币交易所 Coinbase 成立了 Coinbase Cloud 部门，允许其他公司使用 Coinbase Cloud API 和区块链基础设施开发加密项目——包括加密支付、交易、数据访问和质押。

Coinbase 首席执行官 Brian Armstrong 在 11 月的摩根大通加密经济论坛上表示：“这有点像用于加密的 AWS，我们正在尝试将我们必须构建的一些服务外部化。如何存储加密货币并集成所有区块链、监控交易以实现反洗钱目的以及进行交易和质押等等，我们已经进行了很多艰难探索”。

据彭博社上周报道，谷歌高管 Shivakumar Venkataraman 已负责牵头了一个新的区块链团队，这与面向数字资产的云团队是分开的。

谷歌还在探索NFT领域。比推此前报道，YouTube 首席执行官 Susan Wojcicki 在致内容创作者的年度信中表示，YouTube 视频服务受到了 Web3 的深刻启发。她写道：“过去一年，在加密货币、非同质代币 (NFT) 甚至去中心化自治组织 (DAO) 的世界里，这是一个以前难以想象的机会，可以加强创作者与其粉丝之间的联系”。
