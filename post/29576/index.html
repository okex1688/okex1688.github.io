在讨论如何使用NFT之前，让我们花些时间了解它是什么？

到底什么是NFT？

NFT是一种数字资产，代表了一系列虚拟和现实的事物，不能用赝品替代或交换。它们在可以溯源到原始数据的区块链上创建，可以是艺术品、照片、音乐、视频、收藏品、纪念品、合同、优惠券、真伪证书、身份证件、健康记录等等。

直到 2017 年，随着 CrypoKitties 和CryptoPunks的推出，NFT 才开始备受关注，CrypoKitties 和CryptoPunks都是不同的数字交易卡的集合。根据nonfugible.com的数据，2018 年 NFT 的总销售额超过了 4000 万美元。截止 2020 年底，NFT年度采用率和销售额继续增长，逐渐成为了主流。

归功于NBA Top Shot 的推出，NFT的年销售额达到了3.38亿美元。而此时，部分NFT的单个售价已超过了5000万美元。在用户购买了NFT之后，可以将其在线或离线存储于数字钱包中妥善保管。

NFT的持续增长

根据《NFT2021年第二季度报告》，2021年第二季度的三个月窗口期打破了NFT的行业记录。nonfungible.com 的报告显示，2021 年第一季度，NFT交易总额超过了 5 亿美元。第二季度飙升了 40%，超过 7 亿美元。

第二季度的活跃钱包数量也创下了新的历史纪录，超过了175,000个，同时NFT显示出了强劲的价格增长。此外，用户数量也在增加。买家数量增长超过38%，卖家数量增长超过25%。这意味着NFT的采用率越来越高。

NFT 越来越多地被采用和销售量增加的很重要的原因是人类喜欢收集东西，同时NFT被编程为稀缺的。基本经济理论指出，几乎任何有需求的东西的稀缺性都会抬高其价格。

NFT 的其他属性是：

  • 数字持久性：只要它们所建立的区块链继续存在，它们就会持续存在。

  • 不可伪造：任何人都可以通过检查区块链来认证其真实性。

  • 可证明的所有权：如果 NFT 被编程为转售，则可以转让。

使用NFT促进小公司的业务

所有这些都表明小企业有机会利用当前的 NFT 热潮。我们总结了小公司采纳NFT的五种方式，使其发展业务并留住客户。

一次性促销工具的防伪发行：优惠券、促销折扣、活动门票、BOGO 优惠等等。区块链代码或二维码作为不可复制、防篡改的可访问折扣资产，具有广泛的用例。

建立客户忠诚度计划：考虑使用数字 NFT 忠诚度卡。客户注册后，他们会将其下载到他们的移动设备上。每当他们访问您的企业时，都会扫描忠诚度 NFT 代码以累积忠诚度积分。在某些阈值下，您可以推送奖励、折扣或区块链代币以便其在商店中使用。

公益营销与慈善支持：在美国人口外流之后，NFT被用来筹集资金来帮助保护阿富汗妇女保护她们的权利。类似的计划还可以在社区、城镇、城市或州一级复制，以刺激地域相似性营销或支持当地慈善机构。

为企业扩张筹集资金：如果企业正在考虑开设新的网点或扩大现有的规模。NFT可能是与传统银行贷款相比的低成本融资替代方案。我们可以把它想象成一个可以防骗局的Kickstarter。参与众筹的用户可以跟踪这笔钱，在满足活动开始时建立的智能合约条例的情况下，才能发放。

进行抽奖或游戏化促销活动：这类营销活动的运行成本可能很高，通常只对大型企业有意义，例如麦当劳的大富翁游戏、出版商结算所的抽奖营销。而NFT可以凭借一点点的创造力，为小型企业带来了高度参与游戏化营销的可能。例如，商会可以发起涉及数十家本地企业的数字 NFT 寻宝活动，以将客户流量吸引到每个地点。而在这一方面NFT的潜力仅受想象力的限制。

开启NFT之旅

如果小企业主精通互联网编程或者有机会接触一些大学实习生，就可以通过观看一些免费的文章和视频资源来DIY您的NFT。如果你没有时间、兴趣或耐心来完成所有的这些。可以根据自己的创造力聘请一名自由职业者来完成整个项目。

无论小型企业从哪里开始，都可以尝试一下 NFT。这对这些企业而言几乎没有损失，却存在一些潜在的受益。NFT的受欢迎程度还在上升。
