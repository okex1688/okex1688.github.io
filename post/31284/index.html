启动P2E游戏

游戏公会以相对较低的估值投资游戏的种子轮，然后通过Discord和社交媒体将游戏推广到社区。随着游戏发行NFT，公会购买和培育NFT，创造了对代币/NFT的更多需求，导致更高的代币/NFT价格 -> 围绕游戏的更多炒作 -> 更高的用户增长 -> 更高的协议收益 -> 更高的估值以及代币价格。

游戏公会可以从 a) 代币价格上涨和 b) NFT收益农业的早期代币激励中获利。这实际上是一个自我实现的预言，使公会能够加倍下注。另一方面，也有风险，游戏可能无法获得任何非公会驱动的有机增长，从而导致公会投资消亡。

现有游戏公会

游戏公会不是赢家通吃的市场，因为会有多个大型公会共存，共同投资。Yield Guild Games 和Merit Circle的资产管理规模累计超过20亿美元。这个数字是将分配给购买区块链游戏NFT 和代币的支配“火药”。公会AUM主要由原生治理代币组成，尽管两个公会都在讨论拥有更多样化的国库。

与传统的封闭式基金不同，公会DAO的结构是常青结构，没有终止日期。投资回报可以再循环到新的NFT/代币投资中，公会可以选择通过代币交换（类似于PIPE投资）通过出售国库中持有的一些治理代币来筹集稳定币。

YGG的完全稀释市值为90亿美元，MC为100亿美元。Avocado Guild筹集了1800万美元，GuildFi筹集了600万美元，Polemos筹集了200万美元，Good Games Guild筹集了170万美元。非上市公会也有望发行他们的原生代币，创造更多的资本需求，以满足不断增长的游戏内 NFT供应。

玩家（scholar）和玩家候选人

世界上有80亿人；超过60亿人生活在发展中国家或非发达国家。通过每天在家玩2小时的视频游戏，每天赚取1-2美元的副业收入对他们中的大多数人来说是轻而易举的事。Axie的DAU为 250万；YGG有5000玩家，Ready Player DAO有1800，Merit Circle有1500，GuildFi有1500。

SLP现在被菲律宾的一些商家和租房者接受。区块链游戏有潜力让60亿人加入加密货币，并通过DeFi提供对金融基础设施的无银行账户访问。大多数Axie玩家位于菲律宾和委内瑞拉。居住在印度、巴基斯坦、埃塞俄比亚、尼日利亚的大多数居民并不了解区块链游戏的副业创收潜力。我们还为时过早，玩家候选人增长的潜力巨大。

公会工具

管理公会运作很复杂；a) 采购、面试、聘用、入职玩家和玩家候选人，b) 监控玩家表现，c) 定期向玩家付款，d) 采购、调查、监控 NFT/代币投资。随着公会规模的扩大，他们需要工具来更有效地管理它们。一些自由开发人员一直在为公会开发Axie自动化脚本。BlockchainSpace最近筹集了375万美元，用于为公会构建终极ERP软件。他们已经在与2000+ 个公会合作，并且可以提供公会基准数据。

公会2.0

公会应该考虑如何从竞争中脱颖而出，并为他们的观众和游戏合作伙伴增加更多价值。以下是一些可以实现的潜在功能：

1、玩家市场： Axie经理和玩家没有玩家市场。运行Axie玩家计划（采购、选择、入职、监督学者、发送二维码、付款）几乎是一份兼职工作。独立经理可能宁愿让其他人管理运营。估计超过60%的Axie用户是玩家，估计超过80%的Axie玩家由独立经理或小公会管理。总共有120 万名玩家。许多即将推出的区块链游戏都计划启用玩家计划，并且需要一个可以审查、评分、筛选玩家的市场。一些玩家是有竞争力的游戏用户，可以通过他们的游戏技能提供更高的回报，但管理者可能还需要能够提供其他技能的玩家，例如社区管理、领导力、营销、公关、设计、数据、研究。随着区块链游戏生态系统整长，需要具有不同技能的玩家。

2、玩家评分： 拥有对玩家和玩家候选人进行评分的评分算法，可以提供一种数据驱动的、定量的、客观的方法来聘用、监控绩效和解雇玩家。该算法可以获取游戏内和游戏外的性能。赛外表现的衡量更具挑战性，大多数玩家候选人不会有任何游戏内数据。评分可以作为一种额外的激励机制来激励玩家候选人执行某些操作，例如每天发送一条Discord消息、设置 Metamask钱包、完成Serum交换、将资金桥接到Avalanche、玩特定游戏、回答一些调查问题。这可以使勤奋的玩家从人群中脱颖而出，并允许公会做出更多基于绩效的玩家聘用决定。

3、无担保DeFi贷款： 得分高的玩家和玩家候选人有资格获得无担保小额贷款。这可能是尝试无担保DeFi贷款的绝佳受众。

4、固定收益国库： 尽管散户/机构投资者可能有兴趣投资于特定游戏的产生收益的NFT，但他们可能没有时间或专业知识来设置钱包或选择NFT。拥有国库产品可以使用户能够投资任何金额并开始通过公会管理的NFT和玩家奖学金赚取收益。

5、NFT抵押贷款： 另一种玩家奖学金模式可以向玩家提供NFT抵押贷款。Axie玩家可以拥有一个他在整个奖学金期间一直在玩的Axies团队，以换取在此期间没有收到任何SLP付款。

6、游戏发现平台： 最大的公会可以成为web3时代的Steam，让玩家能够发现各种区块链游戏。

7、收入分享结构： 围绕分享玩家收入可以有不同的结构，例如动态份额而不是固定份额，或者在奖金池中保留一定比例的玩家收入以分配给表现最佳的玩家。

结论

区块链游戏愿意让游戏公会投资，因为它们提供必要的玩家、NFT、代币流动性来启动游戏。游戏公会最终将取代早期区块链游戏风险投资公司，并开始领投融资。来自即将到来的游戏公会和公会工具供应商的竞争对整个领域都至关重要，因为它促使最大的公会变得更具创新性和更有生产力。
