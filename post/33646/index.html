人人都在讨论的Web3是网络的未来还是流行词？

最近，科技、加密货币和风险投资领域都热衷于讨论一个流行词汇，而且当今的对话中充满了这个词，甚至如果你不把它添加到你的 Twitter 个人简介里， 就说明你不关心未来，这个词就是Web3。

它是一个涵盖性术语，用以表示那些希望消除互联网上中间商的理念。在这个新的领域，浏览网页不再意味着需要登陆 Facebook、 Google 或 Twitter。

你可以这样想: Web1 是上世纪90年代互联网的雏形，当时网页被认为是一种让大众获取信息的方式，但你只能通过你朋友的 GeoCities 导航页面才能更好的寻找信息，整体来说，网页信息是相当混乱和庞杂的。

然后，Web2 在2000年代中期开始出现。像谷歌、亚马逊、脸书和推特这样的平台的出现使得互联网的连接和在线交易变得更加容易，从而给互联网带来了秩序。但批评人士说，这些公司随着时间的推移同时也积累了过多的权力。

Web3 正是为了夺回一些权力。

“所有这些平台掌握在一小部分公司手中，虽然我们使用它们，而且我们为这些平台的成功做出了贡献，但我们并不拥有它。”来自Mat Dryhurst说，他目前是一名来自柏林的艺术家和研究员，在纽约大学教授有关互联网未来的课程。

因此，根据 Dryhurst 和其他 Web3 粉丝的看法，Web3 是互联网的新的迭代，同时新的社交网络、搜索引擎和市场也会出现，从而取代了公司垄断。

它们是去中心化的，建立在一个被称为区块链的系统之上，这个系统是比特币和其他加密货币的基础。你可以把它想象成一种账本，大量的计算机同步保存着任何人都可以搜索到的数据。它是由用户集体运营，而不是通过公司。参加者可获得代币奖励，这些代币可以用来对决策进行表决，甚至可以升值。

在 Web3的世界里，人们控制自己的数据，使用一个个性化的账户就可以访问社交媒体，发送电子邮件以及购物，同时所有这些活动记录都会创建在区块链上。

加州大学伯克利分校的企业家和区块链讲师Olga Mack说: “对于普通人来说，这听起来确实像巫术，但是当你按下开灯的按钮时，你了解电是怎么产生的吗？你并不需要知道电是如何工作的就可以了解到它的好处，区块链也正是如此。”

现在，重塑互联网的想法听起来像是遥远的数字乌托邦。但 Web3 正在推动新的对话，并吸引了大量来自加密投资者的资金。

Web3 起初令人困惑，但正变得越来越主流，科技公司也开始注意到这一点

Web3 运动的发展得益于NFT的兴起，NFT 是一种可以用加密货币买卖的数字收藏品。最近，一群加密爱好者联合起来，试图用数字货币购买一份美国宪法。他们组建了ConstitutionDAO (DAO 代表去中心化自治组织，同时也指在线的加密爱好者集体，他们聚集在一个区块链和代币管理的组织中，非常有 web3的特色。)

Dryhurst 认为试图解释清 Web3可能会令人苦恼，因为它是一个定义松散的术语，其定义描述会随着不同人而略有不同，但他说，所有新的技术前沿都有这样的问题。

“网络里每一次新发现在初期都令人困惑,”他说。

对于技术专家和加密学者来说，Web3多年来一直有一个理论上的宏大愿景。但是最近几个月，关于区块链未来的讨论已经在某些圈子里的科技会议和社交媒体上占据主导地位。它甚至迫使大型科技公司组建了专门研究 Web3 的团队。

这给 Web3 的发展带来了一定的讽刺意味: 支持者希望他们不再需要通过这些大型科技的产品分享照片、与朋友交流和在线购物，而是选择区块链上众多具有竞争力的产品。例如，每次你发布状态时，你就可以赚取一个代币奖励，从而拥有了平台的部分所有权，并有朝一日可以从中兑现。

理论上讲，这意味着我们不再需要向这些科技平台付费，而且可以免受其规则和限制。但一些大型的科技平台也在试图采用这种做法。

Twitter 高级项目经理 Esther Crawford 说，“这意味着创造的所有价值可以在更多人之间分享，而不仅仅是限于公司所有者、投资者和员工。”。

Crawford 说，Twitter 正在研究如何将 Web3的概念融入到社交网络中，比如未来可以通过一个加密货币账户，而不是 Twitter 账户进行登录并发布推文。她对未来的看法有所不同: 在标准 Twitter 的基础上引入了 Web3 功能，而不是去开发一个加密版本的 Twitter。

“很长一段时间以来，Web3 非常理论化,” 她说。“但现在正在形成新一股势头。”

Web3 会是新规范吗？

专家说，对于 Web3 爱好者最好的情况是，这项技术将与 Web2 一起并存，而不是完全取代 Web2。

换句话说，基于区块链的社会网络，交易和企业将在未来几年蓬勃发展。然而，根据技术学者的观点，彻底淘汰 Facebook、 Twitter 或 Goggle 的可能性不大。

“我无法判断谁会赢,” Dryhurst 说。“但是 Web2公司将会把 Web3的理念整合进他们的服务中，从而与其保持关联。”

他认为，许多人希望能够将自己的数据和交互历史记录保存到互联网上的任何地方，而不是保存在单一的网络平台上，也就是有些人所说的大型科技公司的“围墙花园”。

Dryhurst 说: “这与我们今天所习惯的体验完全不同。”

但他承认，无限的自由会导致一些令人不安的后果。”没有什么可以阻碍人们建立他们想要的社区，这一点令人兴奋，但同样，我也不无法阻止人们建立一些糟糕的东西。”

但总的来说，对参与规则的集体投票将比在今天主流社交媒体平台的用户体验要更好。

他表示: “如果我们停留在目前的范式中，那将在一个由少数人运营的少数公司公司中越走越远，在那个世界里，大型科技公司的问题更加严重。”

当然，另一个问题是政府监管。基于区块链的代币现在处于监管的灰色地带，但随着拜登政府开始为该行业制定新的规则，这种情况可能很快就会发生改变。

Web3 如何与互联网未来的另一个愿景元宇宙相契合？

Facebook 最近更名为 Meta，并表示其首要任务是建立**“** 元宇宙”， 它是一个人人都在虚拟现实中生活、互动和工作的数字化未来。

其特点之一是具有“强大的互通性”，即用户可以将他们的账户或头像从一个网站无缝转移到另一个网站或服务上，而不再需要每次访问新网站时必须登录由不同公司控制的账户。

这也是Web3的理念之一。

但真正的信仰者说，在 Web3的世界里，不管Facebook 如何努力地想成为下一代互联网的一分子，它都没有没有立足之地。

Williams说，“ Facebook 只会在其原有的基础上变得更大更强，而这不是网络空间应该有的管理的方式。”

Web3有可能只是一个被过度炒作的幻想吗？

Web3的怀疑者也到处都是。

研究法律和技术的康奈尔大学教授 James Grimmelmann 已经公开表达了他的疑虑。" Web3是一种雾件。”雾件指的是已经宣传但根本就不会问世的产品。

“这是一个听起来前景无限的未来互联网，它可以解决人们对当前互联网诟病的所有问题，即使有些问题是矛盾的。”

他说，如果一部分动机只是为了抵制把个人数据交给大型科技公司，那么区块链并不是解决方案，因为这将使更多的数据被公开。

“这是说不通的，这种观点认为互联网的问题在于太多中心化的中介机构。那如果我们不再使用不同的应用程序和网站，而是把它们都放在区块链上，同样这也是在把它们放在了一个地方。”

对于 Grimmelmann 来说，Web3代表了技术技术人员在互联网初期所追求的理想主义精神，即每个人都可以自由地使用信息高速网络！但这种精神在很久以前就被科技公司取代了。

互联网的发展是一场在碎片化和集中化之间的拉锯战，他说。当它向一个方向摆动过大时，其反作用力就会把它拉向相反的方向。

“区块链很有趣，而且以一种新的方式解决了一些难题,”他说。“他们可能最终会成为下一代互联网的工具包，但这并不意味着互联网将围绕区块链构建。”

但是，许多在新冠流行期间通过投资加密货币致富的人，正在四处寻找除了无聊猿——NFT 界的卡通“游艇俱乐部”之外的新的投资机会。

他说，现在的主要问题是，Web3主要还是理论性的。

“有很多人有钱可以投资,”他说。“但他们投资之前希望看到项目真正的愿景。”
