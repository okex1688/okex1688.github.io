今天你铸造 NFT 了吗？截至3月9日，全球持有 NFT 的用户已经上涨到195万人 ，创造了超170亿美元 的市场。

图源：NFTGO

自从 NFT 火出圈，各种品牌、明星纷纷跨界，利用自己的明星效应打造限量版 NFT，创造更大收益。

2021年10月，王家卫与苏富比合作，将电影《花样年华》1分31秒的未发行片段制成 NFT，取名为《花样年华——刹那》进行拍卖，最终成交价428万 港币。

但要说从 NFT 中收益良多的，应该还是各路艺术家，比如画师、音乐创作者、作家。因为有 NFT，更多独立创作家有了更多的机会向大众展示自己的作品，并从中获取更多可能的收益。

去中心化音乐平台 ROCKI 允许独立音乐人在其上创建和售卖音乐 NFT，以此拓展他们的收益来源。

这一方式越来越受各路艺术家欢迎，NFT 行业也涌入越来越多的创作者。但如果想玩转 NFT，首先你需要拥有一个 DID 。

和 Web2 中独立的、临时的身份不同，Web3 身份，即 DID 应该让用户完全拥有对自己数字身份和数据的所有权、管理权和控制权，也可以展示用户的链上声誉/信誉。用户可以自主管理例如：使用、隐藏、展示、撤销证明，同时别人始终能校验证明的真实性和有效性。

DID 的意义主要有：

  • 链上数据保证用户身份和行为的透明和不可篡改，建立信任基础 。

  • Web3 中的通用性具体由 dApp 数据 可以在各个项目读取调用体现。

在 NFT 的各类细分赛道，DID 也需要与其有机结合，并提供契合其需求的功能。

身份展示

PFP，Picture For Proof / Profile Picture，即图片凭证，可以作为链上身份的一种形式。

PFP 的热度在 NFT 细分赛道中一直居高不下，作为“蓝筹”的 BAYC、Azuki 等等更是持续性地在扩张影响力。自从推特支持头像展示 NFT 之后，其它社交媒体也开始对 PFP 类的 NFT 支持跃跃欲试。

图源：Hypebeast

PFP 作为 NFT 的最简单用途 ，也很好地起到了对 NFT 的宣传作用 。不同于其它类别的 NFT，因为 PFP 的简易性，使得其在各种场景下展示的门槛都极低 。毕竟使用一个心仪的头像是人类使用社交媒体之后就自然形成的习惯。

与此同时，DID 在 PFP 中的作用不可小觑，具体表现主要有两点：

  • DID 可以“包容”、“涵盖”PFP。例如，PFP 所具备的“展示”特性能够使得 DID 在表现形式上会更有特点。

  • DID 也能进一步在 PFP 的交易上增加透明度和安全性。例如，买卖双方可以通过 DID 进行身份与藏品的验证。

创作者经济

Web3 中的创作者经济和 NFT 密不可分，创作者经济的核心之一就是 NFT 化用户的作品 ，无论是文章、音乐还是其它形式的作品，而在此之上能进一步保护 用户权利的就是 DID。

作为链上身份系统，DID 对用户的保护是多元的 ：既可以对发布者的属性进行判断 ，也可以进行一定程度的 KYC ，保证用户和作品的相关性，杜绝盗版或者黄牛；在隐私方面，DID 可以很好的进行信息展示的选择 ，用户对自己的身份信息拥有控制权 ，而不是像 Web2 中互联网寡头控制了用户数据。

上文提到的 BNB 链上的音乐流媒体平台 ROCKI 就是 DID 应用的一个最好体现。其集成了本体搭建的 ONT ID（ont.id）框架，为音乐人提供可信、高效、低成本的验证以及作者凭证，以及解决盗版问题，并赋予NFT更高的价值——

作品 A 是 ROCKI 平台知名作家 X 发行的限量版 NFT，全球唯一，具有极高的鉴赏和收藏价值。通过身份验证，该条信息为真，那么售价 100个 ETH 不过分吧~

总结

归根结底，拥有一个 DID，不仅是对创作者身份信息的保护，更是对其经济收益的保护。不管是 NFT 的哪个细分领域，乃至整个 Web3、元宇宙，只要你在创作，那么 DID 都将成为全方位为你保驾护航的关键性一步。
