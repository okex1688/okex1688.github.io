原文标题：《ETH 还是 Solana 的 NFT：鸡生蛋的问题》

NFT 市场经过过去半个月的持续震荡之后，重新回归于平静，ETH 链上的 NFT 项目迎来了整个周中的寒冬，相比之下，作为最大的 NFT 交易平台 Opensea 支持了 Solana 链上 NFT 的交易，让更多独特的项目走入大部分人的目光，长期「霸榜」的 YugaLas 在近一周的交易量中也败下阵来，以 Okay Bears 为首的 SolanaNFT 开始崛起，甚至是 Okay Bears 的仿盘们都开始频繁登顶，各大社交媒体对于这个现象呈现出褒贬不一态度。Real Vision 的 NFT 专栏作者 Mr.Fox，总结了过去一周 NFT 市场动态，并且对 Okay Bears 所带来的「新风向」进行了深度的分析与反思。原文翻译如下：

回顾

简单来说，在过去的 7 天里，NFT 市场开始从上周遭受的混乱中，逐渐重建。目前看来，虽然进展比较缓慢，但是这并不意味一切开始停滞不前。回顾过去一周的种种迹象，我们把这些信息拆解来做一次分析。

在过去的一周里，我们看到整体 NFT 市场处于横盘的状态，但是仍然有一些 NFT 的地板价有所上升，让我们看看下面的截图。

但是本周一些 NFT 的情况变得更糟了，特别是在 Fidenzas, Ringers, 和 Azukis 这些项目地板价有了明显的下降。

对于 Fidenzas 和 Ringers 来说，这些损失是从上周两个项目经历的大规模上涨后的回调，同时也因为其他 NFT 迎来下跌的原因。许多人认为，随着人们对 NFT 市场大熊市的担忧加剧，投资者正在把他们的 ETH 从投机性的头像类 NFT 转向到如数字艺术品类等更长期的 NFT 中；因此 Fidenzas 和 Ringers 的上涨趋势如下图所示：

在负面新闻过后之后，还是有很多积极的信号。此外,Bored Ape Yacht Club, PROOF Collective, Moonbirds, Mutant Apes, CloneX, World of Women 和 Meebits 都呈现出了上涨的趋势，而 Doodles, VeeFriends 和 Gutter Cats 的跌幅也相对很小。在这些项目中，也有很多是因为传出了利好的消息的原因。比如，Yuga 终于通过发布 Otherside 游戏玩法的短片，让我们对他们元宇宙 The Otherside 有了一些清晰的认识。关于 The Otherside 的后续步骤的一些细节已经公布，尽管这些信息已经众所周知。其中的重点是，Kodas 未来将能够和土地分离，并且土地资源将是动态的可变化的，它们将随着时间的推移，被用来建设、交易和发展。

Moonbirds 发布了一条非常神秘 Gif 动图的推文。许多人认为这是对他们大约一周后的第一次筑巢的奖励。就项目的行动方面而言，这些就是上周 ETH 链上 NFT 市场的主要发展情况。

Okay Bears

Okay Bears 是一个在 Solana 上的头像类 NFT 项目，也是近期最为瞩目的项目之一。

当我们想到 NFT 时，很少会想到这个 NF 是在哪个链上发行的，这是出于以下几个原因：

它与 NFT 本身并没有多大的关系，除非它是一种有实用性的，要在许多应用或游戏中使用它来进行大量交易。然而，市场习惯的绝大多数 NFT 都是头像类，在购买之后，不需要经常转移。

然而，我们很少考虑我们的 NFT 位于哪个链的最大原因是，我们今天知道的几乎所有 NFT 都存在于 ETH 上，因此大部分人会下意识的觉得 NFT 都应该在 ETH 上发行。

如果上述就是为什么我们只考虑以太坊上的 NFT 的原因，那么我们必须要思考的下一个问题就是，为什么以太坊上的 NFT 可以持续的最受欢迎，而建立在 Solana 和 Immutable X（以太坊 2 层网络）等没有 gas 消耗的公链上的项目，却难以获得同等水平的吸引力？

有一件事是确定的，这不是一个关于实用性的决定，因为 Solana 和 Immutable X 提供与以太坊相同的体验，但要便宜得多。同样，这也不是完全意义上的价格导向的，因为尽管 SOL 的价格比 ETH 波动大得多，但通过前者节省的费用肯定会减少这种波动的影响。在 Immutable X 上，如果用户更喜欢使用 ETH，他们甚至不必使用 L2 的 Token IMX 进行交易。

有趣的是，我们在 NFT 的其他领域，例如二级市场的平台的纷争中，我们也看到了非常相似的动态。尽管 LooksRare 为用户提供了更加经济优惠的交易环境，但它积累的用户仍然只能占 OpenSea 每日活跃用户的一小部分。

那么，为什么会这样呢？当有更便宜的替代品随时可用的情况下，为什么 NFT 市场参与者始终选择更昂贵的方法？

我认为主要有这两个原因：

鸡生蛋的问题

是的，Solana、Immutable X 和 LooksRare 都比竞争对手为用户节省了更多的钱，但这些平台上仍然缺乏产品。就 Solana 和 Immutable X 而言，大多数新的 NFT 项目仍在 ETH 为基础，这使得这些替代链很难建立先发优势。因为人们只在 ETH 链上购买 NFT，就导致没有人在其他链上发行 NFT，因为没有人在其他链上发行，人们只能在 ETH 链上购买 NFT，也因为人们只在以太坊上购买，等等。

虽然有点超出了今天讨论的范围，但对 LooksRare 来说也是如此。从理论上说，如果执行相同的交易，无论是对买卖双方在 LooksRare 都比在 OpenSea 上更划算，但许多人并没有这样做，因为 OpenSea 的竞争对手缺乏真实有效的挂单。如果您想出售 NFT，您的首要任务是流动性（真正能够卖掉 NFT），因此您总是会选择 OpenSea 而不是 LooksRare。由于缺乏真实有效的挂单，买家很难找到更合适的交易。因为没有人挂单，没有人在购买，因为没有人在购买，就没有人在挂单，等等。

用户的主观体验

即使很多新闻已经介绍过了使用其他链可以比使用 ETH 链便宜得多，但人们仍然更喜欢以高昂的 gas 而闻名的 ETH 链。

对此，另一个理论是，归根结底，我们是人，我们的核心是渴望并享受丰富的体验。这一切都回到了起源。显然，选择 ETH 是投资 NFT 更昂贵的路线，但它仍然是顶级公链。似乎许多人宁愿支付额外的 gas，以换取将他们的资产放在世界上最大的公链上。这就是为什么劳力士的手表比其他的手表要卖出得多得多。是的，它们（手表）的功能都是报时，但是当你用劳力士看时间的时候，会有独特的体验。ETH 和其他品牌一样，目前，它是世界上 NFT 最大的连锁品牌。这在很大程度上与上面概述的例子相符。

我们非常喜欢在 Real Vision 的报道中使用「ETH 网络效应」这个词汇，因为每个人都已经在链上了，每个人都在继续在此基础上再接再厉的构建它，这就是为什么这么多人都希望他们的 NFT 也存在与 ETH。Gas fee 只是许多人愿意为更丰富的体验而加入这个网络所付出的成本。

这并不意味着 NFT 未来不会在其他链上或者以太坊 2 层网络上爆发。一切的契机只需要一个特殊的项目或产品，就能带来足够的流量，从而解决先有鸡还是先有蛋的问题。并且这种网络效应的飞轮开始在其他平台上出现了。

这就是现在我们看到的最热门的 NFT 项目吗，Okay Bears?

在过去的 7 天里，Okay Bears 变得炙手可热。成为了迄今为止最成功的 Solana 头像类 NFT 项目。尽管其他 Solana 的 NFT 头像项目也取得了一些成功。例如，DeGods、SMB 等。到目前为止，无论是在交易量上的成功，还是在 Solana 以往项目上Okay Bears 都是领头羊，它项目吸引了所有 NFT 社区的目光。

是什么让 Okay Bears 与众不同？为什么他们做得这么好？

可悲的是，答案是什么都没有。Okay Bears 只是另一个头像类 NFT 项目。他们的路线图或他们所描述的「蓝图」甚至不如今天大多数的项目做那么细致入微。他们承诺的 IRL 活动、合作、未来的铸造等——和其他项目都是一样的。Okay Bears 并没有什么特别之处。

我并无意抨击这个项目。有时最不承诺任何事情，只是让 NFT 的艺术和氛围来引导项目的方向，或许是更好的选择。在某种程度上，这似乎就是在 Okay Bears 中发生的事情。有些项目非常幸运，找到了一个神奇的方法，并最终流行起来。

许多对 Okay Bears 持批评态度的人已经注意到 Twitter 上的持有人在齐心协力的推广这个项目，并声称其增长或多或少是无机的。现在，我无法评论这个项目在 Twitter 上的真实性，但我能说的是，Okay Bears 似乎舆论中最大的赢家。

在 ETH 的 NFT 遭受了巨大打击的同时，整个 Crypto 市场也呈现出下行的趋势。市场参与者正在积极寻找寻找其他的收益渠道。因此，资金只会流向「叙事」更强的项目种。在这种情况下，Okay Bears 将是第一个顶级的 Solana 头像类 NFT 项目，因此，每个人应该接触不同链上的 NFT，每个人都需要在价格涨起来之前抓住一只熊。

当然，他们的官网上列出的 12 人团队，没有一个是实名认证的。对于一个项目的未来来说，这并不是一个好的迹象。

在过去的 24 小时里，Okay Bears 达成了惊人的成交量（比其他任何的 NFT 交易量都多），但从根本上来看，没有任何迹象表明该项目会成功。值得注意的是，NFT 的上涨通常都不是那么理性的，因此 Okay Bears 虽然没有创造出令人惊艳的图片质量，但不意味着它不会持续上涨。我不是说 Okay Bears 只是短期的增长，因为现在他们已经成为了龙头项目，但不可否认的是没有人不想在加密市场持续横盘的过程中，找到任何的赚钱的机会。
