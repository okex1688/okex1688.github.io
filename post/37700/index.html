5月10日消息，NFT二层扩展解决方案Immutable X称，在其被GameStop选为二层协议市场合作伙伴关系以来，Immutable X正在将其全球订单簿整合到 GameStop NFT市场，以及将Immutable X上现有和未来的项目引入GameStop，包括Gods Unchained、Guild of Guardians、Ember Sword、Playco、TikTok、Planet Quest、ESL Gaming、GreenPark Sports、Illuvium等。

今年早些时候，GameStop宣布将在以太坊上建立其第一个不可替代令牌 (NFT) 市场。作为首选的平台合作伙伴，Immutable X已被选中整合其以太坊第 2 层协议市场，并与GameStop 的团队密切合作，为GameStop客户带来下一代游戏NFT项目，为所有玩家创造参与和流动性的重要机会。

此次Immutable X将其全球订单簿整合到 GameStop NFT市场备受期待。两者未来的合作发展也让备受瞩目。

GameStop与Immutable X

据悉，GameStop 是一家从 2021 年初流行的 WallStreetBets 运动中受益的公司，从过去的错误中吸取教训后，他们决心转向数字空间。这一战略的一部分包括进军区块链、NFT和 Web 3.0领域。经过数月的招聘，GameStop于1月7日正式宣布了他们创建NFT市场的计划，其股价上涨了近 20%。

GameStop可以通过将数字游戏变成NFT，并允许用户在交易所转售和交易来改变这一点。这将在二级数字游戏市场创造一个价值数十亿美元的新产业，可以帮助游戏发行商和玩家赚取更多的钱。发行商可以从二级市场上销售的每款游戏获得版税，从而获得他们原本无法获得的销售额，玩家可以通过出售其不想再玩的游戏获利。市场预计，GameStop的市场朝着这个方向发展，将成为NFT前所未有的独特用途，可以将其推向新的采用水平，并可能使得GameStop成为Web 3.0的游戏巨头。

Immutable X是由Immutable开发的，它是以太坊上NFT的第一个第2层扩展解决方案，通过ZK Rollup技术，该项目在以太坊主网之上建设了第2层区块链网络，具有即时交易确认、高交易吞吐量和零 gas 费等优点，同时不会影响以太坊的安全性。据Immutable X官网介绍，Immutable X每秒钟最多可处理超过9000笔交易。

可以说，GameStop 的愿景是为玩家带来力量，并将下一代 NFT 游戏和数字所有权带入其社区。Immutable X是以太坊上非同质代币（NFT）的二层扩展解决方案，拥有世界上一些最大的 NFT 游戏，以及一个全球订单簿，全球订单簿将用于最大限度地提高Immutable X客户的流动性，可以为用户解锁数百万个现有NFT。两者的发展为NFT市场带来更多可能性，因而备受期待。

Immutable X的必要性

在进一步了解Immutable X的必要性之前，首先要认识Immutable。Immutable由Robbie Ferguson和James Ferguson兄弟于2018年6月份创办，总部位于澳大利亚悉尼，专注于开发让玩家可以真正拥有游戏资产的区块链游戏。早在2019年9月，该项目的1500万美元A轮融资即由腾讯最大的股东Naspers旗下的投资公司Naspers Ventures领投。

Immutable联合创始人Alex表示，去年游戏玩家在游戏内物品上花费了超过 1100 亿美元。尽管花费如此巨大，但用户却一无所有——他们无法将它们出售给其他玩家，无法将它们带到第三方市场，并且所有资产都可以被游戏开发商随意没收。如果你在游戏内物品上花费了 100 美元，我们相信你应该能够在完成后转售它（就像您可以出售您购买的任何实物资产一样）。NFT 是解决这个问题的绝佳解决方案，因为它们允许数字资产稀缺且可交易，而无需第三方（例如游戏工作室）成为每笔交易的中间人/账本/真相来源。

目前，大部分NFT交易都发生在以太坊主网，gas费居高不下，用户需要为每笔交易都需要付出几美元到几十美元不等的费用。同时，由于以太坊主网TPS过低，用户交易往往需要至少数十秒才能获得确认，这些问题对大部分普通用户购买与交易NFT形成较大障碍，制约了NFT在大众人群的进一步扩张与普及。因此，Immutable针对该痛点开发了Immutable X。

Immutable和StarkWare合作创建了Immutable X，将 StarkWare 世界级的第 2 层扩展知识与 Immutable 对 NFT 空间的深刻理解相结合。

该项目由以下主要组件组成：

ZK-Rollup L2 Scaling: 通过批量处理大量交易生成“有效性证明”, 然后在以太坊上验证证明。Immutable X的rollup允许每秒超过 9,000次非同质代币转账、交易和铸币，达到并超过主流NFT项目所需的规模。

API Abstraction Layer: 在Immutable X上，每个交互都被抽象为REST API调用。该领域的新进入者，例如成熟的游戏和内容公司，因无需直接与智能合约交互可以更快地推出更好的项目。

默认市场: Immutable X Marketplace是Immutable X协议上NFT交易的默认市场。

相比其它通用智能合约平台，Immutable X主要针对NFT场景，用于解决游戏、游戏市场和游戏玩家在创建和交易 NFT 资产时面临的各种核心挑战，许多技术方案与特点都针对这些场景进行了优化。

例如，允许用户通过输入特征筛选出自己想要购买的NFT资产；链游开发者可以在Validium模式下铸造NFT，NFT交易平台可以在zkRollup模式下运行；为NFT开发人员提供了一个无需许可的网关，集成基本的NFT相关功能模块，可以让他们在几天而不是几个月内构建一个主流的 NFT 游戏、dapp或市场等。

代币情况

Immutable X发行了代币IMX，旨在通过奖励 Immutable X 上的网络活动（例如交易、流动性提供和构建应用程序）来加速 Immutable X 的增长。

IMX是项目的原生代币，当前用例包括：

费用支付： Immutable X协议20%的费用必须以IMX代币支付。用户可以直接以IMX支付，不然Immutable X则会在公开市场上自动将购买货币（例如ETH）换成IMX付费。

抵押： 费用捕获机制一部分收到的IMX代币将被发送到抵押奖励池，并按比例奖励给合格的质押者。

_治理：_代币持有者将能够通过去中心化治理对与代币相关的提案进行投票。 提案可包括如何分配代币储备、对开发者资助、每日奖励以及代币供应变化等主题。

截至2022年1月10日，IMX的最大和总代币供应量分别为2,000,000,000和2,000,000,000。流通供应量为188,160,768（约占最大代币供应量的~9.41%）。

未来目标

目前，Immutable X正在努力吸引更多项目在该网络发行NFT或开发链游。上文提及与GameStop的团队密切合作，Immutable X能为GameStop客户带来下一代游戏NFT项目，为所有玩家创造参与和流动性的重要机会。

Immutable X官方指出，我们的计划是让 Immutable X每天支持超过 2 亿笔交易，同时消耗不到以太坊 30%的容量。我们的目标是让在以太坊上构建游戏、应用程序和交易所变得可行。我们的目标是让以太坊 NFT 的铸造和交易比交易传统数字资产更容易。
