纽约时报：链游扩张之路正遭遇传统游戏玩家的抵制

多年来，Christian Lantz一直在玩 STALKER，这是一款以世界末日后的乌克兰为背景的第一人称射击游戏，因其身临其境的角色扮演而大受欢迎。所以当这位 18 岁的高中生听说今年要制作游戏续作时，他知道他必须买下它。

直到游戏软件背后的乌克兰公司 GSC Game World 上个月宣布，新的 STALKER 将包含基于加密的资产，称为不可替代代币或 NFT。GSC 表示，在新游戏中，玩家可以为游戏中的角色购买和出售服装等物品的 NFT。该公司宣称此举是迈向被称为元宇宙的虚拟世界的“变革性一步” 。

Lantz被激怒了。他在 Twitter 和 Reddit 上加入了成千上万的粉丝，他们在 STALKER 的续作中反对 NFT。他们说，这家游戏制造商只是想从玩家身上榨取更多的钱。强烈反对让 GSC 迅速扭转了局面，放弃了 NFT 计划。

“工作室滥用了它的知名度，”住在安大略省的Lantz先生说。“很明显，这样做是为了利润，而不是仅仅创造一个漂亮的游戏。”

图片来自《STALKER 2》的剧照

一年多来，加密行业一直处于狂热状态。比特币和以太坊等加密货币的价值飙升。NFT等基于加密的资产已经起飞。Twitter 创始人 Jack Dorsey 最近将他的公司之一更名为 Block，以纪念区块链，这是一种为数字货币提供动力的分布式账本系统。Melania Trump拍卖了她自己的 NFT。支持者希望区块链将彻底改变从金融到社交媒体再到艺术的行业。

但对一些人来说，加密热潮已经走得太远、太快了。怀疑论者认为，加密货币和 NFT 等相关资产是数字版庞氏骗局，价格被人为夸大，超出其真实价值。一些人质疑加密货币和区块链这些模糊的概念是否具有任何长期效用。

没有比游戏社区更令人不快的了，用户与育碧、Square Enix 和 Zynga 等主要游戏工作室之间关于加密的冲突越来越多。 在许多遭遇中，游戏玩家都占了上风——至少目前如此。

“人们正在接受流行语，”拥有 300 万订阅者的游戏玩家和 YouTuber 博主 Mutahar Anas说。他说，那些在游戏中推动 NFT 的人“试图向你推销万灵油”。

最近几个月，至少有六家游戏工作室透露了在他们的游戏中添加 NFT 的计划，或者表示他们正在考虑这样做，通过区块链技术验证的数字资产提供真实性和所有权证明。 游戏制造商表示，这为游戏玩家提供了独特的数字物品，可以丰富在线市场上销售 NFT 的人。游戏发行商表示，未来 NFT 也有可能在游戏之间转移，这意味着来自一个游戏特许经营权的项目可能会影响另一个游戏的游戏玩法。

但玩家们表示，他们认为这些举动是公然的现金攫取。

“我只是讨厌他们一直想方设法以任何可能的方式来骗我们，”22 岁的游戏玩家Matt Kee说。Kingdom Hearts 表示正在推进 NFT。“我没有看到任何地方提到这对游戏玩家有何好处，如何改善游戏玩法。它总是关于’我怎样才能从中赚钱？’”

《Kingdom Hearts III》游戏图片  Square Enix出品

他们的大部分不满源于游戏中微交易的侵占。 多年来，游戏制造商通过让用户付费升级角色或提高游戏中的游戏水平，找到了更多从用户那里获利的方法。即使人们已经为游戏预付了 60 美元或更多，他们也被要求为数字物品支付更多的钱，比如服装或角色的武器。

在 2006 年的一个著名事件中，角色扮演游戏“The Elder Scrolls IV: Oblivion”向用户收取了 2.50 美元的费用，以购买他们角色的马的一套盔甲。

“‘只有几块钱，但我记得当时在想，‘他们为什么不给我们马铠？’”来自爱荷华州迪科拉的 31 岁的啤酒酿造商Eric Hild说。“‘为什么让我们为此付出代价？’”

游戏行业网站 Fanbyte 的游戏主播兼编辑 Merritt K 表示，在过去十年中，游戏玩家对这些公司的敌意有所增加，部分原因是微交易数量不断增加。因此，当游戏制造商将 NFT 作为买卖的附加元素引入时，她说，玩家“已经准备好把这些东西轰出去。”

这导致了游戏玩家的愤怒，这让游戏公司感到不安。12 月，Sonic the Hedgehog游戏的开发商 Sega Sammy 在用户的“负面反应”后对其 NFT 和加密计划表示保留。制作刺客信条等游戏的育碧表示，在上个月宣布 NFT 计划后，它误判了客户的不满程度。超过 90% 的观众不喜欢有关此举的 YouTube 视频。

“也许我们低估了反弹的强烈程度 ，”负责这家法国公司新区块链计划的育碧副总裁 Nicolas Pouard 说。

游戏公司表示，他们的 NFT 计划并非出于利润动机。相反，他们说，NFT 为粉丝们提供了一些有趣的收藏，并为他们提供了一种通过出售资产来赚钱的新方式。

“这真的是关于社区的，”移动游戏制造商 Zynga 的高管Matt Wolf说，他正在领导进军区块链游戏。“我们相信让人们有机会玩来赚钱。”

在过去的几年里，在游戏中拥抱加密的热潮愈演愈烈。一些开发者开始在区块链上构建游戏，让玩家可以轻松收集数字资产并证明他们拥有它们。其中一款游戏是CryptoKitties，这是 2017 年的热门游戏，玩家在其中收集数字猫咪，其中一些售价超过 100,000 美元。在新冠疫情大流行中，像 Axie Infinity 这样的基于区块链的游戏（玩家通过赚取和出售 NFT 来赚钱）也变得流行起来。

更大的游戏工作室现在正试图参与其中，尽管他们的一些加密计划仍然含糊不清。

育碧是第一家涉足加密领域的大型游戏发行商。去年 12 月，它宣布了一项名为 Ubisoft Quartz 的计划，以头盔和枪支等数字设备的形式推出了三套 NFT。NFT 在射击游戏 Ghost Recon Breakpoint 中免费提供给游戏中达到一定级别的玩家。该公司表示，游戏玩家可以保留这些物品或在第三方市场上出售。

2019 年洛杉矶电子娱乐博览会上的 Ghost Recon Breakpoint 游戏站

Pouard 先生说，到目前为止，已有 10,000 个数字钱包连接到 Quartz 平台，尽管育碧在第一批中仅铸造了 3,000 个 NFT。他说，这表明未来对更多 NFT 的兴趣。

Pouard 补充说，育碧最终计划从未来 NFT 的销售额中分一杯羹。“我们正在从专注于游戏的商业模式转变为专注于每个玩家都可以成为利益相关者的生态系统的商业模式，”他说。

将被 Take-Two 收购的Zynga在 11 月聘请了游戏行业资深人士 Wolf 先生领导一项加密工作。Wolf 先生说，目标是在区块链上创建新游戏，让玩家更容易获得、拥有和出售 NFT。他几乎没有提供有关这项工作如何运作的细节，包括 NFT 是否可以在 Zynga 游戏之间转移。

“我们仍在开发这一切，”他说。

其他游戏公司也涉足 NFT，呼应了加密货币如何为用户创造新财富。本月，Square Enix总裁Yosuke Matsuda在一封公开信中写道，创建区块链游戏可以让玩家赚钱。他说，这将成为公司的“主要战略主题”。

但随着来自游戏工作室的 NFT 公告的数量越来越多，玩家变得越来越恼火。在用户反对 Sega Sammy 的加密计划后，其一位高管在上个月的一次管理会议上表示，“如果这被认为是简单的赚钱，我想决定不继续进行。”

其他游戏公司已经站出来反对加密货币。微软 Xbox 负责人 Phil Spencer在 11 月告诉 Axios，一些以通过 NFT 赚钱为中心的游戏似乎具有“剥削性”，他会避免将它们放入 Xbox 商店。微软拒绝置评。

拥有在线游戏商店Steam的Valve也在去年秋天更新了规则，禁止允许交换加密货币或 NFT 的区块链游戏。 Valve没有回应置评请求。

游戏 Fortnite 的制造商 Epic Games 的首席执行官Tim Sweeney表示，他的公司将避免在自己的游戏中使用 NFT，因为该行业充斥着“各种棘手的骗局”。（Epic仍将允许开发者在其在线商店中销售区块链游戏。）

这种反弹影响的不仅仅是游戏工作室。受游戏玩家欢迎的社交平台Discord于 11 月在用户威胁要取消其通过加密计划的付费订阅后退缩。Discord 的首席执行官 Jason Citron在 Twitter 上取笑了该项目，引发了叛乱。

“虽然我乐观地认为区块链领域正在发生很多很酷的事情，但也存在很多问题，”Citron 先生在接受采访时说。

游戏玩家Mr. Kee表示，他将继续与游戏公司的加密努力作斗争。他说，STALKER 开发人员在 NFT 上的转变让他希望其他公司可以受到公众舆论的影响。

“这给了我一种很好的感觉，每个人都在反对这一点，”他说。“在过去的 10 年里，我们看到了各种各样的此类计划，我们对此感到厌倦。”
