A&T Capital 介绍了 NFT 应用变迁，并点明了重点关注的 NFT 领域投资机会。

撰文：Jasmine Zhang 与 Fiona He，A&T Capital 合伙人与投资经理

《三体》中，伊文斯创造了三体游戏，通过虹膜链接和整套体感设备的帮助，将虚拟世界作为先导，以三体文明的历程作为蓝本吸引地球文明社会精英去了解三体的高级文明。如果我们把目光移回现实生活中的人类社会，在近几个世纪科技的不断进步下，我们已经初步掌握了实现太空旅行、搭建虚拟现实的技术，并且在不断地向 Web3，去中心化物联万物的时代迈进。同时，我们也面临着地球资源的不断枯竭，生态环境的恶化，和一步步走向灭亡的太阳系—-星际穿越和虚拟现实冥冥之中变成了让人类长久生存并成为高等文明的工具。而在现实世界和数字世界必然交织的未来，我们又会以怎样的方式拥有属于自己的事物呢？

NFT，或许是答案。

NFT 有望成为最快落地的区块链应用

无论是游戏还是艺术品，NFT 都可以助其实现从现实通往虚拟的跨越。从艺术家们的数字作品，到职业运动员场上的高光时刻，再到电子游戏中的稀有卡片，越来越多不一样的目光开始聚焦到 NFT 的玩法上。而活跃于艺术、娱乐、甚至快餐市场的巨头们也开始把注意力投入到 NFT 市场上，如佳士得开始拍卖 NFT 艺术品、NBA 合作推出 Top Shot、迪士尼和 Taco Bell 也陆续推出周边电子收藏品。但 NFT 的发行并不止限于拥有庞大粉丝基数的专业玩家，由于 NFT 市场本身极广的受众面，人们也可以把任何他们认为有独特价值的资产变成无需许可的区块链可编程资产，并通过 NFT 平台铸造属于自己的 NFT，被发现进而进行拍卖。

自 2020 年 8 月再度爆发，NFT 成为不容忽视的赛道

随着 NBA Top Shot 去年 8 月公测，凭借 NBA 强大 IP，带领 NFT 迅速爆发。根据 Coingecko 数据显示，截止到 2021 年 6 月 10 日，NFT 整体市值已超过为 170 亿美元，约占整个加密货币市场市值的 1%，而去年 8 月之前，NFT 市场几乎小的可忽略。从 NFT 市场历史交易额看，根据 nonfungible 数据统计（未包括 NBA Top Shot，仅追踪以太坊上的 NFT 数据） ，2021 年第一季度 NFT 市场交易呈现出历史性高活跃度，并于 5 月第一周录得 1.73 亿美元历史最高的周交易量。虽然目前 NFT 交易量略显颓势，但已成为不可忽视的赛道。

下图介绍了 NFT 市场上几幅值得注意的艺术品，前两幅作品由美国圈内知名图形设计师 Beeple 创作，分别是 NFT 历史最高交易纪录的第一名和第四名。第三幅作品 Stay Free 的发行者的是 NSA 棱镜门揭秘者 Snowden 本人发布的，随后他将拍卖所得的收入全数捐给了版自由基金会（Freedom of the Press Foundation）。而第四幅作品 NYAN CAT 则是由一段十年前风靡网络的视频由来，随后衍生成了如游戏和加密货币等项目，也象征了流媒体在数字世界的生命力。

NFT 加强区块链与传统行业关联性，万物皆可且万物皆愿 NFT

新冠疫情 Covid-19 的爆发迫使传统商业模式由重线下转而重线上，人们将更多行为转到互联网上。比如，艺术品收藏和拍卖一直以线下展览、实物交割等为主，区块链的赋能或许将深刻改变这个行业：a）链上溯源且透明，鉴证真伪，解决信任问题；b）非实物交割，疫情下的倒逼转型。比如球星卡可以直接便捷地收藏在数字钱包；c）提高艺术品等交易的流动性；d）无论对于用户还是创作者，NFT 的链上交易平台提供了一个较低的进入门槛，更易于创造者及其作品价值被发掘。

NFT 作为区块链中的一个资产表示工具，核心在于将独特的价值 Token 化。在此之前，只有 FT，代表的是无差别的价值，而 NFT 进一步拓展了链上资产的种类和场景，使之前无法在链上表示的各种资产成为可能。

在目前新冠疫情倒逼传统行业进一步线上化和数字化的时代，NFT 是最快能落地和赋能传统行业的工具。

我们正经历从创造 NFT 到使用 NFT 到社交 NFT 的变迁

回顾 NFT 发展历史，每次牛市都有一个 NFT 代表项目引领市场。第一次牛市，以 Colored Coin （BitcoinX）为大家所知， Colored Coin 可标记多种特有资产，类似在 1 美元钞票上打上一行信息「可兑换 1 股 Tesla」。这次尝试让更多人了解了区块链和比特币除交易之外的可能。到第二次牛市时，加密猫的爆发造成了以太坊的严重拥堵，使 NFT 第一次进入公众视野。当时加密猫有近 2 万用户，贡献了价值 4000 万美元的交易。最贵的小猫「龙」以 600ETH （彼时折合 17 万美元）的价格售出。第三次牛市即我们所经历的阶段，NBA Top Shot 让 NFT 火速出「圈」，已有超过 100 万用户 ，总销售额超过 6 亿美元，目前最贵的销售单品近 39 万美金，是詹姆斯致敬科比的灌篮时刻。从历次牛熊变迁看到，NFT 从 0 到成为市值 170 亿美元的赛道，趋势从 NFT 创造（Colored Bitcoin，加密猫），到 NFT 使用（NBA Top Shot）的变迁。

我们预测，下一个 NFT 的爆点将会是 NFT 社交。

首先，NFT 为主的社交平台为 NFT 持有者提供了一个展示和炫耀的平台。无论你是投机者，还是投资者，都对一个提供藏品展示及其价格发现的平台有需求。在人类社会的社交环境中，爱好证明（Proof of Passion）给人们提供了交流价值和灵感碰撞的渠道。这些证明可以是一件艺术品，一双限量球鞋，或者一张唱片；在 1961 年，Keith Richards 在英国达特福德的火车站碰见了手里拿着一张蓝调 Chess 唱片的 Mick Jagger, 于是两人展开交谈。几年后，他们两个共同创建了滚石乐队，一并成为摇滚音乐历史上最有影响力的一对音乐创作家。如今，人们已经习惯了用流媒体平台听音乐，并且把收藏的艺术品保存在家里，而 NFT 的到来让他们能够更轻易地通过数字的形式向他人展示自己的爱好证明。

第二，NFT 的代币激励机制能进一步提高社交媒体内容的原创性，鼓励创作者提供更有趣，独特和有价值的内容，最终提高平台的整体价值。我们从社交网络的几个基本要素来比较目前常用的几个社交形式，要素如下 :

  • 传播范围：能触达多少到的人数和社交圈

  • 传播速度和频率：触达方式是否便利从而提高速度和频率

  • 独特和原创性：触达内容是否独特有趣和真诚

第三，社交平台让内容创造者们有了自由推广和分享的渠道，降低了创造者进行展示的门槛，并且能够跳过中间人与粉丝建立更加直接、透明的交流。对于创作者来说，他们可以随时随地通过社交平台发布作品相关的 NFT 来筹资进行创作，而这些 NFT 形式的「作品代币」也会随着作品集一同升值，并且可以作为粉丝们的爱好证明于社交平台上进行展示。Instagram 和 Twitter 是目前中心化社交平台的代表，他们的亮点在于鼓励用户用简短的字数以及图片传递和分享信息，并利用庞大的子社区和精准的算法向用户推荐他们爱好相关的内容；对于 NFT 的社交网络平台，我们认为出‘圈’的机会也存在于能够让用户简单传递并展示 NFT 爱好证明，提供准确地 NFT 内容和价值索引，并且合理设计社区 / 作品代币的平台。

目前 NFT 的用例仅是拼图的一部分，当各个拼图模块逐渐拼合完整，才能看到 NFT 整体的价值。截止 2021 年 6 月 10 日，公链方面，以太 NFT 生态最为活跃，按历史总销售额 排名，前 15 名中有 10 名是以太坊生态上的 NFT 项目。综合交易平台方面，OpenSea 历史总交易额最高，超过5 亿美元 。应用项目方面，总交易额超过 1 亿美元的只有两个项目：Flow 上的明星项目 NBA Top Shot，历史交易额排名第一，达到 6 亿美元；另一个是 CryptoPunks，凭借独树一帜的像素艺术头像，总交易额超过 3 亿美元。可以看出，目前市场最为买单的 NFT 代表了两个极端：有独特且受众广的 IP 资源（与现实世界极强关联）和网络世界独有的加密艺术（与现实世界极弱关联）。

NFT 领域的投资机会

目前 NFT 赛道发展还在早期，项目标的集中在垂直领域，主要涵盖游戏以及艺术品。我们重点关注以下方向：

  • 底层：关注 NFT 底层协议规范。

  • 平台：关注综合性平台，比如是否包含便捷地 NFT 铸造、全面的搜索引擎、传染性的社交或拥有高价值的 IP 资源；

  • 应用：关注两方面与 DeFi 的结合，一是 NFT 作为底层资产衍生的新 DeFi 生态，二是 NFT 作为价值表示工具展现的更复杂的可组合性。

我们从中筛选出了几个重点关注子赛道如下。

底层机会：NFT 的底层协议规范

底层协议运作在公链之上，一定程度上决定了上层 NFT 应用的发展空间，比如 ERC721 是最严格的 NFT 标准，每个智能合约仅对应一个 NFT，无法定义游戏中的 FT 部分，这限制了游戏的场景，比如 1000 件同一等级的盔甲实际是 FT。而 ERC1155 解决了这个问题，允许智能合约代表一系列 FT 的集合 NFT，但缺点在于无法追踪集合里单个资产的信息。

NFT 的可组合性也极大程度上取决于底层协议，尝试包括：ERC998 允许任何一个 NFT 拥有其他 NFT 或 FT，包含多个 ERC721 和 ERC20 代币；Fractional 项目允许拆分 NFT 的所有权，转化为 FT 提高流动性；RMRK 项目在 Kusama 上尝试跨链的 NFT。

还有一些讨论中的协议在探索特定场景下的使用，比如 EIP2615 尝试定义 NFT 的抵押和租赁、EIP1523 尝试定义 NFT 保险单、EIP2981 尝试定义 NFT 版税等等。

总结来看，协议规范各有优劣，ERC721 由于出现较早，成为了试错成本最低的协议而被广泛运用。从需求来看，底层协议需要开放性（开发者友好，限制少，想象空间大）、可组合性（跨链，拆分重塑等）、定义信息的完整性等。

平台机会：NFT 铸造

目前 NFT 的铸造发展非常早期和精英化，匮乏普惠大众和成熟的上链流程，存在链上 NFT 和链下资产脱钩的问题。

首先，大多 NFT 平台的铸造仅向小部分艺术家开放，比如 MakersPlace 和 SuperRare，且筛选标准不透明，由于平台自身需要流量，因此偏向已经比较有名气的艺术家。参照 NFT 研究者 Kimberly Parker 最近的分析，65% 的 NFT 销售价格都低于 300 美元，考虑到 Gas Fee 根本无法赚钱，因此在所有通过筛选的艺术家中，只有少数有名的艺术家比如 Beeple 才能真正享受 NFT 带来的技术变革红利，而无法更大程度赋能整个行业。可以思考 NFT+DAO，如何让 NFT 更加开放和去中心化。

其次，上链铸造成 NFT 过程中，艺术家仅通过完成一系列 问卷 ，并勾选是否原创等信息供平台后台查阅。缺乏一个完整的数据库和确权过程。

最后，大部分 NFT 的底层资产和数据在链下，而 NFT 能多大程度锚定其代表资产取决于数据储存方式，比如 Aavegotchi 可以完全在链上存储所有的媒体和元数据。而 Mintbase 使用以太坊和 Arweave 组合来铸造 NFT；SuperRare 使用 IPFS 存储文件及文件的元数据。

应用机会：NFT 赋能 DeFi，相辅相成

DeFi 当前的借贷和交易等主流应用仅支持 FT，和 NFT 的结合仍在一个割裂独立的状态。早期的尝试有 grab.finance （DeFi）的 cryptowine 形式（NFT）：用户通过在 Uniswap 上提供 Grab/ETH 代币对流动性，获得一定的 tickets，得到的 tickets 可以换取代表随机分数的 cryptowine NFT，而交易手续费则注入进奖池，最终按分数高低平分奖池奖金。这里 NFT 在中间只是可有可无的尝试。

也有项目试图使 NFT 作为 DeFi 底层资产，丰富 DeFi 生态和扩张资产类别，但 NFT 由于流动性较差，对上层 DeFi 应用构成了挑战。尝试有 Whale 项目，基于质押「具有价值的」NFT 发行 FT 代币 Whale，价值判定比较粗暴，NonFungible 每个月为其质押资产池审计，主要依据过去一段时间的各相似 NFT 的交易价格做 估值 ，延时及滞后性严重，目前很难在此之上建立有意义的 DeFi 生态，目前 Whale 资产池估值仅不到 5000 万美元 。然而，一旦能够在 NFT 的流动性和定价上找到解决方案，不仅能建立起新一代 DeFi 借贷及交易（比如新的「MakerDAO」、「Compound」、和「Uniswap」），还能撬动传统领域艺术品、收藏品等弱流动性资产的资本市场。

DeFi 项目的一些 NFT 尝试也值得关注。比如 Aavegotchi NFT，通过抵押 Aave 借贷平台上的付息存款凭证 aToken 获得（DeFi)，向游戏场景拓展了 NFT 的用途。比如 Zapper，通过与平台的交互获得 NFT，但 NFT 使用场景还未打开。还有一些尝试展现了使用 NFT 的真实需求，比如 yinsure.finance 将独特的保单以 NFT代币化 ，可在二级市场流动，增加了保单的流动性；比如 Uniswap V3 将 LP Token 以 NFT表示 ，定义了不同做市策略下的流动性价值，从而使复杂的金融场景随着 NFT 的表示开始呈现在我们面前：多头寸、自动 Rebalancing、借贷、对冲等。

总的来说，我们关注深挖 DeFi 中有必要结合 NFT 特性的场景，而不单单只是简单的在 DeFi 上叠加 NFT 概念，寻找可以被 NFT 表示的价值，能扩大 DeFi 底层资产或加强 DeFi 的可组合性。

NFT 的一些其他机会：游戏与文化

人们在钱包里持有艺术品，但他们除了收藏和展示，目前还没有更高效的使用价值，而游戏可以提供艺术品目前无法做到的互动性及参与度。据 Forte 统计，自 2012 年开始游戏产业的年度收入都在电影和音乐产业的总和之上。NFT 如何赋能游戏行业，老生常谈，过去我们看到的游戏多是将游戏内道具 NFT 化，或是传统游戏区块链化，但我们也在其中一个内嵌市场看到了另外一种思路。据 Statista 统计，2020 年英雄联盟全球总决赛最高时收看数 达到 了 4600 万 。试想，如果能够推出记录电竞运动员精彩一刻（Godlike，Legendary 等） NFT 形式的收藏品，其市场潜力不亚于 NBA Top Shot。而且电竞市场受众和币圈用户特征重合度较大，非常年轻且易于克服区块链使用门槛，教育成本较低。

当前大额的 NFT 交易，仍局限于艺术品交易，背靠的是大 IP 以及知名创作者，获益人仍局限在以欧美审美为主流的小圈子内。这既与现实世界相似（对比欧美剧与国产剧，NBA 与 CBA，德甲与中超），也可比对当前币圈 DeFi 版图与区域基金抱团的意识形态。而中国在国际上影响与日俱增，亚洲区块链基金期待获得来自全球基金与创业者更多的认可，既然有资本也有广阔的市场，我们应具有足够的文化自信，透过 NFT，向世界输出我们自己的东方审美和评判价值。行业内，如果我们看公链、DeFi、二层、跨链赛道等，金字塔顶端的大多是欧美项目；在 NFT 这个冉冉升起的极易出圈的赛道，我们期待看到更多的亚洲创业者能够抓住机会。文化与文明，一直是一个进化的存在，而这个进化的载体是人。我们期待我们这一代币圈耕耘者，可以在发现并支持强大自信的自有文化输出者，更透明和有效的与全球玩家平等的竞争。
