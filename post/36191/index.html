SeeleN主网验证人3月30日宣布，共同出资成立SeeleN Turbo生态基金。SeeleN DAO生态项目扶持计划随之启动，旨在为优秀的开发者和生态项目提供基金扶持，扩展SeeleN DAO生态规模。

对于社区认可的优秀开发者或项目，SeeleN Turbo不仅在孵化期对其进行基金资助，还将在上线并获得社区肯定后，持续进行基金奖励。另外，对于已在其他网络上线的优秀项目，SeeleN Turbo也将提供基金吸引项目入驻生态。

公链竞争一直是区块链领域关注的重点，2021年，更是竞相发力，在生态建设方面，各公链项目都取得了一定成绩，并进入生态爆发期。多链世界已经从理想走入现实，一年内公链数量增长近7倍。在以太坊日趋完善并饱和的前提下，EVM兼容是众多生态优先选择的一条道路。SeeleN网络自更新POS共识机制以来，便支持EVM兼容，并在2021年成功上线主网，引入DAO治理模式，正式开启生态建设。

目前SeeleN DAO拟布局的生态领域包括NFT、区块链游戏、能源、DeFi、文娱、DAO概念等，在此前召开的项目生态大会中，已经公布了一些领域内项目的具体信息。

CandySwap

CandySwap是SeeleN首个生态项目，目前承接了SeeleN生态内资产交易的主要功能。此前SeeleN Turbo基金已宣布投资25万美元用于支持CandySwap发展和后续功能开发。

SeeleN Wallet

SeeleN Wallet是SeeleN生态钱包，目前正在进行封闭测试，将于近期上线。SeeleN Wallet集成了资产管理、应用入口、支付、资讯、社交等功能，并具备兼容性、易用性、开放性等特点，为生态用户提供完善的基础服务。

Project Fantasy

Project Fantasy是SeeleN生态的GameLauncher平台，将致力于SeeleN在区块链游戏领域的持续探索。平台创新性地推出了通行证功能，让玩家可以免费体验平台内游戏，并决定游戏发展方向。在游戏阵容方面，已有两款平台游戏确定在近期发布，更多游戏开发和引进工作也在持续进行中。

Project Lending

Project Lending是针对SeeleN生态金融功能拓展而开发的项目，前期主要以借贷功能为主，并提供极低的抵押担保率，最终形成为个人和机构提供流动性的衍生品交易市场。

Project Stablecoins

Project Stablecoins将为生态提供算法稳定币支持，并为用户存放和制造稳定资产提供流动性，还将支持具备跨链功能的稳定币兑换。后续将支持用户通过抵押不同资产铸造稳定币或合成其他资产。

Project Carbon

Project Carbon是一款以碳资产交易为核心功能的能源领域项目，将打造碳资产形成过程中存储、网络、计算、应用四个层级的数据账户协议族，并保障碳资产相关数据与计算的多方协同与互认。

Project Video

Project Video是在线视频和流媒体相关的区块链应用，依托去中心化的区块链网络，通过点对点的传输方式，来让内容在网络上传输，观众作为节点贡献多余带宽，观看高质量视频的同时还可以获得奖励。

Project Metaverse

Project Metaverse将打造元宇宙中的身份基础设施，为个人用户创造身份标签，并链接其他领域的产业和机构。项目不仅着眼于元宇宙，更将在Web3.0时代建立网络共识。

Project Nature

Project Nature专注于能源项目这一传统领域的区块链化，打破能源项目投资高门槛和封闭性的限制，将能源投资彻底开放给大众，并促进市场更加活跃。

Project Asset Management

Project Asset Management是生态内的去中心化资产管理平台，用户可以通过平台分散投资，管理被动收益并最大化资产价值利用。平台还将支持跟踪他人投资策略，与交易所、借贷平台、资产协议进行外部集成，实现多策略投资。

Project Insurance

Project Insurance专注于针对用户投资行为的保险服务，通过去中心化的基础设施，避免了信任问题，并且无需检查用户身份和信用。项目将在前期与生态内DeFi功能进行深度耦合。

Project Society

Project Society将针对社交领域进行区块链改造，把资料和信息的控制权归还给个人，并为贡献者提供奖励。Project Society将努力终结收割用户信息变现的社交时代。

Project DAO

Project DAO是一个DAO概念项目，由国际知名机构发起。项目旨在通过DAO的方式，促进各部门之间，各领域之间，乃至各国之间的合作，实现可持续发展的目标，并将以合作者的身份参与联合国一系列青年活动。

在如今的区块链领域，新公链生态内循环逐步完善，与比特币的资产解耦已愈发明显，生态的成熟完善和用户圈层的独立，让生态突破了和比特币价格变动的强关联，并逐步走出适合自己的发展道路。但新公链的踊跃是对区块链现有架构的进化而不是对原生系统的否定，正如生产力创新、生产关系透明和充分竞争自由是加密世界发展的原动力。

SeeleN DAO进行生态建设以来，已经与众多项目与开发者建立了良好的合作关系，并将持续在不同领域进行深度探索，拓展生态规模。SeeleN DAO将在生态发展之初，利用高激励政策带动整个生态整体的资产活跃，并在具备一定规模后拓展路径向外部扩张和输出，积极实现生态创新与差异化，并最终实现社区共建的协同自治生态。
