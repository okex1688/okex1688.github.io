概述

Solana 是下一代互联网平台（Web 3.0），其结构是基于区块链的公开交易加密网络。 Solana 的开源软件网络将全球分散的计算机协调成一个完全统一的、用户拥有和运营的云平台。 Solana 支持单层区块链上的高速和低成本交易，从而减轻了对其他网络通常需要的额外扩展解决方案的需求。

SOL 是 Solana 网络的原生代币，代表生态系统中的一部分所有权。现阶段，SOL 代币主要有以下五个功能：

  • 为去中心化应用程序 (dApp) 提供支持

  • 付款

  • 支付网络费用

  • 通过 Staking 质押提供网络安全

  • 促进网络治理。

Solana的区块链解决方案

Solana 网络旨在为开发人员提供高性能的 Web 3.0 云平台，该平台在 Layer 1 区块链上提供可扩展性。这种方法旨在优化速度、成本、并改善网络去中心化程度，消除以太坊 Layer 2解决方案的复杂性。具体来说，这些特征可以量化为：

  • 速度：400 毫秒出块时间

  • 成本：每笔交易 0.000005 SOL（约 0.001 美元）。

  • 去中心化：2,242 个全球节点。

利用具有八项核心创新的独特架构方法并针对不同的技术权衡进行优化，Solana 最终达到了网络高性能水平。但是，这些“权衡”也带来了一些问题，比如网络去中心化程度相对较低（与以太坊等网络相比），结果就是网络存在中断风险，例如 2021 年 9 月 14 日发生的 DDoS 攻击，一度导致 Solana 网络的全部功能停摆了约 24 小时。尽管如此，Solana 仍在吸引喜欢此类网络设计的用户，目前他们正在通过启用以下功能来扩大 Web 3.0 区块链云计算市场的规模：

  • 新用户：Solana 使以前服务不足的成本敏感用户能够访问现有的加密应用程序，如去中心化金融 (DeFi)、非同质化代币代币 (NFT) 和其他 Web 3.0 DApp。

  • 增加使用量：Solana 通过降低交易成本和提高交易速度，使用户能够更积极地采用 Web 3.0 应用程序。

  • 新应用：Solana 可以支持更多新应用并使自己逐步走向主流市场。

技术趋势经常以周期形式出现，Solana 对 Web 3.0 云计算市场的影响与此前推动整体互联网采用的计算平台创新具有相似之处：  • 从大型机到PC：在大型机时代，计算机体积庞大且价格昂贵，这限制了它们接触选定大学的学术研究人员的机会。个人电脑 (PC) 的出现降低了成本并为新一波消费者开放了访问权限，从而改变了当时的状况， Solana 较低的费用在今天也产生了类似的影响。

  • 从本地到云端：在本地网络时代，任何一家组织都必须自己拥有和管理服务器、或是依赖其他计算机基础设施才能访问软件，但是随着云计算的到来，人们已经不再需要涉及本地托管的流程，同时又能大幅提高应用程序敏捷性。 Solana 通过为开发人员提供一个足够强大的单一平台来运行 Web 3.0 DApp，同时又消除了 Layer 2 扩展解决方案的复杂性，从而产生了类似的影响。

  • 从桌面到移动：在桌面时代，占主导地位的应用程序基本上都是为固定的计算平台而设计，因此可以支持的应用范围也十分有限。但是现在，我们的主要生活都在移动设备上进行，也为开发人员提供了一个移动计算平台，从而改变了过去桌面应用的状况，也使得需要基于移动端的新应用程序开发成为可能，例如 Uber。Solana 则是通过提高 Web 3.0 云可扩展性并为开发人员提供新的基础设施媒介来设计 DApp，在今天也产生了类似的影响。

对于 Solana、以太坊等网络来说是，Web 3.0 云计算市场一个巨大且快速增长的机会。许多 Web 2.0 云提供商——包括 AWS、谷歌云、微软 Azure、阿里云和腾讯云——通过服务不同的客户需求取得了成功，Web 3.0 云市场正在遵循类似的路径，以太坊和 Solana 等网络正在成为该领域的主要竞争者。

实际应用

Solana 云经济在过去一年中快速增长，推动生态系统总价值超过 1100 亿美元，目前整个生态系统主要由 Solana 网络（SOL）、以及在网络上发行的资产组成，SOL 社区对生态系统具有管辖权。其中包括在 Solana 网络上发行的数字业务（智能合约 DApp）、数字美元（美元支持的稳定币）或数字商品和艺术品（即 NFT）等资产。以下是这些类别之间的价值分布概览：

  • Solana Network (SOL) 以 730 亿美元的市值占据了数字经济的大部分价值，当前在整个生态系统价值中的占比约为 65%。

  • Solana 网络上的数字业务 (DApp) 在整个生态系统价值中约占 1/3，具体数据大概在 350 亿美元左右，占比或约 30%。

  • 在 Solana 经济中流通的数字美元（稳定币）价值大概有 40 亿美元，约占整个生态系统经济价值的 3%。

  • 基于 Solana 的数字商品和艺术 (NFT) 价值约为 10 亿美元，约占整个生态系统经济总价值的 1%。

自网络启动以来，开发人员对 Solana 的兴趣在过去一年中急剧上升。开源开发人员的关注度是 Web 3.0 云网络发展的一个关键指标，因为正是这些 BUIDLer 创建了最终将用户吸引到 Solana网络 的应用生态系统。为此，Solana 一直在积极推动黑客马拉松活动并以此吸引越来越多的开发者注册和提交项目，进一步推动 DApp 生态系统的发展。

Solana 是一个用于开发 DApp 的智能合约平台或通用加密云平台，Solana 社区在网络启动后很短时间内便建立了一个由 500 多个 DApp 组成的大型生态系统。Solana DApp 设计的领域主要有DeFi、Web 3.0 和 NFT ，具体包括：

  • DeFi：用例包括开放式订单簿交易所、自动化做市商、借贷平台、资产管理软件和支付。

  • Web 3.0：用例包括Solana 域名服务、数据隐私网络浏览器和链下数据预言机。

  • NFT：用例包括铸造平台、市场、游戏、音乐流媒体、社交媒体和分布式自治组织 (DAO)。

在过去一年中，Solana DeFi 协议的总锁仓量 (TVL)所上升，目前已经占到所有加密项目总锁定量 (TVL) 的 6% 左右，取得这样的成绩主要归功于用户对 Solana 日益增长的兴趣、不断增加的应用程序、强劲的 SOL 网络性能以及 Solana 链上数字美元发行量增加（Solana 网络上的稳定币总市值已经达到约 40 亿美元）。

金融并不是 Solana 数字经济中唯一增长的领域——在消费方面，Solana 的链上表现同样不俗。最近几个月，Solana 链上代表数字财产（例如艺术品、消费品或其他资产）所有权的非同质化代币 (NFT) 也有所增长。基于 Solana 区块链的 NFT 总市值已升至 10 亿美元以上， Solana NFT 月销售额已增至约 2.5 亿美元，而独立买家数量在过去三个月中增加了约 4 倍，达到约 60,000 人。

随着用户、应用程序和使用量的持续增长，Solana 的网络收入在过去一年中有所增加。到 2021 年 11 月，Solana 用户为发送付款、运行智能合约程序、发行新资产和对网络共识投票等交易支付的总费用达到约为 45 亿美元，比年初增长了 100 多倍。

Solana 协议通过消除 50% 的总交易费用来减少 SOL 供应量，而剩余的 50% 则会支付给网络的计算基础设施提供商（验证者），质押 SOL 代币可以获得激励。

由于费用收入减少了供应量，而支付给质押者的费用会刺激更多人持有代币，在这种代币经济下 SOL 的基本价值模型得到了有效支撑，并使数字代币 SOL 成为一种生产性资本资产。

图 8：SOLANA 链上每月收入、以及按月计算的年收入趋势

由于 Solana 采用了供应通缩的机制，未来 SOL 总供应量数字不会确定。 虽然 SOL 总供应量将根据网络收入而有所不同，但新代币的发行速度已被写入 Solana 协议。

Solana 主网测试版于 2021 年 3 月上线，当时 SOL 的供应量为 5 亿枚。 2021 年 2 月期间，SOL 供应通胀率从 0.1% 调整为初始通胀率 8%，之后 8% 的初始通胀率又按照 15% 的年化反通胀率（annualized dis-inflation rate）下降，直到达到 1.5% 的长期通胀率。 截至 2021 年 11 月，Solana 网络代币流通量约为 5090 亿枚 SOL，总通货膨胀率估计约为 7.3%。

Solana 简史

Solana 创始人是 Anatoly Yakovenko， 他在 2017 年底首次构思了该项目及其技术方法的关键要素。 Solana 网络的内部测试网和官方白皮书于 2018 年初发布，Solana 软件协议主网 Beta 版于 2020 年 3 月推出。

在2018 年至 2020 年初期间，Solana 背后的开发公司 Solana Labs 进行了一系列融资交易。

竞争优势

Solana 具有以下几个竞争优势：

  • 团队：Solana 拥有一支强大的核心技术团队，由来自高通、谷歌、Dropbox 和苹果的前员工组成。

  • 技术：Solana 拥有一种新颖技术的区块链解决方案，可提供高可扩展性和低交易费用，使其有别于其他网络。

  • 社区：Solana 拥有一个由用户、开发人员、行业合作伙伴和投资者组成的强大而活跃的社区。

  • 生态系统：Solana 建立了一个庞大且快速增长的 DApp 生态系统，新用例不断涌现。

潜在风险

  • 在网络层面，Solana 也面临着一些潜力风险，同时也会有一些来自加密行业竞争对手和外部因素的压力，其中包括：

  • 竞争网络：Solana 面临来自其他具有智能合约功能的区块链竞争，例如以太坊、币安智能链、Internet Computer Protocol、雪崩 Avalanche 等。

  • 经济和估值：与以太坊等其他区块链相比，Solana 网络费用收入相对较低。除非费用因新应用程序、使用量增加或费用提高而增加，否则可能无法支持基于现金流的估值模式。

  • 中心化程度：如果有一天，一个或一组实体控制了大部分 SOL 代币供应，Solana 网络可能会变得过度中心化。此外，如果想要参与 Solana 网络，用户可能需要更专业的设备来提高支持，这意味着该网络无法吸引大量用户，继而可能无法实现较高程度的去中心化。

  • 监管不确定性：Solana 可能面临各种监管机构的审查，迄今为止，监管机构通常只将比特币和以太坊确定为不属于证券的资产。

  • 网络安全：Solana 利用多种新技术方法有利也有弊。首先，Solana 共识机制使用了一种未广泛使用的新区块链技术，可能无法按预期运行；其次，网络底层的密码技术可能存在缺陷，包括影响 Solana 网络功能或使网络容易受到攻击的缺陷。最后，Solana 的经济激励措施可能无法按预期发挥作用，这可能会导致网络不安全或表现不佳。

总结

毫无疑问，诸如 Solana 这样加密网络平台能为下一代网络和云浪潮提供支持。Web 3.0 DApp 可以为用户提供许多创新福祉，并且也产生强劲的消费者需求。

然而，目前许多头部区块链云计算平台性能并不好，交易处理缓慢且成本昂贵，有的还需要复杂的扩展解决方案提供支持。这些问题限制了加密经济被大规模采用。 Solana 采用了独特的技术方法，优化权衡以优先考虑高可扩展性，使链上交易比许多其他网络更快、更便宜。 Solana 获得了庞大且快速增长的活跃用户群、开发者社区和 DApp 生态系统，同时也一直在推动网络快速增长。

Web 2.0 云计算市场年收入已经达到 3500 亿美元，市场价值更是高达 4.6 万亿美元，Solana 已将自己定位为传统市场的挑战者，而且已经开始攫取 Web 2.0 的市场份额了。
