9月24日，中国人民银行等十部门联合发布《关于进一步防范和处置虚拟货币交易炒作风险的通知》，明确指出开展法定货币与虚拟货币兑换业务、虚拟货币之间的兑换业务、作为中央对手方买卖虚拟货币、为虚拟货币交易提供信息中介和定价服务、代币发行融资以及虚拟货币衍生品交易等虚拟货币相关业务活动涉嫌非法发售代币票券、擅自公开发行证券、非法经营期货业务、非法集资等非法金融活动，一律严格禁止，坚决依法取缔。对于开展相关非法金融活动构成犯罪的，依法追究刑事责任。

就在《通知》发布后两天，9月26日，Huobi Global快速响应发布《关于逐步有序清退中国大陆地区存量用户的公告》，公告显示Huobi Global已于2021年9月24日（UTC+8）停止了中国大陆地区新用户注册。对身份认证为中国大陆地区的存量用户，计划于2021年12月31日（UTC+8）24:00之前，在保证用户资产安全的前提下完成有序清退。

12月14日与12月15日，Huobi 将分别停止中国大陆用户的充币与币币交易业务。Huobi 的清退工作的重头戏正式到来。12月31日 Huobi 将关闭人民币的OTC业务。至此，Huobi 全球站的中国业务将彻底关闭。

 2020年，Huobi 曾披露拥有1500万用户。排除其中一些无意义的纯羊毛数据的，考虑中国用户的占比很高，本轮清退规模至少是数百万量级以上，绝对是加密行业乃至商业历史上罕见。

据吴说区块链描述，“Huobi 与众不同的是采取了彻底坚决清退的策略，几乎不留任何余地，内部强调“无论是谁也不能阻拦清退工作”。一个显著特点是，用户哪怕使用境外公司主体注册，但主要注册/最终受益人是中国大陆，那么依然不被接受。”

因此将资产存放在火币的用户，当前最好的做法就是尽快完成提币、转移资产，而不是到「Last Day」再进行操作。那么，接下来的问题就是，火币中提取出来的资产，如何妥善保管。

用户存放在交易所的资产，为了流动方便，交易所会将相当比例的资产存放在自身的热钱包中，而热钱包的安全系数较低，在过去几年的时间里，不乏头部交易所资产受到黑客攻击的新闻出现，那么有人会说，我手中的资产并没有多少，放在交易所里。交易所会赔的。头部交易所可能出于维持业务的需求，对用户进行赔偿，而小交易所，则可能直接选择跑路：

无数前车之鉴告诉我们，不要心怀侥幸心理，在当下时间节点，你的资产，应该自己做主，自己保管！是时候选择一款适合自己的硬件钱包以存储资产了。

Ledger系列

Ledger当前推出的硬件钱包有两种：Nano S、Nano X

Nano X实际上是Nano S的升级版，在实际使用上，两款的区别不算太大，因此Nano S完全作为Nano X的平替版。

整体来说，Nano X在使用便携性上会比S更胜一筹，但相应的，Nano X的价格也更高，X 系列芯片为CC EAL5+，这也是当前市场上安全系数较高的硬件钱包。

Ledger 一直相信“安全元件是防止物理攻击和设备拦截的安全架构的核心部分”，Ledger Nano X 的设计延续这一理念，采用CCEAL5 +认证的商用级安全芯片以保证硬件的高安全性。Ledger Nano X 也是目前支持币种最多的硬件钱包，同时支持蓝牙功能，可以连接PC或智能手机来管理数字资产。适合持有多种类数字资产的用户使用。

Ledger使用的是安全芯片，安全等级是CC EAL5+，国际标准化组织将CC评估等级由低到高分为EAL 1-7等级，一般而言，商用产品的安全要求在CC EAL 5以内，生活中大众使用的银行芯片借记卡要求CC EAL5+安全认证，更高的加密级别如CC EAL6和CC EAL7属于军事级别。不过Ledger用了两个芯片，一个是安全的，一个是非安全的，这两个芯片之间在传输信息过程中就存在安全隐患。Ledger也曾爆出过安全漏洞，黑客通过强制升级非安全芯片的方式，在不拆除外壳的前提下，能一步步取得钱包的PIN码。PIN码相当于钱包的密码，有了它，就可以打开钱包，把钱转走。

Trezor系列

Trezor硬件钱包有两款：ONE、Model T

ONE系列在市场中已发售良久，在此主要描述 Model T：

Trezor Model T 是一款黑客级的数字钱包产品。“在安全性和可用性之间寻找最佳平衡”一直是Trezor 的目标，Trezor 在保证高安全标准的同时，尽可能为用户提供简洁友好的操作界面和操作方式。同样，Trezor Model T 从硬件到软件，代码全部开源。且常见主流加密资产钱包均已支持。另外Trezor Model T不仅仅是一个硬件钱包，还有许多丰富实用的功能，如果币币交易功能，密码管理器、双因素身份（FIDO U2F）验证、SSH Agent 认证等。

Trezor采用的是普通芯片，黑客如果对硬件钱包进行电子攻击，普通芯片类的钱包就束手无策。2017年，在美国拉斯维加斯举行的世界黑客大会上，国外某个安全团队现场演示了如何破解Trezor钱包，黑客通过拆除其外壳，就可以利用漏洞转走比特币，这个过程最快只需要15秒。

HyperMate系列

HyperMate旗下有两款硬件钱包，HyperMate G、HyperMate Pro。

HyperMate G 是第一代产品，G是Genesis的缩写，代表创世纪的意思。HyperMate G 是一款超轻薄、操作便捷、极具安全性的加密货币硬件钱包，获得美国 FCC 授权认证，经过全球知名安全公司审计，内置 CC EAL6+ 安全芯片，HyperMate支持主链及其合约币如BTC、ETH、BSC、HECO、OEC、Polygon、BCH、EOS、LTC、HC、QTUM、XRP、DASH、TRX、FIL，并支持符合ERC20、TRC20、QRC20标准代币，未来可通过固件升级以支持更多币种。

Pro版本在G的基础上有很多细节和使用体验上的创新，例如交易二次确认，指纹识别，中文显示，12/24位助记词等。

使用体验对比

HyperMate Pro 和 Ledger Nano X 使用时需要下载桌面客户度或者移动客户端。

Trezor Model T 客户端为网页，无需安装客户端。

三款钱包配置流程基本相同，均包括设置PIN码、记录助记词等基础步骤。HyperMate Pro和Trezor Model T 完成基础配置后既可用来管理数字资产。而Ledger Nano X 需要有额外的步骤 – 安装币种应用程序及创建账户，才可以用来管理数字资产。

HyperMate Pro 和 Trezor Model T 相较于 Ledger Nano X 更容易上手使用，更适合新手用户。

另外中文支持方面，Ledger Nano X 桌面客户端已支持中文，移动客户端暂未支持中文。HyperMate Pro移动客户端支持中文，Trezor Model T 网页客户端支持中文。

总结

硬件钱包一直被认为是行业中最安全的资产保管形式。用户无需担心资产被盗风险，投资界讲究“落袋为安”，只有财富真正落到自己的口袋里才安稳，这一点在加密市场尤为适用，资产放在交易平台也好，放在其他线上平台也好，都会面临遭受攻击，而在火币掀起撤退序幕的当下，想必很多交易所都将面临被挤兑的风险，所以，只有挑选一款硬件钱包，将资产存放在其种，与网络世界隔绝，才能更好的保护自身的资产安全，希望各位投资者都能重视资产的安全防护。
