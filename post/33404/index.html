不管你是嘻哈音乐的粉丝还是区块链的玩家，黄立成 aka 麻吉大哥，这个名字都是极为响亮的存在。

2003 年嘻哈团体 Machi 横空出世，那句「我的麻吉置叨」成了当时红遍大街小巷的流行语;走之后，市场敏锐度极高的大哥，开始将商业版图扩展到其他领域，从服饰品牌、夜店到直播平台，每项事业都掷地有声。
近几年，他将重心逐渐从演艺圈转到区块链产业，今天我们就来听听他是如何开始投身加密货币以及 NFT 的故事。
HR：当初怎么开始接触加密圈？
“2017 年还在做 17 直播的时候，加密货币开始夯，一堆人都在 ICO，我就跟股东说这是趋势要做这个！ 结果他们都说是诈骗，Okay Fuck It，我就自己来搞。”
“2018 年就做了秘银，还推出社交挖矿，鼓励用户上照片或影片来挖矿。 算是走得很前面，但就是因为太前面了，很多细节没有想好，结果币价就崩掉了。 虽然可惜，但也学到了很多经验，后来又做了CREAM。”
HR：但 CREAM 之前不是被骇？
“对，那次真的惨，我算是上了一堂昂贵的课，体会到 Defi 水有多深，当时我们被骇走差不多一亿四千万美金，但后来有想办法补偿客户，现在 CREAM 也持续获利。

2020年11月的时候，我选择把CREAM的主控权转交给Yearn的创办人Andre Cronje，这是一个很好的决定，因为Andre真的很厉害，我把CREAM交给他也很放心，之后疫情爆发在家没事，就开始把关注力都放在NFT上了。”
HR：对！ 我们就是想听 NFT 的故事。
“最早开始接触 NFT 大概也是 18 年的时候，当时就想过把音乐切割做成 NFT 放上网卖，但又走太早所以才去搞 Defi。 真正全力投入 NFT 市场的原因，是朋友之中有一位台湾女孩叫 Emily Yang，她是很有名的 NFT 艺术家。

当时她有个项目是卖画并捐款给 #StopAsianHate 机构，这么有意义的事情当然要挺啊！ 就号召几个朋友，以 Emily 的昵称 pplpleasr 集结成一个组织，于是 “PleasrDAO” 就诞生了。”
“一开始只是好玩，但随着越来越多有影响力的大咖，或是有经济实力的创业者加入 PleasrDAO，我们决定要把这个 DAO 打造成一个去中心化的艺术投资组织。 前阵子神鬼黑客 Edward 推出名为『Stay Free』的 NFT。

我们觉得很有价值，不管对手开多少都一定要标到，但过程异常惊险，原本以为没希望了，但最后五分钟突然有朋友问说还缺多少，他可以帮忙，所以最后才得以用 540 万美元成功拍下，Doge 迷因图的 NFT 也在我们手上。”

HR：那么无聊猿（BAYC）呢？
“我其实观察BAYC很久了，当时有好多天可以Mint，最后一天半夜我一个人先Mint了40只，然后也Call朋友来买，那个17创办人Sam很凶啊！ 随手就 Mint 了 120 只，WTF！ 不能输啊！ 我就卯起来继续追，Mint 完了还到二级市场去收。

别人开一颗 ETH、两颗 ETH 照买，最后收到差不多上百只猴子。 等到天亮 PleasrDAO 团员起来都在该说没买到，我就想说都是大家都来玩才好玩，想都没想就送了 30 多只出去。”
HR：艺人圈的猴子风气也是您带起的吧！
“我记得第一个给的艺人是 Kai Kai（柯震东），柏霖的猴子也是我给他的，柏霖眼光很好，一选就选到我收藏里头最限量的，好像排名前 50 吧！ 那个时候还没现在这么贵，但也不便宜，大概要一万多美金吧。”
“然后你看 Stephen Curry 买了、Snoop Dogg 买了，就连 Eminem 也都买了，这就是社群的力量，也是 Crypto 有趣的地方。 BAYC 虽然不是我的项目，但是喜欢的人都可以尽一份力，你可以用自己影响力去感染别人，把这东西让更多人知道。”

 
HR：但除了换头像之外，NFT还可以做什么？
“NFT（Non-fungible tokens） 意思是非同质化代币，应用范围可以很广，什么房地产证明、艺术版权，甚至身分证明也可以是 NFT;这些技术早就成熟了，问题是在政策面上，国家政府有没有这个认知，或是有打算往这个方向前进，这得打个问号。”
“另外，大众也多抱持着看热闹的心态，除了觉得这个是骗局会泡沫，更多只在意手上的币、NFT 可以涨多少，谁管去中心化在干嘛？ 这都很正常，网络 Web 1.0 刚出来的时候根本也想像不到未来会长成怎样，一切都是过程。”

HR：大哥可以透露你手上的 NFT 资产吗？
讲数字就俗气了，区块链有趣的就是帐目公开，你直接去看我钱包就好了。“ （既然如此，我们就不客气查了他的钱包：3200 个 NFT 收藏，总价值 4000 多万美金。 ）

 
HR：最后请给想参与 NFT 的新手一些建议。
“大家常会说一句话：『NFA（Not Financial Advice ）』，赚钱的时候没人感恩你，赔钱的时候大家会恨你，我能给的建议就是，有兴趣就多去 twitter、Medium、Discord 上面鬼混一下，看大家都在聊什么。 看不懂就 Google，等你看懂了就自然会知道消息在哪里，趋势在走什么方向，就算赔钱受伤也都是经验。”
“这个道理跟学滑板跳街舞一样，你在那边问半天，一直担心危不危险，不亲自下场尝试怎么会知道诀窍在哪？ 设定好可承受的风险范围，其他就努力学习吧！"

后记：

当我们最后问麻吉大哥，能不能让我们拍几张照片呢？ 他说他年纪大了不喜欢拍照，但可以用他的虚拟分身（无聊猿头像）替代，“大家都知道无聊猿，也知道我永远支持无聊猿，之前我还开了一个玩笑，说自己是 “ChairApe of the Bored”（无聊猿主席）。 让虚拟回归虚拟，这不就是区块链真正有趣的地方吗？”
“所以，不用拍照了，你就用我 IG 跟钱包上的猴子吧。”
