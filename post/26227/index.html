今年，NFT成为了很多头条新闻报道的热点。似乎每个媒体都在试图回答诸如“为什么有人会花钱购买任何人都可以截取屏幕截图的 jpeg？”之类的问题。

在本文中，我们深入探讨了 NFT 生态系统及其最新发展。

3月11日开创了数字艺术和不可替代代币的电子时代。Mike Winkelmann，又名数字艺术家 Beeple，以 6900 万美元的价格出售了他的 NFT 艺术品 Everyday。这个价格不仅创下了纯数字艺术品的新纪录，而且使 Beeple 的作品成为有史以来在世艺术家拍卖会上售出的第三贵的作品。

几周后，Twitter 首席执行官杰克·多尔西 (Jack Dorsey)以近 300 万美元的价格出售了他 2006 年第一条推文的数字签名副本。

诸如此类的事件引起了人们对 NFT 领域的浓厚兴趣。尽管最近几个月销售额有所下降，但我们看到越来越多的创始人、艺术家、名人和投资者进入这个生态系统。

而数字艺术仅仅是个开始。从Punk、加密猫和NBA球星卡到抵押贷款、虚拟运动鞋和元宇宙中的数字土地：NFT 生态系统可以提供很多东西，让我们一起来看看吧。

1.市场

数字市场是 NFT 生态系统不可或缺的一部分。虽然数字加密艺术的兴起推动了供应方（艺术家将他们的作品作为 NFT 出售），但越来越多的收藏家和投资者也正在进入这一领域。

OpenSea是最早期和完善的市场之一，它是用户创建和交易 NFT 的一站式平台。今年 3 月，OpenSea 筹集了 2300 万美元的资金，由 Andreessen Horowitz 领投，天使投资人包括 Naval Ravikant 和 Mark Cuban。

另一个市场是Rarible。数字艺术市场最近宣布了来自 Venrock Capital、CoinFund 和 01 Advisors 的1420 万美元 A轮融资。该平台的核心是其基于以太坊的代币 RARI。通过拥有 RARI 代币，收藏家可以对影响平台的提案、审核创作者和策划特色艺术品进行投票。

策展与开放市场

在评估市场时，一个主要区别是策展。虽然OpenSea和Rarible等平台无需许可，但其他市场（包括SuperRare、Nifty Gateway、Foundation、Known Origin和Makersplace）需要特殊的申请程序。

在Foundation 上，艺术家需要得到另一位创作者的邀请才能列出他们的数字作品。SuperRare和Nifty Gateway等其他平台要求艺术家上传应用视频来解释他们的经历和动机。

版税

转售版税是创作者从其艺术品的二次销售中获得的百分比。值得注意的是，不同市场对版税的处理方式不同。在 SuperRare 和 MakersPlace 上，艺术家获得 10% 的版税，而 Rarible、Foundation 和 NiftyGateway 上的艺术家可以自己指定版税。

其他值得注意的市场包括AsyncArt、Mintable、Zora、NFT Showroom、Blockparty、Cargo、Mitable和ArtBlocks。

2.数字收藏品

数字收藏品是 NFT 生态系统中的第二个支柱，包括功能很少但情感价值高的物品（想想棒球或足球卡）。

Punks、加密猫和勒布朗詹姆斯

第一个基于以太坊的 NFT 实验是 CryptoPunks，它由 10,000 个独特的可收藏朋克组成，每个朋克都有一组独特的特征。最昂贵的拍卖是 CryptoPunk #7523，在苏富比拍卖行举办的拍卖会上以 1180 万美元的价格售出。

CryptoPunk #7523（来源：Larva Labs）

然而，正是CryptoKitties将 NFT带入了主流。CryptoKitties 于 2017 年在加拿大的 ETH 滑铁卢黑客马拉松上创建，其特色是一款允许用户收集、繁殖和交易虚拟猫的游戏。“第 0 代”猫在拍卖中出售，新猫也可以在 OpenSea 等二级市场上转售。最昂贵的销售品是名为 Dragon 的 CryptoKitty，在 2018 年以600 ETH 或 172,000 美元的价格出售。

这个领域的另一个著名项目是NBA Top Shot，这是一个由 DapperLabs 与美国国家篮球协会合作创建的篮球时刻交易卡平台。4 月，勒布朗·詹姆斯模仿著名的科比·布莱恩特扣篮的 NBA 最佳投篮时刻以 387,600 美元的价格售出。

其他收藏品包括Sandbox、CryptoStamp、Avegotchi、MLB Champions、Meebits、TokenTrove、Hashmasks和Avastars。

3.游戏

游戏平台是NFT生态系统中的重要支柱，因为它们将非加密用户吸引到 NFT 生态系统。

一个著名的例子是Sorare，这是一个位于巴黎的游戏平台，用于创建和交易足球NFT。在游戏中，玩家可以组建一个五张牌的小队来对抗其他Sorare粉丝，并根据足球运动员在足球比赛中的实际表现来衡量他们的表现。比赛的获胜者可以获得可以兑换其他卡或 ETH 的奖励。Sorare 在 A 轮融资中筹集了 4000 万欧元，由 Benchmark 领投，另外还有来自 Accel 和包括足球运动员 Antoine Griezmann 和 Rio Ferdinand 在内的天使投资。

最著名的加密游戏之一是越南开发商 Sky Masis 于 2018 年推出的Axie Infinity。该游戏遵循游戏赚取模式，玩家可以在其中战斗、饲养和交易名为 Axies 的数字生物来建立虚拟王国。今年 5 月，母公司Mavis Sky从 BlackTower Capital 和 Mark Cuban 那里筹集了 875 万美元的资金。

根据 Coindesk 的一篇文章，随着 Covid-19 大流行袭击菲律宾，一些失去工作的菲律宾人能够以玩和交易 Axies为生。

“通过玩 Axie 赚到的钱，来自 Philipiness 的游戏玩家购买了从婴儿尿布和牛奶到工作面试穿的鞋子和衬衫的所有东西。”

其他著名的 NFT 游戏包括Gods Unchained、F1 Delta Time、Guild of Guardians、Splinterlands、CSC、League of Kingdoms、Lost Relics、ChainZ Arena、Crazy Kings、Evolution Land和MyCryptoHeroes。

4.基础设施

以太币在 NFT 生态系统中占据主导地位，但仍有改进的空间。截至 2021 年 7 月，按销售额排名前 10 的 NFT 项目中有 8 个是建立在以太坊上的。然而，高昂的 gas 费用（在 Ethereum 上进行交易的成本）、网络拥塞和不满意的用户体验仍然是一个问题。这为新玩家提供了市场机会。

必须在第 1 层和第 2 层解决方案之间做出主要区别。第 1 层“基础层”是指底层的主区块链架构，而第 2 层是存在于底层区块链之上的覆盖网络。

第 1 层 - 基础层：Ethereum、Flow

第 2 层和侧链 ：WAX、Polygon、Enjin、Charged Particles、Tari

5.金融

许多有趣的项目正在去中心化金融 (DeFi) 和 NFT 的交汇处发展：

NFTfi：NFT 抵押贷款的首批平台之一。用户可以通过将 ETH 借给将其NFT作为抵押品的用户来赚取收益。

NIFTEX：一个 NFT 部分所有权的平台。它于 2020 年 5 月推出，早在 NFT 炒作之前。

其他参与者包括Armor.fi（DeFi 资产覆盖公司）和NFT20（NFT 指数基金提供商）。

6.域名

虽然传统域主要只是网站地址，但区块链域与钱包地址相关联，从而更容易发送和接收付款。域名 NFT，用一个简单的、人类可读的名称替换复杂的加密货币钱包地址。

该领域的领先者是Unstoppable Domains、Ethereum Name Service (ENS) 和Handshake。

7.跟踪/分析

跟踪和分析平台发挥着重要作用，因为它们为 NFT 生态系统带来了透明度。

领先的数据库和分析平台是Nonfungible.com、DappRadar、NFTbank和Upshot。

8.音频

音频 NFT 平台支持音乐家，就像数字艺术市场支持艺术家一样：它们为创意人员开辟了新的途径，将他们的作品转化为数字资产。

音频 NFT 消除了集中的唱片标签，将有助于为艺术家带来更大的自由，为用户带来更大的开放性和可用性。

著名的去中心化音乐流媒体服务有Audius、Rocki和Opus。

9.去中心化自治组织

DAOs“去中心化自治组织”是存在于一组智能合约上的成员所有的社区。在 NFT 背景下，DAO 可以用作 NFT 项目的投资工具。DAO 中的投资者可以根据智能合约中建立的规则对提案进行投票，而每个人的投票都由拥有的代币数量加权。如果提议的投资获得足够多的赞成票，智能合约会自动触发交易。

Flamingo：一个以 NFT 为中心的 DAO，使其成员能够开发和部署以 NFT 为中心的投资策略。

ArkGallery：一个购买 Cryptopunks 的 DAO。

WHALE： $WHALE 代币是一种社交代币，由NFT 最大的个人买家之一WhaleShark创建。该代币的所有者获得 Whale Shark 的 NFT 收藏的部分股份。

FWB：$FWB 是代表 NFT 艺术家和创作者社区的社交代币。

如果您想了解有关 DAO 领域的更多信息，这篇文章是一个很好的资源。

10.数码时尚

NFT 是关于收藏品的，时尚界也是如此。该领域最著名的初创企业之一是虚拟时尚品牌 RTFKT，该品牌最近宣布了来自 Andreessen Horowitz 和 C Ventures 的 800 万美元种子轮融资。

NFT 时尚品牌 Overprice 推出的一款虚拟连帽衫以 26,000 美元的价格售出。所有者可以通过在智能手机上扫描 VR 代码来虚拟地“穿着”这件衣服。

其他 NFT 时尚品牌包括DressX、Digitalax和MetaFactory。

11.展示柜

您现在可能想知道：“如果您不能向公众展示它们，那么购买所有这些 NFT 有什么意义？” 有多种方式，并且在该领域有一个全新的行业正在发展。

虚拟画廊：加密艺术博物馆、Flawnt和MOCDA是允许收藏家和艺术家展示他们的 NFT 的虚拟画廊。

数码相框：来自Blackdove、Qonos和Tokenframe 的数码相框/显示器允许 NFT 所有者在现实生活中展示他们购买的艺术品。

该领域的其他有趣项目是Showtime（社交网络）和Infinite Objects（NFT 打印服务）。

12.虚拟世界

你玩过 Minecraft、Fortnite 或 Sims 吗？那么虚拟世界可能适合你。

虚拟世界是计算机模拟的世界，用户在其中相互联系，就像在现实世界中一样。

Decentraland是最著名的虚拟世界之一，它提供了一种全新的体验：Decentraland 中的大多数东西都是 NFT。从虚拟土地到虚拟画廊墙壁上的艺术品 - 玩家可以购买、收集和出售仅存在于虚拟世界中的资产。Decentraland 分为不同的区域，从赌博区维加斯市和购物区 Metajuku 到教育和商业区，用户可以在这里见面和协作。

用户可以购买虚拟衣服的购物区Metajuku（来源：Decentraland）

6月，Decentraland 的一块虚拟土地以超过 90 万美元的价格售出，成为有史以来最有利可图的 NFT 土地购买。

其他平台，如Somnium Space、Upland和Cryptovexels（也被称为“区块链 Minecraft”，因为其是用积木建造）也执行相同的任务。它们允许用户购买虚拟土地、建造物品并通过在 OpenSea 等平台上进行交易来将它们货币化。

展望未来

当比较从 5 月到今天的 NFT 销售历史时，NFT 泡沫似乎已经破灭。根据Nonfungible 的数据，整体销售额从 5 月 9 日的 7 天峰值 1.76 亿美元暴跌至 7 月中旬的约 800 万美元。

许多市场观察家将此现象与90 年代末的互联网泡沫相提并论。随着危机后科技行业的增长趋于稳定，谷歌、Facebook 和亚马逊等 Web2 公司出现并重塑了我们的日常生活。

哪些公司将引领 Web3 领域？也许他们已经在这张地图上，我们很快就会知道。
