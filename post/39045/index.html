当我们称其为建设市场时，我们不仅仅指 Crypto 和 Web3 协议。现在也是建设自己的时候了。

虽然 Crypto 市场现在显然非常惨淡，但 Web3 不会就此消失。这个行业比以往任何时候都更有资金和更有弹性。

因此，尽管市场价格下跌，但该行业本身已准备好继续建设。这意味着现在是投身于崭露头角的加密职业的好时机。

现在是您发展和过渡技能、磨练您的价值主张并建立自己的社区的机会。

今天，William 列出了启动你的加密职业生涯的五个黄金技巧，这样你就可以把握下一个牛市。

如何开始你的 Crypto 职业生涯

所以你有兴趣开始你的 Crypto 职业生涯，但你不确定从哪里开始。那么你可能是面临一些问题，其中最常见的问题包括：

  • 我可以靠 Crypto 谋生吗？

  • 我必须成为 Crypto 专家才能开始吗？

  • 有哪些类型的工作可供选择？

是的，你可以靠 Crypto 谋生！

一般而言，非技术和技术 Crypto 经济职位要求的薪资都高于其相同“标准”的职位。

刚从大学毕业，我的第一次写作稿酬是每字 0.03 美元。如今，我的 Crypto 写作——平均而言——通常会达到这个数字的 20 倍以上。部分原因在于我的退伍军人身份和不可否认的天赋（你能阅读这篇文章，就说明了这一点，对吧？）。

另一个主要促成因素是整个 Crypto 经济对写作的需求广泛。这意味着：文案、新闻、技术写作、研究、翻译，不胜枚举。Crypto 需要这一切。

至于第二个问题：您是否必须成为专家才能开始 Crypto 职业？不，你当然不需要。我以一般技术作家的身份进入这个生态系统，然后在 2017 年夏天才深入了解了 Crypto。

我的许多同事在工作中同样学习了 Web3，而且许多人现在都是各自领域中备受尊敬的专家。你也可以！这个行业的每个人都曾经是新手——而且可能比你想象的更小白。

最后，至于加密中可用的工作类型，简单的答案就是所有这些。开发人员和工程师是需求量最大的，但目前也在招聘各种非技术工作。

包括：

  • 社区管理员 + 版主（Discord、Twitter 等）

  • 内容创作者（平面设计师、新闻作家、播客等）

  • 数据分析师

  • 经济学家（代币经济学）

  • 编辑

  • 营销人员（合作伙伴关系、社交媒体、讲故事等）

  • 产品经理

  • 项目经理

  • 翻译人员

  • & 更多！

即使考虑到熊市，Crypto 就业机会也比比皆是。虽然你如何入行是一个个人的旅程，但我开发了一个简单的程序，我可以推荐给所有人。

以下是我提供给你开启加密职业生涯的 5 个步骤：

1. 确定你的目标

如果你现在有一份体面的工作，你可能需要花时间过渡到你的密码学梦想的全职 Web3 职位。

如果您想尽快迈出第一步，您可能需要考虑所有事情：兼职、赏金、合约、空投等。

当您开始时，请考虑您的迫切需求并从那里制定战略。花一些时间准确地写下你的 Crypto 职业目标是什么，你为什么要这样做，以及你可以在哪里提供价值。

你重视公司氛围是否冷清？带薪吗？是否是具有创意的 NFT 项目？稳定的工作环境？上述所有的？你想做什么类型的工作？我已经与想要转向加密领域的非技术职业的技术专业人士和想要转向技术工作的非技术专业人士等进行了交谈。

加密货币的参与规则不那么严格。所以你想要什么？

如果你坐下来设定基本目标、愿望和界限，你就会扫清前进的道路。它将帮助您避免选择不适合您的项目、在死胡同上浪费时间或被支线任务所困扰。

2. 整理你的（其他）作品集

一旦你确定了你想从加密职业中得到什么，就该向其他人强调你可以带来哪些技能。为此，展示和发布基本作品集的一种简单方法是创建自己的网站。

例如，早在 2019 年——在之前熊市的阵痛中——我希望将我的加密职业提升到一个新的水平。当时，我记得看到 Spencer Noon 建议创建一个个人网站。

所以我做了！

从那以后，我就一直使用那个网站 wmpea.GitHub.io/portfolio 作为我的作品集，并且在当时它带来了很多很棒的工作机会。我绝对建议使用您的基本背景信息创建一个作品集网站。你最终如何做到这一点取决于你。我通过 GitHub Pages 免费创建并维护，所以如果您对这条路线感兴趣，我建议您查看这个有用的 GitHub Pages 快速入门指南。

如果您正在尝试构建您网站的“代表作”部分，但在大型加密出版物中还没有进行过很多（或任何）署名，不用担心！考虑通过 web3 发布项目 Mirror 就您热衷的主题创建一两个帖子，然后在您网站上放上这些帖子的链接，以展示您是一个思想家和行动者。

3. 让自己沉浸这个文化中

“在申请 web3 职业时，我最大的建议是始终在链上和链下‘提出你的理由’。参与聊天，开始使用 DAO，在可能的地方加入，转发你感兴趣的东西，收集 POAP NFT——在您评估加密货币就业市场并找到适合您的职位的同时”——Michael Wong，Bankless 文化负责人

获得和维持有意义的 crypto 职业将需要随着时间的推移成为加密原生。这意味着成为生态系统细微差别及其项目的专家。

一开始，这些都是很难理解的。我最近看到了 Dune Analytics 的数据科学家 Jackie Zhang 的一篇文章，题为《如何将 crypto 作为一种新语言来学习》。这是一篇精彩的文章，讲述了进入加密职业生涯就像学习新文化和新语言一样。

我认为学习制作一个框架是完全正确，可以让你为即将发生的事情做好准备。你需要学习新单词、新概念、新习惯和新工具。进行这些研究的最好方法就是让自己沉浸在这个空间中。做到入乡随俗。

从简单的东西开始，然后从那里继续深入。在不知不觉中，您的 crypto 技能树将迅速开花。您将通过 dogfooding、构建网络和成为建设者来自动升级。

4. 建立自己的品牌

您也可以将此步骤称为“成为权威”或“实践和发布”。该过程的这个阶段是通过为公众创建内容将您的技能和学习应用于生产方式。

这一步可以有多种形式。可以是为 YouTube 创建主题加密解释器内容。也许可以是在 Twitter Spaces 或播客期间记笔记并将它们发布到您的博客。或者是写一篇关于你认为值得关注的利基 web3 主题的时事通讯。

这么做的好处是多方面的：

教别人是深入自我学习的最好方法之一

为他人提供有用的服务是建立信誉、人脉和雇佣能力的捷径。

为公众创建内容可为您提供大量材料，以扩展和改进您的个人作品集。

2020 年 1 月，我推出了 DeFi Arts Intelligencer 时事通讯，以了解有关 NFT 的更多信息、提供公共服务并同时支持我的投资组合。后来我在那里的努力为我赢得了 Bankless 的邀请，让我创建 Metaversal 时事通讯。

直到今天，我还在写 Metaversal。我喜欢这份工作！这个机会来自我作为 NFT 分析师公开建立我的个人品牌。随着您建立自己的品牌，您的职业生涯同样会成为焦点。

5. 全力以赴

所以你已经确定了你的目标，完善了你的作品集，埋头于 加密文化，并开始发布内容。然后呢？

一方面，您可以坐下来，创建内容，然后等待，看看是否有任何团队最终会接触你，有兴趣聘请您提供服务的团队。耐心点，一定会有！

然而，如果您想更加积极主动，您可以积极寻找您认为最适合自己的职位。因此，这需要时刻留心并申请您遇到的有趣职位。

Bankless 运营着一个 Pallet 工作委员会，您可以在其中随时找到并申请生态系统中的一些最佳工作机会。

另一种途径是在社交媒体上关注你喜欢的项目，然后跟踪他们的团队是否宣布他们正在招聘。最近的一个例子是 PartyDAO——一个 NFT 团体竞标项目——它刚刚宣布它正在寻找协议工程师。

关注您最喜欢的项目及其团队成员，然后与他们建立个人联系。
