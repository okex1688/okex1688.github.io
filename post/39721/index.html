价值有多种形式。根据自己的世界观，可能更容易高估或不成比例地贬低某物的价值。尽管如此，价值代表了无法估量的变量纠缠，从根本上讲，这些变量始终是主观的。

有些人花费大笔资金购买课程寻找答案，但数据和分析模型或许会给到一些方向。NFT行业的发展速度本身就是在当下成就一个NFT项目的原因，3-6 个月后，原有的认知便可能会经历翻天覆地的变化。

这种复杂且有些难以理解的价值表述在 NFT 市场中具有新的意义。Beeple 的 NFT 杰作 《Everydays: The First 5000 Days》 怎么能卖到超过 6900 w 美元？有人会说“它只是一个jpeg”。但事实证明他们的看法只是一种主观情绪，并不一定是正确的，只要市场允许且有人愿意为之付出，那它的价值就是 6900 w 美元。

除了主观情绪之外，我们还能从哪些方向来判断一个 NFT 项目呢？

1.效用

NFT 仅仅几年从8位像素图像和弹球式的区块链游戏发展到高保真的3D动画和复杂的 “P2E” 机制。

NFT的核心用户识破谎言和骗局的能力越来越强，这意味着对于大多数项目 Mint 总量会减少。因此我们很容易看到 NFT 总 Mint 量的下降，但现实情况是，NFT 圈内需要大量的更替，才能淘汰那些并非认真做事的项目，并更有效地激励这个领域真正的创作者。

虽然 NFT 是偏向于文创类的，故事属性为根本，但显然，随着 NFT 市场开始走向成熟，优质的东西更容易被主流获取，有应用的 NFT 会有更大的优势。NFT 的潜在价值主张将更加透明和可靠，有实际效用的 NFT 项目，将更容易被用户所传播。

简单来看，可以分为以下几种类型：

  • 应用支撑，GameFi、P2E、元宇宙等等，用户所购买的 NFT 可以在其中有实际用途，类似于装备、道具等等。

  • 实物（权益）支撑，比如：买了某个艺术品的 NFT ，则有去实物观赏该艺术品、甚至获得该实物艺术品的权利（更像是凭证）。

  • Pass ，Pass 更像是把会员卡上链，做成了 NFT 的形式。听起来似乎并不是很“高大上”，但却具备很强的实用性，不管是从转售、功能嵌入等，Pass 都具备传统会员卡没有的便利性。

  • IP 。比如，和某个明星联合、获得某个 web 2 知名事物的版权（ 比如：Dapper Labs 的 NBA Top Shot），或是自己创造IP；人们总会为了自己喜欢的 IP 去买单，就像会去购买自己喜欢明星的专辑一样。

2.社区

社区是决定项目能否成功最关键的因素之一。社区文化能够影响项目的成败，这样的例子比比皆是。当前，多数社区通过 Discord 进行日常交流，有时也会依赖Twitter彼此沟通。项目社区就像专属俱乐部，让你感觉自己参加了一场潮流派，或是遇到了一群志同道合的家人。

可以从以下几点来判断项目的社区：

  • 是否活跃？在主流平台上能够定期进行交流和沟通。请记住，新成立的社区和小型社区可能在成立之初甚至整个运行期间都不太活跃，而较大的社区活跃度要高得多。

  • 是否真实？在 Discord 中，真实有效的对话非常难得。有些项目会举行活动竞赛，项目成员只会一遍又一遍地发送群发垃圾消息来提高自己的排名，争取被列入白名单。这会让一部分人望而却步。

  • 是否营造出氛围感？社区氛围大有裨益。这种氛围可以体现为言谈中的语气和能量。无论是冷淡平静，还是嘈杂喧哗，亦或是称兄道弟，不管氛围如何，确保这是你感兴趣并且愿意长期参与的事情。

  • Twitter 粉丝数是有多少？Twitter 为NFT提供了优质的平台。粉丝数越多，获得的曝光量也就越大。新项目的粉丝数可能有限，但它们是否有规律地增长？都是真实的粉丝吗？

  • Discord 成员有多少？Discord 服务器对推动项目讨论度、促进社区成员就感兴趣的事项进行对话很有帮助。较小的服务器可能看上去更为安静，因为成员遍布世界各地，很难保持实时对话。你也期待它能成为你提供最新的新闻资讯。新项目的 Discord 成员数可能偏少，但它是否有规律地增长？

  • 规模是否够大？一个颇具规模的社区对于持有和转手转手的意义都很关键。理想情况下，投资者寻找的社区应当能够支持每名成员拥有约2枚NFT。因此，对总量为 10000 枚的 NFT 而言，其 Discord 应当拥有至少5000名成员。

  • 是否能够及时回答群友的提问？你可以从社区和团队那里找到本文提出的许多话题和问题的答案。如果你没有得到任何答案，试着去挖掘原因。有些项目希望隐藏部分细节，以待日后公开。但是，如果你不仅没有得到任何答复，甚至没有任何迹象表明团队会对问题作出回答，那么这很可能是个不好的征兆。

3.团队

这是项目成功的另一个关键所在。大多数时候，项目都有过度承诺和交付不足的现象发生。这是因为在项目开始时，很难考虑完整应用程序的形态展现；具备效用的愿景可能随着项目的普及和发展就磨灭了，最终甚至可能变成了一个 “收藏品” 。把项目的团队成员和做事方式弄明白，将更有助于你你判断该项目“适不适合”你。

                                                        Moonbirds 创始人KevinRose为传统界顶级 VC

不妨判断下，这支团队：

  • 是否公开？是否能在Twitter、Linkedln、项目官网或其他社交媒体上找到团队成员？知道团队不仅仅由匿名人士建立无疑是一颗定心丸，因为一旦情况急转直下时，愿意分享真实生活的人往往很难隐藏自己。

  • 是否活跃？如果他们参与社区事务，答疑解惑，引导人们回答问题，或是与其他社区成员进行交流，这表明他们有意愿成为事情的一部分，也表明他们十分重视自己的社区，愿意为之投入时间。

  • 是否有经验？在一个全新的领域，拥有经验会到来很大的帮助，因为有经验的人往往对自己的行动方向比较清晰，或者至少知道不该做什么。此外，经验丰富的工程师、艺术家、项目经理、营销人员、社区经理和其他重要角色也能给项目本身带来许多经验。这些经验能在一定程度上加深人们对项目的信任。团队中经验丰富的人越多，项目就越有可能成功。

从 Azuki 创始人被曝出黑料导致 Azuki 价格下跌，我们就不难看出团队的重要性，或者说团队实力对于用户的重要参考意义。

4.价格和流动性

当前的环境下，Mint 的价格是逐步降低的，甚至多数项目都是免费的 Free Mint，因此项目团队将不得不努力提高项目的价格和流动性，以此来获得版税收入，而并非像以前许多项目一样一样只需要发售一波就可以溜之大吉。因此，对于 Mint 价格高但效用几乎为零的项目需要格外谨慎——毕竟对于项目方来说，高 Mint 价格破发之后拉盘的成本实在太高了。

另一方面，流动性也是一个 NFT 项目最为重要的判断依据。虽然许多用户购买的第一个 NFT 的原因是喜欢它，但更多人购买还是为了二次转售盈利，而非收藏。有些人很喜欢，想长久持有，但也有一些人面临着 NFTs 无法卖出的情况。NFT 的流动性不光代表了转售的难易程度，更是代表了它的流通性、热度甚至是共识强度。

我们从下面两个角度来看：

  • 收藏品类 NFT。如头像和艺术品，普遍被认为流动性较低。如果这些物品的所有者遇到合适的买家，便可以按当前市场价值进行交易。然而，如果想尽快售出，有可能不得不降价出售，显然降低了此类资产的流动性和价格判断。然而，当对某一物品的需求突然增加时（例如，市场对某一 Collection 的热情高涨），流动性将被重新评估。

  • 相对流动性。一是过去的成交数据，但一个周全的买家会考虑到许多其他因素，例如，与其他同类型的产品相比的流动性，简称，相对流动性。道理很简单，较高的交易数量某种程度上代表较高的流动性。这个数字不是从货币的角度，而是从生产力的角度来证明产品的市场规模。然而，一些交易量小的市场（比如知名艺术家推出的作品集）也可以被认为一个非常独特和有吸引力的流动性市场，因此要分别讨论。

                                                                              数据来源：nftgo.io

如果一个项目不具备一定的流动性，那即使它天价也只会有价无市，变成只有收藏属性的“藏品”。

5.价值驱动 / 权益

NFT 是否有价值驱动也是关键所在。

项目方往往会对 NFT 持有者们格外关照，赋予他们更多的权益。如数月前蓝筹项目的 Air Drop 热潮曾把项目价格及市场情绪带到了顶点。投资者们也希望寻找创新点，从而提升购买NFT能够获得的价值。许多路线图勾勒了持有者们在未来能从该项目中得到的回馈。

                                                   Azuki  发布空投给项目带来极大的涨幅

除此之外，以 BAYC 为核心的 NFT 项目还会授予持有者 “著作权授权” ，给予持有者个人使用和商业使用的权利，如：李宁、绿地等购买了 BAYC 后便可以进行二次创作，以联名的形式进行发售营销或获利等。而这也是 Web3 和现实世界的一种连接尝试，给予整个 Web3 注入了新的生命力。

对于任何希望将自己创作的 NFT 做大做强的项目方来说，持有者权益的设计永远是放在头部的。

6.美学 / 艺术性

成功的项目通常具有引人注目而又与众不同的艺术性。虽说各花入各眼，但从过去数月间人猿、东方元素、再到像素风、3D 艺术的流行，或许你能逐渐看到一些项目审美的变化趋势。

当然，每个用户的审美都不尽相同，但总体“美”的范围是有一个阈值所在，如果一个项目过于猎奇，那它极有可能只有少部分受众，很难成为蓝筹（MAYC 受益于BAYC，除外），亦或是很可能只在短时间内有热度。

                                                                                数据来源：nftgo.io

在去判断一个项目时，可以看一下该 NFT 的图片本身：

  • 是否夺人眼球？好的项目会具备独特的艺术性，且设计在细节上区分度明显，每个单独的 NFT 都有能够区分的元素。具备艺术性的项目总能在诸多项目中脱颖而出。

  • 是否独一无二？独特的项目完全由不同的个体组成。在收藏中，每个NFT看上去都与其他的不同。通常，人们偏好拥有独特的NFT，这类NFT能够吸引不同类型的投资者。

  • 是否在“美”的阈值内？这与行业的“趋势”发展息息相关，因为当前的流行风格会影响项目是否受欢迎，但趋势往往只是暂时的，所以盛行一时的收藏很可能在未来成为明日黄花。虽然项目团队做事的情况对项目的影响最为关键，但具备“美”元素的项目往往更能持久。

  • 以及最重要的——你是否喜欢它？你是否愿意把该 NFT 用作自己的头像，甚至是把它打印出来挂在家中某处？每当你在钱包中看到它时，是否心生喜悦？以及如果你真的喜欢它（或是其背后的文化），那么想必即使最后转售，你也会从持有的过程中获得不一样的感受。

7.路线图

对于项目成功而言，路线图的重要性正与日俱增。几个月前，推出 NFT 项目尚且不需要路线图。团队只需要抛出一堆 NFT，人们就会自发把它们拾起来。但情况已经有所变化。团队意识到如果拥有阶段性产品（路线图），他们就可以推动NFT持续升值，让持有人长期持有 NFT，而非让项目昙花一现。投资者也有所改变，用户理所当然期待看到一张有规划路线图，不管它在一开始有多简单。这一路线图能告诉投资者和持有人该项目未来的发展方向。

问问自己，这张路线图：

  • 是否真实可信？如果内容听起来美好得令人难以置信，那很可能就是镜花水月。通常，完成路线图需要一个由开发人员、设计师、项目成员等人组成的庞大团队。如果团队规模没有这么大，他们就不大可能达成路线图的规划。

  • 是泛泛而谈还是言之有物？详细的路线图不仅能振奋人心，还能让关心项目未来的人心中有数。路线图也能帮助社区了解他们能够期待的阶段性成就。

  • 是否有具体的时间规划？你想要找到业务细节或具体的截止日期，如Q1、一月初或11月27日等。这些日期或时间线为路线图提供了更多细节，能给成员带去更多信心，减少困惑和模糊不清之处。

  • 是否有竞争力？路线图细节能帮助你将其与其他项目进行比较，要素包括项目大小、铸造日期、铸造价格等。通过对比，你能看到该项目的竞争力。如果其中一项要素远超出标准水平，尝试找到原因。也许该项目的社区十分庞大，由于社区内争夺激烈，导致 Mint 价格攀升。

8.愿景

愿景不光是项目方对于项目的规划，更应该是你对项目的期待。你可以因为喜欢 Azuki 的东方元素而购买它，可以因为对 Mfers 的躺平文化有感触而购买它，可以钦佩 Moonsbirds 的创始人有头脑而去参与，亦或是仅仅判断某个项目有升值空间而入场。

在 Web3 的世界中，你是自由的，你不需要传统艺术品购买所需的验资、身份等繁琐的门槛，你可以购买到你喜欢的任何数字艺术品，你可以和一大群志同道合的人一起交流、伴随项目成长，以及在不同的项目中感受到到各种各样的文化。

社区是 Web3 的基础，你的 NFT 会让你与社区有更紧密的联系；社区文化和路线图是发展的核心，用户的愿景更像是社区文化的推进器。
