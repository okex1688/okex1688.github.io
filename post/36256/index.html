金色财经 区块链4月4日讯   提到2021 年最火爆的数字资产，非NFT莫属。如今， NFT 风靡全球之势已经无法阻挡，全世界不少知名球星、影星、歌星也都纷纷趁热推出了自己的 NFT 系列。

也许有很多人觉得 “NFT” 这个词理解起来有些困难，毕竟其中涉及到的技术术语非常晦涩难懂。其实，用通俗易懂的话来说，NFT就是存储在区块链上的独立、不可互换的数据单元。而区块链则是一种数字分类账簿， NFT 持有者需要从数字分类账簿中获得所有权证明——这样解释是不是比较好理解？

通常情况下，NFT 代表着虚拟艺术，并且分为好几种不同类型，接下来就让我们一起来看看吧。

1. 艺术品  

目前，这是最受欢迎的一类 NFT 。每一个数字艺术品都有代表自己真实性和所有权的证书，而这些证书就是由存储它们的数字分类账簿所颁发。截至目前，售价最贵的 NFT 数字艺术品是艺术家 Beeple 于 2021 年 3 月在佳士得拍卖会上售出的《The First 5000 Days》，当时的拍卖价格高达 6930 万美元。   

2. 收藏品

收藏品是有史以来推出的第一个 NFT 种类。它们的灵感来自一些实体收藏品，比如宝可梦（Pokemon）卡片或是一些老式玩具等，只不过采用的都是数字形式。其实许多人并不清楚， Curio Cards 是有史以来推出的第一个大型 NFT 收藏品，而在这之后推出的其他几个收藏品，如 Bored Ape Yacht Club、Cryptopunks、Cat Colony、Meebits 等如今都已火遍全球，成为了收藏家们追捧的对象。其中，Bored Ape Yacht Club 是目前为止最有价值的数字收藏品。      

3. 体育类藏品

体育类藏品是当下最热门的 NFT 类别之一，其中最著名的 NFT 要数 NBA Top Shot 了。这类 NFT 通常会是一段精彩的体育竞技瞬间的视频剪辑，而这其中最有名的一段剪辑就是湖人队球员勒布朗·詹姆斯（LeBron James）的扣篮视频 ——Throwdowns系列，目前它的售价已超过 380,000 美元，是有史以来最昂贵的体育收藏品之一。

4. 数字资产类游戏

在这类基于 NFT 的数字类游戏中，玩家可以通过参与游戏而获得加密货币、数字资产或是其他 NFT 的奖励。该领域出现的第一个 NFT 数字资产类游戏是 Axie Infinity，之后又涌现出了大批类似的游戏如 CryptoKitties、Gods Unchained、Sorare 等，并逐渐在玩家中变得流行起来。

5.虚拟土地

虚拟土地包括数字类游戏和元宇宙中的土地。别看这类资产看起来很鸡肋，没有什么实用价值，但其实潜力巨大。这些虚拟土地可以被用来在数字类游戏中投放广告、创建虚拟资产等。

6. Meme

几年前，人们可能还无法想象 meme 也能够有任何的附加经济价值。然而就在这波 NFT 浪潮中，meme 也成为了一种非常有价值的数字资产。最早的 Doge meme诞生于 2021 年 6 月，当时的售价为 400 万美元。除此之外，Disaster Girl、Bad Luck Brian、Success Kid、NyanCat 等一批meme的估值也很高，售出的价格堪称天价，其中名为”Disaster Girl”的meme售价就超过 47 万美元。这些meme的创作者也因此变得富有起来，meme市场变成了一个利润丰厚的市场。

7. 域名

这些在币安智能链或是以太坊等区块链上铸造的加密域名总共有 500 多个域名后缀，其中最受欢迎的是“.eth”。加密域名不依赖于任何一个中心化集权，是一种人人都向往的资产。此外，它们还可以链接到加密钱包。不过，加密域名最大的缺点就是大多数浏览器目前都不支持它们。

8. 音乐

音乐算是 NFT 热潮中的新兴领域。通常情况下，艺人会先在 NFT 市场上预发行自己的专辑，然后再到传统流媒体平台上进行发布。买家可以在预发行阶段先购买专辑的一部分NFT，类似入股的形式，然后当专辑通过传统渠道发布时，买家就能分享到专辑带来的收益。虽然这种商业模式目前还未成为所有音乐 NFT 的交易方式，但已经有相当多的交易开始遵循这种模式。例如，2021 年 3 月，美国摇滚乐队莱昂国王（Kings of Leon）的专辑《When You See Yourself》就以类似的方式将自己的专辑拆分成了多个 NFT 进行销售。

9. 票务

NFT 的出现给日常交易方式带来了各种创新，当然也不会落下票务领域。现在，利用区块链平台可以铸造各类活动的门票，之后再由组织者进行拍卖，当然也可以以一口价的形式出售。可以肯定的是，NFT形式的门票出售方式可以显著减少门票伪造行为并且能让买家产生一种购买收藏品的感觉（因为这些门票也可以当作收藏品来保存和转售）。

10. 现实世界资产

读到这里，您可能很想知道是否 NFT 仅仅只与数字和虚拟资产挂钩？答案是否定的， NFT 也有与现实世界中的资产相关联的应用。现实世界资产 NFT (rwANFt) 就是一种代币，代表了对于实物的虚拟所有权。如同数字 NFT 一样， rwANFT 代币不仅证明了物品的所有权，而且还具备了保修、保险、可执行性等方面的法律效应。由于代币不能被伪造，并且数字分类账簿是透明化、防欺诈的，因此 rwANFT 相当于有了一个额外的安全层。  

11. 时尚商品  

这似乎听起来很疯狂，一般人很难理解时尚居然也能虚拟化。然而令人意想不到的是，时尚NFT 不仅非常真实而且市值也非常可观。时尚 NFT通常包括为数字形象和视频游戏角色定制的服装和配饰。不仅如此，奢侈品品牌 Louis Vuitton 和 Burberry 也推出了一系列时尚 NFT ，包括为数字形象打造的和服和运动鞋。

12. 身份

不得不承认，NFT 已经获得了现实世界的巨大关注，这得益于它的不可替代性以及数字分类账簿的透明性这些特点。身份类NFT在现实世界中的一个实际应用就是为个人提供身份证明，而自我主权身份和桥接协议又是在个人身份识别领域中的一些用例。

13. 其他在线项目

此类别包括了前面12种类别未涵盖到的所有内容，包括推文、博客、Instagram 帖子等等。换句话说，任何能在数字分类账簿上铸造的并且具有不可替代性的东西都属于这一类。例如，推特前CEO杰克·多尔西（Jack Dorsey）的第一条推文就售出了 290 万美元的价格。

应该没有人能想到如今的NFT市场会如此蓬勃的发展，每天都有无数新的NFT项目问世。新技术不断地被创造出来，满足我们的基本需求和愿望。如今，NFT 不仅与数字资产和虚拟资产紧密关联，而且随着技术的不断突破，与现实世界相关的NFT应用也被创造了出来，交易从此将变得更加透明和安全。

尽管如此，NFT 的发展还只是刚刚起步而已，未来的NFT将对我们的生活、社交和投资方式产生更深远的影响，集成不同NFT效用的能力也将为我们带来难以想象的价值。
