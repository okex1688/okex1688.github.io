原文作者：chainleft.eth

NFT 这三个字母缩写相信绝大多数人并不陌生，但就算是加密行业资深玩家也无法给 NFT 下一个清晰的定义。大家可能都清楚 NFT 缩写展开是「Non-Fungible Token」，即非同质化代币，但它真正代表什么含义，其实存在很多争议和讨论。 例如，NFT 仅仅是一个数据凭证吗？它是真正的艺术品吗？非同质化的含义是什么？什么是代币……

其实在 NFT 流行之前，数字艺术已经存在了。早期的数字艺术通常会代表创作者的一些开创性成就，所以本身有一定的收藏价值。那么早期的数字艺术是 NFT 吗？

本文将讨论历史发展过程中关于 NFT 定义、以及「非同质化」和「代币」含义的争论。在进入正题之前，首先了解一下 NFT 有哪些独特的功能。

NFT 的重要意义

NFT 为数字艺术领域带来哪些变化？

数字艺术在 NFT 出现之前就已经存在，甚至加密艺术品也在 NFT 之前出现一段时间了。但是他们从未像 NFT 一样吸引如此多的收藏家。

早在 2011 年就有人在比特币区块链网络上创建了「ASCII 艺术」，以表示对最近去世的早期比特币开发者 Len Sassaman 的致敬。它包含两张 ASCII 图像，一张是 Len Sassaman，一张是当时的美联储主席 Ben Bernanke。

BitLen ASCII

「BitLen ASCII」是区块链历史上一件值得铭记的艺术品，它永久地留在区块链上供所有人查看。但我们回到最初的问题：为什么 NFT 会比早期加密艺术品更受欢迎？

NFT 的出现，让数字艺术品在市场中流通起来，人们可以通过参与市场行为进行买卖 NFT，进而实现其所有权的转移或交换，这与 Beanie Babies 或 Pokemon Cards 等公司发行的收藏品明显不同。NFT 的收藏价值越高，财富效应就会越大，而巨大的财富效应吸引了很多收藏家，同时激励了更多的数字艺术家和创作者参与到 NFT 的发展和创建当中。这也是通常情况下人们将 NFT 去中心化所有权可转移性带来的收藏价值和财富效应视为这场盛宴的开端。

图片来源

「代币」是一个抽象术语，实际上并不存在，只是调用确定可拥有资产代码的简写方式。不可替代的代币或 NFT 满足了人类天生渴望拥有独特事物的欲望，但是「非同质化」的定义却有着很大的分歧。

对于 「非同质化」的讨论

半同质化

在一些 Discord 服务器聊天中，人们会反对将一些著名的数字艺术称为 NFT，例如 Rare Pepes、Curio Cards 或 JakNFT 等，因为他们是半同质化的。这些数字艺术同一个代币的不同版本是可以在没有任何价值损失的情况下进行同等交换，但他们又不同于完全可替代的代币，因为他们在不同数字艺术集合里是不可替代的。

无疑这些都可以称为去中心化数字收藏品，但如果我们要把称为 NFT，那么就要给 NFT 一个清晰的诠释，即 NFT 对艺术家和收藏者有什么独特的意义。

同质化

同质化代币彼此之间可以互换，有些是为了完全可替代的目的而创建的，例如大多数投资者交易的加密资产是可互换的。另一方面，一些同质化代币仍然具有一定的收藏价值，并且存在于去中心化网络上，具有收藏价值的同质化代币也会声称自己是 NFT，但遭到了质疑者的反对。那么问题变成了什么是「收藏价值」？我们尝试从一些收藏家的观点中理解它。

稀有资产具有收藏价值，资产可能在发行时具有一定的特征而变得具有稀缺性，或者说在时间轴上象征某一里程碑事件。例如在 2014 Dogeparty 发行 JOLLYROGER，它是 Dogeparty 代币，在发行时附有图像。虽然每个代币在 JOLLYROGER 中都是可替代的，但由于发行时附有图像，因此 JOLLYROGER 在创作过程中具有收藏价值。著名加密艺术家 Rhea Myers 在 2014 年创作了 MYSOUL 概念艺术品，MYSOUL 虽然没有附加图像，但可拥有且可交易性使它具有收藏价值。同样 Counterparty 上的 XCPinata 资产也具有艺术价值，虽然实际图像在后来才附上的，但是最初的创作意图使它成为一个重要的收藏品。

有一些加密资产在创建时并没有独特的收藏价值，但现在追溯起来会具有特殊的纪念价值，例如 2014 年第一个 Counterparty 代币 TEST，供应量只有 400 个。因为 TESR 的首次发行导致了加密艺术运动（Rare Pepes）的兴起，而且 TESR 本身具有稀缺性，所以现在看来它具有一定的收藏价值。

在早期区块中开采的比特币是否具有收藏价值呢？他们没有图像也并不稀缺，唯一让他们不同的是所定义每个区块链交易的开始和结束位置的 UTXO 机制。正式因为这种可追溯性和独特性让早期开采的比特币具有收藏价值。Casey Rodarmor 创建的 ordinals 项目，旨在让 satoshis 被编号并在交易中进行跟踪，通过这种方式可以观察收藏家对代币发行时间序列的态度。

一些具有收藏价值的半同质化和同质化代币

Namecoin NFT

关于 Namecoin NFT 的争论集中在「代币」的所有权。

最早的 NFT 就是 Namecoin NFT，在 2011 年 4 月 21 日铸造的文本内容为「d/bitcoin」的域名成为第一个 NFT。Namecoin 资产是非同质化的、可拥有的、可转让的代币，这一点没有具体的争论。以太坊创始人 Vitalik Buterin 在以太坊白皮书的第一页提到了 Namecoin NFT。他在白皮书中使用的是「non-fungible assets」，而 NFT 这个词正式出现是在 2017 年。

Namecoin NFT 是第一个 NFT 在很大程度上是被接受的，更多的争议是关于到期时间的设定：如果所有者没有及时续订他们的域名，九个月之后就会过期。

有些人认为如果持有者必须续订他们的域名，这更像是租赁，而非真正的所有权转移，违背了 Web3 的初衷。

这种观点是错误的，因为 Namecoin NFT 为持有者提供了完全的所有权。与租房不同，继续持有所有权的决定完全取决于业主自己。更新主要是为了维护网络，需要向矿工支付的极低费用，更像是基础设施税。在撰写本文时，50 年的续订费用约为 1 美元。

关于 Namecoin NFT 更大的问题是溯源问题，Namecoin NFT 过期之后可以重新注册，这样就会存在持有人变更的问题。

关于 Namecoin NFT 溯源的技术讨论

当域名所有权到期并在一段时间之后重新注册时，Namecoin 域名会发生什么？

当一个域名第一次注册时，会有一个特定的标识符 UTXO，通过特定的 UTXO 就可以查询域名的交易历史数据。UTXO 是一种特殊的代币，本文称这种特殊的标识符为「colored coin」。当域名到期时，「colored coin」将无法使用，即被「烧毁」。之后重新注册该域名时，会有一个新的 UTXO 生成，也就是说当域名过期时，该域名的唯一标识就会无法识别。域名本是是唯一的，所有的更新和交易都会保存在链上。

编者注：UTXO 是一种用于分发加密货币数据位的协议，它允许用户跟踪查询加密货币的所有权交易记录。「colored coin」粗略地描述了一类在比特币区块链上表示和管理现实世界资产的方法，资产发行人承诺可以将其兑换成某些商品和服务，从而将显示世界价值附加到代币上。

不仅可以在区块浏览器上通过 UTXO 查询域名的交易过程，还可以利用域名的功能查询。域名开始就有一个内置的 PRC 模型：「name_show」，通过内置的「name_show」，即使在域名过期期间也可以返回有关域名的信息。

name_show 函数

事实上过期的域名仍然可以正常工作，返回本地状态 name_show 时会表示处于过期状态，如果解析器重新设置，该域名仍然可以工作。同样域名中过期的「id/」名称仍然可以用作登录名。

另一个内置 RCP 方法是 name_history，此功能可以提取域名的所有原数据和历史记录，这个功能表明域名在设计之初就有着可追溯的特性。

有关「代币」的解释

解释 1：「代币」是「colored coin」，它由唯一标识符的域名表示。域名过期之后重新注册时，会分配到一个新的「colored coin」，从而形成新的资产。

解释 2：「代币」本身就是域名，都是具有唯一标识性的资产。Namecoin 是以域名为主要的唯一标识符设计的。当域名到期时，该域名的数据仍然保存在链上，仍然是可以追溯的。

解释 3：「代币」是「colored coin」，它是域名的加密表示。当域名被重新注册时，新的「colored coin」变成新的 NFT，但它仍然代表被重新注册前的资产价值，也就是说「colored coin」注册前后只有所有权发生改变，而所代表的资产价值没有发生变化。

这三种解释将「代币」分为三个种类

  • 加密货币

  • 加密资产

  • 链下资产（适用于引用现实世界资产的情况，例如： Twitter Eggs、Blockheads、Quantum 等。对于 Punycodes、Damselfly、Identities、Bit Domains 不使用）

对于收藏家和去中心化爱好者来说，重要的是以去中心化的方式拥有具有收藏价值的资产。就算是过期的 Namecoin 域名的历史数据仍然存在于去中心化区块链上。这些数据可以以去中心化的方式拥有，所以它仍然被称为去中心化收藏品。

上文一直在从技术原理的角度讨论，接下来开始考虑伦理，哲学和法律角度展开阐述，这些角度具有一定的主观性，但值得借鉴。

Namecoin NFT 溯源的道德与哲学讨论

2021 年 由 NFT 引发的加密市场大牛市让很多 Web3 参与者坚信区块链去中心化、可追溯性等特质会带来一场所有权归属的范式转变。这种观点的持有者支持 CC0（不保留任何权利），对于现实世界的论点甚至是法律不太关心。

编者注：CC0 使科学家、教育工作者、技术加和其他受版权或数据库保护的内容完全开源，以便于其他人可以自由地使用和补充，不受版权法的限制。

这种观点的批评者认为，区块链可以为创作者和收藏家带来很多价值，但是不能忽视物理、伦理和普遍真理。

Larva Labs 与 v1Punks 之间的冲突反映了这两种截然不同的观点。作为 CryptoPunks 的原始创建者的 Larva Labs 长期以来一直认为有缺陷的 v1 合约并不代表他们的真实艺术品（艺术家声明和意图），而 v1Punks 的持有者则认为区块链时间戳反映了 v1 仍然是去中心化收藏品，具有收藏价值。Namecoin NFT 也发生了类似的争论。

图片来源

如果一件艺术品是由创作者在 Namecoin 上创作的，过期之后由同一位创作者重新注册，在哲学意义上所有权并未发生改变或转移。但是如果过期之后被其他人重新注册这种情况就存在很大的争议。

解释 1：在哲学意义上，艺术家创作出作品之后，后来他人重新注册后不能声称自己是创作者；站在这个角度，盗用别人的作品并出售是不道德的。

解释 2：Namecoin 提供 100% 的所有权，是否延长所有权取决于所有者。在哲学意义上，如果艺术家不延长他们的所有权，就表示决定将他们的作品公之于众，任何人都可以认领。如果后来持有者通过营销提高其认可度，那么从获取经济利益是合乎道德的。

当艺术品链接到原始创作者拥有控制权的外部服务器时，情况会变得更加复杂。我们将链下信息考虑在内，那么艺术品到底指的是什么？是链下文件、链接所附的域名或域名对应的「colored coin」其中之一还是全部，这个问题是具有争议的。

出于道德因素的溯源

最近 Punycodes 社区发现了 966 Punycodes 的原始创建者，Punycodes DAO 立刻邀请 haluciphile 成为 DAO 成员，并赠送了多个 Punycode。Punycodes DAO 还承诺将分配以太坊上保险库合约的部分版税给 haluciphile。随后 Punycodes 宣布了一则通告：如果 Punycode 的原始创建者通过钱包验证消息流程证明他们他们的身份，Punycodes DAO 承诺尽最大努力确保原始创建者获得价值，例如代币资产和版税。

收藏价值的基本面和指标

  • 时间戳：区块链上不可更改的客观数据。

  • 稀缺性：集合或版本中的资产数量。

  • 创造力：艺术和概念意图。

  • 技术成就：推动 NFT 发展的技术创新。

  • 铸造的区块链：游戏 NFT 对区块链的互交性和可组合性要求较高，去中心化治理模式仍然很重要。

结论

我们大多数人不会关注于半同质化、收藏价值的含义、所有权到期的争论，但这不影响人们参与到 NFT 带来的这场盛宴。只要可收藏资产可以进行所有权转移或交易，并以去中心化的方式被拥有，它们就可以为 NFT 运动做出贡献。随着人们的大规模使用，令人困惑的 NFT 术语、去中心化收藏品、区块链收藏品之间的定义也会更加清晰。
