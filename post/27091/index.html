你以为新的一天会有一个新的 NFT 项目。不，不对，实际上更像是每天会有十到二十个新 NFT 项目冒出！

是吧，登录一下 Twitter ，你就会看到推特上的 NFT 大 V 们正在便宜兜售一些新的表情包。扯淡！ 这些人一定知道一些（我不知道的）事。他们赚了钱。我是不是该跟风买进（行话叫做「ape」）这个狗屎图片的版权？ 它还很便宜，再加几个旧表情包 remix 一下怎么样？ 我怎么会赚不到钱？在这插入一些旧的表情包！那张狗屎图片的低价已经涨了 5 倍了，我绝对会发大财！

这听上去很熟悉吗？如果你在过去几个月里关注了 NFT 社区，上面那些话应该绝不陌生。过去数周以来，我一直看到萌新在交易根本没什么价值的狗屎图片。追为了逐收益，被割了的韭菜，在没有引导的情况下胡乱操作。

最近，在精彩的 Pixel Vault Discord 中，我们就 NFT 领域投资理论的价值进行了精彩的对话。受这一对话的启发，我想扩展我自己的 NFT 投资理论，讨论一些实例，并希望启发普通投资者开发自己的理论，以实现自己的目标。

话不多说，我接下来介绍 0xBEW NFT 投资理论：（新颖 * 永恒 * 实用）^ 社区。

新颖

Bored Ape #3817

新颖可以有多种形式，并且通常与第一个出现有关（尽管不一定必须是第一个）。一个新颖的项目是尝试做一些以前没有做过的事情，或者以创造性方式对一些已经做过的事情构建其基础。

对我而言，这些项目的新颖性通常取决于其抱负和创造力。

一般来说，我认为新颖项目高风险高回报。因为以前没有做过，或者它是对现有项目的全新演绎，你的投资很容归零。同理，如果该项目成功，你将获得在其他地方无法匹敌的巨大收益。

当我第一次了解 NFT 及其底层技术时，我立即想到了数字授权。使用代币作为开启访问之门的钥匙，似乎是一个完美的产品市场契合点，我有点惊讶我没有看到更多。当我第一次了解到「无聊猴游艇俱乐部」（Bored Ape Yacht Club/BYAC）时，我觉得我终于找到了一个完全符合我预期的项目。

BAYC 「仅限会员」制度正是我从一开始就一直在考虑的方向。BAYC 是一个充满新颖事物的项目。成败取决于人们对加入这一俱乐部的渴望程度，这很令人兴奋。这个项目牵扯的不仅仅是一张个人头像照片，而是更多的东西。

几个月后，我认为可以肯定地说 BAYC 的成功远远超出了最乐观的「猿人」最疯狂的想象。那些参与其 NFT 铸造的人已经赚得盆满钵满，而那些近期刚入场的人也轻松赚到 10 倍回报。 从商品空投到「Bored Ape Yacht Club Bathroom」再到元宇宙（ Metaverse）会所，BAYC 创造了一个人们真正想参与其中的东西。

BAYC 对新颖性的押注得到了回报，如果说跟风山寨是一种奉承的话，整个 NFT 领域显然完全赞同。

永恒

Tyler Hobbs: Fidenza #131

永恒是另一个可能与成为第一相关的原则。不过，一个项目不一定要成为第一个才能永恒。

Ringers 并不是 Art Blocks Curated 市场发布的第一个产品，但我相信它们是永恒的。同样，CryptoPunks 不是第一个链上 NFT，但同样，它们是永恒的。投资永恒的东西是一个长期的赌注，而关于你正在投资的东西，可能还会有额外的理论。这也是最难评估自己是否能成功的原则。

对我来说，最能体现永恒的项目是 ArtBlocks。 正如我之前提到的，我关于 ArtBlocks 的理论还有另外一个论点：加密艺术是一种合法的艺术运动，而我们正处于艺术史掀开新篇章的早期阶段。考虑到这一点，我相信 ArtBlocks 秉承链上生成和一流策展的宗旨，将通过 ArtBlocks 平台铸造的项目自然地置于这一新艺术运动的高层梯队中。

我没有期望我现在的论调能在五年或更长时间后依然是对的。正如我前文所说，这个有很长的前导时间。也就是说，我相信早期迹象验证了我的投资。

Ringers 和 Fidenza 系列的作品非常受欢迎，这表明 NFT 社区对高品质艺术的需求巨大，而通过苏富比拍卖行（Soethby）出售的大量 19 件 ArtBlocks 作品，则显示了传统艺术世界中的机构对 ArtBlocks 的认可，认可 ArtBlocks 将成为艺术史上这一新兴时期的定义性平台。

实用

PUNKS Issue #1

实用性可以归结为「不仅仅是一张 JPEG 图片」。

今年早些时候，盲目跟风者（Normie）一再向 NFT 领域发出的批评言论，都集中在为什么有人会为一张 JPEG 图片花费数百万美元！你只需右键单击保存一下！OK，现在你也有了这张 JPEG 图片。

请原谅这种过于简化的描述方式，但 NFT 的底层技术不仅仅用于容纳一张图片的元数据。关注利用 NFT、将 JPEG 变成不仅仅是 JPEG 的项目，是寻找附加价值以及潜在收入和流动性的好方法。

这个原则让我想到了 Punks Comics 和 Pixel Vault。我在 Twitter 上关注 Beanie 有一段时间了，因为他一直在梳理这一漫画。我对媒体公司利用 NFT 围绕早期区块链场景构建知识产权的想法很感兴趣（看起来很新颖）。 不过，当 Punks 原创问题的细节发布时，伴随复杂的代币经济学的完善，该项目的潜在效用成为最大的吸引力。

真的不仅仅是一个 JPEG，原创问题是效用的体现，再加上新颖的博弈论以及引人注目的代币经济学。

我积极投资 Punks 的理论是，考虑到每个代币的实用性，每个代币的价值都非常高。经过对该项目几个月的研究，我对 Punks 或 Pixel Vault 的信念从未如此强烈。随着 Punks Comics 二级市场的大幅增长，团队为开发具有这种实用性的项目所付出的辛勤工作开始显现出成果。

社区

Life Well, Parallel Alpha Masterpiece

在我的投资理论中，社区是放大器。一个糟糕的社区甚至可以让一个伟大的项目瘫痪，而一个优秀的社区可以让好的项目势不可挡。

与我们前面谈论的三个原则不同，如果你尽早投资，社区问题可能是你最难获得准确信息的一个方面。在这种情况下，你需要更坚定的信念，随着社区的发展，紧紧把握脉搏是很重要的。相反，如果你入场较晚，项目社区可能是你最有信心的维度。

在社区中以真正得到切实感受的唯一方法是参与：加入项目 Discord，在 Twitter 上关注团队并参与互动。如果你给他们机会，他们就会暴露真实的自己。不要害怕与其他匿名人士混在一起。你怎样能知道一个项目的社区是好是坏？就像政府对色情的态度，你看到它时就会知道了。

当然，BAYC 是一个很好的例子，但 ArtBlock、Pixel Vault 和 Parallel 都进一步演示了一个伟大社区是什么样子。花些时间参与这些项目的 Discords，你就会明白为什么。

我文中提到的这些项目清单绝非完整，仅供参考。我的投资理论会是一份动态文件。我会通过不断学习，不断更新我的权重，并修改我的内部决策树。

NFT 领域也是动态的，一直在变化。你的投资理论应该足够灵活以应对变化，最重要的是，它应该与你本人相匹配。

我写这篇文章并不是要吹嘘我总是投对赢家项目——不是这样吧（安息吧，我对 Nekos 表情包的投资）。像每个人一样，我也曾投过失败的项目。撰写投资理论的目标不是抹去失败投资的记忆（这是不可能的），而是准备好在你认为最有信心的项目上进行更大的尝试。

这可能显而易见，但很难不被 「害怕错过」FOMO 和 pysop 的情绪所裹挟。 要使投资理论编程成功的实践，它必须基于对自身局限性和目标的诚实描述。也许这看起来很难，那是因为确实如此。

也就是说，如果你能将理论投入实践，我真的相信随着时间的推移，会产生更好的结果。你可能会错过狗屎图片的空投，但在一堆垃圾中错过一两个赢家项目，没什么大不了，可错过投资你真正深思熟虑的项目，那可不一样。

朋友，直到下一次…
