亲爱的Bankless Nation，

当谈到优秀的 NFT 项目时，我们将才华横溢的创造者、创新机制和充满活力的社区视为重要优势。 

然而，糟糕的 NFT 项目具备哪些特征？ 

需要注意哪些关键的危险迹象？

下面，我汇总了十几个危险信号，供您在研究新的 NFT 项目时参考。危险信号范围从疏忽到彻底操纵。

NFT 危险信号列表

1）过高的Mint价格

有了无穷无尽的创意可能性，关于什么构成“合理”的 NFT Mint价格有很大的回旋余地。这里没有单一的正确答案。但是，如果一个新团队的收费远高于其前辈或同类项目的收费，这表明他们可能更专注于保护你的 ETH。 

 2）粗略的社交媒体活动

一个 NFT 项目突然向您发送消息，以将您的注意力引向即将到来的下降，实际上您在大约几个小时前已经看到了一条推文。现在您单击该集合的个人资料并注意

1)您的关注者都没有关注该项目

2)对于一个新项目，它有非常异常的大量关注者。这些都是危险信号！合法项目不会通过 DM 或使用 Web2 广告来引起个人的注意；他们有自己的吸引力，不必购买追随者。 

3）地板价格维护

看到项目领导讨论或努力维持某个 NFT地板价？那就逃跑，快速逃跑。他们只是在运行一个财务计划，对文化价值或社区几乎没有真正的兴趣。这种不真诚会导致他们在未来遇到的所有麻烦，所以要小心。  

4）滥用 Discord 行为

一个令人担忧的巨大原因是，当 NFT 团队禁止人们提出合理的问题时，它会禁止其 Discord。你希望看到项目是包容和直率的，而不是好斗和神秘的。团队没有充分的理由滥用他们的审核职责。  

5）无审核

来自安全公司的审计报告并不完美，但它们确实提供了一些基本的安全检查。如果您在 Web3 中，您希望使用经过审计的 DeFi 和 NFT 项目，因为未经审计的基础设施更容易被滥用或利用。问题是每个审计公司最近都被需求淹没了。一些新的 NFT 项目不想永远等待，因此通过启动未经审计的基础设施来抓住机会。相应地谨慎对待此类项目。  

6）糟糕的设计

一个设计糟糕的项目可能会导致bug、低效的gas、可利用的mint等更多问题。对于新手和大量非技术专家来说，评估糟糕的代码等等是很困难的，所以跟随那些知道得更多的人并从他们的判断中学习是关键。

7）劣质艺术品

如果您遇到的艺术项目看起来像是由 Fiverr 内容农场在一个小时内构思并完成的，那么也许就是这样。因此，如果 NFT 团队本身并没有非常重视他们的内容，那你又何必呢？请记住，有品位的/迷人的/滑稽的/怪异的极简主义作品和庸俗的废话之间存在着巨大的鸿沟。你来判断吧，但根据我的经验，这个范围内的项目是不言自明的

8）出售白名单

我开始看到的一个新现象是NFT团队出售他们自己的NFT白名单。哇。这是抢钱行为。

9）过度承诺

新 NFT 收藏的创造者是否说他们的作品是“蓝筹质量”，并会成为下一个无聊猿游艇俱乐部（BAYC）？该项目的路线图是隐喻地延伸到月球，还是声称为月球上第一个 NFT交易市场？避免纯粹的夸夸其谈。 

10）完全匿名的团队

拥有一个完全匿名的团队本身并没有问题。人们有隐私权，在许多情况下，项目和代码库的质量等，将为自己说话。然而，在最坏的情况下，完全匿名的团队可能会造成严重的损害，比如增加流动性，然后几乎没有责任。

11）缺乏跟踪记录

团队中缺乏NFT经验丰富的项目比那些只有一个熟练NFT的团队更容易失败。

12）在 Etherscan 上未经验证

通过Etherscan验证你的代码可以确保公众可以在以太坊上正常工作。使用未经验证的智能合约意味着您没有这种基本的性能保证，因此在进行NFT活动时，您一定要注意这一点。例如，您可以看到Cool Pets智能合约是如何通过其选中状态来验证的。对于没有类似选中标记的项目要小心。

13）拥有者分布不均

让我们假设一个10k的PFP项目吸引了你的注意，但是你去查看它的OpenSea收集页面，发现项目的NFT的所有权分布在几百个人身上。这是一种疯狂的所有权集中，意味着所有这些人都可以对市场施加巨大影响。

结论

上面所描述的一些危险信号应该是自动的交易破坏者，如滥用Discord行为。其他的，像完全匿名的团队，并不总是重要的，但至少是值得注意的。养成发现这些危险信号的习惯，这样你就可以更精明地实施你的NFT交易行为。
