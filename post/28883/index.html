a16z合伙人Chris Dixon发表最新的观点，他将互联网的浪潮分为了两个时代，一个是拟物时代，类似于将线下邮件搬到了线上变成了电子邮件，另一个是数字原生时代，比如网站，是互联网时代开启后独有的，而Token同样是数字原生，大多数的web3仍然只是停留在拟物时代。

以下为观点全文：

1/ 代币是一种新的数字原始物，类似于网站。

2/ 主要的计算浪潮一般有两个时代：拟物时代和原生时代。

3/ 在拟物时代，设计思维在很大程度上是从较早的领域改编的。例如，早期的网络大多是对前互联网活动的数字化改编，如写信和邮购，那时的网站大多是只读的。

4/ 大约十年后，技术专家才开始认真探索网站可以是读/写的，由用户产生内容的想法。这导致了网络原生类别的发展，如社交网络、众筹和社会生产力应用程序。

5/ 这种模式在加密货币/Web 3中不断重复。有一些伟大的原生Web 3产品，但总体而言，我们仍然处于拟物时代。许多Web 3产品是对旧领域的改编。

6/ 流行的拟物 Web 3想法包括离线票务、供应链管理和离线资产的记录保存。这些可能是好主意，就像只读网站是个好主意一样，但它们只触及Web 3的表面。

7/ 今天的很多NFT都是从艺术品和收藏品的线下世界改编而来。这导致人们认为NFTs仅限于这些领域，就像人们曾经认为网络仅限于宣传册和杂志一样。

8/ 代币–实物和NFT–最好被认为是新的数字基元，在灵活性和通用性方面类似于过去的数字基元，如网站。

9/ 代币赋予用户产权：拥有互联网的一部分的能力。

10/ Web 2遗漏了数字产权。当你使用一个网站（或应用程序）时，它只会让你借用或租用东西。想象一下，如果在现实世界中，你每次去一个新的地方都要从头开始买东西，这就是Web 2。

11/ 像网站一样，代币是数字基元，可以被概括为代表几乎任何东西–金钱、艺术、照片、音乐、文本、代码、游戏项目、控制权、访问权，以及人们在未来梦想的任何东西。

12 /用户现在可以在他们的钱包里有一个持久的物体库存，他们可以从一个应用程序带到另一个应用程序。如果他们的物品升值，用户就会得到升值的好处。这与Web 2有很大的不同，在Web 2中，涨价的部分主要由科技公司获得。

13/ 我们仍然处于Web 3的拟物时代，但开始看到新一轮的原生应用，这些应用之前没有类似的东西，以前根本不可能存在。

14/例如，在DeFi企业家开创的机制设计的基础上，新一波的DAO正在探索团体聚集、汇集资源、建造东西和自我管理的方法。

15/ 像Loot这样的可组合的NFT游戏激励社区围绕一组NFT构建整个世界–如果没有Web 3所带来的所有权和可移植性，这种活动是不可能的。

16 /可替换的代币没有任何内在的东西需要与货币和金融相关，而NFT也没有任何内在的东西需要与艺术和收藏品相关。

17/ 这些都是很好的初始应用，而且可能仍然非常重要，但代币最好被认为是一种新的数字原始物，类似于网站–围绕互联网新时代的原子单元。
