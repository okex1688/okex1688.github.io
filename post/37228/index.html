作为当下最好用的 NFT 聚合平台，Gem 被 NFT 收购似乎最开心的只有当事双方和背后的机构。从链上数据看，Gem 只用了不到半年时间就开始成为部分用户的交易入口，不论从产品体验和快捷程度，Gem 确实要比 OpenSea 实用几个量级。

不过收购并不被用户认可，OpenSea 早已走了 Web2 之路，而 Gem 被收购也大概率不能为用户提供 Token 的权益，即便他们在社交媒体表示仍有发行 Token 的可能，但用户们并不认可。

除 Gem 以外，用户还有哪些还没有 Token 的 NFT 交易平台可以值得尝试一下？

Genie

除了 Gem 外最大的聚合平台，用户可免费使用 Genie List，当前该平台已支持 OpenSea、LooksRare、X2Y2、NFTX、NFT20、Rarible 交易市场。据官方披露，用户最多节省的 gas 费可达到 40%。与 Gem 一样，Genie 也有批量交易的功能，可将选中的 NFT 加入购物车一次性购买多个 NFT。

Genie 创始人兼首席执行官 Scott 曾发文暗示，或将发行名为 GENIE 的 Token。不过当前还未发币，需注意风险。除此之外，还可关注一些 NFT 的 P2P 交易平台。

sudoswap

sudoswap 是一个点对点交易平台，支持 ERC-20 Token 及 ERC-721、ERC-1155 NFT 资产之间相互交易，在交易对象接受交易之后，资产才会自用户钱包中转移，交易过程中只需花费 gas 费用。用户可在 sudoswap 中设置到期时间，在截止周期内可自由交易，到期后交易自动失效，不必再耗费精力前去取消。

该项目如今在开发 NFT AMM 自动做市功能，这个专为 NFT 设计的集中流动性 AMM 协议名为 sudoAMM，不仅交易者在 NFT 与 Token 之间进行互换时可享受较低滑点，流动性提供者也可以更好地控制其 LP 价格范围。与其他 NFT 市场不同的是 sudoAMM 使用链上流动性池而不是链下订单簿，这也为 NFT 与 DeFi 之间架起了一座桥梁。

项目计划发布的原生 Token 为 SUDO，随之 sudoAMM 协议也将由 SUDO Token 治理，在此之后，将构建 DAO 实现最后的去中心化。

NFT Trader

与 Sudoswap 类似，NFT Trader 的主要功能也是买卖双方两者进行点对点的置换交易，支持 ERC-721、ERC-20、ERC-1155 这几种类型 Token，以及 ETH。在双方确认交易之前，用户资产会暂时托管到平台部署的智能合约之中。 

NFT Trader 企图将 NFT 交易游戏化，以游戏排名奖励的方式激励用户交易，用户还可从该平台铸造自己的 NFT。

Swap.kiwi

Swap.kiwi 也是如此，与前两者相比关注者较少，该平台支持 ERC-721、ERC-1155 和 ETH，同样不属于 NFT 销售平台，而是 P2P 交换平台，提供将 NFT 换成 NFT 以及将 NFT 换成 NFT + ETH 的服务。

用户使用 P2P 功能主要是为了免除掉平台版税以及艺术家收入，当双方达成共识后，只需支付 gas，在一定程度上缩减了成本，降低了交易摩擦，这也提升了 NFT 的流动频率。

这些平台还处于发展初期，不同于传统的 NFT 交易市场，为用户之间 NFT 的流转提出了新范式，较之前的 NFT 交易市场相比，交易过程简单、透明、自由度较高，期待未来 NFT 市场随着用户要求的提高逐步完善。
