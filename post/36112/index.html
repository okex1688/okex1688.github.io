周三，全球最大电影和电视娱乐制作公司之一的华纳兄弟（Warner Bros）在赞助的<NFT | LA>线下峰会上宣布，旗下的 DC 漫画公司（DC Comics）将与以太坊侧链 Palm 在 4 月 26 日发行以蝙蝠侠为主题的 3D NFT—蝙蝠面具（Bat Cowl），总量 200,000 个，售价为 300 美元，并承诺为这个系列的 NFT 赋能。

作为 DC 最畅销、卖座的超级英雄 IP，蝙蝠侠在诞生的 80 余个春秋岁月中，在全球范围内累积了数亿粉丝，激励了一代又一代读者和观众，给无数孩子心中种下了一个除暴安良的梦，而蝙蝠面具形象的改变也是最具代表性的。

据悉，每个蝙蝠面具 NFT 都有一组特征，通过引入精心设计的颜色、纹理和升级，伴随着 NFT 数字所有权的属性，持有者将以 NFT 作为独特身份穿梭在 DC 漫画宇宙未来的规划中，并能够访问 DC 宇宙粉丝论坛的访问权限，包括幕后内容、精选粉丝活动、实物收藏品和限量商品等。

除此以外，更引人注意的当属该 NFT 系列的创建者已经制定了一项为期两年的路线图，每 52 天就为 NFT 持有者在 VR、元宇宙和钱包功能等领域构建应用程序。 

DC 野心的开始

NFT（非同质化通证）在过去一年陡然盛行，并受到各方的追捧，例如数字艺术家、名人、传统行业品牌、音乐人、服装设计师以及运动员。即使是之前从不愿意接近加密货币的全球 500 强企业也不禁对 NFT 产生了浓厚的兴趣，其中不乏 NBA 这样的体育联盟，百事、可口可乐这样的饮料公司，环球音乐集团这样的音乐公司，Line 这样的社交媒体，以及 Gucci、阿迪达斯这样的服装品牌。

NFT 是一种特殊类型的加密货币，其中存储了唯一的可识别信息，以证明数字资产的所有权，如图像、音频、视频，或其他类型的收藏品，如书籍、博客、帖子或备忘录。

NFT 如此盛行的原因之一正是许多公司发现了它们的固有价值。除了稀缺性和观赏性以外，NFT 的属性，使得它可以在现实社会中为艺术创作者、收藏家、内容创作者等涉及知识产权类的人群提供保护，从根源上杜绝山寨抄袭现象带来的损失。 因为所有人都可以通过区块链查询它是否是出自正规授权发行。

那么兼具观赏性和收藏性的漫画与 NFT 的“交叉”就容易理解了。成立于 1934 年的老牌漫画公司 DC 以超级英雄而闻名，旗下包括蝙蝠侠、超人、神奇女侠、绿灯侠等。然而，DC 与 NFT 的故事，还要回到一次版权争议事件上。

去年 3 月，曾在 1976 年至 1981 年期间绘制了神奇女侠漫画、现年 87 岁的意大利资深漫画家 José Delbo 创作了以神奇女侠为主题的 NFT，并销售了 185 万美元，其中的一些收益被捐给了非营利组织 Girls Who Code。值得注意的是，Delbo 的 NFT 作品上并没有 DC 的官方 Logo，这说明 Delbo 并不是根据过去的创作为基础，也没有获得 DC 的授权。

图片：José Delbo 的神奇女侠 NFT

一周后，DC 的法律事务高级副总裁 Jay Kogan 写给全体自由漫画家的一封信被曝光出来，Kogan 在信中表示，DC 正在探索进入市场的机会，以通过 NFT 进入 DC 原创数字艺术的分销和销售市场，包括专为 NFT 市场创作的新艺术，以及为 DC 漫画出版物提供的原创数字艺术。

此外，DC 还警告其自由漫画家出售包含某些角色的 NFT 的法律后果：“请注意，无论是否为 DC 的出版物呈现或在与 DC 的合同约定范围之外呈现，都不允许出售任何具有 DC 知识产权的数字图像，无论是否具有 NFT。”

自此，DC 算是正式承认探索 NFT。

同年 10 月，DC 举办了 DC FanDom 2021，据 DC 公布的数据，全球有 6600 万人观看了活动的直播。DC FanDom 是 DC 一年一度的线上展览与官方资讯的发布会活动，DC 将会在这一天公布相关的电影、游戏、剧集等的最新资讯和独家爆料，这一天也被粉丝亲切的称之为“DC 粉丝年会”。

除了想要了解 DC 未来资讯的粉丝以外，这次活动还引起了加密社区的关注。

在活动举办前的一个月，DC 与专注于 NFT 的以太坊侧链 Plam 合作，计划为注册参加 DC FanDom 的粉丝免费发放 FanDome NFT，粉丝还可以通过在社交媒体平台上分享有关活动的信息来解锁第二个免费的 NFT。每件 NFT 都是由 DC 的首席创意官 Jim Lee 手动精选的。根据 Palm 在活动后透露，共为粉丝发放了数十万个 FanDom NFT。

鉴于活动吸引了 6600 万粉丝的观看，所以这可能是有史以来最大的 NFT 发行活动之一。

通过 Plam 将品牌连接到 Web3.0 

说到 DC，就不得不来聊一聊它的“老东家”华纳兄弟。

作为全球最大的电影和电视娱乐制作公司之一，华纳兄弟出品了太多太多我们熟悉的电影，例如《黑客帝国》、《金刚》、《我是传奇》等等。有如此多的经典 IP 攥在手里，华纳兄弟自然成为了首批探索 NFT 的电影公司。

去年 7 月，华纳兄弟与基于 Palm 网络的 NFT 市场 Nifty’s 合作，后者获授权推出了基于电影《空中大灌篮 2》的 NFT 艺术品，并向所有注册该系列 NFT 画廊的用户免费发放一个 NFT。华纳兄弟全球品牌与体验总裁 Pam Lifford 接受采访时表示：“以这种新方式让我们的观众与他们最喜欢的角色互动、与其他粉丝交流并在屏幕内外欣赏我们的电影。”

而参与这个电影 NFT 创作的，也有 Palm。

随着 NFT 热潮传遍全球，有关“NFT会危害环境”、“以太坊速度太慢了”、“购买NFT的手续费太贵”等观点也随之而来。为了解决这些问题，很多 NFT 领域的开发人员一直在研究解决方案，其中便包括 Palm。

Palm 是一个完全兼容以太坊 EVM 的侧链，允许用户在网络上建立 NFT，并在不牺牲互操作性的情况下提高交易速度。在 Palm 网络或以太坊主网上创建的任何 NFT 资产都具有相同的形式和规范，并且可以通过跨链桥在以太坊主网与 Palm 网络之间互相发送。

与 Avalanche、Polygon、Harmony 这类公链不同的是， Palm 诞生的初衷就是为了缓解 NFT 可持续性影响，根据可持续经济基础设施公司 Patch 的研究发现，Palm 网络上一笔交易的碳足迹仅仅相当于发送三封电子邮件。此外，Palm 在其配置中还内置了碳税，用户在使用网络资源时强制支付。

Palm 创始人之一的 David Heyman，曾作为电影制片人参与制作多部华纳兄弟公司的高质量电影，例如《哈利波特》、《地心引力》等，在电影行业享有极高的美誉。正是他的坐镇，让 Palm 在启动之初便与华纳兄弟、美国职业棒球大联盟、英国知名艺术家 Damien Hirst 合作。

Palm 网络的开发公司 Palm NFT Studio 于去年 12 月完成了 2700 万美元的 B 轮融资，由微软的 M12 风险基金领投，华纳兄弟、Griffin Gaming Partners、SK Inc.、RRE、Third Kind Venture Capital、Sfermion、The LAO 参投。

M12 在新闻稿中称赞道：“技术和社会范式的转变正在迎来 Web3.0，这将从根本上改变现有的技术、商业模式和我们互动的方式。Palm NFT Studio 是 Web3.0 的关键组成部分之一——在品牌、企业、创作者和消费者向 Web3.0 推进的过程中连接它们。 ”

蝙蝠面具 NFT 将被赋能

虽然 DC 并不是首次发行 NFT，也不是发行 NFT 的首家漫画/影视公司，例如其竞争对手已经多次在数字资产市场 VeVe 上发行基于超级英雄的“3D数字雕像”，而《哥斯拉大战金刚》、《教父》、《环太平洋》等电影已经被授权发行了相关的 NFT，但他们中很少有为 NFT 制定未来计划，大多仅以收藏品为主。不过， DC 已经承诺为 200,000 个蝙蝠面具 NFT 绘制了一幅未来两年赋能的路线图。

通过路线图我们可以了解到：

1. 蝙蝠面具 NFT 的持有者将可以访问 DC 宇宙网站上的私人粉丝论坛，以及访问粉丝活动、实物收藏品、独家商品和幕后内容。而蝙蝠面具 NFT 也将融入未来蝙蝠侠漫画的故事中。

2. DC 将每 52 天为 NFT 推出新的实用程序，为持有者解锁更广阔的功能。此外，DC 还计划将通过“元宇宙”和 AR 体验等功能来扩展功能集。

3. 在 4 月底发行后，5 月将可以在 Palm 网络上进行交易，包括 FanDome NFT。

4. 优先访问未来发行在 Palm 网络上更多的 NFT。

5. NFT 未来可以转移到 Metamask 等非托管钱包中。

去年免费领取到 FanDome NFT 的粉丝将获得提前购买蝙蝠面具 NFT 的机会。
