当你听到 “NFT “这个词时，你首先想到的是什么？你们中的一些人可能会想起某些人的名字，他们在出售NFT藏品后一夜之间成为千万富翁。其他人可能会想到那些允许他们日入斗金的play-to-earn游戏，或者正在进入NFT领域的公司名单。

你也可能是 那些 对这个新的流行语 仍持怀疑态度的人，认为它就像郁金香泡沫。

我们将解释人们对NFT的一些最常见的误区，你可能会有新的看法。

误区一：NFTs毫无价值

在我们深入研究这个例子之前，讨论一下什么是NFT才是公平的。NFT是区块链上非同质化代币。任何数字资产都可以成为NFT–从数字绘画、备忘录、音乐、推文、GIF等。

有人可能会说，NFT没有价值，因为在线用户可以 “简单地右击并保存数字资产”，将其放在他们的桌面上。不幸的是，这是人们最大的误解之一。

NFTs的所有权被记录在一个基于区块链的网络中，这意味着没有人可以伪造NFTs。有人可能会说，人们可以简单地对数字资产进行截图，但事实是：与其说是 “偷 “数字资产，不如说是在网上分享截图的行为可能会变成一个导火索，导致NFT成为病毒似营销，并提高其正品价格。

如果你是一个数字艺术家，NFT允许你证明你的创作的所有权，并以一种安全和有效方式收取版税。另一方面，如果你是投资者或买家，NFTs允许你在经济上支持你喜欢的艺术家（当然，也有合法的吹嘘权利，说你拥有艺术作品）。

同样，如果你是一个游戏玩家，你将有机会在享受游戏中的NFT资产的同时赚钱，而这些资产的所有权是可以转让的，被大型游戏巨头所控制。

误区二：NFTs会让你暴富

虽然有些人认为NFTs “一文不值”，但另一个极端认为NFT是所有人快速致富的解决方案。真相是，NFTs并不是一经铸造就会自动成为有价值的东西。有许多因素在背后发挥作用，使NFT具有其目标价值。

首先，NFT背后的创造者确实在决定价值方面发挥了作用。他们可能是一个有影响力的人，有预先存在的社区支持者，或者他们的创造有一些数字媒体价值的意义。

NFT领域每天都充斥着暴富神话故事：

就在去年，“无聊的猿猴游艇俱乐部 “（bayc）背后的四个三十多岁的年轻人–发行了一万个NFT，这些NFT包括了卡通猿猴类动物，创建了一个价值10亿美元的卡通猿猴生态系统,并解锁了他们生活的虚拟世界–在这之前他们生活方式平淡，白天工作，现在，他们成了亿万富翁，靠着前卫的、杂乱无章的艺术作品大获成功，这些作品也是一个疯狂社区的会员卡。还有什么比这更像加密朋克呢？

事实上，当你放大观察BAYC可能并没有什么意义。它可能看起来不公平、不合理，而且在许多情况下。炒作是成功的首要条件，而不管艺术是否好。如果背后有炒作，一个NFT项目几乎总是会做得很好。相反，可能有一个NFT项目花了大量的时间来建立艺术和实用性，市场仍然不会买单。

正确投资NFT

你对一个项目的研究应该远远超出它是否销售的基础。谁是社区，谁是团队，他们提供什么，他们的效用是什么，他们的计划是什么？重要的是，你要做出明智的决定，因为尽管在市场被FOMO驱动时，你可以轻松实现盈利，但明确的决定才会对你有长远的帮助。

要有探究精神，要有好奇心,这不仅能帮助你了解更多关于项目的信息。为自己设定标准，该项目是否符合你所要求的期望？

NFT的市场是7*24小时。你不可能跟踪所有的事情和信息。学会与社区接触，寻求支持和洞察力。

那么，NFT会让人们瞬间致富吗？总是需要努力和一些运气的成分来实现它。

误区三：NFT是最方便的洗钱工具

只要有炒作，就会有骗子利用它的机会。NFT也不例外，特别是人们不必在NFT市场上使用他们的真实身份。

然而，将NFT与洗钱和其他非法活动联系起来是不太公平的，因为它只占加密市场的一小部分，而且实现NFT的区块链技术是完全透明的。换句话说，每笔交易都被记录下来并可追踪–这使得人们很容易发现骗局。

根据周三发布的Chainalysis 报告，通过购买和出售( NFT ) 进行的洗钱是犯罪活动的一个小部分。

该报告称：“虽然实体艺术品中的洗钱活动难以量化，但由于区块链固有的透明度，我们可以对基于 NFT 的洗钱活动做出更可靠的估计。”

Chainalysis 追踪了价值至少 442 亿美元的加密货币发送到 ERC-721 和 ERC-1155 合约（与 NFT 市场和收藏品相关的两种以太坊智能合约）， 高于 2020 年的 1.06 亿美元。

报告称，“与我们在整个 2021 年追踪的 86 亿美元的基于加密货币的洗钱活动相比，所有这些NFT交易活动都是九牛一毛，”建立对 NFT 的信任的主要风险，应由市场、监管机构和执法部门更密切地监控。”

同一份 Chainalysis 报告还指出了清洗交易的增长，NFT 所有者通过从他们控制的加密货币钱包中向自己汇款来“出售”NFT 的 做法，并创造了价值的虚假印象。长期以来，清洗交易一直是加密货币交易所的一个问题——它给人一种交易量的误导。

事实上，NFT平台必须采取行动，防止此类问题进一步升级。其一，他们可以考虑监测他们的平台，并制定严格的规则，反对清洗交易，以建立信任。

误区四：NFTs对环境有害

鉴于用于铸造、购买和出售NFT的加密货币有很大的碳排放，NFT经常被认为是对环境的危害。

大多数 NFT 使用以太坊区块链，该区块链使用工作量证明系统来确认每个新区块。这意味着网络上的所有计算机（即矿工）竞相解决一个复杂的问题。网络上第一台解决它的计算机将获得确认区块的权利，并收取与区块中每笔交易相关的gas费，以及奖励。

随着以太币价格的攀升和gas费的增加，在工作量证明系统中解决该问题的价值也在增加。结果，矿工被激励投资更多的计算机能力，这从电网中汲取了越来越多的能量。如果该能源来自排放温室气体或对环境有害而没有任何抵消它的来源，它将对环境产生负面影响。

但很难确定 NFT 如何具体影响 环境 。NFT 代表以太坊区块链上的一小部分交易。目前尚不清楚他们是否有意义地改变了矿工为解决这些复杂问题而添加到网络中的计算机能力。但它的影响可能没有加密货币和去中心化金融 ( DeFi ) 的 广泛采用那么大。

环保的 NFT

涉及改变 NFT 的底层技术，可以使用依赖于更节能的股权证明系统（POS)的区块链可以减少归因于 NFT 的 消耗和温室气体排放。

一些区块链已经使用了权益证明系统并支持 NFT，包括Cardano ( ADA )和Solana ( SOL )。以太坊一直在努力迁移到称为以太坊 2.0 的权益证明系统。不过，它多年来一直在努力进行更改，并且没有明确的转换日期。

环境的另一个解决方案是在区块链之上使用Layer 2。第二层将允许任何人在区块链之外进行交易，然后在区块链本身的一次交易中一次性对它们进行批量处理。

与洗钱的误区类似，NFT只占区块链支持的市场交易很小一部分。随着包括以太坊在内的越来越多的区块链公司希望使自己更加节能，我们可以预期在不久的将来，可持续能源将在网络中运行。

在评估NFT是否值得支持或反对它们时，我们必须以一种全面和客观的方式看待问题，即使 NFT 越来越受欢迎，就像任何其他资产类别一样，泡沫破裂的风险也始终存在。NFT 可能是当下的白热化现象，但与任何新技术一样，它们在被广泛接受并成为真正的主流之前还有一段路要走。
