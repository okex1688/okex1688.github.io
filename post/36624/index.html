搭上 Arbitrum 这趟快车

Arbitrum 的现状

图片来自 L2 BEAT

根据 L2 BEAT ‌提供的数据，目前在所有以太坊最先进的扩容方案中，有价值 67 亿美元的总价值锁定（TVL）。相应地，Arbitrum 目前近 40 亿美元的 TVL 意味着该扩展解决方案目前占到了 50% 以上的市场份额。

虽然 Arbitrum 的用户群规模与以太坊的规模相比仍然相形见绌，但 Nasen Arbitrum 的仪表板‌显示，在过去的一个月里，后起之秀 L2 托管了令人印象深刻的大约 15 万个独立活跃地址，自其成立以来托管了近 5000 万地址。作为比较，Optimism L2 在当月托管了大约 4 万个地址，到目前为止托管了超过 2400 万个地址。

挖掘 Nasen，获得更多的精英分析结果

关于 L2 的路线图，扩展解决方案的创造者最近宣布了 AnyTrust Chains，它将沿着 Arbitrum 运行，并提供超低价交易，以及 Arbitrum Nitro，它将优化 L2 的网络并进一步大幅降低其交易费用。

当前 Arbitrum 上的 5 大机会

下面，我整理了一些我认为目前在 Arbitrum 上对冒险者来说最引人注目的机会。

通过 NFTI 获得对蓝筹 NFT 的投资机会

图片来自 Scalara

你需要知道的是：

  • 在 2022 年 2 月，Scalara（以前是 Pulse 公司）披露了 Scalara NFT 指数‌的方法，或称为 NFTI。在几天之内，有人利用该方法通过指数跟踪协议 Kuiper 推出了该指数。

  • NFTI 为顶级 NFT 藏品的地板价表现提供潜在回报。具体来说，该指数作为 NFT 流动性协议 NFTX 的 xToken 的自动加权篮子。此外，NFTX 的库存质押计划为 NFTI 持有人提供了与 NFT 价格表现无关的额外收益。

  • 最初，NFTI 跟踪 CryptoPunks、CrypToadz、Chromie Squiggles、Cool Cats、Doodles、Meebits 和 World of Women 系列。投资者现在可以使用单一的 ERC20，以任何预算获得这些蓝筹 NFT 项目的风险。相比之下，通过同时支付其全额底价来投资这些 NFT 集合，现在就相当于要掏出数百个 ETH。

  • 此前，绝大多数蓝筹 NFT 的流动性都只存在于以太坊上，所以 NFTI 在 Arbitrum 上的到来使得人们有史以来第一次享受到围绕一些最大的 NFT 项目的快速和廉价的二级交易。

如何访问 NFTI

  1. 如果你还没有给你的 Arbitrum 钱包充值 ETH，那么先给你的钱包充值 ETH 作为 gas 费。

  2. 然后前往 Uniswap V3 的 Arbitrrum 部署中的 ETH-NFTI 池‌。

  3. 使用“互换（swap）”用户界面，输入你想交易的 NFTI 的数量，然后用你的钱包完成交易。小心高滑点！

  4. 你已经得到了 NFTI，你可以在你看涨并想接触其收藏品的时候长期持有 NFTI，或者你可以在你准备好把钱拿出来的时候通过同一个 Uniswap V3 池卖出。

用 Yearn 的 L2 Curve Tricrypto 金库进行投资

请鼓掌。。。
Yearn 已经在 Arbitrum 上推出。
就在四个月前，Yearn 增加了对其第二个区块链的支持：Fantom。
今天，Yearn 增加了其第一个以太坊 L2：Arbitrum。

你需要知道的是：

  • 在 2022 年 2 月，领先的 DeFi 收益聚合器 Yearn 背后的团队宣布该协议部署到 Arbitrum。值得注意的是，这次部署标志着 Yearn 支持的第一个 L2。

  • Yearn 团队决定首先试用 Arbitrum，因为该 L2 的成本低，其令人信心十足的 TVL 统计数据，以及它与通过币安和 FTX 等主要加密货币交易所直接充值/提现的兼容性。

  • Yearn 在 Arbitrum 的首发产品是 Curve Tricrypto 金库。Curve 的 Tricrypto 资金池包含 WBTC、WETH 和 USDT，该资金池的储户可以将他们的流动性提供者（LP）代币带到 Yearn 的金库，通过自动的、以 Curve 为中心的 DeFi 策略提高他们的收益。

如何进入 Curve 的 L2 Tricrypto 金库

  1. 在 Arbitrum 上向 Curve 的 Tricrypto 池‌充值。

  2. 带着你的 LP 代币，回到 Curve Tricrypto 金库‌页面。

  3. 使用“充值（Deposit）”用户界面，输入你想提供的 crvTricrypto LP 代币的数量。

  4. 完成审批，然后进行充值交易，你已经完成了。现在，金库正在产生大约 7.4% 的年利率。

在 Stargate Finance 上提供流动性

图片来自 Stargate Finance

你需要知道的是：

  • Stargate Finance 是一个“omnichain’”桥协议，它是第一个在 Layer Zero ‌上推出的项目，Layer Zero 是一个互操作性的解决方案，可以使不同的区块链很容易直接相互沟通。

  • Stargate Finance 目前支持一系列跨越多个区块链的稳定币池，与以太坊虚拟机（EVM）兼容。这些链包括 Arbitrum、Avalanche、BSC，以及其他区块链。

  • 通过向 Stargate Finance 的流动性池添加流动性，用户可以从通过 Stargate Finance 进行的每一笔转账中获得一小部分奖励。此外，还可以将 Stargate FinanceLP 代币进行质押，以赚取以 STG 计价的进一步奖励，STG 是该协议的原始代币。

如何获得 Stargate Finance 的 Arbitrrum 质押的机会

  1. 在你的 Arbitrum 钱包中加入一些 USDC 或 USDT，用于质押。

  2. 然后前往 Stargate Finance Pool‌ 页面，连接你的钱包。

  3. 点击 Arbitrum USDC 资金池或 Arbitrum USDT 资金池选项，存入你想要的金额。

  4. 用你的新 LP 代币，导航到 Farming‌ 页面，点击适用于你的质押机会。输入你想质押的 LP 代币数量，然后完成质押交易。

  5. 你已经完成了！这就是如何加入 Arbitrum 上 Stargate Finance 挖矿的过程。注意 USDC 和 USDT 矿田目前分别提供大约 10% 和 9% 的年利率。

在 GMX 上进行质押，按目前利率赚取高达 47% 的年利率

图片来自 GMX

你需要知道的是：

  • GMX 是一个去中心化的现货和永续交易所，目前在 Arbitrum 和 AVAX 上线。该协议的核心是一个多资产池，通过它，流动性提供者可以通过做市商和支持杠杆交易获得收入。

  • 同名的 GMX 代币是该协议的本地管理和实用代币。持有者可以将 GMX 进行质押，随着时间的推移，通过其他用户的交易费用赚取托管的 GMX（esGMX）奖励和 ETH 收益。

  • GMX 的另一个中心代币 GLP 是该协议的原生流动性提供者代币。简单地说，GLP 是一个资产指数，横跨 GMX 的基础多资产池中存在的代币，而且可以用 GLP 质押来赚取 ETH 和 esGMX 奖励。

如何获得 GMX 或 GLP 进行质押

  1. 进入 GMX 购买页面‌，按照说明获得你想要的 GMX 或 GLP 的数量。

  2. 如果你购买 GLP，你已经完成所有操作了，因为你的 GLP 在购买时自动进行了质押。然后你可以通过 GMX 收益页面‌跟踪你的收益或解除你的仓位。请注意，GLP 质押现在产生 47% 的年利率（27% 的 ETH + 20% 的 esGMX）。

  3. 对于 GMX 的质押者来说，请进入 GMX Earn 页面并连接你的钱包。

  4. 然后向下滚动到 GMX 质押的用户界面，点击“质押（staking）”，用你的钱包完成交易。请记住，GMX 质押目前的年利率为 28%（ETH 的 11% + esGMX 的 17%）。

通过 Vesta 流动性挖矿赚取高达 60% 的年利率

图片来自 Vesta

你需要知道的是：

  • 与 Liquity ‌类似，Vesta 是一个“零利息”借贷协议。

  • 然而，虽然 Liquity 存在于以太坊上，并且只支持 ETH 抵押品，但 Vesta 是建立在 Arbitrum 上的，并支持多种抵押品类型，如 ETH、renBTC、gOHM 和 GMX。

  • 该协议的原生美元挂钩稳定币是 VST，而项目的原生治理代币是 VSTA。为了引导 VSTA 的流动性，Vesta 团队目前正在运行一个 Arbitrum 上以 Balancer 的 ETH-VSTA 池为中心的流动性挖矿项目。因此，现在用户可以将他们池子的 LP 代币质押到 Vesta 上，以获得 42% 的 VSTA 奖励 + 17% 的 BAL 年化奖励。

如何获得 Vesta 的流动性挖矿

  1. 向 Balancer 上的 VSTA-ETH 池‌提供流动性。

  2. 然后进入 Vesta 的流动性挖矿‌流动性挖矿‌页面，点击 VSTA-ETH 选项，会出现上图中描述的用户界面。

  3. 输入你想要的 VSTA-ETH LP 代币数量，然后点击“质押（Stake）”，用你的钱包完成交易。请记住，你的累积质押奖励可能会被相关的 VSTA-ETH 池的无常损失所抵消。

当然，这些项目都伴随着风险，尽管它们是快速和廉价的 DeFi 和 NFT 活动时代已经来临的最好证明。
