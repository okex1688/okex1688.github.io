简介—平行链已上线

2021年12月17日，在成立五年后，Polkadot成功地完成了其多阶段的启动，拉开了平行链的序幕。现在平行链已经上线，终端用户可以在Polkadot上进行交易。

什么是平行链？平行链是使用Polkadot的区块链开发框架Substrate开发的针对具体应用、高度定制的Layer1。与传统的Layer1类似，它有自己的经济、token、治理和用户。然而，与传统的Layer-1不同，平行链不是孤立的。相反，它与Polkadot的另一个主要技术组件中继链相连。

什么是中继链？中继链是Polkadot的核心。它是容纳所有验证器的Layer-0基础链，负责保障、治理和连接平行链。除了中继链的管理外，平行链也有主权治理权限，这将在下面的类比中进一步解释。中继链可以支持大约100到250个平行链。平行链的上限主要是由扩展Polkadot的跨链消息传递（XCMP），以及维持最佳的10:1验证人与平行链的比例以确保去中心化和表现的两者成本决定。

Polkadot token（DOT）有什么作用？Polkadot的token DOT是一种公用事业和治理token，用于参与治理、交易费用、质押的安全性，以及为平行链卡槽提供保证金。截至2022年3月，DOT是加密货币基金中最常持有的资产。

除了风险投资外，Polkadot的独特架构与Substrate的灵活性相结合，成功跻身为第二大开发者社区。现在，平行链已经上线，大量VC资金也得到了确认，开发者的劳动成果可以为大众所享用。在这份报告中，我们将深入研究Polkadot的前11个平行链。

生态系统概述—11条平行链

如上所述，中继链可以支持大约100到250个平行链。要想获得中继链提供的优势，它必须通过质押的DOT来赢得平行链插槽的蜡烛式拍卖。

潜在的平行链竞拍者通常通过他们的社区筹集DOT，以换取他们的本地token。在蜡烛拍卖的随机时间里，拥有最多质押DOT的平行链赢得卡槽拍卖。在租赁期结束时，贷款人的质押DOT被归还（他们将保留平行链token）。

经过11次拍卖，1.26亿DOT（占总供应量的11%；价值21亿美元）已经被质押。第一批中标者（拍卖1-5）于2021年12月17日登上了Polkadot。第二批中标者（拍卖6-11）于2022年3月11日入驻Polkadot。每位获奖者都为Polkadot的生态系统带来了独特的功能和用例。

Acala阿卡拉

Acala赢得了Polkadot的第一次拍卖，筹集了32,515,980 DOT。Acala是一个建立在Polkadot上的去中心化金融网络，它为aUSD稳定币生态系统提供动力。Acala拥有一套以aUSD为中心的金融功能，包括：

阿卡拉美元（aUSD）

 Acala Dollar（aUSD）是一种由Honzon协议产生的稳定币—一种受Maker DAO（DAI）启发的机制。用户通过使用经批准的储备资产（如DOT、LDOT和ACA）创建一个过度抵押的债务头寸来铸造aUSD。该团队计划将抵押品扩大到BTC、ETH和其他面值的原始token。Acala与最初的九个Polkadot 平行链团队和几个支持的风险基金一起，推出了一个2.5亿美元的aUSD生态系统基金。截至2022年3月8日，已经发行了617万枚aUSD。

流动性质押（L-Token）

Acala通过Homa协议实现的流动性抵押功能，用户质押他们的token以获取L-token（例如LDOT），它代表了质押的主要资产和不断累积的质押收益。L-Tokens可作为ERC-20和基于Substrate的token使用，也可在Acala的DeFi协议中使用。

Acala Swap

Acala Swap是一个恒定产品的AMM DEX，支持ERC-20和本地基于Substrate的token。Acala Swap正在主导一个引流活动以庆祝其启动，用户可以用任何token在Acala交换上支付gas费用。

Acalatoken - ACA

Acalatoken（ACA）是一种效用和治理token，用于参与治理、交易费用和激励Acala收集者节点。

此外，Acala还与一个社区运营的财政部门的金融科技独角兽Current建立了合作关系，同时Acala EVM+实现了EVM和Substrate之间的全栈可组合。截至2022年3月初，Acala拥有5.28亿美元的TVL。

Moonbeam

Moonbeam赢得了Polkadot的第二次拍卖，筹集了35,759,931 DOT。Moonbeam是一个兼容EVM的跨链智能合约平台，其允许开发者在极少甚至不修改代码的情况下迁移他们的项目，并为启动新项目提供一个开发者友好的环境。

以太坊式的统一账户

任何平行链都可以通过启用Parity和Moonbeam设计的frontier pallet来实现EVM兼容。然而，兼容EVM的平行链需要用户拥有EVM和Substrate账户的私钥。Moonbeam提供统一账户–用户只需要一个私钥（EVM式密钥）就可以与EVM和Substrate互动。统一账户功能允许用户在Moonbeam上使用其现有账户，并使用Metamask钱包。

XC-20资产

Moonbeam推出了XC-20token类别，这是一个新的token类别，提供基于Substrate资产的原生互操作性，同时允许用户和开发人员通过熟悉的ERC-20接口进行交互。

Moonbeam token - GLMR

Moonbeam token（GLMR）是一种效用和治理token，用于参与治理、交易费用（80%烧掉，20%分配给链上国库），激励Moonbeam收集者节点，并支持智能合约执行的gas计量。

截至2022年3月初，Moonbeam拥有1.41亿美元的TVL，生态系统不断壮大。

Astar Network

Astar Network赢得了Polkadot的第三次拍卖，筹集了10,333,552 DOT。Astar是一个支持WASM和EVM虚拟机的多链智能合约平台，为开发者构建智能合约提供多种解决方案。

跨虚拟机(X-VM)

到目前为止，虚拟机通常是独立断开并且很难交互的，例如在EVM合约中调用Substrate合约。为了统一各种智能合约引擎，Astar正在开发跨虚拟机（X-VM），X-VM将允许智能合约从不同的虚拟机和语言中执行调用和读取存储数据。

ZK解决方案

Astar公司一直在开发ZK的扩展解决方案，在不久的将来，Astar将实现 ZK Plonk pallet （该plonk被称为通用ZK Snark）。

DApp质押

 在构建应用程序时，开发人员往往面临着大量的支出费用而没有相抵消的收入。DApp 质押是对智能合约的开发人员或管理员的奖励机制。Astar的质押奖励是根据动态的奖励模式发给应用开发者的，最终，Astar将把50%的质押奖励分配给开发者。Astar还计划推出功能，允许开发者直接从支持者获取项目提供资金。

Astar token - ASTR

Astartoken（ASTR）是一种实用和治理token，用于参与治理、交易费用、激励Astar收集者节点、DApp定投和Layer2应用支持。

此外，Astar公司通过TVL ArthSwap支持Polkadot公司最大的DEX，近日在Polychain的领导下筹集了2200万美元的资金，兼具有两个以太坊桥接器。截至2022年3月初，Astar有3.86亿美元的TVL。Astar计划与Cosmos连接，并将增加大量的额外功能。

Parallel Finance

Parallel Finance赢得了Polkadot的第四次拍卖，筹集了10,751,519 DOT（79%通过Parallel拍卖贷款平台筹集）。Parallel是一个拥有完整功能的DeFi平台。

货币市场

Parallel的去中心化货币市场协议可提供借贷和抵押功能。选择质押的用户可以参与杠杆质押，杠杆质押允许用户同时出借和质押，从而获得双倍收益。如何实现？xDOT可以借回给Parallel，以便在目前的质押收益之外获得借贷费用。用户可以进一步选择用xDOT进行借贷以增加杠杆。Parallel Finance 平台通过算法选择性能最高的节点，从而最大化获取质押收益。

拍卖贷款平台

 拍卖贷款平台为潜在的平行链筹集DOT，贡献者被激励通过拍卖贷款平台借出他们的DOT，因为他们可从 Parallel 和潜在的平行链获得额外的奖金，并通过流动的质押衍生品cDOT保持流动性。

Parallel token - PARA

Parallel token（PARA）是一种实用和治理token，用于参与治理、交易费用、激励平行整理器节点，并在安全模块中被抵押。

此外，Parallel计划发布一个具有Stable Swap功能的自动化做市商AMM，一个多链钱包，以及系列的附加功能。截至2022年3月初，Parallels拍卖贷款平台已经筹集了5.53亿美元，占有22%的市场份额。

Clover Finance

Clover Finance赢得了Polkadot的第五次拍卖，筹集了9,752,487 DOT。Clover是一个兼容EVM的跨链基础设施平台，为开发者提供迁移和扩展其应用程序的工具。

通用跨链

Clover兼容各种钱包用于与Polkadot和Kusama上基于Substrate的链、EVM链和Solana上的应用互动。用户可以在不同的网络中发送、接收、包装和解除跨链资产，而无需在它们之间进行导航（类似于Zapper）。

Clover钱包

Clover Wallet是一个非托管的跨链钱包，可帮助用户在Polkadot和Kusama的基于Substrate的链、EVM链和Solana上管理、跟踪和交互资产。Clover钱包可在移动、扩展和网页端使用。

Clover跨链资源管理器

Clover Cross-Chain Explorer是一个跨链区块探索器，提供跨Polkadot和Kusama上基于Substrate的链、EVM链和Solana的多链索引，用户可以通过区块、交易和账户进行搜索。

Clover token - CLV

Clover token（CLV）是一种实用和治理token，用于参与治理、交易费用、激励四叶草整理者节点和财政管理。

此外，Clover计划发布跨链桥、云服务API、gas费用再分配（与开发者分享一定比例的交易费用）、低gas终端用户体验，以及基于身份和声誉的费用，为有信誉的用户提供优惠的价格。

Efinity

Efinity赢得了Polkadot的第六次拍卖，筹集了7,695,377 DOT。Efinity是由Enjin团队开发的专门用于NFTs的区块链，旨在支持任何区块链上的NFTs。

token标准

Efinity正在为可互操作的、跨链的token引入一个新标准。它将与基于底层的链Polkadot和Kusama、ERC-20、ERC-1155、ERC-721和其他链及其标准兼容。该token标准支持同质化token、非同质化token和分组NFTs。

Fuel Tanks 

Fuel Tanks 用于交易费用，开发者将EFI存入他们的Fuel Tanks以补贴其用户/玩家的成本。

离散账户（Discrete Accounts）

离散账户通过不要求新用户开设加密货币钱包来实现用户的简易入局，链上物品可以放在开发者控制的钱包里，玩家可以在以后的时间里索取。

NFT.io市场

Efinity和Enjin正在开发一个名为NFT.io的NFT市场，预计将在3月进行封闭测试，NFT.io是一个跨链NFT市场，通过Efinity提供新的价格发现机制。

EFI

Efinity token（EFI）是一种实用和治理token，用于参与治理、交易费用、促进流动性、奖励网络参与者、资助Fuel Tanks，以及增加Enjin的JumpNet EVM链上的交易限额。

此外，Efinity将推出新颖的用户友好功能，并提升Enjin的现有基础设施的利用度。超过100个项目已经承诺在Efinity和Enjin上建立应用程序和游戏。

Composable Finance

Composable Finance赢得了Polkadot的第七次拍卖，筹集了6,075,487 DOT。Composable是一个为跨链互操作性设计的DeFi基础设施平台，其正在建立一个全面的技术栈。

跨链虚拟机(X-VM)

跨链虚拟机（XCVM）是虚拟机，可为开发者提供运行跨链智能合约功能的环境。XCVM通过其创新可用层将Substrate连接到其他网络，使用路由层确定指令的最佳路线。

Mosaic

Mosaic是一个跨链转移可用层，与EVM兼容的链集成，包括Ethereum、Arbitrum、Avalanche、Polygon、Fantom、Moonriver等。Mosaic使用多种保证来确保转账的交付，包括流动性预测、流动性金库的再平衡和社区运行的流动性机器人，开发人员可以通过使用Mosaic SDK来构建Mosaic。

Mural

Mural促进了利用Mosaic的NFTs的跨链转移。

Composable  token - LAYR

Composable token（LAYR）是一种实用和治理token，用于参与治理、交易费用和激励Composable收集者节点。

Centrifug

Centrifuge赢得了Polkadot的第八次拍卖，筹集了5,435,161 DOT。Centrifuge为DeFi带来真实世界的资产融资，以降低小企业的资金成本，并为DeFi投资者提供与加密货币市场无关的稳定收益来源。

点对点网络

Centrifuge的点对点网络提供了一种安全的方法来创建、交换和验证合作者之间的资产数据，并将资产token化为NFTs。资产发起人可以有选择地与服务提供商分享资产细节，而服务提供商可以评估数据并为铸造的NFT贡献信息，数据来源可以使用加密签名进行验证。

Tinlake

Tinlake是一个充当现实世界资产的公开市场应用。企业将其金融资产token化为NFTs，并将NFTs作为其Tinlake池的抵押品。用户可以访问所有的池子，并将DAI投资到他们选择的池子中，每个池子包含两个风险级别，具有不同的风险回报水平。

Centrifug token - CFG

Centrifuge token（CFG）是一种实用和治理token，用于参与治理、交易费用、激励Centrifuge收集者节点，并奖励网络参与者。

此外，Centrifuge利用现有的法律和商业框架为现实世界的资产进行安全融资。截至2022年3月初，Centrifuge有7600万美元的TVL。

HydraDX

HydraDX赢得了Polkadot的第九次拍卖，筹集了2,462,543 DOT。HydraDX是一个跨链流动性协议，通过将许多加密货币资产纳入单一的交易池来减少流动性的分散。

全能池（Omnipool）

HydraDX Omnipool是一个服务于所有资产的流动性池。Omnipool中的所有交易都以LERNA（LRNA）计价，这意味着LRNA允许资产相互定价。池中可用的LRNA数量与总的流动性相关而波动–当流动性增加时，LRNA被铸造，而当流动性被移除时，LRNA被烧毁。由于Omnipool铸造LRNA以匹配每一个增加的流动性，LPs能够提供他们想要的任何资产的流动性，并在这些资产从池中购买时产生交易费用。

HydraDXtoken - HDX

HydraDXtoken（HDX）是一种实用和治理token，用于参与治理和稳定LRNA价值。

Interlay

Interlay赢得了Polkadot的第十次拍卖，筹集了2,751,954 DOT，Interlay可将比特币交互至任何区块链上。

InterBTC

InterBTC是一个完全抵押的、可相互操作的BTC挂钩衍生品。为了铸造interBTC，用户将他们的BTC锁定在一个保险库中。保险库可是现有的，也可是BTC所有者自己创建的。所有的金库用特定的资产进行超额抵押，若金库内发生恶意活动，超额抵押的资产将起到保险作用。若恶意活动实际发生，BTC储户将以溢价的方式获得被砍掉的抵押品的补偿。第一批金库将以DOT作为抵押。

InterBTC与竞争者

InterBTC是一个去中心化和无信任化的BTC挂钩衍生品。如何实现？任何人都可以成为InterBTC的金库（去中心化），金库不能阻止用户铸造InterBTC（抗审查），金库将抵押品锁定在不同的资产中（资本效率高），如果BTC丢失或被盗，用户可以用抵押品以有利的利率偿还（无信任）。

Interlay token - INTR

Interlay token（INTR）是一种实用和治理token，用于参与治理、交易费用和激励Interlay收集者节点。该团队计划将INTR集成到interBTC中以实现高级功能、更好的费率和更高的保险范围。

此外，Interlay计划增加新的金库抵押品，并与其他区块链包括Ethereum、Cosmos和Solana连接。Interlay的目标是成为一个去中心化的比特币银行，使金库能够重新使用将托管的BTC和存入的抵押品用于结构化金融产品。

Nodle

Nodle以2,475,528 DOT的筹款额赢得了Polkadot的第11次拍卖。Nodle是一个分散的无线网络，为物联网设备提供安全、低成本的连接和数据流动性。Nodle网络通过数以百万计的智能手机和路由器利用蓝牙低能量（BLE），使企业和智能城市能够以低成本将物联网设备连接到互联网，并保持隐私和安全性。

Nodle Cash App

Nodle Cash App是一个移动应用程序，使用户能够连接到被称为公民网络的Nodle网络。一旦下载该应用程序，用户必须在手机设置中打开蓝牙和定位服务。Nodle Cash应用程序将检测传感器和智能设备，并匿名连接到它们。对于所提供的服务，应用程序用户会收到Nodle Cash，也被称为NODL令牌。

Nodle token - NODL

Nodle token（NODL），也被称为Nodle Cash，是一种实用和治理token，用于参与治理、交易费用、奖励Nodle网络的参与者，以及激励Nodle收集者节点。NODL每六秒向Nodle网络的参与者分发一次。

此外，Nodle网络拥有500万部每日活跃的智能手机和3000万台物联网设备，每天在100多个国家移动约100GB的数据。
