据媒体报道，虎年新春上海海昌海洋公园以“元宇宙”为主题，线上线下联动全球首发数字盲盒“虎鲸骑士团海洋NFT数字盲盒”，力求带给游客惊艳与完美，发售NFT数字盲盒的不仅仅有上海海昌海洋公园。

NFT数字盲盒之所以广受追捧，原因在于基于区块链技术可实现确权、不可篡改和可追溯等特征，用户可收藏、展示、转售，满足人们的观赏、收集、社交需求，成为不可多得的“数字珍藏品”。NFT技术与盲盒模式的结合，既丰富了NFT产品使用场景，又增强了盲盒活动的技术含量，可谓相得益彰，但其中确实也存在一些问题，我们从盲盒经营活动的特征加以分析。

盲盒经营活动及其特征

根据《上海市盲盒经营活动合规指引》（简称《合规指引》）的定义，盲盒经营活动是指经营者在合法经营范围内，在事先不告知商品具体型号、款式或服务内容、提供方式的情况下，通过互联网、实体店、自动贩卖机等形式，以消费者随机抽取的方式销售特定范围内商品（包括服务）的经营模式。从以上定义来看，盲盒经营活动有以下几个特征。

一是在合法经营范围内。 盲盒经营活动必须是在合法经营的场景中的一种营销方式，是基于正常主营业务基础上开展的增强服务体验和促进消费的手段。

二是经营者以多种形式提供该类商品或服务。 包括互联网、实体店、自动贩卖机等线上、线下或线上线下相结合的形式。

三是消费者以一定方式参与。 有时候也以赠送的形式，但是这种赠与同样也是基于消费者的参与行为基础上。

四是消费者所获得的商品具有一定的随机性。 消费者在获得之前一般并不知道其具体型号、款式或具体服务内容，当然经营者一般会告知其大概情形。

盲盒经营活动基本要求及其对消费者权益的影响

盲盒经营活动是一种新兴的营销方式。我们结合盲盒经营活动的具体环节来分析其基本要求及其对消费者权益产生影响。

（一）关于盲盒相关商品或服务

盲盒销售需要履行正常商品销售的相关要求，包括以下几个方面。

盲盒需要遵守商品销售基本要求， 如盲盒经营者应按照法律法规规定，通过产品标签、消费提示等方式充分告知消费者盲盒商品的产地、成分、生产日期、保质期、使用存储条件、安全警示、“三包”条款等关系商品质量的基本必要信息，标注的信息应当真实、准确。

盲盒应保证产品质量。 盲盒经营者应建立健全企业质量保障体系，加强生产、仓储、物流等环节的管理，保证商品来源可靠、质量合格。不得利用盲盒销售“三无”产品、伪劣产品。

盲盒需要遵守相关认证规定。 盲盒商品属于强制性认证范围的，应当取得强制性认证，并标注“CCC”标识。

盲盒需要遵守相关标准。 盲盒商品没有国家标准、行业标准和地方标准的，盲盒经营者应制定相应的企业标准，对供应渠道、原材料、设计安全性能等方面严格把关，降低使用安全风险隐患。

互联网销售盲盒可不适用七日无理由退货，但全包形式除外。 盲盒经营者通过充分告知提示，并经消费者确认后，以互联网形式销售的盲盒商品可以不适用七日无理由退货。但以全包形式销售整套系列商品的，其标的物清楚确定，并非盲盒，经营者应依法执行七日无理由退货规定。

商品质量主体责任制。 盲盒经营者应当依法承担商品质量主体责任，履行退换货、修理等经营者义务，不得推诿、拖延、拒绝处理消费者投诉。

建立健全投诉处理机制。 盲盒经营者应建立便捷、有效的投诉处理机制，公开投诉方式、处理流程、退换货标准等信息，加强与监管部门、消费者权益保护委员会的沟通协作，提高消费争议解决效率。

另外需要强调的是，盲盒商品不能是法律法规明确规定禁止销售、流通的商品，以及对销售资质、存储运输、使用条件等方面有严格要求的特殊食品、药品、医疗器械、有毒有害物品、易燃易爆物品，包括活体动物；其价格应当与其实际正常售卖的价格相当。《合规指引》建议绝对价值不超过200元；不得侵害他人商标、专利等知识产权以及其他合法权益。

（二）不得以非法经营方式销售盲盒

盲盒经营 不得从事金融类金融活动 。 盲盒经营者不得违法利用盲盒开展金融经营活动和类金融经营活动；不得违法开展二手盲盒经营活动，诱导非玩家用户加入投机炒作，故意造成二手市场价格剧烈波动，即不得主动二级市场炒作，扰乱正常交易秩序。

在信息 内容管理 方面遵守相关规定。 盲盒商品不得含有歪曲历史、封建迷信、淫秽色情、恐怖暴力等法律法规禁止以及违背公序良俗的内容。

不得从事博彩 、彩票等经营活动。 盲盒经营者不得以盲盒名义从事赌博、彩票销售或变相开展赌博、彩票销售。

不得 以欺诈操纵方式 天价炒作，不得过度营销，不得饥饿营销 。 盲盒经营者不得传播虚假市场信息进行天价炒作；不应过度营销助长消费者炫耀攀比心理；不应开展饥饿营销故意调低特定商品的抽取概率。

（三）关于销售方式

经营者要落实相关制度。一是经营者应保留活动相关规则和实施情况的记录备查。 盲盒经营者应保留抽取概率设定、结果抽取的完整记录，并以此为依据进行实际的市场投放和向消费者发放盲盒。经营者应建立追踪记录制度，确保消费者所抽取的商品发放到位，并自觉接受监督。相关记录留存时间一般不少于3年。二是针对二级市场应落实相关责任和制度。 网络交易平台、二手交易平台应落实企业主体责任，建立健全相关制度，依法配合政府部门加强对盲盒经营活动的监管。三是针对未成年人保护的特别规定。 盲盒经营者不得向8周岁以下未成年人销售盲盒。向8周岁以上未成年人销售盲盒商品，应通过销售现场询问或网络身份识别等方式，确认已取得相关监护人的同意。

盲盒经营者在销售中要保障消费者知情权。 盲盒经营者应将商品种类、抽盒规则、商品分布、商品投放数量、隐藏款抽取概率、商品价值范围以及每个系列可以抽取的次数、金额限制等关键信息以显著方式对外公示，保证消费者在购买前知晓。

经营者不得欺诈或操纵方式侵害消费者权益。 盲盒经营者不得通过后台操纵改变抽取结果、随意调整抽取概率等方式变相诱导消费。不得以折现、回购、换购等方式拒绝或故意拖延发放盲盒。不得设置空盒。

NFT盲盒经营有关问题

盲盒经营最重要的问题可能是其规则公开透明以保障消费者知情权等权益上。从逻辑上说，基于区块链技术基础上的NFT技术应用在这方面有天然的优势。由于NFT盲盒的销售规则和过程基于智能合约以保证公开透明，当满足智能合约所确定的相应开奖条件后，只要消费者点击，开奖结果便已经生成，无法篡改。从这个角度来说，盲盒结合NFT技术应用可以更好地保障消费者权益，让经营活动接受有关机关和社会的监督。但现实情况并不总是如此，NFT盲盒经营中的问题除了正常的商品品质等类似情况外，其他归集起来其主要包括信息不透明导致的欺诈误导、市场操作和类金融操作等问题。

（一）NFT盲盒经营中的欺诈问题

某些NFT盲盒的具体操作中，NFT协议其有些关键环节并未完全做到去中心化、不可篡改。正如Signal 创始人在《我的 Web3 初体验》中提到的，现在很多NFT产品不是在链上存储数据，而是包含一个指向数据的URL，而位于URL的数据并无哈希承诺，因此NFT的URL所指向的可能并非是某一个特定内容，可能只是指向某个服务器，而在这个服务器是处于中心化机构的控制之下，理论上其可以修改该NFT指向的具体内容。无独有偶，据媒体报道NFTSCAN 联合创始人拾二也认为，“如果 Token URI 的 Jason 是存储在中心化服务器，项目方可以随便改，想什么时候改，就什么时候改。如果 token URI 的 Jason 是存储在 IPFS 的，那就不能改了。”。

对于一些 NFT盲盒发行者而言，由于缺少审计监督，前期可操作空间较大。在这种情况下有可能出现的情况是，开盲盒的游戏从一开始用户处于信息不对称状态，其权益保障可想而知。如果NFT数字盲盒经营者发行时有这种行为，则属于欺诈经营应当承担法律责任。

（二）NFT盲盒经营中的价格操纵问题

目前市场上一些NFT产品存在一些随意定价和炒作问题，屡屡曝出的天价交易很可能只是背后操作者的左右手，其性质上是一种市场操纵。NFT产品结合盲盒玩法以后可能更加让人眼花缭乱，导致其中的市场操纵与炒作得以掩盖。如某NFT产品发售时，先给出高价，然后把交易人为分为若干次限量发售，最后一次则以相对低价以盲盒形式出售。这种做法，很难不让人联想，其前几次操作属于饥饿营销，只为了在最后一轮拍卖中让消费者买单。

（三）NFT盲盒经营中的类金融操作

《合规指引》提到，盲盒经营不得从事金融类金融活动。盲盒经营者不得违法利用盲盒开展金融经营活动和类金融经营活动；不得违法开展二手盲盒经营活动，诱导非玩家用户加入投机炒作，故意造成二手市场价格剧烈波动，即不得主动二手交易炒作，扰乱正常交易秩序。

有些NFT盲盒发行方将NFT产品中赋予其股权权益，拥有NFT产品的人可享有股权收益或相应分红；有些NFT盲盒发行方将某些特定的资产收权益与NFT产品结合在一起，比如与若干特定作品版权收益权结合在一起，本质上是把NFT 产品当成了类似证券化产品；有些NFT盲盒发行方要求用户基于NFT产品享有某个特定投资产品的收益，其本质是将其当成了基金或其他相关的理财产品；还有一些NFT盲盒发行方经营二手交易，甚至客观上形成了交易所经营。这些类金融操作一个核心问题是，其本质无法解决相应收益权的中心化经营、管理和分配问题，因此这类活动必须根据有关金融规定加以监管，因此参与这些活动可能存在极大的法律风险。
