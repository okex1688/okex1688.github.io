中本聪在2008年写下的比特币白皮书，是web3先驱意识的觉醒，呼唤着人们去追求一个更加自由、高效、真实的去中心化世界。在web3时代，数据将被永久地去中心化储存在多个节点，不可篡改也无法窃取，人们可以自由地在平台表达观点，隐私得到安全保护。虚拟经济崛起且可持续，人们可以有另一个数字身份在虚拟空间里工作、学习、社交，超越时空的阻碍，满足精神和物质需求。基于区块链智能合约的DAO会渐渐成为主流，组织内的成员众包、众筹、众智，个人利益与组织利益达到了高度一致，极大地提升组织的运行效率。

区块链、游戏、VR和5G网络这些基础设施的进步给web3的到来提供了基础和保障。未来已来，让我们一起看看有哪些项目正在为web3的美好时代夯实基础。

存储和数据基础设施

随着5G、物联网、大数据、区块链等新技术的发展，我们迎来了数字经济时代，随之而来的是是海量数据的的产生和存储需求，云存储市场蓬勃发展，预计2025年会将到1,373亿美元。但当今以亚马逊、阿里为代表的主流中心化存储方案并不能给数据的生产者带来收益，还时不时出现严重的隐私泄露问题，甚至被单点攻击造成数据永久性丢失。将数据去中心存储以增加安全性，以及将数据的使用权、收益权归还给用户的呼声越来越大。当去中心化存储和区块链相结合，在技术创新的基础上配套了共识激励机制，星际文件系统为代表的新一代去中心化存储掀起新的革命浪潮。

Filecoin

基于区块链的去中心化数据存储协议。为了解决了数据的存储问题，利用闲置的带宽与硬盘空间提供数据存储和检索服务，给存储提供商提供了获利的机会，还降低了用户数据的存储和使用成本。Filecoin具有自我修复过程，当个别节点出现问题时，可将其存储的数据转移到其他可靠的节点，提高了数据的安全性。作为IPFS的激励层，目前在存储赛道的市值排名在第一位。

Internet Computer

将公共网络重塑为区块链计算机，共识算法高速可扩展，能够与任何防御女巫攻击的共识机制结合，为开发人员提供强大的软件框架，可以轻松构部署网站、Dapp、DeFi等，使其实现代币化。9月份通过了为比特币提供智能合约的提案，将采用链式密钥密码术，使比特币停留在比特网络的情况下，通过智能合约直接传输、接收比特币，确保交易的真实性，来消除对存在安全隐患的封装与桥的需要。

Arweave

NFT/web3项目采用的首选存储方式之一。建立在blockweave技术的去中心化存储网络，采用创新的Proof of Access共识机制，调节节点保存的数据量，达到数据的均匀分布。用户一次性消费即可永久储存数据。

Crust Network

波卡上面的Filecoin，致力于建立保障所有权和隐私权的去中心化云生态系统。支持包括IPFS的多种存储层协议，且为应用层提供了接口支持。采用有担保的权益证明共识和有意义的工作量证明，技术栈支持去中心化计算层。

Theta

老牌的分布式流媒体协议。使用点对点网络，任何人都可以通过空闲的带宽和计算把视频转发给其他用户，获得代币奖励。视频转播者的边缘节点相对中心化CDN服务器，更靠近用户，降低运营成本。

Livepeer

去中心化、高度可扩展的流媒体层协议，内容创作者分享实时直播，通过视频转码获得代币奖励，可作为web3堆栈中的实时媒体层。

LINK

老牌预言机，允许用户将智能合约连接到不同的数据源，实现链上数据和链下数据的准确传输，且不可篡改，使智能合约在金融、供应链等行业得以应用。

The Graph

以太坊和IPFS等网络的数据索引查询。开发人员可以创建被称为子图的开放API，定义访问、提供区块链数据的可验证方式，使数据查询更高效。

Polkadot

Polkadot 是一种开源分片多链协议，可促进任何数据或资产类型的跨链传输，从而使各种区块链可以相互操作。这种互操作性旨在建立一个完全分散的私有网络，由其用户控制，并简化新应用程序、机构和服务的创建。Polkadot 协议连接公有链、私有链、免许可网络、预言机和未来技术，允许这些独立的区块链通过 Polkadot 中继链无信任地共享信息和交易。

Polkadot 有四个核心组件：中继链、平行链、平行线程和桥。中继链是Polkadot 的“心脏”，帮助在不同链的网络上建立共识、互操作性和共享安全性。平行链是可以拥有自己的代币并针对特定用例进行优化的独立链。平行线程类似于平行链，但具有经济的即用即付模型的灵活连接。桥允许平行链和平行线程与以太坊等外部区块链连接和通信。

另外，Polkadot创始人是区块链web3概念的提出者，Polkadot下面有web3基金会。

隐私

web3精神强调的一个重点就是注重保护个人隐私，用户不再希望通过隐私来获取免费的服务。比特币、以太坊网络是公开透明的，上面的每一笔转账记录都可以看得一清二楚，黑客可以通过链上交易习惯推测出背后地址背后的主人。暴露个人钱包地址相当于暴露了个人银行卡里的金额，人们开始将目光投向可以保护隐私的新公链。此外，隐私公链在供应链金融、博彩行业、匿名投票、匿名招标、NFT等应用场景也大有可为。

Oasis Network

伯克利华裔教授的Oasis lab孵化的第一个支持开放金融和可靠的数据经济的隐私公链，让用户控制、安全共享生成的数据。区块链底层分开，共识层处理简单的交易，Paratime执行层处理复杂的工作量。Paratime层是去中心化的，用户可以建立自己独立的ParaTime来满足各自的应用。网络精密的差异检测使 Oasis 用更小的复制因子就达到了和分片、平行链相同的安全级别。

Secret Protocol

生态最丰富的隐私公链。Secret上的节点不需要接触原始数据，在可信执行环境对加密后的数据执行通用计算。这使其智能合约能够使用私密和敏感数据作为输入。Secret Network的重点是隐私计算，而不仅仅是交易隐私，这对去中心化金融、Web3、机器学习、访问控制等许多领域都至关重要。Secret还与opensea合作推出第一个包含公共和私有内容的隐私NFT，私有内容只对NFT持有人可见。

PlatON

万向区块链孵化的隐私和智能计算公链。“计算互操作性”是其核心特征。PlatON 通过构建由可验证计算、安全多方计算、零知识证明、同态加密等密码算法和区块链技术组装而成的计算系统，为人工智能、分布式应用程序开发人员、数据提供者和有计算需求的组织、个人提供公共基础设施。

NuCypher

NuCypher 是一种去中心化的加密、访问控制和密钥管理系统 ，使用户可以去中心化存储和在公共共识网络使用代理重加密技术（PRE）进行多人的隐私数据分享。

Mina Protocol

Mina 的网络的大小只有 22 KB（比特币的 300 GB），是最轻的“简洁区块链”，在使用量增长的情况下大小保持不变，减少计算需求，更有效地运行 DApp。此外，它在安全性和去中心化方面保持平衡。

游戏

链游是指区块链上运行的游戏，使玩家第一次成为游戏真正的主人。玩家在游戏中所拥有的的装备是可以自由买卖的NFT，不受游戏开发者的控制。其他游戏开发商可以针对玩家的NFT构建新的游戏。如果原先的游戏开发得不好，玩家可以将NFT拿到新的游戏里去玩。

链游要求玩家投入资本、时间和劳动力等成本，所获得的NFT在链游的经济模型里有相应的用途，所以是和其他工作一样有价值的。

链游帮助很多发展中国家的玩家在疫情期间获得高于当地经济水平的收入。链游虽然可能没有比传统游戏更好玩，但比传统工作有趣得多，所以就很容易被人们接受。经济发达国家的玩家出于娱乐的目的，承接了发展中国家玩家的装备，满足自身精神需求。

Axie Infinity  

是基于以太坊以及Ronin二层扩容侧链的数字宠物世界游戏。玩家可进行宠物对战、收藏、繁殖和土地家园建设。此外还开发了自己的去中心化交易所Katana和游戏链Ronin链。

Decentraland  

最早的去中心化元宇宙。用户可以创建、体验和发现内容，并与其他人进行互动。主要场景是用户可以通过基于区块链的土地账本宣称对虚拟领地的所有权，进行开发建设。

Sand

没有主线剧情和唯一结局的沙盒游戏，玩家可以在游戏中自由地创建游戏，拥有和售卖游戏的NFT，可玩性强。

Illuvium

基于Immutable X的RPG 和 NFT 收藏游戏，以科幻作为游戏背景，所有玩家都是一艘坠毁在外星的星际飞船上的幸存者，玩家组队与怪兽Illuvials交战，捕获的怪兽可进行升级、对战或者交易。

Alice

基于BSC的多人创作游戏，玩家可以购买和拥有虚拟土地，收藏和开发物品，结实新的朋友。这个游戏既为普通玩家提供了有趣的故事情节，又为想要收藏和交易NFTs的玩家提供了生态系统。此外，Alice向不同的网络节点支付费用，使玩家在没有加密货币的情形下也可以参与游戏。

游戏公链/侧链

从9月以来，区块链游戏的融资金额超过14亿美元。软银、红杉中国、罗斯柴尔德信托资本等传统资本纷纷入局。各种新链游项目层出不穷，风头无两。而此时为DeFi设计的以太坊Layer2 Rollup或者高性能L1公链在高峰期时有卡顿，甚至瘫痪，这明显难以应付满足大规模协作的扩容需求。在此背景下，专为链游而设计的游戏公链/侧链就显得尤为重要。

Ronin

Axie自己开发的以太坊侧链， 大幅降低了用户的使用成本，提升了操作的响应速度，满足大规模协作游戏的要求。近期推出了RON代币，未来有计划上线更多的游戏和其他产品。

Chromia

BSC上多项游戏使用的侧链,旨在解决DdApps的可扩展性问题。CHR的关系型数据库架构使其在存储和处理数据方面更有效率，实施储存虚拟世界的逻辑、信息以及玩家进度，使得DApp可以在去中心化服务器的情况下运作。作为游戏的后台支持，对DApp开发者采取灵活的收费模式，例如开发者支付CHR来购买存储和计算能力，让其玩家在不持有加密货币的情况下参与游戏。原生的NFT 标准Chromia Originals，集合了更高级的元数据，具有新的功能和更多的灵活性。

Flow

2017 年，Cryptokitties 引发了以太坊的大拥堵，Flow在这背景下应运而生，旨在为下一代DApp、游戏和NFT提供可高度扩展的区块链平台。生态系统丰富，包括Cryptokitties、华纳音乐、NBA Top Shot。

Immutable X

基于ZK Rollup技术的二层网络NFT可扩展交易平台，使玩家享受更快的游戏体验的时候无需支付Gas费。

Efinity

ENJ建立在波卡网络上可访问、可拓展的 NFT 跨链网络，交易费用被设计在后台，使用户可在链上自由铸造和交易各种标准的NFT。

应用

去中心化应用可以用任何语言编写，前端代码托管在去中心化存储上，能够调用在去中心化 点对点网络上运行的后端代码。用户界面友好，与web2应用的使用感觉没什么区别。但web3的应用强调去中心化，不受中心化个人或者团体控制，即使大比例参与者受到攻击而下线，网络仍可正常运作，不存在单点故障的问题。网络上的任何人都可以在无需许可的条件下有使用应用服务，拥有较大自由。Web2 应用提供服务来换取用户的个人数据，而web3注重用户的隐私权、专利权、数据所有权和自主选择权。由此可见web3的应用为一个更加民主化、自由化的社会提供了不可或缺的基础工具。

Ethereum Name Service

以太坊名称服务 (ENS) 是以太坊区块链上的去中心化域名系统。用户将将人类可读的域名， 例如 “Satoshi.eth” 映射到机器可读的标识符，例如数字货币地址、去中心化的内容哈希和元数据。相比DNS，ENS域由智能合约所有，采用去中心化自治管理。用户可将DNS域名（例如 Satoshi.com）映射到ENS, 以Satoshi.com 接收到加密货币的付款。

Basic Attention Token

隐私浏览器协议Braven的代币。Brave 浏览器的广告尊重用户的隐私，不会主动跟踪用户，除非用户自己选择加入。当用户浏览网页并观看广告时会获得得 BAT 奖励。如果内容创作者能够留住优质用户，也会获得 代币奖励。而广告商能够通过这种机制接触到愿意关注广告的目标客户，获得更好的广告投资回报。

Bright ID

web3社交身份网络, 为人类造福的公共产品。采用非侵入性、去中心化、开源的技术，允许人们向应用程序证明他们没有使用多个帐户。它通过创建和分析社交图来解决唯一身份问题。它提供的社交恢复功能可以保护让用户拥有、掌控自己的数字身份。

Radicle

去中心化的Github。它的点对点网络与可选的以太坊智能合约相结合，使开发人员能够在不依赖可信平台中介的情况下，在协议上就代码进行协作。

Litenry

跨链身份数据协议。去中心化身份认证和用户活动数据管理基础设施。基于Substrate 信用计算网络，为以用户为中心的区块链网络提供支持，包括一个基于身份的网络和相关工具，身份匹配和身份质押机制。

Ocean Protocol

区块链上的数据交易市场。发行人可以在保留隐私和拥有数据控制权的情况，将数据货币化。消费者现在可以购买、访问以前不可用或难以找到的数据集，对数据集进行操作。

Ethereum Push Notification Service

web3通知协议。通过使用该协议，DApp、智能合约以及服务商都可以以与平台无关的方式向用户的钱包地址发送重要的通知。用户拥有是否接收通知的自主权，若接收实际通知可以赚取代币奖励。

Rabbit Hole

颠覆了付费教学的模式，实现“边学边赚”。各种协议赞助任务，新用户可以完成这些任务可以获得奖励。协议的用户越多，协议就变得更有价值。协议为了获取新用户，需要让潜在用户先学习如何使用。通过奖励的方式，用户学习一项新技能，更新“链上简历”，协议获得了用户，实现双赢。目前还未发币。

Numbers protocol

由用户治理的去中心化图片网络，利用IPFS 存储数据，使每一张登记的图片或视频带有创作者签名、所有权证明、时空信息、完整的交易记录，使图片或视频可溯源、被验证，保护创作者的版权，提供了可验证真实性的平台, 使Web3的数字图像资产货币化成为可能。

社交

与去中心化金融、数字收藏品相似的，社交网络也是一个容易吸引、容纳大量用户群体的应用场景。

Web2时代的社交平台拥有较大的控制权，容易进行产品升级和商业变现。但是，随着平台的发展，这种中心化的运作模式越来越难以适应不同文化背景的用户和监管方，无法处理大量的误导性信息，屡屡出现的隐私泄露丑闻也让平台出现信任危机。此外，出于商业化利益的考量，平台的推荐算法倾向于讲用户的注意力转向易引发噱头的内容。

在这种背景下，人们开始思考在web3时代，社交网络应该由基于平台向基于开放、去中心化的协议转变，把内容决定权、数据控制权交还给用户，探索在新的社交场景下协调各方的激励机制。接下来，让我来看看有哪些可能改变人们社交方式的应用。

BitClout

去中心化的社交网络。重新搭建可自定义的区块链，架构和比特币相似，支持复杂的社交网络数据，例如帖子、个人资料、粉丝、投机行为等，并且具有更大的吞吐量和规模。创作者可以发布优质内容而获得代币奖励，使创作者和粉丝建立更紧密的关系。

Mask

社交金融协议。使用户能够在web2社交平台如Twitter和Facebook使用web3的功能，是web2和web3的桥梁。其具有的特色功能包括在Twitter和Facebook发布加密信息，允许用户直接从Twitter上对Gitcoin上的项目进行捐赠。Mask允许用户在不离开现有社交网络的情况下进行去中心化交易所交易、去中心化文件存储、去中心化电子商务等活动。

Argent

无需私钥的去中心化社交钱包。用户手机存有一个私钥，一个智能合约存用户的资金。如果用户丢失了手机，可以通过之前添加的信任的守护人来分配一个新手机作为资金的拥有方。

还可以设置每日的限额以及大额转账确认功能，媲美银行级别的安全性。内嵌的生物识别技术使钱包更加易用、安全。

RSS3

Web3时代的RSS, 去中心化内容和社交网络的开源协议。RSS3旨在促进信息传递的高效、自由，保护数据安全且去中心存储。该协议分布式托管索引文件和内容文件，让用户分布式控制自己的数据，通过密钥对 决定编辑文件的权利。RSS3强调包容性，不仅来支持不同类型的应用程序，还对不同功能的模块开放，这为其将来的发展提供了灵活性和可行性。

Poap

POAP是一种可靠地记录生活经历的新方式。每次参加活动时，POAP 收藏家都会获得一个由加密记录支持的独特数字收藏品NFT。POAP收藏允许人们低成本地创建可靠的去中心化身份，不依赖于典型的个人身份信息，例如姓名、性别、国籍、护照号码。

创作者经济平台

聚集大量用户的网络平台如Facebook、Youtube、微信、抖音在最近几年赚得盆满钵满。他们的成功离不开平台上的创作者。创作者发布优秀作品，吸引大量用户，平台拿着活跃的用户数据获得高估值融资。创作者经济的价值在今天超过1000亿美元，为超过5000万的内容创作者提供了岗位。顶尖的自媒体影响力甚至超过传统主流媒体。然而，创作者在平台中处于弱势地位，发布的作品可能因为不符合平台的审核标准而被一键删除甚至删号，也难分享平台成功的果实。这都抑制了创作者经济的进一步发展。是时候发展为创作者经济服务的去中心化平台了。Web3的创作者经济平台应该由创作者建立、经营以及共同拥有，平台发展的同时也有利于创作者和粉丝，实现财富积累过程的民主化。

Audius

Audius 是一种去中心化的音乐共享和流媒体协议，让创作者自由地发布音频内容，将作品商业化，与听众直接进行交易。

Rally

早期知名的社交代币发布平台。Rally 是一个由创作者驱动的去中心化网络，KOL、明星、品牌可在Rally平台上发售定制化的个人代币，实现作品商业化，也提供了和粉丝进一步交流的途径。粉丝可通过功能代币RLY来购买喜欢的创作者代币对其进行支持，获取限量版内容或者商品，甚至参与私人活动。

Chiliz

领先的体育产业的粉丝经济代币平台。它运营基于区块链的体育娱乐平台 Socios，粉丝可以通过原生代币CHZ参与初始FTO（粉丝代币发行）购买粉丝代币，参与喜爱的体育品牌的治理。对于体育俱乐部来说，粉丝代币提供了一种与粉丝建立联系并开启新收入来源的方式。俱乐部可以决定球迷影响力，投票的范围可以从新球衣设计到表演赛中的球队对决。

Mirror

去中心化的Medium。创作者可以发布文章和NFT、发起众筹、进行拍卖、协同合作，甚至进行社区投票，使有效信息迅速获得价值反馈。Mirror不仅仅是为区块链行业的内容平台，它永久存储、抗审查的功能同样适合更广泛的内容创作者或者普通人。以内容写作工具为出发点的mirror, 在最近开放所有功能后，有扩展为去中心化协通合作平台的潜质。

Gitcoin

最大的资助web3新基础设施的捐赠平台，一个由开发人员、协议和捐赠者共同创建，促进开源领域的发展，旨在构建具有数字民主和重视社区建设的数字生态系统。

代表性DAO

以智能合约为纽带的是去中心化自治组织（DAO）严格基于智能合约对共同拥有的资金进行分配管理，提升了资本筹集速度，增加投票的透明度。相比传统公司形式，具有更快的执行效率，也消除了对法律中介层的依赖。DAO使更多投资者得到参与早期投资的机会，也拓宽了优秀项目的融资渠道。下面介绍具有代表性的DAO。

Yield Guild Games

最出名的去中心化游戏公会。Yield Guild Games 建立了一个玩家和投资者社区，他们通过投资用于链游的NFT 来赚钱，目标是发展世界上最大的虚拟经济，将资产效用最大化并与利益相关者分享利润。YGG的主要收入来自直接或间接租赁拥有的NFT资产给公会成员使用，换取部分游戏奖励。非公会成员可以在游戏的土地进行经济活动产生收入。在疫情期间，菲律宾有很多人通过YGG获得了比当地平均工资水平还高的收入，缓解了就业压力。YGG还投资了元宇宙DAO Merit Circle。不过MC目前也主要专注于链游，和YGG的业务相似，等后期资金充裕才会发展其他方向。

BanklessDAO

加密媒体 Bankless 推出的DAO， 协调参与媒体节点的分工，通过教育、媒体和文化推广以太坊、DeFi和比特币等无银行货币系统。

Friend with Benefits

a16Z投的链上链下结合的社交DAO。要加入这具有超过两千名会员的DAO，不仅需要严格的身份认证，还需要购买接近一万美元的FWB代币作为门票。成员主要有创作者，艺术家，反抗权威人士，思想家和建立组织来改变未来的实践者。通过聚集一群想要改变世界的人，FWB以各种方式促进web3的发展，如举办NFT gallery、虚拟音乐工作室。

BitDAO

bybit发起的专注于DeFi的去中心化组织，通过研发、流动性引导和提供资金的形式为DeFi、DAO、NFT和链游项目提供支持。其核心产品包括一套链上和链下治理解决方案，对DAO金库进行投资管理。BitDAO通过代币交换和共同开发与其他项目合作，吸引出色的开发人员，也使金库积累了顶级项目代币。

MetaCartel Ventures

2020年推出的第一个投资DAO，MetaCartel 社区创建，它投资于早期去中心化应用程序，其成员必须是 MetaCartel Ventures的合格投资者或员工，聚集了行业顶尖大咖。MetaCartel Ventures 目前不对新会员开放。

The LAO

The LAO 由 OpenLaw 建立的投资DAO。LAO 的成员贡献了超过 15,119 个 ETH。新会员必须贡献至少 310 ETH来获得1%股份的LAO 会员资格。任何基于以太坊的项目都可以在启动日期之向 LAO成员推荐他们的公司，优秀的项目将被提交给所有 LAO成员集体决策。

DAO工具

Aragon

Aragon成立于2016年，提供一套模块化的应用程序来建设、治理Dapp、加密协议和DAO，拥有包括LidoDAO在内的知名用户。它的主要产品是Aragon client，帮助用户在以太坊上建立DAO。此外，该项目也在建设基于权益共识机制的Aragon Chain。

Gnosis

多签资产管理工具。Genosis基于以太坊生态系统开发了多项基础设施，包括Gnosis Safe、CowSwap、预测市场和拍卖。Gnosis Safe的多重签名功能使用户能够定义访问控制权限，安全管理DAO资产。

Syndicate Protocol

去中心化的链上投资协议和投资社交网络, 在区块链上降低投资辛迪加的门槛，给创业者们搭建了获取风险投资的渠道。Syndicate记录链上与链下的交易，实现存款自动化，管理小额投资者的股权和收益分配。

Snapshot

是一个去中心化的投票系统，提供了灵活的投票权重的选择，以满足组织的不同需求。在 Snapshot 上创建提案和投票是在链下执行，因此不会产生 gas，对用户友好。

Collab.land

基于用户代币数量的社区管理工具，创建一个特定的社交空间。将机器人添加到您的Telegram群组或Discord公会，他们就会为您的管理员。持有代币的用户可以被机器人允许进入社区。一旦用户出售代币，机器人就会将他们清楚社区。

SourceCred

衡量个人对DAO贡献的工具, 来激励社区成员积极参与Discord 讨论，在论坛提出治理提案。SourceCred跟踪用户发布的消息数量和社区参与度(例如点赞数、回复数、代码提交数)，并依据这些客观数据给用户分CRED积分。DAO金库可根据CRED分配代币奖励。

Superfluid

链上流支付和资金管理工具。用户能够按照预定义的规则进行链上转账，资金将实时从用户的钱包流向不限人数的收款人，没有资金锁定。非常适用于DAO的工资分发和奖励分配。

DAOhaus

DAOhaus 是一个用于启动和运行 DAO 的无代码平台，用户可以通过平台加入、建立DAO，发起提案,查看其他人的提案内容和投票情况等活动。已经为不少知名DAO服务，如MetaCartel Ventures和The LAO。DAOHuas是由社区捐赠驱动的，因此也提供了CCO（Community Contribution Offering）的发行方式。用户可以通过捐赠入股获得喜爱的DAO的不可转让股份。DAO使用捐赠资金时，需要发起提案，捐赠者将按照所占份额获得HUAS空投和项目空投。

指数

NFTX

以太坊上面的NFT指数，跟踪广受欢迎的 NFT 收藏品，例如 CryptoKitties、CypherPunks 等，人们可以将其在Sushiswap 和 Uniswap 等去中心化交易所（DEX）上交易NFTX代币，这为想投资NFT的投资者提供了便捷，降低了进入门槛和费用成本，分散风险。

Metaverse Index

Index Coop推出的指数，旨在跟踪娱乐、体育和元宇宙主题的ERC20代币表现。指数主要选取流通市值超过5000 万美元、在去中心化交易所流动性好的主题代币，用户可以将MVI / ETH LP 在 Index Coop 进行质押。
