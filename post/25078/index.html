DeFi热点

1.PeckShield：BSC链上Merlin Lab疑似遭到攻击

6月29日消息，PeckShield派盾预警显示，BSC收益聚合器Merlin Lab疑似遭到攻击。

2.PeckShield：BSC链上Merlin Lab遭攻击源于收益逻辑漏洞

PeckShield派盾安全人员定位发现，BSC链上收益聚合器 Merlin Lab 遭到黑客攻击源于 MerlinStrategyAlpacaBNB 中存在的逻辑漏洞，合约误将收益者转账的 WBNB 作为挖矿收益，使得合约增发更多的 $MERL 作为奖励。

3.DeFi 概念板块今日平均涨幅为4.53%

金色财经行情显示，DeFi 概念板块今日平均涨幅为4.53%。47个币种中41个上涨，6个下跌，其中领涨币种为：COMP(+28.95%)、BAL(+14.52%)、ZRX(+12.28%)。领跌币种为：HDAO(-6.16%)、DMG(-5.23%)、YFV(-2.85%)。

4.算法稳定币项目SafeDollar被黑后SDO币价归零

据Decrypt 6月29日消息，算法稳定币项目SafeDollar被黑后SDO币价已归零。周一，SafeDollar团队表示，已暂停SafeDollar活动并正调查此事，调查结束后将发布计划向流动性提供者提供补偿。不过，该团队尚未结束对攻击的完整报告。此前28日消息，算法稳定币项目SafeDollar遭到黑客攻击，损失达25万美元。

5.Polygon推出可扩展的数据可用性解决方案Avail

Polygon宣布推出一种通用的、可扩展的数据可用性解决方案Avail。根据周一发布的新闻稿，Polygon透露Avail将用作侧链、独立网络和第二层协议等执行层的数据可用工具。

Heco

1.YouSwap将于7月上旬启动“火星矿池"联动计划

7月9日，YouSwap将迎来HECO首次减产，现决定于7月上旬开启HECO生态链特"火星矿池"计划，同时开设"火星挖矿专区"。届时将会为参与“火星矿池”计划的项目方提供总价值超过10万USDT的奖励，所有用户均可参与此次计划进行流动性挖矿。 截至6月29日17:00，YouSwap累计交易总额达87,229,078 USDT，YOU总销毁数量达640241，累计挖矿总产值4,488,635USDT。

NFT

1.Square CEO：Tidal有兴趣探索NFT机会

据The Block 6月29日消息，Square的首席执行官Jack Dorsey表示，音乐流媒体服务Tidal对探索NFT感兴趣，这可能是补偿艺术家的一种新方式。据悉，Square今年早些时候收购了Tidal的多数股权。

2.推特CEO Jack Dorsey：音乐平台Tidal有兴趣拓展NFT业务

6月29日，推特 CEO Jack Dorsey称音乐平台Tidal有兴趣拓展NFT业务以支持艺术家创作。据悉，JackDorsey作为推特和Square的CEO在今年早些时候收购了Tidal的多数股权。

3.NFT市场活跃钱包数量突破1.2万个，创1个月来新高

据官方消息，昨日NFT市场活跃钱包数量突破1.2万个，创1个月来新高。今天NFT市场活跃钱包数量依旧在1.2万个上方，为12367个。

4.喜剧演员Eric Andre拍卖首个NFT

上周末，喜剧演员Eric Andre拍卖了他的第一个NFT作品“Non-Flushable Turd”。据悉，Andre最出名的脱口秀节目为“The Eric Andre Show”。

5.加密社交+NFT平台Torum获得AWS孵化计划Grant

据官方消息，亚马逊网络服务（Amazon Web Services，缩写为 AWS）与日本传统机构 Saison Capital 于2021年6月联合发起Fintech Forward 孵化计划，这项计划专注于寻找、孵化并投资东南亚地区优秀的金融科技、区块链、及加密货币相关的新创公司。 一站式 NFT + 加密社交平台Torum 获得 AWS 本次孵化计划Grant与技术支持。Amazon Web Services 云端平台通过全球资料中心提供超过200项功能完整的一站式云端服务。 据悉，Torum是⼀家专为加密货币⽤⼾及项⽬⽅而设的社交平台，致⼒于以社交平台作为⽣态圈的基础，并通过整合搭建于币安智能链上的DeFi 平台、NFT商城等功能，目前平台已拥有来自十余个国家近6万名的注册用户，已连续数月保持10%-30%用户增长率。

6.SumSwap定制NFT作品已在opensea.io进行交易，最低价格4000USDC

据官方消息，数学创新型去中心化协议SumSwap已为500多名会员每人成功发放一枚NFT作品，剩余NFT作品正在发放中。当前已发放的NFT作品已经在opensea.io去中心化NFT交易平台交易，最低价格4000USDC。 据悉，SumSwap是英国数学专家及区块链极客打造的一款数学创新型去中心化协议，近日SumSwap为每位会员发放一枚定制NFT，该NFT代表最早加入SumSwap一批元老用户，具备收藏价值。

7.梁和生先生携手（福、禄、寿）系列作品即将登录SilkSwap

据官方消息，梁和生先生携手（福、禄、寿）系列作品即将登录SilkSwap去中化交易所，该系列作品曾被艺术家、企业家，明星收藏，锻造成NFT数字艺术品。 据悉，SilkSwap是结合NFT+DeFi跨级加密艺术平台，SilkSwap利用这一特性，结合区块链的不可篡改、可追溯的技术原理，通过 NFT+DeFi 的技术理念来重新定义艺术品。SilkSwap同时支持单双币挖矿，交易挖矿、流动性 LP 挖矿、无损质押挖矿（机枪池）挖出双币，单币为流动性挖矿。Silkswap致力于成为去中心化、社区人人自治的交易所。

8.Jay-Z和JackDorsey拥有的音乐流媒体服务可以采用NFT和智能合约

在6月28日在Twitter Spaces举办的会议上，Jay-Z和Jack Dorsey两人强调了推出能够赋予使用该平台的艺术家权力的功能的重要性，并举出智能合约、NFT和分析工具作为例子。Jay-Z和Jack Dorsey暗示，他们的音乐流媒体服务Tidal可能正在考虑支持不可伪造代币（NFTs）和智能合约的整合。

9.英国艺术家PhilipColbert将在Decentraland上推出NFT艺术展和音乐表演

英国艺术家PhilipColbert计划在虚拟世界Decentraland中举办NFT艺术展，还有与美国传奇艺术摇滚乐队Devo合作的名为“LOBSTERDE-VO”的音乐表演，PhilipColbert目前已在Decentraland中推出了“Lobsteropolis City”和“Lobster Land Museum”，并将Metaverse视为NFT展览的所在地。

10.NFT指数基金NFTX发布V2版，新增特定NFT赎回功能

NFT指数基金NFTX发布V2版本，目前处于测试阶段，V2版将与V1一起运行以允许库存有序迁移。V2版新增功能包括针对Vault的自定义合格模块（如使用链上元数据），能以额外费用（5%的溢价）赎回特定的NFT，LP持有者智能合约从Vault周转中收取协议协议费用，代币化库存供应（xTokens）以及vToken（Vault代币）闪电贷，普通变化包括每个组合激励一个单一的Vault，鼓励meme友好且可现金标记（cashtag-able）的资产代码、术语层面将「Fund」替换为「Vault」。
