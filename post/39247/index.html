Chris Dixon 曾在 2021 年发表的《NFTs and a Thousand True Fans》一文中分享了创作者如何利用 NFT 从一千个忠实粉丝中变现的方法。随后在创作者经济领域，NFT 的用例开始成倍增加。其中一个典型的用例就是 NFT 会员：要求粉丝持有特定的 NFT 才能加入社区并享有特权。NFT 会员可以帮助创作者加强品牌和建立更有价值的社区，同时粉丝也可以随着社区发展获得收益。

在未来，NFT 会员将会成为与广告和订阅相媲美的创作者获利方式。

接下来将通过实际案例讨论 NFT 会员的优势，它们在实践中的使用方法，以及创作者、Web3 初创公司如何利用 NFT 会员扩大影响力。

NFT 会员案例

虽然所有的 NFT 都是存储在区块链上的数字资产，但是不同类型的 NFT 可以发挥不同的作用。例如 PFP 类型 NFT 可以当做头像使用，而 NFT 会员就可以作为一个演唱会等专属团队的门票使用。项目方或者社区只需要在区块链网络上查看用户地址是否持有有 NFT 就可以判断他们能否拥有特定权利。

越来越多的创作者利用 NFT 会员建立社区，并因此涉足 Web3。对于创作者来说，NFT 会员可以通过多种方式取代传统的订阅和基于广告的收费模式。

创作者可以通过细分粉丝的付费意愿，从 NFT 中获得更多收入。 在 YouTube 等传统平台上，针对不愿意为内容付费的用户来说，创作者只能通过投放广告获利；对于愿意支付少量费用的用户，创作者可以通过订阅获利；而对于一些有意愿支付更多费用的忠实粉丝，创作者并没有很好的获利方式。针对不同付费意愿的粉丝，创作者可以推出不同类型的 NFT。例如，创作者可以以忠实粉丝群体为目标用户推出限量版 NFT，也可以为一般粉丝推出普通 NFT。

NFT 帮助创作者根据支付意愿细分用户群体

创作者可以通过降低平台抽成从 NFT 中获得更多收益。 传统平台会从订阅和广告中抽取很大一部分收入，并可以在未经创作者许可的情况下随意更改规则。如果使用 NFT，创作者可以保留他们出售 NFT 时获得的大部分收益（具体请参考《The NFT Mint Sweet Spot : Data on Early Decisions》），还可以从版税中获得额外收入。

NFT 可以直接激励早期支持者，并帮助创作者取得成功。 在传统平台上，早期粉丝会给创作者带来很多必要的帮助和鼓励，甚至为其后来的成功奠定重要基础，但是早期支持者的身份很难得到证明，且不太可能带来切实的回报。NFT 为粉丝提供了一种证明他们早期参与者身份的方式，创作者可以为早期粉丝准备特别活动等额外福利，并且当创作者获得更大的成功时，其 NFT 价格通常也会大幅增长，早期持有者可以获得较大收益。

NFT 会员机制所面临的挑战

创作者无法直接控制铸造后 NFT 的价格。 尽管创作者可以设定初始 NFT 铸币价格和他们在二级版税中的份额，但是他们无法控制 NFT 未来价格的变动，如果整个 NFT 市场崩溃，创作者最终获得的收入可能会低于传统渠道。投机者也可能通过购买创作者的 NFT 并抬高价格，导致真正的粉丝没有参与的机会。针对这个问题，创作者可以在早期铸造时制定粉丝白名单，或者禁止个人一次购买过多的 NFT。

如果 NFT 价值不明确，创作者可能会失去粉丝的支持。 粉丝更加希望 NFT 会员可以提供长期效用，所以创作者需要明确 NFT 会员所带来的价值，以免让粉丝感到失望。在没有明确如何为 NFT 会员持有者提供价值时启动 NFT 项目，可能会导致创作者声誉受损，失去粉丝的支持。

NFT 会员社区治理是一项艰巨的工作。 社区治理是创作者实现他们内容制作计划的重要方式。为了减轻工作量，创作者可以聘请专业团队来保持社区的活跃度，并鼓励粉丝自发维护社区环境，扩大社区影响力。作为创作者品牌的核心，创作者也需要时刻关注社区发展情况。

创作者如何在实践中使用 NFT 会员

创作者发行 NFT 会员最低要求是什么？成功发行的社区通常包含三个组成部分：

  1. 创作者需要有一定的粉丝基础，可以向其发出 NFT 会员购买邀约

  2. 粉丝愿意购买 NFT 以加入社区，并帮助增加会员价值

  3. 创作者及其社区通过现实世界实际利益的形式为会员提供效用。

下面我们观察三个 NFT 会员社区的例子。

VeeFriends：它是 Gary Vaynerchuk 发行的 NFT 藏品，除此之外，Gary Vaynerchuk 还与他人共同创立了预订平台 Resy 和数字机构 VaynerMedia，他是该公司的首席执行官。

  • 创作者：凭借过往信誉， Gary 建立了庞大的多样化受众群体。

  • 社区：Gary 通过相关培训，让粉丝了解 NFT 为何重要以及如何首次购买 NFT，便于粉丝购买 NFT 并加入社区。并且创作者为项目的所有角色都创造了背景故事，增强 NFT 的叙事性。

  • 实用性：所有 VeeFriends 持有者均可免费参加为期三年的 VeeCon 会议。在 2022 年 5 月举行的首届 VeeCon 大会上，Snoop Dogg 和 Beeple 等重量级嘉宾出席并做了相关演讲，同时 VeeFriends 的持有者在这次活动组织中发挥了关键作用。

  • Moonbirds：这是由 Proof Collective 推出的 NFT 藏品，而 Proof Collective 是 Kevin Rose 发起的社区。

  • 创作者：Kevin 是流行的加密播客《Modern Finance》的主理人。

  • 社区：Proof Collective 的第一个 NFT 系列有 1000 个 pass，许多收听 Kevin 播客的顶级 NFT 收藏家已经购买了它们，形成了对 Moonbirds NFT 的潜在需求。

  • 实用性：Proof Collective 社区会以解锁独家奖励的方式，例如一些商品或者线下活动，激励 Moonbirds 持有者长期持有其 NFT。

Developer DAO：由 Nader Dabit 在离开 AWS 之后推出。

创作者：Nader Dabit 在 YouTube 建立了 Web3 开发者教育频道，吸引了大批粉丝。之后他启动了 Developer DAO，旨在加速对 Web3 建设者的教育和影响。

社区：开发者和其粉丝购买了 Developer DAO NFT，增强了 NFT 会员的吸引力。

实用性：成为社区成员是该 NFT 的主要优势，成员在举办活动、开发项目和工作推荐等方面相互提供帮助。

Web3 公司如何帮助创作者发起 NFT 会员项目？

创作者和社区正在以新的方式尝试 NFT 会员项目。随着这一趋势的加速发展，从 Web2 跳转到 Web3 的创作者在整个项目建设中需要额外的帮助：

创建 NFT

  • 定义 NFT 会员详细信息（例如，目标、受众、数量、价格、福利）。

  • 招募团队或选择无代码解决方案来搭建艺术品、智能合约和网站。

向粉丝介绍 NFT 会员

  • 建立社区并告诉粉丝 NFT 会员的价值。

  • 使用白名单为粉丝和投机者保留 NFT 铸造名额

与社区成员合作，为 NFT 会员创造价值

  • 了解哪些持有者对社区贡献最大。

  • 与持有者合作，为 NFT 会员提供持续的效用。

管理业务

  • 管理并找到为国库提供资金的方法，例如寻找专业资管人士成为合作伙伴。

  • 管理税收和法律合同。

许多公司已经在应对上述挑战，希望更多的创作者和社区使用 NFT 会员来创造价值和分享收益。
