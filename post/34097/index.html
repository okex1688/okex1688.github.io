原文标题：《 Vampire attack! LooksRare vs. OpenSea 》

Coinbase Ventures 的Around the Block系列博客致力于 Crypto 主要趋势的研究。本篇文章的作者是 Connor Dempsey、Justin Mart 以及 Mike Cohen。

只有 Crypto 领域才会出现的现象是：一个由匿名创始人在几周内建立起来的平台，就能与业内处于领先地位的平台进行抗衡。

新上线的 NFT 平台LooksRare便生动诠释了这一现象，其 1 月份的交易量达到了 90 多亿美元，几乎是 OpenSea 同期交易量的三倍。不仅如此，在上线的 30 天内，LooksRare 就取得了3.07 亿美元的协议收入，而同期 OpenSea 只有 1.1 亿美元。

然而，这些原始数据并不能反映全部的问题。

在本期博客中，我们将探讨 LooksRare 是如何对行业领先的 NFT 市场 OpenSea 进行吸血鬼攻击的。

OG 的吸血鬼攻击

吸血鬼攻击是一种只存在于 Crypto 或 Web3 中的现象。广义上来讲，吸血鬼攻击是指通过向用户提供某种激励（通常为 Token），从而将用户从现有平台吸引到新兴平台上的方法。

最臭名昭著的一次吸血鬼攻击发生在 2020 年，当时 SushiSwap 推出了一个与行业领先的 Uniswap 平台非常相似的去中心化交易平台，它会给予那些将资金从 Uniswap 迁移到 Sushiswap 的用户 SUSHI Token 作为奖励。除了能享受到 SushiSwap 上更低的交易费用以外，SUSHI Token 持有者还将获得平台的治理权。

最终，由于 Uniswap 用户将 12 亿美元的资金转移到 Sushiswap，Uniswap 的流动性出现了短时内的下降，而 Sushiswap 也兑现了自己的承诺，向用户发放了 Token 奖励。之后 Uniswap 的流动性也恢复到了原先的水平，而且他们也推出了自己的 Token。即便如此，Sushiswap 的流动性在较短的时间内还是得到了引导。

LooksRare 的吸血鬼攻击

LooksRare 的吸血鬼攻击所遵循的是那套经典的操作步骤，具体如下：

1. 确定出行业领军平台 

2. 建立兼具竞争力和战略性优势的平台 

3. 为那些迁移过来的用户提供优厚的奖励

SushiSwap 和 LooksRare 的吸血鬼攻击的主要区别在于，SushiSwap 几乎复制了 Uniswap 的全部代码（也被称为分叉），并基于此制定了 Token 激励措施，而根据 LooksRare 的文件显示，他们似乎建立了自己的智能合约。不过，二者其余的做法都是一致的：

1. 确定出行业领军平台：在这一点上，它比 OpenSea 要做的好。 

2. 建立兼具竞争力和战略性优势的平台：LooksRare 搭建了 looksrare.org。 

3. 为那些迁移过来的用户提供优厚的奖励：发放LOOKS Token。

初始激励措施

由于区块链的开源性质，LooksRare 团队能够识别出那些在此前六个月中至少交易了价值 3 个 ETH 的 NFT 的 OpenSea 用户，并向他们空投 LOOKS Token。然而，要想获得这些 Token，用户必须先在 LooksRare 交易平台发布一个 NFT。

将 Token 空投给那些活跃的 NFT 交易社区效果显著，而 NFT 也因此涌入了新的市场。LOOKS Token 在发布 10 天之后就上涨到了大约 7 美元的价格，并顺势取得了 10 亿美元的市值。

不过，Token 空投并不是唯一一种激励措施。

其他激励措施

除了最开始的空投以外，LooksRare 平台的用户也可以通过质押他们的 LOOKS Token 和他们在平台上交易的 NFT 来赚得更多的 Token。LooksRare 会对每笔交易收取 2% 的手续费（而 OpenSea 则为 2.5%），而质押之后，也就是 LOOKS Token 被锁定在智能合约中以后，用户便有权获得全部的手续费收入。除此之外，质押者还有机会获得额外的 LOOKS Token。

在我写这篇文章的时候，LOOKS 的质押收益年利率已经超过了 500%，十分惊人。

LooksRare 在交易上的激励措施颇为有趣，而且原理也非常简单：用户只需要在 LooksRare 上买卖 NFT 便可以赚取 LOOKS Token。LooksRare 会根据每日的交易量来划定奖励比例，且目前来看，LooksRare 每天发放大约 280 万个 LOOKS，其市场价约为 1 千万美元。

可以肯定地说，免费的空投就已经足以让人们来到这个平台了，而每天 1000 万美元的奖励更是让 LooksRare 的每日交易量完胜 OpenSea。不过，如果我们仔细观察 LooksRare 每日用户人数的话，就会发现它惊人的交易量并没有我们想象中的那么简单。

刷单交易

虽然 LooksRare 在任何一天的交易量都是 OpenSea 的两倍，但 OpenSea 的活跃用户却比 LooksRare 多 20-40 倍。这表明，LooksRare 的交易量是少数交易者为了赚取 LOOKS 而投机取巧的结果。

由于 LooksRare 上每天的奖励金额是由当日的交易量所决定的，所以即便存在着交易费用，用户也愿意用自己的钱包来回买卖同一个高价的 NFT。这样一来，如果有人可以通过刷量完成当日 10% 的交易量的话，那么他就可以获得价值 100 万美元的 LOOKS。

但问题在于，每笔交易中用户都必须支付 2% 的手续费，并以 ETH 来核算。计算结果显示，交易者用 ETH 支付的费用与获得的 LOOKS 奖励金额大致相等。因此，交易者基本上是在用 ETH 来交换 LOOKS，当 LOOKS 相对于 ETH 升值时，用户便会取得相应收益。

从本质上讲，这就是一场博弈，因为如果 LooksRare 平台能取得成功的话，该平台的早期用户就能因此获利。

LooksRare 上的真实交易量

诚然，LooksRare 惊人的交易量确实只是归功于他们各类的激励措施。不过，这也并不意味着 LooksRare 只是虚有其表。

Crypto Slam 曾于 1 月底预计说，今年 1 月份 LooksRare 上 90 亿美元的交易量中约有 80 亿美元都是虚假交易。然而，剩余的 10 亿美元 NFT 交易量也仍然超过了 Rarible、SuperRare、Foundation、Makersplace 和 Aysnc 等其他 NFT 市场2021 年全年交易量的总和。

就算消除了虚假交易的影响，LooksRare 也依然是一个成功的平台。另外，他们还提供了其他有趣且新颖的功能，比如无手续费的私人买卖，或对整个 NFT 收藏进行单一报价。因此，LooksRare 的「真实交易量」正在持续增长。

交易奖励能否持续

LooksRare 的交易奖励将在未来两年内逐渐减少，届时，它将不得不依靠其产品和社区来参与市场竞争。LooksRare 能否在没有优厚奖励的情况下继续扩大其市场份额，还将取决于其背后的匿名团队。

我们可以从 SushiSwap 身上大致窥探一下 LooksRare 的未来。从许多方面来看，SushiSwap 在其最初的奖励发放完毕以后还在继续发展。通过不断创新与多链部署，今天 SushiSwap 的总锁仓价值已经达到了将近 50 亿美元。

不过，SushiSwap 的发展并没有影响到 Uniswap 的地位。Uniswap 最终推出了自己的 Token（有人猜测 OpenSea 有一天也会这样做），并依然是 Crypto 领域中占主导地位的 DEX，如今已经有超过 75 亿美元的总锁仓价值了。不仅如此，一直以来还没有任何有争议性的硬分叉（如 Bitcoin Cash、Ethereum Classic 等）或吸血鬼攻击者（SushiSwap、Swerve 等）曾取代过现有平台。

虽然 Sushiswap 现在已经获得了成功，但它也曾经历过一些挫折。其核心团队和 SUSHI Token 持有者之间的内部矛盾曾导致 Sushiswap 丧失了几位核心团队成员，其中就包括项目的首席技术官。当 LooksRare 团队将治理权移交给 LOOKS Token 持有者以后，它可能才会发现DAO 治理目前还尚未成熟，且问题丛生。

最后，对于 LooksRare 的用户来说，匿名团队随时都有可能跑路。此外，LooksRare 的智能合约是在未经审计的情况下推出的，并且没有公开的 GitHub 库，所以各位交易者还是要保持谨慎。
