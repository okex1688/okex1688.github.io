一年一度的报税季又来啦，NFT的崛起催生了新一代投资者，对于当今的NFT投资者来说，牢记税收影响很重要。否则，他们可能会陷入税收噩梦。

在 2017 年牛市之后，许多加密货币交易员发现自己陷入了困境。他们在市场上涨时积累了巨额税负，但在崩盘后不再有钱支付税款。这些交易者中的许多人根本没有意识到他们的交易对税收的影响，也没有做好相应的准备。

本文将分享每个NFT投资者需要知道的关于税收的三件事，可以避免与美国国税局(IRS)发生纠纷。

购买 NFT 时，您很可能要交税

处置您的加密货币被视为应税事件，使用以太坊 (ETH) 或其他加密货币购买 NFT 就属于这一类。根据您最初收到加密货币后价格的变化情况，您将产生资本收益或损失。

许多NFT交易员背负着巨大的税务，因为他们的代币自最初收到以来价格已经大幅上涨。为了避免遇到纳税问题，您应该计算您进行的每笔交易的潜在税单，并在纳税季之前将钱存起来。

出售 NFT 时需要纳税

无论您是出售法定货币、加密货币还是将其兑换成不同的 NFT，出售 NFT 也被视为应税事件。NFT 的征税方式与加密货币类似——出售 NFT 的应税收入是通过计算购买 NFT 时的原始成本基础与出售所得总收益之间的差额来确定的。

如果您的 NFT 的价值自您最初收到以来有所下降，只要您将 NFT 作为投资而非个人使用，您就可以申请资本损失并减少您的税务。

您可以通过查看购买原因来确定 NFT 是用于投资还是个人使用。您是打算赚取利润，还是打算只享受NFT供自己使用，而不考虑资产是否会升值。

投资的资本损失可以抵消您当年的资本收益和高达 3,000 美元的普通收入。个人使用造成的资本损失不可扣除。

您的 NFT 可能被视为收藏品

出于税收目的对 NFT 进行分类之所以如此困难，部分原因在于它们是一种新型资产类别。不幸的是，这意味着美国国税局尚未就某些 NFT 是否将被视为收藏品并以更高的税率征税发布明确的税收指南。

根据税法，某些实物资产被视为收藏品。这包括艺术品、黄金等金属以及邮票或棒球卡收藏品。当这些资产在一年后出售时，它们的最高税率为 28%，而典型的长期资本利得率在 0% 到 20% 之间。

可以合理地得出结论，某些 NFT 艺术品将被视为税收目的的收藏品。

那么，像Bored Ape Yacht Club 这样的头像系列呢？很容易理解为什么它们会被美国国税局认为是收藏品，因为 10,000 张独特的图像都是“收藏品”的一部分。然而，这个问题仍然没有完全解决。

如果没有额外的 IRS 指导，任何不是艺术品的 NFT 都可能不符合可征税规则。例如，可以合理地假设代表 Uniswap v3 流动性头寸的 NFT 不会被视为收藏品。

一些 NFT 投资者正在采取更激进的税收选择。他们认为，如果没有 IRS 的指导，NFT 不应被视为收藏品，因为它们是无形的。这些投资者之所以采用这种方法，是因为与收藏品相关的税法提到了有形财产——这使情况变得混乱。

这似乎是一个很难向 IRS 提出的案例。但是，如果没有指导，很难确定，可能会有纳税人决定倾向于不确定性并采取更积极的税收方法，因为他们知道 IRS 在这个问题上的指导可能需要数年时间。

有关收藏品的税法很复杂，在评估您数字资产的收藏状态时，最好与税务专业人士交谈。

当然，目前这个问题对大多数 NFT 投资者来说可能并不重要。由于 NFT 非常新，因此大多数销售可能涉及持有时间不到 12 个月的 NFT。无论它们是否被归类为收藏品，这些 NFT 都按普通税率作为短期销售征税。

牢记 NFT 的税收影响，您可以避免在未来一年在不知不觉中招致非常大的税收负担。请记住，当您买卖 NFT 时，您很可能会纳税，但要确定您的 NFT 是否属于需要纳税的收藏品，则需要仔细研究。

您可能还在指望IRS在如何对NFT进行分类方面更加清晰，这种明确性可能不会很快到来。短期内，美国国税局可能会转而关注那些选择完全不纳税的NFT投资者。
