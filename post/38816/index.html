通过对主流的 NFT 进行分类，我们观察到 NFT 所涉及的领域已经远远超越了传统的艺术品 / 收藏品。越来越多的 NFT 项目开始赋能 NFT 多样化的公用事业，为 holders 带来更好的使用体验。在熊市期间，那些 100% 由投机构成的 NFT 基本彻底凉凉，而那些有故事、有实用性和社区的 NFT 则可以持久发展下去。

我们根据其效用对市场上的 NFT 进行分类：

艺术

在过去的几十年中，艺术形式一直在不断变化。它现在正在进入数字世界，由实体往线上转变。对于艺术家和买家来说，普遍担心宣传成本、转售成本和流通等风险，且不易存储收藏——就像近期，《蒙娜丽莎》遭到破坏被恶意扔蛋糕。

而 NFT 的出现，则将这些风险最小化。毕竟通过线上艺术家们的作品就能有很好的曝光——除了 Instagram 和 Twitter，OpenSea 是艺术家展示其 NFT 作品并获得公众曝光的另一个主要平台。此外，购买艺术品时最大的问题之一是，买家是否具备艺术品高消费力的门槛。随着 NFT 的出现，它允许买家通过智能合约以挂牌价格提供报价或购买。

且通常来说，在一级市场倒卖艺术品是禁忌，但对于 NFT 艺术品，这并不是一个令人头疼的问题。除此之外，NFT 还带来了额外的许多益处——如：尽管艺术家从初始销售中获得了，但 NFT 允许艺术家从其艺术品的所有二级销售中获得版税，并且不管转售几次该版税机制一直存在，给予激励。

艺术类别涵盖了很多领域。任何东西都可以被认为是艺术。因此，这就是为什么每个 NFT 都属于上述三个类别之一的原因。因为如果它不属于其他两个，它可以归结为，“某人，某处，认为这看起来很酷，想买它。”

关键是，人们将某些东西视为宝贵的奢侈品。其他人可能不会以同样的方式看待它们。每个人都喜欢不同的东西。让人们享受事物。毕竟，这不是你的钱。所以，很多类型的NFT都可以归为艺术。

PFP

PFP 是“profile picture”（个人资料图片）的缩写，或通常称为头像类。截至目前，NFT PFP 仅在 Twitter 、Discord 等少数应用中常用，当然其他很多社交媒体平台也正计划在其应用程序中采用 NFT PFP。当前，NFT PFP 的最主要问题 holder 购买的 NFT PFP 可以被复制，但显然多数应用的产品经理都考虑了这个问题，如：Twitter 将验证 NFT 个人资料图片，如果有效，用户将获得六边形轮廓；这使得用户能够很好地分辨该 NFT PFP 的持有者是不是 holder，同时 holder 也有了可以炫耀的资本。

除了可以让用户展示获得满足外，它还可以作为类似 Pass 的功能使用，让用户成为社区成员、DAO 开发者，以及一个潜力巨大的项目的早期投资者。此外，PFP 的所有者也可能有机会访问该社区创建的游戏或其他产品，亦或者是获得能和某个名人持有同一系列 NFT，拉近距离感。

虚拟土地

NFT 虚拟土地是元宇宙平台上可拥有的数字土地区域。这些土地类似于《我的世界》，用户可以自由创作，只是数字土地完全是用户所有，不像《我的世界》中微软拥有用户的创作。当用户可以完全拥有该数字土地时，可以获得许多益处，例如：能够将 NFT 土地用于广告、社交、游戏和工作，以及其他用例。

可能听起来模糊，我们来简单解释下：

1、广告：如果用户的虚拟土地位于元宇宙中的核心（热门）区域，能够吸引很多游客，用户可以收取广告位费用。之前就有用户购买了临近 Binance 名下虚拟土地旁边位置的土地，收取了天价广告费。

2. 社交：用户可以在他们的数字土地上举办活动，包括音乐会、会议和社区聚会，亦或是用户可以构建自己想要的应用场景在自己的土地上。

3. 游戏：用户的 NFT 土地也有机会在 NFT 游戏中得到使用。有些游戏需要用户持有土地才可以参加，亦或是用户可以在自己的土地上打造属于自己的 NFT 游戏

4、工作：可以用3D头像探索的土地，可以作为虚拟办公空间，也可以提供数字服务。

游戏

NFT 代表加密游戏中的头像、游戏装备和游戏卡等数字资产。NFT的吸引力在于其独特性、稀缺性、所有权可证明性，可转移性以及不可分割性，这对于提升游戏内物品价值无疑具有巨大的意义。值得一提的是，NFT还嵌入了元数据，可以用来证明资产的真实性。

实际上 NFT 最先的应用也是游戏。在 NFT 游戏最为流行的时候，菲律宾每天玩 NFT 游戏的人数多达 100 万，这期间产生的收入甚至超过了他们的工资，改善了就业危机。随着 NFT 游戏的普及，游戏种类也大大增加，从 Axie Infinity（灵感来自 Pokemon Go）到 StepN（ Move to Earn ）等各种模式的游戏或相关应用都在增加。

会员资格

“会员”这个词在 Web2.0 中已经存在了很长时间。然而，用户的隐私、数据处理和所有权仍然是关键问题。会员 NFT （ Pass 卡 ） 可以通过解决上述问题来潜在地改变会员体验。首先，我们不必记住各种平台的一堆密码，而只需记住一个钱包助记词——毕竟不同平台的验证逻辑也是通过链接钱包验证钱包中的 NFT。此外，将我们的钱包连接到后端的智能合约允许我们访问多种功能。最后，我们还可以有资格在二级市场上转售会员以获取利润，比起传统会员的转售方便了不少。一些会员还具备使用热门项目跟踪、Smart Money 、资源社群等功能的权限。

社交

具备社交属性的 NFT 可以为持有者提供包括参加在线和面对面社交聚会在内的诸多权益。随着 NFT 变得越来越流行，诸多社交网络应用正计划使用 NFT 作为媒介。一个成功的社交 NFT 项目需要有影响力的人员（ Kol ）以及他活跃的社区。例如：VeeFriends NFT 由 Gary Vaynerchuk 创建，他是一位有影响力的 YouTuber 和企业家。通过拥有 VeeFriend NFT，所有者立即成为 VeeFriends 社区的一员并获得 VeeCon 的访问权限。不同的 VeeFriends 的 NFT 持有者可以参与 Gary Vaynerchuk 举办的不同社交活动。

以及治理。NFT 可用于向粉丝和客户群开放艺术和商业决策，目前多用于文娱领域。例如，可以向购买 NFT 的粉丝发放 NFT，以支持前期制作中的戏剧表演。作为购买 NFT 的交换，该资产的所有持有者都可以就主角 T 恤的颜色、角色名称甚至节目名称等事项获得投票。

这就是粉丝的未来。粉丝群在他们最喜欢的创作者的项目中有发言权。就像前阵子有人自建了一个王心凌 DAO，发的 NFT 很快就被抢完了。有趣的事和应用会持续下去。

音乐

音乐 NFT 中快速增长的生态系统也很引人注目。粉丝奖励和版税是 Web2.0 和 Web3.0 音乐收藏品之间的两个显著区别。许多平台旨在创建一个更具协作性的音乐市场或流媒体平台。例如：尽管拥有音乐 NFT 的所有权，但sound.xyz还授予持有者对歌曲发表公开评论；艺术家可以在Pianity 上将他们的音乐分成 4 个稀缺级别，从而使独特的版本更有价值。至于版税，http://Royal.io和http://Rocki.com等平台允许收藏家以 NFT 的形式投资流媒体版税。换句话说，流媒体产生的收入将根据版税分成，NFT持有者可以根据其持有比例获得。

品牌

NFT 具备作为未来数字资产的潜力，且该领域的火爆让各大品牌纷纷涌入 NFT 领域。在这些品牌中，阿迪达斯和耐克是其中的佼佼者。耐克通过收购 RTFKT 进入 NFT 领域（RTFKT 是 元宇宙数字运动鞋资产公司），他们使用最新的游戏引擎和 NFT 技术来创造一种数字运动鞋，如：RTFKT 设计了许多酷炫的 3D 耐克鞋 NFT，允许所有者“穿”并在“Snapchat”等社交媒体上展示。

另一方面，阿迪达斯通过与 BAYC 合作推出了自己的 NFT。阿迪达斯 NFT 的有趣用途是，阿迪达斯 NFT 所有者不仅可以访问数字阿迪达斯的虚拟世界，还可以获得阿迪达斯的实体产品，如运动服、连帽衫和无檐帽。

域名

NFT 还可用于与. com域名相似的域名服务，但不同的是域名 NFT 基于去中心化技术。Ethereum Name Service (ENS) 将可读的名称映射到区块链和非区块链资源机器可以辨识的标识，如Ethereum地址等。简单来说，ENS是把.eth域名解析为以太坊地址。ENS可以让以太坊地址更容易记忆，ENS域名可以在二级市场上买卖。

同时，区块链应用还可以和域名联动（如：Mirror ），链接了域名后还会有其他额外的功能。

Defi

在 Uniswap V3 上，流动性提供者 (LP) 头寸表示为 NFT（非同质化 ERC-721 代币），而不是 Uniswap V1 和 V2 上的同质化 ERC-20 代币。

根据在流动性提供界面上选择的池和参数，可铸造一个独特的 NFT，代表特定池中的位置。Uniswap NFT 显示有关用户的流动性头寸的最重要信息，包括交易对、所选层级、Token 符号和池地址。NFT 所有者可以在 Uniswap V3 中修改、赎回甚至出售头寸。

结语

如果我们对 NFT 进行最为详细的分类，可能会有几十种。这里我们选取了当前阶段，最具代表性的十大种类。可能在数个周期之后再回过头看，NFT 会有更多当前阶段难以想象的分类和应用。

有些时候太多的主观情绪反而会遏制想象力，所以我在做这篇文章的修缮时，也扩散开想到了很多可能存在的用例。这些可能在未来出现的用例和其更好的形态，也会在整理出来后在公众号/社群给大家分享。
