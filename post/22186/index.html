在 Meme 币种夸张的涨跌行情之下，NFT 话题的主流讨论度有所降低。

不过，如果说 Meme 币种让更多人普通投资人关注到了数字货币，那么 NFT 所做出的「破圈」努力仍在继续。

NFT 的存在为普通网民进入加密世界构建一条非常有创意的接口，而目前这种接口的影响力正在扩大。

Blocklike 观察发现，二手交易平台闲鱼、网购零售平台淘宝也成为了 NFT 的新接口，在这些平台中搜索 NFT，就能够看到熟悉的 NFT 、加密艺术、知名项目等已被上架了的可售产品。

NFT 走进「闲鱼」、「淘宝」

NFT 领域的造富神话和名人效应仍在扩散，更多人了解到数字艺术的巨大潜力后，正在努力参与其中。当购买需求增大，NFT 正在多种意想不到的渠道进行着渗透。

在众多讨论 NFT 主题的社区中，闲鱼被冠上了「中国最大的 NFT 交易市场」的名号。

这个名号并非空穴来风，倘若在闲鱼平台上搜索「NFT」， 最相关的交易产品卖家有二、三十个，卖家们设置的关键词还包括了「加密艺术」、「Opensea」、「Cryptopunks」、「全球最火」等标签。同样的信息也出现在了淘宝中，传统互联网网购平台成为了更广范围的投资人接触 NFT 产品的有趣入口。

根据观察，卖家在闲鱼上提供的产品主要分为两类：

第一类直接售卖 NFT 产品，买家与卖家直接商谈原创 NFT 的交易事项，或者商谈出售已购的二手 NFT 产品，这些交易者往往对数字资产已经有了一定的认知；

第二类则主要以代购与咨询为主，买家嘱托卖家购买指定平台中的 NFT 产品，卖家主要提供全程的代操作或全程指导，在「代购」这种方式中，买家只需提供平台中所挑好的 NFT 产品相关链接即可，其他操作都可由代购完成。除了「代购」 NFT，同样还有「代投放代管理」，帮助买家将自己的创作的作品在指定平台中上链。

第二类往往针对于 NFT 的「入门级」玩家，也是咨询人数与成交数量更多的一类。值得注意的是，这些卖家非常细心，对于那些初次接触 NFT 或加密艺术的「新人」，卖家往往会先行介绍「比特币」、「虚拟钱包」、「NFT」等基础知识，帮助买家先对数字货币有一个更好的理解。

在费用上，除了 NFT 本身购买（或生成）费用及相关 Gas 费用，卖家大多会收取「代拍费」、「手续费抽成」等费用。此外，倘若成交金额或成交数量比较大，交易双方也会讨论一些优惠方式，比如是否会「包 Gas 费」。

闲鱼的「求购」功能中，也能看到一些对于 NFT 充满兴趣的人，直接表达着自己对于 NFT 的交易需求。

虽然目前 NFT 热度一度被「动物园」所掩盖，但这种基于游戏或艺术的出圈产品，仍然不断地被更多普通投资人接纳。

前段时间社区热议的「加密艺术正在出圈」话题发酵至今，最终在闲鱼这样的「下沉」渠道中被具象化。

NFT：渠道正在打开

前段时间，区块链娱乐工作室 Virtually Human Studio（VHS）主管 Mac Ocampo 在分析 NFT 所面临的问题时就提出，NFT 游戏行业面临的障碍在于：并非所有游戏玩家都是数字货币交易者，也不是所有的数字货币交易者都是游戏玩家。

这个观点不止适用于 NFT 游戏领域，更是整个 NFT 交易市场所面临的问题。虽然很多平台致力于降低用户进入门槛，但对于「圈外」的人来说，购买数字资产时所需的操作流程、代码知识、Token ID 等流程，仍需要花费大量的时间学习与研究。而对部分数字货币交易者而言，可能他们并不关心 NFT 产品存在的文化价值，追逐的仅仅是热度引发的「财富效应」。

在由传统艺术媒体平台中央美院艺讯网上，一篇介绍 NFT 的论文中，需要附上 11 个加密市场基础术语用于科普及市场教育，才能让传统领域的读者更好地理解，NFT 的理解门槛最终还是围绕着加密行业的专业认知水平。

（中央美院艺讯网在介绍 NFT 时加入大量行业术语进行科普）

购买流程的复杂程度令买家望而却步。有观点认为，这个弊端是深深植根于加密世界底层，对于新入场的个人投资者来讲，整个购买体验并不友好。

据 nonfungible.com 数据显示，近一个月的 NFT 销售数量平均约 11 万左右，单月销售额达 3.03 亿美金，但二手交易量却呈现下降趋势，平均销售数量为 35626 件。

一份 2021 年一季度的 NFT 市场研究报告显示，2 月上旬至 3 月上旬的独立地址数，数据显示，买家人数超过 300 人的 NFT 应用仅 17 家，考虑多个地址归属于同一个买家的情况下，每天有 10 个及以上真实买家的应用数量仅在 10 至 15 个左右。这一数据显示了当前 NFT 持有地址间的悬殊发展现状。NFT 「出圈」更多的是表现在概念上，而非真正参与其中的持有者上。

（数据来源：nonfungible.com ） 

不过，一些平台正在对用户操作体验上进行改善，OpenSea 及 KnownOrigin 等 NFT 平台就参考了传统意义上网购平台的交互方式，在艺术品旁加上「输入出价」和「立即购买」功能。

除了闲鱼这样「被动式」开启 NFT 交易流转的平台之外，海外一些大型平台正在主动介入 NFT 交易。

5 月 12 日，电商巨头 eBay 北美市场高级副总裁兼总经理 Jordan Sweetnam 公开表示，将允许在其平台上出售数字收藏品 NFT，包括有关 NFT 交易卡、图片或视频片段等数字收藏品。eBay 将在短期之内向符合 eBay 标准的卖方提供 NFT 产品清单。但有外媒信息称，目前 eBay 上在售的 NFT 基于 WAX 区块链，且只能以美元进行支付。

风险与成长

在 NFT 交易现状没有出现更好的解决办法之前，NFT 的交易门槛似乎让众多名人效应所带来的新投资者有所流失。闲鱼及淘宝上的 NFT 成交案例告诉我们，市场对于 NFT 的交易需求，或许比我们预计的还要多。

但另一方面，由于虚拟物品交易安全风险，在闲鱼这类平台进行 NFT 交易的风险仍然存在。 目前 NFT 尚处于发展阶段，一方面是机会与蓝海，另一方面则是需要解决的关键技术和安全问题。

在 Blocklike 所观察到的一例发生纠纷的 NFT 案例中，买家对卖家重复上架已售出 NFT 的操作提出不满。另一个案例中，有买家认为自己买到了「盗版」的 Cryptopunk。而 NFT 市场上，平台如何分辨及把控上传作品的原创性，是一直以来保守争议的一个话题。

倘若你想买一个正版的 Cryptopunk，目前最便宜的价位也在 15 ETH 左右，但如果去 NFT 交易平台多找找看，那么一个「差不多」的高仿版本可能仅需十几美元。

海外的一位加密艺术家、NFT 用户 @Neitherconfirm 曾表示，进入到 opensea 之后，即使该 JPG 在拍卖状态或者该 JPG 本不属于他，但用户仍可以去改变这张图。

某种程度上讲，NFT 的「唯一性」尚无法强调其所对应的内容必定唯一，NFT 作为一种收藏品的数字证书和真实性证明，但现实中大部分情况下的实用性也饱受争议。此外，由于大部分数字作品仍储存在链下服务器中，其所有权也一直被社区进行讨论。

海外 NFT 社区整理出了给投资者在购买 NFT 之前的几个建议，主要有：

1、在购买 NFT 之前，检查艺术品的来源，考量其真实性、脆弱性和存在时间。 
2、对 NFT 进行验证，搜索其创建者，并检查其智能合约是否已存在于 Etherscan 上。 
3、提前了解 NFT 可能面临的技术或金融风险。 
4、如果 NFT 的资产并未存储在区块链和/或 IPFS 上，则不能确保购买者真正意义上的「拥有」。 
5、倘若使用链下存储，则 NFT 的所有权与项目团队息息相关。

NFT 以区块链技术作为基础，但 NFT 所带来的优越性并非绝对的。随着 NFT 的破圈效应快速蔓延，用户数量的增加和交易需求或许也走在了前面， NFT 交易方式仍充满着机会与挑战。
