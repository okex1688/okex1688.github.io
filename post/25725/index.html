如果今年年初，有人走到我面前说，NFT的销售额将轻松超过10亿美元，GaryVee将推出NFT项目，Axie Infinity 成为前五名 NFT 市场之一，我会回答：“我会相信三个中的一个。”但是，上个季度（21 年第二季度）前五大收藏品的收入超过 7.5 亿美元，Axie 的市场产生了 770 万美元的费用，而 GaryVee 已经从他的 NFT 项目中赚取了超过 100 万美元的版税。让我们看看具体的情况。

NFT是一场泡沫吗？

媒体叙述的关于NFT市场已经死亡或泡沫已经破裂的说法，客观上是错误的。 媒体主要关注加密艺术和收藏品（例如 NBA Top Shot），但 NFT 生态系统仍然在继续增长。事实上，就 NFT 销量而言，OpenSea 在 6 月份创下了有史以来最好的销售量记录。

Rarible 的糟糕表现（相对于 OpenSea来说）在很大程度上可以归因于市场没有提供对过去几个月蓬勃发展的一些 NFT 类别的访问，包括 NFT 头像、艺术块、体育收藏品和虚拟世界。

OpenSea 展示了煤矿中的金丝雀 NFT 生态的增长。6 月，OpenSea 为近 40,000 名活跃交易者中提供了超过 211,000 笔 NFT 销售。

总体而言，6 月是 OpenSea 有史以来 NFT 交易量最好的一个月 ，NFT 销售量创历史新高，也是活跃交易者数量第二好的月份。

虽然 OpenSea 仍然是整体市场的领导者，但 Axie Infinity 在整个季度飙升，成为四大 NFT 市场。值得注意的是，Atomic Market——WAX 上 NFT 的首选市场——现在在 NFT 市场中排名第 8。

Axie Infinity 的增长是在 Ronin 的第 2 阶段迁移之后实现的，Ronin 是其与以太坊相关的侧链，可提供更低的 gas 费用和更无缝的用户体验。虽然 Axie 可以在其他平台上进行交易，但97% 的 Axie 交易量发生在 Axie 市场上，该市场将费用（以 AXS 和 ETH 形式）交给 Axie 金库。

Axie 金库现在的价值已经超过 2100 万美元，其中 85%（1800 万美元）是在上个季度通过繁育和市场费用累积的 。Axie Infinity 已经与其他加密游戏不可相提并论，并且它在继续上升。它的下一个重要里程碑将是在第三季度推出 AXS 质押，以及在第四季度推出其虚拟世界 Lunacia 的 alpha版本。

Avatars：不是最后的NFT

=========================

也许上个季度最重要的趋势之一是 NFT 头像项目的延续。NFT Avatars——专门为个人在互联网上代表自己而设计的收藏品（例如 CryptoPunks）——继续流行，因为新兴行业在整个 21 年第二季度产生了近 3.5 亿美元的二级销售额，并在 2021 年上半年产生了近 6 亿美元。

神奇的数字：10,000

=====================

正如 DeFi 的夏天见证了每一个成功的 DeFi 协议基于食物的分叉，像 CryptoPunks 这样的 NFT 蓝筹项目正在经历类似的分叉。

这些推出的新项目大多具有CryptoPunk风格特征 ，包括数量上限约10,000个NFT，每个NFT具有不同稀有度的独特属性，以及专注于宣传有关其化身项目未来价值的社区。

这些推出的项目包括：

Meebits – 由 Larva Labs（CryptoPunks 的创造者）设计的 20,000 个独特的 3D 角色，旨在成为虚拟世界的头像。现有的 CryptoPunks 和 Autoglyphs 所有者也可以获得免费的 Meebit。

Bored Ape Yacht Club – Bored Ape Yacht Club (BYAC) 是一个由 10,000 个独特的 Bored Ape NFT 组成的集合，可以访问“Swap Club”，这是一个仅限 Bored Ape 会员的团体。

Bored Ape Kennel Club – Man’s Ape 最好的朋友。与 Meebits 类似，每个 BAYC 都提供了一只免费的狗供“领养”，我猜因为每个猿都需要一只狗。

Wicked Craniums – 10,762 个 NFT的集合，其中每个 Cranium 都是您进入“The Cradle”（一个仅限 Cranium 成员的团体）的门票。

Bulls on The Block – BAYC 的早期社区成员，他们希望为二次销售创建一个钱包，以便为社区积累。有 10,000 只与 BAYC 类似的独特公牛。

My Fking Pickle – 10,000个泡黄瓜

Slumdog Billionaires——因为加密的每个方面都需要 10,000 只狗。

Larva Labs 团队对他们的 Meebits 项目非常有想法。由于 Meebits 可由 Punks 或 Glyph 持有者认领，Meebits 为现有的 Larva Labs 社区提供了附加值。这创造了 Meebits 与 CryptoPunks 以及朋克价值叙事相关的感知，而不是从一个较新的、不相关的 NFT 贬低他们现有的收藏品的感知。此外，Bored Apes 在 21 年第二季度的销售额明显超过 6000 万美元，目前在 NFT 化身排行榜上排名第三（排在 Punks 和 Meebits 之后）。

然而，这些 NFT 头像项目中的大多数可能会沦为与 Hashmasks 相同的命运 ，Hashmasks 是一个一度著名的 NFT 项目，自 2 月推出以来一直停滞不前，在上个季度仅产生了 200 万美元的二次销售。

NFT 头像的一个更大的问题是，这些社区中的大多数都相对较小，并且由少数个人拥有。Meebits 拥有最大的所有者基础，尽管与大多数其他 NFT 头像项目相比，它们的 NFT 数量是其两倍（20,000 vs 10,000）。尽管如此，所有这些项目都非常集中，像 Gary Vaynerchuk这样的富豪透露，他拥有 52 个 Punks 和 54 个 Meebits。

最终，集中的所有权有助于推动稀缺性叙述，但除非个人出售，否则会减少社区的增长。与奢侈品行业类似，买家数量有限。此外，随着其他类型的知识产权推出 NFT 头像（想想蝙蝠侠 NFT），对加密货币原生头像的需求可能会减少。Genies——Flow 上的虚拟化身 NFT——最近在 5 月份筹集了 6500 万美元的 B 轮融资，并与包括Rihanna、Shawn Mendes和Cardi B在内的多位名人建立了合作伙伴关系。随着各自的明星向他们的忠实粉丝出售NFT，这些类型的NFT很可能会吸引大量关注。

NFT Avatar是一个新兴领域，但最终将面临幂律分布，一些将有很高价值，而大多数的价值将大大降低。 在加密原生头像项目中，CryptoPunks 已经建立，并且可能会继续成为加密中 NFT 蓝筹股的标准，而其他加密原生头像项目由于名人和 IP 风格化身项目的涌入而难以扩大采用。

Web3的采用已达到临界点

======================

人们对Web3的普及期待已久，但由于需要计算、索引、数据管理、托管、存储和其他重要服务的基础设施，因此未能充分体现。然而，经过多年的构建和持续增长，许多 Web3 协议开始进入其发展阶段。

多年来一直在不断构建的中间件层现在正在经历产品与市场的契合。Livepeer是一个视频基础设施供应商和流媒体应用市场，每周处理数百万个视频并产生数千协议费用收入。

Graph 正式迁移到它的主网，并在那里部署了几个协议的子图，现在每周产生数千美元的费用。

 来源：Web3Index

Ocean Protocol 的数据市场（允许个人发行数据集作为平衡器流动性池）在 419 个数据集池中拥有超过 330 万美元的 TVL。

文件和数据存储网络也继续被采用：

存储在 Arweave 上的数据又增加了 1.8 TB（1,800 GB）

Sia 上使用的存储容量达到了 1.09 PB（1 PB = 1,000 TB）的历史新高，总容量为 3.25 PB。

Filecoin 上存储的总数据现在超过 23 PB。

在其他地方，像 Handshake 这样的 Web3 协议的使用正在增加，目前有近 150 万个注册的 Namebase 域名和超过 100,000 个 Handshake 域名在使用。Helium 的网络每周增加数千个节点，导致花费了 1470 亿个数据点数（相当于约 3,500 GB），总净收入略高于 140 万美元（主要用于添加新热点）。

尽管如此，虽然 Web3 的许多独立组件都已准备好迎接它们的突破时刻 ，但整个技术栈尚未找到与 DeFi、NFT 或其他 Web3 协议进行互操作的方法。Web3 不再是遥远未来的星辰，技术栈正在聚集在一起，而星星们正在快速对齐。

名人、影响者和品牌 NFT

======================

害怕它，逃离它，但命运依旧会降临。

随着像 GaryVee 这样的名人深入研究加密货币，其他名人和影响者的进入风险就降低了。

Gary Vaynerchuk，通常被称为 GaryVee，现在正通过他的新 NFT 项目——VeeFriends——进入加密货币行业。自 5 月初推出以来，这个双 NFT 项目和品牌货币化实验在本季度取得了超过 1100 万美元的二次销售额。

此外，像 Jay-Z 这样的知名艺术家也正在进入加密货币领域，他使用 CryptoPunk 作为他的 Twitter 头像。更重要的是，这位说唱届的巨星进行了“Heir to the Throne”的拍卖，他的首张专辑的NFT收藏品最终以 $139,000 拍出。

历来难以将其作品货币化（通过音乐会除外）的音乐家已成为 NFT 实验的先行者。Yellow Heart（一个 NFT 票务市场）已经与 Maroon 5 和 The Kings of Leon 一起开发和发行了 NFT 收藏品，这些收藏品还带有额外的福利，比如前排座位、参加试听会、与专辑配套的肖像等等。

如果个人通过 NFT 实现品牌货币化，那么大型知名品牌也不会滞后太久。Twitter 最近在 Rarible上推出了“Twitter 140 系列”，这是一套包含 Twitter 历史和流行文化的 NFT。

第二季度收官

===============

有时，一个季度过去的时候，几乎没有人注意得到。其他时候，一个季度会出现突破趋势。对于Web3来说，本季度处于中间位置。Web3 部门经历了一些震荡增长并达到了一部分里程碑，但总体呈稳步上升趋势。

在 NFT 中，从大众消费者的角度来看，该季度的表现较为平静，但从分析的角度来看则不然。 艺术品销售下降，但 NFT 投机者转向更绿色的牧场（就像所有投机者一样）。尽管如此，NFT 的格局仍在继续扩大，扩展到尚未探索的各种新领域，尽管这意味着我们必须让名人进入 OpenSea 大门。下个季度会发生什么是任何人难以预料的，但如果这些趋势继续下去，对于那些关心 Web3 和 NFT 采用的人来说，今年下半年将是值得期待的。
