就在上个月，BAYC (Bored Ape Yacht Club）正式推出了属于无聊猿的“猿宇宙”代币 ApeCoin，链茶馆也对其进行了介绍。通过收购 Crptopunks、Meebits 以及与 Animoca Brands 合作进行新游戏开发，BAYC 向外界传达了他的雄心壮志。

BAYC 和 Crptopunks 两个在 NFT 领域具有统治地位的顶级项目合并，究竟会爆发出什么样的火花，截至今天，BAYC 官方除了精致的宣传片外还未给出答案，毕竟才过一个月。但民间已经已经给出了第一个答案：“相亲、结婚、生孩子”。

Rich Baby 的 Rich 爹妈

Rich Baby 是三月底推出的在以太坊上的 NFT 项目，字面意思是富有的宝宝，而富有的宝宝则需要富有的爸爸和妈妈。没错，每对个 Rich Baby 都需要由一个 BAYC 和一个 CryptoPunks 配对“生出来”。

既然是父母的娃，每对生出来的 Rich Baby 都会直接遗传他爹妈的特征，一个给 BAYC 家长，一个给 CryptoPunks 家长，两个娃大概率长得不一样，但有小概率生出几乎一模一样的双胞胎。

比如下面这张全家福，Rich Baby 就继承了 CryptoPunks 的肤色和 BAYC 的胡子。一个学他爹 BAYC，闭着眼睛学穿红衣，一个学他妈 CryptoPunks 睁着眼睛戴着灰帽学。

Rich Baby 的发行分为三个阶段，目前处于阶段一：只有拥有 BAYC 或 CryptoPunks 的用户可以配对铸造，属于“真正“的 Rich Baby，毕竟 BAYC 和 CryptoPunks 的地板价都来到快 500 ETH。

第一阶段发行的 Rich Baby 都带有独特的金奶嘴元素，并且都不需要铸造费用，只需要用户负担以太坊的 Gas 费。第二阶段依旧发生在公开铸造之前，团队将会在未来公布可以免费铸造 Rich Baby 的 NFT 项目。第三阶段是公开铸造，有钱即可。

Rich Baby 的社交属性

NFT 从根本上就是寻共识，而 BAYC 和 CryptoPunks 两款 NFT 在这点上已经做到了。Rich Baby 通过撮合两者来为自己的项目带来流量与热度，可以说是非常聪明。而免费铸造也表现了项目的诚意与格局，毕竟拥有这两种 NFT 的人群不大会在意他的价格。

一时间，Twitter 上出现很多了为自己的 BAYC 或者 CryptoPunks 找媳妇、找老公的推文，Rich Baby 社区甚至出现了相亲角，来方便双方配对，毕竟铸造 Rich Baby，爹妈缺一不可。

如果你是花钱买来的 Rich Baby，官方还提供了溯源找爸妈的专区，用户可以在这里输入 ID 来找到自己的 NFT 父母、兄弟姐妹以及全家福。但买别人 NFT 的儿女，这个操作确实有点魔幻。

Rich Baby 价值何在

热度高的 NFT 项目往往都不可避免的会出现相应的仿盘，但 Rich Baby 从创意到质量都属于仿盘中不可多得的项目。尤其是通过配对即可免费铸造的方式，为自己赚取了不少流量。

截至本文发稿，已有 1101 个 Rich Baby“呱呱坠地“，总量是 10000。而交易数据上，Rich Baby 地板价在 0.421 ETH，交易额 241 ETH。

得益于其高贵的出生，他的地板价在新项目里算比较高，但交易量并不大，一方面可能怕被占便宜？花钱买个 NFT 当别人 NFT 的娃确实有点哭笑不得。

但更重要的一方面是需要回 NFT 的核心，也就是共识，Rich Baby 在这点上很难想到能做什么。两个兄弟或者姐妹 Rich Baby 以共同的爹妈为共识确实太离谱。

此外，就目前来看项目方并没有路线图来表明未来发展方向，只是在官网提到想以此来建设一个社区，但 Discord 热度和推特关注量并不高。

除了一张质量不错的全家福和 Rich Baby 的 Jpg，我唯一能想到的一点作用就是相亲：

想铸造 Rich Baby，爹妈缺一不可，也就是相亲的双方；而爹妈还都需要是拥有极其昂贵的 NFT 用户，这就为相亲双方从资产、价值观两方面做了筛选；每个 BAYC 或 Punk 都仅能生成一次 Rich Baby，这表明相亲对象此前没有与他人有过 Rich Baby。
