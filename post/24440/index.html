Oscar Civit在成为NFT艺术家之前是一名西班牙的插画师。他的作品在几个同类型的NFT平台上都有很好的销售记录。当他想走进亚洲市场的市场的时候，他选择了CryptoArt.Ai这个平台渠道。

在短短一段时间内，Oscar已经在亚洲范围内斩获了属于自己的藏家。我很久没有看着这么有潜力的艺术家，他的作品包涵着激情，对于色调和光影的运用让人感叹。当然有人会说Oscar的作品在技术层面过于简单，而我想在现在艺术早已经超越了技术层面的约束，它探索人类灵魂深处真正关心的问题，精神自由，这是人性之中最根本的东西。用艺术为载体来传达一个人对生命的理解和价值观，也正是这代艺术家要做的事情。

1.你能向喜欢你的藏家介绍自己吗？

Civit：我是一名插画家和视频艺术家，喜欢尝试使用各种技术，我最喜欢的主题是未来主义、科幻小说和超现实主义题材。

 CCTV2

2.在成为 NFT 艺术家之前，你是一名插画家。你能向我们解释一下你的艺术作品中蕴含的哲学吗？

Civit：是的，在这之前我已经是一名插画师了……我的哲学有点酸……我喜欢黑色幽默，我喜欢表现21世纪人类丑陋的事物或矛盾……批评我们所有人。

Mode 77

3.你最喜欢的工作是什么？你有最喜欢的 NFT 艺术家吗？为什么？

Civit：我最喜欢的艺术家是莫比乌斯，目前我喜欢的艺术家有好几个……Baka、Javier Arrés、Fuckrender、Mattia Cuttini、Robness、Entter……每个人风格不同，但他们都给我带来了很多灵感。

Nightcrawler

4.你认为 NFT 艺术是一场革命吗？你认为他是一个反传统的艺术流派吗？

Civit：当然，这是一场伟大的革命。多亏了它，艺术家可以成为他们事业的真正所有者，而不受中介的限制。此外，它是一个世界舞台……我可以在中国，美国或日本出售我的作品……不仅仅是让西班牙的观者看到……另外，我不认为这是“反传统”，所有的艺术运动都在与先例决裂。NfT艺术不遵循传统艺术市场的规则这一点，我认为……很棒！

Model700

5.以后有什么打算吗？你会考虑举办个人艺术展吗？

Civit：我的计划是每天继续画画……是的，考虑举办个人展览总是很有趣，但目前我没有这个计划。

 pl4st3R

6.你如何理解那些质疑 NFT 艺术市场的人？你如何定义 NFT 艺术？

Civit：有些人害怕变化，并受到那些墨守成规的人的影响。NFT 艺术市场有一些可以完善的东西……但它的本质是好的。Nft艺术是数字世界中的艺术。

7.NFT 艺术给你的生活带来的最大改变是什么？

Civit：独立！
