以太坊上的价值抽取：简介与项目

DeFi 领域之所以能焕发如此强劲的生命力和创新力，主要原因之一是人们可以在以太坊上编写灵活的智能合约，并创建具有可组合性的货币积木。

然而，由于组合这些积木块所带来的复杂性，区块链矿工在创建区块时有机会通过剔除交易、打包交易或调整交易排序来抽取价值。这就是所谓的 MEV（矿工可抽取价值）。将来，随着跨链交易增多，复杂性只会越来越高。

为了更好地理解 MEV，我们来举个简单的例子：在 Uniswap 等自动做市商平台上，一笔大单交易通常会导致流动性池价格失衡，从而产生套利机会。假如你发现了这种机会，发送了一笔 gas 价格为 80 Gwei 的套利交易。这时，套利机器人会注意到你的交易，并发送一笔同样的交易，但是将 gas 价格设成了 120 Gwei，让它的交易抢先你的一步执行。最后，你要么就是以更高的价格买入，要么就是完全错过套利机会。这类 MEV活动很常见，被称为抢跑交易。

- 呈指数级增长的 MEV（来源：Flashbots Dashboard） -

根据 Flashbots 的数据追踪面板所示，MEV 已累计达 5.59 亿美元。随着 DeFi 领域的交易量增加，MEV 有可能成为未来最大的叙事之一。如果得不到解决，MEV 将造成更大的问题。在本文中，我们将介绍一些致力于解决 MEV 问题的项目。

1. Flashbots

Flashbots 是一个开源研发组织。他们致力于构建各种工具来让所有人都能分析并获取 MEV。从理论上来说，凡是需要具备大量智能合约知识才能理解的复杂交易环境，往往都存在 MEV。Flashbots 从以下三个方面降低了复杂性：

  1. 提供工具来探究并量化 MEV 的影响 —— 这有助于降低信息的不对称性，让人们更好地理解 MEV 问题及其影响。（MEV-Inspect）

  2. 实现 MEV 抽取民主化 —— 鉴于抽取 MEV 的权力集中在少数人手中，Flashbots 向所有人提供可以抽取 MEV 的工具。（MEV-Geth）

  3. 重新分配 MEV 收益 —— 目前，矿工和验证者是 MEV 的最大受益者。因此，在所有参与者之间重新分配收益非常重要。

只要是关于 MEV 的讨论都会提到 Flashbots。Flashbots 在吸引大众关注 MEV 问题上发挥了重要作用，未来将会继续如此。

2. KeeperDAO($ROOK)

若要利用去中心化交易所和衍生品平台上的套利和清算机会，你需要有足够的资源持续寻找这些机会，并在时机到来时拥有足够的资金把握住这些机会。KeeperDAO 专门为套利者创建了一个资金池：任何人都能向这个资金池借钱套利，并获得套利收益的一部分。

KeeperDAO 生态系统内有两类利益相关者：管理者和流动性提供者。管理者采取策略把握套利和清算机会，流动性提供者为这些机会提供资金。

我们举个例子来理解 KeeperDAO 的原理。假设你注意到了一个套利机会，你可以从 KeeperDAO 的资金池中获得一笔零息闪电贷。你拿了这笔贷款，执行套利/再平衡交易。一旦套利成功，本金和收益就会归还给协议。协议会以 $ROOK 代币（KeeperDAO 的原生代币）的形式向你发放收益分成。

闪电贷是一类特殊的零质押即时贷款，贷款会在同一笔交易内归还给资金池。但是，闪电贷安全吗 ？在大多数情况下是安全的。借出的资金必须在同一笔交易内归还给资金池，如果贷款者不遵守这个规则，就拿不到贷款。我之所以强调是在 “大多数情况下”，是因为任何 DeFi 应用都存在智能合约风险。如果你是传统金融行业出身，那么这一切对你来说可能有点荒谬。如果你想要了解更多关于闪电贷的信息，可以观看这个视频。

3. Secret Network($SCRT)

像以太坊这样的区块链是公开的，这就意味着任何人都能在 etherscan 上查看所有以太坊交易。如果有人碰巧知道了你的地址，就可以轻而易举地查看你的所有交易。Secret Network 通过赋予智能合约隐私性解决了这些问题。这类私密智能合约被称为 Secret 合约。基于 Secret Network （采用其原生代币标准 SNIP-20）构建的平台可以加密所有交易信息，从而提供完整的隐私性。

- Secretswap.io -

听起来是不是很棒！但是，这对解决 MEV 问题有什么帮助？ 以基于Secret Network 的平台 Secret Swap 为例。Secret Swap 是一个跨链自动做市商。然而，不同于其它自动做市商，Secret Swap 平台上所有交易的输入、输出和状态都经过加密，避免向外界泄漏任何交易信息。这样可以有效防止矿工根据交易费对区块内的交易进行重新排序，还可以防范来自做恶者的抢跑攻击。

Secret Network 的生态系统内有两种代币 $SCRT 和 $SEFI。在这个网络中，交易的 gas 费是以 $SCRT 代币的形式支付的，用来激励网络验证者。$SEFI 是治理代币，其持有者可以通过创建和票决提议来决定 Secret Network 生态的发展方向。Secret Network 还提供跨链桥，支持多种资产在其生态中流通。总的来说，Secret Network 有着宏大的愿景，旨在让所有人都能在其基础上构建应用。我很期待看到 Secret Network 生态未来将如何发展。

4. ArcherDAO($ARCH)

Secret Network 和 KeeperDAO 都致力于根治 MEV 问题。ArcherDAO 则反其道而行之，帮助矿工获得无害的 MEV。通过提供无害于交易者的 MEV 交易，ArcherDAO 实现了正和博弈，让矿工在不牺牲交易者的前提下获利。ArcherDAO 建立在现有的 Flashbots 交易再排序系统之上，并对其进行了优化，让矿工在套利的同时不伤害交易者。矿工只需花费 5 分钟即可集成 ArcherDAO，利用可持续 MEV 来提升自己的收益。

EIP 1559 一旦实行，将极大改变生态系统中的交易方式。在以 ETH 支付的交易费中，base fee 部分将被销毁以减少 ETH 的供应量，tip （小费）部分则支付给矿工作为报酬 。通过在链上识别 MEV，ArcherDAO 为矿工提供了一个可持续的无风险收入来源，未来将发挥更重要的作用。

- Archer Swap -

ArcherSwap 是 ArcherDAO 产品的 demo，可以让用户使用来自 Uniswap/Sushiswap 的流动性，执行零 gas 费、零滑点的套利交易，并向矿工支付少量 tip 。ArcherDAO 希望能尽快与 Sushiswap 和 1Inch 展开合作。像 ArcherDAO 这样的项目很有趣，因为它们兼容 KeeperDAO 生态的管理者策略。最近，Vitalik Buterin 对SHIB 和 ELON 币发动了大屠杀，在第一笔交易失败后，他使用 ArcherDAO 抛售了 ELON 币（还有什么是比 Vitalik Buterin 本人亲自用过更靠谱的代言呢！）ArcherDAO 的未来发展值得我们关注。
