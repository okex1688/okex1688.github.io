Web 3.0的未来之路

“因为稀有才是珍贵的，而水是最便宜的，但最好的。”

——柏拉图，《欧西德摩斯》

30多年前，第一个网页是在欧洲核子研究中心所创建的，由Tim Berners-Lee爵士和其他科学家想象构建。

他们当时对其的畅想：_网络应该是一个供所有人使用的开放标准系统。_Web 2.0将互联网从共享信息过渡到社交媒体时代，允许用户创建、共享和协作，而无需网页设计或技术技能。

回顾一下到目前为止的网络迭代。在Web 3.0中，一切都与昂贵的商业服务和电子商务有关，只有强者才能生存。与模拟时代一样，信息仍然从一个流向多个。

在Web 2.0时代，在Web上创建服务变得非常容易且成本低廉，社交网络也开始腾飞。博客是Web 2.0的第一次迭代，Twitter是第一个最新出来的社交网络。

Web3.0不会去任何地方，这是未来

但现在在线世界不再是通过PC查看服务器上的一系列文件。它是一系列设备和对象。对我们来说，“在线”经济包括不时掉在门垫上的棕色亚马逊包裹、iPod…….，而是存储在那里一生的CD收藏、Kindle等等。

而关于Web 3.0将会是什么，有很多关于未来的研究。有一种观点认为，Web 3.0是一种设备驱动的版本，而不是PC/浏览器驱动的Web 2.0版本——在这种版本中，围绕信息设置商业围栏的能力最终成为社会可接受的用户，因为他们不再能够免费获得他们想要的东西，需要支付费用才能获得。

Web 3.0 的一个更激进的愿景是蒂姆·伯纳斯·李（Tim Berners Lee）的“语义网络”想法，计算机能够从现有信息中获得如此多的意义，以至于它们可以开始代表人类相互交互。这将是一个无缝的信息网络，既是社会构建的又是机械分类的。

当然，这也被描述为元宇宙1.0——一个信息和人类生活开始通过设备无缝交互的世界。詹姆斯·卡梅隆的电影《阿凡达》捕捉到了这一愿景。

方法

在不断变化的社会中考虑信息通信技术时，我们当然也必须问——从什么角度来看？

Web 3.0，一个不再局限于浏览器或even到屏幕的网络——是一个多设备，多渠道和多方向信息吞吐量世界中的网络，涉及传感器和许多其他我们从未见过的设备。这所代表的变化是巨大的。

Web 3.0的谷歌，你将不得不付费

Web 3.0是一个网络，其中ICT显然是围绕着我们，我们的信息，我们的需求以及实时的信息：一些人开始将其称之为“流"”的网络。我们希望认证思考这个最新的趋势，Web 3.0，并提出一个关于它可能意味着什么的建议，以及可能需要理解它的观点。

互联网领域——通常被描述为大数据——将Web 3.0定义为“集成到或增强大规模Web应用程序的语义Web技术”。

这与我们认为对Web 3.0的更准确描述的物联网定义非常不同。许多其他人在对现象学的理解中呼应了这种广泛的技术重点。与Web 2.0一样，除了使用社交网络理论之外，大多数解决这一现象的论文很少使用任何真正的理论视角来接近它。

相比之下，在（较小的）第二类中，Fuchs等人专注于社会政治后果，将Web 3.0称为合作网络——可以说与Web 2.0作为读写网络的描述没有太大区别。对于其中许多人和其他作者来说，Web2.0被广泛认为是一种“受商业修辞深刻影响的文化结构”，与Web 3.0将大致相同。

在第三类（甚至更小）中，一些人似乎同意“一个新的在线环境”的定义，它将整合用户生成的数据以创造新的意义。与Web 2.0相比，Web 2.0被理解为基于用户的参与，而Web 3.0将基于用户的协作。

FAANG的大佬们不得不采用一些Web3技术

但他们仍然批评了“诸如Web 1.0之类的概念，2.0和3.0可以成为社会分析的可行和成功的理论模型”，而实际上它们本身就是“文化结构”。

当然，还有一类人对Web 3.0概念的看法是，这是一个有用的区别类，就像区分主要是农业经济和工业经济一样，尽管现在总是两者的混合，而且就网络而言，这种差异更多的是社会性的而不是技术性的。

这种文化结构，尽管它们具有历史和理论的偶然性——以及区域的特殊性——最终是我们唯一的窗口，而忽视它们就是否定所有理论。

对“Web开发的理解”，倾向于线性发展到共存的社会和技术趋势，这是有问题的。政治经济学和新自由主义的新Web应用程序的话语不能被希望消失，互联网时代资本主义的逻辑确实是快节奏的，并且伴随着庞大的人口。

文化、背景和商业——Web3.0时代重新审视品牌增长的三位一体

是的，Web 2.0和Web 3.0的概念通常带有用户实践的假设：Web 2.0被视为使用户参与，而Web 3.0被视为触发用户合作。

技术创新和系统与使用有关，以及我们拥有的东西如何参与并融入我们的日常活动中。

#uksnow事件

在许多人看来，Web 3.0比迄今为止许多文章中设想的要复杂得多。以几年前#uksnow事件为例。

意见领袖保罗·克拉克在他的博客上沉思了整个寒冷的早晨，他以英国下雪的自然现象设想了“#uksnow”应用程序，如此，众包数据会发生什么。几个小时后，敏锐的程序员和意见观察者BenMarsh创建了Twitter-GoogleMaps混搭所需的代码，#uksnow可供博客圈使用。

在博客上发表评论的聚合网站，并在Twitter上突出显示“趋势”主题，发现了这种混搭的存在，并且，就像博客圈一样，受欢迎程度助长了人气增长。

按照Ben Marsh网站上的说明，成千上万的人使用微博网站Twitter，在推特上发布了两条简单的信息：他们当前位置的英国后码的前三到四位数字，以及他们所在位置的雪的沉重程度的粗略测量，作为五分之一的标记，例如“BL7 2/5”。

#uksnow地图程序，在此APP上的内容可以分享到推特上，并带有标签

人们向#uksnow提供了这些信息，在Twitter上“标签”（例如，他们在推特上发布了“#uksnow BL7 2/5”）。这些推文，它们是在全国范围内发布的，创造了所谓的Twitter流。这样流可以使用简单的搜索工具捕获，并且可以由Web应用程序显示或用于其他目的。

来自远程收集的流#uksnow标签提供了数据，将五张不同大小的雪花图片中的一张放到英国的GoogleMap上，从而在http://www.benmarsh.co.uk/snow/创建了英国的实时雪地地图。

很明显，人们因为跟风潮流站在外面拍摄下雪图片，并使用他们的移动互联网设备提供所需的推文。这很容易从Twitpic照片的流行中推断出来，这些照片的简短网址伴随着推文中的定位和下雪数据。

到第一天结束时，微软已经创建了一个克隆程序。随着降雪的继续，该应用程序又存活了几天，但是在最初的36小时内，该应用程序获得了大量推文和用户访问。

随手拍的下雪照片，推特上的推文《下雪的纽伯里❄❄❄️5/10 #uksnow》

当时，棒球世界大赛在美国同时举行，但就Twitter流量而言，由于推特上关于（和#uksnow）的人数而短暂黯然失色。

这个#uksnow事件就是一个很好的例子，我们在这里将其描述为Web 3.0。

借用和扩展Orlikowski的脚手架概念，物理基础设施很容易被识别为移动互联网，包括支持互联网的移动设备本身，在每个小区内广播和接收信号的信号杆，以及托管和路由所涉及的数百万个数字文件的服务器群。

该技术支架与穿过不列颠群岛的雪云气象支架，降水根据大气条件和地形高程而变化。这种组合的是社交网络的文化现象，以及使用最小截断（140个字符）版本的（160个字符）短消息服务标准的微博技术，盐稀缺的政治局势，痴迷于灾难的媒体以及英国人对天气的古老迷恋。

由此产生的“混搭”在狭义上作为基于Web的应用程序，在广义上作为不同情况的特定事件，代表了一种受益于Orlikowski脚手架社会物质的理解的情况。

来自Netflix的“社会困境”的剧照

试图从技术中心或纯粹以人为中心的角度来理解和解释这种现象，都会错过现代生活中这种相互关联的方面。

此外，使#uksnow成为可能的脚手架展示了新兴的文化和社会实践如何结合在一起，不仅对天气，而且对移动互联网技术脚手架中新固有的可能性和微博的潜力。

一旦雪融化，表演就结束，就会被遗忘和丢弃。它还代表了所涉及的技术及其直接迎合的文化痴迷的一系列极其灵活的潜在影响。

后结构主义思想

过去，在万维网出现之前，计算系统是小的、离散的、短的和可控的。

自从信息通信技术问世以来，随着Web 1.0、Web 2.0和现在的Web3.0的出现，重要的是要问：我们的“系统”概念是否足够广泛？

复杂性的洞见所隐含的这种对系统的重新想象的核心哲学含义是，正如我们过去所设想的那样，系统的安全、清晰的完整性和边界开始消失。

Web 3.0承诺了一个去中心化且更加互联的网络空间

在一些文献中通常讨论的简单计算信息系统之外，扩展了我们对系统的理解，20世纪结构主义思想的核心见解之一是：在19世纪及之前，我们以那些东西为中心。

事实上，“系统”的广泛概念并不是中心，而是由系统本身决定的

1960年代及其后的后结构主义思想进一步揭示了这种结构本身往往甚至不是“系统性的”：在社会历史背景下，开放系统是自我组织和自我决定的，并且不断变化，以至于任何系统定义一旦被提出就变得多余。

当然，大多数信息技术专家都熟悉的系统最基本的、传统的定义是，它由一个具有边界、内部和外部的综合整体组成。信息系统同样可以被定义为个人和组织用来创建、收集和处理数据的硬件和软件的集成系统。

然而，正如解释主义的研究人员会立即提出，信息系统不仅被人和组织使用，而且该系统可能被认为包括那些人与组织，在变化和约束的复杂相互关系中，因此，系统作为具有内部，外部和边界的集成整体的概念本身正在受到这些新的复杂理解的挑战。

Cryptopanties艺术/时尚作品的创作旨在弥合物理世界和数字世界之间的差距

我们认为，Web 3.0既是这些实用的复杂系统的典范，也是理解它们所需的哲学转变的范式。

例如，为了理解——正如第一类评论家（由亨德勒所阐述的那样）Web 3.0中发生的大部分事情的本质，作为一个传统的“信息系统”，比如，就像使用土壤科学一样——尽管对于研究土壤本身是有用和准确的——试图了解整个农业。

这种方法是忽略所有丰富的附加材料，例如种子的不同种类和等级，动物和鸟类的注意力，天气的变化以及为管理土壤而投入使用的各种不同机器。

此外，这种方法甚至没有开始考虑人类农业社区和农业经济使用这些机器，采购和种植的整个历史、文化、区域和跨国复杂性。

因此，正如土壤科学家必须与种子科学家合作，灌溉技术人员与农业设备制造商合作，最后与农民合作一样，农民必须与种子批发商和他/她的产品市场变幻莫测地合作，所以信息技术专家必须明白，被设想为“系统”在信息技术专家中，实际上要复杂得多，也更为偶然。

飞鱼俱乐部：“世界上第一个会员唯一的私人餐饮俱乐部，会员资格在区块链上作为NFT购买”

必须开发理论工具，不仅将信息通信技术（ICT）视为开放，而且简而言之，将其视为复杂 系统，并且不仅在纳入一系列人为因素和其他因素的意义上开放，而不是信息技术本身，而是在持续时间意义上开放。

简而言之，在归类为Web 3.0的东西中，大部分发生的事情都是涌现的，正如考夫曼等复杂性理论家所使用的，对他们来说，一个复杂的整体可以驱逐集体属性，即本身就是合法的“涌现”特征。

所以，Web 3.0道路并不平坦和光明。由分布式账本概念源起，区块链技术正在为人类踏上Web3.0星球迈出左脚，而距离真正实现美好的数字未来还有多远，乐观者说5年，先行者则走了更久。

而毛球科技作为国内本土最早的区块链技术企业之一，一直在探索未知的Web 3.0的道路上步履不停，相信在未来，毛球科技可以凭借强大的区块链技术，支撑数据互通的Web3.0发展需求，希望能在Web3.0世界看到更加公平、透明，更加便利的场景。
