NFT 的价格非常不稳定。牛市中，很多人都会在交易中亏损，熊市中更是如此。

随着交易量的增长，越来越多的名人入场，越来越多的 VC 也开始投资 NFT 项目，NFT 正迅速成为一种另类投资。

虽然人们对 NFT 的价值和可持续性议论纷纷，但很多名人确实相信选择前景好的 NFT 项目是一门深奥的学问。

即使某些 NFT 系列很吸引大家的眼球，但它仍然需要一些指标来帮助评估项目的潜力及价值所在。

以 Ethereum 上的 NFT 为例，了解分析 NFT 的 5 个指标

1、项目团队背景

团队的背景和经验对 NFT 项目的价值至关重要。

什么样的团队对项目有利？

不妨看看 Bored Ape Yacht Club 和 Otherdeed 背后的团队。Yuga Labs 就是一个完美的例子。该团队成立于 2021 年 2 月，创建了 BAYC，又在 2022 年 3 月收购了 CryptoPunks 和 Meebits。后来，他们发行了 ApeCoin，为 Otherside 项目做铺垫，推动了 Otherside 项目的发展。

熊市当前的 3 月，该团队完成了 4.5 亿美元的一轮融资，估值达 40 亿美元。

如果想找到有潜力的 NFT 项目，项目背后的团队应该：

  • 发行过成功的 NFT 项目

  • 明确的长期增长计划

  • 在熊市中保持估值和获得融资的能力

这些都是影响 NFT 项目价值的重要驱动因素。

另一个符合上述标准的团队是创建 Azuki 的团队 Chiru Labs。它们拥有大型传统科技公司的背景，如 Google 和 Facebook，这是找到有潜力 MFT 的另一个标准。

2、稀有性

稀有性是最基本的数据指标之一，与项目的内在质量、效用或美学价值完全分开。

基本上，供应和需求表明，NFT 越稀有，其价值越高。

在 NFT 世界中，大约 8,500 个为一个系列的 NFT 的供应量被认为是非常小的。也有一些项目，如 Meebits，有高达 20,000 个。

CryptoPunks 是以太坊上最早的 NFT 项目，截至 6 月 17 日 Punk 的地板价为 48.9 ETH，总共只有 9,999 个 Punks。为了让更多人接触到 NFT，Larva Labs – CryptoPunks 背后的团队推出了 Meebits，一个拥有 20,000 个 NFT 的 3D 体素集合。Meebits 的底价比 CryptoPunks 低 15 倍。

稀有性在 NFT 中也很重要，包括独特的特征、属性以及它们的组合。虽然没有硬性的规则来评估 NFT 项目的稀有性，但学习和理解 NFT 稀有性对项目的影响是很重要的。

3、交易量与流动性

交易量是衡量市场对 NFT 需求的重要指标。较低的交易量通常是熊市到来的信号，而较高的交易量意味着它的流动性风险较小。

随着 NFT 的快速发展，众多项目如雨后春笋般出现。虽然许多 NFT 项目都有着令人惊叹的独特设计，但市场已经饱和，并不是每个新系列都能成为下一个 CryptoPunks 或 CryptoKitties。

许多在 Ethereum 上发行的 NFT 都处于 0 交易状态。这表明很多 NFT 都缺乏流动性，这是 NFT 行业发展的阻碍之一。

4、价格墙

通过价格墙可以看到一个项目在什么价格上可能会出现上行阻力，也可以从该系列的 NFT 价格来判断其上升空间。

如果用户无法判断 NFT 的买卖价格，可在二级交易市场查看项目 NFT 集合的价格。例如，Azuki 有 10,000 个流通单位，只有 5 个单位低于10 ETH 标记，最高的 NFT 为 9,999 ETH。这表明 Azuki 的价格是有很大的溢价上升空间。

5、社区力量

社区的力量是决定一个项目成功与否的最关键因素之一。

简而言之，社交媒体是否活跃可以衡量一个项目的受欢迎程度。

例如，让我们通过社交媒体活动来比较两个大型的、趋势性的 NFT 项目。 首先要看的地方是 Discord 和 Twitter，它们是 NFT 领域最常见的沟通渠道。当你登陆项目的网站、Discord 或 Twitter 时，你应该看看项目的粉丝数量、参与度与粉丝比例、是否通过赠品等策略人为提高参与度、官方和用户之间的互动等等。

以下通过官方 Twitter 数据对 Otherside 和 Goblins 的社区力量进行的分析。

  • 从粉丝数量来看，Otherside 的粉丝数量是 Goblins 的 11 倍。这意味着Otherside 比 Goblins 更受用户欢迎和关注。

  • 从官方与用户互动来看，在发布话题方面，Otherside 仍然比 Goblins 更活跃。
