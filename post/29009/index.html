这些 NFT 衍生物的主要来源是铸造者所持的 NFT 头像，以 0N1、Cool Cats、Pudgy Penguins、无聊猿（及变异猿）、World Of Women 等热门 NFT 系列为主。

在海外社区，这些 NFT「配件」受到了加密爱好者的广泛支持，很多 NFT 头像的持有者在积极铸造属于自己的 NFT 衍生物，其中一部分衍生物已经被放在 Opensea 上进行交易，产生收益。

NFT 头像衍生物似乎正在为市场带来新的乐趣。

用你的 NFT 铸造加密「配件」

这些 NFT 二次创作产品的发展速度也非常夸张，根据 nft mint alert 的统计，在过去的 1 个小时，10KTF 已经铸造了 1042 件产品。

上文所展示的大部分 NFT 头像衍生运动鞋就来自 10KTF 平台。该平台以配件制作工匠「Wagmi San」的形象作为展示，为 NFT 持有者们铸造新的衍生产品。

在用户这位工匠的简单交互中，可以直接看到钱包地址中所持 NFT 能够进行生成的运动鞋样式，并选择是否付费铸造。

该平台提供了一个简单明了的铸造逻辑：10KTF 构建了 ERC-721 兼容智能合约，用户可通过已持有的 NFT 来铸造衍生 NFT。在链接用户的钱包之后，拥有某一 NFT 头像商业权利的持有人，可以选择铸造相对应的 High Top（高帮运动鞋）系列运动鞋。

（* ERC-721 是目前主流的 NFT 头像及游戏所采用的智能合约通用标准，CryptoKitties   加密猫 CryptoKitties、CryptoPunks 加密朋克等头部项目就采用了这种合约。）

10KTF 的智能合约将确保仅 NFT 的当前所有者才能创建衍生 NFT，并且每个对应的父系 NFT 只能铸造一个 NFT。以目前社区中所分享的衍生鞋 Ape High Tops 系列为例，仅有 Bored Ape（无聊猿）的所有人才能够铸造这些 Ape High Top，且每个 Bored Ape 只能铸造一个，Bored Ape 与 Ape High Top 之间是一一对应的。

在以太坊上铸造这些衍生物运动鞋的价格是 0.05 ETH 以及 Gas 费用，此外，运动鞋设计的 PNG 版本可免费下载。这对于大部分 NFT 持有者来讲，成本并不算高。

短短几周，10KTF 的 Discord 社区中已经吸引了 3600 多位感兴趣的 NFT 持有者。目前，已有 2400 名 NFT 持有者制作出了 NFT 衍生运动鞋并于 Opensea 上售卖，这些产品的数量超过 5500 件，平均售价在 0.3 ETH 左右。在过去的 7 天里，10KTF 共售出超过 1700 件产品，总销售额达 561.28 ETH。

而这些 NFT 头像衍生运动鞋的交易量正在缓慢增加，受到了很多人的认可和欢迎。

（数据来源：icy.tools）

根据公开信息，这些衍生运动鞋目前仅为数字化产品，没有对标相应的实体，这些产品是否能在数字虚拟世界中佩戴仍然未知。

不过，已经有热情的投资者自发地寻找将这种鞋子实体化的方法。在推特上，有加密爱好者正试图通过 Vans 网站上传图片定制来获取心仪的鞋子。

运动鞋：Crypto 最受欢迎的切入点之一

对于 NFT 爱好者来说，运动鞋是一个易于鉴赏的领域。因此，无论加密市场历经过几轮牛熊，运动鞋始终是 NFT 领域的探索方向之一。

2019 年，美国运动服饰巨头 Nike 推出一项专利「CryptoKicks」，在这项专利中，区块链技术将 Crypto 的概念与数字资产与实体运动鞋联系到了一起：当消费者购入一双「CryptoKicks」加密鞋时，同时也将收到基于该运动鞋而生成的 Token（NFT），这枚代币将记录该运动鞋的所有信息，该 NFT 用于认证和交易实体鞋。

此外，CryptoKicks 还设想了一种双币模型，每个实体鞋还将对应一个用于访问、培育和交易数字鞋的代币，模仿加密猫（CryptoKitties）的模式，「繁衍」新鞋型。

能够看到，「CryptoKicks」的主要思路仍在于加密货币对于实体运动鞋的对应。

2021 年初，加密社区迎来了 DeFi、NFT 概念的爆发期，虚拟服饰的热度也随之增高，数字运动鞋逐渐受到市场的关注。由 RTFKT Studios 设计「虚拟鞋」就是极具代表性的项目之一， RTFKT 推出的独一无二的运动鞋在设计上极具想象力，并且能够让用户在游戏中穿戴，彰显个性。

这时的数字运动鞋已经开始脱离对「实体运动鞋」的限制，想着「NFT 和数字艺术」的方向发展，增强了用户佩戴数字运动鞋的体验，为虚拟世界中的社交来服务。

目前，这类产品也被视为基于区块链的「元宇宙」概念中重要又有趣的组成部分，也是 GameFi 背景下值得期待的功能性「皮肤」，对于这些概念来讲，NFT 是不可或缺的。

相比之下，10KTF 似乎找到了另一个切入口，它正在现有规模不断扩大的、已经持有 NFT 头像类产品的加密用户进行服务。 随着头像类 NFT 产品的数量大规模增加，NFT 爱好者们需要挖掘所持有 NFT 的品牌、IP、创造价值、商业价值等新的的玩法，这些「周边配件」恰好符合了这种情况。

不过，除了铸造这些链上的运动鞋，NFT 爱好者们仍然期望该平台的下一步发展：比如推出 NFT 头像的其他配件或周边、在虚拟世界中可穿戴、可通过版税等方式赚取收益等等。

NFT 头像衍生品再续热度？

这种 NFT 衍生物制作能给市场带来新的热度吗？

在加密社区，主流的评价 NFT 价值的方式里，将 NFT 头像的价值主要分为三个部分：内在价值（Intrinsic value）、 使用价值（utility value）、社区溢价（community premium）。

内在价值主要包含上市价格(mint price)，这些往往是初创团队及艺术家获取的收益。比如，前段时间曾风靡一时的 NBA 球星卡一个卡包的上市价格为 9 美金。

在使用价值中，主要描述了 NFT 的实用价值或附加权益，例如，持有足球俱乐部粉丝代币的人们可以获取该俱乐部的一些决策权；或者持有门票类 NFT 的所有者可参加演唱会等等。

社区溢价主要针对二级市场由于 NFT 稀有性或其他特性给出的溢价。CryptoPunk 头像就是很好的案例之一，这些头像在加密社区的高度认可及追捧下，该头像的持有者之间能够相互认同，这种社交价值也直接推升了该系列的价值。

大部分 NFT 头像都适用于这个评价体系。有趣的是，铸造 NFT 头像衍生运动鞋系列中的主力军无聊猿，就曾以周边产品的影响力带来了巨大的影响力。

无聊猿是一个「周边产品」的有趣案例，现在这些无聊猿的交易人数已超过 8000 人，相比 CryptoPunk 的 4800 余次交易，更多的持有者展示出了更高的社区活力。

这与去中心化的社区是分不开的，早些时间，这些头像在 Twitter 上一度形成了「无聊猿头像互粉」默认规则，能够为为社交账号吸引大量粉丝，这加速了该系列头像在社交平台上的快速扩散。

随着持有人数量快速增加， 无聊猿的「Meme」现象在制作周边这一环节上尤为突出，各种周边商品在社交平台上被分享，包括无聊猿 T 恤、无聊猿咖啡、无聊猿面膜、无聊猿衍生 NFT 等等，在传播过程中扮演了相当重要的角色。

这就不难理解 NFT 「周边产品」所散发的吸引力与创新的生命力了。在目前的 Opensea 上，除了 10KTF，在基于已有 NFT 产品推出「NFT 周边」、「NFT 配件」、「NFT 图片衍生物」的平台及产品正在变多。

这些周边产品往往基于某一 NFT 系列进行二次创造， 由于目前 NFT 的持有者社区数以百计。NFT 的持有们出现了自发的创新现象，自制相关游戏、制作周边产品等等。在这样的「二次创作」中，NFT 持有者的创造力再度被激发，为回馈社区带来了新的方式。在 NFT 铸造并被售出之后，持有人及社区成为了市场拓展的新力量。

通过这些 NFT 周边平台，NFT 头像被加入了更多附加价值和新生玩法，并为铸造者带来新的收益，展示出了新的生命力。

2021 上半年，NFT 头像市场在短短数月创造出巨大的财富效应，头像 NFT 的制作和销售热度高涨，并有很多知名人士、加密 KOL、品牌 IP 参与进来，逐渐形成更加完整和丰富的 NFT 生态。

目前，头像 NFT 的「数字身份」正在变得重要，代表着持有者对于社交和身份价值的多重需求，这种 NFT 头像二次创作方式的出现，或将大面积激发加密社区的创新热情。
