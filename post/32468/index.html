对于加密货币投资者来说，今年已经成为名副其实的“NFT 年”。其实 NFT 的概念已经有几年的历史了，只是相对来说最近才正式“出圈”。这同时也进一步促进了 NFT 相关项目的快速增长。年末已至，《比推》团队将仔细解析这一现象级浪潮，并重温 2021 年最火热的，也是最赚钱的十个 NFT 项目。

首先，什么是 NFT ？

非同质化代币（NFT）是区块链中的加密资产，具有独一无二的识别代码和元数据，因此可以将它们彼此区分开来。与普通加密货币不同，它们不能在同等价值的基础上进行交易或交换。这使它们区别于同质化代币，后者相互之间是价值等价的，因此可以作为商业交易的媒介。

NFT是目前最有潜力在现实世界中得到广泛应用的加密技术之一。例如，它们是房地产和艺术品等实物资产数字化的理想载体。由于 NFT 是基于区块链存在的，NFT 也可以用来消除非同质化商品交易过程中的中介机构，以及加强内容创作者和他们的粉丝（消费者）社区之间纽带，让内容创作者能更好得从自己的内容创作中获益。

目前的NFT市场大部分是围绕着数字艺术、卡片和稀有物品等收藏品展开叙事。一个典型的例子就是NBA Top Shot平台，它以 NFT 的形式收集大量与 NBA 有关的独特时刻。其中一些卡片甚至已经卖出了数百万美元的价格。

NFT另一个有趣的用途是记录现代互联网和加密市场的里程碑式事件。Twitter首席执行官Jack Dorsey曾在推特上发布了一个有史以来第一条推文的代币化版本的链接。其NFT版本以250万美元售出。

像实物货币一样，加密货币是同质化的，这意味着它们有良好的流动性（便于买卖），且易于交换。例如，一个比特币永远都等于另一个比特币的价值。同样地，一个以太坊也总是等于另一个以太坊的价值。这种可互换性的特点使得加密货币天生就适合作为数字经济中的流动性工具。

而 NFT 通过使每个代币都独一无二、不可替代，创造了一个新的模式。此外，在某些情况下，NFT 可以相互结合，创造出具有其他独特标识的新数字对象。这为其游戏化提供了大量的可能性。

大多数基于以太坊网络的NFT是ERC-721代币。其代码与ERC-20代币代码不同，后者通常用于创建可互换的基于以太坊网络的代币。主要区别在于 ERC-721代币是独一无二的。其中最重要的技术参数是tokenId，通过它就可以用来区分不同的 NFT。

为什么NFT如此受欢迎？

ERC-721标准是在2017年9月发明的，但“独一无二的数字对象”这种概念，至少在几年前就已经在加密社区中广泛讨论过了。2018年，CryptoKitties项目引发了第一波肉眼可见的 NFT热潮。这是一款早期链游，用户在其中交换以“猫”为主题的相关 NFT。而它的主要特点是能够通过组合和创造新的代币来 “繁殖 “虚拟宠物。

CryptoKitties在当时掀起了一阵“养猫”狂热，以至于以太坊区块链上所有交易的11 %以上是通过该项目的NFT进行的，而且在交易高峰期还造成了多次网络拥堵，其热度可见一斑。最有趣的是，CryptoKitties并不是第一个NFT项目，首个 NFT 项目的头衔属于CryptoPunk，它在交易量方面几乎碾压了 CryptoKitties，尽管CryptoPunk真正的爆发期是2021年 。那么， NFT 能像病毒一样蔓延开来的原因是什么？在这里，它有几个绝对优势值得强调下。

  • 独一无二：持有 NFT 就和持有现实世界的某些藏品一样，从代码角度来看它仅此一个。

  • 不可篡改：关于NFT的数据会被记录在区块链上，不能被删除或篡改。

  • 稀缺性：NFT代币的集合有相对较少的实例（例如，CryptoPunk系列只有10000个 NFT）。

  • 数字化：由于NFT，几乎任何实物都可以通过它，与区块链中的信息绑定，让其具有额外的价值。

2021年，NFT的优势极大地吸引了大量投资者的关注。正因为如此，在短短几个月的时间内，与 NFT 有关的项目也出现了前所未有的回报。《比推》将带您纵览2021 “NFT年”。

2021年十大NFT项目

下面我们来看看最受欢迎的一系列 NFT、交易所平台和其他与新趋势相关的项目。排名不分先后，按编者创作随机排列。

NBA Top Shot

NBA Top Shot是由加拿大的区块链公司Dapper Labs与NBA合作创建的一个平台，允许用户以NFT 的形式收藏 NBA 中的一些最佳时刻。要想获得所谓的 “moments”（时刻），用户可以购买相应的套装，或者更简单一些，直接在二级市场上购买由玩家开出的“时刻”。

这些 “时刻 “有不同程度的稀有性，其价值取决于以下规律：

  • Normal（普通，代表着有超过 1000 份数字拷贝的时刻）：它们可以普通套装的形式购买，例如价格为 9 美元的 9 个时刻。

  • Rare （稀有，150-999份数字拷贝）：这些时刻可以在稀有套装中找到，价格从22美元起，一个有七个普通时刻和至少一个稀有时刻。

  • Legend（传奇，25-999份数字拷贝）：这些可以在传奇套装中找到，价格为230美元起。

  • Platinum Ultimate（白金终极版，3份数字拷贝）：仅通过拍卖获取。

  • Genesis Ultimate（创世纪终极版，只有1份数字拷贝）：仅通过拍卖获取。

自成立以来短短两年左右的时间里，NBA Top Sho的活跃用户数量已经突破了100万大关。同时，一年中篮球比赛的独特时刻的交易量超过了10亿美元的水平。许多来自NBA系列的代币自年初以来价值增长了几十倍，并且其价格还在随着NFT的普及而继续增长。

CryptoPunk

2017年，Larva Labs 创始人Matt Hall 和 John Watkinson创建了一个程序，生成了数千种不同的像素图片。起初，他们认为自己创造了一个智能手机应用程序或游戏的雏形。但这个简单的项目最终为数字艺术市场开辟了一条崭新的路，并首次在加密领域挑战了 “所有权 “的概念。

CryptoPunk是一个24×24的像素艺术风格图像集合。总共有整整1万个代币，每个代币都有自己的个性和独特的、随机生成的特征组合。有6,039名男性 “CryptoPunk"和3,840名女性 “CryptoPunk”。他们根据不同的特征被区分开来，如头发颜色、是否有配件等等。还有八个没有任何显著特征的朋克，有时他们也被称为Genesis punk（创世朋克）。

Larva Labs的网站上有一张所有10,000份 “CryptoPunk “的合成图片。每个 NFT 也有自己的详情页面，其中详细介绍其特点和全部交易历史。任何人都可以保存该图像文件的副本，但只有一个人可以拥有对应的NFT，因为CryptoPunk的每个 NFT 所有权是在以太坊区块链代码中制定的。

Matt Hall 和 John Watkinson创造的 CryptPunk 有时被比作拥有一件永久借给公共博物馆的实物艺术品。因此CryptoPunks作为NFT世界的先驱，当之无愧地获得了相当高的热度，以及与之匹配的价值。

Rarible

Rarible是一个基于以太坊的 NFT 交易平台。尽管Rarible是作为一个中心化的平台推出的，但其创建者计划随着时间的推移逐渐将其去中心化，最终将控制权移交给一个去中心化的自治组织（DAO），由Rarible社区管理。Rarible计划通过其RARI治理代币来实现这一目标，该代币使用户有权参与平台的管理。2021年，Rarible宣布将推出Flow，这是一个来自Dapper Labs（CryptoKitties开发团队）的快速、去中心化且对开发者很友好的区块链（公链）。旨在托管去中心化的应用程序并实现快速交易。

要使用Rarible，用户需要一个以太坊网络的钱包，如Metamask、Argent或Coinbase，以及一些ETH来支付佣金。用户可以用信用卡直接在平台上购买ETH。链接钱包之后，用户可以浏览NFT Rarible交易平台，寻找自己青睐的 NFT 。该平台将 NFT 分为图片、游戏和meme等不同类别供投资者检索。

此外，针对艺术家或创作者群体，他们可以使用Rarible来直接创建自己的NFT。首次创造NFT时，Rarible 用户必须支付两笔费用–一笔是其钱包与Rarible互动的权限费用，另一笔则是 “调用铸币功能"的费用，简而言之它会将创作者的文件放入平台间文件系统（IPFS）和以太坊区块链中。

这两项费用都是网络费，即与区块链互动的成本，本身并不算平台收入。在首次授权之后，用户再想创建新的 NFT 将只需要支付gas费即可，而该费用会根据以太坊网络的负载情况而变化。当创作者决定出售其NFT时，Rarible Art交易平台允许创作者从后续的销售中赚取一定的 NFT交易费用，让创作者真正的受益于自己的作品。

OpenSea

OpenSea 是另一个去中心化的交易平台，它允许用户购买和出售 NFT 。OpenSea成立于2017年，最初主要用于CryptoKitties NFT的交易，但后来服务范围逐渐扩大。现在的 OpenSea支持相当庞杂的数字资产类型，其中包括艺术品、域名、游戏项目、音乐等的NFT。

OpenSea是建立在以太坊区块链上的，所以用户也必须有一个以太坊钱包才能使用该应用。OpenSea也支持其他几个区块链，包括Polygon和Klaytn。用户可以选择241种支付方式，其中也包括稳定币，所以几乎任何人都可以在平台上购买和出售数字物品。

Hashmasks

Hashmasks是由与Suum Cuique Labs 合作的70名艺术家团队创作的虚拟艺术品。每个 “Mask面具 “都有一个独特的特征，其中最引人注目的是肤色、性格和眼睛颜色。Hashmask与其他普通的NFT绘画不同，因为每个面具没有名字。代币的所有者负责为他们的Hashmask命名，从而参与到艺术品的制作过程中。

该项目的创作者已经制作了16,384个 NFT 和相关图像。这些面具在价值上也有巨大的差异 。这批 NFT 中最昂贵的一笔交易是以 420 枚 ETH（约合169万美元）出售的面具。

Enjin

ENJIN在2009年,即所谓的“古典互联网时代”注册于新加坡，成立之初推出了名为“ENJIN Network”的游戏社区平台。目前其全球用户数已经超过2000万人，旗下游戏社区超过25万个，其中包含了电竞团队、游戏服务器、粉丝网站和Cosplay社团等多种业务。

2017年，ENJIN正式入局加密领域，并旨在打造链游、元宇宙和 NFT 的基础设施。

Enjin币（ENJ）是该项目的原生代币。它有大约 8.35 亿个 ENJ 在流通，而最高限额为10亿。今年1月至4月期间，ENJ 出现了巨大的涨幅，其价格在短短几个月内从14美分上涨到3.46美元。

Enjin目前拥有着一个庞大的用户社区。该平台的成功是由于NFT和元宇宙的人气大幅上升。而这些领域将在下一阶段主导加密市场，因此该平台未来的需求前景将是不可限量的。

Flow

Flow是一个去中心化的生态系统和基于区块链的平台，具有可扩展的解决方案，用于交互式互动、去中心化应用项目和链游领域。Flow网络专注于提供一个可扩展的环境和架构，以实现无拥堵的快速交易，为NFT、dApps和游戏创造生长的沃土。Flow团队也参与了前文中提到的NBA Top Shot开发。

Flow在没有使用分片的前提下，解决了以太坊目前所面临的可扩展性问题，并且 Flow 网络也是基于 PoS （权益证明）共识机制。通过这种方式，Flow能确保整个网络平稳运行，并将dApps和游戏的网络拥堵降到最低。

Flow将网络上的工作分配给支持该系统的节点，不同的节点根据其技术能力被分配不同的任务。通过将非确定性进程（如将交易形成一个区块）与确定性进程（如执行该交易区块）分开，实现了区块链吞吐量的增加。

为了减少以太坊网络的负载，Flow使用了一个多角色的多节点架构。为了使整个网络的扩展能够以有效和经济的方式进行，Flow使用网络参与者的四种角色来分解区块验证过程，将不同的任务分配给不同类型的节点。

Decentraland

Decentraland也是一个基于以太坊网络的元宇宙项目，旨在打造一个全球性的元宇宙。Decentraland的用户可以通过在这个虚拟世界中探索、互动和玩游戏，还可以购买和出售数字房地产。随着时间的推移，该平台现已实现互动应用，内部支付，甚至是一个虚拟通信等多项业务。

Decentraland使用两种类型的原生代币。

  • LAND，这是一种 NFT，主要用于确定代表数字房地产的土地所有权。

  • MANA，这是其系统内的流通代币，便于购买LAND以及Decentraland中使用的虚拟商品和服务。

Axie Infinity

在《Axie Infinity》中，玩家通过购买、繁殖和与被称为Axies的怪物对战来赢得游戏中的SLP货币。这种商业模式被称为 “Play to Earn（玩赚）"，玩家通过玩虚拟游戏的时间来赚取真实奖励。

每个Axie都是一种 NFT 和可收集的物品，可以在游戏外的交易所购买或出售。玩家必须至少有三个Axie才能开始战斗，最便宜的Axie目前售价约为200美元。而最贵的Axie早在2020年就以 300 ETH的价格出售。

这款游戏在网络上的病毒式蔓延帮助开发工作室Sky Mavis在今年赚了数千万美元。该公司从游戏中的交易中获得了7亿美元的收入，其中 17 %由开发商拥有。预计到今年年底，其总流通资本将稳定在10亿美元以上。

该游戏开发商最近在B轮融资中筹集了1.52亿美元，由Andreessen Horowitz（a16z）领导，估值为30亿美元。由于该公司并不拥有Axie Infinity的所有代币，其估值低于其货币的市值。Sky Mavis表示，它将利用新的资本推出一个改进的战斗系统，一个去中心化的交易所，以及一个新的游戏。

The Sandbox

The Sandbox是一个基于以太坊网络的元宇宙和游戏生态系统，用户可以在一个虚拟世界中创造、交换和货币化资源和游戏体验。Pixowl的项目将颠覆游戏行业，在传统游戏行业中，平台拥有和控制用户生成的内容，创作者和玩家的权利受到限制。在沙盒中，用户完全拥有他们的虚拟物品。

该项目由三个关键部分组成。

  • VoxEdit：允许用户在元宇宙中创建和动画化3d物体。这些对象被称为资产，并使用ERC-1155代币标准，允许使用单一智能合约创建同质化和非同质化代币。

  • The Sandbox Market：这是一个市场，用户可以在首次上传他们的资产到InterPlanetary 文件系统（IPFS）后，在这里发布和出售他们的资产。

  • 沙盒游戏制作: 允许用户免费创建3D游戏。

虽然参加沙盒不需要任何编程知识，但其强大的工具包允许用户在快速增长的网络游戏和虚拟世界领域创造和嵌入有价值的资产。这就是促使其原生代币 SAND在2021年创下历史新高的主要原因。

总结

在NFT出现之前，虚拟世界中的物体仅仅因为其存在的事实而没有基本价值。换句话说，一个普通的计算机文件可以被复制许多次，而不会以任何方式改变其属性。随着NFT的出现，人们能够使虚拟内容货币化，这打造了新时代区块链网络的基础。因此，NFT 成为了数字资产行业的一个非常独立的领域。相信在不久的将来，NFT 将在新世界经济体系中扮演举足轻重的角色。
