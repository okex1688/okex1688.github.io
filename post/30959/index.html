金色财经报道，昨日，区块链技术公司ConsenSys表示，已经以32亿美元的估值从汇丰银行和美国对冲基金Third Point等投资者处筹集了2亿美元。其他投资者包括Marshall Wace、Think Investments、Dragonfly Capital、Electric Capital、Spartan Group、Coinbase Ventures、DeFiance Capital和Animoca Brands等。此次融资后，ConsenSys计划招募多达400名新员工以壮大团队。

ConsenSys在业内闻名已久，旗下有6大产品业务，都是整个区块链行业举足轻重的产品。今天我们将其6大产品资料整理如下，帮助大家了解ConsenSys。

6个产品一起被称为ConsenSys产品套件，分别为金融开发工具集Codefi、审计公司Diligence、以太坊接口服务Infura、去中心化钱包MetaMask、网络协议Quorum和Hyperleger Besu客户端、开发工具Truffle。

Codefi

ConsenSys Codefi于2019年9月推出，包含九个产品模块，这些模块协同工作，旨在通过领先的金融科技基础设施增加去中心化金融技术的使用。

其中，为了促进资产数字化和业务流程自动化的采用，Codefi 开发了五种机构产品：

使用 Codefi Assets 创建和管理数字资产。资产管理公司、私募基金和其他金融机构可以通过端到端的数字资产管理克服传统金融效率低下的问题。

使用 Codefi Markets 解锁数字资产流动性。交易所和代币化平台可以利用集成的价格信息聚合和场外交易来实现快速、安全和结构优化的点对点传输。

使用 Codefi Workflow 优化业务流程。大型公司可以跨多个组织以数字方式连接数据并进行业务运营。

使用 Codefi Compliance 的规定。加密货币交易所和以太坊项目可以利用可靠的 AML-CFT 解决方案来防止欺诈和金融犯罪。

使用 Codefi Orchestrate 加速企业区块链解决方案部署。企业可以通过行业验证的核心技术区块链组件降低研发成本并加快上市时间。

为了满足寻求去中心化金融机会的机构的需求，MetaMask 与Codefi 合作提供其市场模块：

使用 MetaMask for Institutions 交换、借入、借出、投资和与 DeFi 协议交互。加密货币基金、托管人和专业交易员可以利用内置于世界上最值得信赖的以太坊钱包中的企业级安全性、操作和报告功能。

许多中央银行正在探索零售或批发 CBDC 和跨境支付用例：

使用 Codefi Payments 发行和管理 CBDC 和稳定币。中央银行、金融机构和企业可以通过新的支付基础设施改善批发、零售和跨境支付。

以太坊项目需要原生的数字基础设施、扩展解决方案和支持增长和可持续运营的治理机制。

Codefi Activate 已经帮助促进 SKALE 的代币销售，部署 Eth2 存款启动板，构建 Filecoin 的存储平台和 DeFi 桥，并管理 AirSwap 的新治理模型。

使用 Codefi Activate 启动和扩展去中心化网络。去中心化网络可以建立具有活跃且受过良好教育的代币持有者社区的高性能网络。

另外，使用 Codefi Staking 在 Ethereum 2.0 上获得奖励。机构可以利用以太坊 2.0 的收入机会，而无需运行独立验证器的技术和操作复杂性。

Codefi正在通过支持去中心化网络和应用程序来改变这种状况，使每个人都能使用 DeFi 进行交易、借贷、投资和储蓄。

Codefi Markets 为MetaMask Swaps 提供支持，该功能可汇总来自整个 DeFi 生态系统的最佳定价数据，同时促进世界上最受欢迎的以太坊钱包中的高效点对点交易。使用 MetaMask Swaps，消费者可以在整个资产生命周期内保持对其资产的控制。

Codefi Activate 有助于发展去中心化网络，并使用户能够因参与网络治理和效用而获得奖励。通过 Activate，消费者可以质押 SKL 并扩展以太坊网络，将 FIL 变成 renFIL，质押AST 并对 AirSwap 治理进行投票。

最后，通过 Codefi Data，DeFi 用户可以检查 DeFi 协议风险并比较跨借贷平台的收益机会，以做出明智的财务决策。

使用 Codefi Data 评估 DeFi 协议安全性和数字资产性能。用户可以通过全面的数据和风险分析来评估 DeFi 协议的安全性和数字资产性能。

借助这些工具，让全世界的人们都能以中心化服务的可靠性、易用性和安全性访问去中心化金融服务。

Diligence

Diligence公示了如下的审计工具，包含但不仅限于这些。

MythX

MythX是一种安全分析服务，可以在开发过程中发现以太坊智能合约代码中的 Solidity漏洞。

Harvey

用于以太坊智能合约的灰盒模糊器。它集成了各种新技术，可以更快、更可靠地发现错误。

Mythril

Mythril是一个开源组件，是 EVM 字节码的安全分析工具。支持为以太坊、Quorum、Vechain、Roostock、Tron 和其他兼容 EVM 的区块链构建的智能合约。

Scribble

Scribble 是一种验证语言和运行时验证工具，可将高级规范转换为可靠代码。它允许您使用属性注释 Solidity 智能合约。

Solidity Visual Auditor

Solidity Visual Auditor 是一个 Visual Studio Code 扩展，旨在让智能合约审计员的工作更轻松。它为 Visual Studio Code 用户提供安全感知语法和语义突出显示、详细的类大纲和高级 Solidity 代码洞察。

Visual Studio Code

Visual Studio Code 的 Ethereum Vyper 语言支持和语法突出显示。此扩展提供安全增强装饰、通用语言结构的代码片段以及更多功能。

Surya

Surya 帮助审计员理解和可视化 Solidity 智能合约。它提供有关合约结构的信息并生成调用图和继承图。它还支持通过多种方式查询函数调用图，以帮助手动检查合约。

Karl

 Karl 是一个智能合约监视器，它使用 Mythril 检测引擎检查安全漏洞。它可用于实时监控以太坊区块链中新部署的易受攻击的智能合约。它通过在区块链的虚拟副本中运行候选合约来消除误报。

Theo

Theo 是一个具有类似 Metasploit 界面的开发工具，可将您带入 Python REPL 控制台，您可以在其中使用可用功能进行智能合约侦察、检查存储、运行漏洞利用或针对特定智能合约的前端或后端事务。

AraGraph

一种可视化 Aragon DAO 的权限关系和其他细节的工具。

Legions

方便的（安全）研究人员探索以太坊节点和合约的工具包，现在具有光滑的命令行界面，具有自动完成命令和历史记录。

SWC注册

智能合约弱点分类注册表是 EIP-1470 中提出的弱点分类方案的实现。它与通用弱点枚举 (CWE) 中使用的术语和结构松散一致，同时覆盖了特定于智能合约的各种弱点变体。

Infura

区块链产品可以将 Infura 集成到产品的整体战略中来使基础设施更具弹性和可靠性。Infura 在内部实现了反向代理模式，可以成为基础架构堆栈中高度可靠和高性能的节点。

可以选择将您的数据库作为主数据库，将 Infura 作为备份，反之亦然。通过这种方式，Infura 将充当您数据库的永远在线备份，提供单个数据库运行时无法提供的灵活性。

反向代理是通常位于基础设施边缘的服务器。它负责接收来自 Web 客户端的请求，然后将它们路由到一个或多个后端 Web 服务器。

使用反向代理服务器的主要原因之一是负载平衡，确保客户端和服务器之间的流量保持顺畅。这是同时使用您自己的节点和 Infura 的主要优势之一。此外，它还允许您缓存内容并提供 Web 加速。最后，反向代理允许更高效、更安全的 SSL 加密和免受 DDoS 攻击和相关的安全问题。

MetaMask

MetaMask 的创建是为了满足基于以太坊的安全和可用网站的需求。特别是帐户管理并将用户连接到区块链。MetaMask 允许用户以多种方式管理帐户及其密钥，包括硬件钱包。

11月18日，ConsenSys宣布，MetaMask月活跃用户达到了2100万。MetaMask运营主管Jacob Cantele表示，最近的大部分增长都是由垂直游戏的增长推动的。

Quorum

Quorum 是一个开源协议层，可为开发人员提供使区块链应用程序成功所需的可配置项目。使用 Quorum，您可以访问将区块链解决方案从概念到生产的所有必要组件。目前在这个部分包含四个项目，分别是Hyperledger Besu、GoQuorum、Tessera、EthSigner。

Hyperledger Besu 是在 Apache 2.0 许可下开发并用 Java 编写的开源以太坊客户端。它运行在以太坊公共网络、私有网络和 Rinkeby、Ropsten 和 Görli 等测试网络上。Besu 实施工作证明（Ethash）和权威证明（IBFT 2.0、Clique 和 QBFT）共识机制。

GoQuorum 是在 LGPL 许可下开发并用 Go 编写的开源以太坊客户端。GoQuorum 是一种基于以太坊的协议，运行私有的、许可的网络。GoQuorum 实施权威证明（IBFT、QBFT、Raft 和 Clique）共识机制。

Tessera 是在 Apache 2.0 许可下开发并用 Java 编写的开源私有事务管理器。Tessera 的主要应用是作为启用隐私的以太坊客户端（例如GoQuorum和 Hyperledger Besu ）的隐私管理器。

EthSigner 是一个开源的、客户端不可知的、以太坊交易签名者，在 Apache 2.0 许可下开发并用 Java 编写。EthSigner 通过使用可以在各种云提供商中保护或在本地磁盘上加密的私钥对交易进行签名，从而将私钥管理与交易验证分开。

Truffle

Truffle是使用以太坊虚拟机 (EVM) 的世界级开发环境、测试框架和区块链资产通道，旨在让开发人员更轻松。使用Truffle，您将获得：

内置智能合约编译、链接、部署和二进制管理。

用于快速开发的自动化合同测试。

可编写脚本、可扩展的部署和迁移框架。

用于部署到任意数量的公共和专用网络的网络管理。

使用 EthPM 和 NPM 进行包管理，使用ERC190标准。

用于直接合同通信的交互式控制台。

支持紧密集成的可配置构建管道。

在 Truffle 环境中执行脚本的外部脚本运行器。
