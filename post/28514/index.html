NFT，是在区块链上唯一和可验证的资产，一直是今年的热门话题，但我在与专家的交谈中得到的一个共同的看法是，它没有任何意义。这也是主流媒体眼中的一个常见的观点。例如，主流媒体报道的新闻通常是关于某个数字艺术家以数百万美元的价格出售 Jpg（图片）的新闻。这将使大部分人更难理解NFT 是什么，因为人们看到新闻后最直接反应是：NFT 市场是一个大泡沫。

而就在一周前，NFT 市场中又崛起了一个新星—Loot，Loot 在上线交易市场后的 24 小时内以超 1 万ETH 的成交额登顶销售榜，而许多圈内爱好者却不明白这个黑底白字的图片究竟是个什么？因为黑底白字是这个项目的标志，也因此，该项目被称作是TXT（文本），以与被戏称是“JPG”的 NFT 项目形成对照。“越看不懂的越值得购买，市场从JPG到TXT”，市场上有人总结道。

但是，如果有人愿意花这么多钱在这些图片上，那么它对某人来说是有价值的。我敢肯定，你已经看过一些新闻文章，其中有一张照片是关于一颗超级岩石以 130 万美元的价格出售的，一家无聊的游艇俱乐部(Ape Yacht Club)以同样的价格(400 ETH)出售 Ape，或者是 Visa 以 15 万美元的价格买了一张《CryptoPunk》。这些 NFT 大买卖的新闻正把越来越多的人带到加密/区块链行业，这正是我们希望看到的“出圈效应”。

以区块链和加密货币来说，改变我们持有和管理资金的方式是必要的。有些人认为，去中心化金融(DeFi)将成为加密货币被大规模采用的原因，但似乎只有 NFT 才能实现这一目标。NFT不仅仅是对 Jpg  图片的买卖，还有更多好玩的东西。

下面我将解释 Jpg 和 TXT 图片狂热背后的需求，并讨论 NFT 的其他用途。

OpenSea

在讨论 NFT 的不同使用方式之前，我想谈谈 OpenSea 交易额的疯狂增长。OpenSea 是迄今为止最大的 NFT 交易市场，自 2017 年成立以来，它的交易量呈指数级增长。上周，它的每周销售额达到10 亿美元，仅 8 月份的销售额就达到 29.3 亿美元。在他们的网站上，您可以按艺术、音乐、域名、虚拟世界、交易卡、收藏品、体育和实用程序来筛选适合你的 NFT。这只是显示了这个市场的广阔性和独特所有权的潜力。他们目前有 37 名员工管理所有这些交易量，并且迫切希望雇用更多人。

图片/头像的狂热

每天都有许多 NFT 项目在启动，试图从 Jpg 的故事中分一杯羹。这些项目中的大多数都依赖于社区来为其提供价值，但也有很多买家和项目试图快速赚钱。这意味着数百个项目归零，您最终会拥有价格过高的 Jpg。大多数项目都使用 CryptoPunks 作为蓝图，并发布 10,000 张通常随机生成的具有不同特征的独特图像。这些特征的稀有程度各不相同，为 NFT 的盲盒经济提供了“运气”这一个指数。你的特征越稀有，你的 NFT 就越昂贵。

CryptoPunks 在 2017 年免费提供它们，但今天大多数的持有者以 0.04-0.1 ETH（130-320 美元）的价格出售它们。

经过 4 年的沉淀，如今这个系列已经一种社交身份，象征着地位、文化趋同、财富地位。

那么为什么有些图片有这么大的价值呢？CryptoPunks 被认为是 2017 年创建的第一个 NFT。因此，它不仅仅是一张像素化的图片，而是一段加密历史，是对 NFT 市场走向的预测。CryptoPunks 实际上并不是许多人认为的第一个项目，但我认为它是第一个具有这种性质的项目。这就是为什么人们四处寻找这些 NFT 的原因，也是为什么一张图片能有这么离谱的价格的原因。这就是区块链的美妙之处，总有证据证明它最初是何时铸造的。

而目前较新的项目具有不同的吸引力，两个主要原因是影响力和社区。这与今天人们穿的昂贵的衣服非常相似。很多时候这些衣服甚至不好看，但每个人都知道它们要花很多钱（Balenciaga鞋）。人们愿意花很多钱来炫耀。随着这些照片价格的上涨，成为这些项目社区的一员变得越来越受追捧。通过将你的 Twitter 头像或个人资料图片设为 NFT 的图片，这就表明要么你很早就在玩 NFT ，要么表明你很富有。

独特的艺术和音乐

NFT 市场正在改变艺术家赚钱和在此过程中照顾他们的方式。纵观艺术品历史，曾有艺术家拍卖自己的作品，当他们成名后，他们的作品拥有者变得非常富有，但艺术家们却一无所有。NFT 正在改变这一点。例如，艺术家现在可以永久地为其每件艺术品的销售附加 5% 的折扣。他们可以决定金额，这将被纳入 NFT 的只能合约中，因此没有人可以对此提出异议。这种艺术永远无法伪造，因为区块链上有无可辩驳的证据。

数字艺术家因其出色的作品而受到美术机构的认可。拍卖行佳士得拍卖了艺术家 Beeple (Mike Winkelmann) 的一件作品，成交价为 69,346,250 美元。Beeple 的艺术品在 NFT 狂热开始时就引起了 NFT 市场的广泛关注。他是一位制作原创艺术品的数字艺术家，他曾每天创作一件作品，连续坚持创作了 5000 天（超过 13 年）。而在佳士得拍卖的作品是这 5000 件作品的拼贴画。Beeple 一夜之间成为了百万富翁。随后，他开始通过抽奖系统赠送个人作品，这会立即使他的粉丝赚到数千美元，而且他的很多作品都可以在创作共享中免费下载。于是，Beeple 被授予 GQ 年度艺人奖。

在音乐行业，有很多关于低薪音乐家和音乐公司剥削艺术家的新闻。而 NFT 也可以摆脱这一问题，并使音乐家能够直接向他们的粉丝销售。可能性是无限的，他们可以将活动门票作为 NFT 出售，以记录这一事件，他们可以将音乐专辑作为 NFT 发行，他们可以发布他们的曲目的歌词，只有一定数量的粉丝才能拥有，他们还可以与某些 NFT 持有者（粉丝）见面以达到销售 NFT 的效果。

在 NFT 的世界，艺术家可以做很多事情来与他们的粉丝建立更亲密的关系。这显然不仅限于音乐家，它使人们能够将更多的个人品牌 IP 货币化。

NFT的碎片化所有权

一些艺术品以数百万美元的价格出售，似乎普通人无法参与这些市场。事实并非如此，这些昂贵物品的所有者可以将他们价值数百万美元的 NFT 变成碎片化所有权的 NFT。他们可以决定要将其分成多少个部分，然后他们可以将这些部分作为自己的 NFT 出售。这意味着任何人都可以投资于这些艺术品，并围绕艺术品创建一个社区，并保持其价值。

Play to Earn

区块链游戏中的道具装备是在 NFT 领域对于被大规模采用最重要的一点。与 NFT 的其他一些用途相比，新用户的基础知识障碍非常低。这些用户可以在游戏中游玩，一些游戏玩家甚至已经在游戏内的道具购买上花费了数千美元。虽然 Fortnite 是一款免费游戏，但仅仅是 2019 年的道具购买费用已经让开发团队赚了 18 亿美元。现在想象一下，玩家可以拥有他们在游戏中购买或解锁的所有道具装备，如果认为它有价值，则可以出售以获取利润。这对玩家和游戏行业来说都是双赢的。大多数游戏的利润来自游戏的销售，仅此而已。现在，他们可以通过运行游戏内资产的市场来持续赚钱，并从交易市场中抽取一小部分的手续费用作为利润。

但是目前，虽然这些区块链游戏非常基础，但已经让人们赚了很多钱。玩家每天可以通过玩 Axie Infinity 赚取50 美元，这是一款可以繁殖宠物宠物、战斗和交易的游戏。Axie’s 的收藏品 NFT 在目前的链游市场中是最昂贵的，仅 2021 年 6 月的销售额就为 4200 万美元。Axie Infinity 的代币价格从 2021 年 6 月的 3 美元飙升至今天的70 美元左右，让很多早期的游戏玩家变得富有。

生活成本低的国家的人们玩这些游戏赚的钱比他们在正常工作中赚的钱多。这为缩小全球收入差距创造了机会。如果美国的某个玩家比第三世界国家的人更重视他们的时间，那么激励美国的人玩游戏需要更多的激励。一旦美国的玩家开始入场，而身处在第三世界国家的玩家早已经在玩了，并且随着游戏的流行，他的资产价值也会增加。

元宇宙（虚拟世界）

如果您看过电影《玩家一号》，那么您就会了解虚拟世界的全部内容。它基本上是一个数字世界，用户可以在其中控制化身在世界各地漫游并相互互动。直到现在虚拟世界还没有大规模流行的原因是所有权问题。一个没有所有权的虚拟世界并没有给人们足够的动力来继续回来。现在，随着所有权和社交模式相结合，人们纷纷涌向这些虚拟世界。

目前，不论是哪一个元宇宙项目，大部分虚拟土地已被挖出，你只能在二级市场上购买。但是，随着成千上万的用户涌入这些虚拟世界，那么这些虚拟土地的价值只会增加。这些虚拟世界的惊人之处在于，这正是它们的本质，“一个世界”。

你在这些虚拟世界上可以自由控制你所扮演的角色。你可以和其他人交谈，你可以在你的土地上建造任何你想要的东西，甚至可以举办虚拟活动和虚拟会议。

虽然目前元宇宙的仅仅处于概念化的阶段，但是已经有不少玩家率先感受到了它独特的“赚钱效应”。比如在虚拟世界 Decentraland 中销售和服的设计师 Hiroto Kai 。他在三周内赚了大约 20,000 美元，Decentraland 虚拟世界中的玩家已经可以穿这些和服了。每件衣服都是独一无二的，它们就相当于 NFT，因此可以将其出售给愿意支付价格的任何人。

在区块链的世界中，有各种各样的虚拟世界，每个世界涵盖的内容略有不同。目前在行业中领先的是Decentraland、Sandbox、Cryptovoxels 和 Somnium。虽然它们在图像质量、虚拟土地、用户数量和游戏内货币的价值方面有所不同，但它们的相似之处在于你可以在其中自由的创建与交互。我在者四个平台上能找到的最便宜的虚拟土地的价格大约是 1 ETH（±3300 美元）。

信任很容易破碎，但是区块链基于信任。

一般来说，区块链领域的神奇之处在于，一切都是基于信任。你拥有你的资金，你可以做你想做的任何事。有许多的竞争对手都提供类似的服务（例如DeFi），这迫使他们必须努力拓展创新、积极维护服务原则。而这恰恰也是推动整个区块链行业迈向更大创新的动力。
