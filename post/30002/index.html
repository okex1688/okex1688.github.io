DeFi热点

1. 数据：Terra链上DeFi协议总锁仓量突破100亿美元 创历史新高

10月22日消息，Defi Llama数据显示，Terra链上DeFi协议总锁仓量达100.8亿美元，创历史新高。目前，锁仓量排名前5位的公链分别为以太坊（1614.6亿美元）、BSC（204.9亿美元）、Solana（123.7亿美元）、Terra（100.8亿美元）、Avalanche（75.8亿美元）。

2. DeFi借贷协议Timeswap完成Timeswap领投的种子轮融资

10月22日消息，DeFi借贷协议Timeswap完成种子轮融资，MulticoinCapital领投，Mechanism Capital和Defiance Capital参投，该项目将于近期推出测试版本。Timeswap是一个受Uniswap AMM机制启发，采用三变量AMM机制的借贷协议，主要特点包括：无权限（允许任何人创建借贷池而无需投票），市场驱动的借贷利息和所需抵押品，固定期限资金池（可以创建固定期限的金融产品如债券和期权），单独的资金池（一个资金池的崩溃不会影响其他资金池）等。

3. DeFi总锁仓量突破1500亿美元，创历史新高

10月22日消息，据DeBank数据显示，全网DeFi总锁仓量超1500亿美元，截至昨日该数据为1519.1亿美元，创历史新高。当前锁仓金额排名前三的DeFi协议分别是PancakeSwap（245亿美元）、Maker（157亿美元）、Curve（150亿美元）。

4.DeFi 平台 EQIFI 推出加密万事达卡

金色财经报道，DeFi平台EQIFI宣布推出其全球安全的加密万事达卡。这些借记卡允许平台用户利用数字资产在店内和在线进行实时支付。EQIFI 得到了财富科技创新者 EQIBank 的支持，它是一家受监管的无国界数字银行，用于加密和传统资产，使 DeFi 平台的用户可以访问一系列银行产品和服务。 EQIFI 首席执行官 Brad Yasar 表示：“我们非常高兴将 EQIFI 的借记卡带给我们当前和未来的用户。这是将 DeFi 的优势与主流、传统银行和支付流程相结合的重要一步。拥有可以让我们的用户融入日常生活的有形有形产品，说明 DeFi 正在稳步改变金融实践。通过结合 DeFi、加密和传统支付轨道，金融的未来就在这里。

NFT热点

1. 腾讯NFT平台幻核与腾讯课堂联合发售霸符系列NFT，以庆祝1024程序员节

10月22日消息，为庆祝1024程序员节，腾讯NFT平台幻核与腾讯课堂联合发售霸符系列NFT。据悉，本次活动将发行5种NFT，每种类型500枚，共计2500枚。该系列NFT的存储全部基于至信链，且全部通过抽取形式发售。幻核方面表示，抽取到金霸符的用户可兑换限量版NFT，而集齐5款普通NFT的用户可获得定制T恤。

2. 数据：以太坊链上NFT销售总额突破80亿美元 创历史新高

金色财经报道，根据最新数据显示，以太坊链上NFT销售总额突破80亿美元，创下历史新高，本文撰写时为8,041,063,403美元，链上交易数量为3,575,620笔。此外，NFT链上销售总额突破120亿美元，目前为12,146,606,738美元，同样创下历史新高，其中以太坊链上NFT销售额最高，Ronin链上NFT销售额位居第二，为2,587,527,511美元。

3. NFT市场Kalamint与Wert合作，提供信用卡充值服务

10月22日消息，Tezos网络上第二大NFT市场Kalamint已经通过Wert实现了钱包充值。用户现在可以在Kalamint主页上用信用卡充值他们的XTZ钱包。在获得Draper Goren Holm基金和Amesten Capital投资后，Kalamint现已成为使用最多的十大NFT市场之一。

4. 综合 NFT 平台SuperFarm在Immutable X上推出 NFT 发布平台

10月22日消息，综合 NFT 市场平台SuperFarm宣布在 NFT 二层解决方案Immutable X上推出 NFT 发布平台（NFT Launchpad），并推出详细指南教用户如何发布自己的 NFT。SuperFarm是一个综合 NFT 市场平台，有 NFT 发布平台 （NFT Launchpad） 、NFT挖矿 （NFT Farming） ，还将推出 NFT 生成工具、NFT市场平台以及 NFT 社交平台。

5. 说唱明星Snoop Dogg与哈林花式篮球队合作出演 NFT 情景喜剧

10月22日消息，说唱明星Snoop Dogg宣布将与哈林花式篮球队（Harlem Globetrotters）合作出演NFT情景喜剧《Da Dogg Gone Gym》，讲述上世纪70年代的故事，Snoop Dogg扮演该篮球队的教练。该剧还伴随相关的限量版Snoop Dogg与哈林篮球队的NFT发售，通过网络门户网站wide.app独家提供。

6. NFT 游戏Parallel获Paradigm的5000万美元投资

10月22日消息，NFT 科幻卡牌游戏Parallel宣布以5亿美元估值获得Paradigm的5000万美元投资。Prarllel表示该项目之前的投资者包括YouTube联合创始人 Chad Hurley、Focus Labs、OSS Capital 和 Yunt Capital 等。Parallel 项目是围绕一个幻想故事情节展开的，讲述了人类在试图解决全球能源危机的世界末日之后逃离太空的故事。
