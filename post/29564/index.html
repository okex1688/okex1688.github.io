以太坊将作为全球结算层，或者从更加技术的角度来说，是全局安全和数据可用性层 。

那时的以太坊上有一个蓬勃发展的外部执行层生态系统 ，比如 Rollups 和 Volitions ，这里是所有用户和 dApps 的所在地 。

这些执行层不仅仅是“可扩展性的解决方案 ”，更是充满活力的社区 ，拥有自己的文化、身份、治理和经济。这是所有创新将发生的地方，而以太坊则蜕变为一个些许僵化的结算层 。这些不同的社区将相互交流、竞争，但也会为以太坊这个整体而携手并进。

如上图所示，有些执行层会选择成为基于以太坊的 Rollups ，完全由以太坊 L1 主网保护；有些可能会选择成为具有中心化的数据可用性的 Validiums ，其中一些 Validiums 有着半去中心化的数据可用性，也即拥有其自身的数据可用性共识机制；其他则会给用户选择。

还有些会选择作为拥有独立主权的侧链 ，这些侧链有着相对中心化的共识机制，无须向以太坊 L1 提交任何证明 (proofs)。这种情况下，它有可能是一条与以太坊无关的独立 L1 公链 (比如比特币)，或者实际上是一条对以太坊怀有敌意的以太坊侧链 (比如 BSC)，亦或者是一条对以太坊友好的以太坊侧链 (比如 Polygon PoS)。虽然相对于 Validiums 或者 Volitions 而言，这是一种在技术上较差 的方案，但其市场营销、模因和网络效应将继续存在。鉴于此，在 2021 年更加务实的一些 L1s 将会转而成为 Volitions 或者 Rollups ，但如前所述，另一些 L1s 则会坚持其整体单片式区块链模式 (monolithic blockchain model)，尽管这种模式存在严重的效率低下问题。一个可能的例外是 zkL1s (使用零知识证明的 L1s)，条件是它们足够去中心化。

那用户呢？

  1. 普通用户 ：他们将使用某种形式的中心化聚合器 ，这些聚合器将在 Rollups 或者直接在以太坊上结算。可以将它们看作是类似于 CEXs (中心化交易所)，只不过它们将扩展其功能，集成 DeFi、NFTs 和社交功能等 。

  2. 精通技术的用户 ：能够自我托管的技术娴熟用户将使用智能合约社交恢复钱包 。这些钱包将非常先进，其背后有各种聚合机制 ，可以跨多个 Rollups 聚合智能合约。设想这样一个场景：用户将法币存入智能钱包，并希望获得利息。他们要做的就是将资金存入其中。智能钱包将 a) 把法币转换为稳定币，b) 自动将稳定币存入到有着最佳利息收益的机会中。当然，届时可能有多个类似于 Yearn 这样的利息聚合池。所有这些将从用户端抽离出去，用户只会看到他们正在赚取的利息收益 ！类似地，大多数受欢迎的 dApps 将集成到智能钱包中 ，这样你就不需要在前端和 Rollups 之间来回跳转了。

  3. 特定于应用程序的用户 ：有些应用将完全在前端完成，而后端对用户不可见 。事实上，我们已经在 Sorare (一款基于以太坊的 NFT 梦幻足球游戏) 身上看到了这一点：该应用的前端与中心化的梦幻足球游戏没什么区别，而其后端则使用了基于 StarkEx 的 Validium 进行 NFT 结算。

  4. 爱好者用户 ：这些用户将很可能直接使用 Rollups 进行交易；而对于成本更加敏感的爱好者将使用 Volitions 或者 Validiums 方案。

  5. 金融机构、政府 & 鲸鱼 NFT 收藏家 ：最终，将只有非常有限的参与者会直接在以太坊 L1 上进行结算 ，其中一些是以聚合的方式将用户/公民的活动提交至以太坊 L1 进行结算；而其他的则是希望利用以太坊来消除他们的数据责任。比如，Libra (现改名为 Diem) 就会遇到这个问题：如果他们在以太坊上部署一条 Rollup 链，而不是创建自己的共识机制，则该项目可能已经成功了 (当然，Rollups 在当时还不存在)。

对于用户方面的畅想的另一种方式是：

谁将使用以太坊？ 金融机构、政府、企业、鲸鱼。而且显然，Rollups、Volitions 和 Validiums 将在以太坊上结算，侧链/其他 L1s 链将桥接至以太坊。

谁将使用 Rollups？ 那些更喜欢自我托管和直接与 Rollups 链进行交互的爱好者用户。

谁将使用 Validiums 或者 Volitions？ 同上，但更有可能是那些更关注成本，想要以更低的成本来安全地进行交易的用户。

我预计大多数人将使用智能钱包和中心化的聚合器；人们将很少像现在这样与离散的前端进行交互。

应用

我不认为我们能够想象出将有哪些类型的应用将搭建在 Rollups 之上。这是一个完全开放的设计领域，将会有很多在传统的 L1s 区块链上根本不可能实现的应用出现 。非共识的数据可用性将取得重大进展，“使用以太坊进行登入”也是如此，这两者都将至关重要。一个崭新的创新之门将打开，搭建出创新的应用，这些应用将改变人们和社会的运作方式。

社交网络、游戏、身份、治理和金融系统之间的界限将越来越模糊。所有这些都将汇聚在一起，形成许多人现在所说的「元宇宙 」，其中大部分活动将在以太坊上结算——即便用户可能不会意识到这一点。他们将是以太坊的用户，即便他们从未直接使用以太坊。

以太坊：基础层

在以太坊基础层，将有数百个数据分片 (data shards) 处于活动状态。状态到期 (state expiry) 将上线，从而使以太坊执行层更具可持续性。区块构建者/提议者分离，信标链有着单个 Slot 终结性。此外，在以太坊以太坊基金会和更广泛的以太坊研究和客户端开发社区还有最后一个重大项目——对一切实现 zk-SNARK/STARK ！

我们甚至可能会看到一些非 EVM 的 zkVMs (零知识虚拟机)，可能是一个专门为结算来自 Rollups 或其他先进的密码学的零知识证明而构建的 zkVM 分片，与现有的 EVM 并行。最终，以太坊 EVM 执行层和信标链本身将会实现 zk-SNARK/STARK。这是一项艰巨的工程，需要几年的时间才能完成。

我在畅想的东西还有很多，但就说到这里了。你认为 2025 年以太坊将会是怎样的呢？
