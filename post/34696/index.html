 离开《纽约时报》给我额外带来的好处是，我终于能玩转 Crypto 了。虽然我还有很多顾虑，但 NFT 那超高的热度和趣味性还是不由得让我大吃一惊。

我记得 2017 年我看到Fred Wilson谈论自己发现的 NFT 时，这在我看来似乎还只是一个愚蠢且昂贵的在线卡牌游戏——充其量是一个兄弟式的凯恩斯主义版选美比赛。不过，在我买了几个 NFT 之后，就突然明白为什么有这么多人喜欢它了。

我承认，这个领域充斥着各类骗局，而且我觉得它们其中 95-99% 都会归零。但是，NFT 也让我第一次对 Crypto 的创新性有了直观的了解，也明白了持有特殊数字物品的意义所在。有了 NFT 这样一种数字物品，你们不仅可以将其在不同生态系统之间转移，而且如果你能证明自己拥有该 NFT 的话，你也可以用它来做一些有趣的事情。同时，NFT 也不受任何公司管控，它们不能剥夺我对该物品的访问权限。不同于可以互相替换的同质化 Token，每个 NFT 都是一个独特的物品。不过要想持有 NFT 的话，你目前只能使用自己的钱包，而不能像其他 Crypto 交易一样托管给第三方机构完成。

我已经看到Moxie Marlinspike等人对 NFT 行业背后的所有中心化基础设施产生了质疑，但很多创新型的链上项目也正在着手应对这些质疑。

NFT 确实有时只是一张图片，但事实证明，这个图片比股票报价机更容易得到人们的重视，因为即便你把报价机卖掉了，也可以轻易地将其购回（即使是由公司支持的报价机）。某些因 NFT 而成长起来的社区确实令人不快，但也有一些 NFT 系列收藏在 Discord 上发展出了自己奇特且有趣的社区，社区成员可以在其中找到自己的归属感，并和有相同的兴趣的人一同交流。

对于数字所有权来说，目前很多相关的重要法律问题还未能被解答，比如是否能右键保存等等。而这背后还有一个更主要的问题：对于那些才华横溢的年轻人来说，这个领域真的值得他们投入时间与金钱吗？我的一个朋友在 Crypto 交易上花的时间远比他想象的多，他曾说过：「我很清楚把时间浪费在 Crypto 上对我没有一点好处」。但 NFT 就是能让你近乎疯狂的一心追求更高的收益，而忽略掉其他可能更具价值的兴趣。

撇开其他的不谈，如果 NFT 不是以美元计价的话，人们就更愿意花更大的价钱，就像在赌场里买下筹码一样。我的那个朋友还说：「我今晚花了 7000 美元买了一张照片，照片里是一位握着剑的年轻人。我这辈子从来没有在一件东西上花过那么多钱。」

不过，在 NFT 领域中，很多项目并不单纯是为了挣钱，而是想利用数字身份和数字所有权的概念进行创新，从而探索建立数字游戏、品牌、社区和经济的新方法。

NFT 的快速发展像是 2017 年 IC0 热潮的一个升级版本。其中大多数项目都失败了，而且似乎去中心化的出现只是为了创造出更多种类的欺诈，但它们之中也不乏有几个 IC0 项目成为了价值数十亿美元的风投项目。所有 IC0 中的元素在 NFT 世界中也出现了，但由于 NFT 的理解难度更低且实用性更高，与 2017 年人们创建的大多数 Token 不同，人们可以快速了解并判断其基础技术，并区分出好坏。

我在 NFT 市场中曾经也遇到过骗局，当时我在一个假的 NFT 上签了名，然后我的钱就被卷走了。但我也在 NFT 上取得过一些战绩，比如我曾买到过几个Doodles和一个Azuki。我卖掉了这几件 NFT，并用赚来的钱尝试了更多新奇的创新型项目，比如NFT Worlds，一个类似于 Minecraft 的元宇宙，很多大型 NFT 社区已经在上面搭建属于自己的领地了。

我在玩EtherOrcs的过程中也了解到了很多有趣的事物，它似乎是目前最认真也最受尊重的 NFT 项目之一。不仅如此，我还通过EtherOrcs知道了Anonymice、Realms以及TreasureDAO。在过去的几周里，这最后一个项目已经让我深深着迷，而这都要感谢Bram Kanstein的一条推文。

TreasureDAO正在尝试引领一个新的游戏生态系统。它开发出了一款奇特但非常挣钱的战略游戏——Bridgeworld，用来测试系统并分配资源，游戏内部使用的是MAGIC。虽然整个游戏都建立在各种疯狂的博弈理论上，但你又这确实可以利用这些理论挣到钱。

Bridgeworld 在一开始就取得了巨大的收益，许多 NFT 领域的开发者都期待能在 TreasureDAO 上开发自己的游戏。在这个过程中有一个不错的趋势，那就是所有在 TreasureDAO 上创建游戏的团队都会发放免费的 NFT 初始空投，而如果你能在铸造阶段就得到他们的 NFT，就不用太担心游戏团队卷钱跑路了。

我不得不承认，NFT 市场有可能会走向衰败，许多新的探索与尝试也可能都会以失败告终，因为毕竟现在的市场还充斥着大量泡沫。但我们是否也可以静观其变，享受其中的乐趣？在这一过程中，我一直在寻找那些看到一些希望但也在阻止谣言散播的 NFT 专家。我从0xBender、NFT Alpha Beta、Nate Rivers和NFT Ethics那里也学到了很多东西。如果你还知道有哪些人值得我关注，请务必告诉我。
