一个好的故事会让你有所感触。如果对故事拥有所有权，会让这些情绪更加强烈。用 NFT 讲故事，可以成为讲故事的一种有力方式：对新旧故事都是如此。

过去几个月，多个项目一直在探索这个故事与 NFT 的交叉点。不过，各项目倾向于采用更类似自上而下的方法，而 NFT 可以且应该探索自下而上的讲故事方式，成为新的传播媒介。

自上而下的故事是要告诉我们一个怎样的故事。比如，你去电影院观看《星球大战》系列影片，或者玩 Bethesda 出品的新角色扮演游戏时。而自下而上的故事是我们自己编纂关于虚构世界的故事。比如，当你买了星战的光剑，在院子里跑来跑去，嘴里发出嘶嘶声伴奏，用力把网球扔向你的狗时。当你玩《魔兽世界》游戏时，在暴风城外的小酒馆角色扮演时。

我在本文中想分享 NFT 在实践中的样子，以及我对未来的看法。

自上而下讲故事的 NFT 项目

这些示例的主要目的是用媒体向观众讲述故事。它确实包含粉丝贡献和互动的例子，但主要故事仍然由创建个人 / 团体 /DAO/ 企业主导。

自上而下的故事可以在 NFT 中采用直接文学的形式。

Excerpts of Gridlock (Dec 2020)

这是我自己对直接文学 NFT 的尝试。这些是我的小说处女作《Hope Runners of Gridlock》的摘录。您还可以点击查看我对作家的 NFT 藏品的 更多初步想法。

Few Understand (2021)

加密作家 Kalen Iwamoto 系列小说 《Few Understand》的一条故事链。

还有一个新兴的诗歌场景，增加了口语和视觉元素！

Wild One (2021)

阿尔忒弥斯·怀尔德 (Artemis Wylde) 一首诗的朗诵，伴有视觉元素。无法在此处嵌入视频，请点击观看！

一系列新项目已经出现，其目的是讲述发生在 NFT 之外的虚构故事。在这些示例中，NFT 是故事本身的载体、元素或角色。从某种意义上说，这些项目将 NFT 视为可销售的商品。 这是一种新的众筹形式（有提前预售，也有后期销售）。因为这些项目还包括选择「你自己的冒险」或「贡献自己想法」的能力，所以它更多地介于自上而下和纯粹自下而上的小说之间。

Stoner Cats (2021)

Stoner Cats 售出了超过 800 万美元的斯通纳猫（stoned cat）。除此之外，还推出一部讲述这些猫冒险经历的系列动画片。NFT 持有人将能够对其中一些故事进行投票和共同创作。

Glue Factory (2021)

与 Stoner Cats 类似， 粉丝可以购买即将上映的系列动画片中的马匹。他们还在试验作家小屋和人才基金（如果你所持有的品种出现在片中，就可以从中获利）。

Deadheads (2021)

类似的构思。动画片+ 影响故事的部分能力 + 3D 藏品。

Aku (2021)

Aku 讲述了一个想成为宇航员的年轻黑人男孩的故事。他的太空头盔让他可以探索很多世界。 Aku 分章节发布，实际视频片段被发行为 NFT。它已被选中制作成电影。

Untitled Frontier (2021)

在这里安利一下我自己的项目。Untitled Frontier 制作免费科幻小说，并以 NFT 的形式销售故事中的数字商品。我们切实计划及时激励更多自下而上的故事。

Pillheads (2021)

Pills 是围绕以太坊的叙事包装，渲染了以太坊相关的项目和个人。Pills NFT 持有者可以投票和为内容做出贡献。在某些情况下，他们探索做「任务」，这是一种独特的体验，粉丝实际上与以太坊互动，以增加故事和叙事。

Elektra (2021)

一个神奇的项目，粉丝们共同创造一个故事+音乐。

自下而上讲故事的项目

虽然这些自上而下的项目旨在促进交互性和贡献，但它们仍然主要由项目本身「认可」或「实施规范化」。当粉丝以不请自来的方式，凭借拥有 NFT 取得掌控权，将故事修改为自己的版本时，就有了自下而上的讲故事。而不仅仅局限于允许粉丝投票或贡献内容。

这就是将 NFT 嵌入讲故事的新奇有趣之处。与之前的其他传播媒介不同，它更直接地让粉丝可以将情感时间投入到世界的构建中，而且还可能因此获得相应的经济回报。同人小说并不新鲜，在过去我们已经看到了一些大获成功的案例。《五十度灰》最初是《暮光之城》的同人小说。 《时间的救赎》（The Redemption of Time）最初是刘慈欣《地球往事》三部曲（《三体》、《黑暗森林》、《死神永生》）的同人小说，最终作为重量级作品出版。不过上述几例是极其罕见的。同人小说存在巨大长尾。包括正式出版的作品，也包括孩子们在后院自己创造的角色。

NFT 带来的是能够将故事和溯源添加到 NFT 中，作为虚构世界的一部分。如果成功，它实际上可以为该作家带来价值。一个模拟示例如下：粉丝从商店购买了一把光剑，对其进行了一些修改，然后开始专门为那把光剑写一个故事。如果同人小说取得成功，他们可以将光剑作为豪华套装商品出售，代表添加到这一世界中的那个故事。你可以看到：在前数字时代的模拟世界中，很难做到这一点。

NFT 能更容易赋予所有权 + 溯源。它可以将同人小说提升为融合到核心产品，融入介于商品 + 同人小说 + 投资 + 所有权之间的新传播媒介。想象一下能够购买 / 拥有星战中的绝地大师欧比旺·克诺比（Obi-Wan Kenobi）并添加到角色的故事中。社交媒体用户挂上他们的头像 NFT 就是这些承诺的典型例子。嘻哈巨星 Jay-Z 的 CryptoPunk 有了新的出处，仅仅是因为他在 Twitter 上「佩戴」了它。这其中没有制作出任何故事，但部分故事也来自元互动。Jay-Z 的 CryptoPunk 积累了更多价值，仅仅因为他拥有它。

以下是一些自下而上讲故事的早期项目示例，粉丝接管 NFT 并在原始项目之外添加新故事。

Punks Comic (2021)

一个有趣的实验，围绕 CryptoPunks 讲故事。要阅读相关漫画，至少需要拥有 10,000 个 NFT 之一。如果粉丝选择「烧毁」其漫画，还有一个额外的机制允许他们访问 DAO。

Jenkins The Valet (2021)

可能是我个人最喜欢的例子。团队选择了一只「无聊的猿猴」来作为一个「贴身男仆」，成为「游艇俱乐部」所有故事中的主角。粉丝可以购买作家小屋 NFT 来为故事做出贡献，该故事最终也将发行为唯一的 NFT。其中存在很多层次，但主要有趣的部分是这种特定的猿猴已经积累了广泛的新出处和故事。

Citizen 2890 (2021)

虽然没有明确添加新小说，但由 FlamingoDAO 购买的一个外星 cryptopunk 在其 Twitter 账户中尽情展示。这个想法也是 Fuckin’s Trolls 项目想要激励的东西。

透露个小秘密，一个值得关注的公司 / 项目是 Alethea。它让 NFT 持有者可以在其 NFT 后面添加一个 GPT-3 角色，允许他们将原生和简单的故事嵌入其 NFT 中。NFT 持币者因此更容易进入一个虚构的世界。

前进

除了可以供 Twitter 用户使用头像 NFT 装扮个人资料之外，可能还有更多自下而上讲故事的例子，但尚未得到广泛尝试。这个领域里有很多选择，粉丝们可以聚集在一起，购买现有项目的 NFT，给它一些润色和新故事，如果效果很成功，继续这样做，或者在投入时间和精力值后选择出售 NFT。也许 PartyBids 是一个很好的开始方式。

弄清楚如何促进自下而上讲故事并不容易。它是凌乱的、仓促的，还可能有一些令人讨厌的陷阱。贡献的质量可能很差，也可能是彻头彻尾的仇恨。当前的加密市场有大量对社会所有成员都不友好的 Meme。话虽如此，但在 NFT 出现之前，有一些合作讲故事的项目取得成功的例子。这些都不是新问题。我在一篇 相关阅读 里讲述了与有限与无限故事写作的更广泛的主题。

围绕这些自下而上项目的一个大问题是知识产权（IP）问题，以及如何对待 IP。一些自上而下的项目现在和将来可能仍会尝试保留对 IP 的权力。但是我认为，从长远利益来看，这是错误的方法。NFT 之所以能产生价值，正是因为它可以嵌入到新的环境中并吸引自己的故事。 如果限制获取，就意味着关上了这扇大门。大部分价值将归于 NFT，而不是 IP。话虽如此，一些项目会弄清楚如何有效地管理这个中间地带。不过，把握航向确实难度更大。

就像（thejaymo.net 创始人） Jay Springett 回复的那样。你想要的是 宽松式 IP 许可和能力强大的粉丝。我们将迎来小说和讲故事的新时代，我很高兴看到这一切全面绽放！如果我错过了任何例子，敬请分享！
