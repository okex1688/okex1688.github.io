热潮涌动，出圈的NFT玩儿法也越来越“野”。

前有周杰伦等众多明星艺人，后有冰墩墩等冬奥新宠，各路顶流都在进军NFT。然而，尽管更多人听说过 NFT，但大多数人仍然没有拥有它们。这也使得加密货币交易平台、传统巨头强势跨界，纷纷打造所谓的易用的NFT交易平台。

今天就来盘点一下跨界的NFT交易平台，看看谁更有优势。

交易所：强强联手

集成了多种资源的交易所开辟新的市场并不稀奇。对他们而言，其拥有强大的品牌效应，无论是调动内容资源、资金还是宣发造势、用户群体等方面都有一定的优势。目前这一赛道，集合了FTX、Binance以及Coinbase等头部交易所。

FTX从体育领域全面进军 ，先后成为NBA 华盛顿奇才队和 NHL 华盛顿首都队的官方加密和 NFT 合作伙伴，并与金州勇士队签署了NFT协议，同时FTX 还与一些体育明星建立了直接的合作关系，包括 Tom Brady、Steph Curry 和入选棒球名人堂的红袜队传奇 David Ortiz。

币安从娱乐领域开始 ，与韩国知名的国际化娱乐公司YG Entertainment Inc. (YG) 建立战略合作伙伴关系。YG旗下拥有BIG BANG、BLACKPINK、WINNER、iKON、AKMU、TREASURE等国际知名Kpop明星，在NFT等数字资产领域有着巨大的发展潜力，其知识产权也拥有广泛的商业应用前景。作为双方战略合作的一部分，币安将与YG在多个区块链项目上进行合作。在NFT领域，币安将提供NFT平台及技术基础设施，YG则将提供内容和游戏资产。此外，币安与YG还计划开发BSC（币安智能链）游戏、打造元宇宙生态，并积极寻找数字资产领域的机会，为粉丝提供独特体验与服务。

Coinbase与万事达卡建立合作，为人们提供更多支付选择，让任何人都可以更轻松地使用万事达卡购买 NFT。这次合作也是万事达卡更广泛工作的一部分，该工作旨在通过用于加密钱包和网络安全保护的新工具来支持加密生态系统。

综合看来，交易所布局NFT市场更是强强联合。强大的品牌效应和加密基因使得这些头部交易所开辟NFT市场上一呼百应。但问题也显而易见——那就是如何应对监管。这其中，NFT的属性、税务等问题，都是值得探讨关注的问题。

传统巨头：新颖前卫

如果说交易所开辟NFT市场只是完善生态，那么传统科技巨头进军NFT尤其是开发NFT交易平台则是妥妥地备受期待的跨界。目前这一赛道中，三星集团、日本通讯公司Line、以及乐天株式会社（Rakuten Group, Inc.）正摩拳擦掌。

2022年1月3日，韩国最大的科技公司三星在其官网宣布旗下新型智能电视将允许用户购买NFT。具体来说，三星将于2022款的新MICRO LED、Neo QLED 和 The Frame 三个系列的电视型号中上线NFT交易平台，将支持用户预览、购买和交易数字艺术品。三星对外表示“随着对 NFT 的需求不断增加，对当今分散的观看和购买格局的解决方案的需求从未如此强烈。”这是首个基于电视屏幕的 NFT 浏览器和市场交易平台，也是一个具有开创性意义的平台，能为日益增长的 NFT 需求提供“一站式服务”。

通讯公司Line投入区块链领域多年，继去年 6 月份于日本发表 NFT Market后，在LINE TAIWAN TECHPULSE 2022 开发者大会上透露，旗下子公司 LINE NEXT 所打造、面向国际市场的 NFT 交易平台DOSI将在今年第一季度正式上线 ，目前开始接受企业与客户的预约注册。据悉，DOSI 支持 8 种语言、于全球 180 个国家推出。DOSI 推出的服务预计将包括DOSI Store、DOSI Wallet以及DOSI Support ，其中 DOSI Store 提供企业品牌利用模板或 API 快速打造自家的 NFT 品牌市场、自由铸造 NFT；而 DOSI Wallet 则与社群平台结合，用户可利用其参与元宇宙的各项活动，首季将优先支持信用卡支付，未来，透过 NFT 专用钱包，用户还可通过比特币、以太坊、LINK 等多种加密货币来进行交易以及参加社群活动；至于 DOSI Support 则由 Line 的技术与营销团队支援，提供企业主技术面与产品面的服务，协助企业主最佳化其 NFT 商店的规划与营运。

Line NFT 机制

经过数月研发，日本最大电商平台乐天“Rakuten NFT”将于本月25日正式上线。 它允许用户在体育、音乐、动画及其他娱乐领域买卖NFT，并为IP持有者发行和销售NFT提供平台。目前，乐天NFT已于朝日电视台（TV Asahi ）等达成合作伙伴关系。最值得关注的是，乐天NFT将集成到乐天生态系统中， 这意味着乐天在日本的数百万会员中的任何一个都可以使用他们现有的乐天 ID 登录和结帐。未来，乐天还将支持客户使用乐天的忠诚积分购买 NFT 。此外，乐天株式会社本身就拥有众多的与体育有关的合作伙伴。该公司本身赞助了日本足球联赛的一支职业球队，也是世界重量级足球俱乐部 FC 巴塞罗那队的球衣赞助商。乐天还运营着许多基于娱乐的产品，包括视频点播平台-乐天电视。

显然，传统巨头布局NFT市场，有着天然的优势：用户量级和优势资源，这是现阶段加密货币交易所难以匹敌的。传统科技巨头资金实力雄厚，从自身基因和擅长的领域出发，更容易捕获更广泛的市场。
