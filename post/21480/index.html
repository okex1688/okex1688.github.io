波卡一周观察，是我们针对波卡整个生态在上一周所发生的事情的一个梳理，同时也会以白话的形式分享一些我们对这些事件的观察。本文旨在提供信息和观点，不为任何项目作背书。任何提及的项目，均不作投资推荐，仅作信息展示。

1.观察与观点

本周受到美国征收资本利得税的影响，加密货币市场遭到重挫，波卡自然不能幸免。而由于Kusama插槽拍卖迟迟未上，许多人已经在唱衰波卡了。对此，我们不妨换一个视角来看， 如果波卡必定会因为插槽拍卖的上线而迎来一波上涨，那么在这个利好出来之前遇到的大跌都是加仓的好时机。

当然，如果以就事论事的角度来讲，技术研发本来就很难按照计划准时完成。研发总会面临各种困难，遇到困难再想办法解决困难，这是最正常不过的事了。如今，波卡在全球有一百多位工程师参与研发，是目前区块链领域里最大的研发团队之一，研发团队都是多年区块链技术专家，更何况创始人还从零到一构建了以太坊的原型。所以，这样的技术实力，我们要再多给他们一些时间和信心，静待问题的解决。

而且，其实官方也在悄悄地下一盘大棋，在本周就推出了三个重要的举措。也许在本周惨淡的行情下，大家会忽略掉这些信息的价值，我们不妨试着把它们拎出来，跟大家捋一捋这其中的真意。 

第一个 是Gavin Wood发布 Kusama 平行链的上线清单。在清单里，除了披露了启动Kusama平行链功能的三大阶段，还提及了一系列待完成事项。这其中，公益平行链中的通用资产链Statemint将在平行链插槽拍卖之前上线，尤为值得注意。或许，许多朋友还不太了解Statemint，官方对它的定义是，将提供在Polkadot和Kusama网络中部署资产的功能——从token化的艺术品到稳定币和中央银行数字货币（Central Bank Digital Currencies） ，并以比现有解决方案更低的费用为用户提供更好的体验。简单的理解为波卡上可以发行所有通用资产的地方，包括了NFT，稳定币和央行数字货币CBDC。

 这一条链，将会是波卡生态中举足轻重的一条链，为什么这么说呢？

因为无论是这次的NFT还是DeFi的火热都是围绕着两点，一个是资产，一个是应用，就好比是DeFi的火热也助推了USDT和USDC这些稳定币体量的爆发式增长。同样，Statemint也将会是未来波卡生态中DeFi的重要资产诞生地。 而且此前，USDT的发行方Tether公司就已经宣布会在Statemint上发布USDT了，这也意味着市值第五大的数字资产USDT将会很快接入波卡。同时，我们也看到像Acala这样的稳定币发行方也正积极地在Statemint上做测试，由此可见，Statemint将会是各个数字资产发行方的兵家必争之地，关注这些资产究竟会更多地流向哪些DeFi中，将会是抓住波卡未来DeFi大爆发红利的绝佳观察点。

而另一个层面来说，更贴近于传统资本市场和企业的NFT以及CBDC将会为波卡整个生态带来巨大的增量资金，Statemint便是这些资金的入口，海量的外部资金也将会助推整个波卡生态快速繁荣。从可行性上来看，为波卡做研发的其中一些团队，比如Sora，就已经为越南央行发行的Bakong稳定币提供过技术支持了，所以，这是完全在技术上和商业上都可以被验证的逻辑。

第二个 重要进展是Parity推出Builders for Builders计划。在这之前，Parity官方已经推出了Substrate Builder Program计划，这个计划相当于是在技术、财务、营销以及业务反馈上对项目提供支持，相当于是Parity官方的一个孵化计划，目前已经有50多个项目参与了这个计划。 但是项目孵化成功后，又有另一个问题，这些项目基本都是发链的项目，他们也需要去发展自己的生态，吸引更多的开发者在链上开发出DApp，但是往往这些项目的开发者还不太了解如何做好一个生态。于是乎，官方替你解决这个问题，也就有了这次的Builders for Builders计划。这个计划简单来讲，就是官方教这些发链的项目方如何构建机制去孵化DApp。这两个计划连起来看，可以理解为手把手一条龙服务地把一个项目从头到尾拉扯大。  

虽然说手把手地拉扯这一个用词有点戏谑，但其实认真地说，这一步是非常有必要的，因为光是孵化了发链的项目，让他们顺利完成项目的开发，这还只是第一步。如果这些项目后续的生态做不好，导致项目最终死掉，这也会影响整个波卡生态的发展。所以，通过这个计划，我们可以看到官方的良苦用心，我们还从未在其他项目上见到有官方如此耗费心力和资金来支持生态建设的，这也是波卡为什么能快速建立庞大的生态的原因之一，也更让我们相信波卡一定会成功地实现它成为底层区块链基础设施的目标。

第三个 重要进展是Wococo测试网的上线，它将为以太坊桥的测试做准备 ，这意味着距离将以太坊生态纳入波卡生态的时间越来越近了，其意义就不必多说了。

2.波卡官方动态

【Gavin Wood发布 Kusama 平行链的上线清单】

据Polkaworld报道，波卡创始人Gavin Wood在 Github 上发布启动 Kusama 平行链功能的三大阶段以及一系列待完成事项。从发布的清单来看，在开启正式的拍卖之前，会先上线 Shell chain 和之前已经介绍过的公益平行链 Statemint。截止发文还没看到波卡官方有公布任何关于 Shell chain 的信息，但猜测应该也是一个公益平行链。

【Parity推出Builders for Builders 计划！支持平行链团队发展自己的建设者生态】

据Polkaworld报道，Builders for Builders 是一项新计划，旨在赋能构建基于 Substrate 的平行链的团队，开发和发展在自己的平行链上开发的建设者生态。作为 Builders for Builders 计划的一部分，Substrate Builders 计划团队将帮助平行链团队建立自己的类似程序，以便团队在其平行链上构建解决方案。这些被称为 Parachain Builders Programs（PBP）的计划将让平行链团队成为其他团队的导师，并发展自己蓬勃发展的建设者社区，从而丰富整个 Polkadot 生态系统。

【Rococo：已上线公益平行链Statemint测试网】

官方消息，目前平行链测试网 Rococo已经上线公益平行链 Statemint，并开始测试资产发行。目前，Acala、Zenlink、ChainX、Bifrost已经在 Statemint 测试网发行了他们自己的测试 token。

【波卡周报：Parity上线Wococo测试网】

据Polkaworld波卡周报显示，Parity上线Wococo测试网，为以太坊桥测试做准备。1.Gavin发布Kusama平行链上线清单。2.Kusama的106号公投已经通过并执行，Kusama目前正在运营runtime v2030的版本。3.Kusama理事会通过了CryptoLocally提出的集成 Kusama 的国库议案。4.目前Rococo上有四个平行链测试网Statemint、Tick、Trick和Track。5.Statemint测试链正在Rococo上运行。6.为了准备Kusama的平行链，Rococo目前正在扩展到一个类似于Kusama的网络拓扑和等效参数。

3.生态大新闻

【Deeper Network：Bione Exchange未经官方授权强上DPR】

Web3基础设施、去中心化网关Deeper Network（DPR）在推特上提醒称，加密货币交易所Bione Exchange强行上线DPR，并未与官方沟通，请投资者谨慎投资和交易。

【Findora联合9家机构发起的国内首个隐私计算技术联盟成立，Manta Network、Phala Network等波卡生态隐私项目也是发起机构】

2021年4月16日，由金融隐私公链Findora联合9家机构发起的国内首个区块链隐私计算领域去中心化治理组织-隐私计算技术联盟（Privacy Computing In China，简称“PCIC”）宣布成立。PCIC 9家创始成员机构为：Findora、Oasis Network、CertiK、Manta Network、Phala Network、Cabin VC、Candaq、中国技术经济学会区块链分会、Blocklike。PCIC旨在为隐私计算领域打造一个有公信力、开放、共赢的自治理组织。PCIC将通过轮值机制，由核心发起成员轮值管理隐私计算技术联盟社区，定期向市场传递隐私计算领域技术研讨及生态进展。

【Clover Finance将构建跨以太坊和Polkadot的多链钱包】

据Cointelegraph消息，Polkadot生态智能合约平台Clover Finance将构建一个一站式的基础架构平台，使DeFi项目的开发人员可以将其DApp从以太坊迁移到Polkadot上。Clover的这种多链钱包允许用户跨区块链查看和处理其所有资产（包括第二层资产）。

【波卡生态去中心化存储项目Crust发布Kusama平行链插槽竞拍计划】

据官方消息，波卡生态去中心化存储项目Crust发布Kusama平行链插槽竞拍计划，Crust平行网取名“Shadow”，Crust Shadow将参与Kusama平行链插槽竞拍，Crust Shadow作为使用Kusama平行链资源的桥接，功能是负责承接Kusama平行链和Crust network（主网）之间的信息交换。未来，Crust network会参与波卡平行链插槽竞拍，力求接入平行链，通过平行网Crust Shadow来桥接Kusama和波卡之间的信息交换。

【Punk.Network在波卡公益平行链Rococo Statemint启动测试】

据官方消息，波卡生态公链项目Punk.Network在波卡平行链测试网Rococo的公益平行链Statemint上启动测试，Statemint具备在Polkadot和Kusama网络中部署资产的功能。同时Punk.Network已被Polkaproject收录。此外，Punk.Network为国内卡牌游戏《天际征服》中NFT资产发行提供技术支持，《天际征服》已正式上线Android平台，相关NFT游戏卡包将在近期上架NFTCastle.io。

Web3基础设施、去中心化网关Deeper Network（DPR）发推称，DPR/USDC流动性质押第一季已结束，总共发放500万枚DPR奖励。DPR/USDC流动性质押第二季开始于UTC时间4月26日5点（北京时间13点）开始，并于UTC时间5月10日5点（北京时间13点）结束。第二季奖励共计250万枚DPR，每天将提供166666枚DPR奖励。第一季已经参与质押的用户将自动加入第二季活动。

波卡生态跨链项目Darwinia Network在推特上表示，已成功在波卡公益平行链Statemint上铸造代币资产

5.生态合作

【波卡生态智能合约平台Clover Finance与跨链项目Darwinia Network达成合作】

据官方消息，波卡生态智能合约平台Clover Finance宣布与波卡生态跨链项目Darwinia Network（达尔文）达成合作，将从技术、经济、效率三个方面共同研究异构链资产交互的跨链解决方案，以将更多以太坊、BSC等公链资产引入波卡生态系统。同时，双方将共同研究DeFi和NFT领域的相关跨链标准。

【波卡生态货币市场协议 Konomi 与 DAO 基础设施 Dora Factory 达成战略合作】

据官方消息，波卡生态货币市场协议 Konomi 与 DAO 基础设施 Dora Factory 达成战略合作。Konomi 将使用 Dora Factory 的工具来管理社区。同时，双方将共同资助与 DeFi 协议相关的链上治理方案的研究以及产品的开发。Konomi 基于 Substrate 底层技术框架，提供跨链资产管理解决方案，旨在让以太坊上的金融应用无缝衔接到波卡生态。Dora Factory 是 Substrate 上的可编程 DAO 服务开放基础设施。

【波卡生态智能合约平台 Clover Finance 与 MathWallet 达成战略合作】

据官方消息，波卡生态智能合约平台Clover Finance 宣布与 MathWallet 达成战略合作，双方将在行业研究、技术开发、资产安全、社区服务等领域继续展开深度合作，共同推进 Polkadot、Layer2 生态发展。

【波卡生态项目 Darwinia Network 与区块链游戏平台 GamyFi 达成合作】

据4月22日消息，Darwinia Network 与 GamyFi 游戏平台建立战略合作伙伴关系，GamyFi 计划部署 Darwinia 跨链桥，以在以太坊和其他 Polkadot 生态系统链之间转移资产。跨链桥同时支持 NFT 的跨链传输，将为 GamyFi 平台提供基础架构支持。

【Plasm网络与Bondly达成战略合作】

据官方消息，Bondly与Plasm网络双方达成战略合作伙伴关系，Bondly将使用Plasma网络通过其即将发布的NFT launchpad、NFT交易平台：BondSwap，以及即将发布的BondProtect来进一步扩展到Polkadot的生态系统中，Plasm将会在Bondly launchpad上发布一个特别NFT。据官方介绍，NFT Launchpad由BondSwap驱动，NFT Launchpad将允许合作伙伴的项目来使用Bondly的基础设施与网络。

【波卡生态混合式抵押弹性供应稳定币Standard Protocol与Litentry达成战略合作】

Standard Protocol与波卡去中心化身份聚合器项目Litentry达成战略合作，将在下一个版本更新中将采用Litentry的跨链身份验证技术，充分评估用户各方面贡献值（例：抵押物价值、社群维护、内容生产数量与质量）。这将使得Standard Protocol生态中用户有更大动力在各Dapps上建设与活跃，提升社区治理的权利，实现更去中心化。

同时，Litentry的身份聚合器技术将与Standard的预言机系统相结合，完善并提升Standard生态中预言机喂价者的的身份验证流程与检验准确度。未来，当Standard上的合成资产在全球范围内流通时，双方的合作领域将扩大至符合各国各地区监管政策的必要的KYC验证等方面。
