第三方跨链桥通常比官方桥速度更快，特别是用户从 L2 退出到 L1 时不需要等待 7 天，不过需要付出额外的成本。

在 Optimism 和 Arbitrum 正式对外开放主网测试后，除了应用部署的数量在快速增加之外，如何将大量资产以更低成本迁移至 L2 网络中，也是很多人关心的话题。另一方面是，如果用户需要将 L2 的资产快速转移至 L1 中，官方桥无法实现，所以只能借助其他的第三方工具。

而随着公链和 L2 数量的增加，资产跨链需求也快速增长，各种各样的跨链桥工具成为了用户的必备的工具之一。而针对 L2 这类特殊的跨链场景而言，又诞生了一批专为 L1 和 L2 之间这一特定场景提供跨链服务的桥。

综合来看，第三方跨链桥工具比官方桥的跨链速度更快，特别是当用户需要将资产从 L2 退出到 L1 时，不需要等待 7 天的时间。不过用户需要付出额外的成本，支出一部分费用给为跨链提供资产流动性的「做市商」或者「协议」，也就是将资金的时间价值进行了折现，减少了 7 天空置资产带来的潜在损失。

从一个 L2 去另一个 L2 非常耗时耗钱

不同的跨链桥所使用的技术都不太相同，比如常见的会采用哈希时间锁、状态通道，或一些偏中心化的解决方案等。这些资产跨链桥对资产的支持也不太相同，所以我们体验了下目前已经上线的跨链桥（连接至 Optimism 或 Arbitrum），包括 Hop Protocol、cBridge、DeGate、RenBridge 和 Anyswap，并整理了这些平台的优缺点和对应的成本估算（按照近期平均的 Gas 成本）。

由于以太坊社区对于 ETH、USDC 和 USDT 这三类资产的需求量显著高于其他资产，所以对这三类的资产支持是对跨链桥评估的核心指标之一。

L2 官方提供的跨链桥

L2 团队都会提供一个官方的跨链桥，这类桥提供了最基础的功能，安全性也是其最重要的特点，不过由于 Arbitrum 和 Optimism 采用的扩容技术的机制，从 L2 退回 L1 需要等待 7 天的时间。

Arbitrum 刚更新了官方桥的网页 UI

  • 支持方向：以太坊 <> L2 网络（Arbitrum/Optimism）

  • 资产：已支持 USDC、USDT 和 ETH，以及白名单中的代币，之后会逐渐开放

  • 优点：相对最安全，不限制交易规模

  • 缺点：退出期需要 7 天

  • 跨链成本：仅需网络的正常交易费（约 20 至 30 美元）

Hop Protocol

Hop 是由此前智能合约钱包开发团队 Authereum 开发的跨链桥，他们方案中设计了通用的资产桥，以及通过引入「自动做市商」 （AMM） 组件和「连接器」 （Bonder） 角色，最终实现 Layer 2 网络之间的资产快速迁移。

使用 Hop 的方案时，资产需要通过 Hop 流转到 Layer 2 网络中，比如通过 Hop 的资产桥进入二层的 ETH 被称为 Hop ETH （或 hETH） 。hETH 和 ETH 是完全等价的，至少可以通过 Hop 完成兑换。

但是 Layer 2 网络中同时还有「官方」版本的 ETH，也就是更多人普遍使用的 ETH 版本，官方版 ETH 和 hETH 理论上应该也是完全等价的，但是由于流动性的原因，可能会存在一些价差。

所以 Hop Protocol 引入了 AMM 的组件以及「连接器」，AMM 是为了解决官方版 ETH 和 hETH 之间的短时间波动的价差而设计的，而「连接器」 （Bonder） 角色则是可以为需要提前释放流动性的用户提供流动性，同时也可以获得部分的收益 （因为他为用户节省了 7 天的提款周期） 。

Bonder 可以通过观察不同 Layer 2 之间的交易数据，为网络提前垫付官方版 ETH，而不同 Layer 2 网络之间的套利者也会不断再平衡 （以获得收益） ，将 AMM 的价格维持在一个比较合理的范围内。

  • 支持方向：以太坊 <> Arbitrum；以太坊 <> Optimism；Arbitrum <> Optimism

  • 资产：已支持 USDC 和 USDT，未支持 ETH

  • 优点：退出期不需要 7 天

  • 缺点：协议中资产的流动性会限制跨链的资产规模

  • 跨链成本：网络本身的交易费用（在 L1 上的交易约 30 美元）+协议交易费（0.04%），另外还存在交易滑点

cBridge

cBridge 是由 Celer 团队开发的跨链桥工具，基于原子交换技术实现。

cBridge 表示将在短期内陆续支持更多 Layer2 和 Layer1 区块链的跨链和跨层转账，此外任何人都将能通过运行 cBridge 节点加入 cBridge 网络以提供跨链跨层流动性，并获取对应的手续费收益。

Celer 刚刚发布了 2.0 的规划，能进一步提高跨链资产的流动性，具体可参考这篇文章：cBridge 公布 2.0 升级计划，速览新版设计取舍及优化。

  • 支持方向：以太坊 <> Arbitrum；以太坊 <> Optimism；Arbitrum <> Optimism

  • 资产：已支持 ETH、USDC 和 USDT

  • 优点：退出期不需要 7 天

  • 缺点：协议中资产的流动性会限制跨链的资产规模

  • 跨链成本：网络本身的交易费用（在 L1 上的交易约 30 美元） +支付给中继节点的运营费用（由节点自己定价）

  • 链接

DeGate

以太坊二层交易协议 DeGate 已在 Arbitrum 网络部署了资产桥的 Beta 版，该资产桥采用了稳定币 AMM 曲线来提高资金的利用率。

DeGate 表示，用户可以通过该资产桥实现跨层资产转移，且无需等待原生桥的挑战期。在 Beta 版本阶段，流动性注入仅向白名单合作伙伴开启。

  • 支持方向：以太坊 <> Arbitrum

  • 资产：已支持 ETH、USDC，未支持 USDT

  • 优点：退出期不需要 7 天

  • 缺点：协议中资产的流动性会限制跨链的资产规模

  • 跨链成本：网络本身的交易费用（在 L1 上的交易不到 10 美元）+协议交易费（暂时免费），另外还存在滑点

  • 链接

RenBridge

RenBridge 是由 Ren Protocol 开发的跨链桥，主要的特点是他们关注与其他公链之间的跨链解决方案，目前最核心的是支持 BTC 与 L2 之间的直接跨链。

  • 支持方向：BTC <> Arbitrum

  • 资产：仅支持 BTC （和其他公链代币），未支持 ETH、USDC 和 USDT

  • 优点：支持 BTC 跨链

  • 缺点：仅支持 Arbitrum 与 BTC 之间的跨链

  • 跨链成本：由于不触及以太坊 L1，所以仅需比特币矿工费（～3 美元）+Arbitrum 交易费（～3 美元）+ 协议收费（0.15%）

  • 链接

Anyswap

Anyswap 是通用的跨链协议，此前关注在公链之间的跨链技术，近期刚刚加上支持 Arbitrum 网络。

  • 支持方向：ETH <> Arbitrum

  • 资产：已支持 USDC，未支持 USDT 和 ETH

  • 优点：退出期不需要 7 天

  • 缺点：未支持 Arbitrum 之外的 L2 网络

  • 跨链成本：网络本身的交易费用（在 L1 上的交易约 30 美元）+协议交易费（0.1%）
