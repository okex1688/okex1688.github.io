前两天与⼀个 MakerDAO 早期⽀持者喝酒，⼤佬解答了我⼀个⼀直没想明⽩的问题，⾦融机构的能⼒应该是创造信⽤，优化资⾦配置，但像 MakerDAO 这种超额质押，更接近于典当铺的形式，资⾦利⽤效率并不⾼，与他的去中⼼化央⾏远景不符。⼤佬说，你换个⻆度想，MakerDAO 现在是只⽀持抵押以太坊⽐特币，但他的想象空间是让所有资产都可以发币，产⽣流动性，随着抵押物的多元化，发⾏货币权会慢慢移植到私⼈企业和个⼈⼿中，这才是最厉害的。你看这瓶啤酒，如果⼤家都认可他的价值，那他的所有者和⽣产企业就可以进⾏⼀定货币化，⽽不需要中⼼化⾦融机构参与。

于是，我开了个脑洞，如果未来把特斯拉抵押发币，应该会是什么样的流程呢？

第⼀步，对⻋辆价格进⾏估价

对⻋辆价格评估已经是⾮常成熟的体系，现有⻋商都有⼀个成熟的数据库，输⼊⻋辆 id(VIN) 就可以 知道⻋辆具体型号，年代，是否出过⼤修等等最重要估值依据。⽽⻋检的实际状况，可以通过⾮常简单的算法完成，只要对⻋固定位置进⾏拍照扫描，现有的机器学习模型可以轻松判定出⻋漆和内饰状况，对⻋辆⾥程真伪进⾏验证。如果特斯拉开放⻋辆内部传感器芯⽚权限给估值⼯具，就可以更好读取机器运⾏状况，精确⻋辆价值。假设⼗年后，特斯拉依然是市场销量第⼀的新能源汽⻋，⼆⼿⻋市场⼜⾮常透明化，那么这个估值模型可以⾮常准确。

第⼆步，将⻋辆 NFT 化

这⾥需要把⻋辆的所有权控制权 token 化，只要特斯拉愿意开放接⼝，也是⾮常容易的事情。把评估出来的⻋辆信息（资产信息）录⼊到⼀个 token 中，这个 NFT token 就代表着⻋辆的所有权，token 转移就等同于⻋辆所有权转移，每⼀个 token 相当于是⻋辆的唯⼀⻋钥匙。Token 转移后，原⻋主就失去了⻋辆所有权，⽽新⻋主就获得了⻋辆的钥匙。

第三步，⻋辆定价、交易、抵押以及投资

畅想 10 年后的 OpenSea 或其他 NFT 交易平台，已经⽀持特斯拉⼆⼿⻋NFT 交易，这时，对于⻋主可以有两种选择，⼀⽅⾯，可以直接交易，随着 NFT 的销售，所有权直接转移，⻋主拿到资⾦，买家得到 NFT 后，⻋辆可以⾃动驾驶到买家⻔⼝? ，⽤数字货币钱包内的 token 激活⻋⻔。

另⼀⽅⾯，⻋主也可以抵押⻋辆的 NFT，前提是 DAO 同意把某年某型号加⼊质押资产类别，⼀个 NFT 对应的是这个标准⻋型价值的⽐例，⻋主在 MakerDAO 或其他质押借贷平台上，质押⾃⼰⻋⼦的 NFT，获得美元等值的稳定币，⽤以满⾜⽇常⽣活开销，⻋还依然归⻋主开，并且可以把资⾦投资到各种 DeFi 的⽣息资产和交易中，⻋主要定期⽀付平台⼀个利息，相当于⼜租借了⻋⼦使⽤权。

这⾥对于质押借贷平台⽅最⼤问题，是如何去做预⾔机，去取⼀个合理资产价值，既不被企业操 纵，也不会被⻋商操控，估计到时候会有⼀套特斯拉维护的中⼼化交易所⼆⼿tesla-u 价格，以及⻋商维护的去中⼼化交易所⼆⼿tesla-u 价格。如果随着固定折旧，⻋主并不归还所借资⾦，或者⻋辆出现重⼤事故，⻋上传感器报警，⻋辆价值暴跌，⻋的所有权都会彻底不属于⻋主，⻋会⾃动驾驶开回到基⾦会某个⽹点，完成对质押物清算。

整个过程对于特斯拉⻋主和特斯拉来讲，其实就是进⾏了⼀笔没有银⾏的抵押借贷，完成了创造货币过程，并没有依赖任何中⼼化⾦融机构参与（除了特斯拉企业本身），⼏乎⽆磨损且超⾼效率。除了政策原因，特斯拉的⽀持不能保证，以及⻓途的⾃动驾驶部分，还有些⽞幻，整个过程⽤到的基本技术都已经很成熟，离实际商⽤可能就是个时间问题。

区块链如果想被⼤规模接受的应⽤，⾸先是有赚钱效应，第⼆是可以标准化，有⾮常简单的估值逻辑，⽐如之前公链看 TPS、DeFi，看 TVL，第三是操作⽅便。NFT 现在集中在艺术品，赚⾜了流量和眼球， 并且满⾜了短期赚钱效应，但因为艺术品特殊性，估值逻辑差距⽐较⼤，⽽且线下艺术品确权和流通⽐较复杂。

我个⼈是⽐较看好原⽣的数字资产如游戏道具、域名、图标设计等的 NFT 化，并通过叠加 DeFi 完成货币创造。⽽如果能进⼀步引⼊像特斯拉使⽤权的资产包，区块链真的会⼀步步颠覆中⼼化⾦融。
