自今年年初，推特 CEO Jack Dorsey 把他在 2006 年发布的首条推文作为一个 NFT 以超过 290 万美元的高价卖掉时，「沉寂」了好几年的 NFT 再次被市场引爆，并迅速出圈：美国《时代周刊》拍卖 3 张封面 NFT，《纽约时报》专栏作家 Kevin Roose 的一篇专栏文章 NFT 最终以 350ETH 成交，中国拍卖公司永乐拍卖将进军 NFT……

作为 NFT 的始祖，起源于 2017 年的 CryptoPunks 像素头像项目也再次焕发青春。5 月 12 日，由 Larva Labs 设计并发行的 9 枚 CryptoPunks NFT 完成拍卖，共拍出 16,962,500 美元的高价。此前，Larva Labs 还推出了一款新的 NFT 藏品 meebits —— 20000 个独特的 3D 像素风人物，并在开售后的 6 个小时内将其中的 9000 个 meebits 全部售完，总售价超过 7000 万美元。

NFT 的市场增长有多快？

据 Dune Analytics 的数据显示，仅以太坊上的去中心化 NFT 交易平台 OpenSea，其用户数量就已经突破了 11.2 万 ，而在今年年初这个数据是 3.6 万，短短 4 个月的时间用户数量翻了 2.1 倍 。OpenSea 的使用门槛非常高，而且与合约交互所需要的 ETH 手续费很高昂，但尽管如此，今年前 4 个月 OpenSea 分别创下了 3.22 亿、1.47 亿、0.93 亿、1.05 亿美元 的成交额。

NFT 市场之大，正在吸引大量的资本、机构、交易所、开发人员、艺术家进入这个领域，连币安也高调宣布入局，并将于 2021 年 6 月正式上线 NFT 平台。

新兴的 PoW 公链中发展最好的 Nervos，其生态在今年第二季度正式开始爆发：第一款基于 Nervos CKB 主网的 NFT 平台，由 Nervina Labs 倾力打造的金色传说（Golden Legend），目前已经正式上线。
不同于以太坊、BSC 等区块链上的 NFT 平台，金色传说的操作门槛极低，用户体验感极佳，是一款在某种程度上可以媲美互联网级别应用的区块链产品 。通过金色传说平台发行的 NFT，可以通过 Nervina Labs 开发的网页版钱包进行接收，这款钱包被亲切地称为「秘宝（Crypto Treasure）」。

下面，我们对金色传说和秘宝做一个简单的介绍。

金色传说

「金色传说」这个名字取自暴雪推出的炉石传说这款游戏。在炉石传说里，有一款出现概率极低、卡面为金色的卡牌，非常珍贵，因此被称为金色传说。

从本质上来看，Nervina Labs 推出的金色传说是一款专门发行 NFT 的 SaaS 平台，主要针对艺术家、NFT 发行者，具备以下 4 个特性：

1. 简单流畅的使用体验

无需和复杂的密钥管理打交道，金色传说会提供安全的密钥托管、签名和交易管理服务；

2. 灵活的创作专属 NFT

支持进行批量创作 NFT，可设置限量发行，自定义套装，专为真实的应用场景设计；

3. 高效的操作管理平台

创作 NFT 后即可进行快速批量分发，贴心的交易管理，让用户告别漫长的链上等待和不确定性；

4. 丰富的生态组件

已有配套的秘宝账户、秘宝浏览器、SDK 等组件，大大减少了基础设施的开发量，轻松链接用户。
这里需要额外说明的一点，是金色传说丰富的生态组件。大量的 SDK 和 API，使得金色传说在后续的发展过程中，非常容易被其他互联网 App 或者区块链上的 dApp 所集成。 想象一下，以后打开你的微信小程序，或者某款常用的手机软件，就可以进入到金色传说的后台，创作和发行自己设计的 NFT，多么方便，多么酷！
Nervina Labs 创始人兼 CEO Cipher 透露说：

我们希望能推动区块链技术的落地并为传统互联网应用赋能，金色传说和秘宝钱包是一套 NFT 技术整体解决方案，用户通过 API 可以非常方便地将我们的方案集成到公司旗下的软件产品中，为自己的软件、App 赋能。我们目前的客户以自带流量的企业用户为主。目前，Nervina Labs 已同网易文创三三工作室、区块链咨询公司 WXY 等多家平台、公司和个人 IP 达成合作，将从娱乐、艺术、体育、文创、游戏、金融、时尚等多个领域进行技术服务和商业场景落地，为各领域和行业有影响力的机构和个人提供更好的商业增值服务。

秘宝

在金色传说这个平台上，秘宝有两个含义：一个是指通过金色传说这个平台发行的 NFT，另外一个含义是特指 Nervina Labs 推出的 NFT 网页版钱包。

用户可以通过手机号或者邮箱注册一个秘宝账户，也可以通过直接连接以太坊钱包（比如 MetaMask、imToken 等）创建账户，无论是完全不懂区块链的圈外人，还是拥有多个钱包的老炮，在用户体验上秘宝考虑得非常周到。
在这个小小的注册环节，其实也隐藏了两项「黑科技」：Unipass 和互操作性 2.0 技术。
用户之所以可以通过手机或者邮箱进行注册，是因为秘宝这款产品使用了 Lay2 Tech 提供的 Unipass 组件，你可以认为它是 CKB 上的 MetaMask 密钥管理器，但是它做到了互联网级别的用户体验和区块链世界的高安全性，并且资产由用户完全掌管。通过 Unipass 注册的账户，本质上也是 CKB 的加密钱包账户，这要感谢 CKB 底层对密码学原语的灵活支持，把邮箱、手机号码等使用的密码学搬到 CKB 上来。所以，通过 Unipass，用户无需助记词，就可以顺滑地生成加密账户。
通过连接以太坊钱包进行注册和登录，很多人早已习以为常，因为以太坊、BSC、HECO 上的很多 DApp 都是这样操作的，但这里的 NFT 是发行在 CKB 上的，CKB 和 ETH 是两条完全不同的底层公链。用以太坊钱包接收、发送 CKB 上的 NFT，这正是 Nervos 提出的互操作性 2.0 的应用案例之一。

创建好秘宝账户后，用户就可以随时随地查看、管理自己的 NFT 了，非常方便。

核心优势

「金色传说+秘宝」，让 NFT 的发行方、NFT 的收藏爱好者都得到了互联网产品般的体验，在使用上几乎无任何门槛。
作为 NFT 发行方，他不需要懂区块链技术，不需要懂合约代码的部署，甚至也不需要懂创作和发行一款 NFT 需要多少手续费、网络是否拥堵之类的问题。他只需要知道自己发行的 NFT 要取个什么名字、是否限量发行、发行多少数量即可，完全是互联网产品的体验。
创作完 NFT 之后，发行方可以通过金色传说平台进行批量分发，而且 NFT 自带盘缠，完全不需要考虑分发时的手续费，也不需要额外购买 CKB 当作手续费。在金色传说这个平台上，创作和发行 NFT 只需要消耗特定数量的「能量点」，这点类似于很多游戏中的能量值，完全没有任何陌生感。
至于 NFT 的爱好者、收藏者，他也不需要懂区块链技术，直接用手机号或者邮箱注册一个秘宝账户即可。在秘宝账户中，他可以实时预览账户里的 NFT，这里还有一个小细节：打开账户里的任何一个 NFT，将鼠标放在 NFT 图片上面移动，图片会跟着鼠标移动的方向动起来，这点和 MetaMask 小狐狸钱包的动态效果非常类似，让人很有亲切感。
如果要将秘宝里的 NFT 转让给别人，同样不需要额外去购买 CKB 或者考虑手续费等情况，直接像普通转账一样转给别人即可。通过金色传说发行的每一个 NFT，都自带 1CKB 的盘缠，按照目前的网络手续费情况，可以转账 7000 多次。这对于换手率较低的 NFT 而言，非常够用了，即便以后出现了因为转账次数过多导致手续费不足的情况，后续也依然通过简单的方式给 NFT「加油」。
另外，无论是金色传说，还是秘宝账户，用户都无需额外管理自己的公钥、私钥，所有的操作全程上链记录，保证账户安全，确保 NFT 的归属和价值准确无误。
目前，市面上已经有不少 NFT 发行平台了，但对比下来，在用户体验上做得最好的，非 「金色传说+秘宝」 莫属。

金色传说是一款主要针对 B 端用户的产品，所以对于企业和机构用户而言，它还有另外一项优势：通过 API 可以非常方便地将金色传说、秘宝集成到公司旗下的软件产品中，为自己的软件、App 赋能。

与实际商业场景结合的应用点

1. 娱乐圈

代表形式：明星个人 IP，影视周边及记录等，周边衍生 IP，动漫 IP，影像记录，图片传播，音乐版权，声音记录。

2. 艺术圈

代表形式：画作，画展，设计作品，设计展，数字艺术作品，传统行业设计物品（包含玉，木，泥雕等），传承文化艺术工艺，现代艺术制作作品，行为艺术衍生作品。

3. 商业活动

代表形式：活动发布平台高净值活动电子凭证－数字纪念物品，IP 活动活动形式及周边衍生，活动IP数字物品及周边。

4. 图书出版

代表形式：撰写人版权，发行人确权，稀有作品发布等。

5. 体育

代表形式：个人 IP 发售，团队 IP 发售，精彩瞬间铸造，周边及衍生数字物品，体育用品，故事情节等。

6. 文创

代表形式：动漫 IP，二次元 IP，文创物品等（例如故宫的文创）。

7. 奢侈品

代表形式：实物数字编号结合，粉丝认证，数字产品。

8. 电竞行业

代表形式：战队 IP，个人 IP，物品 IP 等。

9. 游戏行业

代表形式：经典道具，人物，情节，IP等。

10. 金融行业

代表形式：股票，私募，基金确权等。

11. 时装行业

代表形式：设计样式，版权，周边，等。
