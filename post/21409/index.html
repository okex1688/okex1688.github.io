物以稀为贵。

独一无二的物品总是能吸引着收藏家的关注，他们也愿意支付高价购买藏品，不管是绘画、棒球卡、邮票、古董卡、以及稀有货币等。现在，随着区块链技术的发展，这些项目正在进入数字领域。

在几种不同区块链网络上，绘画、棒球卡、邮票、古董卡、以及稀有货币等项目被称为“唯一数字资产”，即非同质化代币 NFT。区块链技术的引入，意味着这些项目的真实性和所有权可以被轻松验证，尤其是稀缺性在资产评估中起着至关重要的作用。

今年三月，英国最具标志性的街头艺术家 Banksy 将一副价值 7 万英镑的艺术品「Morons」（白痴）在纽约布鲁克林一处秘密地点销毁，销毁过程向全球观众直播，之后他将这幅作品制成了 NFT，因为他认为数字化艺术形式将是行业未来。这幅价值 7 万英镑的作品内涵旨在嘲笑收藏家购买昂贵艺术家，最终 NFT 以近 40 万美元的价格被收储。

正如 NFT 数字艺术品平台 Blind Boxes 首席执行官 Anita Moore 所说，通过去中心化和真实性验证的概念，NFT 改变了人们对所有权和价值的思考方式。

什么是 NFT 交易卡？

NFT 交易卡是一种基础资产的虚拟表示，由于所有信息都是通过区块链存储，因此 NFT 交易卡可以被赋予不可变性和所有权的公共验证，即使物理版本丢失或毁坏，只要 NFT 存在，那么就能一直在区块链上“生存”下去。

人们可以在以太坊、或是其他智能合约区块链上创建代币，并用这些代币来代表这些 NFT 交易卡的虚拟表示，这些代币是“非同质化”的，其中可以包含NFT交易卡的元数据，比如交易卡图像，所有这些都可以通过支持 NFT 的钱包进行存储、查看和传输。

人们可以在许多平台上创建、购买和出售 NFT 交易卡，目前最大的一些交易平台包括 OpenSea、SuperRare 等。

作为一种投资标的，数字藏品行业正在快速增长

交易卡，本质上与艺术品非常相似。

作为一种独特的数字藏品，数字交易卡进入数字领域的步伐正在加快，而且已经成为全新的投资类别。不仅如此，就算在加密货币领域之外，交易卡销售也一直非常火爆。

NFT 平台 Ethernity Chain 创始人兼首席执行官尼克·罗斯（Nick Rose）认为，越来越多名人和有影响力的行业大咖开始支持NFT代币，也给这种新兴资产赋予了真实价值，他解释说：

“很明显，来自真实世界中的体育人士和娱乐人士都开始拥抱NFT收藏品，这些名人背书给 NFT 市场注入了真实性，比如滑板运动巨星Tony Hawk、拳王 Muhammad Ali、以及传奇足球运动员球王贝利，从而为 NFT 创造了实际潜在价值。”

在 NFT 藏品领域中还有许多其他成功创业的例子，比如 NFT 交易平台 StockX 上的 Mint Condition Card 价格从一年前的 280 美元跃升至当前 775 美元均价。最近，一张美式橄榄球巨星汤姆·布雷迪（Tom Brady）交易卡在 StockX 平台上以 130 万美元的价格售出，因为这张卡非常稀有，仅在他新秀赛季发行了 100 张。

许多传统投资者已开始转向风险较高的资产类别，例如加密货币和区块链企业。但是，当被问到最新的 NFT 趋势时，区块链游戏平台 Hoard Exchange 首席执行官 RadekZagórowicz 提出警告，他认为盲目跟随“炒作”会非常危险：

“NFT 是一场新的数字革命，但与每一项新技术一样，NFT 可能经常会被滥用。一些项目可能只是为了把 NFT 当做噱头或作为促销策略，而不是去探索其真正的价值，投资者必须非常小心。”

为了遏制新冠病毒疫情，许多国家不得不选择封城，这一措施导致数以百万计的人员被解雇，也给经济增长带来的负面影响。在这种情况下，美国、巴西、德国和日本等国家都大幅下调利率并购买了政府债券，结果导致投资者对传统金融产品吸引力不断降低。

与此同时，美联储实施的货币量化宽松还引发了人们对未来恶性通货膨胀的担忧，让传统投资者更加远离股票和法定货币而转向贵金属和加密货币，其中他们最感兴趣的就是 NFT。

主流艺术家、企业开始试水 NFT

到目前为止，NBA Top Shot 已经吸引了超过 23 万名NBA球迷加入，他们都渴望拥有 NBA 球星“高光时刻”的 NFT。另一方面，美国职棒大联盟 MLB 及其球员也已经宣布与本土交易卡制造商 Topps 合作推出全新 NFT 交易卡。

三月底，NBA Top Shot 开发团队 Dapper Labs 宣布完成一笔 3.05 亿美金融资，投资机构包括 Coatue Management、篮球巨星迈克尔·乔丹、凯文·杜兰特参投。Coatue 是一家市值高达 10 亿美元投资公司，目前持有 Airtable 和 Instacart 等公司股权。这些投资机构对 Dapper Labs 产生兴趣主要是因为近期 NFT 热潮引发市场关注，而 NBA Top Shot 是目前用户量最大 NFT 平台之一。根据 The Information 称，目前 Dapper Labs 最新一轮融资其实还没有完成，如果后续有更多投资者加入的话，预计其估值还会进一步提升。

YouTube 网红明星洛根·保罗（Logan Paul）也加入“NFT 热潮”，通过出售 3,000 张 NFT（包括 Pokemon 交易卡）获得了超过 500 万美元的收入。另一方面，市场对NFT的需求在一定程度上是受游戏的推动，比如 Pokemon 交易卡就可以直接加入游戏，其中三个首版 NFT 套件估值已经达到 40,000 美元。

越来越多艺术家、名人和主要体育组织也开始加入 NFT 交易卡行列，以此吸引新的受众，但许多人认为 NFT 才刚刚开始展现其真正的潜力。 去中心化应用程序开发平台 Ava Labs 总裁 John Wu 解释说：

“目前我们仍然只是在挖掘 NFT 的潜力，但是许多全球知名品牌已经展现对 NFT 的兴奋，这点非常真实。从体育到活动，NFT 项目覆盖了许多领域，并看到许多有潜力的收藏品正在适应互联网经济发展。”

真正的资产所有权不是终点

实际上，在实体世界里，可收藏的交易卡行业已经存在了很长时间，之所以没有发展壮大，很可能由于身份验证和保存模式不成熟有关——区块链技术可以为这些问题提供解决方案。更重要的是，随着交易卡的稀缺性价值能够被充分体现，区块链技术已经开始给整个收藏市场带来改变。

尽管有些人通过参与 NFT 这个新兴市场而发了大财，但需要注意的是，这个行业也确实存在很高的风险。2020年，实体体育明星交易卡和纪念品市场价值估计刚刚超过 54 亿美元，相比之下，单单 Dapper Labs 这一家数字交易卡公司的市值就达到 26 亿美元（一说投后估值已升至 75 亿美元），但其销售总额“只有”大约 2.3 亿美元（数据截至 2021 年 2 月底）。

从数字上来看，目前 NFT 市场的确有很大发展空间，NFT 也的确包含可以推动市场进一步发展的宝贵资产。此外，虽然 NFT 是“唯一的”，但并不意味着 NFT 对应的内容也是唯一的，NFT 只是一种收藏品的数字证书和真实性证明，但现实中有人其实并不在乎是否需要拥有一个所谓的“真实性证书”，但也有不少人需要这样的“证明”。就像比特币一样，虽然你可能不需要一个真实存在的实体硬币去“证明”比特币的存在，但至少需要一个可以存储比特币的钱包，这样才能“证明”你持有资产的所有权。

比特币、以及其他加密货币和加密代币的作用之一，就是让人们“自己真正拥有自己的钱”——现在，NFT 也发挥了相同的作用，只是其标的不是“钱”，而是其他资产类别。可以说，NFT 的实际应用范围可能会更广泛，正如一站式 NFT 交易平台 APP Curate 首席执行官James Hakim 所说：

“我们正朝着更具实用性的 NFT 迈进，而不仅仅是去标记数字艺术。很快，我们将在 NFT上做更多的事情，例如质押、支付、认证和忠诚度奖励。”
