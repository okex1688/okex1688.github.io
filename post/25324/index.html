Gavin Wood 在柏林 Web3 峰会上

作为一个天生好奇的小孩，当 Gavin 的母亲自作主张给他弄来一台邻居的旧电脑时，9 岁的 Gavin 非常兴奋。不过，她应该不知道这台旧东西会激起他的好奇心，让他成为建立以太坊、Solidity 和 Web3 基金会，为互联网新时代奠定基础的先驱。2014 年，他的好奇心将他带到了迈阿密，在那里他与 Vitalik Buterin 和其他六人共同创立了去中心化智能合约平台以太坊。

**做以太坊之前 **

Gavin James Wood 博士虽然继承了他母亲对电力事故的恐惧，但后来逐渐喜欢上了技术和博弈论。他对计算机和技术的热爱，进一步得到了他在兰开夏郡的邻居肖恩的帮助，肖恩教他编写了第一个计算机程序，后来兰开斯特皇家文法学校的老师激励他忠于自己的热情。从兰开斯特皇家文法学校毕业后，他继续攻读软件工程，并在约克大学攻读工程学硕士学位。至于博士学位，他在研究生母校约克大学选择了人机界面 (HCI) 的音乐可视化作为首选研究领域。

Wood 博士在 2000 年代初期继续积累他的开发者经验，使用可延展且可扩展的音频信号细化和 Exscalibar 微调架构。完成博士学位后，他为 Frontier Developments 开发了下一代跨平台游戏引擎。2007 年，他创办了 Quid Pro Code，这是一家软件商店，在那里他设计并实现了第一个 C++ 语言工作台 Martta，同时他就嵌入式领域特定语言的技术方为 Microsoft Research 提供了咨询。2011 年，作为 Lancaster Logic Response 的技术总监，他利用博士学位的技术开发了一个系统，用于为音乐创建实时灯光秀，并部署到伦敦的一家顶级夜店。在 LLR，Wood 博士还参与了 Noted 的开发，这是一种可扩展的音频分析和符号 C++ 环境。

以太坊时期

2013 年，Wood 博士正在建立 OxLegal，当时他通过一个共同的朋友认识了 Vitalik Buterin，并开始从事一个不在他的博士研究课题范围的项目。在 Buterin 的帮助下，他开始做一个去中心化平台，该平台与现有的加密货币比特币非常相似，但在互联网去中心化的愿景和追求方面截然不同。2014 年 1 月，Gavin 与 Buterin、Charles Hoskinson、Anthony Di Iorio、Wendell Davies 和其他人一起窝在迈阿密的海滨别墅里。在那里，他编写了以太坊的第一个功能实现，称为 PoC-1 或 alpha 版本）。他们五人将在 2014 年 1 月在迈阿密举行的北美比特币会议上展示 PoC-1。Wood 博士成为联合创始人和首席技术官，撰写了以太坊黄皮书，黄皮书中提出了以太坊虚拟机规范，它是编写区块链状态机的第一个正式规范。

在接下来的两年里，Wood 博士全心专注于以太坊的开发，其中包括通用平台架构、大部分 C++ 以太坊客户端以及 Solidity 的初始设计，Solidity 是一种用于编写以太坊使用的智能合约的面向对象的编程语言。

到 2016 年，Wood 博士扩大了他的关注范围并创立了 EthCore，并迅速筹集了前种子轮资金。EthCore 在 15 个国家/地区拥有 60 多名开发人员，后来发展成为 Parity Technologies。

**从以太坊到 Parity **

成立以太坊两年后，Wood 博士与以太坊校友 Jutta Steiner 博士（安全审计负责人）、Aeron Buchanan 博士（财务和运营负责人）和 Ken Kappler（通讯主管）继续前行。在 Parity，他构思了许多项目来创新当前的去中心化应用程序环境，从最快和最先进的以太坊客户端 Parity Ethereum，到 DIY 区块链框架如 Substrate，这是一种灵活的技术堆栈，让用户能够从零构建自己的区块链。

Wood 博士与 Ewald Hesse、Ana Trbovich 博士和其他三位区块链先驱共同创立了 Grid Singularity，这是一个基于区块链技术的开放式互联网去中心化能源数据交换平台，延续了他的去中心化互联网愿景。2016 年，他还与 MelonPort AG 一起参与了多个区块链的去中心化资产管理。

Wood 博士创立了 Web3 基金会，以培育去中心化 Web 软件协议领域的技术和应用，通过在去中心化网络中应用现代加密方法来稳定 Web 3 生态系统。

Web3 基金会支持对 Whisper 协议的研究，Whisper 协议是 Wood 博士于 2014 年构想的，该协议有望在未来的 Web 3.0 中实现秘密的分布式消息传递。

Web3 基金会推出了 “金丝雀网络” Kusama，允许开发者在真实经济环境中探索 Polkadot 更精细的复杂性，然后再在主网上推出他们的技术。Kusama 的发布被证明是 Polkadot 网络启动的宝贵转折点，Polkadot 网络是 Wood 博士在离开以太坊后共同创立的分片多链互操作性网络。

当 Gavin Wood 博士不痴迷于推动区块链基础设施的成功时，他会抽出时间滑雪，并在 Medium 和 Twitter 上分享他对行业趋势的见解。
