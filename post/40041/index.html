**日本加密资产交易市场 **

对于任何进入加密市场的投资者，其加密生涯的第一站是中心化交易所。

1.1 日本国民如何参与加密市场投资

在日本国内，购买加密资产需要开通获准加密资产交易商个人或者法人加密（暗号）资产账户，需要完成实名认证（KYC）并且居住地在日本国内，即可绑定日本的银行卡，购买并交易加密（暗号）资产，日本国内凡是满 18 岁到 80 岁（含）的用户都可以开通加密（暗号）账户。

需要注意的是日本所有加密资产交易商仅提供各种代币和法币（日币、美元）的交易对，不提供任何代币交易对服务，类似与股票市场。比如支持购买 BTC/JPY、BTC/USD，但不能购买 BTC/ETH 币对。

1.2 日本加密资产监管制度

日本加密资产监管由行政监管机构与自律监管组织负责，监管对象为加密资产交易商、加密资产投资基金公司和云挖矿企业，监管范围涵盖加密资产现货与衍生品交易、ICO 业务，其加密资产监管制度在全球处于领先水平。

1、行政监管机构：日本金融厅、国家税务管理局

监管内容：日本金融厅负责加密资产交易商注册登记，制定与修改加密资产和ICO 业务等方面的监管法规，加密资产的业务报告审核和信息披露;日本国税局则负责制定加密资产方面的税收制度与执行细则。

2、自律监管组织:日本暗号資産ビジネス協会（JCBA）、日本暗号資産取引業協会

监管内容：制定自律规则、监管和向会员单位提供信息、对会员单位进行业务指导、建议和处置、接受会员单位的商业咨询、加密货币业务投诉处置、加密资产相关衍生品业务外包给证券和金融工具保证咨询中心（FINMAN）、加密资产交易统计调查。

1.3 日本国民参与加密市场投资注意事项

1、日本国民如果用虚拟货币获利必须进行纳税，虚拟货币的利润被归类为杂项收入，如果收入（从利润中减去必要费用所得的金额）为 200,000 日元或以上，则需要进行最终纳税申报。计税方式为综合计税。

**日本 NFT 市场分析 **

3.1 日本仅有 8000 名 NFT（PFP）买家

3.3 对于日本 NFT 市场的一些思考

1、宏观经济

自金融危机后，日本国长期维持低通胀环境，日元保值能力稳定，也形成了日本央行长期维持宽松的状态和预期，然而受到俄乌冲突造成的大宗与原材料价格抖升，日本作为长期依赖进口原材料国家，“输入性通胀”造成的被动压力过强，这是日本国内通胀与全球通胀的矛盾。

另外日本央行和市场也存在矛盾，日本央行行长黑田东彦坚持“安倍经济学”，倚靠低利率来支持疲弱的经济，容忍“弱日元”来促进出口市场，日元的贬值增加了进口成本，这导致通胀压力在聚集，同时美国的滞胀压力以及美联储加息的上升，美债利率、美元指数持续走强，日本市场正面临强势的“美元回流”，这或许是麻绳专挑细处断，噩运只找苦命人。

需要认识到日元的贬值颓势任在继续，日本部分国民开始将自己的储蓄与投资转换为美元以及尝试投资加密资产，令人兴奋的是，参与上述 NFT 项目的日本加密投资者都获得超额利润。

2、项目方布局日本 NFT 市场

在日本，仅 8000 人在购买 NFT（PFP 向），每人的投资预算大概是 1 ETH 左右，在此不建议对日本市场进行专门营销，市场规模顶多 8000 ETH，况且市场上有更多的 NFT 项目在消耗这些购买力，但我会给出一些后续布局日本 NFT 市场的建议。

第一个建议 ——等待市场成熟，在买家数量如此少的情况之下，没有必要推出项目，如果再等六个月，日本市场的 NFT 买家数量会增加到 20000 左右，且时以太坊主网升级，也利于更多买手进入到 NFT 市场。

第二个建议 ——降低售价以及免费铸造，这种销售策略有以下优点：

  • 可以吸引新手 NFT 投资者；

  • 在发售后（MINT）立即启动交易，这利于海外交易者通过交易量排名参与投资；

  • 价格越底会更容易产生的投资倍数，就像 0.01 到 0.1 比 0.4 到 4 更加容易，而这种多倍投资回报会形成更强的 FOMO 效应；（图源A16z的分析报告）

第三个建议 ：采取 High supply collection，单钱包多 Mint 策略；

第四个建议 ：为保持项目活性，请在短周期内采取销毁原生 NFT 孵化新 NFT 策略；

第五个建议 ：近期一个叫做 Flower Lolita Collections 针对日本市场发行的项目发生了破发，这是一个成熟的项目，经营主体明确，设计也是原创，营销也很扎实，出现这种情况是令人意外的，原因也很简单——忽视了社区本土意见领袖的培养，广为人知的是日本国民对于英语的心理障碍强烈，请务必做好项目本土化，参考 STEPN 在日本的成功，不难察觉本土意见领袖的羊群效应十分显著；

第六个建议 ：日本人普遍金融知识水平较低，投资者教育会是一个切入方式；

第七个建议 ：与日本 KOL 合作并不是最优选择，日本 KOL 相当谨慎细微，对项目助力能量有限，尝试利用 CEX 机构信誉，顺带一提，日本投资者私下最愿意使用 BINANCE；

3.4 关于个人投资者布局日本 NFT 市场的三个建议

第一个建议 ：可以将网络 IP 切换为日本登入推特，算法推荐也会有更多惊喜；

第二个建议 ：值得关注的日本 NFT 向 KOL；

第三个建议 ：保住本金。
