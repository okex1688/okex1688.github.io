截至8月15日，30日间成交价格排名前五的 NFT 均来自 CryptoPunks。以太坊伦敦升级后，24小时内燃烧 ETH 最多的地址与加密朋克是什么关系？“仿盘”本身有没有突围之路？本文将尝试向各位读者解释「加密朋克」的概念及回答上述问题。

仿盘首次“出圈”

8月5日20:33 (UTC+8)，以太坊于区块高度12,965,000完成伦敦升级，EIP-1559 提案也随之生效。这意味着以太坊网络开始根据交易需求和区块大小来动态地调节每笔交易的基础费用，而这部分费用将直接销毁，某个应用一段时间内销毁数量的排名一定程度上也反映了它在以太坊生态内的相对活跃度。

如下图所示，截至以太坊伦敦升级完成后的11小时内，已经燃烧超过2899枚以太坊（截至8月18日15:00已燃烧59,632枚以太坊），其中燃烧以太坊最多的地址（0xE4CfAE3Aa41115Cb94cFF39Bb5dBaE8BD0Ea9d41）当时尚未进行标记，后经查实，该地址为 CryptoPunks 仿盘——COVIDPunks 的地址，引起市场极大关注度。在以太坊燃烧仪表盘上短暂与 OpenSea、Uniswap、Axie Infinity 及其他稳定币等以太坊头部应用并列，这在某种程度上意味着它可能从 NFT 加密头像的小众圈子跳脱出来浮现在更为广泛的加密圈子面前。

仿盘“仿”的是什么？

COVIDPunks 的灵感源自原生赛博朋克艺术品项目 CryptoPunks。2017年，CryptoPunks 由 Larva Labs 推出，其创始人是 John Watkinson 和 Matt Hall。起初，John 制作了一款“像素角色生成器”，本想用于游戏，但均被否定不可行。他们后来了解到以太坊并将此带至区块链领域，于是诞生了 CryptoPunks，他们称之为这是一场实验，因为彼时的技术和人们对虚拟收藏品的接受度都是影响这场实验能否持续的不确定性因素。但作为历史存续时间最悠久的加密头像 NFT 项目，多个因素推动 CryptoPunks 成为一场“成功的实验”：

  • 社区共识，自2017年起坚持运营5年，拥有强大的社区，被行业 kol 作为 Twitter 和 Discord 个人头像；

  • 先发优势，作为以太坊上第一批具有验证所有权内置方法的数字艺术品，CryptoPunks 能比其它加密藏品捕获更多价值；

  • 资本青睐，除了 NFT 信仰者将 CryptoPunks 作为币本位的基础资产外，新入场的资本如 3AC 等机构也会优先选择资金承载量大的 NFT 资产。

以 COVIDPunks 为代表的仿盘很难短期内获得如上所述三大优势，但从其项目本身特质来说，都是基于 CryptoPunks 项目做出部分元素改变或添加以及叙事性方面展开的想象。从估值来说，市场最高承接度与 CryptoPunks 也高度统一，稀有度分布特征基本一致。

一般而言，CryptoPunks 的估值通常由两部分构成：

  • 一部分是其 NFT 本身的客观稀有度，包括 CryptoPunk 的类型、属性的数量及属性的独特性；

  • 另一部分则是基于客观属性之上出价者的主观审美。

以 CryptoPunks 历史成交价最高的 NFT 为例，从客观稀有度来说，其类型属于9/10000的外星人类型，这本身就体现了它们的绝对稀有度，加之最高出价者所青睐独特的属性如眼睛和烟斗等，则构成了大众对于加密朋克类型估值最高 NFT 的一般印象。因而在仿盘历史成交价最高的 NFT 上，我们可以看到较为同质化的特征：

  • 清一色的外星人；

  • “晶莹”或黑色的眼睛；

  • 一字型的嘴巴以及硕大的烟斗。

但同时仿盘也做出了一些“微创新”调整，可以看到最高价的 COVIDPunk #2338 是在 CryptoPunk #7804 的基础上增加了其最显著的口罩元素；而最高价的 Fast Food Punk #001 则是将 CryptoPunk #3100 的发带元素替换为麦当劳帽子。

从叙事性来说， COVIDPunks 加入了时代特征，融入口罩元素，象征着新冠或将与人类长存或是纪念抗疫。同时，COVIDPunks 也作为富有叙事性的反射性资产 (Reflexive Asset)，继承了 CryptoPunks 易于波动的特征，因为叙事往往是反复无常的。

仿盘百花齐放“反哺”生态

加密朋克“仿盘”的出现相当于降低了整个加密朋克 NFT 生态的地板价 (Floor Price)，大幅降低了 NFT 的入门门槛，对做大群体相当有益。

从上图可以看出，每一个仿盘后来者的发行价都不可避免地低于此前的相对同质化盘子，从另一个角度来说，具有一定热度的加密朋克仿盘（OpenSea 前150名的 NFTs）构成的整个生态，其成交均价一直维持在一个相对平滑的低水平区间。相对而言，整个仿盘生态的地板价实际上也就维持在相对平滑的较低区间，极大地降低了朋克生态的门槛，这给整个加密朋克生态乃至加密头像圈带来更多的受众。当然这其中一部分用户会往下沉淀，同时另一部分用户也会上浮到原生“正品”盘子中反哺生态。

仿盘的经济周期及影响与今年上半年的狗狗币、SHIB 等 Memes 类似。狗狗币今年把很多本不会买比特币和以太坊的人带进了圈子，这其中一部分人下沉到狗狗币的诸多仿盘或是可以加杠杆的合约中，其中部分群体用户通过赚钱的正反馈过程更加深入地了解了加密货币生态，从而配置了一部分比特币和以太坊抗风险资产，这也相当于反哺了生态。

从上图可以看出，加密朋克及其仿盘成交额体现一定的周期性和传递性，从存续时间最久的 CryptoPunks 到存续时间最短的仿盘依次爆发；其次，存续时间越久的朋克项目在新的仿盘诞生时越有可能第二次爆发，CryptoPunks、Fast Food Punks 和 COVIDPunks 均在存续时间最短的前三个项目爆发时同时二次引燃，从成交额角度来说，有较大可能是仿盘新引入的流量与人口对生态的一个反哺。

截至8月15日，从30日内 NFT 的成交价情况来看，NFT 成交价排名的前五位均被 CryptoPunks 纳入怀中，加密朋克的中长板不断延伸。从地板价来看，在其仿盘“百花齐放”的这半个月中，其地板价自 21.9ETH 涨至 44ETH，无论是仿盘引起的目光聚集，还是 3AC 等顶级资本的青睐，整个生态都趋向于正向发展。

对小众长尾的圈子来说，只要做大人口数就是好事。新人口会带来新的关注度、流量和资金，不管是好流量还是坏流量都对圈子有益。仿盘的问题在于流动性不行，大家诟病的只是怕它凝聚不了价值，这块儿市场自然会给出答案。

访谈观点

一艘捕鲸船

关注 Cryptopunk、小企鹅和其他 NFT 收藏品的投资机会时，多注意最高成交价和地板价两个指标。

最高成交价是长板，起到捕获市场注意力，吸引新玩家入场系列作品的作用。地板价是短板，起到作为最低入场门槛，支撑整体市值的作用。类似于炒币时的币价，地板价的涨跌是短期炒作需要关注的关键。

大橙子

CryptoPunks 是一种社区文化，他在早期很好的吸引了一波海外大V，并且在大V内迅速 Fomo，他已经脱离了必须有 IP 才能火的 NFT 传统思维，我觉得它会长期火下去，因为社区支持太强了。

COVIDPunks 一度霸占以太坊燃烧第一，头矿热度极高，不过产品只是简单的 Fork 了前者，而且还打着帮助印度疫情的旗号，但是从他们捐赠数据看可以说毫无诚意，社区在头矿热度后也选择了用脚投票。

陈威廉

仿盘比较不容易在 Opensea 上“加V”，所以价值往往很难与其他原创项目相比。但是作为新手入门，以及散户体验来说，仿盘是拉进人们与昂贵的正版 NFT 项目之间距离的良好过渡。

另外，仿盘的价值核心是关注 Twitter 热度和 Opensea 的成交量。如果想体验同时想参与炒作的话，一定要关注这两个指标。

NFT Labs

现阶段的 NFT 市场并不规范，投机炒作的项目，大量 CryptoPunk 和 BAYC 等项目的仿盘。比如 BAYC 的社区运营在成功后，就被许多项目复制。大量的仿盘会加速 NFT 泡沫，等这轮泡沫破灭后，NFT能迎来新的发展。NFT 在估值上的挑战是因为他包括功能价值(实用性+所有权历史+ 数字稀缺性 + 供求关系 + 未来价值 + 流动性溢价)和享受价值。

结论

  • 仿盘只可仿其表，难仿其本质，且存在和狗狗币类似的经济周期及影响；

  • 仿盘的流动性不行，但它的出现一次次巩固和提升了原生盘子的价值，具有反哺生态的作用；

  • 仿盘本身有没有突围之路并不重要，对于小众长尾圈子来说，做大人口基数本身就是一件好事。
