不可替代代币 (NFT) 正在风靡全球。 在本文中，我们将超越市场炒作，涵盖您需要了解的有关 NFT 的所有信息。

介绍

不可替代代币 (NFT) 正在风靡全球。 Steve Aoki 和 Jay-Z 等名人都自豪地在 Twitter 上展示他们的 NFT 作为他们的个人资料图片。 阿迪达斯和耐克等主流品牌已进入该领域。 阿迪达斯通过推出自己的 NFT 系列筹集了 2300 万美元，该系列可作为获得阿迪达斯独家实物和数字产品的通行证。 耐克收购了 NFT 领域的虚拟设计工作室 RTFKT，以进军元宇宙。 NFT作品正在苏富比和佳士得以数百万美元的价格被拍卖。

随着 NFT 的主流采用越来越多，越来越难以忽视这一上升趋势。 在本文中，我们将超越炒作，涵盖您需要了解的有关 NFT 的所有信息。

什么是 NFT？

不可替代代币或简称 NFT 是独一无二的加密代币，代表存储在区块链上的一组不可互换的元数据。另一方面，可替代代币是可互换的。比特币和以太坊等代币就是可替代代币的例子。 1 比特币 = 1 比特币，用户对他们购买或接收的确切比特币没有区别。每个 NFT 都是唯一可识别的，在大多数情况下，用户会关心他们购买或接收的确切 NFT。

区块链是分布式和去中心化的数字分类账。它们由按时间顺序链接在一起的数据块组成。目前 NFT 最常用的区块链是以太坊。我们也开始看到以 NFT 和游戏为核心构建的利基区块链的兴起，例如 Flow 和 Ronin。

NFT 的元数据可以在存储在区块链块中的智能合约中找到。关于 NFT 的一个常见误解是，显示的图像或代表 NFT 的 JPEG 存储在区块链上。虽然对于少数项目（Anonymice、Chain runners）确实如此，但大多数项目将图像存储在外部存储平台（IPFS、Arweave）上，并且区块链上的元数据包含指向该图像的链接。由于图像的文件大小，将图像存储在区块链上的成本非常高，这使得项目完全上链是不经济的。

NFT 解决了哪些问题？

互联网通过数字化一切来解决物理世界的问题。拥有无限规模并与全球观众建立联系比以往任何时候都容易。分发成本也大大降低了——通讯vs.报纸。但是，它缺乏物理世界的好处。所有权和稀缺性几乎不存在，与实物商品相比，验证数字资产的能力要困难得多。尽管“右键单击并保存”NFT 是一种模因，但它反映了互联网的当前状态。数字资产没有所有权，价值增值很少。

NFT 的核心是提供当今数字经济中缺少的所有权证明。它创造了将数字世界的好处（全球受众、规模和分布）与物理世界的好处（所有权和真实性）结合起来的能力。

在 NFT 兴起之前，数字资产很难积累和获取价值。这使得它对用户来说非常便宜，但对创作者不利。今天很难在互联网上追踪和归属真正的所有权，而 NFT 解决了这个问题。创作者因其工作而获得更准确的奖励，并且能够从他们创造的价值中提取更大比例的价值。用户受益于能够获得对数字资产的所有权并能够轻松交易和评估它们。成为当今世界上最大乐队的早期粉丝并没有给你带来额外的好处，但有了 NFT，消费者就有可能通过做他们每天已经做的事情来获得回报。

NFT 有什么用途？

艺术

NFT 最流行的用例是将 NFT 作为可收藏的艺术品。 NFT 为艺术家提供了一种在确保所有权的同时轻松将其艺术作品出售给全球观众的方法。艺术 NFT 可以大致分为两类：非生成艺术和生成艺术。

非生成艺术

当我们想到艺术时，我们通常会想到非生成艺术作品。它们是由人类在没有自主计算机系统的帮助下创建的。数字艺术家可以在 NFT 市场上列出他们的作品，世界上任何地方的任何人都可以查看和购买他们的作品。在今天的传统市场上，艺术家只能从作品的初始销售中获得收入。 NFT 让艺术家可以选择将版税嵌入到他们作品的智能合约中。这使他们能够更公平地分享其艺术作品的价值。 NFT 使验证艺术品的真实性和所有权变得容易，这是该技术带来的巨大好处。该领域的著名名称：Beeple、pplpleasr、ferocious、Pak。

照片 NFT 是 NFT 的一个不断上升的用例。摄影师将他们的作品作为 NFT 出售，并获得与将作品作为 NFT 出售的艺术家类似的好处。 Justin Aversano 是该领域最知名的人物之一。当他的“双生火焰(Twin Flames)”系列中的一件 NFT 在佳士得拍卖时，他创造了历史。此类 NFT 的热门市场是 Opensea、SuperRare、Foundation、Rarible 和 Nifty Gateway。

生成艺术

除了由艺术家单独绘制和创作的艺术之外，艺术 NFT 的兴起很大程度上归功于生成艺术。 生成艺术是通过使用自主计算机系统（通常是算法计算机程序）创造的艺术。 诸如 CryptoPunks 和 Bored Ape Yacht Club (BAYC) 之类的个人资料图片 (PFP) 集合是在此类程序的帮助下创建的。 创作者完成了个体特征，程序通过整合这些特征随机组合并创建独特的角色。 有完全由计算机程序完成的完全生成的收藏品，除了创建算法之外没有人工输入。 此类别下最受欢迎的系列是 Art Blocks 的子系列，例如 Snowfro 的“Chromie Squiggle”和 Tyler Hobbs 的“Fidenza”。 OpenSea 是生成艺术 NFT 的领先市场，而 LooksRare 是一个受欢迎的新推出的NFT市场。

游戏

艺术 NFT 是受到主流关注的用例，然而，就销量而言，有一个收藏品超过了所有艺术 NFT 收藏品。 NFT 游戏 Axie Infinity 目前占据桂冠（不包括 LooksRare 上的清洗交易）。 NFT 技术与游戏的结合催生了一种新的游戏范式：Play-To-Earn/Play-And-Earn/GameFi。

音乐

音乐 NFT 是 NFT 的新兴用例。就像艺术 NFT 一样，音乐 NFT 是艺术家获取他们创造的大部分价值的关键。艺术家通常只获得他们为音乐行业创造的收入的 12% 左右。制作人/艺术家 Daniel Allan 在 Opensea 上以 1.1 ETH 的价格出售了他制作的音乐 NFT，相当于 Spotify 上的 120 万个流。

真正的粉丝也受到这种趋势的影响。他们能够直接从艺术家那里支持并拥有他们最喜欢的音乐艺术家创作的作品的副本。这为艺术家创造了新的方式来吸引他们的粉丝，达到现有方法难以实现的水平。通过拥有艺术家作品的原件，粉丝们能够与艺术家未来的成功息息相关。艺术家得到更公平的回报，作为粉丝，你可以在早期参与支持艺术家的好处。 Sound.xyz 和 Catalog 是两个项目，它们为艺术家提供了一个平台，可以将他们的作品直接出售给他们的粉丝。

体育收藏品

体育 NFT 是 NFT 的一个不断增长和蓬勃发展的用例。与棒球交易卡类似，体育 NFT 允许球迷建立他们最喜欢的球队和球员的收藏品。目前最受欢迎的两个项目是 NBA Top Shot 和 Sorare。两者都与相关组织建立了官方合作伙伴关系，并拥有官方许可证。

NBA Top Shot 本质上是在 NBA 比赛期间创建关键时刻的 GIF，并将它们随机捆绑到数字卡包中，每个时刻都是 NFT。根据玩家、重要性和当下的病毒式传播，卡包分为不同的稀有度。时间越少，供应就越少。一包普通物品的起价为 9 美元，传奇物品的起价为 999 美元。用户可以在 NBA Top Shot 的官方市场上交易 NFT。勒布朗·詹姆斯在 2020 年总决赛中赢得总冠军的传奇扣篮以 23 万美元的价格售出，是迄今为止的最高成交价。

当梦幻足球遇到 NFT 时，您会看到Sorare。玩家卡是 NFT，奖励以 ETH 支付。用户可以在 Sorare 市场上交易球员卡。 Sorare 的好处类似于 NFT 游戏，对游戏内资产的所有权以及兑现和从游戏中提取价值的自由。

通行证

由于易于身份验证和验证所有权的能力，NFT 可以充当访问通行证。 此类系列的一个例子是 Flyfish 俱乐部，它充当私人餐饮俱乐部的会员通行证，而阿迪达斯系列则让持有者可以独家访问该品牌的实体和虚拟商品。 虽然有些收藏品的唯一目的是成为会员通行证，但这个用例是许多收藏品的首要主题，所有形式的 NFT 都可以充当各种访问通行证。

促成 Art NFT 兴起的一个关键因素是支持该项目的社区。 NFT 持有者可以独家访问某些Discord频道，并且在社区内有一种友情。大型项目 (BAYC) 为拥有 NFT 的人举办了实体聚会和独家派对。持有某些 NFT 可能会让您提前获得来自同一团队或通过与新项目合作推出的新 NFT 收藏品。与阿迪达斯 NFT 类似，一些系列（BAYC、Creature World）已经发布了自己的实物商品，并且 NFT 的持有者获得了早期/独家访问权。音乐艺术家和游戏能够为其 NFT 的持有者提供类似的福利。

去中心化自治组织 (DAO) 和游戏公会是 NFT 可以充当访问通行证的另一个领域。相关 NFT 的所有权将为您提供独家会员访问权限，并可用作投票代币。 Yield Guild Games 就是以这种方式使用 NFT 的游戏公会的一个例子。 NFT 的持有者可以访问网站上的独家功能，并在公会中注册各种计划。

你如何创建 NFT？

创建 NFT 的方法有很多，您可以手动完成，也可以借助 OpenSea 和 Rarible 等 NFT 市场创建的系统来完成。除了创建 NFT 的技术性之外，要考虑的重要一点是您想要在其上构建 NFT 的区块链：以太坊、Solana、Avalanche、Polygon 等等。不同的区块链有不同的权衡和社区，这会对您的项目的成功产生很大影响。

OpenSea 在此处提供了有关如何使用其平台在以太坊上轻松创建自己的 NFT 的精彩指南。

如何购买 NFT？

就交易量而言，最大的市场是 OpenSea，它很可能是您 NFT 之旅的起点。虽然当今最流行的 NFT 收藏品是建立在以太坊上的，但在 Solana、Terra 和 Polygon 等其他区块链上也有蓬勃发展的项目。其他受欢迎的市场包括适用于 Ethereum/Polygon 的 LooksRare 和 Rarible，适用于 Solana 的 Magic Eden 和 Solanart，适用于 Terra 的 Random Earth。

您需要设置一个与构建 NFT 项目的区块链兼容的加密钱包，以便与市场进行交互。最常用的钱包是以太坊的 MetaMask、Solana 的 Phantom 和 Terra 的 Terra Station。设置好钱包后，将 ETH/SOL/LUNA 从您选择的中心化交易所转移到钱包中，您就可以开始了！

你如何选择 NFT 进行投资？

在购买 NFT 作为投资时，重要的是要从整体上评估项目，而不是仅仅依靠它的外观。与挑选要投资的加密项目类似，创始团队的质量、经过验证的执行和交付能力以及项目的路线图是需要考虑的重要因素。也许最重要和最难衡量的因素是该项目设法建立的社区的实力。 NFT 的价值主要来自其社区，拥有强大社区的项目将能够承受看跌期。我们甚至看到社区通过在创始团队失败的地区交付项目而团结起来并恢复项目的案例。这种情况的最新例子是矮胖企鹅。

真正衡量一个社区的实力的唯一方法是与他们同在，成员的质量比数量更重要。加入Discord，在频道里闲逛，感受成员的氛围。如果谈话主要围绕底价而环境是有害的，那么这可能不是你想赌的项目。

对 NFT 的常见批评

“为什么要花数千美元购买我只需右键单击并保存的 JPEG 文件？”

是的，你可以通过这样做获得艺术作品的副本，但这并不等同于拥有实际的 NFT。就像假冒奢侈品永远无法接近正品的估值一样，即使材料和制造质量完全一样，“右键保存”版本的 NFT 也永远不会具有相同的价值。仿冒品或在这种情况下，“右键单击保存”版本只会增加原件的价值。它创建了一个反馈循环 - 原始项目被复制 > 获得病毒式传播和吸引力 > 价值上升 > 项目被更多地复制。在世界范围内看到的特定图像越多，原始副本就越具有病毒性，更具标志性和更有价值。推动蒙娜丽莎成名的是这幅画在被盗时的传播力，而不是它是由达芬奇绘制的。被贴在世界各地报纸的头版上，使它在国际上享有盛誉并成为家喻户晓的名字。

您可以争辩说，实物仿冒品与原品价值不一样的原因是它不是 100% 相同的物品。这对于实物商品是正确的，对于 NFT 也是如此。这个概念更难理解，因为 NFT 是完全数字化的，而且从表面上看，复制的 JPEG 与原始 100% 相同，但是，就像实物一样，真正重要的是下面的内容。 “右键保存”版本在区块链上不会有相同的底层数据。它不会使您享有拥有真正 NFT 所提供的相同利益和权利。它不会授予您访问事件或地点、访问拥有 NFT 的个人社区的访问权、在市场上转售作品的能力、访问游戏内体验等。

NFT 的实施并不意味着用户必须支付更多费用，它只是意味着用户可以选择。不想为 JPEG 支付数千美元？当然，如果您愿意，可以免费下载并使用它。那些想要通过购买和拥有他们的作品来支持他们最喜欢的创作者的人现在有能力以比现有方法更公平的方式这样做。看看 OpenSea 的销量（2021 年的销量约为 170亿美元），很明显有相当数量的资本对此感兴趣。‍

“NFT 对环境有害”

虽然以太坊等工作证明 (PoW) 区块链确实是能源密集型的，但许多较新的区块链甚至以太坊本身都在转向权益证明 (PoS)。诸如 Solana 之类的 PoS 区块链消耗的能源要少得多，并且可能是碳中和的。最重要的是，NFT 不太可能导致额外的能源使用，因为如果 NFT 不存在，其他应用程序（例如 DeFi）可能会占用区块链上的任何空闲交易容量。

NFT 的未来

未来的 NFT 可能看起来与我们今天看到的完全不同。虽然 NFT 已将自己确立为艺术家可行且增值的工具，但我们仅处于利用该技术的起步阶段。

鉴于它们的主要功能是所有权代币，NFT 可以是任何资产的代币化表示，包括物理和数字资产。住房契约、学校证明、身份证只是可以通过代币化版本改进的一些潜在项目。由于区块链是不可变的并且数据很容易验证，如果一个人的学术证书和过去的工作经历都存储在区块链上，机构可以无缝地验证它。

NFT 不会作为资产存在于孤岛中，但可以与 DeFi 协议集成以创建一个流动的加密生态系统。想象一个世界，您可以使用 NFT 作为抵押从 DeFi 协议（例如 Aave 或 JPEG’D）中提取贷款。目前，NFT 是相对缺乏流动性的资产，高价位阻止了较小的参与者参与某些收藏品的上涨。 NFT 的碎片化将有助于增加空间的流动性，并允许较小的参与者分享上行空间。跟踪 NFT 总市值的 NFT 指数可以创造新的投资途径，并为想要接触该行业但不想持有特定 NFT 的投资者打开空间。

流行的 NFT 收藏可能会开始成为虚拟世界中非常珍贵的资产，并且可能需要出于广告目的使用 NFT。在现实世界中，品牌与名人接触，利用他们的品牌和影响力来推广他们的产品。特定的 NFT 可能会成为元宇宙的影响者，品牌会为此类情况租用 NFT。

随着时间的推移，成功的项目本身可以成长为大品牌。一旦一个项目在流行文化中巩固了自己，获得了足够的思想份额和文化意义，它们可能会与 LVMH 等传统公司竞争。从该项目中拥有 NFT 成为一种强烈的财富社会信号，并且变得非常受欢迎，类似于拥有劳力士手表。

结论

NFT 为我们提供了一种利用物理世界和数字世界的优势为创作者创建新的分销渠道和货币化模式的方法。它为我们提供了新的方式来吸引和激励志同道合的人社区，并有可能改变我们协调社会群体的方式。它们使我们能够更好地归属和确定数字资产的价值，并使相关的创造者能够从他们创造的价值中获得更大的份额。 “右键单击并保存”将始终是互联网的一项功能，但 NFT 让创作者和用户可以选择拥有所有权并对其资产进行更大程度的分配控制。右键单击并保存也会将价值返回到原始副本，而不是减少它的价值。

目前该技术的实施是有限的，并不完美。但是，这并不能消除未来将出现的长期潜力和未来用例。我们只是处于虚拟经济兴起的第一局，我们有责任找到新的、令人兴奋的方法来利用 NFT 的好处。
