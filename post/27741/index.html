继美国跨国金融服务公司Visa斥巨资15万美元购买Crypto Punk后，其又有了进一步动作——发布了NFT白皮书《Engaging Today’s Fans in Crypto and Commerce》（让如今的粉丝参与加密和商业）。

这份白皮书分享了对 NFT 前景的见解和观察，以及如何评估和扩展 NFT 机会的可操作指南。

如何定义NFT

与加密货币一样，NFT 在区块链上发行，用来指定特定资产的所有权。每个NFT都与一些独特的数据相关联，通常是某种数字化的内容文件（或对其的引用），并由智能合约管理。将媒体文件转换为NFT的过程被称为「铸造」NFT，而且就像加密货币一样，该 NFT 就被写入适用的区块链数据库中。与加密货币不同的是，NFTs是不可替代的，这意味着每个 NFT 是唯一的，不能与其他 NFT 互换

三大NFT应用场景

Visa 认为目前 NFT 的主要应用场景主要是三个领域：收藏品、艺术品和游戏。

收藏品：nft所实现的数字稀缺性自然适合那些价值依赖于有限供应的收藏品或资产。如CryptoKitties、CryptoPunks、以及NBA Top Shot。

艺术品：NFT 使艺术家能够以自然的形式出售他们的作品，而不是必须印刷和出售艺术品。此外，与实物艺术不同的是，艺术家可以在二次销售或拍卖中获得收入，从而确保他们的原创作品在后续交易中得到认可。如 Nifty Gateway。

游戏：由于NFT所引入的所有权机会，它也为游戏提供了重要机遇。虽然人们在数字游戏资产上花费了数十亿美元，如在《堡垒之夜》中购买皮肤或服装，但消费者并不一定拥有这些资产。NFT将允许玩加密游戏的玩家拥有资产，在游戏中赚取资产，将其移植出游戏，并在其他地方(如开放市场)出售资产。

此外，Visa指出，现阶段，销售艺术品和收藏品是NFT的主要应用场景。不过在粉丝经济方面仍有潜在的发展机会。

Visa表示，NFT的一个令人兴奋的方面是它们的可组合性。随着生态系统的开发，NFT可以以一种跨越多个用例的方式进行设计。这种长期效用能够让玩家更深入地参与游戏，并最终创造出更有价值的NFT，从而在二次销售或拍卖中产生额外收益。这一领域潜力巨大，可以跨越客户体验的多个阶段：粉丝激励、元宇宙、票务、粉丝治理和决策等。

而在Visa购买NFT后，Visa加密货币业务负责人Cuy Sheffield也曾表示，在可预见的未来，NFT 将在零售、社交媒体、娱乐和商业等领域发挥关键作用。根据Sheffield的说法，Visa 相信 NFT 将在未来的商业中发挥重要作用，并可以以新的方式帮助个人内容创作者和中小型企业。

如何创建NFT

1，选择合适的区块链

选择合适的区块链需要评估多个维度的权衡，包括吞吐量、交易成本、现有的应用生态系统和去中心化程度。理想的基础设施将可支持多个区块链，并在区块链之间实现资产的互操作性。

对此，Visa 提到了以太坊和Flow。

Visa指出，以太坊是最大的区块链之一，拥有强大的用户、开发者、钱包和应用程序生态系统。大多数NFT建立在以太坊之上。不过，以太坊Gas费对于NFT小型交易者来说并不友好， Layer2的应用可以很好地解决这些问题。

Flow也备受NFT市场关注。考虑到它还处于早期阶段，开发者和应用生态系统还比较有限。不过，Flow已经拥有了相当成功案例NBA Top Shot。像 Flow 这样的区块链正在获得更多第三方的采用。

2，创建NFT

有几个平台可以帮助铸造NFT。 OpenSea以及Rarible定位于任何创作者，Bitski、Nameless（NFT42）更专注于品牌支持和更大的创作者。

3，存储NFT

Visa指出了三种存储NFT的形式：

直接存储在区块链：由于区块链软件分配的存储容量有限，允许的文件大小可能相当小。至少目前，许多公司发现直接在区块链上存储数字内容成本高昂。

去中心化存储：去中心化存储不依赖于单个实体。这些点对点存储协议的开发人员为不同的价格点提供不同程度的存储持久性。如 Arweave 。

中心化存储：云存储。

4，存储和访问NFT

钱包有两种主要的模式：“托管”或“非托管”。与加密货币互动的消费者通常更喜欢“非托管”模式，因为它使他们能够完全控制自己的资产。例如，像 SuperRare 和 OpenSea 这样的平台集成了非托管钱包，这意味着消费者负责安全地存储私钥，使他们能够访问和交易他们的NFT每个平台都可以连接到特定的第三方加密钱包，为用户提供加密和安全。

相比之下，托管类解决方案可以帮助更广泛的受众，轻松访问企业平台。如果使用托管类解决方案，最重要的是选择一个值得信赖并具有很强的安全性的品牌，因为它将代表消费者负责保管的NFT。

Visa 的加密计划

目前，Visa正与50余家领先的加密平台合作，以使在全球接受Visa卡7000万商家中，加密资产转换与消费变得简单方面。

Visa正在升级其基础设施，以实现稳定币结算，通过以太坊网络，从USDC开始。

另外，Visa 还成立了区块链领域的研究团队，从商业角度探索加密货币和 NFT 领域。白皮书中指出，作为数字支付领域的领导者， Visa 有责任引领并推动形成商业模式的讨论，包括加密货币和 NFT 领域。Visa 在几年来一直在探索区块链技术，比如离线的数字货币交易和新型的可以保护隐私的密码学技术等。
