Part 1 本周深度主题

No.1 Bye, See you.

中国全面禁止挖矿已成定局。6 月 19 日，四川关闭所有矿场，一段小视频在行业内广为流传，视频中一排排绿光，随着开关的「咔嗒」一声，逐个变暗。

四川一直是比特币算力聚集地，受此影响，比特币全网平均算力跌至 102 E，距离高点 181 E 跌去约 43%。6 月 20 日数据显示，中国头部矿池 F2Pool、AntPool、Poolin 等算力均有大幅下跌，与之相对的，北美矿池 Foundry USA 算力上涨 90%（该矿池隶属于 DCG，灰度母公司)。这是监管带来的比特币算力「去中国化」进程中的浓重一笔。

舆论方面，比特币挖矿产生的碳排放问题引起行业外的诸多批评。事实上，该问题长期以来持续被外界诟病，加密人士则予以反驳，譬如：

从数据角度， Galaxy Digital 发布定量研究报告指出，传统银行系统所消耗的能源是比特币所消耗能源的两倍以上；

从效率角度， Coinbase 发文认为能源使用是否合理取决于从中获得的价值，而比特币资源使用效率上，比许多行业都高得多；

从价值角度， 加密研究者 Gianmarco Guazzo 撰文指出，比特币所消耗的能量是必要的，目的是使加密货币免受网络攻击和篡改协议内的数据。

然而，这些观点大多建立在对比特币网络价值的认同的基础上，对于原本对比特币价值存疑的人则很难成立。这次，被迫直面环保问题，矿企和组织纷纷表明态度或予以相应行动：

马斯克与 Michael Saylor 牵头，与多家北美的主要比特币矿企组建了比特币挖矿委员会，同意在全球范围内提高能源使用的透明度，加快可持续发展计划。

许多矿企过去已经在尝试使用水能、太阳能、风能、天然气等可再生能源挖矿。

知名矿企 Argo Blockchain 今年 3 月即宣布启动纯清洁能源驱动的比特币矿池。

Terra Pool，Neptune Digital Assets 和 Link Global 同月也宣布将在加拿大启动由太阳能、风能和天然气驱动的比特币矿池。

如果说，环保问题一直是比特币推广到主流社会的阻力，那么这些调整对于比特币矿业的可持续发展、比特币主流化则起到一定程度的推动作用。

No.2 五年来最重要美联储会议带来的启示

在史无前例的天量刺激政策下，美国 5 月通胀率已经高达 5%，创 2008 年以来新高；但与此同时，非农就业岗位月均增幅仅 46 万个，整体新增就业岗位只恢复到去年年底时损失的 1/4。众所周知，保就业，保经济增长和保通货膨胀是央行的责任，而这三者是一个不可能三角。如今，通胀「爆表」，就业「缓慢」，双重目标背离。在此背景下，加息预期、QE 政策牵动着金融市场的表现，对加密市场也是如此。

6 月 15-16 日，美联储公开市场委员会（FOMC）举行货币政策会议，会议宣布：

贷款利率维持在 0%-0.25% 区间，将在 2023 年底前开始上调基准利率，比原计划提前了一年。 此消息一出，比特币和美国股市大跌。此前，随着美联储将贷款利率维持在 0%-0.25% 区间，比特币从 2020 年 3 月的 3858 美元跃升至 2021 年 4 月的近 65000 美元，同期标准普尔 500 指数上涨逾 95%。

链上分析师 Willy Woo上周五表示，股市崩盘加上美元升值可能会增强比特币的看跌前景。 他解释说：“如果股市下跌，比特币会有一些下行风险，美元指数(美元强势)大幅反弹，这是资金转向避险资产的典型表现。 ”

Scion Asset Management 负责人 Michael Burry 也对比特币和股市即将崩盘敲响了警钟。

Stock-to-Flow 比特币价格预测模型的创造者 PlanB 则根据 S2F 模型作出预测：即便是“最坏的情况”，比特币在8月份的交易价格仍然会达到 4.7 万美元。9 月份的小幅回测将 9 月的最低目标价定为 4.3 万美元，10 月份的最低目标价为 6.3 万美元，接近目前的历史高点。然后市场就会升温，11 月将达到 9.8 万美元，到年底将达到 13.5 万美元的目标。

No.3 API 经济与可组合性

如果说当前的互联网是「信息互联网」，区块链则是「价值互联网」。

互联网催生了一个概念叫做「API 经济」。API 是对应用进行封装、对外开放访问的接口，以便被其他应用或者客户端访问。在大约 16000 个开放 API 的支持下，开发者们可以在几天内实现一个想法。几乎每一个受欢迎的手机 APP，都在通过 API 带来的杠杆力量创建功能，开发者们不需要创建自己的地图、交流服务、数据服务等等，他们聚焦于在 API 之上创造价值和创新。

在区块链上，与 API 经济对应的则是可组合行。传统上，银行需要花费数周时间和大量的文书工作来验证一个人的资产状况，然后发放贷款。而有了智能合约，利用一段代码，就可以根据个人提供的抵押品自动发放贷款。

关于可组合性带来的强大作用，Linda Xie 通过一些典型项目予以说明：https://mp.weixin.qq.com/s/rF8tsWTcH1awEt1W7TMh0g

NFT 抵押类应用

使用 NFT 作为贷款的抵押品（如 NFTfi），出租或赚取 NFT 的利息（如 RenFT，Charged Particles）等。

假设你在 Decentraland 或 Cryptovoxels 拥有一块有价值的房地产，你可以用该资产借入 USDC 来为现实生活中的房子支付首付。或者，你在一个 NFT 游戏中拥有一款理想的皮肤—你可以将该 NFT 出租给其他玩家，就像在 Airbnb 上出租一个房间一样。

Mirror+Zora

通过组合 Mirror（一个去中心化的出版平台）和 Zora（一个用于创建和交易NFT的协议），允许 Mirror 的作者将他们的帖子转换为 NFT，并且可以在他们的帖子中嵌入 NFT 的拍卖链接。

闪电贷

闪电贷允许某人在同一笔交易中借入资金并偿还贷款；如果没有在同一笔交易中偿还，那么该交易将失败。

No.4 与 DeFi 代币价格息息相关的指标

Messari 通过对 DeFi 基本面指标和代币价格表现的相关性分析https://mp.weixin.qq.com/s/gNfWH73zmcGTvnzjyQmU2A，得出如下结论：

1、活跃用户的数量。基本面和运营统计数据对代币价格有重要影响。活跃用户指标是一个有价值的与价格相关联的信号，因为它显示出有意义的相关性。

2、DEX 交易量。DEX 的价格表现与交易量相关，这与假设一致，即基本面和运营指标很重要。

3、代币持有人可得的费用。能够向代币持有者分享费用的代币在熊市中表现较好。

4、锁定的总价值（TVL）。TVL 并不是一个可靠的价格行动指标，因为流动性挖矿激励、攻击或去杠杆化等因素，资本是临时性的和雇佣性的。
