在 PolkaWorld 社群和公众号后台，经常有人会问 “XXX 项目是不是波卡生态项目？”。对此，之前我们也写过一篇「真假波卡生态项目鉴别指南」来帮助大家鉴别。

最近，波卡官方文档 Polkadot Wiki 更新了一章 “如何鉴别合法项目” ，提到了一些很实用的鉴别方法，对波卡生态项目感兴趣的朋友们可以仔细读读。本文由 PolkaWorld 翻译。

有很多项目都建立在波卡/Kusama 上，或正打算建立在两者之上。但是，也有许多声称这样做的人，实际上并没有这种意图。他们没有资源来真正解决问题，或者他们完全是通过滥用波卡和 Kusama 的品牌来欺骗大家。

从 “不太诚实” 的项目中区分出合法项目，并不总是一件容易的事。 本指南旨在帮助您自己进行研究，来确定某个项目否合法。但它做不到的是，直接帮你做决定，将某个特定的项目界定为合法或不合法。本文只是提供一系列合法项目和不合法项目相比的常见特征，从而帮助您做出自己的决定。

最后，一个合法的项目并不一定意味着它也会成功，因此本指南也绝不能被视为财务或投资建议。除了自己研究以确定项目的合法性之外，在决定是否进行投资之前，你还需要对项目的成功前景进行研究。

关于 “由 Polkadot 提供支持（Powered by Polkadot）” 声明

许多项目网站上都有此声明，这经常会引起混淆。这个声明通常意味着该项目正在使用 Substrate 在或打算在 Polkadot 生态中进行构建。但是任何项目都可以声明这一点，因此，项目网站上存在此声明，不能推断出该项目合法，并且这类声明一定不是 Web3 基金会的 “批准书”。

合法项目的迹象

1、它已经获得了 Web3 Grant

Web3 基金会 Grants 是一种补贴基于波卡构建的项目的方法。他们有非常具体的申请标准，并且由审核小组进行全面审核。因此，其申请通过，是团队致力于他们的项目的第一个迹象。一旦交付了第一个里程碑，他们就可以在项目网站上添加 “Web3 Grants” 徽章。

获得 Grant 是一个项目合法的有力指标。 如果他们实现了所有里程碑，那么质量大概率会不错。但是，Grant 并不能保证项目的连续性，如果项目违反了程序规则，则 Grant 可以随时终止，所以需要注意这一点。

最后，Grant 并不代表 Web3 基金会的认可或与 Web3 基金会的伙伴关系。

你可以在下列链接中找到成功申请 Grant 的项目的完整列表，你可以在其中查看每个项目已交付了哪些内容，以及哪些项目的 Grant 被终止了：

https://github.com/w3f/General-Grants-Program/blob/master/grants/accepted_grant_applications.md

2、开源

开源项目可以提高透明度、建立信任，并确保项目团队没有在幕后做任何可疑的事。 而且，它还让跟踪项目进度、查看团队的开发活跃程度变得非常容易。

但是，这并不意味着任何闭源项目都是不合法的，或者其背后的团队在掩盖什么。许多团队会选择将其代码保密，以保护其知识产权。已经有几个这样做的团队获得了 General Grant，Grant 审核小组的成员审核了他们的私有代码。

因此，闭源项目不一定是一个危险信号，但开源无疑是一大优势。

3、活跃的开发

如果项目团队不断更新他们的产品，那么这永远是一个表明该项目合法的很好的指标。定期发布新功能和升级、修复错误、更新网站并通知社区这些更改 ，这些都是合法项目的明显标志。

而且，如上文所述，开源项目还具有另一个优势，就是大家可以通过项目的 GitHub 存储库轻松地观察到开发活动。

4、白皮书

对于任何想被认真对待的项目，白皮书的存在都应视为强制性的。但是，白皮书不仅仅需要一些信息图表、一些要点和代币经济学。一个合法的项目，或者更进一步一个严肃的项目，都将深入分析其项目的各个方面：用例、要解决的问题以及项目的技术方面。

一份写得好的、详细的、更侧重于技术而非代币经济学的白皮书 ，始终是该项目合法的一个好兆头。当然同样的道理也可以应用于项目本身，也就是说始终专注于技术而不是 token sale，永远都是加分项。

5、知名的、信誉良好的团队

合法的项目通常会在网站上突出显示其团队成员以及开发团队成员的社交媒体资料 （通常是 LinkedIn）和 GitHub 帐户。这让潜在的用户和投资者能够验证该团队的资历、过往背景和专业度。此外，让核心团队成员直接与社区互动可以增进信任和项目的透明度。

除了他们自己的社区之外，认真考虑在波卡上进行建设的项目通常会与 更广泛的波卡社区进行 互动 。他们活跃在各种波卡和 Kusama 频道中，如果团队中有成员是波卡大使，也总是一个好兆头。

6、没有在别的链上发行 token

由于 Polkadot 和 Kusama 尚不支持智能合约，因此项目有一种惯例的普遍做法，就是在 Ethereum 或币安智能链其他链上发行代币来筹集资金，并承诺在时机成熟后将代币迁移到波卡上。

但是，如果一个项目没有在另一条链上发行代币，则表明它们对生态系统的执着精神，以及对真的要建立在波卡上的承诺。他们破釜沉舟，限制了延迟上线，并消除了他们可能最终决定不建立在波卡上，转而使用另一个区块链的可能性。

不过，有几个项目在宣传其 “多链” 解决方案。在这样的情况下，项目在多条链上发行 token 是很自然的，但同时也增加了项目最终可能不在波卡上构建的可能性。

7、平行候选人

项目可以在波卡上构建的方式有很多。但是，最直接和最有影响力的方式是计划成为平行链。查看项目是否真的正在构建平行链的方法特别简单：就是通过 polkadot.js.org 上的 Rococo 测试平行链和平行线程页面 。每个旨在成为平行链的项目，在迁移到 Kusama 和最终的波卡之前，都会在 Rococo 链上作为平行链或平行线程，来测试它们的链。

他们还会尝试在 Rococo 测试网上进行众贷（Crowdloan）和拍卖，并且应该活跃在 Rococo 聊天室中。

当然，项目并不能保证自己可以在波卡或 Kusama 上拥有一个平行链插槽，项目需要赢得他们感兴趣的插槽的拍卖，否则他们可能选择成为平行线程（这一点应该向其社区说明）。但是，在 Rococo 测试网上测试他们的软件，无疑是该项目合法性的有力指标。

非法项目的迹象

1、网站质量差

识别非法项目的一种简单方法是根据其网站的质量。网站做得很差，带有错别字和语法错误，或者样式不佳；网站是用模板做的，且没有进行任何认真的努力来改进或更改它；网站上几乎没有有关该项目的信息，并且没有指向其 GitHub 或其他资源的链接；网站通常让人觉得不专业。这些迹象都表明该团队对该项目并不认真。

但是，要小心！许多没有计划进行实质性建设的项目，却仍然拥有不错甚至很漂亮的网站。他们在视觉呈现方面投入了大量资源，以此来误导他人。所以，一个好的网站不一定标志其是合法项目，但一个坏的网站却通常标志其是非法项目 ，但是在得出结论之前，你始终需要检查此处列出的所有内容。

2、匿名的或虚假的团队

与一个实名的、参与度高的团队相反，如果一个项目提到了团队成员，但没有 提供有关他们的信息，也没有提供社交媒体或 GitHub 帐户的链接，则表明存在一些问题。用 Google 搜索 团队成员，如果搜不到，或者唯一的搜索结果是与该项目有关的，那么这可能表明他们的团队成员的身份是假的。

他们网站上的照片（如果有照片的话）也可能是图库照片，换句话说，也是伪造的。这些通常很容易识别，但是如果您想更全面地了解，这里提供了有关如何进行反向图片搜索的指南。

图片搜索指南：

https://helpdeskgeek.com/how-to/free-reverse-image-search-tools/

当然，如果根本没有提到过团队成员，并且只能在他们的 Telegram 组中找到网名，这也是一个危险信号。

3、不要轻信社交媒体里的行为

尽管拥有活跃的社区，是一个合法项目的不错指标，但许多假项目会用机器人 来在群组中灌水，来伪造较多的成员数和参与度。这在 Telegram 和 Discord 中尤其常见，尽管根据 Discord 的的条款，任何机器人用户都需要标注其身份，但诈骗者不会去管这些条款和条件。

因此，不要轻信 Telegram 或 Discord 里的行为，和人们在这些群里说的话。他们很有可能是机器人账号。

4、使用公共邮箱

如果项目或团队成员的邮件来自 Gmail 或 Yahoo 或其他公共的邮件提供商，则表明该项目并不认真，尤其是在这些邮箱似乎不是个人邮箱，甚至不是有效电子邮件的情况下。任何对项目认真的团队都将使用其域名来注册邮箱。

5、代码是复制的且没有归属

Fork 来的代码仓库很容易被发现，因为它指向原始 repo。使用 fork 的代码并试图据为己有，是大大的红牌警告。在某些情况下，使用其他人的代码是没问题的，但是对于任何一个合法项目来说，在任何情况下都必须归属于原始项目。如果只有部分代码是从另一个项目复制的，也同样应该如此。

实际上，这种情况将终止 Web3 Grant 授予，并且在过去曾经发生过。

6、虚假宣传

许多不合法的项目，会声称各种根本不是真的的事情，来抬高自己，给自己增加可信度。他们通常会把自己与知名机构扯上联系来做到这一点。

如果你发现一个项目说自己和 Web3 基金会合作，那么你可以从一开始就知道这是一个错误的声明。Web3 基金会不会为生态中的任何项目背书，或建立合作伙伴关系。

对于所有其他声明，你都应检查其有效性。如果他们声称自己获得了 Grant，请检查是否属实且尚未终止。如果他们声称与多家公司和风险投资基金建立伙伴关系，或从中获得资金，请在这些公司的网站上交叉验证这些声明。如果知名媒体曾报道过他们，请确认这是真正的新闻报道，而不是付费新闻稿。

最后一点可能包含所有其他内容：不要相信表面上显示的内容。深入挖掘并始终进行验证。

众贷（Crowdloan）

随着平行链即将发布，许多旨在成为平行链的项目都发起了众贷，以筹集必要的资金参与平行链拍卖。伴随着上线在即带来的热潮，还可能出现许多骗局。因此，我们把众贷部分单独列了出来，确保你以安全的方式参与众贷。

首先，只有旨在成为平行链的项目才应该进行众贷。如果项目不是平行链候选者，则不应有与之相关的众贷。

参与平行链众贷的最可靠方式是本地方式，即通过 polkadot.js.org 上的 Crowdloan 模块进行 ，你可以在 Rococo 测试网上进行测试。这将发起一项特殊交易，锁定你的资金，直到平行链插槽租赁期结束为止。你可以在 Wiki 上了解有关众贷的更多信息。

体验 Rococo 测试网众贷：

https://polkadot.js.org/apps/?rpc=wss%3A%2F%2Frococo-rpc.polkadot.io#/parachains/crowdloan

合法的团队绝对不会要求你给他们打钱来参与众筹。 话虽这么说，也有些团队在进行 token sale 或兑换，以期抢先为即将到来的拍卖筹集资金，但这些并不是众贷，这些方式意味着对团队的完全信任。因此，如果你计划参与这些 token swap，请确保该项目信誉良好，并通过其官方网站和社交媒体渠道获取正确的信息。

同样，一些中心化交易所 正在推出一些方法来让大家通过其平台参与众贷，也有一些钱包 正在将众贷功能集成到其应用中，并且肯定还会有更多人参与。任何合理的努力都应在后台使用本地众贷模块，并为用户提供更简化的体验。在使用这些服务之前，请确保情况属实，但是无论如何，通过这些平台操作，仍然涉及信任 交易所或服务提供者。
