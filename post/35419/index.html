原文标题: 《CryptoPunks vs BAYC, Who won the Blue Chip Battle?》

  • 上周，Yuga Labs从Larva Labs手中收购了CryptoPunks和Meebits的合集，收购完成后Yuga Labs将拥有这两个系列的品牌、艺术版权和其他知识产权，以及423个CryptoPunks和1711个Meebits，这也带来关于 NFT 的版权问题讨论。

  • CryptoPunks 在经历了去年 NFT 暴涨后，面对加密市场走熊，其平均价格下跌了 61.40%。收购后其地板价与之前相比变化不大。

  • 尽管 ETH 大幅下滑，但 BAYC 的估值并未受到加密货币价格的太大影响。平均价格与三个月前相比增长了 88%，收购后地板价有所上涨。

PFP 市场中的领先者

目前，PFP项目是NFT市场上最受欢迎的类别之一，根据NFTGo.io 的排行数据，PFP 项目占据了 70% 的NFT市场份额。

数据来源：NFTGo.io

而在PFP项目中，CryptoPunks和BAYC分别是两大头部蓝筹。自从NFT summer以来，这两个项目的排名一直位居前列，分别是第一和第二名。

数据来源：NFTGo.io

成立于 2018 年的加密朋克（CryptoPunks）一直在 NFT 市场大放异彩。作为 NFT PFP 的 OG，CryptoPunks 是 NFT 爱好者身份和资历的代表。目前，加密朋克占据整个NFT市场份额的11.3%

不同项目占据市场份额的饼图 

数据来源：NFTGo.io

无聊猿（BAYC）也是一个广受关注的顶级蓝筹，已达到超过8亿美元的市值，并从 a16z 获得了 50 亿美元的估值。 除了BAYC，Yuga Lab还拥有MAYC(Mutant Ape Yacht Club)和BAKC(Mutant Ape Kennel Club)系列，总共创造了超13亿美元的收入。

CryptoPunks vs BAYC，谁更胜一筹？

加密朋克的市场表现

在 2021 年 10 月的第一周，市场的狂热将朋克的平均价格提高到 137 ETH。 值得一提的是，当时1个ETH 的价格是4151美元，而今天1个ETH的价格是2700美元左右。与此同时，CryptoPunk 交易打破了有史以来最昂贵的 PFP NFT 交易记录。

然而，尽管 CryptoPunks 过去表现惊人，但最近项目经历了严重的价格暴涨和暴跌。将价格图表与 2021 年夏季进行比较，市场低迷对 CryptoPunks 的影响变得更加明显。

数据来源：NFTGo.io

目前，很多投资者认为购买几百万美元的像素 JPEG 并不具有价值，因为市场不像以前那样看涨，FOMO情绪逐渐从市场中淡去。尽管如此，CryptoPunks 仍然是市场上巨鲸最多的 OG NFT 项目，并且几乎是入门 NFT 领域的必知项目。

BAYC的市场表现

目前，在过去一周 NFT 市场低迷的情况下，BAYC 交易量增加了近 88.92%。

数据来源：NFTGo.io

上周，BAYC 的最高销售额从 120 ETH 到 250 ETH 不等。不久前也有 666 ETH 的成交记录。从高额交易和大量巨鲸的入场，可以看出，他们仍然认为持有 BAYC 是一种有价值的资产，并且愿意在市场低迷的情况下持续购入。

数据来源：NFTGo.io

通过观察 NFTGo 的历史 BAYC 价格图表，可以看到，与 CryptoPunks 不同的是，尽管 ETH 的价值急剧下降，但 BAYC NFT 的 ETH 价格一直在上涨，达到了高价位。

目前，无聊猿的平均价格已上涨至 88.2 ETH，ETH 的当前价值约为 2700 美元。地板价也在过去7天中上涨了41%。

数据来源：NFTGo.io

我们可以通过将当时的平均价格乘以 ETH 价格，然后将价值与今天的估值进行比较，来计算这两个顶级蓝筹 NFT 项目的净价值损失。

事实上，随着 ETH 价格下跌，CryptoPunks 的估值净变化为负，BAYC 的净变化为正。CryptoPunks 在 NFT 夏季依靠其创世纪的故事和发现引领市场走向，但面对加密市场下跌，其价值也下跌了超过 50%，和市场情绪的相关性关联较大。然而，尽管以太坊大幅下滑，但 BAYC 的估值并未受到加密货币价格的太大影响，仍然保持其原有价值。

品牌效应和叙事风格

BAYC的团队重视会员资格，对于持有者给予BAYC的会员资格，并且围绕着BAYC正在逐步完善仅限会员的生态内容，包括会员专属的BATHROOM，供他们进行创作、涂鸦、交流等，T恤、帽衫等周边产品也是会员专属的实体赋能。

此外，除了BAYC，Yuga Labs还将产品横向扩展到了BAKC和 MAYC，其中BAKC中的宠物对于每个BAYC的会员都是免费的，仅需支付gas，Yuga团队下的IP之间实现了互相的价值支撑。

BAYC NFT的商业使用权也给到了具体的持有者。在2021年12月22日，33只BAYC持有者与台湾电商集团合作，推出3D立体公仔、时尚服饰等业务。这使得BAYC的商业价值的实现方式有了更多的可能性。Yuga Labs和著名的音乐经理人Guy Oseary签署了一份IP共同开发协议。与此同时，他们还与Jimmy McNiles合作，甚至与环球音乐合作，与他的四只猿组成虚拟乐队，共同拓展音乐市场。BAYC的发展离不开其对于NFT商用版权的下放以及围绕着商用版权打造的日益完善的生态圈。

而CryptoPunks的定位更像是一种NFT届的创世藏品，其背后的价值支撑来自其所承载的开创性叙事元素，就像是静待收藏的古董，更偏向于一种价值存储手段。

强强联合下的版权问题

一直以来，Larva Lab 对于CryptoPunks的发展沉默不语，对于商业应用和版权问题等也一直没有明确说明。2021年8月，Larva Lab将各种影视、娱乐平台的出版曝光机会权交给好莱坞的代理机构UTA而非交给持有者、创始人出售v1 CryptoPunks获得大额收益后对v1 CryptoPunks版权的模糊等问题都引发了持有者的不满情绪。

在收购后的申明中，Yuga Labs强调了要做的第一件事是给予CryptoPunks和Meebits NFT 持有者完全的商业权利，这也可以追溯到Yuga Labs起初便授予了BAYC 和 MAYC 所有者所有权和商业使用权，Yuga Labs还计划与社区成员一起将这两个NFT系列更多的与Web3相结合。

通常来说，NFT的版权不属于买家。而Yuga Labs授予了BAYC的所有权和商业使用权，在发生交易时，都会被授予给买家。BAYC向 Bored Ape NFT 持有人授予使用、复制和展示所购买的收藏品的完整商业使用权，以及使用它制作创意衍生作品。

数字原生资产的版权保护一直是个难题，因为 “复制粘贴 “太过容易，导致数字确权维权变得非常困难，再者关于如何定义NFT的版权问题随之而来。

2018年，加密猫创作公司Dapper Labs发布了 NFT License，明确规定，NFT持有人不得任意改变其持有的NFT的艺术形象，不得将其用于商业营销或三方产品的生产。此外，NFT 持有者不得与不正当的形象产生关联，也不得注册商标或获取其他知识产权。可以说，这一标准基本上排除了持有人的大部分商业利益。

但这样会有利于生态吗？

聪明的项目创造者明白，只有当整个IP的影响力增长时，它才能真正为每个持有者带来好处。Yuga Lab的案例就是典型之一。加密运动倡导开放、透明和共享的世界，其基本单位是开放社区。任何版权限制似乎都与加密世界的价值观不相容。

假设版权战的结局是一个象限。X轴最左边是完全封闭旧世界的版权标准，最右边是完全开放的CC0版权协议。值得注意的是，NFT 版权最后走向取决于社区和项目方的态度和推动。

结语

朋克创作者在网络上表示，他们并不“非常适合”支持该项目。这两个最强大的 NFT 项目的合并也向我们表明，一些项目可以通过优势互补，共同构建一个更好的社区。

两个项目持有者的反应也预示着未来的增长预期。粉丝们在Twitter上发起了#ApesandPunksTogetherStrong 活动，表达了他们对这次收购的兴趣。

当前 NFT 增长率下降也可能表明市场将会变得更加成熟，随着市场逐步扩大，未来也会有更多类似版权买卖场景。此次合作或许是 NFT 生态里程碑事件，这两个顶级蓝筹 NFT 项目所产生的力量足以改变 NFT 市场的发展轨迹，并随之为所有 NFT Hodler 带来更宝贵的机会。
