Web 3.0，正在成为VC的重点关注对象。

2022年3月16日，微软、日本软银和新加坡淡马锡宣布完成对初创公司ConsenSys新一轮4.5亿美元融资，大举进军Web 3.0领域。

3月17日，风险投资机构The Spartan Group宣布推出价值1亿美元的Web 3.0基金。

同样是2022年3月，红杉资本也在Web 3.0赛道出手频频。继2021年以4.5亿美元领投区块链公司Polygon后，红杉资本在2022年3月9日，领投了一家Web 3.0电子协议平台EthSign的1200万美元种子轮融资；并在3月7日参与了一家专注Web 3隐私系统Espresso Systems的3200万美元融资。

更早些的时候，红杉资本悄悄上线一只6亿美元规模计划专注于投资Web 3.0相关技术的创业公司。

该基金负责人Michelle Bailhe认为，无论是游戏、消费还是直播购物，都将转向Web 3.0，就像人们从PC转向移动互联网一样。

无论其他VC机构此时是否准备好迎接Web 3.0，这家1972 年创立、此前只投能源、金融、互联网赛道的资本巨头已经做出表率：将注意力转移至更为前沿的新一代互联网技术。

实际上，VC早已进入Web 3.0跑马圈地的战争。

比较激进的是另一家硅谷风险投资平台：a16z。

2022年3月17日，a16z等机构完成对前Facebook团队成员的Aptos项目完成2亿美元注资。

2022年1月，a16z正在募集45亿美元组建新基金，其中10 亿美元将用于其在 Web 3.0 领域的种子投资。

该基金甚至吸引国内了资金认购。A股上市公司蓝色光标近日宣布，将使用不超过3500万美元的自有资金参与认购，并成为其有限合伙人。

有机构预测，2022年Web 3.0在应用端的市场规模将超过500亿美金。

空间巨大、资本涌入，Web 3.0迎来了春天？

为什么是Web3.0？

搞懂Web3.0并不是一件简单的事，这里有个关于Web3.0的通俗解释：1.0只能读信息，2.0可读写、书写信息，3.0可读、写，并拥有信息。

2004年以前，互联网处于Web 1.0时代，其主要特征是人们从互联网上单向获取信息，信息和知识由大公司进行生产和组织，例如雅虎、新浪等各类门户。

2004年至今，互联网进入Web 2.0时代，知识和信息更多地由用户进行生产，商业机构搭建平台、搜集用户信息，进行有效信息分发。

在知乎分享一篇观点，在抖音分享一段原创歌曲，这些都在基于Web 2.0的操作。此外，微博、Youtube、小红书，也都是Web 2.0的产物。

这些互联网平台无一不以兜售用户信息分发广告作为主营收。据统计，2021年拼多多、微博的广告收益占比总收入80%以上，快手、百度则超过50%。

但这样合理么？试想下，创作者付出巨大时间、精力成本，但最大获利者则是平台。我们不仅成为了大型公司的盈利工具，个人数据隐私也受到了威胁。人们逐渐意识到大数据杀熟、隐私滥用、诱导广告等问题。

Web 2.0时代，迫于各国政府部门对互联网数据的监管压力，用户数据隐私管理才逐渐被互联网平台重视，我们看到Facebook被送上听证会，苹果迫于压力将IDFA开启权限让渡给用户。

数据权还给用户的呼声越来越高。2015年，万维网之父Tim Berners-Lee、以太坊共同创始人Gavin Wood等人提出以保护用户隐私为目标的Web 3.0，旨在将互联网“去中心化”：将个人信息让渡给用户自行保管，用户可以选择向平台发送，也可选择保留，这些数据不可被追溯、也不会泄露。

在向 Web3.0演进过程中，有很多介于Web 2.0和Web 3.0的混合形态产品出现，例如NFT交易平台Opensea，艺术创作者将作品挂在平台上，并获得最大化收益，平台则以收取手续费为盈利，这类似与将传统的电商与去中心化交易模式结合。

2021年，随着NFT交易市场的火爆，Opensea交易额激增，数据显示仅8月单月交易额便超过34亿美元。而此前，OpenSea以15亿美元“低估值”获得了a16z、Coinbase等知名投资者的大额融资。

此外，据投中网统计，海外VC已有桥水基金、红杉资本、VISA等机构入局NFT。

为什么Web 3.0在国内的投资少之又少？

区块链从业者梦波向投中网介绍，多数Web 3.0项目规模小、市场短期见不到回报，而且多数VC采用传统投资架构，投资组合里只能选出具规模的股权项目，这样一来筛掉了大多数Web 3标的。

以开头的提到红杉资本为例，其架构已在2021年得到调整，打破传统VC的十年周期制投资模式，力图打造一个永久开放的结构，这为直接投资Web 3.0一类投资回报周期长、价值难以评估的项目扫平了制度上的障碍。

该基金负责人Michelle Bailhe认为VC和所有生物一样，要么进化，要么死亡。并预测在接下来的十年中，Web 3.0的用户数量和开发人员数量将增加10倍甚至100倍，而红杉资本将继续投注上亿美元资金。

那么红杉资本如何Pitch项目？Michelle坦言，大部分Web 3.0项目是在推特上找到的。投中网发现，Michelle在推特上已有5000+粉丝，在她的关注列表，前几位用户全部为Web 3.0创业者，其中不乏竞争对手a16z投过的Paradigm、Coinbase、OpenSea等明星项目创始人。

据a16z创始人马克·安德森撰文介绍，他们之所以能够提前以天使投资者身份投进这些规模不大的Web 3项目，原因在于其基金可以“开出小至25000美元，大至数亿美元的支票”。

长期持有且小额的投资策略无疑在区块链赛道是成功的。据统计，仅在2021年，a16z的投资组合中就有43家公司完成IPO或并购，业务范围涉及Web3.0、DAO、NFT等新兴概念。其中一个著名案例，便是a16z从2012年一直持有至今的Coinbase股份，IPO当天回报率达到了惊人的4000倍。

受困与传统投资策略，国内VC能对上眼的Web 3.0项目少之又少。

“大多数投资者都会称自己在投资Web 3.0乃至下一代互联网，但实际还是Crypto那套逻辑。”对于国内Web 3.0的投资，梦波表示并不乐观。

另一个角度看，国内市场对Token的流动性监管同样抑制了Web 3.0创业活跃度。NFT平台创业者臧成都认为，此前Token作为区块链项目除股权融资外最主流的融资渠道，如果社区运维的好，其发行的Token则同时具备高流动、高溢价的优势；而没有了流动性，投资人自然没有了积极性，反而将目光转向海外市场。

有关Web3.0的争论

当然，疯狂的资本加持并不能让所有人对Web 3.0保持乐观。如此超前且魔幻的概念，引得老牌实业家纷纷站出来炮轰这个硬是被VC“炒”出圈的新名词。

前Twitter CEO杰克·多西对Web3.0极其不满：“你们并不拥有Web 3.0，风投VC和他们的有限合伙人拥有，Web 3.0永远不会逃脱他们的（资金）刺激。”

特斯拉CEO埃隆·马斯克同样表示怀疑，发推特问：“是否真的有人看过Web 3.0？反正我没有。”并发了一张图片来讽刺那些逢人便推销Web 3.0概念，并鼓动用户加入DAO中的人。

“它介于a和z之间”杰克·多西在马斯克这条推特下面回复道，并暗示Web 3.0正在被a16z控制。

A16z合伙人Chris Dixon反击道：“首先他们无视你，而后嘲笑你，接着批斗你，再来就是你的胜利之日。”

随后，杰克·多西在推特上取关了Coinbase首席执行官Brian Armstrong以及a16z创始人马克·安德森，而安德森也不客气，直接将多西拉黑。

那么，Web 3.0真的是VC割韭菜的新工具么？

一位区块链项目经理向投中网分享了她的观点，Web 3.0的确存在，但也是个见仁见智的概念。国内很多所谓Web3项目的，本质连去中心化都没实现，目前赛道鱼目混珠。

另一位区块链技术人员从技术角度向投中网介绍道，当前多数Web 3.0项目，只能算是Web2.5，数据计算的基础层限制了Web 3.0发展，区块链的核心基础“脱链计算”还未得到解决，目前光靠共识机制和跨链远未及Web 3.0的标准，但5年后或许会实现。
