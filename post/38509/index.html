Web3 的繁荣催生了大量基于 Token 的创新商业模式，从 Play-to-Earn 到 Move-to-Earn。现在，一家名为“SEXN”的初创公司别出心裁地推出了新概念“Sex-to-Earn”，将实体生物识别设备、APP和 NFT 结合在一起，让人们在身心愉悦的过程中有利可图。

根据 SEXN 公司官方社交媒体介绍显示，他们希望为用户提供的服务与人类最喜欢做的两件事有关：一个是“为爱鼓掌”，另一个是赚钱。SEXN 的商业模式可能受到了 Axie Infinity 和 SETPN 的启发，与通过步行、慢跑或其他运动获取 Token 激励不同，SEXN 希望用户在与某人建立亲密关系的时候赚取加密货币和 NFT。

利用智能可穿戴设备打通线上线下

根据《Psychological Reports》发布的研究报告显示，每周一到两次“为爱鼓掌”有助于增强免疫系统，尤其是男性，与频率较低的男性相比，每周至少“为爱鼓掌”两次的 50 多岁男性患心脏病的风险较低，女性也能降低高血压和中风风险。

在尝试 SEXN 服务之前，用户需要拥有自己的智能手表或手环，该公司创始人 Max 表示，他们能够利用智能可穿戴设备中的 GPS 传感器来区分“为爱鼓掌”和其他有氧运动，比如跑步，他进一步解释说：

“首先，为爱鼓掌的时候 GPS 位置不会有明显移动，另外，为爱鼓掌的时候心率增长曲线和其他运动完全不同，呼吸的变化曲线也不一样，氧饱和度的变化与其他运动更是有较大差异。不仅如此，啪啪啪时手部动作与其他运动不同，在综合这些要素之后，SEXN 就能确认具体的运动行为。如果你想通过出去跑步让自己出汗并诱发体温升高，那么 APP 就会判断出你其实并没有在为爱鼓掌，因此就不会获得 Token 奖励。”

简单来说，SEXN APP 将根据智能传感器（例如可穿戴手镯、手表或手机）的生物特征信息识别用户行为。 

SEXN Token 经济模式

SEXN 推出了两种类型 Token，一个是治理 Token“SST”，另一个是应用和激励 Token“SOT”。 用户在完成运动之后可以获得 SOT。此外，SEXN 还设计发行了四种 NFT，分别是：Condom NFT、Masturbation Toys NFT、BDSM Toys NFT 和 Honeymoon Ticket，只有持有 NFT 才可以在 APP 内进入 “为爱鼓掌模式”，不同类型 NFT 可以进入不同的“模式”，不同模式给予的 Token 激励也不一样，目前 SEXN 提供了四种“模式”，分别是 Coitus Mode、Masturbation Mode、S&M Model 和 Super Mode。

用户在 App Store 或 Google Play 下载 App 之后，首先需要连接钱包登陆，值得注意的是，为了保护隐私，SEXN 并不需要任何个人信息，无论性别/年龄，每个人都可以加入。接下来，用户探索“Sex-to-Earn”之前要做的一件事就是购买 NFT，因为 SEXN NFT 相当于是启动“Sex-to-Earn”的通行证。

举个例子，持有 Condom NFT 的用户每周至少可以进入 1 次“Coitus Mode”，最多 4 次。在 SEXN 市场里找到 Condom NFT 之后，用户可以连接 BSC 钱包并使用 BNB 购买，然后选择”Coitus Mode”并单击“开始”按键，接下来 APP 就会根据传感器的生物特征信息确定您是否在“为爱鼓掌”，幸福之后，单击“完成”按钮停止该过程后可以获得 200 SOT 奖励。

同理，如果要进入“Masturbation Mode”，用户需要持有 Masturbation Toys NFT；想要进入 S&M Model 模式，则需要持有 BDSM Toys NFT，完成运动后即可获得相应的 SOT Token 激励。（超级模式 Super Mode 需要 Honeymoon Ticket 才能进入，而且可能需要你连续几天“为爱鼓掌”才能获得奖励，因此轻易不要尝试。）

值得一提的是，由于 SEXN 已经在苹果 App Store 和 Google Play 上架，意味着这款 Web3 APP 严格遵守 Google 和 Apple 公司的隐私要求，因此基本上不会从用户数据中获利。

总结

现阶段，通过绑定在身体上的智能可穿戴设备去测试运动频率并鼓励健康的性行为，这样的 Web3 项目似乎并不多见。坦率地说，不管是 Play-to-Earn、Move-to-Earn，还是 Sex-to-Earn，想要不陷入死亡螺旋必须更多地去创造外部价值，比如 Move to earn 的外部价值可以是与耐克等运动品牌合作，让更多不运动的人养成运动的习惯并增加更多社交属性。

“Sex-to-Earn”的确是一个新奇的想法，至于最终能否另辟蹊径在日益激烈的 Web3 竞争中杀出一条血路，或许只有时间才能给出答案。
