人类总是有一种与生俱来的收集东西的欲望。 然而，我们所抓住的完全是基于品味的。在某些情况下，只需看看传统收藏品市场，看看这个行业有多大。

一系列让我们思想上难以理解、难以接受的新事物和新模式随着区块链的发展而崛起。近期崛起的NFT头像类项目就是一个典型的案例。以常人的思维看，这些头像仅仅只是一个个图片而已，它怎么能值这么多钱？这不是炒作是什么？

NFT 被我们赋予意义

NFT英文全称为Non-Fungible Token，翻译成中文就是：非同质化 token ，每一个NFT，艺术品，虚拟资产都是独一无二的，无法拆分，无法和别的资源替换。NFT的代表了持有人对这个资产的所有权，而并不阻止其他人视察它或阅读它，NFT只是捕捉信息记录持有人的信息并且标记在区块链上，让全世界都知道这个数字资产的所有人是谁。

如果说比特币、以太币等同质化token可以被比作钞票，NFT更像是一件不可复制的艺术品的原件。区块链技术的不可篡改给NFT token 带来了真伪鉴定的最好手段和证明：如果这些图片背后锚定的事物是可以任意篡改的，那这些图片就不会有今天的价值。NFT token 的独一无二给这些图片附上了稀有的属性，使得这些图片不再单纯的是图片而成为“限量版”的数字物品。

NFT头像token大事近期盘点

库里把自己的推特头像换成了一个猴子的形象，而这个看起来平平无奇的头像居然是他用了55个以太坊（大约18万美金）的价格买下的。看到自家Idol换了头像，球迷们纷纷追随库里的脚步，把头像都换成了“无聊猿（Bored Ape）”系列，瞬间球迷们就变身了成了猴子军团，库里也坐上了新一代花果山美猴王的宝座。

7月31日，CryptoPunks系列5217号NFT以 2250枚ETH的价格成交，约合547.2万美元。该NFT上一次交易发生在3年前，成交价仅为2.5 ETH；朋克风像素头像CryptoPunks #3100，在今年3月份最高拍价4200 ETH,按照当时的以太坊价格计算，约7500万美元；卡通胖企鹅图像Pudgy Penguins#5687在上周成交价为100 ETH，按照以太坊现价折合为30万美元。

数据网站Cryptoslam 显示，仅仅CryptoPunks近7天交易量达到48988 ETH，约合1.21亿美元，环比增长了1251.95%。其交易量仅次于Axie Infinity（近期最火热的区块链游戏），是当前市场中第二大的NFT项目，领先于NBA Top Shot 等NFT系列作品。

为什么NFT头像火爆起来了？

区块链技术的可追溯性给NFTtoken带来了历史价值：加密朋克如果不是诞生在2017年而是诞生在今天，它一定不会有今天这样的历史地位。

现代人追求的是特立独行，独一无二，没有人想和其他人撞衫、撞头像，更不想为一张图片的归属斤斤计较。区块链技术的不可篡改和NFTtoken的独一无二都给这些图片附上了稀有的属性，使得这些图片不再单纯的是图片而成为“限量版”的数字物品。

和DeFi项目相比，对于普通大众来说，参与这类NFT的门槛要低得多。在DeFi中，普通大众要弄清楚什么是流动性挖矿、如何进行流动性挖矿是一个有相当门槛的活动。而参与这类项目，大家只需要在官网或者OpenSea上面买一个 token ，然后将头像挂到自己的推特或社交工具上，这个 token 的内在作用和价值就非常直观地表现出来了，并且这种价值和体验能够通过推特或其它社交工具迅速传导到整个虚拟空间。

而选择的NFT市场最大的问题是，除非有人知道你的虚拟钱包地址，否则没人可以验证你是否真的买了这个艺术品。也就是说，我省吃俭用买了一个图片，然后，别人无法证明我真的买了它？

为什么要购买NFT头像？ 按照之前的理解，一个独一无二的NFT头像是加密世界社会地位的象征，也被一部分人认为是未来元宇宙的身份标识，“就像拥有一个限量版奢侈品包包，或是一台法拉利”。而现在，购买NFT头像有了更直接的理由：拥有，然后等待暴富。

NFT头像重点项目

头像NFT鼻祖CryptoPunks

CryptoPunks被视为最早、最古老的NFT项目之一，它的出现甚至比此前以太坊最火的NFT项目CryptoKitties“加密猫”还要早。

CryptoPunks是建立在以太坊上的最早期图像类(JEPG)NFT项目，2017年由手游开发公司Larva Lab创建。CryptoPunks共发行了1万个NFT，每个NFT代表不同的「朋克风」像素头像。CryptoPunks的总数固定，不会再生成新的NFT。在创建早期，拥有以太坊钱包的用户可以免费领取CryptoPunks头像。如今，用户只能在LarvaLabs本站和Opensen等二级交易场所购买。

CryptoPunks像素头像图片由算法生成，主要分为像素人物、外星人、僵尸及人猿四大类别。所有的头像都走朋克风路线，每个都样式独特，如男女性别之分，是否戴了帽子，项链的款式也不同，发型各异，肤色也不尽相同。在CryptoPunks网站上，每个朋克像素头像都有自己的资料展示，显示他们的属性特征及所有权和出售状态。由于CryptoPunks像素头像是最早的加密艺术图片类NFT，再加上区块链技术带来的加密性和稀缺性，它如同艺术品一样具有收藏价值。

TheBoredApeYachtClub

每个Bored Ape猿猴头像NFT都有独特的特征，团队塑造了猿猴的不同表情、头饰、服装等特征，在链上生成了各具特点的头像。Bored Ape猿猴头像是一种身份象征，你购买Bored Ape NFT，不仅仅是买入一个数字文创品，你还可以获得进入BAYC俱乐部的资格，你拥有的Bored Ape NFT是你的数字身份，可以享受俱乐部才有的福利和产品。

PudgyPenguins

Pudgy Penguins上共发行了8888个形态各异的卡通胖企鹅头像NFT,有些企鹅戴着帽子或眼镜，有些穿着一件T恤或戴着一条围巾。Pudgy Penguins胖企鹅NFT由于外形可爱，受到不少推特用户的喜爱，其中Reddit联合创始人lexis Ohanian在推特上发布了他持有的Pudgy Penguins，《纽约时报》对此事件进行了专门报道。

总结

即使一个NFT会被许多没有所有权的人用作头像，但在CryptoPunks像素头像的圈子里，就有着不成文的规定：“将一个不属于你的CryptoPunk设置为你的头像是羞耻的。”因此，NFT在版权保护方面有其独特的优势，通过区块链，不仅艺术品的交易更加快捷便利，鉴定成本也大大减少，让艺术品交易更加安全。

NFT也让艺术具有了更多可能性。NFT出现后，可编程的数字艺术也得到了关注，甚至在艺术品、文化产品的所有权上也有新的探索。2020年，世纪罕见的货币政策来以及新冠疫情加速了虚拟世界与资产生态的发展。未来会有更多大公司入场，期待着未来NFT市场更加繁荣！
