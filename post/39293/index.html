Web3 社交赛道蓬勃发展，50 个入选项目来自社交通讯、社交网络、DID、声誉、创作者经济等 7 个板块。

撰文：Mask Network

开源软件和社区捐助平台 Gitcoin 已启动第 14 轮捐助活动，活动时间从 6 月 8 日至 23 日，用户可以通过该平台为优秀的开源软件团队进行资助。

Mask Network 非常高兴能够举办首次 Web3 社交生态系统捐助活动，并提供 50000 美元的奖金。在几轮审查之后，有 50 个项目入选这一轮捐助活动。同时 Mask Network 也非常高兴地见到大约一半的项目来自于 Mask Network 的推荐，而其余的是来自自然流量，这也显示了 Web3 领域社交赛道的蓬勃发展景象。

为开源项目提供资金可以促进整个区块链生态的发展，Gitcoin 采用二次方投票的模型对项目进行众筹实现资金的配置，解决了以太坊生态中的公共产品融资低效的问题，并鼓励小众群体投票。在每轮的 Gitcoin 捐助活动中，除了各项目方收到的捐款外，Gitcoin 还会以此作为参考协调以太坊基金会的基金分配以便让优质项增加目获得投资的机会。

Gitcoin 在前 13 轮赠款活动中已为数千个开源项目提供了 6260 万美元资金，也引入了若干生态轮和倡议轮，不仅帮助 Web3 生态系统中的建设者和创造者，而且还使用二次融资机制来支持气候、乌克兰和加密倡导等事项。

我们将这些项目分为了 7 个大类，以下是我们对本社交生态轮项目（50 个项目）的推荐与评语

  • 社交通讯 Communication

  • 社交网络 Social Network

  • 社交身份 Social Profile & DID

  • 链上声誉 Reputation

  • 创作者经济 Creator Economy

  • 去中心化自治组织 DAO

  • 元宇宙 / 游戏身份 Metaverse/Gaming Identity

社交通讯

SwapChat 

介绍：SwapChat 是一个 Web3 原生跨平台聊天应用，用户可以跨平台给地址发送信息，如 OpenSea、Twitter 等。Swapchat 与 Next.ID 可以使用户将地址与社交档案联系起来。同时 SwapChat 还集成了交易工具和协议，如 SudoSwap 产品，使得用户在 SwapChat 上进行交易要比在 Discord 和 Telegram 进行交易更加安全。

SwapChat 上的所有用户均通过 Twitter 验证并以钱包为导向。因此用户可以与任何其他使用 SwapChat 的钱包交谈。此外，即使平台没有聊天功能，Swapchat 也让用户拥有可以聊天的机会，比如用户在 Opensea 上看到喜欢的 NFT 时，只需创建一个 SwapChat 就可以 NFT 的所有者交谈，提出报价，并进行交易。用户的信息将以去中心化和加密的方式传输。

推荐理由：在保障安全的前提下，SwapChat 填补了用户在 Opensea 上缺少与卖家交流工具的空缺，有机会成为 Opensea 旺旺。同时也输出 sdk，接入大量插件，钱包，应用等，有机会成长为跨平台协议级别通讯产品。它背后的 Web3MQ 协议有机会成为下一代的 SMTP/POP3

Metamail 

介绍：MetaMail 是一个利用 Web3 优势解决传统电子邮箱安全性以及垃圾邮件等问题的邮箱应用。MetaMail 通过使用 Web3 域名或地址，来发送和接收电子邮件，与传统邮箱兼容，同时没有 Gas 费用，能够使用密钥，对邮件进行签名以及端到端的加密，从而达到极致的安全性，并大幅减少垃圾邮件。

推荐理由：邮件是最经典的通讯产品，MetaMail 的无 Gas 费用、加密安全的兼容方案是 Web3 邮件产品的理想形态

Dmail Network

介绍：Dmail 是 ICP 链上的 Web3 信息协同平台，集消息通信、资产管理、数据存储、工作空间等功能于一体。用户可以通过邮件进行消息和资产的收发，也可将 NFT 设为自己的头像。

最终，Dmail 将成为用户进入 Web 3.0 时代的基础基础设施工具、流量入口和 DID，以及将 Web 2.0 用户轻松连接到区块链服务的桥梁。

推荐理由：Dmail 有望成为 ICP 生态的信息协同基础设施。

Star

介绍：Star 是一个加密和去中心化的聊天应用协议。虽然 Star 本身是一个桌面应用程序，但协议容易集成到任何其他项目包括私人 IPFS 网络。Star 是一个 IRC as a Service 的模式。

Star 已建立可继承的去中心化聊天协议，开发者很容易在上面建立应用程序。与许多中心化的聊天方式不同 – 使用 Star 的人越多，Star 就越稳定。

推荐理由：Star 对于中心化聊天协议的继承性使其对其他通讯工具开发者友好的特点尤为突出。

Daopanel

介绍：Daopanel.chat 是一个开源的加密去中心化的钱包对钱包（或 ENS），使用 XMTP 的聊天工具。

Daopanel 的团队历史可以追溯到 2017 年的 MakerDAO，daopanel 是代表们和 DAO 讨论、分享、辩论和成功的地方。目前上线的 daopanel.chat 是 daopanel 基础设施的一部分。DAO 的决策者有不可审查的和私人的 Web3 通信的需求。DAO 中的内部事件，如代币销售、DAO 之间的合并，以及投票应该是加密的，并以用户的 ENS 或钱包身份进行，同时保持完全匿名。

推荐理由：Daopanel 在 DAO 治理方向的应用上有强大的积累。

Beoble

介绍：Beoble 是一个带有 API 和 SDK for Web 3.0 的社交模块。借助 beoble，Web 3.0 服务可以在 5 分钟内轻松地将社交资料、消息传递以及发布和提要等社交功能集成到其服务中。

目前 DEX、P2E 游戏和 NFT 项目在内的许多 Web 3.0 服务都在考虑添加社交功能，但构建成本高且与其他服务不兼容。同时，现有的 Web 2.0 社交服务缺乏对区块链和 Web 3.0 的支持。 beoble 旨在通过提供社交模块来解决这些问题

推荐理由：Beoble 的模块化改造使其更容易得被第三方应用集成；可以较灵活帮助项目方添加社交功能。

社交网络

Lenster 

介绍：Lenster 是一个基于 Lens 协议的去中心化、无权限社交媒体应用。Lenster 建设在 Polygon 网络中，用户可以免费与 dApp 进行无缝互动，即使管理员也无法从该应用中删除任何内容。

用户除了提交用户名（自愿提交）和钱包地址意外不需提供任何个人信息，Lenster 也不收集任何敏感信息。Lenster 或 Lens 团队没有权限对用户的帖子进行审核。在 Lenster 中，用户所有的行为都需要签名。

推荐理由：Lens 生态应用中的头牌，做的是 Web3 Twitter 的方向，除了发帖评论等基本的社交网络功能外，还能建立社群、众筹、成立 DAO。目前测试版的体验还是比较流畅的，而在拥有 Lens 社交图谱的加持下，也会比其他同类项目更有优势。

Crypto Twitter Clone

介绍：Crypto Twitter Clone 是 ICP 链上的 Web3 克隆版 Twitter，内嵌了基于 AMM 的市场预测、订单簿、NFTs、DAOs 等功能。目前 MVP 已经开发完成，未来会计划加入比特币和以太网网络的链接。

推荐理由：Web3 版 Twitter 一直是大家讨论的焦点，基于市场预测、订单簿、NFTs、DAOs 等 Web3 特点集成有较大的探索空间

Fuse Pass

介绍：Fuse Pass 是一个为招聘而设计的 Web3 原生职业身份网络。通过利用 NFT 徽章、链上和链下数据，Fuse Pass 为用户提供了高度的可组合性的方案来帮助用户匹配他们的职业身份，并努力在合法性和用户的能力证明之间取得平衡。Fuse Pass 还提供人才发现引擎，通过挖掘职业社交图谱来帮助招聘者与「合适」 的人才进行匹配。

推荐理由：Web3 linkedin 的一种巧妙切入方式。

Metajam 

介绍：MetaJam 正在建立一个 Web3 BUIDL 网络，用户可以在上面寻找与发现 Web3 产品 / 应用程序，并通过 BUIDL 协议的证明（一个去中心化的 Web3 BUIDL 声誉协议）来建设 Web3 项目。

用户可以在上面建立 Web3 简历、验证技能并打造你的业绩记录，通过建设 Web3 项目赚取信誉积分和通证奖励。而项目方则可以在上面展示自己的项目和社区，通过链上的简历招募人才，募集资金。

推荐理由：MetaJam 有点像 Web3 Product Hunt，可以更好的帮助潜在 Web3 的贡献者找到契合的项目积累声誉。

CryptoIn 

介绍：CryptoIn 旨在创建 Crypto 里的 LinkedIn，用去中心化的技术打造一个为区块链从业人士服务的社交平台。用户可以介绍自己专业技术、寻找合适的工作机会或创业伙伴；项目方可以寻求合作，募集资金，寻找 Web3 专业人才。所有数据以及聊天内容都是去中心化储存，让用户在 Crypto 的世界里自由连接。

推荐理由：基于去中心化存储的 Web3 LinkedIn。

Crosssync 

介绍：CrossSync 是一个运行在 Crossbell 上的面向社交的 Web3 Dapp。 CrossSync 旨在允许用户拥有的社交信息流。

Crossbell 允许用户收回他们的数据所有权； CrossSync 让拥有您的数据成为一种有趣且引人入胜的社交体验。

推荐理由：RSS3 的相关项目，能将 Twitter 上的所有推文同步上到 Crossbell 链上，相当于 web2 资产 Web3 化，还是挺有意思的，6.23 号之前还可以免费 claim 域名。

Blocktracker 

介绍：Blocktracker 由前 Tiktok 工程师构建，利用 AI 从区块链中提取数据，并将钱包地址与 Twitter 帐户匹配。用户可以从他们关注的人那里看到 Web3 信息，例如最新的 NFT 交易、终身收益率等。 

推荐理由：非常实用的 Twitter 改造插件，能直接在 twitter 上看到用户最新的 NFT 交易情况以及整体收益率，是挖掘潜在大腿、辨识 KOL 的利器

Shorum 

介绍：Shorum 是一个平台，用户可以在其中支持 Web3 用户并与您的支持者分享您的利润，或者我们可以将其称为抽象的原生 Web3 社交支付渠道。Shorum 利用 Knn3 发现用户的链上关系以邀请用户的潜在支持者。

推荐理由：值得探索的重要方向，社交媒体中的利润分配是 Web3 需要解决的重要核心问题之一；同时基于 KNN3 的发现系统能更好的帮助创作者招募早期支持者。

PartyBoard

介绍：PartyBoard 被定义为一个基于场景 / 主题的 Web3 音频，内置游戏和社交元素的社交应用程序。

PartyBoard 为 Metaverse 运营中最核心的社交模块建立了主体模型，治理模型、共识模型、经济模型、沟通模型、神谕模型。同时，Partyboard 利用智能合约赋予上层应用访问 Metaverse 社交网络的能力。多链 Web3 Socialfi 协议和 Metaverse 社会网络与 Gamefi。

推荐理由：音频在链接社区与社交行为中往往扮演必不可少的媒介作用；Web3 社区中音频又能起到什么样的作用？

社交身份

Jolocom

介绍：自 2014 年以来，Jolocom 一直在支持和推动自主权身份的使命。Jolocom 是 DWeb、INATBA、Bundesblock 的创始成员，也是 DIF、IIW 和 W3C 的长期支持者和积极成员。 

Jolocom 致力于维护 SSI 的原则，并为此开发了一个完全开源的技术栈，包括一个资源库、一个 SDK 和面向终端用户的智能钱包，同时也是德国政府 3/4 安全数字身份的展示项目中选定的技术提供商。Jolocom 未来想要做的是一套开源、非盈利的 SSI (自主权身份) 标准，包括了 web2 和 Web3 的交互。

推荐理由：自主权身份是一个比较重要的概念，尤其在未来会是 Web3 与传统中性化社会融合的关键，背后需要能沉下心来的技术团队，同时也要获得以政府为代表的核心中心化机构的认可，这两点上 Jolocom 都展现出来了不错的实力。

Def Network 

介绍：Def Network 是一个 Web3 用户标签系统，通过收集链上地址行为，例如合约交互和 token 交易行为，让开发者和分析师更容易创建链上用户画像。用户可以基于 Def 找到拥有特定标签组合的人群，可以用于生成白名单、搭建 SocialFi、开发交易工具、Profile、数据分析等。

推荐理由：Web3 用户除了荣誉标签外，理应拥有更多维度的兴趣用户画像；Def 为项目提供了可以标准化自定义的标签组合系统，有很强的实用性价值。

Dragd

介绍：Dragd 是一个简单的无代码网站建设工具，能够用户创建个人资料页面，并将其连接到用户的 Web3 域名。用户可以用 Dragd 建设一切，从 NFT 收藏到 DApp 的复杂智能合约前端都包含在内。Dragd 的组件库包括各种可重复使用的组件，可以用来构建用户自己的 dApp。Builder 应用程序类似 Figma，不需要任何编程经验，但同时也支持自定义代码，用于第三方集成和为用户的 dApp 提供更复杂的功能。

推荐理由：Web3 WordPress，可以很好的帮助用户在标准化 / 个性化建站中做选择，降低了 Web3 个性化 Profile 的建立门槛。

Soulcard

介绍：一个名为灵魂卡的 Web3 名片解决方案，展示一个地址或一个人的信息和社会关系，并尝试构建以建造者为中心的 did 解决方案！

相对于 LinkedIn 等主页传统的个人信息展示，SoulCard 具备更高的信息密度，因为其可以被嵌入到多种社交场景内。如 Mirror 文章、Twitter 等；其次，SoulCard 的可靠性在使用时也得到保障，因为其在铸造为 NFT 时需 DAO Owner 进行验证； Soulcard 还有很多其他的特点，包括：SoulCard 上记载的技能等相关信息会有 Challenges 记录、Github PR 记录等进行证明；在 SoulCard Owner 加入 DAO 之后，SoulCard 上回会附带其参与的 DAO 的信息，从而高效、清晰地对 Owner 的族群进行标记。

推荐理由：ETHshanghai 决赛选手，Web3 名片的一种优雅解决方案

Proof of Soul 

介绍：受 Glen Weyl、Puja 和 Vitalik 关于去中心化社会和灵魂绑定代币的最新论文的启发，Proof of Soul 是一个无需三方许可的灵魂绑定代币（SBT）发行 / 证明协议，使用户能够将各种关系带到链上，如结婚、共享资产、社交恢复、通过信用上链进行无抵押借贷。

Proof of Soul 将在 7 月下旬在主网和 L2 推出更完整的产品，包括撤销 SBT，搜索、发行、证明 ENS，以及链接到各种 DID 服务中，如 Proof of Humanity 和 brightID.

推荐理由：Soul Bound Token 大赛优秀选手。

0xScope Protocol 

介绍：0xScope 是一个协议，通过建立知识图谱来规范 Web2 和 Web3 数据。0xScope Protocol 能够整合不同来源的数据，并结合了各种现代机器学习算法，例如 GPL，为用户提供实时分析、信号和报告。

0xScope Protocol 的产品可以赋能新一代的社交媒体平台、引擎推荐算法，还能通过纳入其知识图谱帮助政府机构的工作流程。

推荐理由：用户通过搜索其他用户的大号地址，就能立刻搜出其小号的地址。Web3 的社交网络已经不再是钱包地址 1 对 1 的关系，需要像 0xScope 这样的产品进行链上网络的识别、归一与分析，为新一代推荐算法赋能。

Interface

介绍：Interface 是一个社交应用程序，将钱包各种来源（链上、协议、合约、平台）的活动汇总起来，并将它们变成一个用户友好的社交摘要。用户可以通过滑动社交摘要来看看到朋友们的动态，包括新的 NFT 系列的铸造，在 DEX 的兑换，向跨链桥桥梁提供流动性。

推荐理由：输入你的一个身份（不论 web2 账号还是 Web3 地址），就可以得到你的身份全貌，NextID 的一个很好的前端应用场景。

Keychain

介绍：KeyChain 是一个使用 ENS 协议、SoulBound 代币和链上证明，以将 Web3 身份与其声誉匹配的 Web3 开发平台，将成为应对 Web3 诈骗、欺诈、抗女巫攻击的一种解决方案。

推荐理由：SBT 大赛的又一位参赛选手，把灵魂绑定落实到了可供其他人调取的开发平台上。

Kola

介绍：Kola 提供无代码工具和基于 AI 的基础设施，以创建具有高度参与社区的海量 Web3 音乐资产。

在 Kola 的社区中，人们可以利用跨越背景和国界的创意资源。创作过程的每个贡献部分都可以获得由智能合约执行的公平、持久的特许权使用费。

推荐理由：一个挺有意思的将 ENS 域名音频化的无代码工具，使用非常简单，多媒体域名是是一个比较新颖的概念，随着去中心化域名逐渐的身份化，像 Yat 和 kola 这类的项目则在尝试将这种身份变得更加多样，可以想象未来的身份域名会变得十分丰富有趣。

A3S

介绍：A3S 协议是下一代地址标准的多链协议，使地址能够安全地交易、租赁和托管。它为地址配备了流动性和可积性，从而允许它们作为基本的链上服务定价。

对于用户来说，这是第一次由地址携带的交互记录能够持续。用户不再需要因为私钥泄露或共享而放弃自己的地址和有价值的记录。

推荐理由：虽然信息很少，但可转移地址的概念非常有意思，如果真的成立，则会成为 SBT、NFT 社区、防女巫攻击的头号公敌，还是挺好玩的。

Wordblock
