2022开年，周杰伦没有发专辑，而是高调进军元宇宙。

周杰伦与好友共同创办的潮牌PHANTACi发售NFT项目Phanta Bear（幻影熊），单价0.26ETH（以太坊），发售当天火爆非凡，1万件幻影熊不到一小时售罄，总价达6200万元。 之后，周杰伦也换上了Phanta Bear的头像。

根据官网介绍，Phanta Bear项目由 PHANTACi 和 Ezek 联合发起，Ezek是一个建立在区块链技术上的数字娱乐平台。Phanta Bear 是10,000 个随机生成的数字收藏品，同时可兼作 Ezek Club 的会员卡。每只Phanta Bear都是独特的，都会解锁不同的等级和会员权益，持有者未来可以使用NFT门票参加虚拟演唱会。

万物皆可NFT

当游戏服装、视频片段、数字艺术甚至是自拍照与NFT加密技术结合，就能在数字时代谱写出一段段造富神话。

抓不到也摸不着，NFT到底为什么值钱？

NFT全称为Non-Fungible Token，即非同质化代币，被称为元宇宙的基础设施。与市场上常见的比特币等同质化代币不同，NFT是数字资产真实性与所有权的可靠证明。

这相当于在区块链、虚拟货币技术的基础上，通过加密凭证的方式，给每枚代币绑定上某个物品，比如图片、视频、知识产权….购买者就有了不可篡改、公开透明、独一无二的“署名证书”。

NFT技术能清楚地显示所有交易记录和产权归属，所有权和流通过程将被永久刻在区块链上，透明度高，无法造假，大大提升了交易的便捷性与安全性。另外，NFT没有中心化的服务器，一旦发行就永远存在，与发行NFT的平台存在与否无关。

国际领先的NFT交易平台OpenSea，当前拥有8千万件数字藏品，2021年累计交易额已突破100亿美元。新技术与新概念的融合，让NFT具有价值成倍增长的潜力，不断产生新的财富故事。

2021年3月11日，艺术家Beeple的NFT作品《每一天：第一个 5000 天》在佳士得拍卖行拍出天价6000万美元，约合人民币4.5亿元。 Beeple成为在世艺术家中第三高价的艺术家，这场拍卖打开了虚拟世界和NFT艺术的新篇章。

2021年10月，王家卫将电影《花样年华》1分31秒的未发行片段制成NFT，取名《花样年华——刹那》，内容是1999年开机当天拍摄的片段，卖出428万港币，实现经典电影片段在虚拟世界的永生。

NFT独一无二的属性，解决了艺术品和收藏品等的防伪难题，艺术收藏成为NFT应用广受关注的领域。

明星成为数字世界风向标

明星总是走在潮流一线，随着NFT艺术品风靡全球，不少明星开始买进或推出自己的NFT作品。

2021年9月，篮球明星库里花了约18万美元买了一只蓝色的无聊猿头像（ Bored Ape ）。Bored Ape Yacht Club是NFT作品中异军突起的新秀，一万只猿猴形象拥有不同的服装、表情、风格，受到火热追捧**。Eminem、林俊杰、余文乐、陈柏霖等明星也纷纷入坑，换上了无聊猿头像。**

余文乐目前已经收藏了79个NFT作品，佳士得拍卖行于去年9月进行余文乐珍藏线上拍卖，拍卖品里不仅有实体艺术品，其中还有他珍藏的NFT作品，如NFT头像鼻祖CryptoPunks。

徐静蕾也是NFT头像的狂热爱好者，她收藏的NFT高达700多个，不仅包括大热门无聊猿系列， 也有具有小众特色的小熊NFT系列。除此之外，鹿晗也花费4450美元将Instagram头像更换为NFT项目Kaiju Kingz，它由3333个像素画组成。

明星入局NFT已不是奇事，越来越多艺人通过购买NFT标榜自己独特的审美品味，将自己入手的艺术作品换成社交媒体头像，分享自己的NFT艺术品。同时，也有艺人借助NFT这一全新形式发掘潜力市场，探索数字艺术收藏的可能性，或是寻找新的投资风口，积累财富资源。

早在Phanta Bear之前，方文山与索卡艺术（Soka Art）合作，推出了方文山X周杰伦联名限量版《庞克猫史汀》（青花瓷款）收藏品公仔，并且附有NFT防伪认证。

2021年5月，歌手阿朵发行国内首支 NFT 数字音乐作品《WATER KNOW》，采用 NFT 加密技术，并将封面和歌曲的署名权公益拍卖，成交价超过30万元。

1月9日，NFT 项目The Heart Project 宣布，The Heart Project 与陈冠希的合作项目将于2 月1 日发布，The Heart Project NFT 持有者可优先参与。陈冠希也在社群活动中表示，与周杰伦的合作NFT项目将在年底推出。

明星不断入局，让NFT成为从科技界、资本圈到街头巷尾的热门话题，在如今流量当道的时代大放光彩。

粉丝经济搭上元宇宙快车

周杰伦高调“带货”的Phanta Bear，在发行第十天时平均价格达到最高点7.87ETH，是发行价格的30倍；截至1月22日19点，最低价格为2.5ETH，是发行价格的9.6倍。

尽管杰威尔音乐官方微博声明周杰伦本人并未参与发行NFT的策划经营，也未取得任何收益，但不得不说，很多人都是冲着对周杰伦的喜爱、周杰伦的号召力而购入这一NFT。

发行方Ezek的官网也显示，购买Phanta Bear可以拥有JAY同款—每只Phanta Bear的服装和配饰都由周杰伦和RIC的时尚品牌PHANTACi设计。

Phanta Bear发行当天，由于涌入大量用户，网页一度陷入卡顿状态，更有真爱粉用三个专属周杰伦的日子（生日、出道日、国际周杰伦日）标价，表达对周杰伦的支持。

NFT已经成为社交身份和财富地位的象征，NFT头像更像是网络社交的一张通行证。 名人使用这类 NFT 不仅激发了大众对头像的兴趣，也使持有者产生与名人使用同种身份的认同感和满足感。

NFT也是名人注意力和影响力变现的重要途径，对粉丝而言，购买明星背书的NFT是对自己粉丝身份的官方认证，也能享受专属的粉丝权益。在某种程度上，明星NFT是粉丝经济的延伸。

人人都谈论NFT，但并非人人都懂NFT。

当元宇宙的概念加上明星效应，NFT成为人人注目的舞台，不断开辟人类的想象力，拓宽认知的边界。从NBA Top Shot的数字版球星卡掀起球迷狂热，到新加坡女网红Irenezhao的照片NFT蹿红市场，NFT 在全世界范围拥有着越来越大的影响力，也证明了社交资本、趣缘连接的巨大力量。

明星NFT光环下的隐忧

NFT、区块链技术、元宇宙……人们对未来的期待被倾注在这些新鲜词汇所代表的虚拟世界，无数人跟风追随，想要在这片暗藏玄机的淘金圣地里大显身手。然而现实中这一全新的领域，更像是科技、想象和泡沫的集合体，面临着项目发行不透明，发行门槛低等行业风险。

目前国内对于NFT的法律性质、交易方式、监督主体、监督方式等尚未明确。根据国家现有规定，如果想要在正式的交易所开放NFT交易，必须经省级人民政府批准设立，或者取得国务院或国务院金融管理部门的批准。

尽管腾讯、阿里、京东等互联网巨头均开始布局NFT，但阿里的鲸探、腾讯的幻核等平台上都不直接提及NFT，而是使用了“数字藏品”这一称呼。目前国内数字藏品的主要功能是收藏、研究、社交，而不是用于商业和炒作。

在尚不完备的监管体系和底层架构环境下，NFT交易主要局限在数字藏品的一级交易市场，其他交易则产生在闲鱼等二手交易平台或社交媒体上。

另一方面，就数字收藏艺术品而言，NFT仅仅锚定了产权，对版权并没有限制作用，随着更多玩家加入，更多的版权问题将被暴露。大众对 NFT 投资的热情不减，OpenSea等多个NFT交易平台也存在大量艺术窃贼以极低的成本盗卖别人创作的NFT作品。

不少仿冒NFT的作品，仅仅需要保存图片，修改参数就能在平台上进行交易，但由于这些行为发生在区块链外，因此平台、艺术家和买家都无法追责，创作者的维权之路也异常艰辛。

名人效应下NFT不断翻高的要价，也让人不禁担忧数字作品的价格泡沫。当明星掀起NFT头像大流行时，不少人指出，一张头像真的值成千上万吗？随着NFT的艺术价值不断受到质疑，周杰伦带货Phanta Bear后也被网友指为“割粉丝韭菜”，因此明星在涉足NFT领域时也要注意舆论风险。

明星入局NFT到底是看准风口的投资，还是“击鼓传花”式的炒作，目前在NFT赛道上还看不到结果。 但明星招致的注意力流失后，人们购买的NFT是否能一直保持热度难以说准。目前NFT市场存在着肆意抬价、套取投资等乱象，而国内NFT的许多探索仍停留在概念层面，尚未形成确定性的投资价值，不应急功近利地思考变现。

2021被称为“NFT元年”，NFT甚至当选《柯林斯词典》2021 年度词汇，使用频率在过去一年暴涨了11000%。信仰者为它吟诵赞歌，反对者视之为地狱。

对于普通人而言，面对遍地开花的NFT，比起盲目跟着明星入局，陷入投机陷阱，也许保持谨慎和理性，找到自己的方向才是更好的选择。
