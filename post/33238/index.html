NFT 领域繁荣背后有无数个投资陷阱在等着我们，本文所提出的6个维度，将是调研一个 NFT 项目是否值得长线投资的基本考察点。

根据 NFTSCAN 浏览器的实时数据显示，截至 2022年1月14日，以太坊上累计铸造了 3943万枚 NFT 资产，部署了 29570 个 NFT 资产合约，链上 NFT 资产相关交易记录多达 6473 万条，涉及到的钱包地址数量有 3797 万个。 而且，以太坊上的 NFT 资产铸造数量，平均每天在以 10 万枚的速度新增。 无疑，我们所关注的 NFT 领域是一个极富有创新力和创造力的新兴市场。

在过去的 2021  年，NFT 领域开启了狂暴之路。我们每天睁开眼，都可以看到新面孔。从 Cryptopunks 到 BAYC（Bored Ape Yacht Club），从 JPG 头像到 Loot 可编程资产，从 Land 到 Metaverse，NFT 领域从未停止过创新涌现。无论是昙花一现，还是遥不可及，不断有新鲜血液加入的NFT 领域总能给你带来惊喜。历经市场沉浮，能够沉淀发展起来的 NFT 项目却屈指可数。下面这张图囊括了NFT领域的佼佼者们。

按照当前的数据，以太坊上部署的 29570  个智能合约，对应当前市场上你能叫上得名字的30 来个优质 NFT 资产项目，那么 NFT 项目成功率仅为0. 1 %。仔细想一下，这个数字太可怕了，1000  个 NFT 资产项目中才能诞生出一个值得投资的蓝筹 NFT 项目。对于普通投资者来说，这个数字意味着 NFT 领域繁荣背后有无数个投资陷阱在等着我们。所以，基于这一点，我以 NFT 领域一线从业者和 NFT 投资者的视角，来分享一些自认为有价值的经验，仅供各位参考。

如何判断一个 NFT 项目是否有潜力发展成为头部蓝筹，我认为可以从以下几个方面来进行分析：

利益声明：我非常喜欢也持有Doodles 这个项目，它是我心目中的行业标杆。所以，在接下来的内容中，我会时不时的拿 Doodles 来进行举例，方便理解和描述。

NFT 资产的协议标准

目前常见的 NFT 资产协议标准主要是 ERC-721 和 ERC-1155 协议，投资属性较强的 NFT 资产一般是 ERC-721 协议标准，而 ERC-1155 协议更偏向于功能属性较强的 NFT 业务场景。

谈到 NFT 资产协议标准，主要是从 NFT 资产合约代码层面进行分析，考察其是否遵循 ERC-721 协议标准来进行的代码编写。这里举出一个反面素材 ：Cryptopunks，由于其资产合约代码部署较早，所以当时的开发者没有遵守协议标准来进行开发，结果可以用四不像来形容。Cryptopunks 既不是 ERC-20 Token 类资产，也不是标准的 ERC-721 NFT 类资产，而且还内置了自己的交易功能。这导致后面包括 Opensea、NFTSCAN 在内的 NFT 基础设施，全部都需要主动去兼容它，这为开发者制造了很多麻烦。

所以，这里对 NFT 资产的协议标准考察的结果就是：确保该 NFT 项目的资产合约要基本遵循 ERC-721 协议标准，为后面的发展打好基础。

NFT 资产 metadata 的存储方式

如果某 NFT 仅仅是个票据或凭证，那 metadata 是否可访问倒也无所谓了；但是目前大部分我们接触的 NFT 资产都是具有投资价值的，更加关注其金融属性，所以这时候 metadata 的安全性就变得非常重要了，它是构成该 NFT 能够成为投资资产的核心要素之一。

根据 NFTSCAN 的数据分析，目前 NFT 资产 metadata 存储的方式主要有以下四种：

存储在区块链网络的区块中。通常是 Base64 格式：data:image/svg+xml;base64,PHN2ZyB3aWR0aD0iMjkwIiBoZ…

比如：CryptoPunks 和 Uniswap V3 版本的 LP NFT，都是存储在区块中的，这是安全级别最高的存储方式。

存储在基于 IPFS 协议的节点服务器中。格式为：QmYBYWMwGcyn1GoettFnn4yzpzRUwtaYrt3XscsxcyN6hD

比如 Doodles、BAYC 等头部项目使用的是 IPFS 协议存储，安全级别相对较高。

存储在中心化服务器，各种云服务器上。常见格式为：https://lh3.googleusercontent.com/oLO3z8c7Mxiu5PfIeSn6lPzWfhxr1gzZ1CcawJ3HK2bK0S7E6sv0yfuV-s1YCDKAV1JcA0vhU6xypYbPDixnMWMocd8eF4crksKU

这是最常见的存储方式，也是安全级别最低的存储方式。

此外，还有一小部分项目是使用了 Arweave 网络进行存储。

通过上面的分类，我们可以看出，NFT 资产 metadata 的存储方式是参差不齐的，没有进行标准化。同时，这些规模庞大的非标准化的 NFT 数据，也给行业内的开发者带来了很多麻烦，他们需要投入很大的人力和算力去进行数据解析，建立标准化。最近，Numbers 联合 Protocol Labs、NFTSCAN 三方团队，准备对 NFT 资产 metadata 统一进行 IPFS 存储，进行访问路径的标准化。以后，使用 NFTSCAN Open API 的开发者，可以享受该福利。

NFT 资产创作者版权的开放程度

版权开放程度对一个 NFT 项目社区来说非常重要，因为它代表着后续该 NFT 社区的发展方向和发展规模。如果版权完全保留，那么社区很难基于该 NFT 资产进行扩展，无论是二次创作层面的，还是商业化层面的。

现在常见的版权声明大概有 3 类 ：

版权完全开放。即版权跟着 NFT 持有人走，谁持有 NFT 资产，谁就拥有该 NFT 素材的版权，并且可以进行商业化，典型的案例就是：BAYC NFT。

创作者团队完全保留版权。NFT 持有人不能享受该 NFT 资产携带的任何元素的任何版权，典型的案例就是：CryptoPunks。

创作者团队有限开放版权。允许 NFT 持有人小规模化进行商业化尝试，典型的案例就是：Doodles，允许 10 万美金以下的商业化尝试。

我们当然选择版权完全开放的 NFT 项目，这更加符合区块链本质精神，更受社区欢迎！

NFT 资产自身元素的可扩展性

我把 Doodles 和杰伦熊 PhantaBear 放在一起，大家就明白什么意思了。

很明显，Doodles NFT 的基础元素的扩展性非常强，而且很具有特色和吸引力。但是反观 PhantaBear，扩展性就比较差了，甚至无法扩展。这里可能有人会说，我被打脸了，杰伦熊非常成功。那我只能说杰伦熊的成功不在于 NFT 本身的设计，核心在于是杰伦，而无关它是不是熊，亦或是其他什么动物。

NFT 项目团队的运营能力

我们都知道，同质化资产的Token，比较依赖流动性；到了 NFT 这里，因为 NFT 是非同质化资产，所以其非常依赖Web2流量。是的，你没看错，就是互联网流量。因为只有有了足够大的流量和品牌曝光度，才有可能转化更多的购买力进来。在第 4 点中，提到杰伦熊是当做反面教材的，到了这里，杰伦熊就是成功案例了。因为杰伦熊捕获了杰伦的影响力和流量，所以才有我们看到的所谓的成功。

NFT 项目团队的运营能力，代表着其捕获流量的能力。在如此庞杂的 NFT 市场中，如何在 1000 个 NFT 项目中脱颖而出，除了 NFT 自身的质量外，就是要看团队的运营能力了。

我们都知道，区块链网络和协议，本质是由冰冷的算法构成的，这些东西非常的客观。流量和资本永远无法把一个存在逻辑错误的协议做成功，但是可以把像杰伦熊这样的熊给捧上天。因为，后者并不是客观的，它的存在就是从主观出发进行创造的。

NFT 资产的金融属性

这里提到金融属性，直接一点就是持有这个项目的 NFT 资产能有什么收益。

一般有 2 种权益：使用权和收益权（分红权）

使用权是指持有 NFT 资产能获取的某些场景的使用权限或者说 PASS。收益权是指持有 NFT 资产我能获取多少投资收益或分红。

这里以 Doodles 举例，其设有 DoodleBank 用来价值捕获和收益分配。

Doodles 项目在市场上获取的版权收入以及衍生品项目主动上交的收入，都会进入到 DoodleBank 中。当然，收入来源其实可以有很多，这里就不描述了。

所以考察一个 NFT 项目的最后关键一步，就是分析其 NFT 资产的金融属性，看它是否能够捕获价值，并且是否可持续。此外，金融属性的强弱，对 NFT 自身的估值是非常直接的。金融属性越强，该 NFT 资产在市场上越受欢迎，抗风险能力越强。

大部分 NFT 资产，在牛市的时候是资产，在熊市的时候是空气；或者说，有流量的时候是资产，没流量的时候是空气。那么怎么样才能改变这个局面呢？那就大力发展其金融属性，当 NFT 资产有了较强的金融属性的时候，它在熊市的时候依然是资产，同时它也不会缺流量。

综上所述，以上 6 点，是调研一个 NFT 项目是否值得长线投资的基本考察点。

此外，由于 NFT 资产属于长尾资产，流动性特别差，所以要有更大限度的耐心和毅力来对待它。流量来的时候，要懂得急流勇退，高效的变现才能体现投资的本质。
