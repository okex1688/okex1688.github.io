GameFi 赛道增长趋缓

进入 12 月，自今年下半年以来一直火热的 GameFi 赛道开始「偃旗息鼓」，增长态势出现了大幅回落；一些链游开始崩盘或币价大幅腰斩，甚至连一些知名的 GameFi 项目也未能幸免，接连遭受重创。 如 CryptoMines、Cyber Dragon、Metamon、Farmers World 等，无论是从游戏内 Token、NFT 的交易额，还是玩家用户，都相继跳水。

The Block 数据显示，目前区块链游戏中 NFT 资产的交易额主要来自 Axie Infinity、Sorare 和 Alien world、CryptoKitties、Gods Unchained 五款游戏，但在进入 12 月后开始逐步下降。截至 12 月 12 日的最近数据，NFT 资产的总交易额为 2136 万美元，相比高点 2.19 亿美元已下降 90%；甚至占到总交易额 90% 的链游龙头项目 Axie Infinity，其交易额同样迎来了腰斩再腰斩，下降至 1923 万美元，相比峰值 2.15 亿美元跌去 90%。

另据 DappRadar 数据，所有区块链游戏的玩家人数在 11 月 30 日达到了最高 147.7 万人后，开始不断走低；单日的总交易额则在 11 月 8 日达到峰值为 5 亿美元后急转直下。此外，按单款游戏的交易额来看，目前排在前三的是 Axie Infinity、DeFi Kingdoms 和 Bomb Crypto，而这三款游戏分别位于不同链上，为以太坊侧链 Ronin、Harmony 和 BSC。

而从 Token 市值来看，据 Coingecko 12 月 15 日数据，全球 GameFi 项目 Token 总市值目前为 257.2 亿美元，相比峰值同样出现了较大的跌势，赛道内项目的估值正经受着极大的考验。

的确，此前 GameFi 赛道处于过热的阶段，整个市场由不理性的资金和新涌进的用户支撑着项目的价值。而当市场开始进入调整，甚至随着整个加密行业波动而波动时，GameFi 项目内游戏资产价值和币价则开始逐步回归其真实的价位。

但值得庆幸的是，短暂退潮过后，GameFi 领域留下的并不是一地鸡毛，仍留存了许多令行业所称道的项目。如龙头项目 Axie Infinity，广泛地普及了 Play to Earn 的概念后，成功点燃了此次 GameFi 的热潮，许多东南亚的居民因新冠疫情的封锁限制甚至开始靠玩这款游戏来谋生。而另外一款链游 Thetan Arena，则给游戏注入了更多的可玩性，不再是过度关注获益而忽视了游戏原本很重要的可玩度，因而获得了广泛玩家的青睐，在上线一个月后已突破 600 万用户。

优质GameFi项目具有哪些特征

那么经历过这一轮 GameFi 热潮洗礼后，留存下来的优质链游项目都具有哪些特征呢？

首先，当前的 GameFi 游戏还处于 Play to Earn 的早期， 一款游戏能否在一定时间内赚取收益依然是用户最为关注的地方。此外，众多头部 GameFi 游戏的营销推广能力都很强，能否首先砸当地社群和游戏公会中获得认可，这关系到早期玩家的加入，以及是否能激励到新用户的持续涌进并向其中投入资金，以维持游戏生态经济的健康平衡。

而在用户同样较为关注的回本周期上，则要稳定，项目方不能随意更改规则。如 Axie Infinity 在其各项数值上就设计得十分良好，通过拉长玩家经济活动的链路与获取报酬的周期，使每个用户都需要一定的投入与时间沉淀才有办法获得长期的获益。

对于当前，游戏的画面精致度以及故事情节、可玩性，虽还不是最为主要的，只是锦上添花，但也不可矫枉过正过度忽视。制造粗糙的游戏在收益逐步降低后，就会不可避免的导致用户大量流失，因而只能靠前期的高额收益来吸引用户。但风险往往与收益并存，这类项目多类似 DeFi 中的土矿，将沦为少部分高风险爱好者的短期投机工具。

在 Token 的机制设计上，目前的链游项目多采用双 Token 模式代替单 Token。 单 Token 模式的优势在于降低了资源整合的难度，方便了玩家交易，促进了游戏内资金的流动。不过，当早期玩家获益而大量出售时，则会导致 Token 价值迅速贬值，后进入的玩家也会因收益的降低而失去玩下去的动力，从而加速了游戏内经济生态陷入死亡螺旋。

多 Token 模式则分为游戏内 Token 和平台治理 Token，治理 Token 可用于激励玩家在 PvP 战斗等任务中赚取 Token 而获得对于社区的治理权，进而强化了用户粘性以及增加玩家在游戏中的经济消耗。此外，多 Token 模式还可通过资源整合实现自我的调整，以降低二级市场对于游戏的影响，增加游戏内经济系统的稳定。

另外，还有一些头链游平台项目正建立起多游戏互通的经济生态。 如 Gala Games、Mobox 等项目在其平台发行了多款游戏，通过流量与平台资金来吸引开发者使用平台工具开发，这样不仅提升了平台内游戏的丰富度，还使 Token 的经济体系贯穿于平台内的多款游戏中更加稳定，而非受限于单一游戏。

去中心化的Web3游戏

当前的 GameFi 游戏还将如何演化，相比传统的游戏目前又改变了什么？

在传统游戏领域，无论是游戏的机制设计还是资产的所有权上，都存在着诸多弊端。游戏往往被开发者、第三方运营商所掌控，数据缺乏透明性，一旦游戏厂商更改了游戏参数，就会对玩家的利益造成实质性的损害；同时，玩家无法拥有游戏资产的所有权，其交易行为也会受到严重的限制。另一方面，对于玩家而言，玩游戏只是单纯的一种娱乐消遣方式，不能为其带来实际的收益，缺乏投资价值。

而在加密领域，以区块链技术构建的游戏则可以很好地解决这些中心化、数据不透明等问题，并且玩家可以通过 NFT 发行的游戏资产获得其所有权，以及在游戏中进行 Play to Earn，为自己带来实际收益。

而 GameFi 仍在不断发展迭代，未来Web3 的游戏形态究竟是怎样的还未可知。通过将游戏内部分数据上传到链上解决透明性问题已是大势所趋，而 GameFi 能否为玩家带来稳定的收益，真正地实现边玩边赚仍是困扰着众多项目方一大考验。

另外游戏中资产的所有权是最重要的吗？拿法币来说，我们拥有对于自己手中法币的所有权，它们唯一、可溯源，甚至看得见摸得着。但这些法币背后的价值却随着通货膨胀不断地在逃离我们，或许对于 Web3 的游戏，所有权仍不是最重要的，如果没有去中心化的赋能，就不会有去中心化的所有权。

未来，Web3 的游戏或将还是由 DAO 所主导，从一款游戏的最开始的想法，到开发制作、以及推广宣传，都将由一款游戏的爱好者来共同完成，然后根据每个人的贡献来分配相应 Token 以及游戏内 NFT 资产。

好在传统游戏领域里的巨头们也看到了这一趋势，并相继涌入加速这一进程的推进。其中，大型游戏厂商 Ubisoft 在早年就已对加密游戏进行了布局，不仅发布了多款区块链游戏，还于近期推出支持 NFT 的游戏平台。而 Electronic Arts 的 CEO Andrew Wilson 也曾表示在关注 NFT 与 Play-to-Earn，并描述这或将成为未来游戏行业的一个重要组成部分。此外，Twitch 联合创始人 Justin Kan 也宣布将推出基于 Solana 的游戏 NFT 市场。

Web3 的游戏形态究竟是怎样的，或许都还在盲人摸象，但却也正在不断接近它的真实面貌，而这将需要大家共同地去探索。
