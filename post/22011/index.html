为了帮助大家更好地理解 LP NFT 的价值与应用逻辑，避免发生类似的损失，我们希望通过本篇文章，帮助解答大家在使用中遇到的疑惑。

NFT 是什么

简单来说，NFT 是非同质化代币的简称，专门用于表示具有唯一性且难以分割的资产。然而，由于近期以 NFT 为主要存储载体的收藏与艺术品价格大幅上涨，并引起了瞩目的财富效应，导致许多用户便习惯性的将 NFT 与收藏品划上了等号。

其实，除了收藏品以外，现实中具有唯一、不可分割等属性的资产还有很多。如游戏中的道具、皮肤，以及金融市场中的各种非标准化票据等等。例如，你在银行柜台购买了一张大额存单，那么这张存单上标明的金额、日期、利率、储户姓名等具体属性都会因人而异。同时，银行也会为每一张存单赋予一个独一无二的编号。像这种具有唯一属性的金融票据，就非常适合使用 NFT 标准的代币来表示。

V3 中 LP NFT 的意义与价值

对于许多普通用户来说，V3 中的 NFT 似乎就是一张在 Uniswap V3 提供流动性后，系统自动赠送给你的一张纪念卡片，然而这是一种极其错误的观念。

律动 Blockbeats 在此要郑重提醒大家：V3 中的 LP NFT 不是你提供流动性做市后送给用户的收藏品，这个 NFT 代表着你对你存入资金的所有权，任何得到这个 NFT 的用户都有权利将其对应的资金进行赎回。如果你出售或者转移了这张 NFT，那么你对之前存入资金的赎回权，也将一同被转移出去。

如何生成 LP NFT

用户并不需要单独进行什么专门的操作来「生成」LP NFT，只要按照提示在 V3 中成功提供了流动性 （具体步骤可参考这篇文章：《Uniswap V3 第一时间上手教程：全新的做市体验》），V3 版的智能合约就会自动铸造一个代表你提供资金头寸的 NFT 发送到你的地址上。 就如同你在 V2 中成功提供流动性后，会自动生成 LP Token 发给你一样。

1.交易对的名称

2.资金池对应的手续费率

每一个交易对在 V3 中都可以自由的创建至少三个资金池，每一个资金池对应不同的交易费率，这里显示的是你提供流动性对应资金池的手续费率。

3.LP 提供的资金在原始恒定乘积曲线上的聚合程度

这个曲线的形状表示了资金的聚合程度。曲线的弯曲度越大，表示 LP 提供资金的集中程度越小，覆盖的价格范围越广。

同时，这条曲线也能表示你提供的流动性，目前是否在有效范围内，是否在为你赚取收益。如果曲线的两端都是一个圆点，表示当前代币的价格处于你提供的流动性范围之中。如果价格超出了你做市的价格范围，这条曲线就会变成一端有顶点而另一端无限延伸的射线形状，表示你存入的流动性处于闲置状态。

4.NFT 的编号

每一张 NFT 在被创建的那一刻，都会按顺序被分配一个独一无二的编号，也就是我们在图中看到的 ID，创建的越早 ID 的值便会越小。同时，通过当前 ID 编号的最大值也可以轻松的查询目前已经生成 NFT 的数量。

5.资金分配的价格范围

Tick 是在 V3 中新增加的概念。在 V3 中，每一个 tick 的编号代表了一个完整价格曲线上的一个独立的价格点。

6.曲线在完整曲线上的位置

这是一张缩略图，表示你提供的流动性在完整的价格曲线内的位置。

7、8. 交易对对应的代币地址

如果点开卡片的链接，可以看到周围的两串代码正在围绕着卡片转动。这两串看似无规律的代码，分别代表着资金池中两个不同代币的合约地址。由于在链上创建 ERC20 代币成本极低且可以任意命名，所以合约地址是唯一能够帮你确认你买到的代币是真的代币的方式。

因此，如果你想通过直接购买的方式获得 V3 资金池的流动性，请在购买前一定注意核实代币的合约地址是否真实，以免买到由假币组成的流动性头寸。
