网传今年火热的NFT已经走到了熊市，近两个月全靠元宇宙挑起新风口大梁，不过谣言归谣言，这一周卖出天价的CryptoPunk可不管这些，成交价格涨到了天际。

Larvalabs官网显示，当前在售的这些Punk头像都有着不低的价格，动辄数百万美元，令人震惊。 就在7月31日，下图中的5217号Punk就以 2250 ETH的价格成交，折合约547万美元。

另外，编号2140的猿人Punk也在31日以1600ETH，约合392万美元的高价转手，它的购买者是Gary Vaynerchuk，曾参与领投过Uber、Facebook、Twitter 等科技公司。

这是继之前Beeble的作品拍出6900万美元价格后的又一突破，花这么多钱买个头像，这位购买者迅速引起围观者关注，这究竟是前瞻性地的投资，还是一时冲动的愚蠢行为？

对于这一疑问，Vaynerchuk本人在接受采访表示，他对这项资产有着难以置信的信心。

购买者本人的自信和CryptoPunk天价级的现象的不断出现，让人不经思考：这究竟是区块链行业发展到现在，专属于NFT不可阻挡的气势，还是过快成长推动来的泡沫泛滥。CryptoPunk究竟是什么，NFT发展到哪了，又将走向何方？

比加密猫诞生还早的CryptoPunk是什么？

CryptoPunks（加密朋克）并不是近期才出现的新事物，早在2017年就已经诞生，发布时间甚至比以太坊最火的NFT项目CryptoKitties（加密猫）还要早。

2017年，在纽约有一家名为Larva Labs的软件公司创建了一个像素角色生成器，该程序可以生成数千个不同的角色，创始人John 和 Matt 将这些像素头像带到了区块链上，让他们的独特性得到验证并让它们可以被人拥有，也就是CryptoPunks：由24x24、8bit样式的不规则像素组成的10,000个艺术图像。

这1万个以 20 世纪密码朋克运动为灵感的CryptoPunks头像中，有3840个女性头像、6039个男性头像、9个外星人头像，24个猿人头像，以及88个僵尸系列头像，每个头像都有自己的显示属性、所有权、出售情况等。

它的创始人Matt Hall和John Watkinson曾表示，创造这些作品只是想要表达一种“朋克精神”，在刚开始发布时，他们给自己留了 1000 个 CryptoPunk，其他的全部免费分发给过以太坊用户，一个星期左右所有的 CryptoPunk 便全部被认领完了。

随着这两年NFT的热潮兴起，CryptoPunks才再一次被带动了出来。今年5月，佳士得拍卖中出现了9个由Larva Labs提供的CryptoPunks NFT。这件事之后，大量新的NFT甚至开始模仿CryptoPunks。

Cryptoslam显示，近一周CryptoPunk的其交易量仅次于近期最火热的区块链游戏Axie Infinity，一跃成为当前市场中第二大的NFT项目。

True Ventures 风投公司合伙人的Kevin Rose发推称未来十年，CryptoPunk将超越标普500的表现，并声称会为自己的言行负责。

当然，CryptoPunks并不只是Larva Labs 的唯一一个，之后他们还推出了致敬 60 年代生成艺术的Autoglyphs 系列NFT ，以及被称为升级版CryptoPunks的Meebits 。

这些新事物的出现，不断给NFT生态注入着新的血液。

NFT生态逐渐繁荣

综合性交易平台、基础设施、收藏品、游戏

回顾NFT生态的发展，2017到2018年加密猫占据NFT交易量绝大多数，NFT主要是游戏，后来NFT生态系统出现了大规模增长。

据NFT Labs统计，2018到2020年NFT 市场规模增长了 825%，活跃地址数增长了 201%，发展趋势显著，形成了最初的生态规模。

但在2021年，即便不包含 NBA Top Shots 、Nifty Gateway 等项目，第一季度 NFT 市场交易额就已超过 2020 年全年 8 倍，增长速度令人咂舌。

今年以来，继Axie Infinity之后，CryptoPunks 又把NFT向前推了一大步，NFT和传统艺术品金融、娱乐传媒等结合发展，逐渐有了更多的应用场景，生态发展不断扩增。

目前NFT生态主要分为交易平台、基础设施、游戏和艺术收藏品几个板块。 在NFT出圈市场中，收藏品、艺术品占据着绝大部分位置，而游戏、体育、音乐类等NFT品类也在慢慢崛起。

综合性交易平台

NFT交易平台中的龙头OpenSea 成立于 2018 年，是目前全球最大的加密数字藏品市场，商品数量很多，囊括了各种类型，今年 3 月，OpenSea 筹集了 2300 万美元的资金。

在上周短短两天内曾创造了 9500 万美元的销售额，相比2020 年该市场全年的销售额为 2100 万美元而言，直接昭示着NFT的热度。

另外，Rarible也是一个发展的还不错的综合性的交易平台，几乎包含了艺术、收藏品、虚拟现实、游戏、实用品和体育等所有类型。 据 Crunchbase的数据，Rarible交易平台已经筹集了 180 万美元的资金。

基础设施

目前为止，按销售额排名前 10 的 NFT 项目中有 大部分是建立在以太坊上的，因此高昂的 gas 费用、网络拥塞仍然是一个问题，因此这里的基础设施主要是指 NFT的公链和侧链。

首先第 1 层基础层是Ethereum和Dapper Labs开发的Flow。 除了使用的以太坊之外，为提高用户体验，Dapper Labs 实现了在 Layer1上打造区块链的高扩展性，创建了一条为游戏和 NFT 的公链——Flow。

据Flow官网显示，与传统架构相比，在不影响安全性或去中心化的前提下，Flow的吞吐量提高了约50倍，交易费也显著降低。

第 2 层和侧链 ：WAX、Polygon、Enjin开发的波卡上的公链Efinity等。 WAX区块链是专门为NFT交易而设计的，作为一家全球虚拟资产交易平台，被称为NFT市场快速崛起的幕后推手之一，凭借着OPSkins（全球最大的数字市场的运营商）在成立之初便坐拥百万级用户基础。

Polygon 生态中游戏、NFT 和 DeFi 占据主导，目前有Showcase 、Terra Virtua等热门应用。NFT在波卡生态的发展中，Efinity较为有名，开发者可在 Efinity 链上开发 NFT 应用。

数字收藏品平台

主要功能是发行收集类NFT，这些Token可以用于游戏或是玩家之间互动，如上文提到的 CryptoPunks就属于其中一种，将 NFT带入了主流。

另外，还有NBA Top Shots，是Flow 平台上领先的 NFT 项目，是第一款基于区块链的 NBA 数字收藏品游戏 ，由 NBA 与 CryptoKitties 创始团队 Dapper Labs 合作推出，并受到不少球星的投资支持，一举成为当前市场上最为火热的NFT产品之一。

游戏

最近在NFT领域在火爆的游戏无疑就是Axie Infinity，它的收入在圈内碾压了Uniswap，圈外秒杀了王者荣耀，今年6 月以来，Axie Infinity 活跃用户持续增长，其Token AXS 也屡创新高。

另外，The Sandbox 游戏虚拟世界出售了 300 万美元的 LANDS NFT，超过 2019 及 2020 年度总和。 其他著名的 NFT 游戏包括Gods Unchained、Guild of Guardians、Splinterlands等。

不仅如此，NFT在金融、域名、跟踪分析、音频和虚拟游戏等方面的生态也都在逐步建立。 作为一个新兴的小众加密领域，尽管目前市值规模依旧较小，越来越多的名人、画家、球星、歌手、收藏家、艺术家参与投资NFT此产品，整个生态更加丰富繁荣。

备受关注的NFT面临新困境

随着NFT生态的逐步完善，近两年以来，除了艺术家、收藏家等参与投资NFT，阿里、推特、可口可乐等大厂也纷纷布局NFT。

根据Google Trends显示，关键词“NFT “全球的搜索量在2021年剧增，NFT的关注度正在不断攀升。

NFT凭借其唯一性和稀缺性的特质，实现了连接虚拟世界资产和现实资产的难题，未来或许会有更多的实物上链，整个生态将越发丰富。

但市场炒作的异常高涨，很容易让普通人对于其真实价值和目前进展有所迷失，理智地看待现存问题和未来走向也变得异常重要。

首先在基础设施方面，由于当前 NFT 应用多仍以以太坊为底层开发，因此以太坊链上 GAS 费过高，网络拥堵等问题依旧也是影响 NFT 生态稳定发展的重要问题。 尽管目前NFT 生态已经开始寻求 layer 2 或侧链解决方案，但依旧还在探索之中。

其次，NFT依旧属于小众圈层，市场的资产类型多样性不足，因此流动性有限。 不少买家购置了资产却砸手里卖不出去，未来价值依旧有极大的不确定性。

另外，数字资产确权方案并不完善。 在数字化资产上链方面，没有中心化信用机构背书，怎么确保资产的所有权是唯一且确定的，仿冒、重复出售等问题依怎么解决，暂时还没有一个闭环完整的解决方案。

目前单个 NFT 价格往往都很夸张，参与者也并非对NFT抱有绝对信心。 比如Beeple在接受《福克斯新闻》采访时曾说这绝对是一个泡沫。

前佳士得拍卖行Charles Allsopp也表示，购买NFT的概念“毫无道理”，买不存在的东西的想法很奇怪。

小结

新事物的出现总是伴随着同等的机遇和风险，在NFT的猛烈的发展势头中，有多少消费者会愿意用这么多的钱来购买一件虚拟的奢侈品，NFT究竟是不是“昙花一现”，泡沫和价值的占比，或许是一个时刻需要审视的问题。

不过对于普通人而言，NFT尽管火爆全球，但依旧处于早期阶段，在氛围感爆棚的市场中，NFT不断带来的新玩法、新规则，会不会淹没在下一个区块链风口中，让我们拭目以待。
