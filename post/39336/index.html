目前，NFT（非同质化通证）在监管上还处于初级阶段。NFT作为区块链等新型技术应用的产物，难以在既有的法律框架下对其进行定性和监管。相反，其与艺术品、商品、金融产品和加密货币等多种产物有着部分相关的特性，这为监管带来挑战。监管不会迟到，只是时间未到。随着NFT的市场日益增长，以及相关NFT案件的出现，NFT监管出现了几大方向。

此外，与NFT相关的政策监管上也呈现出地区差异。一方面，在NFT市场比较活跃的国家和地区，对NFT的合规监管也有着较为迫切的需求。另一方面，NFT在元宇宙和Web3.0框架下是重要的一环，而对于日韩等将其纳入发展战略的国家而言，探索NFT相关政策方针也显得较为重要。

一、NFT监管要点

目前，NFT在监管上存在许多要点及难点。其一，NFT作为数字资产的一种承载形式，存在所有权和许可权的争议，并带来诸如知识产权的问题。其二，NFT在不同定性下存在不同的监管要求，例如其被视为证券时将面临来自金融业的合规监管。其三，NFT同样面临来自加密货币相关的风险，例如反洗钱和内幕交易等。

以上监管难点同样也是目前对NFT监管的几大主要方向。

（一）所有权和许可权

所有权和许可权是 NFT 的门槛问题。 智能合约能够写入多种规则，如仅写入所有权，或者写入版权，这意味着拥有者将拥有给予个人的非商业权利，甚至广泛的商业化权利。

  • 对于CryptoPunks（加密朋克），与购买艺术品的传统方式一样，买者并没有得到该NFT的版权。这意味着拥有者可以将其向朋友展示收藏，但不能够卖出印有该收藏品衣服和杯子。

  • 对于Bored Ape Yacht Club（无聊猿），NFT所有者则拥有更广泛的许可权。例如NFT 艺术品收藏家Jimmy McNelis与环球音乐合作后选出4个无聊猿形象组成乐队KINGSHIP，艺术家丹·罗尔曼用他的无聊猿为角色创作了一部喜剧。

无论NFT上承载何种权利，许可条约都应该准确描述相应权利。卖方应该向购买者准确传达其授予买方的权利，买方对有效合同的肯定接受也很重要。买方对于条款的不准确认知，以及卖家的恶意营销或将引发法律索赔等问题。

（二）知识产权

NFT在发行和流通过程中面临着来自知识产权诉讼的风险。 从铸造中看，将不拥有有效许可的内容，如艺术品、音乐或者IP等数字资产上链铸造成NFT，将面临承担侵权责任的第三方知识产权。在流通的过程中，拥有者无权授予代币购买者在NFT中没有的权利。对于平台，出售或展示包含第三方版权或商标的数字资产，这部分交易所或平台也可能面临知识产权诉讼。

  • 5月18日消息，美国演员Seth Green 遭遇钓鱼攻击，致4个 BAYC系列的NFT 被盗。其中，名为BAYC #8398的NFT被转售出去，该NFT正被用于推进一个名为White Horse Tavern的电视节目，附着于该NFT之上的商业使用权也转移到了新的拥有者手上。在向盗窃负责人上诉失败后，Green只能以165ETH（260,000美元）的价格买回 NFT以收回其合法权利。

（三）金融风险

NFT的金融化带来了金融监管问题。 一般来说，仅代表单一独特资产且拥有单一所有权的NFT并不是证券，而当NFT涉及资金融通，或者展现出较强的流动性时，其有可能被判定为证券，并受到证券法的约束。目前，NFT更可能在以下情况中牵涉到证券法：

汇集NFT以分享收入，或分割NFT，使得多个投资者拥有NFT的部分所有权。

NFT代表对数字资产如歌曲的使用许可权，或资产收入的份额，如销售额的百分比。

NFT代表预售以在尚未建成的平台上使用的数字资产，且销售所得用于构建平台。

  • 2021年10月8日，虚拟股票NFT市场Visionrare在官网宣布关闭公测，并退还此前用户购买VisionShares的资金。据悉，Visionrare是一个虚拟股票NFT市场，用户可以在拍卖会上竞标数百家不同初创公司的NFT股票（VisionShares），并竞争出表现最佳的虚拟投资组合。当Visionrare开启公测，由于“低估了法律的复杂性”，其或因涉及证券法，已暂停公测，并后更改为一款免费的投资游戏。

（四）反洗钱

美国财政部发布了《通过艺术品交易开展洗钱和恐怖融资的研究》报告，将NFT归类为新兴在线艺术品市场，并指出该市场可能会带来新的风险。包括数字艺术品领域内存在洗钱风险。

报告指出，NFT可用于进行自我洗钱。犯罪分子可以用非法资金购买NFT并与自己进行交易，以在区块链上创建销售记录；然后，NFT将被出售给一名不知情的个人，罪犯将从后者与犯罪无关的干净资金中获利。

  • 根据分析平台Chainalysis分析，在262名疑似习惯性洗售交易者（向自筹资金地址出售 NFT超过25次）中，110家盈利的洗售交易者从这项活动中总共获得了近890万美元的利润，更糟糕的是，这890万美元很可能来自向毫无戒心的买家销售。

（五）内幕交易

内幕交易是非法交易者获利最常见的欺诈方式之一。 通过重要的非公开信息，交易者能够预先购买或者出售资产以获取利益。在NFT发行和交易平台上，NFT预售分配和相关信息需要得到谨慎处理，否则，内幕交易将会不当地操纵NFT的价格和交易量。目前，最大的NFT交易市场Opensea已经被爆出内幕交易事件。

  • 2022年6月，OpenSea的前产品经理Nate Chastain被控内部交易。据悉，Chastain负责筛选登上网站主页上的NFT。由于预先知道哪些NFT将出现在网站上，其提前购买了一批即将登上OpenSea主页的NFT，随后在NFT登上OpenSea主页后，Chastain又以最初购买价格的两到三倍的价格进行出售。在2021年6月至2021年9月期间，Chastain利用机密信息购买了几十个NFT，并被指控电信欺诈和洗钱。

此后，该公司发布了两项新的员工政策，包括禁止OpenSea团队成员在公司介绍或推广藏品或创作者时进行买卖行为，以及禁止员工“使用机密信息购买或出售任何NFT内容，无论它是否在OpenSea平台上提供”。法律消息人士称，这些行为可能会引起对该行业进行进一步的严格审查。

二、海外NFT政策

从全球NFT的监管表现来说，目前还处于探索阶段。其中，NFT与加密货币在去中心化等方面的相同点使其面临着来自各个国家的反洗钱和金融相关方面管制，合规性是目前政策的主要方向。此外，各国在政策上也开始出现了对知识产权保护、税收等方面的考量。

从地区上来说，欧美地区，尤其美国，是NFT相关产业最活跃的地区，也是在NFT定性和合规方面走在最前面的地区之一。此外，韩国和日本纷纷将发展元宇宙、Web3.0等产业纳入国家发展方针，因此在NFT的政策发展方向上也有一定探索。

（一）欧美国家

2021年10月28日，全球反洗钱机构金融行动特别工作组（FATF）发布了针对处理加密货币和虚拟资产公司的最新指南，并将DeFi、NFT和DAO等新领域纳入其框架。根据 FATF 的最新指南，NFT不可互换，是独一无二的，且被用作收藏品，而不是作为支付手段或投资工具。 然而，如果 NFT 被用于“投资目的”，NFT将会成为该指南的监管对象。

随着加密执法加强打击勒索软件，NFT需要加强合规性以免受到制裁的影响。 2021年10月15日，美国财政部外国资产控制办公室 (OFAC) 发布了针对加密货币行业的定制指南，其中强调了制裁合规要求，并就OFAC的合规预期提供了针对行业的建议。11月，OFAC制裁了一家位于拉脱维亚的交易所Chatex，包括Chatex相关的某些指定加密地址持有42个NFT，价值约531600美元。

欧盟《加密资产市场条例》草案（MiCA）公布将对NFT带来新的合规挑战。3月14日，欧洲议会经济与货币事务委员会对《加密资产市场条例》（MiCA）草案进行了表决。此次主要监管对象是稳定币，但是MiCA对加密资产的定义并不局限于稳定币，而是以分散式账本技术（DLT）为核心。这样一来，NFT就存在被MiCA纳入监管范围的可能性。文件称，这意味着“NFT 发行人必须是法人”而不是去中心化实体，并且还必须向当局注册并遵守法律规定的其他消费者保护措施。

当然，也不乏支持NFT的声音。

2022年2月，纽约证券交易所(NYSE)向美国专利商标局(USPTO)提交申请 ，为各种数字商品提供在线市场，包括NFT、加密货币、数字媒体和艺术品。如果交易所推进这些计划，它将与OpenSea和Rarible等公司竞争。据法律人士指出，如果申请通过，纽交所将成为加密货币和NFT的交易所，且发行NFT的项目方和NFT本身都将受到纽交所的管理。

美国田纳西州立法者提议允许该州投资加密货币和NFT ，目前，该法案已传递至众议院财政小组委员会以供进一步审议。据悉，田纳西州通过吸引加密公司和投资者，已经成为一个适度的技术中心，而对加密货币友好的法律将有助于吸引更多企业进入该州。

（二）日韩国家

韩国的NFT市场较为活跃，其金融监督局同样也关注到NFT存在的风险。2022年2月，韩国金融监督局（FSS）宣布将加强对包括NFT在内的新交易资产的监控， 包括加强对NFT和元宇宙等新兴市场企业IPO的核查，并将针对快速增长的数字资产市场中对消费者造成损害的因素制定对策。

韩国金融服务委员会 (FSC)则将遵循FATF关于NFT及其监管的指导 ，即NFT通常不属于虚拟资产的定义范围，也不会受到此类监管。“如果 NFT 属于虚拟资产，他们将被要求根据《特定金融交易信息法》报告以遵守反洗钱义务。”一位 FSC 官员解释说。

2021年9月，日本金融监管机构金融服务局（FSA）的政策文件显示，该机构将开始监管NFT代币以及初始交易所发行（IEO）领域。虽然该文件没有具体解释FSA将对NFTs、DeFi、CBDCs和IEO采取什么行动，但在文件中提到的事实表明，为NFT建立监管框架的这些事项已经在其议程上。

除了对NFT相关问题进行监管，日韩国家也在政策层面上探讨发展NFT的可能性。

韩国首尔政府就出台政策、制定规划,宣布将在5年内打造首尔元宇宙平台，而NFT作为元宇宙中重要的一环同样也受到关注。目前，韩国或将出台首个 NFT 监管规则。2022年4月12日，韩国政府将准备一个 NFT 版权政策，以解决未经版权所有者同意发行的“虚假 NFT”的问题。 目前，韩国文化体育观光部已经发布了“元宇宙、NFT 相关版权问题研究服务”的招标公告，并正准备选择相关研究公司及正式的签约程序。

2022年 1月28日，日本执政党成立NFT政策审查项目组 ，以将区块链和NFT相关业务纳入政府的增长战略。根据工作组成员发布的白皮书，日本必须“推动 Web3.0 时代的创新”，在此过程中，政府必须将NFT 部门视为“新的增长引擎”。

（三）海外其他地区

俄罗斯下院议员向国家杜马提交关于将NFT纳入立法的法案。 2022年6月，提交的法案旨在通过将NFT一词引入俄罗斯联邦民法典。该草案的作者表示，“那些拥有非同质化代币的人的权利需要得到保护，目前俄罗斯人在处理 NFT 时要自担风险。”

2022年2月，印度财政部宣布了该国的预算，透露将以30%的税率对数字资产征税。根据预算，“就本条款而言，‘不可替代代币’是指中央政府可能通过官方公报通知指定的数字资产。”这意味着，印度政府将有权利指定或认证NFT，并对其进行征税。

阿布扎比自由区就 NFT 规则征求意见。 2022年3月开启征求，阿布扎比全球市场（ADGM）的金融服务监管局（FSRA）目前“不提议为 NFT 建立正式的监管框架”。FSRA市场执行董事 Simon O’Brien 表示，根据建议草案，在交易和托管等服务方面获得ADGM许可和监管的公司也将能够为NFT提供这些服务。
