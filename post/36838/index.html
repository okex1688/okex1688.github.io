与权益证明（PoS）的合并（The Merge）将是迄今为止对以太坊最深刻的升级，其重要性仅次于创世区块。这将是让世界看到一个主要的、分散的系统如何在实际上消除其碳足迹的例子。

近日，以太坊基础设施开发公司ConsenSys发推称，以太坊合并将彻底改变构建和访问以太坊的方式，合并将为以太坊发展带来4大支柱。 

自 2015 年成立以来，ConsenSys 在以太坊协议的持续发展中发挥了基础性作用。从“Eth2”规划的第一天起，ConsenSys 协议工程团队就一直在不懈地努力实现合并：

（ConsenSys 协议团队的主要贡献）

ConsenSys 团队指出，我们相信像以太坊这样的去中心化网络可以帮助社会取得更多成就，我们贡献了我们的人员、产品和资源，将 PoS 带到了以太坊。随着合并的临近，我们希望通过说明以太坊将如何升级来重振生态系统的信心。这是合并的四大支柱。 

支柱一、多元开放

权益证明将使网络参与民主化，并通过新的激励机制、验证者和创新更公平地分配奖励。

新的激励措施旨在鼓励单节点验证者更多地参与，同时要求更大的参与者承担责任。这方面的一个例子是反相关机制，它对单节点验证者施加较小的惩罚，而像 Coinbase 或 Lido 这样运营数千个节点的大型质押者。反相关机制贯彻“权力越大，责任越大”的原则，鼓励大型参与者与客户多样性等网络目标保持一致。

升级到PoS将使以太坊更加开放和多样化。客户端、验证器和第二层的多样性也将确保一个健康的、可扩展的网络，使参与民主化。

PoS 不仅对验证节点的资源要求较低，而且旨在奖励所有参与者，而不仅仅是那些具有最大计算能力的参与者。虽然仍然会有拥有更多节点和鲸鱼的网络参与者，但每个节点将有平等的机会获得奖励。

同样重要的是要记住，虽然 PoS 将通过引入新的参与者来帮助以太坊更加多样化，但验证者仍然有责任通过选择多样化的软件客户端来帮助促进客户的多样性。ConsenSys 提供免费和开源的以太坊客户端软件，例如 Besu 和 Teku ，因为我们努力争取在我们自己之外的客户端的均匀分布。

像质押池这样的新创新降低了参与门槛，并允许多方汇集他们的 ETH 以满足 32 ETH 的最低质押门槛。ConsenSys 是Rocket Pool DAO 中支持完全去中心化质押池的 14 个组织之一。

支柱 2. 能源效率

不断发展网络的共识方法可确保以太坊能够在不损害地球的情况下可持续地支持下一代 Web3创造者和开发者。

PoS 将减少每笔交易使用能源的99.95%，大致相当于荷兰的能源消耗。以太坊的存在是为了改善世界，而不是伤害世界。 

PoS的升级也使以太坊更加模块化。这种模块化将计算密集型执行与共识层分开，是减少网络运行能耗的原因之一。

L2 rollup 和分片的其他效率将使以太坊消耗的能量减少几个数量级。

PoS的升级也使以太坊更加模块化。这种模块化将计算密集型执行从共识层中分离出来，是减少网络运行能耗的原因之一。

支柱 3. 无缝过渡   

在以太坊上工作的开发人员可以高枕无忧，因为用于合并前后架构的通用 API 以及对现有组件（如以太坊执行客户端 Besu 和 Geth）的重用将确保无缝过渡的不间断开发人员体验。 这意味着现有应用程序的返工率很低甚至没有，但是我们建议您进行测试以完全自信。

PoW 和 PoS 接口的相似性意味着不需要更改 API。Engine API从一开始就与现有的基础设施和开发人员工具兼容。利用现有组件还可以确保无需重写智能合约或 dapp 或迁移基础设施。对于依赖随机操作码的 dapp，可能需要更新一些基础设施。如果您是以太坊开发人员，并且希望提前开始测试您的 dapps 和基础设施，我们鼓励您使用最近推出的Kiln 测试网来这样做。

增加的客户端选择和灵活性将允许开发人员混合和匹配以满足他们的需求，例如在执行层使用轻量级客户端和在共识层使用重量级质押客户端。

迁移到PoS还意味着更可预测的阻塞时间和费用，这将减少设计高流量解决方案时的不确定性，该解决方案以12秒心跳为周期进行阻塞，并使用EIP-1559建立基准费用。

支柱 4. 超健全货币

合并将带来更新的加密经济模型，减少以太币的发行，并为最终用户和 DeFi 行业提供更强大的安全保障。合并还将限制以太坊的供应，使其更稀缺，因此更有价值。

网络经济本质上是动态的，但是值得注意的是，一些潜在的结果与向 PoS 的迁移相结合。一是减少为支付验证者而创建的代币数量（减少 90%），二是销毁作为交易费用收集的代币，而不是将它们传递给矿工。

攻击链的成本也将显着增加（10-20 倍）。民主化的参与确保了几乎不可能受到攻击。在当前 350K 验证节点（质押 1000 万 ETH）的水平上，攻击链的成本是 34B 美元的 51%。PoS 与 EIP-1559 的通缩影响，以及合并后质押 ETH 的预期增加，将加剧这种攻击成本。 

更重要的是，攻击将变得更容易恢复，部分原因是削减将破坏用于攻击的权益，需要进一步投资来维持攻击。少数用户激活的软分叉也可以由社区协调以销毁攻击者的货币。

合并即将来临

我们希望这四大支柱已经阐明了以太坊将如何通过迁移到 PoS 来升级，并恢复您的信心：

1.开放和多样化：所有类型的参与者都可以更轻松地访问以太坊，从而实现更大的去中心化。

2.节能：所需的计算能力减少了几个数量级，将以太坊每笔交易的碳足迹降低了 99.95%。

3.无缝过渡：当前在网络上工作的构建者不会中断，继续构建，继续交付，并全力以赴进行测试。

4. 超健全货币：更新的加密经济模型使以太币更加安全、稀缺，并提高了交易的最终确定性。

ConsenSys 认为，像以太坊这样的去中心化网络通过更深入地联系我们并消除权力中心及其固有的缺点，使社会更加繁荣。我们将继续贡献我们的人员、产品和资源来完成合并。

此外，有关 Merge 的定期更新和新闻，请访问ConsenSys Merge 知识库，您还可以在其中找到 Merge 档案，其中包含已实现的关键里程碑以及开发人员的其他重要资源。
