这一届 NFT 正用躺平和暗黑对抗熊市

皮之不存，毛将焉附。

在加密熊市背景下，NFT 市场已经从「我们都会成功」变成了「我们都会死去」（we’re all gonna die）。在过去的几周时间，加密行业发生了翻天覆地的变化，NFT 市场情绪也变得低迷且“反乌托邦”，一些 NFT 创作者为了宣泄自己的感受，开始推出一些暗黑系 meme，甚至开始拿“厕所文化”来嘲讽当前熊市。

于是，我们看到 OpenSea 平台的交易排行榜上出现了各种奇怪的 NFT 项目，比如「哥布林地精」Goblintown、「暗黑死亡系」We Are All Going to Die 等数字藏品涌入了 OpenSea 平台，其中不少项目一度出现在顶级收藏品排行榜前几名。数据显示，过去一周交易量排名前 20 名的数字藏品中，这些“怪异”的 NFT 项目交易量占比一度达到了 45%。

正如「无聊猿」Bored Ape Yacht Club 社区用户 @professorM 所说：

“熊市让每个人都变得有点疯狂，如果你做一些不同的事情，就很容易抓住市场注意力。”

「哥布林」Goblintown  的快速蹿红

去年，以太坊上的 NFT 交易异常火爆，交易额直线飙升至 200 多亿美元。当时整个加密市场的关注焦点都集中在 CryptoPunk、无聊猿 BAYC 以及后来出现的 Doodles、Cool Cats、Azuki 等项目上。然而随着近期加密市场逐渐走向冷清，NFT 加密社区的一句流行语「WAGMI」（We All Gonna Make It/我们都会成功的）已经慢慢不再被人提起，显然投资者们开始意识到并不是每个人都能获得「成功」的。

然而，尽管目前的市场环境充斥着低迷气息，却依然有着这样一个以太坊 NFT 项目出乎意料地蹿红了。这个名为「Goblintown」的 NFT 项目由一群相貌丑陋、造型荒诞的「哥布林地精」头像构成，目前在去中心化交易所 OpenSea 上的交易额位居榜首。5 月 21 日，该项目的 9999  个尖耳妖精头像免费派放给 NFT 玩家，用户仅需支付链上 Gas 费即可。然而谁也没想到的是，就在此后短短的两周半时间内，该系列 NFT 的总交易量就超过了 32000 ETH（5760 万美元）。

实际上，「Goblintown」这个词是用来描述熊市的一个术语，然而就在过去的一年，它已成为加密货币玩家心目中越来越流行的 meme 文化代表了。坦率地说，「Goblintown」用其丑陋、怪诞的画风挑战人们的审美，刮起了一阵故意丑化 NFT 的风潮，有人直言「这件艺术作品实在是太糟糕了，但人们一心想成为 degen，这样他们就可以翻身赚大钱了。」

「degen」其实是英文单词「degenate」的缩写。对于每天长期在线并耗费大量时间研究和交易加密资产的投资者来说，这个词已经成为了一个标榜他们的惯用绰号（带着点自嘲的意味），意思是这类人通常会以牺牲生活中的其他事情来作为投资的代价。

受近期加密熊市的影响，连「蓝筹」NFT（如无聊猿 BAYC）也开始出现暴跌，这让加密投资者们倍感不安，因此开始寻找新的赚钱方法。NFT 投资者 ConciergePhysician 认为，丑陋的艺术是没有前途的，并不值得为此冒险，对投资者来说，即便在熊市中，其实也不缺乏赚钱的机会。

然而对于那些持有 Goblintown NFT 的「degens」来说，Goblintown 已经成为投资者追随 meme 文化，也是加密熊市中寻求获利机会的一个新载体。实际上，成为爆款后的 Goblintown 已经引来了 10 多个仿盘，他们大都艺术风格前卫、色彩新鲜，相比之下，大多数符合大众审美的蓝筹 NFT 反而让人觉得有一些「丑陋」。

推特用户 Dgen.eth 目前拥有 12 个 Goblintown NFT，其中包括一个 1 of 1 Goblintown wizard NFT。对于这个略显怪异的项目，他这样描述道：

“在 Twitter Space 里，Goblintown 成员会使用语音合成器模仿哥布林的声音，这些有趣的行为也给该项目平添了几分神秘感。不得不承认，通过图像、声音与界面的结合，Goblintown 让 Web 3 支持者变得疯狂。”

“摆烂”正在成为一种趋势

继 Goblintown 蹿红之后，另一个 NFT 项目「We Are All Going to Die (WAGDIE)」也异军突起，其交易量在 OpenSea 本周排名中位居第二。值得一提的是，该项目也是一个免费铸造的 NFT 系列，在 OpenSea 上将自己标榜为一个「没有发展规划，只有死亡」的项目，换句话说该项目并没有像其它 NFT 项目那样为投资者提供一个清晰的未来发展规划，像极了现在流行的「躺平族」。

推出后仅一周，WAGDIE 便产生了超过 9,000 ETH（1620 万美元）的销售额。很快，WAGDIE NFT 系列就在社交媒体上引起轰动，因为该项目团队以 21 ETH（约合 37,800 美元）价格购买「变异猿」Mutant Ape Yacht Club # 20438 之后，仅仅过了一个小时便将其销毁。

毫无疑问，这种不走寻常路的操作让 WAGDIE NFT 项目名声大噪，也在短时间内吸引到大量关注——据说有 1500 个粉丝关注了「变异猿 NFT 销毁」话题。据说，在 Goblintown 和 Cereal Club NFT 持有者的 Twitter Space 中，疯狂的粉丝在群里大喊：

“把那些高价 NFT 都 TM 烧掉！”

在 NFT 市场中，这些人似乎已经形成了一股新兴力量，他们热衷于追逐带有虚无主义色彩的 NFT。正是在这样的潮流下，除了「哥布林」Goblintown 和 WAGDIE 之外，一批类似的 NFT 项目也如雨后春笋般出现，比如 ShitBeast、Ill poop it nft、mcgoblin.wtf、Goblin Grlz、Leave Me Alone NFT、troll-town.wtf、Begin as Nothing 和 ShitPlunger 等。

这些「躺平系」NFT 项目看似和其他 NFT 项目没有太大区别，但基本上没有 Discord 社区和网站，在谈到这些 NFT 项目问题时，NFT 投资者 ConciergePhysician 补充说：

“如果说四个、五个、甚至更多暗黑系或躺平系 NFT 由同一个人经营，我不会感到惊讶。”

实际上，从这些 NFT 项目的名字就能看出加密社区当前的情绪和主题——虚无主义和「身体排泄物」正在流行。举个例子，「ill poop it nft」在其描述中写道——你的 NFT，不过是 JPG 图片，你所认为的蓝筹 NFT 都是「狗屎」。但略显反讽的是，「ill poop it nft」地板价却不降反升，单个「poop」JPEG NFT 的价格超过 1.1 ETH（约合 1,980 美元），根据 NFTGo.io 数据显示，当前「ill poop it nft」交易额已接近 850 万美元，市值超过 400 万美元，但地板价已出现大幅下降。

坦率地说，这些 NFT 中有很多是「复制粘贴」项目，而且 90%、甚至 95% 都会失败，或许在两年、三年或五年之后，我们才能看清这些 NFT 项目的最终走势和结果。

隧道尽头的曙光

虽然有些人不看好「躺平系」、「暗黑系」、「摆烂系」NFT，但这些项目的市场前景并非一片黯淡。根据加密分析网站 Dapprader 最近发布的一份报告显示，相比于加密货币市场，NFT 市场显然更具弹性，而且 NFT 行业更加成熟，甚至可能与加密市场脱钩，这意味着很可能会有几个怪异的 NFT 项目会杀出一条血路。

另据 Dune Analytics 数据，虽然 OpenSea 月交易额从 4 月的约 35 亿美元下降到 5 月的 26 亿美元，但如果与 OpenSea 去年 11 月的交易额数据相比，你会发现该指标并没有出现大幅走低。

很多玩家说越来越看不懂现在的 NFT，但其实，一旦更广泛的加密市场暴跌，NFT、DeFi、以及其他垂直领域都会受到影响，而「躺平系」、「暗黑系」、「摆烂系」NFT 或许只是为了苦中作乐，让加密社区成员在压抑的市场情绪中找到一点乐趣。

至少现在，这些 NFT 已经被加密社区接受。
