NFT 的浪潮一波接着一波，CryptoPunks 的余温尚在，Loot 就携带 NFT 乐高的概念席卷社区。热点更迭背后，NFT 整个生态在繁荣发展，新玩法和新项目层出不穷。由此催生了对 NFT 发行的需求，Solana 生态 NFT 社交平台 Only1 正在探索 NFT 发行的新领域，其计划与 10 月 7 日推出 NFT Launchpad 平台。

与此同时， NFT Launchpad 还将进行首次 NFT 发行（Only1 Initial NFT Offering，简称 INO），预计将售出 1 万枚 Only1 NFT，筹集到的 SOL 将用于回购 LIKE 以奖励早期 Only1 用户。INO 是一种新型 NFT 发行方式，它能够在分发 NFT 的同时进行融资。

首次 NFT 发行（INO）是什么？

人们从首次代币发行的设计中吸取灵感，相继推出了其他加密众筹机制，如 IDO （首次 DEX 发行）等。首次 NFT 发行或称「INO」也是基于以上基本概念诞生的一种新的 NFT 发行模式。

NFT 是非同质化代币（non-fungible token）的缩写。作为存储在区块链上的数据单元，NFT 具备证明数字资产独一无二、不可替代的属性，因而成为今年市场追逐的热点。

目前有越来越多的平台向其社区提供 NFT 空投。这些空投的 NFT 代币由很多随机生成的、具有可变稀有属性的图像组成，且往往能够为团队募集到一笔资金。INO 便基于此背景诞生。

INO 为 NFT 项目和创作者提供了通过发行 NFT 来吸引支持者和社区的能力；另一方面，INO 让投资者能够从 NFT 固有的供应稀缺性中受益。

随着 NFT 市场日渐成熟，首次 NFT 发行或将作为刺激和奖励投资者和社区的解决方案而异军突起。

Only1 INO 具体工作机制

一般情况下，创作者和项目会首先向社区公布他们的技术文档、计划和路线图，并通过先到先得或乐透模式启动项目，以公平分配他们的 NFT，从而提高社区参与度。不同的项目方可以根据自身情况制定自己的标准和游戏化选项。

在 Only1 的 launchpad 将发行销售一组限量版 NFT，数量为 1 万个。Only1 将根据用户持有的原生加密代币（LIKE）的数量进行 NFT 分配。Only1 鼓励用户质押原生加密代币，以便从即将 INO 的 NFT 项目中获得分配。在 INO 中，投入资金量属于最高层级的用户会在每个项目中获得极其稀有且人气很高的 NFT。

然后 INO 募集到的资金将分配给项目所有者，以执行其路线图。

Only1 INO 有何优势？

随着 NFT 市场不断成长，NFT 发行的融资性质正在变得日益清晰。INO 旨在改变蓬勃发展的 NFT 生态系统，与其他发行方式比较，Only1 INO 具备以下优势：

NFT 项目启动门槛低

任何人都可以通过 Only1 的去中心化平台发行限量版 NFT。INO 使每个人形色各异的构想都能被更广泛的受众认可。很多仍在努力输出的创作者需要新的资金来资助其项目，通过 INO，艺术家可以在创作之前筹集所需的资金。

构建早期支持者社区

Only1 允许社区投票选出有潜力的项目在平台上架，并对获奖者予以奖励，这种对 NFT 项目的曝光鼓励潜在投资者研究项目。这一机制使创作者能够围绕 NFT 项目建立一个早期支持者社区，吸引更多活跃的交易者和经验丰富的投资人。

降低交易成本

Only1 INO 在 Solana 区块链上运行，最大限度地减少了创建者和投资者面临的交易费用。

社交功能

INO 让参与的投资者能够从 NFT 固有的供应稀缺性中受益。Only1 具有原生社交平台功能，可以提高某些创作者的知名度，从而提升其 NFT 的价值。

结语

随着 NFT 市场继续壮大，NFT 项目和创作者需要一种更加便利的方式来推出 NFT，市场需求或将刺激 INO 模式产品的发展。但同时，INO 需要借助新的技术手段和面对陌生的应用场景，其未来发展有也变数。不管怎样，Only1 在这一新领域的大胆尝试值得期待。
