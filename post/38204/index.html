KNN3 Network是一个web3关系数据图谱解决方案，类似于The Graph，KNN3 提供了 GraphQL API 供 web3 开发者访问，web3构建者可以通过KNN3发现多区块链之间的深度关系。

在 web3 世界中，用户在区块链上产生了大量的足迹，这些用户行为不仅反映了用户的行为模式，还解释了彼此之间的联系。KNN3 将这些足迹转换为关系连接器，可以构建一个海量的拓扑图。

KNN3提供了可组合和可优化的图形解决方案，与实时区块链无缝同步，因此，web3 构建者可以构建具有实时链上关系的 dApps。

KNN3 定义了 3 个维度的用户足迹：Action、Status和Links。

动作——web3 用户已经完成的动作。例如：参加的活动、交易/转账记录等。

状态— 用户当前获得的状态。例如：NFT/代币的所有权、会员资格、名称服务等。

链接——用户明确拥有的现有连接。例如：RSS3/Twitter 的社交关系。

KNN3 Network作为产品是一种可组合且无缝的 web3 关系数据服务，可应用于巨大的 web3 原生场景。

Web3营销

KNN3可以为 web3d/App 构建者构建一个发现引擎，根据他们的历史行为和现有关系发现潜在的目标用户。可以利用全面的 web3 用户数据和深入的关系来推广活动，提高准确性、降低成本。

Web3 算法基础设施

使用 KNN3 网络，算法开发人员和 web3 社交应用程序构建者能够使用全范围数据和web3 关系来训练算法，甚至可以在应用程序启动前，KNN3 网络就开始为算法赋能。

金融赋能

定量因子挖掘——KNN3 网络图谱数据结构对许多高级 AI 模型的算法友好。

社交交易——通过准实时（小于 2 分钟）链上流数据馈送，允许交易者深入研究链上关系以找到隐藏的 alpha。

Web3 数据构建模块

KNN3 Network 是一个完全开放的数据库，可供 web3 建设者社区免费访问。KNN3 的 GraphQL API 已发布。
