一、数字经济的现实

1.仅仅24小时内，比特币价格跌幅已近12%，以太坊跌幅8.71%。

5月10日晨间，比特币价格直线跌破30000美元，最低触及29730美元，创下2022年的最低点，也是自2021年7月以来的最低点。

币价的下跌直接传导至衍生品、DeFi、NFT等生态领域，例如据币coin数据显示，过去 1 小时全网爆仓达 1.49 亿美元，过去 24 小时全网爆仓达10.3亿美元。

2.NFT头部项目同样无一幸免，据 NFTGo.io 数据显示，“无聊猿”BAYC、“红豆”Azuki 等蓝筹 NFT 系列地板价 24 小时跌幅均超 20%。其中“无聊猿”BAYC 系列 NFT 地板价现报 89.9 ETH，24 小时跌幅 20.49%；Azuki 系列 NFT 地板价现报 17 ETH，24 小时跌幅 25.27%。

而近月大热的“跑鞋”项目StpenN也未能幸免，其项目代币GMT价格24小时跌幅达到30%，7天内下跌近40%。

3.本轮加密货币的持续下跌与美股保持一致。

美股七大科技巨头——苹果、微软、特斯拉、亚马逊、英伟达、Meta（Facebook母公司）、Alphabet（谷歌母公司）在3日内市值蒸发超1万亿美元（约合人民币7万亿元）。

4.引发此次美股暴跌的导火索是美联储近期的加息动作。

美联储不顾本国已经非常严重的通胀率，依然强硬地通过加息手段收割全球经济。

5.可以看到，当前的世界经济逻辑，依然是美国通过控制美元，继而影响美股，然后美股会很自然传导到所谓“数字货币”或者“加密货币”，继而影响整个区块链生态。

所以，有人告诉你，加密货币是“去中心化”“可靠的跨境资产”，你还信吗？

二、什么是Web 3.0

一千人的眼中，就有一千种Web 3.0。

但是对于幕后资本玩家，世界上只有一种规则，就是：如何最大化剥削大众的剩余价值。

Web本身只是对于互联网的一个名词术语，类似于一块土地，本身是“技术中立”的。

Web 1.0时代，被资本文艺的表达为“只读”时代，即用户只能阅读由企业提供的内容，所谓“中心化”。

也就是地主花钱拉拢或者迫使农民去地里种庄稼，收益都是归地主。

这种情况下，所有生产资料都是由地主提供，产销过程都是由地主负责，生产效率不高。

Web 2.0时代，被资本文艺的表达为“读+写”时代，即用户不仅可以阅读企业提供的内容，更鼓励用户自己贡献内容，所谓“平台化”。

也就是地主告诉农民，这块土地实行了承包制，农民种庄稼收益归自己，地主仅仅是“收取一小部分佣金”。

为了鼓励农民的主动承包热情，在生产初期，地主还会提供工具补贴、种子补贴、化肥补贴等方式，促进承包进程。

这种情况下，农民的生产热情获得极大的鼓舞，逐步依附在土地上成为长期的佃农。虽然生产效率在达到一定阶段后同样不再增长，但是由于固化了人力资源和生产生活方式，地主可以长期、稳定地获得收益，并通过增加工具租用费、提高水费、电费、化肥价格等方式，扩大利润率。

Web 3.0时代，被资本更加文艺的表达为“读+写+共有产权”时代，即用户完全是自己生产活动的产权拥有者，所谓“去中心化”。

在Web 3.0时代，不再有地主，而是所有农民基于“共识”，集体开荒了一块新的土地，在这个土地上，每个人都有产权所有者，根据自己的贡献，比如是否参与了开荒或者耕种，或者是否进行了浇水或者收割，包括最后的存储、买卖，只要在整个生产过程中有贡献，就会根据相应的贡献值获得收益。

这种情况下，农民的生产热情获得了完整、充分地释放，既然农民已经是土地的主人，理论上就是最好的生产关系。

嗯，互联网的“乌托邦”。

Web 3.0这种没有地主的状态，叫做“去中心化”。这种基于“共识”的管理形式，叫“DAO”（去中心化自治组织）。

为了保障这种生产关系的维系，区块链技术就成为底层架构，来满足“不可篡改”“分布式记账”“加密交易”等等应用需求。

种地当然不是个快速赚钱的好生意，因此Web 3.0在应用层上，首先推向社会的是加密货币，然后正在推进的是NFT。

NFT，全称为Non-Fungible Token，指非同质化代币，是用于表示数字资产（包括jpg和视频剪辑形式）的唯一加密货币令牌，设计原理是可以买卖。

从Web 1.0到Web 2.0，再到Web 3.0，本质上是生产关系的演进过程，在这个演进过程中，资本增值变得更有效率。

三、关于元宇宙

元宇宙，是“下一代互联网”的通称。

曾经，马化腾在腾讯大会上提出了“全真互联网”，其实本质上是说的一个意思。

元宇宙既然是“下一代”的互联网，那么就不可能基于现实条件来进行完整准确的表述。这就像在iPhone诞生之前，完全无法想象现在的移动互联网。

但就目前可感知的理论和实践，元宇宙已经至少有三个方向，值得你我这样的普通人关注。

1.加密数字凭证

这里包括比特币、以太坊这样的加密货币，也包括NFT数字藏品等等形式，它们本质上都是一种“加密数字凭证”。

虽然在中国国内，“加密数字凭证”依法不得具有金融属性，但是在海外已经有众多的场景可以进行合法使用。从全球化视野看待，加密货币和NFT数字藏品都有巨大的增长价值，无论从技术角度还是商业角度，都值得学习和关注。

即便在国内，随着监管逐步规范行业健康发展，在合法合规的基础上，加密数字凭证行业依然可以在“确权”“权益”等场景，通过产品创新，有望实现大规模应用。

2.Web 3.0

在多个不同的组织机构和专家学者的表述里，“元宇宙”和“Web 3.0”的内涵互相复用，读者很难理解究竟两者的关系是什么。

资深互联网人士张栋伟的个人理解，是可以将“移动互联网”和“Web 2.0”的之间的关系来借用，即“元宇宙”的下一代互联网的公众表达，“Web 3.0”是下一代互联网的行业表达。

搁置争议，共同开发，实干兴邦，空谈误国。

Web 3.0应用包括对 数字藏品、游戏、虚拟数字人、社交、供应链管理等多个领域的区块链技术重构，产生新的技术应用和实践，并创造出新的商业价值。

谷歌、亚马逊近期正在建立开发工具平台和团队，以支持全球开发者利用其资源，为Web 3.0开发应用程序。

3.VR/AR/MR

最近入手了几台Pico VR，体验效果相比前几年的VR眼镜，已经是质的提升。

我组织的元宇宙社群，已经有群友的企业，开发出一些支持裸眼的VR/AR/MR设备和应用。

VR/AR/MR是消费者最容易感知的产品形式，又是最早可能与教育、娱乐、工业设计等领域融合的应用。

更重要的是，目前在VR/AR/MR领域，我国政策是处于鼓励时期，而不是监管整顿时期。

大有可为。

尾声：

今天是2022年5月11日，以此撰文分享一下截止当前的个人认知。

无论元宇宙还是Web 3.0，都是新兴事物，一如我1998年看到互联网，盲人摸象，管中窥豹。

你对“元宇宙”和“Web 3.0”怎么看，欢迎留言。
