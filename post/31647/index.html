跟随“聪明的钱”一直是投资的高性价比路径。真正难的是，判断谁什么时候聪明，以及了解并同步他们的操作。花费大量时间逐个研究一级市场项目并不适用于大多中小投资者，或许投资某个主题板块、大盘股或白马股的集合对新手更为友好。

加密指数便是其中不错的选择。

12月8日，加密货币指数协议Index Coop宣布完成225万美元融资，红杉资本印度（Sequoia Capital India）、Blockchain Ventures和White Star Capital联合领投。Index Coop业务发展工作组负责人Simon Judd表示，新融资将用于扩大其全职团队，并确保协议在财务上保持稳定。

Index Coop曾于今年7月完成775万美元融资，Galaxy Digital和1kx领投，1confirmation、LD Capital、Defiance Capital、Assembly Capital、Wintermute、Laconia Ventures以及前财富管理高管Phil DiIorio参投。当时的融资主要用于雇佣更多的工程师，将其类似ETF的产品扩展到其他网络，并加强该协议的长期财务稳定性。

Index Coop的运作方式类似Vanguard或Merrill Lynch这样的资产管理公司：它创建一篮子加密货币——混合了比特币、以太坊和其他加密货币——供人们投资。只不过，Index Coop以DAO形式运作，更像是去中心化的资管机构。指数的构成也由DAO成员使用INDEX代币来投票（有种相信群体智慧的味道）。 该项目的匿名领导者之一Lemonade Alpha表示，就Index Coop而言，DAO是DeFi专家、业务开发人员和市场营销人员的集合。每个人或实体——Set Labs、DeFi Pulse以及参与最新一轮融资的投资者——都持有不同规模的治理权力。尽管Index Coop DAO在其Discord服务器上有6000多名成员，但其负责日常运营的（由DAO支付工资的）核心团队大约有100人。

Index Coop还开发了一系列机构级DeFi产品，包括DeFi Pulse Index（DPI）、Metaverse Index（MVI）、ETH2x-FLI和BTC2x-FLI（杠杆产品）、Bankless BED Index（BED）和Data Economy Index（DATA）。散户和机构投资者都可购买以获得加密市场的各个敞口。

Index Coop的BD负责人在接受Decrypt采访时表示，“我们的使命是将安全高效的投资方式带给全世界，如果人们是从跟随趋势和一篮子加密货币开始接触这一投资领域，就再简单不过了。”

Index Coop管理资产曾一度突破5亿美元，但在经历最近的市场回调后下降至约4.19亿美元。 其最受欢迎的产品是ETH 2x Flexible Leverage Index（包括Compound Ether和稳定币USDC），市值近1.9亿美元。

Index Coop的演进过程和产品更新

2020年10月7日，去中心化资产管理协议Set Protocol宣布建立加密货币指数社区Index Cooperative，并推出治理代币INDEX。Set Protocol表示Index Coop首个管理的指数为DeFi Pulse Index（DPI），该指数此前由Set Protocol与DeFi实时数据分析平台DeFi Pulse联合推出，已有200万美元锁定资产和1300位唯一持有人。Set Protocol还阐述了治理代币INDEX的细节：70％的代币将分配给社区，除了向此前的DPI持有者进行INDEX初始分配之外，还提供DPI/ETH Uniswap初始流动性挖矿池。INDEX持有人可参与添加指数、分配社区资金、更新管理等费用设定、提升指数绩效等社区关键决策。

2021年3月23日，Index Coop宣布推出链上灵活杠杆指数（FLI），首个产品为基于Compound的ETH两倍杠杆指数。Index Coop为完善DeFi中的杠杆流程和体验，与DeFi Pulse合作推出了建立在Set Protocol和Compound之上的灵活杠杆指数（FLI）产品，FLI的最初版本是基于借贷协议Compound来实现其杠杆作用，标的资产为ETH，杠杆率为2，用户可在通过Uniswap、TokenSets购买该指数。

4月9日，Index Coop推出关注NFT的指数Metaverse Index（MVI），该指数追踪NFT和虚拟元宇宙行业中的领先代币和项目。截至8月5日，MVI管理规模超过1000万美元，追踪的18种加密货币中排前五的分别为AXS、ILV、ENJ、MANA和SAND，其占比之和超过总权重的64%。

6月15日，Index Coop通过由Bankless提出的BED指数的提案。BED指数跟踪加密货币的排名前3名的可投资资产，由33.3%比特币、33.3%以太坊、33.3% DeFi (DPI) 组成，将于每个月的第一个星期五进行再平衡。该指数的交易费用为0.25%，其中0.125%归Bankless，0.125%归Index Coop。

7月21日，Index Coop称，所有产品都已在以太坊二层解决方案zkSync L2上线。DPI、MVI、ETH2x-FLI和BTC2x-FLI是第一批使用其无许可上币新功能的代币。

8月21日，Index Coop推出基于Uniswap V3的DPI/ETH流动性挖矿计划，活动持续至9月4日，奖励为2000个INDEX代币。

9月24日，Index Coop推出数据经济指数代币DATA，追踪LINK、RENFIL、GRT、BAT、LPT、OCEAN、NMR七种代币的表现。算法做市商Wintermute已向SushiSwap流动性池提供100万美元的流动性，并为DATA代币的OTC提供超过10万美元的资金。Index Coop还将开放SushiSwap上的流动性挖矿。
