数字藏品的技术原理-NFT

NFT 提供了一种利用区块链标记原生数字资产所有权的方法,赋予了其真实性和稀缺性，包括但不限于数字画作、图片、音乐、视频、3D 模型等各种形式，具有不可篡改、不可分割，也不能互相替代的链上权利。这项技术可帮助例如艺术/收藏品、影视作品等文化内容生产行业数字化升级，并带来更多的商业模式创新。

国内主流数字藏品平台

目前国内发行的数字藏品，技术原理与 NFT 基本一致，但在具体的创作机制和交易方式等方面存在差别。投资者和创作者使用的平台有两类，主流是以蚂蚁链的鲸探和腾讯链的幻核为代表的联盟链，这类平台主推数字藏品概念，比较注重合规管理，发行和流通过程均不与虚拟货币挂钩，目前仅以私下渠道联系艺术家与艺术馆的形式发行数字艺术品，并未开放私人创作，发行产品也以收藏为主，因此二次交易上有较为严格的限制。另一类是以 NFTCN 为代表的垂直交易市场综合性平台，采用的是以太坊的侧链，平台对任何完成注册的用户都开放了创作权限，同时盲盒发行、拍卖、定价等多种发行模式，但需要收取一定的手续费。除了上架自有平台的市场，这类平台还会将作品同步到 OpenSea 等海外几个最大的 NFT 市场，以解决流动性问题。

对比这类基于公链的 NFT 平台，像鲸探，幻核这样的联盟链平台的去中心化和不可篡改的特征属性偏弱，而去中心化和不可篡改恰好又是保证收藏品唯一性和安全性的重要技术手段，因此某种程度上也制约了国内数字藏品的流动性和实际价值。但另一方面，结合国内大环境来看，想促进数字藏品的良性发展，我们其实更需要的是合法合规的交易平台，需要一个健全的市场来流通，从现阶段看，我国并不具备这样的流通环境。当前国内 NFT 市场还存在技术不成熟、市场不规范、价值不清晰、监管政策有待明确等一系列风险和挑战，且在 NFT 市场诞生初期，数字藏品为噱头进行的投机、炒作、欺诈、传销甚至是非法集资等行为层出不穷，这违背了我国寻求金融长期稳定发展的初衷。

国内向来对虚拟货币也就是同质化代币的监管历来十分严格。2021 年 9 月 24 日，国家发改 委等 11 部门发布《关于整治虚拟货币“挖矿”活动的通知》，宣布虚拟货币“挖矿”活动将被正式列为淘汰类产业；而在几周后的 10 月 8 日，人民银行，网信办，最高人民检察院与工信部等官方机构联合发布的《关于进一步防范和处置虚拟货币交易炒作风险的通知》再一次明确强调了虚拟货币相关业务活动属于非法金融活动。并从“建立健全应对虚拟货币交易炒作风险的工作机制”、“加强虚拟货币交易炒作风 险监测预警”等角度防范和处置虚拟货币交易炒作风险。海外 NFT 架构在完全去中心化的公链上，具有货币属性，存在与虚拟货币相似的炒作问题，而且其铸造、发行、销售和流转的信息需要通过挖矿来上链记录，能源消耗严重。因此，按照国内对虚拟货币和“挖矿”活动的监管态度，长远来看，这类海外 NFT 的发展模式在国内并不可行。

数字藏品的未来

但可以肯定的是，在未来的数字经济时代，基于 NFT 的个性化、多元化的数字藏品将会成为一种被广泛接受的类似于艺术品一样的存在，并成为一种数字资产的投资方式，在未来架起连接现实世界资产和数字世界资产的桥梁，这也是最切合元宇宙的发展方向之一。而数字藏品真正的价值本质上还是要回归其背后的 NFT 技术，它补全了当前数字产权领域的空白，并让数字内容得以资产化，并高效流通，这背后的市场才是真正的蓝海。数字藏品想要在国内获得真正发展，需要的是健康合规与产业的正视，也需要积极拥抱监管的态度与监管层更宽大胸怀的共建。

我认为随着 NFT 应用的推广和流行，未来 NFT 的铸造、发行、销售、流转都会有监管的介入，NFT 的法律性质、交易方式、监督主体、监督方式将逐步明确。参照虚拟货币的相关监管法律规定体系，未来相关监管部门或将从“明确 NFT 和相关业务活动本质属性”“建立健全应对 NFT 交易炒作风险的工作机制”“加强 NFT 交易炒作风险监测预警”等角度来防范和处置 NFT 多类违法风险。

NFT 在中国的发展路径将遵循不同于海外市场的商业模式，国内数字藏品相关企业更多从版权保护和资产确权切入，发挥 NFT 数字产权证明功能，探索无币化的发展道路或是以人民币作为标价和交易货币保证其价值的稳定，并在数字藏品的发行、销售、流通等方面探索和建立中国化的规则和标准。相信随着数字世界的蓬勃发展，更多的标的物将以数字原生形式出现，让我们拭目以待。

（作者陈晓华：北京邮电大学科技园元宇宙产业协同创新中心执行主任、中国移动通信联合会区块链专业委员会主任委员兼首席数字经济学家）
