2021 年，Chris Dixon 写了一篇关于创作者如何使用NFT来货币化他们的“1000 名铁粉”的文章。从那时起，NFT 用例成倍增加。举个例子：NFT 会员的兴起，要求粉丝持有特定的 NFT 才能加入私人社区并享受其特权。NFT 会员为创作者提供了一种加强他们品牌，建立有价值的社区，并让粉丝随着社区发展分享收益的方式。 

从我在 Twitch 和 Facebook 致力于创作者增长，以及撰写关于创作者经济的文章，我相信NFT会员将成为与广告和订阅相媲美的创作者获利渠道。根据现实世界中的例子，我已经分析了为什么创作者应该考虑 NFT 会员，它们在实践中的使用方式，以及创作者经济和 web3 初创公司如何从其日益流行中获利。

NFT 会员的用例

虽然所有 NFT 都是存储在区块链上的数字收藏品，但不同的类型可以实现不同的目的。例如，人们经常购买个人资料图片NFT (PFP) 来用作头像。然而，有了NFT会员资格，NFT就是一张进入一个专属团体的门票。想想看：Soho House 或乡村俱乐部，但会员资格是线上优先的，进入这里取决于是否是代币持有者vs.是否拥有正确的连接。

越来越多的创作者通过为他们的粉丝建立 NFT 社区来涉足 web3。对于有商业头脑的创作者，NFT 会员可以在几个方面取代和补充传统订阅和基于广告的收入模式：

创作者可以通过细分粉丝的付费意愿，从 NFT 中获得更多收益 。在 YouTube 等传统平台上，创作者使用广告来锁定不会为内容付费的普通粉丝，并使用订阅来锁定愿意支付少量费用的粉丝。结果是，他们没有从愿意支付更多的忠实粉丝那里获得最大收益。NFT 通过针对各个层次的粉丝，让创作者获得更多收益。创作者可以针对他们最忠实的粉丝推出限量版 NFT 系列，后续针对更多的普通粉丝推出其它系列。

创作者可以通过降低中间商费用从 NFT 中获得更多收益 。传统平台从创作者的订阅和广告收入中抽取很大一部分，并且可以在未经创作者许可的情况下随时更改政策。有了NFT，创作者可以保留他们从最初的铸造中获得的大部分收入（请参阅此NFT 铸币指南），然后还可以从版税中获得额外收入。

创作者应该特别感谢早期粉丝 。在传统平台上，随着创作者越来越受欢迎，创作者的早期支持者不会获得任何回报。当然，有人可以说他们在一位YouTuber成名之前就发现了他——但这很难证明，而且早期粉丝的身份不太可能带来切实好处。另一方面，NFT为粉丝提供了一种方法，以明确证明他们对创作者的早期追捧者。创作者可以用实际好处来回报这些早期粉丝，例如参加活动和特别发行版本。此外，创作者（从音乐家到运动员）的早期收藏品往往会根据创作者的成功而升值。通过这种方式，NFT 直接激励粉丝帮助创作者取得成功。 

NFT 会员面临的挑战

创作者还需要注意 NFT 会员的潜在困难方面。

创作者无法直接控制 NFT 铸造后的价格 。尽管创作者可以设定初始 NFT 铸造价格和他们在二级版税中的份额，但他们对NFT 未来价格的控制力较弱。例如，如果整个 NFT 市场崩溃，创作者最终从 NFT 中获得的收入可能会低于传统渠道。或者，投机者可以购买创作者的 NFT 并抬高价格，而不给真正的粉丝参与的机会。为了防止后一种现象，创作者可以在投机者了解该项目之前尽早向粉丝推广NFT的“白名单”位置。他们还可以制定规则，禁止任何一个人一次购买过多的 NFT。

如果 NFT 的价值不明确，创作者可能会疏远粉丝 。创作者需要明确阐释其 NFT 会员的价值，以免让粉丝失望。粉丝们要求创作者信守诺言，并期望他们为 NFT 会员提供长期效用。在没有明确计划如何为会员提供价值的情况下，推出NFT会员资格会使创作者的声誉面临风险。

管理 NFT 会员社区是一项艰巨的工作 。社区管理是创作者现有内容生产计划之上的一项重大承诺。为了帮助减轻他们的工作量，创作者可以聘请一个团队来保持社区的参与度，并鼓励粉丝在提升会员价值方面发挥作用。但他们不太可能将维护参与度的职责完全外包——毕竟，创作者是他们品牌的核心。

创作者如何在实践中使用 NFT 会员

要让NFT会员制落地，最少要满足哪些要求？成功的社区通常包含三个组成部分：

  1. 1.创作者已经有一个有参与度的粉丝群，可以邀请这个群的粉丝来购买他的NFT。

  2. 2.粉丝购买 NFT 加入社区，并帮助增加会员价值。

  3. 3.创作者及其社区为会员提供现实世界利益效用。

让我们看看这三个NFT会员社区的例子，可以很好地说明这种模型。 

VeeFriends是一个来自Gary Vaynerchuk的NFT系列，此外，Gary Vaynerchuk 是预订平台 Resy 和数字机构 VaynerMedia的联合创始人，是这些公司的CEO。

  • 创作者： Gary 凭借他的企业信誉，建立了庞大的多平台受众群体。

  • 社区： Gary 通过教育的方式使他的听众加入VeeFriends社区，了解为什么NFT很重要以及如何首次购买。他为他的NFT系列中所有角色绘制并创造了背景故事，为代币注入了人物联系。

  • 实用性： 所有 VeeFriends 持有者均可免费获得VeeCon会议的三年通行证。2022年5月举行的首届VeeCon会议，邀请了Snoop Dogg和Beeple等重量级发言人。VeeFriends 持有者在组织这次活动中发挥了关键作用。

Moonbirds，来自Kevin Rose的私人社区Proof Collective的第二个NFT系列。 

  • 创作者： Kevin 主持流行的加密播客_Modern Finance_。

  • 社区： Proof 的第一个NFT系列有 1000 张通行证。许多收听 Kevin 播客的顶级 NFT 收藏家购买了这些通行证，这为社区注入了人才，并为Moonbirds创造了潜在的需求。

  • 实用性： Moonbirds 持有者被激励去长期持有他们的NFT，以解锁专属奖励，例如商品和现实生活中的活动。

Developer DAO，开发者Nader Dabit在离开AWS专注web3后推出的DAO。

  • 创作者： Nader Dabit在他的YouTube 频道上收获了一批受众，该频道专注于开发者教育。然后，他推出了 Developer DAO，以加速 web3 建设者的教育和影响。

  • 社区： 开发者和其他人才购买了Developer DAO NFT，增强了会员资格的吸引力。

  • 实用性： 访问社区本身已成为 NFT 的主要优势。成员们互相帮助举办活动、建设项目和找工作。

公司如何帮助创作者管理 NFT 会员

创作者和社区正在以新的方式试验 NFT 会员。随着这一趋势的兴起，从 web2 跳转到 web3 的创作者在整个用户旅程中将需要帮助。为了支持创作者，初创公司可以：

创建 NFT 会员

  • 定义会员详细信息（例如，目标、受众、数量、价格、福利）。

  • 招募团队或选择无代码解决方案来制定艺术品、智能合约和网站。 

向粉丝介绍会员资格

  • 建立一个社区，并告诉粉丝为什么 NFT 会员资格很有价值。

  • 使用白名单为粉丝而不是投机者保留 NFT。

与社区合作，为 NFT 会员创造价值

  • 了解哪些NFT持有者对社区贡献最大。

  • 与NFT持有者合作，为 NFT 会员提供持续的效用。

管理业务

  • 管理并找到为金库提供资金的方法，例如通过合作伙伴关系。

  • 管理税收和法律合同。

许多公司已经在应对上述挑战。我很高兴看到更多的创作者和社区使用 NFT 作为创造价值和分享收益的工具。
