从 DeFi 到 DAO

随着时间的推移，虚拟房地产市场可能会开始看到类似于现实世界的服务，包括信贷、抵押贷款和租赁协议。 然而，随着去中心化金融 (DeFi)的出现和基于区块链代币的数字资产的可组合性，下一代融资公司可能会利用数字服装作为抵押品来承销虚拟土地和财产抵押贷款。 事实上，它可能根本不是一家公司，而是一个自组织的、基于任务的社区，也称为去中心化自治组织（DAO）。DAO 可能已将其原始资产负债表植入一个多签名钱包，以创建抵押贷款。

在虚拟世界中工作

供需动态正在推动更多人进入元经济。反过来，这将需要开发新技能，并创造新的赚钱机会。毕竟，人们将不得不开发和制造在虚拟世界中消费的产品——为创造者经济创造巨大的机会。RTFKT是一个虚拟运动鞋品牌，最近被耐克收购。它在数字商品和实体商品之间架起了一座有趣的桥梁（NFT 所有者会收到与其 NFT 相匹配的实体运动鞋），除了可穿戴设备外，它还在构建一个由虚拟形象、空间和数字产品组成的生态系统。在最近的一次拍卖会上，RTFKT运动鞋的售价高达每双1万美元。相比之下，有些人在元宇宙中提供服务。例如，如果您计划举办一个虚拟派对并想要音乐，您可以聘请歌手或 DJ 来表演。虚拟音乐会有可能比实体音乐会更有利可图，因为实体音乐会存在容量和停车等障碍。最近，Fortnite 举行的一场大型虚拟音乐会吸引了4500万人观看，票房收入约为 2000 万美元，其中包括商品销售。由于地理或成本原因，原本无法获得此类体验的人现在可以参与。

我们将拭目以待，看看营销和广告的效果如何。由于拥有大量受众，元宇宙中的品牌推广和身临其境的广告体验有巨大的机会。与当代社交媒体公司一样，广告是许多平台用来创造收入的模式。到2027年，游戏中的广告支出将达到184.1亿美元。

跨越行业和国家的大量机会

元宇宙将为企业提供巨大的机会，以一家正在为其设备购买新部件的制造商为例。目前，这一过程包括收到实体手册或通过电子邮件发送的带有静态2D图片的PDF。在元宇宙中，用户可以在虚拟环境中以较低的成本测试产品。

元宇宙的巨大可能性之一是，它将为新兴和前沿经济体的消费者扩大进入市场的机会。互联网已经解锁了对以前遥不可及的商品和服务的访问。 现在，低收入国家的工人可能无需移民就可以在西方公司找到工作。教育机会也将扩大，虚拟现实世界成为一种低成本且有效的培训方式。随着这些发展，还必须有明确的治理。

从企业的角度来看，存在大规模扩张的机会。一家大型零售商可能会在元宇宙建立一个能够服务数百万客户的全球中心，而不是在每个城市都开设门店。除了零售商，元宇宙将加速游戏、体育博彩从现金到加密货币的转变。

Sightline Payments等公司已经建立了基础设施，为无现金数字游戏提供支持，随着这些行业迅速转型，为元宇宙中的游戏玩家提供服务，这些公司可以利用这一机遇。

炒作VS现实

尽管元宇宙的未来令人兴奋，但为了充分发挥其参与、社区建设、表现力和商业的潜力，需要进一步开发关键领域。我们将这些视为团队、项目、DAO、企业、技术提供商和金融机构在充满活力的生态系统中共同繁荣和协作的新机遇，涉及以下领域：技术、商业基础设施、隐私和身份、未来劳动力、监管、税收、会计和社会基础设施。

技术

从技术的角度来看，元宇宙发展的一些关键要求是：

改进了化身移动和环境渲染的性能

减少环境“分片”，使所有参与者可以在同一地点进行互动

更大容量以支持设计和交互的复杂性

减少与 3D 渲染进行复杂交互的本地硬件需求

扩展跨设备（包括移动设备）的可访问性

开发互操作性或跨虚拟世界交互，以及跨这些平台管理参与度和数字资产的方法

定义数字资产（可穿戴设备、物品、品牌）和虚拟/加密货币的标准，以便它们可以在不同的虚拟世界中转移

先进的发现机制，例如寻找朋友或推荐要做的事情、参观的地方和定制虚拟商品的能力

扩展虚拟空间的数据分析和报告。这些将专门用于商业和营销用途，并将跟踪业务关键绩效指标

商业基础设施

支持商业和元经济的一些关键要素仍需要确定和调整，包括：

Web 3.0 虚拟世界与传统传统金融支付轨道（例如信用卡、银行支付、借记卡、电汇）的集成

Web 2.0 虚拟世界与加密支付和 NFT/数字资产市场商务的集成

单一钱包用户体验，包括：

  • Web 2.0 传统金融支付轨道、虚拟货币和虚拟资产

  • Web 3.0 加密、NFT 和数字资产

  • 数字身份可验证凭证、了解您的客户 (KYC)/反洗钱 (AML) 合规密钥和信誉积分

  • 多个别名以保护隐私并实现数字自由

通过使用贷款模式或利用去中心化金融（例如，NFT 抵押品支持的虚拟世界抵押贷款），改进虚拟/加密货币和数字资产支持的融资和抵押

创建跨境和跨元宇宙外汇和流动性解决方案

改善主流采用非加密/Web 3.0 原生产品的用户体验

扩展解决方案和服务，以支持基于 DAO 社区的项目、商业企业和投资

隐私和身份

用户身份识别和隐私保护对于元宇宙中的交互和交易都至关重要。

  • 保留一个人拥有多个化身/身份的能力，并增加：

私人 KYC/AML 兼容的商业和支付

可验证的凭据，可以轻松构建以更轻松地识别社区/团队成员，或启用对不同虚拟世界位置和体验的可配置访问

防止跨虚拟世界的网络欺凌或在线骚扰/攻击

  • 扩展 NFT 代币门控空间，包括创建私人互动、讨论和消息传递

未来劳动力

元宇宙中未来的工作会是什么样子？ 需要考虑的一些关键领域包括：

  • 利用虚拟世界特定软件开发工具包 (SDK) 培训新的设计师和开发人员，专门从事虚拟化身、可穿戴设备、对象、建筑和室内设计的 3D 建模

  • 内容创作者可以跨越虚拟世界、虚拟商品、NFT、商业和社区

  • 重新部署专门从事元宇宙体验（音乐会、派对、艺术展览、社区会议、体育赛事、观光和旅行）的活动制作人

  • 创建社区经理，专注于建立强大的线上和线下关系建立和互动，以加强平台参与度和粘性

监管、税收、会计和社会基础设施

当涉及到管理元宇宙的监管框架时，有许多悬而未决的问题需要解决，包括：

  • 为 Web 3.0 数字房地产/财产和虚拟文字商业交易的监管、税务和会计处理铺平道路

  • 熟练掌握主要使用 Web 3.0 数字资产的 Web 2.0 虚拟世界的监管、税务和会计处理

  • 支持全球可访问的虚拟世界的解决方案和服务，但可能需要遵守当地的商业和支付管辖要求和规则

  • 社区治理的演变（例如，谁在虚拟世界中制定规则？谁治理？）

你的“元宇宙策略”是什么？

在元宇宙中，我们熟悉的一些现有服务和商业模式将继续存在，但元宇宙开辟了一个全新的参与方式，我们预计这将带来独特的新服务和商业模式。并非元宇宙中的所有内容都与每项业务相关。但是，借此机会去探索也没什么坏处。

学习

对元宇宙有许多不同的解释，除了这里的内容之外，还有不同的角度可以了解。

评估

虽然元宇宙将跨越许多行业垂直领域，但请评估您的业务是否有相关机会。

创建

通过内部发展获得经验。借此机会通过新的渠道、服务、体验、数字商品和资产探索创造性的消费者参与。从小处着手，快速行动，测试和学习。

连接

开始构建元宇宙生态系统参与者网络。世界仍然很小，因此在合理的投资水平上仍然可以获得高质量的合作和伙伴关系。与新一代建立联系并利用替代人才库。

建立

让您的业务走出去，并定位自己，吸引科技前沿社区。

我们对元宇宙的态度

在元宇宙中构建和扩展的成功取决于拥有一个强大而灵活的金融生态系统，该生态系统将允许用户在现实世界和虚拟世界之间无缝连接。我们的支付和金融基础设施方法将使这种互操作性得以增强。

我们相信，现有的虚拟游戏环境（每个虚拟世界都有自己的人口、GDP、游戏内货币和数字资产）具有与现有全球经济类似的元素。这就是我们在跨境支付、外汇、金融资产创造、交易和保管方面的长期核心竞争力，以及我们大规模的消费者立足点，可以在元宇宙中发挥重要作用的地方。

我们正在构建和扩展新兴技术，以实现基础设施和商业模式的现代化，包括但不限于代币化和数字身份，因为我们力求在去中心化网络中不断创新和更好地组织金融交易和支付。
