UST脱锚的前情就不多说了，这个目前已有很多分析文章，本质问题都分析的很透彻。

我也是第一次见第一天跌了95%+的代币，第二天还能继续跌99%，而此种情况下，交易所还暂没下架，借贷协议也没暂停借贷。

从这里可以分析出很多，争取通过这个事件，让自己能学习到更多的风险管理知识、套利策略和风险规避动作。

整个事件的分水岭，应该是DK的发言。在这之前，大家主要关注有什么措施能让UST重新锚定。

而DK的发言， 清晰的表明他们没有任何外部解决方法。从推文可以看出，他先是说了UST稳定的原理，引用了一个DEPEG的详细说明：

之后这两条是重点，他明确只能依靠DEPEG机制，不断的燃烧UST，增发LUNA，通过套利的形式让UST回到1。为了增加这一进程，社区还在投票一个增加了燃烧效率的提案，以便加快UST套利速度。

看到这两条消息，我的第一反应是项目完蛋了，如果这么燃烧，按照LUNA当时的市值基本很快就会归零。

第一个机会点是消息面做空。风险是交易所关闭充提，杀空头。

DK下午发言后，LUNA在交易所价格也很快到了0.7附近。但这时交易所关闭了充提，LUNA价格被拉到7。

此后大概进入了美区交易时间，LUNA价格进入了市场博弈，没有再次暴跌，稳定在1附近。

5月11日，大家主要关注点在LUNA和UST的价格上，关注消息面大多是恢复锚定措施，以及在交易所的多空厮杀。

从LUNA增发量看，5月11日的增发体量也并没有超过认知范畴，价格和增发量也基本可以对齐。

如果5月11后交易所直接下架交易对，后面512的大戏也就不会上演，链上并没有那么大的池子能容纳这巨量的增发套利。

关键点就在于512一整天，大所并没有关闭交易对，让整个套利链条能完整运行。

还有如果不了解这个机制的人，是无法想象会有如此巨大的增发量的。交易量有那么大，就是因为这里有巨大的信息差，有人觉得已经跌了99%，有那么大的生态，可以抄底搏一搏。

巨大的信息差形成了巨大的交易量，让luna在大量增发的同时，价格并没有跌的像增发量那么快。

LUNA的价格，在不同的市场已经有巨大偏差，而突破口主要是2个地方，一个是官方的兑换渠道，预言机价格按照1美元计价，另一个是交易所渠道，有巨大交易量托底。

这里的主要套利路径就变成：交易所购买UST-> 提到terra station兑换LUNA -> 充值到交易所卖成USDT -> 再购买成UST

这其中，在station兑换会有4%左右的折价。不过相对UST和USDT的折价，这个空间还是巨大的。如果UST/USDT为0.5，96% - 50% = 46% 依次套利就有接近46%的利润。当UST到0.7时，也有26%利润。

这个过程也需要2个价格保护，一个是购买UST时，一个是兑换LUNA时。

在DK的推文中，有提到社区的一个提案，增加UST的燃烧效率，我看了下这个提案的状态，目前还是在投票中。也就是说，即便没增加效率，市场的动作已经充分的发挥了作用。

其实，站在理性的角度考虑。当DK说让UST通过DEPEG机制恢复锚定时，只要是对算稳有了解的，就一定知道这就是宣布死亡。这个过程太快了。

而LUNA没有立即死掉，这里交易所有一天多硬生生没下架，不知道是咋想的，让大量不了解情况的人在里面厮杀。

站在套利的角度，市场价格没有反应项目实际情况时，就有套利空间。能做到可能需要懂得原理，会相关操作，找得到具体数据，做好风险规避动作，最后还要有点胆识，或者是已经练就的肌肉记忆。

从增发量和市场价格走势看，512一天LUNA按照每一个块的上限稳定增发，价格也基本是持续走低，又继续跌了99%。

513，交易所下架了LUNA和UST，大戏至此落幕。

其实从UST的价格走势可以看出，支撑这一大戏的核心是交易所上有这2个币种交易对，因为中心化有很多不透明因素，因此存在信息差。

512早9点，UST在交易所价格达到3d高点，此后一路走低，LUNA的价格也是。

从511晚7点到512早9点这一阶段，UST的价格从0.35恢复到了0.8。如果不了解机制的朋友，可能以为这是市场在恢复信心，在自愈。

通过结合增发数据看，这一时间段LUNA大概增发了3倍，UST价格回升实际是被套利者购买上去的。

而512的9点后，进入东8区交易时间，大家发现了增发量如此大，价格才再次下行。而UST在下行过程一直没有大的瀑布，也是套利在不断套利的表现。

一切还是从DK那条推文开始，它那14条其实就一个意思，你们去套利吧。正常来说，创始人这么说了，大概率直接到底。套利机制能生效，是需要有对手盘来支撑的。如果是链上项目，一次性池子就干了。而有了交易所的存在，让这个盛宴能维持一天多。

说的有点散，简单总结下：

时间线：
• DK发推，宣布躺平
• 市场反馈价格从3.5-0.7
• 交易所关充提，价格到7
• 511晚-512早9，市场价格博弈，没有大波动
• 此时间段内LUNA增发3倍左右，套利者隐秘行动
• 512早9，UST恢复到0.8，套利效应比较明显
• 512早9后，东8区时间，发现大量增发后再次恐慌盘
• 此后估计是全民套利，也有乱冲的很多，当了对手盘
• 513交易所下架LUNA，大戏落幕
• UST价格0.1
• LUNA供应量万亿+

需要了解的知识和信息面：
• UST的机制
• UST和LUNA的关系
• 铸造和烧毁UST的操作方法
• 套保的操作方法
• 各交易所和链上的价差
• LUNA增发的数据
• 预言机价格和实际价格差
• 市场对当前实际情况的反映程度

可能存在的风险：
• 交易所关闭充提
• 套保单被爆
• 撸一半下架
• 各种未知的中心化风险

以上内容纯属虚构，散了吧。
