环顾四周，一些惊人的新兴的 NFT 项目正在悄悄崛起。 尽管最近几周“NFT 终于死了”的讨论在主流圈子里沸沸扬扬，但生态系统中的创造者和梦想家们仍在坚持，并没有被所有的噪音所吓倒。
我一直在寻找围绕 NFT 的有趣工作，一直在探索那些能给我留下深刻印象的新项目。
想知道它们是什么吗？以下。

1) Archipelago

是什么：Archipelago 由 Art Blocks 爱好者创建，是一个为所有生成艺术定制的 NFT 市场。在写这篇文章的时候，该项目已经上线一个多星期了，而且在这么短的时间内，该平台已经因其优势和前所未有的设计元素收到了许多好评。

为什么重要：当涉及到满足生态系统中蓬勃发展的生成艺术场景的细微差别时，其他 NFT 市场还有很多不足之处。 在专门迎合生成艺术爱好者的需求方面，Archipelago 相应地会在短期内成为 NFT 领域的一个重要活动中心。

2) Astaria

是什么：Astaria 由前 Sushi 首席技术官 Joseph Delong 领导，是一个即将推出的 NFT 借贷协议，它将使用一种新颖的点对点方法，并以三种类型的利益相关者为中心：评估者、借款者和贷款者。

为什么重要：在过去的一年里，随着各项目竞相实现产品与市场的契合，NFT 借贷项目的场景一直在扩展。这个领域有大量的需求，也有许多项目正在解决这个问题，但似乎还没有项目能决定性地“破解密码“。Astaria 前所未有的设计可能最终代表了这一领域的一个飞跃。

3) Bueno

是什么：Bueno 是一套针对 NFT 创作者的无代码工具，该项目第一个旗舰产品是其 NFT 生成器功能，允许你上传艺术图层，例如 Photoshop，然后亲自简单定制你的 NFT 系列的规则和稀有性。该项目还即将发布一个无代码的智能合约部署工具，因此你可以随时定制你的艺术，并从 Bueno hub 部署你的 NFT 合约。

为什么重要：Bueno 的工具将使非技术性的创作者更容易部署他们自己的高级 NFT 项目，这可能有助于 NFT 生态系统在未来几年大大扩展。

4) ClubNFT

是什么？ClubNFT 是一个 NFT 基础设施项目，由长期加密艺术专家和创作者 Artnome 领导。由于早期的收藏品使用私人服务器，Artnome 个人损失了许多无价的 NFT。然而，私人服务器仍然被许多 NFT 所使用！ClubNFT 旨在帮助用户检查哪些 NFT 面临存储风险，并备份以备后用。

为什么重要：人们购买 NFT 是希望可以无限期地访问它们。这就是为什么许多人如果现在不采取积极的解决方案来备份他们脆弱的 NFT，就会在未来发现 NFT 丢失的原因。ClubNFT 使这一过程变得简单，该平台准备在未来几年内促进有弹性的 NFT 生态系统的发展。

5) Spores

是什么：Spores 是一个建立在 Zora 协议之上的音乐 NFT 混合器应用。该项目允许用户对音乐 NFT 进行混音，然后将这些混音作品作为“Spores”存储在 IPFS 数据存储网络上。

为什么重要：有时，在 web3 的所有喧嚣中，人们会忽略去中心化应用程序是否真的有趣。对我来说，Spores 在很大程度上通过了乐趣测试。有一天我发现自己在这个项目上玩了一个多小时，只是静下心来享受我的小音乐 NFT 的 DJ 狂欢。我很想看到更多像这样有创意且令人愉悦的项目，如果 Spores 是一个早期的迹象，我们将看到更多这样的项目。
