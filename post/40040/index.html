什么是音乐 NFT？

NFT 是记录在区块链上的独一无二的加密代币，它可以指向特定的数字或实物资产，包括视频、图像，或你能想到的任何类型的资产。恰如其名，音乐 NFT 是与一段音乐作品相关的 NFT。音乐 NFT 可以是一支单曲、整张专辑、MV，甚至是一段 生成性音乐，即使用计算机程序将随机的模式、颜色、声音或形状用算法生成的音乐作品。

当然，音乐 NFT 也可以指向其它跟音乐相关的东西，比如演唱会门票或专辑封面。我们无需过于纠结定义，总之音乐 NFT 和＂音乐＂类别下的数字资产密不可分。

音乐 NFT 可以铸造为 1/1 或者＂版本＂两种形式。

1/1 铸造是指该 NFT 仅有一枚，独一无二。开放版本是指某个 NFT 可以无限铸造；而限量版则受限于预先设定的铸造数量。一些艺人选择 将专辑中的每首曲目按 1/1 铸造，并单独进行拍卖；而有些艺人则 将曲目铸造为限量版或开放版。每种发行方式都有各自的优缺点。

1/1 增强了 NFT 的稀缺性，从而有可能增加其价值。虽然版本类铸造的稀缺性较低，但通常对建设社区和提高粉丝忠诚度来说是更好的选择。例如，如果你希望将来为 NFT 持有者提供独家服务，比如门票预售，那么版本类铸造可能是更好的选择。

NFT 如何让音乐人受益

许多音乐人说音乐产业已经支离破碎。尽管流媒体平台的兴起为粉丝们创造了一个黄金时代，让他们可以轻松访问几乎任何音乐，但它对艺人来说并不那么友好。音乐行业目前的商业模式似乎施惠于大厂牌和平台，却往往 以牺牲独立创作人的利益为代价，而他们才是行业的命脉。

事实上，通常艺人只能从每次流媒体播放中挣到 0.003 至 0.005 美元。总体收入情况又如何呢？让我们来看一看 Spotify 的情况：Spotify 公司市值 430 亿美元，但每年能够在其平台上挣到超过十万美元的艺人只有 大约 7,500名。基于以上数字可以发现，流媒体似乎给消费者和供应商带去的好处远大于艺人。

有了 NFT，艺人们便能够选择一种全新的经济模式，在不需要第三方中介参与的情况下，靠自己独特的知识产权获得收益。歌曲的发行和流传，不必再依靠某家厂牌或流媒体服务——二者都会从中分一杯羹——取而代之的是，艺人们独自就可以在区块链上铸造并出售音乐 NFT。

此外，得益于区块链的内在机制，铸造 NFT 的音乐人还可以从二次销售中获得版税收入。这缓解了由于作品销量不佳，或中心化流媒体服务付费极低所造成的 音乐人挣钱难 的问题。简言之，许多人购买音乐 NFT 是因为这样可以为自己喜爱的创作人赋能，给予他们经济上的支持。

音乐 NFT 的区块链

选择在哪条区块链上铸造 NFT 时，要考虑到这些要点：网络规模，创建成本，消费者的消费习惯，安全性和社区输入。以下是对 NFT 区块链的高度概括：

以太坊 ：对 NFT 来说最受欢迎的区块链是以太坊。以太坊上有着一些最热门的交易市场、 NFT 项目（ 加密朋克 和 无聊猿 ）和 NFT 艺术家（ Pak 和 Beeple ）。虽然在所有选择中它费用最贵，但是，鉴于其受欢迎程度、安全性、去中心化程度和易用性，以太坊应该作为优先考虑。

Solana ：这是加密领域速度最快的可编程区块链之一。Solana 别出心裁地将历史证明 （PoH） 和权益证明 （PoS） 共识机制结合在一起，从而大幅压缩了验证时间和费用。真的，它的交易费用已经低到不能再低了。美中不足的是，Solana 的采用率不如以太坊那么高。

币安智能链（BSC） ：如果想在费用和性能之间获得平衡，BSC 是个好选择。问题在于，它的高速交易和低费用是以去中心化为代价的。币安是文中列出的所有选项里中心化程度最高的一个。所以如果你特别看重去中心化，那还是看看别的选项吧。

Tezos ：如果你想要更环保的替代方案，Tezos 是不二选择。Tezos 于 2021 年初开始流行，它消耗的能量比以太坊少两百万倍，而且费用还低于以太坊（但高于 Solana）。

如何铸造音乐 NFT

市面上几乎每个月都会出现新的 NFT 市场和 Web3 流媒体服务，所以，想要铸造和推广音乐的音乐人选择还是挺多的。铸造音乐 NFT 的方式有两种：使用市场铸造或者自己铸造。

但无论选择哪种方式，都离不开智能合约。以下是两者之间差异的详细解析。

NFT 市场：市场允许用户在其智能合约上铸造 NFT，由它们来负责铸造。选择 NFT 市场的额外好处在于，大多数平台已经拥有庞大的用户群，这意味着你可以直接吸引现成的发烧友来抢占社区建设的先机。但有一点需要注意：通过市场铸造的话，你对作品系列或店面无法拥有完整的所有权或控制权。

创建自己的智能合约 ：考虑到作品溯源问题，拥有自己的合约可谓无价。无数艺人选择在自己的合约上铸造 NFT，从而能够有效地掌控 NFT 制作的整个过程。自己创建智能合约费时费力，但 Manifold 之类的服务让这项工作比以前容易多了。

对于初学者来说，与市场合作可能是最好的选择。虽然不能随心所欲，但在你对 NFT 音乐生态更加熟悉之前，这是个良好的起步。市场提供的一体化解决方案还可以帮助你更快地启动项目。

选择 NFT 市场

通用 NFT 市场

选择市场时要考虑：是一次铸造一枚 NFT 进行拍卖，还是铸造一个作品系列，或者一批 NFT，并且需要对每个 NFT 单独定价。对于后者，请考虑一些世界上最大的 NFT 市场，如 OpenSea，LooksRare 和 Rarible。OpenSea 是最受欢迎的 NFT 市场，平台上有超过 100 万个活跃用户钱包。LooksRare 和 Rarible 是 OpenSea 最强大的两个竞争对手。另一方面，如果要铸造 1/1 NFT 歌曲，那么像 SuperRare，Foundation 和 Zora 这样的平台是最好的选择（但 SupeRare 和 Foundation 都是邀请制）。

而且要做好准备，铸造有初期费用。大多数时候只需要 支付 gas fee（交易费），但有时市场会附加额外的费用。如果费用一开始不明显，那么接下来你有可能还会支付其它费用。

同样，在研究版税分成时，请确保做了应有的调查。OpenSea 或 Rarible 等平台不能保证你获得跨平台版税。像 CXIP 这样的智能合约和铸造工具可以解决这个问题；此外，无论你的 NFT 转售到哪里， 0xSplits 都可以帮助你实现二级销售的自动版税分成。

专用音乐 NFT 市场

Catalog 是流媒体服务和 NFT 市场的混合体。在 Catalog 上，你可以在开放的音乐库中添加一条记录，设置一个＂立即购买＂的价格；将其加入逆向拍卖列表，或接受收藏者提出的报价。虽然该平台完全是一个 1/1 市场（没有版本类作品），但初次销售的收入可以 100% 由艺人获得，也能够从转售中获得收益。

Sound.xyz 是一种将流媒体与音乐 NFT 铸造相结合的服务。在 Sound 上，艺人可以使用一系列开放或限量 NFT 为新歌发行举办欣赏派对。艺人可以保留 100% 的销售收益。该平台也不局限于单曲，甚至还举办过像 Soulection+ 这样由 18 位不同艺人通过自动分账的形式演出的活动。

Async Music 是著名的 NFT 平台 Async Art 的一部分，专注于音乐领域。在 Async Music上，每个音轨（吉他，钢琴，人声，打击乐）都单独上传，并具有许多不同的版本。这让艺人能动态地铸造音乐──把多个音轨组合成一个主音轨。

Mint Songs 是唯一没有关闭注册或仅限邀请的音乐 NFT 平台之一。其核心使命是帮助艺人通过音乐创作获得足以持续维生的收入。为了帮助艺人将他们的音乐变成 NFT，它提供了各种工具，并且铸造费为 0。而且它运行在 Polygon 上，这使得该平台即便在没有高昂交易费用的情况下也非常容易访问。

Royal 是 3LAU 的新音乐 NFT 市场，NFT 所有者能够获得在该平台上售出歌曲的相应权利。这意味着，那些收集了 Royal NFT 的人将能够获得该曲目在传统流媒体平台（Spotify，Apple 等）上产生的部分版税。用户可以使用信用卡购买 NFT（无论以何种标准衡量，这个功能都独具特色），但目前该平台的内容是经过高度策划的，网站上的每一位艺人和所铸造的曲目都由 3LAU 和他的团队精心挑选。

OneOf 托管在 Tezos 区块链上。该平台的内容目前是高度策划的，不允许独自铸造，只能收集，但它是一个发布作品的好地方。Objkt 事实上已经成为 Tezos 链上 NFT 交易市场，也是铸造音乐 NFT 的好地方。Kalamint 是另一个不错的市场，值得一看。

如何为 NFT 定价

如果你通过平台铸造 NFT，通常会收到有关定价的意见。如果你自己铸造 NFT，则可能很难为你的作品定价，尤其是这个数值是用加密货币的小数来表示的。

这个部分需要你做上一些调研。没有人能确切地说清楚你的 NFT 值多少钱，或者人们愿意掏钱买什么。但是，你可以查看一些艺术家的情况，来得到为自己的 NFT 定价的一般思路。价格每个月都有变化，因此你应该等到接近发布时再进行调研。去上一节中提到的任何一个平台查看系列作品卖什么价。你甚至可以把 OpenSea的音乐分区 都过一遍，看看现在趋势怎么样，成交价格情况如何。

最重要的是，要灵活。每个人都想立即以 1 ETH 的价格卖掉某个 NFT。然而，现实情况是，即使知名艺人有时也很难找到收藏者。给你的 NFT 标上个理想的价格是可以的，但如果这个价格不起作用，降低一点也不会损害你的声誉。

事实上如果你刚起步，设定低价并且调低 NFT 的铸造数量甚至对你有利。低价（约 0.05 至 0.1 ETH）有助于你将现有的粉丝转化成收藏者，而不会因为高标价把他们吓跑。先以可承受的入门价格创造稀缺性，将最终让早期持有者为支持过你而自豪，并为未来的投放创造更多需求。

如何围绕你的 NFT 建立社区

NFT 的市场营销与社区建设相辅相成。记住，即使你已经建立了社区并取得了一定程度的成功，也需要对你的社区进行持续建设。这事没有终点。一旦开门纳客，你就需要不断为留住他们而努力。这意味着不断地传达你的价值。

将粉丝们聚集到一处好方法是拥有自己的 Discord 或 Twitter 群组。然而，也有很多成熟的空间可供你推广自己的音乐，在 NFT 社区中打响名声。

音乐 NFT 平台

上一节中列出的大多数 NFT 平台都建立起了自己的繁荣社区。通过平台提交表格后，便可以加入他们的 Discord 并向他们介绍自己。许多在这些平台上发行作品的艺人也参与到了他们的社区中。多跟那些积极开发工具以帮助音乐人在 NFT 领域取得成功的人们接触，可以学到很多东西。

Friends With Benefits

Friends With Benefits（FWB）由创作者和意见领袖组成，他们使用 web3 工具来建立社区和培养创意代理人。该组织主要活跃在 Discord 和 Twitter 上，以 $FWB 代币为中心。但并不是任何人都可以加入，你需要购买特定数量的 $FWB 才行。一旦加入他们，你就能访问FWB 的各种线上和线下活动，研讨会，以及更多有关 web3 音乐和文化的活动。

SongCamp

SongCamp 是一个 web3 音乐社区和 NFT 实验。该组织举办为期一个月的营地活动，基本上一半是歌曲创作，一半是 web3 黑客马拉松。这些营地活动汇集了来自世界各地的新生艺人，词曲作者，制作人和黑客，共同创作音乐作品和视觉艺术，最后还会推出 NFT。SongCamp 已经运营了将近一年，成为 web3 和 NFT 音乐的大熔炉。这是个建立人脉并了解其他艺术家在 NFT 领域中的动向的好地方。

Twitter Spaces

在过去的一年里，推特空间已经成为 NFT 社区事实上的市政厅。许多 NFT 网红，艺人和开发者定期主持 Twitter Space，讨论 NFT 生态中正在发生的事情，聚焦新鲜有趣的艺术家和项目。有很多这样的例子，比如 NFT NOW 的＂绿屋子＂，专门针对音乐、Web3 和 NFT 的交叉领域。绝对值得花一些时间研究这里的活跃个人和组织。

驾驭 NFT 中的版权问题

在任何创意领域，版权问题都是一个不可避免的话题，尤其是像 NFT 这种新颖而晦涩难懂的领域。一些出版物对 NFT 中版权问题的影响进行了广泛的概述。但是，一切都太新了，因此对法庭来说，司法指导并不太多。

但是，我们确实知道一些重要事项，你应当加以了解以便自我保护。

NFT 的所有权归谁?

NFT 所有权的历史被记录在区块链上，可以通过区块浏览器（如 Etherscan ) 访问。根据区块链记录，我们能够分辨出谁创建了 NFT，它此后在何时何地出现过，以及在它的整个历史中通常的售价是多少。

对 NFT 来说，这个交易账簿就可以用来证明某个 NFT 属于你。任何人都能够找到你用来铸造某个 NFT 的地址，并且从理论上讲，能够根据你在社交帐户和各种其他 NFT 帐户上提供的信息来查询它。

某个人拥有你的某个 NFT，并不意味着他对你的歌曲有任何商业所有权（母带，出版，机械等）。这是肯定的，除非你在发布时做出了明确声明。否则，你将保留该歌曲的全部所有权。

如果 NFT 被盗怎么办？

自互联网诞生以来，坏人们一直处心积虑从艺术家的画作，音乐和著作中渔利。这些窃贼会公然在他们的网站、个人资料、博客等上发布实际上并不拥有的内容，以吸引观众并获得广告收入──当然，在 NFT 领域也不例外。

艺术家的内容受国际版权法的保护，这意味着当他们的作品由无权发布的人发布时，他们拥有法律追索权。这就是1998年通过《数字千年版权法》（DMCA）以保护数字内容的确切原因。

通过提交删稿请求，艺术家可以从网站、社交媒体或应用程序中移除他们的作品。NFT 市场也是如此。这只是假设艺术家自己遇到了假冒作品，然而盗贼在原创作者不知情的情况下剽窃和转售艺术品的情况并不少见。

删稿通知的另一个主要麻烦是，通知必须发给正确的人。我们并不总是能弄清楚某个网站或市场的所有者到底是谁或哪个组织。向错误的人或地方提出申请是无效的，欺诈性的内容就会继续存在。

此外，艺术家需要挨个儿向每个发布作品的地方提交删稿通知。如果只是有人剽窃了某一个NFT，操作起来还是简单的；但是当多个店面或作品系列都使用了被盗内容时，情况就不那么简单了。

尽管如此，DMCA 删稿仍然是防止被盗 NFT 和艺术品被转售的最佳方法之一。

音乐 NFT 的创新者──学习的榜样

当然，本文开头分享的图表对构成 NFT 生态的诸多独特艺术家，社区，平台等进行了广泛而全面的概述。单就艺术家而言，有一些已经真正在 NFT 领域为自己开辟了一片天地。

这些人值得关注。每个 NFT 都会在区块链上留下痕迹，你甚至可以顺藤摸瓜，回顾这些艺术家的历史，看看他们的故事是从哪里开始的，看看他们的作品和售价随时间演变的轨迹。

Latashá

Latashá 是一名在 NFT 领域另辟蹊径的艺术家，既 大卖特卖 又提高了音乐 NFT 社区的地位，可谓一举两得。作为一个独立的创意人和 Zora 的社区编程主管，拉塔莎不仅有助于人们对 Web3 音乐艺术家的认识，而且还为新兴艺术家保有其知识产权开创了新的途径。

Ibn Inglor

Ibn Inglor 是新的 Web3 音乐商业模式的领跑者。与超级明星 3LAU 对 Royal 的做法类似，Inglor 招募粉丝群体和社区来帮助他的音乐项目进行众筹。作为回报，他的社区可以通过版税分成的方式分享财富。在＂DANGER ZONE＂上，Inglor 通过销售 $DANGER 代币筹集了 92,000 美元，这些买家现在可以分享机械灌录发行版税中高达 40% 的份额。Inglor 将这种独立模式描述为＂一种将音乐掌握在社区和我自己手中的方式"。

Heno

Heno 是第一代埃塞俄比亚/厄立特里亚裔美国艺术家和音乐家，他的作品侧重于反映黑人意识。Heno 将他与执法部门不愉快的经历转化为一张 14 轨的专辑，名为＂I’m Tired Of Being Hypersurveilled＂，这张专辑正在发起 20ETH 的众筹。为了更好地了解广泛的音乐 + NFT 生态系统在过去一年中的发展情况，请阅读我们的 2021 年 21 大音乐 NFT 时刻。

Daniel Allan

Daniel Allan 已经成为音乐 NFT 市场上最响亮的名字之一。这位制作人兼艺术家同时也是一个普通的 Web3 爱好者。他首先为自己的迷你专辑＂Overstimulated＂众筹 了近 50 ETH（14万美元），又通过 Catalog 和 Sound 等音乐 NFT 策展平台销售一系列令人印象深刻的作品，在 NFT 生态系统中开辟了一片独特的天地。到目前为止，他的努力无疑激励了更多的人参与到 NFT 音乐领域，甚至登上了《时代》杂志的专题。

余音绕梁

音乐 NFT 仍在广泛开发之中。在 NFT 的市场大潮中，音乐家和艺术家仍然不得不埋头工作，努力在 NFT 领域赢得成功。但随着时间的推移，这似乎正变得越来越容易。

随着更多的成熟艺术家进入 Web3 和 NFT 领域，我们无疑将看到音乐 NFT 生态的指数级增长。然而，与对小艺人不屑一顾的传统音乐产业相比，中产阶级似乎真的出现在了音乐 NFT 领域。

摆在眼前的是一个巨大的机会，让我们可以把权力还给所喜爱的音乐创作者。借由 NFT，我们能够为繁荣昌盛创造一个更加公平的模式。但它不会自己产生，还得依靠那些勇于迈出脚步尝试新事物的文化创造者。如果你正好身处这个领域，希望本文可以对你有所激励。
