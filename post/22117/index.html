近期，Gavin 在 Github 上发布了启动 Kusama 平行链的步骤和准备事项，Kusama 的插槽竞拍已经到了最后筹备阶段 。Bifrost 作为波卡生态底层的 DeFi 协议，对此次 Kusama 插槽竞拍，做了充分的准备，引弦待发。

同时，Bifrost 已提交 SALP (Slot Auction Liquidity Protocol) 的 Grant 申请文档，借助 SALP 协议，Bifrost 将通过生成衍生品 vsToken（vsDOT/vsKSM） 和 vsBond 的方式来释放插槽竞拍中锁定的 DOT/KSM 的流动性。SALP 通过波卡跨链协议 XCMP 完全去中心化的创建衍生品，让用户可以安全 、无信任壁垒 、释放流动性 的情况下，参与 Crowdloan，以更低的机会成本，支持自己喜欢的平行链项目，对于平行链项目方而言，SALP 也可以作为竞拍策略中的特色功能，提高平行链插槽竞拍成功率。

随着 Kusama 插槽拍卖的临近，Bifrost 的 SLP 业务 和 SALP 业务一步步完成探索，进入落地阶段。除此之外，通过在 Rococo 测试网模拟 Crowdloan 竞拍并成功上线测试平行链，Bifrost 已经做好接入 Kusama 中继链 前的充分测试，正式平行链插槽竞拍的进程现在开启。

铸造 vsKSM 参与竞拍

在本次 Bifrost Kusama 插槽竞拍中，将首次启动 vsKSM 的铸造，其中参与 vsKSM 铸造的 KSM 会用于 Bifrost Kusama 平行链插槽竞拍，用户将获得释放流动性的vsKSM 与丰厚的 BNC 插槽竞拍奖励 ，目前 vsKSM Mint Drop 页面已经上线，你可以通过访问 https://ksm.vtoken.io 来提前进行了解，同时，也可以通过邮箱订阅 ，第一时间获取 Bifrost 平行链插槽竞拍的最新消息。

vsKSM 及 BNC 奖励实时展示

vsKSM Mint Drop 将主要分为两个阶段：预约阶段 和正式阶段 。

预约阶段

预约阶段将于5月14日 19 点 正式开启，将持续至 Kusama 卡槽正式开始竞拍之前，您可在此期间通过填写他人邀请码并完成 0.01 个 KSM 的铸币，获得优先铸币权 和个人专属邀请码 。

  • 优先铸币权：可在正式铸币开启前，提前 24 小时开始铸币
  • 个人专属邀请码：随时邀请其他好友建立绑定关系，好友参与铸币，您赚取邀请奖励

正式阶段

将于预约阶段结束后开启，持续 4-8 周 ，在此期间将先后开启优先铸币与正式铸币，同时获得多重 BNC 奖励，该阶段可邀请他人但不再生成新的邀请码。所有参与 vsKSM Mint Drop 铸币的 KSM 都将用于参与 Bifrost Kusama 插槽竞拍 ，整个过程透明公开，所有参与记录将公示在铸币界面 中。

铸币奖励

正式阶段，铸币通道开启，您将可以通过 Bifrost 官方网站的活动页面参与铸币，获得所投 1 KSM : X BNC 奖励（X 将在正式阶段开始时公布，比例至少大于 1 KSM : 10 BNC），其中包含:

1. 即时奖励： 铸币即获得，无关插槽竞拍成功与否
2. 邀请奖励： 填写正确邀请码，获得即时奖励的 5% 加成
3. 成功奖励： 仅在插槽竞拍成功后发放，成功后立即解锁 50%，其余部分将在 336 天内（48周插槽租赁期）线性解锁

vsKSM Mint Drop 奖励构成

由于活动页面的参与入口内置了 SALP 协议，KSM 铸币即获得等量的衍生品 vsKSM 与 vsBond ，用于释放插槽竞拍流动性，同时也将获得相应的 BNC 作为奖励。BNC、vsKSM 及 vsBOND 的交易与兑换功能将在 Bifrost 竞拍成功后解锁 。

邀请奖励

当有人通过您的邀请码 参与铸币时，您将获得邀请奖励 。

您将获得对方最终结算奖励的 5% 的额外奖励 ，可在活动页面中查询被邀请人地址、铸币量和时间信息。通过扩散自己的邀请码，让更多的朋友使用。一般情况下，预约阶段提前建立的绑定关系越多 ，获得的 BNC 奖励提成越丰厚 ，同时作为被邀请人，也可以额外获得 5% 的即时奖励。

需要注意的是：

所有奖励将在铸币过程中实时统计与记录，待竞拍结果公布后统一发送至你的铸币地址 。

在预约阶段，要获取邀请码，必须先填写他人的邀请码，一旦预约成功则邀请码不可更改。Bifrost 将首先通过合作伙伴，社区，KOL 发放首批邀请码 。正式铸币阶段则不同，用户在参与 Bifrost vsKSM 铸币时，邀请码是选填项。

如何参与预约活动？

Bifrost 将会联合媒体、合作方及 KOL 联合发放首批创世邀请码。创世邀请码将在预约活动正式开启时公布 ，但请注意在填写邀请码之前需要通过 Polkadot js 插件登录个人的 DOT/KSM 地址 ，每个地址仅可以填写一个邀请码。活动入口：https://ksm.vtoken.io

操作步骤如下：

1. 填写邀请码

2. 铸造 0.01 个 KSM

3. 点击预约

4. 预约成功

小结

SALP 协议 是 Bifrost 的一大创新，解决了很多伙伴想参与 Crowdloan，又担心损失机会成本的问题。此次 vsKSM Mint Drop 把 KSM 插槽竞拍 和 SALP 相结合，既能保留 Bifrost 插槽竞拍奖励 ，又能释放 KSM 流动性 ，随着插槽成功竞拍和 Bifrost 平行链成功上线，将掀起新一轮的行业浪潮。

Bifrost 是什么？

Bifrost（彩虹桥）是波卡生态 DeFi 基础协议 ，致力成为质押资产 提供流动性的基础设施，推出面向 Staking 和波卡平行链卡槽（Crowdloan）的衍生品 vToken 。目前已获得 NGC、SNZ、DFG、CMS 等机构数百万美金融资 和 Web3 基金会 Grant，同时也是 Substrate Builders Program 、Web3 Bootcamp 成员。

vToken 可在 DeFi、DApp、DEX、CEX 等多场景下优化交易，通过 vToken 实现 Staking、Crowdloan 等质押权益的转让通道，实现质押资产风险对冲，扩充场景如 vToken 作为抵押物进行借贷时，其 Staking 收益可抵销部分利息，实现低息借贷 。
