2021年在很大程度上是由Layer-1的崛起决定的。继以太坊在2020年夏天崛起后，Solana、Terra、Avalanche和Fantom等区块链在网络活动和其原生代币的价格方面都出现了指数级增长。

这些区块链的迅速崛起表明了两件事：

  1. 对于那些探索和采用新兴网络的富有冒险精神的用户来说，存在着巨大的机会。

  2. 单一的、单片的区块链存在着固有的重大技术限制。

进入Cosmos

Cosmos能够创建高度可定制的区块链，无论它们是通用的，还是特定的应用。Cosmos旨在通过所谓的区块链间通信（IBC）来连接这些网络，允许在区块链间进行信任最小化、且近乎即时的价值和数据转移。这种模块化设计有可能解锁一个新的可互操作的跨链应用设计空间，同时提供一条持续扩展的潜在路径。

尽管Cosmos的许多潜在变革性技术还处于起步阶段，但该生态系统已经开始看到了有意义的牵引力，并在协议和应用层带来了许多创新。

而随着创新带来的机会–由于许多人可能熟悉Cosmos链，如Terra和ThorChain，让我们深入了解一下该生态系统中的其他五个投资者应该密切关注的代币。

Cosmos Hub (ATOM)

  • 代币类型：L0/L1

  • 市值：82亿美元

  • FDV: n/a (永续通胀)

  • 可冲指数（Apeability）: 5/5

协议概述

尽管它是Cosmos的旗舰代币，但由于ATOM在生态系统中不断发展，它也值得仔细研究。作为有史以来的第一条Cosmos链，Cosmos Hub作为基础设施发挥着重要作用，作为IBC传输的终端，并将在未来几个月通过提供链间安全（Interchain Security）看到进一步的效用。

特定应用链面临的一个核心挑战是需要启动他们自己的验证器集。与rollups不同，Cosmos链不从父链继承安全性。相反，每条链都负责自我保护，需要独立的源验证器，并确保他们的原生代币维持价值。虽然允许更大程度的主权和可定制性，但相对于以太坊等共享安全的Layer-1，它可能会导致特定链的安全性急剧下降。虽然许多用例的安全需求不同，但有些用例，如DeFi，需要一个高度强大和去中心化的验证器，以抵御潜在攻击。

Cosmos Hub旨在通过推出Interchain Security来发挥信标链的作用。这将使Cosmos链能够将其安全需求外包给Cosmos Hub，从其超过52亿美元的ATOM质押中获益。Interchain Security可以为生态系统带来更多的高价值用例，预计将在2022年第三季度开始推出。

代币经济学

虽然可以说Cosmos Hub是一个"0层"而不是1层，但ATOM代币在实践中的作用与传统的L1代币类似。ATOM被用来保证Hub的安全，持有者有资格获得通胀奖励以及IBC转账的交易费。

相对于其他L1代币，ATOM的一个优势是它作为 “空投磁铁 “的作用。Cosmos生态系统中的许多网络和项目，如Osmosis、Juno和Shade Protocol，已经或计划将其代币供应的一部分空投给ATOM持有者。

不太清楚的是，ATOM是否会在不同程度上获得其他L1代币所持有的货币溢价。例如，ATOM是上述Osmosis DEX上第二大流动性代币，仅次于OSMO，这表明它可能在各种链上扮演第二位的原生代币。

要点

虽然它的市值已经超过80亿美元，而且作为一项资产，它在Cosmos DeFi用户中的受欢迎程度还不清楚，但ATOM可能是投资者接触Cosmos生态系统最安全、最广泛的方式。该代币准备从Interchain Security的推出所带来的效用和需求的增加中获益，而且至少在不久的将来，有可能通过空投为其持有人产生额外的现金流，从而进一步增加其吸引力。

Osmosis (OSMO)

  • 代币类型：L1/DeFi - DEX

  • 市值：28亿美元

  • FDV: 8.5亿美元

  • 可冲性: 5/5

协议概述

Osmosis是一个去中心化交易所和特定应用链，拥有超过16.6亿美元的TVL，最近几周促成了7-17亿美元的交易量。与Terra的UST稳定币类似，用户和应用能够利用网络建立在AMM之上的能力。

来源:info.Osmosis.zone

Osmosis与Balancer最为相似，因为它允许创建高度可定制的多资产池，但也融合了其他AMM的元素，同时也有自己的独特功能。例如，每个流动性池都是"自我管理的”，流动性提供者对其参数有管理权。像Curve一样，LPs必须锁定他们的代币以参与治理，另外，流动性挖矿奖励与锁定期的长度成正比。

代币经济学

OSMO代币在Osmosis生态系统中扮演着几个关键角色，具有Layer-1和DeFi代币的特征。

像ETH、AVAX、SOL和其他L1代币一样，用户通过质押OSMO从Gas支付中获得通胀奖励和交易费用，以换取网络安全。此外，与后两种代币一样，OSMO也被用来对全网治理提案进行投票。

此外，OSMO与DeFi代币也有相似之处，它也被用于AMM级治理。例如，该代币用于确定所有Osmosis池的基本交换费，并且像Curve一样，用于投票决定对不同的OSMO计价器的奖励。

OSMO也被用于所谓的"Superfluid Staking”（超流质押），这将允许OSMO在被质押的同时，作为AMM内的流动性被利用。

更多关于超流质押的信息，请点击这里。

要点

Osmosis既是一个非常独特的区块链，也是AMM。OSMO受益于所有加密货币中最独特的代币设计之一，其实用性和需求驱动来自于其作为L1代币、AMM治理代币和超流质押。

Juno (JUNO)

  • 代币类型：L1

  • 市值：14亿美元

  • FDV: 56亿美元

  • 可冲性：2/5

协议概述

Juno是一个通用的智能合约平台。与Cosmos生态系统中的许多其他链不同，Juno允许使用CosmWasm（一种使用Rust作为编程语言的虚拟机）构建的智能合约的无权限部署。JunoSwap是该网络最受欢迎的应用，持有超过4200万美元的TVL，而在未来，该链将利用IBC来实现真正的跨链应用。

来源：Juno Documentation

Juno最近正处于一个极具争议性的治理建议中心。作为其代币分配的一部分，JUNO将其47%的供应量分配给ATOM质押者。虽然有"鲸鱼上限"，以防止交易所等实体接收JUNO，但有一只鲸鱼设法欺骗了空投，积累了250万JUNO，按当前价格计算，价值约7500万美元。这占代币初始流通供应量的7.5%，占团队12年归属期结束时预期供应量的1.3%。

经过激烈的讨论，在一个被称为 “提案16 “的极具争议的提案中，社区通过链上治理投票决定将鲸鱼的余额削减到50,000 JUNO，即初始空投期间每个地址的最大分配量。

译者注：3月16日凌晨，uno Network关于移除巨鲸账户资产的16号提案获得通过，其中40.85%的投票者同意该提案。

代币经济学

与ATOM、OSMO和其他PoS网络一样，JUNO被用于网络安全，而质押者能够获得通胀奖励和交易费用。与前两种代币一样，JUNO被用作网络上的Gas，并显示出成为该链储备资产的迹象，因为它是JUNoSwap上最受欢迎的基础对资产。

要点

提案16的结果对Juno的可信中立性构成了巨大的风险，可能会阻碍开发者的兴趣和资本流入生态系统。然而，如果社区团结一致，如果网络硬分叉，那么作为第一批去权限、通用的CosmWasm L1之一，JUNO可能对逆向投资者来说是很有吸引力的。

Secret Network (SCRT)

  • 代币类型：L1

  • 市值：8.05亿美元

  • FDV: n/a (永续通胀)

  • 可冲性: 3/5

协议概述

Secret Network是一个通用的智能合约平台，它（不辜负它的名字）允许创建面向隐私的应用程序，被称为Secret Apps。Secret Apps可以使用能够加密用户数据的 “Secret合约 “来构建。由于强调隐私，Secret有可能在Cosmos生态系统中发挥 “隐私链 “的宝贵作用。通过对整个第一层的交易数据进行加密，该网络结合了Zcash和Monero等隐私币以及Tornado Cash等应用程序的优点。

图源：Secret Analytics

由于部署在网络上的应用数量有限，Secret只看到了最早的牵引迹象。例如，SecretSwap是该网络最大的AMM，但每天仅有100-300万美元的交易量，仅持有3910万美元的TVL。此外，尽管Secret也强调了与其他网络的互操作性，但从以太坊、BNB Chain、Monero以及通过IBC桥接的资金只有4620万美元。

代币经济学

与Juno类似，SCRT是一种L1代币，用于保障网络安全，支付Gas，并用于协议层面的治理。然而，如上所述，SCRT可以有多种“角色”，也可以作为一种私人货币，具有与ETH等L1代币类似的广泛效用。

要点

作为隐私公链，Secret Network和SCRT有可能在Cosmos生态系统和整个加密领域中提供一个非常有价值且重要的利基。然而，鉴于迄今为止，该网络看到的牵引力有限，SCRT似乎最适合具有长期视野的投资者。

5. Astroport (ASTRO)

  • 代币类型：DeFi - DEX

  • 市值：2.74亿美元

  • FDV：21亿美元

  • 可冲性：4/5

协议概述

Astroport是Terra上最大的去中心化交易所，拥有超过13亿美元的TVL，交易量持续在1亿美元以上。通过一个独特的三阶段 “锁定”，结合了空投和吸血鬼攻击的元素，DEX作为Uniswap和Curve的组合，Astroport促进了波动性交易对之间的交换，以及其他类似资产。

该协议善用其所谓的 “生成器”（generators），其功能类似于Curve和Frax计价器，通过治理投票获得释放分配的交易对。生成器也能够自动将LP代币存入第三方协议，这些协议目前正在提供激励措施，以便通过向用户提供双重挖矿奖励来提高资本效率。

代币经济学

ASTRO采用了两种代币模式。第一种是流动性资产xASTRO，持有者，即ASTRO质押者，将获得代币并有权获得治理权以及平台产生的交易费份额。

第二种，vxASTRO，是一种不可转让的代币，与veCRV最为相似。ASTRO持有者可以锁定他们的代币以获得vxASTRO，这使他们有权获得相对于xASTRO的更大份额的治理权和交易费，以及在平台上提供流动性时获得更多奖励。

要点

作为Cosmos生态系统中最大区块链上最大的DEX治理代币，ASTRO似乎准备成为Terra增长的主要受益者。此外，由于它有能力引导交易所内各交易对的流动性，该代币可能在Terra原生协议中需求量很大，或许会掀起"ASTRO战争”。

结语

虽然它们在现今的可操作性方面各有不同，但在Cosmos生态系统中，有许多引人入胜的链、协议和代币值得关注。对于加密原住民而言，虽然它们中的很多都处于采用周期的早期阶段，但它们在网络、应用和代币设计方面带来了令人兴奋的创新。

它们能把投资者带到Cosmos中去吗？
