我最近有机会见到了一些考虑进入 web3 的优秀 web2 创始人。

Web3 现在很流行，因此重要的是要消除噪音并突出强调正确的长期理由来启动 web3 项目。

以下是构建 web3 项目的一些充分理由：

产品

维基百科只是比传统百科全书更好的百科全书，因为它利用了社区的知识和创造力。

DAO 是一种新颖的 web3 结构，通过提供更复杂的协调方法和对互联网原生金融资源的控制来增强在线社区。

在过去的一年中，数千个 DAO 被创建，涵盖广泛的类别，包括金融服务、投资、内容创作、游戏、社交俱乐部等。

也就是说，我们仍处于探索 DAO 设计空间的早期阶段。可能会有机会将 DAO 应用于新领域并创造突破性产品。

经济学

Web3 通过去除中介层并显着降低收取率（中介收取的费用）改变了互联网服务的经济性。

这为初创公司提供了一个机会，通过为创造者、开发者和其他网络参与者提供卓越的经济性来吸引和培育社区。相比之下，例如，Instagram 和 Twitter 等社交网络不会与让它们受欢迎的创作者分享任何收入。

在 web3 中，创作者可以通过销售 NFT 之类的东西直接接触到他们的受众，从而获得他们所产生收入的 90% 或更多。

Web2 社交网络是通过说服人们免费赠送他们的创意作品而建立的。

一旦创作者意识到有可靠的替代方案，这将停止工作。

NFT 的整体销售额平均每月达到数十亿美元。

按照目前的趋势，web3 创作者收入将很快超过 web2 创作者收入。

营销

Web3 解锁了强大的新营销技术。

首先，web3 使项目能够在原生网络效应较弱的早期阶段向用户提供代币激励。

这有助于克服过去曾击败过许多有前途的早期初创公司的引导问题。

Helium 就是一个很好的例子，它使用代币激励措施建立了一个国家级的草根电信服务，以应对电信老牌企业。

其次，在 web3 中，代币是自我营销的。 当某人真正拥有某样东西并在游戏中切身感受之后，他们想要宣传它。

web3 不是在现有平台之上构建您的服务或使用付费广告，而是让您通过用户的热情来构建它。

这极大地降低了客户获取成本，并让您建立更高质量的社区。

生态系统

世界上运行的绝大多数软件都是开源的，由开发人员社区创建。

这是因为具有相关专业知识的聪明人无处不在。

正如比尔·乔伊的名言：“无论你是谁，大多数最聪明的人都会为别人工作。”

Web3 项目可以通过创建和培育开发者生态系统来挖掘更大的人才库。

开发人员想知道他们正在构建一个规则不会改变的可信的中立平台。

HTTP 和 SMTP 等 Web1 协议在这方面做得很好，导致了创新的爆炸式增长。

Web2 试图通过承诺（“不作恶”）向开发人员做出承诺。

正如我们多年来看到的那样，这不起作用。 Web2 平台通常会弃用 API 并更改服务条款，而很少考虑第三方开发人员。

在 web3 中，对开发人员的承诺通过不可变的开源代码（“不可能是邪恶的”）嵌入到系统中。

这意味着开发人员可以依靠可靠的中立性，就像他们在 web1 中可以做到的那样。

这将导致新一波广泛的、无需许可的创新。

精神

最后，也是最重要的一点是 web3 精神。

互联网越来越被少数几家现有的科技公司主导。

web3 将我们带回到互联网最初的愿景：一个由开放协议管理的去中心化网络，其中权力和财富流向边缘而不是中心化中介。

作者：A16z合伙人Chris Dixon
