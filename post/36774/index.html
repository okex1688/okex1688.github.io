在如今这种信息高速发展的时代中，注意力的价值将会超过信息。

所谓注意力经济（Attention Economy），是将人类的注意力作为一种稀缺的资源并且运用经济学理论来进行运营。

而诺贝尔奖获得者 Herbert Alexander Simon 曾经指出：

在如今这种信息高速发展的时代中，注意力的价值将会超过信息。

而 NFT 作为 Web3 时代的信息载体，必然摆脱不了注意力经济的影响。

当然，NFT 本身的文化（包括艺术、Vibe、Meme 等）是社区的内核与灵魂，但这并不在本文要讨论的范围之内。

NFT 的注意力经济

当我们打开类似 opensea 这样的 NFT 交易平台时，我们首先观看的大概率会是 24 小时交易排行榜前 100 的 NFT 项目，这是因为我们本能上会去先关注热点，而这就是注意力经济的一种体现。

再然后，我们会尝试去找出这些热点 NFT 项目吸引到资金关注的原因，而通常的原因会有以下这些：

  1. Mint 抢购热度高，而没有 Mint 到的用户会将注意力转移到 opensea 这样的二级交易市场；

  2. 该 NFT 项目方将为 NFT 持有者空投新的衍生品；

  3. 该 NFT 项目方或社区将为 NFT 持有者空投 token;

  4. 该 NFT 项目方将为 NFT 持有者推出一款游戏；

  5. 该 NFT 项目方与某些大 IP 达成合作；

  6. 拥有巨大影响力的巨星或鲸鱼投资者成为了该 NFT 的持有者；

  7. 该 NFT 本身具有某些病毒式传播的因素（搞笑、可爱、meme 等），并因为某些原因导致了大范围的传播，等等；

而一旦注意力开始消散，或者转移到新的 NFT 项目上，我们就会看到这些 NFT 的价值会发生转移。

这也是绝大多数的 NFT 图片都会归零的根本原因。

自上而下 VS 自下而上

而在影响注意力经济的因素方面，运营的方式是「自上而下」，还是「自下而上」，会起到非常大的作用。

回顾加密历史，中本聪在吸引到足够多的加密朋克开发者和拥护者之后，其才选择了放权，时至今日，比特币已经发展成为「自下而上」发展的最佳案例，这个去中心化的系统拥有十分强大的反脆弱性。

但之后绝大部分采用「自下而上」发展方式的加密项目，它们都消失在了人们的视线当中，例子有 BTS、Grin 等等。

根据笔者目前观察到的情况来看，采用「自上而下」运营方式的 NFT 项目在初期阶段会拥有巨大的优势，以目前影响力最大的 “ape 宇宙” 运营方 Yuga Labs 为例，他们可以通过融资去做各种各样的事（例如制作游戏、空投 Apecoin、收购 Punk、推出影视作品，与大 IP 达成合作等等），以此吸引更多的注意力。

这就是无聊猿在短短一年的时间里逆袭 punk，并成功登顶 NFT 龙头宝座的最大原因。

然后简单谈谈采用「自下而上」运营方式的 Loot，这个曾一度被寄予厚望的链上元宇宙种子的 NFT 项目，在其迎来短期注意力的巅峰期之后，开始了漫长的挤泡沫过程。

Loot 的创始人 Dom Hofmann 在发布完这个作品后就退出了项目引导人的角色，而是将其交给了社区，让社区来决定这个 IP 的未来。

但我们看到的是，大量低质量的项目耗散了这个 IP 在前期吸引到的注意力，而尽管最近类似 hyperloot 和 Realms 这样的 Lootverse 生态项目有了较大的改善，但它们目前表现出来的还仅仅是头像与地图，因此吸引到的注意力也是非常的有限。

举这两个例子，一是再次强调注意力经济的重要性，二是对比中心化运营和去中心化运营的优势与劣势。

当然，采用「自下而上」构建方式的 NFT 也有自己的优势，那就是去中心化系统具有的反脆弱性优势。

而像无聊猿这样的 NFT 蓝筹，一旦失去 Yuga labs 的支持，那整个生态的成长大概率会陷入一种停滞状态，这就是依赖的影响。

总的来说，「自上而下」的方式更适合中短期运营，而「自下而上」的方式，则更适合后期运营。

如果过早的采用放权的方式来发展一个 NFT，则需要依赖生态的参与者自己去建设，这具有非常大的不确定性。

增加短期 NTF 注意力的一种新方式：FloorDAO

除了以上谈到的影响 NFT 注意力的一些运营手段和方式之外，目前还出现了一种名为 FloorDAO 的地板 DAO 协议可以短期聚焦 NFT 注意力，其旨在通过 NFT 做市和其他策略赚取收益。

以其最新投票的结果来看，tubby cats 和 MiladyMaker 这两个“可爱风”的 NFT 头像项目成为了最新的受益者，而这两个 NFT 项目的地板价在最近几天都出现了快速飙升。

这就是注意力经济的一种运用。

当然，这些 NFT 接下来的发展，最终将取决于其能否持续吸引到更多的注意力。
