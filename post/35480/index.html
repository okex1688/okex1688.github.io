Gala Games 是如何工作的？

Gala Games 是一个区块链游戏聚合平台，游戏以及游戏内资产（NFT）发行在以太坊上。未来，Gala Games 计划推出对游戏用户和开发者更友好的游戏公链。很多人都将其理解为 Web3 世界中的 Steam 平台。

据 OKLink 数据显示，得益于 Play2Earn 模式的爆发，2021 年 GameFi 市场日活跃人数得到爆发式增长。在此期间，Gala Games 乘势而上，在 2021 年获得了长足的发展。

据 Gala Games 官方数据，其每月的活跃用户已经超过 130 万，有超过 26000 件 NFT 售出。但是随着 GameFi 市场的冷淡，在抵达新高后，GALA 代币价格在 2021 年 11 月开始逐步回落。虽然从表现来看，Gala Games 代币价格表现与 GameFi 市场强相关，但 Gala Games 的游戏 P2E 属性并不强，正如团队在官网中所写，Gala Games 对于游戏的要求更侧重「Fun First」。

他们认为：「We believe the blockchain should be invisible in our games. We use simple game mechanics that all players can enjoy, whether or not they consider themselves to be blockchain pros. 我们认为区块链在我们的游戏中应该是隐形的。我们使用简单的游戏机制，所有玩家都可以享受，无论他们是否认为自己是区块链专家。」

目前，Gala Games 已经明确将上线 8 款游戏，分别为 Legends Reborn、Spider Tanks、Town Star、The Walking Dead: Empires、Mirandus、Legacy、Last Expedition、Echoes of Empire、Fortified。这八款游戏涵盖了卡牌、策略、RPG、MMORPG、格斗、模拟经营、FPS、TD 等多种游戏模式。

游戏内容涵盖了太空、中世界、僵尸、朋克、冒险、休闲、种植等风格。其中，模拟经营类游戏 Town Star 已经正式运营，以网页作为前段供玩家参与游玩。剩余 7 个游戏处于开发、测试和等待节点投票等几个阶段。

接下来本文将以节点、GALA 代币、游戏内容、团队、竞品、风险等角度详细分析 Gala Games 的基本面和未来的发展潜力。

节点

去中心化节点网络是 Gala Games 的基础。通过 Gala Games 的创始节点，用户可以投票来决定 Gala Games 将上线哪些游戏。针对单独的游戏，Gala Games 同样设置了游戏节点，比如 Town Star，以减少 Gala Games 服务器压力。

Gala Games 创始节点恒定上限为 50000 个，据官方数据显示，目前已经有超过 16000 个由玩家运行的创始节点，共同支撑着 Gala Games 的生态发展。

对此，我们可以将其理解为 Gala Games 作为一个主体，通过分散节点来将整个游戏网络去中心化。主体 Gala Games 负责为整个去中心化网络提供运营、策展、市场营销和维护等服务，这些服务内容涵盖了 Gala Games Network、节点软件更新、维护开发等，最终与社区共同推动 Gala Games Network 的发展。

据官方解释，Gala Games 节点网络生态由五部分组成，分别是 Gala Games 主体、游戏中的 NFT、随机分配算法、公众数据库和节点。

Gala Games 主体

Gala Games 主体负责各类游戏在治理的基础上进行关于游戏层面（游戏上线等）的策划和执行。同时，Gala Games 希望为更多的工作室提供工具包以制作发布 NFT，用一个开放的生态迎接更多的游戏工作室。

据官方解释，「Gala Games 的目标是让任何公司都可以轻松快速、轻松地将他们的游戏添加到 Gala Games 生态系统，并使用工具包生成标准化的加密包、NFT 库和钱包以集成到他们的游戏中」。可以认为，游戏工作室的多寡决定 Gala Games 生态网络的未来发展，也是我们评估 Gala Games 潜力的重要指标之一。

公共数据库

Gala Games 社区已经通过 GALABASE.io 成立了一个公共数据库。我们可以将其理解为区块链浏览器。公共数据库的职能在于通过易于访问的特性，帮助开发者社区更轻松地参与 Gala Games 生态。Gala Games 生态下的游戏工作室是其开发者社区的主要构成部分。

此外，公共数据库还是其随机分配算法的支撑。我们也可以通过其来查询不同节点对于 Gala Games 生态的支持。但是经过测试，公共数据库 GALABASE 的应用主要体现在 NFT 查询上，对于 Gala 节点的贡献等信息的展现比较模糊。

创始节点和 NFT

成为 Gala 创始节点的收益主要体现在两方面：

  • 1、创始节点可进行投票决定平台上线的游戏；

  • 2、创始节点获得 Gala Games 和游戏（Town Star）的运行收入；

此外，根据随机分配算法，新游戏的 NFT 也会空投给创始节点。新游戏中的 NFT 由游戏工作室们提供，并在 Gala Games 的商店中交易。

除了投票外，虽然成为 Gala 创始节点需要提供电脑等设备，但就像上文提及的，成为 Gala 节点具体需要贡献的内容并不清晰，对外也没有具体信息的披露。并且，其数据库中的活跃节点数据更新是在 3 月前。

可能在未来更多游戏上线的情况下，Gala 创始节点才能体现出其更多的职能。

节点所有者@炼金术认为，「节点售卖是项目方变相募资手段，项目方自己说过，卖节点是为了规避美国监管风险。目前还有 1 万个节点待售，项目方能卖多少钱呢？ 差不多 7.5 亿美金」。

GALA 代币

GALA 为 Gala Games 生态系统的 ERC 原生 Utility Token，目前可跨链至 BSC，总量为 500 亿，目前流通量为 69.8 亿。目前 GALA 市值为 17 亿美金，市值排名为 63。

据悉，GALA 代币未进行 ICO，已登陆 Binance、Coinbase、Crypto.com、OKX、Kucoin、Gate.io、Uniswap 等多家交易所，涵盖了 GALA/BNB、GALA/USD、GALA/USDT、GALA/ETH 等多个交易对。

Gala Games 节点可以通过运行节点 6 小时以获得每日的积分，之后通过积分来兑换 GALA 代币奖励。积分越多，便可以在每日的 GALA 代币分配中获得更多份额。除了运行节点外，邀请用户加入节点、邀请用户节点运行 6 小时以上，邀请人也可以获得积分奖励。

每年 7 月 21 日，GALA 代币分配的数额将减半——随着时间的推移，GALA的抛压越少，但成为节点的回本周期越长。并且，Gala Games并没有进行融资——从结果来看，Gala Games 的创始节点其实便是 Gala Games 的投资人。

通过创始节点和节点销售，Gala Games 已经获得足够支撑他们持续发展的资金，据头等仓统计，截止1月11日，Gala Games 在节点销售中获得了 5.1 亿美金以及 8681.8 ETH 的收益。

除此之外，Gala Games 还拥有另外一项收入来源：黄金会员订阅。黄金会员拥有优先获取信息、获得特殊 NFT 奖励、抢先体验游戏测试机会等权利。

GALA 代币充当着 Gala Games 生态中所有事物的支付媒介，包括节点、黄金会员、商店中的 NFT 销售等事宜。这也是 GALA 代币的主要赋能。之后在 Gala Games 推出 Project GYRI（Gala 生态的公链项目）之后，GALA 代币或许会在其中充当 Gas 代币。

未来，除了在 Gala Games 生态中充当重要的交易媒介外，GALA 代币也是 Gala Games 早期投资者，也就是 Gala 创始节点收回投资成本的重要手段。

如果 GALA 代币价格未能达到预期，新晋 Gala 节点收回成本的周期变长，潜在投资者成为节点的欲望变弱，Gala Games 也无法获得充足的资金进行积年累月的长期运营。当然，截止目前为止，Gala Games 的资金依然充裕。

2 月 8 日，Gala Games 证实，将在未来一年内拨款 50 亿美元拓展其 NFT 等产品。其中预计 2 亿美元将用于游戏 NFT，10 亿美元将用于工作室的音乐 NFT，10 亿美元用于电影 NFT，Gala 主题公园 NFT 将获得剩余的 10 亿美元。

节点数量的增长也将成为 Gala Games 重要的 KPI 指标，只有通过不断推动 Gala Games 自有生态的发展，提升投资者对于 GALA 代币的价值预期，才能够吸引更多的节点和玩家参与。这是一个正反馈循环。

游戏

Town Star

如今 Gala Games 唯一上线的游戏便是 Town Star（Spider Tanks 还在测试阶段）。Town Star 是一款搭建在网页端的 Play2Earn 区块链游戏。

它是一款模拟经营类游戏，玩家可以通过在游戏中建设农场、牧场、伐木场、工厂等一系列基础设施，雇佣工人、工厂加工、贩卖资源来建设一个动态发展的城镇体系。虽然 Town Star 是一个搭建在网页的游戏，但是其游戏机制非常全面，且玩家可以免费游玩，没有门槛。

对于基础的 GameFi 玩家而言，Town Star 难度很高，有一定的上手门槛。虽没有传统模拟经营游戏那样复杂，但是玩家也需要花费精力和时间熟悉游戏流程。

首先玩家需要了解工人、材料、基础设施的作用和花费，平衡发展，直到各类资源可以支撑炼油厂的加工需求。其中，汽油是游戏中最重要的资源，我们需要汽油来运行汽车贩卖自己的商品以获取利润。游戏无法购买，只能通过炼油厂获得汽油。一旦汽油不足，我们又无法生产汽油，那游戏便进入了死亡循环——我们只能再开一局。

游戏中的 Play2Earn 模式并非传统 GameFi 市场中的模式，即 DeFi 游戏化。传统 GameFi 是通过 NFT 和 FT（同质化代币）的结合，为参与者们带来一些游戏体验，但本质还是 DeFi。而 Town Star 践行了 Gala Games「Fun First」的初衷，更重视玩家的游玩体验——即使玩家无法获得快速的经济效益。

Town Star 的 Play2Earn 模式主要体现在以下三个方面：

  • 每日挑战：完成每日挑战即可获得代币奖励；

  • Gala Power：Gala Power 的设置鼓励玩家持有 GALA 代币和 TOWN 代币——两者数量决定了玩家Gala Power的等级。Gala Power等级越高，玩家在Town Coin中能够获得的NFT越多；

  • 季票：通过季票 NFT，玩家可以在规定时间内完成比赛以获得 TOWN 代币奖励。

当然，Town Star 也提供了付费内容（NFT）以帮助玩家掌握更高的赚钱效率。同时这些付费内容（NFT）也被支持在市场中随意交易。玩家可以在 Gala Games 商店中购买那些 NFT，NFT 越稀有，赚取 TOWN 的效率也就越高。

而玩家通过游玩游戏或者在第三方交易所购买的 TOWN 代币拥有以下三种用途：

  • 购买 TownCoin 的独家 NFT 物品；

  • 使用 TownCoin 进行游戏内的升级和福利；

  • 持有 TownCoin 可以获得更高的 Gala Power 等级。 

除了通过 Play2Earn 赚取 TOWN 代币外，TOWN 代币的另外一种获取方式是成为 Town Star 的运营节点。节点运营商在每日分发中收到 TOWN 代币作为运营其 Town Star 创始人节点的奖励。

其实我们可以从 Town Star 的例子中看到 Gala Games 对于生态游戏的态度。就像我们在上文中多次提及的「Fun First」，Gala Games 生态中的游戏更有可能向「普通玩家享受游戏，专业玩家赚取赏金」的模式发展，更像是 Contest2Earn，而非 Play2Earn。

这也是上文中将 Gala Games 认定为 Web2.5 产品的另一个原因。一款好玩的游戏受众将不止于加密用户，它同样会吸引更多的普通玩家参与进来，更像是 Web2 与 Web3 之间的桥梁。

同时，非加密行业的玩家可以通过 Gala Games 的邮箱注册来创建自己的加密钱包（仍需记录助记词），不需要下载 Metamask 等钱包软件。这在一定程度上降低了玩家对于区块链/加密世界的认知门槛。

Gala Games 生态 NFT VOX

VOX NFT 是一个基于 Gala Games 生态的 NFT，首发共计 8888 枚。 VOX NFT 可以被应用于 Gala Games 生态中的游戏内，比如它可以被质押在 Town Star，以获得 TOWN 代币奖励。

目前，灵感来自 MMORPG 的 MirandusVOX 第二系列 VOX Collectibles: Mirandus 已经推出。

未来，在 Gala Games Metaverse 推出后，它将在其中充当 Avatar。并且，Metaverse 参与者还可以通过 VOX NFT 赚取 VOX Coin。VOX Coin 将被应用于 Metaverse 中，或兑换成其他 Cryptocurrency。

创始人和团队

在 Gala Games Medium 的介绍中提到，Gala Games 由前 Zynga 联合创始人 Eric Schiermeyer、加密货币领域的首批矿工以及区块链技术的多项专利持有人 Wright Thurston 和 Farmville 2 等风靡一时的游戏背后的创意总监 Michael McCarthy 创立。

其中，Eric Schiermeyer 担任 Gala Games 首席执行官。他在创立 Gala Games 之前是 Zynga 联合创始人。作为游戏公司，Zynga 推出了许多脍炙人口的网页/手机游戏，比如 Farmville、Farmville 2、Zynga Poker。

为方便理解，我们可以将 Farmville 理解为在国内风靡一时的 QQ 农场，只不过 Farmville 是基于 Facebook 社交网络建立的。Farmville2 是 1 代的 3D 升级版。而 Zynga Poker 是一个以德扑作为游戏内容的手机游戏。

Eric Schiermeyer 没有大型游戏的开发经验，从结果上来看，Farmville 等游戏所展现的更多是其在营销层面的能力。不过区块链游戏尚处于初级发展阶段的时期，3A 游戏大作的水土不服将给予轻量级游戏更多的发展空间。

除了主要创始人外，游戏部总监 John Osvald 曾经同样是在 Zynga 工作，从事游戏设计工作。他的从业经历基本都集中在游戏领域，曾任职于 EA、Carnivore Inc.等公司，经验主要集中在移动游戏板块。

T. Elliot Cannon 担任过 Gala Games 第一个游戏 Town Star 总监，拥有 27 年的游戏从业经历，曾参与设计过虚幻引擎、Unreal Tournament（虚幻竞技场）、孤岛危机、Doom 等游戏大作。在今年 2 月，T. Elliot Cannon 已经开始筹备下一个 Gala Games 即将推出的游戏。

竞品分析

作为一个游戏聚合平台，Gala Games 同样拥有很多的竞争对手，比如开启了 GameFi Play2Earn 时代的 Axie Infinity，抑或是致力于建设 NFT Layer2 的 Immutable X。

与 Axie Infinity 的对比

为解决以太坊可扩展性、费用高昂的问题，Axie Infinity 搭建了一条以太坊的侧链 Ronin 以供游戏玩家使用。除了解决以太坊本身的痛点外，Ronin 还集成了支付网关以方便玩家出入金。可以认为，Ronin 是 Axie Infinity 成功的基础 。

随着 Play2Earn 的乏力，Axie Infinity 的发展重心更倾向于侧链 Ronin 的建设，而非对于更多游戏内容/机制的探索。随着 Ronin 上的去中心化交易所 Katana 的推出，投资者的注意力也从 GameFi 转移到了 DeFi，Katana 为 SLP（Axie Infinity 为玩家提供的奖励通证）提供了新的赋能——只要将 SLP/ETH 组成 LP 便可以参与 Ronin 原生通证 RON 的流动性挖矿 。在推出后，在 2021 年 12 月 3 日，Ronin 的锁仓资金达到 15 亿美金，创下历史新高。

但在 PoA（Proof-of-Authority）共识下，Ronin 网络相对中心化，Sky Mavis 计划在 Ronin 发展得更为完善与稳定之后，届时可能才会有更多新的游戏。

上文提及，Gala Games 同样正在测试并希望推出一个名叫 Project GYRI 的公链项目。与 Axie Infinity 不同的是，Gala Games 推出的 GYRI 是第一个针对游戏开发人员在游戏中面临的需求的区块链 。按照官方所宣称，「对于“游戏第一，区块链第二”的立场，我们从未动摇过。 」

另外，凭借着 Gala Games 同时推进的八个游戏，届时 Gala Games 的游戏生态必定极具竞争力。量变引发质变，新的挑战者也会从籍籍无名的众多区块链游戏中脱颖而出，挑战 Axie Infinity 的领先地位。

与 Immutable X 的对比

与 Gala Games 所推出的 Project GYRI 类似，Immutable X 是一个专门为游戏和 NFT 提供基础设施服务的 Layer2。

Immutable X 希望着手解决当前以太坊 NFT 生态所面临的主要挑战：

  • 可扩展性；

  • 确认时间长（交易需要几分钟才能确认）；

  • Gas 花费；

  • 流动性（大多数 NFT 从未进行过交易）；

  • 开发者体验（NFT 项目团队将更多的时间放在了技术研发上，而非项目改进）；

  • 环境影响（主流采用的障碍）。

这是以太坊 NFT 市场当下最迫切的需求。Immutable X 将这些需求作为出发点，与 StarkWare 合作，使用 ZK-Rolllup 技术将 Immutable X 构建为一个对 NFT 友好的 Layer2。

虽然 Immutable X 希望承接以太坊的 NFT 生态，但是现实是，只有足够的利益才能吸引更多流动性资金的涌入——尽管基础设施拥有足够的优势。生态的最初发展大多来源于官方基金会的激励，只有更多优秀的区块链游戏构建在 Immutable X 上，Immutable X 才有机会获得更长远的发展。

这也是 Gala Games 与很多区块链基础设施发展思路的不同。Gala Games 更像一个 Web2.5 产品，由中心化实体和去中心化节点组成，以游戏为基础，发展各类满足游戏开发者的工具和基础设施。

相较 Immutable X，我们无法武断地认为两者孰优孰劣，但单从游戏层面来看，Gala Games 的运营和营销思路无疑是成功的。而 Immutable X 在推出基础设施服务后，仍需要时间的沉淀和发展。

风险

第一，与其他 Web3 项目不同，Gala Games 的代码、白皮书均未开源展示 。Gala Games 更像一个 Web2 公司，不透明以及半中心化将对加密投资者的投资热情产生影响。

第二，Gala Games 目前的游戏并不能满足加密玩家的打金需求 ，以 Town Star 为例，相较其他 GameFi 几天、一周，或是一个月的回本周期。据统计，Town Star 回本周期在一年以上。一个小体量的游戏无法满足玩家所需要的沉浸感，这意味着玩家无法长时间将时间花费在一个小游戏中。或许在后面推出的游戏将改变这一点。

第三，Project GYRI 公链是否能够达到游戏开发者的预期，也是一个潜在的风险点。如果 Project GYRI 的表现不尽人意，会让更多的开发者望而却步 ——这对于Gala Games生态的发展是一个严重的威胁。

总结

从本质来看，Gala Games 的成功与创始人丰富的游戏开发和营销经验脱不开关系，其经验在 Gala Games 的节点销售和游戏运营中发挥了巨大的作用 。由 Gala Games 生态游戏吸引过来的用户也将被留存于 Gala Games 的社区和社交媒体中，新游戏上线的短期流量需求也将被快速满足。

而 GALA 代币作为在 Gala Games 生态中的交易媒介，因此消费场景和应用场景丰富，从而也可以获得来自生态用户的有效支撑。Gala Games 生态中的 NFT 也获得了相比其他生态中 NFT 更大的流动性——Gala NFT可以被应用于生态的各类游戏当中。
