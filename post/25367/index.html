NFT的困境：想要改变艺术行业 NFT必须更加安全

2021年，新兴的非同质化代币 (NFT) 市场已经达到了许多引人注目的里程碑，该市场的价值已经从 2020 年第四季度增长了 2100 倍，消费者支出超过 20 亿美元 。尽管媒体的头条新闻一直被 NFT 破纪录的销售额所占据，但新投资者不断增长的需求却常常被忽视。

据追踪 NFT 交易的 NonFungible 数据，今年第一季度有 7.3 万名 NFT 买家和 3.3 万名 NFT 卖家 。尽管这些数字看起来令人印象深刻，但与全球艺术品市场相比，它们实际上相对较小：2018 年，全球艺术品市场的价值为 647 亿美元，其中美国、中国和英国占全球市场的 84%。 

在一个日益网络化和全球化的世界里，由经销商和拍卖行等中间商主导的艺术品市场的传统基础设施已经显得过时了。在这个世界里，新兴市场对艺术品的需求只会不断增长 。当人们回顾 COVID-19 时，可能会认为这场疫情是变革现有艺术市场基础设施的催化剂。与此同时，NFT 市场让我们得以一窥智能合约技术 如何被应用，以确保第三方和中间商能够被移除。 

然而，就目前的情况来看，当前的 NFT 市场基础设施有太多的缺陷和用户错误的可能性 ，使其无法作为当前 (传统艺术品市场的) 验证、分发、拍卖和所有权认证的替代方法。 

今天，没有办法通过查看 NFT 中包含的数据来确定谁是某件艺术品的真正人类创造者 。其结果是越来越多的 NFT 伪造案例 ，其中骗子创建 NFT 并将其呈现为某个特定的知名艺术家的作品。在谷歌网站上快速搜索一下这个话题就会发现，NFT 伪造是一个迅速增长的问题 。在某些情况下，骗子会获取某位艺术家创作的一幅真正艺术作品的图像/照片，将其转化为 NFT，然后假装自己就是该艺术家并出售该 NFT。 

此外，当某个 NFT 代表着重要的相关内容或数据 (如图像) 时，该数据不会存储在区块链上 。相反，该 NFT 会包含一个数据链接，最常见的是通过互联网上的超链接 。如果该超链接末端的数据 (例如图像) 发生了变化或消失，就没有办法从区块链数据中知晓或证明与该 NFT 相关的实际图像是什么。 

因此，没有办法保护 NFT 相关数据的持久性。这是令人震惊的，但实际情况确实如此。这意味着与 NFT 相关的实际图像或数据可能会被更改或删除，从而破坏 NFT 的价值。还有可能出现用户错误 ，即人们错误地复制了长而复杂的地址或遭受中间人攻击，这可能导致数百万美元被发送到错误的地址或永久被盗。 

真实性的验证

 在艺术的现实世界里，艺术家在他们的作品上签名以验证其真实性，而艺术品的所有者则通过将其安全地存放在他们信任的地方来确保其持久性。为了让 NFT 获得长期的成功，区块链技术必须实现类似的功能，并且以一种去中心化的、自主的方式加以实现。 

我们不知道正在发生的 COVID-19 疫情将对艺术界产生什么样的长期影响。人们回头看时可能会发现它是一种催化剂，引发了早就该出现的行业变革，并加剧了对这个本质上仍由高端拍卖行和各种声誉的经销商组成的垄断的竞争。智能合约技术已经展示了 NFT 如何能够消除这些中间人；然而，尽管有明确的需求，但操作风险和潜在的欺诈交易使得目前的 NFT 交易模式风险太大，无法实现规模化交易 。 

防止 NFT 艺术品伪造并永久保护数据，对于区块链生态系统中 NFT 艺术品的使用的持续增长至关重要，以此确保为艺术品买家和卖家建立一个更加公平和透明的系统。未来的艺术生态系统是显而易见的，我们需要做的就是开始构建它。

撰文：Luke Stokes
