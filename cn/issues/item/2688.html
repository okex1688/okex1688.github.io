如何管理子账户功能及常见问题

 交易常见问题    |      2018-12-06
子账户功能
常见问题
子账户功能

什么是子帐户？

对于用户想透过设立多个帐户进行交易且需要在不同帐户间进行权限管理，子帐户可帮助您明确分工，更有效率安全的管理交易。
子帐户功能适用于企业帐户和拥有VIP1及以上等级的个人用户（需要完成KYC个人身份认证和开启币安/谷歌验证）。

子帐户管理有什么功能？

子帐户的创建与管理

操作方法：
1. 鼠标移至【我的帐户】- 点击【子帐户】-【 帐户管理】 - 点击【创建子帐户】。
2. 创建帐户时可使用一般邮箱或虚拟邮箱来创建。
Tips:
  • 当使用真实邮箱创建时需进行邮箱二次验证以确保邮箱地址正确性。
  • 当子帐户使用虚拟邮箱创建时仅可由母帐户通过API进行操作，无法使用虚拟帐户进行登入。
3. 创建完成后即可在帐户管理列表查看该帐户并可在此进行帐户权限管理。

帐户权限管理及操作说明

  • API管理：您可于此创建、编辑子帐户API KEY与API权限。
  • 资产管理：您可以查看主帐户和所有子帐户的余额，可使用比特币或法币表示价值。您只需要点击【划转】就能立即将资产在主帐户和子帐户之间免费划转。
  • 订单管理：您可以查看所有子帐户的“挂单”、“订单记录” 和“交易记录”。您也可以按日期和交易对来筛选订单。
  • 划转历史：选择【转出账户】或【转入账户】即可查看所有帐户的资金转帐记录。
  • 登录历史：您可以按“日期”、“IP 地址”和“位置”来查看所有子帐户登入记录。
1. 所有子帐户都享有和主帐户一样的交易手续费折扣吗？
VIP 等级和相对应的折扣 将取决于所有子帐户和母帐户的交易总量，这样可以让您在交易时享有更有利的费用。
2. 我要如何计算子帐户的交易推荐佣金？
所有子帐户的交易量将计入母帐户，即返佣关系将继承母帐户，子帐户的交易手续费返佣将反佣给母帐户的推荐人帐户。
3. 我的子帐户享有相同的 API 订单限额吗？
是的。各子帐户的 API 订单限额都和母帐户相同，子帐户享有各自独立的api下单限额：
合约订单
每 10 秒 300个订单
每1分钟1200个订单
现货订单
每10秒50个订单
每 24 小时 160,000 个订单
4. 子帐户是否需要二次验证？
仅在第一次登录子帐户时系统将会进行提示，用户需要进行邮箱二次验证，因此建议您将子帐户开启币安/谷歌二次验证功能以确保帐户安全。
5. 子帐户可进行提领资产吗？
不可以。只有母帐户可以进行资产划转管理，子帐户间与第三方帐户不可进行资产划转。
