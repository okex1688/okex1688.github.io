双赢期权

 交易常见问题    |      2020-10-19
幣安期權在app手機端上推出除了看漲期權與看跌期權之外的第三種期權 - 雙贏期權。
目前，幣安期權僅支持在手機端 (App) 上交易，請下載最新版本的手機應用程序。雙贏期權上線版本：蘋果 iOS 2.20.0 安卓 Android 1.31.0
一、雙贏期權是什麼？
雙贏期權也叫多頭跨式期權，是一種中性期權交易策略。系統會自動爲用戶同時買入同一標的物，相同行權價格和到期日期的看跌期權和看漲期權，讓用戶在標的價格上漲或下跌超過權利金的幅度後都能盈利。
用戶無需猜測漲跌方向，只需判斷市場是否即將出現大波動，無論方向是漲或跌，標的物往任意一邊走，當價格波動大於盈虧平衡價格/保本價格（執行價格+/-權利金）即可盈利，虧損時最多隻損失權利金。
但相較於同一行權價格、相同到期時間的看跌期權或看漲期權，雙贏期權的權利金較貴。
雙贏權利金 = (看漲權利金 + 看跌權利金) * 0.95
* 約5% 折扣
例如，如果交易者預期比特幣的價格在減半發生後將產生巨幅波動，就可以買入雙贏期權。在這種情況下，交易者可以7000美元價格同時分別買入1天期的看漲和看跌期權，期權費爲100美元。如果比特幣的價格波動超過了兩個期權的總盈虧平衡價格，交易者即可獲利退出。
使用買入雙贏期權，只要標的資產往有利的方向波動足夠劇烈，收益是無上限的。
雙贏期權的盈虧公式 = Max [ ( 標的價格 - 行權價格) * 合約數量 - 權利金，( 行權價格 - 標的價格) * 合約數量 - 權利金， - 權利金]
二、如何在App上買入雙贏期權？
如果您判斷即將出現大波動，可以在App上買入雙贏期權。
1. 選擇行權的到期時間。到期時間分爲5分鐘、10分鐘、30分鐘、1小時、4小時、8小時、12小時和一天。接下來，選擇訂單類型“雙贏”，並輸入要訂購的數量。幣安期權細則
2. 輸入數量後，點擊跳出下單確認框，確認合約數量，行權價格，權利金 （輸入數量實時計算獲得的權利金，未輸入數量時爲0）和保本價格。
*注意雙贏期權有兩個保本價格，分別對應看漲與看跌期權：行權價格 + 權利金 / 行權價格 - 權利金
3. 您可以選擇設置兩個目標價格（上方/下方），可以獨立設置或不設置，未設置展示“--"。目標價格的設置是按行權價格+或-多少價格(X)，可輸入或點擊加減按鈕進行調整。界面下方會根據您設置的目標價格顯示預期盈虧。
*預期盈虧公式如下：
  • 上方 = Max [ (目標價格 - 行權價格)*合約數量 - 權利金, - 權利金 ]
  • 下方 = Max [ (行權價格) - 目標價格)*合約數量 - 權利金, - 權利金 ]
4. 按鈕點擊 “確認” 後下單買入的雙贏期權。
5. 下單後，您可以查看您的持倉，包括未實現損益、到期時間和損平價格。此外，您的期權也會顯示在k線圖表上， VOL 表示雙贏訂單。
6. 用戶可提前行權或到期自動結算。
1) 提前平倉
由於幣安期權是美式期權，您可在到期日前隨時選擇行權。在“持有倉位”界面，點擊“行權”以實現您的期權實現盈虧。
2) 到期
如果您沒有在到期日之前行權，那麼當您的期權到期時，它將自動行權。
